Create AWT Radio Buttons Using CheckboxGroup Example

In this example, we are going to see how to create AWT radio buttons using CheckboxGroup. The CheckboxGroup class allows the user to select one radio button at a time. When a radio button is selected, it will be deselected any other radio button in the group.

Creation of components

The CheckboxGroup class is used to group radio buttons together. We can create a new RadioButton object by passing a label, a CheckboxGroup and a boolean specifying the state of the checkbox as parameters to its constructor. 

          
           
CheckboxGroup cbg = new CheckboxGroup();
RadioButton rb1 = new RadioButton("Male", cbg, false);
RadioButton rb2 = new RadioButton("Female", cbg, true);

Adding components to the frame

We can add the radio buttons to the Frame using its add method. 

          
           
frame.add(rb1);
frame.add(rb2);

Setting the size and visibility of the frame

We can set the size of the frame by calling the setSize method and make the frame visible to the user by calling the setVisible method. 

          
           
frame.setSize(400, 400);
frame.setVisible(true);

Example Program

Here is the complete example program to create AWT radio buttons using CheckboxGroup. 

          
           
import java.awt.*;  
class MyRadioButton{  
     MyRadioButton(){  
        Frame f= new Frame("Radio Button");  
        CheckboxGroup cbg = new CheckboxGroup();  
        Checkbox radioButton1 = new Checkbox("A)",cbg,false);    
        radioButton1.setBounds(100,100, 50,50);  
        Checkbox radioButton2 = new Checkbox("B)",cbg,true);    
        radioButton2.setBounds(100,150, 50,50);  
        Checkbox radioButton3=new Checkbox("C)",cbg,false);  
        radioButton3.setBounds(100,200, 50,50);    
        f.add(radioButton1); f.add(radioButton2); f.add(radioButton3);    
        f.setSize(400,400);  
        f.setLayout(null);  
        f.setVisible(true);  
    }  
public static void main(String args[]){  
    new MyRadioButton();  
    }}
