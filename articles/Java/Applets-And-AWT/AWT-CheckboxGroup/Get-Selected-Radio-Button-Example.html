Java: Get Selected Radio Button Example

Introduction

Java is a powerful programming language used to create a wide range of applications. This tutorial will demonstrate an example on how to get the selected radio button from a group of radio buttons in a Java application.

Radio Button Example

The following example creates a group of radio buttons, and then retrieves the selected radio button from the group. 

          
           
JRadioButton radioButton1 = new JRadioButton("Option 1");
	JRadioButton radioButton2 = new JRadioButton("Option 2");
	JRadioButton radioButton3 = new JRadioButton("Option 3");

	ButtonGroup bg = new ButtonGroup();
	bg.add(radioButton1);
	bg.add(radioButton2);
	bg.add(radioButton3);

	// iterate over the radio buttons
	for (Enumeration<AbstractButton> buttons = bg.getElements(); buttons.hasMoreElements();) {
	    AbstractButton button = buttons.nextElement();

	    if (button.isSelected()) {
	        System.out.println("Selected: " + button.getText());
	    }
	}
This example prints the text of the selected radio button. In this case, it would be “Option [X]”, where [X] can be either 1, 2 or 3.

Conclusion

In this tutorial we demonstrated an example on how to get the selected radio button from a group of radio buttons in a Java application. We saw how to iterate over the radio buttons and retrieve the text of the selected radio button.
