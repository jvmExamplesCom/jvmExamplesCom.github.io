Set Selected Radio Button Example in Java

Java is a popular programming language due to its scalability, portability and many other features. In this article, we will take a look at how to set a selected radio button example in Java. We will be using the following basic HTML code for our example: 

          
           
<form name="myForm">
  <input type="radio" name="gender" value="male"/> Male
  <input type="radio" name="gender" value="female"/> Female
  <input type="radio" name="gender" value="other"/> Other
</form>
To set the default selection of a radio button in Java, we need to use the method “setSelected()” which takes a boolean argument. The following code snippet shows how to set the male option as the default selection: 

          
           
<script>
RadioButton maleRadioButton = (RadioButton) findViewById(R.id.button_male); 
maleRadioButton.setSelected(true); 
</script>
The output of this would be the Male option selected by default upon loading the page.

Dynamically Setting the Selected Radio Button

In some cases, the selected radio button might need to be changed dynamically based on the user input or other factors. To do this, we can use the following code snippet: 

          
           
<script>
String gender = "female"; // This can be retrieved from the DB or user input
String radioButtonId = gender + "_rb";
RadioButton genderRadioButton = (RadioButton) findViewById(R.id.radioButtonId); 
genderRadioButton.setSelected(true); 
</script>
In the above code snippet, we are retrieving the gender of the user from the database or user input. We then create a string that contains the ‘radioButtonId’ of the corresponding gender. We then use this id to locate the RadioButton view and call the setSelected() method on it. Finally, if we want to reset all the radio buttons in the form we can use the following code snippet: 

          
           
<script>
RadioGroup radioGroup = (RadioGroup) findViewById(R.id.radioGroupId); 
radioGroup.clearCheck(); 
</script>
This will clear all the selections made to the radio buttons in the form.
