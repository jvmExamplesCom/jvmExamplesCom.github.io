Get Red, Green, and Blue (RGB) Components From Color Example

Java provides several ways to retrieve the Red, Green, and Blue (RGB) components of a given color. The following examples demonstrate various methods for obtaining such information.

Using Color.getRGB()

The easiest way to obtain the RGB components from a Color instance is via the getRGB() method: 

          
           
Color c = Color.RED;
int red = c.getRed();
int green = c.getGreen();
int blue = c.getBlue();

Using Color.getComponents()

The getComponents() method of Color provides the rgb components in the form of an array of floats, with the index order being Red first, followed by Green, and Blue last: 

          
           
float [] rgb = c.getComponents(null);
int red = (int)rgb[0];
int green = (int)rgb[1];
int blue = (int)rgb[2];

Using Color.getHSBColor()

The getHSBColor() method of the class Color can be used to retrieve the color's Hue, Saturation, and Brightness components. These components can then be used to obtain the equivalent RGB values: 

          
           
float [] hsb = Color.RGBtoHSB(c.getRed(),

int green = (int)(hsb[1] * 255);
int blue = (int)(hsb[2] * 255);

Using Bitwise Operators

Bitwise operators can also be used to extract the RGB components from a Color instance: 

          
           
int colorInt = c.getRGB();
