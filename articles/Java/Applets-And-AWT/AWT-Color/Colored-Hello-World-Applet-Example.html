Colored Hello World Applet Example

In this tutorial, we will learn how to create a very simple Hello World applet in Java. An applet is a graphical program that runs inside a web browser, and it could be written in different languages.

Let's first have a look at the code: 

          
           
import java.awt.*;

public class HelloWorld extends java.applet.Applet {

    public void paint(Graphics g) {
        g.setColor(Color.red);
        g.drawString("Hello World!", 25, 50); 
    }

}

This code extends the java.applet.Applet to create the HelloWorld applet. We need the Graphics class to draw the String, and the Color class to set the color of the text.

The paint method is where we do the actual drawing. It first sets the color as red by using the setColor method, and then draws the String "Hello World!" with the drawString method. The drawString method takes two parameters: the String to draw and the x and y coordinates of the text.

To compile the code, we can use the following command:


          
           
javac HelloWorld.java

Once compiled, we can run the applet in our web browser by using the following HTML code:


          
           
<html>
    <head>
       <title>Hello World Applet</title>
    </head>
    <body>
       <applet code="HelloWorld.class" width="300" height="200"></applet>
    </body>
</html>
