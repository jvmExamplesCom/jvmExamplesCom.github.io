Set Drawing Mode to XOR Example

The XOR drawing mode is a powerful graphics feature that enables graphical applications to draw shapes on the screen, while also making them easily removable. In this tutorial we'll explore how to use XOR drawing mode in Java programs. Before we dive into the examples, let's review the basics of XOR drawing mode. XOR drawing mode is a type of bit-operation. It works by taking two values, one from the first shape (the 'background') and one from the second shape (the 'foreground'), and performing a logical XOR operation on them. Then it assigns the result to the output color. What this means is that when you draw a shape onto a background that already contains a shape, the result will be a third unique shape. This can be used to create special effects like removals and edits, as the previous shapes are still underneath the new shape, just blended together.

Using XOR Drawing Mode in Java

In Java, you can use the Graphics2D class to enable XOR drawing mode. This requires calling two functions: setXORMode() and setComposite(). Here is an example: 

          
           
Graphics2D g = (Graphics2D)graphics;
g.setXORMode(Color.white);
g.setComposite(AlphaComposite.getInstance(AlphaComposite.SRC_OVER, 0.5F));
This code sets the XOR drawing mode with a white background color. In addition, it also sets the transparency of the resulting shape to 0.5F. You can adjust this value according to your needs.

A Simple XOR Drawing Mode Example

Now let's look at a simple example that demonstrates how to use XOR drawing mode in Java. For this example, we'll create two shapes: a square and a circle. The result will be a combination of the two shapes, thanks to the XOR drawing mode. Here is the code: 

          
           
Graphics2D g = (Graphics2D)graphics;
g.setXORMode(Color.white);
g.setComposite(AlphaComposite.getInstance(AlphaComposite.SRC_OVER, 0.5F));

// Draw a square
g.drawRect(100,100,100,100);

// Draw a circle

If you run this code, you should see a combined shape of a square and a circle being drawn on the screen.
