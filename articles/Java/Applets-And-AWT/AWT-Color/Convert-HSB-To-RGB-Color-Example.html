Convert HSB To RGB Color Example in Java

Java is a versatile language that allows developers to convert colors from one color system to another. In this example, we'll be taking a look at converting an HSB color (Hue, Saturation, and Brightness) to it's equivalent RGB value.

HSB vs RGB Colors

Before getting into the code example, it's helpful to understand the difference between these two color types. RGB (Red, Green, Blue) colors are additive colors; meaning they are created when light is mixed together in various amounts. It is generally used for displaying text and images on a computer monitor. HSB (Hue, Saturation, and Brightness) are different types of colors that are used to represent colors in a more intuitive way. Hue is the ‘color’ attribute of a color, saturation is its ‘intensity’ and brightness is its ‘lightness’.

Example Code

Now, let's take a look at the code. We'll start with creating an HSB color object using the static Color.getHSBColor() method: 

          
           
Color hsbColor = Color.getHSBColor(h, s, b);
In the above code, parameters 'h', 's' and 'b' represent the Hue, Saturation and Brightness values respectively. We can now use the method getRGB() to convert this HSB color to its equivalent RGB value. The getRGB() method takes no arguments and returns an int value containing the RGB levels of the HSB color. 

          
           
int rgbLevels = hsbColor.getRGB();
Finally, we can pass this int value to the static Color.decode() method which will convert it to an RGB color object. 

          
           
Color rgbColor = Color.decode(rgbLevels);

Example Output

Now, let's take a look at a short example showing how this code can be used. First, let's create an HSB color object with some arbitrary values: 

          
           
Color hsbColor = Color.getHSBColor(0.2f, 0.8f, 0.5f);
Next, we can get the equivalent RGB values by calling the getRGB() method. 

          
           
int rgbLevels = hsbColor.getRGB();
Finally, we can convert this int value to an RGB color object. 

          
           
Color rgbColor = Color.decode(rgbLevels);
The output of this code is the RGB color object with the following RGB values: (170, 191, 128).

Conclusion

In this example, we looked at how to convert an HSB color to its equivalent RGB value using the Color class in Java. We saw how to create an HSB color object; how to get the equivalent RGB values; and how to convert these values to an RGB color object.
