Create Custom Color Using HSB Example

Java lets developers create custom colours with the HSB (hue, saturation, brightness) model. In this article, we will look at some examples of how to create a custom colour using the HSB model in Java.

Creating a Custom Colour from HSB Values

The simplest way to create a custom colour from HSB values is to use the Color.getHSBColor(float, float, float) method. This method returns a Color object given three floats for the hue, saturation and brightness. For example, to create a deep blue colour from HSB values, we could use the following code: 

          
           
Color deepBlue = Color.getHSBColor(0.60f, 0.90f, 0.50f);

Creating a Custom Color from HSB Integer Values

If you need to create a custom colour from HSB integer values, you can use the Color.getHSBColor(int, int, int) method. This method takes three int values ranging from 0 to 255 as parameters and returns a Color object. For example, to create a pastel green colour from HSB integer values, we could use the following code: 

          
           
Color pastelGreen = Color.getHSBColor(120, 25, 100);

Converting HSB Values to RGB Values

The Color.HSBtoRGB(float[]) method can be used to convert HSB values to RGB values. This method takes a float array with three elements for the hue, saturation and brightness, and returns an integer that represents a colour in the RGB color model. For example, to convert the HSB values for the deep blue colour mentioned above to RGB values, we could use the following code: 

          
           
float[] hsbValues = new float[]{0.60f, 0.90f, 0.50f};
int rgbValue = Color.HSBtoRGB(hsbValues);

Using the Color Class Constructors

In addition to the methods described above, we can also use the Color class constructors to create custom colour with the HSB model. The Color(int, boolean) constructor takes an int parameter and a boolean parameter, and returns a Color object. The int parameter specifies a colour in the HSB color space, and the boolean parameter determines whether the resulting color is opaque or transparent. For example, to create a light pink colour with the HSB model, we could use the following code: 

          
           
Color lightPink = new Color(0xFF7F90, true);
We can also use the Color(float, float, float) constructor to create a custom colour from HSB values. This constructor takes three float parameters for the hue, saturation and brightness and returns a Color object. For example, to create a golden yellow colour with the HSB model, we could use the following code: 

          
           
Color goldenYellow = new Color(0.15f, 0.75f, 1.00f);

Conclusion

In this article, we looked at some examples of how to create a custom colour using the HSB model in Java. We looked at several methods including the Color.getHSBColor(float, float, float) and Color(int, boolean) methods.
