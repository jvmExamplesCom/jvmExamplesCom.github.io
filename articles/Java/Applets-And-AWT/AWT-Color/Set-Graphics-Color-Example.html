Java: Set Color Graphics Example

Setting graphics color with Java is a task that requires a few lines of code. The following example will demonstrate how to set graphics color in Java.

Prerequisite

Before diving into the example, it is important to have the basic Java knowledge, such as initializing and working with objects, calling instance methods, calling static methods, etc.

Example

In this example, we will create a class called 
MyColor
, which will contain a method to set the graphics color. 

          
           
public class MyColor {

	public static void setColor(Graphics g, Color c) {
		g.setColor(c);
	}

}
In the example above, we create a class 
MyColor
, which contain two static methods: 
setColor()
, which takes two arguments, a 
Graphics
object and a 
Color
object. Inside the 
setColor()
method, we simply call the 
setColor()
on the 
Graphics
object and pass in the 
Color
object.

We can now use this method to set the color of any 

Graphics

MyColor.setColor(g, Color.RED);
