Create Custom Color Using RGB Example

Custom colors can be created in a Java program using RGB (Red Green Blue) values. This allows you to retrieve and modify the exact color you need. The following example demonstrates how to create custom colors using RGB and HTML code.

Java Code

The following code shows how to create custom colors using RGB values: 

          
           
public static Color customColor(int r, int g, int b){
    return new Color(r,g,b);
}
In the above code, the “r”, “g”, and “b” parameters refer to the red, green and blue values respectively. These values must range from 0 to 255.

HTML Code

To retrieve and display the custom color in an HTML document, you must use the following code: 

          
           
<div style="background-color:rgb(r, g, b)"></div>
For example, if you wanted to create a custom color with red = 100, green = 75 and blue = 50, you could use the following code: 

          
           
<div style="background-color:rgb(100,75,50)"></div>
