Java: Get Checkbox State Example

In this article, we will learn how to use the Java programming language to get the state of a checkbox. We'll look at the different ways in which this can be achieved and provide code examples for each one. Firstly, we need to create a simple form containing a checkbox. We'll use the HTML element . In order to access the value of the checkbox from a Java program, we need to give it a name attribute. 

          
           
<form>
  <label for="cb">Do you want to receive email notifications?</label>
  <input type="checkbox" name="cb" />
</form>
The next step is to write some Java code to get the state of the checkbox. We can do this using either the Java Servlet API or the JavaServer Faces API. The code example below demonstrates both of these methods. 

          
           
// Using the Java Servlet API
String cbValue = request.getParameter("cb");
boolean checked = (cbValue != null &amp;&amp; cbValue.equals("on"));

// Using the JavaServer Faces API
FacesContext context = FacesContext.getCurrentInstance();
UIComponent component = context.getViewRoot().findComponent("cb");
boolean checked = false;
if (component instanceof HtmlInputCheckbox) {
  checked = ((HtmlInputCheckbox) component).isSelected();
}
If the checkbox is selected, the variable "checked" will be set to true. Otherwise, it will be set to false. We can also use the JSP Expression Language (EL) to access the value of the checkbox. This requires us to set the "name" attribute of the checkbox to a variable in the JSP page. 

          
           
<form>
  <label for="cb">Do you want to receive email notifications?</label>
  <input type="checkbox" name="${user.notificationsEnabled}" />
</form>
And then we can use the EL expression ${user.notificationsEnabled} to access the value of the checkbox from the JSP page. 

          
           
boolean checked = ${user.notificationsEnabled};
Finally, if you are using the JavaServer Pages Standard Tag Library (JSTL), you can use the tag to test the state of the checkbox. 

           
            
<c:if test="${user.notificationsEnabled}">
  The checkbox is selected
</c:if>
