Get Available Font Objects Example with Java

In this article, we will discuss how to get a list of the available font objects for use with the Java programming language. This example uses the GraphicsEnvironment class’s getAvailableFontFamilyNames() method to return an array of strings containing the names of all fonts available on the system. We will also demonstrate how to print these font names using a simple loop and a System.out.println() call.

Example: Get Available Font Objects in Java

Let's look at a code example of how to get a list of the available font objects for use with the Java programming language. 

          
           
GraphicsEnvironment ge = GraphicsEnvironment.getLocalGraphicsEnvironment();
String[] fontNames = ge.getAvailableFontFamilyNames();
for (String fontName : fontNames) {
    System.out.println(fontName);
}
The code above creates an instance of GraphicsEnvironment called ge and then calls the getAvailableFontFamilyNames() method to get a list of all available font family names. We then loop through the array of strings named fontNames and print each one to the console using a System.out.println() call.
