Get Font Information Example in Java

Java provides multiple APIs to get font information. This article discusses each of the available options and provides a code example.

Using the GraphicsEnvironment Class

The 

          
           
GraphicsEnvironment
class provides information about the available graphics devices, such as the fonts installed on the system. The following code example uses the GraphicsEnvironment class to list the available fonts on the system: 

          
           
import java.awt.GraphicsEnvironment;

public class GetFontsExample {
  public static void main(String[] args) {
    // Get the list of available fonts
    String[] fontNames =
        GraphicsEnvironment.getLocalGraphicsEnvironment().getAvailableFontFamilyNames();
    System.out.print("Available Fonts: \n");
    for (String font : fontNames) {
      System.out.println(font);
    }
  }
}
Output: 

          
           
Available Fonts:
Arial
Times New Roman
Verdana
Comic Sans MS
...

Using the Font Class

The 

          
           
Font
class is used to create and manipulate different fonts in Java. It provides methods to obtain the attributes of a font, such as its name, size, style, and so on. The following code example uses the Font class to get font information: 

          
           
import java.awt.Font;

public class GetFontInfoExample {
  public static void main(String[] args) {
    // Create a font
    Font font = new Font("SansSerif", Font.PLAIN, 12);

    // Get font attributes
    String fontName = font.getName();
    int fontStyle = font.getStyle();
    int fontSize = font.getSize();

    // Print font attributes
    System.out.println("Font Name : " + fontName);
    System.out.println("Font Style : " + fontStyle);
    System.out.println("Font Size : " + fontSize);
  }
}
Output: 

          
           
Font Name : SansSerif
Font Style : 0
Font Size : 12
