Get Available Font Family Names Example

Introduction

Java has a powerful library of APIs to help developers access a variety of fonts. In this article, we will explore an example of how to use the FontFamily.getAvailableFontFamilyNames() method to get a list of available font families in Java.

Prerequisites

Before proceeding with this example, you should have a basic understanding of the Java programming language and working with Fonts in Java.

Get the List of Available Font Families in Java

The basic syntax for calling the FontFamily.getAvailableFontFamilyNames() method is as follows: 

          
           
FontFamily.getAvailableFontFamilyNames();
This method returns an array of strings containing the names of all the available font families.

Example: Get Available Font Family Names

Let's create a simple Java program to demonstrate how to use the FontFamily.getAvailableFontFamilyNames() method: 

          
           
import java.awt.*;

public class FontFamilyNameExample {

    public static void main(String[] args) {
        // Get the font family names
        String[] fontFamilies = GraphicsEnvironment
                .getLocalGraphicsEnvironment()
                .getAvailableFontFamilyNames();
                
        for (String fontFamily : fontFamilies) {
            System.out.println(fontFamily);
        }
    }
}
In this example, we imported the java.awt package, which contains the classes for dealing with graphics in Java. We then used the GraphicsEnvironment.getLocalGraphicsEnvironment().getAvailableFontFamilyNames() method to retrieve an array of strings containing the names of all the available font families. Finally, we iterated through the array and printed out each font family name.

Output

When we run the above program, the output will be a list of all the available font families on our system. Here is an example of the output: 

          
           
Arial
Courier New
Times New Roman
Verdana
...
