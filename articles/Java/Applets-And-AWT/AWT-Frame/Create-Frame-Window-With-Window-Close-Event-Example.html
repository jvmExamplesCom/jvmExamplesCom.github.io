Create Frame Window With Window Close Event Example

Java is a powerful programming language that allows developers to create graphical user interfaces. One of the most common examples is creating a frame window with a window close event. In this article, we'll take a look at how you can do this in Java.

The Code

Here's the code you'll need to create a frame window with a window close event.


          
           
import javax.swing.*;
import java.awt.*;

public class CloseEventWindow {

  public static void main(String[] args) {
    Frame frame = new Frame("Close event test");
    frame.setSize(400, 300);

// Create a window close listener 
frame.addWindowListener(new WindowAdapter() {
    public void windowClosing(WindowEvent we) {
        System.out.println("Window closed");
        System.exit(0);
      }
        });

// Show the frame
frame.setVisible(true);
  }
}

Finally, this line sets the frame to be visible on the screen.
