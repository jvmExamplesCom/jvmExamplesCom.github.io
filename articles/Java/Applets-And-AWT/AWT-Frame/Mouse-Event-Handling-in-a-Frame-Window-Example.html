Mouse Event Handling in a Frame Window Example

Java provides a platform-independent way to detect and respond to mouse events. Every GUI component on the screen is actually a class that extends the abstract Component class. Java toolkits, such as the AWT (Abstract Window Toolkit), provide classes that extend Component and add their own functionality. The AWT contains a base class called Frame, which is used to create windows.

Example

This example creates a window with a single button and responds to mouse events. This example uses the Listener interface, which allows you to handle events in an object-oriented fashion. 


          
           
import java.awt.*;
import java.awt.event.*;

public class MouseEventDemo extends Frame {

    public static void main(String args[]){
        Frame frame = new MouseEventDemo();
        frame.setSize(200, 200);
        frame.setVisible(true);
    }

    // Constructor
    public MouseEventDemo() {

        // Create the button
        Button btn = new Button("Press me!");

        // Add the button to the frame
        this.add(btn);

        // Create and register a listener
        btn.addActionListener(new ActionListener(){
            public void actionPerformed(ActionEvent e){
                System.out.println("Button pressed!");
            }
        });
    }
}

This example creates a window with a single button. When the button is clicked, an event is generated. In this case, the event is an ActionEvent. The application listens for this event and responds when it is triggered. This is done by creating an instance of the Listener interface and registering it with the button.

The Listener class has a single method: actionPerformed(). This method is called when an ActionEvent is triggered. In this example, it simply prints a message to the console.

Output

When the application is run, the window appears on the screen. When the button is clicked, a message is printed to the console. 


          
           
Button pressed!
