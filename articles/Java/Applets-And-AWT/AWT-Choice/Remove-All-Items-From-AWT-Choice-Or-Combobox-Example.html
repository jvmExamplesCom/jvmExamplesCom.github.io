Remove All Items From AWT Choice Or Combobox Example In Java

In this article, we will discuss how to remove all items from an AWT Choice or Combobox example in Java.

Example 1: Remove All Items From AWT Choice

The following example demonstrates how to remove all items from an AWT Choice using the removeAll() method. 

          
           
// create a new AWT Choice 
Choice choice = new Choice();
 
// add some items 
choice.add("Apple");
choice.add("Mango");
choice.add("Orange");
 
// remove all items 
choice.removeAll();
In the above example, we have created an AWT Choice and have added some items such as Apple, Mango, and Orange. Finally, we have used the removeAll() method to remove all the items from the AWT Choice.

Example 2: Remove All Items From AWT Combobox

The following example demonstrates how to remove all items from an AWT Combobox using the removeAllItems() method. 

          
           
// create a new AWT Combobox 
ComboBox comboBox = new ComboBox();
 
// add some items 
comboBox.addItem("Apple");
comboBox.addItem("Mango");
comboBox.addItem("Orange");
 
// remove all items 
comboBox.removeAllItems();
In the above example, we have created an AWT Combobox and have added some items such as Apple, Mango, and Orange. Finally, we have used the removeAllItems() method to remove all the items from the AWT Combobox.
