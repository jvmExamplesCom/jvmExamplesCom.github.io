Count Number Of Items in a AWT Choice or Combobox Example

The AWT class Choice is a subclass of Component and an implemented interface ItemSelectable. It provides the user a list of choices, from which the user can choose only one item at a time. The Choice class also defines two methods for retrieving the number of items in the list. In this article, we will explore some code examples to count the number of items in a AWT Choice or Combobox.

Example 1: Counting the Number of Items in a Choice

In this example, we create a Choice object and add some items to it, then use the getItemCount() method to print the total number of items.


          
           
import java.awt.*;

public class ChoiceExample {
    public static void main(String[] args) {
        Frame frame = new Frame();
        Choice choice = new Choice();

        // Add items to the Choice
        choice.add("Item 1");
        choice.add("Item 2");
        choice.add("Item 3");
        choice.add("Item 4");
        choice.add("Item 5");

        // Get the total number of items in the Choice
        int numItems = choice.getItemCount();
        System.out.println("Number of items in the Choice: " + numItems);

        // Add Choice to the frame
        frame.add(choice); 
        frame.pack();
        frame.setVisible(true);
    }
}
Output: 

          
           
Number of items in the Choice: 5

Example 2: Counting the Number of Selected Items in a Choice

The Choice class also provides getSelectedIndexes() method which returns an array of indexes corresponding to the selected items. We can use this method to count the number of selected items in a Choice.


          
           
import java.awt.*;

public class ChoiceExample {
    public static void main(String[] args) {
        Frame frame = new Frame();
        Choice choice = new Choice();

        // Add items to the Choice
        choice.add("Item 1");
        choice.add("Item 2");
        choice.add("Item 3");
        choice.add("Item 4");
        choice.add("Item 5");

        // Select multiple items
        choice.select(0);
        choice.select(3);

        // Get selected indexes
        int[] selection = choice.getSelectedIndexes();

        // Print the total number of selected items
        System.out.println("Number of selected items: " + selection.length);

        // Add Choice to the frame
        frame.add(choice); 
        frame.pack();
        frame.setVisible(true);
    }
}
Output: 

          
           
Number of selected items: 2
