Create AWT Choice Or Combobox Example

Creating a Choice or ComboBox in Java is simple with the help of the Abstract Window Toolkit (AWT). This tutorial will provide an example on how to create and use such components in Java.

Creating an AWT Choice or ComboBox

To create a Choice or ComboBox in Java, you will need to first import the components you need from the java.awt package. 

          
           
import java.awt.*;
Next, declare a Choice or ComboBox variable. For example: 

          
           
Choice choice;
ComboBox comboBox;
You will then create an object of this variable, set its size, and then add items or choices to it. Items can be added using strings, as shown below. 

          
           
choice = new Choice();
choice.setSize(150, 150);
choice.add("Item 1");
choice.add("Item 2");
choice.add("Item 3");
Or if you want to add choices with specific values, you can do so by using the addItem method. 

          
           
comboBox = new ComboBox();
comboBox.setSize(150, 150);
comboBox.addItem("Item 1", "Value 1");
comboBox.addItem("Item 2", "Value 2");
comboBox.addItem("Item 3", "Value 3");
Finally, you can set a default item or value as the selected choice. 

          
           
choice.select("Item 1");
comboBox.setSelected("Value 1");

Using an AWT Choice or ComboBox

You can use the Choice or ComboBox to allow the user to select an item or value. You can retrieve the selected value using the getSelectedItem or getSelectedItem methods. 

          
           
String selectedItem = choice.getSelectedItem();
String selectedValue = comboBox.getSelectedItem();
You can also add an action listener that will be executed when the user selects an item or value. 

          
           
choice.addActionListener(e -> {
    // Do something when an item is selected
});

Example

The following is a complete example that demonstrates how to create and use an AWT Choice or ComboBox. 

          
           
import java.awt.*;

public class ChoiceOrComboBoxExample extends Frame {

    public ChoiceOrComboBoxExample() {
        Choice choice = new Choice();
        choice.setSize(150, 150);
        choice.add("Item 1");
        choice.add("Item 2");
        choice.add("Item 3");
        choice.select("Item 1");
        this.add(choice);

        ComboBox comboBox = new ComboBox();
        comboBox.setSize(150, 150);
        comboBox.addItem("Item 1", "Value 1");
        comboBox.addItem("Item 2", "Value 2");
        comboBox.addItem("Item 3", "Value 3");
        comboBox.setSelected("Value 1");
        this.add(comboBox);

        this.setSize(400, 400);
        this.setVisible(true);

        choice.addActionListener(e -> {
            System.out.println("Selected Item: " + choice.getSelectedItem());
        });

        comboBox.addActionListener(e -> {
            System.out.println("Selected Value: " + comboBox.getSelectedItem());
        });
    }

    public static void main(String[] args) {
        new ChoiceOrComboBoxExample();
    }
}
