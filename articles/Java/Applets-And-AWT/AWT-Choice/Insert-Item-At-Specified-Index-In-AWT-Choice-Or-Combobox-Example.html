Insert Item At Specified Index In AWT Choice Or Combobox Example

Java AWT (Abstract Windows Toolkit) library enables developers to build Graphical User Interfaces. One of the components offered by AWT is the Choice element, popularly known as Combobox. It is a dropdown list for selection between multiple items. This article provides an example of how to insert Choice elements at a specified index in AWT Choice or Combobox.

Example Code

The following code snippet demonstrates inserting Choice elements at a specified index in AWT Choice or Combobox. 

          
           
public class insertChoiceAtIndexExample { 
 
    public static void main(String[] args) {
        
        //create a frame
        Frame frame = new Frame("AWT Insert Choice Example");
       
        //create a choice box
        Choice cb = new Choice();
       
        //add items to the choice box
        cb.add("Item 1");
        cb.add("Item 2");
        cb.add("Item 3");
       
        //insert item 4 at index 1
        cb.insert("Item 4", 1);
       
        //print the items of the choice box
        for (int i = 0; i < cb.getItemCount(); i++)
            System.out.println(cb.getItem(i));
    }
}

Output

When the above code is executed, it produces the following output. 

          
           
Item 1
Item 4
Item 2
Item 3
