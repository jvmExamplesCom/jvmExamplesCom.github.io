Create AWT Label Example With Java

Java supports GUI designing through various components and packages such as awt, swing etc. The AWT label component is used to create labels in a Java application. It is an abstract class which extends Component class and implements the Accessible interface. This tutorial will demonstrate how to create an AWT Label example in Java.

Create AWT Label Example in Java

We can use the Label class of the java awt package to create labels. Let's create an AWT Label example program in Java. 

          
           
import java.awt.*; 
import javax.swing.*; 
  
public class AWTLabelExample { 
    public AWTLabelExample() { 
        // Create a frame 
        JFrame frame = new JFrame("AWT Label Example"); 
  
        // Set FlowLayout as layout manager 
        frame.setLayout(new FlowLayout()); 
  
        // Set size of the frame 
        frame.setSize(400, 150); 
  
        // Create a label 
        Label label = new Label("Welcome to TutorialsPoint"); 
  
        // Add label to frame 
        frame.add(label); 
  
        // Set the frame visible 
        frame.setVisible(true); 
    } 
  
    public static void main(String[] args) { 
        new AWTLabelExample(); 
    } 
}
The output of the above program will be a window with a label displaying the text “Welcome to TutorialsPoint” as shown below: In this program, we have imported all the classes from 'java.awt' and 'javax.swing' packages and then defined class AWTLabelExample. In the constructor of the class, a new frame has been created. A flow layout has been set for the frame and its size has been set using the setSize() method. After that, a new label has been created using the Label class and added to the frame. Finally, the frame is made visible by the setVisible() method.
