Change Text Font Of Label Example In Java

This tutorial will provide a code example to change text font of label in Java. We will use the JavaFX Label class to create and add labels to the application window. We will also use the setFont() method to set a font on the label.

JavaFX Labels

Labels are used to display text, images and icon in a GUI application. The JavaFX Label class has the following declared constructors: 

          
           
Label()
Label(String text)
Label(String text, Node graphic)
The first constructor creates an empty label. The second constructor creates a label with text. The third constructor creates a label with text and an image.

Set Font On Label

We can set a font on a label using the 

          
           
setFont()
method. This method takes a Font object as parameter and sets that font as a label's font. Let's see an example. 

          
           
import javafx.application.Application;
import javafx.scene.Scene;
import javafx.scene.control.Label;
import javafx.scene.text.Font;
import javafx.scene.layout.StackPane;
import javafx.stage.Stage;
 
public class ChangeFontExample extends Application {
   
   public static void main(String[] args){
      launch(args);
   }
 
   @Override
   public void start(Stage stage){
      //create label
      Label greeting = new Label("Hello World!");
      
      //set font on label
      greeting.setFont(Font.font("Verdana", 15));
      
      //add label to StackPane 
      StackPane root = new StackPane();
      root.getChildren().add(greeting);

      //create scene and add to stage
      Scene scene =  new Scene(root, 500, 200);
      stage.setScene(scene);
      stage.show();
   }
}
In this code example, we have created a label with the text "Hello World!". Then we set the font on the label using the 

          
           
setFont()
method. We pass two arguments to the setFont() method, a Font object and the font size. Finally, we add the label to a StackPane layout and set it on the scene. Running this program will produce the output like following: This is how we can set the font on labels in JavaFX applications.
