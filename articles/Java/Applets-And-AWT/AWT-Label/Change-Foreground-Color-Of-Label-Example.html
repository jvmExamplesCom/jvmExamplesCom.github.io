Change Foreground Color Of Label Example In Java

This example explains how to change the foreground color of a label in Java. A label is a component for displaying text in Swing programming. It has various properties such as background color, foreground color, font settings and so on. We can change all these properties as per our need. In this example, we will see how to change the foreground color of a label. This can be done by setting the new color in the setForeground() method of the Label class.

Example

To demonstrate the example, first we create a label instance and set some text on it: 

          
           
Label label = new Label("Demo Label");
Then create a Color object to define the new foreground color: 

          
           
Color color = new Color(255, 0, 0);
Finally set this color on the label using the setForeground() method: 

          
           
label.setForeground(color);
That's it! This will set the desired color for the label.

Output

After executing the program, you will see the following output. The label text will be displayed with the new foreground color that we set:
