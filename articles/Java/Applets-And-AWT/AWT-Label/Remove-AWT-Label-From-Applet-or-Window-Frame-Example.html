Removing an AWT Label From an Applet or Window Frame Example

In this article, we will learn how to remove an AWT label from an applet or window frame. We will look at a code example of the process, and then go through the steps one by one. By the end of this article, you should be able to remove an AWT label from an applet or window frame.

Code Example

The following is a code example of removing an AWT label from an applet or window frame:


          
           
import java.awt.Frame; 

// Create the frame
Frame frame = new Frame();

// Create the AWT label
Label label = new Label("This is a label"); 

// Add the label to the panel
frame.add(label);

// Remove the label from panel
frame.remove(label);

    frame.add(label);
