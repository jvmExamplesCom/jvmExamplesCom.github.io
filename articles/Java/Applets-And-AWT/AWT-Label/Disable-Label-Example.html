Disable Label Example with Java

The following example shows how to create a basic label that can be disabled. A disable label is useful when the text in the label needs to remain viewable, but not selectable or clickable. To create a disable label, we use the 

          
           
setEnabled(false)
method for the 

          
           
JLabel
class. 

          
           
import javax.swing.*;

public class DisableLabelExample {
    public static void main(String[] args) {
        // Create label
        JLabel label = new JLabel("This is a disabled label.");

        // Set label to be disabled
        label.setEnabled(false);

        // Create a frame to display label
        JFrame frame = new JFrame("Disable Label Example");

        // Add label to frame
        frame.add(label);

        // Set size and visibility of frame
        frame.setSize(300, 200);
        frame.setVisible(true);
    }
}
When the code above is compiled and executed, it produces the following result: 

          
           
This is a disabled label.

Disable Button Example with Java

In the following example, we show how to create a basic button that can be disabled. A disabled button is useful when the user needs to be able to view the text of the button, but not be able to interact with it. To create a disabled button, we use the 

          
           
setEnabled(false)
method for the 

          
           
JButton
class. 

          
           
import javax.swing.*;

public class DisableButtonExample {
    public static void main(String[] args) {

        // Add button to frame
        frame.add(button);

        // Set size and visibility of frame
        frame.setSize(300, 200);

    }
}
