How to Change Background Color Of Label in Java

Changing the background color of a label is one of the basic operations that every Java programmer needs to learn. In this tutorial, we will learn how to change the background color of a label in Java.

Prerequisite

Before working with this example, you should have basic knowledge of Java programming language and Swing library. Also, we are using Eclipse IDE for development.

Example â€“ Change Background Color of Label in Java

To change the background color of a label, let us create a simple program. Create a new Java project and add a frame. Let us name it as LabelBackgroundColor.


          
           
import javax.swing.*;
 
public class LabelBackgroundColor {
     public static void main(String[] args) {
         JFrame frame = new JFrame();
         frame.setSize(400, 400);
         frame.setVisible(true);
     }
}

Now let us add few labels to the frame. In Eclipse, drag the label component from palette and drop it on the frame. Make sure to set its text and position it properly.


          
           
JLabel label1 = new JLabel("Label 1");
label1.setBounds(50, 50, 100, 50);
frame.add(label1);
 
JLabel label2 = new JLabel("Label 2");
label2.setBounds(150, 50, 100, 50);
frame.add(label2);

Now let us set the background color of the labels. To do this, we can use setBackground() method of the JLabel class. The method takes an argument of type Color.


          
           
label1.setBackground(Color.BLUE);    
label2.setBackground(Color.RED);

We can also specify the color using its RGB values. Here is an example:


          
           
label1.setBackground(new Color(255, 0, 0)); // red
label2.setBackground(new Color(0, 255, 0)); // green

That's it. We have successfully changed the background color of the labels in our program.

Conclusion

In this tutorial, we have learned how to change the background color of a label in Java. We have seen how to do it using basic color values as well as RGB values. We have also seen how to set the color of the label using the setBackground() method.
