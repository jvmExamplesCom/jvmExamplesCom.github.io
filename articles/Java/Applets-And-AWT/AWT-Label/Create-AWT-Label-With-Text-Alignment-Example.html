Create AWT Label With Text Alignment Example

In this example we'll show how to create a Label object in AWT and set its text alignment. We'll also provide a code sample along with a screenshot of the output.

Creating the Label

To create a new AWT Label, we use the following line: 

          
           
Label labelObj = new Label();
Once we have our Label object, we can set its text alignment using one of the following values:
  • Label.LEFT
  • Label.CENTER
  • Label.RIGHT
For example, if we want to set the text alignment to center, we would use the following code: 

          
           
labelObj.setAlignment(Label.CENTER);

Example Use

Let's take a look at a complete example where we create a Label object and set its text alignment to the center: 

          
           
import java.awt.*;

public class LabelExample {
   public static void main(String[] args){
      Label labelObj = new Label();

      //Setting alignment to center
      labelObj.setAlignment(Label.CENTER);

      //Adding label to Frame
      Frame frame = new Frame();
      frame.add(labelObj);
      frame.setSize(300, 300);
      frame.setVisible(true);
   }
}
