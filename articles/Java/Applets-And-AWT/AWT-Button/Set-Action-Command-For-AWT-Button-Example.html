Set Action Command for AWT Button Example

This tutorial will guide you through setting an action command for a Java AWT button. First, we'll review what the action command does and then show you how to set it with code examples.

What is an Action Command?

An action command is a user-defined string that is associated with a particular GUI component or control. It is used as an identifier so that when an event is generated, you can determine which of the action commands is being invoked. For example, let's say you have three buttons on your screen - "Start", "Stop", and "Reset". You can assign the action commands "start", "stop", and "reset" respectively to those buttons. When the user clicks on the "Start" button, your program will be able to detect that the "start" action command has been invoked.

Setting an Action Command for an AWT Button

The Java AWT (Abstract Windowing Toolkit) is a set of classes that provide basic graphical user interface components, such as buttons, checkboxes, labels, etc. Let's look at a simple example of setting an action command for an AWT button. 

          
           
import java.awt.*;

public class ActionExample {
    private Frame mainFrame;
    private Button btnStart;

    public ActionExample() {
        mainFrame = new Frame("Action Example");
        btnStart = new Button("Start");
        btnStart.setActionCommand("start");
     
        mainFrame.add(btnStart);
        mainFrame.setSize(200, 200);
        mainFrame.setVisible(true);
    }
}
In this example, we create a frame with a single button on it. We then set the action command of the button to "start". This means that when the user clicks on the button, the program will be able to detect that the action command "start" has been invoked.

Conclusion

In this tutorial, we have seen how to set an action command for an AWT button in Java. We have also learned what action commands do and how we can use them to identify events triggered by the user. With this knowledge, you should have no problem setting action commands for other AWT components.
