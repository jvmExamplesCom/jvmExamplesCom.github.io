Disable Button Example in Java

In this article, we will demonstrate how to disable a button in Java. Disabling a button means that it can no longer be clicked. This is useful in situations such as applications where there are certain buttons that should only be available during particular parts of the program.

Example Using an Anonymous Inner Class

In this example, we use an anonymous inner class to create a listener for a button and then disable it. 

          
           
Button myButton = new Button("Click me!");
myButton.addActionListener(new ActionListener() {
    public void actionPerformed(ActionEvent e) {
        myButton.setEnabled(false);
    }
});
In this example, we create a button with the text "Click me!" and then add an anonymous inner class which handles the action performed event. The action performed event is fired when the button is clicked, so inside the event handler we set the button's enabled state to false. This effectively disables the button so it can't be clicked again.

Example Using a Lambda Expression

We can also use a lambda expression to achieve the same effect. 

          
           
Button myButton = new Button("Click me!");
myButton.addActionListener(e -> myButton.setEnabled(false));
In this example, we create a button and add an action listener with a lambda expression. The lambda expression takes a single parameter which is the action event. Inside the lambda expression, we set the enabled state of the button to false.

Example Using a Method Reference

Finally, we can also use a method reference to disable the button. 

          
           
Button myButton = new Button("Click me!");
myButton.addActionListener(myButton::setEnabled(false));
In this example, we create a button and set a method reference on it for the action listener. The method reference calls the setEnabled() method on the button and sets the enabled state to false. This effectively disables the button.
