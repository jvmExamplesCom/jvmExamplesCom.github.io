Determine If The Button Is Enabled Example

In this article we will dive into how to determine if a button on a Java application is enabled.

Button.isEnabled() Method

The most obvious way to check if a button is enabled or not is by using the 

Button.isEnabled()
method. This will return a boolean value based on whether or not the button is enabled. 

          
           
boolean isEnabled = btn.isEnabled();
System.out.println(isEnabled);
If the button is enabled the output will be 
true
, otherwise it will be 
false
.

Action Event Listener

Another way to determine if a button is enabled or not is by using an action event listener. This listener can be set up when the page is first loaded and will listen for any interaction with the button, including a mouse click or pressing of the enter key.


          
           
btn.addActionListener(new ActionListener() {
    public void actionPerformed(ActionEvent e) {
        System.out.println("Button enabled: " + btn.isEnabled());
    }
});

If the button is enabled, the above code will print out 

true
. Otherwise, it will print out 
false
.

By Checking the Style Class

If you have access to the component of the button, you can also check its style class to determine if it is enabled or not. The style class of the component will usually contain the string 

disabled
if the button is disabled. 

          
           
String styleClass = btn.getStyleClass();
boolean isEnabled = !styleClass.contains("disabled");
System.out.println(isEnabled);
If the button is enabled, the output will be 
true
, otherwise it will be 
false
.

Conclusion

In this article we have discussed how to determine if a button is enabled or not in Java. We have looked at the 

Button.isEnabled()
method, the action event listener, and checking the style class of the component. Any of these methods can be used to determine if a button is enabled or not.
