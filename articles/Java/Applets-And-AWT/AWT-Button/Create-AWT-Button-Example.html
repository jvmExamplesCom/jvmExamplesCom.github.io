Create AWT Button Example in Java

Creating AWT buttons in Java is a simple task and requires minimal coding. The AWT class provides a number of methods for creating different types of buttons, including push buttons, check boxes, and radio buttons. In this article we'll take a look at how to create an AWT push button.

The AWT Button Constructor

The most basic way to create an AWT button is using the 

          
           
Button(String label)
constructor. This constructor takes a single String argument which represents the text that is displayed on the button. For example, the following code creates an AWT button labeled "OK": 

          
           
Button btnOK = new Button("OK");

Adding an Action Listener

In order to perform an action when a button is clicked, you need to add an action listener to the button. An action listener is a special type of Java object which is capable of responding to events, such as mouse clicks on a button. The following code adds an action listener to the button created in the previous example: 

          
           
btnOK.addActionListener(new ActionListener() {
  public void actionPerformed(ActionEvent e) {
    // Code to be executed when button is clicked
  }
});
The code inside the 

          
           
actionPerformed()
method will be executed when the button is clicked. Note that you can also add multiple action listeners to a single button if desired.

Setting the Button's Size

By default, the size of an AWT button will be determined by its label. However, you can also set a specific size for the button using the 

          
           
setSize()
method. This method takes two integer arguments, specifying the width and height of the button, respectively. The following example sets the size of the button to 100x50 pixels: 

          
           
btnOK.setSize(100, 50);

Displaying the Button

Finally, you need to add the button to a container in order to actually display it. This can be done using the 

          
           
add()
method, which takes the button to be added as the first argument and an optional layout manager as the second argument. The following example adds the button to a frame: 

          
           
Frame frame = new Frame();
frame.add(btnOK);
That's all there is to creating an AWT button in Java. Once the button has been added to a container, it can be displayed by invoking the 

          
           
setVisible()
method on the container. For example, the following code displays the frame containing the button: 

          
           
frame.setVisible(true);
