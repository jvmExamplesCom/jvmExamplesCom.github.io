Change Button Foreground Color Example

The following code demonstrates how to change the foreground color of a button in a Java Swing application. 

          
           
JButton btn = new JButton("Change color");
btn.setForeground(Color.RED);
The above code sets the foreground color of the button to red.

Using ActionListener to Change Color

The following code demonstrates how to use an ActionListener to change the foreground color of a button in a Java Swing application. 

          
           
JButton btn = new JButton("Change color");
 btn.addActionListener(new ActionListener() {
	@Override
	public void actionPerformed(ActionEvent e) {
		btn.setForeground(Color.BLUE);
	}
 });
In the above code, when the button is clicked, the foreground color of the button is changed to blue.
