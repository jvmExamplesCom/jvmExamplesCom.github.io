Java Hide Button Example

The code below demonstrates how to programmatically hide a button in Java. This code utilizes the Java AWT package for the graphical component and uses the setVisible() function to hide the button. 

          
           
Button btn_hide = new Button("Hide");
btn_hide.setVisible(false);
When the above code is run, the "hide" button will not be visible on the screen.

Example of Hiding a Button With an Action Event Listener

In this example, we will use an action event listener to programmatically hide the button when it is clicked. This code utilizes the same Java AWT library and the setVisible() function. 

          
           
// Define hide button.
Button btn_hide = new Button("Hide");

// Add action event listener to hide button
btn_hide.addActionListener(new ActionListener(){
    public void actionPerformed(ActionEvent e) {
        btn_hide.setVisible(false);
    }
});
The above code adds an action event listener to the hide button, so when the button is clicked, it will not be visible on the screen anymore.

Example of Revealing a Hidden Button

In this example, we will programmatically reveal a hidden button. This code utilizes the same Java AWT library and the setVisible() function. 

          
           
// Define show button.
Button btn_show = new Button("Show");

// Add action event listener to show button
btn_show.addActionListener(new ActionListener(){
    public void actionPerformed(ActionEvent e) {
        btn_hide.setVisible(true);
    }
});
The above code adds an action event listener to the show button, so when the button is clicked, it will become visible on the screen again.

Complete Code Example

Below is a complete code example demonstrating both hiding and revealing a button using Java AWT library. 

          
           
// Define hide and show buttons
Button btn_hide = new Button("Hide");
Button btn_show = new Button("Show");

// Add action event listener to hide button
btn_hide.addActionListener(new ActionListener(){
    public void actionPerformed(ActionEvent e) {
        btn_hide.setVisible(false);
    }
});

// Add action event listener to show button
btn_show.addActionListener(new ActionListener(){
    public void actionPerformed(ActionEvent e) {
        btn_hide.setVisible(true);
    }
});
The above code demonstrates how to hide and reveal buttons programmatically in Java.
