Handle Action Events for AWT Button Example

Java provides several methods of adding interaction to applications using the AWT (Abstract Window Toolkit). One of those methods is through Action Events. This article will discuss how to create, register and handle Action Events for AWT buttons.

Creating an Action Event Listener

At the core of handling Action Events is creating an Action Event listener. An Action Event listener implements the ActionListener interface, which has one method that must be implemented: 

          
           
public void actionPerformed(ActionEvent event)
The Action Event listener class can then be used to listen for action events from components that support it, such as Buttons. A simple Action Event listener class might look like this: 

          
           
import java.awt.event.*;

public class MyActionListener implements ActionListener {
  public void actionPerformed(ActionEvent event) {
     //handle action here
  }
}

Registering an Action Event Listener

Now that an Action Event listener has been created, it needs to be registered with a component that supports Action Events. In our example, we will be registering the listener with an AWT button. To do this we call the 

          
           
addActionListener()
method on the button with the Action Event listener as an argument. For example: 

          
           
MyActionListener myActionListener = new MyActionListener();
Button myButton = new Button("My Button");
myButton.addActionListener(myActionListener);
The above code creates a new instance of an Action Event listener and a button, and registers the action listener with the button.

Handling Action Events

Once the Action Event listener has been registered with the component, it can begin handling Action Events. This is done in the 

          
           
actionPerformed()
method of the Action Event listner class. The method takes an ActionEvent object as an argument, which contains information about the event, such as its source. For example: 

          
           
public void actionPerformed(ActionEvent event) {
  Object source = event.getSource();
  if(source == myButton) {
    //handle action here
  }
}
The above code checks to see if the source of the event is the button, and if it is then it can perform an action.

Conclusion

Action Events are a powerful way to add user interaction to Java applications. This article has discussed how to create, register, and handle Action Events for AWT buttons.
