Get Number Of Visible Rows Of AWT List Example

Getting the number of visible rows of an AWT List can be done through a few simple steps. In this example, we will create a List and set its visibleRowCount to 3. Then, we will use the getVisibleRowCount() method to get the number of visible rows of the list.

Create List And Set Visible Row Count

Below is the code for creating an AWT List and setting its visible row count to 3: 

          
           
List list = new List();
list.setVisibleRowCount(3);

Get Number Of Visible Rows

We can use the getVisibleRowCount() method to get the number of visible rows of the list: 

          
           
int visibleRows = list.getVisibleRowCount();

Print The Result

Finally, we can print the result using the following statement: 

          
           
System.out.println("The number of visible rows is: " + visibleRows);
When run, the above code prints the following result: 

          
           
The number of visible rows is: 3
