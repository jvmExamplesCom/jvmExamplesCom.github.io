Get Selected Indexes From Multi Select AWT List Example

This example demonstrates how to get the selected indexes of a multi select AWT List. The AWT list is a subclass of the AWT Component class which allows the user to select one or more items from a list.

Example Code

The following example code shows how to get the selected indexes of a multi select AWT List. The code is written in Java and uses the AWT List class.


          
           
List list = new List(4, false);

// Add items to the list
list.add("Item 1");
list.add("Item 2");
list.add("Item 3");
list.add("Item 4");

int[] selectedIndexes = list.getSelectedIndexes();

for (int i : selectedIndexes) {
    System.out.println("Selected index: " + i);
}

Output

When this code is executed, it will print out the index values of the selected items in the list.


          
           
Selected index: 0
Selected index: 2
