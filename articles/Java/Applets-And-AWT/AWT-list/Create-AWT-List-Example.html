Create AWT List Example in Java

AWT List is an example of AWT GUI component used to display list of items. The AWT List class is used to create a List object, which can then be used to show the list of items in AWT GUI window. Frame frame = new Frame("AWT List Demo");
frame.setSize(250, 150);
frame.setVisible(true);
This code creates a Frame object with its title set as "AWT List Demo". It also sets the size of the Frame at 250px by 150px, and makes it visible on the screen. Next, we need to create an AWT List object that will be placed within the Frame. To do this, we can use the following code: 

          
           
List list = new List();
list.add("Item 1");
list.add("Item 2");
list.add("Item 3");
list.setMultipleMode(true);
frame.add(list);
This code creates a List object, and adds three items to it. We also set the MultipleMode property to true, so that multiple items can be selected simultaneously. Finally, we add the list to the Frame. Now, we have created an AWT List object in Java, and added it to the Frame.
