Get Selected Index From Single Select AWT List Example

In this tutorial, we will learn how to get the selected index from a single select AWT list example. We will also look at the code examples of how to get the selected index from a single select AWT list.

Example 1: Get Selected Index From Single Select AWT List

In this example, we will create a single select list using the AWT List class and then get the selected index from it using the getSelectedIndex() method.


          
           
List myList = new List();
myList.add("A");
myList.add("B");
myList.add("C");

// Get selected index
int selectedIndex = myList.getSelectedIndex();
System.out.println("Selected index: " + selectedIndex);
Output: 

          
           
Selected index: 0
