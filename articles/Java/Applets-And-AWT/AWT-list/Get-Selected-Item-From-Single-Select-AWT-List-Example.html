Get Selected Item From Single Select AWT List Example

The AWT List class provides a way for users to choose a single item from a list of items. This tutorial explains how to use the AWT List class to create a single-selection list and get the selected item.

Creating the List

The List class can be used to create a single-select list. The List constructor takes two parameters: the parent component, and the style (which should be either List.SINGLE or List.MULTIPLE). The following example code creates a single-select list with four items: 

          
           
List list = new List(parent, List.SINGLE);
list.add("Item 1");
list.add("Item 2");
list.add("Item 3");
list.add("Item 4");

Getting the Selected Item

The List class provides a method to retrieve the index of the selected item. This method is called getSelectedIndex(). The following code example retrieves the index of the selected item and prints it to the console: 

          
           
int index = list.getSelectedIndex();
System.out.println("Selected index: " + index);
The List class also provides a method to retrieve the item at a given index. This method is called getItem() and it takes an index parameter. The following code example retrieves the item at the selected index and prints it to the console: 

          
           
String item = list.getItem(index);
System.out.println("Selected item: " + item);

Example Program

The following program demonstrates the use of the AWT List class to create a single-select list and get the selected item. 

          
           
import java.awt.*;

public class ListExample {

    public static void main(String[] args) {
        Frame frame = new Frame("List Example");
        Panel panel = new Panel();

        List list = new List(4, false);
        list.add("Item 1");
        list.add("Item 2");
        list.add("Item 3");
        list.add("Item 4");

        panel.add(list);
        frame.add(panel);

        frame.setSize(200, 100);
        frame.setVisible(true);

        int index = list.getSelectedIndex();
        String item = list.getItem(index);

        System.out.println("Selected index: " + index);
        System.out.println("Selected item: " + item);
    }
}
This program will produce the following output: 

          
           
Selected index: 0
Selected item: Item 1
