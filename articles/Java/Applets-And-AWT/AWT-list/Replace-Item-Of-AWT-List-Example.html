Replace Item Of AWT List Example

Replacing an item of an AWT list model can be a tricky task, but this example will walk you through the process.

Create the List Model

The first step is to create a new AWT List Model. This will include creating an array list containing strings to be added to the list. The following example uses Java's ArrayList class to add the items:


          
           
ArrayList<String> items = new ArrayList<String>();
items.add("One");
items.add("Two");
items.add("Three");
items.add("Four");

Now that the array list has been created, we need to create the AWT List Model. This is done using the AWT List Model class, as shown below:


          
           
AWTListModel<String> listmodel = new AWTListModel<String>(items);

Replace an Item

Now that we have a list model, we can replace an item within it. To do this, simply call the 


          
           
listmodel.replaceItem(index, item)
method, where 
index
is the index of the item to be replaced, and 
item
is the new item to replace it with.

For example, to replace the item at index 0 with the string "Foo", we would call:


          
           
listmodel.replaceItem(0, "Foo");

This will replace the item at index 0 with the string "Foo".

Retrieve an Item

Once an item has been replaced, you may want to retrieve it. To do this, simply call the 


          
           
listmodel.getItem(index)
method, where 
index
is the index of the item to be retrieved.

For example, to retrieve the item at index 0, we would call:


          
           
String item = listmodel.getItem(0);
This will return the string "Foo".

Conclusion

Replacing an item of an AWT List Model is relatively straightforward. By following the steps outlined in this example, you should have no problem replacing and retrieving items from the list.
