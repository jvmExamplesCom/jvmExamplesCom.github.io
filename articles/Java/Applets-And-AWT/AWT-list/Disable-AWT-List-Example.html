Disable AWT List Example

AWT (Abstract Window Toolkit) is a GUI library used in Java to create windows and graphics. One of the most common components used in AWT is the List, which is a scrollable list of items. This article will discuss how to disable items in a list in an AWT program.

Example Code


          
           
import java.awt.*; 
import java.awt.event.*; 
  
public class DisableListExample { 
    public static void main(String[] args) 
    { 
        Frame frm = new Frame(); 
        frm.setSize(400, 400); 
        frm.setLayout(null); 
        frm.setVisible(true); 
  
        // Creating a List 
        final List lst = new List(); 
  
        // Adding elements to the List 
        lst.add("Item 1"); 
        lst.add("Item 2"); 
        lst.add("Item 3"); 
        lst.add("Item 4"); 
        lst.add("Item 5"); 
        lst.add("Item 6"); 
  
        // Setting bounds of the List 
        lst.setBounds(100, 100, 75, 75); 
  
        // Adding the List to the frame 
        frm.add(lst); 
  
        // disabling the List by item 
        lst.disable(4); 
    } 
}

In this example code, we are creating a List with some elements and setting its bounds. We then add the list to a frame and disable the last item in the list, which is “Item 5”.

Output

The output of the above code is a List with some items. The last item, “Item 5”, appears in grey and cannot be selected.

Conclusion

In this article, we discussed how to disable items in AWT List. We saw how to do this using the “disable” method of the List class. We also looked at an example code and its output.
