Java: Get Item Count Of AWT List Example

In this article, we discuss how to get the item count of an AWT list using the getItemCount() method. This example is written using Java using the Abstract Window Toolkit (AWT). We look at the code example and discuss the output.

The Code Example


          
           
import java.awt.*;

public class GetItemCountOfAWTListExample {
    public static void main(String[] args) {
        Frame frame = new Frame();
        List list = new List();
        list.add("A");
        list.add("B");
        list.add("C");
        list.add("D");
        frame.add(list);
        frame.setSize(400, 400);
        frame.setVisible(true);
        System.out.println("Size of AWT List =  " + list.getItemCount());
    }
}

Output


          
           
Size of AWT List =  4
