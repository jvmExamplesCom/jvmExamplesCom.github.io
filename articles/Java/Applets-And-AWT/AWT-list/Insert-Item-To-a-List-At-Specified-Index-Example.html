Insert Item To a List At Specified Index Example

Java is a great language for building applications and working with data structures. One of the more powerful features of Java is the ability to easily manipulate lists. This article will explore how to insert an item into a list at a specified index.

Creating a List

The first step is to create a List in Java. You can do this using the 

ArrayList
class. This class provides a constructor that takes an array of items as an argument and creates a new List from those items. 

          
           
String[] names = {"John", "Sue", "Anna"};
List<String> list = new ArrayList<>(Arrays.asList(names));

This code creates a List from an array of Strings and assigns it to the 

list
variable. The List now contains the elements 
John
, 
Sue
and 
Anna
.

Inserting an Item at a Specified Index

Now that we have a List, we can insert an item into it at a specified index. We can do this using the 


          
           
add(int index, E element)
method on the 
List
. This method takes two arguments: an index and an element to insert. The element is inserted at the specified index, and all elements after it are shifted one position to the right. 

          
           
list.add(2, "Jack");
This code inserts the String 
Jack
at index 2 in the 
list
. The resulting List contains the elements 
John
, 
Sue
, 
Jack
and 
Anna
. Note that the elements at index 0 and 1 remain unchanged.

Inserting Multiple Items at a Specified Index

It is also possible to insert multiple items at a specified index. To do this, we can use the 


          
           
addAll(int index, Collection<? extends E> c)
method on the 
List
. This method takes two arguments: an index and a 
Collection
of elements to insert. The elements in the 
Collection
are inserted at the specified index, and all elements after it are shifted to make room for the new elements. 

          
           
List<String> names = Arrays.asList("Ben", "Charlie", "Daniel");
list.addAll(3, names);
This code inserts the three Strings in the 
names
List at index 3 in the 
list
. The resulting List contains the elements 
John
, 
Sue
, 
Jack
, 
Ben
, 
Charlie
, 
Daniel
and 
Anna
. Note that the elements at indices 0, 1 and 2 remain unchanged.

Inserting an Item at the End of the List

Finally, it is possible to insert an item at the end of the List. To do this, we can use the 

add(E element)
method on the 
List
. This method takes one argument, the element to insert, and inserts it at the end of the List. 

          
           
list.add("Emily");
This code inserts the String 
Emily
at the end of the 
list
. The resulting List contains the elements 
John
, 
Sue
, 
Jack
, 
Ben
, 
Charlie
, 
Daniel
, 
Anna
and 
Emily
.

Conclusion

In this article, we explored how to insert an item into a List at a specified index in Java. We saw how to create a List from an array of items, how to insert a single item at a specified index, how to insert multiple items at a specified index, and how to insert an item at the end of the List.
