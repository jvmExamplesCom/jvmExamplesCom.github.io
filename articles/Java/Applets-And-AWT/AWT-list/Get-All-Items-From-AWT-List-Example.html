Get All items From AWT List Example

In this guide, we'll be discussing how to get all items from an AWT list. By the end of this article, readers should have a good understanding of the various ways to code a program to do this task in Java.

Using For-Loop:

The simplest way to get all items from an AWT list is to use a 

          
           
for-loop
: 

          
           
List list = new List(); 
for (int i = 0; i < list.getItemCount(); i++) 
    System.out.println(list.getItem(i));
Here, a list instance is created using the 

          
           
List
class. We then iterate over the list using a for-loop and print out each item using the 

          
           
getItem()
method.

Using Enhanced For-Loop:

An 

          
           
enhanced for-loop
can also be used to get all items from an AWT list: 

          
           
List list = new List(); 
for (Object element : list.getItems()) 
    System.out.println(element.toString());
The same list instance is created and within the loop, the 

          
           
getItems()
method is used to get all elements from the list. The 

          
           
toString()
method is then invoked on each object.

Using ForEach:

The 

          
           
forEach()
method can be used to get all items from an AWT list as well: 

          
           
List list = new List(); 
list.getItems().forEach(System.out::println);
Here, the 

          
           
getItems()
method is used to retrieve all items from the list and then a method reference is used to print out each element.

methods.
