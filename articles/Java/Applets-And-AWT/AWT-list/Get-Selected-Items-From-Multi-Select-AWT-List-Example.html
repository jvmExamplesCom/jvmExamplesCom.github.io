Get Selected Items From Multi Select AWT List Example

In this tutorial, we will discuss how to get the selected items from a multi select AWT list in Java. We will be using the 

          
           
java.awt.List
class for this purpose. First, let's create an instance of this class and add some items to it: 

          
           
List l = new List();
l.add("Item 1");
l.add("Item 2");
l.add("Item 3");
l.add("Item 4");
l.add("Item 5");
Now, we can set the list to be multiple selection by calling the 

          
           
setMultipleMode()
method: 

          
           
l.setMultipleMode(true);
Finally, we can get the selected items by calling the 

          
           
getSelectedItems()
method: 

          
           
String[] selectedItems = l.getSelectedItems();
The 

          
           
selectedItems
array will contain the selected items from the AWT list. Let's look at a complete example where we are creating a Frame with a list and a button: 

          
           
import java.awt.*;
import java.awt.event.ActionEvent;
import java.awt.event.ActionListener;

public class GetSelectedItemsFromMultiSelectAWTListExample extends Frame implements ActionListener {

    List l;

    public GetSelectedItemsFromMultiSelectAWTListExample() {
            // Create frame
        Frame f = new Frame("Get Selected Items From Multi Select AWT List Example");

            // Create List
        l = new List();

            // Add items to list
        l.add("Item 1");
        l.add("Item 2");
        l.add("Item 3");
        l.add("Item 4");
        l.add("Item 5");

            // Set list to multiple selection
        l.setMultipleMode(true);

            // Add list to frame
        f.add(l);

            // Create button
        Button b = new Button("Get Selected Items");

            // Add action listener
        b.addActionListener(this);

            // Add button to frame
        f.add(b, BorderLayout.SOUTH);

            // Set the size of frame
        f.setSize(400, 400);

            // Set the visibility of frame
        f.setVisible(true);
    }

    public void actionPerformed(ActionEvent e) {
            // Get selected items
        String[] selectedItems = l.getSelectedItems();

            // Print selected items
        for (int i = 0; i < selectedItems.length; i++) {
            System.out.println(selectedItems[i]);
        }
    }

    public static void main(String args[]) {
        new GetSelectedItemsFromMultiSelectAWTListExample();
    }
}
If we run this program, selecting the first, third and fifth item in the list, we will see the following output: 

          
           
Item 1
Item 3
Item 5
This is how you can get the selected items from a multi select AWT list in Java.
