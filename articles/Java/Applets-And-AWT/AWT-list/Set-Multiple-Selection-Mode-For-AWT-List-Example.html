Set Multiple Selection Mode For AWT List Example

Multi selection mode allows users to select multiple items from a list. This can be achieved using the AWT ("Abstract Window Toolkit") API, which is part of the Java SE platform. This tutorial will demonstrate how to set multiple selection mode for an AWT List example.

Creating a New AWT List Component

The first step is to create a new AWT List component: 

          
           
List list = new List();
This will create a new AWT List that is capable of displaying multiple items.

Adding Items to the List

Next, we can add items to the List. We can do this by using the 

          
           
add()
method of the List object. For example, to add the items "Apple", "Banana", and "Cherry" to the List, we could use the following code: 

          
           
list.add("Apple");
list.add("Banana");
list.add("Cherry");

Setting Multiple Selection Mode

Finally, we can set the List to multiple selection mode by using the 

          
           
setMultipleMode()
method. We can call this method on the List object with a boolean argument that specifies whether multiple selection mode should be enabled (true) or disabled (false). For example: 

          
           
list.setMultipleMode(true);
This will enable multiple selection mode on the List, allowing users to select more than one item at a time.

Conclusion

In this tutorial, we have demonstrated how to set multiple selection mode for an AWT List example. We have seen how to create a new AWT List component, how to add items to the list, and how to set multiple selection mode.
