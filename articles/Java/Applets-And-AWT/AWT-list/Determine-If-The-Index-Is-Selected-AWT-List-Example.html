Determine If The Index Is Selected AWT List Example

This tutorial provides a sample code snippet to illustrate how to determine if the index is selected from a Java AWT List. To check if a specific item in the list is selected or not, the isIndexSelected() method of List class should be used. This method takes one argument: an index of the list item, and returns a boolean value, true if the particular item is selected, false otherwise. In the following example, we will create a AWT based user interface with a List that contains some items by using AWT Components. Then, we will iterate through its items and check if each one of them is selected or not by using the isIndexSelected() method of List class. 

          
           
import java.awt.*;  
import javax.swing.JFrame; 

public class DetermineIfTheIndexIsSelectedExample {  
    public static void main(String[] args) {    
        //Creating the Frame  
        JFrame frame = new JFrame("Determine If The Index Is Selected Example");  
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);  
        frame.setSize(400, 400);  

        //Creating the List  
        List list = new List(); 
        list.add("Item 1");  
        list.add("Item 2");  
        list.add("Item 3");  
        list.add("Item 4");  
        list.add("Item 5"); 

        //Adding the Items to the frame  
        frame.add(list);  
        frame.setVisible(true);  

        //Iterating through the list items  
        for (int i = 0; i < list.getItemCount(); i++) {  
            //Checking if the item is selected  
            if (list.isIndexSelected(i)) {  
                System.out.println(list.getItem(i) + " is selected");  
            } else {  
                System.out.println(list.getItem(i) + " is not selected");  
            }  
        }  
    }  
}
When we run the above program, we will get the following output: 

          
           
Item 1 is not selected
Item 2 is not selected
Item 3 is not selected
Item 4 is not selected
Item 5 is not selected
