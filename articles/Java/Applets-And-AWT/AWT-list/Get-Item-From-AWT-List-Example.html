Get Item From AWT List Example

In this example, we’ll look at how to get an item from an AWT list using Java. We’ll walk through the steps for setting up an AWT list component and then retrieving an item from it.

Setting Up An AWT List Component

To set up an AWT list component, we first need to import the necessary classes: 

          
           
import java.awt.*;
import java.awt.event.*;
We then create a Frame object: 

          
           
Frame frame = new Frame("My Frame");
We then create an AWT List object: 

          
           
List list = new List(3);
In this example, we’ve created a List component with 3 item slots. We can add items to the list like this: 

          
           
list.add("Item 1");
list.add("Item 2");
list.add("Item 3");
We then add the List component to our Frame object: 

          
           
frame.add(list);

Retrieving An Item From An AWT List

To retrieve an item from an AWT list, we first need to get the index of the item we want to retrieve. We can do this by calling the 

          
           
getSelectedIndex()
method on the List component: 

          
           
int index = list.getSelectedIndex();
We can then retrieve the item by passing the index to the 

          
           
getItem()
method: 

          
           
String item = list.getItem(index);

Full Example Code

Here is the full example code needed to set up an AWT list component and retrieve an item from it: 

          
           
import java.awt.*;
import java.awt.event.*;

public class AWTExample {

    public static void main(String[] args) {
        Frame frame = new Frame("My Frame");
        List list = new List(3);

        list.add("Item 1");
        list.add("Item 2");
        list.add("Item 3");
        frame.add(list);

        int index = list.getSelectedIndex();
        String item = list.getItem(index);

        System.out.println(item);
    }
 
}
When we run this code, it will print out the retrieved item on the console.
