Remove All Items From a List Example in Java

Removing all items from a list is a common operation that can be performed using different methods in Java. In this article, we will discuss how to remove all elements from a given list using Java. There are multiple ways to achieve this and we will discuss each of them in detail with code examples.

Using clear() Method

The easiest way to remove all elements from the list is by using the clear() method. This method is available in the java.util.List interface and it removes all items in the list. Here is an example of how it can be used: 

          
           
List<String> list = Arrays.asList("a", "b", "c");
list.clear();
System.out.println(list);  // []
This approach is most suitable when you want to completely empty out the list and not retain the reference to it. The clear() method returns no value.

Using removeAll() Method

The removeAll() method is another way of removing all items from the list. It takes a collection argument and removes any elements contained in the collection argument from the list. Here is an example of how to use it: 

          
           
List<String> list = Arrays.asList("a", "b", "c");
Collection<String> c = Arrays.asList("a", "b");
list.removeAll(c);
System.out.println(list);  // [c]
The removeAll() method returns true if the list was modified or false otherwise.

Using removeIf() Method

The removeIf() method is another way of removing all elements from the list in Java. This method takes a Predicate object as an argument and removes any elements from the list that match the predicate condition. Here is an example of how it can be used: 

          
           
List<String> list = Arrays.asList("a", "b", "c");
list.removeIf(s -> s.equals("b"));
System.out.println(list);  // [a, c]
The removeIf() method returns true if the list is modified or false otherwise.

Using Iterator

Another way to remove all items from the list is by using an Iterator. The iterator() method of the list returns an Iterator instance and by looping through it, we can remove each element in the list. Here is an example of how it can be done: 

          
           
List<String> list = Arrays.asList("a", "b", "c");
Iterator<String> iterator = list.iterator();
while (iterator.hasNext()) {
    iterator.next();
    iterator.remove();
}
System.out.println(list);  // []
The Iterator.remove() method removes the last element returned by the next() method.

Conclusion

In this article, we saw how to remove all items from a list in Java. We saw multiple ways of doing so, such as using the clear(), removeAll() and removeIf() methods, as well as using an Iterator. All of the approaches discussed here are valid solutions depending on the use case.
