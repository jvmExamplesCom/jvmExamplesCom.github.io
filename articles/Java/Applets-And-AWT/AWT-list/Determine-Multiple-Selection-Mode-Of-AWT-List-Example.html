Determine Multiple Selection Mode Of AWT List Example

Java's Abstract Window Toolkit (AWT) provides the List class which allows developers to easily create a scrolling list of items. Each list item can be selected or deselected. This article will discuss how to determine the multiple selection mode of an AWT list.

Multiple Selection Modes

The AWT List class allows developers to specify whether users can select one list item or multiple list items. The two possible modes are SINGLE_SELECTION and MULTIPLE_SELECTION. SINGLE_SELECTION mode allows only one list item to be selected at a time. MULTIPLE_SELECTION allows the user to select multiple list items by holding the CTRL key while selecting the required item.

Example

This example demonstrates how to determine the selection mode of a list in AWT. Firstly, create an instance of AWT List object: 

          
           
List list = new List();
To set the selection mode, use the 

          
           
setMultipleMode()
method. This method takes a single argument that specifies the new selection mode which can be either SINGLE_SELECTION or MULTIPLE_SELECTION. If you want to allow users to select multiple list items, use: 

          
           
list.setMultipleMode(true);
To get the current selection mode of a list, use the 

          
           
getMultipleMode()
method, as follows: 

          
           
boolean isMultipleMode = list.getMultipleMode();
If the return value of this method is true, it means the selection mode is MULTIPLE_SELECTION. Otherwise, the selection mode is SINGLE_SELECTION.

Complete Source Code

Here is the complete source code of this example: 

          
           
import java.awt.*;

public class AwtList {
  public static void main(String[] args) {
    Frame f = new Frame("AWT List Example");
		
    List list = new List();
		
    // Set the selection mode of the List 
    list.setMultipleMode(true);
		
    // Check the selection mode of the List 
    boolean isMultipleMode = list.getMultipleMode();
    System.out.println("Selection mode: " + (isMultipleMode ? "MULTIPLE_SELECTION" : "SINGLE_SELECTION"));
		
    f.add(list);
    f.setSize(400, 400);
    f.setLayout(null);
    f.setVisible(true);
  }
}

Output

When you run the above program, the output will be: 

          
           
Selection mode: MULTIPLE_SELECTION
