Java Applet Info Example:

In this tutorial, we will demonstrate how to use the Java Applet API to retrieve important information about an applet. We will show how to query an applet's document base and code base, its parameters, and other pertinent information.

Getting Started:

Before we can access any of an applet's information, we must create an instance of the applet using Java's Applet class. This class provides several useful methods that can be used to get information about an applet. The basic syntax for creating an instance of an applet is as follows:


          
           
Applet myapplet = new Applet();
myapplet.init();
myapplet.start();
The 
init()
and 
start()
methods are necessary for any applet to work properly; they must be called after the applet is created. After calling these methods, the applet should be ready to accept requests for information.

Retrieving an Applet's Document Base:

The document base of an applet is the URL of the HTML page in which it is embedded. It is simple to get the document base of an applet using the 

getDocumentBase()
method. This method returns a 
URL
object representing the document base of the applet. Here is an example: 

          
           
URL documentBase = myapplet.getDocumentBase();
System.out.println(documentBase.toString());

This code prints out the document base of the applet to the console. Note that the 

getDocumentBase()
method can only be called once the applet has been initialized and started.

Retrieving an Applet's Code Base:

The code base of an applet is the URL of the directory containing the applet's compiled class files. It can be retrieved using the 

getCodebase()
method. This method also returns a 
URL
object representing the code base of the applet. Here is an example: 

          
           
URL codeBase = myapplet.getCodebase();
System.out.println(codeBase.toString());

This code prints out the code base of the applet to the console. Note that the 

getCodeBase()
method can only be called once the applet has been initialized and started.

Retrieving an Applet's Parameters:

Applets can contain parameters, which are specified in the HTML page in which they are embedded. These parameters can be accessed using the 

getParameter()
method. This method takes a string argument, which is the name of the parameter being retrieved. Here is an example: 

          
           
String myParam = myapplet.getParameter("myparam");
System.out.println(myParam);

This code retrieves the parameter named "myparam" from the applet, and prints it out to the console. Note that this method can only be called once the applet has been initialized and started.

Conclusion:

In this tutorial, we have demonstrated how to use the Java Applet API to retrieve important information about an applet. We have shown how to query an applet's document base and code base, its parameters, and other pertinent information. We hope you have found this tutorial helpful!
