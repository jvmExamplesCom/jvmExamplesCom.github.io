Java Applet Example

A Java applet is a graphical program written in the Java programming language and stored on a Web server. It can be shared among Web browsers and can be embedded in HTML pages. This tutorial demonstrates how to create a simple Java applet and use it on a web page.

Creating the Applet

The following example creates a simple Java applet named HelloApplet that displays a greeting message:


          
           
import java.awt.*;
import javax.swing.*;

public class HelloApplet extends JApplet {
  public void paint (Graphics g) {
    g.drawString("Hello world!",25,25);
  }
}

The class name specified in the example must be the same as the filename of the source code file for this applet. For example, the source code for this applet must be saved as HelloApplet.java in the directory where the applet will be compiled.

Compiling the Applet

To compile this applet, use the following command:


          
           
javac HelloApplet.java

This command creates a class file named HelloApplet.class. This class file contains the code necessary to run the applet in a Web browser.

Embedding the Applet in an HTML page

The compiled HelloApplet class can be embedded in an HTML page as follows:


          
           
<html>
  <head>
    <title>Hello Applet</title>
  </head>
  <body>
    <applet code="HelloApplet.class" width="200" height="200">
    </applet>
  </body>
</html>

The code attribute specifies the name of the class file created when the Java applet was compiled. The width and height attributes specify the size of the applet on the web page. When the HTML page is loaded in a browser, the applet is dynamically downloaded from the server and then loaded into the browser's Java plugin. After the applet is loaded, it is executed and the output is displayed in the browser.

Sample Applet Output

When the HTML page with the HelloApplet is loaded in the browser, the following output appears:


          
           
Hello world!
