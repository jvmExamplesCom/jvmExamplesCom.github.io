Java: Resize Applet Window Example

Overview

Java allows developers to interact with various components of the user's computer system. The Java Applet API, which lets a programmer embed a Java program in web page and provides a method to resize the applet window. This tutorial will demonstrate how to use the Java Applet API to resize the applet window.

Code Example

The following code example shows how to use the Java Applet API to resize an applet window: 

          
           
// Get the width and height of the applet window
int width = getSize().width;
int height = getSize().height;

// Resize the window
setSize(width + 100, height + 50);
This code creates two variables, "width" and "height", that store the current size of the applet window. Then, it uses the setSize() method to increase the size of the window by 100 pixels in width and 50 pixels in height.

Testing

To test the code example above, we will create a simple Java applet that displays a cyan-colored rectangle on the screen. We will use the code from the previous example to resize the applet window when the button is clicked. 

          
           
import java.awt.*;
import javax.swing.*;

public class ResizeApplet extends JApplet {
    public void init() {
        // Create a button
        JButton btn = new JButton("Resize");

        // Add action listener
        btn.addActionListener((e) -> {
            // Get the width and height of the applet window
            int width = getSize().width;
            int height = getSize().height;

            // Resize the window
            setSize(width + 100, height + 50);
        });

        // Add button to content pane
        getContentPane().add(btn);
    }

    public void paint(Graphics g){
        // Draw a cyan rectangle
        g.setColor(Color.CYAN);
        g.fillRect(20, 20, getSize().width - 40, getSize().height - 40);
    }
}
We can now compile and run this applet in a web browser. When we click the "Resize" button, the applet window should increase in size by 100 pixels in width and 50 pixels in height.

Conclusion

In this tutorial, we demonstrated how to use the Java Applet API to resize an applet window. We used a simple code example to show how the setSize() method can be used to increase the size of an applet window. Finally, we created a simple Java applet and tested our code example to make sure it works as expected.
