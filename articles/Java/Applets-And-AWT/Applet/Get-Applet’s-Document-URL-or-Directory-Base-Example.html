Get Applet's Document URL or Directory Base Example

The document base of an applet is the directory from which it was loaded. When a browser loads an applet, it sets the applet's codebase to the directory of the HTML file that contains the 

         
           tag. The document base can also be specified using the 
          
codebase
 attribute within the 
          
           
             tag. 
            


            
The following code example shows how to get the applet's document URL or directory base. It uses the methods 

            
getCodeBase()
 and 
            
getDocumentBase()
 of the 
            
Applet
 class, which return the directory base.
            


            

              
               
public void init() {
    String codeBase = this.getCodeBase().toString();
    String documentBase = this.getDocumentBase().toString();

    System.out.println("Code base: " + codeBase);
    System.out.println("Document base: " + documentBase);
}

              



            The output of this code example will be:


            

              
               
Code base: http://example.com/
Document base: http://example.com/path/page.html
