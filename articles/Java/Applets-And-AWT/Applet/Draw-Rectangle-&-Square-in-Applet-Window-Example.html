Draw Rectangle & Square in Applet Window Example

Java is an object-oriented programming language that allows developers to create various graphical objects and images. In this article we will discuss how to draw a rectangle and a square in an applet window using Java.

Creating the Applet Window

The first step to drawing a rectangle or square in an applet window is to create the applet window itself. To do this, we need to create a class that extends the 

          
           
java.applet.Applet
class and override its 

          
           
paint
method. This 

          
           
paint
method will be called by the browser when the window is opened and it will be used to draw the rectangular and square shapes. 

          
           
import java.applet.Applet;
import java.awt.Graphics;

public class DrawRectangleSquareExample extends Applet {

    public void paint(Graphics g) {
        // Code to draw rectangle and square goes here.
    }
}

Drawing the Rectangle and Square

Once the applet window has been created, we can then use the 

          
           
Graphics
object provided in the 

          
           
paint
method to draw the rectangle and square. The 

          
           
Graphics
object provides several methods for drawing shapes, including 

          
           
drawRect
for drawing rectangles and 

          
           
drawSquare
for drawing squares. The 

          
           
drawRect
method takes four parameters - the x and y coordinates of the top left corner of the rectangle, and the width and height of the rectangle. Similarly, the 

          
           
drawSquare
method takes two parameters - the x and y coordinates of the top left corner of the square, and the length of each side. For example, to draw a 20x20 square in the top left corner of the window: 

          
           
// Draw the square
g.drawRect(0, 0, 20, 20);
And to draw a 30x50 rectangle starting at x=20 and y=20: 

          
           
// Draw the rectangle
g.drawRect(20, 20, 30, 50);
