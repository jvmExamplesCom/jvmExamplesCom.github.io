Draw Oval & Circle in Applet Window Example

Introduction to Java Applets

Java applets are small programs that can be embedded in a web page and executed within a web browser. They are typically used for displaying animation, enhancing existing HTML forms and providing interactive elements such as image viewing and data entry fields. Applets allow for user interaction and manipulation of their content using the Java language.

Drawing an Oval & Circle in an Applet Window

The following example demonstrates how to draw an oval and circle in an applet window. The example uses the 

          
           
drawOval()
and 

          
           
fillOval()
methods of the Graphics class to draw an oval and circle respectively. The example also sets the colour of the oval and the circle using the 

          
           
setColor()
method of the Graphics class. 

          
           
import java.applet.Applet; import java.awt.*;

public class OvalApplet extends Applet {
    public void paint(Graphics g) {
        // Set the color to red
        g.setColor(Color.red);

        // Draw an oval
        g.drawOval(20, 20, 100, 100); 

        // Set the color to blue
        g.setColor(Color.blue);

        // Draw a filled oval
        g.fillOval(180,20,100,100);
     } 
}
