Background Color Of an Applet Window Example

This article will explain how to set the background color of an Applet Window using the setBackground() method that is available with the AWT (Abstract Window Toolkit) and Java AWT. The AWT is a platform independent API for creating and manipulating GUI components in Java. This example is implemented using Java programming language so you can use it with any other platform.

Code Example

The following code shows how to set the background color of an Applet window:


          
           
import java.awt.Color;
import java.applet.Applet;

public class MyApplet extends Applet {

   public void init() {
      Color c = new Color(255, 0, 0); // Red color
      setBackground(c);
   }
}

In the above example, we are using the setBackground() method to set the background color of an Applet window. This method takes a single parameter, which is an instance of the 

java.awt.Color
class. You can also pass any one of the predefined colors as argument to the setBackground() method like 
Color.RED
, 
Color.BLUE
, 
Color.GREEN
, etc.

Once you have set the background color, it will be visible on the Applet window when it is executed.

Output

The output of this example will be a Applet window with a red background color.
