Using Applet Dimension to Print Center Aligned Text Example in Java

Printing center aligned text is a fairly easy task to accomplish in Java, but it requires you to have knowledge of a few simple concepts first. In this article, we’ll be examining how to use the Applet Dimension class to print center aligned text in a Java program.

What Is the Applet Dimension Class?

The Applet Dimension class is a part of the Java AWT library that provides access to information about an applet’s size, namely its width and height. This information can be used when printing text, as we will explain below.

Using the Applet Dimension Class to Print Center Aligned Text

To use the Applet Dimension class to print center aligned text in your Java program, you need to do the following:

  1. Instantiate an Applet Dimension object and pass in the desired width and height.
  2. Use the getWidth() method to get the width of the applet window.
  3. Calculate the coordinates of the center of the applet window by subtracting half of the width from the origin (x = 0).
  4. Use the Graphics drawString() method to print the text at the newly calculated coordinates.

A Sample Program to Print Center Aligned Text Using the Applet Dimension Class

Here is a sample program that shows how to use the Applet Dimension class to print center aligned text in Java:


          
           
import java.applet.Applet;
import java.awt.*;
 
public class AppletDimensionExample extends Applet {
 
    private AppletDimension screen;
 
    public void init() {
        screen = new AppletDimension(getWidth(), getHeight());
    }
 
    public void paint(Graphics g) {
        // Get the width of the applet window
        int width = screen.getWidth();
 
        // Calculate the coordinates of the center of the applet window
        int x = (int)Math.round(width / 2);
 
        // Print the text at the center coordinates
        g.drawString("Hello World!", x, 50);
    }
}

In this sample program, we first instantiate an Applet Dimension object by passing in the desired width and height. We then use the getWidth() method to get the width of the applet window, which we then use to calculate the coordinates of the center of the applet window. Finally, we use the Graphics drawString() method to print the text at the newly calculated coordinates.
