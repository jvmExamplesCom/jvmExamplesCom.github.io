Set Status Message in Applet Window Example

Java applets can display messages in an applet window. This guide will show you how to set the status message in an applet window in a Java program.

Example Code

The following code example shows how to set the status message in an applet window: 

          
           
appletWindow.setStatus("This is the status message");
The code above sets the status message of the applet window to the string "This is the status message".

Example Output

When the program is run, the output in the applet window will be the status message set in the code example: "This is the status message".

Compiling and Running the Program

To compile the program, use the 

          
           
javac
command: 

          
           
javac MyApplet.java
To run the program, use the 

          
           
appletviewer
command: 

          
           
appletviewer MyApplet.java

Full Code Example

The following is the full code example for setting the status message in an applet window: 

          
           
import java.applet.*;

public class MyApplet extends Applet {
    public void init() {
        AppletWindow appletWindow = this.getAppletWindow();
        appletWindow.setStatus("This is the status message");
    }
}
