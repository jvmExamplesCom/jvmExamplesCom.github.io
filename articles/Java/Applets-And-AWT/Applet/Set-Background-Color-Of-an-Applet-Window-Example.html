Setting the Background Color of an Applet Window Example

The following example demonstrates how to set the background color of an applet window.

Code Example


          
           
// Applet Window 
JFrame frame = new JFrame("Applet Window"); 
frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE); 

// Applet 
JPanel panel = new JPanel(); 
frame.add(panel); 

// Set Background Color 
panel.setBackground(new Color(200, 175, 255)); 

// Display 
frame.pack(); 
frame.setVisible(true);
The 

          
           
new Color(200, 175, 255)
sets the background color of the applet window to a medium shade of purple.
