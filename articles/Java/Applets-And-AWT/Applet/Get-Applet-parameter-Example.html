Get Applet Parameter Example in Java

Java provides the 

          
           
getParameter
method to retrieve parameters from an applet. To get an applet parameter programmatically, use the 

          
           
getParameter
method of the 

          
           
Applet
class. This method takes the parameter name as the argument and returns a 

          
           
String
, containing the parameter's value.

Syntax: 


          
           
public String getParameter(String parameterName)

Example:

Let’s take an example to understand the concept of using 

          
           
getParameter
to retrieve applet parameter: 

          
           
import java.applet.Applet; 
import java.awt.Graphics; 

public class GetParamEx extends Applet { 
	String phrase; 
		
	public void init() { 
		phrase = getParameter("greetingMsg"); 
	}	 
	// We will print the parameter by overriding paint() method. 
	public void paint(Graphics g) { 
		g.drawString(phrase, 10, 50); 
	} 
}
The HTML code for the above example is shown below: 

          
           
<html>
  <body>
    <applet code="GetParamEx.class" width="200" height="200">
      <param name="greetingMsg" value="Welcome! User."/>
    </applet>
  </body>
</html>
When the above code is compiled and executed, it will produce the following output − 

          
           
Welcome! User.
