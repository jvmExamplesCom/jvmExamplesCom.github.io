Load New HTML File Using Applet Context Example

This example shows how to use the 

          
           
getAppletContext()
method of 

          
           
Applet
class and the 

          
           
showDocument()
method of 

          
           
AppletContext
interface to load a new HTML file from the same directory in which an applet .class file is maintained.

Example Code


          
           
import java.applet.*;
import java.awt.*;

public class LoadHtmlFile extends Applet {
   Button b1;

   public void init(){
      b1 = new Button("Click");
   }
 
   public void start(){
      add(b1);
      b1.addActionListener(ae -> getAppletContext().showDocument(getCodeBase() + "html_file.html"));
   }
}
The code above creates a button called 

          
           
b1
, which is added to the applet when 

          
           
start()
method is called. The 

          
           
b1
button has an action listener registered, which calls the 

          
           
showDocument
method of the 

          
           
AppletContext
object with the URL of the HTML file which needs to be loaded.

Output


          
           
When the user clicks on the "Click" button, the HTML file "html_file.html" from the same location as that of the .class file of the applet is opened.
