Draw Smiley In Applet Example

In this article, we’ll learn how to draw a smiley in an applet example. This example will demonstrate the use of the Graphics class, the Applet class and the Color class of Java.

Java Applet Overview

An applet is a small Java program that runs inside a web browser. It is used for creating interactive web applications. To create a Java applet, we need to extend the Applet class. Then, we'll override the paint() method of the Applet class.

Graphics Class Overview

The Graphics class contains a number of methods that can be used to draw shapes, such as lines, rectangles, circles, etc. The Graphics class also supports drawing text, setting font and color, printing images etc.

Color Class Overview

The Color class is used to define colors and is a part of java.awt package. A Color object can be created by specifying its red, green and blue values. The Color class also provides some predefined constants, such as RED, GREEN, BLUE, etc.

Draw Smiley In Applet Example

Let’s create a simple example to draw a smiley in an applet. The following code snippet creates a class called SmileyApplet that extends the Applet class. We have overridden the paint() method to draw the smiley.


          
           
import java.awt.*;

public class SmileyApplet extends Applet {
  public void paint(Graphics g) {
    // Set background color
    setBackground(Color.BLUE);
    
    // Draw the face
    g.setColor(Color.YELLOW);
    g.fillOval(10, 10, 200, 200);
   
    // Draw the eyes
    g.setColor(Color.BLACK);
    g.fillOval(55, 65, 30, 30);
    g.fillOval(135, 65, 30, 30);
    
    // Draw the mouth
    g.setColor(Color.RED);
    g.fillArc(50, 110, 120, 60, 0, -180);
  }
}

Here, we have first set the background color to blue using the setBackground() method. Then, we have drawn the face of the smiley by using the fillOval() method. Next, we have drawn the eyes of the smiley by using the fillOval() method. Finally, we have drawn the mouth of the smiley by using the fillArc() method.
