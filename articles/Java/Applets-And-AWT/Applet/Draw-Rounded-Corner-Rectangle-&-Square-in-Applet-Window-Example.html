Draw Rounded Corner Rectangle & Square in Applet Window Example

The following Java applet example demonstrates the usage of classes from 

java.awt
package to draw a rounded corner rectangle and squares in a window.

Code Example


          
           
import java.applet.Applet; 
import java.awt.Graphics; 
import java.awt.Color; 
  
public class RoundedRect extends Applet { 
      
    public void paint(Graphics g) { 
        g.setColor(Color.gray); 
        g.fillRoundRect(120, 15, 75, 75, 50, 50); 
          
        g.setColor(Color.green); 
        g.fillRoundRect(20, 15, 75, 75, 50, 50); 
          
        g.setColor(Color.orange); 
        g.fillRect(20, 135, 95, 75); 
          
        g.setColor(Color.blue); 
        g.drawRoundRect(20, 15, 75, 75, 50, 50); 
    } 
}

The following HTML code illustrates how to call the above code example within an applet tag:


          
           
<applet code="RoundedRect.class" width="400" height="400"> </applet>
