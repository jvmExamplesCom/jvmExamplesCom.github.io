The following example demonstrates how to fill a rectangle in an Applet window. It uses the Window class to create the Applet window. The Paint method is used to draw the rectangle. The Graphics object is used to access the graphical properties of the window. The drawRect() and fillRect() methods are used to draw and fill the rectangle. 

          
           
import java.awt.*;
import java.applet.*;

// Create Window class
public class FillRectangleSquare extends Applet {
   private Window window;

      window = new Window(400, 400);

   }
