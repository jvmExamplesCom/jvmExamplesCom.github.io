Window window = new Window (400, 400);

Graphics Object

The Graphics object is used to access the graphical properties of the window. It has many useful methods including drawRect() and fillRect(). Example: Filling a Rectangle

The following example demonstrates how to fill a rectangle in an Applet window. It uses the Window class to create the Applet window. The Paint method is used to draw the rectangle. The Graphics object is used to access the graphical properties of the window. The drawRect() and fillRect() methods are used to draw and fill the rectangle. 

          
           
import java.awt.*;
import java.applet.*;

// Create Window class
public class FillRectangleSquare extends Applet {
   private Window window;
 
   // Constructor
   public FillRectangleSquare() {
      window = new Window(400, 400);
   }
   
   // Paint method
   public void paint(Graphics g)
   {
      // Draw rectangle on the Applet window
      g.drawRect(100, 100, 200, 100);

      // Fill rectangle on the Applet window
      g.fillRect(100, 100, 200, 100);
   }
}
