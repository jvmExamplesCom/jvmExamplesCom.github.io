Draw Line in Applet Window Example

Java applets can be used to create graphical applications. In this tutorial, we will learn how to draw a line in the applet window with an example. To draw a line in an applet window, the following lines of code are required: 

          
           
import java.awt.*;  
public void paint(Graphics g) 
{  
	g.setColor(Color.RED);  
	g.drawLine(20, 30, 80, 100);  
}
Let's understand each line of code one by one:

Import Statement

The first line of code imports the java.awt package. The java.awt package provides access to classes for creating user interfaces and for painting graphics and images.

paint() Method

The second line of code is the paint() method which contains all the statements for drawing the line.

g.setColor() Method

The third line of code sets the color of the line using the setColor() method which takes a Color object as its parameter.

g.drawLine() Method

The fourth line of code draws the line in the applet window using the drawLine() method which takes four parameters - the x-coordinate of start point, the y-coordinate of start point, the x-coordinate of end point and the y-coordinate of end point.

Output

When the above program is compiled and run, it produces the following output: 

          
           
A red line is drawn in the applet window from (20, 30) to (80, 100).
