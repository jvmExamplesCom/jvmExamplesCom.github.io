Get Maximum Memory Available to Java Virtual Machine (JVM) Example

Java Virtual Machine (JVM) provides a dynamic environment for executing Java applications. It allocates the memory needed by the application and makes sure the application never runs out of memory. However, the memory that is available to the JVM has a maximum limit. The following example shows how to get the maximum memory available to the JVM.

Example

The following code creates a Runtime instance and then uses it to access the maximum amount of memory available to the JVM: 

          
           
Runtime rt = Runtime.getRuntime(); 
long maxMemory = rt.maxMemory(); 
System.out.println("Maximum memory available to JVM: " + maxMemory);
The output will look something like this: 

          
           
Maximum memory available to JVM: 134217728
In this example, the maximum amount of memory available to the JVM is 134,217,728 bytes (128 MB).
