Terminate Java Virtual Machine (JVM) Example

In this article, we'll demonstrate how to terminate JVM in Java. This is a useful process, as it will allow you to free up memory and CPU resources that would otherwise be taken up by Java programs running on the JVM. We'll also offer some code examples to help you terminate JVM in your own application.

What is JVM?

Java Virtual Machine (JVM) is a virtual machine that enables a computer to run a Java program. It provides the runtime environment in which Java bytecode can be executed. The JVM is a crucial component of the Java platform. It is a part of the Java Runtime Environment (JRE) and is responsible for its execution.

How to Terminate JVM in Java?

You can use the 

          
           
Runtime.getRuntime().halt()
method to forcibly terminate JVM in your Java application. It takes an exit status argument and terminates the instance of JVM with the specified exit status. Here is an example code snippet for using the halt() method to terminate JVM in Java: 

          
           
int exitStatus = 0;
Runtime.getRuntime().halt(exitStatus);

The above code snippet terminates the JVM instance with the given exit status. The exit status can be any integer value.

Terminate JVM with Shutdown Hook

Another way to terminate JVM in Java is to use the shutdown hook feature. Shutdown hook is a thread that runs when the JVM is shutting down. This can be used to perform any clean-up tasks that need to be done before the JVM is terminated. Here is an example of using shutdown hook to terminate JVM in Java:


          
           
public class ShutdownHookExample {
    public static void main(String[] args) {
        Runtime.getRuntime().addShutdownHook(new Thread() {
            @Override
            public void run() {
                System.out.println("Terminating JVM...");
                Runtime.getRuntime().halt(0);
            }
        });
    }
}

The above example creates a new thread and adds it as a shutdown hook to the JVM. When the JVM is shutting down, it will run the thread and call the 

halt()
method to terminate the JVM instance.

Conclusion

In this article, we discussed how to terminate JVM in Java using the 

halt()
method and shutdown hook. We also provided some code examples to illustrate the usage of these methods. Terminating JVM can be a useful process, as it allows you to free up memory and CPU resources that would otherwise be taken up by Java programs running on the JVM.
