Get Available Processors using Java Runtime Example

The Java Runtime Class provides the getAvailableProcessors() method that can be used to find the number of available processors in a system. This tutorial will explain how to use the getAvailableProcessors() method to find the number of available processors in a system using Java.

Java Runtime getAvailableProcessors() Method

The Java Runtime getAvailableProcessors() method returns the number of processors available to the Java virtual machine. It returns an int value which describes the number of processors available. Here is the syntax for the method: 

          
           
public int getAvailableProcessors()

Java Runtime getAvailableProcessors() Example

Here is an example of how to use the getAvailableProcessors() method in a Java program. This example prints out the number of available processors in the system. 

          
           
import java.lang.Runtime;

public class JavaRuntimeGetProcessorsExample {

   public static void main(String args[]) {

      // get the current runtime object
      Runtime runtime = Runtime.getRuntime();

      // print the total number of processors
      System.out.println("Number of processors: " + runtime.availableProcessors());
   }
}
When you compile and run this Java program, it will produce the following output: 

          
           
Number of processors: 8
