JVM to Run Object Finalization Example

Java consists of an integrated development environment,JVM(Java virtual machine), and a class library. The automatic memory management system closes all the existing objects from the JVM before the program is terminated. This process is known as Object Finalization in Java. This article discusses how developers can use the JVM to demonstrate the object finalization process in Java.

Object Finalization: What is it?

Object finalization in Java is a special method that is used to remove unused objects from the JVM. This means that when the JVM is about to exit, it calls the finalize() method on all the objects that are still active in the application. This helps in cleaning up resources that are no longer needed by the application or by the user.

Object Finalization Example

Let’s take a look at an example of Object Finalization in Java. 

          
           
class Foo {
    public void finalize() {
        System.out.println("Finalizing object...");
    }
}
 
public class Test {
    public static void main(String[] args) {
        Foo f = new Foo();
        f = null;
        System.gc();
    }
}

Foo
which has a 

          
           
finalize()
method. We then create an instance of 

          
           
Foo
and set it to 

          
           
null
. Finally, the 

          
           
System.gc()
method is called which instructs the JVM to perform garbage collection. When it does, it will call the 

          
           
finalize()
method on all the objects that are no longer in use. As a result, the output of this program will be: 

          
           
Finalizing object...

Conclusion

Object finalization is an important concept in Java and is used to remove unused objects from the JVM. It is especially important when using a JVM because it helps the JVM clean up resources that are no longer needed.
