Get File Separator Using System Class in Java

The 

          
           
File.separator
or 

          
           
System.getProperty("file.separator")
is a static variable from java.io.File class and system environment. It is used to get standard separator for platform-independent usage. In simpler words, it returns the character that is used to separate directory names in the file system.

Syntax of File Seperator


          
           
public static final String separator

Example of File Separator

Let's take a look at an example. 

          
           
public class TestFileSeparator {
    public static void main(String[] args) {
        // Print the file seperator
        String seperator = System.getProperty("file.separator");
        System.out.println("File Separator = " + seperator);
    }
}
In the above code snippet, we have used 

          
           
System.getProperty("file.separator")
to get the file separator.
