Get system properties using System Class

Java applications typically rely on system properties to configure their behavior. System properties can be set directly from the command line, or programmatically via the System class. In this tutorial, we'll explore how to use the System class to get and set system properties.

Using the getProperty() Method

The most straightforward way to get a system property is to use the 

          
           
System.getProperty()
method. This method takes a single argument, which is a String representing the name of the property you wish to retrieve. For example, the following code retrieves the value of the "user.home" property: 

          
           
String homeDir = System.getProperty("user.home");
In this case, the 

          
           
homeDir
variable will contain the path to the user's home directory. There is also an overloaded version of 

          
           
getProperty()
which takes two arguments. The first argument is the same as the one-argument version, and the second argument is the default value to use in case the specified property does not exist. This can be useful if you want to ensure that your application still functions even if certain properties are missing. For example, the following code retrieves the value of the "my.property" property, and falls back to a default value if it does not exist: 

          
           
String myProp = System.getProperty("my.property", "defaultValue");

Using the getProperties() Method

Another useful method is 

          
           
System.getProperties()
, which returns an instance of 

          
           
java.util.Properties
. This is a convenience method which allows us to traverse all system properties at once. For example, the following code retrieves all system properties and prints them to the console: 

          
           
Properties props = System.getProperties();
for (String key : props.stringPropertyNames()) {
    System.out.println(key + "=" + props.getProperty(key));
}

Using the setProperty() Method

It's also possible to set system properties programmatically. This is done using the 

          
           
System.setProperty()
method, which takes two arguments: the first is the name of the property to set, and the second is the value to assign to the property. For example, the following code sets the value of the "my.property" property to "someValue": 

          
           
System.setProperty("my.property", "someValue");
Once a property has been set, it will supersede any value associated with it in the system properties. However, this change may only be visible within the current application. In other words, setting a system property will not affect the value of that property in other applications.

Conclusion

In this tutorial, we've explored how to use the System class to get and set system properties. We looked at the 

          
           
getProperty()
and 

          
           
getProperties()
methods, as well as the 

          
           
setProperty()
method.
