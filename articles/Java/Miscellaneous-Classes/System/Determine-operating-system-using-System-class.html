Determining Operating System using System class

The 

          
           
System
class in Java provides a great way to detect the operating system (OS) on which your program is running. This is useful for writing code that adapts to the OS environment, or to conditionally compile code based on the OS. To use the 

          
           
System
class, you will first import it into your program. This is done by placing the following line of code at the top of your file: 

          
           
import java.lang.System;
Next, you can use the 

          
           
getProperty()
method to determine the OS. This method takes two parameters: the first is a string that specifies the OS property that you are trying to get and the second is a default value that is returned if the property is not found. In order to determine the OS, we will use the "os.name" property. The following code example shows how to use the 

          
           
getProperty()
method to detect the OS: 

          
           
String osName = System.getProperty("os.name", "default");
The variable 

          
           
osName
will now store the name of the OS. You can then use an 

          
           
if/else
statement or a 

          
           
switch
statement to check the value of 

          
           
osName
and take appropriate action based on the OS. Here is an example of a 

          
           
switch
statement that checks the value of 

          
           
osName
and prints out a message depending on the OS: 

          
           
switch(osName) {
  case "Linux":
    System.out.println("You're running Linux!");
    break;
  case "Mac OS X":
    System.out.println("You're running Mac OS X!");
    break;
  case "Windows 10":
    System.out.println("You're running Windows 10!");
    break;
  default:
    System.out.println("I don't recognize your OS!");
    break;
}
That's all there is to it! With just a few lines of code, you can easily detect the operating system on which your program is running.
