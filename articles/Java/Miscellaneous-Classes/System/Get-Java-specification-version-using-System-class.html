Get Java Specification Version Using System Class

Java provides the 
System
class that contains various methods to get system information such as version, name and path. This class is available in the 
java.lang
package. One of the useful methods is 
getProperty()
which is used to return values of the specified property. The following code snippet shows how to use 
System
class to get Java specification version: 

          
           
String javaSpec = System.getProperty("java.specification.version");
System.out.println("The Java specification version is: " + javaSpec);
The output of the code above will be: 

          
           
The Java specification version is: 10
We can also use 
java.version
property to get the same result: 

          
           
String javaVer = System.getProperty("java.version");
System.out.println("The Java version is: " + javaVer);
The output of the code above will be: 

          
           
The Java version is: 10.0.2+

To get the Java implementation version, we can use the 


          
           
java.runtime.version
property: 

          
           
String javaRuntimeVersion = System.getProperty("java.runtime.version");
System.out.println("The Java runtime version is: " + javaRuntimeVersion);
The output of the code above will be: 

          
           
The Java runtime version is: 10.0.2+19
