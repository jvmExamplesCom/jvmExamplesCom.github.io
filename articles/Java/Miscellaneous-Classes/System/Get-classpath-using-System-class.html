Getting the Java Classpath Using System Class

The 

          
           
System
class in the Java language provides a collection of useful methods and variables, such as the classpath. In this tutorial, we'll look at how to use the 

          
           
System
class in order to get the classpath from a Java application.

Getting the Classpath

The first step to getting the classpath is to create a new instance of 

          
           
System
. This can be done by using the following code: 

          
           
System system = new System();
Once you have an instance of the 

          
           
System
class, you can use the 

          
           
getProperty()
method to retrieve the classpath. This is done by passing the argument 

          
           
"java.class.path"
to the 

          
           
getProperty()
method. The following code snippet shows an example of how to do this: 

          
           
String classPath = system.getProperty("java.class.path");

Using the Classpath

Once you have the classpath retrieved, you can then use it however you'd like. One common use case is to use the classpath to add additional JARs to your classpath. This is done by adding the JARs to the 

          
           
ClassLoader
object for the current application. The following code snippet shows an example of how to do this: 

          
           
ClassLoader classLoader = Thread.currentThread().getContextClassLoader();
classLoader.addURL(new URL(classPath));
This will add all of the JARs in the classpath to the 

          
           
ClassLoader
.
