Get System Time using System Class

Java provides the 

          
           
System
class which provides access to system resources, such as system time. In Java, the current system time can be obtained using the 

          
           
currentTimeMillis()
method of the 

          
           
System
class. This method returns the total number of milliseconds since Jan 1, 1970 UTC.

Example

Below is an example of how to get the system time in Java: 

          
           
long currentTime = System.currentTimeMillis();
System.out.println("Current time: " + currentTime);
The output of the above code will be the system time in milliseconds since Jan 1, 1970 UTC. For example: 

          
           
Current time: 1587410402207
In order to convert this long type timestamp into a 

          
           
Date
object, we can use the 

          
           
Instant
class, as shown below: 

          
           
long currentTime = System.currentTimeMillis();
Instant instant = Instant.ofEpochMilli(currentTime);
Date date = Date.from(instant);
System.out.println("Current time (date object): " + date.toString());
The output of the above code will be the Date object representing the current system time. For example: 

          
           
Current time (date object): Mon Apr 20 17:20:02 EDT 2020

Conclusion

In this tutorial, we looked at how to get the current system time in Java using the 

          
           
System
class. We saw how to convert the system time in milliseconds to a 

          
           
Date
object using the 

          
           
Instant
class.
