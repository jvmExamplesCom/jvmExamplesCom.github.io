Java Double isInfinite Example

A double data type is used to represent the primitive value of the double type in Java. The double data type has a precision of at least 15 decimal places. It is a 64-bit IEEE 754 floating-point number. In this article, we will learn about the Java Double isInfinite example.

What is the isInfinite() Method?

The Java Double isInfinite() method is a static method of the Double class that indicates whether the specified double value if infinite or not. The method returns a boolean value, which is true when the double value is infinite, and false otherwise. Syntax of the method is as follows: 

          
           
public static boolean isInfinite(double value)

Code Examples

Let’s look at some code examples to understand how the isInfinite() method works. Example 1: 

          
           
public static void main(String[] args) {
    double value = 10.0;
    boolean res = Double.isInfinite(value);
    System.out.println(res);
}
In the above example, we are passing the double value 10.0 to the isInfinite() method. Since, 10.0 is not an infinite number the output will be false. Example 2: 

          
           
public static void main(String[] args) {
    double value = Double.POSITIVE_INFINITY;
    boolean res = Double.isInfinite(value);
    System.out.println(res);
}
In the above example, we are passing Double.POSITIVE_INFINITY to the isInfinite() method. Since, Double.POSITIVE_INFINITY is an infinite number the output will be true. Example 3: 

          
           
public static void main(String[] args) {
    double value1 = Double.POSITIVE_INFINITY;
    double value2 = Double.NEGATIVE_INFINITY;
    System.out.println(Double.isInfinite(value1));
    System.out.println(Double.isInfinite(value2));
}
In the above example, we are passing both the Double.POSITIVE_INFINITY and the Double.NEGATIVE_INFINITY values to the isInfinite() method. The output will be true for both.
