Java Double isInfinite Example

A double data type is used to represent the primitive value of the double type in Java. The double data type has a precision of at least 15 decimal places. It is a 64-bit IEEE 754 floating-point number. In this article, we will learn about the Java Double isInfinite example.

What is the isInfinite() Method?

The Java Double isInfinite() method is a static method of the Double class that indicates whether the specified double value if infinite or not. The method returns a boolean value, which is true when the double value is infinite, and false otherwise. Syntax of the method is as follows: 

          
           
public static boolean isInfinite(double value)
