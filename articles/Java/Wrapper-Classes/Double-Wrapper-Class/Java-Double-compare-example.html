Java Double Compare Example

Double variables in Java are used to store double-precision values. The double data type is a primitive type and it is used when the precision of float is not enough. Comparing two doubles can be tricky because they are not always accurate. This article explains how to compare two Double variables in Java with examples.

Java Double Comparison Code Examples

The below code shows how to compare two double variables in Java.


          
           
double d1 = 0.1;
double d2 = 0.2;
if (d1 == d2) {
    System.out.println("The doubles are equal");
}
else {
    System.out.println("The doubles are not equal");
}
The output of this code will be: 

          
           
The doubles are not equal

The above example used the equality operator (==) to compare two double variables. This method is not recommended because of rounding errors when checking for equality. Therefore, it is better to use a comparison method.

The below code compares two double variables using the Math.abs() method.


          
           
double d1 = 0.1;
double d2 = 0.2;
if (Math.abs(d1 - d2) < 0.00001) {
    System.out.println("The doubles are equal");
}
else {
    System.out.println("The doubles are not equal");
}
The output of this code will be: 

          
           
The doubles are equal

In the above example, the difference between the two double variables is compared to the threshold value of 0.00001. The threshold value can be adjusted according to the precision required. If the difference between the two doubles is less than the threshold value, then the doubles are equal.

Conclusion

Comparing two double variables in Java requires special care. The equality operator (==) should not be used as it can lead to incorrect results due to rounding errors. Instead, the difference between the two doubles should be compared with a threshold value that is suitable for the required precision.
