Convert Java Double to Numeric Primitive Data Types Example

In this article we will discuss how to convert a Java double to numeric primitive data types. We will also look at different code examples and understand the syntax and usage of each example.

Java Double to Integer Conversion

We can convert a Java double to an integer by using the casting operator. For instance:


          
           
double d = 5.0;
// Cast double to int
int i = (int)d;
System.out.println("Integer value is " + i);

The output of the above code will be: Integer value is 5.

Java Double to Byte Conversion

We can also convert a Java double to a byte by using the casting operator. For instance:


          
           
double d = 127.0;
// Cast double to byte
byte b = (byte)d;
System.out.println("Byte value is " + b);

The output of the above code will be: Byte value is 127.

Java Double to Long Conversion

We can also convert a Java double to a long by using the casting operator. For instance:


          
           
double d = 4561234.0;
// Cast double to long
long l = (long)d;
System.out.println("Long value is " + l);

The output of the above code will be: Long value is 4561234.

Java Double to Float Conversion

double d = 3.14;
// Cast double to float
float f = (float)d;
System.out.println("Float value is " + f);

The output of the above code will be: Float value is 3.14.

Java Double to Short Conversion

We can also convert a Java double to a short by using the casting operator. For instance:


          
           
double d = 32767.0;
// Cast double to short
short s = (short)d;

The output of the above code will be: Short value is 32767.
