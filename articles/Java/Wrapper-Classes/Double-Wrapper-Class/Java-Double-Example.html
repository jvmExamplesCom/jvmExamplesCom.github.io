Java Double Example

Double is a primitive data type in Java programming language. It is one of the eight primitive data types provided in Java and has a wide range of uses. This article will discuss how to create and use double variables and how to perform common operations.

Creating a Double Variable

To create a double variable, use the double keyword: 

          
           
double myVar = 0.5;
This assigns an initial value of 0.5 to the variable named “myVar.”

Using a Double Variable

Double variables can be used in calculations, comparisons, or assignments to other variables. For example, to add two double values together, use the following code: 

          
           
double sum = 3.2 + 4.1;
This code creates a new double variable called “sum” and assigns the result of the addition operation to it. Similarly, the “==” operator can be used to compare two doubles for equality, as with the following code: 

          
           
boolean equal = 3.0 == 3.0;
This Boolean variable “equal” will be set to true since both values are the same. Additionally, doubles can also be assigned to other variables: 

          
           
double x = 4.7; 
double y = x;
In this case, the value of the “x” variable is assigned to the “y” variable.

Formatting Double Values

By default, double values are printed using scientific notation. However, it is also possible to format doubles to print out in human-readable formats. To do so, use the String.format() method: 

          
           
double number = 3.145678; 
String formattedNumber = String.format("%.3f", number);
System.out.println(formattedNumber); // Prints “3.146”
In this example, the “number” double is formatted to have three decimal places and then used to print out the formatted string.

Converting Doubles to Strings

The Double class provides a handy method to convert a double to a String. To use it, first import the Double class: 

          
           
import java.lang.Double;
Then use the Double.toString() method: 

          
           
double myValue = 3.14; 
String str = Double.toString(myValue);
System.out.println(str); // Prints “3.14”
This code creates a String that holds the same value as the double.

Rounding Double Values

Finally, it is also possible to round double values to a certain number of decimal places. To do this, use the Math.round() method: 

          
           
double x = 3.145678; 
int roundedVal = (int) Math.round(x * 1000) / 1000;
System.out.println(roundedVal); // Prints “3.146”
This code multiplies the double by 1000, rounds it to the nearest integer, and then divides it by 1000 to get the desired result.

Conclusion

Double is a versatile primitive data type in Java and can be used in many different ways. Hopefully this article has given you a better understanding of how to create, use, and format double variables.
