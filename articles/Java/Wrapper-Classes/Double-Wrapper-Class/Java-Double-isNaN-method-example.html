Java Double isNaN Method Example

The Double class provides the method isNaN() to check if an object contains value that is not a number. It is part of the java.lang package and can be used to detect whether a double or float value is not a valid number. In this article, we will explore the usage of the Java Double isNaN() method with examples.

Syntax

The syntax of the Java Double isNaN() is as follows: 

          
           
public static boolean isNaN(double d)
This method takes in a double value as an argument and returns a boolean value. The returned boolean value is true if the passed value is NaN (Not a Number) else it will return false.

Example

Let us consider a simple example to understand the usage of the isNaN() method. 

          
           
double x = Double.NaN;
boolean result = Double.isNaN(x);
System.out.println("Result: " + result);
Output: 

          
           
Result: true
In this example, we have created a double variable x and assigned it with a Double.NaN value. To check if x contains a valid number, we use the isNaN() method. As expected, the call to isNaN() returned true indicating that x is a Not a Number.
