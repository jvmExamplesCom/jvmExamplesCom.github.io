Converting Double Object to String Object in Java

Java has built-in methods and classes to support converting a Double object to a String object. In this tutorial, we'll learn the different ways of achieving this conversion and see how they work together.

Using the toString() Method

The most straightforward way to convert a Double object to String is by using the 

          
           
toString()
method. The syntax for calling the 

          
           
toString()
method is: 

          
           
doubleValue.toString();
Here, 

          
           
doubleValue
is the Double object that you want to convert to a String. For example, let's say that you have a 

          
           
Double
object named 

          
           
dNum
with a value of 3.14159. To convert it to a String, you would write: 

          
           
String str = dNum.toString();
After executing this statement, the value of 

          
           
str
will be 

          
           
"3.14159"
.

Using the String.valueOf() Method

Another way to convert a Double object to a String object is by using the 

          
           
String.valueOf()
method. The syntax for calling the 

          
           
String.valueOf()
method is: 

          
           
String.valueOf(doubleValue);
Here, 

          
           
doubleValue
is the Double object that you want to convert to a String. For example, let's say that you have a Double object named 

          
           
dNum
with a value of 3.14159. To convert it to a String, you would write: 

          
           
String str = String.valueOf(dNum);
After executing this statement, the value of 

          
           
str
will be 

          
           
"3.14159"
.

Using the String Constructor

Another way to convert a Double object to a String object is by using the String constructor. The syntax for calling the String constructor is: 

          
           
new String(doubleValue);
Here, 

          
           
doubleValue
is the Double object that you want to convert to a String. For example, let's say that you have a 

          
           
Double
object named 

          
           
dNum
with a value of 3.14159. To convert it to a String, you would write: 

          
           
String str = new String(dNum);
After executing this statement, the value of 

          
           
str
will be 

          
           
"3.14159"
.

Using the DecimalFormat Class

If you need to format your Double object before converting it to a String, then you can use the 

          
           
DecimalFormat
class to do so. The syntax for creating a 

          
           
DecimalFormat
object is: 

          
           
DecimalFormat fmt = new DecimalFormat(pattern);
Here, 

          
           
pattern
is a string specifying the format of the output. For example, let's say that you have a 

          
           
Double
object named 

          
           
dNum
with a value of 3.14159. To convert it to a String, with two decimal places, you would write: 

          
           
DecimalFormat fmt = new DecimalFormat("#.00");
String str = fmt.format(dNum);
After executing these statements, the value of 

          
           
str
will be 

          
           
"3.14"
.

Conclusion

In this tutorial, we learned the different ways of converting a Double object to a String object in Java. We saw that the most straightforward way is to use the 

          
           
toString()
method, but if you need to format the output, then you can use the 

          
           
DecimalFormat
class.
