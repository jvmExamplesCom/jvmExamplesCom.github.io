How to Find Square Root of BigInteger Example in Java

The BigInteger class in Java provides support for arbitrary-precision integer arithmetic, allowing you to compute the square root of a BigInteger. In this article, we'll look at an example program that finds the square root of a BigInteger in Java.

Example Program: Find Square Root of BigInteger

Here's an example program that finds the square root of a BigInteger in Java. The program takes a single argument - the BigInteger whose square root needs to be calculated - and prints out the result. 

          
           
import java.math.BigInteger;

public class BigIntSqrt {
  public static void main(String[] args) {
    if (args.length != 1) {
      System.err.println("Syntax: BigIntSqrt <number>");
      System.exit(1);
    }

    BigInteger n = new BigInteger(args[0]);
    BigInteger sqrt = bigIntSqrt(n);
    System.out.println("The square root of " + n + " is " + sqrt);
  }

  public static BigInteger bigIntSqrt(BigInteger x) {
    BigInteger div = BigInteger.ZERO.setBit(x.bitLength()/2);
    BigInteger div2 = div;
    // Loop until we hit the same value twice in a row, or wind
    // up alternating.
    for(;;) {
      BigInteger y = div.add(x.divide(div)).shiftRight(1);
      if (y.equals(div) || y.equals(div2))
        return y;
      div2 = div;
      div = y;
    }
  }

In this program, we are using the BigInteger class from the Java Math library to calculate the square root of a BigInteger. First, we read the value of the BigInteger to be squared from the command line arguments. Then, we use the bigIntSqrt() method to calculate the square root of the number. The bigIntSqrt() method is a simple recursive algorithm that uses repeated divisions and shifts to calculate the square root. We start by setting the div variable to half of the bit length of the number, then loop until we reach a steady state or alternating values. The y variable is the value of the result, computed by adding the div variable and the result of dividing the number by the div variable, then shifting the result right by one bit. Finally, we print out the result.
