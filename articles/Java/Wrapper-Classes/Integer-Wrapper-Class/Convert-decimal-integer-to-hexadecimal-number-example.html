Converting Decimal Integer to Hexadecimal Number

In some programming applications, it is necessary to convert a decimal integer into its corresponding hexadecimal equivalent. This is particularly useful when working with Color values, where the color codes must be specified in hexadecimal values. This article will explain how to convert a decimal integer into its hexadecimal equivalent in Java.

Example Conversion

To demonstrate an example of converting a decimal integer to its hexadecimal equivalent, we will use the following integer:


          
           
int decimal = 15;

To convert this number to its hexadecimal equivalent, we can use the following Java code:


          
           
String hex = Integer.toHexString(decimal);
The output of this code is as follows: 

          
           
0xF

The "0x" prefix indicates that the output value is a hexadecimal number. Thus, the hexadecimal equivalent of our original decimal integer is 0xF.

Longer Code Example

We can further elaborate on our example by including an example of a complete program which takes a decimal integer as input, and prints its hexadecimal equivalent. The following code snippet is an example of such a program:


          
           
public class DecimalHex {
    public static void main(String[] args) {
        // Take input from command line
        int decimal = Integer.parseInt(args[0]);

        // Convert decimal to hexadecimal
        String hex = Integer.toHexString(decimal);

        // Print result
        System.out.println("Decimal Value: " + decimal);
        System.out.println("Hexadecimal Equivalent: 0x" + hex);
    }
}

This program takes a decimal integer as input, and prints its hexadecimal equivalent. For example, if the following command is used:


          
           
java DecimalHex 15
The output of the program would be: 

          
           
Decimal Value: 15
Hexadecimal Equivalent: 0xF

As mentioned before, the "0x" prefix indicates that the output value is a hexadecimal number.

Conclusion
