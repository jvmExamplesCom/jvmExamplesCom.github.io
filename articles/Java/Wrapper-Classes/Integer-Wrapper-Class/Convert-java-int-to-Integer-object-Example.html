Converting an int to Integer Object in Java

Java's Integer class provides several methods and properties for converting an int data type to an Integer Object. This tutorial will show how to convert an int to an Integer Object in Java using three different methods.

Using the valueOf() Method

The most common way of converting an int to an Integer Object is to use the static 

          
           
valueOf()
method of the Integer class. This method has two versions: one which takes a primitive int type and one which takes a String as its argument.

For example, to convert the int 12 to an Integer object: 


          
           
Integer intObj = Integer.valueOf(12);

Using the Constructor

You can also create an Integer object from an int value by using the Integer Class constructor. This constructor takes an int as its parameter and returns an Integer object.

For example, to convert the int 12 to an Integer object: 


          
           
Integer intObj = new Integer(12);

Using the parseInt() Method

The 

          
           
parseInt()
method of the Integer class can also be used to convert an int value to an Integer object. This method takes a String as its argument and returns an Integer object.

For example, to convert the int 12 to an Integer object: 


          
           
Integer intObj = Integer.parseInt("12");
