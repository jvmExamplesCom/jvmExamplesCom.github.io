Convert Java String to Integer Object Example

Objective

The objective of this article is to show how one can convert a string in a Java program into the Integer object. We will go into detail, showing sample code and explaining the different methods that can be used to achieve a successful conversion.

Overview

In Java, the Integer object is a wrapper class that wraps a primitive type int around an object-oriented interface. It is useful when you need to store numbers, compare values, or perform calculations on them. Converting a string to the Integer object is one of the most common operations performed by Java developers.

Methods of Conversion

There are three different methods for converting a string to an Integer object. The 

          
           
Integer.parseInt()
method can be used to convert strings to integers. This method takes a string as input and returns an integer value. It throws a NumberFormatException if the string does not contain a valid integer. The 

          
           
Integer.valueOf()
method is similar to the 

          
           
parseInt()
method but returns an Integer object instead of an int value. The 

          
           
valueOf()
method can also take a String as input and return an Integer object. It throws a NullPointerException if the String is null. The 

          
           
new Integer(String)
constructor can also be used to convert strings to Integer objects. This constructor creates a new Integer object and initializes it with the value of the given string. It throws a NumberFormatException if the string cannot be parsed into a valid integer.

Examples of Conversion

Using the 
parseInt()
Method

We can use the 

          
           
parseInt()
method to convert a string to an int value like so: 

          
           
String str = "123";
int i = Integer.parseInt(str);
System.out.println(i); // prints 123

Using the 
valueOf()
Method

We can use the 

          
           
valueOf()
method to convert a string to an Integer object like so: 

          
           
String str = "123";
Integer i = Integer.valueOf(str);
System.out.println(i); // prints 123

Using the Constructor

We can also use the constructor to convert a string to an Integer object like so: 

          
           
String str = "123";
Integer i = new Integer(str);
System.out.println(i); // prints 123

Conclusion

In this article, we have looked at how to convert a string to an Integer object in a Java program. We have discussed three different methods for achieving this conversion and have shown examples of each.
