Converting Decimal Integer to Octal Number Example in Java

In this article, we will discuss how to convert a decimal integer to octal number in Java. We will also provide example code snippets to illustrate the process of conversion.

Understanding Octal Numbers in Java

An octal number is a base-8 number that consists of digits from 0 to 7. Therefore, when converting a decimal number to an octal number, we need to use the base-8 numbering system. In Java, octal numbers begin with the prefix ‘0o’ or '0O', followed by one or more octal digits (0-7).

Algorithm for Converting Decimal to Octal in Java

We will use the following algorithm for converting a decimal integer to octal number in Java: 1. Divide the decimal number by 8 and store the remainder. 2. Divide the number by 8 again and store the remainder. 3. Repeat steps 1 and 2 until the number becomes 0. 4. Reverse the order of the remainders to get the octal number.

Java Code Example

Now let us take a look at a Java code example that shows how to convert a decimal integer to an octal number. 

          
           
public static int convertDecimalToOctal(int num) {
    // Initialize result
    int result = 0;

    // Set base and initial power
    int base = 1, power = 0;

    // While num is greater than 0
    while (num > 0) {
        // Get the last digit in the number
        int lastDigit = num % 8;
 
        // Add the last digit to the result
        result += lastDigit * base;
 
        // Increase the base and power
        base = base * 10;
        power++;
 
        // Remove the last digit from the number
        num /= 8;
    }
 
    // Return the result
    return result;
}
In the above code example, we first declared a variable to store the result of the conversion. Then, we initialized variables for the base and power. In each iteration, we obtain the last digit of the given decimal number and add it to the final result. We also update the base and power accordingly. Finally, we divide the decimal number by 8 to obtain the remaining part of the number, which is processed in the next iteration.
