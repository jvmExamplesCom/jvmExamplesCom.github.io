Converting String to int in Java

Java is an object-oriented programming language that can be used to create complex applications. One of the most common tasks when programming with Java is to convert a String to an int. This conversion can be done in different ways, depending on the format of the String and what type of integer you wish to obtain. In this article, we will look at some examples of how to convert a String to an int in Java.

Using the Integer.parseInt() Method

The Integer.parseInt() method is one of the most common and straightforward ways of converting a String to an int in Java. It takes a string representation of a number as an argument, and returns an int as the result.


          
           
String strNumber = "123";
int number = Integer.parseInt(strNumber);
System.out.println(number);
// Output: 123

Using the Integer.valueOf() Method

The Integer.valueOf() method is similar to the Integer.parseInt() method, but it returns a wrapper class (Integer) instead of a primitive int. The syntax for using the Integer.valueOf() method is almost identical to the syntax for using the Integer.parseInt() method:


          
           
String strNumber = "456";
Integer number = Integer.valueOf(strNumber);
System.out.println(number);
// Output: 456

Using the DecimalFormat Class

The DecimalFormat class can be used to convert Strings that contain numeric values to ints. The DecimalFormat class provides a variety of formatting options, which makes it very useful when dealing with Strings representing numbers in different formats. To use the DecimalFormat class, you must first create an instance of it and pass in the desired formatting pattern. Then, you can call its parse() method to convert a String to an int:


          
           
String strNumber = "789.0";
DecimalFormat decimalFormat = new DecimalFormat("#");
int number = decimalFormat.parse(strNumber).intValue();
System.out.println(number);
// Output: 789

Using the NumberFormat Class

The NumberFormat class is another way to convert a String to an int. Unlike the DecimalFormat class, which only supports formatting strings containing numeric values, the NumberFormat class supports a variety of number formats, including currency and percentage formats. To use the NumberFormat class, you must first create an instance of it and pass in the desired number format. Then, you can call its parse() method to convert a String to an int:


          
           
String strNumber = "$1,234";
NumberFormat numberFormat = NumberFormat.getCurrencyInstance();
int number = numberFormat.parse(strNumber).intValue();
System.out.println(number);
// Output: 1234

Conclusion

In this article, we have looked at several different ways of converting a String to an int in Java. We have seen how to use the Integer.parseInt() and Integer.valueOf() methods, as well as the DecimalFormat and NumberFormat classes. Each of these methods has its own set of advantages and disadvantages, so it is important to choose the right one for your specific situation.
