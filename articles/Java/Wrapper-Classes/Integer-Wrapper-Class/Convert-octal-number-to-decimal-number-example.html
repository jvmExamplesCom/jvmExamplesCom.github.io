Converting Octal Number to Decimal Number Using Java

Java is an incredibly popular and versatile language with a wide range of applications. It can be used to perform various operations such as converting numbers between different bases, including octal and decimal.

Overview of the Conversion Process

The process of converting an octal number, or base 8, to its equivalent decimal number, or base 10, is relatively straightforward. Each position in an octal number represents a power of 8, starting with 20 (1) and moving left. Starting from the rightmost digit, multiply each digit in the octal number by its corresponding power of 8 and sum the results to obtain the decimal equivalent.

Example Conversion

To illustrate the conversion process, let’s take a look at the following example of converting the octal number 213 to decimal:


          
           
Octal number: 213
Position  | 7 | 6 | 5 | 4 | 3 | 2 | 1 | 0
Powers of 8: 1 | 8 | 64 | 512 | 4096 | 32768 | 262144 | 2097152
Digit     | 2 | 1 | 3 |
Product   | 2 | 8 | 192 |
Sum       | 194

We can see here that the resulting decimal number is 194.

Java Code for Conversion

Now that we understand the overall process of converting from octal to decimal, let’s take a look at how this could be implemented using Java.

First, we will define a class, 
OctalToDecimal
, and create a constructor that takes the octal number as an argument: 

          
           
class OctalToDecimal {
    int octalNumber;

    public OctalToDecimal(int octalNumber) {
        this.octalNumber = octalNumber;
    }
}

Next, we will create a method to perform the conversion. This method will have a single parameter, 

octalNum
, which is the octal number we want to convert. The method will return an integer, 
decimalNum
, which is the equivalent decimal number. 

          
           
public int octalToDecimal(int octalNum) {
    int decimalNum = 0;
    int i = 0;
    while (octalNum != 0) {
        decimalNum += (octalNum % 10) * Math.pow(8, i);
        ++i;
        octalNum /= 10;
    }
    return decimalNum;
}

The while loop iterates through each digit of the octal number starting from the rightmost. For each iteration, the current digit is multiplied by the corresponding power of 8 (obtained using the 

Math.pow()
function) and added to the total. Once the loop completes, the 
decimalNum
variable will contain the equivalent decimal number.

Testing the Conversion

Finally, let’s create a simple test program to test our conversion method. We will create an instance of the 

OctalToDecimal
class and invoke the 
octalToDecimal()
passing in the octal number 213 as an argument. 

          
           
public static void main(String[] args) {
    OctalToDecimal octalToDecimal = new OctalToDecimal();
    System.out.println(octalToDecimal.octalToDecimal(213));
}

When we execute this program, we should see the result 

194
printed to the console. This confirms that our conversion code is working correctly.
