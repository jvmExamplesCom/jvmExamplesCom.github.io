Convert Long Primitive to Long Object Example

The following example demonstrates how to convert a long primitive to a Long object in Java. To convert a long primitive to a Long object, use the valueOf() method of the Long class. 

          
           
public class ConvertLongPrimitiveToLongObjectExample {

    public static void main(String[] args) {
        //convert long primitive to Long object
        long l = 10;
        Long LObj = Long.valueOf(l);
        System.out.println("Long primitive to Long object: "+LObj);
    }
}
