Convert Short to Numeric Primitive Data Types Example

In this article, we will discuss how to convert a short type to different numeric primitive data types. We will look at some examples in which we use the 

shortValue()
method of the 
Short
class in Java to convert a short type to different primitive data types.

Example 1: Convert Short To int

In this example, we will convert a short value to an int value. We will use the 

shortValue()
method of the 
Short
class to perform the conversion. 

          
           
public static void main(String[] args) { 
    //declare a short variable 
    short sh = 100; 
    
    //convert it to an int 
    int i = sh.shortValue(); 
    
    //print the int value 
    System.out.println("int value: " + i);
