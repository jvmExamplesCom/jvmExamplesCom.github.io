Convert Short to Numeric Primitive Data Types Example

Example 1: Convert Short To int

In this example, we will convert a short value to an int value. We will use the 

shortValue()
method of the 
Short
class to perform the conversion. 

          
           
public static void main(String[] args) { 
    //declare a short variable 
    short sh = 100;
