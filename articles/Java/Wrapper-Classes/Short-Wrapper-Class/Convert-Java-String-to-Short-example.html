Java String to Short Conversion

The 'String' class in Java provides a method called 'parseShort' which is used to convert a string into a primitive short data type. This conversion is possible when the string contains numerical values. In this tutorial, we will discuss the steps on how to convert a given string into a short data type in Java.

Example 1: Convert a string to short using 'parseShort' method


          
           
String s = "12345";
short sh = Short.parseShort(s);
System.out.println(sh);

In the above code snippet, we define the string 's' with a numerical value. Then, we use the 'parseShort' method to convert the string variable 's' into the short variable 'sh'. Finally, we print out the converted variable 'sh' to see the result. The output will look like this:


          
           
12345

Example 2: Convert a string to short using 'valueOf' method


          
           
String s = "54321";
short sh = Short.valueOf(s);
System.out.println(sh);

In the above code snippet, we define the string 's' with a numerical value. Then, we use the 'valueOf' method to convert the string variable 's' into the short variable 'sh'. Finally, we print out the converted variable 'sh' to see the result. The output will look like this:


          
           
54321

Example 3: Convert a string to short using Long and shortValue method


          
           
String s = "100";
long l = Long.parseLong(s);
short sh = (short) l.shortValue();
System.out.println(sh);

In the above code snippet, we define the string 's' with a numerical value. Then, we use the 'parseLong' method to convert the string variable 's' into the long variable 'l'. Next, we use the 'shortValue' method to convert the long variable 'l' into the short variable 'sh'. Finally, we print out the converted variable 'sh' to see the result. The output will look like this:


          
           
100

As seen in the examples above, it is easy to convert a string into a short data type in Java. The 'parseShort', 'valueOf' and 'shortValue' methods can be used in conjunction with the Long and Short classes to achieve the desired result.
