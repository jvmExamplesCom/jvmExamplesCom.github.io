Converting String to short Primitive

Java provides several ways to convert a String to a short primitive. This tutorial examines each of these options, providing examples for each.

Parsing Strings as Short Numbers

The easiest way to convert a String to a short primitive is to parse it using the 

          
           
Short.parseShort(String)
method. Here is an example: 

          
           
String strVal = "400";
short shVal = Short.parseShort(strVal);
System.out.println(shVal); // Outputs 400

Using the valueOf() Method

Alternatively, you can use the 

          
           
Short.valueOf(String)
method to convert a String to a short primitive. Here is an example: 

          
           
String strVal = "500";
short shVal = Short.valueOf(strVal);
System.out.println(shVal); // Outputs 500

Using the Decode() Method

You can also use the 

          
           
Short.decode(String)
method to convert a String to a short primitive. Here is an example: 

          
           
String strVal = "600";
short shVal = Short.decode(strVal);
System.out.println(shVal); // Outputs 600

Using the Short Constructor

Finally, you can use the 

          
           
Short(String)
constructor to convert a String to a short primitive. Here is an example: 

          
           
String strVal = "700";
short shVal = new Short(strVal);
System.out.println(shVal); // Outputs 700
