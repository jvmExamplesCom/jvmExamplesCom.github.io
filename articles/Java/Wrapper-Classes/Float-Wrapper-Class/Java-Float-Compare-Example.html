Java Float Compare Example

Below is an example of comparing floats in Java. 

          
           
float a = 5.6f;
float b = 5.5f;
 
if (a > b) {
   System.out.println("a is greater than b");
}
else if (a < b) {
   System.out.println("a is less than b");
}
else {
   System.out.println("a is equal to b");
}
This code compares the two float values and prints the respective output depending on if they are greater than, less than, or equal to each other.

Comparing the Equality of Two Floats

While it is not generally recommended to compare floats for equality, it can be done with the following code snippet. This code will return true if the two floats are equal within a set tolerance value. 

          
           
private static final float EPSILON = 0.0001f;
public static boolean equals(float a, float b) {
    return Math.abs(a - b) < EPSILON;
}
Here, the constant EPSILON is the tolerance value.

Comparing Edge Case Values

Floating point numbers have a range of special values such as NaN, +Infinity, and -Infinity. These values require special consideration when comparing them. 

          
           
if (Float.isNaN(a)) {
   System.out.println("a is NaN");
}
if (Float.isInfinite(b)) {
   System.out.println("b is infinite");
}
The Float class provides isNaN and isInfinite methods that return true if the value is one of the special values mentioned earlier.
