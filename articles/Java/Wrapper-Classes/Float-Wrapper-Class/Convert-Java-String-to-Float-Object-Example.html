Converting Java String to Float Object Example

Java provides various ways to convert a string into a float object. In this article we'll discuss the different methods for string-to-float conversion and explain the advantages and disadvantages of each.

Method 1: Using the Float.parseFloat Method

The Float.parseFloat() method is the most commonly used approach for converting a string to a float object. This method takes in a single argument, which is a string representation of a number, and returns a float object. Here's an example of this method being used: 

          
           
String str = "12.3";
Float obj = Float.parseFloat(str);
System.out.println("Float value = "+obj);
Output: 

          
           
Float value = 12.3
The parseFloat() method can be used to both convert a numeric string to a float object and a non-numeric string that can be coerced into a number (e.g.  "123abc" → 123.0).

Method 2: Using the Float.valueOf Method

The Float.valueOf() method is another way to convert a string to a float object. This method takes in a single argument, which is a string representation of a number, and returns a float object. Here's an example of this method being used: 

          
           
String str = "12.3";
Float obj = Float.valueOf(str);
System.out.println("Float value = "+obj);
Output: 

          
           
Float value = 12.3
The valueOf() method also works with both numeric strings and non-numeric strings that can be coerced into a number. This method is slightly more performant than the parseFloat() method as it uses a cached lookup approach.

Method 3: Using the Float Constructor

The Float constructor is another way to convert a string to a float object. This method takes in a single argument, which is a string representation of a number, and returns a float object. Here's an example of this method being used: 

          
           
String str = "12.3";
Float obj = new Float(str);
System.out.println("Float value = "+obj);
Output: 

          
           
Float value = 12.3
The Float constructor also works with both numeric strings and non-numeric strings that can be coerced into a number. This method is slightly less performant than the parseFloat() and valueOf() methods as it requires the construction of a new Float object.

Conclusion

In this article we discussed the different methods for converting a string to a float object in Java. The most commonly used and performant approach is the Float.parseFloat() method. However, there are other options available such as the Float.valueOf() and Float constructors.
