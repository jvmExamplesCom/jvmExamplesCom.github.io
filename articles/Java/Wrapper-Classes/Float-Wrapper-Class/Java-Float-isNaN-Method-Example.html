Java Float isNaN Method Example

The Float class in Java provides useful methods to manipulate float data types. One of these methods is the isNaN() method which is used to determine if a float value is a Not-a-Number (NaN). In this article, we will discuss how to use the Float isNaN() method with the help of examples.

Syntax:


          
           
public static boolean isNaN(float f)

Example 1: Simple Float isNaN() Method Usage

In this example, we will use the isNaN() method to check if a float value is Not-a-Number. 

          
           
// Program to demonstrate Float isNaN() method 
public class FloatNaNExample1 
{ 
    public static void main(String[] args) 
    { 
        // Create a float variable 
        float f1 = Float.NaN; 
  
        // Check if it is Not-a-Number 
        boolean b1 = Float.isNaN(f1); 
  
        System.out.println("Is "+f1+" Not-a-Number: "+b1); 
    } 
}
Output: 

          
           
Is NaN Not-a-Number: true

Example 2: Using Float isNaN() Method to Check Non-NaN Float Values

In this example, we will use the isNaN() method to check if a float value is Not-a-Number or not. 

          
           
// Program to demonstrate Float isNaN() method 
public class FloatNaNExample2 
{ 
    public static void main(String[] args) 
    { 
        // Create a float variable 
        float f1 = 10.5f; 
  
        // Check if it is Not-a-Number 
        boolean b1 = Float.isNaN(f1); 
  
        System.out.println("Is "+f1+" Not-a-Number: "+b1); 
    } 
}
Output: 

          
           
Is 10.5 Not-a-Number: false

Example 3: Comparing Floats with the isNaN() Method

In this example, we will compare two floats with the isNaN() method. 

          
           
// Program to demonstrate Float isNaN() method 
public class FloatNaNExample3 
{ 
    public static void main(String[] args) 
    { 
        // Create two float variables 
        float f1 = 12.5f; 
        float f2 = 10.5f; 
  
        // Check if they are equal using isNaN() method 
        boolean b1 = Float.isNaN(f1 - f2); 
  
        System.out.println("Are "+f1+" and "+f2+" equal: "+b1); 
    } 
}
Output: 

          
           
Are 12.5 and 10.5 equal: false
