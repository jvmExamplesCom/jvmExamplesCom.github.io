Convert Java String to Byte Example

Java provides several ways to convert a String object into a byte array. This article will discuss the various methods of converting a Java String to a byte[].

Using java.io.ByteArrayOutputStream

The ByteArrayOutputStream class is one of the Java IO Streams which contains an internal buffer that can be used to write a sequence of bytes and then get the resulting bytes in the form of a byte array. The following code example shows how to use the ByteArrayOutputStream to convert a String to a byte array.


          
           
ByteArrayOutputStream bos = new ByteArrayOutputStream();
bos.write(str.getBytes());
byte[] data = bos.toByteArray();

Using java.nio.charset.CharsetEncoder

The CharsetEncoder class provides methods to convert a sequence of characters into a sequence of bytes. The following code example shows how to use the CharsetEncoder to convert a String to a byte array.


          
           
Charset charset = Charset.forName("UTF-8");
CharsetEncoder encoder = charset.newEncoder();
byte[] data = encoder.encode(CharBuffer.wrap(str)).array();
