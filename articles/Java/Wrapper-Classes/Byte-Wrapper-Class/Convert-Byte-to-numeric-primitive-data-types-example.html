Converting Byte to Numeric Primitive Data Types Example in Java

Java provides a wide array of data types and the logical conversion of one type to another is an important concept. One such example is the conversion of a byte data type to a numeric primitive data type. Knowing how to successfully convert between bytes and numeric primitive data types is essential for any programmer.

What is Byte?

Byte is simply a storage unit and is used to store a single byte of information. It can represent any type of data from integers, to floating-point, to character data. A byte can be stored in memory, but more often it is used when transferring data between two devices.

Convert Byte To Numeric Primitive Data Types Example

The following examples demonstrate how to convert a byte to a numeric primitive data type in Java.

Example 1: Convert a Byte to Integer

This example shows how to convert a Byte value to an Integer. 

          
           
// Create a Byte object
Byte b1 = new Byte((byte) 97);

// Convert it to an integer
int i1 = b1.intValue();

// Display the result
System.out.println("Byte Value: " + b1);
System.out.println("Integer Value: " + i1);
When the above code is compiled and executed, it will produce the following output: 

          
           
Byte Value: 97
Integer Value: 97
