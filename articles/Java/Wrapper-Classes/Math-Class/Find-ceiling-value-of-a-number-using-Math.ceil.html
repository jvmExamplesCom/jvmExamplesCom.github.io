Find Ceiling Value of a Number using Math.ceil

In mathematics, the ceiling value of a number is the smallest integer not less than the given number. Essentially, it rounds a real number up to the next larger integer. Java provides us with the Math.ceil function, which allows us to find the ceiling value of a given number quickly and easily.

Using Math.ceil

To use the Math.ceil function, you must first import the java.lang.Math module at the top of your program:


          
           
import java.lang.Math;
Then, you can call the ceil function as follows: 

          
           
double ceiling = Math.ceil(number);

The ceil function takes in a double and returns the ceiling value of that number as a double.

Example

Let's look at a quick example of how to use Math.ceil. Consider the following code:


          
           
import java.lang.Math;

public class CeilingValue {
    public static void main(String[] args) {
        double number = 3.14;
        double ceiling = Math.ceil(number);
        System.out.println("Ceiling value of " + number + " is " + ceiling);
    }
}

This simple program imports the Math module, and then finds and prints the ceiling value of 3.14 by calling Math.ceil. The output of this program will be:


          
           
Ceiling value of 3.14 is 4.0
As you can see, the ceiling value of 3.14 is 4.0.

Conclusion

Math.ceil is a very useful and powerful function in the Java math library. It allows us to quickly and easily round real numbers up to the nearest larger integer. Hopefully this article has given you an understanding of how to use this function and when it may be useful.
