Round Java float and double numbers using Math.round

In Java, it is possible to round both float and double numbers using the 

Math.round()
function. This function takes a single parameter – the number to be rounded – and rounds it to the nearest integer value. It also returns an int value, so we must explicitly cast it back to a float or double type. Here are some examples of how to use the 
Math.round()
function to round Java float and double numbers.

Round Float Using Math.round()

The following example shows how to use the 
Math.round()
function to round a float to the nearest integer: 

          
           
float num = 3.6f;
float roundedNum = (float) Math.round(num);
System.out.println(roundedNum); // Outputs 4.0

Round Double Using Math.round()

Similarly, the following example shows how to use the 

Math.round()
function to round a double to the nearest integer: 

          
           
double num = 3.6;
double roundedNum = Math.round(num);
System.out.println(roundedNum); // Outputs 4.0

Round Double to N Decimal Places

In addition to rounding to the nearest integer, you can also use the 

Math.round()
function to round a double to a specified number of decimal places. The following example rounds a double to two decimal places: 

          
           
double num = 3.6786;
double roundedNum = Math.round(num * 100.0) / 100.0;
System.out.println(roundedNum); // Outputs 3.68

In this example, we multiply the number by 100.0 before calling the 

Math.round()
function. This ensures that it is rounded to two decimal places instead of one. Then, we divide the result by 100.0 to get the correct value.

Conclusion

In this article, we have looked at how to round both Java float and double numbers using the 

Math.round()
function. We have seen how to round a number to the nearest integer as well as to a specified number of decimal places. Hopefully, this will help you round your numbers with ease!
