Find exponential value of a number using Math.exp

The function 

          
           
Math.exp
is used to calculate the exponential value of a given number. This can be done in Java by passing the numerical value as an argument to the Math.exp method and the result will be returned.

Examples


          
           
double result = Math.exp(4);
// Will return 54.59815003314423
In this example, the number 4 is passed to the Math.exp method, with the result being a double value of 54.59815003314423. 

          
           
int result = Math.exp(6); 
// Will return 403.4287934927351
In this example, the number 6 is passed to the Math.exp method, with the result being an int value of 403.4287934927351. 

          
           
float result = Math.exp(2);
In this example, the number 2 is passed to the Math.exp method, with the result being a float value of 7.38905609893065.

Conclusion

The 

          
           
Math.exp
function can be used in Java to find the exponential value of a given number. The number should be passed as an argument to the method and the result will be returned as a double, int, or float depending on the data type of the argument.
