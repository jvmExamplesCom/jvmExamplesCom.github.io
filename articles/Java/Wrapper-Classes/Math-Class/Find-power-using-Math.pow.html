Finding Power in Java using Math.pow

Java has the ability to calculate the power (exponent) of a given number using the 

          
           
Math.pow(base, exponent)
method. This can be used when performing mathematical calculations that involve an exponent, such as calculating the area of a triangle. In this article, we will look at some examples of how to use the Math.pow() method in Java.

Example 1: Calculating the Area of a Triangle

Consider a triangle with sides of length 3, 4 and 5 cm. To calculate its area, we can use the Heron formula, which states that the area 

          
           
A
of a triangle with side lengths a, b and c is 

          
           
A = \sqrt{s(s - a)(s - b)(s - c)}
where 

          
           
s = \frac{a + b + c}{2}
is the semi-perimeter of the triangle. Using the given values for the sides of the triangle, we can calculate the area as follows: 

          
           
double a = 3;
double b = 4;
double c = 5;

double s = (a + b + c) / 2;
double area = Math.pow(s * (s - a) * (s - b) * (s - c), 0.5);

System.out.println("Area of triangle = " + area);
When this code is executed, the output will be: 

          
           
Area of triangle = 6.0

Example 2: Calculating the Length of a Diagonal

We can also use Math.pow() to calculate the length of a diagonal of a rectangle. The formula for the length of the diagonal 

          
           
d
is given by 

          
           
d = \sqrt{a^2 + b^2}
where 

          
           
a
and 

          
           
b
are the lengths of the sides of the rectangle. Let's say we have a rectangle with sides of length 5 cm and 8 cm. To calculate the length of the diagonal, we can use the following code: 

          
           
double a = 5;
double b = 8;

double diagonal = Math.pow(Math.pow(a, 2) + Math.pow(b, 2), 0.5);

System.out.println("Length of diagonal = " + diagonal);
The output of this code will be: 

          
           
Length of diagonal = 9.433981132056603

Conclusion

In this article, we have seen how to use the Math.pow() method in Java to calculate the power of a number. We have looked at two examples where this method can be used to calculate the area of a triangle and the length of a diagonal of a rectangle.
