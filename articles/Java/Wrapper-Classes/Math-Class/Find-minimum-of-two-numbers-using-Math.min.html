Find Minimum of Two Numbers Using Math.min

Java has a built-in method to find the minimum of two numbers, Math.min(). It returns the smallest number between its two arguments. This can be useful when you need to determine the lowest value between two numerical values.

Syntax

The syntax for Math.min() is as follows: 

          
           
Math.min(firstNumber, secondNumber);

Examples

We can use this method in various combinations of types and data structures. Here are some examples of how Math.min can be used:


          
           
Math.min(5, 10); // Returns 5
Math.min(-5, 4); // Returns -5
Math.min(-4.3, -4.2); // Returns -4.3
Math.min(2, 3, 4); // Returns 2
Math.min(2.2, 3.3, 4.4); // Returns 2.2
Math.min(true, false); // Returns false
Math.min(-Infinity, Infinity); // Returns -Infinity
Math.min('A', 'b'); // Returns 'A'
Math.min(new Date(), new Date(2021,0,1)); // Returns new Date()

Math.min() can also be used to find the minimum value in an array or list of values. For example, we could use it to determine the smallest number from a set of numbers:


          
           
Math.min(1, 2, 3, 4, 5); // Returns 1

Or we could find the earliest date from a list of dates:


          
           
Math.min(
  new Date(),
  new Date(2021, 0, 1),
  new Date(2022, 1, 1)
); // Returns new Date()
