Using Math.floor to Find Floor Value of a Number

Java's 
Math.floor()
function is useful for finding the floor value of a given number. This means the greatest integer less than or equal to the number.

Example 1 - Integer

In the following example, we are using the 
Math.floor()
function to find the floor value of the given number 
4.8
: 

          
           
int floorVal = (int) Math.floor(4.8);
System.out.println(floorVal);
The output will be: 

          
           
4

Example 2 - Double

In this example, we are using the 
Math.floor()
function to find the floor value of the given 
double
number 
4.8
: 

          
           
double floorVal = Math.floor(4.8);
System.out.println(floorVal);
The output will be: 

          
           
4.0

Example 3 - Negative Number

In this example, we are using the 
Math.floor()
function to find the floor value of the given negative number 
-2.5
: 

          
           
int floorVal = (int) Math.floor(-2.5);
System.out.println(floorVal);
The output will be: 

          
           
-3
