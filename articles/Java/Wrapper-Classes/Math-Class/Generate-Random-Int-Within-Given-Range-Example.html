Generate Random Int Within Given Range Example

Java provides developers with the ability to generate a random int within a given range. This is a useful tool for various applications, from card games to simulation programs. In this article, we will discuss how to generate a random int within a given range using Java.

Math.random() Example

The Math.random() method returns a pseudorandom double type number greater than or equal to 0.0 and less than 1.0. We can use this method to generate a random int value within a specified range by making some calculations. For example, if we want to generate a random int value between 1 and 10, we can do the following: 

          
           
//Generate a random int between 1 and 10
int num = (int)(Math.random() * 10) + 1;
System.out.println(num);
The code above will generate a random int value between 1 and 10.

Random Class Example

The Random class provides us with an alternative way to generate a random int value within a given range. For example, if we want to generate a random int value between 1 and 10, we can do the following: 

          
           
// Generate a random int between 1 and 10
Random rand = new Random();
int num = rand.nextInt(10) + 1;
System.out.println(num);
