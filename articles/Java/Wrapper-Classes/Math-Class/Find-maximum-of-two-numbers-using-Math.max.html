Find Maximum of Two Numbers Using Math.max

Java provides a versatile way of finding the maximum of two numbers using 

Math.max
static member. The purpose of this article is to provide some examples and discuss how the maximum can be found.

Syntax and Parameters

The syntax for 
Math.max
is as follows: 
Math.max(x, y)
x
and 
y
are referred to as parameters and represent the two numbers that you would like to compare. These can be of any data type, including int, double, float, and long.

Examples

The following are some examples of how to find the maximum of two numbers using 

Math.max
: 

          
           
// Find the maximum of two integers
int maxInt = Math.max(1, 2);
System.out.println("The maximum of 1 and 2 is: " + maxInt);
// Outputs: The maximum of 1 and 2 is: 2

// Find the maximum of two doubles
double maxDouble = Math.max(3.14, 5.67);
System.out.println("The maximum of 3.14 and 5.67 is: " + maxDouble);
// Outputs: The maximum of 3.14 and 5.67 is: 5.67

// Find the maximum of two floats
float maxFloat = Math.max(4.5f, 6.4f);
System.out.println("The maximum of 4.5f and 6.4f is: " + maxFloat);
// Outputs: The maximum of 4.5f and 6.4f is: 6.4

// Find the maximum of two longs
long maxLong = Math.max(5L, 8L);
System.out.println("The maximum of 5L and 8L is: " + maxLong);
// Outputs: The maximum of 5L and 8L is: 8

Return Value

The return value of 
Math.max
is the maximum of the two parameters. If the two parameters have different data types, then it will return the parameter with the higher numerical value, or the one with the larger data type. For example, if a 
float
is compared with an 
int
, the 
float
will be returned since it has a higher numerical value.

Conclusion

In conclusion, Java provides an easy and efficient way of finding the maximum of two numbers using 

Math.max
. As shown in the examples above, 
Math.max
can be used to compare a variety of data types. The return value of 
Math.max
is the maximum of the two parameters, which will always be the parameter with the highest numerical value or the one with the larger data type.
