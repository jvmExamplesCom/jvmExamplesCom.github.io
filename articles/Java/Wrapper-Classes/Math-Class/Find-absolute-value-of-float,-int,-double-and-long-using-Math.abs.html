Find Absolute Value of float, int, double and long using Math.abs

One of the most commonly used methods in Java is 
Math.abs()
. This method returns the absolute value of a number, no matter if it's a float, int, double or long. Let's look at some examples using this function.

Find Absolute Value of a Float

To get the absolute value of a float, we can use the Math.abs() method.


          
           
float x = -50.25f;
float y = Math.abs(x);
System.out.println(y);
The output of this code is 
50.25
.

Find the Absolute Value of an Integer

We can also use Math.abs() to find the absolute value of an integer:


          
           
int x = -100;
int y = Math.abs(x);
System.out.println(y);
The output of this code is 
100
.

Find the Absolute Value of a Double

Using the same method, we can find the absolute value of a double as well:


          
           
double x = -49.99;
double y = Math.abs(x);
System.out.println(y);
The output of this code is 
49.99
.

Find the Absolute Value of a Long

Finally, we can use Math.abs() to get the absolute value of a long value:


          
           
long x = -1000L;
long y = Math.abs(x);
System.out.println(y);
The output of this code is 
1000
.

As you can see, Math.abs() is a very useful and versatile method for getting the absolute value of any numerical data type in Java. You can use this method with confidence, knowing that it will return the absolute value of any number you pass into it.
