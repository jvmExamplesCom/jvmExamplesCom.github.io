Converting a String to a Java Wrapper Object

Java provides many wrapper classes to encapsulate primitive data types. This is necessary to pass primitive data types as objects and to be able to store them in collections like ArrayList and HashMap. In this tutorial, we will learn how to convert a String to a wrapper object in Java.

String to Integer

To convert a 

          
           
String
to an 

          
           
Integer
, you can use the 

          
           
Integer.valueOf()
method. The syntax for using this method is as follows: 

          
           
Integer intObj = Integer.valueOf(str);
where 

          
           
intObj
is the 

          
           
Integer
object and 

          
           
str
is the 

          
           
String
. Let's see an example of how to use this method. 

          
           
String str = "42";
Integer intObj = Integer.valueOf(str);
System.out.println(intObj);
The output of the code above would be: 

          
           
42
