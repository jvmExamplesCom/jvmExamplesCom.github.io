Convert Byte Object to byte, short, int, long, float, and double

Java supports 8 primitive data types - byte, short, int, long, float, double, char, and boolean. A byte object is an object version of a byte datatype. This article will discuss the ways to convert a byte object to other primitive data types.

Converting Byte Object to byte

The simplest way to convert a byte object to a byte is by invoking the byteValue() method on the object. Consider the following example: 

          
           
Byte b= new Byte((byte) 5);
byte b2 = b.byteValue();
System.out.println(b2); 
// Outputs 5

Converting Byte Object to short

To convert a byte object into a short, we can use the shortValue() method present on the Byte class. Consider the following example: 

          
           
Byte b= new Byte((byte) 5);
short s = b.shortValue();
System.out.println(s); 
// Outputs 5

Converting Byte Object to int

We can convert a byte object to an int using the intValue() method. Consider the following example: 

          
           
Byte b= new Byte((byte) 5);
int i = b.intValue();
System.out.println(i); 
// Outputs 5

Converting Byte Object to long

The longValue() method can be used to convert a byte object to a long. Consider the following example: 

          
           
Byte b= new Byte((byte) 5);
long l = b.longValue();
System.out.println(l); 
// Outputs 5

Converting Byte Object to float

The floatValue() method can be used to convert a byte object to a float. Consider the following example: 

          
           
Byte b= new Byte((byte) 5);
float f = b.floatValue();
System.out.println(f);
// Outputs 5.0

Converting Byte Object to double

The doubleValue() method can be used to convert a byte object to a double. Consider the following example: 

          
           
Byte b= new Byte((byte) 5);
double d = b.doubleValue();
System.out.println(d);
// Outputs 5.0
