Converting Java Boolean Primitive to Boolean Object

Java has a primitive data type for boolean values which is either true or false. There may be situations where we need to convert a boolean primitive type value to the object representation of a boolean.

We can use the Boolean class’s static method, valueOf(), to convert a boolean primitive type to the object representation of the boolean.

Example 1 - Convert boolean primitive to Boolean object

In this example, we will convert a boolean primitive type to the object representation of a boolean.


          
           
boolean primitiveBoolean = true;
Boolean objectBoolean = Boolean.valueOf(primitiveBoolean);
System.out.println(objectBoolean);

When the above code is compiled and executed, it produces the following result:


          
           
true
