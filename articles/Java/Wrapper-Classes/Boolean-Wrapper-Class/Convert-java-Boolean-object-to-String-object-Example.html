Convert Java Boolean Object to String Object Example

This article explains how to convert a Java Boolean object to a String object. This task can be accomplished by using the toString() method of the Boolean class. In this tutorial, we will demonstrate how to convert a Boolean object to a String object and print the value to the console.

Example 1: Convert a Boolean object to a String object using the toString() method

In this example, we will create a Boolean object and use the toString() method to convert it to a String object. We will then print out the value of the Boolean object to the console window. 

          
           
public static void main(String[] args){
   
       // Create a Boolean object
       Boolean b = new Boolean(true);
       
      // Convert the Boolean object to a String object
       String str = b.toString();
       
       // Print out the value
       System.out.println("The value of the Boolean object is: " + str);
   }
The output of the above code will be: 

          
           
The value of the Boolean object is: true

Example 2: Convert multiple Boolean objects to String objects

In this example, we will demonstrate how to convert multiple Boolean objects to String objects. We will create an array of Boolean objects and use the toString() method for each one. We will then print out the value of each Boolean object to the console window. 

          
           
public static void main(String[] args){
   
   // Create an array of Boolean objects
   Boolean[] bArray = { new Boolean(true), 
                        new Boolean(false), 
                        new Boolean(true) };
   
   // Iterate over the array and convert each Boolean object to a String object
   for (int i = 0; i < bArray.length; i++){
       String str = bArray[i].toString();
       
       // Print out the value
       System.out.println("The value of the Boolean object at index " + i + " is: " + str);
   }
}
The output of the above code will be: 

          
           
The value of the Boolean object at index 0 is: true
The value of the Boolean object at index 1 is: false
The value of the Boolean object at index 2 is: true
