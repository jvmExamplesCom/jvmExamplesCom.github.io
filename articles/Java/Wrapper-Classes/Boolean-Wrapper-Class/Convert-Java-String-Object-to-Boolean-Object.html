Converting Java String Object to Boolean Object

A Java string object can be converted to a boolean type object given the right techniques. In this article, we will discuss how to convert a java string object to a boolean object.

Using if-else Statement

The most straightforward way of converting a string object to a boolean object is by using an if-else statement. This method checks the value of the provided string object to determine its boolean type.


          
           
String s = "true";
Boolean b;
 
if(s.equals("true")) {
   b = true;
} 
else {
   b = false;
}

This method can be useful for certain situations where you need to evaluate the value of the string prior to determining the boolean type.

Using parseBoolean Method

The parseBoolean method can be used to convert a string object directly to a boolean object without having to use an if-else statement. This method evaluates the string value and checks to determine if it is equal to either "true" or "false". If the string is equal to either value, then the corresponding boolean type is returned.


          
           
String s = "true";
Boolean b = Boolean.parseBoolean(s);

Using ValueOf Method

The ValueOf method works similar to the parseBoolean method in that it takes a string object as a parameter and returns a boolean type. However, the ValueOf method also takes other values such as integer and double types that are then evaluated to determine the boolean type.


          
           
String s = "true";
Boolean b = Boolean.valueOf(s);

Using Boolean Constructor

The Boolean constructor is another method for converting a string object to a boolean object. This constructors takes a string object as a parameter and immediately evaluates the value of the string in order to determine the boolean type.


          
           
String s = "true";
Boolean b = new Boolean(s);

These are just some of the ways to convert a java string object to a boolean object. Each of these methods can be useful depending on the goal of the program.
