Java Break Statement Example

The break statement is used to end a loop early when the value of a specific condition is met. In Java, this statement is used to exit a loop while it is still in the middle of iterating. This can be useful in cases where you want to skip the rest of the iterations and move on to other statements.

Example 1: Using The Break Statement To Exit A While Loop Early

In this example, we are using a while loop to loop through an array. We have set a condition that if the value of the array at index 2 is equal to 5, then we will use the break statement to exit the loop early and move on to the next statement. 

          
           
int[] arr = {1, 2, 5, 7, 8};
int count = 0;

while(count < arr.length) {
    if (arr[count] == 5) {
        break;
    }
    count++;
}
System.out.println("The loop has ended!");
The output of this code would be "The loop has ended!".

Example 2: Using The Break Statement To Exit A Do-While Loop Early

In this example, we are using a do-while loop to loop through an array. We have set a condition that if the value of the array at index 2 is equal to 5, then we will use the break statement to exit the loop early and move on to the next statement. 

          
           
int[] arr = {1, 2, 5, 7, 8};
int count = 0;

do {
    if (arr[count] == 5) {
        break;
    }
    count++;
} while(count < arr.length);
System.out.println("The loop has ended!");
The output of this code would be "The loop has ended!".

Example 3: Using The Break Statement To Exit A For Loop Early

In this example, we are using a for loop to loop through an array. We have set a condition that if the value of the array at index 2 is equal to 5, then we will use the break statement to exit the loop early and move on to the next statement. 

          
           
int[] arr = {1, 2, 5, 7, 8};

for (int i = 0; i < arr.length; i++) {
    if (arr[i] == 5) {
        break;
    }
}
System.out.println("The loop has ended!");
The output of this code would be "The loop has ended!".
