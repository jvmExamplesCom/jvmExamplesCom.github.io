Continue Statement in Java

The continue statement is used to break the current iteration of a loop and skip ahead to the next iteration. The continue statement is generally used in the looping statements like while, do-while and for loop, but can also be used inside an if-else control structure as well. In this article, we will take a look at an example of using the continue statement with a while loop and discuss how this statement works.

Java Continue Statement Example

Let's see an example of using the continue statement with a while loop. In the following code, we are looping through the numbers 1 to 10 and printing each number to the screen. The continue statement skips printing the number 5 to the screen: 

          
           
int i = 0;
while (i++ < 10) {
    if (i == 5) {
        continue;
    }
    System.out.println(i);
}
When the above code is executed, the output will be: 

          
           
1
2
3
4
6
7
8
9
10
Here, we can see that the number 5 was not printed to the screen due to the continue statement.

Understanding the Continue Statement

Let's take a closer look at how the continue statement works in the above program. First, we declared and initialized the variable i to 0. We then created a while loop that runs until the value of i is less than 10. Inside the loop body, we have an if-else statement that checks if i is equal to 5. If it is, the continue statement is executed and the rest of the loop body is skipped, moving control to the next iteration. If i is not equal to 5, the else branch is executed and we print the value of i to the screen. After the print statement, control moves to the next iteration of the loop. This process continues until the value of i reaches 10 and the loop breaks.
