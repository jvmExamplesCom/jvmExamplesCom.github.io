Java continue Statement With Label Example

The Java 

          
           
continue statement
with label enables the programmer to skip some part of the loop and comes to the direct iteration of the loop. The label provides the reference of the loop, which is being skipped by the statement. The below example provides a description of the continue statement with a label:

Program code


          
           
public class Test {
    public static void main(String[] args) {
        for(int i = 1; i <= 10; i++){
            if (i == 5){
                continue;
            }
            else
            {
                System.out.println(i);
            }
        }
    }
}
In the above program, the continue statement skips the iteration when the value of the variable i is 5. The output of the program will be:

Output


          
           
1
2
3
4
6
7
8
9
10
Now, if you need to skip or terminate all the remaining iterations of the outer loop, when a condition is fulfilled in an inner loop, then the same can be achieved using labels, as shown in the program given below:

Program code


          
           
public class Test {
    public static void main(String[] args) {
        OUTER_LOOP:
        for(int i = 1; i <= 3; i++){
            for(int j = 1; j <= 3; j++){
            	if (i * j > 3)
            		continue OUTER_LOOP;
            	System.out.println(i*j);
            }
        }
    }
}
In the above program, when the condition 

          
           
i * j > 3
is fulfilled, the continue statement skips all the remaining iterations of the outer loop. The output of the program will be:

Output


          
           
1
2
2
3
3
