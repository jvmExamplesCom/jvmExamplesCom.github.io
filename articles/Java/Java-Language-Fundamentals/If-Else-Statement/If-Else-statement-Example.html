Java: If Else Statement Example

If Else statements are a common feature of programming and scripting languages, including Java. In its simplest form, an If Else statement looks like this: 


          
           
if (condition) 
    statement;
else
    statement;
This means that if the condition (in parentheses) is true, the first statement will be executed, otherwise the second statement.

Examples

Here are a few examples of If Else statements using different types of conditions and statements.

Checking a boolean variable

The following example checks a boolean variable named 

flag
to see if it is 
true
or 
false
: 

          
           
if (flag) 
    System.out.println("Flag is true!");
else
    System.out.println("Flag is false!");

Comparing two integers

In this example, two integers ( 
x
and 
y
) are compared to see which is greater: 

          
           
if (x > y) 
    System.out.println("x is larger than y!");
else
    System.out.println("x is smaller than or equal to y!");

Comparing a string to a character

This example compares a character in a string to an individual character:


          
           
if (string.charAt(0) == 'A') 
    System.out.println("First character of string is 'A'");
else
    System.out.println("First character of string is not 'A'");

Checking for a null value

The following code checks for a null value: 

          
           
if (value == null) 
    System.out.println("Value is null!");
else
    System.out.println("Value is not null!");

Accessing an array element

This example checks if a given array element is empty:


          
           
if (array[index] == null) 
    System.out.println("Array element is empty!");
else
    System.out.println("Array element is not empty!");

Using multiple conditions

You can use multiple conditions in an If Else statement by checking them with logical operators such as 

&&
(AND) and 
||
(OR): 

          
           
if (flag1 &amp;&amp; flag2) 
    System.out.println("Both flags are true!");
else
    System.out.println("At least one flag is not true!");

Conclusion

This article has provided some examples of how to use an If Else statement in Java. Remember that the condition must always evaluate to a boolean value (i.e. 

true
or 
false
) and that you can combine multiple conditions with logical operators.
