Java If Else-If Statement Example

The if statement allows you to control the flow of your program based on a condition. In this example, we will look at how to use an if else-if statement in Java.

Syntax

The syntax for an if else-if statement looks like this: 

          
           
if (condition1) {
    // Code to be executed if condition1 is true
} 
else if (condition2) {
    // Code to be executed if condition2 is true
} 
else {
    // Code to be executed if neither condition1 nor condition2 are true
}
Let's take a closer look at how it works.

Example

In this example, we have a simple if else-if statement to determine if a given number is positive, negative, or zero. 

          
           
int num = 5;

if (num > 0) {
    System.out.println("Positive number");
} 
else if (num < 0) {
    System.out.println("Negative number");
} 
else {
    System.out.println("Zero");
}
In this example, we have a variable called num with a value of 5. If num is greater than 0, it is a positive number, if it is less than 0, it is a negative number, and if it is equal to 0, it is zero. The output of this code will be: 

          
           
Positive number

Multiple Conditions

You can also use multiple conditions in an if else-if statement. Let's take a look at another example. 

          
           
int num = 5;

if (num > 0 &amp;&amp; num < 10) {
    System.out.println("Number is between 0 and 10");
} 
else if (num > 10 &amp;&amp; num < 20) {
    System.out.println("Number is between 10 and 20");
} 
else {
    System.out.println("Number is not between 0 and 20");
}
In this example, we have a variable called num with a value of 5. If num is greater than 0 and less than 10, then it is between 0 and 10. If num is greater than 10 and less than 20, then it is between 10 and 20. Otherwise, it is not between 0 and 20. The output of this code will be: 

          
           
Number is between 0 and 10
