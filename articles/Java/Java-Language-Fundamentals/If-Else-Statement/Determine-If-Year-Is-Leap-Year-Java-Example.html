Determine If Year Is Leap Year Java Example

Leap years are years with an extra day, February 29th. To improve the accuracy of calendar based activities, an additional day is included every four years. Different algorithms can be used to determine if a given year is a leap year or not. The following code example demonstrates one way to determine if a given year is a leap year in Java.

Simple Java Code Example to Determine If Year Is Leap Year

The following simple Java code example determines if a given year is a leap year: 

          
           
public static boolean isLeapYear(int year) {
    if (year % 400 == 0) {
        return true;
    } else if (year % 100 == 0) {
        return false;
    } else if (year % 4 == 0) {
        return true;
    } else {
        return false;
    }
}
In this example, the method 

          
           
isLeapYear()
takes an 

          
           
int
as an argument and returns a 

          
           
boolean
value. The method evaluates the given year according to the following conditions:
  • If the year is divisible by 400, it is a leap year.
  • If the year is divisible by 100 but not 400, it is not a leap year.
  • If the year is divisible by 4 but not 100, it is a leap year.
  • Otherwise, it is not a leap year.

Java Program to Demonstrate Leap Year Example

The following program demonstrates how the code example can be used in a Java program to determine if a given year is a leap year:

    public static void main(String[] args) {

        // Check if 2020 is a leap year
        int year = 2020;

        boolean isLeapYear = isLeapYear(year);

        if (isLeapYear) {
            System.out.println("Year " + year + " is a leap year.");
        } else {
            System.out.println("Year " + year + " is not a leap year.");
        }
    }

    public static boolean isLeapYear(int year) {
        if (year % 400 == 0) {
            return true;
        } else if (year % 100 == 0) {
            return false;
        } else if (year % 4 == 0) {
            return true;
        } else {
            return false;
        }
    }
}
