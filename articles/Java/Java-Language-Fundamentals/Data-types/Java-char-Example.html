What is Java char?

Java char is a primitive data type used to represent single characters. It is defined as an unsigned 16-bit integer. It supports a full range of Unicode characters, allowing developers to store characters from various languages in the same data type.

Declaring a Java char

To declare a Java char, use the syntax 
char myChar = 'a';
. The character value can be expressed either as a character constant or as its corresponding integer value.

Examples of Java char

To demonstrate Java char with examples, consider the following code snippet:


          
           
char myChar1 = 'a'; 
char myChar2 = 97; 
char myChar3 = (char)200;

System.out.println("myChar1 is " + myChar1);
System.out.println("myChar2 is " + myChar2);
System.out.println("myChar3 is " + myChar3);

When this code is executed, the following output will be produced:


          
           
myChar1 is a
myChar2 is a
myChar3 is È

As you can see from this example, the character constant 'a' and its corresponding integer value 97 are both interpreted as the character 'a'. The character constant 200 is converted to its corresponding Unicode character 'È'.

Java char Comparison

Two char values can be compared with the equality operator ==. To demonstrate, consider the following code snippet:


          
           
char c1 = 'a';
char c2 = 'b';

if(c1 == c2) {
    System.out.println("c1 and c2 are equal!");
} else {
    System.out.println("c1 and c2 are not equal!");
}
