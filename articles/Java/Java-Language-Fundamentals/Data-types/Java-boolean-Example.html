Java Boolean Example

A boolean is a data type used in programming to represent two possible values - either true or false. In java, the predefined type boolean is used to have true or false values. This example demonstrates how it can be used in java.

Declaring a Java Boolean

In java, a boolean variable can be declared using the following syntax: 

          
           
boolean x;
This statement declares a variable x of type boolean, which will store either true or false values.

Assigning Values to Java Boolean Variables

After a boolean variable is declared, it can be assigned a value using the assignment operator, which is a single equal sign (‘=’). Any value which is not equivalent to zero (such as strings, numbers, and objects) will be treated as true. The following are some examples of assigning values to boolean variables: 

          
           
boolean x = true;
boolean y = false;
boolean z = 1;
boolean w = "test";
In the above example, x is assigned the value true, y is assigned the value false, z is assigned the value 1 and w is assigned the value "test".

Comparing Values Using Boolean Expressions

Boolean expressions are used to compare two values to evaluate whether they are equal or not. The following are some examples of boolean expressions: 

          
           
x == 1 // returns false
y != 0 // returns true
z < 5 // returns true
w > "apple" // returns true
In the above example, x is compared with 1 and returns false because x is assigned the value true, y is compared with 0 and returns true, z is compared with 5 and returns true and w is compared with "apple" and returns true.

Using boolean Variables in Java

Boolean variables can be used in various ways in java. For example, they can be used in if statements, for loop and while loops, and for comparisons between two variables.

If Statements

In java, if statements are used to execute a code block based on the boolean expression's result. If the expression returns true, the code inside the if statement will be executed. Otherwise, the code will be skipped. The following is an example of using a boolean variable in an if statement: 

          
           
if(x){
    System.out.println("x is true");
}
In the above example, the code inside the if statement will be executed if x is true.

For Loop

In java, for loops are used to iterate a code block a certain number of times. The following is an example of using a boolean variable in a for loop: 

          
           
for(boolean i = false; i; i = !i){
    System.out.println("Looping...");
}
In the above example, the code inside the for loop will be executed once, because i is false at the beginning of the loop.

While Loops

In java, while loops are used to iterate a code block until the condition becomes false. The following is an example of using a boolean variable in a while loop: 

          
           
boolean x = true;
while(x){
    System.out.println("Looping...");
    x = false;
}
In the above example, the code inside the while loop will be executed once, because x is true at the beginning of the loop and is set to false inside the loop.

Comparing Two Variables

Boolean variables can also be used to compare two variables. The following is an example of comparing two variables using a boolean expression: 

          
           
int a = 5;
int b = 10;
boolean x = a < b;

System.out.println(x); // prints "true"
In the above example, the boolean expression a < b is used to compare the values of a and b. Since a is less than b, the expression returns true and the value is stored in the boolean variable x.
