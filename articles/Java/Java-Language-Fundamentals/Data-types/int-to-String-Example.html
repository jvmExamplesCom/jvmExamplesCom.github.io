int to String Example in Java

Java provides various ways to convert int to string. This article is a walkthrough of different methods which can be used to convert an integer value to its corresponding string representation.

Method 1: Using Integer.toString()

The Integer class has a method toString() which can be used to convert any integer to its corresponding string representation. It takes a single parameter i.e. the Integer value that needs to be converted and returns the corresponding string representation of it. 

          
           
int num = 123;
String str = Integer.toString(num);
System.out.println(str);
Output: 

          
           
123

Method 2: Using String.valueOf()

The String class also provides a built-in method valueOf() which can be used to convert an integer to its corresponding string representation. It takes a single parameter i.e. the integer value to be converted and returns the corresponding string representation. 

          
           
int num = 123;
String str = String.valueOf(num);
System.out.println(str);
Output: 

          
           
123

Method 3: Using StringBuilder.append()

The StringBuilder class also provides a built-in method append() which can be used to convert an integer to its corresponding string representation. It takes two parameters i.e. the object on which the append method is being called and the integer value to be converted. It returns the object on which the append method is being called with the value appended to it. 

          
           
int num = 123;
StringBuilder sb = new StringBuilder();
String str = sb.append(num).toString();
System.out.println(str);
Output: 

          
           
123

Method 4: Using DecimalFormat.format()

The DecimalFormat class also provides a built-in method format() which can be used to convert an integer to its corresponding string representation. It takes one parameter i.e. the integer value to be converted and returns the corresponding string representation. 

          
           
int num = 123;
DecimalFormat df = new DecimalFormat("#");
String str = df.format(num);
System.out.println(str);
Output: 

          
           
123
As seen from the above examples, there are various ways of converting an integer to its corresponding string representation in Java. Each method provides a different way of achieving the same conversion and depending on the use case, one method can be more suitable than other.
