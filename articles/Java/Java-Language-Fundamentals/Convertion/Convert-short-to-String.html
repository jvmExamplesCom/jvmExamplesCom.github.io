Converting Short to String In Java

One of the most common tasks in Java programming is converting a short data type to a String. This can be done either manually by appending a short value to an existing String object, or programmatically using built-in functionality. Below, we'll discuss both approaches in detail.

Manually Appending a Short to a String

The simplest approach to converting a short to a String is to simply append the short value to an existing String. To do this, we first create a short variable and assign it a value: 

          
           
short myShort = 10;
Next, we create a String variable and add the short value to the end: 

          
           
String myStr = "My short is " + myShort;
The above code will create a String containing the following text: 

          
           
My short is 10

Programmatically Converting a Short to a String

Java also provides built-in functionality for programmatically converting a short to a String. To do this, we use the static 

          
           
toString()
method of the 

          
           
Short
class. Like this: 

          
           
short myShort = 10; 
String myStr = Short.toString(myShort);
The above code will create a String containing the same text as the previous example ( 

          
           
"My short is 10"
).

Conclusion

In this article, we discussed two approaches for converting a short to a String in Java. The simplest approach is to simply append the short value to an existing String. toString()

Short
class.
