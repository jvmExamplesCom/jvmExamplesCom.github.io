How to Convert String to Double in Java

Java provides a range of primitive data types for storing numeric values. The double type is one of these, and it is used for storing large or small numbers with decimal points. Knowing how to convert a String to a double type is important for dealing with numerical inputs from users. This article will discuss the process of converting String objects to double in Java.

Overview of String and Double Data Types

A String is an object class in Java which represents text-based data. Strings are surrounded by quotation marks, and contain characters which can be used to represent words and phrases. The double type is used for storing floating-point numbers, which are numbers that have decimal points. This type is useful for representing decimal fractions and for working with large or small numeric values.

Using the Double.valueOf() Method

The easiest way to convert a String object to a double is to use the 

          
           
Double.valueOf()
method. This method will take the String object as its parameter and return a double value. The following example demonstrates this: 

          
           
String strValue = "10.5";
double doubleValue = Double.valueOf(strValue);
System.out.println("The double value is:" + doubleValue);
// Outputs: The double value is: 10.5

Using the Double.parseDouble() Method

Another way to convert a String to a double is to use the 

          
           
Double.parseDouble()
method. This method takes the String object as its parameter and returns a double value. The following example shows how to use this method: 

          
           
String strValue2 = "20.7";
double doubleValue2 = Double.parseDouble(strValue2);
System.out.println("The double value is: " + doubleValue2);
// Outputs: The double value is: 20.7

Using the NumberFormat Class

The NumberFormat class also provides a way to convert a string to a double. This class can be used to format numbers, but it also provides methods for parsing strings. To use this class, the following example shows how to create a 

          
           
NumberFormat
object and use its 

          
           
parse()
method to convert a string to a double: 

          
           
String strValue3 = "30.9";
NumberFormat numberFormat = NumberFormat.getInstance();
double doubleValue3 = numberFormat.parse(strValue3).doubleValue();
System.out.println("The double value is: " + doubleValue3);
// Outputs: The double value is: 30.9

Conclusion

In this article, we discussed how to convert a String to a double in Java. We looked at three different methods for doing this: the Double.valueOf() method, the Double.parseDouble() method, and the NumberFormat class. All of these methods are easy to use and provide a convenient way for working with numeric input.
