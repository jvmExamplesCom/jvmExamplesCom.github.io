Convert String to float in Java

Java provides several ways to convert a String to float. The following are the commonly used methods to convert any given String to float.

Using Float.parseFloat() Method

The 

          
           
Float.parseFloat()
is an inbuilt method in Java, declared in the java.lang package. It returns a new float initialized to the value represented by the specified 

          
           
String
, as done by the 

          
           
valueOf
method of one of the following classes: 
  • Float
  • Double
  • BigDecimal
Syntax: 

          
           
float f = Float.parseFloat(String s);
Example: 

          
           
public class Main{ 
  public static void main(String[] args) { 
    String s = "3.14"; 
  
    // Converts the string to float 
    float f = Float.parseFloat(s); 
  
    System.out.println("String to float conversion: " + f); 
  } 
}
Output: 

          
           
String to float conversion: 3.14

Using Float.valueOf() Method

The 

          
           
Float.valueOf()
is an inbuilt method in Java, declared in the java.lang package. This method behaves identically to the 

          
           
valueOf
method of one of the following classes: 
  • Float
  • Double
  • BigDecimal
Syntax: 

          
           
float f = Float.valueOf(String s);
Example: 

          
           
public class Main{ 
  public static void main(String[] args) { 
    String s = "3.14"; 

    // Converts the string to float 
    float f = Float.valueOf(s); 
  
    System.out.println("String to float conversion: " + f); 
  } 
}
Output: 

          
           
String to float conversion: 3.14

Using DecimalFormat Class

The 

          
           
DecimalFormat
class is a class in Java that allows us to format numbers according to a specified pattern. We can use this class to convert a string representation of a number to a float. Syntax: 

          
           
import java.text.DecimalFormat; 
public class Main { 
  public static void main(String args[]) { 
    DecimalFormat df = new DecimalFormat("#"); 
    String s = "3.14"; 

    // Converts the string to float 
    float f = (float)df.parse(s); 
  
    System.out.println("String to float conversion: " + f); 
  } 
}
Output: 

          
           
String to float conversion: 3.14
