Convert String to Any Primitive in Java

Java provides a way to convert String objects into primitive types like int, float, double, etc. This is necessary because String objects are used throughout the language and often need to be converted to other types when they are used in functions or operations that involve arithmetic or comparison with other types. In this article we will discuss how to use Java's built-in methods to perform this type of conversion.

Using Java's parseInt() Method

The simplest way to convert a String object to an integer in Java is using the parseInt() method. This method takes a String object as its parameter and returns an int value. For example, the following code will take the String "1" and convert it to an int: 

          
           
String text = "1";
int number = Integer.parseInt(text);
System.out.println(number); // prints 1
The parseInt() method is also capable of converting Strings with decimal points and negative numbers, though the decimal point must be specified as a period (not a comma): 

          
           
String text = "3.14";
double number = Double.parseDouble(text);
System.out.println(number); // prints 3.14

          

          
           
String text = "-1";
int number = Integer.parseInt(text);
System.out.println(number); // prints -1

Using Java's valueOf() Method

Another way to convert a String object to a primitive type is using the valueOf() method. This method works similarly to parseInt(), but instead of returning an int it returns the primitive type specified by the parameter. For example, the following code will take the String "true" and convert it to a boolean: 

          
           
String text = "true";
boolean bool = Boolean.valueOf(text);
System.out.println(bool); // prints true

Using Java's toString() Method

The opposite operation is also possible in Java, which is to convert a primitive type to a String object. This is done using the toString() method and is useful when a primitive type needs to be printed out or inserted into a collection or data structure. For example, the following code will take the double 3.14 and convert it to a String: 

          
           
double number = 3.14;
String text = String.valueOf(number);
System.out.println(text); // prints 3.14

Conclusion

In this article we discussed how to convert a String object to a primitive type in Java. We demonstrated how to do this using both the parseInt() and valueOf() methods, as well as how to convert a primitive type to a String using the toString() method. These methods are useful when dealing with Strings and primitives in Java.
