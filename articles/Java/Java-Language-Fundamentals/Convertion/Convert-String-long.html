Converting Strings in Java

Java is an incredibly powerful language and among its many features is the ability to convert strings. This article will provide an overview of how this can be accomplished. We will explore several methods of conversion and provide code examples of each.

String to Primitives

The simplest form of conversion is from a String to a primitive. In order to accomplish this, the string must contain a valid representation of the primitive type. For example, if the string is "3.14" it can be parsed into a double primitive. To do so, the Double.parseDouble() method is used like: 

          
           
String myStr = "3.14"; 
double myDouble = Double.parseDouble(myStr);
This code would set myDouble to 3.14. Similar methods exist for parsing other primitives, such as Integer.parseInt(), Float.parseFloat(), and Short.parseShort(). Note that if the string contains an invalid value for the primitive, these methods will throw a NumberFormatException.

String to Objects

String values can also be converted to objects. This is done by using the String constructor. For example, a String object can be created from a String literal like: 

          
           
String myStrObj = new String("Hello World!");
This creates a String object with the content of "Hello World!"

Objects to String

Once we have objects, we can use the .toString() method to convert them back to Strings. If the object does not override this method, the result will usually be a description of the object, such as its memory location or type. For example, 

          
           
String myStrObj = new String("Hello World!"); 
String myStr = myStrObj.toString();
The code above will set myStr to "Hello World!".

Formatting Objects

It is often useful to format objects into specific Strings. This can be accomplished using the String.format() method. This method takes a formatting string followed by arguments to be inserted into the formatting string. For example, the following code formats a number into a string with a maximum of two decimal places: 

          
           
float myFloat = 3.14159265f; 
String myStr = String.format("%.2f", myFloat);
This code would set myStr to "3.14". The % specifier is used to denote a value to be replaced, the .2 defines two decimal places, and f denotes the type, which is a float.

Wrapping Up

We have explored several methods for converting between Strings and primitives/objects. The key takeaway is that there are multiple ways to achieve the same result and it is up to you to decide which one is best for your particular use-case.
