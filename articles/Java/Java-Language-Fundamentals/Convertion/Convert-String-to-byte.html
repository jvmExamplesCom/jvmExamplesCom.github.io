Converting String to Byte

Java provides a number of ways to convert a string to a byte. These methods can be used for a variety of purposes, depending on the type of data being stored in the string and the desired output.

Using getBytes() Method

The simplest method for converting a string to a byte is to use the 


          
           
<a href="https://docs.oracle.com/javase/8/docs/api/java/lang/String.html#getBytes--" target="_blank" rel="noopener noreferrer">String.getBytes()</a>
method. This method takes no parameters and returns a byte array containing the contents of the string. 

          
           
String str = "Hello World";
byte[] bytes = str.getBytes();
System.out.println(Arrays.toString(bytes));
// Output: [72, 101, 108, 108, 111, 32, 87, 111, 114, 108, 100]

Using Arrays.copyOf() Method

Another method for converting a string to byte is to use the 


          
           
<a href="https://docs.oracle.com/javase/8/docs/api/java/util/Arrays.html#copyOf-byte:A-int-" target="_blank" rel="noopener noreferrer">Arrays.copyOf()</a>
method. This method takes two parameters: an array of bytes and the desired length of the output byte array. The resulting byte array will contain the same contents of the input array, except it will be truncated or padded with zeros to the desired length. 

          
           
String str = "Hello World";
byte[] bytes = str.getBytes();
byte[] res = Arrays.copyOf(bytes, 16);
System.out.println(Arrays.toString(res));
// Output: [72, 101, 108, 108, 111, 32, 87, 111, 114, 108, 100, 0, 0, 0, 0, 0]

Using Unsafe Class

Finally, if you need more control over the conversion process you can use the 


          
           
<a href="https://docs.oracle.com/javase/8/docs/api/sun/misc/Unsafe.html" target="_blank" rel="noopener noreferrer">Unsafe</a>
class, which is a low-level API to access memory and manipulate objects. This class has a 

          
           
<a href="https://docs.oracle.com/javase/8/docs/api/sun/misc/Unsafe.html#putByte-long-byte-" target="_blank" rel="noopener noreferrer">putByte()</a>
method which can be used to write a byte value to a specified address in memory. This method can be used to create a byte array from a string by writing each character of the string to a byte array. 

          
           
String str = "Hello World"; 
byte[] bytes = new byte[str.length()]; 
Unsafe unsafe = Unsafe.getUnsafe(); 
for (int i = 0; i < str.length(); i++) { 
    unsafe.putByte(bytes, i, (byte) str.charAt(i)); 
} 
System.out.println(Arrays.toString(bytes));
// Output: [72, 101, 108, 108, 111, 32, 87, 111, 114, 108, 100]

String.getBytes()
and 
Arrays.copyOf()
methods are recommended. If more control is needed, the 
Unsafe
