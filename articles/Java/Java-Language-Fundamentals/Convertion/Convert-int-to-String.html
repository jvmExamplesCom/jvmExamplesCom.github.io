Converting int to String in Java

Java provides multiple ways of converting an integer to a string and each way has its own advantages. The most common ways are to use the 

          
           
Integer.toString()
method, 

          
           
String.valueOf(int)
method or the newer 

          
           
String.format()
method.

Using Integer.toString()

The Integer.toString() method is the simplest way to convert an 

          
           
int
to a 

          
           
String
. It takes an 

          
           
int
value as an argument and returns the corresponding string representation of the integer. 

          
           
//Example
int number = 45;
String numStr = Integer.toString(number);
System.out.println(numStr);
The output of the above code is: 

          
           
45

Using String.valueOf()

The 

          
           
String.valueOf()
method is another simple way to convert an 

          
           
int
to a 

          
           
String
. It takes an 

          
           
int
value as an argument and returns the corresponding string representation of the integer. It can be used in place of the 

          
           
Integer.toString()
method. 

          
           
//Example
int number = 45;
String numStr = String.valueOf(number);
System.out.println(numStr);
The output of the above code is: 

          
           
45

Using String.format()

The 

          
           
String.format()
method is a newer and more versatile way to convert an 

          
           
int
to a 

          
           
String
. It takes a format string as the first argument and an 

          
           
int
value as the second argument and returns the formatted string representation of the 

          
           
int
value. 

          
           
//Example
int number = 45;
String numStr = String.format("%d", number);
System.out.println(numStr);
The output of the above code is: 

          
           
45

Conclusion

In summary, there are three different ways to convert an 

          
           
int
to a 

          
           
String
in Java: using the 

          
           
Integer.toString()
method, 

          
           
String.valueOf()
method, or the 

          
           
String.format()
method. Each of the methods has its own advantages and it is up to the programmer to decide which one to use depending on their needs.
