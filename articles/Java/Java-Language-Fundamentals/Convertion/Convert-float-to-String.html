Convert Float to String in Java

Java provides a variety of ways to convert float to String. In this article, we'll go over three of the most common ways to convert float to String.

Using String.valueOf() Method

The 

          
           
String.valueOf()
method is the easiest and most straightforward way to convert float to String. 

          
           
float foo = 3.14f;
String bar = String.valueOf(foo);
System.out.println("The string value is:" + bar); 
// output: The string value is:3.14

Using Float.toString() Method

The 

          
           
Float.toString()
method is another easy way to convert float to String. 

          
           
float foo = 3.14f;
String bar = Float.toString(foo);
System.out.println("The string value is:" + bar); 
// output: The string value is:3.14

Using DecimalFormat Class

The 

          
           
DecimalFormat
class is a third way to convert float to String. This way is slightly more complicated, but it allows for more control over the decimal format. 

          
           
float foo = 3.14f;
DecimalFormat df = new DecimalFormat("#.##");
String bar = df.format(foo);
System.out.println("The string value is:" + bar); 
// output: The string value is:3.14
That's all there is to it! As you can see, converting float to String in Java is simple, and with the right methods, it can be done quickly and easily.
