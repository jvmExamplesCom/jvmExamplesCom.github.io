Java Final Variable Example

A final variable is a constant value which cannot be changed by the program during normal execution. The final keyword can be used with variables, methods and classes. In this article we will look at examples of using the final keyword in Java.

Final Variables

Final variables are declared with the final keyword followed by the type and the name of the variable: 

          
           
final int MAX_SIZE = 20;
Once a final variable has been assigned, it can't be changed or reassigned. Any attempts to modify it will result in a compile-time error. Here's an example of attempting to assign a final variable twice: 

          
           
final int MAX_SIZE = 20;
MAX_SIZE = 10; // Compile-time error: cannot assign a value to final variable MAX_SIZE

Final Methods

Final methods are declared with the final keyword followed by the return type, the name of the method and its parameters. A final method cannot be overridden by any subclasses: 

          
           
public final void setName(String name) {
    this.name = name;
}

Final Classes

Final classes are declared with the final keyword followed by the class name. They cannot be extended, meaning that no subclasses can be created from them. Here's an example of a final class: 

          
           
public final class Circle {
   ...
}

Conclusion

In this article we looked at examples of using the final keyword in Java. Final variables are constants which cannot be reassigned, final methods cannot be overridden, and final classes cannot be extended.
