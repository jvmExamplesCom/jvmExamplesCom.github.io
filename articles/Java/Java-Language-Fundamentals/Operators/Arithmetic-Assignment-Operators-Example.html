Java: Arithmetic Assignment Operators Example

Arithmetic assignment operators are an important part of Java, as they allow you to make use of some powerful mathematical operations. Whether you want to increase a variable's value by a certain amount, or decrease it by some factor, arithmetic assignment operators help make it happen. In this article, we'll go over the basics of arithmetic assignment operators in Java, by walking through a few examples of each of the available operations. We'll also provide example code snippets and some accompanying output to help illustrate the various operations' behaviors.

Adding with the += Operator

The += operator is used to add a value to a given variable. The result is then assigned to the same variable. Here's an example that shows how to add 5 to a given int value: 

          
           
int num = 10;
num += 5;
System.out.println(num);
Output: 

          
           
15
The += operator can also be used to add two values together and assign the result to one of the values: 

          
           
int x = 5;
int y = 3;
x += y;

System.out.println(x);
Output: 

          
           
8

Subtracting with the -= Operator

The -= operator is used to subtract a value from a given variable. The result is then assigned to the same variable. Here's an example that shows how to subtract 10 from a given int value: 

          
           
int num = 20;
num -= 10;
System.out.println(num);
Output: 

          
           
10
The -= operator can also be used to subtract two values, and assign the result to one of the values: 

          
           
int x = 5;
int y = 3;
x -= y;

System.out.println(x);
Output: 

          
           
2

Multiplying with the *= Operator

The *= operator is used to multiply a value with a given variable. The result is then assigned to the same variable. Here's an example that shows how to multiply 10 with a given int value: 

          
           
int num = 5;
num *= 10;
System.out.println(num);
Output: 

          
           
50
The *= operator can also be used to multiply two values together and assign the result to one of the values: 

          
           
int x = 5;
int y = 3;
x *= y;

System.out.println(x);
Output: 

          
           
15

Dividing with the /= Operator

The /= operator is used to divide a value by a given variable. The result is then assigned to the same variable. Here's an example that shows how to divide 10 by a given int value: 

          
           
int num = 20;
num /= 10;
System.out.println(num);
Output: 

          
           
2
The /= operator can also be used to divide a value by another value, and assign the result to one of the values: 

          
           
int x = 15;
int y = 3;
x /= y;

System.out.println(x);
Output: 

          
           
5

Modulo with the %= Operator

The %= operator is used to calculate the modulo of a given variable. The result is then assigned to the same variable. Here's an example that shows how to get the modulo of a given int value: 

          
           
int num = 15;
num %= 10;
System.out.println(num);
Output: 

          
           
5
The %= operator can also be used to calculate the modulo of two values, and assign the result to one of the values: 

          
           
int x = 23;
int y = 7;
x %= y;

System.out.println(x);
Output: 

          
           
2

Conclusion

Arithmetic assignment operators in Java are an easy and effective way to perform various mathematical operations on variables. This article has gone through a few examples of the various operators, illustrating their usage and output. With a better understanding of these operators, you can now confidently make use of them in your own Java programs.
