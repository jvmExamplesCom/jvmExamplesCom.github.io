Java: Modulus Operator Example

In Java, the modulus operator (%) is an arithmetic operator that takes two numeric expressions and divides them. The remainder of the division is then returned. The modulus operator is useful to obtain the last digits of a number, check whether a number is even or odd, or generate a repeating decimal.

Basic Modulus Examples

The most basic use case of the modulus operator is to obtain the last digit of a number. Using the modulus operator, the following code returns the last digit of the number 1356:


          
           
System.out.println(1356 % 10); // prints 6

The modulus can also be used to check if a number is odd or even. An even number will have a remainder of 0 when divided by 2, while an odd number will have a remainder of 1.


          
           
System.out.println(1356 % 2); // prints 0, thus the number is even
System.out.println(1355 % 2); // prints 1, thus the number is odd

Generating a Repeating Decimal

The modulus operator can be used to generate a repeating decimal. For example, consider the fraction 1/3. When computed normally, the result is 0.3333333333..., where the 3s continue indefinitely. This can be replicated using a loop and the modulus operator. The following code prints out the repeating decimal for 1/3:


          
           
int i = 1;
while (true) {
  System.out.print(i % 3);
  i++;
}
// prints 0.33333333333....

Finding Prime Numbers

The modulus operator can also be used to find prime numbers. A number is prime if it is only divisible by 1 and itself. To test if a number is prime, you need to loop through all the numbers from 2 to the number itself, and check if the modulus is 0. If it is, the number is not prime, since it is divisible by at least one number other than 1 and itself. Otherwise, the number is prime. This is demonstrated in the following code:


          
           
public static boolean isPrime(int n) {
  for (int i = 2; i < n; i++) {
    if (n % i == 0) return false;
  }
  return true; // number is prime
}
Using this code, we can check if 1 is prime: 

          
           
System.out.println(isPrime(1)); // prints false

Conclusion

The modulus operator is an extremely powerful and versatile tool for mathematics. It can be used to obtain the last digit of a number, check if a number is even or odd, generate a repeating decimal, and find prime numbers.
