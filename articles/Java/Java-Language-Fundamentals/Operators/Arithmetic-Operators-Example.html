Java - Arithmetic Operators Example

Arithmetic operators are operators for performing mathematical operations on values. In the Java programming language, these operators are used to perform addition, subtraction, multiplication, division, and modulus. This article will demonstrate how to use arithmetic operators in example programs.

Addition Operator

The addition operator is used to add two values together. The following example code demonstrates this: 

          
           
int a = 5;
int b = 8;

int c = a + b;

System.out.println(c);
The output of this code is 

          
           
13
.

Subtraction Operator

The subtraction operator is used to subtract one value from another. The following example code demonstrates this: 

          
           
int d = 10;
int e = 5;

int f = d - e;

System.out.println(f);
The output of this code is 

          
           
5
.

Multiplication Operator

The multiplication operator is used to multiply two values together. The following example code demonstrates this: 

          
           
int g = 7;
int h = 4;

int i = g * h;

System.out.println(i);
The output of this code is 

          
           
28
.

Division Operator

The division operator is used to divide one value by another. The following example code demonstrates this: 

          
           
int j = 25;
int k = 5;

int l = j / k;

System.out.println(l);
The output of this code is 

          
           
5
.

Modulus Operator

The modulus operator is used to calculate the remainder when one value is divided by another. The following example code demonstrates this: 

          
           
int m = 17;
int n = 8;

int o = m % n;

System.out.println(o);
The output of this code is 

          
           
1
.
