Java: Increment and Decrement Operators Example

Increment and decrement operators are unary operators that add or subtract one, respectively, to or from their operand, which has to be of a numeric type. They are most often used in looping operations to achieve an effect similar to "counting". Java supports both pre-increment and post-increment, as well as pre-decrement and post-decrement operators.

Pre-Increment Operator

The pre-increment operator ++x increments x before returning the result, so it can be used to increment and assign: 

          
           
int x = 1;
x = ++x;
System.out.println("x="+x);
// Output: x=2
In this example, the expression "++x" adds one to x and then assigns the new value (2) back to x. The result is that x is incremented and assigned the new value 2.

Post-Increment Operator

The post-increment operator x++ increments x after returning the result, so it can be used to assign and increment: 

          
           
int x = 1;
x = x++;
System.out.println("x="+x);
// Output: x=1
In this example, the expression "x++" returns the original value of x (1) and then adds one to the original value of x. The result is that x is assigned the original value (1) and then incremented to 2.

Pre-Decrement Operator

The pre-decrement operator --x decrements x before returning the result, so it can be used to decrement and assign: 

          
           
int x = 3;
x = --x;
System.out.println("x="+x);
// Output: x=2
In this example, the expression "--x" subtracts one from x and then assigns the new value (2) back to x. The result is that x is decremented and assigned the new value 2.

Post-Decrement Operator

The post-decrement operator x-- decrements x after returning the result, so it can be used to assign and decrement: 

          
           
int x = 3;
x = x--;
System.out.println("x="+x);
// Output: x=3
In this example, the expression "x--" returns the original value of x (3) and then subtracts one from the original value of x. The result is that x is assigned the original value (3) and then decremented to 2.
