Java Static Member Variable Example

In this article, we will discuss Static Member Variables in Java with examples. In Java programming language, every class can contain its own set of variables and methods that are commonly known as members of the class. These members can be accessed through their instance or through a class name.

Static Member Variables, also known as Class Variables, are those which are shared by all the instances (or objects) of the same class. All objects share the same copy of a static member variable and any changes made to it are reflected across all instances.

Syntax


          
           
access_modifier static data_type var_name;
Here, 
access_modifier
is the access specifier related to the variable, 
data_type
specifies the type of the variable and 
var_name
is the name of the variable.

Example

Let's consider an example to understand the concept of Static Member Variables.


          
           
public class Test {  
    //static variable  
    public static int b;  
}
Here 
b
is a static variable.

Static Variable Initialization

As soon as the class is loaded into memory, all its static variables get initialized automatically. In the case of numeric variables, they get assigned with a default value as 

0
, for Boolean variables, it's 
false
and for object references, it's 
null
. We can also explicitly assign a value to a static variable at the time of its declaration, like so: 

          
           
public class Test {  
    //explicit initialization  
    public static int b = 10;  
}
Here 
b
is initialized to 10.

Accessing Static Variable

We can access the static variable from the main method or from outside the class. It can be accessed either by class name or by instance.

By Class Name:


          
           
//Class Name  
Test.b

By Instance:


          
           
//Creating Object  
Test t=new Test();  

//Accessing member through instance  
t.b

Modifying a Static Variable

We can modify the value of a static variable by using an instance or a class name. Below is a simple program to demonstrate the concept.


          
           
public class Test {  
    //static variable  
    public static int b;  

    public static void main(String args[]){  
        //Accessing static variable through instance  
        Test t=new Test(); 
        t.b=20;  

        //Accessing static variable through class name  
        Test.b=50;  

        //printing value of static variable  
        System.out.println(t.b);  
        System.out.println(Test.b);  

    }  
}
Output: 

          
           
50
50

In the above program, we have accessed and modified the static variable 

b
using both instance and class name. In both the cases, the value of 
b
is changed to 50.

Conclusion

To summarize, Static Member Variables are those which are shared by all the instances of the same class and are mainly used to define properties at a class level that can be accessed without creating any instance of the class. They are initialized at the time of class loading and any modifications to them are reflected across all the instances.
