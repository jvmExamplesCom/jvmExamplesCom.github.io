Calculate Circle Area using Java Example

Java provides multiple ways of calculating the area of a circle. In this article, we will explore three different options for computing the area of a circle in Java.

Option 1: Math.pow Method

One approach is to use the Math.pow method to calculate the area of a circle as shown below. This is a straightforward way of calculating the area of a circle using the radius as an input. 

          
           
public static double CalculateCircleArea1 (double radius) {
    return Math.PI * Math.pow(radius,2);
}

Option 2: Multiplication of Radius

A second approach is to use the multiplication of radius to calculate the area of a circle, as shown below. This approach is based on the concept that the area of a circle is equal to πr2. 

          
           
public static double CalculateCircleArea2 (double radius) {
    return Math.PI * radius * radius;
}

Option 3: Using Math.PI

The final approach is to use the Math.PI method to calculate the area of a circle, as shown below. This approach is based on the concept that the area of a circle is equal to πr2. 

          
           
public static double CalculateCircleArea3 (double radius) {
    return  Math.PI * Math.PI * radius;
}
