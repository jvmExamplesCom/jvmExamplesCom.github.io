Java Interface Example

In this article, we will discuss how to create a Java interface and declare functions within it. We will also provide code examples along with their outputs.

Declaring an Interface

An interface in Java is a reference type, similar to a class, that can contain only constants, method signatures, and nested types. An interface can be declared using the keyword 

interface
. Here is an example of an interface named 
Animal
: 

          
           
public interface Animal {
  public void eat();
  public void sleep();
  public void move();
}
Here, we have declared three methods 
eat()
, 
sleep()
, and 
move()
within our interface 
Animal
. Note that all methods declared in an interface must be public and abstract. Therefore, no method body is required for interface methods.

Implementing an Interface

Once an interface is declared, it can be implemented by a class using the 

implements
keyword. A class can implement multiple interfaces. Here is an example of a class 
Dog
implementing the 
Animal
interface: 

          
           
public class Dog implements Animal {

  // Implementation of eat(), sleep(), and move()
  public void eat() {
    System.out.println("The dog is eating");
  }
  public void sleep() {
    System.out.println("The dog is sleeping");
  }
  public void move() {
    System.out.println("The dog is moving");
  }
}
After declaring our 
Dog
class and implementing the 
Animal
interface, we have provided an implementation for each of the methods declared in the 
Animal
interface.

Accessing Methods of an Interface

Once we have both an interface and a class implementing it, we can create an instance of the class and access the methods within the interface. Here is an example of a class that creates an instance of the 

Dog
class and accesses each of the methods declared in the 
Animal
interface: 

          
           
public class Main {
  public static void main(String[] args) {
    // Create dog object
    Dog myDog = new Dog();

    // Call eat(), sleep(), and move()
    myDog.eat();
    myDog.sleep();
    myDog.move();
  }
}

When this code is executed, the output will be as follows:


          
           
The dog is eating
The dog is sleeping
The dog is moving

We have successfully implemented an interface and accessed its methods.

Conclusion

In this article, we discussed how to create and implement an interface in Java. We also created a class that implements the interface and accessed the methods within it. Interfaces are a powerful concept in Java that helps us write custom, reusable code.
