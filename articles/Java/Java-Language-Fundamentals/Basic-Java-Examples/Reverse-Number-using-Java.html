Reverse Number Using Java

Java provides a simple way to reverse any numerical value as it is an object oriented programming language. The USe of the StringBuilder and StringBuffer classes can be used to reverse a number in Java. This article will discuss the various ways to reverse a number in Java with the use of examples.

Using StringBuilder Class

The StringBuilder class has a reverse() method that allows it to reverse the order of characters in a string. This method can be used to reverse any numerical value. The following example code reverses the value “123”: 

          
           
String str = "123";
StringBuilder sb = new StringBuilder(str);
String rev = sb.reverse().toString();
System.out.println(rev);
//Output: 321
In the above example, the string “123” is stored in the str variable then passed as an argument to the StringBuilder constructor. The reverse() method is called on the StringBuilder object which reverses the string and returns a StringBuffer object. The toString() method is then used to get the reversed string stored in the rev variable. This string is printed to the console using the print statement.

Using StringBuffer Class

The StringBuffer class also has a reverse() method which can be used to reverse any numerical value. The example code below performs the same task as the above example but uses the StringBuffer class instead: 

          
           
String str = "123";
StringBuffer sb = new StringBuffer(str);
String rev = sb.reverse().toString();
System.out.println(rev);
//Output: 321
In the above example, the string “123” is stored in the str variable then passed as an argument to the StringBuffer constructor. The reverse() method is called on the StringBuffer object which reverses the string and returns a StringBuffer object. The toString() method is then used to get the reversed string stored in the rev variable. This string is printed to the console using the print statement.

Using Iterative Approach

Iterative approach involves using loops such as for and while loops to reverse a number. The following example shows how to use a while loop to reverse a given number: 

          
           
public static int reverseNum(int num) {
    int rev = 0;
    while (num != 0) {
        rev *= 10;
        rev += num % 10;
        num /= 10;
    }
    return rev;
}

int num = 123;
int rev = reverseNum(num);
System.out.println(rev);
//Output: 321
In the above example, the reverseNum() method takes an integer value as a parameter and reverses it. This method uses a while loop to go through each digit in the number and adds it to a reverse variable. After all the digits have been processed, the reversed number is returned from the method. The main method then calls this method and stores the reversed number in the rev variable. This reversed number is printed to the console using the print statement.

Using Recursive Approach

Recursive approach involves using recursive functions to reverse a number. The following example shows how to use a recursive function to reverse a given number: 

          
           
public static int reverseNum(int num) {
    if (num == 0) 
        return 0;

    return (num % 10 * (int) Math.pow(10, String.valueOf(num).length() - 1))

int num = 123;
int rev = reverseNum(num);
System.out.println(rev);
//Output: 321
In the above example, the reverseNum() method takes an integer value as a parameter and reverses it. This method uses a recursive approach to add the last digit in the number to a variable and pass the remaining value to the same method. After all the digits have been processed, the reversed number is returned from the method. The main method then calls this method and stores the reversed number in the rev variable. This reversed number is printed to the console using the print statement.
