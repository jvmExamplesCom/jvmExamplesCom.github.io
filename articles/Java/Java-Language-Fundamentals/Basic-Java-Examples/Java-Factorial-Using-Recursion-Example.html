Java Factorial Using Recursion Example

Factorial of a number is defined as the product of all the numbers from 1 to that number. Factorial of a number can be computed using recursion as well as without using recursion. In this article, we will explain how to find the factorial of a number using recursive approach in Java.

Factorial Java Program Using Recursion

Let's see the factorial program in java using recursion. The program takes a number as input and finds the factorial using recursive method.


          
           
public class FactorialExample {

    // Java program to find Factorial of a number 
    // using recursion 
    static int factorial(int n) 
    { 
        if (n == 0) 
            return 1; 
       
        return n*factorial(n-1); 
    } 
       
    public static void main(String[] args)  
    { 
        int num = 4; 
        int fact = factorial(num); 
        System.out.println("Factorial of "+ num + " is " + fact); 
    } 
}
Output: Factorial of 4 is 24

Explanation of Factorial Program in Java

The main method calls a recursive method 'factorial()' which takes an integer as a argument. In the beginning, if the number is equal to 0 then it returns a value 1 and stops executing further. If the number is greater than 0, the method call itself recursively and multiplies it with the number passed as an argument until it is equal to 0. This process continues until the base condition (number=0) returns a value 1.

Therefore, the factorial of 4 will be calculated as 4*3*2*factorial(1)=4*3*2*1=24. For larger numbers, the recursive call continues until base condition returns a value.
