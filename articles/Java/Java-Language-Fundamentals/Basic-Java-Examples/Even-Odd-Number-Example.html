Java: Even Odd Number Example

An even number is an integer that can be divided by 2 without any remainder. An odd number can not be divided evenly by 2. In this section, we will provide some examples of how to determine whether a given number is even or odd using Java programming language.

Example 1

The following code example will check whether a given number is even or odd by using the modulus (%) operator. The modulus operator returns the remainder after dividing a number by another number. If the remainder is 0, then the number is even, else it is odd. 

          
           
public class Main {
    public static void main(String[] args) {
        int number = 4;
        if (number % 2 == 0) {
            System.out.println("Even");
        } else {
            System.out.println("Odd");
        }
    }
}
Output: 

          
           
Even

Example 2

Another way to check for even or odd numbers is to use the bitwise AND operator (&). We can simply compare the last bit of a given number to 1. If the result of the comparison is 1, then the number is odd, else it is even. 

          
           
public class Main {
    public static void main(String[] args) {
        int number = 4;
        if ((number &amp; 1) == 1) {
            System.out.println("Odd");
        } else {
            System.out.println("Even");
        }
    }
}
Output: 

          
           
Even

Example 3

Finally, we can also check for even and odd numbers using Java 8 Streams API. The following example shows how to use Java 8 Streams and Lambda expression to determine whether a number is even or odd. 

          
           
public class Main {
    public static void main(String[] args) {
        int number = 4;
        
        boolean isOdd = IntStream.of(number)
            .anyMatch(x -> (x &amp; 1) == 1);

        System.out.println(isOdd ? "Odd" : "Even");
    }
}
Output: 

          
           
Even
