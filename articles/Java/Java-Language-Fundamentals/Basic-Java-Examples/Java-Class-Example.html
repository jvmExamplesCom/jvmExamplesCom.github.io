Java Class Example

Creating a Class in Java

Creating a class in Java is relatively straightforward. A class is a template for an object and is used to define the attributes and methods that objects of this type possess. A class definition starts with the keyword "class" followed by the class name, and then the class body, enclosed by curly braces: 

          
           
public class MyClass {

}

Java Class Attributes

Attributes are variables defined within a class. An attribute has a type, such as int or String, and a name. All instances of the class will share the same attributes. To define an attribute in a class, you can use the following syntax: 

          
           
public class MyClass {
	// attributes
	int myAttribute;
	String myString;
}

Java Class Methods

Methods define the behavior of a class. A method consists of a name, a list of parameters (which can be empty), and a body. The body contains the code that is executed when the method is called. To define a method in a class, you can use the following syntax: 

          
           
public class MyClass {
	// methods
	void myMethod(int parameter) {
		// method body
		System.out.println("Hello World!");
	}
}

Calling a Method

Once a class has been defined and it contains methods, you can create an instance of the class and call the methods on it. To create an instance of a class, you can use the new operator followed by the class constructor: 

          
           
MyClass myObject = new MyClass();
To call a method on an instance, you can use the dot operator followed by the method name and its parameters. In the example below, we call the myMethod() method on the myObject object, passing in the parameter 10: 

          
           
myObject.myMethod(10); // prints: Hello World!

Conclusion

In this article, we have seen how to create a class in Java and how to define its attributes and methods. We have also seen how to create an instance of a class and call its methods. This should give you a basic understanding of classes in Java.
