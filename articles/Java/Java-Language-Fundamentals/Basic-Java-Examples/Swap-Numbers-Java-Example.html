Swap Numbers Java Example

Swapping two numbers means to exchange the values of the two variables with each other. This can be done using a temporary variable or by using arithmetic or bitwise operations. This article will explain how to swap two numbers using Java and provide some examples.

Using a Temporary Variable to Swapping

A common way to swap two numbers is to use a temporary variable. The logic behind this method is simple: save the value of one of the variables in the temporary variable and assign the second variable to the first one, and then assign the original value of the first variable back to the second one. This can be done with the following code: 

          
           
int num1 = 10;
int num2 = 20;
int temp;

temp = num1;
num1 = num2;
num2 = temp;
The result is that num1 has the original value of num2, and num2 has the original value of num1.

Using Arithmetic Operations to Swap

This method uses a combination of arithmetic operations to swap two numbers without the need for a temporary variable. It works by adding the two numbers together and then subtracting the second number from the sum, which results in the first number. It then subtracts the first number from the sum, which results in the second number. The code looks like this: 

          
           
int num1 = 10;
int num2 = 20;

num1 = num1 + num2;
num2 = num1 - num2;
num1 = num1 - num2;
Again, the result is that num1 has the original value of num2, and num2 has the original value of num1.

Using Bitwise Operations to Swap

A more efficient way to swap two numbers is to use bitwise operations. Bitwise operations allow us to manipulate the underlying binary representation of the numbers to achieve the desired result. The code to accomplish this is as follows: 

          
           
int num1 = 10;
int num2 = 20;

num1 = num1 ^ num2;
num2 = num1 ^ num2;
num1 = num1 ^ num2;
The XOR operator (^) is used in this method because it essentially flips the bits from 0 to 1 and vice versa. This results in the same effect as swapping the two numbers. This method is typically much faster than the previous two methods since it does not involve any arithmetic operations.

Conclusion

Swapping two numbers can be done in multiple ways using Java. The most common method is to use a temporary variable, which is straightforward but requires additional memory. Alternatively, arithmetic and bitwise operations can be used to swap two numbers without the need for a temporary variable, which can make the process faster and more efficient.
