Java Bubble Sort Descending Order Example

The bubble sort algorithm is a sorting technique used to sort elements of an array in descending order. It is one of the simplest algorithms to use and relatively fast compared to more complicated ones. In this article, we will look at an example of how to implement bubble sort in Java for a descending order.

Bubble Sort Algorithm Explanation

The bubble sort algorithm works by comparing each element of an array to its adjacent element and swapping them if their positions do not match the desired order. The algorithm continues to run until no swaps occur, which indicates that the array has been completely sorted. This is also known as the 'bubble sort' as the smallest element 'bubbles' to the top (or bottom) of the array. In this example, we will be using the bubble sort for a descending order.

Java Code Example

Below is the Java code to implement the bubble sort in descending order. The comments inside the code explain how the sorting works. 

          
           
// Function to implement bubble sort  
static void bubbleSort(int arr[], int n)  
{  
    int i, j, temp;  
    boolean swapped;  
    for (i = 0; i < n - 1; i++)  
    {  
        swapped = false;  
        for (j = 0; j < n - i - 1; j++)  
        {  
            if (arr[j] < arr[j + 1])  
            {  
                // swap arr[j] and arr[j+1]  
                temp = arr[j];  
                arr[j] = arr[j + 1];  
                arr[j + 1] = temp;  
                swapped = true;  
            }  
        }  
  
        // IF no two elements were  
        // swapped by inner loop, then break 
        if (swapped == false)  
            break;  
    }
}

If we input the following array into the above code 

          
           
int values[] = { 6, 3, 2, 9, 7, 1, 5, 4, 8 };
The output will be 

          
           
8, 7, 6, 5, 4, 3, 2, 1, 9
