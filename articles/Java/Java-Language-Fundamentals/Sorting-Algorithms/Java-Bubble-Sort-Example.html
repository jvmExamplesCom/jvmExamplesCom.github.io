Java Bubble Sort Example

Bubble Sort is a simple and popular sorting algorithm. It works by comparing two adjacent elements and swapping them if they are not in the correct order. This process continues until the entire list is sorted. Bubble Sort is also known as "Sinking Sort" and is most efficient for small lists.

Source Code For Bubble Sort Algorithm


          
           
public class BubbleSort {
    public static void sort(int[] arr) {
        int n = arr.length;

        // One pass of bubble sort. After
        // this pass, the largest element 
        // is moved (or bubbled) to end.
        for (int i=0; i<n-1; i++) 
            for (int j=0; j<n-i-1; j++) 
                if (arr[j] > arr[j+1]) { 
                    // swap arr[j] and arr[j+1]
                    int temp = arr[j]; 
                    arr[j] = arr[j+1]; 
                    arr[j+1] = temp; 
                } 
    }
}

Example I/O


          
           
Input: [7, 4, 5, 2, 8]
Output: [2, 4, 5, 7, 8]

Java Program to Demonstrate Bubble Sort


          
           
public class Main {
    public static void main(String[] args) {
        int[] arr = {7, 4, 5, 2, 8};

        System.out.println("Input array: " + Arrays.toString(arr));

        // calling bubble sort method 
        BubbleSort.sort(arr); 

        System.out.println("Sorted array: " + Arrays.toString(arr));
    }
}
