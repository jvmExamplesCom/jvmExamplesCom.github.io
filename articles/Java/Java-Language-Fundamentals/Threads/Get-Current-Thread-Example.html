Get Current Thread Example in Java

Java provides two methods that can be used to get a reference to the current thread of execution, they are 


          
           
Thread.currentThread()
and 

          
           
Thread.currentThread().getName()
. This article will show how to use these methods in a simple example.

Method 1: Thread.currentThread()

The first method is 

          
           
Thread.currentThread()
, which returns an instance of 
java.lang.Thread
that represents the current thread of execution. 

          
           
public static void main(String[] args) {
    Thread currentThread = Thread.currentThread();
    System.out.println("Current thread is: " + currentThread);
}
This code will print out the 
currentThread
object, which will look like this: 

          
           
Current thread is: Thread[main,5,main]

Method 2: Thread.currentThread().getName()

The second method is 

          
           
Thread.currentThread().getName()
, which returns the name of the current thread of execution as a 
String
. 

          
           
public static void main(String[] args) {
    String threadName = Thread.currentThread().getName();
    System.out.println("Current thread's name is: " + threadName);
}

This code will print out the name of the current thread of execution, which will look like this: 


          
           
Current thread's name is: main

By using either one of these methods, you can access the current thread of execution in Java and use it for whatever purpose you may have.
