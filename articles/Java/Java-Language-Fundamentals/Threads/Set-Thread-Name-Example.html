Java Set Thread Name Example

The purpose of this article is to provide an example of how to set a thread name in Java. We will look at two different ways of doing this, one using the Thread class and the other using a Runnable interface.

Thread Class

Using the Thread class is relatively simple and straightforward. You can use the setName() method to specify the thread’s name, like so: 

          
           
Thread t = new Thread();
t.setName("My Thread");
Or, if you don’t specify the thread name, the thread name will be the same as the thread ID generated by the JVM. 

          
           
Thread t = new Thread();
Finally you can also pass a String parameter to the thread constructor to set the thread name when instantiating the Thread object. 

          
           
Thread t = new Thread("My Thread");

Runnable Interface

When using the Runnable interface you can specify the thread name using the Thread.currentThread().setName() method. 

          
           
Thread t = new Thread(new Runnable(){
    public void run(){
        Thread.currentThread().setName("My Thread");
    }
});
It is important to note that the thread name must be set before the start() method is called on the thread.

Conclusion

Setting thread names in Java can be done easily using either the Thread class or the Runnable interface. The method used will depend on the overall structure of your program and which approach makes more sense for your use case.
