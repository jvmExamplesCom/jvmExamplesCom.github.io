Pause Thread Using Sleep Method Example

In Java, the Sleep method of Thread class is used to pause the execution of a thread for a specified amount of time. This method can be used to pause the thread if an operation has to be performed after a specific period of time. By using the sleep method, it is possible to put the current thread to sleep for a given amount of time in milliseconds.

Syntax

The syntax of this method is as below − 

          
           
public static void sleep(long millis) throws InterruptedException
This method can throw an InterruptedException which needs to be either caught or thrown.

Example

We will now see an example of using this sleep method to pause the thread. We have a simple program in which a thread is created and its sleep method is used to pause it for 1000 milliseconds (1 second). 

          
           
public class PauseThread {
   public static void main(String[] args) {
      Thread t1 = new Thread() {
         public void run() {
            for (int i = 0; i < 5; i++) {
               System.out.println("I am running " + (i+1));
               try {
                  Thread.sleep(1000);
               } catch (InterruptedException e) {
                  e.printStackTrace();
               }
            }
         }
      };
      t1.start();
   }
}
When we compile and execute the above program, it produces the following output − 

          
           
I am running 1
I am running 2
I am running 3
I am running 4
I am running 5
This shows that the program has paused for exactly 1000 milliseconds after the thread runs for each of its iterations.
