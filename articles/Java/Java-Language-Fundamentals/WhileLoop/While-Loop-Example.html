Java - While Loop Example

A while loop is a control flow statement that allows code to be executed repeatedly based on a given boolean condition. The while loop can be thought of as a repeating if statement. The syntax for a while loop in Java is: 

          
           
while (condition) {
   // code block to be executed
}
The condition is evaluated before executing the code block. If the condition evaluates to true, the code in the code block will be executed. After the code block has been executed, the condition will be evaluated again. This process is repeated until the condition becomes false. Let's look at a simple example of a while loop in action: 

          
           
int x = 0;

while (x < 10) {
   System.out.println("x is " + x);
   x++;
}
In this example, we are using a while loop to print out the value of the variable x, which starts at 0 and is incremented by 1 each time the loop is executed. The loop will continue until x is no longer less than 10. The output of this program would look like this: 

          
           
x is 0
x is 1
x is 2
x is 3
x is 4
x is 5
x is 6
x is 7
x is 8
x is 9
Another useful feature of the while loop is that you can use it with an array. For example, if you have an array of integers, you could use a while loop to iterate through the array and print out each element: 

          
           
int[] myArray = {1, 2, 3, 4, 5};
int i = 0;

while (i < myArray.length) {
   System.out.println(myArray[i]);
   i++;
}
The output of this program would be: 

          
           
1
2
3
4
5
You can also use the while loop with a Scanner object to read lines of text from the keyboard. For example, you might use a while loop to read in user input and process it until the user types in the word "done": 

          
           
Scanner scanner = new Scanner(System.in);
String line;

while (!(line = scanner.nextLine()).equals("done")) {
   System.out.println("You entered: " + line);
}
This program would keep reading lines from the keyboard until the user types in the word "done". It would then print out each line entered by the user. While loops are a very useful and powerful tool for writing programs in Java. They allow you to create loops that execute repeatedly until a certain condition is met. You can use them with arrays, scanners, and other objects to read and process data in your programs.
