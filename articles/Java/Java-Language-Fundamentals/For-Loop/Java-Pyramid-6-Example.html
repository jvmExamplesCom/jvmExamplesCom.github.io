Java Pyramid 6 Example

This tutorial will explain how to create a pyramid of numbers using Java. We will use the for loop in Java to construct this pyramid structure. A pyramid consists of a row of numbers that are arranged in such a way that each consecutive number is one less than the number preceding it.

Example 1

In this example, we will create a pyramid of five numbers, starting from 5 and ending at 1. 

          
           
public class Pyramid {
    public static void main(String[] args) {
        int num = 5;
        for (int i = 0; i < num; i++) {
            for (int j = 0; j < num - i; j++) 
                System.out.print(num - j);
            System.out.println();
        }
    }
}
The output of this code will be: 

          
           
54321
543
54
5
