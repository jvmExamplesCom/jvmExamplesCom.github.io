Java Pyramid 4 Example

We will also declare an integer variable to store the current row number and an integer variable to store the maximum number of columns. 

          
           
int numRows = 4;
int row = 0;
int maxCols = (2 * numRows) - 1;

We will use a for loop that will start at zero, increase by one for each time through the loop, and end when the current row number is equal to the number of rows. for (row = 0; row < numRows; row++) {
    for (int col = 0; col < maxCols; col++) {
        if ((col >= numRows - row - 1) &amp;&amp; (col <= numRows + row - 1)) {

        }
    }

}
