Java Pyramid 4 Example

In this article, we will be discussing the Java Pyramid 4 program with code examples and their outputs. This program can be used to print out a pyramid-like pattern of numbers in Java. The output of this program is shown below:

  121 
 12321

To create a pyramid at the command line using Java, we first need to declare a few variables. We need to create an integer variable that will be used to store the number of rows. We will also declare an integer variable to store the current row number and an integer variable to store the maximum number of columns. 

          
           
int numRows = 4;
int row = 0;
int maxCols = (2 * numRows) - 1;
Next, we need to create a loop that will allow us to print out each row of the pyramid. We will use a for loop that will start at zero, increase by one for each time through the loop, and end when the current row number is equal to the number of rows. Inside the loop, we will create a series of if statements that will be used to determine whether or not to print out a number in the current column. 

          
           
for (row = 0; row < numRows; row++) {
    for (int col = 0; col < maxCols; col++) {
        if ((col >= numRows - row - 1) &amp;&amp; (col <= numRows + row - 1)) {
            System.out.print(row + 1); 
        } else {
            System.out.print(" ");
        }
    }
    System.out.println();
}

If both of these conditions are satisfied, then it prints out the number of the current row plus one in the output. Otherwise, it prints out a space instead.
