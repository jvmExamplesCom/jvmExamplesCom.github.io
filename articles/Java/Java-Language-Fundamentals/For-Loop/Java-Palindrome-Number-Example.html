Java Palindrome Number Example

Palindrome numbers are those which represent same value if reversed. For example, 121, 454, 1221 etc are all palindrome numbers. In this article, we will look at a Java program to check if a given number is a palindrome or not and also discuss the algorithm behind it.

The Algorithm

The algorithm to check if a number is a palindrome is quite simple. First, we need to store the number in a variable. Then, we reverse the number and store it. Finally, we compare both the numbers and check if they are equal.

Program to Check for Palindrome Number

Below is the complete program to check for a palindrome number using Java: 

          
           
public class PalindromeNumber {

  public static void main(String[] args) {

    int num = 1221;
    int temp = num;
    int reversedNumber = 0;
    while (temp != 0) {
      int remainder = temp % 10;
      reversedNumber = reversedNumber * 10 + remainder;
      temp /= 10;
    }
    if (num == reversedNumber)
      System.out.println("Number is a palindrome.");
    else
      System.out.println("Number is not a palindrome.");
  }
}
The program first assigns the number to a variable num. We define another variable reversedNumber and set to 0. Then we use a while loop to reverse the number. Inside the loop, we get each digit from num using the modulo operator % and store it in the reversedNumber. We also divide num by 10 in each iteration. Finally, we compare num and the reversedNumber using an if else statement. If the two numbers are equal, it prints “Number is a palindrome.” otherwise it prints “Number is not a palindrome.”

Output

When we run the above program, it produces the following output: 

          
           
Number is a palindrome.
