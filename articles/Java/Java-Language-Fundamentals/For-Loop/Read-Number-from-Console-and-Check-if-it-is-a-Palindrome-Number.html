Read Number from Console and Check if it is a Palindrome Number in Java

A palindrome number is one that reads the same backward or forward. Examples of palindrome numbers are 8118, 7227 and 12321.

In this article, we will learn how to read a number from console and then check if it is a palindrome. We will also discuss a few programming challenges related to checking if a number is a palindrome.

Reading Numbers from Console

We can use the Scanner class in Java to read numbers from keyboard input. Here is an example code snippet which demonstrates how to read an integer number from console:


          
           
Scanner scanner = new Scanner(System.in);
int number = scanner.nextInt();
System.out.println("You have entered " + number);

In the above code snippet, we have created an instance of the Scanner class. We then read an integer using the 

nextInt()
method and print it out to the console.

Checking if a Number is Palindrome

We can use the loop and modulo operator to check if a number is a palindrome. Here is an example code snippet which demonstrates how to check if a number is a palindrome:


          
           
int temp = number;
int reverse = 0;
while (number > 0) {
    int lastDigit = number % 10;
    reverse = (reverse * 10) + lastDigit;
    number = number / 10;
}
if (temp == reverse) {
     System.out.println("The number is a palindrome");
} else {
     System.out.println("The number is not a palindrome");
}

In the above code snippet, we first store the original number in a temporary variable. We then use a while loop to reverse the digits of the number and store the reverse in another variable. Finally, we compare the original number with the reversed number and print out the result.
