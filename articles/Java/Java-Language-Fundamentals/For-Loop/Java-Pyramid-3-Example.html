Java Pyramid 3 Example

The classic pyramid 3 example of Java is an introductory program that shows how to print a right-angle triangle pattern in the Java language. It uses nested for loops to accomplish this goal. The following sections will provide a code example and explain how the code works.

Code Example 


          
           
public class Pyramid3 {
    public static void main(String [] args) 
    {
        for (int i = 0; i < 9; i++) 
        {
            for (int j = 0; j <= i; j++) 
            {
                System.out.print("*");
            }
            System.out.println();
        }
    }
}

Code Explanation

This code example consists of a main method, which contains two nested for loops. The outer loop is defined as 


          
           
for (int i = 0; i < 9; i++)
and the inner loop is defined as 

          
           
for (int j = 0; j <= i; j++)
. This code will run nine times, with the value of the variable i starting at 0 and increasing with each iteration.

The outer loop is designed to print a row of asterisks for each iteration it executes. The inner loop prints an asterisk for each iteration and stops when it has printed as many asterisks as the current value of the variable i. When both loops have finished executing on the same row, the program prints a new line with the command 


          
           
System.out.println();
, in order to shift the output to the next row.

The result of this code is a right-angle triangle with nine rows. The first row will contain one asterisk, the second row two asterisks, and so on. By changing the values of the variables, the user can control the size of the output.
