Introduction to Infinite For Loop in Java

An infinite for loop in Java is a loop that continues executing until some condition is met. It is used when we do not know how many times a code should repeat, or when the loop should be repeated indefinitely. The condition in the for statement is left empty so it does not terminate until a break statement is encountered.

Syntax of an Infinite For Loop in Java

A for loop consists of three parts: 

          
           
for (initialization; condition; increment/decrement) {
	// code block to be executed
}
In an infinite for loop, only the initialization part is ever executed. The condition and the increment/decrement parts are left empty. This means that the loop will not terminate until some condition is met or until a break statement is encountered. The syntax of an infinite for loop in Java can be written as follows: 

          
           
for (initialization; ; ) { 
	 // code block to be executed  
}

Example 1: Printing Numbers from 1 to 10 Using an Infinite for Loop

In this example, we will use an infinite for loop to print numbers from 1 to 10. 

          
           
public class InfiniteForLoopExample { 
        public static void main(String[] args) {

            // from 1 to 10 
            for (int i = 1; ; i++) {
                System.out.println(i);
