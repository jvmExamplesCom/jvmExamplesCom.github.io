List Even Numbers Java Example

In this article, we will look at the process of listing out all the even numbers in Java. We will first look at how to use the for loop to list all even numbers from 0 to a given value. We will then look at a recursive approach and see how to list all the even numbers from 0 to a given value using recursion. Finally, we will also look at another approach where we can list all even numbers from a specific range of numbers.

Using a For Loop

The easiest way to list all even numbers up to a given value is by using a for loop. We can use a for loop to iterate over the numbers starting from 0 and ending at the given value. At each iteration, we can check the current number is even or not; if it is even we can add it to the list. The following example shows how it can be done: 

          
           
public static List<Integer> listEvenNumbersUsingForLoop(int n) {
    List<Integer> evenNumbers = new ArrayList<>();
    for (int i=0; i<=n; i++) {
        if (i % 2 == 0) {
            evenNumbers.add(i);
        }
    }
    return evenNumbers;
}
In the above example, we iterate over all numbers from 0 to the given value and check if the current number is even or not; if it is even, the number is added to the list.

Using Recursion

An alternative approach to listing even numbers up to a given value is to use recursion. The idea is to divide the problem into smaller subproblems until the problem is so small that it can be solved directly. In this particular case, for each odd number, we can use recursion to list all the even numbers before it. In other words, for each odd number, we can simply call the same method with 'n-1' as the argument and list all the even numbers any less than or equal to n-1. 

          
           
public static List<Integer> listEvenNumbersUsingRecursion(int n) {
    List<Integer> evenNumbers = new ArrayList<>();
    if (n == 0) 
        evenNumbers.add(0);
    else if (n % 2 == 0) {
        evenNumbers.add(n);
        listEvenNumbersUsingRecursion(n - 1);
    }
    else {
        listEvenNumbersUsingRecursion(n - 1);
    }
    return evenNumbers;
}

Listing Even Numbers From a Specific Range

The previous two methods can only list even numbers up to a given number whereas we may need to list even numbers from a specific range of numbers. For example, we may want to list all the even numbers from 20 to 50. In such cases, we can again use a for loop and iterate over the numbers from the lower bound to the upper bound. At each iteration, we can check if the number is even or not and add it to the list. The following example shows how it can be done: 

          
           
public static List<Integer> listEvenNumbersFromRange(int lower, int upper) {
    List<Integer> evenNumbers = new ArrayList<>();
    for (int i=lower; i<=upper; i++) {
        if (i % 2 == 0) {
            evenNumbers.add(i);
        }
    }
    return evenNumbers;
}
In the above example, we iterate over all numbers from the lower bound to the upper bound. For each number, we check if it is even or not and add it to the list if it is.
