Generate Pyramid For a Given Number Example in Java

Generating pyramid using numbers is a common programming exercise that can help new developers understand the concept of loops and a basic understanding of Java language. Here we'll provide an example of generating a number pyramid for a given number, as well as step-by-step instructions for how to complete this task.

Java Program to Generate Pyramid for a Given Number

The following program is written in Java and can be used to generate a pyramid for a given number: 

          
           
public class Pyramid {
  public static void main(String[] args) {
    int rows = Integer.parseInt(args[0]);

    for (int i=1; i<=rows; i++) {
      for (int j=1; j<=i; j++) {
        System.out.print(j);
      }
      System.out.println();
    }
  }
}

Explanation of Code

In the code above, we first declare a class called 

          
           
Pyramid
. This is where our program logic resides and is responsible for printing out the pyramid for the given number. We then have a main method, which is where the program starts execution. At this point, the program takes in the number of rows to print out as an argument and stores it in the integer 

          
           
rows
. We then create two 

          
           
for loops
- one to loop through the rows and one to loop through the columns. In each iteration, we print out the respective column number. After printing out the whole row, we move onto the next one.

3
