Java Pyramid 2 Example

A pyramid is one of the most commonly used structures in programming. This tutorial will explain how to create a pyramid using the Java programming language.

The program below is a basic example of how to create a pyramid using Java. It will use both a for loop and a while loop:


          
           
public class JavaPyramid2Example { 
   public static void main(String[] args) { 
      // Create pyramid using a for loop 
      for (int i = 0; i < 5; i++) { 
         for (int j = 0; j <= i; j++) { 
            System.out.print("*" + " "); 
         } 
         System.out.println(); 
      } 
                   
      // Create pyramid using a while loop 
      int x = 4; 
      while (x >= 0) { 
         int y = 0; 
         while (y <= x) { 
            System.out.print("*" + " "); 
            y++; 
         } 
         System.out.println(); 
         x--; 
      } 
   } 
}
