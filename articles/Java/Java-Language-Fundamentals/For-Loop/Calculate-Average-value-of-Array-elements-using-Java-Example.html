Calculate Average Value of Array Elements Using Java Example

Java is a powerful programming language used by software developers around the world. It provides a simple, easy to use syntax for writing efficient and robust code. In this tutorial we will look at how to calculate the average value of an array's elements using Java.

Example Code

First, let's define an array of integers to use in our example: 

          
           
int[] myArray = {1, 2, 3, 4, 5};
We can now write a simple loop to iterate over the elements of our array: 

          
           
int sum = 0;
for (int i = 0; i < myArray.length; i++) {
    sum += myArray[i];
}
Once we have the sum of all the elements, we can calculate the average value: 

          
           
int average = sum / myArray.length;
To test our code, we can print out the average value to the console: 

          
           
System.out.println("Average is: " + average);
The complete code for calculating the average value of an array's elements can be seen below: 

          
           
int[] myArray = {1, 2, 3, 4, 5};
int sum = 0;
for (int i = 0; i < myArray.length; i++) {
    sum += myArray[i];
} 
int average = sum / myArray.length;
System.out.println("Average is: " + average);

Output

When the code is run, it should produce the following output: 

          
           
Average is: 3
