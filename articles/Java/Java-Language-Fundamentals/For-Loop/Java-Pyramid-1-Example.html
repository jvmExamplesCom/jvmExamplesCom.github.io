Java Pyramid 1 Example

This tutorial will provide an example of a Java pyramid program with both explanation and code. This example is for intermediate/advanced Java developers, so if you are just starting out you should start off with the basics first.

Explanation

A Java pyramid is a program that prints out a pyramid made out of numbers. The numbers are arranged in increasing order with the biggest number at the top followed by smaller numbers on each level. The total number of levels is based on the user’s input. The following is an example of a 5-level Java pyramid. 

          
           
1
   212
  32123
 4321234
543212345
In this example, the user has inputted a 5 as the number of levels. Each row of the pyramid reflects the number of levels - 5 rows in this case. Also, each row has an increasing number of numbers - from 1 number in the first row to 5 numbers in the last row.

Code

The following code is an example of a Java program that prints a pyramid for the user’s input. 

          
           
public class Pyramid {
    public static void main(String[] args) {
        int numOfLevels = 5; // user's input
        int count = 1;
        for (int i = 0; i < numOfLevels; i++) {
            for (int j = numOfLevels - i; j > 1; j--) {
                System.out.print(" ");
            }
            for (int k = 0; k <= i; k++) {
                System.out.print(count + "");
                count++;
            }
            for(int l = i; l > 0; l--){
                count--;
                System.out.print(count + "");
            }
            System.out.println();
            count = 1;
        }
    }
}
The first line declares the Pyramid class. The second line declares the main function with its argument list. This is where the program’s instructions will be written. The third line declares the variable numOfLevels. This variable is assigned the user’s input which is the number of levels they want in the pyramid. The fourth line initializes the count variable to 1. This variable is used in the loops to determine which numbers to print on each row. The fifth line is a for loop. The loop is run the same number of times as the user inputted. This loop is responsible for creating each row of the pyramid, one row for each iteration. The sixth and seventh line create another two for loops. These loops are responsible for controlling the spacing between the numbers and printing the appropriate numbers to the screen. The eighth line of the code will output a newline after each row. This will ensure that each row prints out on its own line. Finally, the ninth line resets the count variable back to 1 for the next row.

Output

When the program is executed, it will produce the following output. 

          
           
1
   212
  32123
 4321234
543212345
