Java JLabel Get Tooltip Example

In this example, we will learn how to use the getToolTipText() method of JLabel class. The 

          
           
JLabel
class is a component for creating text labels for GUI applications. This class provides methods for displaying, setting, and retrieving text labels. The 

          
           
getToolTipText()
method of 

          
           
JLabel
class is used to get the tooltip (text description) of the label. If a tooltip is not set, this method returns 

          
           
null
.

Example

Let us look at an example of using the 

          
           
getToolTipText()
method of 

          
           
JLabel
class. 

          
           
import javax.swing.JLabel;

public class JLabelGetTooltipEx {
    public static void main(String[] args) {
        // Create a JLabel with a text description 
        JLabel lbl = new JLabel("Text Description");
        lbl.setToolTipText("This label contains a text description");
        
        // Get the tooltip text
        System.out.println(lbl.getToolTipText());
    }
}

          

          
           
This label contains a text description

JLabel

setToolTipText()
method. Then, we use the 

          
           
getToolTipText()
method to get the tooltip text of this label.
