Align Icon in JLabel Example

This example code demonstrates how to align an Icon in a JLabel.

The example code below is a basic implementation of how to align an Icon in a JLabel. In this example, the Icon is aligned to the right side of the label which is done via the IconTextGap property.


          
           
JLabel label = new JLabel("Label"); 
label.setIcon(new ImageIcon("myImage.png"));
label.setHorizontalTextPosition(SwingConstants.RIGHT);
label.setIconTextGap(6);

The IconTextGap property sets the gap between the icon and text. This is an integer value expressed in pixels.

It is also possible to align the Icon to the left side, or to the top or bottom of the JLabel. To do this, use the setHorizontalTextPosition() and setVerticalTextPosition() methods, like so:


          
           
// Align icon to the left
label.setHorizontalTextPosition(SwingConstants.LEFT);

// Align icon to the top
label.setVerticalTextPosition(SwingConstants.TOP);

// Align icon to the bottom
label.setVerticalTextPosition(SwingConstants.BOTTOM);

Output

The output of the above code will be a JLabel with the Icon aligned to the right side.

You can also experiment with the IconTextGap property to further adjust the Icon's position within the JLabel.
