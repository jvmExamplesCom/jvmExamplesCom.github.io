Set Horizontal Alignment of JLabel Example

Java Swing provides a wide range of GUI controls and components. JLabel is one of the most used components in Swing applications. This tutorial will demonstrate how to set the horizontal alignment of a JLabel component using Java Swing.

Creating the JLabel Object

To begin, we need to create an instance of the JLabel class.


          
           
JLabel label = new JLabel("My Text");

Next, we can set the Horizontal Alignment of the JLabel component by using the setHorizontalAlignment() method.

Setting Horizontal Alignment

The setHorizontalAlignment() method takes an int argument which correspond to the constants declared in the SwingConstants interface.


          
           
label.setHorizontalAlignment(SwingConstants.LEFT);

This example sets the horizontal alignment to LEFT. The following constants are available:


          
           
CENTER, LEFT, RIGHT, LEADING, or TRAILING

Example Program

The following example program demonstrates how to set horizontal alignment of a JLabel component.


          
           
import javax.swing.*;  
import java.awt.FlowLayout;  
public class HorizontalAlignmentExample {  
    public static void main(String[] args) {  
        //Create the frame  
        JFrame frame = new JFrame("Horizontal Alignment Example");  
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);  
        frame.setSize(300, 200);  
        frame.setLayout(new FlowLayout());  
          
        //Create the label  
        JLabel label = new JLabel("My Text");  
        label.setHorizontalAlignment(SwingConstants.LEFT);  
          
        //Add the label to the frame  
        frame.add(label);  
          
        //Show the frame  
        frame.setVisible(true);  
    }  
}
