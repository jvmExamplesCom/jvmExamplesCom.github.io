X and Y Coordinates of JLabel Example

The Java language allows developers to create complex graphical interfaces for applications. In this tutorial, we'll explore how to retrieve the coordinates of a 

JLabel
. This can be useful for many applications â€“ allowing users to click within a canvas or perform other operations on it.

Creating the Label

We'll start by creating a 
JLabel
that is 200 pixels wide and 50 pixels tall. This will be our canvas for our example code. 

          
           
JFrame frame = new JFrame("Coordinates");
frame.setSize(200, 50);

JLabel label = new JLabel();
frame.add(label);

frame.setVisible(true);

Retrieving the X and Y Coordinates of the Label

Now that we have our canvas, we can use the 

          
           
getLocationOnScreen()
method to retrieve the X and Y coordinates of the label. We'll store the 
Point
object returned by the 

          
           
getLocationOnScreen()
method and then extract the x and y coordinates from it. 

          
           
Point point = label.getLocationOnScreen();
int xCoord = (int)point.getX();
int yCoord = (int)point.getY();

We now have the X and Y coordinates of the label stored in the variables 

xCoord
and 
yCoord
, respectively. We can now use these values for any purpose within our application.

Example Output

Using the code above, we can output the coordinates of the label like so:


          
           
System.out.println("X Coordinate: " + xCoord);
System.out.println("Y Coordinate: " + yCoord);
This would result in the following output: 

          
           
X Coordinate: 10
Y Coordinate: 10
