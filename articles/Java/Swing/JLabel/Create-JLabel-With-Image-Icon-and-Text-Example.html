Create JLabel With Image Icon and Text Example

In this article, we’ll discuss how to create a JLabel with an Image Icon and Text using the Java Swing package. Swing is one of the most popular GUI libraries for Java and is used to create applications with a graphical user interface (GUI). In this example, we’ll create a simple Label that displays an image icon along with some text.

Creating the Label

The first step is to create a new Label object which will contain our image icon and text:


          
           
JLabel label = new JLabel();

Setting the Icon

Once the Label has been created, we can set an image icon to be displayed on it. This can be done by passing the ImageIcon object to the label's setIcon() method:


          
           
ImageIcon icon = new ImageIcon("icon.jpg");
label.setIcon(icon);

Setting the Text

Once the icon has been set, we can set the text to be displayed on the label. This is done by calling the Label’s setText() method:


          
           
label.setText("Hello World!");

Adding the Label to a Container

Now that our Label has been created, we need to add it to a container such as a JPanel. This is done by calling the container’s add() method: 


          
           
JPanel panel = new JPanel();
panel.add(label);

Displaying the Label

Finally, we need to make sure that the Label is displayed on the screen. We can do this by calling the JFrame’s pack() method:


          
           
JFrame frame = new JFrame();
frame.add(panel);
frame.pack();
frame.setVisible(true);

The output should look like the following: 

          
           
<img src="output-example.png" alt="example output">

Conclusion

In this article, we’ve discussed how to create a JLabel with an Image Icon and Text using the Java Swing package. We created a Label object and set the icon and text to be displayed. We then added the Label to a container such as a JPanel and called the JFrame’s pack() method to finally display the Label on the screen.
