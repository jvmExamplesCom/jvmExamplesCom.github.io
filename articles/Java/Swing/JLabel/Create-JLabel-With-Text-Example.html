Create JLabel With Text Example

This article will provide you with an example of how to create a 

          
           
JLabel
with text using the Java programming language. The 

          
           
JLabel
class is part of the Swing library which facilitates the creation of graphical user interfaces (GUIs). A 

          
           
JLabel
is a special type of text field that displays non-editable text.

Create JLabel Object

To create a new 

          
           
JLabel
object, use the following syntax: 

          
           
JLabel label = new JLabel("My Label");
This will create a 

          
           
JLabel
object named 

          
           
label
with the text 

          
           
"My Label"
.

Set Text for JLabel

You can change the text for your 

          
           
JLabel
object by using the 

          
           
setText()
method. For example: 

          
           
label.setText("My New Label");
This will set the text for the 

          
           
JLabel
object named 

          
           
label
to 

          
           
"My New Label"
.

Create JLabel With Text And Icon

You can also create a 

          
           
JLabel
with both text and an icon. To do this, use the following syntax: 

          
           
ImageIcon icon = new ImageIcon("myicon.png");
JLabel label = new JLabel("My Label", icon, JLabel.LEFT);
In this example, 

          
           
icon
is an 

          
           
ImageIcon
object created from the image file 

          
           
myicon.png
. The third argument for the 

          
           
JLabel
constructor specifies the position of the icon relative to the text. In this case, the icon is positioned to the left of the text.

Add JLabel to Container

Once you have created your 

          
           
JLabel
object, you need to add it to a container in order for it to be displayed. For example, if you have a

object named 

          
           
panel
: 

          
           
panel.add(label);

JLabel

panel
.
