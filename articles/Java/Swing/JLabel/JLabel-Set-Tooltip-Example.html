Java JLabel Set Tooltip Example

JLabel
using the 

          
           
setToolTipText()
method in Java.

setToolTipText()

Example

The following example creates a 

          
           
JLabel
and sets the tooltip text for it using the 

          
           
setToolTipText()
method. 

          
           
JFrame frame = new JFrame("Demo");
frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
frame.setSize(200, 200);

JLabel label = new JLabel("My Label");
label.setToolTipText("This is my tooltip text");
frame.add(label);

frame.setVisible(true);
