Set Background Color of JLabel Example in Java

In this article, we are going to discuss how to set background color of 

JLabel
in Java. 
JLabel
is a component used to display a short text string or an image, or both. It is one of the most commonly used components in Swing-based applications. The default background color for 
JLabel
is transparent. If you want to change it, you can use the following methods.

Using JLabel setBackground Method

The 
setBackground()
method of 
JLabel
can be used to set a new background color. The syntax for this method is as follows. 

          
           
public void setBackground(Color color)
Where, 
  • color
    : Represents the new background color of the 
    JLabel
    component.
Example: 

          
           
JLabel label = new JLabel("Hello World!");
label.setBackground(Color.RED);

Using JLabel setOpaque Method

The 
setOpaque()
method of 
JLabel
can also be used to set a new background color. This method must be called before invoking the 
setBackground()
method. The syntax for this method is as follows. 

          
           
public void setOpaque(boolean opaque)
Where, 
  • opaque
    : Represents the boolean value whether the 
    JLabel
    component is opaque or not.
Example: 

          
           
JLabel label = new JLabel("Hello World!");
label.setOpaque(true);
label.setBackground(Color.RED);
This is how you can set background color of 
JLabel
in Java.
