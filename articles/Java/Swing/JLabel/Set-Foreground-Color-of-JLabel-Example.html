Set Foreground Color of JLabel Example

This tutorial will demonstrate how to set the foreground color of a Java Swing 

JLabel
object, using Java code. In order to set the foreground color, you will use the 
setForeground()
method of the 
JLabel
class.

Example Code


          
           
// Create a new JLabel instance
JLabel label = new JLabel("Hello World!");

// Set the foreground color
label.setForeground(Color.green);

The output of the code above would be a Java Swing 

JLabel
object with the text "Hello World!" and the foreground color set to green.
