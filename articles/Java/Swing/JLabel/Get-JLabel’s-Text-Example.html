Get JLabel Text Example In Java

In this tutorial, we will learn how to get the text from a 

JLabel
component in Java. The 
JLabel
class is used to create labels which are components which display text or images. We can use the 
setText()
method of the 
JLabel
class to set the text of the label, and the 
getText()
method to retrieve the text of the label.

Creating the JLabel

First, let's create a simple 
JFrame
window and add a 
JLabel
to it. 

          
           
JFrame frame = new JFrame("My Frame");

JLabel label = new JLabel("This is my label");
frame.add(label);

frame.pack();
frame.setVisible(true);

Getting the Text

To get the text from the 
JLabel
, we can use the 
getText()
method. This method will return the text of the label as a 
String
. 

          
           
String text = label.getText();

Example

Let's look at a complete example that creates a 
JFrame
window with a 
JLabel
and prints out the text of the label. 

          
           
import javax.swing.*;

public class MyClass {
    public static void main(String[] args) {
        JFrame frame = new JFrame("My Frame");

        JLabel label = new JLabel("This is my label");
        frame.add(label);

        frame.pack();
        frame.setVisible(true);

        // Get the text from the label
        String text = label.getText();
        System.out.println(text); // Prints "This is my label"
    }
}
When we run this program, the output will be: 

          
           
This is my label
