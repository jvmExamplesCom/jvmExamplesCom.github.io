Set Icon for JLabel Example in Java

This article is about how to set an icon for a JLabel in Java. We will explore the various ways to add an icon to a label and provide code examples to illustrate each approach.

Using the setIcon Method

The simplest way of setting an icon for a JLabel is to use the 

          
           
setIcon()
method. This method takes as its parameter a 

          
           
javax.swing.Icon
object. The following code example uses this method to set a simple icon on a 

          
           
JLabel
instance: 

          
           
JLabel label = new JLabel("My Label");
ImageIcon icon = new ImageIcon("icon.png");

label.setIcon(icon);
label.setAlignmentY(JLabel.CENTER_ALIGNMENT);
This code creates a new 

          
           
JLabel
instance with the text “My Label” and then creates an 

          
           
ImageIcon
instance using the file named “icon.png”. The 

          
           
setIcon()
then sets the icon on the label and the 

          
           
setAlignmentY()
method centers the icon within the label.

Using the JLabel Constructor

It is also possible to set an icon on a 

          
           
JLabel
by passing the icon as a parameter when creating a new instance of 

          
           
JLabel
. The following code example shows how to do this: 

          
           
ImageIcon icon = new ImageIcon("icon.png");
JLabel label = new JLabel("My Label", icon, JLabel.CENTER);
In this example, the 

          
           
JLabel
constructor takes three parameters: the text of the label, the icon for the label, and the alignment for the icon. In this case, the alignment is set to 

          
           
JLabel.CENTER
.

Using the setIconTextGap Method

By default, the icon is placed to the left of the text for a 

          
           
JLabel
. To place the icon on the right side, use the 

          
           
setIconTextGap()
method. This method takes an integer that represents the number of pixels between the icon and the text of the label. The following example shows how to use this method: 

          
           
JLabel label = new JLabel("My Label");
ImageIcon icon = new ImageIcon("icon.png");

label.setIcon(icon);
label.setAlignmentY(JLabel.CENTER_ALIGNMENT);
label.setIconTextGap(20);
In this example, the 

          
           
setIconTextGap()
method is used to set the gap to 20 pixels, which places the icon on the right side of the text.

Using the setHorizontalAlignment Method

It is also possible to set the horizontal alignment for a 

          
           
JLabel
with the 

          
           
setHorizontalAlignment()
method. This method takes one of the following constants: 

          
           
JLabel.LEFT
, 

          
           
JLabel.CENTER
, or 

          
           
JLabel.RIGHT
. The following example shows how to use this method: 

          
           
JLabel label = new JLabel("My Label");
ImageIcon icon = new ImageIcon("icon.png");

label.setIcon(icon);
label.setAlignmentY(JLabel.CENTER_ALIGNMENT);
label.setHorizontalAlignment(JLabel.RIGHT);
In this example, the 

          
           
setHorizontalAlignment()
method is used to set the alignment to 

          
           
JLabel.RIGHT
, which places the icon on the right side of the text.

Conclusion

In this article, we have explored different ways to set an icon for a JLabel in Java. We have seen how to use the 

          
           
setIcon()
, 

          
           
JLabel
constructor, 

          
           
setIconTextGap()
, and 

          
           
setHorizontalAlignment()
methods to achieve this. With these techniques, you should now be able to easily set an icon for a JLabel in your own Java applications.
