Set Vertical Alignment of JLabel Example

In this article, we are going to look at how to set the vertical alignment of a 

          
           
JLabel
object in Java. It is possible to set the vertical alignment of a 

          
           
JLabel
using the 

          
           
setVerticalAlignment()
method of the 

          
           
JLabel
class. This method takes a single parameter, which is an integer representing the vertical alignment desired. The valid values for this parameter are defined in the 

          
           
SwingConstants
class, which is a part of the Java Swing API. Below is a full example of setting the vertical alignment of a JLabel in Java: 

          
           
// Create JLabel
JLabel label = new JLabel("This is my Label");

// Set the vertical alignment of the JLabel
label.setVerticalAlignment(SwingConstants.TOP);
In this example, we have created a 

          
           
JLabel
object and set its vertical alignment to TOP, as defined by the 

          
           
SwingConstants.TOP
constant. In addition to TOP, there are also constants for CENTER and BOTTOM, which can be used to set the vertical alignment of the label in the same way. For example, this code sets the vertical alignment to BOTTOM: 

          
           
label.setVerticalAlignment(SwingConstants.BOTTOM);
Alternatively, you can also set the vertical alignment to one of the predefined numeric values as an int instead of using the constants, although this is not recommended: 

          
           
label.setVerticalAlignment(0);
The first two parameters represent TOP and CENTER respectively, with BOTTOM being represented by the number 2. In addition to setting the vertical alignment, it is also possible to get the current vertical alignment using the 

          
           
getVerticalAlignment()
method, as seen in this example: 

          
           
int alignment = label.getVerticalAlignment();
The value returned from this method will be one of the predefined constants from the 

          
           
SwingConstants
class. It is worth noting that setting the vertical alignment of a 

          
           
JLabel
is not necessary in all cases. In some cases it may be desirable to allow the default vertical alignment to be used.
