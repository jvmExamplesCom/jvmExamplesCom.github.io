Horizontal Alignment of JLabel Example

The following example code demonstrates the horizontal alignment of a Java Swing 

          
           
JLabel
component. 

          
           
import java.awt.*;
import javax.swing.*;

public class HorizontalAlignTest extends JFrame {

  public HorizontalAlignTest() {
    super("Horizontal Align Test");
    this.setSize(250, 200);
    getContentPane().setLayout(null);

    JLabel lbl = new JLabel("Example String");
    lbl.setBounds(50,20,150,20);
    lbl.setHorizontalAlignment(SwingConstants.LEFT);
    getContentPane().add(lbl);
  }

  public static void main(String[] args) {
    HorizontalAlignTest ha = new HorizontalAlignTest();
    ha.setVisible(true);
  }
}
The example creates a 

          
           
JFrame
and adds a 

          
           
JLabel
. The frame size is set to 250x200 pixels and the label bounds are set to 50,20,150,20. The label's text is "Example String" and its horizontal alignment is set to 

          
           
SwingConstants.LEFT
, which aligns it to the left side of the window. The output from this example would be a window with a 

          
           
JLabel
in the top left corner containing the text "Example String". Here's another example that shows how to right-align a
