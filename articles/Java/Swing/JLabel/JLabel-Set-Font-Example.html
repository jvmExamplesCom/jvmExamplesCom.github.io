JLabel Set Font Example

Java's JLabel class offers several methods of setting font. This article will explore a few examples demonstrating how to set font in a JLabel instance. Before we dive into examples, it's important to note that the font settings are inherited from the parent component. Therefore, setting font for JLabel can also be done with its parent component, such as a JPanel if desired.

Using setFont()

The simplest way to set font for JLabel is by using the setFont() method. The setFont() method takes a java.awt.Font object as parameter and sets it as the font of the JLabel. Here is an example: 

          
           
Font font = new Font("Arial", Font.PLAIN, 14);
JLabel label = new JLabel("My JLabel");
label.setFont(font);
This code snippet creates a Font object with the desired style and size and assigns it to the JLabel using the setFont() method. The setFont() method takes a Font object as parameter and sets it as the font of the JLabel.

Using setFont(String, int)

The setFont() method also takes two additional parameters- String and int. The String parameter is used to specify the font name (e.g. “Arial”) and the second int parameter is used to set the font size. Here is an example: 

          
           
JLabel label = new JLabel("My JLabel");
label.setFont("Arial", 14);
This code snippet creates a Font object with the specified font name and size and assigns it to the JLabel.

Using setFont(Font)

The setFont() method also accepts a Font object as parameter and sets it as the font of the JLabel. To create a Font object, you can use the Font constructor or one of its static factory methods. Here is an example: 

          
           
Font font = Font.createFont(Font.TRUETYPE_FONT, new File("myfont.ttf"));
JLabel label = new JLabel("My JLabel");
label.setFont(font);
This code snippet creates a Font object from a TrueType font file and assigns it to the JLabel.

Conclusion

In this article, we explored a few examples demonstrating how to set font in a JLabel instance. We saw how to use setFont(), setFont(String, int) and setFont(Font) methods.
