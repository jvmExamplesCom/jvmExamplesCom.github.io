Disable JLabel Example

Disabling a 
JLabel
in Java is a simple task that requires just two lines of code. The first line sets the label’s enabled state to false, and the second line removes the label from the container in which it resides. This can be useful when trying to hide certain labels or display different labels depending on user input.

Example 1: Simple JLabel Disable


          
           
JLabel label = new JLabel("My Label");
label.setEnabled(false);

In this example, we create a new JLabel with the text “My Label”. Then we call the setEnabled() method on our label object, passing in false as the first argument. This will set the enabled property of our label to false, but the label will still be visible in the user interface.

Example 2: Remove JLabel from Container


          
           
JLabel label = new JLabel("My Label");
container.remove(label);

In this example, we create a new JLabel with the same text “My Label”. Then we call the remove() method on our container object, passing in our label object as the first argument. This will remove the label from the specified container, making it invisible to the user.

Example 3: Disable and Remove JLabel


          
           
JLabel label = new JLabel("My Label");
label.setEnabled(false);
container.remove(label);

In this example, we do both steps together in one statement. After creating our JLabel with the text “My Label”, we call setEnabled() with false as its argument and then remove() with our label as the argument. This will both disable the label and remove it from the container, making it completely invisible to the user.
