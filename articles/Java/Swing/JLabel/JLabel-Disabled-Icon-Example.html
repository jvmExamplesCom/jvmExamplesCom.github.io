JLabel Disabled Icon Example

The JLabel Disabled Icon Example explains how to set the disabled icon of a JLabel. JLabels are components used to create text labels in swing applications. They can also be used to add an image with an optional border. The example below shows how to create a disabled icon for the JLabel.

Create a JLabel with a Disabled Icon

To create a JLabel with a Disabled Icon, use the following code: 

          
           
JLabel label = new JLabel("Text Label");
label.setDisabledIcon(new ImageIcon("disabledIcon.png"));
This will create a JLabel with the text “Text Label” and a DisabledIcon set to the image specified.

Get the Disabled Icon from a JLabel

To get the Disabled Icon from a JLabel, use the following code: 

          
           
ImageIcon disabledIcon = (ImageIcon)label.getDisabledIcon();
This will retrieve the Disabled Icon from the JLabel.

Set the Disabled Icon of a JLabel

To set the Disabled Icon of a JLabel, use the following code: 

          
           
label.setDisabledIcon(new ImageIcon("disabledIcon.png"));
This will set the Disabled Icon of the JLabel to the image specified.

Example

The following example shows how to create a JLabel with a Disabled Icon: 

          
           
import javax.swing.*;
 
public class Demo {
    public static void main(String[] args) {
        JFrame frame = new JFrame();
        JLabel label = new JLabel("Text Label");
        label.setDisabledIcon(new ImageIcon("disabled.png"));
        frame.add(label);
        frame.setVisible(true);
    }
}
The output of this code will be a JLabel with the text “Text Label” and a DisabledIcon set to the image specified by “disabled.png”.
