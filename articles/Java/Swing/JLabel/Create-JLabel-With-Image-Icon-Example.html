Create JLabel With Image Icon Example

In this tutorial, we will learn how to create a 

          
           
JLabel
with an image icon in Java. 

          
           
JLabel
is a class of Java Swing used to create labels and make them more attractive. There are many different icons that can be used to add interest to a 

          
           
JLabel
; these include various shapes, such as circles or rectangles as well as images.

Setting up the Project

First, we need to set up our Java project. We will use the NetBeans IDE 8.0.2 for this example. To begin, create a new project by selecting 

          
           
File → New Project
from the menu bar. Select 

          
           
Java Application
and click 

          
           
Next
. 

          
           
Project Name: JLabelImageExample
Click 

          
           
Finish
to create the project.

Adding the Image

Now we need to add the image file that we want to use for our icon. This can be any image type supported by Java, such as PNG, GIF, or JPG. Create a new folder in your project directory and name it 

          
           
images
. Copy the image file into this folder, and then drag it from the 

          
           
Files
window into the 

          
           
Source Packages
window.

Creating the JLabel

Now let's create the 

          
           
JLabel
. The first step is to get the 

          
           
ImageIcon
object. The 

          
           
ImageIcon
class is used to represent an image icon. To do this, we need to use the 

          
           
getResource()
method of the 

          
           
Class
class: 

          
           
ImageIcon icon = new ImageIcon(getClass().getResource("images/image.png"));
Next, we create the 

          
           
JLabel
object and set its text and icon properties: 

          
           
JLabel label = new JLabel();
label.setText("Label Text");
label.setIcon(icon);
Finally, we add the 

          
           
JLabel
to our frame: 

          
           
frame.add(label);

Running the Program

When you run the program, the 

          
           
JLabel
should appear like this: 

          
           
Label Text
That's it! We have successfully created a 

          
           
JLabel
with an image icon in Java.
