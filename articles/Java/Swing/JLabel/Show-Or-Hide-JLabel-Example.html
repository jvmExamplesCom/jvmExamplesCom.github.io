Java: Show Or Hide JLabel Example

In this article, we will discuss an example of how to show or hide a JLabel in Java. We will first create a sample program which will contain a JLabel, and then we will demonstrate how to show or hide it.

Create a sample program

Let's start by creating a simple program which will contain a JLabel. The code for the program is given below: 

          
           
import javax.swing.*;
public class ShowHideExample 
{
    public static void main(String[] args)
    {
        JFrame f = new JFrame("Show/Hide JLabel Example");  
        JLabel label = new JLabel("This is a label!");    
        label.setBounds(50, 50, 100, 30); 
        f.add(label);  
        f.setSize(400, 400);  
        f.setLayout(null);  
        f.setVisible(true);    
    }
}
In the above program, we have created a JFrame and added a JLabel to it. The JLabel has been given a size and position, and the JFrame has been set to visible.

Show/Hide JLabel

Now that we have created a sample program, let's see how we can show or hide the JLabel. To show or hide a JLabel, all we need to do is set its visibility: 

          
           
label.setVisible(true/false);
The setVisible() method takes a boolean as a parameter. If the boolean value is true, then the JLabel will be visible, and if the boolean is false, then the JLabel will be hidden. So, to show the JLabel, we will use the following code: 

          
           
label.setVisible(true);
To hide the JLabel, we will use the following code: 

          
           
label.setVisible(false);

Complete Program

Let's now see how to use the show/hide feature in a complete program. The code for the program is given below: 

          
           
import javax.swing.*; 
import java.awt.event.*; 

public class ShowHideExample 
{
    public static void main(String[] args) 
    {
        JFrame f = new JFrame("Show/Hide JLabel Example");  
        final JLabel label = new JLabel("This is a label!");    
        label.setBounds(50, 50, 150, 30); 
        f.add(label);  
        JButton btn = new JButton("Show/Hide");
        btn.setBounds(50, 100, 95, 30);
        f.add(btn);  

        btn.addActionListener(new ActionListener() 
        {
            public void actionPerformed(ActionEvent e) 
            {
                label.setVisible(!label.isVisible()); 
            }
        });

        f.setSize(400, 400);  
        f.setLayout(null);  
        f.setVisible(true);  
    }
}
In the above program, we have added a button which will be used to show or hide the JLabel. The button has an action listener attached to it, and when the button is clicked, the actionPerformed() method is called. In this method, we have used the setVisible() method to show or hide the JLabel, depending on its current visibility.

Conclusion

In this article, we discussed an example of how to show or hide a JLabel in Java. We first created a sample program which contained a JLabel, and then we showed how to show or hide the JLabel using the setVisible() method.
