Set Background Color of JTextField Example

Using setBackground() Method

The 

          
           
setBackground()
method is used to set the background color of a 

          
           
JTextField
component. The syntax of the method is as follows: 

          
           
public void setBackground(Color bgcolor)
Here, 

          
           
bgcolor
is the specified Color object. To illustrate this example, we have created two classes – MyFrame, and BackgroundExample. All the code snippets mentioned in this article can be found in our GitHub repository.

MyFrame Class

The MyFrame class has been created to create the window frame. We can customize the frame as per our requirements. In our example, we have used the following code: 

          
           
import javax.swing.*;
public class MyFrame extends JFrame {
    public MyFrame() {
        setSize(400, 300);
        setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        setLayout(null);
        setVisible(true);
    }
}

BackgroundExample Class

In the BackgroundExample class, we have used the 

          
           
setBackground()
method to set the background color of a 

          
           
JTextField
component. The following code illustrates how to use this method: 

          
           
import java.awt.Color;
import javax.swing.*;
public class BackgroundExample extends MyFrame {
    public BackgroundExample() {
        JTextField txtField = new JTextField();
        txtField.setBounds(50, 50, 100, 30);
        txtField.setBackground(Color.GREEN);
        add(txtField);
    }

    public static void main(String[] args) {
        new BackgroundExample();
    }
}
