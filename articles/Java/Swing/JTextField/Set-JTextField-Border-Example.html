JTextField Border Example

Java provides a 

          
           
setBorder()
method to create a border around a 

          
           
JTextField
. This method takes an argument of type 

          
           
Border
, which can be any of the pre-defined borders defined in the 

          
           
javax.swing.border
package.

Creating Borders

The 

          
           
javax.swing.border
package contains several subclasses of 

          
           
Border
, all of which can be used to construct a border for a 

          
           
JTextField
. Some of the most commonly used borders are: 
  • LineBorder
    : Creates a line border with the specified color and thickness.
  • BevelBorder
    : Creates a beveled edge border with the specified type (raised or lowered) and color.
  • EmptyBorder
    : Creates an empty border with the specified border width.
  • CompoundBorder
    : Combines two borders into one.

Using 
setBorder()

Once you have created a 

          
           
Border
instance, you can use the 

          
           
setBorder()
method on an existing 

          
           
JTextField
instance to set the border. The following code shows an example of how to do this: 

          
           
JTextField textField = new JTextField();
Border lineBorder = BorderFactory.createLineBorder(Color.BLACK, 1);
textField.setBorder(lineBorder);
In the code example above, we create a 

          
           
LineBorder
with a black color and a thickness of 1 pixel. We then use the 

          
           
setBorder()
method to set the border of the 

          
           
JTextField
to the newly created 

          
           
LineBorder
.

Example Code

Now that we have seen how to create and use a 

          
           
Border
, let's look at a complete example of creating a 

          
           
JTextField
with a 

          
           
BevelBorder
. Note that the 

          
           
BevelBorder
allows us to specify the type of bevel (raised or lowered) as well as the color of the border. 

          
           
JTextField textField = new JTextField();
Border bevelBorder = BorderFactory.createBevelBorder(BevelBorder.RAISED, Color.RED);
textField.setBorder(bevelBorder);
In the code example above, we create a 

          
           
BevelBorder
with a raised style and a red color. We then use the 

          
           
setBorder()
method to set the border of the 

          
           
JTextField
to the newly created 

          
           
BevelBorder
.

Compound Borders

The 

          
           
CompoundBorder
class allows us to combine two borders into one. This can be useful if you want to create a border with more than one component. For example, let's say we want to create a 

          
           
JTextField
with a border composed of a 

          
           
LineBorder
and a 

          
           
BevelBorder
. Here is the code: 

          
           
JTextField textField = new JTextField();
Border bevelBorder = BorderFactory.createBevelBorder(BevelBorder.RAISED, Color.RED);
Border lineBorder = BorderFactory.createLineBorder(Color.BLACK, 1);
Border compoundBorder = BorderFactory.createCompoundBorder(lineBorder, bevelBorder);
textField.setBorder(compoundBorder);
In the code example above, we create a 

          
           
BevelBorder
and a 

          
           
LineBorder
, and then combine them into a single 

          
           
CompoundBorder
. We then use the 

          
           
setBorder()
method to set the border of the 

          
           
JTextField
to the newly created 

          
           
CompoundBorder
.

Conclusion

In this article, we have learned how to create and use a 

          
           
Border
on a 

          
           
JTextField
. We have seen how to create the different types of borders available in the 

          
           
javax.swing.border
package, and how to use the 

          
           
setBorder()
method to apply the border to a 

          
           
JTextField
. Finally, we have also seen how to use the 

          
           
CompoundBorder
class to combine two borders into one.
