JTextField Show Hide Example

Introduction

This article will provide an example of how to use JTextField to show or hide text fields in Java. The example will be written using the Swing library and will show how to create a text field that can be either visible or hidden at the click of a button. This example will also demonstrate how to use a ActionListener to control the visibility of the text field.

Example Code

The following code is a complete example of how to create and use a JTextField. We will create a GUI with two buttons: one to show the text field, and one to hide it. 

          
           
import javax.swing.*;

public class JTextFieldShowHideExample {
    
    public static void main(String[] args) {
        //Create the window
        JFrame frame = new JFrame("JTextField Show/Hide Example");
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setSize(400, 400);

        //Create the text field
        JTextField textField = new JTextField();
        textField.setVisible(false);

        //Create the buttons
        JButton showButton = new JButton("Show");
        JButton hideButton = new JButton("Hide");

        //Add action listeners to the buttons
        showButton.addActionListener((e) -> textField.setVisible(true));
        hideButton.addActionListener((e) -> textField.setVisible(false));

        //Create the panel
        JPanel panel = new JPanel();
        panel.add(showButton);
        panel.add(hideButton);
        panel.add(textField);
        frame.add(panel);

        //Display the window
        frame.pack();
        frame.setVisible(true);
    }
}
