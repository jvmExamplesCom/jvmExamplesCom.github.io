JTextField ToolTip Example

In this example, we will learn about how to create and use JTextField ToolTip. The Java Swing toolkit provides a class named JTextField which is used to create a textbox. It also provides a property named setToolTipText to display a tooltip when the user hovers the mouse over a JTextField component.

Example 1: Simple JTextField ToolTip

In this example, we have created a JTextField using its constructor, and then set a tooltip to it using setToolTipText() method. 

          
           
import javax.swing.*;  
public class JTextFieldTooltipExample{  
    public static void main(String[] args) {  
        JFrame f= new JFrame("JTextField Tooltip Example");   
        //Creating textfield  
        JTextField text = new JTextField("Welcome");             
        //setting tooltip text  
        text.setToolTipText("This is a TextField");              
        f.add(text);  
        f.setSize(400,400);    
        f.setLayout(null);    
        f.setVisible(true);    
    }  
}
