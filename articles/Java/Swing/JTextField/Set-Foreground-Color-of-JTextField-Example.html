Setting Foreground Color of a JTextField – Example

Java swing provides various options to customize various components. One of the component is JTextField, the text field component of swing. The foreground color of a text field can be changed using the setForeground()method of JTextField class. This tutorial will help you understand how to set foreground color of a JTextField.

Example Code: Setting the Foreground Color of a JTextField

In this example, we’ll create a JTextField and set its foreground color to red. After that, we’ll add it to the frame.


          
           
JFrame frame = new JFrame("Set Foreground Color of JTextfield Example");
 
// Create a text field  
JTextField textField = new JTextField(20);
 
// Set foreground color to red  
textField.setForeground(Color.red);
 
// Add the text field to frame  
frame.add(textField);
 
// Set the size of frame  
frame.setSize(400, 400);
 
// Set the visibility of frame  
frame.setVisible(true);
