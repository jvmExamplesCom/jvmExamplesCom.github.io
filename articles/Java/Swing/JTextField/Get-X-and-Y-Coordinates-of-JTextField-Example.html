Get X and Y Coordinates of JTextField Example

Many applications require knowing the position of the cursor within a window or graphical user interface. Often this is done using the mouse cursor's coordinates to reference where the user's selection (or lack thereof) on the screen is located. This article will discuss how to use the Java swing library to find the x and y coordinates of a JTextField so you can easily determine where the user has selected within the textfield.

JTextField Setting Up

Before you can begin to obtain the x and y coordinates of a JTextField instance, you must first set up the Java Swing environment by creating an instance of a JTextField. Below is an example of the syntax used to create a JTextField instance: 

          
           
JTextField textField = new JTextField();
This creates a new instance of a JTextField, which is stored in the variable textField. Now, you can use textField to get the x and y coordinates of the JTextField.

Getting the X and Y Coordinates of JTextField

To get the x and y coordinates of the JTextField instance, the getLocationOnScreen() method must be called on the textField variable. The getLocationOnScreen() method returns a java.awt.Point object, which contains the x and y coordinates of the JTextField. Below is an example of how a java.awt.Point object can be stored in a variable, called point, and used to get the x and y coordinates of the JTextField instance, stored in the variable textField: 

          
           
java.awt.Point point = textField.getLocationOnScreen();
The x coordinate of the JTextField instance can be obtained by calling the x property of the point variable, as shown in the following example: 

          
           
int x = point.x;
Likewise, the y coordinate of the JTextField instance can be obtained by calling the y property of the point variable, as shown in the following example: 

          
           
int y = point.y;

Putting it All Together

Below is an example that combines all of the previous steps into one, showing how to store the x and y coordinates of a JTextField instance in two variables, x and y: 

          
           
JTextField textField = new JTextField();
java.awt.Point point = textField.getLocationOnScreen();
int x = point.x;
int y = point.y;
Now that you know the x and y coordinates of the JTextField instance, you can easily determine where the user's selection is located in the window or graphical user interface.
