JTextField Horizontal Alignment Example

In this tutorial, we'll learn how to align the text within JTextField horizontally. Aligning text in a JTextField can be tricky, but knowing some basic Java code can help us get it done quickly.

Creating a TextField Object

Let's start by creating a JTextField object. We can do this with the following code. It will create a new text field with a width of 10 and height of 20 pixels:


          
           
JTextField tf = new JTextField(10,20);

Aligning Text Within the Text Field

Now that we've created our text field object, we can use the 


          
           
setHorizontalAlignment()
method to set the alignment of the text within the text field. The alignment can be one of four values: 
  • JTextField.LEFT
    , which aligns the text to the left
  • JTextField.CENTER
    , which aligns the text to the center
  • JTextField.RIGHT
    , which aligns the text to the right
  • JTextField.LEADING
    , which aligns the text to the leading edge of the text field (typically the left edge)

For example, to align the text within our text field to the right, we can use the following code:


          
           
tf.setHorizontalAlignment(JTextField.RIGHT);

Example Program

Here is a complete program that demonstrates how to use the 


          
           
setHorizontalAlignment()
method to align the text in a text field: 

          
           
import javax.swing.*;

public class TextFieldExample {

  public static void main(String[] args) {

    JFrame frame = new JFrame("TextFieldExample");
    frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
    frame.setSize(400,200);
    frame.setLayout(null);

    // Create and position text field
    JTextField tf = new JTextField(10,20);
    tf.setBounds(50,50,200,30);

    // Set alignment
    tf.setHorizontalAlignment(JTextField.CENTER);

    frame.add(tf);
    frame.setVisible(true);
  }
}
