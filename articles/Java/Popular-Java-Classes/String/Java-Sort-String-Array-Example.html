Java Sort String Array Example

In this article, we are going to learn how to sort a string array in Java. We will look at different ways to sort an array in Java, including the Arrays.sort() method and the Collections.sort() method.

Using the Arrays.sort() Method

The Arrays class in java provides a static method called sort(). This method uses the merge sort algorithm to sort an array of strings. The syntax for the sort() method is:


          
           
Arrays.sort(String[] array);

Let's see an example of how to use the sort() method. In this example, we have an array of strings that we want to sort in ascending order:


          
           
String[] cities = {"New York", "Chicago", "Los Angeles", "Houston"};
We can use the sort() method to sort the array: 

          
           
Arrays.sort(cities);

The resulting array will be sorted in ascending order:


          
           
["Chicago", "Houston", "Los Angeles", "New York"]

Using the Collections.sort() Method

The Collections class in java provides a static method called sort(). This method uses the insertion sort algorithm to sort an array of strings. The syntax for the sort() method is:


          
           
Collections.sort(List<String> list);

Let's see an example of how to use the sort() method. In this example, we have a list of strings that we want to sort in ascending order:


          
           
List<String> cities = Arrays.asList("New York", "Chicago", "Los Angeles", "Houston");
We can use the sort() method to sort the list: 

          
           
Collections.sort(cities);

The resulting list will be sorted in ascending order:


          
           
["Chicago", "Houston", "Los Angeles", "New York"]

Conclusion

In this article, we learned how to sort a string array in Java. We looked at two methods for sorting an array, the Arrays.sort() method and the Collections.sort() method. Both methods use different algorithms to sort the array.
