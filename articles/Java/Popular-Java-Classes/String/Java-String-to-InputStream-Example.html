Java String to InputStream Example

This article provides a comprehensive introduction to the conversion of a Java String to the InputStream object. It will cover various methods that can be used and showcase examples with complete code snippets. It is important to understand that String is an immutable sequence of characters and InputStream, on the other hand, is a stream of data, containing as bytes or characters. This means we need to explicitly define the transformation that occurs when converting a String to an InputStream.

Methods to convert String to InputStream

There are various options available when it comes to converting a String to InputStream. Below, we have covered some of the most common methods employed by developers.

Using ByteArrayInputStream

The ByteArrayInputStream class is one of the simplest ways to convert a String to an InputStream. This method requires the use of a byte array to pass the data of the string into the input stream. By using this approach, there is no need to write any additional code to achieve the desired result. The code below shows a sample conversion of a String to an InputStream using ByteArrayInputStream: 

          
           
String str = "Hello World!";
InputStream is = new ByteArrayInputStream(str.getBytes());

Using StringReader

Another method for transforming a String into an InputStream is via the use of a StringReader instance. The StringReader provides an interface for reading characters from a given string. The example below outlines a basic implementation: 

          
           
String str = "Hello World!";
Reader reader = new StringReader(str);
InputStream is = new ReaderInputStream(reader);

Using CharArrayReader

Next, we have the CharArrayReader which is another useful option for converting a String to an InputStream. This method requires the usage of a character array as the source for the data. The following code provides an example of how this method can be used: 

          
           
String str = "Hello World!";
Reader reader = new CharArrayReader(str.toCharArray());
InputStream is = new ReaderInputStream(reader);

Using PipedInputStream

Another way to perform the conversion of a String to an InputStream is through the use of a PipedInputStream instance. This method is based on the concept of "piping" data between two threads. Here, we create a separate thread which writes the data of the string into the piped output stream. The example code below indicates how this can be carried out: 

          
           
String str = "Hello World!";
PipedInputStream pis = new PipedInputStream();
PipedOutputStream pos = new PipedOutputStream(pis);

Thread t = new Thread(new Runnable(){
    public void run(){
        try {
            pos.write(str.getBytes());
            pos.close();
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
});
t.start();
InputStream is = pis;
