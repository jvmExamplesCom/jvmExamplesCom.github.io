Java String Array Length Example

In this article, we will explain how to use the Java String Array Length and provide some code examples. The Java String Array Length is a useful tool that returns the number of elements in a specified array. This can be used to check if a certain array has any elements in it, or to loop over all of the elements in an array.

Using the Java String Array Length

To use the Java String Array Length, you must first declare an array, assign values to the elements, and then call the array's length method. The length method will return the number of elements in the array. 

          
           
String[] myArray = {"Element 1", "Element 2", "Element 3"};
int myArrayLength = myArray.length;
System.out.println(myArrayLength); // Outputs "3"
In this example, the array named 

          
           
myArray
has three elements, so the length method will return 

          
           
3
. It is important to note that the length only includes the elements that have been assigned in the array. If an element has not been assigned, it will not be included in the count.

Example Code

Let's look at a few more examples of using the Java String Array Length. 

          
           
int[] myIntArray = new int[5];
int myIntArrayLength = myIntArray.length;
System.out.println(myIntArrayLength); // Outputs "5"
In this example, we create an array of integers called 

          
           
myIntArray
. Since it has been declared with five elements, the length method will return 

          
           
5
.
