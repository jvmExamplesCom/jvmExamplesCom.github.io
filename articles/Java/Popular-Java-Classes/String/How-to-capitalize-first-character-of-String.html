How to Capitalize the First Character of a String in Java

Java makes capitalizing the first character of a string simple and straightforward. Whether you’re creating a new string or modifying an existing one, you can utilize several approaches to achieve the effect. Let’s investigate the various ways to capitalize the first letter of a string in Java.

Option 1: Using the substring() and toUpperCase() Methods

If you want to capitalize the first character of a string, you can make use of the 

          
           
substring()
and 

          
           
toUpperCase()
methods. In order to do this, create a new string with the first letter of the original one in uppercase. To do this, use the following syntax: 

          
           
String str = "my string";
String first_letter_uppercase = str.substring(0,1).toUpperCase();
In this example, 

          
           
str
is a string of your choice and 

          
           
first_letter_uppercase
is a new string with the first letter capitalized.

Option 2: Using the replaceFirst() Method

Another way to capitalize the first character of a string is by using the 

          
           
replaceFirst()
method. This approach uses regular expressions to match and replace the first letter of a string. The syntax for this method is as follows: 

          
           
String str = "my string";
String first_letter_uppercase = str.replaceFirst("^.", str.substring(0,1).toUpperCase());
Here, the regular expression 

          
           
"^."
matches the first character of the string, while 

          
           
str.substring(0,1).toUpperCase()
capitalizes it.

Option 3: Using StringBuilder/StringBuffer Class

You can also capitalize the first character of a string by making use of the 

          
           
StringBuilder
or 

          
           
StringBuffer
class. This approach modifies an existing string in-place, and the syntax is as follows: 

          
           
String str = "my string";
StringBuilder str_builder = new StringBuilder(str);
str_builder.setCharAt(0, Character.toUpperCase(str.charAt(0)));
String first_letter_uppercase = str_builder.toString();
In this example, we create a new 

          
           
StringBuilder
object called 

          
           
str_builder
and set its value to our string. Then, we use the 

          
           
setCharAt()
method to set the first character of the string as capitalized. Finally, we convert the 

          
           
StringBuilder
object back to a string.

Option 4: Using Apache Commons Library

The Apache Commons library offers a “StringUtils” class with a method called 

          
           
capitalize()
. This method takes a string and capitalizes the first letter of each word in the string. The syntax for this approach is as follows: 

          
           
String str = "my string";
String first_letter_uppercase = StringUtils.capitalize(str);
As you can see, using the Apache Commons library makes capitalizing the first letter of a string simple and straightforward.

Conclusion

In this article, we explored four ways to capitalize the first character of a string in Java. We investigated the 

          
           
substring()
, 

          
           
toUpperCase()
, 

          
           
replaceFirst()
, 

          
           
StringBuilder/StringBuffer
, and 

          
           
StringUtils.capitalize()
methods. With any of these approaches, you can quickly and easily capitalize the first letter of a string.
