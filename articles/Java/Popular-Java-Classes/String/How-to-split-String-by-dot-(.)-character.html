Splitting Strings By Dot (.) Character With Java

Java provides a variety of ways to split strings by dot (.) character. In this article, we will explore different methods and their use cases.

Using String.split() Method

The simplest way to split a string based on dot (.) character is to use the 

          
           
String.split()
method. It takes a regular expression as an argument and returns an array of strings split by the given regex. For example, the following code splits a string on dots (.): 

          
           
String str = "Hello.world.java";
String[] result = str.split("\\.");
System.out.println(Arrays.toString(result));
The output of the above code would be: 

          
           
[Hello, world, java]
It is important to note that the 

          
           
split()
method will return an array of strings regardless of how many dots (.) are present in the original string. This makes it very versatile and can be used to split a string based on any regular expression.

Using StringTokenizer Class

Another way to split a string based on dot (.) character is to use the 

          
           
StringTokenizer
class. It allows you to split a string into tokens by providing a delimiter. In our case, this delimiter would be a dot (.). Here is an example of how to use the 

          
           
StringTokenizer
class for splitting a string based on a dot (.): 

          
           
String str = "Hello.world.java";
StringTokenizer st = new StringTokenizer(str, "\\.");
while (st.hasMoreTokens()) {
    System.out.println(st.nextToken());
}
The output of the above code would be: 

          
           
Hello
world
java
Similar to the 

          
           
split()
method, the 

          
           
StringTokenizer
class allows you to provide a regular expression as the delimiter which gives it greater flexibility.

Using Regex

The last way to split a string based on dot (.) character is to use regular expressions. The syntax is a bit more verbose but it provides you with complete control over the tokenization process. Here is an example of how to use regular expressions for splitting a string based on dot (.): 

          
           
String str = "Hello.world.java";
Pattern p = Pattern.compile("\\.");
String[] result = p.split(str);
System.out.println(Arrays.toString(result));
The output of the above code would be: 

          
           
[Hello, world, java]
Regular expressions are especially useful when the tokenization process needs to be more complex than just splitting on a single character.

Conclusion

In this article, we explored different ways to split strings by dot (.) character in Java. We saw how the 

          
           
String.split()
, 

          
           
StringTokenizer
and regular expressions can be used for this purpose. Each of these methods has its own advantages which makes them suitable for different use cases.
