How to Convert String to Title Case in Java

Title case is a style of writing that capitalizes all first letters of words, and lowercases all other letters. Converting a string to title case is an essential skill for any programmer, and it is quite easy to do in Java. In this article, we will explain how to convert a String to title case in Java.

Using the String.toUpperCase() and String.substring() Methods

In order to convert a string to title case in Java, we can use the 


          
           
String.toUpperCase()
and 
String.substring()
methods. Before doing so, we will have to split the string into an array of words.

Let's consider the following example: We have a string "the quick brown fox", and we want to convert it to title case. The code would look like this:


          
           
String str = "the quick brown fox";

String[] words = str.split("\\s+");

for (int i = 0; i < words.length; i++) {
    String firstLetter = words[i].substring(0,1);
    String restLetters = words[i].substring(1);
    words[i] = firstLetter.toUpperCase() + restLetters.toLowerCase();
}

str = String.join(" ", words);

System.out.println(str);
// Outputs: The Quick Brown Fox
Here, we use the 
String.split()
method to split the string into an array of words, then we loop through each word and use the 
String.substring()
method to get the first letter and the rest of the word. We then use the 

          
           
String.toUpperCase()
and 

          
           
String.toLowerCase()
methods to capitalize the first letter and lowercase the rest of the letters, respectively.

Finally, we join the words back together using the 

String.join()
method and print out the title-cased string.

Using Apache Commons Library

Alternatively, if you are using the Apache Commons Library, you can use the 


          
           
WordUtils.capitalizeFully()
method, as shown below: 

          
           
String str = "the quick brown fox";

String titleCasedStr = WordUtils.capitalizeFully(str);

System.out.println(titleCasedStr);
// Outputs: The Quick Brown Fox

As you can see, this approach is much simpler and shorter, and does not require any looping or string manipulation.

Conclusion

In this article, we have explained two ways to convert a string to title case in Java. We have used the 


          
           
String.toUpperCase()
, 
String.substring()
, and 
String.join()
methods, as well as the 

          
           
WordUtils.capitalizeFully()
method from the Apache Commons Library.
