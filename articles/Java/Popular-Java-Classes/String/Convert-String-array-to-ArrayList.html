Converting String Array to ArrayList

Java provides many ways to convert a String array to an ArrayList. The most commonly used methods are the following: 

          
           
ArrayList<String> list = new ArrayList<String>(Arrays.asList(arr));
or 

          
           
List<String> list = Arrays.asList(arr);
The following code example shows how to convert the contents of a String array to an ArrayList. 

          
           
public class Test {
	public static void main(String[] args) {
		String[] arr = {"a", "b", "c"};
		List<String> list = Arrays.asList(arr);
		System.out.println(list);  
	}
}
Output: 

          
           
[a, b, c]

Using for loop

You can also use a simple for loop to convert a String array to an ArrayList. The following code example shows how to use a for loop to convert the contents of the String array to an ArrayList. 

          
           
public class Test {
	public static void main(String[] args) {
		String[] arr = {"a", "b", "c"};
		List<String> list = new ArrayList<String>();
		for (String str : arr) {
			list.add(str);
		}
		System.out.println(list);
	}
}
Output: 

          
           
[a, b, c]

Using Collections.addAll() Method

You can also use the Collections.addAll() method to convert a String array to an ArrayList. The following code example shows how to use the addAll() method to convert the contents of the String array to an ArrayList. 

          
           
public class Test {
	public static void main(String[] args) {
		String[] arr = {"a", "b", "c"};
		List<String> list = new ArrayList<String>();
		Collections.addAll(list, arr);
		System.out.println(list);
	}
}
Output: 

          
           
[a, b, c]
