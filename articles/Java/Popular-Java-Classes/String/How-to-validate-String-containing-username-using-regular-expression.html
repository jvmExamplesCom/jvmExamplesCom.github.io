How to Validate Strings Containing Username Using Regular Expression

Regular expression is a powerful tool for validating strings. It can be used to check whether a given string contains certain characters or words and can be used to filter out undesired inputs. In this article, we will learn how to use regular expressions to validate strings containing usernames.

Overview

A username is usually a combination of alphabets, numbers, and symbols that one chooses to identify themselves on a website or application. It usually has restrictions like the length of the username, the characters used in the username, etc. It is important to validate usernames so that only valid usernames are accepted. This can be done using regular expression.

Implementation

Let's say you want to validate the following criteria for a username: 1. Must contain only alphabets and numbers 2. Must start with an alphabet 3. Can have a minimum length of 4 and maximum length of 12 characters We will use the following regular expression for this purpose: 

          
           
/^[a-zA-Z][\w]{3,11}$/
In the above regular expression, * '^' - indicates the start of the string * '[a-zA-Z]' - indicates that the first character must be an alphabet * '[\w]' - indicates that the next three to eleven characters must either be an alphabet or a digit * '$' - indicates the end of the string We can use the regex in the following way: 

          
           
if (userName.matches("/^[a-zA-Z][\w]{3,11}$/")) {
  System.out.println("Valid Username!");
}
else {
  System.out.println("Invalid Username!");
}
The above code snippet checks if the entered username matches the given regular expression. If it does, the message "Valid Username!" is displayed else "Invalid Username!".

Conclusion

In this article, we learned how to use regular expression for validating strings containing usernames. Regular expressions can be used for many other purposes as well such as validating email addresses, URLs, phone numbers, and ip addresses.
