Keep Only Numbers in String - Java

When working with strings in Java, sometimes you need to keep only numbers. For example, if you have a string like “ABC123” and you want to remove the letters and just keep the numbers, you’ll need to use a few methods to manipulate the string. In this article, we’ll go over how to keep only numbers in a String in Java.

Using Regular Expressions

The easiest and most straightforward way to keep only numbers in a String is to use regular expressions. With regular expressions, you can specify a pattern that you want to match and then remove everything else. For example, let’s say you have a String that contains the following: “ABC123456”. To keep only the numbers, you can use the following code: 

          
           
String input = "ABC123456";
String output = input.replaceAll("[^0-9]", "");
System.out.println(output); // 123456
Here, we’re using the replaceAll() method to replace all characters that are not 0-9 with an empty string (“”). This will effectively remove all the letters and keep only the numbers.

Using Substrings

Another way to keep only numbers in a String is by using substrings. To do this, you’ll need to loop through the string until you find the first and last numbers in the string. Once you have the indices of the first and last numbers, you can then use the substring() method to extract just the numbers. For example, let’s say you have the same string as above: “ABC123456”. To keep only the numbers, you can use the following code: 

          
           
String input = "ABC123456";

int firstNumberIndex = input.indexOf("1");
int lastNumberIndex = input.lastIndexOf("6");

if (firstNumberIndex >= 0 &amp;&amp; lastNumberIndex >= 0) {
    String output = input.substring(firstNumberIndex, lastNumberIndex + 1);
    System.out.println(output); // 123456
}
Here, we first use the indexOf() and lastIndexOf() methods to find the indices of the first and last numbers in the string. We then use the substring() method to extract the substring from the first number to the last number.

Using Char Arrays

Finally, you can use a char array to keep only numbers in a String. With char arrays, you can loop through each character of the string and check if it’s a number or not. If it’s a number, then you can add it to a new string. If it’s not a number, then you can simply skip it. Let’s say you have the same string as before: “ABC123456”. To keep only the numbers, you can use the following code: 

          
           
String input = "ABC123456";
String output = "";

char[] chars = input.toCharArray();

for (char c : chars) {
    if (Character.isDigit(c)) {
        output += c;
    }
}

System.out.println(output); // 123456
Here, we first convert the String to a char array using the toCharArray() method. We then use a for-each loop to loop through each character of the array. Finally, we check if the current character is a digit or not using the isDigit() method. If it’s a digit, then we add it to the output string.

Conclusion

In this article, we went over a few different ways to keep only numbers in a String in Java. Specifically, we discussed using regular expressions, substrings, and char arrays. Each approach has its own pros and cons, so pick the method that works best for your situation.
