Converting a String Array to a String

Java provides multiple ways to convert a string array to a single string. The most efficient way depends on the specific requirements of the application being developed. In this article we will take a look at a few of Java's options for converting an array of strings into a single string.

Using the StringBuilder Class

The most efficient method for converting a string array to a single string is using the StringBuilder class. This class allows developers to efficiently append strings together, and is well suited for the task at hand. 

          
           
String[] strings = {"This", "is", "a", "string", "array"};
StringBuilder builder = new StringBuilder();
for (String s : strings) {
    builder.append(s);
}
String result = builder.toString();
System.out.println(result); // Thisisastringarray
In the example above, we use a 

          
           
for
loop to iterate over the 

          
           
strings
array, and append each string to the 

          
           
builder
. Once all the strings have been added, we can call the 

          
           
toString()
method which will return the combined string.

Using the Joiner Class

Another convenient way to join strings together is using the 

          
           
Joiner
class from the Guava library. This class allows us to specify a delimiter which will be used between each string. 

          
           
String[] strings = {"This", "is", "a", "string", "array"};
String result = Joiner.on(" ").join(strings);
System.out.println(result); // This is a string array
The 

          
           
Joiner
class makes it easy to specify a delimiter between each string when joining them together. We simply pass in the delimiter as the first parameter, and the array of strings as the second. This will return the joined string.

Using the Stream API

The Stream API in Java 8 also provides a way to join strings together. This approach is slightly less efficient than using the StringBuilder, as we must iterate over the array twice. However, if performance is not an issue then this approach is useful. 

          
           
String[] strings = {"This", "is", "a", "string", "array"};
String result = Arrays.stream(strings)
                       .collect(Collectors.joining(" "));
System.out.println(result); // This is a string array
In the example above, we are using the 

          
           
Arrays.stream()
method to convert the 

          
           
strings
array into a stream. We then call the 

          
           
collect()
method, and specify the delimiter we want to use to join the strings. This will return the joined string.

Conclusion

In this article, we discussed three different ways to convert an array of strings into a single string. Each approach has its own merits and drawbacks, depending on the specific requirements of the application. Hopefully this article provided some insight into the various approaches to this problem.
