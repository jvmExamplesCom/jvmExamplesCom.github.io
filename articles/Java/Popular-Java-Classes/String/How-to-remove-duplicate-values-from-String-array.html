How to Remove Duplicate Values From a String Array in Java

Duplicate values in a string array can lead to unexpected behavior in a program. To prevent undesirable results, it is best to remove duplicate values from the array. In this article, we'll show you how to do this using Java.

Using Sets

The easiest way to remove duplicates from a string array is to use a set. A set is a type of data structure that does not allow for any duplicate elements. We can convert a string array into a set, and then convert it back into an array.


          
           
String[] myArray = { "Apple", "Banana", "Apple"}; 
Set<String> mySet = new HashSet<>(Arrays.asList(myArray)); // Create a set from the array 
String[] myNewArray = mySet.toArray(new String[0]); // Convert the set back into an array

The array now only contains unique strings. Note that the order of elements may be different to the original array.

Using Loops

We can also use loops to remove duplicate values. Using a nested loop, we can compare each element of the array with every other element. If the elements are equal, we can remove the duplicate element.


          
           
for (int i = 0; i < myArray.length; i++) {
  for (int j = i + 1; j < myArray.length; j++) {
    if (myArray[i].equals(myArray[j])) {
      myArray[j] = "";
    }
  }
}

Finally, we can create a new array and copy only the non-empty elements from the original array. This will give us a new array with no duplicate values:


          
           
String[] myFinalArray = Arrays.stream(myArray)
    .filter(s -> !s.isEmpty())
    .toArray(String[]::new);

Conclusion

In this article, we learned how to remove duplicate values from a string array in Java. We looked at two different approaches, both of which produce the same result. Which method you choose to use will depend on your preferences and needs.
