How to Sort a String Array in Java

Sorting a string array alphabetically is not a difficult task when using the Java programming language. It is possible to sort an array of strings of any length, as long as each element can be accessed and compared. In this article, we'll discuss how to sort an array of strings alphabetically in Java.

Using the Arrays Class

The Arrays class provides several methods which can be used to sort arrays of any type. The simplest way to sort an array of strings is to use the Arrays.sort() method. This method takes an array of strings and sorts them in ascending lexicographical order: 

          
           
String[] names = {"John", "Alex", "Ryan", "Jane"};
Arrays.sort(names);
// Output: Alex, Jane, John, Ryan

Using the Comparator Interface

If you need to sort an array of strings in a different order than ascending lexicographical order, you can use the Comparator interface. A Comparator can be used to sort an array in any order that you specify. For example, you could use a Comparator to sort an array of strings in descending lexicographical order: 

          
           
String[] names = {"John", "Alex", "Ryan", "Jane"};

Comparator<string> comparator = new Comparator<string>(){
    public int compare(String s1, String s2){
        return s2.compareTo(s1);
    }
};

Arrays.sort(names, comparator);
// Output: Ryan, John, Jane, Alex

Using the Collections Class

The Collections class also provides a method for sorting an array of strings. The Collections.sort() method has a more efficient algorithm than the Arrays.sort() method, and it is recommended for sorting large arrays of strings. 

          
           
String[] names = {"John", "Alex", "Ryan", "Jane"};
Collections.sort(Arrays.asList(names));
// Output: Alex, Jane, John, Ryan

Conclusion
