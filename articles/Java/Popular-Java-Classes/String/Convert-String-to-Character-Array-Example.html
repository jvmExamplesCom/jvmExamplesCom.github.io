Convert String to Character Array Example

Java provides an easy way of converting a String to a character array. This can be achieved using the toCharArray() method of String class. It is a method of String class which returns a character array which contains all the characters from the String object.

Syntax:


          
           
char[] chArr = str.toCharArray();

Example:


          
           
String str = "Programming";
char[] chArr = str.toCharArray();

for(int i = 0; i < chArr.length; i++) {
  System.out.println(chArr[i]);
}

g

Explaining the code:

First, a String value “Programming” is stored in a variable “str”. Then, this String is converted into a character array by the String class' toCharArray() method and is stored in the variable “chArr”. In the following for loop, the length of the character array is determined. Using that, the loop executes till each element of the character array is printed on the screen. Finally, the output is displayed as the individual characters of the String one after other on separate lines.
