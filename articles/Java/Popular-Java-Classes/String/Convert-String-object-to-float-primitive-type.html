Converting String Object to Float Primitive Type in Java

Converting a 

          
           
String
object to a 

          
           
float
primitive type in Java is a helpful process when you need to work with numerical values represented as strings. It ensures the accuracy of your program by understanding and representing the data you are working with correctly. Although there are several ways to accomplish this task, below we provide some of the most common methods for reference and implementation.

Method 1: Using Float.parseFloat() Method

The 

          
           
Float.parseFloat()
method converts a given 

          
           
String
into a 

          
           
float
value. To use this method, we need to pass the string to the 

          
           
parseFloat()
method as an argument. Let’s look at an example: 

          
           
String s= "43.56";
float f = Float.parseFloat(s);
System.out.println(f); // Outputs 43.56
In the example above, we declare a 

          
           
String
named 

          
           
s
with the value of “43.56”. We then use the 

          
           
Float.parseFloat()
method to convert the 

          
           
String
to a 

          
           
float
primitive type, which is assigned to the variable 

          
           
f
. The final line of code prints out the value of the 

          
           
float
, which is “43.56”.

Method 2: Using Float.valueOf() Method

The 

          
           
Float.valueOf()
method is another way to convert a 

          
           
String
to a 

          
           
float
. This method is slightly different from the 

          
           
Float.parseFloat()
method in that the former returns an object of type 

          
           
Float
while the latter returns a 

          
           
float
primitive type. The syntax for using this method is similar to that of the 

          
           
Float.parseFloat()
method: 

          
           
String s= "67.83";
Float f = Float.valueOf(s);
System.out.println(f); // Outputs 67.83
In this example, we declare a 

          
           
String
named 

          
           
s
with the value of “67.83”. We then use the 

          
           
Float.valueOf()
method to convert the 

          
           
String
to a 

          
           
Float
object, which is assigned to the variable 

          
           
f
. The final line of code prints out the value of the 

          
           
Float
, which is “67.83”.

Method 3: Using DecimalFormat Class

The 

          
           
DecimalFormat
class, a part of the 

          
           
java.text
package, is a method to convert a 

          
           
String
to a 

          
           
float
. When using this method, you must create a 

          
           
DecimalFormat
instance and set the pattern for it before you can call the 

          
           
format()
method. Here's an example that demonstrates the usage of this class: 

          
           
String s= "128.54";
DecimalFormat df = new DecimalFormat("0.00");
float f = df.format(s);
System.out.println(f); // Outputs 128.54
In this example, we again declare a 

          
           
String
named 

          
           
s
with the value of “128.54”. We then create an instance of the 

          
           
DecimalFormat
class and set the pattern to “0.00”. We then call the 

          
           
format()
method and pass our 

          
           
String
as an argument. The 

          
           
format()
method returns a 

          
           
float
, which is assigned to the 

          
           
float
variable 

          
           
f
. The final line of code prints out the value of the 

          
           
float
, which is “128.54”.

Conclusion

String

float

Float.parseFloat()
