How to Check if a String is a Palindrome in Java

A palindrome is a word, phrase, number, or other sequence of units that can be read the same way in either direction. In this tutorial, we will discuss how to check if a given string is a palindrome in Java. We'll look at two methods for accomplishing this task, using a for loop and using recursion.

Using a For Loop

The first method we'll look at uses a for loop to iterate over the characters in the given string. For each iteration, we compare the current character with its mirror image in the opposite direction. If any of these comparisons fail, we conclude that the string is not a palindrome and return false. If all the comparisons succeed, we conclude that the string is a palindrome and return true. 

          
           
public static boolean isPalindrome(String str) {
    int n = str.length();
    for (int i = 0; i < n/2; i++) {
        if (str.charAt(i) != str.charAt(n-i-1)) {
            return false;
        }
    }
    return true;
}

Using Recursion

The second method we'll look at uses recursive approach for comparing characters of a string. At each step of recursion, we compare the current character with its mirror image in the opposite direction. If any of these comparisons fail, we return false. If all the comparisons succeed, we return true. 

          
           
public static boolean isPalindrome(String str, int i, int j) {
    // base case
    if (i == j) {
        return true;
    }

    // If first and last characters do not match
    if (str.charAt(i) != str.charAt(j)) {
        return false;
    }

    // If there are more than two characters, check 
    // if middle substring is also palindrome or not.
    if (i < j + 1) {
        return isPalindrome(str, i+1, j-1);
    }

    return true;
}
We also need a wrapper function to call the recursive function with the appropriate arguments. 

          
           
public static boolean isPalindrome(String str) {
    int n = str.length();
    return isPalindrome(str, 0, n-1);
}
