Converting a String to an ArrayList

Java provides us with a few ways to convert a string into an ArrayList. This is useful when working with large strings or when we need to work with each character individually. In this article, we'll explore some of the most common techniques for doing this conversion.

Using split() Method

The split() method of the String class splits a string into a sequence of substrings and returns an array of all the substrings. We can specify the delimiter using this method. The following code demonstrates how to use the split() method to convert a string to an ArrayList. 

          
           
String str = "Hello World";
String[] strArr = str.split(" ");

List<String> arrList = new ArrayList<>();
Collections.addAll(arrList, strArr);

System.out.println(arrList); // Output: [Hello, World]

Using Regular Expressions

Regular expressions provide us with a powerful way to match patterns in strings. We can use regular expressions to parse a string and return a list of all the characters it contains. The following code demonstrates how to do this: 

          
           
String str = "Hello World";
String regex = "[A-Za-z ]+";

Pattern pattern = Pattern.compile(regex); 
Matcher matcher = pattern.matcher(str);

List<String> arrList = new ArrayList<>();
while (matcher.find()) {
    arrList.add(matcher.group(0));
}

System.out.println(arrList); // Output: [Hello, World]

Using forEach() Method

The forEach() method of the Java 8 Stream API allows us to iterate over the characters of a string and add them to an ArrayList. The following code demonstrates how to use the forEach() method to convert a string to an ArrayList: 

          
           
String str = "Hello World";
List<String> arrList = new ArrayList<>(); 
str.chars().forEach((i) -> arrList.add(Character.toString((char) i))); 

System.out.println(arrList); // Output: [H, e, l, l, o, , W, o, r, l, d]

Using Stream API

We can also use the Stream API to convert a string to an ArrayList. The following code demonstrates how to do this: 

          
           
String str = "Hello World"; 
List<String> arrList = str.chars()
                              .mapToObj(c -> Character.toString((char) c))
                              .collect(Collectors.toList());

System.out.println(arrList); // Output: [H, e, l, l, o, , W, o, r, l, d]
