Java String to String Array Example

In this tutorial we will take a look at how to convert a Java String to a String Array. We will explore various strategies and methods for the conversion process, as well as look at some examples of how to use the converted String Array.

Overview of Converting a String to an Array

Converting a String to an array in Java can be done many different ways. The common approaches generally involve using String methods such as split(), which takes a delimiter as an argument, or using a combination of loops and substring() to pull out each element. The most important thing to consider when converting a String to an array is what delimiter should be used to separate the data inside the String. Common delimiters that are used include commas, whitespace characters, and tabs. Once the delimiter is determined, it can be used to split the String into an array of separate Strings. Depending on the situation, these separate Strings may need to be further parsed into individual values.

Examples of Converting a String to an Array

We will now look at some examples of how to convert a String to an array in Java.

Example 1: Comma Separated Values

In this example, we have a String of comma separated values, and we want to convert it to an array. To do this, we can use the 

          
           
String.split(",")
method, which takes a delimiter as its argument. This will split the String into individual substrings at each comma, and return an array of those separate Strings. 

          
           
String str = "1,2,3,4,5,6";
String[] arr = str.split(",");
The above code will result in 

          
           
arr
being an array with the following contents: 

          
           
["1", "2", "3", "4", "5", "6"]

Example 2: Tab Separated Values

This example is similar to the first one, but this time the String contains tab separated values. To split the String on each tab, we again use 

          
           
String.split("\t")
, where "\t" is the escape character for the tab. 

          
           
String str = "10\t20\t30\t40\t50\t60";
String[] arr = str.split("\t");
The above code will result in 

          
           
arr
being an array with the following contents: 

          
           
["10", "20", "30", "40", "50", "60"]

Example 3: Whitespace Separated Values

This example is similar to the two previous ones, but this time the String contains whitespace separated values. To split the String on each whitespace, we again use 

          
           
String.split("\\s+")
, where "\s+" is the regular expression for matching one or more whitespace characters. 

          
           
String str = "Adam Bob Carl Dave";
String[] arr = str.split("\\s+");
The above code will result in 

          
           
arr
being an array with the following contents: 

          
           
["Adam", "Bob", "Carl", "Dave"]

Example 4: Looping and Substring

In this example, we will look at how to use a loop and substring() method to loop through a String and extract each character as an element in the resulting array. 

          
           
String str = "A1B2C3";
int arrLength = str.length();
String[] arr = new String[arrLength];

for(int i=0; i<arrLength; i++) {
    arr[i] = str.substring(i, i+1);
}
The above code will result in 

          
           
arr
being an array with the following contents: 

          
           
["A", "1", "B", "2", "C", "3"]

Conclusion

In this tutorial, we looked at how to convert a String to an array in Java. We explored several different methods, such as using the split() method, or using a combination of looping and substring(). We also looked at some examples of how to use the converted String array.
