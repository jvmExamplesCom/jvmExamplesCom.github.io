Convert primitive long value to String object Using Java

A String object in Java is an immutable sequence of characters. This means that any changes made to the String object result in a new String object being created. It is therefore essential to understand how to convert a primitive long value to a String object.

Using Long.toString() Method

The simplest and most straightforward way to convert a primitive long value to a String object is to use the Long.toString() method. This method takes a single argument, which is the long value to be converted. It then returns a String object representing the same value.


          
           
long longValue = 123456789;
String stringValue = Long.toString(longValue);
System.out.println("long value: " + longValue + " | string value: " + stringValue);
The output of the code snippet above will be: 

          
           
long value: 123456789 | string value: 123456789

Using String.valueOf() Method

The String.valueOf() method can be used to convert any type of primitive value to a String object. The method takes one argument, which can be any type of primitive value (byte, short, int, long, float, double, char or boolean). The method will return a String object representing the same value passed as argument.


          
           
long longValue = 123456789;
String stringValue = String.valueOf(longValue);
System.out.println("long value: " + longValue + " | string value: " + stringValue);
The output of the code snippet above will be: 

          
           
long value: 123456789 | string value: 123456789

Using Long.toString() and StringBuilder()

It is also possible to convert a primitive long value to a String object using the Long.toString() and StringBuilder() methods. The Long.toString() method is used to convert the long value to a string and the StringBuilder() method is used to create a new String object representing the same value.


          
           
long longValue = 123456789;
StringBuilder sb = new StringBuilder();
sb.append(Long.toString(longValue));
String stringValue = sb.toString();
System.out.println("long value: " + longValue + " | string value: " + stringValue);
The output of the code snippet above will be: 

          
           
long value: 123456789 | string value: 123456789

Conclusion

This article has demonstrated three ways of converting a primitive long value to a String object in Java. All three methods are relatively straightforward, with the first two requiring only a single statement and the third requiring just two statements. Any of these methods can be used to convert a long value to a String object in a Java application.
