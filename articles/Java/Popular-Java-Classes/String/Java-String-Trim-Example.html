Java String Trim Example

The trim() method in Java is used to remove the leading and trailing spaces from a string. This tutorial will take you through an example of how to use the trim() method in your code.

Using trim()

The syntax for using the trim() method is simple and straightforward. It is called on the string object, and it doesn't require any parameters: 

          
           
String s = "   Hello World!   ";
String trimmedString = s.trim();
System.out.println(trimmedString);
The code above prints out the trimmed string: "Hello World!".

Example with whitespace character

In this example, we will use the trim() method to remove the whitespace character from a string. We will use the Unicode representation of a whitespace character (\u0020) in the string: 

          
           
String str = "\u0020Hello World!\u0020";
String trimmedStr = str.trim();
System.out.println(trimmedStr);
Running the code above prints out the same result as before: "Hello World!".

Comparing trim() and replaceAll()

The trim() method is often confused with the replaceAll() method. Both methods do remove the whitespace characters from a string. However, the trim() method just removes the leading and trailing whitespace characters, whereas the replaceAll() method removes all instances of whitespace characters from the string. Here's an example that compares the two methods: 

          
           
String str = "\u0020Hello World!\u0020";
String trimmedStr = str.trim();
String replacedStr = str.replaceAll("\\s+", "");
System.out.println(trimmedStr);
System.out.println(replacedStr);
The code above prints out the following strings: 

          
           
Hello World! 
HelloWorld!
As you can see, the trim() method only removed the leading and trailing whitespace characters, while the replaceAll() method removed all instances of whitespace characters.

Conclusion

In this tutorial, we discussed how to use the trim() method in Java. We looked at an example of how to use the trim() method and compared it to the replaceAll() method. With this information, you should have a good understanding of how to use the trim() method in your own code.
