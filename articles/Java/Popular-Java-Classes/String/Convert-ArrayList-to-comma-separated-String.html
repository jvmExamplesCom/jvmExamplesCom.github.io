Converting an ArrayList to a Comma Separated String

The Java programming language offers a number of ways to convert an array or an ArrayList into a comma-separated list of strings. In this article, we will examine some of the most common techniques for achieving this result.

Using StringJoiner

Java 8 introduced the StringJoiner class that allows us to join elements in a collection into a single string, with a separator in between each element. We can use it to convert our ArrayList into a comma-separated list, as shown in the following example: 

          
           
ArrayList<string> myList = new ArrayList<>();
myList.add("item1");
myList.add("item2");
myList.add("item3");

StringJoiner joiner = new StringJoiner(",");
for (String item : myList) {
    joiner.add(item);
}
String listAsString = joiner.toString();
System.out.println(listAsString); // prints: item1,item2,item3
</string>

Using String#join

Java 8 also introduced the String#join method, which provides a simpler way of joining elements in a collection into a string, using a specified delimiter. This method can be used to easily convert an ArrayList into a comma-separated string, as shown in the example below: 

          
           
ArrayList<string> myList = new ArrayList<>();
myList.add("item1");
myList.add("item2");
myList.add("item3");

String listAsString = String.join(",", myList);
System.out.println(listAsString); // prints: item1,item2,item3
</string>

Using StringBuilder

We can also use the StringBuilder class to concatenate all the elements of an ArrayList into a single string. The following example shows how to use this approach to convert an ArrayList into a comma-separated list: 

          
           
ArrayList<string> myList = new ArrayList<>();
myList.add("item1");
myList.add("item2");
myList.add("item3");

StringBuilder builder = new StringBuilder();
for (String item : myList) {
    builder.append(item).append(",");
}
String listAsString = builder.deleteCharAt(builder.length() - 1).toString();
System.out.println(listAsString); // prints: item1,item2,item3
</string>

Using Streams

Finally, we can use the Stream API to collect our elements from the ArrayList into a single string. This approach allows us to make use of the Collectors#joining collector, which takes care of all the work of joining the elements into a single string. Here is an example of how we can use this approach: 

          
           
ArrayList<string> myList = new ArrayList<>();
myList.add("item1");
myList.add("item2");
myList.add("item3");

String listAsString = myList.stream()
    .collect(Collectors.joining(","));
System.out.println(listAsString); // prints: item1,item2,item3
</string>
In this article, we have examined several different techniques for converting an ArrayList into a comma-separated string in Java. Each technique has its own merits, so it is important to consider the specific requirements of your application before making a choice.
