How to Keep Only Letters in String

In Java, there are various ways you can go about keeping only letters in strings. This tutorial will cover the most common methods, including using regular expressions (Regex), ASCII ranges, and a few other techniques.

Using Regex

Using regular expressions (Regex) is considered a more efficient approach when compared to other methods. With Regex, you can match any string that contains only letters. The following example of Regex code will match any string containing only letter characters:


          
           
String str = "123abc456";
String pattern = "[a-zA-Z]+";

if (str.matches(pattern)) {
   System.out.println("The string only contains letters.");
} else {
   System.out.println("The string contains non-letter characters.");
}
The following snippet of code uses a 
replaceAll()
statement to remove all characters except letters. This code will then assign the remaining letters to a new 
String
variable named 
result
: 

          
           
String str = "123abc456";
String result = str.replaceAll("[^a-zA-Z]", "");

System.out.println(result); //Outputs "abc"

Using ASCII Ranges

String str = "123abc456";
StringBuilder sb = new StringBuilder();

for(int i = 0; i < str.length(); i++) {

   }

String result = sb.toString();
