How to Remove the Last Character from a String in Java

String manipulation is a common task that developers perform and Java makes it easy to do. In this article we will look at how to remove the last character from a String in Java. Removing the last character from a String in Java allows you to easily manipulate your String data without the need for additional libraries or tools. This can be done using the String class functions or by using regular expressions.

Using String.substring() Method

The 

          
           
String.substring()
method is the easiest way to remove the last character from a String in Java. The 

          
           
substring()
method takes two parameters, the start index and the end index, and returns a new String with the characters between the specified indices. To remove the last character from a String, we simply need to specify the start index as 0 and the end index as one less than the length of the String. 

          
           
String str = "Example";
String lastCharRemoved = str.substring(0, str.length() - 1);
// Output: Examp

Using Regular Expressions

Regular expressions are powerful tools for manipulating text and can also be used to remove the last character from a String in Java. To do this, we use the 

          
           
replaceAll()
method of the String class with a regular expression. The regular expression specifies the pattern of the character we wish to remove and the resulting String is returned with the specified character removed. 

          
           
String str = "Example";
String lastCharRemoved = str.replaceAll(".$", "");
// Output: Examp

Using StringBuilder

The 

          
           
StringBuilder
class provides an efficient way to remove the last character from a String in Java. It has a 

          
           
deleteCharAt()
function which takes the index of the character to be deleted from the String as its parameter. To remove the last character from a String, we pass the length of the String minus one as the argument to the 

          
           
deleteCharAt()
method. 

          
           
String str = "Example";
StringBuilder sb = new StringBuilder(str);
sb.deleteCharAt(str.length() - 1);
String lastCharRemoved = sb.toString();
// Output: Examp

Conclusion

In this article we looked at how to remove the last character from a String in Java. We saw that there are multiple ways of accomplishing this task, including using the 

          
           
substring()
, 

          
           
replaceAll()
, and 

          
           
StringBuilder.deleteCharAt()
methods. Each of these methods offers different advantages, so it is important to choose the right one for the task at hand.
