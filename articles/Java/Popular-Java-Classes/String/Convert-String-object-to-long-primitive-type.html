Converting a String Object to a Long Primitive Type in Java

Java provides a number of primitive types that can be used to store data in certain formats. The 

long
type is used for storing integer values that are too long for the 
int
primitive type. Converting a 
String
object to a 
long
primitive type is relatively simple in Java.

Using the parseLong() Method

The most common way to convert a 
String
object to a 
long
primitive type is to use the 
parseLong()
method. This method takes an input 
String
object and attempts to parse it as a long value. If successful, the 
parseLong()
will return the value of the 
long
primitive. Here is an example: 

          
           
String str = "12345"; 
long result = Long.parseLong(str); 
System.out.println("Result: " + result);
This code will output the following: 

          
           
Result: 12345

It is also possible to provide an additional argument to the 

parseLong()
method specifying the radix (or base) of the input 
String
object. This argument must be an integer value between 2 and 36. If a radix is not provided, the default value of 10 will be used.

Using the ValueOf() Method

Another option for converting a 
String
object to a 
long
primitive type is to use the 
valueOf()
method. The 
valueOf()
method works similarly to 
parseLong()
but it returns a 
Long
object instead of a 
long
value. To get the actual 
long
value, you can simply use the 
longValue()
method on the 
Long
object. Here is an example: 

          
           
String str = "12345"; 
Long result = Long.valueOf(str); 
System.out.println("Result: " + result.longValue());
This code will output the following: 

          
           
Result: 12345
As with the 
parseLong()
method, it is also possible to provide an additional argument to the 
valueOf()
method specifying the radix (or base) of the input 
String
object.
