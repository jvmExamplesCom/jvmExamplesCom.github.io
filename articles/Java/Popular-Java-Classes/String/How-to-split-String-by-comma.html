How To Split A String By Comma

Split a string by comma to create an array is one of the most common tasks in Java programming. This article will show you several ways to accomplish this and discuss the pros and cons of each.

Using the split() Method

The easiest way to split a string by comma is to use the split() method. This method takes in a single argument, the delimiter character, which in this case is a comma. The method returns an array containing each of the elements of the original string, separated by the delimiter. 

          
           
String str = "A,B,C";
String[] arr = str.split(",");
Output: 

          
           
[A, B, C]

Using a Regular Expression

Another way to split a string by comma is to use a regular expression. This approach requires a bit more knowledge of Java, but allows for more flexibility in what characters can be used as the delimiter. 

          
           
String str = "A,B,C";
Pattern pattern = Pattern.compile("\\s*,\\s*");
String[] arr = pattern.split(str);
Output: 

          
           
[A, B, C]

Using StringTokenizer

The StringTokenizer class is a legacy class that was released in Java 1.0, and is still maintained by Oracle. It provides a slightly more flexible approach than split(). It takes in two arguments - the string to be split and the delimiter character. 

          
           
String str = "A,B,C";
StringTokenizer st = new StringTokenizer(str, ",");
while (st.hasMoreTokens()) {
    System.out.println(st.nextToken());
}
Output: 

          
           
A
B
C

Using a for Loop

The final approach we will discuss is using a for loop. While this is not the most memory efficient approach, it allows for additional flexibility in terms of what characters can be used as the delimiter. 

          
           
String str = "A,B,C";
String[] arr = new String[str.length()];
int j = 0;
for (int i = 0; i < str.length(); i++) {
    if (str.charAt(i) == ',') {
        j++;
    } else {
        arr[j] += str.charAt(i);
    }
}
Output: 

          
           
[A, B, C]

Conclusion

Splitting a string by comma to create an array is a very common operation when working with strings in Java. This article has provided four different approaches to accomplish this task. Each approach has its own pros and cons, so which one you choose will depend on the specific needs of your project.
