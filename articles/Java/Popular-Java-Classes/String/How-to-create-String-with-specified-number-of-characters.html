How to Create String with Specified Number of Characters in Java

In Java, there are various ways to create a string with a specified number of characters. In this article, we will explore multiple methods using which you can create a string of certain length as per your requirement.

Using StringBuilder

StringBuilder class can be used to create a string of specified length. All the operations of this class such as append(), insert(), replace(), delete() etc. can be used to modify the string. To create a string of given length we start by initializing an instance of StringBuilder with an empty String and then use the StringBuilder’s append() method to add characters to the string. 

          
           
public static String createString(int length) { 
    StringBuilder sb = new StringBuilder(""); 
    for (int i=0;i<length;i++) { 
        sb.append("*"); 
    } 
    return sb.toString(); 
}
In the above code snippet, we created a string containing only asterisk (*) character of given length by using the append() method of the StringBuilder. This method also allows us to append other character(s) as well in the same way.

Using CharArray

Another approach to create a string of given length is to use the toCharArray() method of String class. This method takes a character array of given length and returns a string with the elements of that array. 

          
           
public static String createString(int length) { 
    char[] charArray = new char[length]; 
    for (int i=0;i<length;i++) { 
        charArray[i] = '*'; 
    } 
    return new String(charArray); 
}
In the above code snippet, we created a string consisting of only asterisk (*) character of given length by using the toCharArray() method of the String class. This method also allows us to specify the type of characters we want to append in the same way.

Using Repeat() Method

Java 11 introduced a new repeat() method of the String class which can be used to create a string consisting of a specified number of characters. This method takes an integer value as an argument and returns a string containing the same character repeated as many times as specified by the argument. 

          
           
public static String createString(int length) { 
    return "*".repeat(length); 
}
In the above code snippet, we created a string consisting of only asterisk (*) character of given length by using the repeat() method of the String class. This method also allows us to specify any character(s) which we want to repeat.

Using Char() Method

The Char() method of the Character class can also be used to create a string of given length. This method takes an integer value as parameter and returns a character for the corresponding code point. To create a string of given length, we need to call this method for each code point to get the corresponding character and use it to construct the string. 

          
           
public static String createString(int length) { 
    StringBuilder sb = new StringBuilder(""); 
    for (int i=0;i<length;i++) { 
        sb.append(Character.toChars(i)); 
    } 
    return sb.toString(); 
}
In the above code snippet, we created a string consisting of numbers from 0 to the length of the string - 1 by using the toChars() method of the Character class. This method also allows us to specify any character(s) which we want to include in the string. Thus, you can use any of the above methods to create a string with the specified number of characters as per your requirement.
