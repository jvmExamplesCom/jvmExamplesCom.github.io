Getting Substring from a String in Java

In Java, you can use the 
substring()
method of the 
String
class to get a substring from a string. Before we explain how to use the 
substring()
method, let’s explore various ways to create strings in Java.

Creating Strings in Java

There are a few ways you can create strings in Java:

  • Using the 
    String
    class
  • Using the 
    StringBuilder
    class
  • Using the 
    StringBuffer
    class
Once you’ve created your string, you can use the 
substring()
method to get a substring from it.

Using the String Class

The simplest way to create a string is by using the 

String
class. You can create a new string using the following syntax: 

          
           
String str = new String();

You can also pass a string literal as an argument to the 

String
constructor: 

          
           
String str = new String("My String");

Using the StringBuilder Class

Another way to create a string is by using the 
StringBuilder
class. You can use the following syntax to create a new 
StringBuilder
instance: 

          
           
StringBuilder sb = new StringBuilder();

You can also pass a string literal as an argument to the 

StringBuilder
constructor: 

          
           
StringBuilder sb = new StringBuilder("My String");

Using the StringBuffer Class

You can also create a string using the 
StringBuffer
class. You can use the following syntax to create a new 
StringBuffer
instance: 

          
           
StringBuffer sb = new StringBuffer();

You can also pass a string literal as an argument to the 

StringBuffer
constructor: 

          
           
StringBuffer sb = new StringBuffer("My String");

Getting Substring from a String

Now that you’ve created your string, you can use the 

substring()
method of the 
String
class to get a substring. The 
substring()
method takes two arguments: the start index and the end index. The start index is inclusive, while the end index is exclusive. Here’s an example: 

          
           
String str = "Hello World";
String subStr = str.substring(6, 11);
System.out.println(subStr);
The above code will print 
World
to the console. The start index is 6, which corresponds to the ‘W’ character, and the end index is 11, which is the index of the last character in the substring. The 
substring()
method will return a new 
String
object that contains the characters between the specified indexes.

You can also omit the second argument if you want to get a substring that extends to the end of the original string:


          
           
String str = "Hello World";
String subStr = str.substring(6);
System.out.println(subStr);
The above code will print 
World
to the console. The 
substring()
method will return a new 
String
object that contains the characters from the specified index to the end of the string.

Conclusion

In this article, we’ve explored various ways to create strings in Java and how to use the 

substring()
method to get a substring from a string. We hope you’ve found this tutorial helpful!
