Java String Length Example

This tutorial will provide an example of using the String 

          
           
length()
method in Java. This function returns an integer that represents the number of characters in a String object. Let's look at an example: 

          
           
String myString = "Hello, World!";
int length = myString.length(); //length is 12
In this simple example, we declare a String object and assign it the value 

          
           
"Hello, World!"
. We then use the 

          
           
length()
method to get the length of the String. The value returned is 

          
           
12
, which is the number of characters contained in the String. Let's look at another example: 

          
           
String myLongerString = "Some longer text with multiple words!";
int length = myLongerString.length(); //length is 27
For this example, we declared a new String object and assigned it 

          
           
"Some longer text with multiple words!"
. When we used the 

          
           
length()
method on this String, the value returned is 

          
           
27
, which is the number of characters contained in the String. The 

          
           
length()
method can be used to determine the size of a String before and after operations. For instance, if we wanted to take the first 5 characters from our example String with 

          
           
myLongerString
, we could do something like this: 

          
           
String firstFive = myLongerString.substring(0, 5);
int length = firstFive.length(); //length is 5
In this example, we are using the 

          
           
substring()
method to return the first 5 characters of our String. We then use the 

          
           
length()
method to get the length of this new smaller String. The result is 

          
           
5
, which is the correct number of characters contained in the new String. It is also possible to use the 

          
           
length()
method with other data types, such as Arrays and Lists. For example, let's say we have an Array of Strings: 

          
           
String[] arrayOfStrings = {"A", "B", "C"};
int length = arrayOfStrings.length; //length is 3
In this example, we declare an Array of Strings and assign it 3 values. We then use the 

          
           
length
property to get the size of the Array. The value returned is 

          
           
3
, which is the number of elements contained in the Array. Finally, let's look at how we can use the 

          
           
length()
method with a List: 

          
           
List<String> myList = new ArrayList<String>();
myList.add("A");
myList.add("B");
myList.add("C");
int length = myList.size(); //length is 3
In this example, we create a List of Strings and add 3 elements to it. We then use the 

          
           
size()

3
, which is the number of elements contained in the List. As you can see, the 

          
           
length()
method can be used in many different ways in Java. It can be used to determine the size of Strings, Arrays and Lists in your program.
