Java String to Date Example

The Java String to Date Example explains how to convert String object containing date to java.util.Date object. This example is very simple and straight-forward. It includes some examples with relevant code and output.

String to java.util.Date Conversion

The following example converts a String to java.util.Date by creating a SimpleDateFormat object, passing the date in String format and applying the parse() method on the SimpleDateFormat object. 

          
           
SimpleDateFormat sdf = new SimpleDateFormat("dd-M-yyyy hh:mm:ss");
String dateInString = "31-08-1982 10:20:56";
Date date = sdf.parse(dateInString);
System.out.println("Date : " + date);
Output: 

          
           
Date : Wed Aug 31 10:20:56 IST 1982

String to java.time.LocalDateTime Conversion

In addition to java.util.Date, you can also convert a String to java.time.LocalDateTime using DateTimeFormatter. Below is an example code. 

          
           
String date = "2017-12-03T10:15:30";
DateTimeFormatter formatter = DateTimeFormatter.ofPattern("yyyy-MM-dd'T'HH:mm:ss");
LocalDateTime ldt = LocalDateTime.parse(date, formatter);
System.out.println("LocalDateTime : " + ldt);
Output: 

          
           
LocalDateTime : 2017-12-03T10:15:30

String to java.time.LocalDate Conversion

Similarly, the following example shows how to convert a String to java.time.LocalDate. 

          
           
String dateInString="2017-11-17";
DateTimeFormatter formatter=DateTimeFormatter.ofPattern("yyyy-MM-dd");
LocalDate localDate=LocalDate.parse(dateInString, formatter);
System.out.println("LocalDate : "+localDate);
Output: 

          
           
LocalDate : 2017-11-17

String to java.time.LocalTime Conversion

Finally, you can also convert a String to java.time.LocalTime using DateTimeFormatter. 

          
           
String timeInString="11:22";
DateTimeFormatter formatter=DateTimeFormatter.ofPattern("HH:mm");
LocalTime localTime=LocalTime.parse(timeInString, formatter);
System.out.println("LocalTime : "+localTime);
Output: 

          
           
LocalTime : 11:22
