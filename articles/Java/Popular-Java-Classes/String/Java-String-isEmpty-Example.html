Java String isEmpty Example

Java provides two methods to check the empty string in java. The first one is isEmpty() method and the other one is length() method. In this tutorial, we will discuss both the methods in detail. The syntax for the isEmpty() method is as follows: 

          
           
boolean isEmpty()
The isEmpty() method returns true if the given string is empty otherwise it returns false. Let's take a look at a simple example to understand the concept more clearly. 

          
           
public class Main { 
  public static void main(String[] args) 
  { 
    String s = ""; 
    System.out.println("Testing isEmpty() on an empty String : "+s.isEmpty()); 
    String s2 = "GeeksforGeeks"; 
    System.out.println("Testing isEmpty() on a non-empty String : "+s2.isEmpty()); 
  } 
}
Output: 

          
           
Testing isEmpty() on an empty String : true
Testing isEmpty() on a non-empty String : false
In the above example, we have created two strings. The first one is an empty string and the second one is a non-empty string. We have used the isEmpty() method to check if the String is empty or not. The isEmpty() method returns true for an empty string and false for a non-empty string. The other way to check an empty string in Java is using the length() method. The syntax for the length() method is as follows: 

          
           
int length()
The length() method returns the length of the given string. If the given string is empty then it returns 0. Let's take a look at another example to understand the concept more clearly. 

          
           
public class Main { 
  public static void main(String[] args) 
  { 
    String s = ""; 
    System.out.println("Testing length() on an empty String : "+s.length()); 
    String s2 = "GeeksforGeeks"; 
    System.out.println("Testing length() on a non-empty String : "+s2.length()); 
  } 
}
Output: 

          
           
Testing length() on an empty String : 0
Testing length() on a non-empty String : 12
In the above example, we have created two strings and used the length() method to check if the given string is empty or not. The length() method returns 0 for an empty string and the length of the string for a non-empty string. Therefore, in this tutorial, we discussed two ways to check an empty string in Java. We can use either the isEmpty() method or the length() method depending on our requirement.
