Java: Convert First Character to Lowercase

Java is an object-oriented programming language commonly used by developers around the world. One of the common tasks a Java developer might need to do is convert the first character of a string to lowercase. This tutorial will walk through how to do this in Java.

String Class toUpperCase and toLowerCase Methods

The 

          
           
String
class in Java provides us with 

          
           
toUpperCase()
and 

          
           
toLowerCase()
methods which allow us to convert strings to either upper or lower case. We can use the 

          
           
toLowerCase()
method to easily convert the first character of a string to lowercase. In order to do this, we need to call the 

          
           
substring()
method on our string to get the first character of the string as a separate substring. We then call the 

          
           
toLowerCase()
method on this substring to convert it to lowercase. Finally, we combine the lowercase character with the rest of the string to get our desired result. Here is an example of how we can use the 

          
           
substring()
, 

          
           
toLowerCase()
, and 

          
           
concat()
methods together to convert the first character of a string to lowercase in Java: 

          
           
String myString = "MyString";
String lowerCaseCharacter = myString.substring(0,1).toLowerCase(); 
String convertedString = lowerCaseCharacter.concat(myString.substring(1));
System.out.println(convertedString); // Prints 'myString'
In this example, 

          
           
substring(0,1)
returns the first character of the string, which we then convert to lowercase using 

          
           
toLowerCase()
. We then combine the lowercase character with the rest of the string using the 

          
           
concat()
method, and print out the resulting string.

Character Class toLowerCase Method

We can also use the 

          
           
Character
class in Java to convert the first character of a string to lowercase. The 

          
           
Character
class provides us with a 

          
           
toLowerCase()
method that takes in a character and returns its lowercase equivalent. Here is an example of how we can use the 

          
           
charAt()
, 

          
           
toLowerCase()
, and 

          
           
concat()
methods together to convert the first character of a string to lowercase in Java: 

          
           
String myString = "MyString";
char firstChar = myString.charAt(0);
char lowerCaseChar = Character.toLowerCase(firstChar);
String convertedString = lowerCaseChar + myString.substring(1);
System.out.println(convertedString); // Prints 'myString'
In this example, we use the 

          
           
charAt()
method to get the first character of the string, which we then convert to lowercase using 

          
           
toLowerCase()
. We then combine the lowercase character with the rest of the string using the 

          
           
concat()
method, and print out the resulting string.

Conclusion

In this tutorial, we learned how to convert the first character of a string to lowercase in Java. We looked at two ways to do this, using the 

          
           
String
and 

          
           
Character
classes. We saw how to combine the 

          
           
substring()
, 

          
           
toLowerCase()
, and 

          
           
concat()
methods to convert a character to lowercase. We also saw how to use the 

          
           
charAt()
and 

          
           
toLowerCase()
methods from the 

          
           
Character
class to convert a character to lowercase.
