Java String Contains Example

In this article, we'll look at different ways to check if a Java String contains a specified word or substring. We'll examine the most common methods and discuss their advantages and disadvantages.

The contains() Method

The simplest way to check if a Java String contains a given character or substring is to use the 

          
           
contains()
method of the Java String class. This method takes a CharSequence as an argument, so we can easily pass any type of character or substring. 

          
           
String str = "Hello World!";
if (str.contains("World")) {
   System.out.println("String contains substring!");
}
In the above example, we check if the "Hello World!" String contains the word "World". If it does, we print out a message.

The matches() Method

Another way to check if a Java String contains a given character or substring is to use the 

          
           
matches()
method. This method takes a regular expression as an argument, which allows us to search for a pattern within a String. 

          
           
String str = "Hello World!";
if (str.matches(".*World.*")) {
   System.out.println("String contains substring!");
}
In the above example, we check if the "Hello World!" String matches the regular expression ".+World.+". If it does, we print out a message.

The indexOf() Method

Finally, we can check if a Java String contains a given character or substring by using the 

          
           
indexOf()
method. This method returns the index of the first occurrence of the specified substring, or 

          
           
-1
if the substring is not present. 

          
           
String str = "Hello World!";
if (str.indexOf("World") != -1) {
   System.out.println("String contains substring!");
}
In the above example, we check if the "Hello World!" String contains the word "World". If it does, we print out a message.

Conclusion

In this article, we've looked at three different ways to check if a Java String contains a given character or substring. The 

          
           
contains()
method is the simplest option, but it doesn't provide any control over the search. The 

          
           
matches()
method allows us to use regular expressions to search for patterns, but this method can be slow. Finally, the 

          
           
indexOf()
method allows us to search for a specific substring, but it has the potential to return false positives.
