How to Split a String in Java

Java provides several ways to split a string into multiple parts. Knowing the different options for splitting a string can help you write effective and efficient code. This article covers the various ways to split a string in Java, from the simplest to more complex algorithms.

Split Using Stringâ€™s split() Method

The simplest way to split a string in Java is using the split() method of the String class. This method takes a regular expression (regex) as the parameter and splits the string around matches of this regex. For example, if we have the following string: 

          
           
String str = "This is a test string";
We can use the space character as the regex argument to split the string around the spaces and produce an array of substrings. 

          
           
String[] subStrings = str.split("\\s+");
The array returned by this code will contain the following elements: 

          
           
["This", "is", "a", "test", "string"]

Split Using StringTokenizer

Another way to split a string in Java is by using the StringTokenizer class. This class provides methods for tokenizing a string. A token is a substring of the original string that is separated by one or more delimiters (characters that separate tokens). For example, to tokenize the same string as in the previous example, we can use the following code: 

          
           
StringTokenizer strTok = new StringTokenizer(str, "\\s+");
Then, we can loop through the tokens and print them: 

          
           
while(strTok.hasMoreTokens()) {
    System.out.println(strTok.nextToken());
}
The output of this code will be: 

          
           
This
is
a
test
string

Split Using Scanner Class

The Scanner class was added in Java 1.5 and provides a convenient way to tokenize strings. It has several methods for splitting a string. For example, the following code will tokenize the same string as in the previous examples: 

          
           
Scanner scanner = new Scanner(str);
We can then loop through the tokens and print them: 

          
           
while(scanner.hasNext()) {
    System.out.println(scanner.next());
}
The output of this code will be the same as with the StringTokenizer class: 

          
           
This
is
a
test
string

Split Using Regular Expressions

Regular expressions are powerful tools for working with strings. They allow us to match patterns in strings and use those matches to split a string. For example, to split the same string as in the previous examples by a specific character, we can use the following code: 

          
           
String[] subStrings = str.split("e");
This code will return the following array: 

          
           
["This is a t", "st string"]

Conclusion

In this article, we looked at different ways to split a string in Java. We saw how to use the split() method of the String class, the StringTokenizer class, the Scanner class, and regular expressions.
