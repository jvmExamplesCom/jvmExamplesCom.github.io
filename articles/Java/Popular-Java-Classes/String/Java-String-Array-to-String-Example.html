Java String Array to String Example

In this article, we will learn how to convert a Java String array to a string. Java strings are stored as character arrays and are accessed by index number. To convert an array of strings to a single string, use the join() method of the String class.

Example 1: Using for loop to convert String array to String

In this example, we will use a for loop to convert a String array to a String.


          
           
// Java program to convert String array to String 
public class JavaStringArrayToString {  
    public static void main(String[] args) { 
        // Declare a String array 
        String[] words = { "Hello", "World" }; 
  
        // Declare an empty String 
        String resultantString = ""; 
          
        // Iterate through the array of Strings and add them to resultant string 
        for(int i = 0; i < words.length; i++) { 
            resultantString += words[i];
        } 
          
        // Display the resultant String 
        System.out.println("Resultant String = " + resultantString); 
    } 
}
Output: 

          
           
Resultant String = HelloWorld

Example 2: Using Java 8 Stream API to convert String array to String

In this example, we will use Java 8 Stream API to convert a String array to a String.


          
           
// Java program to convert String array to String using Stream API 
import java.util.Arrays;  
import java.util.stream.Collectors; 
public class JavaStringArrayToString {  
    public static void main(String[] args) { 
        // Declare a String array 
        String[] words = { "Hello", "World" }; 
  
        // Using Stream API to convert String array to String 
        String resultantString = Arrays.stream(words).collect(Collectors.joining()); 
          
        // Display the resultant String 
        System.out.println("Resultant String = " + resultantString); 
    } 
}
Output: 

          
           
Resultant String = HelloWorld

Example 3: Using StringJoin class to convert String array to String

In this example, we will use StringJoin class from Java 11 to convert a String array to a String.


          
           
// Java program to convert String array to String using StringJoin class 
import java.lang.StringJoiner; 
public class JavaStringArrayToString {  
    public static void main(String[] args) { 
        // Declare a String array 
        String[] words = { "Hello", "World" }; 
  
        // Using StringJoin class to convert String array to String 
        StringJoiner joiner = new StringJoiner(""); 
        for (String word: words){ 
            joiner.add(word); 
        } 
        String resultantString = joiner.toString(); 
          
        // Display the resultant String 
        System.out.println("Resultant String = " + resultantString); 
    } 
}
Output: 

          
           
Resultant String = HelloWorld
