Convert String to Uppercase in Java

In this article, we'll discuss how to convert string to uppercase in Java. We'll explain the different syntaxes and provide useful tips and code examples.

Method 1: toUpperCase()

The easiest way to convert string to uppercase is to use the built-in 

toUpperCase()
method of the 
String
class. This method returns a copy of the string, with all characters converted to uppercase according to the rules of the default locale. 

          
           
String str = "hello world";
String uppercaseStr = str.toUpperCase();
System.out.println(uppercaseStr); // Outputs: HELLO WORLD

Method 2: toUpperCase(Locale)

If you need to convert the string to uppercase based on a specific locale, you can use the 

toUpperCase(Locale)
method of the 
String
class. This method takes a 
Locale
parameter and returns a copy of the string, with all characters converted to uppercase according to the rules of the specified locale. 

          
           
String str = "hello world";
Locale locale = Locale.US;
String uppercaseStr = str.toUpperCase(locale);
System.out.println(uppercaseStr); // Outputs: HELLO WORLD

Method 3: toUpperCase(Charset)

If you need to convert the string to uppercase based on a specific character set, you can use the 


          
           
toUpperCase(Charset)
method of the 
String
class. This method takes a 
Charset
parameter and returns a copy of the string, with all characters converted to uppercase according to the rules of the specified character set. 

          
           
String str = "hello world";
Charset charset = StandardCharsets.UTF_8;
String uppercaseStr = str.toUpperCase(charset);
System.out.println(uppercaseStr); // Outputs: HELLO WORLD
