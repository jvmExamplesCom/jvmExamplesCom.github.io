Java String startsWith Example

The String.startsWith() method checks if the given string begins with the given prefix.

Example 1

Let's see a basic example of the 

          
           
startsWith()
method. We will create a Java class and define two strings 

          
           
s1
and 

          
           
prefix1
. The 

          
           
s1
string will be checked with the 

          
           
startsWith()
method and the 

          
           
prefix1
string will be passed as an argument: 

          
           
public class JavaStringStartsWithExample {
    public static void main(String[] args) {
        String s1 = "Java String startsWith Example";
        String prefix1 = "Java";

        boolean b1 = s1.startsWith(prefix1);
        System.out.println("b1 = " + b1);
    }
}
This code will print out the following output: 

          
           
b1 = true

startsWith()

s2

prefix1

startsWith()

String s2 = "String startsWith Example";
