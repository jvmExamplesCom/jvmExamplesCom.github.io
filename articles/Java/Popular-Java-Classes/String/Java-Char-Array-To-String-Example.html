Java Char Array To String Example

To convert a char array to a string in Java, we can use the String class constructor that accepts a char array as an argument. The constructor creates a new String instance from the given char array. For example: 

          
           
char[] arr = {'J', 'a', 'v', 'a'};
String str = new String(arr);
System.out.println("String is: "+ str);
The output of the above code is: 

          
           
String is: Java
We can also use the String.valueOf() method to do the same thing. Here is an example of how to convert a char array to String using the String.valueOf() method: 

          
           
char[] arr = {'J', 'a', 'v', 'a'};
String str = String.valueOf(arr);
System.out.println("String is: "+ str);
The output of the above code is: 

          
           
String is: Java
In addition to the above two methods, we can also use a for loop to iterate over each character of the array and append it to a StringBuilder object. This is a good approach if you want to process each character before appending it to the desired string. Here is an example of how to convert a char array to String using a for loop and a StringBuilder object: 

          
           
char[] arr = {'J', 'a', 'v', 'a'};
StringBuilder sb = new StringBuilder();
for (char c : arr) {
    sb.append(c);
}
String str = sb.toString();
System.out.println("String is: "+ str);
The output of the above code is: 

          
           
String is: Java

int[] arr = {1, 2, 3, 4};
String str = Arrays.stream(arr) 
                      .mapToObj(String::valueOf)
                      .collect(Collectors.joining(","));
System.out.println("String is: "+ str);

String is: 1,2,3,4
