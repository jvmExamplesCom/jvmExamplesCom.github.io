How to Check if the String is in Lower Case

String checking is a common task in Java programming. Checking whether a string is in lower case can be useful in many cases, such as making sure the user's input text is uniform or validating certain values. This article will explain how to easily check if a String is in all lower case letters in Java.

Using the String class

The best and most reliable way to check if a String is in lower case is to use the 

          
           
equalsIgnoreCase()
method of the String class. This method takes another String as an argument and returns a boolean value. If both strings are the same with respect to case, then the result will be 

          
           
true
. Here's an example: 

          
           
String str = "example";
boolean isLowerCase = str.equalsIgnoreCase(str.toLowerCase()); 
System.out.println(isLowerCase); // prints "true"
In this example, the 

          
           
toLowerCase()
method is used to convert the original String to lower case. The 

          
           
equalsIgnoreCase()
method compares the resulting String to the original one, ignoring the case. If they are the same, it returns 

          
           
true
.

Using Regular Expressions

Regular expressions can also be used to check if a String is in lower case. The 

          
           
matches()
method of the String class takes a regular expression pattern as an argument and returns a boolean depending on whether or not the original String matches the given pattern. To check if a String is in lower case, the following pattern should be used: 

          
           
^[a-z]+$
This pattern matches only lowercase letters and nothing else. Here's an example: 

          
           
String str = "example";
boolean isLowerCase = str.matches("^[a-z]+$");
System.out.println(isLowerCase); // prints "true"
In this example, the 

          
           
matches()
method is used with the regular expression pattern above to check if the given String is in lower case. If it is, then the result will be 

          
           
true
.

Conclusion

Checking whether a String is in lower case can be useful in many cases, such as validating user input or ensuring uniformity in certain values. In Java, this can be done by using the 

          
           
equalsIgnoreCase()
or the 

          
           
matches()
methods. Both of these methods take other arguments, such as a String to compare with or a regular expression pattern to match against, and return a boolean value if the given String is in lower case.
