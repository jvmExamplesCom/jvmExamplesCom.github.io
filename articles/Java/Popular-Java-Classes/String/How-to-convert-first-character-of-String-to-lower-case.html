How to Convert the First Character of a String to Lower Case in Java

Java provides many different methods for manipulating strings. In this tutorial, we will learn how to convert the first character of a string to lower case in Java.

Using String.substring()

One way to convert the first character of a string to lower case is to use the 

          
           
String.substring()
method. This method takes two parameters, a start index and an end index, and returns a substring starting at the start index and ending at the end index. We can use this method to partially extract the first character, convert it to lower case, and then combine it with the rest of the string. For example, consider the following code: 

          
           
String name = "John";
String first_lower = name.substring(0,1).toLowerCase() + name.substring(1);

System.out.println(first_lower);
Here, we extract the first character of the string 

          
           
name
, convert it to lower case using the 

          
           
toLowerCase()
method, and then combine it with the rest of the string. The output of this code is: 

          
           
john

Using charAt() and toLowerCase()

Another method of converting the first character of a string to lower case is to first use the 

          
           
charAt()
method to extract the character and then use the 

          
           
toLowerCase()
method to convert it to lower case. For example, consider the following code: 

          
           
String name = "John";
char first_char = name.charAt(0);
String first_lower = Character.toLowerCase(first_char) + name.substring(1);

System.out.println(first_lower);
Here, we first extract the first character of the string 

          
           
name
using the 

          
           
charAt()
method, and then convert it to lower case using the 

          
           
toLowerCase()
method. We then combine it with the rest of the string. The output of this code is: 

          
           
john

Using replaceFirst()

Finally, we can also use the 

          
           
String.replaceFirst()
method to convert the first character of a string to lower case. This method takes two parameters, a regular expression and a replacement string, and replaces the first occurrence of the regular expression with the replacement string. We can use this method to replace the first word in the string with a word converted to lower case. For example, consider the following code: 

          
           
String name = "John";
String first_lower = name.replaceFirst("\\b[A-Z]", 
            (name.charAt(0) + "").toLowerCase());

System.out.println(first_lower);
Here, we use the 

          
           
replaceFirst()
method to replace the first word in the string, which is the first character, with a word converted to lower case. The output of this code is: 

          
           
john

String.substring()
, 

          
           
charAt()
, and 

          
           
replaceFirst()
