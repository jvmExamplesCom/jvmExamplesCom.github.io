Java String Compare Example

In this Java tutorial, we will explore how to compare two strings in Java. We will look at different ways to compare strings and cover some of the best practices to use while comparing strings.

String Equals Method

The simplest way to compare two strings is to use the ‘equals()’ method provided by the String class. This method compares two strings lexicographically and returns a boolean value indicating if the two strings are equal. 

          
           
String str1 = "Hello World";
String str2 = "Hello World";
boolean result = str1.equals(str2);
System.out.println(result); // prints true
Note that this method is case-sensitive, so if you want to do a case-insensitive comparison, you need to change both strings to the same case before performing the comparison.

Comparing Strings with == Operator

You can also compare two strings using the ‘==’ operator. It returns true if both strings refer to a same object. 

          
           
String str1 = "Hello World";
String str2 = "Hello World";
boolean result = (str1 == str2);
System.out.println(result); // prints true
However, this is not a reliable way to compare two strings as it only checks if they are pointing to the same object. If the strings are constructed separately but with the same content, the result will be false since they will point to different objects in memory.

String CompareTo Method

Another way to compare two strings is to use the ‘compareTo()’ method provided by the String class. This method is used to compare two strings lexicographically. It returns an integer indicating whether the first string is less than, equal to, or greater than the second string. 

          
           
String str1 = "Hello World";
String str2 = "Hello World";
int result = str1.compareTo(str2);
System.out.println(result); // prints 0
If the first string is lexicographically less than the second string, the result will be a negative number. If the first string is lexicographically greater than the second string, the result will be a positive number.

String RegionMatches Method

The ‘regionMatches()’ method is another way to compare two strings. However, unlike the other methods mentioned above, this one allows you to compare substrings of two strings rather than the full strings. This method can be useful when you are looking for a match within the strings instead of an exact match. 

          
           
String str1 = "Hello World";
String str2 = "Hello";
boolean result = str1.regionMatches(0, str2, 0, 5);
System.out.println(result); // prints true
This method takes four arguments - char array, starting index of each string, length of the substring to compare, and a case-sensitivity flag. The last argument is a boolean which indicates if the comparison should be done in a case-sensitive manner or not.

Best Practices

When comparing strings in Java, the most reliable way is to use the ‘equals()’ method. If you need to do a case-insensitive comparison, make sure to convert both strings to the same case before performing the comparison. It is also important to note that the ‘==’ operator should not be used to compare strings, as it only checks if they are pointing to the same object. For situations where you need to compare substrings, the ‘regionMatches()’ method can be used. Make sure to specify the parameters correctly, such as providing the starting index, desired length, and the case-sensitivity flag, to get the desired result.
