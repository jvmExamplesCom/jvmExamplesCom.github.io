Java InputStream to String Example

It is quite common when working with Java in an application to need to read input from a stream and convert it into a String format. This can easily be done using the 

InputStream
and 
InputStreamReader
classes in Java. In this example, we will show how to convert an 
InputStream
object into a String object.

Read InputStream into String

First, let's create an 
InputStream
object from a String. We can do this by reading the String bytes into a ByteArrayInputStream: 

          
           
String inputString = "This is an example String"; 

ByteArrayInputStream bais = new ByteArrayInputStream(inputString.getBytes());
Next, we will create an 
InputStreamReader
to read from the 
InputStream
: 

          
           
InputStreamReader isr = new InputStreamReader(bais);
Now, to create a String from the 
InputStream
, we will need to read the contents of the 
InputStream
into a 
StringBuffer
using a BufferedReader: 

          
           
BufferedReader br = new BufferedReader(isr);

StringBuffer sb = new StringBuffer();

String line;

while ((line = br.readLine()) != null) {
    sb.append(line);
}

br.close();
Finally, we can convert the contents of the 
StringBuffer
into a String using the 
toString()
method: 

          
           
String outputString = sb.toString();
At this point, the 
outputString
variable contains the String version of the data that was in the 
InputStream
.

Conclusion

In this tutorial, we have shown how to convert an 
InputStream
object into a String object in Java. We have done this by first reading the InputStream into a ByteArrayInputStream, then creating an InputStreamReader object to read from it, and finally reading the contents of the InputStreamReader into a StringBuffer and converting it into a String.
