How to Remove Multiple Consecutive Spaces from a String in Java

Removing multiple consecutive spaces from a string can be achieved in multiple ways using the Java programming language. Here we will discuss four techniques.

Using the replaceAll() Method

The most straightforward way of accomplishing this task is through the use of the 

replaceAll()
method, which takes two parameters: a regular expression and a replacement. By supplying a regular expression matching any number of consecutive spaces, and a replacement of a single space, one can easily modify the string to remove multiple consecutive spaces. 

          
           
String s = "Hello    world!";
String result = s.replaceAll("\\s+", " "); //Hello world!

Using the split() Method

The 
split()
method can also be used to remove multiple consecutive spaces. This method will take the input string and split it into an 
array
of 
String
based on the provided expression. In this case, providing the same expression used in our 
replaceAll()
example above will result in an array of substrings that were separated by multiple consecutive spaces. Finally, by looping over the array, we can recombine the substrings into a new string, omitting the extra spaces. 

          
           
String s = "Hello    world!";
String[] arr = s.split("\\s+"); //["Hello", "world!"]

String result = "";
for (String sub : arr) {
    result += sub + " ";
}
result = result.trim(); //Hello world!

Using the StringTokenizer Class

The 
StringTokenizer
class can be used in Java to break a string into tokens. By providing a regular expression for the delimiter, we can tokenize multiple consecutive spaces as well. The resulting tokens can then be appended to a 
StringBuilder
object to create the new string with no multiple consecutive spaces. 

          
           
String s = "Hello    world!";
StringTokenizer st = new StringTokenizer(s, "\\s+");

StringBuilder sb = new StringBuilder();
while (st.hasMoreTokens()) {
    sb.append(st.nextToken() + " ");
}
sb.setLength(sb.length() - 1);

String result = sb.toString(); //Hello world!

Using the CharSequence Interface

The 
CharSequence
interface can be used to provide a sequence of characters. A 
String
object implements this interface, so we can use it to iterate over the characters of the string and concatenate any character that is not a space, or a group of spaces, to a 
StringBuilder
object. This technique is the least efficient of the four. 

          
           
String s = "Hello    world!";
StringBuilder sb = new StringBuilder();

for (int i = 0; i < s.length(); i++) {
    char c = s.charAt(i);
    if (c != ' ' || (i > 0 &amp;&amp; s.charAt(i - 1) != ' ')) {
        sb.append(c);
    }
}

String result = sb.toString(); //Hello world!

In conclusion, there are multiple ways to accomplish the task of removing multiple consecutive spaces from a String in Java. The choice of approach will depend on the requirements, such as efficiency or readability.
