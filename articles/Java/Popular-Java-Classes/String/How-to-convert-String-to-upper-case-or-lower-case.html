Converting String to Upper and Lower Case in Java

Java provides a range of methods for converting strings from one type of case to another. In this article, we will cover two of them: String.toUpperCase() and String.toLowerCase(). Both are easy to implement and can help us write more efficient code.

String.toUpperCase()

The String.toUpperCase() method takes a string and returns an uppercase version of it with all characters in the original string replaced by their upper-case equivalents. The following example shows how to use this method to convert a simple string to uppercase: 

          
           
String str = "Hello World!";
String newStr = str.toUpperCase();
System.out.println("Old string: " + str);
System.out.println("New string:"+ newStr);

// Output
// Old string: Hello World!
// New string: HELLO WORLD!

String.toLowerCase()

The String.toLowerCase() method takes a string and returns a lower-case version of it with all characters in the original string replaced by their lower-case equivalents. The following example shows how to use this method to convert a simple string to lowercase: 

          
           
String str = "Hello World!";
String newStr = str.toLowerCase();
System.out.println("Old string: " + str);
System.out.println("New string:"+ newStr);

// Output
// Old string: Hello World!
// New string: hello world!
It is also possible to convert strings to mixed case with the help of regular expressions. All you need to do is to specify the desired upper and lower cases as parameters to a Pattern.matcher() method call. For example, the following code snippet converts the first letter of each word to uppercase: 

          
           
String str = "hello world";
String newStr = str.replaceAll( "(\\w)(\\w*)" , "\\u$1\\L$2"); 
System.out.println("Old string: " + str);
System.out.println("New string:"+ newStr);

// Output:
// Old string: hello world
// New string:Hello World
It is also possible to convert multiple words in a string to uppercase or lowercase. To do this, we can use the String.replaceAll() method: 

          
           
String str = "hello world";
String newStr = str.replaceAll( "(\\w)\\w*" , "\\u$1"); 
System.out.println("Old string: " + str);
System.out.println("New string:"+ newStr);

// Output:
// Old string: hello world
// New string:HW

Conclusion

We have seen how to use the String.toUpperCase() and String.toLowerCase() methods to convert a string from one type of case to another. We have also used regular expressions to convert strings to mixed case. With these techniques we can easily write efficient code to manipulate strings.
