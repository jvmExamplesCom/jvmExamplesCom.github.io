Java String to Lower Case

The Java String toLowerCase() method converts all the characters in a string to lower case. In other words, this method returns a string where all the characters in the original string are transformed to lower case characters. This method does not modify the original string, but instead it returns a new string with the modified characters. The syntax for the toLowerCase() method is: 

          
           
String toLowerCase();
where "String" is the name of the string object on which the method is being called.

Java String to Lower Case - Example

Here is an example of the Java String toLowerCase() method: 

          
           
public class Test {

    public static void main(String[] args) {

        String str1 = new String("Welcome to the world of Java");
        //Convert the string into uppercase
        String str2 = str1.toLowerCase();

        System.out.println("Original string : "+ str1);
        System.out.println("Resultant lower case string : "+ str2);
    }
}
This will produce the following result: 

          
           
Original string : Welcome to the world of Java
Resultant lower case string : welcome to the world of java

Java String to Lower Case - Special Character Example

In some cases, uppercase letters may have special character values. For example, the German uppercase letter "ß" is converted to the lowercase "ß". So, the following example shows how the toLowerCase() method handles special characters: 

          
           
public class Test {

    public static void main(String[] args) {

        String str = "This ß one";
        System.out.println(str.toLowerCase());
    }
}
This will also produce the following result: 

          
           
this ß one

The toLowerCase() method can also be used with a locale. For example, the following example uses the toLowerCase(Locale.FRENCH) method to convert the string to lowercase in French.
