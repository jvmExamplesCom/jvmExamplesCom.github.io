How to check if String contains substring using contains method

Java language provides many utility functions and methods that developers can use while working on projects. One of these methods is the 

          
           
contains()
method; this method returns a boolean value indicating whether the specified characters are part of the string. In this tutorial we will learn how to check if a string contains a substring using the 

          
           
contains()
method.

Syntax

The syntax of the 

          
           
contains()
method is as follows: 

          
           
string.contains(substring);

Parameters

The 

          
           
contains()
method takes one required parameter, which is the substring to be checked.

Return Value

The 

          
           
contains()
method returns 

          
           
true
if the specified substring is found in the string, and 

          
           
false
otherwise.

Example

Let's take an example to understand how this method works. The following code snippet checks if the string "Dogs" contains the substring "Do": 

          
           
String str = "Dogs";
boolean result = str.contains("Do");
System.out.println(result);
The output of the above code will be: 

          
           
true
In the above code snippet, we have declared a variable named "str" and assigned it the value "Dogs". Then we invoked the 

          
           
contains()
method to check if the string "Dogs" contains the substring "Do". Since the specified substring is found in the string, the method returns 

          
           
true
. We can also pass a regular expression as the parameter to the 

          
           
contains()
method. For example, the following code checks if the string "Dogs" contains the letter 'o': 

          
           
String str = "Dogs";
boolean result = str.contains("[o]");
System.out.println(result);
The output of the above code will be: 

          
           
true
In the above code snippet, we have passed the regular expression "[o]" as the parameter to the 

          
           
contains()
method. Since the specified regular expression matches the letter 'o' which is present in the "Dogs" string, the method returns 

          
           
true
. We can also use the 

          
           
contains()
method to check if a string contains all the characters of another string. For example, the following code checks if the string "Dog" contains the string "Do": 

          
           
String str1 = "Dog";
String str2 = "Do";
boolean result = str1.contains(str2);
System.out.println(result);
The output of the above code will be: 

          
           
true
In the above code snippet, we have passed the "Do" string as the parameter to the 

          
           
contains()
method. Since the "Do" string is found in the "Dog" string, the method returns 

          
           
true
.

Conclusion

In this tutorial we learned how to check if a string contains a substring using the 

          
           
contains()
method. We also learned how to use the 

          
           
contains()
method to check if a string contains all the characters of another string.
