Converting String to Byte Array in Java

Java provides various ways to convert a String to a byte array. Knowing how to do this conversion can be useful when using file or network operations that require binary data. In this article, we'll explore different methods of converting Strings to byte arrays and vice versa.

Using the String getBytes() Method

The easiest and most straightforward way to convert a String to a byte array is to use the 

          
           
getBytes()
method of the String class. This method will take a String as input and return a corresponding byte array. For example, the following code snippet will convert a String to a byte array: 

          
           
String s = "Hello World!";
byte[] bytes = s.getBytes();
The resulting 

          
           
bytes
array contains the following elements: 

          
           
72, 101, 108, 108, 111, 32, 87, 111, 114, 108, 100, 33

Using the java.nio.charset.Charset Class

Another way to convert a String to a byte array is to use the 

          
           
java.nio.charset.Charset
class. This class provides a mechanism for encoding Strings into a sequence of bytes. We can use this class to encode a String into a byte array. For example, the following code snippet will convert a String to a byte array using the Charset class: 

          
           
String s = "Hello World!";
Charset charset = Charset.defaultCharset();
byte[] bytes = s.getBytes(charset);
The resulting 

          
           
bytes
array contains the same elements as before: 

          
           
72, 101, 108, 108, 111, 32, 87, 111, 114, 108, 100, 33

Using the Base64 Encoding

The 

          
           
java.util.Base64
class provides a way to encode a String into a sequence of bytes that can be represented as a Base64-encoded String. We can use this class to convert a String to a Base64-encoded byte array. For example, the following code snippet will convert a String to a Base64-encoded byte array: 

          
           
String s = "Hello World!";
Base64.Encoder encoder = Base64.getEncoder();
byte[] bytes = encoder.encode(s.getBytes());
The resulting 

          
           
bytes
array contains the following elements: 

          
           
72, 101, 108, 108, 111, 32, 87, 111, 114, 108, 100, 33
We can also decode these bytes back into a readable String using the 

          
           
Base64.Decoder
class. The following code snippet does this: 

          
           
Base64.Decoder decoder = Base64.getDecoder();
String decodedString = new String(decoder.decode(bytes));
System.out.println(decodedString); // Prints "Hello World!"

Conclusion

In this article, we've explored different ways to convert a String to a byte array in Java. We've seen how to use the 

          
           
String.getBytes()
, 

          
           
java.nio.charset.Charset
, and 

          
           
java.util.Base64
classes to do this conversion. Knowing these techniques can be very useful when dealing with file or network operations that require working with binary data.
