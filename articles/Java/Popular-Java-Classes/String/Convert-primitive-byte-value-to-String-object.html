Convert primitive byte value to String object in Java

Java provides a range of primitive types which have a reserved storage location in memory. One of them is the 

          
           
byte
type. It is 8-bit signed two's complement integer, which ranges between -128 and 127. This article discusses how to convert a 

          
           
byte
value to a 

          
           
String
object in Java.

Method 1: With Byte.toString()

The Java 

          
           
Byte
class provides static methods for converting a 

          
           
byte
to a 

          
           
String
object. The following example shows how to convert a 

          
           
byte
value to a 

          
           
String
using 

          
           
Byte.toString()
: 

          
           
byte myByte = (byte) 42;
String myString = Byte.toString(myByte);
System.out.println(myString);
// Outputs "42"
In this method, the 

          
           
byte
value is passed as an argument to the 

          
           
Byte.toString()
static method, which returns a 

          
           
String
object representation of the 

          
           
byte
value.

Method 2: With String.valueOf()

Another way to convert a 

          
           
byte
value to a 

          
           
String
object is to use the 

          
           
String.valueOf()
static method, as shown in the following example: 

          
           
byte myByte = (byte) 42;
String myString = String.valueOf(myByte);
System.out.println(myString);
// Outputs "42"
The 

          
           
String.valueOf()
static method takes the 

          
           
byte
value as an argument and returns a 

          
           
String
object which represents the 

          
           
byte
value.

Method 3: With Byte.toString() and Integer.toString()

The Java 

          
           
Integer
class provides static methods for converting an integer value to a 

          
           
String
object. Using this method, the 

          
           
byte
value can be converted to an 

          
           
int
and then to a 

          
           
String
, as shown in the following example: 

          
           
byte myByte = (byte) 42;
String myString = Integer.toString(Byte.toUnsignedInt(myByte));
System.out.println(myString);
// Outputs "42"
In this method, the 

          
           
byte
value is first converted to an unsigned 

          
           
int
value using the 

          
           
Byte.toUnsignedInt()
static method. The result is then passed as an argument to the 

          
           
Integer.toString()
static method, which returns a 

          
           
String
object representation of the 

          
           
int
value.

Conclusion

This article discussed three ways to convert a 

          
           
byte
