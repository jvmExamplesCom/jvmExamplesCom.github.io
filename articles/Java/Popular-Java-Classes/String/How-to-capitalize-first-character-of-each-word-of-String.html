Capitalizing First Character of Each Word of a String in Java

Capitalizing the first character of each word in a string is a common task for implementing standard text formatting conventions for display. In this article, we'll discuss how to capitalize the first character of each word in a string using Java. We will also provide code examples that demonstrate how to achieve this in different scenarios.

Using the Character Class and toUpperCase() Method

The easiest way to capitalize each word of a string is to use the 

Character
class and its 

          
           
toUpperCase(char ch)
method. The 
Character
class provides several static utility methods that can be used to work with characters, such as converting letter case and comparing characters numerically.

Let's look at a code example that demonstrates how to capitalize the first character of each word in a string: 


          
           
String str = "this is a java program";

StringBuilder result = new StringBuilder();

for (int i=0; i<str.length(); i++){
    char currentChar = str.charAt(i);
    if (i == 0 || Character.isWhitespace(str.charAt(i-1))){
        result.append(Character.toUpperCase(currentChar));
    } else {
        result.append(currentChar);
    }
}

System.out.println("Resulting string: " + result);
The output of this code example is: 

          
           
Resulting string: This Is A Java Program

Using the split() and substring() Methods

Another way to capitalize the first character of each word in a string is to use the 

split()
and 
substring()
methods. The 
split()
method splits a string into an array of substrings based on the given delimiter. In this case, we'll use the whitespace character ( 
" "
) as the delimiter. The 
substring()
method extracts part of a string, starting at the given index.

Let's look at a code example that uses these methods to capitalize the first character of each word in a string: 


          
           
String str = "this is a java program";

String[] words = str.split(" ");

StringBuilder result = new StringBuilder();

for(int i=0; i<words.length; i++) {
    result.append(words[i].substring(0,1).toUpperCase() + words[i].substring(1) + " ");
}

System.out.println("Resulting string: " + result);
The output of this code example is: 

          
           
Resulting string: This Is A Java Program

replaceAll()
