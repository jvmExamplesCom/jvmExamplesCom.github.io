How to Get String Length in Java

Java is a powerful programming language that provides a wide range of features and capabilities. It allows developers to quickly create powerful and efficient applications. One of the features Java offers is the ability to get the length of a string. This article will discuss how to get the length of a string in Java.

Using the length() Method

The simplest way to get the length of a string in Java is to use the 

          
           
length()
method. This method will return the length of the specified string. For example: 

          
           
String str = "Hello World!";
int length = str.length();
System.out.println(length); // Outputs 12
In this example, we create a string called 

          
           
str
, which contains the value 

          
           
"Hello World!"
. We then call the 

          
           
length()
method on this string, which returns the length of the string (in this case, 12). We store this value in an 

          
           
int
called 

          
           
length
, and then print it out using 

          
           
System.out.println()
.

Using a Loop

Another way to get the length of a string in Java is to use a loop. We can use a 

          
           
for
loop to iterate over each character in the string, and increment a counter each time. When the loop reaches the end of the string, the counter will contain the length of the string. For example: 

          
           
String str = "Hello World!";
int length = 0;
for (char c : str.toCharArray()) {
  length++;
}
System.out.println(length); // Outputs 12
