How to Get Count of Substring in a String with Java

Java provides various ways to manipulate strings. One such technique is counting the number of times a substring appears in a string. In this article, we will explore different methods of counting the occurrences of a substring within a string. We will also provide code examples and output to illustrate how these methods work.

Using Java's String.split() Method

The split() method splits up a string into an array and takes a regular expression as the delimiter. The resulting array will contain the substrings matching the delimiter. We can use this method to count the occurrences of a substring and get the total count. For example, let's count the number of occurrences of the word 'the' in the sentence "the cat sat on the mat". We will use the following code to split the sentence based on the word 'the': 

          
           
String sentence = "the cat sat on the mat";
String[] words = sentence.split("the");
After running the code, the value of the words array will be ["", " cat sat on ", " mat"]. The empty space at the front of the array indicates that the first occurrence of 'the' was the beginning of the sentence. The empty spaces at the end of the words 1, 2 and 3 indicate that there was an occurrence of the word 'the' between each of the other words. Therefore, the total count of 'the' in the sentence is 3.

Using Java’s String.contains() Method

The contains() method of the String class returns a boolean indicating whether a string contains a given substring or not. We can use this method to loop through the entire string and check for all occurrences of the substring. We can then sum up the total number of occurrences. Let's use this method to count the occurrences of the word 'the' in the same sentence as before: "the cat sat on the mat". Here is the code: 

          
           
String sentence = "the cat sat on the mat";
int count = 0;
for (int i = 0; i < sentence.length(); i++) {
    if (sentence.contains("the")) {
        count++;
    }
}
System.out.println(count);
The output of the code is 3, which is the same as when we used the split() method.

Using Java’s String.indexOf() Method

The indexOf() method of the String class returns the index of the first occurrence of a substring in a given string. We can use this method to iterate through the entire string and check for all occurrences of the substring. We can then sum up the total number of occurrences. Let's use this method to count the occurrences of the word 'the' in the same sentence as before: "the cat sat on the mat". Here is the code: 

          
           
String sentence = "the cat sat on the mat";
int index = -1;
int count = 0;
while ((index = sentence.indexOf("the", index + 1)) != -1) {
    count++;
}
System.out.println(count);
The output of the code is again 3, which is the same as when we used the split() and contains() methods.

Conclusion

In this article, we have explored three different methods to count the occurrences of a substring in a string using Java. All three of these methods return the same result, which is 3 in our example sentence. However, each of them has its own advantages and disadvantages, so it is worth considering which one best fits the needs of your application.
