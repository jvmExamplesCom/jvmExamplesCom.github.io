Converting Strings to Respective Primitive Values in Java

Java provides an easy way to turn a string value into the associated primitive value. This is useful for many reasons, such as allowing for user input in a program and ensuring that it produces the desired output. In this article, we'll cover how to convert strings to their respective primitive values in Java.

Using parseXxx() Methods

For many primitive types, there are 
parseXxx()
methods available which allow you to quickly and easily convert a String to its corresponding primitive. For example, here is how you would convert a string to an integer: 

          
           
String strInt = "10"; 
int intValue = Integer.parseInt(strInt); //intValue = 10

Likewise, you can use the same approach to convert a String to any of these other primitive types:

  • Long.parseLong()
  • 
            
             
Double.parseDouble()

            
  • Float.parseFloat()
  • Short.parseShort()
  • 
            
             
Boolean.parseBoolean()

            
  • Byte.parseByte()
  • 
            
             
Character.getNumericValue()

Here is a simple example of converting a string to a Boolean value:


          
           
String strBool = "true"; 
boolean boolValue = Boolean.parseBoolean(strBool); //boolValue = true

And here is a simple example of converting a string to a character value:


          
           
String strChar = "3"; 
char charValue = Character.getNumericValue(strChar); //charValue = 3

Using the ValueOf() Method

The 
valueOf()
method is another way of converting a string to its corresponding primitive. Here is an example of using it with an integer: 

          
           
String strInt2 = "10"; 
int intValue2 = Integer.valueOf(strInt2); //intValue2 = 10
The 
valueOf()
method works for all the same primitive types as the 
parseXxx()
methods. The result of calling 
valueOf()
on a String is generally the same as the result of calling 
parseXxx()
, however the two approaches may produce slightly different results in some cases (for example, when dealing with 
float
and 
double
values).

Using Wrapper Classes

If you are dealing with an object-oriented programming language like Java, then you don't need to worry about turning primitive types into objects. There are wrapper classes available for all the primitive types, which provide a way to wrap a primitive type in an object. Here is an example of using a wrapper class to turn a string into an integer:


          
           
String strInt3 = "10"; 
Integer intObj = new Integer(strInt3); 
int intValue3 = intObj.intValue(); //intValue3 = 10

Wrapper classes are especially useful if you want to store the values of the primitive type inside a collection or data structure. You can use the same approach for any primitive type, just by changing the wrapper class that you use.

Conclusion

In summary, Java provides a number of ways to convert strings to respective primitive values. You can use either the 

parseXxx()
methods or the 
valueOf()
method to quickly and easily convert a string to its corresponding primitive type. You can also use wrapper classes if you need to store the primitive value inside a collection or data structure. No matter what approach you use, you are now equipped with the knowledge to quickly and easily convert strings to primitive values in Java.
