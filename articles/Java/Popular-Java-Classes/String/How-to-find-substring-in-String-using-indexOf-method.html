How to find substring in String using indexOf Method

Java provides several methods for finding the substring from a string. The most commonly used method to search for the substring is 

          
           
indexOf()
method. This method allows us to search for the index of a character or a substring in a given string object. In this article, we will discuss how to use the 

          
           
indexOf()
method to find out the substring from a string in Java.

1. Syntax

The syntax of the 

          
           
indexOf()
method is as follows: 

          
           
public int indexOf(int ch)

public int indexOf(int ch, int fromIndex)

public int indexOf(String str)

public int indexOf(String str, int fromIndex)

2. Parameters

The 

          
           
indexOf()
method takes two parameters, the first one is required and the second one is optional.
  • ch: It is an integer value which specifies the character for which we need to search for its index.
  • fromIndex: It is an integer value which specifies the index from where the search should start.
  • str: It is a string which specifies the substring for which we need to search in the given string object.

3. Return Value

The 

          
           
indexOf()
method returns an integer value which is the index of the searched character or substring if it is found in the given string object. If the character or substring is not found, then the 

          
           
indexOf()
method will return -1.

4. Examples

Let's take a look at some examples to understand the working of the 

          
           
indexOf()
method.

Example 1: Find Character Index

In this example, we will use the 

          
           
indexOf()
method to find the index of a given character in a string. 

          
           
public class FindCharacterIndex {
    public static void main(String[] args) {
        String str = "Hello World";
        char ch = 'o';
        int index = str.indexOf(ch);
        System.out.println("Index of '" + ch + "' in '" + str + "' is: " + index);
    }
}
Output: 

          
           
Index of 'o' in 'Hello World' is: 4

Example 2: Find Substring Index

In this example, we will use the 

          
           
indexOf()
method to find the index of a given substring in a string. 

          
           
public class FindSubstringIndex {
    public static void main(String[] args) {
        String str = "Welcome to Tutorialspoint";
        String subStr = "Tutorials";
        int index = str.indexOf(subStr);
        System.out.println("Index of '" + subStr + "' in '" + str + "' is: " + index);
    }
}
Output: 

          
           
Index of 'Tutorials' in 'Welcome to Tutorialspoint' is: 10

Example 3: Find Character Index From Specified Index

In this example, we will use the 

          
           
indexOf()
method to find the index of a given character in a string starting from a specified index. 

          
           
public class FindCharacterIndexFromSpecifiedIndex {
    public static void main(String[] args) {
        String str = "Welcome to Tutorialspoint";
        char ch = 'o';
        int fromIndex = 11;
        int index = str.indexOf(ch, fromIndex);
        System.out.println("Index of '" + ch + "' in '" + str + "' starting from index " + fromIndex + " is: " + index);
    }
}
Output: 

          
           
Index of 'o' in 'Welcome to Tutorialspoint' starting from index 11 is: 15

    }
