Convert Primitive Boolean to String Object

Java provides a number of different ways that you can convert a primitive boolean value (true or false) to a String object. In this article, we'll look at some of the methods available for converting a boolean value to a String and provide sample code for each.

String.valueOf() Method

The simplest way to convert a boolean value to a String is to use the 

String.valueOf()
method. The syntax for this method looks like this: 

          
           
String result = String.valueOf(booleanValue);

For example, if you wanted to convert the boolean value 

true
to a String, you would do the following: 

          
           
boolean myBoolean = true;
String myString = String.valueOf(myBoolean);

String Constructor

Another way to convert a boolean value to a String is to use the String constructor. The syntax for this constructor looks like this:


          
           
String result = new String(booleanValue);

For example, if you wanted to convert the boolean value 

false
to a String, you would do the following: 

          
           
boolean myBoolean = false;
String myString = new String(myBoolean);

Boolean.toString() Method

The third way to convert a boolean value to a String is to use the 

Boolean.toString()
method. The syntax for this method looks like this: 

          
           
String result = Boolean.toString(booleanValue);

For example, if you wanted to convert the boolean value 

true
to a String, you would do the following: 

          
           
boolean myBoolean = true;
String myString = Boolean.toString(myBoolean);

String.format() Method

The fourth way to convert a boolean value to a String is to use the 

String.format()
method. The syntax for this method looks like this: 

          
           
String result = String.format(“%b”, booleanValue);

For example, if you wanted to convert the boolean value 

false
to a String, you would do the following: 

          
           
boolean myBoolean = false;
String myString = String.format("%b", myBoolean);

As you can see, there are a number of different ways that you can convert a boolean value to a String in Java. Each of these methods has its own advantages and disadvantages, and it’s important to understand which is the best one to use in each situation.
