Convert Primitive Double Value to String Object

Converting a primitive double into a String object is a common data manipulation action when working with Java. This guide explains how to convert a primitive double value into a String object using multiple methods.

The String.valueOf() Method

The easiest and most straightforward way to convert a primitive double value into a String object is by using the String.valueOf() method.


          
           
double d = 3.14;
String s = String.valueOf(d);
System.out.println(s); //outputs "3.14"

The String.valueOf() method can also be used directly on the primitive itself, like so:


          
           
double d = 3.14;
String s = d + "";
System.out.println(s); //outputs "3.14"

Using the Double.toString() Method

Another way to convert a primitive double value into a String object is to use the Double.toString() method. This method takes in the double value and returns a String representation of it.


          
           
double d = 3.14;
String s = Double.toString(d);
System.out.println(s); //outputs "3.14"

We can also use this method directly on the double primitive itself, like so:


          
           
double d = 3.14;
String s = d.toString();
System.out.println(s); //outputs "3.14"

Using the DecimalFormat Class

The DecimalFormat class is used for formatting numerical values into strings for user-readable output. This class provides various formatting options to control the display of numerical values. To use this class, we need to create an instance of it and pass in the desired format string.

Let’s see a simple example where we convert our double value into a String with two decimal places.


          
           
double d = 3.14;
DecimalFormat df = new DecimalFormat("0.00");
String s = df.format(d);
System.out.println(s); //outputs "3.14"

We can also use this method directly on the double primitive itself, like so:


          
           
double d = 3.14;
DecimalFormat df = new DecimalFormat("0.00");
String s = df.format(d);
System.out.println(s); //outputs "3.14"

Using the StringBuffer and StringBuilder Classes

We can also use the StringBuffer and StringBuilder classes to convert a primitive double value into a String object. This method involves converting the double value into a string by appending it to an empty string buffer or builder.

Let’s look at a simple example using the StringBuffer class.


          
           
double d = 3.14;
StringBuffer sb = new StringBuffer();
sb.append(d);
String s = sb.toString();
System.out.println(s); //outputs "3.14"

We can also use the StringBuilder class in a similar manner, like so:


          
           
double d = 3.14;
StringBuilder sb = new StringBuilder();
sb.append(d);
String s = sb.toString();
System.out.println(s); //outputs "3.14"

Conclusion

In this guide we discussed a few ways to convert a primitive double value into a String object. We looked at the String.valueOf(), Double.toString(), DecimalFormat, and StringBuffer/StringBuilder methods. Each of these methods can be used to achieve the same result and are dependent on the use case.
