How to Remove Leading and Trailing Spaces from String in Java

Removing leading and trailing whitespace can be a tedious job when dealing with strings in Java. In this article, we'll look at various ways of removing leading and trailing whitespaces from strings in Java.

Trim() Method

The easiest and most common way to remove leading and trailing whitespaces from a string is by using the trim() method. Simply call the trim() method on the string object, and it will remove all leading and trailing whitespaces. This method is also very useful for cleaning up user input before processing. 

          
           
String str = "    Hello, World!    ";
System.out.println(str);   // prints    Hello, World!    

String trimmedStr = str.trim();
System.out.println(trimmedStr);   // prints Hello, World!

Regex Pattern

If you need to remove leading and trailing whitespaces from a string using a regular expression, you can use the following pattern: 

          
           
// Pattern to remove leading and trailing whitespaces
Pattern pattern = Pattern.compile("^\\s+|\\s+$");
To use this pattern, simply compile the pattern using the Pattern.compile() method and then call the Matcher.replaceAll() method on a String object with an empty string as an argument. This will remove all leading and trailing whitespaces from the given string. 

          
           
String str = "    Hello, World!    ";
System.out.println(str);   // prints    Hello, World!    

Pattern pattern = Pattern.compile("^\\s+|\\s+$");
Matcher matcher = pattern.matcher(str);
String trimmedStr = matcher.replaceAll("");
System.out.println(trimmedStr);   // prints Hello, World!

Manual Removal

If you don't have access to the trim() method or a regex pattern, you can still remove leading and trailing whitespaces by manually checking each character. To do this, you can loop over each character in the string and check if it is a whitespace character. If it is, you can skip it, otherwise you can copy it to a new string. 

          
           
String str = "    Hello, World!";
System.out.println(str);   // prints    Hello, World!    

StringBuilder sb = new StringBuilder();
boolean foundStart = false;
for (int i = 0; i < str.length(); i++) {
    char c = str.charAt(i);
    if (c == ' ' &amp;&amp; !foundStart) {
        continue;
    } else {
        foundStart = true;
    }
    if (c != ' ' || (i == str.length()-1)) {
        sb.append(c);
    }
}
String trimmedStr = sb.toString();
System.out.println(trimmedStr);   // prints Hello, World!

Conclusion

In this article, we looked at various ways of removing leading and trailing whitespaces from strings in Java. The easiest and most common way is to use the trim() method, but you can also use a regex pattern or manually check each character if needed.
