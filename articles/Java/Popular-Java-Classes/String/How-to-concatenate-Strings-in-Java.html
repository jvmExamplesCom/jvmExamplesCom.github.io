How to Concatenate Strings in Java

Java provides various ways of concatenating strings. The simplest one is to use the + operator. This operator can be used to concatenate two or more strings. For example: 

          
           
String s1 = "Hello";
String s2 = "World";
String s3 = s1 + s2; 
// s3 will have the value "HelloWorld"
In this example, we have declared two strings and then used the + operator to concatenate them. The String class also provides a method called "concat" which can be used to concatenate strings. This method takes a String as parameter and appends it to the end of the String on which it is invoked. For example: 

          
           
String s1 = "Hello";
String s2 = "World";
String s3 = s1.concat(s2);
// s3 will have the value "HelloWorld"
Here, we have used the concat method to concatenate the two strings. We can also use the join() method of Java's StringJoiner class. This method takes a list of strings as parameter and concatenates them. For example: 

          
           
List<String> list = Arrays.asList("Hello","World");
String s = String.join(" ", list);
// s will have the value "Hello World"
In this example, we have used the join method to concatenate the strings in the list. We can also use the format() method of Java's String class. This method takes a format string and substitutes the placeholders with corresponding arguments. For example: 

          
           
String s1 = "Hello";
String s2 = "World";
String s3 = String.format("%s %s", s1, s2);
// s3 will have the value "Hello World"
In this example, we have used the format method to concatenate the two strings. Java also provides the StringBuffer class, which allows us to efficiently concatenate strings. This class has a method called append which appends a string to the existing StringBuffer object. For example: 

          
           
StringBuffer sb = new StringBuffer();
sb.append("Hello");
sb.append(" ");
sb.append("World");
String s = sb.toString();
// s will have the value "Hello World"
In this example, we have used the append method of the StringBuffer class to concatenate the strings. Finally, Java 8 introduces the StringJoiner class which provides a convenient way to concatenate strings. This class has a method called add which adds a new string to the existing StringJoiner object. For example: 

          
           
StringJoiner sj = new StringJoiner(" ");
sj.add("Hello");
sj.add("World");
String s = sj.toString();
// s will have the value "Hello World"
Here, we have used the add method of the StringJoiner class to concatenate the strings. These are some of the ways we can concatenate strings in Java. Each of these methods have their own advantages and disadvantages and it is up to the developer to decide which one best suits their needs.
