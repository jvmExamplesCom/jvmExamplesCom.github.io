How to Check if String Starts with a Number in Java

Checking if a String starts with a number is a common operation in Java. In this article, we'll explore various ways to check if a String starts with a number in Java.

Using a Regular Expression

One of the most reliable ways to check if a String starts with a number is to use a regular expression. We can use the following expression for our regular expression: 

          
           
Pattern pattern = Pattern.compile("[0-9]");
We can use the above expression to check if a String starts with a integer or a float value. We can then check if the given string starts with a number like this: 

          
           
Matcher matcher = pattern.matcher(inputStr);
if (matcher.find()) {
     System.out.println("String starts with a number");
} 
else {
    System.out.println("String does not start with a number");
}

Using the Character.isDigit() Method

Another way to check if a String starts with a number is to use the Character.isDigit() method. This method takes a single character as an argument and returns a boolean indicating whether that character is a digit or not. We can use the Character.isDigit() method to check if the first character of a String is a digit like this: 

          
           
String inputStr = "29John"; 
char firstChar = inputStr.charAt(0);
if (Character.isDigit(firstChar)) {
    System.out.println("String starts with a number");
} 
else {
    System.out.println("String does not start with a number");
}

Using the StringUtils.isNumeric Method

A third way to check if a String starts with a number is to use the StringUtils.isNumeric method from the Apache Commons Lang library. The StringUtils.isNumeric method is a convenience method that checks whether a given String is numeric or not. We can use the StringUtils.isNumeric method to check if a String starts with a number like this: 

          
           
String inputStr = "29John"; 
if (StringUtils.isNumeric(inputStr.substring(0, 1))) {
    System.out.println("String starts with a number");
} 
else {
    System.out.println("String does not start with a number");
}
In this article, we explored various ways to check if a String starts with a number in Java. We saw how to use a regular expression, the Character.isDigit() method and the StringUtils.isNumeric method from the Apache Commons Lang library for this purpose.
