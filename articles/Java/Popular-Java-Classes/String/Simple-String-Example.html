Java: Simple String Example

A String is an immutable sequence of characters in Java. It cannot be changed once created and is often used to store or transfer data from one class to another. This tutorial will provide a detailed example of creating, accessing and manipulating Strings.

Creating Strings

There are two ways of creating Strings in Java. The first is by declaring a String literal, which is simply writing the desired characters directly in-between double quotes: 

          
           
String myString = "Hello!";
The second is by using the String class' constructor, which allows you to create a String object based on an existing char array: 

          
           
char[] myCharArray = { 'H', 'e', 'l', 'l', 'o', '!' }; 
String myString = new String(myCharArray);

Accessing Strings

Once a String has been created, you can access its individual characters using the charAt() method. This method takes an integer as an argument which corresponds to the position of the character that needs to be accessed in the String. The positions start from 0 and go up until one less than the length of the String: 

          
           
String myString = "Hello!"; 
char myChar = myString.charAt(4);
System.out.println(myChar); // Outputs 'o'
In the above example, the charAt() method returned the 5th character (index 4) of the String, which was 'o'.

Manipulating Strings

Strings are often manipulated to create new Strings with different values. One of the methods used for this is the replace() method. This method takes two arguments. The first argument is the character to find in the String, while the second argument is the character that should replace it. Consider the following example: 

          
           
String myString = "Hello!"; 
String newString = myString.replace('l', 'i');
System.out.println(newString); // Outputs 'Heiio!'
In this case, the replace() method replaced all the occurrences of the character 'l' with the character 'i', resulting in the String being changed from "Hello!" to "Heiio!". Other methods that can be used for manipulating Strings include substring(), concat(), and trim().

Conclusion

This tutorial provided a detailed example of working with Strings in Java. It went over how to create Strings, access their characters, and manipulate them to create new Strings. With this knowledge, it should be easier to work with Strings in your own Java applications.
