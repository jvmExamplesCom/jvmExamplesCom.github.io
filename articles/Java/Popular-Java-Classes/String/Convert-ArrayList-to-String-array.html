Convert ArrayList to String Array

Java provides methods to convert an ArrayList to a string array. This article will provide detailed information about the different ways to convert an ArrayList to a string array, and provide code examples for each approach.

Using the toArray() Method

The simplest way to convert an ArrayList to a String array is to use the 

          
           
toArray()
method. The syntax of this method is as follows: 

          
           
listObject.toArray(new String[0]);
Let's look at an example. Suppose we have a list of Strings declared as follows: 

          
           
ArrayList<String> list = new ArrayList<String>(Arrays.asList("a", "b", "c"));
We can convert this list to a string array using the 

          
           
toArray()
method as follows: 

          
           
String[] arr = list.toArray(new String[0]);
The array will now contain the list items ["a", "b", "c"].

Using the Stream or Collectors Classes

Another approach to convert an ArrayList to a string array is to use the Stream or Collectors classes. The syntax of this approach is as follows: 

          
           
listObject.stream()
        .toArray(String[]::new);
We can again use the list declared in the previous example: 

          
           
ArrayList<String> list = new ArrayList<String>(Arrays.asList("a", "b", "c"));
And convert it to a string array using the Stream or Collectors classes: 

          
           
String[] arr = list.stream()
        .toArray(String[]::new);
Again, the result will be an array containing the list items ["a", "b", "c"].

Using the Iterator Class

We can also convert an ArrayList to a string array using the Iterator class. The syntax of this approach is as follows: 

          
           
ListIterator<String> itr = listObject.listIterator(); 
        String[] strArr = new String[listObject.size()]; 
        int i = 0; 
        while (itr.hasNext()) { 
            strArr[i++] = itr.next(); 
        }
Let's again use the list declared in the first example: 

          
           
ArrayList<String> list = new ArrayList<String>(Arrays.asList("a", "b", "c"));
We can then convert it to a string array using the Iterator class as follows: 

          
           
ListIterator<String> itr = list.listIterator(); 
        String[] strArr = new String[list.size()]; 
        int i = 0; 
        while (itr.hasNext()) { 
            strArr[i++] = itr.next(); 
        }
Again, the result will be an array containing the list items ["a", "b", "c"]. In conclusion, there are three ways to convert an ArrayList to a string array. The simplest approach is to use the 

          
           
toArray()
method. Another approach is to use the Stream or Collectors classes. And the third approach is to use the Iterator class. All three approaches will result in an array containing the list items.
