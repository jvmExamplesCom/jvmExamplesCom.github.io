How to Check if a String Starts With Another String or Substring in Java

In Java, strings are represented by the 
java.lang.String
class. This provides various methods for manipulating strings, including functions for determining whether a string starts with a given substring. In this article, we'll look at the various ways to check if a string starts with another string or substring in Java. We'll also provide code examples to show how each of these methods works.

Using StartsWith()

The most common way to check if a string starts with another string is to use the 

startsWith()
method on the 
java.lang.String
class. This method takes a single string parameter which is the substring you want to check against. It returns a boolean value, 
true
if the string starts with the given substring and 
false
otherwise. Here's an example of using the 
startsWith()
method: 

          
           
String str = "Hello World";
boolean result = str.startsWith("Hello");
System.out.println(result);
This code will print 
true
as the output since the string does start with "Hello".

Using Pattern Matching

Another way to check if a string starts with another string is to use pattern matching. Pattern matching allows you to define a pattern that a string must match in order for it to be considered "starting with" the given substring. To do this, we use the 


          
           
java.util.regex.Pattern
and 

          
           
java.util.regex.Matcher
classes. Here's an example of using pattern matching to check if a string starts with a given substring: 

          
           
String str = "Hello World";
Pattern pattern = Pattern.compile("^Hello");
Matcher matcher = pattern.matcher(str);
boolean result = matcher.find();
System.out.println(result);
This code will also print 
true
as the output since the string does match the pattern defined in the 
Pattern
object.

Conclusion

In this article, we looked at the various ways to check if a string starts with another string or substring in Java. We saw how to use the 

startsWith()
method and pattern matching for this purpose. We also provided code examples for each of these methods to help you get started. With this knowledge, you should be able to easily determine whether a string starts with a given substring in your own code.
