Java String to ArrayList Example

In this article, we will look at an example of how to convert a String to an ArrayList in Java. We will also provide some examples and further explanation about the process.

Example 1: Simple String to ArrayList

In this example, we will use the split() method of the String class which takes a regular expression as an argument. We will also use the ArrayList constructor which takes an array as an argument.


          
           
String str = "Hello World"; 
String[] arr = str.split(" "); 
ArrayList<String> list = new ArrayList<String>(Arrays.asList(arr));

Now if we print out the contents of the ArrayList, we get the following output:


          
           
Hello
World

Sometimes when we are converting a String to an ArrayList, we want to remove any empty elements from the result. This can be done using the removeIf() method of the ArrayList class which takes a predicate as an argument.

String[] arr = str.split("");
