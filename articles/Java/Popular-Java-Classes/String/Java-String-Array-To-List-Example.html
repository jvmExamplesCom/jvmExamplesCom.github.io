Java String Array To List Example

In this tutorial, we will discuss how to convert a Java string array to a list.

Using Java Arrays.asList() Method

The simplest way of converting a string array to a list is by using the Arrays.asList() method in Java. This method accepts an array and returns a list containing the elements of the given array. 


          
           
String[] strArray = {"Hello", "World"}; 
List<String> strList = Arrays.asList(strArray);

Using Java Streams

Another way of converting a string array to a list is by using Java streams. This is a more efficient approach as it does not require any extra memory and is also faster than using Arrays.asList(). 


          
           
String[] strArray = {"Hello", "World"}; 
List<String> strList = Stream.of(strArray).collect(Collectors.toList());

Java 8 List.of() Method

Java 8 provides a convenient method List.of(), which can be used to create a list from an array or any other collection. 


          
           
String[] strArray = {"Hello", "World"}; 
List<String> strList = List.of(strArray);

Using Java for-each Loop

We can also use a for-each loop to iterate through each element of the string array and add them to a List. 


          
           
String[] strArray = {"Hello", "World"}; 
List<String> strList = new ArrayList<>();
for (String s : strArray) {
   strList.add(s);
}

Using Java for Loop

We can also use a for loop to achieve the same result. 


          
           
String[] strArray = {"Hello", "World"}; 
List<String> strList = new ArrayList<>();
for (int i = 0; i < strArray.length; i++) {
    strList.add(strArray[i]);
}

Conclusion

In this tutorial, we discussed how to convert a string array to a list in Java. We saw how to use the Arrays.asList() method, Streams, the List.of() method and the for-each and for loops to achieve this.
