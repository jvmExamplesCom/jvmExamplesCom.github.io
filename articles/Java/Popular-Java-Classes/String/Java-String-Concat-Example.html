Java String Concat Example

Overview

Java String concatenation is the process of joining one or more strings together. This can be done using the + operator and += operator. This guide takes a look at Java String concatenation examples, with code snippets and output for each.

The + and += Operators

In Java, you have two main ways to do concatenation: the + operator and the += operator. The + operator is used for combining two strings. For example: 

          
           
String str1 = "Hello";
String str2 = "World!";
String str3 = str1 + str2;
System.out.println(str3); // Prints "HelloWorld!"
The += operator is used to combine a string with another string or a number. For example: 

          
           
String str1 = "Hello ";
int number = 1;
str1 += number;
System.out.println(str1); // Prints "Hello 1"
You can also use the + operator this way: 

          
           
String str1 = "Hello ";
int number = 1;
str1 = str1 + number;
System.out.println(str1); // Prints "Hello 1"

Integer Concatenation

You can also use the + operator to concatenate an integer to a string. For example: 

          
           
int number = 1;
String str1 = "Number: " + number;
System.out.println(str1); // Prints "Number: 1"

More Examples

You can also use the += operator to concatenate multiple strings together: 

          
           
String str1 = "Hello ";
String str2 = "world!";
str1 += str2;
System.out.println(str1); // Prints "Hello world!"
You can use the + operator to combine multiple strings together, as well as an integer: 

          
           
String str1 = "Hello ";
String str2 = "world!";
int number = 1;
String str3 = str1 + str2 + number;
System.out.println(str3); // Prints "Hello world!1"
You can also combine multiple variables using the += operator: 

          
           
String str1 = "Hello ";
String str2 = "world!";
int number = 1;
str1 += str2 + number;
System.out.println(str1); // Prints "Hello world!1"

Conclusion

Java String concatenation is easy to do and can be achieved using the + operator and += operator. With these operators, you can concatenate strings and integers, as well as multiple strings, variables and integers.
