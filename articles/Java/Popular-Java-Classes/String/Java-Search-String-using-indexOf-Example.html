Java Search String using indexOf Example

The Java 

          
           
indexOf
method is a very useful tool for searching through strings. It enables you to find the position of a substring within a string. This method can be used in many ways, so let's explore some examples.

Basic indexOf Usage

The most basic usage of the 

          
           
indexOf
method is to find the index of a single character or substring in a string. Here is an example of using 

          
           
indexOf
to find the index of a substring: 

          
           
String myString = "This is a test string";
int index = myString.indexOf("test");
System.out.println(index);
The above code will print out the index of the substring "test" which is 10.

Using Regular Expressions with indexOf

You can use regular expressions when searching a string with the 

          
           
indexOf
method. This makes it possible to search for patterns of characters instead of just specific substrings. Here is an example of searching a string with a regular expression: 

          
           
String myString = "This is a test string";
int index = myString.indexOf("t[a-z]st");
System.out.println(index);
The above code will print out the same index as before, since it is searching for the same substring. However, if the string was different, such as "this is a best string", then the index of the substring would be 7.

Using fromIndex Argument

Another useful feature of the 

          
           
indexOf
method is the fromIndex argument. This allows you to specify an index at which to begin searching the string. This can be useful for searching for multiple occurrences of a substring. Here is an example of using the fromIndex argument: 

          
           
String myString = "This is a test string";
int index = myString.indexOf("is", 5);
System.out.println(index);
The above code will print out the index 13, because it begins searching the string at the index 5 (the character "s"). Without the fromIndex argument, the index would have been 2.

Conclusion

The Java 

          
           
indexOf
method is a very powerful and versatile tool for searching strings. It can be used to search for specific characters, substrings, and even patterns of characters. It also provides an option to start the search from a specified index. With these features, the 

          
           
indexOf
method can be used to solve many problems.
