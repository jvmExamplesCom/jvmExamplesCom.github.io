How To Remove Leading Zeroes From String Using Regular Expression

Java provides an easy and efficient way to remove leading zeroes from a string using regular expression. A regular expression, commonly known as regex, is a pattern used to match and manipulate strings. In this article, we will discuss how to remove leading zeroes from a string with the help of a few Java examples. Let's get started.

Example 1: Using replaceFirst() Method

The 
replaceFirst()
method can be used to remove leading zeroes from a given string. In this example, we take the string ‘00004001’ and use the 
replaceFirst()
method to remove the leading zeroes, resulting in the string ‘4001’. 

          
           
// sample string 
String s = "00004001"; 

// regular expression to remove leading zeroes 
String regex = "(^0+)"; 

// remove leading zeroes 
String result = s.replaceFirst(regex, ""); 

// print output 
System.out.println(result); 

// output: 4001

Pattern.compile()
