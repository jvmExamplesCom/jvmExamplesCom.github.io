Splitting Strings by Words Using Java

Java provides a powerful set of tools for working with strings. Splitting strings is a common task, and in some cases it can be difficult to figure out how to approach the challenge. In this article, we'll look at how to split strings by words in Java.

Using the split() Method

The simplest way to split a string in Java is to use the built-in 

          
           
split()
method. This method takes in a delimiter such as a space, comma, or other character, and splits the string into an array based on that delimiter. For example, say you have a string like this: 

          
           
String str = "This is an example string";
You could split this string using whitespace as a delimiter like so: 

          
           
String[] words = str.split("\\s+");
This will create an array of words, each element of which contains a single word from the string. The output would look like this: 

          
           
[
    "This",
    "is",
    "an",
    "example",
    "string"
]

Using a Regular Expression

A slightly more complex way to split a string is to use a regular expression. For example, say you had a string like this: 

          
           
String str = "This;is,an;example-string";
In this case, the delimiter isn't just whitespace, it's a combination of characters such as 

          
           
;
, 

          
           
,
, and 

          
           
-
. To split this string, you would use a regular expression like this: 

          
           
String[] words = str.split("[;,\\-]");
This will create an array of words, each element of which contains a single word from the string. The output would look like this: 

          
           
[
    "This",
    "is",
    "an",
    "example",
    "string"
]

Using a StringTokenizer

Finally, you can also split strings using the 

          
           
StringTokenizer
class. This class is useful if you need to split a string based on multiple different delimiters. For example, say you had a string like this: 

          
           
String str = "This|is&amp;an@example*string";
To split this string, you could use a 

          
           
StringTokenizer
like this: 

          
           
StringTokenizer tokenizer = new StringTokenizer(str, "|&amp;@*");

while (tokenizer.hasMoreTokens()) {
    System.out.println(tokenizer.nextToken());
}
This code will output each word from the string, one line at a time. The output would look like this: 

          
           
This
is
an
example
string

Conclusion

In this article, we looked at how to split strings by words in Java. We looked at three different ways to approach this task: using the 

          
           
split()
method, using a regular expression, and using the 

          
           
StringTokenizer
class. With these techniques, you should have no trouble splitting strings in Java.
