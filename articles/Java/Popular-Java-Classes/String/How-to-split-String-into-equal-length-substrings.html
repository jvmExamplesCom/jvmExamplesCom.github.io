Splitting a String into Equal Length Substrings in Java

Splitting a string into equal length substrings is a common task in many programming languages and is particularly useful when working with strings of an unknown length. In Java, there are several ways to do this including using the String.substring() method or using regular expressions.

Using the substring() Method

The most straightforward way to split a string into equal length substrings is to use the String.substring() method. This method takes two parameters, the start index and the end index (both inclusive), and returns the substring between these indices. By looping over the length of the full string, we can obtain all of its equal length substrings. For example, if we have a string of length 6, and we want to split it into two substrings of length 3, we can do something like this: 

          
           
// Create string 
String myString = "Hello!";

// Split string into 2 substrings 
String s1 = myString.substring(0,3);
String s2 = myString.substring(3,6);

System.out.println(s1);
System.out.println(s2);
This will print out 

          
           
Hel
and 

          
           
lo!
which are equal length substrings of our original string.

Using Regular Expressions

Another way to split a string into equal length substrings is to use regular expressions. Regular expressions can be used to find patterns within strings and by writing a simple expression, we can split a string of any length into substrings of equal length. For example, if we have a string of length 6 and we want to split it into two substrings of length 3, we can do something like this: 

          
           
// Create string 
String myString = "Hello!";

// Split string into 2 substrings using regular expression 
String[] substrings = myString.split("(?<=\\G.{3})");

System.out.println(substrings[0]);
System.out.println(substrings[1]);
This will also print out 

          
           
Hel
and 

          
           
lo!
which are the two equal length substrings of our original string.

Conclusion

Splitting a string into equal length substrings is a common task and there are several ways to do this in Java. The most straightforward way is to use the String.substring() method and loop over the length of the full string. Alternatively, regular expressions can be used to find patterns within strings and can be used to split strings of any length into substrings of equal length.
