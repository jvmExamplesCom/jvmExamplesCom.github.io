How to Check If a String Is in Upper Case

Java is a popular programming language with a wide variety of use cases. It is especially useful for processing strings of text, as it has built-in methods that make it easy to check if a string is in uppercase or not. In this article, we will discuss how to check if a string is in upper case and provide code examples for doing so.

Checking For Uppercase with the matches() Method

One of the most straightforward ways to check if a string is in uppercase is to use the 

          
           
matches()
method. This method takes a regular expression as an argument, and returns true if the string matches the expression. To check for uppercase characters, we can pass in a regular expression that checks for any uppercase letters in the string: 

          
           
String s = "MY STRING";
boolean isUpperCase = s.matches("[A-Z]+");
System.out.println(isUpperCase); // true
In this example, we create a 

          
           
String
variable called 

          
           
s
and assign it the value “MY STRING”. We then call the 

          
           
matches()
method on 

          
           
s
, passing in a regular expression that checks for uppercase letters. The method returns 

          
           
true
, indicating that the string contains only uppercase letters.

Checking For Uppercase With the toUpperCase() Method

Another way to check if a string is in uppercase is to use the 

          
           
toUpperCase()
method. This method takes a 

          
           
String
as an argument and returns a new 

          
           
String
with all the characters converted to uppercase. We can use this method to check if a string is already in uppercase by comparing the original 

          
           
String
to the result of 

          
           
toUpperCase()
: 

          
           
String s = "my string";
boolean isUpperCase = s.equals(s.toUpperCase());
System.out.println(isUpperCase);  // false
In this example, we create a 

          
           
String
called 

          
           
s
and assign it the value “my string”. We then call the 

          
           
toUpperCase()
method on 

          
           
s
and compare the result to the original 

          
           
String
. Since the original 

          
           
String
contains lowercase characters, the result of 

          
           
equals()
is 

          
           
false
.

Conclusion

In this article, we discussed two ways to check if a string is in uppercase in Java. First, we looked at the 

          
           
matches()
method, which takes a regular expression as an argument and returns true if the string matches the expression. Second, we looked at the 

          
           
toUpperCase()
method, which takes a string and returns a new string with all the characters converted to uppercase. With these two methods, we can easily determine if a given string is in uppercase.
