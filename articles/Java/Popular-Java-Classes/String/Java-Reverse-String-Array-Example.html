Java Reverse String Array Example

In this code example, we will be looking at how to reverse an array of strings in Java. We will be using a Java-specific implementation of the ArrayList data structure as well as the Collections class to help us achieve our desired output.

Overview

The goal of this code example is to illustrate how to reverse a string array in Java. We will be using a Java-specific implementation of the ArrayList data structure as well as the Collections class. Creating the ArrayList: We will start by creating the ArrayList that our program will use. We can do so with the following code: 

          
           
ArrayList<String> list = new ArrayList<String>();
Adding Elements to the ArrayList: We will then add some elements to our ArrayList using the .add() method: 

          
           
list.add("One");
list.add("Two");
list.add("Three");
list.add("Four");
list.add("Five");
Reversing the ArrayList: We will then use the Collections.reverse() method to reverse the order of the elements in the ArrayList: 

          
           
Collections.reverse(list);
Printing Out the Results: Finally, we will then use the System.out.println() method to print out the reversed version of the ArrayList: 

          
           
for(String str : list) {
	System.out.println("Reversed Array: " + str);
}

Complete Code Example


          
           
import java.util.ArrayList;
import java.util.Collections;

public class JavaReverseStringArrayExample {
	public static void main(String[] args) {
		ArrayList<String> list = new ArrayList<String>();

		list.add("One");
		list.add("Two");
		list.add("Three");
		list.add("Four");
		list.add("Five");

		Collections.reverse(list);

		for(String str : list) {
			System.out.println("Reversed Array: " + str);
		}
	}
}
