Converting String to Title Case

Title case is a form of writing that capitalizes the first letter of each word in a sentence or phrase. Converting from a normal string to title case involves properly capitalizing each word and ensuring that all punctuation and other characters are in their proper place. In Java, there are several approaches to the task of converting a string to title case.

Using the ReplaceAll() Method

The easiest approach to converting a string to title case is to use the 

String.replaceAll()
method. This method takes two parameters: a regular expression that specifies the pattern to search for, and a replacement string that will be applied to any matches of the specified regular expression. To convert a string to title case, we can use the following code snippet: 

          
           
String str = "this is a sentence";
String titleCase = str.replaceAll("(\\w)(\\w*)", 
    s -> s.substring(0,1).toUpperCase() + s.substring(1).toLowerCase());

In this code snippet, we define a regular expression that looks for alphabetic characters (

\\w
) followed by zero or more other alphabetic characters ( 
\\w*
). These matches are then replaced by a Lambda expression which converts the first character of the matched string to uppercase and the remainder to lowercase. Once this is done, the resulting string will be in title case:

Using the WordUtils Java Library

If you don't want to create your own string conversion logic, you can use an existing library such as Apache Commons Lang's 

WordUtils
. This library provides a number of useful methods for working with strings, including a method for converting strings to title case. To use this library, you first need to add the library to your project: 

          
           
<dependency>
    <groupId>org.apache.commons</groupId>

WordUtils.capitalizeFully()

String str = "this is a sentence";

This will produce the same result as the previous example.

Using Character Iteration

Finally, you can also convert a string to title case using a loop to iterate over the characters in the string and capitalize them as needed. This approach may be useful if you want to customize the conversion process and only capitalize certain types of words, such as proper nouns. Here is an example of how to do this:


          
           
String str = "this is a sentence";
StringBuilder sb = new StringBuilder();
boolean capitalizeNext = true;

for (char c : str.toCharArray()) {
    if (Character.isLetterOrDigit(c)) {
        if (capitalizeNext) {
            sb.append(Character.toUpperCase(c));
            capitalizeNext = false;
        }
        else {
            sb.append(Character.toLowerCase(c));
        }
    }
    else {
        sb.append(c);
        capitalizeNext = true;
    }
}
String titleCase = sb.toString();

The above code will iterate over each character in the string and capitalize it if it is an alphabetic character. All other characters will be left as-is. The resulting string will be in title case.
