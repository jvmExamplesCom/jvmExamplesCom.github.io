Java String toUpperCase() Example

The Java String 

          
           
toUpperCase()
method is used to convert any string to uppercase letters. This method does not affect the original string.

Syntax


          
           
public String toUpperCase();

Example

Let's look at a simple example: 

          
           
String str = "sample string"; 
 
System.out.println(str.toUpperCase());
This code will print out the following output: 

          
           
SAMPLE STRING
In this example, we created a 

          
           
String
variable called 

          
           
str
and assigned it a value of 

          
           
"sample string"
. We then called the 

          
           
toUpperCase()
method on the 

          
           
str
variable, which converted all the characters of the 

          
           
String
to uppercase letters.

s

toUpperCase()

toUpperCase()
method.
