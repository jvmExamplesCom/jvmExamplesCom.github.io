Java String Replace Example

The Java String replace() is an inbuilt function that returns a new string resulting from replacing all occurrences of oldChar with newChar. The function takes three parameters: the string, oldChar, and newChar. For example, consider the following line of code: 

          
           
String str = "John Smith";
str.replace(' ', '_');
This code will replace all whitespace characters in the string with underscores. Let us look at some examples to understand the concept better.

Example 1: Simple String Replace

In this example, we will perform a simple string replace operation. 

          
           
public class Test {
    public static void main(String[] args) {
        String str = "John Smith";
        String result = str.replace(' ', '_');
        System.out.println(result);
    }
}
Output: 

          
           
John_Smith

Example 2: Replacement of a Char Sequence

In this example, we will replace a char sequence with another char sequence. 

          
           
public class Test {
    public static void main(String[] args) {
        String str = "John Smith";
        String result = str.replace("John", "Jane");
        System.out.println(result);
    }
}
Output: 

          
           
Jane Smith

Example 3: Replacement of a Char Sequence Using Regex

In this example, we will use regular expression to replace a char sequence with another char sequence. 

          
           
public class Test {
    public static void main(String[] args) {
        String str = "John Smith";
        String result = str.replaceAll("[a-zA-Z]+", "_");
        System.out.println(result);
    }
}
Output: 

          
           
_ _
