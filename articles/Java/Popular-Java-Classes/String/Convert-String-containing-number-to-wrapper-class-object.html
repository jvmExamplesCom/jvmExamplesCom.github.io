Converting a String Containing Number to a Wrapper Class Object in Java

Java objects provide a variety of methods that allow developers to convert data from one type to another. One such conversion is converting a String containing numbers to a wrapper class object. This tutorial will cover the basics of how to accomplish this task.

String to Primitive Conversion

Before discussing String to wrapper class object conversion, it is important to first understand how java handles primitive types. Primitives are basic data types such as int and double that are used to represent numerical values. Java provides several methods of casting a String to a primitive type, namely: 
  • 
            
             
Integer.parseInt(String)

            
  • 
            
             
Double.parseDouble(String)

            
  • 
            
             
Long.parseLong(String)

            
  • 
            
             
Float.parseFloat(String)
The following code snippet demonstrates how these methods can be used to cast a String to a primitive type. 

          
           
String str = "123";
int i = Integer.parseInt(str);
System.out.println("Integer Value: " + i);
// Output: Integer Value: 123

String str = "123";
Integer intFromString = new Integer(str);
int i = 123;
Integer intFromPrimitive = new Integer(i);

Conclusion

In summary, Java provides multiple ways of converting a String containing numbers to a wrapper class object.
