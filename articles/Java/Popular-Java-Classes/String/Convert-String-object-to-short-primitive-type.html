Converting a String Object to a Short Primitive Type

The Java programming language provides a set of tools and functions to convert objects into primitives. This tutorial will demonstrate the process of converting a String object to a short primitive type in Java.

Example 1: Converting aString To a Short Primitive Type

To convert a String object to a short primitive type, we can create a Short object, passing the String as a parameter to the constructor. The following code example illustrates this approach: 

          
           
String str = "123";
Short s = new Short(str);
System.out.println(s.shortValue());
The output from this code snippet will be: 

          
           
123

Example 2: Parsing a String Into a Short

We can also parse a String directly into a short using the Short.parseShort() method. The following code example illustrates this approach: 

          
           
String str = "123";
short s = Short.parseShort(str);
System.out.println(s);
The output from this code snippet will be: 

          
           
123

Example 3: Converting a String to a Short Using the ValueOf() Method

We can also convert a String to a short using the Short.valueOf() method. The following code example illustrates this approach: 

          
           
String str = "123";
short s = Short.valueOf(str);
System.out.println(s);
The output from this code snippet will be: 

          
           
123
In this tutorial, we have demonstrated three approaches for converting a String object to a short primitive type in Java. We have created a Short object, used the Short.parseShort() method, and used the Short.valueOf() method.
