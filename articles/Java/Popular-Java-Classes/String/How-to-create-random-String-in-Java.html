How to Create Random String in Java

Creating random strings in Java is a helpful tool when working with sorting, encryption, and generating random data. This guide will provide an overview of the process needed to generate a random string in Java.

Creating a Random String

To create a random string in Java, we first create an instance of the 

StringBuilder
class. This is a utility class that allows us to manipulate and build Strings. 

          
           
StringBuilder sb = new StringBuilder();

Next, we need to generate some random characters to add to our String. We can create a random character using the 

random()
method. 

          
           
char c= (char)(random().nextInt(26) + 'a');

'a' - 'z'
. We can use this character to create our random string. 

          
           
for(int i = 0; i < 10; i++) {
    char c = (char)(random().nextInt(26) + 'a');
    sb.append(c);
}
We will then append this character to our 
StringBuilder
instance, looping 10 times to create a 10-character long string. 

          
           
String randomStr = sb.toString();
Finally, we convert our 
StringBuilder

toString()

String

Conclusion
