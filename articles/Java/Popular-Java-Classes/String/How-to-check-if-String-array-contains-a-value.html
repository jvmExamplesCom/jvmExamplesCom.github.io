How to Check if String Array Contains a Value

Java provides many ways to check if an array contains a value. This tutorial explains how to use various approaches to determine whether an array of strings includes a given string.

Using Java 8 Stream API

One of the simplest ways to check if a string array contains a given value is to use the Java 8 Stream API. The following code example shows how to check if a string array contains a given string: 

          
           
String[] arrOfStr = {"Hello", "World"};
boolean contains = Arrays.stream(arrOfStr).anyMatch("World"::equals);
In this example, we create an array of strings and then use the 

          
           
.anyMatch()
method to check if the array contains the given string. The 

          
           
.anyMatch()
method returns true if it finds the given string and false otherwise.

Using for-each Loop

We can also use a for-each loop to check if a string array contains a given value. The following code example shows how: 

          
           
String[] arrOfStr = {"Hello", "World"};
boolean contains = false;

for (String str : arrOfStr) {
    if (str.equals("World")) {
        contains = true;
        break;
    }
}
In this example, we iterate over the array of strings and use the 

          
           
equals()
method to check if the current element is equal to the given string. If it is, then the variable 

          
           
contains
is set to true and the loop is exited.

Using Java Functional Interfaces

We can also use Java functional interfaces to check if a string array contains a given value. The following code example uses a Predicate to check if an array contains a given string: 

          
           
String[] arrOfStr = {"Hello", "World"};
boolean contains = Arrays.stream(arrOfStr).anyMatch(str -> str.equals("World"));
In this example, we create an array of strings and then use the 

          
           
.anyMatch()
method with a predicate to check if the array contains the given string. The 

          
           
.anyMatch()
method returns true if it finds the given string and false otherwise.

Using Java Collections

We can also use Java collections to check if a string array contains a given value. The following code example uses the 

          
           
contains()
method of the 

          
           
List
class to check if an array contains a given string: 

          
           
String[] arrOfStr = {"Hello", "World"};
boolean contains = Arrays.asList(arrOfStr).contains("World");
In this example, we create an array of strings and then use the 

          
           
.contains()
method of the 

          
           
List
class to check if the array contains the given string. The 

          
           
.contains()
method returns true if it finds the given string and false otherwise.

Conclusion

In this tutorial, we have explored several different ways to check if a string array contains a given value in Java. We have discussed how to use the Java 8 Stream API, for-each loop, Java functional interfaces, and Java collections to check if an array contains a given string.
