Java String to java.sql.Date Example

In this tutorial we will be learning how to convert a String to an instance of java.sql.Date using the SimpleDateFormat class. We will start by creating a new instance of the SimpleDateFormat and setting the date format pattern to match the string we wish to convert. Next, we will use the parse() method to parse the string into a java.util.Date object. Finally, we will use the sql Date constructor to create a java.sql.Date object from the java.util.Date object.

Creating a SimpleDateFormat Instance

The first step in converting a String to a java.sql.Date object is to create a new instance of SimpleDateFormat and set the date format pattern. The date format pattern must match the string we wish to convert in order for the parse() method to work correctly. In this example, we will set the date format pattern to ‘yyyy-MM-dd’. 

          
           
SimpleDateFormat sdf = new SimpleDateFormat("yyyy-MM-dd");

Parsing the String

Once we have created a new SimpleDateFormat instance, we can use the parse() method to parse the string into a java.util.Date object. The parse() method takes a String as its argument and returns a java.util.Date object. 

          
           
String str = "2021-05-15";
java.util.Date date = sdf.parse(str);

Creating a java.sql.Date Object

Now that we have parsed the String into a java.util.Date object, we can use the java.sql.Date constructor to create a java.sql.Date object from the java.util.Date object. The java.sql.Date constructor takes a long argument which represents the number of milliseconds since January 1, 1970. We can get this value from the java.util.Date object using the getTime() method. 

          
           
java.sql.Date sqlDate = new java.sql.Date(date.getTime());
