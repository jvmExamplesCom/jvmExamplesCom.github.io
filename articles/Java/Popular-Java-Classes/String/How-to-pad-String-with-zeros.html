Padding a String with Zeroes in Java

Java provides many ways of padding a string with zeroes, depending on the specific use case. In this article, we will discuss each method, as well as provide code examples and output.

Zeros at the Start of a String

If the goal is to pad a string with leading zeroes, use the 

          
           
String.format()
method with the 

          
           
"%0d"
format specifier. The syntax for this method is as follows: 

          
           
String.format("%0nd", var);
Where n represents the minimum length of the output string. For example, if you wanted the output length to be 8 characters, you would use 

          
           
String.format("%08d", var);
. If the length of the string was less than 8 characters, it would be padded with zeroes. If it was longer, it would remain unchanged. Here is a code example: 

          
           
int number = 123;
String result = String.format("%08d", number);
System.out.println(result);
This will produce the following output: 

          
           
00000123

Zeros at the End of a String

If the goal is to pad a string with trailing zeroes, the approach is similar to adding leading zeroes. However, instead of the 

          
           
"%0d"
format specifier, you will need to use the 

          
           
"%-0d"
format specifier. The syntax for this method is as follows: 

          
           
String.format("%-0nd", var);
Where n represents the minimum length of the output string. For example, if you wanted the output length to be 8 characters, you would use 

          
           
String.format("%-08d", var);
. If the length of the string was less than 8 characters, it would be padded with zeroes. If it was longer, it would remain unchanged. Here is a code example: 

          
           
int number = 123;
String result = String.format("%-08d", number);
System.out.println(result);
This will produce the following output: 

          
           
12300000

Alternate Solutions

In addition to the methods outlined above, there are a few alternate approaches that can be used to pad a String with zeroes. These solutions are more generalizable, and can be used to pad a String with any character. One solution is to use the 

          
           
String.replace()
method. This method takes two parameters: the target character or substring, and the replacement character or string. The syntax for this method is as follows 

          
           
String.replace(target, replacement);
For example, if you wanted to replace all occurrences of the letter 'a' with the letter 'b' in the string "Hello World!", you would use 

          
           
String.replace("a", "b");
. Here is an example of how to use this method to pad a String with zeroes: 

          
           
String str = "123";
String padded = str.replace("", "0");
System.out.println(padded);
This will produce the following output: 

          
           
000123

String.format()

"%"

String.format("%x", var);
Where x represents the character or string to be used for padding. For example, if you wanted to pad the output with underscores, you would use 

          
           
String.format("%_s", var);
. Here is an example of how to use this method as a padding solution: 

          
           
int number = 123;
String result = String.format("%_8d", number);
System.out.println(result);

___123
