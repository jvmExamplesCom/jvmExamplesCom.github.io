Convert List to String in Java

Java provides a number of ways to convert a 

          
           
List<String>
to a single string. In this article, we'll go over five different approaches, each with its own advantages and disadvantages.

1. Using StringBuilder

One approach is to use the 

          
           
StringBuilder
class. This is a mutable sequence of characters that allows us to efficiently construct strings by appending characters one by one. We can loop through each element in the list and append it to the 

          
           
StringBuilder
using its 

          
           
append()
method. Finally, we can construct the resulting string using the 

          
           
StringBuilder
's 

          
           
toString()
method. 

          
           
StringBuilder sb = new StringBuilder();
for (String s : list) {
    sb.append(s);
}
String result = sb.toString();

2. Using StringJoiner

Another approach is to use 

          
           
StringJoiner
, which allows us to join strings together with a specified separator. We can loop through each element in the list and call the 

          
           
StringJoiner
's 

          
           
add()
method to add it to the joiner. Finally, we can construct the resulting string using the 

          
           
StringJoiner
's 

          
           
toString()
method. 

          
           
StringJoiner sj = new StringJoiner(",");
for (String s : list) {
    sj.add(s);
}
String result = sj.toString();

3. Using Collectors.joining()

The 

          
           
Collectors.joining()
method is a convenient way to join elements of a 

          
           
Stream
. We can create a 

          
           
Stream
from the input list and use the 

          
           
Collectors.joining()
method to construct the resulting string. 

          
           
String result = list.stream().collect(Collectors.joining(","));

4. Using String.join()

Java 8 introduced the 

          
           
String.join()
static method, which allows us to join a collection of strings with a specified delimiter. We can simply pass in the collection of strings and the delimiter to the 

          
           
String.join()
method to construct the resulting string. 

          
           
String result = String.join(",", list);

5. Using Arrays.toString()

Finally, we can use the 

          
           
Arrays.toString()
static method to convert the input list to a comma-separated string. This approach is the simplest, but it should be used only with lists of strings, not lists of arbitrary objects. 

          
           
String result = Arrays.toString(list.toArray());
