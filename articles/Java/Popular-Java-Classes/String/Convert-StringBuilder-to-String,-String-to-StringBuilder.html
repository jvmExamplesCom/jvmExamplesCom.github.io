Converting StringBuilder to String

In Java, a String is an immutable sequence of characters. A StringBuilder is a mutable sequence used to construct Strings. This article will explain how to convert a StringBuilder to a String.

Using the toString() Method

The most straightforward way to convert a StringBuilder to a String is to use the toString() method. This takes no arguments, and returns the contents of the StringBuilder as a new String object.


          
           
StringBuilder sb = new StringBuilder("Hello");
String str = sb.toString();
System.out.println(str);  // Output: "Hello"

Using the StringBuilder(String str) Constructor

Another way to convert a StringBuilder to a String is to use the StringBuilder(String str) constructor. This constructor takes a String as a parameter and constructs a new StringBuilder object with the same contents as the given String. 


          
           
String str = "Hello";
StringBuilder sb = new StringBuilder(str);
System.out.println(sb.toString());  // Output: "Hello"

Using the append() Method

Another way to convert a String to a StringBuilder is to use the append() method. This method takes a String as a parameter and appends it to the end of the StringBuilder. 


          
           
String str = "Hello";
StringBuilder sb = new StringBuilder();
sb.append(str);
