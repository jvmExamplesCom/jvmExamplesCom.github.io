Convert InputStream to String in Java

Java provides us with various ways to convert an InputStream to a String. Different approaches have different advantages and disadvantages, so let's learn about each of them.

Using Scanner

The easiest way of converting an InputStream to a String is to use the 

          
           
Scanner
class. The following code snippet shows how we can do this: 

          
           
InputStream inputStream = ...;
String text = new Scanner(inputStream).useDelimiter("\\A").next();
The code above creates a 

          
           
Scanner
, passes it the 

          
           
inputStream
and then uses the 

          
           
useDelimiter
method to set the delimiter to the beginning of the string (i.e. "\\A"). Finally, the 

          
           
next
method is used to read the entire contents of the 

          
           
inputStream
into a 

          
           
String
.

Using IOUtils from Apache Commons

If you are using Apache Commons IO library, then you can use the 

          
           
IOUtils.toString
method to easily convert an 

          
           
InputStream
to a 

          
           
String
: 

          
           
InputStream inputStream = ...;
String text = IOUtils.toString(inputStream);
This approach is very simple and easy to use. However, it relies on the Apache Commons IO library which may not always be available.

Using BufferedReader

Another approach to convert an 

          
           
InputStream
to a 

          
           
String
is to use a 

          
           
BufferedReader
. The following code snippet shows how this can be done: 

          
           
InputStream inputStream = ...;
BufferedReader reader = new BufferedReader(new InputStreamReader(inputStream));
StringBuilder out = new StringBuilder();
String line;
while ((line = reader.readLine()) != null) {
    out.append(line);
}
String text = out.toString();
In this approach, we create a 

          
           
BufferedReader
and pass it the 

          
           
InputStreamReader
. We then use a loop to read all lines from the 

          
           
reader
object and append them to a 

          
           
StringBuilder
. Finally, we convert the 

          
           
StringBuilder
to a 

          
           
String
.

Using ByteArrayOutputStream

We can also convert an 

          
           
InputStream
to a 

          
           
String
by first reading its contents into a 

          
           
ByteArrayOutputStream
. The following code snippet shows how this can be done: 

          
           
InputStream inputStream = ...;
ByteArrayOutputStream result = new ByteArrayOutputStream();
byte[] buffer = new byte[1024];
int length;
while ((length = inputStream.read(buffer)) != -1) {
    result.write(buffer, 0, length);
}
String text = result.toString("UTF-8");
In this approach, we create a 

          
           
ByteArrayOutputStream
and read the contents of the 

          
           
inputStream
into it using a loop. Finally, we convert the 

          
           
ByteArrayOutputStream
to a 

          
           
String
by specifying the encoding.

Conclusion

In this article, we have seen various ways of converting an 

          
           
InputStream
to a 

          
           
String
in Java. Each approach has its own advantages and disadvantages, so it is important to choose the right one depending on the requirements.
