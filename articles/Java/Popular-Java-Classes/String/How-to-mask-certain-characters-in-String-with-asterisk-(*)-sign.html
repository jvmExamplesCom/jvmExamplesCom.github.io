How to Mask Certain Characters in String with Asterisk (*) Sign

Java allows strings to be manipulated in many ways. One such method is masking certain characters in a String with an asterisk (*) sign. This can be useful for when you want to hide confidential information such as passwords or bank numbers. In this article, we'll take a look at how to use Java to mask certain characters in a String with an asterisk (*).

Using the replaceAll() Method

The simplest way to mask certain characters in a string with an asterisk (*) sign is by using the replaceAll() method. This method takes two arguments: the regular expression that defines the characters to be replaced and the replacement string (in this case, an asterisk “*”). For example, the following code masks all characters in a string starting from the fifth character onwards: 

          
           
String input = "1234567890";
String output = input.replaceAll("(?<=.{4}).*", "*");
System.out.println(output);
// Output: 1234*****
Here, the regular expression "(?<=.{4}).*" means that all characters after the fourth character should be replaced with an asterisk (*). The output is then “1234*****”, as the first four characters remain intact.

Using the substring() Method

Alternatively, you can also mask certain characters in a string with an asterisk (*) sign by using the substring() method. This method takes two arguments: the start index and the end index of the substring you'd like to return. For example, the following code masks all characters in a string starting from the fifth character (inclusive): 

          
           
String input = "1234567890";
String output = input.substring(0, 4) + "*".repeat(input.length() - 4);
System.out.println(output);
// Output: 1234*****
This code snippet works by first getting the substring of the original string from index 0 (the start index) up to index 4 (the end index). We then append a string of asterisks ("*") to the substring, with the number of asterisks equal to the difference between the length of the original string and the length of the substring (i.e., the total number of characters to be masked). The output is then “1234*****”, as the first four characters remain intact.

Using the replaceFirst() Method

You can also mask certain characters in a string with an asterisk (*) sign using the replaceFirst() method. This method takes two arguments: the regular expression that defines the characters to be replaced and the replacement string (in this case, an asterisk “*”). For example, the following code masks all characters in a string from the sixth character (inclusive) onwards: 

          
           
String input = "1234567890";
String output = input.replaceFirst("(?<=.{5}).*", "*");
System.out.println(output);
// Output: 12345*
This code snippet works by replacing all the characters of the original string from the sixth character (inclusive) onwards with an asterisk (*). The output is then “12345*”, as the first five characters remain intact. In conclusion, you can mask certain characters in a String with an asterisk (*) sign by using the replaceAll(), substring() and replaceFirst() methods in Java. Each of these methods takes different arguments and produces a slightly different result, but they all work to mask certain characters in a String with an asterisk (*) sign.
