Replacing Substring in String Using substring() Method

There are many different ways to replace a substring in a String. One of the most popular and efficient methods is using the Java 

          
           
substring()
method. This tutorial will be outlining what the 

          
           
substring()
method is, how to use it, and providing some examples.

What is the substring() Method?

The 

          
           
substring()
function is a method in Java which extracts a part of a given string and returns that extracted part as a new string. The original string remains unchanged after the 

          
           
substring()
operation. The syntax for the 

          
           
substring()
method is as follows: 

          
           
String substring(int startIndex);
String substring(int startIndex, int endIndex);
The first parameter, 

          
           
startIndex
, is the index from where the extracted string is to begin. The second optional parameter, 

          
           
endIndex
, is the index till where the extracted string should end. If the 

          
           
endIndex
parameter is not mentioned, then the substring returned would be the part of the string starting from the 

          
           
startIndex
till the end of the string.

Example Usage

Now that we understand how to use the 

          
           
substring()
method, let's go through some examples.

Example 1: Substring without End Index

In this example, we look at how to use the 

          
           
substring()
method when we are only specifying the starting index. We have the string 

          
           
"Hello World"
and we want to extract the part from index 4 onwards. In this case, we would use the following code: 

          
           
String str = "Hello World";
String newStr = str.substring(4);
System.out.println("The new string is: " + newStr); // Prints "The new string is: o World"

Example 2: Substring with End Index

Now we look at an example with the 

          
           
substring()
method where we are specifying both the starting and the ending index. Here, we have the same string 

          
           
"Hello World"
but we want to extract the part from index 4 to 8. The code for this would be: 

          
           
String str = "Hello World";
String newStr = str.substring(4, 8);
System.out.println("The new string is: " + newStr); // Prints "The new string is: o Wo"

Conclusion

In this tutorial, we went over how to use the 

          
           
substring()
method in Java to extract a part of a given string. By specifying the starting index, we can easily get the part of the string that we are looking for. We also saw how to specify the ending index so that the returned string has just the desired length.
