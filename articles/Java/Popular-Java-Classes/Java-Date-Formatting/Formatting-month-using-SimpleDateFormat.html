Formatting Month Using SimpleDateFormat

The SimpleDateFormat class in Java provides a formatting mechanism for date and time values. This class can be used to format dates in a specific pattern. It allows developers to specify the format to use when formatting the month portion of a date. This is an essential tool for developing applications that display dates and times in various formats. In this article, we will discuss how to use SimpleDateFormat to format the month portion of a Date.

Using SimpleDateFormat to Format a Month

SimpleDateFormat enables developers to specify a format for dates and times using a pattern string. The pattern string contains letters, symbols, and other characters that indicate the desired format for the date or time. To format the month portion of a date, the pattern string should contain the letter ‘M’.

The letter ‘M’ indicates the month number (1-12). The following example shows a pattern string that contains the letter ‘M’:


          
           
MMMM

This pattern string will result in a date that is formatted like this: ‘January’.

The letter ‘M’ can be used with various optional modifiers. For example, the modifier ‘MMM’ can be used to produce a date that is formatted like this: ‘Jan’. The following example uses the SimpleDateFormat class to format the month portion of a Date:


          
           
import java.text.SimpleDateFormat;
import java.util.Date;

public class Main {

    public static void main(String[] args) {
        Date date = new Date();
        
        SimpleDateFormat sdf = new SimpleDateFormat("MMMM");
        String month = sdf.format(date);
        System.out.println(month);
    }
}

The output of this code snippet would be the name of the current month. For example, if the code is run in January, the output would be ‘January’.
