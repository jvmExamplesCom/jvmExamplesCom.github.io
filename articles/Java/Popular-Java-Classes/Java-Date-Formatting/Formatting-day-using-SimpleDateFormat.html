Formatting Date Using SimpleDateFormat in Java

SimpleDateFormat is a powerful tool in the Java API that allows developers to format and parse dates. SimpleDateFormat follows a specific pattern defined by a String of special characters. It is important to understand the tools and methods available in the SimpleDateFormat class to properly use it.

Using SimpleDateFormat to Format Dates

The most common purpose of SimpleDateFormat is to format dates. This can be done by creating a SimpleDateFormat object with a pattern, then passing a Date object to its format() method. The format() method returns a String of the date formated according to the pattern. Here is an example of formatting a Date object into a MM/dd/yyyy format:


          
           
SimpleDateFormat formatter = new SimpleDateFormat("MM/dd/yyyy");
String formattedDate = formatter.format(date);
In the example above, the 
date
variable is a Date object and the 
formattedDate
variable is a String. The pattern used to instatiate the 
formatter
is "MM/dd/yyyy". This pattern tells SimpleDateFormat to format the Date object into a String of the form "Month/Day/Year".

Using SimpleDateFormat to Parse Dates

SimpleDateFormat can also be used to parse Strings of date into Date objects. To do this, one must create a SimpleDateFormat instance with the same pattern as the String they wish to parse. Then, they can use the parse() method to convert the String into a Date. Here is an example of parsing a String into a Date using a MM/dd/yyyy pattern:


          
           
SimpleDateFormat formatter = new SimpleDateFormat("MM/dd/yyyy");
Date parsedDate = formatter.parse(dateString);
In the example above, the 
dateString
variable is a String of a date and the 
parsedDate
variable is a Date. The pattern used to instantiate the 
formatter
is "MM/dd/yyyy". This pattern tells SimpleDateFormat to parse the String into a Date object in the form "Month/Day/Year".

Conclusion

SimpleDateFormat is an invaluable tool in the Java platform. It allows one to easily format or parse dates. By understanding the different patterns that can be used, along with the different methods availble, one can use SimpleDateFormat to properly format and parse dates.
