Formatting Date in Custom Formats using SimpleDateFormat

The java.text.SimpleDateFormat class is a concrete class for formatting and parsing dates in a locale-sensitive manner. It allows for formatting (date → text), parsing (text → date), and normalization. The formatting and parsing patterns are created from a combination of symbols, letters and other characters.

Overview

SimpleDateFormat can be used to convert a 
Date
or 
Calendar
object into a string representation. This is done using the format method, which accepts a 
Date
object and returns a formatted string representing the date. The format method also accepts a 
Calendar
object. The format method takes a 
String
argument representing a pattern used to convert a 
Date
or 
Calendar
object into a string. The pattern is composed of one or more tokens representing various parts of the date; for example, the letter “m” indicates minutes, and the letter “y” indicates the year. The format method also takes an optional 
Locale
argument, which can be used to customize the formatting of the output depending on the locale. By default, the 
Locale.getDefault()
is used if the 
Locale
argument is not specified.

Examples

Below are some examples of using the 
SimpleDateFormat
class to format and parse dates in different formats. 

          
           
// Formatting a date into a string
SimpleDateFormat sdf = new SimpleDateFormat("yyyy-MM-dd");
String dateString = sdf.format(new Date());
System.out.println(dateString); // 2020-02-01

// Parsing a string into a Date
String dateString = "2011-09-15";
SimpleDateFormat sdf = new SimpleDateFormat("yyyy-MM-dd");
Date date = sdf.parse(dateString);
System.out.println(date); // Fri Sep 15 00:00:00 EDT 2011

Here is an example of how to use the SimpleDateFormat class to format a 

Date
object into a string with a custom format: 

          
           
String dateString = "2011-09-15";
SimpleDateFormat sdf = new SimpleDateFormat("MM/dd/yyyy");
Date date = sdf.parse(dateString);
 
String formattedDateString = sdf.format(date);
System.out.println(formattedDateString); // 09/15/2011
And here is an example of how to format a 
Calendar
object into a string using the same pattern: 

          
           
Calendar calendar = Calendar.getInstance();
calendar.set(2011, 8, 15); // September 15, 2011
 
SimpleDateFormat sdf = new SimpleDateFormat("MM/dd/yyyy");
String formattedDateString = sdf.format(calendar.getTime());
System.out.println(formattedDateString); // 09/15/2011

Common Patterns

The following table lists some common patterns used to format and parse dates and times.

Pattern Description
yyyy-MM-dd Year-Month-Day (e.g. 2016-04-30)
dd/MM/yyyy Day-Month-Year (e.g. 30/04/2016)
MM/dd/yyyy 'at' hh:mm:ss a Month-Day-Year at Hour:Minute:Seconds AM/PM (e.g 05/10/2016 at 02:30:45 PM)
These patterns can be used with either the 
format()
or 
parse()
methods.

Conclusion

The 
SimpleDateFormat
class is a powerful tool for formatting and parsing dates in a locale-sensitive manner. By using custom patterns, it is possible to format and parse dates in a variety of different formats.
