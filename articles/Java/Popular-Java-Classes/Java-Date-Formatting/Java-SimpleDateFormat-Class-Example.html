Java SimpleDateFormat Class Example

The SimpleDateFormat is a class in the Java library used to parse and format dates. It provides a variety of methods to convert date formats from one form to another. In this article, we will take a look at some example usage of the SimpleDateFormat Class.

Creating an SimpleDateFormat Instance

The SimpleDateFormat is created with a constructor that takes two parameters. The first parameter is a string-formatted pattern which is used to parse and format dates, and the second parameter is the locale to be used when parsing the date. 

          
           
SimpleDateFormat sdf = new SimpleDateFormat("MM/dd/yyyy", Locale.US);

Using the SimpleDateFormat Class

Once you have an instance of the SimpleDateFormat class initialized, there are a few methods you can call to format and parse dates.

Formatting a Date

The 

          
           
format()
method takes a Date and returns a String representing the date formatted according to the pattern specified in the constructor. 

          
           
Date date = new Date();
String strDate = sdf.format(date);
// Output: 04/05/2020

Parsing a Date

The 

          
           
parse()
method takes a string and returns a Date object representing the parsed date. This is useful if you have a date stored as a string and need to convert it to a Date object. 

          
           
String strDate = "04/05/2020";
Date date = sdf.parse(strDate);
// Output: Thu Apr 05 00:00:00 EDT 2020

Conclusion

In this article, we looked at example usage of the SimpleDateFormat Class in Java. We learned how to create an instance of the SimpleDateFormat class, as well as how to use the format() and parse() methods to format and parse dates.
