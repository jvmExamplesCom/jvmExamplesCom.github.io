Formatting Hour Using SimpleDateFormat in Java

In Java, the 
SimpleDateFormat
class is used for formatting hours and dates. This class provides various methods to format date into desired format and parse a date string into required date object.

Example

Here is an example of how to format hour using 
SimpleDateFormat
class: 

          
           
SimpleDateFormat sdf = new SimpleDateFormat("HH");
Date date = new Date(); // Get current system date
System.out.println(sdf.format(date));

The above code will print the hour part of the current system date in 24-hour format.

Output


          
           
17

Explaining the Code

In the first line, we are creating a 
SimpleDateFormat
instance with the format "HH". This "HH" denotes that the time should be printed in 24-hour format. Then we are creating a 
Date
instance with the current system date. Finally, we are printing the hour part of the current system date using the 
format()
method of the 
SimpleDateFormat
instance.

Formatting Hours without seconds

If you want to print the hour part of a date excluding the seconds then you can use the following code:


          
           
SimpleDateFormat sdf = new SimpleDateFormat("hh");
Date date = new Date();
System.out.println(sdf.format(date));

The above code will print the hour part of the current system date in 12-hour format.

Output

Formatting Hours with milliseconds

If you want to print the hour part of a date including the milliseconds then you can use the following code:


          
           
SimpleDateFormat sdf = new SimpleDateFormat("HH:mm:ss.SSS");
Date date = new Date();
System.out.println(sdf.format(date));

The above code will print the hour part of the current system date in 24-hour format with milliseconds.
