Converting Dates Using SimpleDateFormat in Java

The Java SimpleDateFormat class is a class for formatting and parsing dates for the Java programming language. It allows developers to specify a date string, in one format, and then convert it to another format. In this article, we will explore how to use SimpleDateFormat to convert date strings from one format to another.

Overview of the SimpleDateFormat Class

The SimpleDateFormat class is used to convert strings of date information into Date objects using a specified format. It is also used to format Date objects into strings in a specified format. Typically, these formats are specified using patterns (e.g., yyyy-MM-dd). The SimpleDateFormat class is part of the java.text package and is included in the Java Development Kit (JDK).

The following code snippet demonstrates how to instantiate a SimpleDateFormat object with a given pattern:


          
           
String pattern = "yyyy-MM-dd";
SimpleDateFormat sdf = new SimpleDateFormat(pattern);

Basic Examples of Formatting and Parsing Dates

Once a SimpleDateFormat object is created, it can be used to format Date objects into strings and parse strings into Date objects. The following code snippets demonstrate Examples of both.

Formatting a Date Object Into a String

The following example demonstrates using the SimpleDateFormat's 

format()
method to convert a Date object into a string with the specified pattern: 

          
           
Date dateObj = new Date();
String dateString = sdf.format(dateObj); // "yyyy-MM-dd"
System.out.println(dateString); //prints 2020-05-21

Parsing a String Into a Date Object

The following example demonstrates using the SimpleDateFormat's 

parse()
method to convert a string with the specified pattern into a Date object: 

          
           
String dateString = "2021-03-11";
Date dateObj = sdf.parse(dateString);
System.out.println(dateObj); // prints Thu Mar 11 00:00:00 IST 2021

Converting Strings From One Format To Another

The SimpleDateFormat class can also be used to convert strings of date information from one format to another. For example, let's say we wanted to convert the following string – “May 18, 2020” – into the following format – “18-May-2020”.

To do this, we first need to create two SimpleDateFormat objects, one for each format:


          
           
String fromPattern = "MMM d, yyyy";
String toPattern = "d-MMM-yyyy";
SimpleDateFormat fromSdf = new SimpleDateFormat(fromPattern);
SimpleDateFormat toSdf = new SimpleDateFormat(toPattern);
We can then use the 
parse()
and 
format()
methods to convert the string from one format to the other: 

          
           
String dateString = "May 18, 2020";
Date dateObj = fromSdf.parse(dateString);
String newString = toSdf.format(dateObj);
System.out.println(newString); // prints 18-May-2020

Conclusion

This article has demonstrated how to use the Java SimpleDateFormat class to format Date objects into strings and parse strings into Date objects, as well as how to convert strings of date information from one format to another. Hopefully, you now have a better understanding of this important Java class and how to use it for working with dates in your Java applications.
