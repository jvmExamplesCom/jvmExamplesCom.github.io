Adding AM/PM to Time Using SimpleDateFormat

Time is often represented in 24-hour format (e.g., 17:00), and while this format is great for computers, it's not always easy to read or understand. Fortunately, it's simple to convert time to 12-hour format using the 

SimpleDateFormat
class in Java. This article will explain how to quickly add AM/PM suffix to a given time.

Using the SimpleDateFormat Class

The 
SimpleDateFormat
class provides several methods for formatting and parsing dates. In this case, it's used to create a pattern for formatting the time in 12-hour format with the AM/PM suffix. To use it, you must first create an instance of the 
SimpleDateFormat
class and set the pattern for the desired output. For example: 

          
           
SimpleDateFormat sdf = new SimpleDateFormat("hh:mm a");

The pattern string consists of two parts, the hour portion and the am/pm portion. In this case, the 

hh
indicates that the hour should be formatted in 12-hour format (i.e., 01 - 12). The 
mm
will format the minutes. To add the am/pm suffix, use the 
a
character. Now that the pattern is set, you can create a 
java.util.Date
object for the given time and use the 
format
method to convert it to a String with the AM/PM suffix: 

          
           
// Create a Date object for the given time
Date date = new Date(17,00);

// Get the formatted string with AM/PM suffix
String formattedTime = sdf.format(date);

The result will be a string containing the time in 12-hour format with the AM/PM suffix appended to it (e.g., 05:00 PM).

Example

Here is a complete example that formats a given time in 12-hour format with the AM/PM suffix using the 

SimpleDateFormat
class: 

          
           
import java.text.SimpleDateFormat;
import java.util.Date;

public class Main {
  public static void main(String[] args) {
    // Set the output pattern
    SimpleDateFormat sdf = new SimpleDateFormat("hh:mm a");

    // Create a Date object for the given time
    Date date = new Date(17,00);

    // Get the formatted string with AM/PM suffix
    String formattedTime = sdf.format(date);

    // Print the result
    System.out.println("Formatted time: " + formattedTime);
  }
}

For the given time of 17:00, the output of this program will be:


          
           
Formatted time: 05:00 PM
