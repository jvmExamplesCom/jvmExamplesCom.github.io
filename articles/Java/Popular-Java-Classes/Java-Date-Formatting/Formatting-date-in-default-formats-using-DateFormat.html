Formatting Date in Default Formats Using DateFormat in Java

Java has been a powerful programming language since its inception. It provides a variety of tools and classes to facilitate formatting dates into various user-defined formats. The java.text.DateFormat utility class is one such tool which provides various static methods to format dates into standard, localized formats.

In this article, we will discuss the major aspects of DateFormat, like the default formats it supports, how to parse and format a date string, and other useful methods. We’ll also look at some code examples for each of these topics.

Default Formats

The default formats provide an easy way to format dates without having to manually define or construct a DateFormat instance. The DateFormat class defines four constants for the commonly used date formats:


          
           
FULL  // full date e.g., Tuesday, April 12, 1952 AD or 3:30:42pm PST
    LONG  // long date e.g., January 12, 1952
    MEDIUM // medium date e.g., Jan 12, 1952
    SHORT // short date e.g., 12/13/52

Each of these constants is sent as an argument to the static method getDateInstance() from the DateFormat class. The getDateInstance() method takes two arguments; the first is constant representing the style of the format (i.e., FULL, LONG, MEDIUM, or SHORT) and the second argument is an optional Locale to set for the date field. Let’s see an example of how to use the getDateInstance() method with the default formats:


          
           
DateFormat df = DateFormat.getDateInstance(DateFormat.FULL);
System.out.println(df.format(new Date());
// Output: Tuesday, October 27, 2020

Using the same method, you can also specify a Locale to set to the date field. For example, if you want to format a date in French, you can pass Locale.FRENCH as the second argument to the getDateInstance() method. This will yield the following output:


          
           
DateFormat df = DateFormat.getDateInstance(DateFormat.FULL, Locale.FRENCH);
System.out.println(df.format(new Date()));
// Output: mardi 27 octobre 2020

Parsing and Formatting a Date String

The DateFormat class also provides several methods to parse and format date strings. The parse() method takes a String object containing a date and parses it into a Date object. The syntax for using this method is as follows:


          
           
DateFormat df = DateFormat.getDateInstance();
Date date = df.parse(dateString);

The other utility method is the format() method which takes a Date object and formats it into a String. The syntax for this method is:


          
           
DateFormat df = DateFormat.getDateInstance();
String str = df.format(date);
Let’s look at an example to see how this works: 

          
           
String dateString = "20 Apr 2018";
DateFormat df = DateFormat.getDateInstance();

Date date = df.parse(dateString);
System.out.println(date);
// Output: Wed Apr 20 00:00:00 PDT 2018

String str = df.format(date);
System.out.println(str);
// Output: Apr 20, 2018

Other Useful Methods

The DateFormat class also provides several other useful methods which can be used to customize the format of the date string. These methods include:

  • setLenient() – this method sets the leniency of the date format; true indicates that the format should be lenient and false indicates that it should be strict.
  • setTimeZone() – this method sets the time zone of the date format; the argument should be a valid TimeZone instance.
  • setCalendar() – this method sets the calendar of the date format; the argument should be a valid Calendar instance.

Let’s take a look at an example of how to use these methods together:


          
           
String dateString = "20 Apr 2018";
DateFormat df = DateFormat.getDateInstance();
df.setLenient(false);
df.setTimeZone(TimeZone.getTimeZone("America/Los_Angeles"));
df.setCalendar(Calendar.getInstance(TimeZone.getTimeZone("America/Los_Angeles")));

Date date = df.parse(dateString);
System.out.println(date);
// Output: Wed Apr 20 03:00:00 PDT 2018

String str = df.format(date);
System.out.println(str);
// Output: Apr 20, 2018

In this example, we have set the time zone and calendar of the date format to America/Los_Angeles and enabled the leniency of the date format. As a result, the date is correctly parsed and formatted to the correct time zone.

Conclusion

In this article, we discussed the various aspects of the DateFormat class in Java. We looked at the default formats it supports, how to parse and format a date string, and other useful methods. We also saw some code examples which demonstrated how to use these features.

I hope these examples have provided you with a good understanding of the DateFormat class and how it can be used to format dates into various user-defined formats.
