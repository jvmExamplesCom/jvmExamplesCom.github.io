Formatting TimeZone using SimpleDateFormat

SimpleDateFormat is a concrete class for formatting and parsing dates in a locale-sensitive manner. It allows for formatting (date ​–​ text), parsing (text ​–​ date), and normalization. It can be used to format dates in a variety of ways, such as displaying a localized version with the day, month, and year, as well as for setting a particular timezone.

Example 1: Setting a Specific Time Zone

The following example shows how to set the time zone for a SimpleDateFormat instance. 


          
           
SimpleDateFormat sdf = new SimpleDateFormat("yyyy-MM-dd HH:mm:ss");
sdf.setTimeZone(TimeZone.getTimeZone("GMT"));
String dateStr = sdf.format(new Date());
System.out.println(dateStr);  // Outputs: 2017-04-14 22:09:27

Example 2: Formatting with UTC Time Zone

The following example shows how to format a given date in UTC time zone.


          
           
SimpleDateFormat sdf = new SimpleDateFormat("yyyy-MM-dd HH:mm:ss");
sdf.setTimeZone(TimeZone.getTimeZone("UTC"));
String dateStr = sdf.format(new Date());
System.out.println(dateStr);  // Outputs: 2017-04-14 20:09:27

Example 3: Formatting with Different Time Zones

The following example shows how to format a given date with different time zones.


          
           
//Create an instance of SimpleDateFormat used for formatting 
//the string representation of date according to the chosen pattern
SimpleDateFormat dateFormatter = new SimpleDateFormat("dd-MM-yyyy HH:mm:ss z");

//Create timezones 
TimeZone india = TimeZone.getTimeZone("Asia/Kolkata"); 
TimeZone germany = TimeZone.getTimeZone("Europe/Berlin"); 

//Set the timezone 
dateFormatter.setTimeZone(india); 
String indiaTime = dateFormatter.format(new Date()); 
dateFormatter.setTimeZone(germany); 
String germanyTime = dateFormatter.format(new Date()); 

//Print the timezones 
System.out.println("India Time: " + indiaTime);
System.out.println("Germany Time: " + germanyTime); 

// Outputs: 
// India Time: 15-04-2017 23:39:43 IST
// Germany Time: 15-04-2017 02:09:43 CEST

Example 4: Parsing a String to a Date Object

The following example shows how to parse a string to a Date object.


          
           
String str = "12/01/2018";
try {
    Date date1 = new SimpleDateFormat("MM/dd/yyyy").parse(str);
    System.out.println(date1);  // Outputs: Fri Dec 01 00:00:00 UTC 2017
