Parsing custom formatted date string into Date object using SimpleDateFormat

Java provides a class – 

          
           
SimpleDateFormat
– for parsing and formatting dates. This class is useful for converting strings in custom date formats into Date objects. In this article, we’ll explore how to use the 

          
           
SimpleDateFormat
class to parse a custom formatted date string into a Date object.

Creating an Instance of SimpleDateFormat

Before we can parse a custom date string, we need to create an instance of the 

          
           
SimpleDateFormat
class. We can do this by calling the constructor, passing in a string that contains the format of our date string. The following examples show two different strings with the format specifier “yyyy-MM-dd” which means that the date string should contain four digits for the year, two digits for the month, and two digits for the day: 

          
           
SimpleDateFormat dateFormat = new SimpleDateFormat("yyyy-MM-dd");

          

          
           
SimpleDateFormat dateFormat = new SimpleDateFormat("yyyy/MM/dd");
In the above examples, both strings specify exactly the same format, but they use different delimiters (the hyphen – in the first example, and the slash / in the second).

Parsing a Custom Date String

Once we have an instance of the 

          
           
SimpleDateFormat
class, we can use the 

          
           
parse()
method to parse a date string in the specified format into a Date object. For example: 

          
           
String dateString = "2020-07-29";
Date date = dateFormat.parse(dateString);
System.out.println(date); // Output: Wed Jul 29 00:00:00 EDT 2020
The 

          
           
parse()
method will throw an exception if it is unable to parse the input string. So it’s important to wrap the 

          
           
parse()
call in a try/catch block, as shown in the following example: 

          
           
try {
    String dateString = "2020-07-29";
    Date date = dateFormat.parse(dateString);
    System.out.println(date); // Output: Wed Jul 29 00:00:00 EDT 2020
} catch (ParseException e){
    System.err.println("Error parsing date string: " + e);
}

Matching Multiple Date Formats

Sometimes you may need to parse a date string that could be in one of many different formats. The 

          
           
SimpleDateFormat
class makes it easy to handle this situation by allowing you to specify multiple patterns in a single instance. For example, the following code snippet creates an instance of the 

          
           
SimpleDateFormat
class that can parse date strings in either the “yyyy-MM-dd” or “MM/dd/yyyy” format: 

          
           
SimpleDateFormat dateFormat = new SimpleDateFormat("yyyy-MM-dd|MM/dd/yyyy");
Parsing a date string with this instance is the same as before: 

          
           
try {
    String dateString = "07/29/2020";
    Date date = dateFormat.parse(dateString);
    System.out.println(date); // Output: Wed Jul 29 00:00:00 EDT 2020
} catch (ParseException e){
    System.err.println("Error parsing date string: " + e);
}

Specifying a Custom Locale

By default, the 

          
           
SimpleDateFormat
class uses the default locale of the system. If you need to parse dates in a different locale, you can specify a custom locale when creating the instance of the 

          
           
SimpleDateFormat
class. For example: 

          
           
Locale locale = new Locale("fr", "FR");
SimpleDateFormat dateFormat = new SimpleDateFormat("dd/MM/yyyy", locale);
The above example creates an instance of the 

          
           
SimpleDateFormat
class that will parse dates in the French locale. You can also use the 

          
           
setTimeZone()
method to specify a custom time zone.

Conclusion

In this article, we explored how to use the 

          
           
SimpleDateFormat
class in Java to parse a custom formatted date string into a Date object. We looked at how to create an instance of the class, as well as how to parse a date string with the 

          
           
parse()
method. We also discussed how to match multiple date formats, and how to specify a custom locale.
