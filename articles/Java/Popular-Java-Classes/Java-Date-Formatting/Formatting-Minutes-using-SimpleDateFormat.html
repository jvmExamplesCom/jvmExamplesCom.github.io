Formatting Minutes using SimpleDateFormat

Java provides a utility class "SimpleDateFormat" that allows developers to format dates. It also allows formatting a date in the form of minutes. In this article, we will discuss how to format minutes using Java's SimpleDateFormat class.

Using SimpleDateFormat

The SimpleDateFormat class provides a method called "format()" which can be used to format a date in the form of minutes: 

          
           
Date date = new Date();
SimpleDateFormat simpleDateFormat = new SimpleDateFormat("mm");
String formattedMinutes = simpleDateFormat.format(date);
System.out.println("Formatted minutes: " + formattedMinutes);
In this example, we create a new "Date" object and then create an instance of the SimpleDateFormat class with the pattern "mm", which is used to format the date into minutes. We then call the "format()" method on our SimpleDateFormat instance and pass in the Date object as an argument. This returns a String representation of the minutes which can then be printed to the console.

Output

The output of the above code will be: 

          
           
Formatted minutes: 21
This is the result of the code when executed. The number represents the current minute of the given date.

Conclusion

In this article, we discussed how to format minutes using Java's SimpleDateFormat class. We looked at the format() method which takes a Date object as an argument and returns a String representing the current minute of the date.
