Formatting Day of Week using SimpleDateFormat

Java provides several ways to format dates and times. One of the most commonly used classes for formatting dates is SimpleDateFormat. SimpleDateFormat allows users to specify the date format to display the day of the week.

Example 1: Formatting Day of Week with Default Pattern

The following example shows how to use the SimpleDateFormat class to display the day of the week with the default pattern: 

          
           
SimpleDateFormat sdf = new SimpleDateFormat();
System.out.println(sdf.format(new Date()));
This will print out the day of the week as two characters: "Fr" for Friday, "Sa" for Saturday, and so on.

Example 2: Formatting Day of Week with Custom Pattern

You can also format the day of the week using a custom pattern. To do this, you need to specify the pattern when creating the SimpleDateFormat instance: 

          
           
String pattern = "EEEE";
SimpleDateFormat sdf = new SimpleDateFormat(pattern);
System.out.println(sdf.format(new Date()));
This will print out the full name of the day of the week: "Friday", "Saturday", and so on.

Example 3: Formatting Day of Week with Different Locales

SimpleDateFormat also allows you to specify the locale when formatting the date. This can be useful if you need to display the day of the week in different languages. For example, to display the day of the week in French, you can specify the locale when creating the SimpleDateFormat instance: 

          
           
String pattern = "EEEE";
Locale locale = Locale.FRANCE;
SimpleDateFormat sdf = new SimpleDateFormat(pattern, locale);
System.out.println(sdf.format(new Date()));
This will print out the day of the week in French: "vendredi", "samedi", and so on.

Conclusion

In this article, we have seen how to use SimpleDateFormat to format the day of the week. We have also seen how to use custom patterns and different locales to display the day of the week in different ways.
