Formatting Seconds Using SimpleDateFormat in Java

The 

          
           
SimpleDateFormat
class in Java can be used to format seconds. It provides a formatting pattern to be applied on a Date object that would represent a time value in milliseconds. In this article, we will discuss how this is done, with some code examples. Before we get started, let's look at how to convert seconds into milliseconds. To do this, we simply need to multiply the value by 1000: 

          
           
long milliseconds = seconds * 1000;
Now, with the milliseconds value obtained, we can use the 

          
           
SimpleDateFormat
class to format it. Here is an example of how you can format milliseconds into a simple date string: 

          
           
SimpleDateFormat sdf = new SimpleDateFormat("yyyy-MM-dd HH:mm:ss");
String formattedDate = sdf.format(milliseconds);
The string passed into the 

          
           
SimpleDateFormat
constructor ("yyyy-MM-dd HH:mm:ss") is known as a formatting pattern. This particular pattern will generate a date string that looks like this: "2020-10-30 12:25:45". Below are some of the commonly used formatting patterns: 

          
           
d - day in month (number) 
dd - day in month (2 digits, padded with 0 if needed)
M - month in year (number)
MM - month in year (2 digits, padded with 0 if needed)
y - year (2 digits)
yyyy - year (4 digits)
h - hour in am/pm (1-12)
hh - hour in am/pm (2 digits, padded with 0 if needed)
H - hour in day (0-23)
HH - hour in day (2 digits, padded with 0 if needed)
m - minute in hour
mm - minute in hour (2 digits, padded with 0 if needed)
s - second in minute 
ss - second in minute (2 digits, padded with 0 if needed)
a - am/pm marker
So, for example, if you wanted to format the milliseconds value in the form of "Saturday, October 30th, 2020 12:25 PM", the formatting pattern would be: 

          
           
SimpleDateFormat sdf = new SimpleDateFormat("EEEE, MMMM dd', 'yyyy h:mm a");
String formattedDate = sdf.format(milliseconds);
You can find out more about 

          
           
SimpleDateFormat
and its formatting patterns in the official java documentation page. Below is a complete example of formatting seconds using SimpleDateFormat: 

          
           
public class TimeFormatter {
    public static void main(String[] args) {
        // convert seconds to milliseconds 
        long seconds = 3600;
        long milliseconds = seconds * 1000;

        // create a SimpleDateFormat object and format the milliseconds value 
        SimpleDateFormat sdf = new SimpleDateFormat("yyyy-MM-dd HH:mm:ss");
        String formattedDate = sdf.format(milliseconds);

        // print the result 
        System.out.println(formattedDate); 
    }
}
The output of the above code is: 

          
           
1970-01-01 01:00:00
As you can see, formatting seconds using SimpleDateFormat can be quite easy. However, there may be times when you need a more complex formatting pattern. In that case, you can refer to the official java documentation page to figure out which formatting pattern to use.
