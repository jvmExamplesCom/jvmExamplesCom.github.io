Java StringTokenizer – Specify Delimiter example

In this tutorial, you will learn how to use the StringTokenizer class to tokenize a given string using a specified delimiter. To demonstrate this, we will use the following example.


          
           
String str = "This-is-a-string";

In the above example, the delimiter is -. We will use the StringTokenizer class to tokenize the string into individual words.

StringTokenizer Syntax

The syntax of the StringTokenizer class is as follows:


          
           
StringTokenizer st = new StringTokenizer(str, delimiter);
  • str: This is the input string to be tokenized.
  • delimiter: This is the delimiter which is used to separate the individual words from the input string.

Example Program

Let us now write a java program to demonstrate the use of the StringTokenizer class.


          
           
public class Example {
   public static void main(String[] args) {
      String str = "This-is-a-string";
      String delimiter = "-";
      StringTokenizer st = new StringTokenizer(str, delimiter);
      
      while (st.hasMoreTokens()) {
         System.out.println(st.nextToken());
      }
   }
}

In the above program, we have defined a string str and a delimiter delimiter. We then use the StringTokenizer class to create an object st. We use the method hasMoreTokens() to check if there are more tokens in the string and use the method nextToken() to get the next token from the string. This loop will continue until all the tokens are processed.
