Java StringTokenizer – Return Delimiters as Tokens example

Example with ReturnDelimiters flag set to true

The following code example shows the usage of 

          
           
StringTokenizer
with the 

          
           
returnDelimiters
flag set to true. This flag when set to true, the delimiter characters are also returned as tokens. 

          
           
public static void main(String args[]) {
   String str = "This is a test ! Okay?";
   StringTokenizer st = new StringTokenizer(str, " !", true);
   System.out.println("---- Split by space and ! ----");
   while (st.hasMoreElements()) {
      System.out.println(st.nextElement());
   }
}
When run, this code produces the following output:

This

is

a

test

!

Okay?

Example with ReturnDelimiters flag set to false

Let’s see the same example but with flag 

          
           
returnDelimiters
set to false. 

          
           
public static void main(String args[]) {
   String str = "This is a test ! Okay?";
   StringTokenizer st = new StringTokenizer(str, " !", false);
   System.out.println("---- Split by space and ! ----");
   while (st.hasMoreElements()) {
      System.out.println(st.nextElement());
   }
}
Produced output when run:

Conclusion

The example above showcases the usage of Java's 

          
           
StringTokenizer
class with the 

          
           
returnDelimiters
flag. Setting this flag to 

          
           
true
will also return the delimiters as tokens, setting it to 

          
           
false
will omit the delimiters.
