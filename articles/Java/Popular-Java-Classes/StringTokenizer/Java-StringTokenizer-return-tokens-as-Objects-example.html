Introduction to Java StringTokenizer Object

Java StringTokenizer is a very useful utility class for tokenizing a delimited string. In this article, we will look at how to return tokens as objects from the StringTokenizer utility class. We will provide code examples and their corresponding output to illustrate this concept.

Using the StringTokenizer Class

To use the StringTokenizer class in Java, we must first create an instance of the class. We can do so by providing a delimited string and a specified set of delimiters to the constructor. For example, let's create a StringTokenizer object that represents the string “A,B,C,D” with a comma (',') as the delimiter: 

          
           
String str = "A,B,C,D";
StringTokenizer st = new StringTokenizer(str, ",");
The StringTokenizer object will now represent our string as an array of four tokens, each separated by a comma.

Returning Tokens as Objects

We can now return our tokens as objects using the hasMoreTokens() and nextToken() methods of the StringTokenizer class. 

          
           
while (st.hasMoreTokens()) {
    String token = st.nextToken();
    System.out.println(token);
}
The above code snippet will print out each token in our delimited string. The output should look like this: 

          
           
A
B
C
D

Additional Examples

Let's take a look at another example, this time we'll create a StringTokenizer object that represents the string “ABCD” with no specified delimiter. 

          
           
String str = "ABCD";
StringTokenizer st = new StringTokenizer(str);
Since we didn't specify any delimiters, the StringTokenizer object will now represent our string as a single token. We can once again return this token as an object using the hasMoreTokens() and nextToken() methods: 

          
           
while (st.hasMoreTokens()) {
    String token = st.nextToken();
    System.out.println(token);
}
The output should look like this: 

          
           
ABCD
