Java StringTokenizer Count Tokens Example

The 

          
           
StringTokenizer
class allows the user to break up a given string into tokens. These tokens can then be used for various purposes. In this example, we will demonstrate how to use a tokenizer to count the number of tokens in a given string.

Example Code

Let's say we have the following string: 

          
           
String testString = "This is the example string";
We can create a 

          
           
StringTokenizer
object and set the string as the parameter: 

          
           
StringTokenizer tokenizer = new StringTokenizer(testString);
To get the total number of tokens, we can use the 

          
           
countTokens()
method on the tokenizer object: 

          
           
int totalTokens = tokenizer.countTokens();
We can then print the result to the console: 

          
           
System.out.println("Number of tokens: " + totalTokens);

Example Output

Below is the output of the code example: 

          
           
Number of tokens: 5
