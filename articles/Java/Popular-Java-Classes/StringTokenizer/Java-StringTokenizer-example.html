Java StringTokenizer Example

StringTokenizer is a powerful Java class to split strings into tokens. It is part of the java.util package and allows us to tokenize strings according to certain delimiters. In this article, we will be looking at different StringTokenizer examples in Java.

Creating a StringTokenizer Object

We can instantiate a StringTokenizer object by passing the string and the delimiter in its constructor. For example: 

          
           
StringTokenizer st = new StringTokenizer("Hi my name is John"," ");
This will create a StringTokenizer object st which tokenizes the string "Hi my name is John" according to the delimiter " ".

StringTokenizer Methods

We can use various methods of the StringTokenizer object to work with the tokenized string. Let's look at some of the commonly used methods.

hasMoreTokens()

This method returns true if there are more tokens present in the tokenized string. 

          
           
boolean hasMoreTokens = st.hasMoreTokens();

nextToken()

This method returns the next token in the tokenized string. 

          
           
String nextToken = st.nextToken(); // this will return the first token "Hi"

countTokens()

This method returns the number of tokens in the tokenized string. 

          
           
int tokenCount = st.countTokens(); // this will return 5 as there are five tokens in the tokenized string.

Example Program

Let's look at an example program that uses the StringTokenizer class. Here, we will tokenize a given string using the space character as the delimiter. We will then iterate over the tokens and print them out. 

          
           
public class StringTokenizerExample {
    public static void main(String[] args) {
        String str = "Hi my name is John";

        // Create a StringTokenizer object
        StringTokenizer st = new StringTokenizer(str," ");

        System.out.println("Tokenizing the string...");
        while (st.hasMoreTokens()) {
            String token = st.nextToken();
            System.out.println(token);
        }
    }
}
