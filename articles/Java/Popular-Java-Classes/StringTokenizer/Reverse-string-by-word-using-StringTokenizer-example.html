Reverse String by Word Using StringTokenizer Example

Java provides an easy way to reverse a string by word. The StringTokenizer class allows you to split the string into tokens and then reverse the order of the tokens. This example will walk through an example of reversing a string by word using the StringTokenizer class.

The Code

The code for this example is very simple. All that is needed is to create a StringTokenizer object, get the tokens from the string and then use a StringBuffer to reverse the order of the tokens. Here is the code: 

          
           
String inputString = "This is a sample string"; 
 StringTokenizer st = new StringTokenizer(inputString); 
 StringBuffer sb = new StringBuffer(); 
 
 while (st.hasMoreTokens()){ 
     sb.insert(0, st.nextToken()+ " "); 
 } 
 
 System.out.println(sb.toString().trim());

The Output

The output of the above code is as follows: 

          
           
string sample a is This

Explanation

The code starts by creating a 

          
           
StringTokenizer
object and passing the string 

          
           
"This is a sample string"
as an argument. The tokenizer then splits the string into individual tokens. The tokens are then stored in a 

          
           
StringBuffer
. A loop is used to get each token from the tokenizer and then insert it at the beginning of the string buffer. This causes the tokens to be reversed. Finally, the 

          
           
toString()
method is called to convert the string buffer to a string and the 

          
           
trim()
method is called to remove any trailing whitespace. The resulting string is then printed to the console.
