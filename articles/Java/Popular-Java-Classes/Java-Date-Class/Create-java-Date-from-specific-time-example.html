Create Java Date from Specific Time Example

Java is a very useful and versatile programming language. It can be used to create a variety of applications, from simple web pages to complex distributed systems. One of the core features of Java is its Date class. The Date class allows developers to create and manipulate dates and times in a variety of ways. In this article, we'll look at how to use Java's Date class to create a date object from a specific time.

Creating a Date Object

The Date class provides several constructors that can be used to create a Date object. To create a Date object from a specific datetime, you can use the following constructor: 

          
           
Date d = new Date(long millis);
This constructor takes a single argument, which is a long representing the number of milliseconds since January 1, 1970 00:00:00 GMT. For example, to create a Date object for noon on May 20th, 2020, you could use the following code snippet: 

          
           
long time = 1590021600000L;
Date d = new Date(time);
System.out.println(d); // prints: Wed May 20 12:00:00 GMT 2020
This code creates a Date object with the specified time, and then prints it out.

Formatting the Date

Once you have created the date object, you may want to format it in a specific way. You can do this using the DateFormat class. The DateFormat class provides several methods for formatting Dates, as well as methods for parsing Dates. To format a Date in a specific format, you can use the following method: 

          
           
String format(Date d, String pattern);
This method takes two arguments, the first being the Date object to format, and the second being a String representing the desired output format. For example, to format the Date created above in the "dd/MM/yy" format, you could use the following code: 

          
           
DateFormat df = new SimpleDateFormat("dd/MM/yy");
String dateString = df.format(d);
System.out.println(dateString); // prints: 20/05/20
This code creates a SimpleDateFormat object and uses it to format the Date object. It then prints out the formatted Date string.

Conclusion
