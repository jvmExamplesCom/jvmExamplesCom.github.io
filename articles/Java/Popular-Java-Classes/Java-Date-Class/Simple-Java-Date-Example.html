Simple Java Date Example

In this article, we will discuss the basics of working with the Java Date object and provide several examples of how to manipulate and display dates. We will cover constructing dates, manipulating them, formatting them for output, and also compare two different date objects.

Creating a Date Object

The Date class in Java is powerful and contains a lot of methods for manipulating dates. To create a Date object, you must use the new keyword. 

          
           
Date date = new Date();
This will create a Date object that holds the current time at the time of instantiation.

Manipulating a Date Object

Sometimes you may need to manipulate a Date object, such as adding or subtracting days, months, or years. This can be done using the Calendar class. 

          
           
Calendar cal = Calendar.getInstance();
cal.add(Calendar.DATE, 3);
Date newDate = cal.getTime();
This example adds 3 days to the current date and stores the result in the newDate object. You can also use the Calendar class to subtract days, months, or years from a Date object.

Formatting Date for Output

Once you have created and/or manipulated your Date object, you may need to format it for output. The DateFormat class can help with this. There are several different types of formats that you can use, such as “MM/dd/yyyy”, “MMM dd, yyyy”, or “dd-MMM-yyyy”. 

          
           
DateFormat df = new SimpleDateFormat("dd-MMM-yyyy");
String output = df.format(date);
This example code will format the date object for output in the “dd-MMM-yyyy” format and store the formatted date in the output string.

Comparing Two Date Objects

It is sometimes useful to compare two Date objects to determine which one is older, newer, or the same. This can be done using the compareTo method. 

          
           
int result = date1.compareTo(date2);
This code will compare the two Date objects and return an integer value. If the result is 0, then the two dates are the same. If the result is negative, then date1 is older than date2. If the result is positive, then date1 is newer than date2.

Conclusion

This article has covered the basics of working with Dates in Java. We discussed constructing and manipulating Date objects, formatting them for output, and also comparing two different date objects. There are many more features and methods available for the Date and Calendar class, so be sure to read the documentation for more information.
