Converting Date into Milliseconds with Java

Java is a useful programming language that can be used to convert dates into milliseconds in order to save data and track changes over time. This guide will explain how to use the Java Date class to convert dates into milliseconds and provide code examples for better understanding.

Creating a Date Instance

Before we can begin converting a date into milliseconds, we must first create a Date instance. To do this, we will utilize the java.util.Date class and call its constructor passing in the year, month and day of the date. We'll also need to create an instance of the GregorianCalendar class in order to set our Date instance. 

          
           
GregorianCalendar calendar = new GregorianCalendar(year, month, day);
Date date = calendar.getTime();

Getting the Time in Milliseconds

Once we have a Date instance, we can use it to get the time in milliseconds. To do this, we will use the getTime() method of the Date class. This will return the number of milliseconds since January 1, 1970 UTC (also known as the Unix Epoch). 

          
           
long millis = date.getTime();

Displaying the Result in Milliseconds

The last step is to display the result in milliseconds. To do this, we can use the System.out.println() method and pass in our millis variable. 

          
           
System.out.println("Time in milliseconds: " + millis);

Example Program Output

Let's take a look at an example program that utilizes the concepts we just discussed. This program will take in a year, month and day and then output the time in milliseconds of that specified date. 

          
           
import java.util.Date;  
import java.util.GregorianCalendar;  

public class DateToMillis {  
   public static void main(String[] args) {  
       int year = 2019;  
       int month = 10;  
       int day = 28;  
       
       GregorianCalendar calendar = new GregorianCalendar(year, month, day);  
       Date date = calendar.getTime();  
       
       long millis = date.getTime();  
       
       System.out.println("Time in milliseconds: " + millis);  
   }  
}
In this program, we have used the GregorianCalendar and Date classes to create a Date instance which we then passed into the getTime() method of the Date class. This yielded us the number of milliseconds since the Unix epoch for our specified date. When we run this code, we should get the following output: 

          
           
Time in milliseconds: 1572225200000

Conclusion

In this guide, we looked at how to use Java to convert dates into milliseconds. We discussed the necessary classes to use and created a sample program that demonstrated our understanding. With this knowledge, you should be able to convert any date into milliseconds in your Java projects.
