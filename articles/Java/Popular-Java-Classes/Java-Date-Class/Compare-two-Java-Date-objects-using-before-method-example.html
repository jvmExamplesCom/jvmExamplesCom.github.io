Comparing Java Date Objects Using the Before Method

The 
java.util.Date
class provides methods for easily comparing two date objects. One such method is 
before()
, which returns 
true
if the invoking Date object is before the specified Date object, and 
false
otherwise. In this article, we will look at an example of using the 
before()
method in Java.

Example

In this example, we will create two Date objects and compare them using the 

before()
method. The code is given below: 

          
           
import java.util.Date;

public class Main {
    public static void main(String[] args) {
        Date date1 = new Date(2020, 1, 2, 10, 15, 0);
        Date date2 = new Date(2020, 1, 3, 10, 15, 0);

        boolean isBefore = date1.before(date2);

        System.out.println(isBefore);
    }
}
In this code, we first have imported the 
java.util.Date
class. Then, we have created two Date objects, 
date1
and 
date2
. We have initialized them with different values. Finally, we have used the 
before()
method to compare them. This returns 
true
since 
date1
is before 
date2
. When we run this code, we get the following output: 

          
           
true
