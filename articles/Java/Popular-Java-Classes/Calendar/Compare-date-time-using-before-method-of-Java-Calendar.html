Comparing Date Time in Java Using the before Method of Calendar

Java has a number of classes, methods and interfaces for dealing with date and time operations. This article examines how to compare two dates using the 

before
method of the 
Calendar
class.

The Calendar Class

The 
Calendar
class is an abstract base class for generating calendar fields and performing calendar arithmetic. It has a number of public and protected abstract methods that should be overridden to provide concrete implementations of the calendar. The 
before
method is one such method which is used to determine if one Date object occurs before another. It takes two arguments - the first is a 
Calendar
object representing the date to compare against, and the second is a 
Date
object representing the date to be tested.

Example Usage

In this example, we will compare two dates to see which one occurs before the other.


          
           
Date date1 = new Date("2021-01-15");
Date date2 = new Date("2021-02-15");

Calendar cal = Calendar.getInstance();
cal.setTime(date1);

if (cal.before(date2)) {
    System.out.println("Date1 occurs before Date2");
} else {
    System.out.println("Date2 occurs before Date1");
}
The above code will print out 

          
           
Date1 occurs before Date2
, since 
date1
is occurring before 
date2
.

Conclusion

The 
before
method of the 
Calendar
class makes it easy to compare dates to determine which one occurs before or after the other. It is important to ensure that the correct parameters are passed to the 
before
method, and that the dates passed in are valid.
