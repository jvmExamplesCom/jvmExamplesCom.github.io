Get Current Date Time Values Using Java Calendar

Java Calendar is a class of the JavaDate package, which provides an easy way to obtain date and time values in Java applications. It contains all the necessary information such as year, month, day, hour, minute, second, millisecond, and so on. The Calendar class can be used to get the current date and time values in different formats. However, this article focuses on getting the current date and time values in Gregorian format.

Example 1 - Using the getInstance() Method

The simplest way to get access to the current date and time values is to use the getInstance() method of the Calendar class. This method returns an instance of the Calendar class with default time zone and locale. 

          
           
Calendar calendar = Calendar.getInstance();
int year = calendar.get(Calendar.YEAR);
int month = calendar.get(Calendar.MONTH) + 1;
int date = calendar.get(Calendar.DATE);
int hour = calendar.get(Calendar.HOUR);
int min = calendar.get(Calendar.MINUTE);
int sec = calendar.get(Calendar.SECOND);

System.out.println("Current date: " + year + "/" + month + "/" + date);
System.out.println("Current time: " + hour + ":" + min + ":" + sec);
The output of the above code will be: 

          
           
Current date: 2019/6/15
Current time: 11:28:2

Example 2 - Using the getTimeInMillis() Method

The getTimeInMillis() method is used to retrieve the time of a Date object in milliseconds. The value returned is the number of milliseconds since January 1, 1970, 00:00:00 GMT. 

          
           
Calendar calendar = Calendar.getInstance();
long millis = calendar.getTimeInMillis();

System.out.println("Time in Milliseconds: " + millis);
The output of the above code will be: 

          
           
Time in Milliseconds: 1560510882945

Example 3 - Using the getTime() Method

The getTime() method is also used to retrieve the time of a Date object in milliseconds. It returns a Date object that represents the time. 

          
           
Calendar calendar = Calendar.getInstance();
Date date = calendar.getTime();

System.out.println("Current Date Time: " + date);
The output of the above code will be: 

          
           
Current Date Time: Sat Jun 15 11:28:42 IST 2019

Conclusion

In this article, we discussed how to get the current date and time values using Java Calendar. We looked at three different ways of doing this: by using the getInstance() method, the getTimeInMillis() method and the getTime() method. Each of these methods has its own advantages and disadvantages, so it is important to choose the right one based on your needs.
