Adding or Subtracting Weeks to Current Date Using Java Calendar

Java's 

          
           
Calendar
class makes it easy to add or subtract weeks to the current date. This tutorial will demonstrate different methods to achieve this goal.

Getting the Current Date

To begin, we must get the current date. We can do this using the 

          
           
Calendar
class's static method 

          
           
Calendar.getInstance()
. This will return a 

          
           
Calendar
instance with the current system date. 

          
           
Calendar calendar = Calendar.getInstance();

Adding or Subtracting Weeks with Add or Subtract Method

The easiest way to add or subtract weeks to a date is to use 

          
           
Calendar
's 

          
           
add()
or 

          
           
subtract()
methods. The first argument specifies the field to be adjusted (in this case, 

          
           
Calendar.WEEK_OF_YEAR
). The second argument specifies the number of weeks to add/subtract. Adding or subtracting a positive number will move the date forward; adding or subtracting a negative number will move it backward. 

          
           
//Add one week
calendar.add(Calendar.WEEK_OF_YEAR, 1);

//Subtract two weeks
calendar.subtract(Calendar.WEEK_OF_YEAR, 2);

Adding or Subtracting Weeks with Set Method

Alternatively, you can use the 

          
           
Calendar
's 

          
           
set()
method to set the date directly. To do this, you must specify the 

          
           
Calendar.WEEK_OF_YEAR
field and the desired week-of-year (which ranges from 1 to 53). 

          
           
//Set to 1st week of year 
calendar.set(Calendar.WEEK_OF_YEAR, 1);

//Set to 10th week of year
calendar.set(Calendar.WEEK_OF_YEAR, 10);

Adding or subtracting Weeks with Roll Method

The 

          
           
Calendar
class also provides a 

          
           
roll()
method which can be used to add or subtract weeks. The first argument specifies the field to be adjusted (in this case 

          
           
Calendar.WEEK_OF_YEAR
). The second argument specifies the number of weeks to add/subtract. Unlike the 

          
           
add()
and 

          
           
subtract()
methods, the 

          
           
roll()
method does not modify the fields that follow. 

          
           
//Add one week
calendar.roll(Calendar.WEEK_OF_YEAR, 1);

//Subtract two weeks
calendar.roll(Calendar.WEEK_OF_YEAR, -2);

Conclusion

This tutorial has demonstrated three different methods to add or subtract weeks to the current date using the Java 

          
           
Calendar
class. With each method, you can easily add as many weeks as you need.
