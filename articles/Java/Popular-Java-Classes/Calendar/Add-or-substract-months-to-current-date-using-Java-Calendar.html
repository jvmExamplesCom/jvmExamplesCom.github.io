Adding or Subtracting Months to the Current Date Using Java Calendar

Java Calendar is a convenient way to work with dates and times in Java. It provides a set of classes, such as 

          
           
Calendar
, 

          
           
GregorianCalendar
and 

          
           
SimpleDateFormat
, that can be used to calculate dates, times, and intervals. With this library, it's possible to add or subtract months from the current date. In this article, we'll look at how to accomplish this using Java Calendar.

Creating a Date Object

To work with dates in Java, first, we need to create a 

          
           
Date
object, which contains the current timestamp. We can do this using the following code: 

          
           
Date date = new Date();
Alternatively, if we want to specify a specific date rather than relying on the current time, we can do so using the following code: 

          
           
// Create a date object for August 14th, 2020
Date date = new Date(2020, 7, 14);

Creating a Calendar Object

Once we have the date object, we can create a 

          
           
Calendar
object from it, which will give us access to the Calendar API's methods for working with dates and times. To do this, we can use the following code: 

          
           
Calendar calendar = Calendar.getInstance();
calendar.setTime(date);

Adding Months to a Date

To add months to our date, we can use the 

          
           
add()
method of the 

          
           
Calendar
object. This method takes two arguments: a field specifying the part of the date that we want to modify (in our case, this will be 

          
           
Calendar.MONTH
) and an integer specifying the number of units we want to add. For example, the following code adds 5 months to the current date: 

          
           
calendar.add(Calendar.MONTH, 5);
We can also use negative integers with this method to subtract months from the date. The following code subtracts 3 months from the date: 

          
           
calendar.add(Calendar.MONTH, -3);

Getting the Modified Date

Once we've added or subtracted the desired number of months, we can get the modified date using the 

          
           
getTime()
method of the 

          
           
Calendar
object. This method returns a 

          
           
Date
object containing the modified timestamp. We can use this as follows: 

          
           
Date modifiedDate = calendar.getTime();

Formatting the Modified Date

If we want to display the modified date in a human-readable format, we can use the 

          
           
SimpleDateFormat
class to do so. This class allows us to specify the output format using a string of symbols. For example, the following code formats a date as “MMM d yyyy” (e.g., “Aug 14 2020”): 

          
           
SimpleDateFormat formatter = new SimpleDateFormat("MMM d yyyy");
String formattedDate = formatter.format(modifiedDate);

Example

Putting all of this together, the following example adds or subtracts months from the current date: 

          
           
import java.util.*;
import java.text.*;

public class DateExample {

    public static void main(String[] args) {

        // Get the current date
        Date date = new Date();

        // Create a Calendar object from the date
        Calendar calendar = Calendar.getInstance();
        calendar.setTime(date);

        // Add or subtract months, as desired
        calendar.add(Calendar.MONTH, 5);

        // Get the modified date
        Date modifiedDate = calendar.getTime();

        // Display the modified date in a human-readable format
        SimpleDateFormat formatter = new SimpleDateFormat("MMM d yyyy");
        String formattedDate = formatter.format(modifiedDate);
        System.out.println(formattedDate);
    }
}
This example will print out the date after 5 months have been added to the current date.
