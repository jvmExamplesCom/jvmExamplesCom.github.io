Display Month of Year Using Java Calendar

Java provides the Calendar class. This class helps in managing date and time through various methods. As the name suggests, the Calendar class is used to perform operations like calculating the day of the week, or displaying the month of the year. In this article, let's explore how to display the month of a particular year in Java.

Creating an instance of Calendar

The Calendar class provides various overloaded constructors that let us create a new instance of the class using different attributes. We can create a Calendar instance from a Date object or from a TimeZone object. Here is an example of both the cases: 

          
           
// Creating Calendar instance from Date object
Date currentDate = new Date();
Calendar calendar = Calendar.getInstance(currentDate);

// Creating Calendar instance from TimeZone object
TimeZone timeZone = TimeZone.getDefault();
Calendar cal = Calendar.getInstance(timeZone);

Using the getDisplayName() Method

Now, let's explore the getDisplayName() method. This method is used to get the full name of the month. For example, it will return “January” for the first month of the year. It expects two parameters – a field and a style. Here is an example of this method: 

          
           
// Get the month of the year
Calendar calendar = Calendar.getInstance();
int month = calendar.get(Calendar.MONTH); // Returns 0-11

// Get the full name of the month
String monthName = calendar.getDisplayName(Calendar.MONTH, Calendar.LONG, Locale.US);
System.out.println("Current Month: " + monthName);

//Output
// Current Month: January
As shown above, we first used the get() method with the MONTH parameter to get the numerical representation of the current month. We then passed the MONTH parameter along with the LONG style and the US locale to the getDisplayName() method. This returned us the full name of the month.

Using the getShortName() Method

The getShortName() method is used to get the abbreviation of the month name. It takes the same parameters as the getDisplayName() method. Here is an example: 

          
           
// Get the month of the year
Calendar calendar = Calendar.getInstance();
int month = calendar.get(Calendar.MONTH); // Returns 0-11

// Get the abbreviation of the month
String monthName = calendar.getShortName(Calendar.MONTH, Calendar.SHORT_FORMAT, Locale.US);
System.out.println("Current Month: " + monthName);

//Output
// Current Month: Jan
As you can see, we used the SHORT_FORMAT style while calling the getShortName() method. This gave us the abbreviation of the month.

Conclusion

In this article, we explored how to display the month of the year in Java. We saw how to use the getDisplayName() and getShortName() methods with various parameters to get the full name and abbreviation of the month respectively.
