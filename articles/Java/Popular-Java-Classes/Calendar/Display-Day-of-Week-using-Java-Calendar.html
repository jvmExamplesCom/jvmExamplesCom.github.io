Display Day of Week using Java Calendar

Java Calendar is a powerful tool for displaying the day of the week. Using the class and methods of the Java Calendar class, you can easily display the day of the week with a simple program. In this article, we will look at some code examples to display the day of the week using Java Calendar.

Code Examples 1: Using getInstance() Method

In this example, we will use the getInstance() method to generate an instance of the java.util.Calendar, and then use its get() method to retrieve the day of the week. This example will look like this:


          
           
// Create an instance of java.util.Calendar
Calendar calendar = Calendar.getInstance();

// Retrieve the day of the week
int dayOfWeek = calendar.get(Calendar.DAY_OF_WEEK);

System.out.println("today is:" + dayOfWeek);
This code will output the following: 

          
           
today is: 4

The above code is a simple way to display the day of the week in a program.

Code Examples 2: Using Date Class

In this example, we will use the Date class to display the day of the week. The Date class is a convenient way to display the current time. Here is the example:


          
           
// Create an instance of java.util.Date
Date date = new Date();

Calendar cal = Calendar.getInstance();
cal.setTime(date);

// Retrieve the day of the week
int dayOfWeek = cal.get(Calendar.DAY_OF_WEEK);

System.out.println("today is:" + dayOfWeek);
This code will output the following: 

          
           
today is:4

The above code is an efficient way to display the day of the week in a program.

Code Examples 3: Using SimpleDateFormat

In this example, we will use SimpleDateFormat to format a string representation of the day of the week. This example will look like this:


          
           
// Create an instance of java.text.SimpleDateFormat
SimpleDateFormat sdf = new SimpleDateFormat("EEE");

// Retrieve the day of the week as a string
String dayOfWeek = sdf.format(new Date());

System.out.println("today is:" + dayOfWeek);
This code will output the following: 

          
           
today is:Wed

The above code is useful to display the day of the week in a more compact manner.

Conclusion

In this article, we have looked at three different code examples to display the day of the week using Java Calendar. We have used the getInstance() method to generate an instance of the java.util.Calendar, used the Date class to display the day of the week, and used SimpleDateFormat to format a string representation of the day of the week. All three examples provide an easy way to display the day of the week in a program.
