Java Calendar Example

This example shows how to use the Java Calendar class to construct a simple calendar application. It demonstrates the use of various 

Date
and 
Calendar
classes, and how to format the output using 
SimpleDateFormat
.

Initial Setup

We start by initializing our 
Calendar
object with the current date and time information. Then, we create a 
SimpleDateFormat
object, which will format the date and time data correctly. 

          
           
Calendar calendar = Calendar.getInstance();
SimpleDateFormat dateFormat = new SimpleDateFormat("dd-MMM-yyyy HH:mm:ss");

Print Current Date/Time Information

Next, we print out the current date and time information in a human-readable format.


          
           
System.out.println("Current Date/Time: "+dateFormat.format(calendar.getTime()));

Add/Subtract Time from Current Date/Time

We can add/subtract individual units of time (day, month, year, etc.) from the current date/time using the 

add()
and 
roll()
methods. 

          
           
// add 1 day
calendar.add(Calendar.DAY_OF_MONTH, 1);
System.out.println("Date after 1 day: "+dateFormat.format(calendar.getTime()));

// subtract 10 seconds
calendar.roll(Calendar.SECOND, -10);
System.out.println("Date after subtracting 10 seconds: "+dateFormat.format(calendar.getTime()));

Set Specific Date/Time

We can also set the date/time to a specific value.


          
           
calendar.set(Calendar.YEAR, 2017);
calendar.set(Calendar.MONTH, 0);
calendar.set(Calendar.DAY_OF_MONTH, 1);
calendar.set(Calendar.HOUR_OF_DAY, 0);
calendar.set(Calendar.MINUTE, 0);
calendar.set(Calendar.SECOND, 0);
System.out.println("Date Set: "+dateFormat.format(calendar.getTime()));

Get Specific Date/Time Info

We can also get the value for a specific unit of time (day, month, year, etc.).


          
           
int day = calendar.get(Calendar.DAY_OF_MONTH);
int month = calendar.get(Calendar.MONTH);
int year = calendar.get(Calendar.YEAR);
System.out.println("Day = "+day+", Month = "+month+", Year = "+year);
