Adding or Subtracting Years to the Current Date Using Java Calendar

For applications that handle, process, and store dates, the Java Calendar is a great tool for manipulating and working with dates. The most common use case for the Java Calendar is to add or subtract years from the current date. This article will discuss how to use the Java Calendar to add or subtract years from a given date.

Basics of the Java Calendar

The Java Calendar class is an abstract class that provides methods for converting between a specific instant in time and a set of calendar fields such as YEAR, MONTH, DAY_OF_MONTH, HOUR, and so on, and for manipulating the calendar fields, such as getting the date of the next week.

The static method 

          
           
Calendar.getInstance()
returns a Calendar object initialized with the current date and time in the default locale and timezone.

Example 1: Adding Years to the Current Date with the Java Calendar

In this example, we will use the Calendar class to add two years to the current date.


          
           
Calendar now = Calendar.getInstance();
  now.add(Calendar.YEAR, 2);
  System.out.println("Current date plus two years: " + now.getTime());

The output of this code would look something like this:


          
           
Current date plus two years: Tue Dec 31 17:32:54 EST 2019

Example 2: Subtracting Years from the Current Date with the Java Calendar

In this example, we will use the Calendar class to subtract two years from the current date.


          
           
Calendar now = Calendar.getInstance();
  now.add(Calendar.YEAR, -2);
  System.out.println("Current date minus two years: " + now.getTime());

The output of this code would look something like this:


          
           
Current date minus two years: Tue Dec 31 16:32:54 EST 2017

Conclusion

In this article, we have discussed how to use the Java Calendar class to add or subtract years from the current date. We have also seen some sample code for both cases. With the help of the Calendar class, you can easily manipulate dates and perform date-related operations in your Java applications.
