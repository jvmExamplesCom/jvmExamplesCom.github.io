Adding and Subtracting Minutes to Current Time Using Java Calendar

Java offers an extensive collection of classes and methods for manipulating dates and times - making it a great language for working with time data. One useful class is the 

java.util.Calendar
, which can be used to easily add or subtract minutes from the current time.

Using Calendar to Add Minutes to the Current Time

To add minutes to the current time, you first need to create a 

Calendar
object. This can be done by calling the static 
getInstance()
method: 

          
           
Calendar calendar = Calendar.getInstance();
Once you have your 
Calendar
object, you can use the 
add()
method to add minutes to it. This method takes two parameters - the type of field to add (in this case, 
Calendar.MINUTE
) and the number of minutes to add: 

          
           
calendar.add(Calendar.MINUTE, 5);

The above code will add 5 minutes to the current time represented by the 

Calendar
object. To get the resulting date and time, you can call the 
getTime()
method: 

          
           
Date date = calendar.getTime();

Using Calendar to Subtract Minutes from the Current Time

Subtracting minutes from the current time is similar to adding minutes. You create a 

Calendar
object and then call the 
add()
method, but this time with a negative value: 

          
           
calendar.add(Calendar.MINUTE, -5);

This will subtract 5 minutes from the current time represented by the 

Calendar
object. As before, you can get the resulting date and time by calling the 
getTime()
method: 

          
           
Date date = calendar.getTime();

Example Program

Let's look at an example program that adds and subtracts minutes from the current time. This program creates a 

Calendar
object and then adds 5 minutes to it and subtracts 10 minutes from it, printing out the resulting dates and times: 

          
           
import java.util.Calendar;
import java.util.Date;

public class TimeExample {
    public static void main(String[] args) {
        // Create a Calendar object
        Calendar calendar = Calendar.getInstance();

        // Add 5 minutes
        calendar.add(Calendar.MINUTE, 5);
        Date date1 = calendar.getTime();
        System.out.println("After adding 5 minutes: " + date1);

        // Subtract 10 minutes
        calendar.add(Calendar.MINUTE, -10);
        Date date2 = calendar.getTime();
        System.out.println("After subtracting 10 minutes: " + date2);
    }
}

When run, this program produces the following output:


          
           
After adding 5 minutes: Tue Nov 17 21:02:12 EST 2020
After subtracting 10 minutes: Tue Nov 17 20:57:12 EST 2020
As you can see, the 
Calendar
class provides an easy way to add or subtract minutes to or from the current time in Java.
