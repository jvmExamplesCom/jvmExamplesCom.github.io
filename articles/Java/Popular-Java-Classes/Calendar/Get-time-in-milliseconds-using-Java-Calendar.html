Get Time in Milliseconds Using Java Calendar

Java is widely used for many applications and its Calendar class provides several methods to get and manipulate date information. One use of this is to get the current time in milliseconds since the 1970 epoch. In this article, we will look at how to use the Java Calendar class to get current time in milliseconds since the 1970 epoch in Java.

Using the Current Time

The easiest way to get the current time in milliseconds since the 1970 epoch is to use the 

          
           
getTimeInMillis()
method of the Java Calendar class. This method returns the current time as a long value representing the number of milliseconds since January 1, 1970, 00:00:00 GMT. Here is a quick example of how to use the 

          
           
getTimeInMillis()
method: 

          
           
Calendar calendar = Calendar.getInstance();
long timeInMillis = calendar.getTimeInMillis(); 
System.out.println("Current time in milliseconds : " + timeInMillis);
When we run this program, it prints the following output: 

          
           
Current time in milliseconds : 1567012265443
As you can see, the output is the current time in milliseconds since the 1970 epoch.

Getting Time From Epoch

We can also use the 

          
           
setTimeInMillis()
method of the Java Calendar class to set the time from a given epoch. This is useful if we need to get a specific time in milliseconds since the 1970 epoch. Here is an example of how to use the 

          
           
setTimeInMillis()
method: 

          
           
// epoch of Jan 1, 1970 0:00:00 GMT
long epoch = 0;

Calendar calendar = Calendar.getInstance();
calendar.setTimeInMillis(epoch);
long timeInMillis = calendar.getTimeInMillis(); 
System.out.println("Epoch time in milliseconds : " + timeInMillis);
When we run this program, it prints the following output: 

          
           
Epoch time in milliseconds : 0
As you can see, the output is the epoch time in milliseconds since the 1970 epoch.

Conclusion

In this article, we looked at how to use the Java Calendar class to get the current time in milliseconds since the 1970 epoch. We saw how to use the 

          
           
getTimeInMillis()
and 

          
           
setTimeInMillis()
methods to get and set the time from a given epoch.
