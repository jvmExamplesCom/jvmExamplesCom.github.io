Comparing Date Time Using After Method of Java Calendar

The 
after()
method of the 
Calendar
class in Java is used to compare two dates and check if one date follows the other. This method returns a boolean value, either 
true
or 
false
, depending on whether or not the first date is after the second date.

To illustrate this method, consider the following code example:


          
           
import java.util.Calendar;
...
Calendar firstDate = Calendar.getInstance(); // today's date
Calendar nextWeek = Calendar.getInstance();
nextWeek.add(Calendar.DAY_OF_YEAR, 7); // next week's date

boolean isFirstDateAfterNextWeek = firstDate.after(nextWeek);
System.out.println(isFirstDateAfterNextWeek);
In this example, the 
after()
method is invoked with two 
Calendar
objects as parameters. The first object ( 
firstDate
) holds today's date, while the second object ( 
nextWeek
) holds a date that is 7 days ahead. The result of the comparison is assigned to a boolean variable – if today's date follows the seventh day from today, the 
after()
method will return 
true
. The
