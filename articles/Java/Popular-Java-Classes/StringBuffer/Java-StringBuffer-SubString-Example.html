Java StringBuffer SubString Example

The StringBuffer class in Java is used to store character strings for manipulation. It provides a range of methods that allow developers to create, modify and manipulate strings. One of the most powerful of the StringBuffer methods is the substring() method which allows a developer to take a portion of their String and return it as a new String object. In this article we'll take a look at how to use the substring() method with a Java StringBuffer object.

Create a StringBuffer

Before we can work with the substring() method, we first need to create a Java StringBuffer object. The following example creates a new StringBuffer containing the phrase "Hello World!": 

          
           
StringBuffer sb = new StringBuffer("Hello World!");

Using the substring() Method

Once we have our StringBuffer object, we can use the substring() method to create a new string from the existing one. This is done by specifying two parameters - the start index and the end index - which define the range of characters to use. For example, the following line of code takes the phrase from our original StringBuffer and extracts the word "World" from it: 

          
           
String world = sb.substring(6,11);
The substring() method starts the extraction at the first index and ends at the second index (the second index is not included in the result). In this case, the first index of 6 corresponds to the letter W in the original phrase and the second index of 11 corresponds to the exclamation mark at the end of the phrase.

Example Output

Here's the output of running the code above: 

          
           
World

Specifying Only the Start Index

In addition to the two parameters, the substring() method also provides an overloaded version where only the start index needs to be specified. Doing so will return the remaining characters from the start index until the end of the String. The following example uses the overload version of the substring() method to extract the remaining characters of the original phrase, starting from the letter W: 

          
           
String world = sb.substring(6);

Example Output

The output of running the code above will be: 

          
           
World!

Conclusion

The substring() method of the StringBuffer class is a powerful tool for manipulating strings in Java. It allows developers to extract a section of their original string and return it as a new String object. In this article, we looked at how to use this method to extract characters from a StringBuffer object.
