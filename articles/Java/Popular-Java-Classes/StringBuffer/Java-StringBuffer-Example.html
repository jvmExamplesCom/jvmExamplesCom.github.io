Java StringBuffer Example

Introduction

Java StringBuffer class is used to create an in-memory mutable sequence of characters. It is related to the java.lang.String class, but it is much more powerful and efficient. With the use of the StringBuffer class, you can add, insert, remove, and replace characters and substrings in a string. Let's take a look at some of the different methods of the StringBuffer class.

Creating StringBuffer Object

To create a StringBuffer object, you simply need to use the constructor. For example: 

          
           
StringBuffer buf = new StringBuffer();
This creates an empty StringBuffer object. You can also pass a String as a parameter to the constructor to create a StringBuffer object with a given string. For example: 

          
           
StringBuffer buf = new StringBuffer("Java");
This will create a StringBuffer object that contains the string "Java".

Appending Strings to a StringBuffer Object

You can append strings to the end of a StringBuffer object using the 

          
           
append()
method. For example: 

          
           
buf.append(" is awesome!");
This will add the string " is awesome!" to the end of the StringBuffer object.

Inserting Strings into a StringBuffer Object

The 

          
           
insert()
method can be used to insert strings into a StringBuffer object at any position. For example: 

          
           
buf.insert(0, "Hello, ");
This will insert the string "Hello, " at the beginning of the StringBuffer object.

Replacing Strings in a StringBuffer Object

The 

          
           
replace()
method can be used to replace strings within a StringBuffer object. For example: 

          
           
buf.replace(0, 5, "Hi,");
This will replace the first five characters (the "Hello" string) with "Hi,".

Deleting Characters from a StringBuffer Object

The 

          
           
delete()
method can be used to delete characters from a StringBuffer object. For example: 

          
           
buf.delete(0, 5);
This will delete the first five characters ("Hi, ") from the StringBuffer object.

Reversing a StringBuffer Object

The 

          
           
reverse()
method can be used to reverse the order of characters in a StringBuffer object. For example: 

          
           
buf.reverse();
This will reverse the order of characters in the StringBuffer object.

Example Usage of StringBuffer Class

Here is an example of usage of the StringBuffer class: 

          
           
// Create a new StringBuffer object
StringBuffer buf = new StringBuffer("Java");

// Append a string
buf.append(" is awesome!");

// Insert a string
buf.insert(0, "Hello, ");

// Replace a string
buf.replace(0, 5, "Hi,");

// Delete a string
buf.delete(0, 5);

// Reverse the StringBuffer object
buf.reverse();

// Print the contents of the StringBuffer object
System.out.println(buf.toString());

// Output: !emosewa si avaJ
The above code will create a StringBuffer object containing the string "Java is awesome!", insert the string "Hello, " at the beginning, replace the first five characters with "Hi,", delete the first five characters, reverse the order of characters and then print the contents to the console. The output will be "!emosewa si avaJ".
