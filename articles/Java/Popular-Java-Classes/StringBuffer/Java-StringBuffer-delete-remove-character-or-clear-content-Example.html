Java StringBuffer delete remove character or clear content Example

This article provides a comprehensive look at how to use the 

delete()
, 
remove()
and 
setLength(0)
methods provided by the Java StringBuffer class to delete remove characters or contents of a string.

Using the 
delete()
Method

The 
delete()
method used by the Java StringBuffer class allows you to delete a subset of characters from a string. The method has two parameters, the beginning index and the ending index of the string that you want to delete. For example, if your string is 
"Hello World"
and you want to delete the word “World” from the string, you can use the following code: 

          
           
StringBuffer strBuf = new StringBuffer("Hello World");
strBuf.delete(6, 11);
System.out.println("StrBuf Value is: " + strBuf);
This will print out the following output: 

          
           
StrBuf Value is: Hello

As you can see, the word “World” has been removed from the string.

Using the 
remove()
Method

The 
remove()
method is similar to the 
delete()
method in that it removes a subset of characters from a string. However, this method takes only a single parameter — the beginning index for the range of characters that you want to remove. This means that the range of characters to be deleted starts at the specified index and ends at the last character of the string. For example, if your string is 
"Hello World"
and you want to delete the word “World” from the string, you can use the following code: 

          
           
StringBuffer strBuf = new StringBuffer("Hello World");
strBuf.remove(6);
System.out.println("StrBuf Value is: " + strBuf);
This will print out the following output: 

          
           
StrBuf Value is: Hello

As you can see, the word “World” has been removed from the string.

Using the 
setLength(0)
Method

The 
setLength(0)
method of the StringBuffer class allows you to set the length of the string buffer to zero. This effectively empties the string buffer of all its contents. This method takes no parameters and is typically used when you want to clear the contents of a string buffer without creating a new one. For example, if your string is 
"Hello World"
and you want to clear the contents of the string buffer, you can use the following code: 

          
           
StringBuffer strBuf = new StringBuffer("Hello World");
strBuf.setLength(0);
System.out.println("StrBuf Value is: " + strBuf);
This will print out the following output: 

          
           
StrBuf Value is:

As you can see, the contents of the string buffer have been cleared.
