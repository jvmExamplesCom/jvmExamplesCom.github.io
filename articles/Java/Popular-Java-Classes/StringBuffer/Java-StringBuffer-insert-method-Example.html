Java StringBuffer insert method Example

The Java StringBuffer class is an excellent tool for manipulating strings in Java. It allows for efficient insertion, deletion, and modification of strings. One particularly useful method of the StringBuffer class is the insert() method. This method allows a character, a string, or an array of characters to be inserted at any specified index in a StringBuffer object. This tutorial will discuss the insert() method and provide several examples of its use.

StringBuffer Insert Method Syntax

The syntax of the StringBuffer insert() method is as follows: 

          
           
public StringBuffer insert(int index, String str)
public StringBuffer insert(int index, char[] str)
public StringBuffer insert(int index, char ch)
public StringBuffer insert(int index, boolean b) 
public StringBuffer insert(int index, int i)
public StringBuffer insert(int index, long l)
public StringBuffer insert(int index, float f)
public StringBuffer insert(int index, double d)
As can be seen from the syntax above, the insert() method takes two arguments: an integer representing the index at which the insertion should occur, and an object of one of the supported types (String, char[], char, boolean, int, long, float, or double).

Java StringBuffer insert Method Examples

Now that we have seen the syntax, let's look at some examples of the StringBuffer insert() method.

Example 1: Inserting a Single Character

In the first example, we will insert a single character at an index of 2 in a StringBuffer object. The code for this example is as follows: 

          
           
StringBuffer sb = new StringBuffer("Hello World!");
sb.insert(2, 'x');
System.out.println(sb);
When the code above is executed, it will print the following output to the console: 

          
           
Hexllo World!
As can be seen, the character 'x' has been inserted at the index specified.

Example 2: Inserting an Array of Characters

In this second example, we will insert an array of characters at an index of 6 in a StringBuffer object. The code for this example is as follows: 

          
           
StringBuffer sb = new StringBuffer("Hello World!");
char[] chars = {'x', 'y', 'z'};
sb.insert(6, chars);
System.out.println(sb);
When the code above is executed, it will print the following output to the console: 

          
           
Hello xyzWorld!
Again, the array of characters has been inserted at the index specified.

Example 3: Inserting a String

In the final example, we will insert a String at an index of 6 in a StringBuffer object. The code for this example is as follows: 

          
           
StringBuffer sb = new StringBuffer("Hello World!");
String str = "xyz";
sb.insert(6, str);
System.out.println(sb);
When the code above is executed, it will print the following output to the console: 

          
           
Hello xyzWorld!
Once again, the String has been inserted at the index specified.

Conclusion

In this tutorial, we discussed the Java StringBuffer insert() method and provided several examples of its use. We saw how to insert a single character, an array of characters, and a String at any given index in a StringBuffer object.
