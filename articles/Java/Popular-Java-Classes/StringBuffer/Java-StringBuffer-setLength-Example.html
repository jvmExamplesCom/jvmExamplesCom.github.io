Java StringBuffer setLength Example

The StringBuffer class in Java contains a method called 

setLength()
to manually set the length of the buffer. It is important to remember that when using this method anything after the given length is truncated and not returned. In this Java StringBuffer setLength example we will discuss how to use this method.

Java StringBuffer setLength Syntax

The syntax for the 

          
           
StringBuffer.setLength()
method is: 

          
           
public void setLength(int newLength)

The parameter passed to this method is the desired length and it must be greater than the current length or no action is taken. If you pass a value less then the current length then all characters past the given length are removed.

Java StringBuffer setLength Example

Let's look at an example of how to use the 

          
           
StringBuffer.setLength()
method. Here we create a StringBuffer and then set the length to 10. 

          
           
StringBuffer sb = new StringBuffer("Hello");
sb.setLength(10);
System.out.println(sb);
When we run the code we get the following output: 

          
           
Hello

As you can see the length is now 10 but because we did not append any characters the string is padded with additional spaces.

Now let's look at a different example. Here we create a StringBuffer and then set the length to 3.


          
           
StringBuffer sb = new StringBuffer("Hello");
sb.setLength(3);
System.out.println(sb);
When we run the code we get the following output: 

          
           
Hel

This time anything after the given length of 3 is truncated and not returned. As you can see the 


          
           
StringBuffer.setLength()
method is a powerful tool that allows you to manipulate the length of the buffer.
