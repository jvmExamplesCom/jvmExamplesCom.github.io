Java StringBuffer Reverse Example

The StringBuffer class in the Java language provides a useful method to reverse the contents of a StringBuffer object. In this example, we'll explore how to use the 

          
           
reverse()
method and some possible variations you might use when reversing a StringBuffer object.

StringBuffer reverse() Method

The 

          
           
reverse()
method is available for all StringBuffer instances. It reverses the characters within the StringBuffer object according to what the JVM considers their order. Let's look at a basic example of the reverse() method: 

          
           
StringBuffer sb = new StringBuffer("Hello World!");
sb.reverse();
System.out.println(sb);
The output of this code would be 

          
           
!dlroW olleH
. Notice how the order of the characters has been reversed.

Looping and Reversing Characters

You can also use a loop to go through each character of a StringBuffer object, reverse it, and place it into another StringBuffer object. This approach gives you more flexibility over the order and types of characters you want to work with. Here's an example of how to do that: 

          
           
StringBuffer sb = new StringBuffer("Hello World!");
StringBuffer revSb = new StringBuffer();
for (int i = sb.length() - 1; i >= 0; i--) {
    revSb.append(sb.charAt(i));
}
System.out.println(revSb);
In this example, we're creating a new StringBuffer object called 

          
           
revSb
which will contain the reversed characters from 

          
           
sb
. The loop goes through each character of 

          
           
sb
in reverse order and adds them to 

          
           
revSb
one by one. The output of this code would also be 

          
           
!dlroW olleH
.

Using Lists and Collections

Collections.reverse()
method. Here's an example of how that would work:

List<Character> chars = new ArrayList<>();
for (int i = 0; i < sb.length(); i++) {
    chars.add(sb.charAt(i));

chars.forEach(revSb::append);
System.out.println(revSb);

revSb

Conclusion

As you can see, the 

          
           
reverse()
