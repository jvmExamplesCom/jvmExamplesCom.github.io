Java: StringBuffer Reset Java Example

The 

          
           
StringBuffer
class in Java represents a mutable sequence of characters. It is used when you want to manipulate strings, such as concatenating strings or replacing characters. In this tutorial, we are going to learn how to reset a 

          
           
StringBuffer
object in Java.

Creating a StringBuffer Object

Before we start resetting the underlying contents of a 

          
           
StringBuffer
, let's create one. We can do that with the 

          
           
StringBuffer
constructor which takes a string argument as its parameter. 

          
           
String s = "Hello World!"; 
StringBuffer sb = new StringBuffer(s);

System.out.println(sb); // prints "Hello World!"

Resetting a StringBuffer Object

Now, we have our 

          
           
StringBuffer
instance, let's look at some ways of resetting it. The most straightforward way of resetting a 

          
           
StringBuffer
is by using the 

          
           
setLength()
method. This method takes an 

          
           
int
argument which represents the length of the string. As a result, if you specify 0 as the length argument, it will reset the 

          
           
StringBuffer
to an empty string. 

          
           
sb.setLength(0);

System.out.println(sb); // prints "" (empty string)
The other way of resetting a 

          
           
StringBuffer
object is by using the 

          
           
delete()
method. This method takes two 

          
           
int
arguments, the first one represents the starting index, while the second one represents the ending index. If you specify 0 as both arguments, it will reset the 

          
           
StringBuffer
to an empty string. 

          
           
sb.delete(0, 0);

System.out.println(sb); // prints "" (empty string)
Finally, you can reset a 

          
           
StringBuffer
object by using the 

          
           
replace()
method. This method takes three 

          
           
int
arguments, the first one and the third one represent the starting and ending index respectively, while the second one does not. If you specify 0 for both the starting and ending index, it will reset the 

          
           
StringBuffer
to an empty string. 

          
           
sb.replace(0, 0, "");

System.out.println(sb); // prints "" (empty string)

Conclusion

In this tutorial, we learned how to reset a 

          
           
StringBuffer
object in Java. We discussed three ways of resetting a 

          
           
StringBuffer
; using the 

          
           
setLength()
, 

          
           
delete()
and 

          
           
replace()
methods.
