Java StringBuffer Append New Line Example

In Java, StringBuffer is a class used for working with text strings. It provides a mutable sequence of characters. StringBuffer implements the CharSequence interface, which enables it to be manipulated just like a regular string. This article will show an example of how to use the append() method to add a new line character to a StringBuffer object.

Example 1: Appending A New Line Using The Append() Method


          
           
StringBuffer sb = new StringBuffer();
sb.append("This is my first line
");
sb.append("This is my second line
");
System.out.println(sb);

In the example above, we create a new StringBuffer object and append two strings, each containing a new line character at the end. The output of the program is:


          
           
This is my first line
This is my second line

Example 2: Appending A New Line With The Append() Method And A Character Literal


          
           
StringBuffer sb = new StringBuffer();
sb.append("This is my first line
");
sb.append('\n'); // Append a new line character
sb.append("This is my second line
");
System.out.println(sb);

In this second example, we use the append() method to append a new line character, represented by the character literal 

'\n'
. As before, the output of the program is: 

          
           
This is my first line
This is my second line

Example 3: Appending A New Line With The AppendChar() Method


          
           
StringBuffer sb = new StringBuffer();
sb.append("This is my first line
");
sb.appendChar('\n'); // Append a new line character
sb.append("This is my second line
");
System.out.println(sb);
In the final example, we use the 
appendChar()
method to append a new line character. As before, the output of the program is: 

          
           
This is my first line
This is my second line

We have seen how to use the append() and appendChar() methods of the StringBuffer object to append new line characters. These methods are useful when you need to append new line characters to a string.
