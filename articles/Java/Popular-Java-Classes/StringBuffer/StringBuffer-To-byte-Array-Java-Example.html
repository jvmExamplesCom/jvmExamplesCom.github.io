Java: StringBuffer to Byte Array Example

The purpose of this article is to demonstrate how to convert a StringBuffer into a byte array in Java. It will walk through the various steps required to do so, as well as provide multiple examples of how to use the code.

Preparing the StringBuffer

The first step is to set up the StringBuffer with the desired string value. This is done with the following code: 

          
           
StringBuffer sb = new StringBuffer("Example String");

Getting the String Values

Once the StringBuffer is prepared, we need to extract the string values for each character in the buffer. This can be done using the 

          
           
charAt()
method to get the characters at a given index: 

          
           
char c = sb.charAt(i);

Converting to Bytes

After the characters are obtained, they need to be converted to a 

          
           
byte
array. This can be done by first constructing the array and then looping over the characters, assigning the 

          
           
byte
versions of each character to the array. The following code shows how this is done: 

          
           
byte[] bytes = new byte[sb.length()];
for (int i = 0; i < sb.length(); i++) {
    bytes[i] = (byte) sb.charAt(i);
}

Printing the Result

Finally, to print out the resulting byte array, we can print out each value of the array with a for-loop: 

          
           
for (int i = 0; i < bytes.length; i++) {
    System.out.println(bytes[i]);
}
