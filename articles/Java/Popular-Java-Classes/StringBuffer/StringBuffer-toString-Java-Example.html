Java: StringBuffer toString Java Example

StringBuffers are powerful tools for manipulating strings in Java. In this article, we will look at how to use the toString() method of a StringBuffer to easily generate a string representation of data stored in a StringBuffer object.

Creating a StringBuffer Object

Before we can use the toString() method, we must first create a StringBuffer object. This can be done with one of the following two methods: 

          
           
StringBuffer sb = new StringBuffer()

          

          
           
StringBuffer sb = new StringBuffer("Hello World!")
The first statement creates an empty StringBuffer object and the second statement creates a StringBuffer object and assigns the string “Hello World!” to it.

Appending Data to the StringBuffer Object

Once we have a StringBuffer object, we can append data to it using the append() method. For example, to add the string “abc” to an existing StringBuffer object: 

          
           
sb.append("abc")
We can also append other data types, such as integers and booleans. For example, to append an integer to our StringBuffer object: 

          
           
int i = 123;
sb.append(i);

Generating a String Representation of the Data

Once we have appended all the data we need to the StringBuffer object, we can generate a string representation of it using the toString() method: 

          
           
String s = sb.toString();
This will generate a string from the data stored in the StringBuffer object. For example, if we had appended the string “abc”, the integer 123 and the boolean true to our StringBuffer object, our string would look like this: 

          
           
abc123true

Conclusion

In this article, we looked at how to use the toString() method of a StringBuffer to easily generate a string representation of data stored in a StringBuffer object. We created a StringBuffer object, appended data to it, and then generated a string representation of the data using the toString() method.
