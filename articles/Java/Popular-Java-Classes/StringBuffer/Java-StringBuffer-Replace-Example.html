Java StringBuffer Replace Example

StringBuffer is a mutable sequence of characters. It is a part of the Java Collection Framework, meaning that it is able to store a sequence of characters and provide functionality to manipulate these characters. In this example, we will explore the use of the replace() method within the StringBuffer class. This method replaces all occurrences of the specified character with another character within the StringBuffer object. The syntax for the replace() method is as follows: 

          
           
public void replace(int startIndex, int endIndex, String str)
The parameters are as follows:
  • startIndex: This defines the starting index where the replacing of the characters should occur.
  • endIndex: This defines the ending index where the replacing of the characters should stop.
  • str: This defines what characters should be used to replace the current characters from start index to end index.
Let's look at a few examples of how the replacing of characters can occur within the StringBuffer object.

Example 1: Replacing Characters in the Middle of a StringBuffer

In this example, we will replace all the characters between index 3 and 5 (inclusive) with the letter ‘j’. 

          
           
StringBuffer sb = new StringBuffer("Hello World");

sb.replace(3, 6, "j");

System.out.println(sb.toString());
The output of this example is 

          
           
Heljlo World
. As you can see, all characters between 3 and 5 index was replaced by the letter ‘j’.

Example 2: Replacing All Occurrences of a Specified Character

In this example, we will replace all ‘o’s in the StringBuffer with the letter ‘j’. 

          
           
StringBuffer sb = new StringBuffer("Hello World");

for (int i = 0; i < sb.length(); i++) {
    if (sb.charAt(i) == 'o') {
        sb.replace(i, i+1, "j");
    }
}

System.out.println(sb.toString());
The output of this example is 

          
           
Hellj Wjrld
. As you can see, all occurrences of the letter ‘o’ were replaced by the letter ‘j’.

Example 3: Replacing All Occurrences of a Specified String

In this example, we will replace all occurrences of the string “Hello” with the string “Goodbye”. 

          
           
StringBuffer sb = new StringBuffer("Hello World");

int startIndex = 0;
int endIndex = -1;

while ((endIndex = sb.indexOf("Hello", startIndex)) != -1) {
    sb.replace(endIndex, endIndex + 5, "Goodbye");
    startIndex = endIndex + 5;
}

System.out.println(sb.toString());
The output of this example is 

          
           
Goodbye World
