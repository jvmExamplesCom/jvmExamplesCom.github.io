Java StringBuffer length Example

The Java StringBuffer class is a powerful tool for manipulating strings. It has a wide range of applications, ranging from creating and modifying strings and manipulating their length. This article shows how to use the length() method of the StringBuffer class to find the size of a given string.

StringBuffer.length() Method

The length() method is a member of the Java StringBuffer class that returns an int representing the length of the string contained within the given buffer. The following example uses the length() method to get the length of the given string.


          
           
StringBuffer sb = new StringBuffer("Hello World!");
int length = sb.length(); 
System.out.println(length); //prints 12

The length() method will count all characters within the string buffer, including whitespace. That means that if the same string was created without any spaces between the words, the method would still return 12.
