Java's 

          
           
java.sql.BatchUpdateException
is a checked exception that occurs when the database does not recognize a batch of SQL statements sent for execution. It is thrown by the 

          
           
Statement.executeBatch()
method when one or more of the commands sent to the database fail to execute correctly.

  int[] numRowsAffected = stmt.executeBatch();
} catch (BatchUpdateException bue) {
  System.err.println(bue.getMessage());
  System.err.println("Error code: " + bue.getErrorCode());
  System.err.println("SQL state: " + bue.getSQLState());

}

When run, this code will execute the INSERT statement, and if an error occurs, the BatchUpdateException will be thrown. The exception contains useful information about the failed query, such as the error code, SQL state and the actual SQL statement.

Statement.executeBatch()
