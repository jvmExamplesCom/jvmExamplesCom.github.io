What is com.hazelcast.config.Config?

com.hazelcast.config.Config is a Java library that provides a set of APIs for configuring and customizing the behavior of Hazelcast-based applications. It provides APIs for setting up network configuration, adding nodes to a cluster, configuring access control lists (ACLs), and more.

Example com.hazelcast.config.Config Code

The following example shows how to use com.hazelcast.config.Config to create a custom configuration for a Hazelcast cluster. 

          
           
Config config = new Config();
config.setClusterName("MyCluster");
 
NetworkConfig networkConfig = config.getNetworkConfig();
networkConfig.setPort(5701);
networkConfig.setPortAutoIncrement(false);
 
JoinConfig joinConfig = networkConfig.getJoin();
joinConfig.getMulticastConfig().setEnabled(false);
joinConfig.getTcpIpConfig().addMember("host1").addMember("host2").addMember("host3");
 
config.getMapConfigs().put("myMap", new MapConfig().setBackupCount(2));
 
HazelcastInstance instance = Hazelcast.newHazelcastInstance(config);
In the above example, we are creating a custom configuration for a Hazelcast cluster. We start by creating an instance of Config. Then, we set the cluster name, port number, and whether port auto-increment is enabled or not. Next, we configure the join configuration for the cluster. Here, we disable multicast and add the members which should join the cluster. Finally, we configure the map configuration by setting the backup count. The backup count defines how many copies of the data should be kept in case of a node failure. Once we have configured the configuration, we can create a Hazelcast instance using the configuration instance.
