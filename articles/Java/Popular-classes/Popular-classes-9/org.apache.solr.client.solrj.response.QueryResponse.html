What is org.apache.solr.client.solrj.response.QueryResponse?

org.apache.solr.client.solrj.response.QueryResponse is a Java class used to represent the response of Solr queries. It can be used to capture information about the document, such as its Solr document id, and its score, as well as highlight snippets. When working with Solr, QueryResponse objects are returned once a query has been executed.

Examples of org.apache.solr.client.solrj.response.QueryResponse

The following is a code example of using org.apache.solr.client.solrj.response.QueryResponse in Java:


          
           
SolrQuery query = new SolrQuery();
query.setQuery("name:John");
QueryResponse rsp = solrClient.query(query);
SolrDocumentList docs = rsp.getResults();
for (SolrDocument doc : docs) {
    System.out.println(doc.getFieldValue("id"));
}

In the above example, we define a query for documents containing the text “John”, execute the query and iterate over the results. The for loop will print out the Solr document id for all documents containing the text John.

We can also use QueryResponse to find highlight snippets from our search:


          
           
SolrQuery query = new SolrQuery();
query.setQuery("name:John");
query.setHighlightSimplePre("<strong>");
query.setHighlightSimplePost("</strong>");
QueryResponse rsp = solrClient.query(query);
SolrDocumentList docs = rsp.getResults();
for (SolrDocument doc : docs) {
  System.out.println(rsp.getHighlighting().get(doc.getFieldValue("id"))
                     .get("name"));
}

In the above example, we again define a query for documents containing the text “John”, but this time we set highlighting parameters to render the search terms in bold. The for loop will print out the highlighted snippets from the “name” fields of documents containing the text John.

We can also use QueryResponse to find the Solr scores associated with each document:


          
           
SolrQuery query = new SolrQuery();
query.setQuery("name:John");
QueryResponse rsp = solrClient.query(query);
SolrDocumentList docs = rsp.getResults();
for (SolrDocument doc : docs) {
  System.out.println(doc.getScore());
}

In this example the for loop will print out the Solr score of each document containing the text John.
