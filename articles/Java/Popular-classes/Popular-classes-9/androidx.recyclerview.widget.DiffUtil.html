Androidx.recyclerview.widget.DiffUtil

Androidx.recyclerview.widget.DiffUtil is a utility class that can calculate the difference between two lists and output a list of update operations that converts the first list into the second one. This can be used to calculate updates for a RecyclerView Adapter. It has methods such as calculateDiff() which takes two lists as parameters and returns a DiffResult object.

Example

Let’s say we have two lists – oldList and newList. We want to calculate the difference between them and update our RecyclerView Adapter.


          
           
List<String> oldList = Arrays.asList("A", "B", "C", "D");

List<String> newList = Arrays.asList("A", "C", "D", "E");

DiffUtil.DiffResult diffResult = DiffUtil.calculateDiff(new DiffUtil.Callback() {
        @Override
        public int getOldListSize() {
            return oldList.size();
        }

        @Override
        public int getNewListSize() {
            return newList.size();
        }

        @Override
        public boolean areItemsTheSame(int oldItemPosition, int newItemPosition) {
            return oldList.get(oldItemPosition).equals(newList.get(newItemPosition));
        }

        @Override
        public boolean areContentsTheSame(int oldItemPosition, int newItemPosition) {
            return oldList.get(oldItemPosition).equals(newList.get(newItemPosition));
        }
    });

diffResult.dispatchUpdatesTo(myRecyclerViewAdapter);

The code above will call the MyRecyclerViewAdapter's update function with an UpdateOperation list and will update the view accordingly.

Using DiffUtil

DiffUtil requires two lists, an old list and a new list, and uses the Callback interface to find the differences between the two. The callback interface needs four functions to be implemented: getOldListSize(), getNewListSize(), areItemsTheSame() and areContentsTheSame().

The getOldListSize() and getNewListSize() functions should return the respective length of each list.


          
           
int getOldListSize() {
    return oldList.size();
}

int getNewListSize() {
    return newList.size();
}

The areItemsTheSame() and areContentsTheSame() functions should compare the items in each list to check for any changes. They should both return true if the items are the same, false if they are different.


          
           
boolean areItemsTheSame(int oldItemPosition, int newItemPosition) {
    return oldList.get(oldItemPosition).equals(newList.get(newItemPosition));
}

boolean areContentsTheSame(int oldItemPosition, int newItemPosition) {
    return oldList.get(oldItemPosition).equals(newList.get(newItemPosition));
}

Once the Callback interface has been implemented, you can call the calculateDiff() method with the two lists and the Callback.


          
           
DiffUtil.DiffResult diffResult = DiffUtil.calculateDiff(new DiffUtil.Callback() {...});

The DiffUtil class will then return a DiffResult object which contains a list of UpdateOperations. This list can be applied to the RecyclerView Adapter in order to update the view.


          
           
diffResult.dispatchUpdatesTo(myRecyclerViewAdapter);
