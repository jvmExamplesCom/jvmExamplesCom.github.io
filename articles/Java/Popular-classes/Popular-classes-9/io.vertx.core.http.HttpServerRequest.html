Introduction To Java io.vertx.core.http.HttpServerRequest

The 

          
           
io.vertx.core.http.HttpServerRequest
is an interface that provides an API to create HTTP server requests in Vert.x. This interface is used in relation with the 
HttpServerResponse
object to provide full functionality for an HTTP request and response. It provides methods to set the response status code, headers and other request-related objects.

Creating an HttpServerRequest Instance

An instance of the HttpServerRequest is created through the use of the 


          
           
HttpServer.requestHandler
method. The following example shows how to create a new instance: 

          
           
HttpServer server = vertx.createHttpServer();

server.requestHandler(request -> {
 HttpServerRequest req = request;
});

The Methods of HttpServerRequest

The 
HttpServerRequest
has numerous useful methods that allow for the manipulation of its related request. Below are some common ones:

setExpectMultipart(boolean expect): Sets whether the request is expecting multipart form data. This should be called before the request body is read.


          
           
req.setExpectMultipart(true);

method(): Returns the request's HTTP method name.


          
           
String httpMethod = req.method();
System.out.println("The HTTP method is " + httpMethod);

uri(): Returns the request URI string.


          
           
String uri = req.uri();
System.out.println("Requested URI: " + uri);

parameter(String name): Returns the request parameter with the specified name.


          
           
String ageParameter = req.parameter("age");
System.out.println("The value passed for age is " + ageParameter);

Conclusion

The 

          
           
io.vertx.core.http.HttpServerRequest
interface provides developers with an API to create HTTP server requests in Vert.x. It contains many useful methods to manipulate the request and set the response status code, headers and other request related objects.
