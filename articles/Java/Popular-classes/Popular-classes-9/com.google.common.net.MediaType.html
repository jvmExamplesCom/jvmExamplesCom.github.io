Introduction to com.google.common.net.MediaType

The 

          
           
com.google.common.net.MediaType
class is a utility class contained within the Google Guava library, which provides support for working with media types such as MIME types and content-encoding. It allows developers to get media type information from a string, create a new media type or update an existing one. It also provides methods to obtain the charset, type and subtype of a media type.

Creating a Media Type Object

A 
MediaType
object can be created from a string representation of a media type. The following example creates a media type from a given string: 

          
           
String contentType = "text/plain; charset=UTF-8";
MediaType type = MediaType.create(contentType);

Accessing Media Type Information

Information about the media type can be accessed using the various getter methods available in the 

MediaType
class. The following example gets the type, subtype, and character set of a media type: 

          
           
String contentType = "image/jpeg; charset=UTF-8";
MediaType type = MediaType.create(contentType);

String type = type.getType(); // image 
String subtype = type.getSubtype(); // jpeg

MediaType.Builder
class. The following example creates an image media type with a custom character set: 

          
           
Charset customCharSet = Charset.forName("UTF-16");
