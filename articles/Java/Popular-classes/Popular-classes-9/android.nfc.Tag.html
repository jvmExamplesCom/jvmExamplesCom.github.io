java.android.nfc.Tag Overview

Java.android.nfc.Tag is a class in the Android framework for providing communication support for Near Field Communication (NFC) tags and other NFC enabled devices. It provides the access to the tag, reading and writing of NDEF messages and other operations system wide. It mainly addresses the needs of developers who are interested creating applications that use NFC tags.

Creating an Instance of Tag

An instance of a Tag can be created as follows:

Method Details

Once an instance of the Tag has been created, there are several methods available to manipulate or query its state.

GetTechList()


          
           
Tag.getTechList()

This method returns a String array containing the list of technologies for which this tag offers support.

GetId()


          
           
Tag.getId()

This method returns a byte array containing the identifier of the tag.

GetMaxTransceiveLength()


          
           
Tag.getMaxTransceiveLength()

This method returns the maximum number of bytes that can be sent to or received from the tag in a single transceive operation.

Connect()


          
           
Tag.connect()

This method establishes a connection with the remote tag. This method should be called before any I/O operations are performed on the tag.

Close()


          
           
Tag.close()

This method closes the connection with the remote tag. Once the connection has been closed, further I/O operations will return errors.
