CloseableHttpAsyncClient

CloseableHttpAsyncClient

interface. The code below creates an asynchronous client that can be used with any HTTP 1.1 compliant server: 

          
           
CloseableHttpAsyncClient httpclient = HttpAsyncClients.createDefault();
