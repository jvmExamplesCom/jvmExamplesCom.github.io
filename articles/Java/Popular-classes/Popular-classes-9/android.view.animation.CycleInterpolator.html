android.view.animation.CycleInterpolator

The android.view.animation.CycleInterpolator class allows us to perform a cycle-based animation in Android. The CycleInterpolator class allows developers to control the number of cycles in an animation, as well as set the rate of the cycle. General Description

The CycleInterpolator class is part of the android.view.animation package and is designed to provide a cycle-based interpolator for animations.
