android.view.animation.CycleInterpolator

The android.view.animation.CycleInterpolator class allows us to perform a cycle-based animation in Android. The CycleInterpolator class allows developers to control the number of cycles in an animation, as well as set the rate of the cycle. This Interpolator is useful for creating animations that appear to repeat themselves and can be used with any type of animation.

General Description

The CycleInterpolator class is part of the android.view.animation package and is designed to provide a cycle-based interpolator for animations. The CycleInterpolator class provides methods to control the number of cycles of an animation, as well as the rate of these cycles. The CycleInterpolator class is useful for creating animations that seem to repeat themselves.

  • public float getInterpolation(float input) - Returns the interpolation value between 0 and 1, based on the input value.
