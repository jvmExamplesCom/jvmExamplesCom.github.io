Understanding org.antlr.v4.runtime.BaseErrorListener

org.antlr.v4.runtime.BaseErrorListener is a class used to define the base class for all ANTLR error listeners. It can be used to create custom error messages when a parser encounters an error. This allows for better control over the output generated by a parser, as well as all types of applications that rely on ANTLR.

Syntax

The BaseErrorListener class is a subclass of org.antlr.v4.runtime.ANTLRErrorListener and provides the following syntax:


          
           
class BaseErrorListener implements ANTLRErrorListener {
    public void syntaxError(Recognizer<?, ?> recognizer, Object o, int i, 
                            int i1, String s, RecognitionException e);

    public void reportContextSensitivity(Parser parser, DFA dfa, int i, int i1, int i2, ATNConfigSet atnConfigSet);
