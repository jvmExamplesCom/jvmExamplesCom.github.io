Exploring org.eclipse.aether.RepositorySystemSession

In this article, we'll take a look at the various aspects of the RepositorySystemSession including its methods and properties and explore code examples for each of them.

Creating a RepositorySystemSession

The first step in using a RepositorySystemSession is to create one. It is typically done by providing a RepositorySystem object along with a RepositorySystemSessionConfiguration to the method createSession().


          
           
RepositorySystem system = Aether.newRepositorySystem();
RepositorySystemSessionConfigurator configurator = new DefaultRepositorySystemSessionConfigurator();
RepositorySystemSession session = system.createSession(configurator);
