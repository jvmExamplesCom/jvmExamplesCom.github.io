org.apache.hadoop.hbase.client.Get

The org.apache.hadoop.hbase.client.Get class is a tool for getting data from an HBase table. It provides a way to retrieve a single row of data from a table, specified by a row key.

public Get(byte[] row, Filter filter)

The constructors take two parameters: the row key that identifies the row to be retrieved, and optionally a Filter instance.

Methods

The Get class provides a number of methods to configure and execute the get operation:


          
           
public void setFilter(Filter filter) 
public void setMaxVersions(int maxVersions) 
public void setTimeRange(long minStamp, long maxStamp) 
public Result get() throws IOException

setFilter()

get()
