org.springframework.beans.factory.ObjectFactory

The 

          
           
ObjectFactory
interface is part of the Spring Framework and is an extension of the 

          
           
FactoryBean
. It is used to provide instances of beans from a 

          
           
BeanFactory
. The 

          
           
ObjectFactory
interface allows a bean to be lazily instantiated on demand from the 

          
           
BeanFactory
. It is useful for situations where the bean must be accessed, but not necessarily needed for the application all at once.

Method Signatures

The 

          
           
ObjectFactory
interface consists of only one method, 

          
           
getObject()
, which returns an instance of the requested bean. 

          
           
public interface ObjectFactory<T> {
    // Gets an instance of the requested bean
    T getObject();
}

Example Usage

To use the 

          
           
ObjectFactory
, we first need to declare a bean that implements the 

          
           
ObjectFactory
interface. The following example shows how to create a simple 

          
           
ObjectFactory
bean. 

          
           
public class MyObjectFactory implements ObjectFactory<MyObject> {

    @Override
    public MyObject getObject() {
        // Code to create and return an instance of MyObject
        return new MyObject();
    }

}

ObjectFactory
bean has been declared, it can be retrieved from the 

          
           
BeanFactory

MyObjectFactory myObjectFactory = beanFactory.getBean(MyObjectFactory.class);
