Introduction to org.apache.hadoop.hbase.client.ConnectionFactory

It enables applications to create reliable database connections without having to manage the configuration themselves. This article will explain how to use the ConnectionFactory API and provide several code examples.

Connection connection = ConnectionFactory.createConnection();

Next, we need to configure the connection by setting the database URL, user name, and password. This is done by calling the setURL, setUsername, and setPassword methods:


          
           
connection.setURL("jdbc:h2:<databaseName>");
connection.setUsername("<username>");
connection.setPassword("<password>");

We can also configure other settings such as the connection timeout and maximum pool size:


          
           
connection.setConnectionTimeout(60_000);
connection.setMaxPoolSize(20);

Finally, we need to open the connection. This is done by calling the openConnection method:


          
           
connection.openConnection();

Closing a Connection using ConnectionFactory

When we are finished with the connection, we can close it by calling the closeConnection method:


          
           
connection.closeConnection();
