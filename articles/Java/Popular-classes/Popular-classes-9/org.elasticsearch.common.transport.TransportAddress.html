Java org.elasticsearch.common.transport.TransportAddress

Java's org.elasticsearch.common.transport.TransportAddress is an abstract base class that allows for network communication between two different endpoints. Creating the TransportAddress

The most basic way to create a TransportAddress is with its constructor:


          
           
TransportAddress address = new TransportAddress("127.0.0.1", 9300);

This will create a new TransportAddress instance using the localhost IP address ("127.0.0.1") and port number (9300) as parameters. This particular address is suitable for server-client communication over the default Elasticsearch transport protocol, which uses TCP/IP.

address.setHostname("example.com");

Using the TransportAddress
