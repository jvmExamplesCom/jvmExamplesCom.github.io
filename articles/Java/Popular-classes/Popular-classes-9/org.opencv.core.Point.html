Introduction to Java org.opencv.core.Point

Java org.opencv.core.Point is a data structure used for representing a point in an image. It consists of two double values: x and y. Point p1 = new Point(x, y);
where x and y are double values. Another way is by passing an array of two double elements: 

          
           
double[] pointData = {x, y};
Point p2 = new Point(pointData);
We can also create a Point from a String: 

          
           
String stringPoint = "x,y";
Point p3 = Point.fromString(stringPoint);

Point p4 = new Point(p2);

Practical Applications
