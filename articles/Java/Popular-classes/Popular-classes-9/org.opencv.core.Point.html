Introduction to Java org.opencv.core.Point

Java org.opencv.core.Point is a data structure used for representing a point in an image. It consists of two double values: x and y. It is used in many OpenCV algorithms for image processing such as object detection, stereo vision, and image stitching. This tutorial will explain how to use it, ways to create points and some practical applications.

Point p1 = new Point(x, y);
where x and y are double values. Another way is by passing an array of two double elements: 

          
           
double[] pointData = {x, y};
Point p2 = new Point(pointData);
We can also create a Point from a String: 

          
           
String stringPoint = "x,y";
Point p3 = Point.fromString(stringPoint);
And finally, we can create a Point from another Point: 

          
           
Point p4 = new Point(p2);

Working with Points

double distance = p1.distance(p2);

Practical Applications

org.opencv.core.Point can be used in many practical applications. One example is object tracking, where we need to track the position of an object in an image sequence. We can use the methods described above to track the position of the object over time. We can also use Point in stereo vision algorithms.
