kafka.javaapi.consumer.ConsumerConnector

The kafka.javaapi.consumer.ConsumerConnector is a Java class used to consume records from Kafka topics. It provides an interface for a typical Kafka consumer application, allowing the user to subscribe to topics, set up message filters and read messages from the topics. Consumer applications using the ConsumerConnector are able to take advantage of many of Kafka's features, such as automatic partition rebalance, fault-tolerance, and offset management.
