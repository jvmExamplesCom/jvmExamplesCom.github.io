What is android.support.design.widget.BottomSheetDialog?

Android.support.design.widget.BottomSheetDialog is a class that provides an alternative to the traditional dialog and popup approach. It allows users to peek at relevant content without having to completely leave their current context.

Overview of android.support.design.widget.BottomSheetDialog

The BottomSheetDialog class is part of the Design Support Library that was released in Android 5.0. The support library brings many features from newer versions of the Android operating system to earlier versions. This allows developers to use the same code base across more than one device running different versions of Android.

The BottomSheetDialog provides an alternative UI pattern to traditional dialogs and popups. It allows content to be presented in a bottom sheet that slides up from the bottom of the screen. Its bottom positioning also allows user interaction with the other elements on the screen while the bottom sheet remains visible.
