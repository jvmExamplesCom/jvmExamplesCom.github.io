What is android.support.design.widget.BottomSheetDialog?

Overview of android.support.design.widget.BottomSheetDialog

The BottomSheetDialog class is part of the Design Support Library that was released in Android 5.0. The support library brings many features from newer versions of the Android operating system to earlier versions. This allows developers to use the same code base across more than one device running different versions of Android.
