java.android.graphics.SweepGradient

The java.android.graphics.SweepGradient class is a type of Shader used to draw a sweep gradient along a line connecting two specified points (x0, y0) and (x1, y1). The gradient is drawn anti-clockwise starting at the start point (x0, y0) and continuing to the end point (x1, y1). It can be used to create radial gradients, which can be used for background, buttons, etc.

Creating a Sweep Gradient

To create a SweepGradient you need to provide the coordinates of the start and end points along with an array of colors. The array of colors should contain a minimum of two values. This array determines the color of the gradient at each point of the sweep.


          
           
float x0 = 0; float y0 = 0; float x1 = 100; float y1 = 100; int[] colors = new int[]{Color.RED, Color.BLUE};

SweepGradient shader = new SweepGradient(x0, y0, x1, y1, colors);

Once created, the shader can be used to create a Paint object that can be used to draw shapes on a Canvas.


          
           
Paint paint = new Paint();
paint.setShader(shader);
canvas.drawCircle(50, 50, 40, paint);

The above code will draw a circle on the canvas, filled with a sweep gradient from red to blue.

Radial Gradients

One common use of the SweepGradient shader is for creating a radial gradient. This can be accomplished by specifying the same start and end points for the sweep.


          
           
float x0 = 0; float y0 = 0; float x1 = 0; float y1 = 0; int[] colors = new int[]{Color.RED, Color.BLUE};

SweepGradient shader = new SweepGradient(x0, y0, x1, y1, colors);

Paint paint = new Paint();
paint.setShader(shader);
canvas.drawCircle(50, 50, 40, paint);

Changing the Orientation of the Gradient

By default, the gradient will be drawn in an anti-clockwise direction, starting at the start point. It is possible to change the orientation of the gradient by specifying a rotation angle when creating the shader.


          
           
float x0 = 0; float y0 = 0; float x1 = 0; float y1 = 0; int[] colors = new int[]{Color.RED, Color.BLUE};
float rotationAngle = 90;

SweepGradient shader = new SweepGradient(x0, y0, x1, y1, colors, rotationAngle);

Paint paint = new Paint();
paint.setShader(shader);
canvas.drawCircle(50, 50, 40, paint);

The above code will draw a circle on the canvas, filled with a sweep gradient from red to blue, with the gradient drawn in a clockwise direction.
