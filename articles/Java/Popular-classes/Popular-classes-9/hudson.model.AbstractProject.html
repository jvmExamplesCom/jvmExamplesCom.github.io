Introduction to hudson.model.AbstractProject

The 

          
           
hudson.model.AbstractProject
class is an abstract base class for all the projects that are maintained in an instance of Hudson. It provides access to useful methods for monitoring builds and setting properties of projects. It also contains the ability to add custom logic to existing methods.

hudson.model.AbstractProject
class provides several methods that can be used when dealing with projects. Some of them include:

  • getBuildsDir()
    – This method returns the full file path to the directory containing the builds of this project.
  • getLastBuild()
    – This method returns the last completed build of this project.
