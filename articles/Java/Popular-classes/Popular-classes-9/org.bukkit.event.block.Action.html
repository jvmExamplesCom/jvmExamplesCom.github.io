org.bukkit.event.block.Action

The org.bukkit.event.block.Action class is a Bukkit event that is fired when a player interacts with a block, either by left-clicking, right-clicking, or placing an item in a slot of a block. This event is not only responsible for firing when a block is interacted with but also any related actions such as opening a chest, using a hopper, changing the direction of a comparator, etc.

Constructors

The org.bukkit.event.block.Action class has three constructors, all of which take different generic Java parameter types: 

          
           
Action(Block block, Player player, Action action)
Action(Block block, Player player, Inventory inventory, Action action)
Action(Block block, Player player, ItemStack item, Action action)
The first constructor creates an instance of the Action class from a block, a player, and the action being taken (e.g. left-click, right-click, etc). The second constructor takes an inventory as an extra parameter; this is typically used for containers such as chests. Finally, the third constructor accepts an item stack as an extra parameter; this is used for when an item is being placed in a slot of a block.

Methods

The org.bukkit.event.block.Action class has several methods which are used to access the data associated with the action being taken. These include the following methods: 

          
           
getAction()
getBlock()
getInventory()
getItem()
getPlayer()
The getAction() method returns an enum representing the action being taken (e.g. left-click, right-click, etc). The getBlock() method returns the block that is interacted with. The getInventory() method returns the inventory associated with the block being interacted with, if any. The getItem() method returns the item associated with the action, if any. Finally, the getPlayer() method returns the player who initiated the action.

Example Usage

The following example shows how the org.bukkit.event.block.Action class can be used to detect when a player interacts with a block. The example shows how to detect when a player left-clicks on a block and prints out a message indicating so. 

          
           
@EventHandler
public void onBlockInteraction(Action event) {  
    if (event.getAction() == Action.LEFT_CLICK_BLOCK) {
        System.out.println("Player " + event.getPlayer().getName() + 
            " left-clicked on block " + event.getBlock());
    }
}
