Java com.intellij.openapi.fileEditor.FileDocumentManager

The com.intellij.openapi.fileEditor.FileDocumentManager class is an abstract class that provides convenient access to IntelliJ projects' open documents. It provides several methods to manage documents of various types, such as TextDocuments, BinaryDocuments, and SourceDocuments.

Methods for Managing Documents

The com.intellij.openapi.fileEditor.FileDocumentManager class has a variety of methods for managing documents. These include:

  • 
            
             
createDocument(VirtualFile file)
    : Creates a new document for the given file.
  • 
            
             
getCachedDocument(VirtualFile file)
    : Returns the cached document for the given file.
  • 
            
             
saveDocument(Document document)
    : Saves the given document.
  • 
            
             
getUnsavedDocuments()
    : Returns a list of all unsaved documents.
  • 
            
             
getModifiedDocuments()
    : Returns a list of all modified documents.
  • 
            
             
reloadFromDisk(VirtualFile file)
    : Reloads the contents of the given file from disk.

Document document = FileDocumentManager.getInstance().createDocument(file);

VirtualFile file = ...;

The above code gets a cached document for the given file.

Getting a List of Unsaved Documents

The above code gets a list of all unsaved documents.

Getting a List of Modified Documents

The above code gets a list of all modified documents.
