org.codehaus.plexus.util.FileUtils

org.codehaus.plexus.util.FileUtils is a class in the Java language that contains several useful file-related utility methods. It can be used to read and write files, list files and directories, create directories, delete files and directories, and more. Reading Files

The FileUtils class provides several methods for reading from files. The most basic one is readFileToString(), which reads the contents of a file into a String object:


          
           
String content = FileUtils.readFileToString(new File("myfile.txt"));

The returned String contains the contents of the specified file. If you want to read the contents of a file as a byte array, you can use the readFileToByteArray() method instead:

Writing Files

FileUtils.writeStringToFile(new File("foo.txt"), "Foo!");

Listing Files and Directories

The listFiles() method can be used to list all of the files and subdirectories in a directory:


          
           
File rootDir = new File("/path/to/dir");
Collection<File> files = FileUtils.listFiles(rootDir, null, true);

for (File file : files) {
    System.out.println(file.getAbsolutePath());
}

The above code prints out the absolute path for every file and subdirectory in the specified root directory. The second argument is a filename filter, which can be used to only include files with a certain name or extension. The third argument controls whether hidden files should be included in the listing.

Creating Directories

The FileUtils class provides several methods for creating directories. The most basic one is forceMkdir(), which creates a directory at the specified path:


          
           
FileUtils.forceMkdir(new File("/tmp/foo"));

If the parent directories of the specified directory do not already exist, they will be automatically created. If the specified directory already exists, it will be left untouched.

Deleting Files and Directories

The FileUtils class also provides methods for deleting files and directories. The deleteQuietly() method deletes a file or directory at the specified path, if it exists:


          
           
FileUtils.deleteQuietly(new File("/tmp/foo"));

If the specified file or directory does not exist, nothing will happen. If the file is a directory, it will be deleted recursively, deleting all of its contents along with it.
