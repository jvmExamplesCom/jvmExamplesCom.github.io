org.codehaus.plexus.util.FileUtils

org.codehaus.plexus.util.FileUtils is a class in the Java language that contains several useful file-related utility methods. Reading Files

String content = FileUtils.readFileToString(new File("myfile.txt"));

FileUtils.writeStringToFile(new File("foo.txt"), "Foo!");

Listing Files and Directories

The listFiles() method can be used to list all of the files and subdirectories in a directory:


          
           
File rootDir = new File("/path/to/dir");
Collection<File> files = FileUtils.listFiles(rootDir, null, true);

for (File file : files) {
    System.out.println(file.getAbsolutePath());
}

The above code prints out the absolute path for every file and subdirectory in the specified root directory. Creating Directories

FileUtils.forceMkdir(new File("/tmp/foo"));

If the parent directories of the specified directory do not already exist, they will be automatically created. If the specified directory already exists, it will be left untouched.

Deleting Files and Directories

If the specified file or directory does not exist, nothing will happen. If the file is a directory, it will be deleted recursively, deleting all of its contents along with it.
