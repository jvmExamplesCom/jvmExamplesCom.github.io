javax.sound.sampled.SourceDataLine

The SourceDataLine class is part of the javax.sound.sampled package and is a subclass of the DataLine class. It can provide a source for data to be played such as audio from a file, from a microphone, or from a text-to-speech system. This class provides methods for reading, writing, and controlling audio data, allowing programmers to create applications that can play or record audio data.

Constructors

The SourceDataLine class has two constructors: 

          
           
public SourceDataLine(DataLine.Info info, AudioFormat format, int bufferSize)throws LineUnavailableException
public SourceDataLine(DataLine.Info info, AudioFormat format, int bufferSize, LineListener listener)throws LineUnavailableException

The first constructor initializes a SourceDataLine with the specified DataLine.Info object, audio format, and buffer size. The second constructor does the same, but with an added listener to listen for events.

Methods

The SourceDataLine class has several methods for reading, writing, and controlling audio data.

Writing data

The SourceDataLine class provides methods for writing data to the source:


          
           
public int write(byte[] b, int off, int len)

This method writes len bytes from the specified b array starting at offset off to the source. The return value is the number of bytes that were written.


          
           
public void drain()

This method drains data from the source, meaning it waits until all pending data has been played. This is useful for ensuring all data has been played before closing the source.


          
           
public void flush()

This method flushes any pending data from the source. This is useful for stopping audio playback without waiting for all pending data to be played.

The SourceDataLine class also provides methods for reading data from the source:


          
           
public int read(byte[] b, int off, int len)

This method reads len bytes from the specified b array starting at offset off from the source. The return value is the number of bytes that were read.

Controlling data

The SourceDataLine class provides methods for controlling the flow of audio data:


          
           
public void start()

This method starts the source playing audio data. This should be called after writing data to the source.


          
           
public void stop()

This method stops the source from playing audio data. This should be called before writing new data to the source.


          
           
public void close()

This method closes the source, releasing any resources associated with the source. This should be called when finished with the source.

Example

This example shows an implementation of a SourceDataLine in an audio player application.


          
           
import javax.sound.sampled.*;

public class AudioPlayer {

    //audio format
    AudioFormat audioFormat = new AudioFormat(8000.0f, 16, 1, true, false);

    //source data line
    SourceDataLine sourceDataLine;

    public void play(String soundFile){
        try{
            //get audio stream from file
            AudioInputStream audioInputStream = AudioSystem.getAudioInputStream(new File(soundFile));

            //create source data line
            DataLine.Info dataLineInfo = new DataLine.Info(SourceDataLine.class,audioFormat);
            sourceDataLine = (SourceDataLine) AudioSystem.getLine(dataLineInfo);

            //open source data line
            sourceDataLine.open(audioFormat);

            //start playing
            sourceDataLine.start();

            //read data from audio input stream
            int count;
            byte tempBuffer[] = new byte[1024];
            while((count = audioInputStream.read(tempBuffer,0,tempBuffer.length)) != -1){
                if(count > 0){
                    sourceDataLine.write(tempBuffer, 0, count);
                }
            }

            //drain and close source data line
            sourceDataLine.drain();
            sourceDataLine.close();

        }catch(LineUnavailableException lue){
            lue.printStackTrace();
        }catch (IOException ioe) {
            ioe.printStackTrace();
        }catch(UnsupportedAudioFileException uafe){
            uafe.printStackTrace();
        }
    }
}

In this example, a SourceDataLine is created using the AudioFormat and DataLine.Info objects. It is then opened, audio data is written to it, and it is drained and closed when playback is complete.
