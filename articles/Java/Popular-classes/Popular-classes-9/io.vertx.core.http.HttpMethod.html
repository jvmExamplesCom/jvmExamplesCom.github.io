Java: io.vertx.core.http.HttpMethod

The 

          
           
io.vertx.core.http.HttpMethod
class defines the HTTP methods. It allows developers to use the corresponding HTTP method represented by an enumerated type.

Methods

Here is a list of available methods for the 

          
           
io.vertx.core.http.HttpMethod
class: 
  • CONNECT
    : Establish a tunnel to the server identified by the given URI.
  • DELETE
    : Delete the specified resource.
  • GET
    : Retrieve whatever information (in the form of an entity) is identified by the Request-URI.
  • HEAD
    : Identical to GET except that the server MUST NOT return a message-body in the response.
  • OPTIONS
    : Returns the HTTP methods that the server supports for specified URL.
  • PATCH
    : Used to apply partial modifications to a resource.
  • POST
    : Submit an entity to the specified resource, often causing a change in state or side effects on the server.
  • PUT
    : Requests that the enclosed entity be stored under the supplied Request-URI.
  • TRACE
    : Used to invoke a remote, application-layer loop-back of the request message.
