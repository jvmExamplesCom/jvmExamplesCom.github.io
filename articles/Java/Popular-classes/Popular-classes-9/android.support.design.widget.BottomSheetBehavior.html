    android:id="@+id/bottom_sheet"
    android:layout_width="match_parent"
    android:layout_height="wrap_content"
    app:behavior_peekHeight="80dp">

    <!--Content goes here-->

</android.support.design.widget.CoordinatorLayout>

android:id="@+id/bottom_sheet"
. This id should be unique and also referenced when creating the BottomSheetBehavior.

Creating the BottomSheetBehavior

The peek height is one of the most important settings for the bottom sheet. This setting determines how much of the bottom sheet is shown when it is collapsed. To set the peek height, we can use the 

          
           
setPeekHeight()
method on the BottomSheetBehavior object. 

          
           
bottomSheetBehavior.setPeekHeight(100);
