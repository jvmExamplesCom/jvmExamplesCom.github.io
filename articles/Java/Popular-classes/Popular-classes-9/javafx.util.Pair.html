JavaFX.util.Pair

JavaFX.util.Pair is a class in Java which enables developers to group two objects together, usually of different types, into one object. This class can be very useful in programming when creating objects that need to contain multiple elements of different types. It is an important part of the JavaFX API, which allows developers to create user interfaces quickly and efficiently.

Creating a Pair

A Pair is created using two parameters, each representing the type of the two objects which will make up the Pair. For example, a Pair containing a String and an Integer may be written as:


          
           
Pair<String, Integer> myPair = new Pair<String, Integer>("Hello", 123);

Accessing the Elements of a Pair

A Pair contains two objects, which can be accessed via the 

getFirst()
and 
getSecond()
methods. These methods return the objects contained in the Pair in the order they were specified when the Pair was created. For example: 

          
           
System.out.println(myPair.getFirst()); // Prints "Hello"
System.out.println(myPair.getSecond()); // Prints "123"

Modifying the Elements of a Pair

The objects contained in a Pair can be changed using the 

setFirst()
and 
setSecond()
methods. These methods take one argument, which is the new object that it should replace the existing one. For example: 

          
           
myPair.setFirst("Goodbye"); // Replaces the first element with "Goodbye"
myPair.setSecond(456); // Replaces the second element with 456

System.out.println(myPair.getFirst()); // Prints "Goodbye"
System.out.println(myPair.getSecond()); // Prints "456"

Comparing Pairs

Two Pairs can be compared with the 
equals()
method. This method takes another Pair as an argument and returns 
true
if the two Pairs are equal, or 
false
if they are not. Equality is determined by comparing both elements, so two Pairs with the same objects but in different orders will not be considered equal. For example: 

          
           
Pair<String, Integer> myOtherPair = new Pair<String, Integer>("Goodbye", 456);

Pair<String, Integer> myThirdPair = new Pair<String, Integer>(456, "Goodbye");
