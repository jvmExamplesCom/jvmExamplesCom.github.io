Java javax.naming.spi.InitialContextFactory

The javax.naming.spi.InitialContextFactory class is a java API class that provides a way for applications to look up objects in the Java Naming and Directory Interface (JNDI). It is used to create an initial context and contains methods to locate objects in the naming and directory service, manage naming services, and bind or unbind objects.

Creating an Initial Context

props.put("java.naming.factory.initial",
        "org.example.MyInitialContextFactory");

InitialContext ctx = InitialContextFactory.getInitialContext(props);

Lookup Objects using InitialContext

Binding Objects to the Context
