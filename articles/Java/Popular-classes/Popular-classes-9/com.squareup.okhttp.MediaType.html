com.squareup.okhttp.MediaType

The MediaType class from the com.squareup.okhttp package provides a way to represent media types such as images, audio, video, and so on and provides convenience methods for easily creating and matching them. parse(String)

MediaType textHtmlUtf8 = MediaType.parse("text/html; charset=utf-8");

MediaType applicationJson = MediaType.create("application", "json");

boolean isTextHtml = textHtmlUtf8.match("text/html");
