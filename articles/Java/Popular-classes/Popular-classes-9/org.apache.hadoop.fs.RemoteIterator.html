Introduction to Remote Iterator in Java

The org.apache.hadoop.fs.RemoteIterator is a Java class from the Hadoop API which enables developers to iterate through remote file systems. The RemoteIterator class is part of the Hadoop API, so developers have access to a variety of features. The iterator can be used when writing map reduce jobs, or when reading/writing files to any remote file system. The following are some of the features of the RemoteIterator:

  • Enables manipulation of data files, as well as reading and writing.
