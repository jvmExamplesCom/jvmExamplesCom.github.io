AbstractAuthenticationProcessingFilter

The AbstractAuthenticationProcessingFilter is a base class for authentication processing filters provided by Spring Security. It is an abstract class which must be extended for a particular authentication mechanism to be supported. It provides essential features such as: requiring that the request already be authenticated, performing additional authentication if necessary and then passing control over to the AuthenticationManager.

Initializing the Filter

In order to initialize the filter, you will need to extend the AbstractAuthenticationProcessingFilter and provide two methods: one for obtain the request path and the other for obtain the request method.


          
           
public class CustomAuthFilter extends AbstractAuthenticationProcessingFilter {

    public CustomAuthFilter(String defaultFilterProcessesUrl) {
        super(defaultFilterProcessesUrl);
    }

    @Override
    protected String getHttpMethod() {
        return "POST";
    }

    @Override
    protected String[] getRequiredAttributes() {
        return new String[] {"username", "password"};
    }

}
