The org.apache.hadoop.mapreduce.InputFormat is a Java API at the heart of Apache Hadoop's MapReduce framework. It provides the ability to read data from an input source and to format it into key-value pairs for processing by the map tasks.

Creating an InputFormat

InputFormats are created in order to read data from an input source. The most basic type of InputFormat is the FileInputFormat, which is created by extending the org.apache.hadoop.mapreduce.InputFormat class. FileInputFormat is used to read files from HDFS or any other file system, and it can be extended to read specific formats such as Avro, SequenceFiles, JSON, and so on.
