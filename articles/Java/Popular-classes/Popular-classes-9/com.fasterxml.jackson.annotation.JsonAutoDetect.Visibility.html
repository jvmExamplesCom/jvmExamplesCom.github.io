Introduction to com.fasterxml.jackson.annotation.JsonAutoDetect.Visibility

com.fasterxml.jackson.annotation.JsonAutoDetect.Visibility is an enum that defines how visible a field or method must be in order to be included in the serialization/deserialization process. PUBLIC_ONLY
