javafx.geometry.Orientation

The javafx.geometry.Orientation class represents the types of orientations available for geometry shapes in JavaFX. It is part of the JavaFX Geometry API, which provides a framework for defining geometrical regions.

The Orientation class defines two constants representing horizontal and vertical orientations: HORIZONTAL and VERTICAL. These constants can be used to specify an orientation when creating geometry shapes, or when setting the orientation of existing shapes.

Creating Shapes With Orientation

When creating a new geometry shape such as a Rectangle, you can provide a specific orientation in the constructor. The following example shows how to create a Rectangle with a vertical orientation:


          
           
Rectangle rect = new Rectangle(width, height, Orientation.VERTICAL);

This creates a new Rectangle with the specified width and height, and a vertical orientation.

Setting Orientation Of Existing Shapes

If you need to set the orientation of an existing geometry shape, you can use the setOrientation() method. For example, if you have an existing Rectangle object called rect, you can set its orientation to horizontal using the following code:


          
           
rect.setOrientation(Orientation.HORIZONTAL);

This sets the orientation of the Rectangle to horizontal, without changing its other properties.

Conclusion

The javafx.geometry.Orientation class provides constants for specifying horizontal and vertical orientations for geometry shapes in JavaFX. You can use these constants in the constructors for geometry shapes, or use the setOrientation() method to change the orientation of existing shapes.
