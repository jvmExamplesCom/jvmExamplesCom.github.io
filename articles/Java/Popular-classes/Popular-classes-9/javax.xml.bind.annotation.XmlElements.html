javax.xml.bind.annotation.XmlElements

The 

          
           
javax.xml.bind.annotation.XmlElements
annotation is used to indicate the allowed elements of a collection element or an indexed property, it is an active representation of xml schema constructs like 

          
           
xs:sequence
and 

          
           
xs:choice
. The 

          
           
XmlElements
annotation is defined as below: 

          
           
@Target({METHOD, FIELD, ANNOTATION_TYPE, CONSTRUCTOR, PARAMETER, TYPE})
@Retention(RUNTIME)
public @interface XmlElements
{
    /**
     * Collection of XML element declarations.
     */
    XmlElement[] value();
}
This annotation groups multiple 

          
           
XmlElement
annotations together. The 

          
           
XmlElement
annotation is used to indicate a class that will be a member of the collection element or indexed property.

Examples

Here's a simple example of how to use the 

          
           
XmlElements
annotation: 

          
           
@XmlElements({
    @XmlElement(name="foo", type=Foo.class),
    @XmlElement(name="bar", type=Bar.class)
})
public List<Object> list;
In this example, we have a 

          
           
List
that can contain objects of type 

          
           
Foo
and 

          
           
Bar
. The 

          
           
XmlElements
annotation is used to indicate that this is the case. Let's look at another example that shows how to use the 

          
           
XmlElements
annotation with an indexed property: 

          
           
@XmlElements({
    @XmlElement(name="foo", type=Foo.class),
    @XmlElement(name="bar", type=Bar.class)
})
@XmlAccessorType(XmlAccessType.FIELD)
public class MyClass {
    @XmlElement(name="item")
    public List<Object> items;
}
In this example, we have a 

          
           
List
that contains objects of type 

          
           
Foo
and 

          
           
Bar
. The 

          
           
XmlElements
annotation is used to indicate that this is the case and the 

          
           
XmlElement
annotation is used to indicate that the 

          
           
items
