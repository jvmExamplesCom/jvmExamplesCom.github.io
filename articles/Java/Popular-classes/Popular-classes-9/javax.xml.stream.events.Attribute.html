javax.xml.stream.events.Attribute

The javax.xml.stream.events.Attribute class represents an attribute in an XML document. An Attribute is reported as a separate event, instead of being part of a StartElement. An Attribute can appear at any position within the scope of a StartElement.

An attribute may be either namespace scoped or not-namespace scoped. To distinguish the two types of attribute the isNamespaceScoped() method is defined. If an attribute is not namespace scoped then it is considered to be an attribute of the current namespace and must not have a prefix. In addition, an attribute may be optional or required.

Example Code

Below is an example of how to create an attribute:


          
           
Attribute attr = f.createAttribute("name", "value");
You can also set the namespace to an attribute: 

          
           
attr.setName("xsi:schemaLocation");
attr.setNamespace("http://www.w3.org/2001/XMLSchema-instance");
To retrieve the value of an attribute: 

          
           
String value = attr.getValue();
And to check if an attribute is namespace scoped: 

          
           
boolean isNamespaceScoped = attr.isNamespaceScoped();
