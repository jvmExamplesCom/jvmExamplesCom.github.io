android.widget.SimpleCursorAdapter

Android’s SimpleCursorAdapter is an adapter for showing data from a Cursor object in a ListView or GridView. It provides a means of displaying and interacting with data collections from a database, making it easier for developers to work with cursors and related data from their Java applications.

Creating a SimpleCursorAdapter

To create a SimpleCursorAdapter, you must provide the context, a layout resource id, and a Cursor. The context is usually the current Activity or Fragment, the layout resource id is a reference to the layout that will be used to display the data, and the Cursor is the result of a query made to the database (or ContentProvider).

For example, to create a SimpleCursorAdapter that shows information from a query made to a SQLite database, the code might look something like this:


          
           
String[] projection = {
  "_id",
  "name",
  "email"
};

// Create cursor and query for data
Cursor cursor = myDatabase.query("contacts", projection, null, null, null, null, null);

// Create SimpleCursorAdapter
int layout = android.R.layout.simple_list_item_1;
SimpleCursorAdapter adapter = new SimpleCursorAdapter(this, layout, cursor, projection, new int[] {android.R.id.text1});

The SimpleCursorAdapter constructor takes the context, layout, Cursor, and the column names from the Cursor itself. It also takes an array of views, which contains the view ids for the views that will be populated with data from the Cursor.

Using a SimpleCursorAdapter

Once the SimpleCursorAdapter has been created, it can be used in a ListView or GridView as the data source. To use it, set the adapter on the ListView/GridView, like so:


          
           
listView.setAdapter(adapter);

Now, when the ListView or GridView is displayed, it will show the data from the Cursor, using the layout provided when creating the SimpleCursorAdapter.

Updating the Data

If the data in the Cursor is changed (e.g. a row is inserted or deleted), then the SimpleCursorAdapter needs to be notified of the changes. This is done by calling the changeCursor() method on the adapter, passing in the new Cursor.


          
           
// Query database and get new Cursor
Cursor newCursor = myDatabase.query(...);
adapter.changeCursor(newCursor);

The SimpleCursorAdapter will then update the view with the contents of the new Cursor.

Android’s SimpleCursorAdapter makes it easy to display and interact with data from a database. To create a SimpleCursorAdapter, you need to provide the context, a layout resource id, and a Cursor. Once the adapter has been created, it can then be used as the data source for a ListView or GridView. If the data in the Cursor is changed, the adapter should be notified of the change by calling the changeCursor() method with the new Cursor.
