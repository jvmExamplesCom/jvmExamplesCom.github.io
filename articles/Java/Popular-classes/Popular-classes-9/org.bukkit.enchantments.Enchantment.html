Introduction to org.bukkit.enchantments.Enchantment

org.bukkit.enchantments.Enchantment is an API class that is used to enchant items in Minecraft. It provides several methods and constants for manipulating enchantments on items, such as adding, removing, and changing their levels. This API class is often used for plugins that provide custom enchantments, or for plugins that need to manipulate existing enchantments.

Enchantment.createEnchantment(String key, int level)

key

level

Enchantment
. Then, you can add this enchantment to an item using the 

          
           
ItemMeta.addEnchant(Enchantment enchantment, int level, boolean ignoreLevelRestriction)

Enchantment

level

ItemMeta

Removing Enchantments from Items

ItemMeta

Changing the Level of Enchantments on Items
