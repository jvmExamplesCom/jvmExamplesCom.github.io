Introduction to org.bukkit.enchantments.Enchantment

org.bukkit.enchantments.Enchantment is an API class that is used to enchant items in Minecraft. It provides several methods and constants for manipulating enchantments on items, such as adding, removing, and changing their levels. This API class is often used for plugins that provide custom enchantments, or for plugins that need to manipulate existing enchantments.

Creating Enchantments and Adding Them to Items

To create an enchantment, use the 

          
           
Enchantment.createEnchantment(String key, int level)
method, where 

          
           
key
is a string representing the name of the enchantment and 

          
           
level
is the enchantment level. This returns an instance of 

          
           
Enchantment
. Then, you can add this enchantment to an item using the 

          
           
ItemMeta.addEnchant(Enchantment enchantment, int level, boolean ignoreLevelRestriction)
method, which adds the 

          
           
Enchantment
with the specified 

          
           
level
to the given 

          
           
ItemMeta
object.

Removing Enchantments from Items

To remove an enchantment from an item, use the 

          
           
ItemMeta.removeEnchant(Enchantment enchantment)
method, which removes the given 

          
           
Enchantment
from the 

          
           
ItemMeta
. Note that this does not decrease the level of the enchantment, only removes it from the item.

Changing the Level of Enchantments on Items

To modify the level of an enchantment on an item, use the 


          
           
ItemMeta.setEnchantLevel(Enchantment enchantment, int level)

Enchantment

level
. The level must be between 0 and 

          
           
Enchantment.getMaxLevel()
, otherwise it will throw an 

          
           
IllegalArgumentException
.

Getting Information about Enchantments

To get information about an enchantment, use the following methods: 


          
           
Enchantment.getName()
, 

          
           
Enchantment.getMaxLevel()
, 

          
           
Enchantment.getKey()

Enchantment.getStartLevel()

The org.bukkit.enchantments.Enchantment class provides an easy way to manipulate enchantments on items in Minecraft.
