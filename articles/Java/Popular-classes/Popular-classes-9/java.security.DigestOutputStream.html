The

A message digest, or cryptographic hash, is a value generated from an arbitrary amount of data that is highly likely to be unique when compared to other values generated from the same bulk data. This is useful for data authentication, where a sender can transmit some bulk data to a receiver along with a message digest and the latter can compute their own version of the digest in order to verify that the data has not been tampered with in transit.
