org.apache.avro.io.EncoderFactory

The EncoderFactory class provides a way to build an Encoder from an implementation-agnostic description. A description is serialized as a JSON object, which contains the following fields:

  • class: The fully qualified classname of the Encoder implementation.
  • config: Any configuration needed by the Encoder implementation.

The EncoderFactory will use the specified class and config to construct an Encoder instance.

Example

The following example shows how to use an EncoderFactory to construct an Encoder.


          
           
EncoderFactory factory = new EncoderFactory();
Map<string, object> config = new HashMap<string, object>();

// Add any additional configuration

Encoder encoder = factory.fromJson(config);
