JavaFX.Scene.Text.TextAlignment

JavaFX.Scene.Text.TextAlignment is an enumeration used to describe the alignment of text within a certain text node. The following enumerations are available: LEFT, RIGHT, CENTER, JUSTIFY and TEXT_BASELINE_JUSTIFY.

Example 1: Using TextAlignment.LEFT

The following example creates a Text object and sets the text alignment to LEFT:


          
           
Text text = new Text("This text is left aligned.");
text.setTextAlignment(TextAlignment.LEFT);

Example 2: Using TextAlignment.RIGHT

The following example creates a Text object and sets the text alignment to RIGHT:


          
           
Text text = new Text("This text is right aligned.");
text.setTextAlignment(TextAlignment.RIGHT);

Example 3: Using TextAlignment.CENTER

The following example creates a Text object and sets the text alignment to CENTER:


          
           
Text text = new Text("This text is center aligned.");
text.setTextAlignment(TextAlignment.CENTER);

Example 4: Using TextAlignment.JUSTIFY
