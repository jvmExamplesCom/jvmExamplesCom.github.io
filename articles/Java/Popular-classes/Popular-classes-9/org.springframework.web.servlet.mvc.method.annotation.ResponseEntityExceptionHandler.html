org.springframework.web.servlet.mvc.method.annotation.ResponseEntityExceptionHandler

The ResponseEntityExceptionHandler class is a convenient base class for any type of custom exception handler that handles exceptions using the ResponseEntity return type on all @ExceptionHandler methods. It provides an option to override the status code via the ResponseEntityExceptionHandler#handleExceptionInternal( Exception, Object, HttpHeaders, HttpStatus, WebRequest ) method.

Example

Here is an example of how to use ResponseEntityExceptionHandler in a Spring Boot application. This example will demonstrate how to handle all exceptions with a custom response object.


          
           
@ControllerAdvice
public class GlobalExceptionHandler extends ResponseEntityExceptionHandler {
 
    @Override
    protected ResponseEntity<Object> handleExceptionInternal(Exception ex, Object body, HttpHeaders headers, HttpStatus status, WebRequest request) {
        ErrorResponse errorResponse = new ErrorResponse(status.value(), ex.getMessage());
        return new ResponseEntity<Object>(errorResponse, status);
    }
}

In the example we are extending the ResponseEntityExceptionHandler class to create a custom exception handler that will respond with a custom error response instead of an HTTP error. The example will catch all errors and return a custom error response with an appropriate HTTP status code.
