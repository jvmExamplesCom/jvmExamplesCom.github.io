javax.xml.bind.annotation.XmlElementRef

The javax.xml.bind.annotation.XmlElementRef is an annotation used to indicate an XML element for a Java property or field whose type is an instance of JAXBElement or an interface or a superclass thereof. It maps an XML element to a Java property, allowing for the correct marshalling and unmarshalling of the objects upon reading and writing data.

Code Examples

Here are some code examples to demonstrate how to use this annotation:


          
           
@XmlElementRef(name = "customer", 
                    namespace = "http://www.example.com/customers", 
                    type = Customer.class) 
protected JAXBElement<Customer> customer;

          

          
           
@XmlElementRefs({
    @XmlElementRef(name = "name", 
                   namespace = "http://www.example.com/customers", 
                   type = Name.class),
    @XmlElementRef(name = "phoneNumber", 
                   namespace = "http://www.example.com/customers", 
                   type = PhoneNumber.class) 
})
The above example shows the use of 
@XmlElementRef
in conjunction with the 
@XmlElementRefs
annotation. The code creates two elements - ‘name’ and ‘phoneNumber’ - each with a corresponding namespace, and with the corresponding class being used for each element type. 

          
           
@XmlElementRef(name = "shippingAddress", 
                    namespace = "http://www.example.com/customers", 
                    type = ShippingAddress.class, 
                    required = false)
protected JAXBElement<ShippingAddress> shippingAddress;

Here, we have provided an example of how to make an element optional. The 

required = false
part of the annotation indicates that the element is optional and doesn't need to be present when marshalling or unmarshalling.

Conclusion

The 

          
           
javax.xml.bind.annotation.XmlElementRef
annotation allows for the correct marshalling and unmarshalling of Java objects when working with XML. This annotation enables you to map an XML element to a Java property, and provides options for making elements optional or required.
