Android ListFragment

Android ListFragment is a type of Fragment that provides an easy adapter-based interface for inserting items into a ListView. ListFragment allows developers to quickly and easily implement a list view in their Android apps. It eliminates the need to write tedious code to dynamically populate the list view, add header and footer items, and manage multiple types of selections.

Implementation

Using the ListFragment is easy. To create a ListFragment, simply create a Fragment class that extends the ListFragment class. Then override the onCreate() method and set the ListAdapter. The ListAdapter is responsible for supplying data to the ListView.


          
           
public class MyListFragment extends ListFragment {
  @Override
  public void onCreate(Bundle savedInstanceState) {
    super.onCreate(savedInstanceState);
    setListAdapter(new ArrayAdapter<String>(getContext(),
        android.R.layout.simple_list_item_1, 
        new String[] { "Item 1", "Item 2", "Item 3" }));
  }
}
