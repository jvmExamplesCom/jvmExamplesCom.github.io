Introduction to android.opengl.EGLSurface

android.opengl.EGLSurface is an API for creating and managing surface for OpenGL ES consumption in Android. This package contains the various classes used to configure, create and destroy surfaces. It also provides support for querying and manipulating EGL contexts associated with a particular surface.

An EGLSurface can be created from a 

          
           
Surface

EGLSurface createWindowSurface(EGLDisplay display, EGLConfig config, Object nativeWindow, Attribute[] attrib_list);
Where 

          
           
display

config
is a valid EGLConfig object, 

          
           
nativeWindow
is the original window type (such as a Surface, SurfaceHolder or SurfaceTexture), and 

          
           
attrib_list

Destroying an EGLSurface

An EGLSurface can be destroyed using the API: 

          
           
void destroySurface(EGLDisplay display, EGLSurface surface);

display

surface

It is possible to query various properties of an EGLSurface, such as its width and height, color depth, and more. The API for doing this is:
