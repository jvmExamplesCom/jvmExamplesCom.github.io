Introduction to org.jboss.arquillian.test.api.ArquillianResource

org.jboss.arquillian.test.api.ArquillianResource is a Java annotation which allows injection of resources into test cases that are run as part of an Arquillian test suite. Resources can include anything from the test class, such as method parameters and fields, to test containers, deployment archives and even URL resources.

Using ArquillianResource in Tests

In order to use the ArquillianResource annotation, simply add it to any test method which accepts an argument of type ArquillianResource. This argument will then be populated with the requested resource automatically by Arquillian.

For example, to inject the current deployment archive into a test method:


          
           
@Test
public void shouldInjectDeploymentArchive(@ArquillianResource Deployment deployment) {
  // Use the deployment instance here
}

The same can be done to inject a URL for the deployed application:


          
           
@Test
public void shouldInjectDeploymentURL(@ArquillianResource URL url) {
  // Use the URL instance here
}

The ArquillianResource annotation can be used to inject many different types of resources. Here are some of the more common ones:

Test Class


          
           
@ArquillianResource TestClass testClass;

Test Instance


          
           
@ArquillianResource TestInstance testInstance;

Container


          
           
@ArquillianResource Container container;

Configuration 


          
           
@ArquillianResource Configuration configuration;

Deployment


          
           
@ArquillianResource Deployment deployment;

URL 


          
           
@ArquillianResource URL url;

Conclusion

The ArquillianResource annotation makes it easy to inject resources into Arquillian tests, allowing for cleaner and more concise tests than if the resources were manually acquired and injected. Although the most common use case is for testing deployments, ArquillianResource can be used to inject almost any kind of resource into a test, making it very versatile.
