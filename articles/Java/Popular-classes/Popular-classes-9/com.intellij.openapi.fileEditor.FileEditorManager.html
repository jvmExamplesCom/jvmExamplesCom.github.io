What is com.intellij.openapi.fileEditor.FileEditorManager?

com.intellij.openapi.fileEditor.FileEditorManager is an interface for services that provide access to the document model associated with file systems. It allows applications to modify and manage documents using the IntelliJ Platform's document model. The FileEditorManager also provides methods to open and close files, access editors associated with files, and more.

Example Code


          
           
FileEditorManager fileEditorManager = FileEditorManager.getInstance(project);
VirtualFile virtualFile = fileEditorManager.getSelectedFiles()[0]; 
fileEditorManager.openFile(virtualFile, true); 
FileEditor[] editors = fileEditorManager.getEditors(virtualFile);

          

          
           
fileEditorManager.closeFile(virtualFile);

Output

The above code opens a file based on the VirtualFile and IntelliJ IDEA project instance, then retrieves an array of editors associated with the file and finally closes the file.
