Introduction to org.apache.hadoop.hdfs.MiniDFSCluster

Hadoop Distributed File System (HDFS) is an open-source distributed file system designed to run on commodity hardware. It provides a reliable, highly available, and scalable way of managing large amounts of data. The org.apache.hadoop.hdfs.MiniDFSCluster class is a small cluster used for testing HDFS and MapReduce programs without needing to deploy a full-scale Hadoop installation. This article will cover the basics of setting up and using a MiniDFSCluster.

Creating a MiniDFSCluster Instance

A MiniDFSCluster instance is created by providing a Configuration object to the MiniDFSCluster constructor. This configuration must contain settings appropriate for the underlying file system. For example, the following code creates a basic configuration for a local file system: 

          
           
Configuration conf = new Configuration();
conf.set("fs.defaultFS", "file:///");
MiniDFSCluster cluster = new MiniDFSCluster.Builder(conf).build();
The MiniDFSCluster instance also needs to be initialized before it can be used. This is done by calling its start() method: 

          
           
cluster.start();
Once the MiniDFSCluster has been started, a FileSystem instance can be created to access the underlying file system: 

          
           
FileSystem fs = cluster.getFileSystem();

Using the Cluster

Once the cluster is up and running, it can be used just like any other HDFS instance. Files can be created, modified, and deleted as normal. The following code demonstrates how to create a file in the file system: 

          
           
Path path = new Path("/file.txt");
FSDataOutputStream out = fs.create(path);
out.write("Hello, World!".getBytes());
out.close();
And the following code demonstrates how to read a file from the file system: 

          
           
Path path = new Path("/file.txt");
FSDataInputStream in = fs.open(path);
byte[] buffer = new byte[1024];
int bytesRead = in.read(buffer);
String content = new String(buffer, 0, bytesRead);
System.out.println(content);
in.close();

Cleaning Up

Once finished with the MiniDFSCluster, it should be shut down by calling its shutdown() method. This will free up the resources allocated to the cluster: 

          
           
cluster.shutdown();
