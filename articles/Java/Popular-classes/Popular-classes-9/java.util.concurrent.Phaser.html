java.util.concurrent.Phaser

Java.util.concurrent.Phaser is a synchronizer that allows multiple threads to wait for each other to reach a common barrier point. It is a reusable synchronization barrier, similar to CyclicBarrier and CountDownLatch but with more flexible usage. The difference between Phaser and these two is that it allows the registration of participants after the initialization, and supports dynamic arrival and departure of threads during execution.

Constructors of Phaser

The java.util.concurrent.Phaser class provides the following constructors.

Important Methods Of Phaser

The java.util.concurrent.Phaser class provides the following methods:

  • arriveAndAwaitAdvance(): It causes the current thread to arrive at this phaser and wait until all other threads in the same phaser arrive as well.
  • awaitAdvance(int): It causes the current thread to wait until a given phase number is reached.
  • arrivedParties(): It returns the number of parties currently registered at this phaser.
  • arriveAndDeregister(): This method allows a party to arrive and concurrently deregister from this phaser.
  • getRegisteredParties(): It returns the number of parties registered at this phaser.
  • bulkRegister(int): It creates and adds a new unarrived party to this phaser.

Example: CountDownLatch vs. Phaser

Let us understand the differences between CountDownLatch and Phaser by the example given below:


          
           
import java.util.concurrent.Phaser;
import java.util.concurrent.CountDownLatch;

public class PhaserDemo {
   public static void main(String[] args) {
      // Creating CountdownLatch
      CountDownLatch latch = new CountDownLatch(3);
 
      System.out.println("CountDownLatch has been created.");
 
      // Creating Phaser
      Phaser phaser = new Phaser(3);
 
      System.out.println("Phaser has been created.");
 
      // Creating 2 threads, t1 and t2
      Thread t1 = new Thread(new Task(latch), "t1");
      Thread t2 = new Thread(new Task(latch), "t2");
 
      // Starting 2 threads, t1 and t2
      t1.start();
      t2.start();
 
      // Waiting for both threads to complete
      try {
         latch.await();
         System.out.println("CountDownLatch completed.");
      } catch (InterruptedException e) {
         e.printStackTrace();
      }
 
      // Waiting for both threads to complete
      phaser.arriveAndAwaitAdvance();
      System.out.println("Phaser completed.");
   }
}
