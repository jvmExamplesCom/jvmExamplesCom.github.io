Java.awt.FileDialog

Java.awt.FileDialog is a class used to create a dialog window for the user to select a file. It is part of the Java Abstract Window Toolkit (AWT). This allows developers to create and display a file selection dialog window with the help of this class. The dialog window can be configured to display different kinds of files, such as image files or text files.

Example 1 - Simple File Dialog

The following code example shows how to create and invoke a FileDialog object to display a file selection dialog window. This example assumes the main method is being called from the same application that contains the class definition.


          
           
//Create a new FileDialog object 
FileDialog fd = new FileDialog(this, "Choose a file"); 

//Set the size and location of the dialog window 
fd.setSize(500, 400); 
fd.setLocation(100, 100); 

//Display the dialog window 
fd.setVisible(true);

Example 2 - Customizing the File Dialog

The following code example shows how to customize the file selection dialog window by setting the filter mode, setting the default directory, and setting the list of available file types.


          
           
//Create a new FileDialog object 
FileDialog fd = new FileDialog(this, "Choose a file"); 

//Set the size and location of the dialog window 
fd.setSize(500, 400); 
fd.setLocation(100, 100); 

//Set the filter mode and list of file types 
fd.setFilterMode(FileDialog.FILES_ONLY); 
fd.setFilenameFilter("jpg, gif, png"); 

//Set the default directory 
fd.setDirectory("/Documents/images"); 

//Display the dialog window 
fd.setVisible(true);

Example 3 - Accessing the selected file

The following code example shows how to access the selected file from a FileDialog object.


          
           
//Create a new FileDialog object 
FileDialog fd = new FileDialog(this, "Choose a file"); 

//Set the size and location of the dialog window 
fd.setSize(500, 400); 
fd.setLocation(100, 100); 

//Display the dialog window 
fd.setVisible(true); 

//Get the name and directory of the selected file

Example 4 - Closing the file dialog after a selection

The following code example shows how to close the file selection dialog window after a file has been selected.


          
           
//Create a new FileDialog object 
FileDialog fd = new FileDialog(this, "Choose a file"); 

//Set the size and location of the dialog window 
fd.setSize(500, 400); 
fd.setLocation(100, 100); 

//Display the dialog window 
fd.setVisible(true); 

//Dispose of the dialog window after the selection has been made
