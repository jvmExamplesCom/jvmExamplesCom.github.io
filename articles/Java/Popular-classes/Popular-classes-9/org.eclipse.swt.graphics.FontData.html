org.eclipse.swt.graphics.FontData

FontData fontData = new FontData(String name, int height, int style)

The font style is a combination of one or more of the following constants: SWT.NORMAL, SWT.BOLD, SWT.ITALIC, and SWT.ITALIC. BOLD. For example, the following code snippet creates a FontData object with name “Arial”, size 12, and style NORMAL: 

          
           
FontData fontData = new FontData("Arial", 12, SWT.NORMAL);

fontData.setName("Times New Roman");
