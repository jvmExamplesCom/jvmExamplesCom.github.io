Overview of org.springframework.security.authentication.AuthenticationServiceException

The 

          
           
org.springframework.security.authentication.AuthenticationServiceException

AuthenticationException

Below is an example of how 

          
           
AuthenticationServiceException
can be encountered in a Java program. 

          
           
try {
    authenticationManager.authenticate(authentication);

    logger.error("Error authenticating user: " + ex);
}
In this example, an authentication request is attempted and an instance of 

          
           
AuthenticationServiceException
is thrown should the authentication be unsuccessful.
