Overview of org.springframework.security.authentication.AuthenticationServiceException

The 

          
           
org.springframework.security.authentication.AuthenticationServiceException
is a subclass of the Spring Security's 

          
           
AuthenticationException
which is thrown when an authentication request could not be processed by the authentication provider due to a system error or misconfiguration. This exception can indicate both a configuration problem with the security context of the application or a system error condition.

Example 1: Authentication Service Exception

Below is an example of how 

          
           
AuthenticationServiceException
can be encountered in a Java program. 

          
           
try {
    authenticationManager.authenticate(authentication);
} catch(AuthenticationServiceException ex) {
    logger.error("Error authenticating user: " + ex);
}
In this example, an authentication request is attempted and an instance of 

          
           
AuthenticationServiceException
is thrown should the authentication be unsuccessful.

can be handled 

          
           
public class MyAppAuthenticationFailureHandler implements AuthenticationFailureHandler {

    private static final Logger logger = LoggerFactory.getLogger(MyAppAuthenticationFailureHandler.class);
