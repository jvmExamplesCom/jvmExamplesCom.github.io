org.redisson.Redisson

Redisson is the open-source Java library for easy use of Redis client-server database. It is designed to help create distributed, scalable, and fault-tolerant applications with ease. Redisson provides advanced data structures such as Queue, Lock, Bloom Filter, Bloom Set, Bloom Map, Cluster Map, Atomic Long, Semaphore and Executor Service. Redisson also offers a distributed compute platform with embedded support for task scheduling, automatic fail-over, and dynamic resource management.

Code Examples

Below are some examples of how to use org.redisson.Redisson:


          
           
// Create Redisson Client instance
Config config = new Config();
config.useSingleServer().setAddress("127.0.0.1:6379");
RedissonClient redisson = Redisson.create(config);

// Create RMap object
RMap<String, String> map = redisson.getMap("map");

// Put values in map
map.put("foo", "bar");

// Get value from map
String value = map.get("foo");

// Print value from map
System.out.println(value); // prints out "bar"

          

          
           
// Create RSet object
RSet<String> set = redisson.getSet("set");

// Add values to set
set.add("foo");
set.add("bar");

// Check if set contains value
boolean containsFoo = set.contains("foo");

// Print if set contains value
System.out.println(containsFoo); // prints out true

          

          
           
// Create RLock object
RLock lock = redisson.getLock("lock");

// Acquire lock
lock.lock();

// Do work

// Release lock
lock.unlock();
