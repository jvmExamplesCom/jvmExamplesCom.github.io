android.widget.ViewFlipper

android.widget.ViewFlipper is an Android class that allows the user to show multiple views in the same activity. It can be used to switch between different views, such as a list view, a grid view, or any other type of view. This can be useful for displaying multiple pieces of information on a single page, such as when displaying a form, or for navigating between different activities without having to reload the entire app.

Creating a ViewFlipper

To create a ViewFlipper, you must first define the layout of the activity containing the views. In this example, we will create a simple activity with two views: one for entering a name, and one for displaying a greeting. 

          
           
<RelativeLayout xmlns:android="http://schemas.android.com/apk/res/android"
    android:layout_width="match_parent"
    android:layout_height="match_parent" >

    <!-- Layout for name entry -->
    <LinearLayout
        android:id="@+id/name_entry_layout"
        android:orientation="vertical"
        android:layout_width="match_parent"
        android:layout_height="match_parent" >

        <EditText
            android:id="@+id/name_input"
            android:hint="enter your name"
            android:layout_width="match_parent"
            android:layout_height="wrap_content" />

        <Button
            android:id="@+id/submit"
            android:text="Submit"
            android:layout_width="match_parent"
            android:layout_height="wrap_content" />

    </LinearLayout>

    <!-- Layout for greeting display -->
    <LinearLayout
        android:id="@+id/greeting_layout"
        android:orientation="vertical"
        android:visibility="gone"
        android:layout_width="match_parent"
        android:layout_height="match_parent" >

        <TextView
            android:id="@+id/greeting_message"
            android:text="Hello! "
            android:layout_width="match_parent"
            android:layout_height="wrap_content" />

    </LinearLayout>


</RelativeLayout>
Once the layout has been defined, we can create our ViewFlipper. To do this, we create an instance of the ViewFlipper from the layout we just defined. 

          
           
ViewFlipper viewFlipper = findViewById(R.id.view_flipper);
We then add the two views to the ViewFlipper, by referencing their IDs. 

          
           
viewFlipper.addView(findViewById(R.id.name_entry_layout));
viewFlipper.addView(findViewById(R.id.greeting_layout));

Using the ViewFlipper

Now that we have our ViewFlipper setup, we can use it to switch between our two views. To show the greeting view, we can write the following code: 

          
           
viewFlipper.setDisplayedChild(1);
This will show the second child in the ViewFlipper's list of views, which is the greeting view. To show the name entry view, we can write the following code: 

          
           
viewFlipper.setDisplayedChild(0);
This will show the first child in the ViewFlipper's list of views, which is the name entry view. We can also use the next() and previous() methods to show the next or previous view in the list.

Example Usage

We can now use our ViewFlipper in our Activity. In this example, we'll create an Activity that allows the user to enter their name, and then displays a greeting upon submission. 

          
           
public class MainActivity extends AppCompatActivity {
    private EditText mNameInput;
    private TextView mGreetingMessage;
    private ViewFlipper mViewFlipper;
    
    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);
        
        // Initialize view components
        mNameInput = findViewById(R.id.name_input);
        mGreetingMessage = findViewById(R.id.greeting_message);
        mViewFlipper = findViewById(R.id.view_flipper);
        
        // Add click listener to submit button
        findViewById(R.id.submit).setOnClickListener(new View.OnClickListener() {
            @Override
            public void onClick(View v) {
                // Get name from input
                String name = mNameInput.getText().toString();
                
                // Set greeting message
                mGreetingMessage.setText("Hello " + name + "!");
                
                // Show greeting view
                mViewFlipper.setDisplayedChild(1);
            }
        });
    }
}
In this code, we first define the views we need to access in our Activity. We then add a click listener to the submit button, which gets the name from the EditText, sets the greeting message, and shows the greeting view.
