Android Content PM ShortcutManager

Android has been providing a wide range of utilities for app development, and one of them is the Android Content PM ShortcutManager. This helps the user to access their apps quickly from the launcher, by creating short cuts that can be customized and accessed quickly. The ShortcutManager provides a way of creating shortcuts and overriding existing ones, and supplies some convenience methods such as disabling, enabling, updating and removing the shortcuts. It is part of the Android framework and is mainly used to create Launcher Shortcuts.

Creating Shortcuts

The ShortcutManager provides an easy way to create and manage your shortcuts. To create a shortcut, you need to provide the following information:
  • Shortcut ID: A unique identifier for the shortcut.
  • Shortcut Intent: An Intent that is used when the shortcut is selected from the launcher.
  • Shortcut Icon: An icon resource to be displayed with the shortcut, or null if no icon is needed.
  • Shortcut Label: A string to be displayed as the label for the shortcut.
  • Shortcut Long Label: A longer string to be displayed as the task description for the shortcut.
Once you have all these details, you can create the shortcut using the following code: 

          
           
ShortcutManager shortcutManager = getSystemService(ShortcutManager.class);
ShortcutInfo shortcut = new ShortcutInfo.Builder(context, "shortcut-id")
  .setShortLabel("Shortcut Label")
  .setLongLabel("Shortcut Long Label")
  .setIcon(Icon.createWithResource(context, R.drawable.icon))
  .setIntent(new Intent(context, Activity.class))
  .build();
shortcutManager.addDynamicShortcuts(Collections.singletonList(shortcut));
This will add a shortcut with the given parameters to the Android Launcher.

Updating Shortcuts

To update a shortcut, you can use the following code: 

          
           
ShortcutManager shortcutManager = getSystemService(ShortcutManager.class);
ShortcutInfo shortcut = new ShortcutInfo.Builder(context, "shortcut-id")
  .setShortLabel("Updated Shortcut Label")
  .setLongLabel("Updated Shortcut Long Label")
  .setIcon(Icon.createWithResource(context, R.drawable.updated_icon))
    .setIntent(new Intent(context, UpdatedActivity.class))
    .build();
shortcutManager.updateShortcuts(Collections.singletonList(shortcut));
This will update the shortcut with the given parameters to the Android Launcher.

Removing Shortcuts

To remove a shortcut, you can use the following code: 

          
           
ShortcutManager shortcutManager = getSystemService(ShortcutManager.class);
shortcutManager.removeDynamicShortcuts(Collections.singletonList("shortcut-id"));
This will remove the shortcut with the given ID from the Android Launcher.

Disabling and Enabling Shortcuts

You can also temporarily disable or enable a shortcut using the following code: 

          
           
ShortcutManager shortcutManager = getSystemService(ShortcutManager.class);
shortcutManager.disableShortcuts(Collections.singletonList("shortcut-id"));
// To enable again
shortcutManager.enableShortcuts(Collections.singletonList("shortcut-id"));
This will disable the shortcut with the given ID, and it will not show up in the launcher. You can enable it again if needed.
