org.antlr.v4.runtime.dfa.DFA

The org.antlr.v4.runtime.dfa.DFA class represents a state machine in the form of Deterministic Finite Automata (DFA). This class provides methods to recognize and control the behavior of such machines, as well as to generate them from regular expressions.

Implementation

A DFA is represented as an array of states, with an initial state at index 0 and final states at any indices greater than that. It also contains internal information about the alphabet of characters used to construct it and the transition functions it implements.


          
           
public class DFA {
    private State[] states;
    private Character[] alphabet;
    private int initialState;
    private Set<Integer> acceptStates;
}

The transition function is implemented as a 2D array of integers, with the first dimension representing a start state, and the second dimension representing a character in the alphabet. The value at each position in the array is the index of the state that the machine will transition to upon seeing the corresponding character in the given state.


          
           
private int[][] transitions;

The DFA can be constructed from a regular expression in the form of a prefix tree, which represents all possible valid strings in the language of the regex. Then, a depth-first search can be performed on the tree to build the transition function of the DFA.

Methods

The DFA class provides several methods for working with its underlying finite automata. For example, the 

accepts()
method checks if a given string could be accepted by the automata, while the 
run()
method runs the machine on an entire string and returns the final state reached. 

          
           
public boolean accepts(String input) {
    int currentState = initialState;
    for (char c : input.toCharArray()) {
        int nextState = transitions[currentState][c];
        if (nextState == -1) {
            return false;
        }
        currentState = nextState;
    }
    return acceptStates.contains(currentState);
}

public int run(String input) {
    int currentState = initialState;
    for (char c : input.toCharArray()) {
        currentState = transitions[currentState][c];
    }
    return currentState;
}
