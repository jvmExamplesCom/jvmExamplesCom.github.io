org.apache.curator.framework.recipes.cache.PathChildrenCache

The org.apache.curator.framework.recipes.cache.PathChildrenCache provides a mechanism for listening to changes in children for a path as maintained by Zookeeper. This allows applications to react to changes in children immediately. PathChildrenCache is particularly useful for reacting to nodes being added, updated, or deleted below a given path.

Code Examples

The following shows an example of how to use the PathChildrenCache:


          
           
PathChildrenCache cache = new PathChildrenCache(client, "/parent/node", true);

// register listener to listen for events
cache.getListenable().addListener((client, event) -> {
   switch (event.getType()) {
      case CHILD_ADDED:
         System.out.println("Child Added: " + event.getData().getPath());
         break;
      case CHILD_REMOVED:
         System.out.println("Child Removed: " + event.getData().getPath());
         break;
      case CHILD_UPDATED:
         System.out.println("Child Updated: " + event.getData().getPath());
         break;
   }
});

cache.start();

This would print out something like the following when an event was triggered:


          
           
Child Updated: /parent/node/child1

The PathChildrenCache can also be used to fetch the current state when it is first started:


          
           
PathChildrenCache cache = new PathChildrenCache(client, "/parent/node", true);

// register listener to listen for events
cache.getListenable().addListener((client, event) -> {
   System.out.println("Event Type: " + event.getType());
   System.out.println("Event Data: " + event.getData());
});

// fetch initial data from zookeeper and stay up-to-date
cache.start(StartMode.BUILD_INITIAL_CACHE);

This would print out something like the following when it is started:


          
           
Event Type: CHILD_INITIALIZED 
Event Data: /parent/node/child1

Finally, the PathChildrenCache can be used to add, update, and remove child nodes with Zookeeper directly:


          
           
PathChildrenCache cache = new PathChildrenCache(client, "/parent/node", true);

// add a child node
String path = cache.create("/child1", byte[] data, CreateMode.EPHEMERAL);

// update a child node
cache.setData("/child1", byte[] newData);

// remove a child node
cache.delete("/child1");
