java.lang.ProcessBuilder.Redirect

The java.lang.ProcessBuilder.Redirect class provides a set of helper classes for redirecting input and output streams of subprocesses, used by the ProcessBuilder class to start processes.

This class is meant to provide an easy way to bypass the complications of setting up specific system-dependent redirection related properties. The helper classes are as follows:

  • INHERIT: inherit the current System.in, System.out or System.err of the current process.
  • PIPE: create an anonymous pipe to connect the standard output/input of the subprocess to the current process.
  • DISCARD: discard the output/input from the subprocess.
  • appendTo(File): redirect the output/input to the given file.

Using ProcessBuilder, you can specify two different types of redirects: input and output.

Input Redirects

Input redirects are used to provide input to a process. It can be done by setting the redirectInput(Redirect) method to an appropriate redirect object.


          
           
ProcessBuilder pBuilder = new ProcessBuilder("cat");
  pBuilder.redirectInput(Redirect.INHERIT);
  Process proc = pBuilder.start();

In the above example, the Redirect.INHERIT is used to inherit the System.in of the current process. This allows the process to accept input from the parent process’ System.in. Here, the process ‘cat’ will print whatever is entered in the command-line.

Output Redirects

Output redirects are used to redirect the output produced by a process either to a file or another process. It can be done by setting the redirectOutput(Redirect) method to an appropriate redirect object.


          
           
ProcessBuilder pBuilder = new ProcessBuilder("ls");
  pBuilder.redirectOutput(Redirect.appendTo(new File("output.txt")));
  Process proc = pBuilder.start();

In the above example, the Redirect.appendTo(File) is used to append the output of the process ‘ls’ to the specified file. Here, the output of the process ‘ls’ will be written to the output.txt file. All the existing content in the file will be preserved.
