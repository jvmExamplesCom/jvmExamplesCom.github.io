javax.swing.ListModel

javax.swing.ListModel is an interface that provides a common protocol for how to interact with lists of objects. It defines the methods which allow it to be used within the Swing framework. It supports the concept of a data model, or a way of representing and manipulating data. This allows it to provide a set of methods which can be used to access and manipulate the data.

ListModel Interface

The ListModel interface has several methods that are used to access and manipulate data within a list. The following is a brief description of each method.

  • int getSize()
    : This method returns the total number of elements that are in the list.
  • 
            
             
Object getElementAt(int index)
    : This method returns the element located at the specified index.
  • 
            
             
void addListDataListeners(ListDataListener listener)
    : This method adds a listener to the list. It will be notified when the contents of the list change.
  • 
            
             
void removeListDataListener(ListDataListener listener)
    : This method removes a listener from the list. It will no longer be notified when the contents of the list change.

ListModel Example

The following example shows how to create a simple list model and add some elements to it.


          
           
public class MyListModel extends AbstractListModel<String> {
  
    private List<String> items = new ArrayList<>();

    public MyListModel() {
        // Add some elements to the list
        items.add("Apple");
        items.add("Orange");
        items.add("Banana");
    }

    @Override
    public int getSize() {
        return items.size();
    }

    @Override
    public String getElementAt(int index) {
        return items.get(index);
    }
}
To use this list model, it needs to be set on a 
JList
as follows: 

          
           
JList<String> myList = new JList<>();
MyListModel listModel = new MyListModel();
myList.setModel(listModel);

The resulting list will contain "Apple", "Orange" and "Banana".
