org.springframework.data.redis.connection.RedisConnection

RedisConnection is an interface from the open-source library org.springframework.data.redis. It provides an API for connecting to Redis databases, executing commands and controlling transactions. This article provides an overview of the interface, details on its methods, and several code examples to demonstrate usage.

Overview

RedisConnection is intended as an abstraction layer across different Redis clients. The interface exposes a consistent set of methods that can be used to perform data access operations such as fetching, inserting, deleting and updating items in a Redis database. It also enables users to start and end transactions and control the behavior of the underlying Redis client.

Methods

RedisConnection contains several methods for data access and control. Here are some of the most commonly used methods:

  • get(byte[] key)
    : Fetches the value of a given key from the Redis database.
  • openPipeline()
    : Opens a new pipeline for batching commands. Commands executed within a pipe will be sent to the server in a single request.
  • multi()
    : Starts a new transaction. All commands executed within the transaction will be atomic.
  • discard()
    : Discards any pending commands in the transaction.
  • exec()
    : Executes all commands in the current transaction atomically.
  • closePipeline()
    : Closes any open pipelines.

Code Examples

The following code examples demonstrate how to use RedisConnection for various data access operations.

Fetching Data

The following example demonstrates how to fetch a key-value pair from the Redis database:


          
           
byte[] key = "key".getBytes();
byte[] value = redisConnection.get(key);
System.out.println("Value of key: " + new String(value));

Inserting Data

The following example demonstrates how to insert a key-value pair into the Redis database:


          
           
byte[] key = "key".getBytes();
byte[] value = "value".getBytes();
redisConnection.set(key, value);

Deleting Data

The following example demonstrates how to delete a key-value pair from the Redis database:


          
           
byte[] key = "key".getBytes();
redisConnection.del(key);

Open Pipeline

The following example demonstrates how to open a pipeline for batching commands:


          
           
redisConnection.openPipeline();
redisConnection.set("key1".getBytes(), "value1".getBytes());
redisConnection.set("key2".getBytes(), "value2".getBytes());
redisConnection.closePipeline(); // send both commands in a single request

Start Transaction

The following example demonstrates how to start a transaction:


          
           
redisConnection.multi();
redisConnection.set("key1".getBytes(), "value1".getBytes());
redisConnection.set("key2".getBytes(), "value2".getBytes());
redisConnection.exec(); // execute both commands atomically

Discard Transaction

The following example demonstrates how to discard a transaction:


          
           
redisConnection.multi();
redisConnection.set("key1".getBytes(), "value1".getBytes());
redisConnection.set("key2".getBytes(), "value2".getBytes());
