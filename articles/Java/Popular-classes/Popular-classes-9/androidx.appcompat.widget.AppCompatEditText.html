What is androidx.appcompat.widget.AppCompatEditText?

androidx.appcompat.widget.AppCompatEditText is an Android library that provides a custom editable text field for developers. Additionally, it allows developers to specify values such as input type, min/max length, and text alignment.

implementation 'com.android.support:appcompat-v7:26.1.1'

<androidx.appcompat.widget.AppCompatEditText
    android:id="@+id/edit_text"
