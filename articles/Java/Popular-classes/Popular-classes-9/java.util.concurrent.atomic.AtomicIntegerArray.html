java.util.concurrent.atomic.AtomicIntegerArray
class provides an array of volatile integers. It is used for atomic, thread-safe operations on an array of integer values. It provides threadsafe methods that can be used to atomically update and retrieve the array elements. The following example creates an AtomicIntegerArray with initial values:


          
           
int[] initialValues = {23, 4, 55}; 
AtomicIntegerArray integerArray = new AtomicIntegerArray(initialValues);
