com.intellij.openapi.util.io.FileUtil

Overview

com.intellij.openapi.util.io.FileUtil is a utility class in the IntelliJ open API library, which contains a wide range of frequently used methods to work with file system operations such as reading, writing, and searching. It provides a number of helper methods to make file IO easier. The FileUtil class is thread-safe and all its methods are marked final to ensure that they are not overridden.

Methods

The FileUtil class provides many useful methods for handling common file system operations. The most commonly used methods are listed below.

String filePath = "/path/to/file.txt";

     System.out.println("File created successfully");

This method is used for reading the contents of a file into a String. It takes two arguments - the file path as a String and the Charset to be used for decoding.


          
           
String filePath = "/path/to/file.txt";
String contents = FileUtil.readFileToString(filePath, StandardCharsets.UTF_8);

This method can be used to copy a file from one location to another. It takes two arguments - the source and destination file paths, both as Strings.


          
           
String sourcePath = "/path/to/source/file.txt";
