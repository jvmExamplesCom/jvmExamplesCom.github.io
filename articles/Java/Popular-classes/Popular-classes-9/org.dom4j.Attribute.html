org.dom4j.Attribute

The org.dom4j.Attribute class is a representation of an XML attribute in a DOM4J tree. It is used to access data stored as attributes within an Element, allowing for the manipulation and retrieval of information from an element's attributes. The Attribute class provides a number of convenience methods for accessing both the name and value of the attribute.

Creating Attributes

Attributes can be created directly from an Element or from a Document. Creating them from an Element allows for the attachment of attributes to that element.


          
           
Element el = new Element("myElement");
Attribute attr = new Attribute("myAttribute", "yourValue");
el.add(attr);

When creating an Attribute directly from a Document, the attribute will exist as its own object, independently of any elements:


          
           
Document doc = new Document();
Attribute attr2 = new Attribute("myOtherAttribute", "anotherValue");
doc.add(attr2);

Retrieving Attribute Values

Attributes can be retrieved from an Element using the getAttribute() method. This method takes a String representing the name of the attribute being searched for as a parameter: 


          
           
Element el = doc.getRootElement();
String attrValue = el.getAttributeValue("myAttribute");

The getAttributeValue() method returns a String representing the value associated with the attribute, or null if the attribute does not exist.

Setting Attribute Values

Attributes can be set using the setAttribute() method. This method takes a String representing the name of the attribute being set, and a String representing the desired value associated with the attribute as parameters: 


          
           
el.setAttribute("myAttribute", "aNewValue");

The setAttribute() method returns a boolean value indicating whether or not the attribute was successfully set.

Removing Attributes

Attributes can be removed from an Element using the removeAttribute() method. This method takes a String representing the name of the attribute being removed as a parameter: 


          
           
Element el = doc.getRootElement();
boolean success = el.removeAttribute("myAttribute");

The removeAttribute() method returns a boolean value indicating whether or not the attribute was successfully removed.
