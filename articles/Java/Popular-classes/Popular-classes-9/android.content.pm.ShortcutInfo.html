What Is android.content.pm.ShortcutInfo?

android.content.pm.ShortcutInfo is a class in the Android API used for managing shortcuts for activities and services. It provides a way for applications to define shortcuts on their launcher, enabling users to quickly access specific tasks or activities. Shortcuts can be user-defined or system-defined. Shortcuts can also be updated, which enables applications to provide custom experiences for users without having to rebuild the entire application.

Creating Shortcuts

To create a shortcut, first create an instance of 

          
           
ShortcutInfo
with necessary parameters. Then add it to the 

          
           
ShortcutManager
. The following example illustrates how to create a shortcut: 

          
           
ShortcutInfo shortcutInfo = new ShortcutInfo.Builder(context)
    .setShortLabel("Short label")
    .setLongLabel("Long label")
    .setIcon(Icon.createWithResource(context, R.drawable.icon_name))
    .setIntent(new Intent(context, MyActivity.class).setAction("action_name"))
    .build();
 
ShortcutManager shortcutManager = getSystemService(ShortcutManager.class);
shortcutManager.addDynamicShortcuts(Collections.singletonList(shortcutInfo));

updateShortcuts()

ShortcutManager

ShortcutInfo oldShortcutInfo = ...; // Retrieve the current shortcut
 
ShortcutInfo newShortcutInfo = new ShortcutInfo.Builder(oldShortcutInfo)
