SslConnectionFactory in Eclipse Jetty

Overview of SslConnectionFactory

The 

          
           
SslConnectionFactory
class is responsible for creating, configuring, and managing secure sockets for communication over the Internet. It is able to create two types of connections: client side, and server side. When a client requests a secure connection, the SslConnectionFactory creates a socket and begins the SSL/TLS handshake process with the client. Once the handshake is successful, a secure connection is established and the server can send encrypted messages to the client. The SslConnectionFactory also provides an authentication mechanism that can be used to authenticate the remote client.

SslContextFactory

SslConnectionFactory

SslContextFactory
. Finally, you’ll need to specify an IP address and port to listen on when the server receives client requests. Here’s an example of a 

          
           
SslConnectionFactory
