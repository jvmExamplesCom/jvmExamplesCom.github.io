SslConnectionFactory in Eclipse Jetty

Overview of SslConnectionFactory

The

Once the handshake is successful, a secure connection is established and the server can send encrypted messages to the client. The SslConnectionFactory also provides an authentication mechanism that can be used to authenticate the remote client.

SslConnectionFactory

SslContextFactory
