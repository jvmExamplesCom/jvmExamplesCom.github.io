java.nio.file.FileStore

The java.nio.file.FileStore class is a Java NIO 2 (New IO) class for accessing file systems and working with file stores. It is an abstract class that provides methods for interacting with file stores, such as manipulating directory listings and retrieving detailed information about file store attributes. This class can be used to traverse the directory structure of a file system, manipulate files and directories, access file system information, check the free space on a file store, and abstract from a file system’s specific capabilities.

Constructor

This class does not have a public constructor, but all its methods are static.

Methods

isReadOnly()

This method returns a boolean value indicating whether the file store is read-only or read-write.

name()

This method returns the name of the file store. 

          
           
String name = FileStore.name();
System.out.println("Name: " + name);

type()

This method returns a string representing the type of file store. This string may be a platform-specific string, e.g. “FAT32” or “NTFS”.


          
           
String type = FileStore.type();
System.out.println("Type: " + type);

getTotalSpace()

This method returns the size, in bytes, of the file store.


          
           
long totalSpace = FileStore.getTotalSpace();
System.out.println("Total Space: " + totalSpace);

getUnallocatedSpace()

This method returns the amount of unallocated space, in bytes, on the file store.


          
           
long unallocatedSpace = FileStore.getUnallocatedSpace();
System.out.println("Unallocated Space: " + unallocatedSpace);

getUsableSpace()

This method returns the amount of usable space, in bytes, on the file store.


          
           
long usableSpace = FileStore.getUsableSpace();
System.out.println("Usable Space: " + usableSpace);

Example

The following code snippet demonstrates how to use the java.nio.file.FileStore class to retrieve information about a file store.


          
           
// Get a list of all the file stores in this system 
Iterable<FileStore> stores = FileSystems.getDefault().getFileStores();

// Iterate through each file store
for (FileStore store : stores) {  
    // Get the name, type, and size of this file store  
    String name = store.name();  
    String type = store.type();  
    long totalSpace = store.getTotalSpace();  
    long unallocatedSpace = store.getUnallocatedSpace();  
    long usableSpace = store.getUsableSpace();

    // Print out the details of this file store  
    System.out.println("Name: " + name);  
    System.out.println("Type: " + type);  
    System.out.println("Total Space: " + totalSpace);  
    System.out.println("Unallocated Space: " + unallocatedSpace);  
    System.out.println("Usable Space: " + usableSpace);  
    System.out.println();  
}
