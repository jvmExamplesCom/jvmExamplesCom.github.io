org.elasticsearch.transport.client.PreBuiltTransportClient

The PreBuiltTransportClient is a factory class designed to create and configure an Elasticsearch client for the transport protocol. It provides the ability to easily add code that can access the cluster and perform queries or other operations.

Example 1: Connecting to Elasticsearch


          
           
Settings settings = Settings.builder()
    .put("cluster.name", "my-cluster")
    .build();

TransportClient client = new PreBuiltTransportClient(settings)
    .addTransportAddress(new InetSocketTransportAddress(
        new InetSocketAddress("localhost", 9300)));
This code example creates a 
Settings
object with the cluster name, then a 
TransportClient
using the 

          
           
PreBuiltTransportClient
factory. Finally, it adds a transport address to the client, which will be used to connect to the cluster.

Example 2: Performing a Query


          
           
SearchResponse response = client.prepareSearch("index")
    .setQuery(QueryBuilders.matchQuery("field", "query"))
    .get();

This code example uses the client to execute a search query on an index. The query is built using the QueryBuilders class, and passed to the SearchRequest. Finally, the get() method is called to execute the query and return a SearchResponse.

Example 3: Closing the Client


          
           
client.close();

This code example closes the client. It is important to close the client when it is no longer needed, to ensure that any resources associated with it are released.
