android.preference.PreferenceGroup

The android.preference.PreferenceGroup class provides an container for a group of Preference objects. It is used to organize and manage the Preferences contained within it. All Preference objects contained within a PreferenceGroup must have unique keys.

The PreferenceGroup class provides a number of methods that can be used by subclasses to add, remove, find and access Preference objects within the group.

Constructor

The PreferenceGroup constructor takes no arguments and creates an empty PreferenceGroup.


          
           
PreferenceGroup preferenceGroup = new PreferenceGroup();

Add Preference

The addPreference() method adds a Preference to the PreferenceGroup. This method takes a single argument, which is a Preference object.


          
           
Preference preference = new Preference(context);
preferenceGroup.addPreference(preference);

Preference preference = preferenceGroup.findPreference("myKey");
