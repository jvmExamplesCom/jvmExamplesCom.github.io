android.support.v4.widget.CursorAdapter

The android.support.v4.widget.CursorAdapter class is a base adapter class that bridges a Cursor to an AdapterView object. It provides a way to access data from a database or other data source without needing to create a custom adapter derived from the BaseAdapter class.

Using android.support.v4.widget.CursorAdapter

To use android.support.v4.widget.CursorAdapter, you need to create a subclass of it that overrides the following methods:

  • newView()
    – creates a view from the layout resource specified in the constructor.
  • bindView()
    – binds the data from the cursor to the view.
  • changeCursor()
    – changes the underlying cursor to a new one. This is used when a new query is made in the same activity or fragment.
