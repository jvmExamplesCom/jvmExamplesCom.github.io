Creating a New JSONObject

Creating a new JSONObject is easy. The constructor takes the form of new JSONObject() and can be called with either no arguments or an existing JSONObject that will be used as a basis for the new object. 

          
           
JSONObject jsonObject = new JSONObject();
System.out.println(jsonObject); // Prints: {}
Alternatively, you can create a new JSONObject based on an existing one as follows: 

          
           
JSONObject existingJsonObject = new JSONObject();
existingJsonObject.put("key1", "value1");

JSONObject newJsonObject = new JSONObject(existingJsonObject);
System.out.println(newJsonObject); // Prints: {"key1":"value1"}

Manipulating Data in a JSONObject

JSONObject provides methods to manipulate the data contained in it. You can add values to a JSONObject using the 

          
           
put()
method. The 

          
           
put()
method takes two arguments, a key (as a String) and a value (which can be any type of Object). 

          
           
JSONObject jsonObject = new JSONObject();
jsonObject.put("key1", "value1");
jsonObject.put("key2", 1234);
jsonObject.put("key3", true);
System.out.println(jsonObject); // Prints: {"key1":"value1","key2":1234,"key3":true}
You can also access and modify existing data in the JSONObject by providing the corresponding key to the 

          
           
get()
and 

          
           
put()
methods. 

          
           
JSONObject jsonObject = new JSONObject();
jsonObject.put("key1", "value1");

Object existingValue = jsonObject.get("key1");
System.out.println(existingValue); // Prints: value1

jsonObject.put("key1", "newValue");
Object newValue = jsonObject.get("key1");
System.out.println(newValue); // Prints: newValue

Iterating Over a JSONObject

JSONObject also provides methods to iterate over its contents. The 

          
           
keys()
method returns an Iterator which can be used to loop through all the keys in the JSONObject. 

          
           
JSONObject jsonObject = new JSONObject();
jsonObject.put("key1", "value1");
jsonObject.put("key2", 1234);
jsonObject.put("key3", true);

Iterator<String> iterator = jsonObject.keys();
while(iterator.hasNext()) {
    String key = iterator.next();
    System.out.println(key + ": " + jsonObject.get(key));
}

/*Prints: 
key1: value1
key2: 1234
key3: true */
The 

          
           
entrySet()
method returns a set of Entry objects which can be used to loop through all the key-value pairs in the JSONObject. 

          
           
JSONObject jsonObject = new JSONObject();
jsonObject.put("key1", "value1");
jsonObject.put("key2", 1234);
jsonObject.put("key3", true);

Set<Entry<String, Object>> entries = jsonObject.entrySet();
for(Entry entry : entries) {
    System.out.println(entry.getKey() + ": " + entry.getValue());
}

/*Prints: 
key1: value1
key2: 1234
key3: true */
