org.springframework.boot.test.autoconfigure.orm.jpa.DataJpaTest

The org.springframework.boot.test.autoconfigure.orm.jpa.DataJpaTest is a specialized test support for JPA-based tests. It configures an embedded in-memory database, as well as a test EntityManager and DataSource suitable for testing JPA based applications.

Creating a DataJpaTest

Creating a DataJpaTest is very easy. You just need to create a new test class and annotated it with @DataJpaTest:


          
           
@DataJpaTest 
class MyJpaTest {
    ...
}

Configuration Options

The configuration options available with a DataJpaTest are the same as with any other test type, such as @SpringBootTest. Options such as @AutoConfigureTestDatabase, @TestPropertySource, etc. can all be used with the DataJpaTest.

Entity Integration

In a DataJpaTest, you can directly use your entity classes in the tests without needing to mock them. The DataJpaTest will automatically set up an in-memory database with the entity classes as tables from which you can query.

Example Test

Let's look at an example of a simple test using the DataJpaTest support:


          
           
import static org.assertj.core.api.Assertions.assertThat;

@DataJpaTest 
class MyJpaTest {

    @Autowired
    private TestEntityManager entityManager;

    @Test
    void whenFindByName_thenReturnEmployee() {
        // given
        Employee alex = new Employee("alex");
        entityManager.persist(alex);
        entityManager.flush();

        // when
        Employee found = employeeRepository.findByName(alex.getName());

        // then
        assertThat(found.getName())
          .isEqualTo(alex.getName());
    }
}

Conclusion

The org.springframework.boot.test.autoconfigure.orm.jpa.DataJpaTest is a great way to write tests for your JPA based applications. It allows you to easily configure an in-memory database and test EntityManager, and provides integration with your entity classes. With this integration, you can use your entity classes directly in your tests without any need for mocking.
