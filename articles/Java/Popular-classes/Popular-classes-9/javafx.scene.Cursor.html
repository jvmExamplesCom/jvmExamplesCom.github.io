JavaFx Scene Cursor

One component of this API is the Java FX Scene Cursor class. This class allows developers to control the appearance of the cursor when it is over the content on the JavaFX scene.

How to Use JavaFX Scene Cursor

  1. Create a scene:
    
            
             
Scene scene = new Scene(root, width, height);
  2. Set the cursor:
    
            
             
scene.setCursor(Cursor.NONE);

    stage.show();
