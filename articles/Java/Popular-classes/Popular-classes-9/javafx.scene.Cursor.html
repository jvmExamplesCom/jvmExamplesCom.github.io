JavaFx Scene Cursor

JavaFX provides an API for creating graphical user interfaces. The scene is where all of the components, such as buttons and shapes are drawn. One component of this API is the Java FX Scene Cursor class. This class allows developers to control the appearance of the cursor when it is over the content on the JavaFX scene.

How to Use JavaFX Scene Cursor

Using the JavaFX Scene Cursor class is fairly straightforward and can be done with the following steps:
  1. Create a scene:
    
            
             
Scene scene = new Scene(root, width, height);
  2. Set the cursor:
    
            
             
scene.setCursor(Cursor.NONE);
  3. Add any desired content to the scene.
  4. Show the scene:
    stage.show();
Additionally, you can set the image of the cursor with the following code: 

          
           
scene.setCursor(new ImageCursor(image));

Available Cursors

The JavaFX Scene Cursor class provides several cursors that can be used. These include: 
  • Cursor.DEFAULT
  • Cursor.CROSSHAIR
  • Cursor.HAND
  • Cursor.MOVE
  • Cursor.TEXT
  • Cursor.WAIT
In addition to the cursors listed above, there are also methods for creating custom cursors. For example, to create a cursor from an existing image, use this code: 

          
           
Cursor cursor = new ImageCursor(image);

Examples
