Java AWT Geom Arc2D

Java AWT Geom Arc2D is an abstract class that defines a 2-dimensional arc, which is a portion of an ellipse specified by a closure type, rectangular framing, and angular extents. It provides methods for interrogating the arc’s framing geometry, determining whether a point lies within the arc’s interior, computing the arc’s bounding box, and transforming the arc.

The following code creates a basic Arc2D object defining a large portion of a circle:


          
           
// Create an Arc2D object
Arc2D myCircle = new Arc2D.Double(0, 0, 100, 100, Math.PI/2, 6 * Math.PI, Arc2D.OPEN);

This code creates an Arc2D object with its center point at (0, 0), and a radius of 100 in both the x and y axes. The angular extent of the circular arc is set from half a turn to six turns. Since the third argument is Arc2D.OPEN, the circular arc does not extend all the way around the circle.

The following code creates a more complex curve: 

          
           
// Create a more complex curve
Arc2D myCurve = new Arc2D.Double(100, 100, 400, 200, Math.PI, 3 * Math.PI/4, Arc2D.CHORD);

This creates an elliptical curve with a center point at (100, 100) and a size of 400 x 200. The angular extent of the curve is 3/4 turns, beginning at a quarter turn. Since the third argument is Arc2D.CHORD, the curve extends all the way around.

// Determine whether a point lies within the curve
