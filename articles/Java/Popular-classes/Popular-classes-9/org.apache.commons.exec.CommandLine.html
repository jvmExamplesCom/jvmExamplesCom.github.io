org.apache.commons.exec.CommandLine

The Apache Commons Exec library provides a powerful way to execute external programs from Java applications. The core class 


          
           
org.apache.commons.exec.CommandLine
provides easy access to external programs and can be used for a variety of tasks.

Creating a CommandLine Instance

A 
CommandLine
instance can be created using one of the several available constructors. For instance, an empty command line is created using: 

          
           
CommandLine cmdLine = new CommandLine();

Alternatively, a command line with a program name can be created with:


          
           
CommandLine cmdLine = new CommandLine("program_name");
In this case, the 
program_name
is the name of the executable file to be executed. It can be a fully qualified path to the executable file or the name if it is located in the system paths.

Adding Arguments

Arguments for the command line can be added using the 

addArgument()
method. For example, the following code creates a 
CommandLine
instance for the 
ls
command and adds two arguments to it: 

          
           
CommandLine cmdLine = new CommandLine("ls");
cmdLine.addArgument("-l");
cmdLine.addArgument("-a");
This is equivalent to the shell command 
ls -l -a
.

Specifying Execution Directory

The directory in which the command should be executed can also be specified with the 


          
           
setWorkingDirectory()
method. The following code creates a 
CommandLine
instance for the 
ls
command and sets the execution directory to 
/home/user
: 

          
           
CommandLine cmdLine = new CommandLine("ls");
cmdLine.setWorkingDirectory(new File("/home/user"));

Executing a CommandLine

Once a 
CommandLine
instance has been created, it can be executed using the 
execute()
method. This method takes as argument a 
Executor
instance, which is responsible for executing the command. For example, the following code executes the 
ls
command created in the previous example: 

          
           
try {
    CommandLine cmdLine = new CommandLine("ls");
    cmdLine.addArgument("-l");
    cmdLine.addArgument("-a");
    Executor executor = new DefaultExecutor();
    int exitValue = executor.execute(cmdLine);
} catch (IOException e) {
    System.err.println(e.getMessage());
}
The 
execute()
method returns the exit value of the process that was executed. A value of 0 indicates that the execution was successful.

Conclusion

The 

          
           
org.apache.commons.exec.CommandLine
class provides a powerful way to execute external programs from Java applications. Using this class it is possible to specify the command to be executed, the arguments to be passed to the command, and the execution directory. The 
execute()
method then executes the command, returning the exit value of the process.
