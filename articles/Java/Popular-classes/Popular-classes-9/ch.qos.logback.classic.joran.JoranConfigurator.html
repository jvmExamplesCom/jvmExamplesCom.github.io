The JoranConfigurator has a few methods that are useful when working with a Logback configuration file: configure() reads the configuration file and sets up the logging system; getContext() returns the Logback Context object, which encapsulates the logback configuration; reset() resets the context to its state before the configuration file was read; and reconfigure() reads and processes a new configuration file, replacing any existing configuration.
