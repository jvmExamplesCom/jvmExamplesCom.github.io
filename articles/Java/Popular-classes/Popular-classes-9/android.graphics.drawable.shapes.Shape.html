android.graphics.drawable.shapes.Shape

The Shape class is a base class for objects that define geometric shapes. It is in the 


          
           
android.graphics.drawable.shapes
package. This class defines methods for drawing, resizing, and transforming the shape.

Class Methods

The following methods are available in the Shape class:

  • void draw(Canvas canvas, Paint paint): Draws the shape on the specified canvas using the specified paint object.
  • void resize(float width, float height): Resizes the shape to the specified width and height.
  • void setBounds(RectF rect): Sets the bounds for the shape.
  • void setStrokeWidth(float strokeWidth): Sets the width of the shape's stroke.
  • void setColor(int color): Sets the color of the shape.
  • void transform(Matrix matrix): Transforms the shape using the specified matrix.

Example Code

To demonstrate how to use the Shape class, let's create a custom shape. 


          
           
Shape circle = new Shape() {
    @Override
    public void draw(Canvas canvas, Paint paint) {
        canvas.drawCircle(50.0f, 50.0f, 20.0f, paint);
    }
};

This creates a Shape object that draws a circle with a radius of 20. 


          
           
circle.setColor(Color.GREEN);
circle.setStrokeWidth(5.0f);
circle.resize(100.0f, 100.0f);
circle.draw(canvas, paint);

In the above code, we set the color of the circle to green, the stroke width to 5, and the size to 100 by 100. We then draw the circle on the specified canvas using the specified paint object.
