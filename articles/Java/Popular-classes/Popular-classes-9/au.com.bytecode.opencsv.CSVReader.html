To get started, you'll need to add the opencsv.jar to your classpath.

The opencsv.CSVReader class provides methods for reading CSV files. To use it, create an instance of the opencsv.CSVReader class, passing in the file path of the CSV file as a parameter. 

          
           
CSVReader reader = new CSVReader(new FileReader("/path/to/file.csv"));
Once you have an instance of the opencsv.CSVReader class, you can use it to read the contents of the CSV file. String[] line;

CSVWriter writer = new CSVWriter(new FileWriter("/path/to/file.csv"));
