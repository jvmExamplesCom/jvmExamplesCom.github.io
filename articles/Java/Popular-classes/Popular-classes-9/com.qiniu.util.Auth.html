Introduction to com.qiniu.util.Auth

Getting Started with com.qiniu.util.Auth

In order to use com.qiniu.util.Auth, you must first install it on your machine. Once installed, you can get started by creating an instance of the Auth class. 

          
           
Auth auth = new Auth();
The Auth class requires two parameters: the Access Key and Secret Key associated with your Qiniu account. These keys can be retrieved from the Qiniu Console. 

          
           
Auth auth = new Auth("[your_access_key]", "[your_secret_key]");
Once the Auth instance has been created, you can then use it to generate URLs for accessing data in the Qiniu cloud platform. The generateDownloadURL() method can be used to generate download URLs while the generateUploadURL() method can be used to generate upload URLs.

Using the generateDownloadURL() Method

The generateDownloadURL() method takes two parameters – the bucket name and the file name of the data to be downloaded. 

          
           
String downloadURL = auth.generateDownloadURL("[bucket_name]", "[file_name]");
