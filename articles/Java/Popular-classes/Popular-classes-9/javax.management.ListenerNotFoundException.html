javax.management.ListenerNotFoundException

The javax.management.ListenerNotFoundException is an exception that indicates a problem has occurred while trying to remove a javax.management.NotificationListener or NotificationFilter from the javax.management.NotificationBroadcaster.

When is ListenerNotFoundException thrown?

The ListenerNotFoundException is thrown when the specified listener is not registered with the NotificationBroadcaster object. It can also be thrown when the specified filter is not associated with any listener.

Example Code

The following code snippet demonstrates how to register a listener to a NotificationBroadcaster and then how to remove it, throwing a ListenerNotFoundException if the listener was not previously registered:


          
           
try {
    NotificationListener listener = ...;
    NotificationBroadcaster broadcaster = ...;

    broadcaster.addNotificationListener(listener, notificationFilter, handback);

    ...

    broadcaster.removeNotificationListener(listener);
} catch (ListenerNotFoundException e) {
    System.err.println("Listener was not registered with the broadcaster.");
}

The following output is produced by the example code:


          
           
Listener was not registered with the broadcaster.
