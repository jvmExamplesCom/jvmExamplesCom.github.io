The javax.net.ssl.SSLEngineResult class defines the result of a secure socket layer (SSL) handshake. It is used to represent the status of an SSL handshake after an SSL handshake has completed. This class is part of the Java Secure Socket Extension (JSSE) package.

State Definition

The following example creates an SSL engine, wraps and unwraps data, and prints out the SSL engine result.


          
           
try {
    // Create an SSL engine
    SSLContext context = SSLContext.getInstance("TLS");
    context.init(null, null, null);
    SSLEngine engine = context.createSSLEngine();

    // Wrap and unwrap data
    ByteBuffer inBuffer = ByteBuffer.wrap("Hello".getBytes());
    ByteBuffer outBuffer = ByteBuffer.allocate(1024);
    engine.wrap(inBuffer, outBuffer);
    engine.unwrap(outBuffer, inBuffer);

    // Print SSL engine result
    SSLEngineResult result = engine.getHandshakeStatus();

}
