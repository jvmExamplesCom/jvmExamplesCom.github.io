android.support.v7.preference.Preference

The android.support.v7.preference.Preference class provides a base class for preferences that are showing in the PreferenceFragment. It allows the user to make and customize different preference settings in an application.

Example 1: Adding a Preference to the PreferenceFragment


          
           
// Create a new Preference
Preference pref = new Preference(getActivity());

// Set the title and key of the preference
pref.setTitle("My Preference");
pref.setKey("my_key");

// Add the preference to the PreferenceFragment
getPreferencescreen().addPreference(pref);

Example 2: Retrieving a Preference from the PreferenceFragment


          
           
// Get the PreferenceFragment
PreferenceFragment prefFragment = getPreferenceFragment();

// Get the Preference with the given key
Preference pref = prefFragment.findPreference("my_key");

// Modify the value of the Preference
if (pref != null) {
  pref.setValue(123);
}

Example 3: Setting a OnPreferenceChangeListener


          
           
// Get the PreferenceFragment
PreferenceFragment prefFragment = getPreferenceFragment();

// Get the Preference with the given key
Preference pref = prefFragment.findPreference("my_key");

// Set a listener to listen for changes in the Preference
if (pref != null) {
  pref.setOnPreferenceChangeListener(new OnPreferenceChangeListener() {

    @Override
    public boolean onPreferenceChange(Preference preference, Object newValue) {
      // Do something...
      return true;
    }
  });
}

// Get the PreferenceFragment
