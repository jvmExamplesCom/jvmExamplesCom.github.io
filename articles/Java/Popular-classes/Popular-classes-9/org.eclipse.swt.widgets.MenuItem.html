org.eclipse.swt.widgets.MenuItem

The org.eclipse.swt.widgets.MenuItem is a class that allows you to create menu items for an Application, usually for a GUI program. Creating a MenuItem

MenuItem menuItem = new MenuItem(shell, SWT.PUSH);
