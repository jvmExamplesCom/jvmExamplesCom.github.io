com.sun.tools.attach.VirtualMachine

The com.sun.tools.attach.VirtualMachine interface in the Java Virtual Machine Tools Interface provides a high-level mechanism for an application to connect to a Java virtual machine (JVM). It allows a tool to: attach to a JVM, list all of the running JVMs on a machine, and get/set system properties in the JVM. The com.sun.tools.attach.VirtualMachine interface provides methods to enumerate currently running virtual machines and to attach to a given virtual machine.

Creating Instance of VirtualMachine

To create an instance of the com.sun.tools.attach.VirtualMachine interface, use the attach API, which creates a connection to a local or remote JVM. This API takes two parameters: the name of the JVM to connect to and a Properties object that can contain additional options. To get a list of running JVMs, use the VirtualMachine.list() static method.


          
           
import com.sun.tools.attach.VirtualMachine;

// Example of using the attach API
VirtualMachine vm = VirtualMachine.attach("name-of-jvm");

// List all running JVMs
List<VirtualMachineDescriptor> vms = VirtualMachine.list();

Operations on VirtualMachine

Once an instance of the com.sun.tools.attach.VirtualMachine interface is created, various operations can be performed on it. The following are some of the operations that can be performed on a given virtual machine.

  • Get System Properties: The getSystemProperties() method can be used to get a list of system properties for the given virtual machine.
  • Set System Properties: The setSystemProperty() method can be used to set a system property for the given virtual machine.

Example of Using VirtualMachine

This example shows how to use the com.sun.tools.attach.VirtualMachine interface to get a list of running JVMs, connect to a given JVM, and get/set system properties.


          
           
import com.sun.tools.attach.VirtualMachine;

public class AttachExample {

    public static void main(String[] args) {
        // List all running JVMs
        List<VirtualMachineDescriptor> vms = VirtualMachine.list();
        for (VirtualMachineDescriptor vmd : vms) {
            System.out.println(vmd.displayName());

        // Connect to a JVM with the given name

        Properties props = vm.getSystemProperties();

        // Set system property
        vm.setSystemProperty("test.prop", "value");
    }
}
