java.util.function.LongConsumer

Java's 
LongConsumer
interface is a functional interface used in the context of lambda expressions and method references to accept a single primitive 
long
-valued argument and return no result. It has been part of Java since the 8th version, and it is used quite extensively when dealing with the 
Stream
and 
OptionalLong
classes.

Syntax


          
           
@FunctionalInterface
public interface LongConsumer {
  void accept(long value);
}

Usage

The 
accept
method of this interface is invoked with the 
long
value passed as its argument. It is typically used to consume values from streams or optional containers. Let's see an example: 

          
           
Stream<Long> longStream = Stream.generate(() -> (long) (Math.random() * 1000)).limit(10);

LongConsumer
