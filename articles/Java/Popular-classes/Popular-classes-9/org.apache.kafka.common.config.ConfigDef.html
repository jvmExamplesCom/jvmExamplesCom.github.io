Introduction to org.apache.kafka.common.config.ConfigDef

Creating a ConfigDef Instance

The 

          
           
org.apache.kafka.common.config.ConfigDef

ConfigDef configDef = ConfigDef.define("config.name", Type.INT, 0, Importance.HIGH, "Configuration Properties");

configDef.define("timeout", Type.LONG, 1000, Importance.MEDIUM, "Timeout value");

map()
method. This method allows fields to be derived from other fields. configDef.map("timeout", "retries"));
