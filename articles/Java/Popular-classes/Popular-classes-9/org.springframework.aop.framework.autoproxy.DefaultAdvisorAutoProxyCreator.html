org.springframework.aop.framework.autoproxy.DefaultAdvisorAutoProxyCreator

The org.springframework.aop.framework.autoproxy.DefaultAdvisorAutoProxyCreator is an auto-proxy creator that resists the use of direct field access for setting and getting property values. Instead, it uses a proxy mechanism to create an advisor chain around the target object. This allows the target object to be wrapped in an object that implements advised methods.

Code Examples

Below are code examples showing how to use the DefaultAdvisorAutoProxyCreator.


          
           
public class DefaultAdvisorAutoProxyCreator {
    public static void main(String[] args) {
        DefaultAdvisorAutoProxyCreator autoProxyCreator = 
            new DefaultAdvisorAutoProxyCreator();
        ProxyFactory proxyFactory = new ProxyFactory();
        proxyFactory.addAdvisor(autoProxyCreator);
        proxyFactory.setTarget(new MyBean());
        MyBean myBean = (MyBean) proxyFactory.getProxy();
        myBean.foo();
    }
}

          

          
           
class MyBean {
    public void foo() {
        System.out.println("This is a foo method");
    }
}

This example sets up a ProxyFactory and adds a DefaultAdvisorAutoProxyCreator to it. Then it sets a target for the ProxyFactory and retrieves the target's proxy. After that, the example calls the foo() method on the proxy.
