What are javax.ws.rs.Priorities?

The javax.ws.rs.Priorities can be used in conjunction with the @Priority annotation when registering components to the JAX-RS runtime. The annotation allows developers to specify the priority of the component which will be used in the runtime order of execution. For example:

public class MyResource implements Resource

    // ...
}
