javax.persistence.EmbeddedId

The javax.persistence.EmbeddedId is a special type of persistent field used to map composite values in a JPA entity. It is used when an entity has a composite primary key and is typically used when the primary key is composed of two or more fields. The EmbeddedId annotation allows a composite primary key class to be treated as a single logical key value.
