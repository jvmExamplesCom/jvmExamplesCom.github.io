Overview

The Stage.Window class provides a windowing system for displaying graphical user interfaces (GUIs). It is the top-level window used by applications to render content. The Stage.Window class enables applications to create and manipulate windows, define their titles, and manage their contents. The Stage.Window class is composed of two primary components: the Scene and the Stage. The Scene is the main container for content displayed in a window, while the Stage manages window-specific properties such as its size, position, and background color.
