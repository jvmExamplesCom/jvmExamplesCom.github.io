javax.management.NotificationEmitter

javax.management.NotificationEmitter is an interface for objects that can emit notifications about events in their life cycles. It allows registration of listeners, and those listeners are then notified when a notification occurs. This class is mainly used to handle panel communication in front-end applications, as well as handling event logging and auditing.

Code Examples

Here are a few code examples demonstrating usage of javax.management.NotificationEmitter.


          
           
NotificationEmitter emitter = someExistingObject;
NotificationListener listener = new MyNotificationListener();
emitter.addNotificationListener(listener);

The above code creates a NotificationEmitter object and registers a listener to it. The listener is then notified when a notification occurs.


          
           
NotificationEmitter emitter = someExistingObject;
NotificationListener listener = new MyNotificationListener();
emitter.addNotificationListener(listener, null, null);

The above code registers the listener with an optional handback object and a list of notification types the listener should be notified about.
