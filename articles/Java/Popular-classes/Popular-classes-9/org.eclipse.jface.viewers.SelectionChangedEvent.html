org.eclipse.jface.viewers.SelectionChangedEvent

The org.eclipse.jface.viewers.SelectionChangedEvent class is an event type used by label providers and content providers to notify the viewers of user selection changes in their viewer. The selection change event can contain information about which elements were selected or deselected, as well as any structural changes to the elements that occur in response to the selection.
