The org.apache.solr.common.SolrInputDocument Class in Java

The 

          
           
org.apache.solr.common.SolrInputDocument
class is a Solr class used for building documents to be indexed and stored within the Solr search engine. It is an object representation of a single document and is used to create, modify and index documents within a Solr core.

Creating a SolrInputDocument

A SolrInputDocument can be created as follows: 

          
           
SolrInputDocument doc = new SolrInputDocument();

This creates a new SolrInputDocument object, but it won't include any fields yet. To populate a SolrInputDocument with content, use the 

addField()
method.

Adding Fields to a SolrInputDocument

We can add fields to a SolrInputDocument by calling the 

addField()
method. Any field added to the document should be a 
String
. This can be done by passing two arguments to the 
addField()
method. The first argument is a 
String
containing the name of the field and the second is a 
String
representing the value of the field. 

          
           
doc.addField("name","John Doe");
doc.addField("city","New York");
doc.addField("age","21");
It is important to note that the two 
String
arguments passed to the 
addField()

Updating Fields in a SolrInputDocument

setField()

Deleting Fields from a SolrInputDocument

We can delete fields from a SolrInputDocument using the 

removeField()
