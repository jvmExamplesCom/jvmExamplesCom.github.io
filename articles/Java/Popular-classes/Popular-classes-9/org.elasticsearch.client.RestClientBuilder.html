Overview of org.elasticsearch.client.RestClientBuilder

org.elasticsearch.client.RestClientBuilder is a Java 8 library for building REST client instances that interact with an ElasticSearch cluster. The library provides easy access to all the features of the ElasticSearch REST API, allowing developers to quickly build powerful applications that leverage the power of the ElasticSearch platform.

Building a REST Client

The RestClientBuilder class provides all the necessary methods for constructing a REST client that interacts with an ElasticSearch cluster. To create a REST client instance, the following code can be used:


          
           
RestClientBuilder builder = RestClient.builder(
    new HttpHost("localhost", 9200, "http"));
RestClient restClient = builder.build();

In this example, the builder is instantiated using the HTTP host that points to the ElasticSearch cluster. Once the builder is set up, it can be used to build the actual client instance.

Configuring the REST Client

The RestClientBuilder allows developers to configure the client in order to enable advanced features such as custom request headers and authentication. For example, the following code sets up basic authentication:


          
           
builder.setHttpClientConfigCallback(httpClientBuilder -> {
    CredentialsProvider credentialsProvider = 
        new BasicCredentialsProvider();
    credentialsProvider.setCredentials(AuthScope.ANY,
        new UsernamePasswordCredentials("username", "password"));
    httpClientBuilder.setDefaultCredentialsProvider(credentialsProvider);
    return httpClientBuilder;
});

Once the client is configured, it can be used to send requests to the ElasticSearch cluster. For example, the following code sends a GET request to the cluster:


          
           
Response response = restClient.performRequest(
