Overview of org.elasticsearch.client.RestClientBuilder

org.elasticsearch.client.RestClientBuilder is a Java 8 library for building REST client instances that interact with an ElasticSearch cluster. Building a REST Client

The RestClientBuilder class provides all the necessary methods for constructing a REST client that interacts with an ElasticSearch cluster. To create a REST client instance, the following code can be used:


          
           
RestClientBuilder builder = RestClient.builder(
    new HttpHost("localhost", 9200, "http"));
RestClient restClient = builder.build();

In this example, the builder is instantiated using the HTTP host that points to the ElasticSearch cluster. Once the builder is set up, it can be used to build the actual client instance.
