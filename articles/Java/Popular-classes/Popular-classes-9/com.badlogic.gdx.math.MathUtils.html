Java MathUtils

The Java MathUtils class provides many useful mathematical utilities, some of which are outlined below.

Round Function

The round function enables you to round a number to its nearest integer, as demonstrated in the example below:


          
           
float rounded = MathUtils.round(4.75);
System.out.println(rounded); // Output: 5

Ceil Function

The ceil function returns the smallest integer greater than or equal to a particular real number, as shown in the example below:


          
           
float ceiled = MathUtils.ceil(4.75);
System.out.println(ceiled); // Output: 5

Floor Function

The floor function returns the largest integer less than or equal to a particular real number, as shown in the example below:


          
           
float floored = MathUtils.floor(4.75);
System.out.println(floored); // Output: 4

Max Function

The max function returns the largest of two numbers, as shown in the example below:


          
           
float maxValue = MathUtils.max(4.75, 3.23);
System.out.println(maxValue); // Output: 4.75

Min Function

The min function returns the smallest of two numbers, as shown in the example below:


          
           
float minValue = MathUtils.min(4.75, 3.23);
System.out.println(minValue); // Output: 3.23

Clamp Function

The clamp function limits a value to the range between two optional min and max values, as shown in the example below:


          
           
float clampedValue = MathUtils.clamp(50, 0, 25);
System.out.println(clampedValue); // Output: 25

Random Function

The random function generates random numbers, as demonstrated in the example below:


          
           
float randomNumber = MathUtils.random();
System.out.println(randomNumber);
