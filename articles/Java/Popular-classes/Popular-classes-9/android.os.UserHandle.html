Introduction to android.os.UserHandle

The android.os.UserHandle class is an object that encapsulates the user assigned to a process. It can also be used to start an activity as a particular user. This class also provides methods to check if the user is in a particular group and to get the user's name and identifier.

Methods of android.os.UserHandle Class

Here are some of the common methods of this class: 

          
           
getIdentifier(): Returns the user identifier for this user.

          

          
           
getName(): Returns the user name for this user.

          

          
           
isOwner(): Checks whether this user is the owner user.

          

          
           
isSameUser(): Checks whether this user is equal to another one.

          

          
           
isUserInGroup(): Checks whether this user is a member of a specific group.

          

          
           
myUserId(): Returns the user identifier for the current user.

          

          
           
readFromParcel(): Creates a UserHandle object from a Parcel.

          

          
           
writeToParcel(): Write this object to a Parcel.

Example of android.os.UserHandle

In this example, we will create a UserHandle, get its user id, name, and check if it is in a specific group. 

          
           
//Create a new UserHandle
UserHandle userHandle = new UserHandle(123);

//Get the user ID
int userId = userHandle.getIdentifier();

//Get the user name
String name = userHandle.getName();

//Check if user is in a given group
boolean isInGroup = userHandle.isUserInGroup("example_group");

System.out.println("User ID is: " + userId);
System.out.println("User name is: " + name);
System.out.println("User is in example group: " + isInGroup);
The output of the above code will be: 

          
           
User ID is: 123 
User name is: User123 
User is in example group: false
