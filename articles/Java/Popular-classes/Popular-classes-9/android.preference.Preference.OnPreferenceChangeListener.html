Android.preference.Preference.OnPreferenceChangeListener

Android.preference.Preference.OnPreferenceChangeListener is an interface for controlling how a change in a Preference will affect the activity. This interface is used to inform the user that a certain preference has changed. It is often implemented directly in the activity hosting the preferences, but can also be implemented as part of another class that can register the listener with the activity.

Overview

Android.preference.Preference.OnPreferenceChangeListener provides a callback method that is called whenever a change is made to a Preference controlled by the activity. This allows the activity to control how the preference is handled and how the UI is updated to reflect the change. This interface can be used in activities that need to know when preferences have been changed or when they are about to be changed, so they can take appropriate action.

Usage

In order to use this interface, one needs to implement the callback method 

          
           
onPreferenceChange(Preference preference, Object newValue)
. This method is called whenever a change is made to a preference controlled by the activity. The first argument, 

          
           
preference
, is an instance of the Preference object that is being changed, and the second argument, 

          
           
newValue
, is the new value of the preference. The implementation of this method should perform the following operations:
  • Check the type of the new value, to make sure it is compatible with the preference.
  • Perform any necessary calculations on the new value.
  • Update the UI to reflect the new value.
  • Save the new value in the Preferences.
  • Return true if the change was accepted, or false if the change was rejected.
One can also add a 

          
           
Preference.OnPreferenceChangeListener
to their activity by calling the 

          
           
Preference.setOnPreferenceChangeListener(Preference.OnPreferenceChangeListener listener)
method. This will allow the specified listener to be notified whenever a change is made to any of the preferences controlled by the activity.

Example

The following example shows a basic implementation of the Android.preference.Preference.OnPreferenceChangeListener interface. In the example, a 

          
           
Preference
is monitored for changes and the value is displayed in a 

          
           
TextView
. 

          
           
public class MyActivity extends AppCompatActivity implements Preference.OnPreferenceChangeListener {

    private TextView textView;

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);

        // find the TextView
        textView = (TextView) findViewById(R.id.text);

        // find the preferences
        SharedPreferences preferences = PreferenceManager.getDefaultSharedPreferences(this);
        Preference pref1 = findPreference("pref1");
        Preference pref2 = findPreference("pref2");
        
        // register the preference change listeners
        pref1.setOnPreferenceChangeListener(this);
        pref2.setOnPreferenceChangeListener(this);
    }

    @Override
    public boolean onPreferenceChange(Preference preference, Object newValue) {
        String key = preference.getKey();
        if (key.equals("pref1")) {
            int value = (Integer) newValue;
            String output = String.format("pref1 changed to %d", value);
            textView.setText(output);
        } else if (key.equals("pref2")) {
            boolean value = (Boolean) newValue;
            String output = String.format("pref2 changed to %b", value);
            textView.setText(output);
        }
        return true;
    }
}

onPreferenceChange
method checks the key of the 

          
           
Preference
and updates the 

          
           
TextView
accordingly. Once the
