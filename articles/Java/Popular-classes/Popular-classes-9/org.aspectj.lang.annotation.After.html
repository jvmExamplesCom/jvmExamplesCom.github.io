Introduction to org.aspectj.lang.annotation.After

The 

          
           
org.aspectj.lang.annotation.After
annotation represents an aspect or a set of aspects that will be executed after a certain set of join points have been executed. It is primarily used to create a post-processing action that can be applied to methods in an abstract class or interface. In some cases it is used to clean up resources or to log information.

Syntax

The syntax for the annotation is as follows: 

          
           
@After(value = "expression")
void aspectName();

The expression is a pointcut expression, which is a String value containing the join points the aspect should be applied to. The aspectName() is the name of the aspect to be executed.

Foo
interface is called: 

          
           
@Aspect
public class FooAspect {

    @After("execution(* Foo.*())")

    }
