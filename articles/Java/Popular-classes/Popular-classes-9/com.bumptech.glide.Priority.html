com.bumptech.glide.Priority

The com.bumptech.glide.Priority class is a priority for Glide requests. It is used to prioritize individual loads and is used to determine the order in which images are processed among multiple requests. The higher the priority, the earlier it will be processed.

  • NORMAL
    - A normal priority. This is usually used for requests that have no particular preference.
  • HIGH
    - A high priority. This is usually used for requests that should be processed as soon as possible.
