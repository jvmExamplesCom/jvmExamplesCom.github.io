Introduction to Nine Old Androids ValueAnimator

ValueAnimator Interpolator

Nine Old Androids ValueAnimator provides several interpolators for defining the relationship between the current and target values.

  • LinearInterpolator – linear transition between values.
  • AccelerateDecelerateInterpolator – smooth transition from slow to fast and back.
  • AccelerateInterpolator – transition from slow to fast.
  • AnticipateInterpolator – transition begins slowly and then accelerates.
  • BounceInterpolator – transition bounces at the end.
  • DecelerateInterpolator – transition from fast to slow.

Interpolators can be set when creating a ValueAnimator instance.


          
           
ValueAnimator animator = new ValueAnimator();
animator.setInterpolator(new BounceInterpolator());
