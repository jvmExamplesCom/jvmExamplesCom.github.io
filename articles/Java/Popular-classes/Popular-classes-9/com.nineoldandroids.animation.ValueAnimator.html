Introduction to Nine Old Androids ValueAnimator

Nine Old Androids ValueAnimator enables developers to animate arbitrary values over a specified duration. This can be used to animate a variety of values such as color, size, and position. The value animator also provides interpolation methods such as linear, cubic, and accelerative-decelerative interpolators for creating smooth transitions between values.

ValueAnimator Basics

The base class for ValueAnimation is com.nineoldandroids.animation.ValueAnimator. To create a ValueAnimator you need to set the duration and a listener instance which will receive animation updates. An optional Interpolator object can also be used to define how the transition between the current and target value takes place.


          
           
ValueAnimator animator = new ValueAnimator();
animator.setDuration(1000);
animator.addUpdateListener(animationListener);
animator.setInterpolator(new AccelerateDecelerateInterpolator());

To start the animation you need to call the start method.


          
           
animator.start();

To set the value to be animated you must use the setObjectValues method. This method takes an array of objects to animate through.


          
           
Object[] values = { 0, 10, 20, 30 };
animator.setObjectValues(values);

If you want to repeat the animation indefinitely you can call the setRepeatCount method, passing in the value ValueAnimator.INFINITE.


          
           
animator.setRepeatCount(ValueAnimator.INFINITE);

ValueAnimator Interpolator

Nine Old Androids ValueAnimator provides several interpolators for defining the relationship between the current and target values.

  • LinearInterpolator – linear transition between values.
  • AccelerateDecelerateInterpolator – smooth transition from slow to fast and back.
  • AccelerateInterpolator – transition from slow to fast.
  • AnticipateInterpolator – transition begins slowly and then accelerates.
  • BounceInterpolator – transition bounces at the end.
  • DecelerateInterpolator – transition from fast to slow.

Interpolators can be set when creating a ValueAnimator instance.


          
           
ValueAnimator animator = new ValueAnimator();
animator.setInterpolator(new BounceInterpolator());

Nine Old Androids ValueAnimator provides an easy way to animate various properties of objects in Android. With the ability to specify durations and interpolators, developers have great control over the speed and feel of their animations.
