javax.xml.transform.sax.SAXTransformerFactory

The javax.xml.transform.sax.SAXTransformerFactory class provides a way of constructing a SAX (Simple API for XML) parser and generating SAX events from an XML document. It is an implementation of the javax.xml.transform.TransformerFactory interface.
