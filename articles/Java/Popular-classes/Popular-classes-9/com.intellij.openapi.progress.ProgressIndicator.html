ProgressIndicator in Java

ProgressIndicator is a class from Intellij API which provides a visual representation of progress in any long-running operations. The ProgressIndicator interface can be used to display progress bars, progress circles and other indicators. The ProgressIndicator class provides methods for setting the progress indicator’s value, color, message, and other characteristics.

Creating a ProgressIndicator Object

A ProgressIndicator object can be created using the static method createProgressIndicator() of the ProgressIndicator class: 

          
           
ProgressIndicator progressIndicator = ProgressIndicator.createProgressIndicator();
It is also possible to pass an additional parameter in the createProgressIndicator() method specifying the start value of the indicator. For example: 

          
           
ProgressIndicator progressIndicator = ProgressIndicator.createProgressIndicator(0.25);

Setting Properties of ProgressIndicator

The progress indicator’s properties can be set using the various setter methods provided by the ProgressIndicator class. For example: 

          
           
progressIndicator.setText("Loading...");
progressIndicator.setFraction(0.5);
progressIndicator.setColor(Color.BLUE);
The fraction property allows you to specify the progress of the operation from 0 (not started) to 1 (completed).

Using a ProgressIndicator in a Task

A progress indicator can be used to show the progress of a task. The ProgressIndicator object can be passed as a parameter to the execute() method of a Task class: 

          
           
Task task = new Task() {...};
task.execute(progressIndicator);
The task will then periodically update the progress indicator with its progress.

Example

The following example creates a progress indicator with an initial value of 0.25 and displays it in a frame window. It then implements a simple task that updates the progress indicator every 0.1 seconds, gradually increasing its value until it reaches 1. 

          
           
import com.intellij.openapi.progress.ProgressIndicator;
import com.intellij.openapi.progress.Task;

ProgressIndicator progressIndicator = ProgressIndicator.createProgressIndicator(0.25);
frame.add(progressIndicator);

Task task = new Task() {
    @Override
    public void run(ProgressIndicator progressIndicator){
        while (progressIndicator.getFraction() < 1.0){
            progressIndicator.setFraction(progressIndicator.getFraction() + 0.1);
            try {
                Thread.sleep(100);
            } catch (InterruptedException e){
                e.printStackTrace();
            }
        }
    }
};

task.execute(progressIndicator);
