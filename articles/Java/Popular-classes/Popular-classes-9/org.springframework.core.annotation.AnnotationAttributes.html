Introduction to AnnotationAttributes from org.springframework.core.annotation

AnnotationAttributes is a utility class that encapsulates an annotation type, its attribute names and their values. It provides convenient methods for accessing annotation attributes and for manipulating them.

Creating AnnotationAttributes

AnnotationAttributes can be instantiated in different ways. The simplest way is to pass an annotation instance to the constructor:


          
           
@MyAnnotation
public class MyClass {

}

AnnotationAttributes attributes = new AnnotationAttributes(MyAnnotation.class);

You can also pass an array of annotations to the constructor:


          
           
@MyAnnotation
@AnotherAnnotation
public class MyClass {

}

AnnotationAttributes attributes = new AnnotationAttributes(MyAnnotation.class, AnotherAnnotation.class);

Finally, you can also create an AnnotationAttributes instance from a 


          
           
Map<String, Object>
: 

          
           
Map<String, Object> map = new HashMap<>();
map.put("name", "foo");
map.put("value", "bar");

AnnotationAttributes attributes = new AnnotationAttributes(map);

Manipulating AnnotationAttributes

AnnotationAttributes provides several methods to manipulate the attributes. You can add, remove or modify attributes with 

setAttribute()
, 
removeAttribute()
and 
getAttribute()
: 

          
           
attributes.setAttribute("name", "foo");
attributes.removeAttribute("name");
Object value = attributes.getAttribute("name");

You can merge another AnnotationAttributes instance into this one with 

mergeAttributes()
: 

          
           
AnnotationAttributes otherAttributes = new AnnotationAttributes(MyAnnotation.class);
attributes.mergeAttributes(otherAttributes);

Finally, you can also convert an AnnotationAttributes instance back to a 


          
           
Map<String, Object>
with 
asMap()

Map<String, Object> map = attributes.asMap();
