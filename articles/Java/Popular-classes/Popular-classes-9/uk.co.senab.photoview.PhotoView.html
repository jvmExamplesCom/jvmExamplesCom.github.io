Introduction to Java Photoview

Java PhotoView is a library for displaying photos and images, created by Senab. It is based on the ImageView widget, and provides an easy way to pan, zoom and rotate photos and images. In addition, it provides support for pinch/zoom and swipe/panning gestures. The library is open source, and can be used in any Android app that requires image-viewing capabilities.

Using the Library

Using PhotoView is quite simple. In order to use the library, you need to add the library as a dependency in your build.gradle. Once added, you can either create a new PhotoView instance in your activity, or add it to a layout file.


          
           
//build.gradle

dependencies {
    implementation 'uk.co.senab:photoview:1.2.4'
}

          

          
           
//Activity
PhotoView photoView = new PhotoView(this);
setContentView(photoView);

Applying Photos

Once the PhotoView is setup, you can then apply a photo to it. In order to do this, you will need to set the PhotoView's image attribute to the Bitmap that contains the photo or image. You can also apply drawables and resources as well.


          
           
//Set Bitmap
Bitmap bitmap = BitmapFactory.decodeFile("/path/to/image.jpg");
photoView.setImageBitmap(bitmap);

//Set Drawable
Drawable drawable = getResources().getDrawable(R.drawable.ic_launcher_background);
photoView.setImageDrawable(drawable);

//Set Resource
int resId = R.drawable.ic_launcher_background;
photoView.setImageResource(resId);
