The Basics of EmbeddedDatabaseBuilder

The EmbeddedDatabaseBuilder allows developers to deal with embedded databases in a systematic way. It enables the configuration of a chosen database platform entirely from Java code and in-memory operations. This allows for the creation of standalone applications using an embedded database of choice – thereby removing the need for a full-scale database server installation. It also allows for the management of more complex database operations such as transactions and stored procedures with relative ease.

Using EmbeddedDatabaseBuilder

EmbeddedDatabaseBuilder

EmbeddedDatabaseBuilder builder = new EmbeddedDatabaseBuilder();
