Introduction to org.apache.hadoop.mapreduce.lib.input.TextInputFormat

The TextInputFormat is an input format for reading text files. It uses a 

LineRecordReader
object to read lines of a text file and convert them into 
key-value
pairs. The keys are line numbers and the values are the complete lines of text. The line number starts from 0.

//Create the Job Configuration

Output from TextInputFormat

The output of the TextInputFormat is a sequence of 

key-value
pairs. The key is a LongWritable which contains the line number and the value is a Text object which contains the line of text. For example, a file with 3 lines of text would produce the following output:
  • Key: 0 Value: "This is the first line of text."
  • Key: 1 Value: "This is the second line of text."
  • Key: 2 Value: "This is the third line of text."

Note that the line numbers start from zero. The output of the TextInputFormat can be used as the input to a MapReduce job or any other data processing application.

Conclusion

The TextInputFormat provides an easy way to read text files in Hadoop. It supports customizing how the text is broken into 

key-value
pairs and the output is a sequence of key-value pairs that can be used as input to any data processing application.
