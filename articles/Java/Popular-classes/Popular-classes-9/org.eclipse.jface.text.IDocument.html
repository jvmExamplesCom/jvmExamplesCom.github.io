org.eclipse.jface.text.IDocument

The org.eclipse.jface.text.IDocument class is a text document interface. It provides methods to access and modify the documents content and is used as the base for text editors in Eclipse.

Constructors

The IDocument class has two constructors: 

          
           
public IDocument()
This constructor creates an empty document. 

          
           
public IDocument(String text)
This constructor creates a document from a given string.

Methods

The IDocument class provides several methods to manage the document content: 

          
           
public void addDocumentListener(IDocumentListener listener)
This method adds a document listener that will be notified when the document is modified. 

          
           
public void removeDocumentListener(IDocumentListener listener)
This method removes a document listener. 

          
           
public void replace(int offset, int length, String text)
This method replaces the text in the document between the given offset and length with the specified text. 

          
           
public int getLength()
This method returns the length of the text in the document. 

          
           
public String get()
This method returns the entire content of the document as a string. 

          
           
public String get(int offset, int length)
This method returns a string containing the text in the document between the given offset and length. 

          
           
public char getChar(int offset)
This method returns the character at the specified offset in the document. 

          
           
public void set(String text)
This method sets the document content to the given string. 

          
           
public void insert(int offset, String text)
This method inserts the given text at the specified offset in the document.

Example

The following example shows how to use the IDocument class to create a document and modify its content: 

          
           
String content = "Hello World!";
IDocument document = new IDocument(content);
document.replace( 6, 5, "Universe");
