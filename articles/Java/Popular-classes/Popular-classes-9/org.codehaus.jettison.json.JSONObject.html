org.codehaus.jettison.json.JSONObject

The org.codehaus.jettison.json.JSONObject class is a standard Java class used to parse and generate JSON (JavaScript Object Notation) data. It provides an easy way to convert JSON objects into Java objects, and vice versa. Through the use of this class, developers can quickly create applications that can interact with other services that use the JSON format.

Creating a JSONObject

A JSONObject can be created by passing a Java Map object to the constructor, or by passing a String representation of the JSON object to the constructor. Here are some examples:


          
           
Map<String, Object> map = new HashMap<>();
map.put("name", "John");
map.put("age", 31);
JSONObject json = new JSONObject(map);
or 

          
           
String jsonString = "{ \"name\": \"John\", \"age\": 31 }";
JSONObject json = new JSONObject(jsonString);

Accessing Values from a JSONObject

Once a JSONObject has been created, values can be accessed using the get() method. The get() method takes a String argument which is the key of the value to be retrieved. It returns the Object associated with the key, or null if no such key exists. Here's an example:


          
           
String name = json.get("name").toString();  // returns "John"
int age = json.getInt("age");  // returns 31

Adding Values to a JSONObject

Values can be added to an existing JSONObject using the put() method. The put() method takes two arguments: the key of the value to add, and the value itself. Here's an example:


          
           
json.put("gender", "male");

Converting a JSONObject to a String

A JSONObject can be converted to a String representation by using the toString() method. This returns a String in the format of a valid JSON object. Here's an example:


          
           
String jsonString = json.toString();  // returns "{\"name\":\"John\",\"age\":31,\"gender\":\"m
