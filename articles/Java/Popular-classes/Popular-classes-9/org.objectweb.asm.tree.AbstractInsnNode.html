AbstractInsnNode Class Overview

The org.objectweb.asm.tree.AbstractInsnNode class is an abstract class used by the ASM bytecode library to represent a single instruction in the bytecode of a Java program. It is part of the ASM tree API, a set of classes whose instances communicate the structure of a class, or the instructions that make up a method, as described in their Java bytecode.

Instances of AbstractInsnNode are actually representations of instructions followed by other instructions. Thus, an instance of AbstractInsnNode cannot exist by itself. It must also have at least one other instruction connected to it, forming a link between the instructions.
