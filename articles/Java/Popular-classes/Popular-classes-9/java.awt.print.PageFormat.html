java.awt.print.PageFormat

Java provides the PageFormat class in the java.awt.print package which allows applications to access information and provides an interface for application-defined page parameters. The PageFormat object encapsulates these parameters, and allows an application to get, set and query them.

Creating a PageFormat Object

A PageFormat can be obtained from a PrinterJob. The PrinterJob will return a default page format if none has been set, or if any of the page parameters were set to their defaults (e.g. 0 for width and height). 

          
           
PrinterJob job = PrinterJob.getPrinterJob(); 
 PageFormat pageFormat = job.pageDialog(pageFormat);
The pageDialog method takes as its argument a PageFormat object. It then creates a print dialog with the page format settings set to those contained within the given PageFormat. This allows users to change the page settings, such as paper size, margins, etc. If OK is selected, the changed PageFormat object is returned; otherwise, the same object is returned.

Querying PageFormat Parameters

Once we have our PageFormat object, we can query it for its various parameters. The methods that allow us to do this are as follows: 

          
           
int width = pageFormat.getWidth();
 int height = pageFormat.getHeight();
 double paperWidth = pageFormat.getImageableWidth();
 double paperHeight = pageFormat.getImageableHeight();
These methods return the width and height of the page, as well as the imageable width and height of the paper itself. The imageable size is the area on the paper where printing can occur. It is usually slightly smaller than the paper size itself, because of the margins that the printer uses.

Setting PageFormat Parameters

It is also possible to set the page format parameters via the various setter methods provided by the PageFormat class. These include: 

          
           
setOrientation(int orientation)
setPaper(Paper obj)
setImageableArea(double x, double y, double w, double h)
The setOrientation method takes an int argument for either portrait or landscape orientation. The setPaper method requires a Paper object as argument, which contains all the settings regarding paper size and margins. Finally, the setImageableArea allows setting of the page size, the x and y offsets of the page, and the width and height of the imageable area.

Conclusion

The java.awt.print.PageFormat class allows applications to access and modify the settings of a printer's page format. This class is useful for applications which require specialized printing parameters, or just need to know the details of the printer's capabilities.
