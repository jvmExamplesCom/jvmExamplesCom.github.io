org.openqa.selenium.remote.CapabilityType
class in Selenium defines a set of capabilities used when creating a new webDriver session. These capabilities can be used to customize the behaviour of the Selenium WebDriver and allow users to configure a session to their specific requirements.

The capabilities in this class can be set using the 

DesiredCapabilities
object, which is created when starting a new session. For example, the following code creates a new instance of the Chrome WebDriver with a custom timeout set: 

          
           
DesiredCapabilities caps = DesiredCapabilities.chrome();
