Java com.squareup.javapoet.AnnotationSpec

JavaPoet is an open source library that provides a Java API for generating .java source files. Its goal is to provide a concise, fluent interface for crafting source code in a typesafe manner. The AnnotationSpec class provides an API for generating annotation declarations in a source file.

Example 1: Basic Annotation


          
           
AnnotationSpec myAnno= AnnotationSpec.builder(Override.class).build();

This code creates an AnnotationSpec object that represents an @Override annotation.

Example 2: Annotation with Properties


          
           
AnnotationSpec myAnno= AnnotationSpec.builder(Deprecated.class)
    .addMember("value", "$S", "Use of this method is discouraged")
    .build();

This code creates an AnnotationSpec object that represents an @Deprecated annotation with one property: value. The property's value is set to "Use of this method is discouraged".

Example 3: Multiple Properties


          
           
AnnotationSpec myAnno= AnnotationSpec.builder(Deprecated.class)
    .addMember("reason", "$S", "Use of this method is discouraged")
    .addMember("since", "$S", "1.0")
    .build();

This code creates an AnnotationSpec object that represents an @Deprecated annotation with two properties: reason and since. The first property's value is set to "Use of this method is discouraged" and the second property's value is set to "1.0"

Example 4: Adding Annotation to Element


          
           
MethodSpec myMethod = MethodSpec.methodBuilder("foo")
    .addAnnotation(myAnno)
    .build();

This code adds the previously created annotation (in example 2-3) to a MethodSpec object. This MethodSpec object can then be added to a TypeSpec object, which can be written to a Java source file.

Conclusion

The com.squareup.javapoet.AnnotationSpec class provides an API for generating annotations in a typesafe manner. Annotations are a powerful part of the Java language and this library makes it easy to craft and add them to your source code.
