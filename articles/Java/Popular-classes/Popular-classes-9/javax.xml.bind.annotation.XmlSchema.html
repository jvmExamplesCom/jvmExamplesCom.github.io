javax.xml.bind.annotation.XmlSchema

The javax.xml.bind.annotation.XmlSchema is an annotation used for binding Java classes to XML schema.

Usage

The XmlSchema annotation can be used to define the mapping between an XML schema and Java elements as follows:


          
           
@XmlSchema(
  namespace="http://example.org/schema", 
  elementFormDefault=XmlNsForm.QUALIFIED
)
package org.example.domain;
In the above example, the 
namespace
attribute defines the namespace of the schema being mapped. The 
elementFormDefault
attribute defines the default form of elements in the schema.

XmlSchemaTypes

The XmlSchemaTypes annotation is used to define types in the schema that are to be bound to corresponding Java classes. The annotation is defined as follows:


          
           
@XmlSchemaTypes({
  @XmlSchemaType(name="dateTime", type=java.util.Date.class),
  @XmlSchemaType(name="duration", type=long.class)
})
package org.example.domain;
In the above example, the 
name
attribute defines the name of the type in the schema, while the 
type
attribute defines the corresponding Java type.

XmlSeeAlso

The XmlSeeAlso annotation is used to define classes that are related to a particular Java class. These classes will be included in the generated schema. The annotation is defined as follows:


          
           
@XmlSeeAlso({
  org.example.domain.Foo.class,
  org.example.domain.Bar.class
})
package org.example.domain;
In the above example, the 
value
attribute defines the list of related classes that will be included in the generated schema.

XmlElementDecl

The XmlElementDecl annotation is used to define an element declaration in the schema. The annotation is defined as follows:


          
           
@XmlElementDecl(
  name="foo", 
  namespace="http://example.org/schema", 
  scope=ScopeType.GLOBAL
)
public JAXBElement<String> createFoo(){ ... }
In the above example, the 
name
attribute defines the name of the element, while the 
namespace
attribute specifies the namespace. The 
scope
attribute defines the scope of the element.

XmlJavaTypeAdapter

The XmlJavaTypeAdapter annotation is used to specify an adapter class for mapping a particular Java type to another type. The annotation is defined as follows:


          
           
@XmlJavaTypeAdapter(MyAdapter.class)
public class MyClass { ... }

value
attribute defines the adapter class for the given type.
