android.hardware.usb.UsbDevice

The android.hardware.usb.UsbDevice class represents a USB device attached to an Android device. This class provides access to information about a USB device, such as its vendor and product ID, the interface protocols and endpoints it supports, and other information.

Create an Instance of UsbDevice

To create an instance of UsbDevice, you must get a handle to the device via UsbManager. You can do this using UsbManager.getDeviceList() or UsbManager.openDevice().

UsbDevice Methods


          
           
public String getSerialNumber()

public String getDeviceName()
This method returns the name of the device, or null if the device does not have a name. 

          
           
public int getVendorId()
This method returns the vendor ID for the device. 

          
           
public int getProductId()
This method returns the product ID for the device.

This method returns the UsbInterface object at the specified interface index, or null if there is no such interface. 

          
           
public List<UsbInterface> getInterfaceList()
This method returns a list of all the UsbInterfaces on the device.

This method returns the UsbEndpoint object at the specified endpoint index, or null if there is no such endpoint.

This method returns a list of all the UsbEndpoints on the device.
