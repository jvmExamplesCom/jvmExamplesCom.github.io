Introduction to org.springframework.data.annotation.CreatedBy

org.springframwork.data.annotation.CreatedBy is an annotation that defines a field used to capture information about the creator of an entity. Usually, the creator of an entity is the user that created it, but can also be a system component or other software.

Annotation Basics

Here is the basic syntax for using the annotation:


          
           
@CreatedBy
private String createdBy;

The annotation takes one optional parameter – “value” – that specifies the name of the field in the entity. By default, this parameter is set to "createdBy".

Example Usage

Let's look at a simple example of how the annotation can be used. We'll define a simple entity class called 

Task
, which has an attribute that holds the user who created it: 

          
           
@Entity
public class Task {
    private Long id;
    private String title;

    @CreatedBy
    private String createdBy;
}

We can now use this annotation to track the entity's creator:


          
           
Task task = new Task();
task.setTitle("Clean the house");
task.setCreatedBy("John Doe");

Now, if we query for the task entity, we will be able to see the creator's name in the result:


          
           
Task savedTask = repository.save(task);
System.out.println(savedTask.getCreatedBy()); // prints "John Doe"
