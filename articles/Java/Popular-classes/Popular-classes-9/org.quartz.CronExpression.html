What is a Cron Expression?

A Cron Expression is a string containing six or seven fields representing different properties of a scheduled job. Using a Cron Expression, the user can define a recurring job at certain times with exact precision. For example, a Cron Expression can be used to define a job that runs every 10 minutes on the first day of the month.
