What is com.google.common.collect.ImmutableMap.Builder?

com.google.common.collect.ImmutableMap.Builder is a class from Guava library which provides a builder API to construct immutable maps.

How to use it?

To use ImmutableMap.Builder you have to first create an instance of it and add entries to the builder calling put method or by calling ‘putAll’ to add all elements at once.


          
           
ImmutableMap.Builder<Integer, String> builder = ImmutableMap.builder();
builder.put(1, "One");
builder.putAll(initialMap);

Once done with adding entries, call the build method which will create an immutable map containing all your entries.


          
           
ImmutableMap<Integer, String> builtMap = builder.build();

Example

Let’s take a simple example where we have an array of integers and strings and we want to convert them into a map.


          
           
int[] numbers = { 1, 2, 3, 4};
String[] names = { "One", "Two", "Three", "Four"};

We can easily iterate over arrays and add them as entries in our map.


          
           
ImmutableMap.Builder<Integer, String> builder = ImmutableMap.builder();

for (int i=0; i<numbers.length; i++) {
    builder.put(numbers[i], names[i]);
}
And finally call build method to create the map. 

          
           
ImmutableMap<Integer, String> map = builder.build();

System.out.println(map); // {1=One, 2=Two, 3=Three, 4=Four}
