What is com.google.common.collect.ImmutableMap.Builder?

ImmutableMap.Builder<Integer, String> builder = ImmutableMap.builder();

Example

Let’s take a simple example where we have an array of integers and strings and we want to convert them into a map.


          
           
int[] numbers = { 1, 2, 3, 4};
String[] names = { "One", "Two", "Three", "Four"};

We can easily iterate over arrays and add them as entries in our map.


          
           
ImmutableMap.Builder<Integer, String> builder = ImmutableMap.builder();
