io.flutter.plugins.GeneratedPluginRegistrant

io.flutter.plugins.GeneratedPluginRegistrant is a utility class for registering plugins with the Android application. It is used as part of the Flutter plugin system, which allows developers to extend Flutter with their own code or libraries. This class is automatically generated when a plugin is added to a Flutter project.

Basic Usage

When a plugin is added to a Flutter project, the Gradle build script will generate the io.flutter.plugins.GeneratedPluginRegistrant class. To register a plugin with the application, invoke its registerWith method on an instance of this class. For example:


          
           
public class MainActivity extends FlutterActivity {
  @Override
  public void configureFlutterEngine(FlutterEngine flutterEngine) {
    GeneratedPluginRegistrant.registerWith(flutterEngine);
  }
}

The class also provides a method for overriding the default registrations with custom logic, if desired:


          
           
GeneratedPluginRegistrant.registerWith(flutterEngine, myCustomRegistry);
