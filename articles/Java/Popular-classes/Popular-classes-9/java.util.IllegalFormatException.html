Java.util.IllegalFormatException

Java.util.IllegalFormatException is an exception class which is thrown when a format string contains an illegal syntax, contains a format specifier that is incompatible with the given arguments, or is missing an argument for the corresponding format specifier.

Example of IllegalFormatException

The following code snippet produces an IllegalFormatException because the format string contains an illegal syntax:


          
           
String fmt = "%d %f";
String s = String.format(fmt, 4);
System.out.println(s);

The output of this code will be an IllegalFormatException: 


          
           
java.util.IllegalFormatException: Unknown conversion type d for the argument '4'
.

Example of Incompatible Format Specifier

The following code snippet produces an IllegalFormatException because the format string contains an incompatible format specifier:


          
           
String fmt = "%d %s";
int n = 4;
String s = String.format(fmt, n);
System.out.println(s);

The output of this code will be an IllegalFormatException: 


          
           
java.util.IllegalFormatException: No conversion available for 'int' to 'String'
.

Example of Missing Argument for Format Specifier

The following code snippet produces an IllegalFormatException because not enough arguments are provided for the format specifier:


          
           
String fmt = "%d %f %s";
String s = String.format(fmt);
System.out.println(s);

The output of this code will be an IllegalFormatException: 


          
           
java.util.IllegalFormatException: Not enough arguments passed to format string.
