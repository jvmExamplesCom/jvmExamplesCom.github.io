org.bukkit.event.block.BlockBreakEvent

The 
BlockBreakEvent
class is part of the Spigot API – it is an event that is triggered whenever a block is broken and is used to control how blocks are broken in Minecraft. This event allows you to customize the breaking behavior of blocks, including checking for certain conditions or denying the break altogether.

Example Usage

Let's say we want to make sure that players can only break blocks with a specific tool. We could use a 

BlockBreakEvent
listener: 

          
           
@EventHandler
public void onBlockBreak(BlockBreakEvent event) {
    if (event.getPlayer().getInventory().getItemInMainHand().getType() != Material.GOLD_PICKAXE) {
        event.setCancelled(true);
    }
}

Here, we check whether the item in the player's main hand is a gold pickaxe. If it isn't, we cancel the event and prevent the block from being broken.

BlockBreakEvent.getPlayer().hasPermission(String permission)

true

false

myplugin.break

@EventHandler
public void onBlockBreak(BlockBreakEvent event) {
