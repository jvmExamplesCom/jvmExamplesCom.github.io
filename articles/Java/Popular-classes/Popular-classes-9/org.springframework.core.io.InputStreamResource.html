How to Use InputStreamResource

Using the 

          
           
InputStreamResource
is relatively straightforward. An instance of it needs to first be created, and then it can be used to access the resource. Let's look at a basic example: 

          
           
InputStreamResource inputStreamResource = new InputStreamResource(new FileInputStream("file.txt"));
