android.media.MediaMuxer

The 

          
           
android.media.MediaMuxer
class implements a multiplexer for a mediacodec recording session. A MediaMuxer can be used to combine multiple media tracks from various formats into a single file. It is the primary class for creating, encoding and streaming multiple audio and video streams.

Overview

The 

          
           
android.media.MediaMuxer
class performs all of the necessary operations required to multiplex audio and video tracks into a single file. It supports a range of audio and video formats such as MP4, 3GP, AAC, MP3, WAV, FLV, and others. It also has the capability to transcode or package existing media files into other formats.

Usage

There are two ways to use the 

          
           
android.media.MediaMuxer
class:
  • Create Muxer with MediaRecorder - The MediaRecorder can be used to capture raw audio and video streams which can then be fed into the MediaMuxer to create a single file. The following code snippet shows how to use the MediaMuxer and MediaRecorder classes to record and multiplex an audio and video stream: 
    
            
             
// Create MediaRecorder 
MediaRecorder mediaRecorder = new MediaRecorder();
mediaRecorder.setAudioSource(MediaRecorder.AudioSource.MIC);
mediaRecorder.setVideoSource(MediaRecorder.VideoSource.CAMERA);
mediaRecorder.setOutputFormat(MediaRecorder.OutputFormat.MPEG_4);
mediaRecorder.setAudioEncoder(MediaRecorder.AudioEncoder.AAC);
mediaRecorder.setVideoEncoder(MediaRecorder.VideoEncoder.H264);

// Create MediaMuxer
MediaMuxer mediaMuxer = new MediaMuxer(outputPath, MediaMuxer.OutputFormat.MUXER_OUTPUT_MPEG_4);

// Add Tracks
mediaMuxer.addTrack(mediaRecorder.getAudioTrack());
mediaMuxer.addTrack(mediaRecorder.getVideoTrack());

// Start recording
mediaRecorder.start();

// Stop recording
mediaRecorder.stop();

// Release resources
mediaMuxer.release();
  • Create Muxer using existing encoded media files - The MediaMuxer can also be used to package existing media files into the desired output format. This allows the user to transcode existing media files into other formats. The following code snippet shows how to use the MediaMuxer to package an existing media file into an MP4 container: 
    
            
             
// Create MediaMuxer
MediaMuxer mediaMuxer = new MediaMuxer(outputPath, MediaMuxer.OutputFormat.MUXER_OUTPUT_MPEG_4);

// Add media sources
mediaMuxer.addSource(audioPath);
mediaMuxer.addSource(videoPath);

// Start transcoding
mediaMuxer.startTranscoding();

// Wait until finished
mediaMuxer.waitForTranscodingFinished();

// Release resources
mediaMuxer.release();

Conclusion

The 

          
           
android.media.MediaMuxer
class provides an easy way to multiplex audio and video streams into a single file or package existing media files into different formats. It is a powerful tool that enables developers to create media-rich applications quickly and easily.
