javax.ejb.Singleton

A javax.ejb.Singleton EJB is a server-side component that contains business logic accessed by Java EE clients in order to request operations and perform tasks. Here are some of the main features of this type of EJB:
  • It is a singleton, meaning there is only one instance of this bean.
  • This bean is shared across all clients, so no two clients can operate on the same data concurrently and they will always get consistent results when they issue requests.
  • It provides a high level of security, as it is only accessible through authorized APIs.
  • Invoking methods on this bean is very efficient and cost-effective as the same instance is used for all requests.

Example

Below is a simple example of a javax.ejb.Singleton EJB:


          
           
@Singleton
public class MyEjb {
    public void doSomething() {
        // Do something here
    }
}

To invoke the method doSomething on this bean, we can use a java ee client as follows:


          
           
MyEjb myEjb = new InitialContext().lookup("java:global/MyEjb");
myEjb.doSomething();

The method doSomething will now be executed on the singleton bean instance.
