android.graphics.drawable.NinePatchDrawable

NinePatchDrawable ninePatchDrawable = new NinePatchDrawable(bitmap, paint);

The NinePatch bitmap contains the necessary information about how to scale the image when drawn on the Canvas. It is important to note that the image must have a one-pixel border around it in order to properly scale. The Paint object is used to define the color and style of the drawable object.

Drawing a NinePatchDrawable
