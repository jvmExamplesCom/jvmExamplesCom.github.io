Introduction to Android WebKit JsPromptResult

Android WebKit JsPromptResult is a class used to provide a JavaScript allow or deny response to a JavaScript prompt. It enables applications to present a JavaScript value or a prompt dialog to the user from an onJsPrompt() call. Applications may use this class to handle both confirm and prompt dialogs.

Code Examples

The following example demonstrates how to use 
JsPromptResult
in an Android application: 

          
           
// Create a new JsPromptResult instance
JsPromptResult result = new JsPromptResult(); 

// Set the result
result.confirm("My Result"); 

// Get the result
String response = result.getStringResult(); 

// Handle the response
if (response.equals("My Result")) { 
    // Do something...
}

JsPromptResult
in an application to display a prompt dialog to the user and handle the user's response:
