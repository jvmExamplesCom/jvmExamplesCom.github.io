javax.enterprise.inject.spi.InjectionPoint

The javax.enterprise.inject.spi.InjectionPoint interface provides access to the information associated with an injection point and requests the container to provide an instance of the required type. It represents an injection point, which is a program construct where a value can be injected.

Basic Usage

The following code examples illustrate basic usage of the InjectionPoint interface:


          
           
public class MyBean implements MyBeanInterface {
    @Inject
    private InjectionPoint injectionPoint;

    public void doSomething() {
        System.out.println("Injection point type: " +
            injectionPoint.getType());
    }
}

          

          
           
InjectionPoint.getMember()

The getMember() method returns the member on which the injection point is declared, or null if the injection point is not declared on a field or parameter of a class.


          
           
public class MyBean implements MyBeanInterface {
    @Inject
    private InjectionPoint injectionPoint;

    public void doSomething() {
        if (injectionPoint.getMember() != null) {
            System.out.println("Member type: " +
                injectionPoint.getMember().getType());
        }
    }
}

          

          
           
InjectionPoint.getQualifiers()

The getQualifiers() method returns the set of qualifiers of the injection point. Any injected instances must have at least all of the qualifiers returned by this method.


          
           
public class MyBean implements MyBeanInterface {
    @Inject
    private InjectionPoint injectionPoint;

    public void doSomething() {
        Set<Annotation> qualifiers = injectionPoint.getQualifiers();
        for(Annotation qualifier: qualifiers) {
            System.out.println("Qualifier: " + qualifier.getClass());
        }
    }
}

          

          
           
InjectionPoint.getType()

The getType() method returns the required type of the injection point. Any injected instances must have the type returned by this method.


          
           
public class MyBean implements MyBeanInterface {
    @Inject
    private InjectionPoint injectionPoint;

    public void doSomething() {
        System.out.println("Required type: " +
            injectionPoint.getType());
    }
}

          

          
           
InjectionPoint.getBean()

The getBean() method returns the bean that declares the injection point, or null if the injection point is not declared by a bean.


          
           
public class MyBean implements MyBeanInterface {
    @Inject
    private InjectionPoint injectionPoint;

    public void doSomething() {
        if (injectionPoint.getBean() != null) {
            System.out.println("Bean name: " +
                injectionPoint.getBean().getName());
        }
    }
}

Conclusion

The javax.enterprise.inject.spi.InjectionPoint interface provides access to the information associated with an injection point and requests the container to provide instances of the required type. It is a powerful tool for creating more robust and flexible applications.
