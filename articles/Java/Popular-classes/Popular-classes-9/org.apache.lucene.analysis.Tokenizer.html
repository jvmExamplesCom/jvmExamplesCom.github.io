About the Java org.apache.lucene.analysis.Tokenizer

The Java org.apache.lucene.analysis.Tokenizer is an invaluable tool for tokenizing text. This API, which belongs to the Apache project, is part of Lucene, Apache’s full-text search library.

Tokenizer API

The Tokenizer API is composed of two main components: the Split class and the TokenStream class. The Split class is used for producing tokens from a source of text split into strings. Each String is then processed with the output being a sequence of Tokens. The TokenStream class is used to analyze the sequence of tokens, to produce useful information such as the frequency of each token, the rank of terms, or to identify and filter out irrelevant words.
