javafx.scene.control.ToggleGroup

The javafx.scene.control.ToggleGroup class is a specialized group of Toggle objects whose selection state are mutually exclusive. ToggleGroup provides extra functionality beyond that of any individual Toggle object. It allows for only one Toggle to be selected at a time, so that no matter how many Toggles are in the ToggleGroup, only one of them can have a selected property set to true. The ToggleGroup also provides API for three specific scenarios: selecting the first toggle by default, retrieving the currently selected toggle, and listening for changes in selection state.
