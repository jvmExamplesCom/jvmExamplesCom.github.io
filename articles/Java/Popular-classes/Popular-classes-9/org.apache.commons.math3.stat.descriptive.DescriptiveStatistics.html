org.apache.commons.math3.stat.descriptive.DescriptiveStatistics

The org.apache.commons.math3.stat.descriptive.DescriptiveStatistics class is a java utility class which provides descriptive statistics on data in an array or comparable collection of numbers. Descriptive statistics include measures of central tendency such as mean and median, as well as measures of spread such as standard deviation and range. The DescriptiveStatistics class is the base class for all descriptive statistical classes in the Commons Math library.

Creating Descriptive Statistics Objects

Creating a DescriptiveStatistics object is as simple as calling the constructor with no argument:


          
           
DescriptiveStatistics stats = new DescriptiveStatistics();

A descriptive statistics instance can also be created from an existing array or collection of numbers by passing in the array or collection:


          
           
double[] data = {1, 2, 3, 4, 5};
DescriptiveStatistics stats = new DescriptiveStatistics(data);

Adding Data to Descriptive Statistics Objects

Data can be added to a descriptive statistics instance either manually, one value at a time, or in bulk from an array or collection of numbers:


          
           
// Adding one value at a time
stats.addValue(6);

// Adding an array of values
double[] values = {7, 8, 9};
stats.addValues(values);

// Adding a collection of values
List<Double> valuesList = Arrays.asList(10.0, 11.0);
stats.addValues(valuesList);

Computing Descriptive Statistics

Once the descriptive statistics object has been populated with data, the descriptive statistics can be computed using the various methods available on the object:


          
           
// Computing the mean
double mean = stats.getMean();

// Computing the standard deviation
double stdev = stats.getStandardDeviation();
