Introduction to org.apache.curator.framework.recipes.cache.ChildData

org.apache.curator.framework.recipes.cache.ChildData is an Apache Curator recipe which enables the user to keep track of changes in ZooKeeper path nodes. ChildData caches a path node's data and stats in memory. This recipe is especially convenient when the user plans to regularly read ZooKeeper paths.
