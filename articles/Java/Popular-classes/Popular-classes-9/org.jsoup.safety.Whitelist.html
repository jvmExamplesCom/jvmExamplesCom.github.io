org.jsoup.safety.Whitelist

The org.jsoup.safety.Whitelist class is a security class used to determine which HTML elements and attributes are allowed in a given input. This is especially useful when dealing with potentially malicious user input, as it can help prevent cross-site scripting attacks and other malicious behavior. The Whitelist class provides numerous methods for configuring a whitelist, such as adding or removing tags, attributes, and protocols, and setting the filtering level.

This constructor creates an empty whitelist which will allow all tags and attributes by default.
