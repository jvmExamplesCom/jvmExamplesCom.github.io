org.jsoup.safety.Whitelist

The org.jsoup.safety.Whitelist class is a security class used to determine which HTML elements and attributes are allowed in a given input. This is especially useful when dealing with potentially malicious user input, as it can help prevent cross-site scripting attacks and other malicious behavior.
