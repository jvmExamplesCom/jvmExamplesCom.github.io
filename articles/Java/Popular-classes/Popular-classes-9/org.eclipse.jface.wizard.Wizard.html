org.eclipse.jface.wizard.Wizard

The org.eclipse.jface.wizard.Wizard class is a base class used to create wizards in the Eclipse platform. It provides support for managing pages, providing help, performing validation, and optionally displaying a progress monitor during lengthy operations.

To create a wizard, you must subclass org.eclipse.jface.wizard.Wizard and override several methods. The following example demonstrates a minimal wizard:

    public boolean performFinish() {

        return true;
    }

}
