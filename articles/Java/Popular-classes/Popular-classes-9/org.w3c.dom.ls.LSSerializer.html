org.w3c.dom.ls.LSSerializer

The org.w3c.dom.ls.LSSerializer interface provides an API to serialize DOM nodes into an XML document. The LSSerializer can be used to turn any DOM document into an XML representation of the document. This can be useful not only for displaying the DOM structure, but also for sending the DOM to a file, or to another application.

DOMImplementation domImpl = new DOMImplementationImpl();

Document doc = domImpl.createDocument(null, "html", docType);
// Create the DOM here

LSSerializer serializer = ((DOMImplementationLS) domImpl).createLSSerializer();
String xml = serializer.writeToString(doc);

    <!-- Content Goes Here -->
