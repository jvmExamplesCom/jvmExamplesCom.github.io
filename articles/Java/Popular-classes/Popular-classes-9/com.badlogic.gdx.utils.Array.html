com.badlogic.gdx.utils.Array

The 

          
           
com.badlogic.gdx.utils.Array
class provides a resizable, ordered array of objects, similar to an ArrayList. This class is the framework for almost all other collection classes provided in libGDX.

Constructors


          
           
public Array()

Constructs an empty array with an initial capacity of 16.


          
           
public Array (boolean ordered, int capacity, Class arrayType)

Constructs an empty array with the specified initial capacity and array type.


          
           
public Array (boolean ordered, int capacity, Class arrayType, Array<?> source)

Constructs a new array containing the elements in the specified array.

Methods


          
           
public void add (Object o)

Adds the specified object to the end of this array.


          
           
public void clear ()
Removes all of the elements from this array. 

          
           
public boolean contains (Object o)

Returns true if this array contains the specified object.


          
           
public Object get (int index)

Returns the element at the specified position in this array.


          
           
public int indexOf (Object o)

Returns the index of the first occurrence of the specified object in this array, or -1 if the object is not present.


          
           
public boolean removeValue (Object o)

Removes the first occurrence of the specified object from this array. Returns true if the object was removed.

public void sort ()

Sorts this array according to the natural ordering of its elements.
