The 

          
           
com.badlogic.gdx.utils.Array
class provides a resizable, ordered array of objects, similar to an ArrayList. This class is the framework for almost all other collection classes provided in libGDX.

Constructs an empty array with an initial capacity of 16.


          
           
public Array (boolean ordered, int capacity, Class arrayType)
