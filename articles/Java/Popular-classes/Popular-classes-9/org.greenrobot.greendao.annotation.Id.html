org.greenrobot.greendao.annotation.Id
annotation is as follows: 

          
           
@Id 
private long id;

Example

org.greenrobot.greendao.annotation.Id
annotation is used: 

          
           
@Entity 
public class Car {

    @Id

org.greenrobot.greendao.annotation.Id
annotation.
