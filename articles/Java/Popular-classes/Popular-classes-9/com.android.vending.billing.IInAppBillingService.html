Introduction to com.android.vending.billing.IInAppBillingService

The com.android.vending.billing.IInAppBillingService, or In-app Billing Service, is a part of the Android in-app billing library that provides developers access to the In-app Billing API. This API allows developers to build applications that sell digital products from within the app. The In-App Billing API can be used to purchase virtual goods, such as coins, extra lives, and special abilities, as well as subscription-based products and services.

Overview

The IInAppBillingService interface is the main entry point for interacting with the In-app Billing API. All methods in the service require authentication, authorization and verification of the Google Play developer as an authorized merchant. It allows developers to:

  • Query the list of products available for purchase.
  • Purchase items.
  • Consume a purchased item.
  • Manage subscriptions and recurring purchases.
  • Notify the application when a purchase has been made.
  • Send request for refunds and adjustments.
