javax.naming.ldap.Rdn stands for Relative Distinguished Name and is part of the Java Naming and Directory Interface (JNDI). This class provides a way for users to parse, construct and compare relative distinguished names (RDNs) which are components of common names in LDAP-based directory systems such as Active Directory.
