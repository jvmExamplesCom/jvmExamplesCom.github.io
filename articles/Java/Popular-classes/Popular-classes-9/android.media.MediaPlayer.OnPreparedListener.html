android.media.MediaPlayer.OnPreparedListener

The 

          
           
android.media.MediaPlayer.OnPreparedListener
interface is used to receive notifications in the MediaPlayer class when a media data source is ready to start playing. This interface provides a callback method, 
onPrepared()
, which is invoked when the MediaPlayer instance has successfully prepared the media for playback.

Implementation

To use the OnPreparedListener interface and receive notifications in the MediaPlayer's callback method, it must be implemented by the activity or fragment that includes the MediaPlayer. Then, it must be set as a listener for the MediaPlayer. For example:


          
           
// Declare MediaPlayer instance as a field
private MediaPlayer mPlayer;

// Initialize it in onCreate()
mPlayer = new MediaPlayer();

// Set listener
mPlayer.setOnPreparedListener(new MediaPlayer.OnPreparedListener() {
    @Override
    public void onPrepared(MediaPlayer mp) {
        // Do something with the media player
    }
});

Calling the Callback

Once the listener has been set, the 
onPrepared()
method will be called each time the media data source has been successfully prepared. For example, after calling the 
prepareAsync()
method, the 
onPrepared()
callback will be triggered when the MediaPlayer is ready to start playback. 

          
           
mPlayer.prepareAsync();

onPrepared Parameters

The 
onPrepared()
callback method takes a single parameter, a 
MediaPlayer
instance. This represents the MediaPlayer object which is the source of the event. It can be used to trigger further actions, such as starting playback. 

          
           
@Override
public void onPrepared(MediaPlayer mp) {
    mp.start();
}

Example Usage

The following example shows how to use 

          
           
android.media.MediaPlayer.OnPreparedListener
to play an audio file when it is ready. The 
onPrepared()
callback is used to start playback when the MediaPlayer is ready. 

          
           
// Declare MediaPlayer instance as a field
private MediaPlayer mPlayer;

@Override
protected void onCreate(Bundle savedInstanceState) {
    super.onCreate(savedInstanceState);

    // Initialize MediaPlayer
    mPlayer = new MediaPlayer();
    mPlayer.setAudioStreamType(AudioManager.STREAM_MUSIC);

    // Set listener
    mPlayer.setOnPreparedListener(new MediaPlayer.OnPreparedListener() {
        @Override
        public void onPrepared(MediaPlayer mp) {
            // Start playback
            mp.start();
        }
    });

    // Start loading the audio file
    mPlayer.setDataSource("http://example.com/audio.mp3");
    mPlayer.prepareAsync();
}
