Introduction to org.apache.commons.lang.time.DateUtils

org.apache.commons.lang.time.DateUtils is a Java library maintained by the Apache Software Foundation. It provides utility methods for manipulating and formatting java.util.Date and java.util.Calendar objects. It is used to quickly manipulate dates, such as compare two dates, truncate dates, or format a date into a string.

Using DateUtils

In order to use DateUtils, you need to add the commons-lang3 dependency to your project. You can do this using maven:


          
           
<dependency>
    <groupId>org.apache.commons</groupId>
    <artifactId>commons-lang3</artifactId>
    <version>3.9</version>
</dependency>

Once the dependency is added, you can start using DateUtils in your code. The following example shows how to compare two dates:


          
           
Date date1 = new Date(); // January 1, 2019
Date date2 = new Date(12345678); // February 27, 1909

int result = DateUtils.compare(date1, date2);

System.out.println("Result: " + result); // Outputs "Result: 1"

The compare method returns 0 if the dates are equal, a positive number if the first date is after the second date, and a negative number if the first date is before the second date.

Truncating Dates

You can also use DateUtils to truncate a date. This means you can get a date without its time component. For example:


          
           
Date date = new Date(); // January 1, 2019 09:13:49

Date truncatedDate = DateUtils.truncate(date, Calendar.HOUR);

System.out.println("Truncated Date: " + truncatedDate); // Outputs "Truncated Date: Tue Jan 01 09:00:00 EST 2019"

In this example, we are truncating the date to the hour, so all time components after the hour will be set to 0. You can also truncate to other levels, such as Calendar.MINUTE, Calendar.SECOND, Calendar.MILLISECOND, etc.

Formatting Dates

DateUtils also provides methods for formatting dates into strings. For example, you can use the format method to format a date into a specific pattern:


          
           
Date date = new Date(); // January 1, 2019 09:13:49

String formattedDate = DateUtils.format(date, "yyyy-MM-dd HH:mm:ss");

System.out.println("Formatted Date: " + formattedDate); // Outputs "Formatted Date: 2019-01-01 09:13:49"

The pattern argument can be any valid SimpleDateFormat pattern. You can also use one of the predefined patterns provided by DateUtils: ISO_DATETIME_TIME_ZONE_FORMAT, ISO_DATE_TIME_ZONE_FORMAT, ISO_DATE_TIME_FORMAT, etc.

Conclusion

This article has shown some of the basic features offered by DateUtils. It is a very useful library when it comes to manipulating and formatting dates. For more information, please refer to the official documentation.
