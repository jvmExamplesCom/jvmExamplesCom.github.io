javax.swing.event.TreeExpansionEvent

Introduction

javax.swing.event.TreeExpansionEvent is a class that defines the event which is fired when a tree node is collapsed or expanded. It is used to notify any TreeExpansionListener instances that have been registered with a JTree.

Creating and Using a TreeExpansionEvent

The TreeExpansionEvent class has two constructors. The one with the signature TreeExpansionEvent(JTree source, TreePath path) is used most frequently. Here, source is the JTree instance which generated the event, and path is the TreePath that identifies the node that has been collapsed or expanded.

Here is an example of how to create and use a TreeExpansionEvent:


          
           
JTree tree = new Jtree();
...
TreeExpansionEvent event = new TreeExpansionEvent(tree, new TreePath(node));
// do something with event

Once a TreeExpansionEvent instance is created, it can be used with any TreeExpansionListener instances. For example, if you want all TreeExpansionListeners to be notified when a node is expanded, you can call the tree's fireTreeExpanded() method like this:


          
           
tree.fireTreeExpanded(event);

Example

Here is a complete example demonstrating the usage of javax.swing.event.TreeExpansionEvent:


          
           
import javax.swing.*;
import javax.swing.event.TreeExpansionEvent;
import javax.swing.event.TreeExpansionListener;
import javax.swing.tree.DefaultMutableTreeNode;
import javax.swing.tree.TreePath;

public class TreeExpansionEventExample {

    public static void main(String[] args) {
        // Create the root node
        DefaultMutableTreeNode rootNode = new DefaultMutableTreeNode("Root");
        // Create the child nodes
        DefaultMutableTreeNode nodeA = new DefaultMutableTreeNode("Node A");
        DefaultMutableTreeNode nodeB = new DefaultMutableTreeNode("Node B");
        // Add the child nodes to the root node
        rootNode.add(nodeA);
        rootNode.add(nodeB);

        // Create the JTree
        JTree tree = new JTree(rootNode);
        tree.addTreeExpansionListener(new MyTreeExpansionListener());

        // Expand the root node
        tree.expandPath(new TreePath(rootNode));
    }

    private static class MyTreeExpansionListener implements TreeExpansionListener {
        @Override
        public void treeExpanded(TreeExpansionEvent event) {
            System.out.println("Tree expanded: " + event.getPath());
        }

        @Override
        public void treeCollapsed(TreeExpansionEvent event) {
            System.out.println("Tree collapsed: " + event.getPath());
        }
    }
}

In this example, a JTree is created with the root node "Root" and two child nodes "Node A" and "Node B". A TreeExpansionListener is added to this tree, and it is used to print out information about the tree when it is expanded or collapsed. The root node is then expanded, triggering the TreeExpansionListener and resulting in the output:


          
           
Tree expanded: [Root]
