com.badlogic.gdx.graphics.Color
class is a java library that enables developers to create colors and manipulate them to create custom effects in their applications. Colors also be manipulated using basic math operations like addition, subtraction, and multiplication of colors, interpolation of colors, and conversion between representation types.

Usage

Creating colors from red, green, and blue (RGB) values: 

          
           
Color color = new Color(red, green, blue); // range 0 - 1
Creating colors from hue, saturation, and value (HSV) values: 

          
           
Color color = new Color().fromHsv(hue, saturation, value); // range 0 - 1

Manipulating colors with multiplication:
