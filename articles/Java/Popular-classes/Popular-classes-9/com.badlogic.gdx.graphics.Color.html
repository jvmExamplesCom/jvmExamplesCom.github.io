Introduction

The 

          
           
com.badlogic.gdx.graphics.Color
class is a java library that enables developers to create colors and manipulate them to create custom effects in their applications. It provides all the necessary functions to easily create colors from standard colors, such as Red, Green, and Blue (RGB) or Hue, Saturation, and Value (HSV) values. Colors also be manipulated using basic math operations like addition, subtraction, and multiplication of colors, interpolation of colors, and conversion between representation types.

Usage

Creating colors from red, green, and blue (RGB) values: 

          
           
Color color = new Color(red, green, blue); // range 0 - 1
Creating colors from hue, saturation, and value (HSV) values: 

          
           
Color color = new Color().fromHsv(hue, saturation, value); // range 0 - 1
Manipulating colors with addition and subtraction: 

          
           
Color resultColor = Color.add(color1, color2);
Color resultColor = Color.sub(color1, color2);
Manipulating colors with multiplication: 

          
           
Color resultColor = Color.mul(color, factor); // factor in range 0.0f - 1.0f
Interpolating between two colors: 

          
           
Color resultColor = Color.lerp(color1, color2, fraction); // fraction in range 0.0f - 1.0f
Converting between representations: 

          
           
Color rgbColor = color.toRgb();
Color hsvColor = color.toHsv();
