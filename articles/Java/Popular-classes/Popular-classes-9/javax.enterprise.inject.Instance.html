javax.enterprise.inject.Instance

javax.enterprise.inject.Instance is an interface used by the Java Enterprise Edition (EE) platform to create objects via Dependency Injection (DI). It is part of the CDI API which makes it possible to inject bean references into classes, and supports type checking, re-injecting objects, and provides access to the context information of beans.

This interface contains a variety of methods for accessing and modifying objects. Some of these methods include: 

get()
, 
select()
, 
isUnsatisfied()
, and 
isAmbiguous()
. These methods make it easy to retrieve and manipulate objects in an application, as well as determine if any unsatisfied or ambiguous dependencies exist.

Example Usage

Let's look at some examples of how to use the javax.enterprise.inject.Instance interface. First, let's create a class called Library which will contain a Book instance.


          
           
public class Library {
    @Inject
    private Instance<Book> bookInstance;

    public void printBooks() {
        while (bookInstance.isUnsatisfied()) {
            System.out.println(bookInstance.get());
        }
    }
}

@Inject

Instance

isUnsatisfied()

Conclusion
