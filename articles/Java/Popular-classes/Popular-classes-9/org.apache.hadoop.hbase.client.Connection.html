org.apache.hadoop.hbase.client.Connection in Java

Java provides the org.apache.hadoop.hbase.client.Connection interface, which enables a client to interact with a HBase cluster. It is typically part of the HBase client library and used to configure connections and manage operations.

Create a Connection With a Configuration Object

Creating a Connection object with a Hadoop Configuration object is the most common way to get access to a HBase cluster. To do this, you can use the 


          
           
ConnectionFactory.createConnection(Configuration conf)
method, as shown here: 

          
           
Configuration conf = HBaseConfiguration.create();
conf.set("hbase.zookeeper.quorum", "localhost:2181");

Connection conn = ConnectionFactory.createConnection(conf);

Using Connection Interfaces

Once you have created a Connection object, you can use its associated interfaces to interact with the HBase cluster. These interfaces include the Admin, Table, and RegionLocator classes, which provide methods for performing operations such as creating and deleting tables, retrieving region information, configuring table properties, and more. TableName tableName = TableName.valueOf("my_table");

Admin admin = conn.getAdmin();
HTableDescriptor tableDesc = new HTableDescriptor(tableName);

// add column families

Close a Connection
