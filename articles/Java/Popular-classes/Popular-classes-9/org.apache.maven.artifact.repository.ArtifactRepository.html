org.apache.maven.artifact.repository.ArtifactRepository

The org.apache.maven.artifact.repository.ArtifactRepository class is a component of the Maven repository system responsible for managing and storing artifacts. It provides methods to support the creation, lookup and retrieval of artifacts from a defined repository.

Create Artifact Repository

You can create an ArtifactRepository instance using the ArtifactRepositoryFactory as follows:


          
           
ArtifactRepositoryFactory factory = new DefaultArtifactRepositoryFactory();
ArtifactRepository repo = factory.createArtifactRepository(repoId, repoUrl, layout);

Lookup and Retrieve an Artifact

You can lookup and retrieve an artifact from the repository using the methods in ArtifactResolver.


          
           
Artifact artifact = new DefaultArtifact("org.apache.maven", "maven-core", "3.5.4");
ArtifactRequest request = new ArtifactRequest().setArtifact(artifact).setRepositories(repositories);
ArtifactResult result = repo.resolveArtifact(session, request);

The lookup and retrieval is based on the artifact coordinates, i.e., groupId, artifactId, version and type. The result contains the resolved artifact, which can be used to read or write the artifact's data.

Find Artifact Versions Available in Repository

You can find all versions of the artifact available in the repository using the method ArtifactVersionsRequest.


          
           
ArtifactVersionsRequest request = new ArtifactVersionsRequest().setArtifact(artifact).setRepositories(repositories);
ArtifactVersionsResult result = repo.resolveVersionRange(session, request);

It returns the ArtifactVersionsResult object, which contains the list of available versions of the requested artifact. You can then use this information to determine the version you need to download.
