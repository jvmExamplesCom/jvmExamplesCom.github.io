org.springframework.scheduling.annotation.AsyncConfigurer

The org.springframework.scheduling.annotation.AsyncConfigurer class provides an annotation-based configuration for asynchronous methods in Spring. It allows developers to easily configure an asynchronous method without the need to manually create one. AsyncConfigurer provides a convenient way to configure and handle task scheduling and execution.

Configure Annotation

You can use the @Async annotation on any method you would like to execute asynchronously. This annotation can be used with a custom Executor, TaskScheduler, or RejectedExecutionHandler. Additionally, the Annotation can be used along with other annotations such as each successful execution. 


          
           
@Async
public void processTask(int number) {
   System.out.println("Processing task: " + number);
}

Java Configuration

To configure AsyncConfigurer, Java configures an AsyncConfigurerSupport class to implement the getAsyncExecutor() and getTaskExecutor() methods. The AsyncConfigurerSupport class implements both the AsyncConfigurer and TaskExecutor interfaces. 


          
           
@Configuration
@EnableAsync
public class AsyncConfiguration extends  AsyncConfigurerSupport {

	@Override
	public Executor getAsyncExecutor() {
		ThreadPoolTaskExecutor executor = new ThreadPoolTaskExecutor();
		executor.setCorePoolSize(2);
		executor.setMaxPoolSize(5);
		executor.setQueueCapacity(500);
		executor.setThreadNamePrefix("AsyncExecutor-");
		executor.initialize();
		return executor;
	}

	@Override
	public TaskExecutor getTaskExecutor() {
		ThreadPoolTaskExecutor executor = new ThreadPoolTaskExecutor();
		executor.setCorePoolSize(2);
		executor.setMaxPoolSize(5);
		executor.setQueueCapacity(500);
		executor.setThreadNamePrefix("TaskExecutor-");
		executor.initialize();
		return executor;
	}

}

XML Configuration

XML configuration of the AsyncConfigurer is also supported by defining a bean of type ThreadPoolTaskExecutor. Additionally, the bean can be configured to use a custom ScheduledExecutorService by injecting it into the constructor. 


          
           
<bean id="threadPoolTaskExecutor" class="org.springframework.scheduling.concurrent.ThreadPoolTaskExecutor">
    <property name="corePoolSize" value="2" />
    <property name="maxPoolSize" value="5" />
    <property name="queueCapacity" value="500" />
    <property name="threadNamePrefix" value="AsyncExecutor-" />
</bean>

Using AsyncConfigurer for Scheduling Tasks

AsyncConfigurer can be used to schedule tasks using either the @Scheduled or @Async annotations. The following code shows how to create a scheduled task using the @Scheduled annotation.


          
           
@Scheduled(fixedRate = 1000)
public void run() {
    System.out.println("Running scheduled task...");
}

@Async

    System.out.println("Processing task: " + number);
