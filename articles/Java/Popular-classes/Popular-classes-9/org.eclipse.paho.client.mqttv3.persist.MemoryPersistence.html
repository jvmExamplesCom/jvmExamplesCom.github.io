Introduction to org.eclipse.paho.client.mqttv3.persist.MemoryPersistence

The org.eclipse.paho.client.mqttv3.persist.MemoryPersistence class is an implementation of the MQTT Client's persistence interface. This class can be used with any MQTT client to provide basic persistence for messages, subscriptions and other data.

Why Use MemoryPersistence?
