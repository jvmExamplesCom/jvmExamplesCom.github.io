javax.xml.validation.Validator

The javax.xml.validation.Validator class is an abstract superclass for all schema validators. This class forms the foundation of XML validation within the Java framework. It provides a set of methods for working with XML documents, schemas, and more.

A Validator object can be created by passing a Source object (which may be a DOM, SAX, or XSLT document) to the ValidatorFactory.newValidator() method. The Validator object is then used to parse and validate the document.

Validate a Schema with Validator

To validate a document against a schema, you can use a Validator instance. First, create a SchemaFactory and a Source instance, which points to the schema:


          
           
SchemaFactory factory = SchemaFactory.newInstance(XMLConstants.W3C_XML_SCHEMA_NS_URI);
Source schemaFile = new StreamSource(new File("schema.xsd"));
Schema schema = factory.newSchema(schemaFile);
Then, create a Validator instance: 

          
           
Validator validator = schema.newValidator();

Finally, use the Validator instance to validate the document:


          
           
validator.validate(new StreamSource(new File("document.xml")));

This will throw an exception if the document is not valid according to the schema. If no exception is thrown, then the document is valid.

Set Error Handler

The Validator class has a setErrorHandler() method, which can be used to customize the handling of validation errors. The ErrorHandler argument should implement the ErrorHandler interface, which defines three methods for handling errors, warnings, and fatal errors:


          
           
public interface ErrorHandler {
    void warning(SAXParseException e);
    void error(SAXParseException e);
    void fatalError(SAXParseException e);
}

For example, you can create a custom error handler that prints out messages to the console:


          
           
validator.setErrorHandler(new ErrorHandler() {
    public void warning(SAXParseException e) {
        System.err.println("[Warning] " + e.getMessage());
    }

    public void error(SAXParseException e) {
        System.err.println("[Error] " + e.getMessage());
    }

    public void fatalError(SAXParseException e) {
        System.err.println("[Fatal Error] " + e.getMessage());
    }
});

Now when the Validator instance validates the document, its custom error handler will be called for each validation error.

Conclusion

In this tutorial, we discussed the javax.xml.validation.Validator class and how to use it to validate an XML document against a schema. We also saw how to customize error handling with an ErrorHandler.
