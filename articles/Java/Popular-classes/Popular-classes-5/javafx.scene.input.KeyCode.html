javafx.scene.input.KeyCode

javafx.scene.input.KeyCode is an enum class that provides key codes for keys and key combinations used in a Scene's input map. It makes it easy for developers to work with the system's keyboard events, including the mapping of physical keys and key combinations to virtual keys.
