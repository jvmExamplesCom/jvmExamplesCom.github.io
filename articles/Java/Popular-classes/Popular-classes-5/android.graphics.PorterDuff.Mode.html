Understanding android.graphics.PorterDuff.Mode

The android.graphics.PorterDuff.Mode class is used to define how two layers interact in an Android UI. It works in conjunction with the Porter-Duff color blending algorithm and can provide a variety of visual effects in image processing. This article will cover what PorterDuff.Mode is, how to use it, and some of the possible effects it can produce.

What is PorterDuff.Mode?

PorterDuff.Mode is an Enum class in the android.graphics package. It defines the blending mode when combining two layers of images in an Android UI. The same blending modes can be used for any combination of drawing primitives and raster operations, including applying filters to images. For example, when drawing on a canvas, you might have one layer representing the background and another layer representing a shape or drawing. PorterDuff.Mode can be used to define how these two layers should interact. Commonly used blending modes include CLEAR, SRC_OVER, SRC_IN, and DST_OUT.

How to Use PorterDuff.Mode

Using PorterDuff.Mode is simple. First, you create a Paint object and set its PorterDuff.Mode. Then, you call the draw(Canvas) method of your Paint object, passing in the two layers of images you want to blend. The result will be a new layer that combines both layers according to the PorterDuff.Mode you set earlier.

Canvas canvas = new Canvas();
canvas.draw(myBackgroundImage, 0, 0, paint);

Possible Effects of PorterDuff.Mode

When two layers are blended together using PorterDuff.Mode, different effects can be achieved depending on which blending mode is used. Some of the most common effects are described below.

CLEAR


          
           
The CLEAR mode replaces the destination image with the source image.

SRC_OVER


          
           
The SRC_OVER mode draws the source image over the destination image.

SRC_IN


          
           
The SRC_IN mode draws the source image within the bounds of the destination image.

DST_OUT


          
           
The DST_OUT mode cuts out the portion of the destination image that lies outside the bounds of the source image.
