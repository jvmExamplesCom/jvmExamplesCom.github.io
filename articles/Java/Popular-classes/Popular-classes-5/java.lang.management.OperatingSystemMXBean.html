Introduction to Java.lang.management.OperatingSystemMXBean

Java.lang.management.OperatingSystemMXBean is an interface which provides a view of an operating system, including operating system specific information such as usernames, supported architectures, and processes. The OperatingSystemMXBean is an MXBean, and therefore it must be accessed through instances of the ManagementFactory. To create an instance of the OperatingSystemMXBean, use the following code: 

          
           
OperatingSystemMXBean osMBean = 
ManagementFactory.getOperatingSystemMXBean();
This will create an OperatingSystemMXBean instance.
