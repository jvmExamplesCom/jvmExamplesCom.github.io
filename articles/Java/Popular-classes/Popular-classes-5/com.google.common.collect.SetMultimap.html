Java: com.google.common.collect.SetMultimap

A SetMultimap is a class from the Google Guava library that allows you to map multiple values to a single key without worrying about duplicates. It is similar to a Map in that it has a key and a value, but with the SetMultimap, you can assign multiple values to the same key.

Creating a SetMultimap Object

The following code example shows how to create a SetMultimap:


          
           
import com.google.common.collect.SetMultimap;

SetMultimap<String, Integer> multimap = HashMultimap.create();

put()
method. The following code example adds three elements to the SetMultimap: 

          
           
multimap.put("a", 1);
multimap.put("a", 2);
multimap.put("b", 3);

Retrieving Elements

Collection<Integer> aValues = multimap.get("a");

Removing Elements

remove()

multimap.remove("a", 2);

Iterating Over Elements
