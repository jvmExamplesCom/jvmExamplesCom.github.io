android.net.http.SslError

android.net.http.SslError is an exception that is thrown when an SSL handshake fails. It provides information about the cause of the error as well as details about the specific SSL protocol version and cipher suite that was negotiated. It can be used to aid in debugging and diagnosing SSL related issues in Android apps. This class is a subclass of IOException.

Code Examples

The following code examples show how to catch an SslError and display its information:


          
           
try {
    // perform an SSL handshake
} catch (SslError e) {
    String message = e.getMessage();
    int index = e.getIndex();
    String protocol = e.getProtocol();
    String cipherSuite = e.getCipherSuite();
    System.out.println("  Message: " + message);
    System.out.println("  Index: " + index);
    System.out.println("  Protocol: " + protocol);
    System.out.println("  Cipher Suite: " + cipherSuite);
}
