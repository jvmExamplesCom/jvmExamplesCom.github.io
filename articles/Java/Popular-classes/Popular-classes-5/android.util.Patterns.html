Android.util.Patterns

Android.util.Patterns is a utility class for providing commonly used regular expressions. It contains both static and non-static methods for reusable text patterns.

Methods

Android.util.Patterns provides several string pattern constants, as well as methods to check the validity of email addresses, domain names, and phone numbers.

Constants


          
           
// Email address regex
public static final Pattern EMAIL_ADDRESS
            = Pattern.compile( "[a-zA-Z0-9\\+\\.\\_\\%\\-\\+]{1,256}" +
                    "\\@" +
                    "[a-zA-Z0-9][a-zA-Z0-9\\-]{0,64}" +
                    "(" +
                        "\\." +
                        "[a-zA-Z0-9][a-zA-Z0-9\\-]{0,25}" +
                    ")+"
             );

// Domain name regex
public static final Pattern DOMAIN_NAME
            = Pattern.compile(
                // minimum 2 characters plus tld
                "(([a-zA-Z0-9]|[a-zA-Z0-9][a-zA-Z0-9\\-]*[a-zA-Z0-9])\\.)+" +
                // tlds
                "(com|org|net|edu|info|biz|gov|name|aero|jobs|mil|co.uk|me|mobi)"
            );

Checking Email Addresses


          
           
// Checks that the given CharSequence is a valid email address
public static boolean isEmailAddress(CharSequence email) {
    if (email == null) {
        return false;
    }
    return EMAIL_ADDRESS.matcher(email).matches();
}

Checking Domain Names


          
           
// Checks that the given CharSequence is a valid domain name
public static boolean isDomainName(CharSequence domainName) {
    if (domainName == null) {
        return false;
    }
    return DOMAIN_NAME.matcher(domainName).matches();
}

Conclusion

Android.util.Patterns provides several useful regex constants and methods for validating commonly used text patterns such as email addresses and domain names. This can be a great timesaver when writing code that works with user input.
