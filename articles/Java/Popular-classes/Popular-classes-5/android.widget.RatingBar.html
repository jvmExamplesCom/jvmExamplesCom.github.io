android.widget.RatingBar

Android RatingBar is a subclass of the ProgressBar and seeks to show a rating in stars. This can be used to allow users to provide feedback in a graphical way. RatingBar is often used for feedback about a piece of content, such as a movie review or product review since it allows the user to easily rate something with a small number of clicks.

Creating the RatingBar

The following XML creates a RatingBar and sets the number of stars to five. 

          
           
<RatingBar 
  android:numStars="5"
  android:layout_width="wrap_content"
  android:layout_height="wrap_content"/>
It is also possible to create the RatingBar programmatically. The following example creates a 5-star RatingBar with a step size of 0.5. 

          
           
RatingBar rb = new RatingBar(this, null, android.R.attr.ratingBarStyleIndicator);
rb.setNumStars(5);
rb.setStepSize(0.5f);

Updating the RatingBar

The RatingBar can be updated programmatically using the setRating() method. The following example sets the rating to three stars. 

          
           
RatingBar rb = (RatingBar) findViewById(R.id.ratingbar);
rb.setRating(3.0f);
It is also possible to listen to changes in the RatingBarâ€™s rating by setting an OnRatingBarChangeListener. The following example listens for changes in the rating and displays a Toast message when the rating changes. 

          
           
RatingBar rb = (RatingBar) findViewById(R.id.ratingbar);
rb.setOnRatingBarChangeListener(new OnRatingBarChangeListener() {
  @Override
  public void onRatingChanged(RatingBar ratingBar, float rating, boolean fromUser) {
    Toast.makeText(getApplicationContext(), "New Rating: " + rating, Toast.LENGTH_SHORT).show();
  }
});

Displaying the RatingBar

The RatingBar should be displayed in an Activity by inflating the XML layout containing the RatingBar. The following example inflates a layout containing a RatingBar and then makes the RatingBar visible to the user. 

          
           
Activity activity = this;
LayoutInflater inflater = activity.getLayoutInflater();
View view = inflater.inflate(R.layout.my_layout, null);
RatingBar rb = (RatingBar) view.findViewById(R.id.ratingbar);
rb.setVisibility(View.VISIBLE);
