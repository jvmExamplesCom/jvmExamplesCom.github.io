javax.swing.JToggleButton

JToggleButton is a Swing component used to display a two-state button, which can be either selected or deselected. It inherits from AbstractButton and implements ItemSelectable. JToggleButton contains both a text label and an image. The button's state may be obtained using the getModel() method.

Example 1: Basic usage of JToggleButton


          
           
JToggleButton button = new JToggleButton("Text");
button.addActionListener(new ActionListener() {
    public void actionPerformed(ActionEvent e) {
        if (button.isSelected()) {
            System.out.println("Button is selected");
        } else {
            System.out.println("Button is deselected");
        }
    }
});

Example 2: Create JToggleButton with image


          
           
JToggleButton button = new JToggleButton(new ImageIcon("image.jpg"));

Example 3: Add item listener to JToggleButton


          
           
JToggleButton button = new JToggleButton("Text");
button.addItemListener(new ItemListener() {
    public void itemStateChanged(ItemEvent e) {
        if (e.getStateChange() == ItemEvent.SELECTED) {
            System.out.println("Button is selected");
        } else {
            System.out.println("Button is deselected");
        }
    }
});

Example 4: Set action command for JToggleButton


          
           
JToggleButton button = new JToggleButton("Text");
button.setActionCommand("myCommand");

Example 5: Get current button state of JToggleButton


          
           
JToggleButton button = new JToggleButton("Text");
boolean state = button.getModel().isSelected();
System.out.println("Current state: " + state);
