org.springframework.util.AntPathMatcher

Using AntPathMatcher, you can match a request path against one or more Ant-style path patterns and then determine which one matches the most accurately. This is particularly useful when attempting to dispatch requests to controllers or methods based on the incoming request path.

    "/foo/bar/**",
    "/baz/*"
};

System.out.println(result);

  • ?

  • *

  • **
    - Matches zero or more directories in a path
