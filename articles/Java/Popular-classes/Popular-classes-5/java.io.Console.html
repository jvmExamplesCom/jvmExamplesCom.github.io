Java.io.Console Overview

The java.io.Console class provides a facility for reading and writing characters or strings from the console. It is an abstract class, so it does not provide any constructors, but its subclasses do. It provides functions for both reading and writing strings.

Reading from the Console

The 

          
           
readLine()
method is used to read a single line of text from the console. It can be used with a prompt string, in which case the prompt string will be displayed on the console before the input is accepted. The code below shows how this can be used: 

          
           
String input = System.console().readLine("Enter a value: ");
System.out.println("You entered: " + input);
The output of this code shows the following on the console: 

          
           
Enter a value: Hello
You entered: Hello
The 

          
           
readPassword()
method is similar to 

          
           
readLine()
, except that it reads a password and masks the input, replacing it with asterisks ( 

          
           
*
) as it is being typed in. This is useful when asking the user to enter a password which must be kept secret.

Writing to the Console

The 

          
           
printf()
method is used to write formatted output to the console. It takes a format string and an optional set of parameters. The code below shows how this can be used: 

          
           
double pi = 3.14;
System.out.printf("The value of pi is %.2f", pi);
The output of this code shows the following on the console: 

          
           
The value of pi is 3.14
The 

          
           
println()
method is used to write a single line of text to the console. The code below shows how this can be used: 

          
           
System.out.println("Hello World!");

Hello World!
