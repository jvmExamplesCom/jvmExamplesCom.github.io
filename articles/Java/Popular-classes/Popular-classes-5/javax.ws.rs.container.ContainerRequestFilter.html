A filter is an object that performs filtering tasks on either the request to a resource (a servlet or static content) or on the response from a resource, or both. Filters do not usually generate a response; they modify or adapt the requests for a resource, or modify or adapt the responses from a resource. The 

          
           
ContainerRequestFilter
class is one such filtered used to filter container requests.

Usage

ContainerRequestFilter
