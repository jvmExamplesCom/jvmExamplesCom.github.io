The 

          
           
org.joda.time.LocalDate
class is a DateTime implementation for representing local dates in the ISO-8601 format, without reference to any time zone. It provides convenience methods for manipulating the date fields, such as add, subtract, and with.

Constructors

There are three main constructors available for creating a LocalDate.


          
           
LocalDate now = new LocalDate(); 
LocalDate today = LocalDate.now(); 
LocalDate specificDate = new LocalDate(2020, 10, 23);

Methods

LocalDate provides a number of methods for manipulating dates.


          
           
// Add three days to the date
LocalDate plusThree = today.plusDays(3);

// Subtract two weeks from the date
LocalDate minusTwo = today.minusWeeks(2);

// Set the date to a specific month

// Get the day of the week
int day = today.getDayOfWeek();
