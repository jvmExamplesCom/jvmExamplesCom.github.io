org.eclipse.swt.events.SelectionEvent

The SelectionEvent class is a subclass of Event and is used as the event type for many SWT widgets that allow user selection. A SelectionEvent is sent when the user selects an item in a widget. This can be due to the user double-clicking an item, selecting the text in a text field, selecting an item in a tree or table, etc. The data field of the SelectionEvent object contains the item that was selected and the x and y coordinates of the mouse click (or -1 if not required).

Examples

The following example shows how to use a 
SelectionListener
to detect when the user selects an item in a text field. 

          
           
textField.addSelectionListener(new SelectionListener() {
       @Override
       public void widgetSelected(SelectionEvent e) {
           System.out.println("Text field selection changed!");
       }

       @Override
       public void widgetDefaultSelected(SelectionEvent e) {
           // Do nothing
       }
});
The following example shows how to use a 
SelectionListener
to detect when the user double-clicks an item in a Tree. 

          
           
tree.addSelectionListener(new SelectionListener(){
  public void widgetSelected(SelectionEvent e){
       // Do nothing
  }
  public void widgetDefaultSelected(SelectionEvent e){
      TreeItem item = (TreeItem) e.item;
      System.out.println("User double-clicked on "+item.getText());
  }
});

The following example shows how to get the item that was selected from the SelectionEvent.


          
           
table.addSelectionListener(new SelectionListener(){
  public void widgetSelected(SelectionEvent e){
       TableItem item = (TableItem) e.item;
       System.out.println("User selected "+item.getText());
  }
