org.apache.http.impl.client.BasicCookieStore

Overview

org.apache.http.impl.client.BasicCookieStore is a class providing an implementation of a cookie store backed by an in-memory collection. It is an important part of the open source Apache HttpComponents project from the Apache Software Foundation. This class provides a threadsafe, mutable object that stores cookies for the execution of multiple requests.

Usage

The BasicCookieStore class is used to store and retrieve cookies. To use, create an instance of the class, and then add cookies to be stored using the addCookie() method. When a request is executed, all available cookies in the CookieStore instance will sent to the remote server. Here is a basic example:


          
           
BasicCookieStore cookieStore = new BasicCookieStore();
cookieStore.addCookie(new BasicClientCookie("cookie key", "cookie value"));
CloseableHttpClient client = HttpClients.custom().setDefaultCookieStore(cookieStore).build();

HttpResponse response = client.execute(new HttpGet("http://example.com/"));
int responseCode = response.getStatusLine().getStatusCode(); // 200 OK

Cookies can also be retrieved from the CookieStore instance:


          
           
List<Cookie> cookies = cookieStore.getCookies();
for (Cookie cookie : cookies) {
    System.out.println(cookie.getName() + ":" + cookie.getValue());
}

The org.apache.http.impl.client.BasicCookieStore implements the org.apache.http.client.CookieStore interface, and provides additional methods such as getCookies() and clearExpired().
