JFreeChart: An Introduction

JFreeChart is a powerful and widely used library for creating a variety of different types of charts and graphs in Java. It provides support for a wide range of chart types, including pie charts, bar graphs, line charts, and many more. This article will introduce the basics of JFreeChart, and provide several code examples to show how it works.

Overview

The JFreeChart library is open source and can be found on GitHub at https://github.com/jfree/jfreechart It is also included as part of the larger Apache Jakarta project. JFreeChart provides a simple, yet intuitive, way of creating charts in Java. It supports a variety of different chart types and provides several methods for customizing and formatting them. It also provides several utility classes for data manipulation and analysis. Using the JFreeChart library is fairly straightforward. All you need to do is include the necessary libraries in your project, create a Chart object, pass in your data, and then render the chart.

Creating a Simple Chart

The most basic type of chart you can create with JFreeChart is a bar chart. Below is an example of how to create a basic bar chart using the JFreeChart library. 

          
           
DefaultCategoryDataset dataset = new DefaultCategoryDataset();
dataset.addValue(15, "Students", "Class A");
dataset.addValue(30, "Students", "Class B");
dataset.addValue(20, "Students", "Class C");

JFreeChart chart = ChartFactory.createBarChart("Number of Students", "Class",  "Number of Students", dataset, PlotOrientation.VERTICAL, false, true, false);

ChartPanel chartPanel = new ChartPanel(chart);
chartPanel.setPreferredSize(new Dimension(400, 400));
In this example, we create a DefaultCategoryDataset object to store our data, then create a JFreeChart object by passing in the data, a title for our chart, and other options. Finally, we create a ChartPanel object to display our chart.

Chart Customization

One of the key features of JFreeChart is its ability to customize charts. With the library, you can change the look and feel of charts by modifying their colors, fonts, and other properties. Below is an example of how to customize the look of a bar chart. 

          
           
chart.getTitle().setFont(new Font("Arial", Font.BOLD, 15));
chart.getCategoryAxis().setLabelFont(new Font("Arial", Font.PLAIN, 12));
chart.getCategoryAxis().setTickLabelFont(new Font("Arial", Font.PLAIN, 10));
chart.getCategoryPlot().getRangeAxis().setLabelFont(new Font("Arial", Font.PLAIN, 12));
In this example, we set the font for the title, category axis labels, and range axis labels. You can also use the setColor() method to set the color of various elements, such as the chart title, axes labels, and plot background.

Conclusion

JFreeChart offers a simple yet powerful way to create charts and graphs in Java. The library provides a wide range of chart types and allows for customization and formatting of all aspects of the chart. With just a few lines of code, you can quickly create and customize beautiful charts in Java.
