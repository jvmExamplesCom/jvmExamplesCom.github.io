android.graphics.Paint.Align

The android.graphics.Paint.Align class is a part of the Android graphics package and is used to specify how text is aligned when drawn on a canvas. It defines the following constants: LEFT, RIGHT, CENTER, TOP, BOTTOM and BASELINE.

Example Usage


          
           
Paint p = new Paint();

// Set text alignment to be LEFT
p.setTextAlign(Paint.Align.LEFT);

// Set text alignment to be CENTER
p.setTextAlign(Paint.Align.CENTER);

Output

The output of this code will be the same regardless of which alignment is set. The only difference is in how the text is aligned when it is drawn onto a canvas.

BASELINE Alignment

The BASELINE alignment is used to align text on its baseline. Baseline alignment is the default alignment for text when drawn on a canvas. This can be seen in the following example: 

          
           
Paint p = new Paint();

// Set text alignment to BASELINE
p.setTextAlign(Paint.Align.BASELINE);

// Draw text on the canvas
canvas.drawText("Hello World", 0, 0, p);
In the example above the text will be aligned exactly on the baseline at position 0,0 on the canvas. It will not be shifted or offset in any way.
