javax.swing.border.LineBorder

The 

          
           
javax.swing.border.LineBorder
class is a Swing class that creates a line border around a container. It has various methods and constructors to create and customize the border.

Constructors

There are four constructors available to be used to construct a LineBorder instance.


          
           
public LineBorder (Color color)
    Constructs a line border with the specified color and a default 1-pixel width.
public LineBorder (Color color, int thickness)
    Constructs a line border with the specified color and thickness.
public LineBorder (Color color, int thickness, boolean roundedCorners)
    Constructs a line border with the specified color, thickness and shape
public LineBorder (ResourceBundle bundle, String prefix)
    Constructs a line border from the specified parameters in the ResourceBundle for the specified prefix

Methods

The 
LineBorder
class also provides a variety of methods that can be used to customize and interact with the border. 

          
           
public Color getLineColor ()
    Returns the color of the line.
public int getThickness ()
    Returns the thickness of the line in pixels.
public boolean isRoundedCorners ()
    Returns whether the corners are round or square.
public void paintBorder (Component c, Graphics g, int x, int y, int width, int height) 
    Paints the border for the specified component with the specified position and size.
public Insets getBorderInsets (Component c)
    Returns the insets of the border.

Example

The following example demonstrates how to use 
LineBorder
class to create and customize a line border. 

          
           
import javax.swing.*;
import javax.swing.border.LineBorder;

public class LineBorderDemo {

    public static void main (String[] args) {

        // Create a frame
        JFrame frame = new JFrame("Line Border Demo");

        // Set the size of the frame
        frame.setSize(400, 300);

        // Create a panel
        JPanel panel = new JPanel();
        
        // Create a line border
        LineBorder lineBorder = new LineBorder(Color.BLUE, 2, true);
        
        // Set the line border to the panel
        panel.setBorder(lineBorder);
        
        // Add the panel to the frame
        frame.add(panel);

        // Set the frame visible
        frame.setVisible(true);
    }
}
