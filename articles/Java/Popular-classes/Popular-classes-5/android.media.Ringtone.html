Android provides Ringtone class to help developers play ringtones, notification tones and alarms in their applications. It is part of the 

          
           
android.media
package. The Ringtone class has various methods which are useful for managing the ringing of the device.

Getting a Ringtone Instance

The first step is to get a ringtone instance which can be done by calling the static 

          
           
getRingtone
method which takes in two parameters: the context object and the ringtone URI which can be obtained from the RingtoneManager class. 

          
           
Ringtone ringtone = RingtoneManager.getRingtone(context, RingtoneManager.getDefaultUri(RingtoneManager.TYPE_RINGTONE));

play()

ringtone.stop();
