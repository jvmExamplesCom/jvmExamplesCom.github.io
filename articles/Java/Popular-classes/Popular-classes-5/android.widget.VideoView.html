Android Widget VideoView

VideoView is an android widget that can be used to play videos. It has a single constructor and several methods to control it.

Constructor


          
           
VideoView (Context context)
The VideoView constructor takes a Context as an argument. This constructor can be used to create a VideoView instance.

Methods


          
           
void start () 
void pause() 
void stopPlayback () 
boolean isPlaying () 
int getDuration () 
int getCurrentPosition () 
void seekTo (int msec) 
boolean canPause () 
boolean canSeekBackward () 
boolean canSeekForward ()
The start method starts playing the video at the current position. The pause method pauses the video playback if it is currently playing. Call stopPlayback to stop the playback and reset the position to 0. The isPlaying method returns true if the video is currently playing. Use the getDuration method to get the video's total duration in milliseconds. The getCurrentPosition method returns the current playback position in milliseconds. The seekTo method changes the current playback position to the specified point in the video. It takes one argument, the millisecond position. Call the canPause method to determine if the video can be paused. Check if the video can be seeked backward/forward by calling canSeekBackward/canSeekForward methods.
