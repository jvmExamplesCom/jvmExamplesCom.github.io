The XML Enum annotation is a utility class used to bind an enum type to a schema element or attribute. This annotation can be used with the @XmlJavaTypeAdapter annotation to map enums values to xml elements.

public enum Color {
    RED("red"),
    BLUE("blue"),
    GREEN("green");
