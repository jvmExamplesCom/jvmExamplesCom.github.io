javax.xml.bind.annotation.XmlEnum

The XML Enum annotation is a utility class used to bind an enum type to a schema element or attribute. This annotation can be used with the @XmlJavaTypeAdapter annotation to map enums values to xml elements.

Example 1 - Defining an Enum with XmlEnum

In this example, we define an enum type called Color and use the XmlEnun annotation to bind each of its values to corresponding XML elements.


          
           
public enum Color {
    RED("red"),
    BLUE("blue"),
    GREEN("green");

    private String value;

    Color(String value) {
        this.value = value;
    }

    @XmlEnum
    public String getValue() {
        return value;
    }
}

            return Color.BLUE;

            return Color.GREEN;

        return null;

    @Override
    public String marshal(Color v) throws Exception {
        if (v == Color.RED) {
            return "Red";
        } else if (v == Color.BLUE) {
            return "Blue";
        } else if (v == Color.GREEN) {
            return "Green";
        }
        return null;
    }
}
