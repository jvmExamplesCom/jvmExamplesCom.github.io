org.apache.commons.lang.builder.HashCodeBuilder

org.apache.commons.lang.builder.HashCodeBuilder is a utility class for constructing hash codes for Java objects. It can be used to generate hash codes for objects, arrays, or collections of objects or arrays. A HashCodeBuilder is created by calling the static method HashCodeBuilder.reflectionHashCode. This method creates a new a HashCodeBuilder that uses the fields of the specified object (and any enclosed objects, such as arrays or collections) to create a unique hash code.


          
           
HashCodeBuilder builder = HashCodeBuilder.reflectionHashCode(myObject);

Generating the Hash Code

The generateHashCode method is used to generate a hash code from the fields that have been added to the HashCodeBuilder. This method returns an int which can then be used in collections operations such as Map.put and containsKey.


          
           
int hashCode = builder.generateHashCode();
