org.apache.commons.lang.builder.HashCodeBuilder

org.apache.commons.lang.builder.HashCodeBuilder is a utility class for constructing hash codes for Java objects. It can be used to generate hash codes for objects, arrays, or collections of objects or arrays. The generated hash code can then be used in collections operations such as Map.put and containsKey to quickly identify an object or collection.

Creating a HashCodeBuilder

A HashCodeBuilder is created by calling the static method HashCodeBuilder.reflectionHashCode. This method creates a new a HashCodeBuilder that uses the fields of the specified object (and any enclosed objects, such as arrays or collections) to create a unique hash code.


          
           
HashCodeBuilder builder = HashCodeBuilder.reflectionHashCode(myObject);

Adding Fields to a HashCodeBuilder

Once a HashCodeBuilder has been created, individual fields can be added to it using the append methods. These methods take a field name and the value of that field as arguments. The type of the value must match the type of the field.


          
           
builder.append("name", myObject.name);
builder.append("age", myObject.age);
builder.append("gender", myObject.gender);

Generating the Hash Code

The generateHashCode method is used to generate a hash code from the fields that have been added to the HashCodeBuilder. This method returns an int which can then be used in collections operations such as Map.put and containsKey.


          
           
int hashCode = builder.generateHashCode();

Generating a Hash Code for a Collection

To generate a hash code for a collection of objects, the same procedure is followed but with the appendCollection method instead of the append methods.
