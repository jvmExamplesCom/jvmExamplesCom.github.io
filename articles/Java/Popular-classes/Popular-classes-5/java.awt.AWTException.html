Java.awt.AWTException

The 

          
           
java.awt.AWTException
is a checked exception that signals that an application or applet cannot invoke an AWT functional feature. It must be caught or declared to be thrown. This exception can occur when an application fails to properly initialize the AWT library or when an application tries to create an AWT component on a system it is not supported.

Examples


          
           
try {
    Component c = new Component();
} catch (AWTException e) {
    System.out.println("Failed to initialize AWT library");
}
In the example above, an AWTException is thrown if the AWT library fails to initialize.
