javax.xml.bind.annotation.XmlElementWrapper

The javax.xml.bind.annation.XmlElementWrapper annotation is used to create an XML element wrapper around its child elements in a Java bean or class.

Overview

This annotation can be used on collection properties to generate the wrapping elements. It should be used in conjunction with @XmlElement annotation. The @XmlElement annotation should be applied for each item in the collection.

Example

Let's consider we have a class named Course which has a list of students as property.


          
           
public class Course {
    private List<Student> students;

    public List<Student> getStudents() {
        return students;
    }

    public void setStudents(List<Student> students) {
        this.students = students;
    }
}

In order to generate the XML element wrapper for collection property, we need to annotate the Course class with two annotations, @XmlElementWrapper and @XmlElement.


          
           
@XmlRootElement
public class Course {
    @XmlElementWrapper(name = "students")
    @XmlElement(name = "student")
    private List<Student> students;

    public List<Student> getStudents() {
        return students;
    }

    public void setStudents(List<Student> students) {
        this.students = students;
    }
}

Now when the above class is marshalled to XML, it will generate the following output.


          
           
<course>
    <students>
        <student>...</student>
        <student>...</student>
    </students>
</course>

Attributes

The @XmlElementWrapper annotation has a few attributes - name, namespace, nillable and required.

name

This attribute is used to specify the name of the wrapping element. If the name is not specified, then the property name will be used.

namespace

This attribute is used to specify the namespace of the wrapping element.

nillable

This attribute is used to indicate whether the element can be nil or not. Its value must be either true or false. Its default value is false.

required

This attribute is used to indicate whether the element is mandatory or not. Its value must be either true or false. Its default value is false.

Conclusion

The javax.xml.bind.annotation.XmlElementWrapper annotation is used to create an XML element wrapper around its child elements in a Java bean or class. It should be used in conjunction with @XmlElement annotation. The @XmlElement annotation should be applied for each item in the collection. The annotation has several attributes which can be used to customize the behavior.
