javafx.util.Duration

JavaFX.util.duration is a utility class used to represent a duration of time, such as years, months, days, hours, minutes, seconds and nanoseconds. It provides methods for manipulating durations, for example adding or subtracting durations from one another or multiplying them. This class is also useful for dealing with temporal calculations.

Creating a Duration

A Duration object can be constructed from an integer value in milliseconds (or nanoseconds) or from two Calendar objects that represent the start and end times. 

          
           
// Create a Duration from an int in milliseconds
Duration duration = new Duration(1000);

// Create a Duration from two Calendar objects
Calendar startTime = Calendar.getInstance();
Calendar endTime = Calendar.getInstance();
endTime.add(Calendar.MINUTE, 5);
Duration duration = new Duration(startTime, endTime);

Manipulating Durations

It is possible to manipulate a Duration by adding, subtracting, and multiplying other Durations. 

          
           
// Add two Durations
Duration duration1 = new Duration(1000);
Duration duration2 = new Duration(2000);
Duration result = duration1.add(duration2);

// Subtract two Durations
Duration result = duration1.subtract(duration2);

// Multiply a Duration by a number
Duration result = duration1.multiply(2);

Retrieving Duration Information

The Duration class contains many methods for retrieving information about the duration, such as the length or the number of days, hours, minutes, and seconds. 

          
           
// Retrieve the length of the Duration in milliseconds
long lengthInMillis = duration.toMillis();

// Retrieve the number of days
int days = duration.toDays();

// Retrieve the number of hours
int hours = duration.toHours();

// Retrieve the number of minutes
int minutes = duration.toMinutes();

// Retrieve the number of seconds
int seconds = duration.toSeconds();

Formatting Durations

It is also possible to format durations using the java.text.SimpleDateFormat class. 

          
           
// Format a Duration using SimpleDateFormat
SimpleDateFormat sdf = new SimpleDateFormat("HH:mm:ss");

JavaFX.util.duration is a useful utility class for dealing with temporal calculations. It provides methods for creating and manipulating durations as well as retrieving information about them and formatting them.
