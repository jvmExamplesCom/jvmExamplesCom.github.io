Using Java Lombok -

The lombok.experimental.Accessors annotation facilitates the creation of automatic getters and setters in your code. It is an Open-Source library created to reduce boilerplate code in Java applications.

Features

This annotation supports a range of features, allowing you to tailor the creation of getters and setters to your needs:
  • Boolean field getter prefixes: use 
    is
    , 
    get
    or nothing.
  • Disable auto getters or setters entirely.
  • Prefix with custom string.
  • Exclude particular fields.
  • Define custom access levels.
  • Supports fluent builders.

Example of Usage

Below is an example of how you would use this annotation: 

          
           
@Accessors(chain = true)
public class Employee {
    private String name;
    private int age;
    
    // getters and setters for "name" and "age" fields
}
 
Employee employee = new Employee().setName("James").setAge(21);

System.out.println(employee.getName()); // Output: James
System.out.println(employee.getAge()); // Output: 21
As you can see, this annotation enables you to easily create getters and setters for the fields of a class. The 

          
           
chain
parameter allows for the creation of fluent builders based off these getters and setters.

Conclusion

In conclusion, the 

          
           
@Accessors
annotation simplifies the process of creating getters and setters for your classes. It saves time and effort and can be used to create fluent builders.
