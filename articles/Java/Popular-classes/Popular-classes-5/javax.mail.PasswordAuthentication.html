public PasswordAuthentication(String username, char[] password)

The first constructor takes a string containing the username and a string containing the password. The second constructor takes a string containing the username and a character array containing the password.

Using a PasswordAuthentication Object

Once a PasswordAuthentication object has been created, it can be used to provide the necessary authentication information when requesting an email service.
