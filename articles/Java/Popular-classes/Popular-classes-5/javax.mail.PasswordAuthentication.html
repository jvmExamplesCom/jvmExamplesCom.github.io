javax.mail.PasswordAuthentication

The javax.mail.PasswordAuthentication class is used to request authentication information such as a username and password from a user or from an authentication database. It can be used to provide authentication information for an incoming or outgoing message to an email server.

Creating a PasswordAuthentication Object

The PasswordAuthentication class has two constructors that can be used to create an authentication object:


          
           
public PasswordAuthentication(String username, String password)
public PasswordAuthentication(String username, char[] password)

The first constructor takes a string containing the username and a string containing the password. The second constructor takes a string containing the username and a character array containing the password.

Using a PasswordAuthentication Object

Once a PasswordAuthentication object has been created, it can be used to provide the necessary authentication information when requesting an email service. For example, the following code shows how to use the PasswordAuthentication object to connect to a mail server using SMTP:


          
           
String username = "myusername";
char[] password = {'m','y','p','a','s','s','w','o','r','d'};
PasswordAuthentication authentication = new PasswordAuthentication(username, password);
Properties properties = new Properties();

properties.put("mail.smtp.auth", true);
properties.put("mail.smtp.starttls.enable", true);
Session session = Session.getInstance(properties, authentication);

In this example, the PasswordAuthentication object is used to provide the username and password to the SMTP protocol. The username and password are then used to authenticate the connection to the mail server.
