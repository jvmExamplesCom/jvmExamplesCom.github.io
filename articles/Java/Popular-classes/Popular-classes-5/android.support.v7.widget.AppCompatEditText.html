android.support.v7.widget.AppCompatEditText

The android.support.v7.widget.AppCompatEditText library provides an easy and convenient way to create an editable text view within an Android application. This library contains a set of classes that help developers create and manage editable text boxes in their Android applications. To create an AppCompatEditText, simply create a new instance of the AppCompatEditText class with the context parameter: 

          
           
AppCompatEditText editText = new AppCompatEditText(context);

AttributeSet attrs = new AttributeSet();
AppCompatEditText editText = new AppCompatEditText(context, attrs);

Editing Text

editText.setText(R.string.hello_world);

editText.setTextSize(16);

editText.setTextColor("#FF0000");
