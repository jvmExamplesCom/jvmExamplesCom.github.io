android.widget.TableLayout

The android.widget.TableLayout is a subclass of ViewGroup that lays out its children in a table. It’s a powerful and flexible layout manager that lets you easily arrange UI elements in a tabular form with multiple rows and columns. Its primary use is to create an efficient way to display data that might otherwise take up too much space on the screen.

Creating a TableLayout

To create a TableLayout, you must first create an instance of it in your Activity: 

          
           
TableLayout tableLayout = new TableLayout(this);
This will create a new TableLayout ready to be populated with child views. You can customize the TableLayout by setting attributes such as padding, background color, and text color. You can also set the number of rows and columns. The following code snippet shows how this can be done: 

          
           
tableLayout.setPadding(10, 10, 10, 10);
tableLayout.setBackgroundColor(Color.BLUE);
tableLayout.setColumnCount(2);
tableLayout.setRowCount(3);

Adding Views to a TableLayout

Once the TableLayout has been created, you can add views to it. There are two ways to do this: 1. By manually positioning the views in the TableLayout by setting their row and column indices. 2. By using TableRow objects to group related views together. The following code snippet shows how views can be added to the TableLayout using both methods: 

          
           
// creating new views
TextView textView1 = new TextView(this);
textView1.setText("Test");

TextView textView2 = new TextView(this);
textView2.setText("Test2");

// manually positioning views
TableLayout.LayoutParams textView1Params =  new TableLayout.LayoutParams(0, TableLayout.LayoutParams.WRAP_CONTENT, 1f);
textView1Params.column = 0;
textView1Params.row = 0;

TableLayout.LayoutParams textView2Params =  new TableLayout.LayoutParams(0, TableLayout.LayoutParams.WRAP_CONTENT, 1f);
textView2Params.column = 1;
textView2Params.row = 0;

// adding views to the TableLayout
tableLayout.addView(textView1, textView1Params);
tableLayout.addView(textView2, textView2Params);

// using TableRow objects
TableRow tableRow = new TableRow(this);
tableRow.addView(textView1); 
tableRow.addView(textView2);

tableLayout.addView(tableRow);
In the above code snippet, two text views are created and then positioned in the TableLayout. The first view is positioned by setting its row and column indices in a LayoutParams object and the second view is grouped with the first using a TableRow object.

Conclusion

The android.widget.TableLayout is a powerful layout manager that allows developers to efficiently display data. It enables them to arrange child views into a tabular form with multiple rows and columns. Views can be added to the TableLayout either manually or by using TableRow objects.
