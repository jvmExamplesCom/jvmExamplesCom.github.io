org.eclipse.swt.widgets.Shell

The org.eclipse.swt.widgets.Shell class provides a desktop window in the SWT application framework. It is used to create top-level windows and dialog boxes. Shells are used to host components such as text fields, list boxes, buttons, and labels.

Creating a New Shell in Java

The following code example shows how to create a new Shell instance in a Java application:


          
           
Display display = new Display();
Shell shell = new Shell(display);
shell.open();
while (!shell.isDisposed()) {
  if (!display.readAndDispatch()) display.sleep();
}
display.dispose();

shell.setText("My SWT Application");

open()

The open() method opens the Shell. This causes the Shell to become visible. The following code example shows how to use this method:


          
           
shell.open();

isDisposed()

The isDisposed() method checks if the Shell has been disposed of. This method is mainly used in conjunction with the readAndDispatch() and sleep() methods, to check whether the Shell has been closed while the application is running:
