Java.awt.image.ColorModel

The Java.awt.image.ColorModel class is a part of the Java Abstract Window Toolkit (AWT) and it provides an abstract class for representing color models used by Image objects in AWT. It defines methods for converting between various color models and for getting various properties of some color models, such as the number of bits used to represent each color component.

Constructors

The ColorModel class has two constructors: 

          
           
public ColorModel(int bits)
public ColorModel(int bits, int[] components)

The first constructor creates a ColorModel with the specified number of bits per pixel used to represent colors. The second constructor creates a ColorModel based on an array of components which is an array specifying the various components used in each pixel. For example, an RGB color model would contain three components – red, green, and blue.

Methods

The ColorModel class provides several methods for converting between color models and for getting various properties of some color models.

  • 
            
             
public static ColorModel getRGBdefault()
    – returns the default RGB color model.
  • 
            
             
public int getPixelSize()
    – returns the number of bits used to represent each color component.
  • 
            
             
public Color convertToComponent(int[] pix, int[] comps)
    – converts the given pixel, represented as an array of integers, to a Color object using the color model components specified in comps.
  • 
            
             
public Object getDataElements(int argb, Object obj)
    – returns an object representing a pixel representation of the specified ARGB color.
  • 
            
             
public static ColorModel getModel(String name)
    – gets the color model of the specified name.
  • 
            
             
public Object getDataElements(int[] pix, Object obj)
    – returns an object representing a pixel representation of the specified pixel colors, represented as an array of integers.

Example

The following example shows how to create a ColorModel, convert a pixel to a Color object, and then get a data element representing that color:


          
           
// Create a ColorModel with 8 bits per pixel.
ColorModel cm = new ColorModel(8);

// Create an array of integers to represent a pixel.
int[] pix = { 0xFF0000, 0x00FF00, 0x0000FF };

// Convert the pixel to a Color object using the ColorModel's components.
Color c = cm.convertToComponent(pix, cm.getRGBdefault());

// Get a data element representing the Color object.
Object dataElement = cm.getDataElements(c, null);

The above code will result in the data element representing the Color object being stored in the dataElement variable.
