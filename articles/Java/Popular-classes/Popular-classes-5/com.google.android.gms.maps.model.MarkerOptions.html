com.google.android.gms.maps.model.MarkerOptions

The 

          
           
MarkerOptions
class of the Google Maps Android API allows developers to define custom markers on a map. It provides various properties and methods to customize markers and allow to add, remove, or animate them.

Setting Marker Properties

The following example shows how to create a marker with a set of properties: 

          
           
MarkerOptions markerOptions = new MarkerOptions();
markerOptions.position(latLng);
markerOptions.title("My Marker");
markerOptions.snippet("This is my marker.");
markerOptions.icon(BitmapDescriptorFactory.defaultMarker(BitmapDescriptorFactory.HUE_MAGENTA));
markerOptions.draggable(true);
markerOptions.visible(true);
The code snippet above creates a new marker object ( 

          
           
markerOptions
) with the following properties:
  • Position: Defines the geographical coordinates of the marker.
  • Title: Sets a text label for the marker.
  • Snippet: Adds a sub-text label for the marker.
  • Icon: Sets a custom icon for the marker.
  • Draggable: Enables or disables the ability to drag the marker.
  • Visible: Shows or hides the marker.

Adding Markers to the Map

We can then add the marker to the map: 

          
           
mMap.addMarker(markerOptions);
In order to add multiple markers to a map, we can use a loop to iterate over a list of markers and add each one to the map: 

          
           
for (MarkerOptions marker : markers) {
    mMap.addMarker(marker);
}

Removing Markers from the Map

remove()
method is used to remove a marker from the map. This method takes a 

          
           
Marker
object as a parameter, which is the marker that needs to be removed from the map. 

          
           
marker.remove();
In order to remove all markers from the map at once, we can iterate over a list of markers and remove them from the map: 

          
           
for (Marker marker : markers) {
    marker.remove();
