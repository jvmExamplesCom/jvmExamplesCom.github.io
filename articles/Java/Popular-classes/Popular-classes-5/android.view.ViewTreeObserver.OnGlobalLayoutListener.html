Overview of android.view.ViewTreeObserver.OnGlobalLayoutListener

The 

          
           
android.view.ViewTreeObserver.OnGlobalLayoutListener
is an interface used within Android to register a callback to be invoked when the global layout state or the visibility of views within the view tree changes. Whenever the global layout state changes, 

          
           
onGlobalLayout()
is called on the ViewTreeObserver's registered OnGlobalLayoutListener objects. This is useful for objects that need to be aware of layout events happening around them, such as when layouts are drawn asynchronously and need further processing beyond the initial layouting.

Code Examples


          
           
ViewTreeObserver viewTreeObserver = 
myView.getViewTreeObserver();
viewTreeObserver.addOnGlobalLayoutListener(
    new ViewTreeObserver.OnGlobalLayoutListener() {
        @Override
        public void onGlobalLayout() {
            // Do Something Here
        }
    });
In the code example above, we are getting the ViewTreeObserver which belongs to our View object (in this case, myView) and registering it as an OnGlobalLayoutListener. The onGlobalLayout method will be triggered whenever the layout of the views in the view tree change. 

          
           
int width = myView.getWidth();
int height = myView.getHeight();
