What is HikariConfig & What Does it Do?

HikariConfig is a Java-based configuration class for the popular Hikari Connection Pool. It enables users to set up their connection pool, specify the necessary parameters and customize the behaviour of their applications accordingly. The HikariConfig API provides support for a variety of features such as automatic connection testing and leak detection. It also supports advanced features like Statement caching and load balancers.

By using HikariConfig, users can customize the settings and parameters of their connection pool without having to write any code. This makes it very easy to set up, modify and debug their applications without needing extensive knowledge of the underlying code.
