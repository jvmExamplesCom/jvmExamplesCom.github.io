java.nio.file.InvalidPathException

java.nio.file.InvalidPathException is a checked exception that occurs when an operation of the java.io.File class is invoked and the given path string is invalid. This exception is typically thrown when attempting to create or delete a file or directory, or other IO operations.

The java.io.File class provides static methods for filesystem operations, including creating and deleting files and directories. InvalidPathException is thrown when one of those methods is given an invalid path string as an argument.
