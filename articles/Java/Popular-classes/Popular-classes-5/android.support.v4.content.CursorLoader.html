android.support.v4.content.CursorLoader

Android provides support for applications to manage Cursor objects. The android.support.v4.content.CursorLoader class provides an easy way to query the content provider, and return the results as a Cursor object.

Using CursorLoader

To use CursorLoader, you need to provide two pieces of information: the URI of the content provider which contains the data, and the projection containing a list of columns that should be returned. 

          
           
Uri uri = Contacts.CONTENT_URI;
String[] projection = { Contacts.DISPLAY_NAME };
You can then create a CursorLoader object with this information: 

          
           
CursorLoader cursorLoader = new CursorLoader(context, uri, projection, null, null, null);
The last three parameters (null in the example) are optional selection criteria for the query. These parameters can be used to filter the data which is returned, using the standard SQL syntax. Once the CursorLoader is created, you can call the loadInBackground() method to execute the query. This method will return a Cursor object containing the results: 

          
           
Cursor cursor = cursorLoader.loadInBackground();

Example

The following example shows how to use a CursorLoader to query the contacts content provider and get the contact name for each contact: 

          
           
Uri uri = Contacts.CONTENT_URI;
String[] projection = { Contacts.DISPLAY_NAME };

CursorLoader cursorLoader = new CursorLoader(context, uri, projection, null, null, null);
Cursor cursor = cursorLoader.loadInBackground();

int nameColumnIndex = cursor.getColumnIndex(Contacts.DISPLAY_NAME);
while (cursor.moveToNext()) {
    String contactName = cursor.getString(nameColumnIndex);
    Log.d(TAG, "Contact name: " + contactName);
}
The example above will output a line of logging for each contact, displaying the contact's name. Note that this example doesn't perform error checking - so make sure to handle any errors which may be thrown by the CursorLoader.

Conclusion

The android.support.v4.content.CursorLoader class makes it easy to perform queries against the content provider and return the results in a Cursor object. Once the Cursor is created, you can use it to iterate through the results.
