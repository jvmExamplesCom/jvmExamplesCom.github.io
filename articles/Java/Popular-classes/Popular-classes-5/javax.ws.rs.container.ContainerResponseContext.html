javax.ws.rs.container.ContainerResponseContext

javax.ws.rs.container.ContainerResponseContext is a class used within Java Enterprise Edition RESTful web services. It provides methods to customize the response body and headers as part of an outbound request. It is often used with the javax.ws.rs.container.ContainerRequestContext class, which is used for customizing inbound requests.

@Path("/example")
  public class ExampleResource {

    @GET
    public Response getExampleResponse(@Context ContainerRequestContext reqCtx,
                                      @Context ContainerResponseContext respCtx) {
        //Customize response body
