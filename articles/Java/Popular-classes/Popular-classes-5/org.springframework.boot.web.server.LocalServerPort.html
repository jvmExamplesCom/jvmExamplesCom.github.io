org.springframework.boot.web.server.LocalServerPort

LocalServerPort is an annotation from the Spring Boot library for indicating the port to use for a web application running in a web server. It is used to automatically detect and configure the port of the local web server, allowing it to be more easily configured and used in development and testing environments.

Getting Started

To start using LocalServerPort, you will need to add it to your project's dependencies. The following example shows how to add it to a Gradle project:


          
           
dependencies {
  compile 'org.springframework.boot:spring-boot-starter-web'
}

Once added, you can use the LocalServerPort annotation to specify the port to use for the web application.

Using LocalServerPort

To use LocalServerPort, you will need to add the annotation to a methods parameter. For example:


          
           
@RequestMapping ("/test")
public String test(@LocalServerPort int port){
    // do something with the port
    return "test";
}

This code will cause the web server to be started on the given port. You can also use it to do other tasks, such as configuring a server or session timeout.

Example

@Configuration
@EnableWebMvc
