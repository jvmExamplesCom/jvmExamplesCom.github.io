android.widget.SeekBar.OnSeekBarChangeListener

The android.widget.SeekBar.OnSeekBarChangeListener is an interface used to monitor changes in a SeekBar instance. This listener works on the basis of callbacks and it implements the SeekBar.OnSeekBarChangeListener interface.

Methods

The OnSeekBarChangeListener interface contains three different methods: 

          
           
public abstract void onProgressChanged(SeekBar seekBar, int progress, boolean fromUser)
public abstract void onStartTrackingTouch(SeekBar seekBar)
public abstract void onStopTrackingTouch(SeekBar seekBar)
The 

          
           
onProgressChanged()
method is triggered when the progress value is changed either by user or by programmatically. It takes three arguments: • SeekBar - the SeekBar whose progress has changed • Progress - the current progress level • FromUser - boolean indicating if the progress update was initiated by the user The 

          
           
onStartTrackingTouch()
method is called many times as the progress position changes. This callback is useful to know when the user starts dragging the progress position. The 

          
           
onStopTrackingTouch()
method is also called multiple times as the progress position changes. This callback is useful to know when the user stops dragging the progress position.

Usage Example

In this example, we will create a SeekBar which will have maximum of 10. User will be able to select the values between 0-10. 

          
           
// Create an instance of SeekBar
 SeekBar seekBar = new SeekBar(context);
 
// Set the maximum Progress
 seekBar.setMax(10);

// Set an OnSeekBarChangeListener
seekBar.setOnSeekBarChangeListener(new SeekBar.OnSeekBarChangeListener() {
    @Override
    public void onProgressChanged(SeekBar seekBar, int progress, boolean fromUser) {
        Log.d(TAG, "Progress:" + progress);
    }

    @Override
    public void onStartTrackingTouch(SeekBar seekBar) {
        Log.d(TAG,"Start Tracking");
    }

    @Override
    public void onStopTrackingTouch(SeekBar seekBar) {
        Log.d(TAG,"Stop Tracking");
    }
});
In the above code snippet, we have first created an instance of SeekBar, then set its maximum progress to 10 and finally attached a listener to monitor the changes in progress. We can also use the interface to update the progress programmatically. To do so, we can simply call the 

          
           
setProgress()
method of SeekBar class passing the progress value as an argument.

method.
