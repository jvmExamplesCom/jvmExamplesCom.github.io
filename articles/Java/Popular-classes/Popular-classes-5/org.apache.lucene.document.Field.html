Introduction to org.apache.lucene.document.Field

Fields in Lucene documents are typically indexed and store values that can be used later to perform searches in the index. In addition to being indexed, fields can also be stored in their original form which can be useful for displaying values from fields in search results. Fields can also be sorted by the values they contain.
