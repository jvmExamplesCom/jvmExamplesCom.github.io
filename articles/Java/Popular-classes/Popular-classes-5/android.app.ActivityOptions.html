ActivityOptions is a class in the Android framework that provides options for Activity operations. It allows you to launch activities with custom parameters and override settings, as well as set up how activities are launched. It's used in situations where you would want to customize the start or launch of an activity, like setting the flag that determines whether it is launched on the current task or a new one. You can also use ActivityOptions to control how animations, shared elements, and launching mode are handled when starting activities.

Creating an ActivityOptions Instance

ActivityOptions opts = ActivityOptions.makeBasic();

bundle.putString("some-param", "value");
opts.setLaunchBundle(bundle);
This will create an instance of ActivityOptions with the launch bundle set to the provided parameter. You can also pass in other parameters such as the flags and animation values.

Launching Activities with ActivityOptions

Intent intent = new Intent(this, MyActivity.class);
startActivity(intent, opts);
This will launch the activity with the options and parameters that you provided.

ActivityOptions Flags

ActivityOptions supports various flags that can be set to control how the activity is launched. These include:

FLAG_ACTIVITY_CLEAR_TOP: Clears the activities on the current task before launching the new activity.

FLAG_ACTIVITY_NO_ANIMATION: Prevents any animations from occurring when launching the activity.
FLAG_ACTIVITY_EXCLUDE_FROM_RECENTS: Excludes the activity from the recent tasks list.
