Introduction to Java com.codahale.metrics.Meter

The 

          
           
com.codahale.metrics.Meter
class is a powerful tool for tracking the rate at which a set of operations occur. It is useful for monitoring the rate of requests within a system, or the rate of data being read from a file. The following code examples will provide a basic introduction to using the Meter class and its associated features.

Creating a Meter Instance

Creating an instance of the 

          
           
Meter

Meter meter = new Meter();

Meter
with no initial state.

Meter

Timer

start()
method of the 

          
           
Meter
: 

          
           
Timer timer = new Timer();
meter.start(timer);
Once the 

          
           
Meter
has been started, it is ready for use.

Marking Events

The 

          
           
Meter
class allows events to be marked. This is done by calling the 

          
           
mark()
method: 

          
           
meter.mark();
Each call to 

          
           
mark()

Meter

Stopping the Meter

When the 

          
           
Meter
is no longer needed, it can be stopped by calling the 

          
           
stop()
method: 

          
           
meter.stop();
This will stop the internal timer, and reset the internal counter of the 

          
           
Meter

Getting Event Rate Information

Once the 

          
           
Meter

getXXRate()

Meter
. These methods include:

  • getFiveMinuteRate()

Meter
class. Note that this example only covers the basic usage, and does not illustrate all possible features. 

          
           
Meter meter = new Meter();
Timer timer = new Timer();
