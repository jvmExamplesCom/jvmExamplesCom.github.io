Introduction to Java com.codahale.metrics.Meter

com.codahale.metrics.Meter
class is a powerful tool for tracking the rate at which a set of operations occur. It is useful for monitoring the rate of requests within a system, or the rate of data being read from a file. :

Once the 

          
           
Meter

Each call to

Stopping the Meter

When the 

          
           
Meter

meter.stop();
This will stop the internal timer, and reset the internal counter of the

. These methods include:

Meter
