Introduction to Java com.codahale.metrics.Meter

The 

          
           
com.codahale.metrics.Meter
class is a powerful tool for tracking the rate at which a set of operations occur. It is useful for monitoring the rate of requests within a system, or the rate of data being read from a file. The following code examples will provide a basic introduction to using the Meter class and its associated features.

Creating a Meter Instance

Creating an instance of the 

          
           
Meter
class is simple: 

          
           
Meter meter = new Meter();
This creates a 

          
           
Meter
with no initial state.

Starting the Meter

Before the 

          
           
Meter
can be used, it must be started. This is done by first creating a 

          
           
Timer
object, and then passing it to the 

          
           
start()
method of the 

          
           
Meter
: 

          
           
Timer timer = new Timer();
meter.start(timer);
Once the 

          
           
Meter
has been started, it is ready for use.

Marking Events

The 

          
           
Meter
class allows events to be marked. This is done by calling the 

          
           
mark()
method: 

          
           
meter.mark();
Each call to 

          
           
mark()
increases the internal counter of the 

          
           
Meter
instance, allowing the rate of events to be calculated.

Stopping the Meter

When the 

          
           
Meter
is no longer needed, it can be stopped by calling the 

          
           
stop()
method: 

          
           
meter.stop();
This will stop the internal timer, and reset the internal counter of the 

          
           
Meter
to zero.

Getting Event Rate Information

Once the 

          
           
Meter
has been started and events have been marked, information about the rates of those events can be retrieved by calling one of the 

          
           
getXXRate()
methods on the 

          
           
Meter
. These methods include: 
  • getOneMinuteRate()
  • getFiveMinuteRate()
  • 
            
             
getFifteenMinuteRate()
These methods return the rate of events per second.

Example Usage

The following is an example of how to use the 

          
           
Meter
class. Note that this example only covers the basic usage, and does not illustrate all possible features. 

          
           
Meter meter = new Meter();
Timer timer = new Timer();
meter.start(timer);

// mark some events
for (int i = 0; i < 10; i++) {
    meter.mark();
}

// get the rate of events

Meter
