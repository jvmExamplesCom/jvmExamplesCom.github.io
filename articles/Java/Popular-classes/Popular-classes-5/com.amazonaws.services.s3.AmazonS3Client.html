com.amazonaws.services.s3.AmazonS3Client

The com.amazonaws.services.s3.AmazonS3Client is an object-oriented Java language interface for working with Amazon S3 resources. This library provides developers with a powerful, convenient and secure way to access and manage their Amazon S3 bucket resources.

Creating an AmazonS3 Client Instance

To start working with the AmazonS3Client library in Java, you first need to create an AmazonS3Client instance with your access key and secret key as parameters.


          
           
AmazonS3Client client = 
    new AmazonS3Client(accessKey, secretKey);

Once the client is created, you can start interacting with Amazon S3 resources.

Getting a List of Buckets

You can use the AmazonS3Client instance to get a list of buckets. The following code snippet shows how to do this:


          
           
List<Bucket> buckets = client.listBuckets();

This will return a list of buckets associated with the S3 account.

Creating a Bucket

You can also use the AmazonS3Client instance to create a new bucket. The following code snippet shows how to do this:


          
           
client.createBucket("my-new-bucket");

Deleting a Bucket
