java.security.spec.AlgorithmParameterSpec

Java.security.spec.AlgorithmParameterSpec is an interface in the Java Security API which provides a specification for parameters used with a specific algorithm. It is used to provide the necessary information to generate an AlgorithmParameters object, which is a repository for a set of algorithm-specific parameter values. This interface is used for symmetric and asymmetric algorithms as well as for key-pair generators.

AlgorithmParameterSpec spec = new RC2ParameterSpec(256, iv);
SecretKey key = KeyGenerator.getInstance("RC2").generateKey();
AlgorithmParameters params = AlgorithmParameters.getInstance("RC2");
params.init(spec);
Cipher cipher = Cipher.getInstance("RC2/CBC/PKCS5Padding");
cipher.init(Cipher.ENCRYPT_MODE, key, params);
