What is an ObjectId?

An ObjectId is a 12-byte identifier that can be used to uniquely identify a document stored in a MongoDB database. Together, these parts form a unique identifier that can be used to quickly and reliably distinguish one document from another. Some common uses for ObjectIds include versioning document states, tracking changes to documents over time, and providing a unique key for each document in a collection.
