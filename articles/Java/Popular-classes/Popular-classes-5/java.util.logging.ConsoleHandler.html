ConsoleHandler handler = new ConsoleHandler();

handler.setLevel(Level.FINEST);

Finally, you need to register the handler with a logger object:


          
           
Logger logger = Logger.getLogger("myLogger");
logger.addHandler(handler);

Once the handler has been configured, you can start logging messages by calling the appropriate methods in the logger object (e.g. log() , info() , warn() , etc.).
