org.eclipse.swt.widgets.Label

The org.eclipse.swt.widgets.Label class is a Swing widget used to provide a simple label for widgets in the user interface. Labels are typically used to provide display text, descriptions and informational messages to the user.

Creating a Label

To create a Label, you must first have an instance of a 

Display
object, which is the main window connection from the application to the user's operating system. Then use the syntax: 

          
           
Label label = new Label (Display, options);
Where 
options
can be one or more of the following constants from the SWT class: 
  • SWT.BORDER
    - displays a border around the label
  • SWT. CENTER
    - centers the label on the display
  • SWT.WRAP
    - tells the label to wrap its text

Setting the Text of a Label

Once the Label has been created, the text that it displays is set using the method 

setText()
. This takes a String argument to set the text displayed by the Label. 

          
           
label.setText("This is the text of the label");

Getting the Text of a Label

To retrieve the text of a Label, the 
getText()
method is called. This returns a String representing the text of the label. 

          
           
String labelText = label.getText();

Aligning the Text of a Label

The text of a label can be aligned using the 
setAlignment()
method. This takes one of the following constants as an argument: 
  • SWT.LEFT
  • SWT.CENTER
  • SWT.RIGHT

          
           
label.setAlignment(SWT.CENTER);

Changing the Foreground Color of a Label

The foreground color of a Label can be changed using the 

setForeground()
method. This method takes a Color object as an argument. To create a Color object use the syntax: 

          
           
Color color = new Color(Display, red, green, blue);
Where 
red
, 
green
and 
blue
are integers between 0 and 255. Then call the 
setForeground()
method: 

          
           
label.setForeground(color);
