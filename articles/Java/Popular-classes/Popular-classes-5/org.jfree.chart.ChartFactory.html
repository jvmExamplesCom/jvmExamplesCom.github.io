Using org.jfree.chart.ChartFactory to Create a Chart in Java

org.jfree.chart.ChartFactory is a class in the JFreeChart library which enables developers to easily create charts in Java applications. Using ChartFactory simplifies the creation of many common chart types and also allows developers to customize their charts with advanced features. In this tutorial, we will explore how to use ChartFactory to create and customize a chart.

Creating a Simple Chart

We will start by creating a simple line chart with some sample data. To create the chart, we first need to define an array containing the data points for the line:


          
           
double[] data = {25, 20, 15, 10, 5};

Next, we can use ChartFactory to create our chart:


          
           
JFreeChart chart = ChartFactory.createLineChart("Sample Chart", 
                                                "Sample X Axis", 
                                                "Sample Y Axis", 
                                                data);

Our chart is now created, but it has no styling or advanced features applied. To make it more visually appealing, we can use a predefined chart theme:


          
           
StandardChartTheme chartTheme = new StandardChartTheme("name");
chartTheme.apply(chart);
Finally, we can display our chart on the screen: 

          
           
ChartPanel chartPanel = new ChartPanel(chart);
frame.setContentPane(chartPanel);
frame.pack();
frame.setVisible(true);

Customizing a Chart

Now that we have created a simple chart, let's look at how we can customize it with more advanced features. We can start by adding a legend to the chart:


          
           
LegendTitle legend = chart.getLegend("legend");
chart.addLegend(legend);

We can also customize the colors used in the chart:


          
           
plot.setDrawingSupplier(new DefaultDrawingSupplier(Color.red,

Finally, we can add annotations to the chart:
