java.util.function.BooleanSupplier

Java.util.function.BooleanSupplier is an interface in the Java programming language. It is a functional interface, meaning it has a single abstract method called getAsBoolean(). It is intended to be used in lambda expressions and method references. The BooleanSupplier interface is used to perform some task that may return a boolean-valued result.

Example 1:

The following example illustrates the usage of BooleanSupplier interface.


          
           
BooleanSupplier bs = () -> true;  
System.out.println(bs.getAsBoolean());
Output: 
true

Here is an example of using BooleanSupplier where you are passing a condition to check if something is true or not. 


          
           
int x = 2;
BooleanSupplier bs = () -> x > 0;
System.out.println(bs.getAsBoolean());
Output: 
true
