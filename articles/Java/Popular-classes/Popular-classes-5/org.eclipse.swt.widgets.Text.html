org.eclipse.swt.widgets.Text

The org.eclipse.swt.widgets.Text class is part of the Standard Widget Toolkit (SWT) and provides a mechanism for displaying and editing text on the screen. It can be used to create user input fields, text fields for displaying results, and other text-based widgets.

Creating a Text Field

The following code snippet shows how to create a new Text field: 

          
           
Text textField = new Text(shell, SWT.BORDER);
The shell parameter is the parent composite object that will contain the Text field. The second parameter is a style flag that specifies how the Text field should be displayed.

Setting Text Field Width and Height

The size of the Text field can be changed with the setSize() method. The width and height are specified in pixels. For example, the following code sets the size of the Text field to 50 pixels in width and 20 pixels in height: 

          
           
textField.setSize(50, 20);

Setting Text Field Position

The position of the Text field can be changed with the setLocation() method. The coordinates are specified in pixels and are relative to the parent composite object. For example, the following code sets the Text field at coordinate (10, 10): 

          
           
textField.setLocation(10, 10);

Setting Text Field Text

The text displayed in the Text field can be changed with the setText() method. For example, the following code sets the text in the Text field to the string "Hello World!": 

          
           
textField.setText("Hello World!");

Getting Text Field Text

The value of the Text field can be retrieved with the getText() method. For example, the following code gets the value from the Text field and stores it in a String variable: 

          
           
String textValue = textField.getText();

Adding a Modify Listener to a Text Field

A ModifyListener can be added to the Text field so that an event can be fired when the value of the field is changed. The following code snippet adds a ModifyListener to the Text field and prints out the new value when it is changed: 

          
           
textField.addModifyListener(new ModifyListener() {
	public void modifyText(ModifyEvent e) {
		System.out.println("Text field value changed to: "+textField.getText());
	}
});

Disabling a Text Field

The Text field can be disabled so that it cannot be edited or interacted with. The following code snippet disables the Text field: 

          
           
textField.setEnabled(false);
