android.graphics.RadialGradient

The android.graphics.RadialGradient class is a type of Shader which can be used to create a radial gradient on shape objects. When used, it will fill the shape with a smooth blend between the colors defined in the constructor.

Creating a RadialGradient

In order to create a RadialGradient, you will need to use the 

          
           
RadialGradient(float centerX, float centerY, float radius, int[] colors, float[] stops, TileMode tileMode);
constructor.
  • centerX and centerY - This is the coordinates of the center of the circle.
  • radius - The radius of the circle.
  • colors - An array of colors to be used for the gradient.
  • stops - An array of positions where each color should be transitioned in the gradient.
  • tileMode - The tile mode. There are three modes: CLAMP, MIRROR, and REPEAT.

Example

Here is an example of creating a RadialGradient: 

          
           
int[] colors = {Color.RED, Color.GREEN, Color.BLUE};
float[] stops = {0.0f, 0.5f, 1.0f};
RadialGradient gradient = new RadialGradient(0.5f, 0.5f, 0.5f, colors, stops, TileMode.CLAMP);
This will create a RadialGradient that blends from red to green to blue in the center of the circle with a radius of 0.5f.

Applying a RadialGradient

Once you have created your RadialGradient, you can apply it to any drawing object such as a Paint or Shape. Here is an example of applying it to a Paint object: 

          
           
Paint paint = new Paint(); 
paint.setShader(gradient);
Now, the RadialGradient will be applied to the object when you draw on the canvas.

Conclusion

The android.graphics.RadialGradient class is a powerful tool for creating radial gradients onshape objects. With it, you can create a smooth blend between multiple colors in your drawings.
