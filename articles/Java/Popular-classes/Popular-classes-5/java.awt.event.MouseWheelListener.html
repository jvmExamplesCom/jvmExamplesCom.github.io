What is Java.awt.event.MouseWheelListener?

Java.awt.event.MouseWheelListener is an interface in the Java AWT (Abstract Window Toolkit) package. The interface provides methods for handling various events that occur when using the mouse wheel.

public class MyMouseWheelListener implements MouseWheelListener {

  }
