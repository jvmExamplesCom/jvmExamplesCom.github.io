javax.swing.text.AttributeSet

javax.swing.text.AttributeSet is an interface used to represent a set of attributes. It is created when text is parsed and contains information about the text such as its font, size, color, etc. AttributeSet is an important component in the Java Swing API as it is used to store, modify, and access text attributes.

Creating an AttributeSet

An AttributeSet can be created by instantiating an attribute map object, such as MutableAttributeSet. An attribute map object stores attributes in the form of key-value pairs. The following example creates an attribute map and sets the font attributes:


          
           
MutableAttributeSet attributeSet = new SimpleAttributeSet();
attributeSet.addAttribute("fontFamily", "Arial");
attributeSet.addAttribute("fontSize", 14);
attributeSet.addAttribute("fontStyle", Font.PLAIN);

Accessing Attributes

An AttributeSet can be used to access and retrieve stored attributes. An AttributeSet can be queried using the 

getAttribute()
method which takes a key as its argument and returns the corresponding value. The following example demonstrates how to query an attribute set to access the font style of a given text element: 

          
           
AttributeSet attributeSet = textElement.getAttributes();
Object fontStyle = attributeSet.getAttribute("fontStyle");

Modifying Attributes

An AttributeSet can be modified by adding or removing attributes. The 

addAttribute()
and 
removeAttribute()
methods are used to add or remove an attribute to/from the AttributeSet respectively. The following example demonstrates how to modify an AttributeSet by adding a double-underline attribute to a text element: 

          
           
AttributeSet attributeSet = textElement.getAttributes();
attributeSet.addAttribute("doubleUnderline", true);
