io.netty.handler.codec.http.FullHttpRequest

The io.netty.handler.codec.http.FullHttpRequest class provides an HTTP request object which holds the request and contains all the information associated with an HTTP request. It also allows for modification of the request itself.

The FullHttpRequest class includes methods for manipulating the request, including adding and removing headers, setting the URI, and setting the body of the request. It also includes several convenience methods for parsing the request parameters, getting the query string, and more.

Creating an instance of FullHttpRequest

Instantiating a FullHttpRequest is easy, simply use the constructor and pass in the URI, method and headers as parameters.


          
           
FullHttpRequest request = new FullHttpRequest(
    HttpVersion.HTTP_1_1, 
    HttpMethod.GET, 
    "http://example.com/index.html", 
    headers
);
