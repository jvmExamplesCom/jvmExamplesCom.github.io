javax.management.remote.JMXConnectorFactory

The javax.management.remote.JMXConnectorFactory is a Java Management Extensions (JMX) API used to create JMX connectors to connect to JMX remote agents. It is used to connect to remote MBeans and their attributes, operations, notifications and other types of communication. It provides a high-level abstraction for the creation of JMX connectors providing a uniform way to access any kind of implementation of JMX remote agents.

Overview

The JMXConnectorFactory enables applications to communicate with remote MBeans over different protocols. It provides APIs to establish managed connections to remote MBeans over various transport protocols such as HTTP, RMI, IIOP, SOAP, etc. It also provides APIs to establish secure connections and authentication of remote MBeans. The JMXConnectorFactory can be used to connect a client to an agent or server running on a different virtual machine or host. It simplifies the process of creating JMX connectors by hiding complexity of the protocol underneath the application layer. Additionally, it provides extensibility of the connection protocol and allows custom extensions to be used.

Usage

The JMXConnectorFactory is used to establish a connection between a remote JMX agent and one or more clients. It provides an easy to use interface for creating various types of connections and performing operations on them. To use the JMXConnectorFactory, first create a ConnectionFactory instance. This is done using the factory method JMXConnectorFactory.newJMXConnector(). The ConnectionFactory requires a URL to specify the remote JMX agent. Once the ConnectionFactory has been created, call the getConnection() method on it to retrieve a connection to the remote agent. Once the connection is established, the application can invoke operations on the remote MBean. Invoking operations on a remote MBean requires the same steps as if the operation was invoked locally. That is, the application must retrieve a proxy for the remote MBean and use that proxy to invoke operations on the remote MBean.

Examples

Creating a ConnectionFactory


          
           
String url = "service:jmx:rmi://hostname:port/jndi/rmi://hostname:port/jmxconnector";
JMXServiceURL serviceUrl = new JMXServiceURL(url);
JMXConnectorFactory connFactory = JMXConnectorFactory.newJMXConnector(serviceUrl, null);

Retrieving a Connection


          
           
JMXConnector conn = connFactory.getConnection();

Invoking an Operation on a Remote MBean


          
           
// Get the MBean Server connection 
MBeanServerConnection mbsc = conn.getMBeanServerConnection(); 
// Get the object name for the remote MBean 
ObjectName mbeanName = new ObjectName("example:type=MyMBean"); 
// Construct the parameters list for the operation 
Object[] params = new Object[] { "foo" }; 
String[] signature = new String[] { "java.lang.String" }; 
// Invoke the operation 
mbsc.invoke(mbeanName, "setFoo", params, signature);

Closing a Connection


          
           
conn.close();
