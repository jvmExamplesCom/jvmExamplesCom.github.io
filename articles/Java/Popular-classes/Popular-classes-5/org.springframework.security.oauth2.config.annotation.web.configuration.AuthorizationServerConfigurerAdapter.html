org.springframework.security.oauth2.config.annotation.web.configuration.AuthorizationServerConfigurerAdapter

The 

          
           
org.springframework.security.oauth2.config.annotation.web.configuration.AuthorizationServerConfigurerAdapter
is a class used to configure an authorization server that provides access tokens for OAuth2 clients.

This adapter class allows developers to customize their configuration by overriding methods. Developers can configure the authorization server’s endpoints, supported grant types, token services, and more.

Example Usage

The following is an example of how an instance of this class could be used in a Spring application.


          
           
@Configuration
@EnableAuthorizationServer
public class AuthServerConfig extends AuthorizationServerConfigurerAdapter {

    @Autowired
    private AuthenticationManager authenticationManager;

    @Override
    public void configure(AuthorizationServerEndpointsConfigurer endpoints) throws Exception {
        endpoints.authenticationManager(authenticationManager);
    }

    @Override
    public void configure(ClientDetailsServiceConfigurer clients) throws Exception {
        clients.inMemory()
            .withClient("client")
            .secret("secret")
            .authorizedGrantTypes("password", "authorization_code");
    }
}
In this example, the 

          
           
@EnableAuthorizationServer
annotation is used to enable an authorization server in the application. Then, the 
configure()
methods allow for customizing the server’s configuration. In the first, the 

          
           
authenticationManager
is wired into the configuration. Then, a client is configured with a client ID and secret, and allowed to use the password and authorization code grant types.

Methods

The following are the most commonly used methods in this adapter class.

  • 
            
             
configure(AuthorizationServerEndpointsConfigurer endpoints)
    : Allows for customizing the authorization server's endpoints, including the authentication manager, token services, and more.
  • 
            
             
configure(ClientDetailsServiceConfigurer clients)
    : Configures the client details service which provides information about the clients that are allowed to use the authorization server.
