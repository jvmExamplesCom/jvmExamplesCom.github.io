com.google.inject.Provides

The 

          
           
com.google.inject.Provides
annotation is used to provide a new instance of any type of object when needed. This annotation can be used to provide a default value for method parameters, return values, and even fields within the class. It is typically used in combination with the @Inject annotation to create a more fluent API.

Example 1: Using Provides to Provide a Default Value for Method Parameters

In this example, we will create a simple method that takes an integer as a parameter and returns a string based on the parameter value. We will use the 

@Provides
annotation to specify the default value of 1 for the parameter. 

          
           
public class TestClass {

    @Provides
    public String testMethod(@Provides(defaultValue="1") int value) {
        if (value == 1) {
            return "Value is one";
        } else {
            return "Value is not one";
        }
    }
}

We can now call the method without providing any arguments, and the method will assume the value of 1 if none is provided.


          
           
String result = testMethod(); // result is "Value is one"

In this example, we will create a method that returns an integer value. We will use the 

@Provides
annotation to specify the default return value of 5. 

          
           
public class TestClass {

    @Provides
    public int testMethod() {
