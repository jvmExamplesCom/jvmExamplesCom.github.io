Introduction to org.reactivestreams.Subscription

org.reactivestreams.Subscription is an interface in the Reactive Streams API that represents a one-to-one lifecycle of a Subscriber subscribing to a Publisher.

It represents the stage in a reactive exchange where a Publisher has a Subscriber and both have called the subscribe() method on the other instance.

It provides methods to request data from a Publisher to a Subscriber, cancel an ongoing subscription and check if a subscription is still valid.

Example of org.reactivestreams.Subscription

The following example demonstrates how to use org.reactivestreams.Subscription.


          
           
public class ExampleSubscription {
  public static void main(String[] args) {
    // Create a publisher
    Flux<String> flux = Flux.just("Hello", "World");
    
    // Create a subscriber
    FluxSink<String> subscriber = flux.subscribe();
    
    // Subscribe to the publisher
    flux.subscribe(subscriber::next);
    
    // Get the active subscription
    Subscription sub = subscriber.currentSubscription();
    
    // Request two items from the publisher
    sub.request(2);
    
    // Cancel the subscription
    sub.cancel();
    
    // Check if subscription is still valid
    System.out.println(sub.isCancelled());
  }
}
