javax.swing.JTextPane

javax.swing.JTextPane is a text component that allows programmatic manipulation of its contents as well as creation of basic text editing functionality. It provides support for text display, printing, and editing syntax highlighting, and undo/redo functionality. It is part of the Java Swing package and is used to create editable text components.

JTextPane textPane = new JTextPane();

To add text to a JTextPane, first get the Document from the JTextPane, and then use the Document's insertString method to modify the contents:


          
           
Document doc = textPane.getDocument();
