javax.swing.JTextPane

javax.swing.JTextPane is a text component that allows programmatic manipulation of its contents as well as creation of basic text editing functionality. It provides support for text display, printing, and editing syntax highlighting, and undo/redo functionality. It is part of the Java Swing package and is used to create editable text components.

Creating a JTextPane

To create a JTextPane, create a new instance of it and pass in any necessary parameters:


          
           
JTextPane textPane = new JTextPane();

Adding Text to a JTextPane

To add text to a JTextPane, first get the Document from the JTextPane, and then use the Document's insertString method to modify the contents:


          
           
Document doc = textPane.getDocument();
doc.insertString(0, "This is an example of adding text to a JTextPane.", null);

Setting the Font of a JTextPane

To set the font of a JTextPane, first create a new Font object and then set the font on the JTextPane:


          
           
Font font = new Font("Monospaced", Font.PLAIN, 12);
textPane.setFont(font);

Displaying HTML Content

The JTextPane can display HTML content by setting its content type to 'text/html', and then setting the document to contain the HTML string:


          
           
textPane.setContentType("text/html");
String htmlString = "<html><body>Hello World!</body></html>";
