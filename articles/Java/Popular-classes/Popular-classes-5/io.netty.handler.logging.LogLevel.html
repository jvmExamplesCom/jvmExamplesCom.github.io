io.netty.handler.logging.LogLevel

There are several different log levels available, ranging from DEBUG, INFO, WARNING, and FATAL. The primary benefit of using LogLevel is that it allows you to quickly configure your application with the appropriate log level without the need to manually configure logging in your application code. Additionally, with the LogLevel enum, you can easily add additional log levels as needed.

Logger.getLogger("MyLog").setLevel(LogLevel.DEBUG);
