The primary benefit of using LogLevel is that it allows you to quickly configure your application with the appropriate log level without the need to manually configure logging in your application code. Additionally, with the LogLevel enum, you can easily add additional log levels as needed.

Logger.getLogger("MyLog").setLevel(LogLevel.DEBUG);
