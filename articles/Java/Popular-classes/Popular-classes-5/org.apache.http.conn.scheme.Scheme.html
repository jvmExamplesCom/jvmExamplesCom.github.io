org.apache.http.conn.scheme.Scheme

The org.apache.http.conn.scheme.Scheme class represents an object used to encapsulate the protocol for a connection. It contains information about the protocol name, default port and a socket factory that is used to create sockets for the given protocol. This class is thread safe and can be used from multiple threads simultaneously.
