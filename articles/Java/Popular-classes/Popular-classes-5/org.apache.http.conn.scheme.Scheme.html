org.apache.http.conn.scheme.Scheme

The org.apache.http.conn.scheme.Scheme class represents an object used to encapsulate the protocol for a connection. It contains information about the protocol name, default port and a socket factory that is used to create sockets for the given protocol. This class is thread safe and can be used from multiple threads simultaneously.

Constructors

The org.apache.http.conn.scheme.Scheme class contains two constructors: 


          
           
public Scheme(String name, SocketFactory factory, int port)
public Scheme(String name, SocketFactory factory, int port, boolean layered)

The first constructor creates a scheme with the specified name, socket factory and port. The second constructor creates a scheme with the specified name, socket factory, port and layered flag.
