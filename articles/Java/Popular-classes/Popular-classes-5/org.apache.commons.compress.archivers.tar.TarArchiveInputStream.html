org.apache.commons.compress.archivers.tar.TarArchiveInputStream

org.apache.commons.compress.archivers.tar.TarArchiveInputStream is an input stream that reads a TAR archive as an ordered sequence of files and directories. The TAR format (Tape Archive) is a popular file archiving format that allows grouping multiple files into a single file.

This class implements the same behavior as the java.util.zip.ZipInputStream, but is optimized for reading archives in the TAR format.

The following code snippet shows how to create a TarArchiveInputStream from a File:

InputStream is = new FileInputStream(file);
TarArchiveInputStream tarIn = new TarArchiveInputStream(is);

Example 2: Read entries from a TarArchiveInputStream

The following code snippet shows how to read entries from a TarArchiveInputStream:


          
           
// create TarArchiveInputStream
File file = new File("archive.tar");
InputStream is = new FileInputStream(file);
TarArchiveInputStream tarIn = new TarArchiveInputStream(is);

// iterate over entries in the archive
TarArchiveEntry entry;
while ((entry = tarIn.getNextTarEntry()) != null) {
    System.out.println(entry.getName());
}

// close input stream
tarIn.close();

Example 3: Read bytes from an entry in TarArchiveInputStream

The following code snippet shows how to read bytes from an entry in TarArchiveInputStream:


          
           
// create TarArchiveInputStream
File file = new File("archive.tar");
InputStream is = new FileInputStream(file);
TarArchiveInputStream tarIn = new TarArchiveInputStream(is);

// find entry in the archive
TarArchiveEntry entry;
while ((entry = tarIn.getNextTarEntry()) != null) {
    if (entry.getName().equals("filename.txt")) {
