com.beust.jcommander.ParameterException
is a checked exception that signals an erroneous parameter was encountered. This exception is thrown during the parsing of the parameters using JCommander.

Example Usage

Here is an example of code using the 
ParameterException
: 

          
           
try {
    //parse command line arguments
    JCommander.newBuilder().build().parse(args);
} catch (ParameterException pe) {
    System.err.println("Error: " + pe.getMessage());
}
