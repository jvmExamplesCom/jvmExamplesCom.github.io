Introduction to Java's com.google.common.primitives.Bytes

By automatically providing methods for operations such as searching, sorting, and filling, it simplifies the task of working with primitive types in Java code. int index = Bytes.searchFor(new byte[] {1, 2, 3, 4}, (byte) 3); // returns 2
int binaryIndex = Bytes.binarySearch(new byte[] {1, 2, 4}, (byte) 3); // returns -3
