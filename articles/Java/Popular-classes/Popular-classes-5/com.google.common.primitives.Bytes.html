Introduction to Java's com.google.common.primitives.Bytes

It provides methods for operating on primitive types such as int, long and double, as well as methods for searching, sorting, filling and reversing the contents of an array. This package is part of the Google's Guava library and the com.google.common namespace. It was designed with the intent of making it easier to work with primitive types. By automatically providing methods for operations such as searching, sorting, and filling, it simplifies the task of working with primitive types in Java code. We'll also look at some of the utility classes contained within the package, such as ByteRange and ByteSet.

Searching Primitive Arrays

The first set of methods we'll look at utilize the searchFor and binarySearch static methods. These methods allow you to search for an item in an array of primitive types and return its index if found. 

          
           
int index = Bytes.searchFor(new byte[] {1, 2, 3, 4}, (byte) 3); // returns 2
int binaryIndex = Bytes.binarySearch(new byte[] {1, 2, 4}, (byte) 3); // returns -3

The reverse static method allows you to reverse the order of the elements in an array. It takes just one parameter, the array to reverse.
