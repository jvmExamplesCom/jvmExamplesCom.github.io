Java NIO File FileAlreadyExistsException

The 

          
           
java.nio.file.FileAlreadyExistsException
exception is thrown when attempting to create a file that already exists. It is a subclass of 

          
           
IOException
and its detail message takes the form of: 

          
           
file<'file'> already exists
where 'file' is the file that already exists. This exception can be thrown by methods that attempt to create a file in a file system. The following example creates a new file in an existing directory, which results in a 

          
           
FileAlreadyExistsException
. 

          
           
try {
    Files.createFile(Paths.get("/tmp/mydirses/myfile.txt"));
} catch (FileAlreadyExistsException e) {
    System.err.println("File already exists");
}
The exception can also be thrown when attempting to create a directory that already exists as follows: 

          
           
try {
    Files.createDirectory(Paths.get("/tmp/mydirses/mydir"));
} catch (FileAlreadyExistsException e) {
    System.err.println("Directory already exists");
}
In both cases, the detail message will contain the path of the file or directory that already exists.

Handling FileAlreadyExistsException

When handling a 

          
           
FileAlreadyExistsException
, one should determine what is the desired action in case of such a conflict. Common solutions are to check if the file exists before attempting to create it, to overwrite the existing file, or to throw an exception and fail. For instance, the following snippet shows how to create a file only if the file does not exist: 

          
           
Path path = Paths.get("/tmp/mydirses/myfile.txt");
if (!Files.exists(path)) {
    Files.createFile(path);
}
One can also explicitly overwrite the existing file, as shown below: 

          
           
Path path = Paths.get("/tmp/mydirses/myfile.txt");
Files.createFile(path, 
    StandardOpenOption.CREATE_NEW, 
    StandardOpenOption.WRITE);
Finally, one could throw an exception in the case of an existing file, as follows: 

          
           
Path path = Paths.get("/tmp/mydirses/myfile.txt");
if (Files.exists(path)) {
    throw new FileAlreadyExistsException(path.toString());
} else {
    Files.createFile(path);
}

Conclusion

In this article, we discussed the 

          
           
java.nio.file.FileAlreadyExistsException
exception and how to handle it. We saw three common strategies for dealing with files that already exist on the file system: checking for the existence of the file before attempting to create it, overwriting the existing file and throwing an exception.
