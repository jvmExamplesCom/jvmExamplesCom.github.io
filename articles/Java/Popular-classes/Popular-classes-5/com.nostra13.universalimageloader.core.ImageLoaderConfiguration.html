com.nostra13.universalimageloader.core.ImageLoaderConfiguration

The com.nostra13.universalimageloader.core.ImageLoaderConfiguration class is used to create instances of ImageLoaderConfiguration and customize the configuration of the Universal ImageLoader. This class enables users to define custom elements including maximum width/height, memory Cache Size, diskCacheSize, decoding options, thread pool size, and display image options.
