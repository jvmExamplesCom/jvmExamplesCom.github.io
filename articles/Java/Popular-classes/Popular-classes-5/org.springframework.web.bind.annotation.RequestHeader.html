org.springframework.web.bind.annotation.RequestHeader

The 
@RequestHeader
annotation is a type of parameter-level annotation used in the Spring Web Framework. It allows the user to access request header information, such as the Accept-Language header, and bind it to a specific method argument.

Syntax

The syntax of the 
@RequestHeader
annotation is as follows: 

          
           
@RequestHeader(name = "headerName") String headerValue
In this syntax, 
name
represents the name of the request header and 
headerValue
is the value of the request header.

Example

The following example demonstrates how to use the 
@RequestHeader
annotation to access the value of the request header 
Accept-Language
: 

          
           
@RequestMapping(value = "/hello")
public String helloWorld(@RequestHeader(name = "Accept-Language") String language) {
   return "Hello world";
}
In the above example, the value of the 
Accept-Language
request header will be stored in the 
language
variable.
