Introduction to java.awt.event.MouseWheelEvent

The Java AWT MouseWheelEvent class is an event class for handling mouse wheel events. It extends the java.awt.event.InputEvent class. The MouseWheelEvent class provides methods for determining the number of clicks on the mouse wheel and the direction of scrolling. It is used for applications that need to track the direction and speed of the mouse wheel when being used.

How to Use MouseWheelEvent?

Using the MouseWheelEvent class is easy and straightforward. The following code example shows how to set up a MouseWheelEvent listener in order to receive mouse wheel events: 

          
           
// Create the MouseWheelEvent listener
MouseWheelListener mwl = new MouseWheelListener() {
  public void mouseWheelMoved(MouseWheelEvent e) {
    // Processing code here
  }
};

// Register the listener with the component
component.addMouseWheelListener(mwl);
Once the listener has been registered with the component, it will be invoked when the user moves the mouse wheel over the component.

Methods of MouseWheelEvent Class

The MouseWheelEvent class provides several useful methods for tracking the mouse wheel movement. The most commonly used methods are the following: 

          
           
getWheelRotation()
This method returns the number of rotations of the wheel; a positive number indicates that the wheel was moved forward, and a negative number indicates that the wheel was moved backward. 

          
           
getPreciseWheelRotation()
This method returns the precise rotation of the wheel, in radians per tick. 

          
           
getScrollType()
This method returns the type of scrolling that occurred. It can be either WHEEL_UNIT_SCROLL or WHEEL_BLOCK_SCROLL.
