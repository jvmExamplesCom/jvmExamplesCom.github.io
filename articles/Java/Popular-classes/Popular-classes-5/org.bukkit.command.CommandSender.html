org.bukkit.command.CommandSender

The CommandSender class provides a way to send commands to a server. It allows for objects that need to send commands to the server, such as players, non-player characters (NPCs), or other entities. It is used in many plugins, particularly those that require communication between the player and the server.

getName()

getName()
method.

//Get the name of the sender
