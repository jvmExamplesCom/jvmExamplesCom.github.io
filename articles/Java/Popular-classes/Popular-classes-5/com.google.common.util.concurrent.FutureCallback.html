Introduction to com.google.common.util.concurrent.FutureCallback

When is com.google.common.util.concurrent.FutureCallback Needed?

Com.google.common.util.concurrent.FutureCallback is called when a thread must wait for a task to complete before continuing. An example of this is when a thread needs to wait for a file operation to complete before continuing.
