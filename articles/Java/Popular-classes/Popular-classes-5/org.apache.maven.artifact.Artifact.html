org.apache.maven.artifact.Artifact

The org.apache.maven.artifact.Artifact class is an interface that allows Maven to interact with different types of artifacts such as JAR, WAR, and ZIP files. This class provides methods for retrieving, managing and manipulating artifacts. It is often used for the purpose of creating new artifacts and for managing a Maven project's dependencies.

Methods

The org.apache.maven.artifact.Artifact class provides several methods for interacting with artifacts. These include the following:

  • getArtifactId(): This method returns the artifact's ID.
  • getGroupId(): This method returns the artifact's group ID.
  • getType(): This method returns the type of the artifact, such as jar or war.
  • getVersion(): This method returns the version of the artifact.
  • setArtifactId(): This method sets the artifact's ID.
  • setGroupId(): This method sets the artifact's group ID.
  • setType(): This method sets the type of the artifact.
  • setVersion(): This method sets the version of the artifact.
  • getFile(): This method returns the artifact's file.
