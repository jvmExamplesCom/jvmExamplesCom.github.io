com.fasterxml.jackson.datatype.jdk8.Jdk8Module

The com.fasterxml.jackson.datatype.jdk8.Jdk8Module is a module used to add support for the Java 8 data types. It enables the JSON serialization of Java 8's temporal classes such as LocalDate, LocalTime and LocalDateTime as well as Optional, Stream, and Duration. It is part of the Jackson data processor which can be used to process JSON in applications.

Adding the Module to an Application

In order to use the Jdk8Module, it must first be added to an application's Jackson ObjectMapper. This is done by calling registerModule on the mapper. 

          
           
ObjectMapper om = new ObjectMapper();
om.registerModule(new Jdk8Module());

Using LocalDate and LocalTime

The Jdk8Module adds support for the Java 8 LocalDate and LocalTime. This allows these types to be serialized to and from JSON without additional configuration or customization. LocalDate and LocalTime can be written to JSON using the ObjectMapper: 

          
           
LocalDate ld = LocalDate.of(2019, 5, 27);
String ldString = om.writeValueAsString(ld);
System.out.println(ldString); // Outputs "2019-05-27"

LocalTime lt = LocalTime.of(14,30);
String ltString = om.writeValueAsString(lt);
System.out.println(ltString); // Outputs "14:30:00"
Similarly, they can be read from JSON as follows: 

          
           
LocalDate ld = om.readValue("\"2019-05-27\"", LocalDate.class);
LocalTime lt = om.readValue("\"14:30:00\"", LocalTime.class);
