javax.jms.ConnectionFactory

ConnectionFactory cf = new JmsConnectionFactory(host, port, user, password);

where host, port, user and password are the parameters required to connect to the JMS provider.

Creating a Connection Instance

Connection conn = cf.createConnection();

Closing a Connection Instance

You can close a connection instance by calling its close() method, like this:
