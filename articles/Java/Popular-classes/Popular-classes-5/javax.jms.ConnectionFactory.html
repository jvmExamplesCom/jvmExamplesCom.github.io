javax.jms.ConnectionFactory

Creating a Connection Factory Instance

To create an instance of a connection factory, use code like this one:


          
           
ConnectionFactory cf = new JmsConnectionFactory(host, port, user, password);

where host, port, user and password are the parameters required to connect to the JMS provider.

Context context = new InitialContext(...);
ConnectionFactory cf = (ConnectionFactory) context.lookup("MyConnectionFactory");

Creating a Connection Instance

Once you have obtained an instance of a connection factory, you can create a connection to the JMS provider using the createConnection() method, as shown below:


          
           
Connection conn = cf.createConnection();

This will create a new connection to the JMS provider. The connection will be in an uninitialized state until you call the start() method.

Closing a Connection Instance

conn.close();
