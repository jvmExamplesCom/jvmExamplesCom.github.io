javax.xml.stream.XMLStreamConstants

The 

          
           
javax.xml.stream.XMLStreamConstants
is an interface that defines constants for various XML elements. It is useful for recognizing characters, strings, comments, entity references, and processing instructions. This interface can also be used for passing information from a reader (such as a DOM parser) to a processor.
