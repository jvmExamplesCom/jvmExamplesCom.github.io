javax.jms.TextMessage

TextMessage is an object of the javax.jms package used for exchanging messages between applications. It is a type of message that contain a text string as well as other data necessary for the reliable delivery of said message.

Creating Textmessage Objects

Creating a new TextMessage object is done via the 

          
           
Session.createTextMessage()
method, passing in a string as the argument. This string can be retrieved later using the 
getText()
method. 

          
           
TextMessage tm = session.createTextMessage("Hello from my application");
System.out.println(tm.getText());
Output: 

          
           
Hello from my application

Setting and Getting Properties

In addition to the text string, TextMessages also let you set and get custom properties. Properties are set via the 

setObjectProperty()
method and can be retrieved using the 
getObjectProperty()
method. When setting and getting the property, a string name should be used as the argument. 

          
           
// Setting a property
tm.setObjectProperty("myProperty", "My Value");

// Getting a property
String value = (String) tm.getObjectProperty("myProperty");
System.out.println(value);
Output: 

          
           
My Value

Delivery Mode

When sending a TextMessage, you may want to specify the delivery mode as either "persistent" or "non-persistent". A persistent message will be saved to a message store until it has successfully been received by the consumer. A non-persistent messge is sent immediately and not stored. The delivery mode is set using the 


          
           
setJMSDeliveryMode()
method. 

          
           
tm.setJMSDeliveryMode(DeliveryMode.PERSISTENT);

Message Receipt

Once a TextMessage has been received by the consumer, it can be acknowledged by the consumer using the 

acknowledge()
method. Acknowledging the message ensures that it is marked as "read" and can be deleted from the message store. 

          
           
MessageConsumer mc = session.createConsumer(destination);
Message message = mc.receive();
