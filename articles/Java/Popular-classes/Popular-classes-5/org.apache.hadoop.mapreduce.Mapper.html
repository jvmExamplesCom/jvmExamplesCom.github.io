Introduction to org.apache.hadoop.mapreduce.Mapper in Java

The Mapper class in Apache Hadoop is one of the core components for MapReduce and provides the map() method for mapping input key/value pairs to a set of intermediate key/value pairs. The Mapper can be used to transform input records into a desired output format and can also be used for filtering out unwanted data. In this article, we will discuss the features of the Mapper class and provide code examples of its implementation.

Features of org.apache.hadoop.mapreduce.Mapper

  • Provides the map() method for mapping input key/value pairs to a set of intermediate key/value pairs.
  • Can perform transformations on input records and extract only desired information.
  • Allows the user to specify the data types of input and output keys and values.
  • Supports both sequential and distributed execution of map tasks.
