com.fasterxml.jackson.annotation.JsonAnyGetter

The  

          
           
&nbsp;com.fasterxml.jackson.annotation.JsonAnyGetter&nbsp;
annotation is used to explain that a method will return additional properties as a Map. It is generally used to return properties from the external sources. This annotation is useful when you have to serialize the entire object into JSON and not just some of its fields.

Examples

Example 1

Let us consider an example of the usage of  

          
           
&nbsp;com.fasterxml.jackson.annotation.JsonAnyGetter&nbsp;
annotation where we will create a Person class and use JsonAnyGetter annotation to show how to return additional properties from a Map. 

          
           
public class Person {
    private String firstName;
    private String lastName;
    private Map<String, Object> otherProperties = new HashMap<>();

    // getters and setters

    @JsonAnyGetter
    public Map<String, Object> getOtherProperties() {
        return otherProperties;
    }
}

Now if we try to serialize this Person object, it will return the value of 

otherProperties
map along with 
firstName
and 
lastName
.
