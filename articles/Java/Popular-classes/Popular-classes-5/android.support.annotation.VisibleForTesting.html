android.support.annotation.VisibleForTesting

The 

          
           
android.support.annotation.VisibleForTesting
annotation is an annotation provided by the Android Support Library to indicate that an element of a program is visible and can be accessed from tests.

By default, Android elements are not visible outside the package in which they are defined. This means that if you want to test one or more elements of your application, you may need to make them visible from outside the package. The 


          
           
android.support.annotation.VisibleForTesting
annotation facilitates this process by allowing you to mark any element as "visible for testing".

Usage

Using the 

          
           
android.support.annotation.VisibleForTesting
annotation is simple: simply add the annotation to any element of your program that you wish to make visible to tests. 

          
           
@VisibleForTesting
public int sum(int a, int b) {
    return a + b;
}

This syntax can also be used to denote an entire class as visible for testing:


          
           
@VisibleForTesting
public class TestableClass {
    //...
}

android.support.annotation.VisibleForTesting
