com.amazonaws.auth.AWSCredentials

com.amazonaws.auth.AWSCredentials is a Java interface provided by Amazon Web Services (AWS) for authentication purposes. The interface provides two methods, getAWSAccessKeyId() and getAWSSecretKey() which return credentials used to access various resources. These credentials are necessary in order to successfully interact with AWS services.

Creating an AWSCredentials Object

The following example code creates an AWSCredentials object and sets the Access Key Id and Secret Key accordingly:


          
           
String accessKeyId = "12345678";
String secretKey = "abcdefgh";

AWSCredentials credentials = new BasicAWSCredentials(accessKeyId, secretKey);

Using an AWSCredentials Object

An AWSCredentials object can be used to create an Amazon S3 client as follows:


          
           
AmazonS3 s3Client = AmazonS3ClientBuilder
    .standard()
    .withCredentials(new AWSStaticCredentialsProvider(credentials))
    .build();

In addition, an AWSCredentials object can be used to authenticate requests to the Amazon API gateway service as described in the API Gateway Developer Guide.
