Android Graphics Drawable Icon

Android provides a class called android.graphics.drawable.Icon that is used for drawing icons. It has a few methods that can be used to create a variety of icon shapes and sizes.

The Basics

The Icon class is used to represent any type of icon - from small icons such as a phone's home screen icon, to large icons such as a map marker. The Icon class can also be used to create bitmaps and shapes. To create an Icon, you must first call the static method createWithResource(). This method takes an integer parameter, which is the resource id of an existing Drawable. For example: 

          
           
Drawable drawable = getResources().getDrawable(R.drawable.icon);
Icon icon = Icon.createWithResource(context, drawable);
You can also create an Icon from a Bitmap: 

          
           
Bitmap bitmap = BitmapFactory.decodeResource(getResources(), R.drawable.icon);
Icon icon = Icon.createWithBitmap(bitmap);
Once you have an instance of an Icon, you can customize it using the setIcon() method. This method takes a Drawable or a Bitmap as a parameter. You can also use the setTint(), setTintMode(), and setTintList() methods to customize the color of an icon.

Creating Shapes

Besides creating icons with bitmaps and drawables, you can also create shapes with the Icon class. The createWithShape() method takes an integer parameter, which specifies the shape of the icon. The different shapes supported are ROUND, RECTANGLE, SQUARE, and CIRCLE. For example, to create a rounded rectangle icon, you can use the following code: 

          
           
Icon icon = Icon.createWithShape(Icon.ROUNDED_RECTANGLE);
To set the width and height of the icon, use the setBounds() method. The method takes two integer parameters - the first is the width and the second is the height. 

          
           
icon.setBounds(30, 30);
To set the color of the icon, use the setTint() method: 

          
           
icon.setTint(Color.RED);

Drawing The Icon

Once you have created your icon, you can draw it on a canvas. To do this, use the Canvas.drawIcon() method. This method takes an Icon as a parameter. For example: 

          
           
Canvas canvas = new Canvas(bitmap);
canvas.drawIcon(icon);

Conclusion

Android's Icon class makes it easy to create a variety of icons and shapes. It can be used to create bitmaps and drawables, as well as custom shapes such as circles, squares, and rectangles. With a few simple lines of code, you can create a unique icon that can be used in your Android application.
