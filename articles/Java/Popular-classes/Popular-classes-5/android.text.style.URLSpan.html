Android Text Style URLSpan

Android Text Style URLSpan is a powerful tool for creating links within the text of an Android application. It allows developers to add a clickable link that will open a browser window with specified URL when clicked by the user, making it easy to direct users to other websites or applications. This tutorial will go over how to use Android Text Style URLSpan, including code examples and output as HTML.

Creating a URLSpan Object

To create a URLSpan object, you can use the following Java code:


          
           
URLSpan urlSpan = new URLSpan("http://example.com");

The constructor accepts a String containing the URL to be opened when the span is clicked. Once the object is created, you can set it on any portion of text in a text view:


          
           
textView.setSpan(urlSpan, 0, textView.getText().length(), Spanned.SPAN_EXCLUSIVE_EXCLUSIVE);

This sets the URLSpan on the whole text view. You can also specify the start and end indices to set the span on only a certain part of the text. The third parameter specifies the type of span; the example above sets a SPAN_EXCLUSIVE_EXCLUSIVE type of span, meaning that none of the text around the span will be affected.

Specifying a Title for the Link

You can also specify a title for the link using the setTitle() method:


          
           
urlSpan.setTitle("Link Title");

When the URLSpan is clicked, the link will appear on the browser window with this title. This is useful if you want to display a descriptive name for the link rather than the full URL.

Outputting the Span as HTML

Once the URLSpan is set, you can output the text view as HTML using the following code:


          
           
String htmlString = Html.toHtml(textView.getText());

This will create an HTML string that contains the URLSpan as a valid HTML link tag, something like this:


          
           
<a href="http://example.com" title="Link Title">Linked Text</a>
