Introduction to org.apache.commons.beanutils.PropertyUtils

The org.apache.commons.beanutils.PropertyUtils class provides a set of methods for getting and setting property values on beans. The class is part of Apache Commons BeanUtils library, used for manipulating JavaBeans.

Getting Property Values with PropertyUtils

The most common use of PropertyUtils is getting property values from beans. The getProperty() method can be used to retrieve a property value from an object: 

          
           
Object value = PropertyUtils.getProperty(bean, "propertyName");

Object value = PropertyUtils.getSimpleProperty(bean, "propertyName");

Setting Property Values with PropertyUtils

PropertyUtils provides the setProperty() method for setting property values on a bean: 

          
           
PropertyUtils.setProperty(bean, "propertyName", value);
