Introduction to org.apache.commons.beanutils.PropertyUtils

The org.apache.commons.beanutils.PropertyUtils class provides a set of methods for getting and setting property values on beans. The class is part of Apache Commons BeanUtils library, used for manipulating JavaBeans.

Getting Property Values with PropertyUtils

The most common use of PropertyUtils is getting property values from beans. The getProperty() method can be used to retrieve a property value from an object: 

          
           
Object value = PropertyUtils.getProperty(bean, "propertyName");
The getProperty() method will return the value associated with the specified property name. If the property does not exist, a NoSuchMethodException is thrown. In addition to the getProperty() method, PropertyUtils also provides the getSimpleProperty() method which is equivalent to the getProperty() method but does not take into account indexed properties or mapped properties. 

          
           
Object value = PropertyUtils.getSimpleProperty(bean, "propertyName");

Setting Property Values with PropertyUtils

PropertyUtils provides the setProperty() method for setting property values on a bean: 

          
           
PropertyUtils.setProperty(bean, "propertyName", value);
The setProperty() method sets the property value on the specified bean instance. The value specified must be of a type compatible with the type of the property. For example, if the property is an integer, then the value specified must be an integer. In addition to the setProperty() method, PropertyUtils also provides the setSimpleProperty() method which is equivalent to the setProperty() method but does not take into account indexed properties or mapped properties. 

          
           
PropertyUtils.setSimpleProperty(bean, "propertyName", value);
