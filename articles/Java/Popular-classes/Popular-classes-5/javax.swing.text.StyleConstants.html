javax.swing.text.StyleConstants

The javax.swing.text.StyleConstants class provides the constants and methods necessary to create a style in the javax.swing.text package. This class helps developers create consistent user interfaces, by allowing them to set: fonts, alignment, sizes and colors of text. Furthermore, it can be used to define the style of Hyperlinks and Components.

Constructors

The javax.swing.text.StyleConstants Constructor initializes a StyleConstants class with no parameters, and can be called as follows:


          
           
StyleConstants newSC = new StyleConstants();

Methods

The javax.swing.text.StyleConstants class contains many methods which allow developers to set style-specific properties. These include:

  • getAlignment: Returns the current alignment property set on the style.
  • setAlignment: Sets the alignment property on a given style.
  • getBackground: Returns the background color of the style.
  • setBackground: Sets the background color of the style.
  • getFontFamily: Returns the font family of the style.
  • setFontFamily: Sets the font family of the style.
  • getFontSize: Returns the font size of the style.
  • setFontSize: Sets the font size of the style.
  • getForeground: Returns the foreground color of the style.
  • setForeground: Sets the foreground color of the style.

Usage Example

Let’s take a look at a sample of how the javax.swing.text.StyleConstants Class can be used:


          
           
JTextPane tPane = new JTextPane();
StyleConstants sc = new StyleConstants();
sc.setFontFamily("Monospaced");
sc.setAlignment(StyleConstants.ALIGN_CENTER);
sc.setBackground(Color.GRAY);
sc.setForeground(Color.WHITE);
sc.setFontSize(20);
tPane.setParagraphAttributes(sc, true);

In this example, we create a JTextPane, then initialize a StyleConstants object, and set several properties such as font family, alignment, background, foreground and font size. Finally, we set the paragraph attributes, using the StyleConstants object.
