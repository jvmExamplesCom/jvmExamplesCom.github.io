org.springframework.data.mongodb.core.mapping.Document

org.springframework.data.mongodb.core.mapping.Document is an annotation used to mark a Java class as a Document in MongoDB. It supports the mapping of a Java class to a MongoDB document in a seamless manner. The mapping is done by converting the class to a BSON (Binary JSON) document, and then storing it into a MongoDB collection. This annotation facilitates easy retrieval and updating of the object from MongoDB.

Annotation Attributes

The Document annotation in MongoDB supports the following attributes:

  • collection - The name of the MongoDB collection where the document will be stored. If not specified, the class name will be used as the collection name.
  • language - The language of the document, if any.
  • readPreference - The read preference for the document. The default read preference is Primary.
  • writeConcern - The write concern for the document. The default write concern is Acknowledged.
  • clientEncoding - The client encoding to use when writing or retrieving the document. The default encoding is utf-8.
