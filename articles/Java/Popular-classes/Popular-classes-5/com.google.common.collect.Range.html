Java - com.google.common.collect.Range

The java.util.Range is a powerful tool for working with ranges in Java. It provides methods for creating, manipulating, and querying ranges of values. Ranges are useful for working with collections of values, such as arrays or lists. For example, one might want to iterate over a particular range of values within the array. Or, one might want to find an element in the array that lies within a particular range. Range makes this possible by providing methods for creating, comparing, and querying ranges.

Creating Ranges

Range provides a variety of methods for creating ranges of values. These include create() and createClosed(). 

          
           
Range range = Range.create(1, 10); // Creates a range from 1 to 10 (inclusive) 
Range closedRange = Range.createClosed(1, 10); // Creates a range from 1 to 10 (both inclusive)
Ranges can also be created from Comparator objects. 

          
           
Range range = Range.from(Comparator.naturalOrder()).lowerEndpoint(1).upperEndpoint(10); // Creates a range from 1 to 10 (inclusive)

Querying Ranges

Once a range is created, it can be queried using the contains() and containsAll() methods. 

          
           
boolean contains = range.contains(5); // returns true 
boolean containsAll = range.containsAll(Arrays.asList(4, 5, 6)); // returns true
