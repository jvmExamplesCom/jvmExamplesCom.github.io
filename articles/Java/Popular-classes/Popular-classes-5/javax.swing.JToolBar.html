javax.swing.JToolBar

The javax.swing.JToolBar class provides a component that is useful for displaying commonly used actions or controls. It gives developers an easy way to create a toolbar of actionable items. A JToolBar can contain buttons, text fields, combo boxes, separators, and other components. It also supports drag-and-drop so users can reorder the components. In this article, we will discuss how to use the javax.swing.JToolBar class to create a simple toolbar.

Adding Components to a Toolbar

The first step in creating a JToolBar is to add components to it. This can be done using the 


          
           
add()
and 

          
           
addSeparator()
methods. The 

          
           
add()
method takes a single parameter which is the component to be added to the toolbar. The 

          
           
addSeparator()
method adds a line which separates different components in the toolbar. An example of adding components to a toolbar is shown below: 

          
           
JToolBar toolbar = new JToolBar(); 
toolbar.add(new JButton("Button 1"));
toolbar.addSeparator();
toolbar.add(new JButton("Button 2"));
toolbar.add(new JTextField("Text Field"));

This code creates a new JToolBar and adds two buttons and a text field to it. The addSeparator() method adds a line which separates the two buttons.

Adding an Action Listener to a Toolbar Button

The next step in creating a JToolBar is to add an action listener to the toolbar button. This can be done by using the 


          
           
addActionListener()
method. This method takes a single parameter which is an instance of the ActionListener interface. An example of adding an action listener to a toolbar button is shown below: 

          
           
JToolBar toolbar = new JToolBar();
JButton button = new JButton("Button");
button.addActionListener(new ActionListener(){
    public void actionPerformed(ActionEvent e){
        // Do something
    }
});
toolbar.add(button);

This code creates a new JToolBar and a new button. It then adds an action listener to the button. Whenever the user clicks on the button, the code in the actionPerformed() method will be executed.

Displaying a Toolbar

The final step in creating a JToolBar is to display it. This can be done by calling the 


          
           
setVisible(true)
method. An example of displaying a JToolBar is shown below: 

          
           
JToolBar toolbar = new JToolBar();
// Add components to toolbar
toolbar.setVisible(true);

This code creates a new JToolBar and sets its visibility to true. This will cause the JToolBar to appear on the screen.

Conclusion

In this article, we discussed the javax.swing.JToolBar class and how to use it to create a simple toolbar. We looked at how to add components to a toolbar and how to add an action listener to a toolbar button. Finally, we discussed how to display a JToolBar. This should give you a good understanding of how to use the JToolBar class.
