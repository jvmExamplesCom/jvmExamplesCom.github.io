What is javax.annotation.processing.SupportedAnnotationTypes?

javax.annotation.processing.SupportedAnnotationTypes is an annotation type used to indicate which annotation types an Annotation Processor supports. This allows the Annotation Processor to identify which annotation types are applicable to it and which may be ignored.

How Does It Work?

The SupportedAnnotationTypes annotation is specified on a class that implements the AnnotationProcessor interface. This annotation specifies which annotation types the processor will process. The value() method of the annotation must return an array of Strings that denote the fully qualified names of the annotations to be processed by this Annotation Processor.
