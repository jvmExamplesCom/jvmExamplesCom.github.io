Introduction to org.springframework.context.annotation.DependsOn

The 

          
           
DependsOn
annotation is a part of the Spring Framework and is used to indicate that a bean must be initialized before the specified beans. @Bean

@Configuration
class. The annotation takes an array of strings containing the names of the beans that must be initialized before the bean with the 

          
           
DependsOn
