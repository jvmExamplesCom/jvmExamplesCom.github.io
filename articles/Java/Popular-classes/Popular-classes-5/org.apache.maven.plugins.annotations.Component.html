org.apache.maven.plugins.annotations.Component is an annotation found in the Maven Plugin API. This annotation denotes a component of a plugin, and is used to bring the component into the plugin. This annotation allows components to be adapted from other components from other plugins.

Usage of org.apache.maven.plugins.annotations.Component

@Component
public class MyComponent {
...
}
