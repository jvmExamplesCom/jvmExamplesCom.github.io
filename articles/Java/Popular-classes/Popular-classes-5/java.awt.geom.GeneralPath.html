java.awt.geom.GeneralPath

The 

          
           
java.awt.geom.GeneralPath
class represents a geometric path constructed from straight lines and quadratic or cubic (Bézier) curves and rendered by the Java 2D ALG Library. It can be used to create line and shape objects. A GeneralPath is an object that encapsulates a sequence of line segments, curves, and move operations that draw a shape.

Creating a GeneralPath

A GeneralPath is created using the 
new GeneralPath()
constructor. This creates an empty GeneralPath object with default winding rule. To specify the winding rule, you can use the 

          
           
GeneralPath(int windingRule)
constructor, where the winding rule can be either WIND_EVEN_ODD or WIND_NON_ZERO. 

          
           
GeneralPath gp = new GeneralPath();
//alternatively
GeneralPath gp1 = new GeneralPath(GeneralPath.WIND_EVEN_ODD);

Adding Segments to a GeneralPath

To add segments to a GeneralPath object, you would use one of the provided ‘moveTo’, ‘lineTo’, ‘quadTo’ and ‘curveTo’ methods. The moveTo method tells the graphics system to move the position of the graphics pen to the specified point without drawing anything. The lineTo method draws a line segment from the current position of the graphics pen to the specified point. The quadTo method draws a Quadratic Bézier curve from the current position of the graphics pen to the specified point. The curveTo method draws a Cubic Bezier curve from the current position of the graphics pen to the specified point. 


          
           
//moving the current position to (0,0)
gp.moveTo(0, 0);
//drawing a line from current position to (10, 10)
gp.lineTo(10, 10);
//drawing a Quadratic Bézier curve from current position to (0, 20)
gp.quadTo(20, 10, 0, 20);
//drawing a Cubic Bézier curve from current position to (50, 30)
gp.curveTo(25, 15, 40, 20, 50, 30);

Closing a GeneralPath

A GeneralPath is a connected sequence of line and/or curve segments, which is closed automatically when a closePath method is called. There are two variants of the closePath method, one with no parameters and one with the parameter fast. Both close the current subpath and cause the plotted points to be connected to the first point in the subpath.


          
           
//closes the current subpath
gp.closePath();
//closes the current subpath but
//renders the path faster
gp.closePath(true);

Rendering a GeneralPath

Once a GeneralPath is created, it can be rendered using the 

Graphics2D#fill
or the 
Graphics2D#draw
methods. The 
Graphics2D#fill
method fills the interior of the GeneralPath with the currently set color. The 
Graphics2D#draw
method draws the outline of the GeneralPath with the currently set color. 

          
           
//fill the interior of the GeneralPath
g.fill(gp);
//draw the outline of the GeneralPath
g.draw(gp);
