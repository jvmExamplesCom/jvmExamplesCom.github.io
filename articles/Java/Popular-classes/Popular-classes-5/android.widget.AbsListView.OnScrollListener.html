What Is android.widget.AbsListView.OnScrollListener

android.widget.AbsListView.OnScrollListener is an interface in the Android SDK used to monitor the user's scrolling behavior in a ListView or GridView widget. It provides an API for tracking the scrolling state of a view, as well as for the position of the scrollable content.

Implementing android.widget.AbsListView.OnScrollListener

To implement android.widget.AbsListView.OnScrollListener, you must use the onScrollStateChanged() method. This method will be called whenever the scroll state of a ListView or GridView changes. The following code will print out the state of the scroll when it changes. 

          
           
AbsListView.OnScrollListener myScrollListener = new AbsListView.OnScrollListener() {
   @Override
   public void onScrollStateChanged(AbsListView view, int scrollState) {
      Log.d("MyScrollListener", "Scroll state changed: " + scrollState); 
   }
   // other methods omitted
}
The int parameter passed to the onScrollStateChange() method is an integer representing the scroll state. The following values are available: 

          
           
SCROLL_STATE_IDLE (0)
The view is not scrolling. 

          
           
SCROLL_STATE_TOUCH_SCROLL (1)
The user is scrolling with their finger. 

          
           
SCROLL_STATE_FLING (2)
The user has released their finger and the view is scrolling automatically.
