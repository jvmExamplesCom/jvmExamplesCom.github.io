About Java com.sun.net.httpserver.HttpExchange

Java com.sun.net.httpserver.HttpExchange is a class from the standard Java library used to create a Java-based HTTP server. It provides a framework for creating HTTP requests and responses, as well as API for processing them.

Code Examples

Here are some examples of using com.sun.net.httpserver.HttpExchange:


          
           
URL url = new URL("https://example.com");
HttpExchange exchange = new HttpExchange(url);
exchange.setMethod("GET");
exchange.send();

The above code creates an HttpExchange instance with a GET request to the given URL.


          
           
ByteArrayInputStream bais = new ByteArrayInputStream("Hello world".getBytes());
HttpExchange exchange = new HttpExchange(bais);
exchange.setMethod("POST");
exchange.send();

The above code creates an HttpExchange instance with a POST request using the provided input stream.


          
           
HttpExchange exchange = new HttpExchange();
exchange.setRequestURI("/some/path");
exchange.setMethod("PUT");
exchange.send();

The above code creates an HttpExchange instance with a PUT request to the given URI path.


          
           
exchange.addRequestHeader("Content-Type", "text/plain");

The above code adds a request header with the content type set to "text/plain".


          
           
String responseBody = exchange.getResponseBody();
System.out.println(responseBody);

The above code prints out the response body received from the HttpExchange request.

Outputting HTML

One can easily output HTML from an HttpExchange instance by sending the necessary headers and a string containing the HTML. Here is an example of a simple HTML page being sent through an HttpExchange instance:


          
           
HttpExchange exchange = new HttpExchange();
exchange.addRequestHeader("Content-Type", "text/html");
exchange.setBody("<html><body><h1>Hello World</h1></body></html>");
exchange.send();

The above code sends an HTTP request with a content-type header set to text/html and the body contains a simple HTML page.
