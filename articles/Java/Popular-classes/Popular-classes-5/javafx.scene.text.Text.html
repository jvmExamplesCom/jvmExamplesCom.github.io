JavaFX: javafx.scene.text.Text

The 

          
           
javafx.scene.text.Text
class provides a node for displaying text. It is an implementation of the Text interface and allows the output of plain text in a JavaFX application.

Creating Text Nodes

The text node can be created with a string argument, e.g. 


          
           
Text textNode = new Text("Hello World!");

This will create a text node with the text “Hello World!”.

Setting Text Color

The text color of the text node can be set with the 

setFill()
method, e.g. 

          
           
textNode.setFill(Color.RED);
This will set the text color to red.

Setting Text Font

The font of the text node can be set with the 
setFont()
method, e.g. 

          
           
textNode.setFont(Font.font("Verdana", 20));

This will set the font to Verdana and the size to 20.

Getting Text

The text of the text node can be retrieved with the 

getText()
method, e.g.: 

          
           
String text = textNode.getText();

This will return the text of the text node as a string.

Setting Text Wrap

The text wrap of the text node can be set with the 

setWrapText()
method, e.g.: 

          
           
textNode.setWrapText(true);
This will enable text wrapping for the text node.

Positioning Text

The position of the text node can be set with the 
setLayoutX()
and 
setLayoutY()
methods, e.g.: 

          
           
textNode.setLayoutX(100);
textNode.setLayoutY(200);

Adding Text To The Scene
