javax.management.ReflectionException in Java

Cause of javax.management.ReflectionException

The underlying cause of the ReflectionException is usually a coding mistake. This can occur when the code attempts to register a component with a component name that already exists, or when attempting to access an attribute or operation of a component that does not exist.
