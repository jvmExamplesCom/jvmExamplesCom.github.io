org.quartz.Trigger

Java's org.quartz.Trigger class is an interface which provides basic operations for creating, managing, and executing triggers. Triggers are used to schedule the execution of tasks or jobs in a background thread.

Creating and Configuring a Trigger

Creating a trigger can be done as follows: 

          
           
Trigger trigger = TriggerBuilder.newTrigger()
    .withIdentity("MonthlyTrigger")
    .startNow()
    .withSchedule(CronScheduleBuilder.monthlyOnDayAndHour(5, 14))
    .build();
