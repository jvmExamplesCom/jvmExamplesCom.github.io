MongoClient mongoClient = new MongoClient("localhost", 27017); 
MongoDatabase db = mongoClient.getDatabase("mydb");
This creates an instance of a MongoDatabase with the name "mydb". You can also use this constructor to create a MongoDatabase without specifying a client: 

          
           
MongoDatabase db = new MongoDatabase("mydb");
