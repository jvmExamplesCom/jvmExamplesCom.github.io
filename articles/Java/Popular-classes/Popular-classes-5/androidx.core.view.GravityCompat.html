GravityCompat in Androidx Core View

AndroidX Core view provides an excellent set of utilities for handling gravity and alignment within a View. GravityCompat is a subclass of the Gravity class which allows developers to access static constants in a consistent way across different version of Android. The GravityCompat class holds static methods that can be used to convert between compatibility gravity and native gravity constants.

Examples

// Convert the compatibility gravity constant 
// to the native API level's gravity
int nativeGravity = GravityCompat.getAbsoluteGravity(Gravity.LEFT, getLayoutDirection());

// Convert the native API level's gravity
