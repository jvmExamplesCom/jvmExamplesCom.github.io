javax.persistence.JoinTable

javax.persistence.JoinTable is an annotation used to define the entities in a relationship and how they are joined together in a relational database. It can be applied to one-to-one, one-to-many, many-to-one and many-to-many relationships. It allows for more flexible configurations of tables than what is possible with the JPA @ManyToMany annotation.

Example 1

In this example, we will create an entity called Post and another entity called Tag. We will create a many-to-many relationship between the two entities, with each post being able to have multiple tags and each tag being able to be associated with multiple posts.


          
           
@Entity
public class Post {
    // ...
    @ManyToMany
    @JoinTable(name = "post_tag",
            joinColumns = @JoinColumn(name = "post_id"),
            inverseJoinColumns = @JoinColumn(name = "tag_id"))
    private List<Tag> tags;
}
 
@Entity
public class Tag {
    // ...
    @ManyToMany(mappedBy = "tags")
    private List<Post> posts;
}
This will create a table in the database called 
post_tag
which will be used to store the relationships between posts and tags. The table will have two columns: 
post_id
and 
tag_id
. Each row in the table will represent a relationship between a post and a tag.

    private List<Group> groups;
}
 
@Entity
public class Group {
    // ...
    @ManyToMany(mappedBy = "groups")
    private List<User> users;

This will create a table in the database called 
user_group

user_id
and 
group_id
. Each row in the table will represent a relationship between a user and a group.

    @OneToMany
