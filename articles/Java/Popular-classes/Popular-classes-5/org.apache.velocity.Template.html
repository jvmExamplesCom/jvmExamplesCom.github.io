org.apache.velocity.Template

The org.apache.velocity.Template class is an important part of the Apache Velocity Template Language (VTL). This class provides a way to represent an individual VTL template. The class is used to store, parse, and render the template text content in order generate output. This article explains how to use the org.apache.velocity.Template class and provides some examples.

Creating a Template Object

A org.apache.velocity.Template object is created from a VTL template file. To create a Template object, the following code must be executed:


          
           
String pathToTemplateFile = “template.vm”;
VelocityEngine velocityEngine = new VelocityEngine();
velocityEngine.init();
Template template = velocityEngine.getTemplate(pathToTemplateFile);

This code will create a Template object from the provided string path to the template file.

Parsing a Template

Before a template can be rendered, it must be parsed. To parse a template, the following code must be executed:


          
           
template.parse();

This code will parse the template before it is rendered.

Rendering a Template

Once a template has been parsed, it can be rendered by executing the following code:


          
           
StringWriter writer = new StringWriter();
template.merge(context, writer);
String outputString = writer.toString();
System.out.println(outputString);

This code will render the template and the output will be sent to standard output.

Example

Consider the following VTL template file: 

          
           
#set($name = "World")
Hello $name!

The org.apache.velocity.Template class can be used to render this template. An example of how this template might be rendered using the org.apache.velocity.Template class is provided below:


          
           
String pathToTemplateFile = “template.vm”;
VelocityEngine velocityEngine = new VelocityEngine();
velocityEngine.init();
Template template = velocityEngine.getTemplate(pathToTemplateFile);
template.parse();

StringWriter writer = new StringWriter();
template.merge(context, writer);
String outputString = writer.toString();
System.out.println(outputString);

// Output
// Hello World!

In this example, the VTL template file is read and a Template object is created. The template is then parsed and rendered. The rendered output is then printed to standard output.

Conclusion

The org.apache.velocity.Template class provides a way to represent and render VTL templates. The class can be used to parse and render VTL templates and generate output. This article has provided an overview of how to use the org.apache.velocity.Template class and some examples.
