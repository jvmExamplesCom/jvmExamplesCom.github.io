Introduction to org.eclipse.swt.layout.GridData

The org.eclipse.swt.layout.GridData class provides support for grid-style user interfaces in SWT applications. It is an abstract class which can be used as the basis for many kinds of grid layouts. This article provides an overview of some of the functionality provided by the class.

Creating a GridData instance

The GridData class provides two constructors: one with no arguments, and another that takes an int argument representing the style. The style argument can take one or more of the following values:


          
           
SWT.BEGINNING 
SWT.CENTER 
SWT.END 
SWT.FILL 
SWT.VERTICAL 
SWT.HORIZONTAL

The default constructor creates a GridData object with style set to 


          
           
SWT.FILL

Setting width, height and anchors

The 

          
           
widthHint
and 

          
           
heightHint
methods can be used to set the desired size for a control contained in the GridData. The 

          
           
grabExcessHorizontalSpace
and 

          
           
grabExcessVerticalSpace
methods can be used to control whether extra space should be taken up by the control if it is available. The 

          
           
horizontalAlignment
and 

          
           
verticalAlignment
methods can be used to control the horizontal and vertical position of the control within the grid cell.

Example

The following example creates a composite containing a label and text field, and adds it to a GridLayout. The text field is given a width hint of 150px and is set to take up any extra horizontal space if available. The label and text field are both aligned to the beginning of their cells.


          
           
Composite parent = new Composite(shell, SWT.NONE);
GridLayout gridLayout = new GridLayout();
parent.setLayout(gridLayout);

Label label = new Label(parent, SWT.NONE);
label.setText("Name:");

GridData data = new GridData();
data.widthHint = 150;
data.grabExcessHorizontalSpace = true;
data.horizontalAlignment = SWT.BEGINNING;

Text text = new Text(parent, SWT.BORDER);
text.setLayoutData(data);
