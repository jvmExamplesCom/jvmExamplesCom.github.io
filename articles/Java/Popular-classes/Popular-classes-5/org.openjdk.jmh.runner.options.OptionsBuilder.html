Introduction

The class provides methods for specifying the configuration of the benchmark, such as the number of iterations, warmup iterations and forks, as well as registering listeners and state objects.

OptionsBuilder Basics

OptionsBuilder options = new OptionsBuilder();

Once you have the OptionsBuilder instance, you can start specifying the configuration of the benchmark. For example, you can set the number of iterations to run the benchmark with:


          
           
options.measurementIterations(10);

options.warmupIterations(2);

options.forks(3);

options.listener(MyListener.class);

Build the Options

Finally, once you have specified the configuration of the benchmark, you can build the options using the build() method:
