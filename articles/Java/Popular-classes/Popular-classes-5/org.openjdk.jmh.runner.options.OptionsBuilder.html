Introduction

The org.openjdk.jmh.runner.options.OptionsBuilder is a utility class that provides convenient ways to build the JMH benchmark options. The class provides methods for specifying the configuration of the benchmark, such as the number of iterations, warmup iterations and forks, as well as registering listeners and state objects.

OptionsBuilder Basics

OptionsBuilder options = new OptionsBuilder();

Once you have the OptionsBuilder instance, you can start specifying the configuration of the benchmark. For example, you can set the number of iterations to run the benchmark with:


          
           
options.measurementIterations(10);

To specify the number of warmup runs before each measurement iteration:


          
           
options.warmupIterations(2);

To set the number of JVM forks to be used in the benchmark:


          
           
options.forks(3);

Register Listeners and State Objects

options.listener(MyListener.class);

A state object is an object which encapsulates the state of the benchmark. For example, you can register a state object like this:


          
           
options.state(MyState.class);

Build the Options

Finally, once you have specified the configuration of the benchmark, you can build the options using the build() method:
