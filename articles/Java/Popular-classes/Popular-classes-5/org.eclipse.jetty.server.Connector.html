Introduction to org.eclipse.jetty.server.Connector

org.eclipse.jetty.server.Connector is an open-source implementation of the Java Servlet API, providing Java-based web applications with a highly configurable and extensible environment. The Connector makes it easy to configure, manage, and deploy web applications to Jetty servers, as well as enabling advanced features like clustering and dynamic content generation.

Quick Start Guide

To get started with Connector, you need to create a Jetty server instance with the Connector. You can do this in code as shown below:


          
           
JettyServer server = new JettyServer();
Connector connector = new Connector("localhost", 8080);
server.setConnector(connector);
server.start();

This will start the Jetty server on port 8080 and will accept connections from localhost. You can also configure the Connector's parameters, like the listener queue size or the maximum connection count, by passing them to the connector constructor like this:


          
           
Connector connector = new Connector("localhost", 8080, 
                                    100, 200);

The first number is the listener queue size, while the second number is the maximum connection count.

Configuration Options

The Connector provides several configuration options that can be used to customize the way it works. These options can be set either in code or in the Jetty configuration file.

In code, you can set the options directly on the Connector instance:


          
           
Connector connector = new Connector("localhost", 8080);
connector.setMaxIdleTime(1000);
connector.setAcceptQueueSize(20);
server.setConnector(connector);

In the configuration file, you can add XML elements like this:


          
           
<Configure id="Server" class="org.eclipse.jetty.server.Server">
  <Call name="addConnector">
    <Arg>
      <New class="org.eclipse.jetty.server.Connector">
        <Set name="port">8080</Set>
        <Set name="maxIdleTime">1000</Set>
        <Set name="acceptQueueSize">20</Set>
      </New>
    </Arg>
  </Call>
</Configure>

The available configuration options are listed in the table below:

Option Name Description
port The port number to listen on.
maxIdleTime The maximum idle time (in milliseconds) allowed for a connection.
acceptQueueSize The size of the listener queue.

Conclusion

The org.eclipse.jetty.server.Connector is a powerful tool for managing and deploying web applications to Jetty servers. With its wide range of configuration options, it makes it easy to customize the way your applications run.
