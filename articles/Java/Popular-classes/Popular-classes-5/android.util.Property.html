android.util.Property

The 

          
           
android.util.Property
class enables better integration between custom views and their animation framework. It helps to provide an interface which allows the view to read and write to a property; which will then be animated in order to perform the desired effect. An example of this is the translation of an image from one point to another.

Creating a Property

Creating a property requires extending 

          
           
android.util.Property
in order to create objects that can be used with various types of animations. An example implementation may look like this: 

          
           
public class CustomProperty extends Property<MyObject, Float> {

    public CustomProperty(Class<Float> type, String name) {
        super(type, name);
    }

    @Override
    public void set(MyObject object, Float value) {
        object.setValue(value);
    }

    @Override
    public Float get(MyObject object) {
        return object.getValue();
    }
}
In the above code sample, you are essentially creating your own object (represented by 

          
           
MyObject
) that can be passed into the animation framework. The value of the 

          
           
MyObject
is a float, so it sets and reads the value as floats. This can be changed, however, to accommodate any type of data. Once you have created the property, it is ready to be used with various types of animations. For instance, the following code will apply a custom property to an ObjectAnimator: 

          
           
ObjectAnimator animator = ObjectAnimator.ofFloat(myObject, new CustomProperty(Float.class, "test"), 0, 100);
animator.setDuration(250);
animator.start();
The above code can be used to animate the property of 

          
           
MyObject
from 0 to 100 over a duration of 250 milliseconds. Note that the call to 

          
           
ObjectAnimator.ofFloat(...)
must specify the type of property being animated - in this case, a float.

Using Multiple Properties

ObjectAnimator

Property<MyObject, Float>
, where each entry in the array is a different property. The following example creates an array of two properties: 

          
           
Property<MyObject, Float>[] properties = new Property[2];
properties[0] = new CustomProperty(Float.class, "test1");
properties[1] = new CustomProperty(Float.class, "test2");

ObjectAnimator

ObjectAnimator animator = ObjectAnimator.ofFloat(myObject, properties, new float[]{0, 100});

        super(propertyName);
        // Compute the sinusoid values

        for (int i = 0; i <= 10; i++) {
            float angle = (float) Math.PI * i / 10;

        // Create the animation
        setFloatValues(values);
    }
}

MyObject
