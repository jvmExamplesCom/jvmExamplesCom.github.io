Introduction to MongoRepository

MongoRepository is an interface in the org.springframework.data.mongodb.repository package of the Spring Data MongoDB project. It is used to create query methods that can access data stored in MongoDB without having to write any MongoDB-specific code.

Creating a MongoRepository

To create a MongoRepository, you need to create a class that implements the MongoRepository interface and define the methods you want to use. Let's look at an example:


          
           
public interface UserRepository extends MongoRepository<User, Long> {
    User findByUsername(String username);
}
This repository defines a method 
findByUsername
that searches for users by their username. The 
User
class is an entity representing a user with two fields: 
username
and 
password
. The 
Long
parameter is the type of the Id field in the 
User
class.

Querying the Repository

Once you have created the repository, you can use it to query for data. Below is an example of how to retrieve a user with the username 

john
: 

          
           
User user = userRepository.findByUsername("john");

The repository will generate an equivalent MongoDB query and execute it against the database. The generated query looks like this:


          
           
db.users.find({"username": "john"})

Constructing Queries With Parameters

The MongoRepository interface also supports creating queries with parameters. This includes support for the following operators:

  • $gt
    — Greater than
  • $gte
    — Greater than or equal to
  • $lt
    — Less than
  • $lte
    — Less than or equal to
  • $in
    — In
  • $ne
    — Not equal to
  • $nin
    — Not in

For example, you can query for users with a username greater than 

john
as follows: 

          
           
List<User> users = userRepository.findByUsernameGreaterThan("john");
This generates the following query: 

          
           
db.users.find({"username": {$gt: "john"}})

Sorting and Pagination

MongoRepository supports sorting and pagination out of the box. To sort the query results, you can use the 

sort
parameter. This parameter takes a list of fields to sort by and an optional order (either ascending or descending). For example: 

          
           
List<User> users = userRepository.findByUsernameGreaterThan("john", Sort.by("username").descending());
This generates the following query: 

          
           
db.users.find({"username": {$gt: "john"}}).sort({"username": -1})
Pagination is also supported via the 
limit
and 
skip
parameters. These parameters can be used to limit the results returned and to skip a certain number of results. For example, if you wanted to return only the first 10 results, you could do the following: 

          
           
List<User> users = userRepository.findByUsernameGreaterThan("john", PageRequest.of(0, 10));
This generates the following query: 

          
           
db.users.find({"username": {$gt: "john"}}).limit(10).skip(0)

Conclusion

MongoRepository is a great way to access data stored in MongoDB without having to write any MongoDB-specific code. It supports a variety of query operations and provides an easy way to sort and paginate query results. Although the interface may seem simple, it is incredibly powerful.
