Introduction to javax.imageio.ImageWriter

The javax.imageio.ImageWriter class is part of the Java Image I/O API, designed to provide a simple way to write images from Java applications. An ImageWriter object utilizes an underlying image stream, called a Destination, to write data to an image format specified by a FormatName. In its simplest form an ImageWriter takes a RenderedImage as input and writes it to the given Destination.

Using the ImageWriter Class

Using the ImageWriter class is relatively straightforward. The first step is to create an instance of the ImageWriter with the desired FormatName, either through the ImageIO.getImageWritersByFormatName method or by providing a File object to the ImageIO.createImageWriterByFormatName method. At this point, if the given format is not supported then an IllegalArgumentException will be thrown. If the format is supported, then the ImageWriter object can be initialized by calling the setOutput method and providing the Destination object such as an ImageOutputStream created by the ImageIO.createImageOutputStream method.


          
           
// Create an instance of the ImageWriter
    ImageWriter writer = ImageIO.createImageWriterByFormatName("jpeg");

    // Initialize the ImageWriter
    writer.setOutput(ImageIO.createImageOutputStream(new File("file.jpg")));

Once the ImageWriter is initialized, the data can be written using the write method, which expects a RenderedImage. For example, if we have a BufferedImage that was created with the ImageIO.read method, then it can be written to a file like this:


          
           
BufferedImage bi = ImageIO.read(new File("file.png"));
    writer.write(bi);

Finally, the ImageWriter should be cleanly closed by calling the dispose method when finished.


          
           
writer.dispose();

Conclusion
