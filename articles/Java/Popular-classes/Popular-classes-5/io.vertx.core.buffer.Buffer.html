Buffer myBuffer = Buffer.buffer();
This will create an empty, zero-length buffer. You can also create a buffer with a specific size by passing an integer argument to the constructor, e.g. 

          
           
Buffer myBuffer = Buffer.buffer(20);

byte[] myData = { 0, 1, 2, 3 };
Buffer myBuffer = Buffer.buffer(myData);

To append a byte array to a buffer:

Reading Data from a Buffer
