android.support.v4.util.ArrayMap

The android.support.v4.util.ArrayMap is a key-value mapping data structure that is designed to be more memory efficient than a traditional HashMap, and to prevent rare collision of hash codes. This class is used as an alternative to the built-in java.util.Map for collections which contain relatively few entries. The design makes it suitable for situations in which many entries are inserted and removed, as compared to a standard HashMap, which is more suitable for a small and fixed number of entries.

Creating an ArrayMap

An ArrayMap object can be created by instantiating the ArrayMap class:


          
           
ArrayMap<String, Object> map = new ArrayMap<>();
