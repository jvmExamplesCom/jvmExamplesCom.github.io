javax.swing.AbstractButton

javax.swing.AbstractButton is part of the Java Swing GUI library and a subclass of javax.swing.JComponent. It is a superclass of all classes that implement an abstract button. An abstract button is a component that can send action events when it is clicked or when a key binding is used to activate it.

Methods

javax.swing.AbstractButton implements many methods, some of which are:

    : Sets the command string associated with this action.

    ActionListener

  • 
            
             
setMnemonic(int mnemonic)
    : Sets the mnemonic for this button.
  • doClick()
    : Programmatically perform a "click" of this button.

Here is an example of how to create a simple button using javax.swing.AbstractButton.

myButton.setActionCommand("my_action");
myButton.addActionListener(new MyActionListener());
