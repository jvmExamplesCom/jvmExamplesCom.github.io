javax.swing.WindowConstants

javax.swing.WindowConstants is an interface in the Java Swing library used for constants that represent various window states. It has several static fields, each of which represents a different state or action associated with a window. The interface also has three predefined variables: DO_NOTHING_ON_CLOSE, HIDE_ON_CLOSE and DISPOSE_ON_CLOSE.

Code Examples:


          
           
// create frame
JFrame frame = new JFrame("My Frame");

// set close operation to do nothing
frame.setDefaultCloseOperation(WindowConstants.DO_NOTHING_ON_CLOSE);

The code above creates a JFrame and sets the default close operation to do nothing.

The code above makes the frame visible.

frame.setDefaultCloseOperation(WindowConstants.DISPOSE_ON_CLOSE);

The code above sets the window operation to dispose on close, which means that when the user clicks the close button, the window will be disposed.

The code above sets the frame location relative to the null coordinates, which means it will be centered on the screen.

The code above sets the window operation to hide when the close button is pressed, which means the window will be hidden but not disposed.
