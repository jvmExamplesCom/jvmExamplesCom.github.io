org.apache.avro.Schema

Avro is a data serialization system that provides a compact binary format for data exchange between systems written in different programming languages. It uses a JSON-based schema definition language to define the data structures that can be encoded and decoded from the binary data format. org.apache.avro.Schema is the Java class that represents the Avro schema.

org.apache.avro.Schema schema = new org.apache.avro.Schema.Parser().parse(json);

org.apache.avro.Schema schema = org.apache.avro.Schema.fromString(json);
