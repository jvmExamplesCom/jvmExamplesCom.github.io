NClob is an interface in the java.sql package which provides API for working with national character large objects stored in a database. It is a sub-interface of Clob and is used to process character data with larger length exceeding the size limitation of a DataBase. It is often used in databases like Oracle, Teradata, MySQL which supports the usage of NClob.
