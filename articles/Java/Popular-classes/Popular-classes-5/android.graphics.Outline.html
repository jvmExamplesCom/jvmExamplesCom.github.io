android.graphics.Outline is a class in the Android graphics package which allows for the creation of outlines for objects. Creating an Outline

To create an outline, first new up an instance of the Outline class: 

          
           
Outline outline = new Outline();

outline.setType(Outline.RECT);
outline.setRect(0, 0, 100, 100);
