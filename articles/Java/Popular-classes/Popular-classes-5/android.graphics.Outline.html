android.graphics.Outline is a class in the Android graphics package which allows for the creation of outlines for objects. An outline can be used to create shapes, mask off portions of an image, or create a “drop shadow” effect. Creating an Outline

To create an outline, first new up an instance of the Outline class: 

          
           
Outline outline = new Outline();
Once the new instance has been created, the outline can be customized by setting the type and shape of the outline: 

          
           
outline.setType(Outline.RECT);
outline.setRect(0, 0, 100, 100);

Drawing the Outline

Once the outline has been set up, it can then be drawn on a Canvas using the draw() method. The draw() method takes two parameters: a Canvas object, and a Paint object. 

          
           
outline.draw(canvas, paint);
