Introduction to org.xml.sax.Locator

The org.xml.sax.Locator class provides information about the current position of an XML parser within a document by returning the line number, column number, and the public identifier of the XML document. The easiest way to create a Locator object is by referencing the org.xml.sax.helpers.LocatorImpl class. An example of this is shown below: 

          
           
Locator locator = new LocatorImpl();

int lineNumber = locator.getLineNumber();
