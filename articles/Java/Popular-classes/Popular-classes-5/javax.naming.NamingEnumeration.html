javax.naming.NamingEnumeration

Java naming services provide access to the naming and directory functionality of the underlying platform. The 

          
           
javax.naming.NamingEnumeration
interface provides the ability to enumerate over the bindings returned in response to a context operation. It is used by many methods in the 

          
           
javax.naming.Context
