Introduction to Java.util.logging.FileHandler

Creating a Log File with FileHandler

In the above example, we specified the log file name “my_log_file.log” and said that the maximum size of the log file should be 1024 bytes, and the number of log files in the rotation set should be 3. Once this is done, the FileHandler object can be used to write log entries to the log file.
