javax.lang.model.type.TypeKind

javax.lang.model.type.TypeKind is a Java class that represents the kind of type in model hierarchies and can be used to determine the subtype of model elements. It is part of the Java Modeling Language (JML), which allows for the specification of types and their associated methods, as well as other aspects of a type system. This class allows developers to create and manipulate type models, and provides the necessary specification for performing type-checking at compile time.
