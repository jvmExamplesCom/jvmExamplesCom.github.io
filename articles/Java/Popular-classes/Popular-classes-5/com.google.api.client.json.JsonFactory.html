Using com.google.api.client.json.JsonFactory

com.google.api.client.json.JsonFactory

Creating a JsonFactory Object

To create a 

          
           
JsonFactory
instance, simply use the 

          
           
JsonFactory.getDefaultInstance()

JsonFactory jsonFactory = JsonFactory.getDefaultInstance();

JsonFactory.fromInputStream(InputStream)

InputStream in = new FileInputStream("data.json");
JsonObject data = jsonFactory.fromInputStream(in);

JsonObject

JsonFactory.toOutputStream(JsonObject, OutputStream)
