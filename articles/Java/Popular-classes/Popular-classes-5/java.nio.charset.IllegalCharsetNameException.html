java.nio.charset.IllegalCharsetNameException

Introduction

Java.nio.charset.IllegalCharsetNameException is an exception class in Java that indicates that a given charset name is illegal or unsupported. It is a subclass of IllegalArgumentException which itself is unchecked exception, meaning it extends RuntimeException. This exception is thrown when invalid charset names are used as arguments to constructors.
