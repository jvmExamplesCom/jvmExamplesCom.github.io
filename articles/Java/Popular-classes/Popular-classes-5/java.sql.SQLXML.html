Java.sql.SQLXML

SQLXML is an interface in the Java programming language which allows applications to manipulate an XML document stored in a database using XML-related technologies such as XPath, XQuery and XSLT. It was introduced in Java SE 6 and is part of the java.sql package.

String sqlQuery = "SELECT xmldocument FROM tablename WHERE id=1";
try {

    Connection con = ...;
    
    // Get the SQLXML instance from the connection
    SQLXML xml = con.createSQLXML(); 
    
    // Execute the query and get the results
    xml.setString(sqlQuery);
    ResultSet rs = xml.getResultSet();
    
    // Iterate through the ResultSet and print out the results
    while (rs.next()) {
        String xmlDocument = rs.getString("xmldocument");

    }

    // Handle any SQL errors
}
