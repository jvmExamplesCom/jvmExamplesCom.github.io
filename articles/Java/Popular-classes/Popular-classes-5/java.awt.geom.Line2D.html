java.awt.geom.Line2D

Java.awt.geom.Line2D is a class that defines a line in 2 dimensional space. It provides methods for constructing,comparing, transforming and other operations on a line. This class is the most basic geometric shape with which all other shapes may be built.

Creating A Line2D Object

A Line2D object can be created using one of the two constructor methods available: 

          
           
Line2D line1 = new Line2D.Double(10.0, 20.0, 30.0, 40.0);
Line2D line2 = new Line2D.Float(100.0f, 200.0f, 300.0f, 400.0f);
The above code creates a line object with start coordinates (10.0, 20.0) and end coordinates (30.0, 40.0). There are two types of Line2D objects - Double and Float. The first example uses the Double constructor while the second one uses the Float constructor.

Methods Of java.awt.geom.Line2D

getP1() and getP2()

The getP1() and getP2() methods return the starting points and ending points of the line respectively. 

          
           
Point2D start = line1.getP1();
Point2D end = line1.getP2();

setLine()

This method sets the start and end points of a line. 

          
           
line1.setLine(20.0, 40.0, 40.0, 20.0);

getBounds2D()

This method returns the bounding box of the line as a Rectangle2D object. 

          
           
Rectangle2D rect = line1.getBounds2D();

contains()

The contains() method checks if the line contains a given point. It takes Point2D as an argument and returns true or false. 

          
           
boolean outcome1 = line1.contains(new Point2D.Double(25.0, 35.0));
boolean outcome2 = line2.contains(new Point2D.Double(250.0, 350.0));

intersects()

The intersects() method checks if the line intersects with another line. It takes Line2D as an argument and returns true or false. 

          
           
boolean outcome3 = line1.intersects(line2);

intersection()

The intersection() method returns the point of intersection between two lines. It takes Line2D as an argument and returns Point2D. 

          
           
Point2D point = line1.intersection(line2);

relativeCCW()

The relativeCCW() method checks the relative positions of two points with respect to the line. It takes two Point2D arguments, and returns -1, 0 and 1 based on the position of the points. 

          
           
int result = line1.relativeCCW(new Point2D.Double(15.0, 25.0), new Point2D.Double(45.0, 35.0);

Conclusion

The java.awt.geom.Line2D class provides a way to represent a line in 2 dimensional space. It consists of constructors and methods that can be used to build, compare, transform and perform other operations on a line object.
