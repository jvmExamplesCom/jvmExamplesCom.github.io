Java.lang.annotation.Repeatable

Java.lang.annotation.Repeatable is an annotation type that enables annotations to be repeated in the same context. This annotation type can be applied to another annotation type, indicating that the latter is a repeatable annotation type. A repeatable annotation type is one whose instances can be applied more than once to the same program element.
