Introduction to org.apache.maven.plugin.logging.Log

org.apache.maven.plugin.logging.Log is an interface that provides logging capabilities for Maven plugins. It allows plugins to log messages and exceptions to the console or log file, as well as presenting output in plain text or HTML format.

The Log interface can be used to log messages from within a plugin, allowing for different levels of verbosity depending on the configuration.

Using the Log Interface

To use the Log interface, obtain a reference to it from the PluginExecutionContext parameter passed to the PluginExecutionListener methods. Then call any of the public methods:


          
           
logger = context.getLog();
logger.info( "My message" );
logger.error( new Exception() );

The methods can be called at any time during the plugin execution.

Configuring Logging Levels

The logging level can be configured by setting the 


          
           
org.apache.maven.plugin.logging.level
property in the 
settings.xml
file: 

          
           
<settings>
    <profiles>
        <profile>
            <id>my-profile</id>
            <properties>
                <org.apache.maven.plugin.logging.level>DEBUG</org.apache.maven.plugin.logging.level>
            </properties>
        </profile>
    </profiles>
</settings>

Logging to File

By default, the log messages will be printed to the console, but they can also be written to a file. To configure the logging to file, set the 


          
           
org.apache.maven.plugin.logging.logfile
property in the 
settings.xml
file: 

          
           
<settings>
    <profiles>
        <profile>
            <id>my-profile</id>
            <properties>
                <org.apache.maven.plugin.logging.logfile>maven-plugin.log</org.apache.maven.plugin.logging.logfile>
            </properties>
        </profile>
    </profiles>
</settings>

If this property is set, the log messages will be printed to the specified file.

    <profiles>

Conclusion

The Log interface provides a simple yet powerful way to log messages from within Maven plugins. It can be used to log messages in different levels of verbosity and in different formats, as well as write them to files.
