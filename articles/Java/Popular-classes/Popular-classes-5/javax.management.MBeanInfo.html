javax.management.MBeanInfo

The javax.management.MBeanInfo is an interface of the Java Management Extensions (JMX) API that provides all of the information necessary to manage a specific MBean, such as its class name, description, attributes, notifications, operations and constructors.

The MBeanInfo interface has several methods which return various pieces of information about the MBean. These methods are:

getClassName()

The getClassName() method returns the class name for the MBean. The class name is a fully qualified Java class name.


          
           
String className = mbeanInfo.getClassName();
System.out.println("Class Name: " + className);

getDescription()

The getDescription() method returns a textual description of the MBean.


          
           
String description = mbeanInfo.getDescription();
System.out.println("Description: " + description);

getAttributes()

The getAttributes() method returns an array of MBeanAttributeInfo objects that describe the attributes of the MBean.


          
           
MBeanAttributeInfo[] attributes = mbeanInfo.getAttributes();
for (MBeanAttributeInfo attribute : attributes) {
    System.out.println("Attribute Name: " + attribute.getName());
    System.out.println("Description: " + attribute.getDescription());
    System.out.println("Type: " + attribute.getType());
}

getNotifications()

The getNotifications() method returns an array of MBeanNotificationInfo objects that describe the notifications emitted by the MBean.


          
           
MBeanNotificationInfo[] notifications = mbeanInfo.getNotifications();
for (MBeanNotificationInfo notification : notifications) {
    System.out.println("Notification Name: " + notification.getName());
    System.out.println("Description: " + notification.getDescription());
    System.out.println("Types: " + Arrays.toString(notification.getNotifTypes()));
}

getOperations()

The getOperations() method returns an array of MBeanOperationInfo objects that describe the operations exposed by the MBean.


          
           
MBeanOperationInfo[] operations = mbeanInfo.getOperations();
for (MBeanOperationInfo operation : operations) {
    System.out.println("Operation Name: " + operation.getName());
    System.out.println("Description: " + operation.getDescription());
    System.out.println("Impact: " + operation.getImpact());
    System.out.println("Signature: " + Arrays.toString(operation.getSignature()));
}

getConstructors()

The getConstructors() method returns an array of MBeanConstructorInfo objects that describe the constructors exposed by the MBean.


          
           
MBeanConstructorInfo[] constructors = mbeanInfo.getConstructors();
for (MBeanConstructorInfo constructor : constructors) {
    System.out.println("Constructor Name: " + constructor.getName());
