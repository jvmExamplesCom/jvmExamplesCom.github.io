javax.xml.bind.annotation.adapters.XmlAdapter

javax.xml.bind.annotation.adapters.XmlAdapter is an abstract class which provides a bridge between Java classes and XML documents. It defines methods which map between XML representations and Java objects. XmlAdapter is used when a custom type mapping is necessary. It enables you to modify the representation of a property or field in the generated XML document from the object's property or field data.

Code Examples

Here is an example of an XmlAdapter implementation:


          
           
public class PointAdapter extends XmlAdapter<Point, Point3D> {
    @Override
    public Point marshal(Point3D point3D) throws Exception {
        return new Point(point3D.getX(), point3D.getY());
    }

    @Override
    public Point3D unmarshal(Point point) throws Exception {
        return new Point3D(point.getX(), point.getY(), 0);
    }
}

Another example is an adapter for BigInteger and byte[] types:


          
           
public class BigIntegerAdapter extends XmlAdapter<byte[], BigInteger> {
    @Override
    public byte[] marshal(BigInteger val) {
        return val.toByteArray();
    }

    @Override
    public BigInteger unmarshal(byte[] val) {
        return new BigInteger(val);
    }
}

You can also use XmlAdapter to convert between a short and a long string, as shown in this example:


          
           
public class LongStringAdapter extends XmlAdapter<String, LongString> {
    @Override
    public String marshal(LongString longString) {
        if (longString == null) {
            return null;
        }
        return longString.getValue().substring(0, 10);
    }

    @Override
    public LongString unmarshal(String val) {
        return new LongString(val);
    }
}

Finally, here is an example of how to use XmlAdapter in an XML Binding class:


          
           
@XmlJavaTypeAdapter(value = PointAdapter.class)

The above annotation tells JAXB to use the PointAdapter class for the Point type.
