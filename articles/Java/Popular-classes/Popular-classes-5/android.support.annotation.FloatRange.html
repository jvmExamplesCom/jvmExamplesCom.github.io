android.support.annotation.FloatRange

android.support.annotation.FloatRange is an annotation that restricts the acceptable values of a given float number. It can be used in the declaration of fields, parameters, and return values.

Syntax


          
           
@FloatRange(from = 0.0, to = 1.0) float f;

Parameters

  • from: minimum allowable value for the float parameter.

  • to: maximum allowable value for the float parameter.

Examples

Let's consider the following interface: 

          
           
interface MyInterface {
   void setValue(@FloatRange(from = 0.0, to = 1.0) float val);
   @FloatRange(from = 0.0, to = 1.0) float getValue();
}

Try to create an implementation of this interface like this one:


          
           
public class MyClass implements MyInterface {
   private float value;

   public void setValue(float val) {
      value = val;
   }

   public float getValue() {
     return value;
   }
}
When we compile the code, we get an error: 

          
           
error: setValue(float) in MyClass&nbsp;cannot&nbsp;implement&nbsp;setValue(float) in MyInterface. incompatible types: float is not applicable for the annotation&nbsp;@FloatRange(from&nbsp;=&nbsp;0.0,&nbsp;to&nbsp;=&nbsp;1.0).
We have to modify our code as follow: 

          
           
public class MyClass implements MyInterface {
   private float value;

   public void setValue(@FloatRange(from = 0.0, to = 1.0) float val) {
      value = val;
   }

   public @FloatRange(from = 0.0, to = 1.0) float getValue() {
     return value;
   }
}
