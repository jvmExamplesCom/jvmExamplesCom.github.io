Introduction to org.springframework.boot.test.SpringApplicationConfiguration

To use org.springframework.boot.test.SpringApplicationConfiguration, you need to create a Java configuration class. The following code example shows the most basic configuration:


          
           
@RunWith(SpringJUnit4ClassRunner.class)

   // ...
