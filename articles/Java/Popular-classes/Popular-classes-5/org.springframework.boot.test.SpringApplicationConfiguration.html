Introduction to org.springframework.boot.test.SpringApplicationConfiguration

org.springframework.boot.test.SpringApplicationConfiguration is a Java configuration class that can be used to configure Spring Boot tests. To use org.springframework.boot.test.SpringApplicationConfiguration, you need to create a Java configuration class. The following code example shows the most basic configuration:


          
           
@RunWith(SpringJUnit4ClassRunner.class)
@SpringApplicationConfiguration(classes = MySpringBootApp.class)

   // ...
