Creating a DashPathEffect is accomplished by calling the DashPathEffect constructor and passing in the desired parameters. A simple example of creating a DashPathEffect with a dashed line is shown below: 

          
           
// Create a new DashPathEffect object

Combining multiple DashPathEffects

It is possible to combine multiple DashPathEffects together to achieve complex patterns and shapes. This is done by calling the Paint.setPathEffect() method and passing in the desired combination of effects. An example of this is shown below:
