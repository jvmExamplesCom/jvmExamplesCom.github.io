Overview of com.google.auto.value.AutoValue

com.google.auto.value.AutoValue is an open source library for creating immutable value classes in Java. It enables developers to easily create classes that can be used as immutable, thread-safe data containers. By using @AutoValue on a class, AutoValue will generate implementation methods and other boilerplate code necessary for the class to function properly.

  public abstract String name();
  public abstract int age();
