java.util.EmptyStackException

The java.util.EmptyStackException is a runtime exception that occurs when a method attempts to pop an empty stack. The EmptyStackException is a class of the Java collection framework and extends a RuntimeException. It is thrown when a method has been called on an empty stack, typically when a pop() or peek() operation is performed on an empty stack.

Example 1

In this example, we create an empty stack and call the peek() method on it: 

          
           
import java.util.*;
 
public class EmptyStackExceptionTest
{
    public static void main(String[] args)
    {
        Stack<Object> stack = new Stack<Object>();
 
        try
        {
            Object element = stack.peek();
        }
        catch (EmptyStackException e)
        {
            System.out.println("Caught the EmptyStackException");
        }
    }
}
When you execute the program, it produces the following output: 

          
           
Caught the EmptyStackException
