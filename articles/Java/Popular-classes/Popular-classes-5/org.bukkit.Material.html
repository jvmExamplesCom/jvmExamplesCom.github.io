The org.bukkit.Material is a class that represents an item in the game, allowing developers to easily manipulate the item in their code.

Code Examples

The following code examples will demonstrate how to get started using the org.bukkit.Material class.


          
           
Material material = Material.getMaterial("DIAMOND");

This code will retrieve a material object for the item with the name "DIAMOND". 


          
           
Material material = Material.IRON_INGOT;

This code will retrieve a material object for the item "Iron Ingot".


          
           
String materialName = material.name();

This code will retrieve the string representation of the material (e.g. "DIAMOND").


          
           
int materialId = material.getId();

This code will retrieve the numerical id of the material (e.g. "264" for "DIAMOND").


          
           
boolean isTool = material.isTool();

This code will check whether or not the material is considered a tool (e.g. "true" for "DIAMOND_PICKAXE").

This code will retrieve an array of all available material objects.
