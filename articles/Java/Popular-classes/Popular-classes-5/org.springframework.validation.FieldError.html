Introduction to org.springframework.validation.FieldError

org.springframework.validation.FieldError is a class from the Spring Framework that allows for rapid and easy data validation. It is designed to make it easier for developers to validate user input, ensuring that only valid information is accepted in an application.

How Does FieldError Work?

FieldError works by comparing user input against a predefined set of rules, called validation constraints. If the input meets all the rules, then it is considered valid and no errors are reported. If there are any errors, then FieldError will return a list of objects which contain information about the errors that it encountered. These objects will report what type of error occurred (such as a missing field), the field that was affected, and the message that needs to be displayed to the user.

Using FieldError

FieldError can be used in conjuction with other validation frameworks, such as JSR-303 or Apache Bean Validation. It can be used to validate user input on both the client-side and the server-side. In order to use FieldError, an application must have the Spring framework on its classpath.

Code Examples

The following code example shows how to use FieldError to validate a form. It uses the JSR-303 validation framework to check whether the fields are filled out correctly. If any of the fields fail the validation check, then FieldError will add a new error object to the list of errors.


          
           
@Valid
public class Form {
  @NotNull
  private String name;
  @NotNull
  private String email;

  public void validate(Errors errors) {
    ValidationUtils.rejectIfEmpty(errors, "name", "name.empty");
    ValidationUtils.rejectIfEmpty(errors, "email", "email.empty");
  }
}

public class MyController {
  public void handleFormSubmit(@Valid Form myForm, Errors errors) {
    if (errors.hasErrors()) {
      for (FieldError error : errors.getFieldErrors()) {
        System.out.println("Field " + error.getField() + ": " + error.getDefaultMessage());
      }
    } else {
      // Save the form
    }
  }
}
