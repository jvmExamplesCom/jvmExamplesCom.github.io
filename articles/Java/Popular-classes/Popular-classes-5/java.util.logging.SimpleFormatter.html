java.util.logging.SimpleFormatter

The 

          
           
java.util.logging.SimpleFormatter
class provides a simple formatter for formatting log records into a single line of text. It can be used to quickly format log records in an efficient manner. The SimpleFormatter can be used in a variety of situations, from creating reports to debugging.

SimpleFormatter Features

  • Can format log records into a single line of text.
  • The log entry is printed in a certain format.
  • Can be used for debugging or creating reports.
  • Log messages can be customized.

SimpleFormatter Usage

Before using the SimpleFormatter, you need to do the following:

  1. Create a Logger object.
  2. Set the level of logging.
  3. Set the Formatter for the Logger.
  4. Add the Handler for the Logger.

The following code snippet demonstrates how to create a SimpleFormatter:


          
           
Logger logger = Logger.getLogger("MyLogger");
logger.setLevel(Level.ALL);
Formatter formatter = new SimpleFormatter();
Handler handler = new ConsoleHandler();
handler.setFormatter(formatter);
logger.addHandler(handler);

The following code demonstrates how to log a message using the SimpleFormatter:


          
           
logger.log(Level.INFO, "This is an informational message");
The output of the above code will look like this: 

          
           
INFO: MyLogger: This is an informational message

The SimpleFormatter can be customized by making use of certain properties. Some of the properties that can be used for customizing the SimpleFormatter are:

  • 
            
             
java.util.logging.SimpleFormatter.format
    – This property allows the user to set the format of the log records.
  • 
            
             
java.util.logging.SimpleFormatter.dateFormat
    – This property allows the user to set the date format of the log records.
  • 
            
             
java.util.logging.SimpleFormatter.timeStamp
    – This property allows the user to set whether the log records include timestamp information.

The following code snippet demonstrates how to customize the SimpleFormatter using these properties:


          
           
Logger logger = Logger.getLogger("MyLogger");
logger.setLevel(Level.ALL);
Formatter formatter = new SimpleFormatter();
formatter.setProperty("java.util.logging.SimpleFormatter.format", "[%1$tF %1$tT] [%4$s] %5$s %n");
