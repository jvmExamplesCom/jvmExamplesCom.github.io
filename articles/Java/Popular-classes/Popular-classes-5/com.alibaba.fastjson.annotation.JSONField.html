Introduction to com.alibaba.fastjson.annotation.JSONField

com.alibaba.fastjson.annotation.JSONField is a Java annotation used to separate JSON fields from Java class fields. It is part of the Fastjson library, an open-source project developed and maintained by Alibaba Group. This annotation can be used to control the mapping of field names and their order in the generated JSON output. It also offers several useful features like allowing the user to exclude some fields from the output and control data format conversion. It is important to note that this annotation is only supported in Fastjson and will not work if used with other JSON libraries such as Gson or Jackson. In this article, we'll look at how to use JSONField and explore its various features.

Using com.alibaba.fastjson.annotation.JSONField

To illustrate how to use the JSONField annotation, let's take a simple example of a Person class with two fields: name and age. We'll use the annotation to control the mapping of these fields in the JSON output. 


          
           
public class Person {
    private String name;
    private int age;

    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }

    @JSONField(name = "full_name")
    public String getName(){
        return this.name;
    }

    @JSONField(name = "years")

        return this.age;
    }
}
