ServerEndpoint

@ServerEndpoint

You can also use the "@OnMessage" annotation to specify a method that will be called when a message is received from the client. Code Examples

ServerEndpoint
subclass. This example creates a simple websocket endpoint which prints out a message when a connection is established and another message when the connection is closed.
