Introduction to android.support.design.widget.TabLayout

The android.support.design.widget.TabLayout is a powerful widget in the Android Support Library which provides an interface for users to navigate between tabs, and make it easier to switch between multiple screens or views. It is mainly used to organize information into separate tabs, and to navigate between these tabs. This widget is designed to be used with the ViewPager , and it allows users to easily and quickly switch between different tabs.

Using the TabLayout Widget

Using the TabLayout widget requires the following steps:
  1. Set up your 
    ViewPager
    that you want to associate with the TabLayout.
  2. Create a 
    TabLayout
    widget and add it to your layout.
  3. Link the 
    TabLayout
    to the 
    ViewPager
    by calling 
    
            
             
setupWithViewPager()
    .
  4. Setup your TabTitle as per your needs using 
    addTab()
    or 
    setText()
    .
  5. Finally, use 
    
            
             
TabLayout.OnTabSelectedListener()
    and the selected tab name to detect selection changes.

Example Code

In the following example, we will create a simple TabLayout, and link it to a ViewPager that contains two pages. 

          
           
// Create a ViewPager 
ViewPager viewPager = (ViewPager) findViewById(R.id.viewpager);
// Set the adapter on the ViewPager
viewPager.setAdapter(new MyFragmentPagerAdapter(getSupportFragmentManager()));

// Find the TabLayout 
TabLayout tabLayout = (TabLayout) findViewById(R.id.tabs);
// Link the TabLayout to the ViewPager 
tabLayout.setupWithViewPager(viewPager);

// Set titles for the TabLayout 
tabLayout.addTab(tabLayout.newTab().setText("Page 1"));
tabLayout.addTab(tabLayout.newTab().setText("Page 2"));

// Add a TabSelectedListener
tabLayout.setOnTabSelectedListener(new TabLayout.OnTabSelectedListener() {
    @Override
    public void onTabSelected(TabLayout.Tab tab) {
        // Handle tab selection
    }

    @Override
    public void onTabUnselected(TabLayout.Tab tab) {
        // Handle tab unselected
    }

    @Override
    public void onTabReselected(TabLayout.Tab tab) {
        // Handle tab reselected
    }
});

Conclusion

The android.support.design.widget.TabLayout is a very useful widget that can be used to provide navigation between different views in an Android application. It is easy to setup and use, and it is also very customizable, allowing developers to change the look and feel of the tabs. By using the TabSelectedListener, it is also possible to handle different actions based on which tab is selected.
