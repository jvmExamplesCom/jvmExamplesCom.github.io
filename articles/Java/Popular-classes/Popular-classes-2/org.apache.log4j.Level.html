org.apache.log4j.Level

The Level class defines a set of standard logging levels that can be used to control logging output. The logging Level objects are ordered and are specified by ordered integers. Enabling logging at a given level also enables logging at all higher levels.

Basic Syntax


          
           
import org.apache.log4j.Level;

// Create a Level object.
Level level = Level.INFO;

// Check if the given level is enabled.
if (level.isEnabled()) {
    // Perform logging.
    Logger.log(level, "Logging message.");
}

Level Constants

The Level class defines following constants. These are usually used for indicating log level.

  • OFF: No logging.
  • FATAL: Highest severity level.
  • ERROR: Error level.
  • WARN: Warning level.
  • INFO: Information level.
  • DEBUG: Debug level.
  • TRACE: Lowest severity level.
  • ALL: Enable all levels.

Below example demonstrates use of Level class and its methods.

    // Perform logging using INFO level.
