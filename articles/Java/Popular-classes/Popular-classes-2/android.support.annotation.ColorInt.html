Understanding android.support.annotation.ColorInt

The android.support.annotation.ColorInt annotation indicates that a field or parameter can be used as a color integer, not just an ordinary integer. This annotation is only applicable to fields, parameters, and return values on methods and constructors.

How It Works

@ColorRes

    return R.color.red;
