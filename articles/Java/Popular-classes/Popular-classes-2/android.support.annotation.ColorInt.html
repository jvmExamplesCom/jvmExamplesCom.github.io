Understanding android.support.annotation.ColorInt

The android.support.annotation.ColorInt annotation indicates that a field or parameter can be used as a color integer, not just an ordinary integer. This annotation is only applicable to fields, parameters, and return values on methods and constructors.

How It Works

This annotation, when applied to a field, parameter, or return value, will automatically resolve any color resource and convert it into an integer suitable for use in the Android SDK. To specify a color resource, you must use the 

@ColorRes
annotation. For example, consider this method:

    return R.color.red;
}

int getColor() {
    return R.color.red;
