Introduction to Java Cache

Java Cache is a library implemented by Google Guava to allow the creation of in-memory caches for efficient application performance. It provides several features like time based expiration, memory based eviction, concurrent access control and various strategies to build a cache. This article will cover the fundamentals of creating and using a Java Cache from scratch.

Creating a Java Cache

To create a Java Cache, you must first create an instance of the 

CacheBuilder
class. This will provide you with the set of parameters necessary to create a Java Cache. 

          
           
CacheBuilder builder = CacheBuilder.newBuilder();
builder.maximumSize(1000);
builder.expireAfterAccess(10, TimeUnit.MINUTES);
Cache<String, String> myCache = builder.build();

In the example above, we have created a new Java Cache that can contain up to 1000 elements, and entries in the cache will expire after 10 minutes of not being accessed.

Putting Entries into Cache

Once a Cache has been created, you can start putting entries into it. To do this, you must use the 

put()
method on the Cache instance. 

          
           
myCache.put("key1", "value1");
myCache.put("key2", "value2");
The 
put()
method will insert the given key-value pair into the Cache, replacing any existing value with the same key.

Retrieving Values from Cache

You can retrieve values from the Cache by typing the key associated with it. To do this, you must use the 

getIfPresent()
method. 

          
           
String value1 = myCache.getIfPresent("key1"); // value1 = "value1"
String value2 = myCache.getIfPresent("key2"); // value2 = "value2"

If there is no entry associated with the specified key, or if the entry has expired, then the 

getIfPresent()
method will return 
null
.

Removing Entries from Cache

You can remove entries from the Cache by using the 

invalidate()
method. 

          
           
myCache.invalidate("key1");
myCache.invalidate("key2");

This method will remove the specified entry from the Cache, if it exists.

Final Thoughts

Java Caches are an invaluable tool to help you improve the performance of your applications. This article has covered some of the basics of how to create and use a Java Cache, but there are many more features available. For more detailed information, please refer to the official Java Cache documentation.
