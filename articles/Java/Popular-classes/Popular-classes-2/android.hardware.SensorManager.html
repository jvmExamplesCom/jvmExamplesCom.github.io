Android Hardware SensorManager

Android's SensorManager is a package that provides access to device's sensors. It is part of the Android core library and is used to detect various physical properties of the device such as its orientation, acceleration, etc. The SensorManager allows apps to access information from different types of sensors in the device such as accelerometers, gyroscopes, light sensors and other types of sensors.
