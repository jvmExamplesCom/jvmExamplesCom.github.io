javax.ws.rs.POST

Introduction

javax.ws.rs.POST is an annotation for the Java API for RESTful Web Services (JAX-RS) 2.0 that is used to designate a resource method as accepting only HTTP POST requests. It indicates that the annotated method responds to HTTP POST requests and thus can receive data from the client. JAX-RS enables developers to build RESTful web services by enabling the developer to annotate the Java source code with additional metadata, which the JAX-RS runtime interprets in order to build and execute the service.

Usage

The javax.ws.rs.POST annotation is targeted to be used on resource methods within classes annotated with the javax.ws.rs.Path annotation. The POST annotation informs the JAX-RS runtime that the method responds to and accepts HTTP POST requests. An example usage of the POST annotation is given below: 

          
           
@Path("/books")
public class BooksResource {

    @POST
    @Consumes(MediaType.APPLICATION_JSON)
    public Response addBook(Book book) {
        // code to add the book...
    }
}
In the above example, the @POST annotation is used to designate the addBook() method as responding to and accepting HTTP POST requests. This makes it possible for the client to send data to the server in JSON format, which is specified in the @Consumes annotation. The addBook() method is then able to use this data to add the book to the resource in question.

Example Output

When the POST request is successfully processed, the method returns a HTTP status code of 200 (OK) along with any response payload. An example of a successful POST request can be seen below: 

          
           
HTTP/1.1 200 OK
Content-Type: application/json

{
   "message": "Book added successfully"
}

Conclusion

The javax.ws.rs.POST annotation is an important part of the Java API for RESTful Web Services (JAX-RS) 2.0 and is used to designate a resource method as accepting only HTTP POST requests. The annotation informs the JAX-RS runtime that the method responds to and accepts HTTP POST requests and thus can receive data from the client. When used in combination with other JAX-RS annotations, the POST annotation can be used to facilitate the building of RESTful web services.
