Introduction to javax.servlet.ServletContextListener

ServletContextListener is an interface in the javax.servlet package that is part of the Java Servlet API. This interface provides a way for developers to create listener classes that can be notified when the ServletContext is initialized, destroyed, or attribute values are changed. This can be useful if you need to perform any type of setup or shutdown operations before and after the web application is started, as well as for listening for changes to attributes in the ServletContext.

Sample Code

The following code demonstrates how to create a ServletContextListener class and implement the methods.


          
           
public class MyServletContextListener implements ServletContextListener { 

    @Override 
    public void contextInitialized(ServletContextEvent event) { 
        // Initialization code goes here 
    } 

    @Override 
    public void contextDestroyed(ServletContextEvent event) { 
        // Cleanup code goes here 
    } 
}

The following code snippet shows how to register the ServletContextListener class with the web.xml file:


          
           
<listener> 
    <listener-class>MyServletContextListener</listener-class> 
</listener>

The following code snippet shows how to access the ServletContext from the ServletContextListener class:


          
           
ServletContext servletContext = servletContextEvent.getServletContext();

One example of when a ServletContextListener can be used is to set up a database connection before the web application is started. In this example, we will create a method called setupDbConnection() which will handle the setup of the db connection and make it available to the web application.


          
           
public void setupDbConnection(ServletContext context) { 
    String dbUrl = ...; 
    String dbUser = ...; 
    String dbPassword = ...; 

    // Create connection with DB and store in the context 
    Connection conn = ...; 
    context.setAttribute("dbConnection", conn); 
}

We can then call this method in the contextInitialized() method of our ServletContextListener class:


          
           
@Override 
public void contextInitialized(ServletContextEvent event) { 
    ServletContext context = event.getServletContext(); 
    setupDbConnection(context);

Now our db connection will be available to all Servlets and other components in our web application. We can now access this connection by calling the getAttribute() method on the ServletContext:
