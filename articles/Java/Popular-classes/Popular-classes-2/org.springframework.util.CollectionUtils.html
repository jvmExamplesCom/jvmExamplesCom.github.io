org.springframework.util.CollectionUtils

org.springframework.util.CollectionUtils is a utility class for working with collections such as Lists, Sets, Maps, and Arrays. It provides various static methods for manipulating collections, including sorting, transforming, filtering, and so on. It is part of the Spring Framework.

Sorting Collections

CollectionUtils provides several static methods for sorting collections, such as sort(), sortByProperty(), and sortByProperties(). The 


          
           
sort()
method takes a list and a comparator and sorts the list using the given comparator. 

          
           
List<Integer> numbers = Arrays.asList(3, 5, 1, 4);
CollectionUtils.sort(numbers, Comparator.naturalOrder());
System.out.println(numbers);
// Output: [1, 3, 4, 5]
The 

          
           
sortByProperty()
method takes a list of objects, a property name and a comparator, and sorts the list based on the values of the specified property using the given comparator. 

          
           
List<Person> people = new ArrayList<>();
people.add(new Person("John", 30));
people.add(new Person("Alice", 20));
people.add(new Person("Bob", 25));
CollectionUtils.sortByProperty(people, "age", Comparator.naturalOrder());
System.out.println(people);
// Output: [Alice (20), Bob (25), John (30)]
The 

          
           
sortByProperties()
method is similar to sortByProperty() but it can take multiple properties and sort by multiple properties simultaneously. 

          
           
List<Person> people = new ArrayList<>();
people.add(new Person("John", 30, "NYC"));
people.add(new Person("Alice", 20, "Chicago"));
people.add(new Person("Bob", 25, "NYC"));
CollectionUtils.sortByProperties(people, "city", "age", Comparator.naturalOrder());
System.out.println(people);
// Output: [Alice (20, Chicago), Bob (25, NYC), John (30, NYC)]

Filtering Collections

CollectionUtils provides several static methods for filtering collections, such as select(), selectRejected(), selectUnique(), and reject(). The 


          
           
select()
method takes a collection and a predicate, and returns a new collection containing only the elements that match the given predicate. 

          
           
List<Integer> numbers = Arrays.asList(1, 2, 3, 4, 5, 6);
Collection<Integer> evenNumbers = CollectionUtils.select(numbers, n -> n % 2 == 0);
System.out.println(evenNumbers);
// Output: [2, 4, 6]
The 

          
           
selectRejected()
method is the opposite of select(). It takes a collection and a predicate, and returns a new collection containing only the elements that do not match the given predicate. 

          
           
List<Integer> numbers = Arrays.asList(1, 2, 3, 4, 5, 6);
Collection<Integer> oddNumbers = CollectionUtils.selectRejected(numbers, n -> n % 2 == 0);
System.out.println(oddNumbers);
// Output: [1, 3, 5]
The 

          
           
selectUnique()
method takes a collection and a predicate, and returns a new collection containing only the elements that are unique according to the given predicate. 

          
           
List<String> strings = Arrays.asList("foo", "bar", "baz", "foo", "bar");
Collection<String> uniqueStrings = CollectionUtils.selectUnique(strings, s -> s.length());
System.out.println(uniqueStrings);
// Output: [foo, bar]
The 

          
           
reject()
method is the opposite of selectUnique(). It takes a collection and a predicate, and returns a new collection containing only the elements that are not unique according to the given predicate. 

          
           
List<String> strings = Arrays.asList("foo", "bar", "baz", "foo", "bar");
Collection<String> nonUniqueStrings = CollectionUtils.reject(strings, s -> s.length());
System.out.println(nonUniqueStrings);
// Output: [baz]

Transformations

CollectionUtils provides many static methods for transforming collections, such as transform(), map(), collect(), and convert(). The 


          
           
transform()
method takes a collection, a transformation function and returns a new collection with all elements transformed by the given function. 

          
           
List<String> strings = Arrays.asList("foo", "bar", "baz");
Collection<String> transformedStrings = CollectionUtils.transform(strings, s -> s.toUpperCase()) ;
System.out.println(transformedStrings);
// Output: [FOO, BAR, BAZ]
The 

          
           
map()
method is similar to transform but it also allows you to specify the type of the returned collection. 

          
           
List<String> strings = Arrays.asList("foo", "bar", "baz");
Set<String> mappedStrings = CollectionUtils.map(strings, String::toUpperCase,
    new HashSet<>());
System.out.println(mappedStrings);
// Output: [FOO, BAR, BAZ]
The 

          
           
collect()
method takes a collection, a collector and returns the result of applying the collector to the given collection. 

          
           
List<String> strings = Arrays.asList("foo", "bar", "baz");
String joinedString = CollectionUtils.collect(strings, Collectors.joining(","));
System.out.println(joinedString);
// Output: foo,bar,baz
The 

          
           
convert()
method takes a collection and a converter, and returns a new collection with all elements converted by the given converter. 

          
           
List<String> strings = Arrays.asList("foo", "bar", "baz");
Collection<Integer> lengths = CollectionUtils.convert(strings, String::length);
System.out.println(lengths);
// Output: [3, 3, 3]
