android.widget.RadioButton

The android.widget.RadioButton class is a subclass of CompoundButton and provides the same basic functionality as a button, with the addition of a radio-style selection of multiple options. RadioButton objects can be configured to display text or an image, or a combination of both.

RadioButton myRadioButton = new RadioButton(this);
