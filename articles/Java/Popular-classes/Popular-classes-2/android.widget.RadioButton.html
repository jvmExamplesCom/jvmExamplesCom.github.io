android.widget.RadioButton

The android.widget.RadioButton class is a subclass of CompoundButton and provides the same basic functionality as a button, with the addition of a radio-style selection of multiple options. The RadioButton class allows the user to make a single selection from a set of two or more pre-configured options. RadioButton objects can be configured to display text or an image, or a combination of both.

Creating a Radio Button

To create a RadioButton you must use the constructor RadioButton() like so:


          
           
RadioButton myRadioButton = new RadioButton(this);

The constructor takes a Context object as its parameter.

Setting Text

myRadioButton.setText("My Radio Button");

Setting Images

You can set images for the RadioButton with the setImageResource() or setImageDrawable() methods like so:


          
           
myRadioButton.setImageResource(R.drawable.my_image);

Adding the RadioButton to the Layout

layout.addView(myRadioButton);

Handling RadioButton Clicks

myRadioButton.setOnClickListener(new OnClickListener() {

    }
