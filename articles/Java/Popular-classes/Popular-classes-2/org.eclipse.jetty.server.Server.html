org.eclipse.jetty.server.Server

Server server = new Server();

setPort()
method: 

          
           
server.setPort(8080);

  • ContextHandler
    – handles requests for web applications
  • ResourceHandler
    – handles requests for static resources
  • ServletHandler
    – handles requests for servlets

server.addHandler(handler);

Starting and Stopping the Server

server.start();

When the server is no longer needed, it can be stopped by calling the
