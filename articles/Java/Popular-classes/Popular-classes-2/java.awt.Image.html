Java.awt.Image

The java.awt.Image class is used to represent graphical images. It is also the superclass of all classes that represent graphical images, such as BufferedImage and ImageIO. It is an abstract class and provides a number of methods for loading images and manipulating them.

Creating an Image Object

An instance of the java.awt.Image class can be created by either loading an image from a file or creating an empty image. The first step is to create an instance of the Image class:


          
           
Image image = new Image();

Loading an Image from a File

Once an instance of the Image class has been created, an image can be loaded from a file using the 


          
           
ImageIO.read()
method. This method takes a File object as an argument and returns an instance of the Image class: 

          
           
File imageFile = new File("image.png");
Image image = ImageIO.read(imageFile);

The file format of the image must be supported by the Java platform. Some of the most common image formats are PNG, JPEG, and GIF.

Creating an Empty Image

An empty image can be created by passing a width, height, and type (color model) to the Image constructor. For example, to create an empty 100x100 RGB (Red-Green-Blue) image, the following code can be used:


          
           
int width = 100;
int height = 100;
int type = BufferedImage.TYPE_INT_RGB;
BufferedImage image = new BufferedImage(width, height, type);

Saving an Image

Once an image has been created or loaded, it can be saved in a variety of formats using the 


          
           
ImageIO.write()
method. This method takes an image, a file format, and a file name as arguments and saves the image to disk in the specified file format: 

          
           
File outputFile = new File("image.png");
String format = "PNG";
ImageIO.write(image, format, outputFile);

Manipulating an Image

Once an image has been loaded or created, it can be manipulated in a variety of ways. For example, it can be resized, cropped, scaled, rotated, flipped, etc. The java.awt.Graphics class provides a number of methods that can be used to manipulate images. For example, the following code can be used to resize an image:


          
           
int width = 200;
int height = 200;
BufferedImage resizedImage = new BufferedImage(width, height, BufferedImage.TYPE_INT_ARGB);
Graphics2D g2d = resizedImage.createGraphics();
g2d.drawImage(image, 0, 0, width, height, null);
g2d.dispose();

The above code creates a new image with the specified size, then uses the drawImage() method to copy the original image into the resized image and finally calls the dispose() method to free up any resources associated with the Graphics object.
