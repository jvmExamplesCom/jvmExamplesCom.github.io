Android App NotificationChannel

NotificationChannel is an Android framework class used to create and manage Notification Channels. This class enables developers to deliver system notifications that are more useful to users. It provides a way to group notifications, assign importance levels to them, and provide a user interface for managing them.

Creating a NotificationChannel

Creating a NotificationChannel requires an instance of the Android Context, a CharSequence for the channel name and a description, and optionally a importance level. The importance level must be one of NotificationManager.IMPORTANCE_DEFAULT, NotificationManager.IMPORTANCE_HIGH or NotificationManager.IMPORTANCE_LOW.


          
           
NotificationChannel channel = 
  new NotificationChannel("channel_id", 
    "My Channel", 
    NotificationManager.IMPORTANCE_DEFAULT);

The channel ID must be unique within the application, and should not contain spaces or special characters.

Enabling the Channel in an Activity

Once the channel has been created, it must be enabled before notifications can be sent. This is done by getting a reference to the current NotificationManager and calling its createNotificationChannel() method with the channel object.


          
           
NotificationManager notificationManager =
  (NotificationManager) getSystemService(Context.NOTIFICATION_SERVICE);
notificationManager.createNotificationChannel(channel);

Sending Notifications

Once the channel is enabled, sending notifications is straightforward. To send a notification, create a Notification object and set the channel ID. Then call the notify() method on the NotificationManager with an ID and the Notification object.


          
           
Notification notification = new Notification();
notification.setChannelId("channel_id");

NotificationManager notificationManager =
  (NotificationManager) getSystemService(Context.NOTIFICATION_SERVICE);
notificationManager.notify(1, notification);

Updating a NotificationChannel

To update a NotificationChannel, first get a reference to the current NotificationManager and then call its updateNotificationChannel() method with the channel instance. This will update any existing fields in the NotificationChannel.


          
           
NotificationChannel updatedChannel = 
  new NotificationChannel("channel_id", 
    "My Updated Channel", 
    NotificationManager.IMPORTANCE_HIGH);

NotificationManager notificationManager =
  (NotificationManager) getSystemService(Context.NOTIFICATION_SERVICE);
notificationManager.updateNotificationChannel(updatedChannel);

Deleting a NotificationChannel

To delete a NotificationChannel, first get a reference to the current NotificationManager and then call its deleteNotificationChannel() method with the channel ID.


          
           
NotificationManager notificationManager =
  (NotificationManager) getSystemService(Context.NOTIFICATION_SERVICE);
notificationManager.deleteNotificationChannel("channel_id");
