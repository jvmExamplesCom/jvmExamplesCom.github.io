The 

          
           
com.facebook.react.ReactPackage
is the main interface for React Native that provides app developers with the necessary tools to build native UI components. It provides a way for developers to create, configure and manage the application's UI components. It also allows for configuration of certain behaviors, such as how elements should be displayed, how to interact with them, and other features.

Creating your React package is done by creating a subclass of 


          
           
com.facebook.react.ReactPackage
. The 
ReactPackage
class must implement three methods: 

          
           
createViewManagers()
, 

          
           
createNativeModules()
, and 
createJSModules()

public class MyReactPackage implements ReactPackage {
  public List<ViewManager> createViewManagers(ReactApplicationContext reactContext) {
    // Create and return a list of view managers here
  }

  public List<NativeModule> createNativeModules(
    ReactApplicationContext reactContext) {
    // Create and return a list of native modules here
  }

  public List<Class<? extends JavaScriptModule>> createJSModules() {
    // Create and return a list of JavaScript modules here
  }

Installing Your React Package

Once you have created your React package, you need to install it in your app. To do this, you need to create a class that extends 

Application

getPackages()
