javax.net.ssl.HostnameVerifier is a secure socket layer used in Java that helps to verify the hostname of a communication endpoint. Methods

verify(String hostName, SSLSession session)
and 

          
           
getAcceptedIssuers()
. The first method accepts a hostname parameter that identifies the server to be verified, and returns true if the hostname matches the one present in the server certificate. The second method returns the list of acceptable certificate issuers.
