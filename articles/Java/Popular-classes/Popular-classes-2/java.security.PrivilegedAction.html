Overview of java.security.PrivilegedAction

The java.security.PrivilegedAction class is used to provide additional privileges to code that is executed within a security context. It defines a single method, run(), which takes no arguments and returns an object result. The class also provides the doPrivileged() static method, which can be used to wrap a security-sensitive operation in a PrivilegedAction instance.

    // create a PrivilegedAction
    PrivilegedAction action = new PrivilegedAction() {
        public Object run() {
            // perform some privileged operation
            return null;
        }
    };

    // execute the action using doPrivileged
    Object result = AccessController.doPrivileged(action);
