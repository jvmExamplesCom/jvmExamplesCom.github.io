Java.time.ZoneOffset

Java.time.ZoneOffset is a time-zone offset from Greenwich/UTC, accounting for the local offset due to daylight savings time. It represents an amount of time that can be added or subtracted from LocalDateTime, LocalDate, and LocalTime objects to obtain a date or time in another time zone.
