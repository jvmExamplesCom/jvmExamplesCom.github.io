java.beans.PropertyChangeListener

The 

          
           
java.beans.PropertyChangeListener
interface is used to receive notifications of changes in properties of an object. The object in question may be a bean, or any other type of object. This interface defines methods to add and remove the listeners, and to fire a notification when a property has been changed.

Using PropertyChangeListener

In order to use the 

          
           
java.beans.PropertyChangeListener
interface, an implementation of the interface must be created. This implementation must define the 

          
           
propertyChange
method, which will handle notifications when a property is changed. Here is an example of a simple implementation: 

          
           
public class MyPropertyChangeListener implements PropertyChangeListener {

	@Override
	public void propertyChange(PropertyChangeEvent evt) {
		System.out.println("Property changed: " + evt.getPropertyName());
	}
}
This class can then be used to register a listener for a bean or other object: 

          
           
MyPropertyChangeListener listener = new MyPropertyChangeListener();
someObject.addPropertyChangeListener(listener);
When the property of the object is changed, the 

          
           
propertyChange
method will be called and the property name will be printed out.

Removing PropertyChangeListener

It is also possible to remove the listener from the object: 

          
           
someObject.removePropertyChangeListener(listener);
This will prevent the 

          
           
propertyChange
method from being called when the property of the object changes.

java.beans.PropertyChangeListener

firePropertyChange

oldValue

newValue
