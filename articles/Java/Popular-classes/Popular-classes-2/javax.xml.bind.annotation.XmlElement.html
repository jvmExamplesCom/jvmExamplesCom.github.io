javax.xml.bind.annotation.XmlElement

The javax.xml.bind.annotation.XmlElement annotation is used to specify an XML element in a Java class.

It can be used to define the name, type and position of a particular element in an XML document. The XmlElement annotation also supports various other attributes such as required, nillable and minOccurs.

Example Usage

Let's look at a basic example of the XmlElement annotation:


          
           
@XmlElement(name = "foo")
private String foo;

In this example, we're declaring an XML element named 

foo
, of type 
String
. This would result in an XML similar to the following: 

          
           
<foo>Some value</foo>

@XmlElement(name = "foo")

<foo>Some value</foo>
