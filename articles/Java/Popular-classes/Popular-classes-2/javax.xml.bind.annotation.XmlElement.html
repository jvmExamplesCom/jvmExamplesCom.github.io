The javax.xml.bind.annotation.XmlElement annotation is used to specify an XML element in a Java class.

Let's look at a basic example of the XmlElement annotation:

private String foo;

foo

String
. This would result in an XML similar to the following: 

          
           
<foo>Some value</foo>
