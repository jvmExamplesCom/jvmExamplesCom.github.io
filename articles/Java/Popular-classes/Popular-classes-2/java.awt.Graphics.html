Java.awt.Graphics

Java.awt.Graphics is an abstract class used for drawing vector graphics on a computer display. It enables developers to draw arbitrary shapes and images on the surface of a window or component. The Graphics class provides several methods for creating and manipulating graphical objects such as lines, rectangles, images, text, colors, strokes, and shapes.

Methods

Let's take a look at some of the methods of the Graphics class:

drawString()

This method draws a String object specified by a given coordinate. The syntax is: 

          
           
public abstract void drawString(String str, int x, int y)
This method requires three parameters:
  • str - The string object to be drawn.
  • x - The X coordinate at which the string should be drawn.
  • y - The Y coordinate at which the string should be drawn.

fillRect()

This method fills a rectangular area with the current color. The syntax is: 

          
           
public abstract void fillRect(int x, int y, int width, int height)
This method requires four parameters:
  • x - The X coordinate of the upper-left corner of the rectangle.
  • y - The Y coordinate of the upper-left corner of the rectangle.
  • width - The width of the rectangle.
  • height - The height of the rectangle.

drawLine()

This method draws a line between two points specified by a given coordinate. The syntax is: 

          
           
public abstract void drawLine(int x1, int y1, int x2, int y2)
This method requires four parameters:
  • x1 - The X coordinate of the start point.
  • y1 - The Y coordinate of the start point.
  • x2 - The X coordinate of the end point.
  • y2 - The Y coordinate of the end point.

drawOval()

This method draws an oval shape with the current color. The syntax is: 

          
           
public abstract void drawOval(int x, int y, int width, int height)
This method requires four parameters:
  • x - The X coordinate of the upper-left corner of the oval.
  • y - The Y coordinate of the upper-left corner of the oval.
  • width - The width of the oval.
  • height - The height of the oval.

drawImage()

This method draws an image at the specified location. The syntax is: 

          
           
public abstract boolean drawImage(Image img, int dx1, int dy1, int dx2, int dy2, int sx1, int sy1, int sx2, int sy2, ImageObserver observer)
This method requires nine parameters:
  • img - The image object to be drawn.
  • dx1 - The X coordinate of the upper-left corner of the destination.
  • dy1 - The Y coordinate of the upper-left corner of the destination.
  • dx2 - The X coordinate of the lower-right corner of the destination.
  • dy2 - The Y coordinate of the lower-right corner of the destination.
  • sx1 - The X coordinate of the upper-left corner of the source.
  • sy1 - The Y coordinate of the upper-left corner of the source.
  • sx2 - The X coordinate of the lower-right corner of the source.
  • sy2 - The Y coordinate of the lower-right corner of the source.
  • observer - The ImageObserver object.

getColor()

This method returns the current color of the graphics context. The syntax is: 

          
           
public Color getColor()
This method does not require any parameters.

Example

Let's take a look at a simple example where we draw a circle in a window. 

          
           
import java.awt.Color;
 import java.awt.Frame;
 import java.awt.Graphics;

 public class DrawCircle {
   public static void main(String[] args) {
     Frame frame = new Frame();
     frame.setSize(400, 400);
     frame.setVisible(true);

     Graphics g = frame.getGraphics();
     g.setColor(Color.RED);
     g.fillOval(100, 100, 200, 200);
   }
 }
The output of this code will be a red circle drawn on a 400×400 frame.
