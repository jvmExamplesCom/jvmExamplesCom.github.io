androidx.recyclerview.widget.RecyclerView

The RecyclerView widget is a more advanced and flexible version of ListView. This widget is a container for displaying large data sets that can be scrolled very efficiently by maintaining a limited number of views. It supports vertical scrolling and horizontal scrolling.

Create the RecyclerView

Creating a RecyclerView requires the following steps:

  1. Add the RecyclerView support library to your dependency list.
  2. Define a model class to use as the data source.
  3. Add a RecyclerView to your activity to display the items.
  4. Create a custom row layout XML file to visualize the item.
  5. Create a RecyclerView.Adapter and ViewHolder to render the item.
  6. Bind the adapter to the data source to populate the RecyclerView.
Let's look at each of these steps in more detail.

1. Add the RecyclerView Support Library to Your Dependency List

The RecyclerView support library is provided as part of the Android Support Library. To add the support library to your project, add the following line to the dependencies section of your build.gradle file:


          
           
dependencies {
    compile 'com.android.support:recyclerview-v7:23.4.0'
}

2. Define a Model Class to Use as the Data Source

A RecyclerView needs a data source from which to get its data. The data items can be of any Java type, but they must be converted into a usable format that the RecyclerView can work with. For this purpose, we will create a simple model class to represent the data items.


          
           
public class Item {
    private String title;
    private String description;

    public Item(String title, String description) {
        this.title = title;
        this.description = description;
    }

    public String getTitle() {
        return title;
    }

    public void setTitle(String title) {
        this.title = title;
    }

    public String getDescription() {
        return description;
    }

    public void setDescription(String description) {
        this.description = description;
    }
}

3. Add a RecyclerView to Your Activity to Display the Items

Next, we add the RecyclerView to our activity's layout. The layout should include a 

RecyclerView
element with an 
id
attribute and a 
layout_height
set to 
match_parent
. 

          
           
<RelativeLayout xmlns:android="http://schemas.android.com/apk/res/android"
    android:layout_width="match_parent"
    android:layout_height="match_parent">

    <androidx.recyclerview.widget.RecyclerView
        android:id="@+id/recycler_view"
        android:layout_width="match_parent"
        android:layout_height="match_parent"/>

</RelativeLayout>

4. Create a Custom Row Layout XML File to Visualize the Item

We need to create a layout for each row in the RecyclerView. This can be done by creating a layout xml file containing the views we need to show for each item.


          
           
<?xml version="1.0" encoding="utf-8"?>
<LinearLayout xmlns:android="http://schemas.android.com/apk/res/android"
    android:layout_width="match_parent"
    android:layout_height="wrap_content"
    android:orientation="vertical">

    <TextView
        android:id="@+id/title_text"
        android:layout_width="match_parent"
        android:layout_height="wrap_content"
        android:textSize="20sp" />

    <TextView
        android:id="@+id/description_text"
        android:layout_width="match_parent"
        android:layout_height="wrap_content"
        android:textSize="16sp" />

</LinearLayout>

5. Create a RecyclerView.Adapter and ViewHolder to Render the Item

We need to create a RecyclerView.Adapter to bind the data source to the RecyclerView. The adapter will also create ViewHolders to contain the views for each row in the RecyclerView.


          
           
public class ItemAdapter extends RecyclerView.Adapter<ItemAdapter.ViewHolder> {

    private List<Item> items;

    public ItemAdapter(List<Item> items) {
        this.items = items;
    }

    @Override
    public ViewHolder onCreateViewHolder(ViewGroup parent, int viewType) {
        View view = LayoutInflater.from(parent.getContext())
                .inflate(R.layout.item_row, parent, false);

        return new ViewHolder(view);
    }

    @Override
    public void onBindViewHolder(ViewHolder holder, int position) {
        Item item = items.get(position);
        holder.titleText.setText(item.getTitle());
        holder.descriptionText.setText(item.getDescription());
    }

    @Override
    public int getItemCount() {
        return items.size();
    }

    class ViewHolder extends RecyclerView.ViewHolder {
        TextView titleText;
        TextView descriptionText;

        ViewHolder(View itemView) {
            super(itemView);
            titleText = itemView.findViewById(R.id.title_text);
            descriptionText = itemView.findViewById(R.id.description_text);
        }
    }
}

6. Bind the Adapter to the Data Source to Populate the RecyclerView

Finally, we need to bind the adapter to the data source and populate the RecyclerView. We do this by getting a reference to the RecyclerView in our activity and setting it's adapter. 


          
           
RecyclerView recyclerView = (RecyclerView) findViewById(R.id.recycler_view);
List<Item> items = getItems();
ItemAdapter adapter = new ItemAdapter(items);
recyclerView.setAdapter(adapter);
