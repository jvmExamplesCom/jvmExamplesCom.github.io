javax.servlet.FilterConfig

The javax.servlet.FilterConfig interface is used for providing configuration information to a filter. It allows servlet containers to pass information to a filter during initialization. A FilterConfig instance is created by the web container for each filter or servlet. FilterConfig objects are passed to a filter's init() method.

The init() Method

The init() method is used to provide configuration information to a filter. This configuration information is provided in the form of the FilterConfig object, which contains the ServletContext and init parameters.     this.filterConfig = config;

}
