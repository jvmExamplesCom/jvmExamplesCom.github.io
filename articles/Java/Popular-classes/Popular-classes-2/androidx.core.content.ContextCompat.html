androidx.core.content.ContextCompat

AndroidX Core Content ContextCompat is a utility library to access features in the Android framework that are not part of the standard Android APIs. It provides wrappers around support libraries and also exposes methods to quickly access context-specific functionality. This class can check if a specific permission is granted, request permissions at runtime, manually check and request permissions, register broadcast receivers, launch activities, get resources, and more.

    This method checks whether a specific permission has been granted or not. It returns an int which can be used to determine whether a permission is granted or not.

                            Manifest.permission.READ_CALENDAR);

    This method is used to request permissions at runtime. It takes the Activity, a String array of permissions, and an int request code as parameters. The application must call the onRequestPermissionsResult() callback method once all requested permissions have been granted or denied.
