org.springframework.core.io.Resource

The 

          
           
org.springframework.core.io.Resource
is an interface in the Spring Framework. It provides an abstract model for handling low-level I/O operations such as reading resources from files, directories on the filesystem, classpath, etc. This is a fundamental interface that allow us to access and manipulate a variety of low-level I/O sources.

Usage

The 

          
           
Resource
interface can be used whenever we need to read data from any of the supported sources. A few common use cases include:
  • Reading a resource file from a classpath
  • Reading a file from the local filesystem
  • Accessing a remote resource over the network
  • Reading a resource from a jar

Methods

The 

          
           
Resource
interface defines a number of methods for retrieving and manipulating resources. The most commonly used methods are listed below:

InputStream getInputStream()
    – Returns an 
    InputStream
    for reading the contents of the resource. 
  • boolean exists()
    – Returns whether the resource actually exists. 
  • URL getURL()
    – Returns a 
    URL
    object representing the location of the resource. 
  • File getFile()
    – Returns a 
    File

  • long lastModified()
    – Returns the last modified time of the resource. 
  • 
            
             
String getFilename()
    – Returns the filename of the resource.
