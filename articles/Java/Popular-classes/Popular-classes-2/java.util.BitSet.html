java.util.BitSet

Java's 

          
           
BitSet
class allows us to manipulate bits of data. It stores each bit in an array of long values, and provides methods to manipulate each bit individually. BitSet simplifies many operations that would otherwise require special-purpose algorithms.

Creating a BitSet

To create a 

          
           
BitSet
, we can use the empty constructor: 

          
           
BitSet b = new BitSet();
This creates an empty BitSet of size zero. We can also pass an integer argument to the constructor to specify the initial size: 

          
           
BitSet b = new BitSet(16);
This creates a 16-bit BitSet with all bits set to false (zero).

Setting and Clearing Bits

We can set any bit to either true or false using the 

          
           
set()
and 

          
           
clear()
methods: 

          
           
b.set(3); // sets bit 3 to true
b.clear(7); // sets bit 7 to false

Testing Bits

The 

          
           
get()
method returns the value of a given bit: 

          
           
boolean bit3 = b.get(3);

Inverting Bits

The 

          
           
flip()
method inverts a bit: 

          
           
b.flip(3); // flips bit 3

Counting Bits

The 

          
           
cardinality()
method returns the number of bits set to true: 

          
           
int count = b.cardinality();

Finding the First Set Bit

The 

          
           
nextSetBit()
method returns the index of the first bit that is set to true: 

          
           
int index = b.nextSetBit(0);
This returns the index of the first bit that is set to true, starting from the given index (in this case, 0).

Iterating Over Set Bits

The 

          
           
stream()
method returns a stream of the indices of all bits set to true: 

          
           
IntStream stream = b.stream();
We can then use the stream to iterate over all bits set to true: 

          
           
stream.forEach(i -> System.out.println("Bit " + i + " is set"));
