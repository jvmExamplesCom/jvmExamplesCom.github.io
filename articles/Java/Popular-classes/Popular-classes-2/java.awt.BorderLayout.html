java.awt.BorderLayout

java.awt.BorderLayout is a class used to lay out components in a container. It allows components to be placed in the four major directions of the container, north, south, east and west. BorderLayout also allows components to overlap each other.

The following example shows a basic layout using java.awt.BorderLayout. A JPanel is created with a BorderLayout and then four JLabels are added to it, one in each direction. All the components have a width and height of 10 pixels.


          
           
JPanel panel = new JPanel();
panel.setLayout(new BorderLayout());
// Set the size of all the components to be 10 pixels
panel.setSize(10, 10);

panel.add(new JLabel("North"), BorderLayout.NORTH);
panel.add(new JLabel("South"), BorderLayout.SOUTH);
panel.add(new JLabel("East"), BorderLayout.EAST);
panel.add(new JLabel("West"), BorderLayout.WEST);

The following example shows how a Button can be added to the center of the BorderLayout container. The button is initialized and then added to the panel with the add method. The first argument of the add method is the component to be added and the second argument is the position to which the component should be added.


          
           
JButton button = new JButton("Click Me!");
panel.add(button, BorderLayout.CENTER);

The following example shows how to set components to overlap, by setting the z-index of the component. First, two JButtons are added to the panel. Then, one of the buttons is set to overlap the other by setting its z-index to a higher value than the other.


          
           
JButton button1 = new JButton("Button 1");
JButton button2 = new JButton("Button 2");

panel.add(button1, BorderLayout.NORTH);
panel.add(button2, BorderLayout.SOUTH);

// Set the z-index of button2 to be higher than button1
button2.setZIndex(1);

Conclusion

java.awt.BorderLayout is a useful layout manager for laying out components within a container. It allows components to be placed in the four major directions of the container, north, south, east and west. BorderLayout also allows components to overlap each other.
