org.springframework.security.core.userdetails.UsernameNotFoundException is an exception that is thrown when a user fails to be authenticated during a login process, usually due to the user not being found in the system. This exception is part of the Spring Security framework and is used to signal that a user cannot access the application.

  UserDetails userDetails = userDetailsService.loadUserByUsername(username);

   System.out.println("Username is not found");
}
