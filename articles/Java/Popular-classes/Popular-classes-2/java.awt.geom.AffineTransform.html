java.awt.geom.AffineTransform

The java.awt.geom.AffineTransform class forms the core of the Java 2D coordinate system. It is responsible for transforming shapes, coordinates, and other elements into different geometric representations. The AffineTransform class is important for its ability to represent transformations in both the x and y plane.

class Description

This class encapsulates a set of 6 coefficients that define an affine transformation that performs a linear mapping from 2D homogeneous coordinates to other 2D homogeneous coordinates that preserves the “straightness” and “parallelness” of lines. The 4th coefficient of the transformation matrix must not be 0 if this transformation is to be invertible. This class can be used to create a wide variety of 2D graphic transformations such as translation, scaling, shearing, sequence of rotated lines and so on.

The following code creates a new AffineTransform object and performs some basic transformations:


          
           
AffineTransform at = new AffineTransform(); // create the affine transform
at.rotate(Math.PI/4); // rotate by 45 degrees 
at.translate(2.0, 3.0); // translate by 2 in x-axis and 3 in y-axis

// scale the transformation by factor of 2
at.scale(2.0, 1.0); 

// calculate the inverse transform
at.inverseTransform();
