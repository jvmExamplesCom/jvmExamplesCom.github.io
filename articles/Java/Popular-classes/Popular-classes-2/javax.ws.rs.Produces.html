javax.ws.rs.Produces

The 

          
           
javax.ws.rs.Produces
annotation is part of the Java API for RESTful Web Services (JAX-RS) 2.0 and is used to specify which media types a resource can produce in its response.

Annotations Overview

Produces annotations can be found at the method level or class level, and indicate which MIME media type is produced by a particular method in a resource or from all methods of a resource.

Syntax

The syntax for the Produces annotation is: 

          
           
@Produces([mime types])

Where [mime types] is a comma separated list of MIME media types that the method produces in its response.

Example

In this example, a method called 
getUserData()
is annotated with the 
Produces
annotation, specifying that it produces either an XML or JSON representation of user data. 

          
           
@GET
@Produces({"application/xml", "application/json"})
public UserData getUserData() {
    // Code for returning user data
}

Output

When the 
getUserData()
method is called, the following output is returned: 

          
           
{ "name": "John Smith", "age": 32 }
