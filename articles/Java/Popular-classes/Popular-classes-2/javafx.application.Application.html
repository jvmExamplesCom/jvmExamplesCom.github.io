What is javafx.application.Application?

javafx.application.Application is a class from the JavaFX library that provides the basic framework for a JavaFX application. It serves as the main starting point for applications that are written using the JavaFX platform. It contains the main() method, which calls the launch() method and initiates the life cycle of a JavaFX application.

Code Examples

The following code is an example of a simple JavaFX application that extends the javafx.application.Application class and prints out a message when the application is launched:


          
           
public class MyJavaFXApplication extends Application {
    public void start(Stage stage) {
        System.out.println("Application has started!");
    }

    public static void main(String[] args) {
        launch(args);
    }
}
