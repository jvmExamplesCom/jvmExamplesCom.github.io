Introduction to org.apache.commons.cli.CommandLine

org.apache.commons.cli.Command Line

Basic Usage and Syntax

To use this class, the user must first create a 

          
           
Options
object that defines all of the possible options that the program can accept. This can be done by calling the 

          
           
new Option()

addOption()

CommandLineParser.parse()
method, passing in the Options and the 

          
           
args[]
array from the main method, to create an instance of the CommandLine class. Once the CommandLine instance has been created, the user can use its various methods to check for the presence and value of any given option specified by the user. -v

public static void main(String[] args) {
	Options options = new Options();
	options.addOption("v", false, "verbose");

	CommandLineParser parser = new DefaultParser();
	CommandLine cmd = parser.parse(options, args);
