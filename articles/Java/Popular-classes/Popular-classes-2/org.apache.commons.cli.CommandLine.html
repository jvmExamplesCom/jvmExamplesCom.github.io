Introduction to org.apache.commons.cli.CommandLine

org.apache.commons.cli.Command Line

Basic Usage and Syntax

Options
object that defines all of the possible options that the program can accept. This can be done by calling the 

          
           
new Option()

addOption()

CommandLineParser.parse()

args[]
array from the main method, to create an instance of the CommandLine class. Once the CommandLine instance has been created, the user can use its various methods to check for the presence and value of any given option specified by the user. public static void main(String[] args) {
	Options options = new Options();
	options.addOption("v", false, "verbose");
