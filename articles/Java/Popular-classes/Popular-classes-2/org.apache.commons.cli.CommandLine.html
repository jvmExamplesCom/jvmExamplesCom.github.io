Introduction to org.apache.commons.cli.CommandLine

Options
object that defines all of the possible options that the program can accept. This can be done by calling the 

          
           
new Option()

CommandLineParser.parse()

args[]

Once the CommandLine instance has been created, the user can use its various methods to check for the presence and value of any given option specified by the user. -v

public static void main(String[] args) {
	Options options = new Options();
	options.addOption("v", false, "verbose");
