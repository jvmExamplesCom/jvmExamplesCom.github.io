org.springframework.context.annotation.Primary

The org.springframework.context.annotation.Primary annotation indicates that a bean should be given priority when multiple candidates are qualified to autowire a single-valued dependency. The bean marked with this annotation will be given precedence over other beans marked with @Qualifier or the default bean. It is useful when multiple beans might qualify as primary autowire candidates and you need to decide which one to use by default.

Using the @Primary Annotation
