org.jsoup.nodes.Document

Introduction

org.jsoup.nodes.Document is the root object of a parsed HTML document. It providess an interface to access, traverse and manipulate the elements in an HTML document. org.jsoup.nodes.Document is a part of org.jsoup Java library.

Creating a Document

A org.jsoup.nodes.Document can be created from a String, a byte array, a File, or an InputStream, using the static method Jsoup.parse().


          
           
String html = "<html><head><title>Sample Title</title></head><body><div>Hello World</div></body></html>";
	
org.jsoup.nodes.Document doc = Jsoup.parse(html);

Accessing Elements

Once a org.jsoup.nodes.Document is created, the elements in the document can be accessed and manipulated. The getElementsByTag() method can be used to access all the elements with a given tag in the document.


          
           
org.jsoup.select.Elements divElements = doc.getElementsByTag("div");

The elementByID() method can be used to access an element by its ID.


          
           
org.jsoup.nodes.Element divElement = doc.getElementById("myDiv");

Manipulating Elements

Elements in the org.jsoup.nodes.Document can be manipulated in multiple ways. For example, the text of an element can be set or retrieved using the text() method.


          
           
divElement.text(" This is new text"); 
System.out.println(divElement.text()); // Prints "This is new text"

The attr() method can be used to set or retrieve attribute values for an element.


          
           
divElement.attr("class", "test"); 
System.out.println(divElement.attr("class")); // Prints "test"
