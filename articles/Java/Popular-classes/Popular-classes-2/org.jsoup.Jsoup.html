Introduction to org.jsoup.Jsoup

org.jsoup.Jsoup is a Java library for working with real-world HTML. It provides a very convenient API for extracting and manipulating data, using the best of DOM, CSS, and jquery-like methods. With jsoup, you can parse HTML from a URL, file, or string and extract and manipulate data, using the best of DOM, CSS, and jquery-like methods.

Getting Started with Jsoup

To get started with jsoup, you will need to include the following libraries in your project: 

          
           
//Adding jsoup dependency
<dependency>
    <groupId>org.jsoup</groupId>
    <artifactId>jsoup</artifactId>
    <version>1.12.1</version>
</dependency>
The next step is to create an instance of a Document (org.jsoup.nodes.Document) class. You can do this by passing a URL or File to the static method Jsoup.parse(). 

          
           
//Parsing a URL
Document doc = Jsoup.parse(new URL("http://example.com"), 5000);

//Parsing a File
File input = new File("input.html");
Document doc = Jsoup.parse(input, "UTF-8", "http://example.com/");

Extracting Data Using Jsoup

Once you have an instance of the Document class, you can use the several accessor methods to extract the data from the HTML. The most common way to extract data is to use the select() method, which takes a CSS-style selector as an argument and returns a list of Elements. 

          
           
//Selecting elements by their tag name
Elements links = doc.select("a");

//Selecting elements by their class name
Elements elements = doc.select(".list-item");

//Selecting elements by their id
Element element = doc.select("#main-content").first();
You can also use the getElementById(), getElementsByTag() and getElementsByClass() methods to get a single element or a list of elements respectively. 

          
           
//Getting an element by its id
Element element = doc.getElementById("main-content");

//Getting a list of elements by their tag name
Elements elements = doc.getElementsByTag("title");

//Getting a list of elements by their class name
Elements elements = doc.getElementsByClass("list-item");
Additionally, there are several methods available to extract the text from the Elements, such as text(), html(), ownText() and val(). 

          
           
//Extracting the text from an element
String text = element.text();

//Extracting the html from an element
String html = element.html();

//Extracting the own text from an element
String ownText = element.ownText();

//Extracting the attribute value from an element
String val = element.val("attribute-name");

Manipulating the Data Using Jsoup

Once you have extracted the data, you can use several methods to manipulate it. Most commonly, you would use the setText() and attr() methods to set the text content or the attributes of an element. 

          
           
//Setting the text of an element
element.text("New text");

//Setting an attribute of an element
element.attr("class", "new-class");
You can also add new elements to the existing document using the append(), prepend(), before(), after() and wrap() methods. 

          
           
//Appending a new element to the document
doc.body().append("<p>Some new text</p>");

//Prepending a new element to the document
doc.body().prepend("<p>Some new text</p>");

//Inserting a new element before an existing element
element.before("<p>Some new text</p>");

//Inserting a new element after an existing element
element.after("<p>Some new text</p>");

//Wrapping an element in another element
element.wrap("<div class="new-div"></div>");
Finally, you can remove an element from the document using the remove() method. 

          
           
//Removing an element from the document
element.remove();
