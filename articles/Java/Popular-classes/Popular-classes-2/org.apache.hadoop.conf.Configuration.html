org.apache.hadoop.conf.Configuration

The org.apache.hadoop.conf.Configuration class provides a way to manage configuration of Hadoop components. It is used by clients, admin tools and daemons to read the configuration information for the entire Hadoop system.

Configuration conf = new Configuration();

Configuration conf = new Configuration();

Configuration conf = new Configuration();
