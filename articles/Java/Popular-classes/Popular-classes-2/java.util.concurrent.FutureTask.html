java.util.concurrent.FutureTask

Java.util.concurrent.FutureTask is a Java class that allows applications to perform asynchronous tasks in a separate thread without having to manage the whole process of thread management. The FutureTask offers an API that can be used to schedule, cancel, and check the status of a task. With this API, it is possible to monitor the progress of the task execution and retrieve results from the task when it is finished.

Creating and Executing the FutureTask

To create a FutureTask, we need to provide it with a Callable object. A Callable is a functional interface that takes no parameters and returns a value. We can define a simple Callable like this:


          
           
Callable<Integer> myCallable = () -> {
    System.out.println("Hello from Callable!");
    return 5;
};

We can now create the FutureTask using this Callable:


          
           
FutureTask<Integer> myFutureTask = new FutureTask<>(myCallable);

Finally, we can execute this FutureTask in a separate thread using the java.util.concurrent.Executor framework:


          
           
ExecutorService executor = Executors.newSingleThreadExecutor();
executor.execute(myFutureTask);

Checking the Status of the Task

Once the FutureTask has been created, it can be queried at any time to get its status. To do this, we can call the isDone() and isCancelled() methods. These methods will return true if the task has completed or been cancelled, respectively.


          
           
if (myFutureTask.isDone()) {
    System.out.println("Task is done!");
}

          

          
           
if (myFutureTask.isCancelled()) {
    System.out.println("Task has been cancelled!");
}

Retrieving Results from the Task

When the FutureTask has finished executing, it's possible to retrieve the result of the Callable using the get() method. This method will block until the FutureTask has finished executing and will then return the result. For example:


          
           
try {
    int result = myFutureTask.get();

    e.printStackTrace();

Cancelling the Task

It is also possible to cancel the FutureTask by calling its cancel() method. This will attempt to interrupt the thread that is executing the task. If the task is already running, this method will have no effect. If the task has not yet started, it will be cancelled and its isCancelled() method will return true. For example:


          
           
myFutureTask.cancel();
