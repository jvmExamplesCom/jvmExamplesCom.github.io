What is XPathConstants?

Below is an example usage of XPathConstants to get a NodeList from an input XML document: 

          
           
String xml = "<root><node>One</node><node>Two</node></root>";
InputSource source = new InputSource(new StringReader(xml));
XPathFactory xpathFactory = XPathFactory.newInstance();
XPath xpath = xpathFactory.newXPath();
