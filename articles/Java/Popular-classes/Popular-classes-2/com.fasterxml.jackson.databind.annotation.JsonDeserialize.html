Introduction to JsonDeserialize annotation in Java

JsonDeserialize annotation is used in Jackson-based applications to deserialize the data from JSON (JavaScript Object Notation) representation into the type of Java object marked with this annotation. Using JsonDeserialize Annotation

@JsonDeserialize(using = CustomPersonDeserializer.class)
