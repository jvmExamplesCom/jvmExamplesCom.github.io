Introduction to JsonDeserialize annotation in Java

JsonDeserialize annotation is used in Jackson-based applications to deserialize the data from JSON (JavaScript Object Notation) representation into the type of Java object marked with this annotation. JsonDeserialize annotation is a marker annotation and it has no specific attributes; however, it can be used in conjunction with JsonDeserializer class for custom deserialization. This article will look into how to use this annotation and its advantages.

Using JsonDeserialize Annotation

JsonDeserialize annotation is used to specify a custom deserialization. Here is an example of how to use this annotation: 

          
           
@JsonDeserialize(using = CustomPersonDeserializer.class)
public class Person {
  private String name;
  private Address address;
  // getters/setters
}
In the above example, annotation is used to specify that the data from JSON should be deserialized with the help of CustomPersonDeserializer class.

@JsonDeserialize(using = CustomDateTimeDeserializer.class)
