Understanding org.springframework.security.core.GrantedAuthority

org.springframework.security.core.GrantedAuthority is an abstract class provided by the Spring Security framework for Java. It is essentially a security-specific representation of an authorization granted to a particular user. It does not provide any policy control in terms of what a particular user can do, but it can be used to implement specific authorization models.

The interface contains a single method - getAuthority() - which returns a representation of the granted authority. This value is typically used as a basis for determining whether a particular user has been granted a certain permission or access level. It is also used to assign roles to users in web applications and to set up access control lists (ACLs) in enterprise applications.
