javax.xml.bind.annotation.XmlAccessType

The javax.xml.bind.annotation.XmlAccessType is an enum class used to specify a mapping between JavaBeans properties and XML elements or attributes in the generated XML documents. The XmlAccessType annotation can be used to change the default JavaBean property to XML element mapping behavior. There are four different values that can be specified with XmlAccessType; PROPERTY, FIELD, PUBLIC_MEMBER, and NONE.

PROPERTY

The PROPERTY value of XmlAccessType instructs the JAXB runtime to bind JavaBean Properties to XML elements or attributes. This is the default XmlAccessType and will automatically be used if the annotation is not specified.


          
           
@XmlAccessType(XmlAccessType.PROPERTY)
public class User {

   private String name;

   public String getName() {
      return name;
   }

   public void setName(String name) {
      this.name = name;
   }

}

FIELD

The FIELD value of XmlAccessType instructs the JAXB runtime to bind fields directly to XML elements or attributes in the resulting XML document. This is the opposite of the PROPERTY access type which binds getter and setter methods to XML elements or attributes.


          
           
@XmlAccessType(XmlAccessType.FIELD)
public class User {

   @XmlElement
   private String name;

}

PUBLIC_MEMBER

The PUBLIC_MEMBER value of XmlAccessType instructs the JAXB runtime to bind public fields and public getter and setter methods to XML elements or attributes in the resulting XML document. This is similar to the PROPERTY access type but it allows for public fields to be bound to XML elements or attributes.


          
           
@XmlAccessType(XmlAccessType.PUBLIC_MEMBER)
public class User {

   @XmlElement
   public String name;

   public String getName() {
      return name;
   }

   public void setName(String name) {
      this.name = name;
   }

}

NONE

The NONE value of XmlAccessType instructs the JAXB runtime to ignore all non static fields and non static getter and setter methods when marshaling and unmarshaling XML documents. This can be useful if you want to bind only a specific subset of properties or fields to an XML element or attribute.


          
           
@XmlAccessType(XmlAccessType.NONE)
public class User {

   @XmlElement
   private String name;

   public String getName() {
      return name;
   }

   public void setName(String name) {
      this.name = name;
   }

}
