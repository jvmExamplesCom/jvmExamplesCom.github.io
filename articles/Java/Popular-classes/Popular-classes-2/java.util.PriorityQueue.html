java.util.PriorityQueue

java.util.PriorityQueue is a class in the Java collections framework that represents an ordered queue. It is based on a priority heap, and operates similarly to a standard FIFO (first-in-first-out) queue, except that objects with higher priority are taken from the queue before objects with lower priority. This class is not synchronized and can therefore be used in a concurrent environment.

PriorityQueue implements Queue interface, which offers multiple methods to work with the PriorityQueue, such as offer(), peek(), poll(), remove() and add(). The offer() method inserts a new element into the PriorityQueue. Peek() retrieves but does not remove the head of this queue, or returns null if this queue is empty. Poll() retrieves and removes the head of this queue, or returns null if this queue is empty. Remove() removes a single instance of the specified element from this queue, if it is present. Lastly, the add() method inserts all of the elements in the specified collection into this queue.

The elements of the PriorityQueue are ordered according to their natural ordering, or by a Comparator provided at queue construction time, depending on which constructor is used.

Example:

The following example shows how to use the PriorityQueue class. In this example, we create a PriorityQueue object and add three string elements, "Apple", "Banana" and "Cherry". Then, we use the peek(), poll() and add() methods to access and manipulate the elements of the PriorityQueue. After, we use the size() method to get the total number of elements in the PriorityQueue.


          
           
import java.util.PriorityQueue;
public class PriorityQueueDemo {
  public static void main(String args[]){

    // Create a PriorityQueue
    PriorityQueue<String> pq = new PriorityQueue<String>();

    // Add elements to the PriorityQueue
    pq.add("Apple");
    pq.add("Banana");
    pq.add("Cherry");

    // Display the head element of the PriorityQueue
    System.out.println("Head element: " + pq.peek());

    // Poll the head element of the PriorityQueue
    pq.poll();
    System.out.println("After poll: " + pq);

    // Add another element to the PriorityQueue
    pq.add("Kiwi");
    System.out.println("After adding Kiwi: " + pq);

    // Get the size of the PriorityQueue
    System.out.println("Size of the PriorityQueue: " + pq.size());
  }
}

When the above code is compiled and executed, it produces the following result:


          
           
Head element: Apple
After poll: [Banana, Cherry, Kiwi]
After adding Kiwi: [Banana, Cherry, Kiwi]
Size of the PriorityQueue: 3
