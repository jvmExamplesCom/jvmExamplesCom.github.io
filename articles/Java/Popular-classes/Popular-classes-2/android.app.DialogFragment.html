Java Android DialogFragment

The 

          
           
android.app.DialogFragment
class is a fragment that displays a dialog window, floating on top of its activity's window. This fragment contains a Dialog object, which it displays as appropriate based on the fragment's state. Control of the dialog (deciding when to show, hide, dismiss it) should be done through the API here, not with direct calls on the dialog. Using this class to create a dialog: 

          
           
final DialogFragment myDialogFragment = new MyDialogFragment();
myDialogFragment.show(getSupportFragmentManager(), "dialog");

Creating a DialogFragment

To create an instance of a DialogFragment, you need to extend the DialogFragment class and implement the onCreateDialog() callback method. This method returns a Dialog instance to be displayed by the fragment. For example: 

          
           
public class MyDialogFragment extends DialogFragment {

    @Override
    public Dialog onCreateDialog(Bundle savedInstanceState) {
        return new AlertDialog.Builder(getActivity())
                .setTitle("My Title")
                .setMessage("My message")
                .setPositiveButton("OK", null)
                .create();
    }
}

Showing a DialogFragment

Once you've created a DialogFragment, you can show it by calling the FragmentTransaction.add() or FragmentTransaction.replace() methods, passing in the DialogFragment as the new fragment. For example: 

          
           
MyDialogFragment myDialogFragment = new MyDialogFragment();
FragmentTransaction ft = getFragmentManager().beginTransaction();
ft.add(myDialogFragment, "dialog");
ft.commit();

Dismissing a DialogFragment

To dismiss a DialogFragment, simply call the dismiss() method on the DialogFragment object, which will close the Dialog window. For example: 

          
           
myDialogFragment.dismiss();
