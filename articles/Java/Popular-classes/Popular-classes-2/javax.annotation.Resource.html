Introduction to javax.annotation.Resource

The javax.annotation.Resource annotation allows developers to inject resources into their applications. This makes it possible to easily change the configuration of an application without the need for changing any code. By using the annotation, a component can be specified in arbitrary locations and referenced by its name. This makes it much easier to maintain an application since the injection points are clear and explicit.

Using @Resource Annotation

To use the @Resource annotation, first the target class must import the resource package: 

          
           
import javax.annotation.Resource;
Once imported, the annotation is then added to the field to which it should inject the resource. The following example will inject the "fooService" resource into the "service" field: 

          
           
@Resource
private FooService service;
In this example, the injected resource must have the same type as the field to which it is being injected. If this is not the case, the resource's type name needs to be specified in the annotation. The following example shows how the resource's type can be specified: 

          
           
@Resource(type = FooService.class)
private Service service;
Alternatively, the resource's name can be specified in the annotation instead of the type. This is useful in cases where a resource is being injected from an external source. The following example shows how a resource can be injected from an external source using its name: 

          
           
@Resource(name = "foo")
private Service service;
Finally, the @Resource annotation also allows developers to specify a default value for a resource. This default value will be used if the specified resource is not available. The following example shows how a default value can be specified: 

          
           
@Resource(defaultValue = "foo")
private Service service;

Example

Here is a complete example of the @Resource annotation in action: 

          
           
import javax.annotation.Resource;

public class Foo {
  
  // Inject the foo service
  @Resource
  private FooService service;
  
  public void doSomething() {
    // Use the foo service
    service.doSomething();
  }
  
}
