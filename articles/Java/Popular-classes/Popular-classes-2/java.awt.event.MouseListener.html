java.awt.event.MouseListener

A 

          
           
java.awt.event.MouseListener
is a class used to capture mouse events and to handle them according to the program logic. It provides methods that are called when various types of mouse actions occur, such as clicking, double-clicking, pressing, and releasing the mouse button. This class is used in both graphical user interfaces (GUIs) and other forms of applications.

Implementing a MouseListener

The most common way to use a 

          
           
MouseListener
is to create an instance of it and add it to the component being monitored, then implement the required methods within it. Here is an example of a basic 

          
           
MouseListener
: 

          
           
public class MyMouseListener implements MouseListener {

    public void mouseClicked(MouseEvent e) {
        System.out.println("Mouse clicked!");
    }

    public void mousePressed(MouseEvent e) {
        System.out.println("Mouse pressed!");
    }

    public void mouseReleased(MouseEvent e) {
        System.out.println("Mouse released!");
    }

    public void mouseEntered(MouseEvent e) {
        System.out.println("Mouse entered!");
    }

    public void mouseExited(MouseEvent e) {
        System.out.println("Mouse exited!");
    }
}
Then the 

          
           
MyMouseListener
class can be added to any component, such as a JButton, like this: 

          
           
JButton btn = new JButton("My Button");
btn.addMouseListener(new MyMouseListener());

Using MouseListener with Anonymous Inner Classes

Alternatively a 

          
           
MouseListener
can be used with an anonymous inner class. This is useful for quickly creating classes on the fly and adding them to components. For example: 

          
           
JButton btn = new JButton("My Button");

    public void mouseClicked(MouseEvent e) {

    }

        System.out.println("Mouse released!");
    }

    public void mouseEntered(MouseEvent e) {
        System.out.println("Mouse entered!");
    }

    public void mouseExited(MouseEvent e) {
        System.out.println("Mouse exited!");
    }
});

Examples of Using MouseListener

Changing the Cursor When Entering a Component

In this example, the cursor will be changed to a hand when the mouse enters a component, and to a default cursor when it exits. 

          
           
btn.addMouseListener(new MouseListener() {
    public void mouseEntered(MouseEvent e) {
        btn.setCursor(new Cursor(Cursor.HAND_CURSOR));
    }

    public void mouseExited(MouseEvent e) {
        btn.setCursor(new Cursor(Cursor.DEFAULT_CURSOR));
    }
});

Adding a Tooltip to a Component

In this example, a tooltip will be added to a component when the mouse enters it, and removed when the mouse exits. 

          
           
btn.addMouseListener(new MouseListener() {
    public void mouseEntered(MouseEvent e) {
        btn.setToolTipText("My tooltip text");
    }

    public void mouseExited(MouseEvent e) {
        btn.setToolTipText(null);
    }
});

Conclusion

The 

          
           
java.awt.event.MouseListener
class is a powerful tool for responding to user input in graphical user interfaces and other applications. It provides a variety of methods that can be used to detect and respond to mouse actions, and can be used with either traditional classes or anonymous inner classes. With these techniques, developers can easily add interactivity to their applications.
