android.preference.PreferenceActivity

Android Preference Activity is an Android-specific activity for creating application preferences that allow users to modify application settings. It provides a consistent approach to handle configuration-related activities in a single unified UI. Preference activities are commonly used to customize an application's behavior, as well as to provide user access to certain features of the application. This article will cover the basics of creating and using a Preference Activity in Android.

Creating a Preference Activity

The first step in creating a Preference Activity is to define the preferences that you want to include in it. This is done by adding a 


          
           
<PreferenceScreen>
XML element to the 

          
           
res/xml/preferences.xml
file. Within this element, you can define the list of preferences that will appear in your activity. For example, the following code snippet defines a preference that allows the user to enable or disable a feature: 

          
           
<PreferenceScreen>
    <Preference
        android:key="enable_feature"
        android:title="Enable Feature"
        android:summary="Turn on or off the feature"
        android:defaultValue="false" />
</PreferenceScreen>

Once the preferences are defined, you can create your preference activity. This is done by subclassing the 

PreferenceActivity
class and overriding the 
onCreate
method. In this method, you can call the 

          
           
addPreferencesFromResource()
method, passing it the resource ID of the 
PreferenceScreen
that contains the preferences for your activity: 

          
           
public class MyPreferenceActivity extends PreferenceActivity {

    @Override
    public void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        addPreferencesFromResource(R.xml.preferences);
    }
}

Now that the activity is created, you can launch it in your application. To do this, just call the 

startActivity()
method, passing it an 
Intent
with the 

          
           
MyPreferenceActivity.class
as the target activity: 

          
           
Intent intent = new Intent(this, MyPreferenceActivity.class);
startActivity(intent);

Accessing Preferences in Code

Once the user has modified the values of the preferences in the activity, you can access the values in your code. To do this, you can get a 

SharedPreferences
object by calling the 

          
           
getSharedPreferences()
method and passing it the name of the XML file that contains the preferences: 

          
           
SharedPreferences prefs = getSharedPreferences("preferences", MODE_PRIVATE);
Now you can use the 
getBoolean()
, 
getInt()
, etc. methods of the 
SharedPreferences
object to retrieve the values of the preferences: 

          
           
boolean enableFeature = prefs.getBoolean("enable_feature", false);

Conclusion

In this article, we have seen how to create and use a Preference Activity in Android. Preference activities provide a convenient way for users to customize the behavior of your app. By leveraging the built-in support for Preference Activities, you can save yourself from having to create your own custom interfaces for handling user configuration tasks.
