org.apache.commons.cli.HelpFormatter

The HelpFormatter class is part of Apache Commons CLI and provides a standard way to present a help message for a command line application. It supports options, parameters and subcommands which can be defined as part of the command line syntax.

Creating a HelpFormatter

The HelpFormatter class is instantiated with a few optional parameters, like the output width, the characters that will form the usage line or whether it should sort the options alphabetically when printing the help message. 


          
           
HelpFormatter formatter = new HelpFormatter();
formatter.setWidth(100);
formatter.setUsageWidth(50);
formatter.setOptionComparator(null);

Setting Options

The HelpFormatter class provides the method 

          
           
addOption
to set one option at a time. This method takes an instance of the 

          
           
Option
class and adds it to the list of options. Options can be configured with a short name, a long name, a description and/or an argument, as well as several other attributes. 

          
           
Option option = Option.builder("s")
    .longOpt("server")
    .desc("Specifies the server URL")
    .hasArg()
    .build();
formatter.addOption(option);

Setting Parameters

The HelpFormatter class also supports the definition of parameters, which are required inputs that don’t have an associated option. Parameters can be added with the method 


          
           
addParameter
. This method takes a name, a description and an optional argument type, as well as other attributes. 

          
           
Parameter param = Parameter.builder("name")
        .desc("The name of the user")
        .required()
        .hasArg(Parameter.REQUIRED_ARGUMENT)
        .build();
formatter.addParameter(param);

Printing Help Message

After configuring all options and parameters, the help message can be printed using the method 


          
           
printHelp
. This method takes a header, a footer and a synopsis as parameters. The header and footer are printed before and after the main body of the help message, while the synopsis is printed between the header and body. 

          
           
formatter.printHelp("myapp", "My App - A sample command line application.",
        " ", options, parameters, 1, 4, null, true);

Outputting Help Message As HTML

The HelpFormatter class also supports outputting the help message as HTML. To do this, you need to instantiate a 


          
           
HtmlHelpFormatter
class with a few optional parameters and invoke the method 

          
           
printHelpAsHtml
when you want to print the help message. 

          
           
HtmlHelpFormatter htmlFormatter = new HtmlHelpFormatter();
htmlFormatter.setWidth(100);
htmlFormatter.setUsageWidth(50);
htmlFormatter.setOptionComparator(null);

htmlFormatter.printHelpAsHtml("myapp",
        "My App - A sample command line application.",
        " ", options, parameters, 1, 4, null, true);
