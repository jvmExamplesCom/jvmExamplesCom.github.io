org.springframework.security.core.AuthenticationException

The AuthenticationException class is a subclass of the Spring Security core exception that provides an authentication-related error code. It is used to indicate an authentication problem has occurred and can be used as part of an overall security system architecture.

Use Cases

AuthenticationException can be thrown from the authentication provider class when authentication has failed due to invalid credentials, users not having the required privileges, or any other authentication-related issues. It is also used to throw exceptions for authorization failures.

  // authentication failed
  throw authEx;
