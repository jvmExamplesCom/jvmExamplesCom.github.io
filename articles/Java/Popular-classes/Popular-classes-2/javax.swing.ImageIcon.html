javax.swing.ImageIcon

javax.swing.ImageIcon is a class in Java that provides the ability to display an icon for graphical user interface components such as labels, buttons, and tables. It is part of the Swing GUI framework and is capable of displaying icons in either PNG, JPEG, GIF, or BMP formats.

Creating an ImageIcon Object

When creating an ImageIcon, you must specify the path to the image file. It is also possible to create an ImageIcon from a URL by providing the URL instead of a local file path.


          
           
// specify the path to the image file
ImageIcon icon = new ImageIcon("my-image.png");

// create an ImageIcon from a URL
URL url = new URL("https://example.com/my-image.png");
ImageIcon icon = new ImageIcon(url);

Displaying an ImageIcon

The ImageIcon can be displayed by using the setIcon() method on the applicable component. This method has been implemented for many Swing components, including buttons, labels, and tables.


          
           
// display an image icon in a label
JLabel label = new JLabel();
label.setIcon(icon);

Scaling an ImageIcon

The size of the ImageIcon can be modified by specifying a width and height when creating the ImageIcon. Alternatively, the scale() method can be used to scale the image to a specified size:


          
           
// create a scaled ImageIcon
ImageIcon scaledIcon = new ImageIcon(icon.getImage().getScaledInstance(100, 100, Image.SCALE_SMOOTH));
