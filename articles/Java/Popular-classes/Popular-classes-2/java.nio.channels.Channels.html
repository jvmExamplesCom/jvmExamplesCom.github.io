java.nio.channels.Channels

The java.nio.channels.Channels class provides methods for creating and manipulating channels for the Java NIO (New Input/Output) API.

A channel is a conduiut for data in Java NIO, similar to a stream in traditional Java I/O; however, instead of reading and writing bytes, a channel can send and receive buffers containing arbitrary data. It provides a convenient way to read and write data from and to a socket, file, or other type of input/output source.

Creating a Channel

There are several different types of channels available in Java NIO, each of which is used to read or write data in a specific way. To create one, you simply call the appropriate static factory method on the Channels class.


          
           
// Create a FileChannel
FileChannel fileChannel = FileChannel.open(Paths.get("filename"));

// Create a DatagramChannel
DatagramChannel datagramChannel = DatagramChannel.open();

// Create a SocketChannel
SocketChannel socketChannel = SocketChannel.open();

Reading from a Channel

Once you have created a channel, you can read data from it using the 

read()
method. This method takes a 
ByteBuffer
argument, which is used to store the data that is read from the channel. The 
ByteBuffer
object must be allocated before calling 
read()
, and the maximum amount of data that can be read is equal to the capacity of the buffer. 

          
           
// Create a ByteBuffer with capacity of 1024 bytes
ByteBuffer buf = ByteBuffer.allocate(1024);

// Read data from the channel into the buffer
int bytesRead = fileChannel.read(buf);

Writing to a Channel

You can also use the 
write()
method to write data from a 
ByteBuffer
to a channel. The 
ByteBuffer
must have an adequate amount of data to be written, or an exception will be raised. 

          
           
// Create a ByteBuffer with capacity of 1024 bytes
ByteBuffer buf = ByteBuffer.allocate(1024);

// Put some data into the buffer
String data = "My data";
buf.put(data.getBytes());

// Set the position to 0 so that data is read from beginning
buf.position(0);

// Write data from the buffer to the channel
fileChannel.write(buf);

Transferring Data between Channels

You can also transfer data directly between two channels using the 

transferFrom()
and 
transferTo()
methods. These methods take a 
ReadableByteChannel
and 
WritableByteChannel
argument respectively, and will copy all the data from one to the other. 

          
           
// Create a FileChannel for source and destination
FileChannel source = FileChannel.open(Paths.get("source.txt"));
FileChannel dest = FileChannel.open(Paths.get("dest.txt"));

// Transfer data from source to dest
source.transferTo(0, source.size(), dest);

// Or, transfer data from dest to source
dest.transferFrom(source, 0, source.size());

Closing a Channel

Once you are done using a channel, it is important to close it. This will release any resources associated with the channel, and avoid any potential memory leaks. You can close a channel by calling the 

close()
method on it. 

          
           
fileChannel.close();
datagramChannel.close();
socketChannel.close();
