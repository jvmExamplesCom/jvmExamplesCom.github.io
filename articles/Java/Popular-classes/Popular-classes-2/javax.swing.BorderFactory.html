javax.swing.BorderFactory

The javax.swing.BorderFactory is a class in the Java Swing GUI library used to create various types of Borders for GUI components.

Creating Borders

Borders can be created from static methods in the javax.swing.BorderFactory class, such as 


          
           
createEmptyBorder()
, 

          
           
createLoweredBevelBorder()
, or 

          
           
createRaisedBevelBorder()
. For example, the following code creates a lowered bevel border and applies it to a JPanel: 

          
           
JButton button = new JButton("Click Me");
JPanel panel = new JPanel();
panel.setBorder(BorderFactory.createLoweredBevelBorder());
panel.add(button);

Applying Borders to Components

Borders created from the javax.swing.BorderFactory class can be applied to any Swing component. To do this, simply call the setBorder() method on the component, passing in the desired Border as an argument. For example, the following code will add a raised bevel border to a JTextField:


          
           
JTextField textField = new JTextField();
textField.setBorder(BorderFactory.createRaisedBevelBorder());

Customizing Borders

Borders created with the javax.swing.BorderFactory class can also be customized. For example, the following code creates a custom line border that is 5 pixels wide, with a blue color:


          
           
Border border = BorderFactory.createLineBorder(Color.BLUE, 5);

This border can then be applied to any Swing component, as described above.
