What is org.springframework.http.HttpMethod?

The 

          
           
org.springframework.http.HttpMethod
class defines the HTTP methods that are available for use with Spring. This is an enumerated type and is provided to facilitate parameter validation when providing web services. It is also useful for determining how requests should be processed by various HTTP clients.

Using org.springframework.http.HttpMethod

The enum values provided in 

          
           
org.springframework.http.HttpMethod
include all of the standard HTTP methods. These can be used to perform operations such as GET, POST, PUT, PATCH, and DELETE. The following code example shows how to access the enum values: 

          
           
HttpMethod[] methods = HttpMethod.values();
for (HttpMethod method : methods) {

}
