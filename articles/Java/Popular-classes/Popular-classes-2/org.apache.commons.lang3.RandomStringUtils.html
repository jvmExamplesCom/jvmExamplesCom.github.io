org.apache.commons.lang3.RandomStringUtils

The org.apache.commons.lang3.RandomStringUtils class is a utility class of the Apache Commons Lang Library. It is used to generate random Strings of various lengths from a given set of characters. This class can be useful for creating randomized strings such as passwords, session keys, and more.

Creating Random Strings

The following example creates a random String of length 10 using the default set of alpha-numeric characters:


          
           
String randomString = RandomStringUtils.random(10);

You can also specify a different set of characters to use when generating the random String. The following example creates a random String of length 10 using only lowercase letters:


          
           
String randomString = RandomStringUtils.random(10, true, false);

Finally, you can specify a custom set of characters to use when generating the String:


          
           
String customChars = "ABCDEF1234";
String randomString = RandomStringUtils.random(10, 0, 0, false, false, customChars.toCharArray());

String randomString = RandomStringUtils.randomAlphabetic(10, 15);
randomString = "prefix-" + randomString;

String randomString = RandomStringUtils.randomNumeric(10);
