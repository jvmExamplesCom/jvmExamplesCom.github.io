org.apache.commons.cli.ParseException

org.apache.commons.cli.ParseException is a checked exception thrown when the command line arguments do not match the expected syntax.

Purpose

The purpose of the org.apache.commons.cli.ParseException is to provide an error message indicating that the syntax of the command line arguments is invalid, and to provide a reference to the option that caused the error.
