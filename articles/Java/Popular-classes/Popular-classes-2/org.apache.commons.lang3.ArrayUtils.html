org.apache.commons.lang3.ArrayUtils

The org.apache.commons.lang3.ArrayUtils class is part of the Apache Commons Lang 3 library and provides various useful methods for working with arrays. It is a utility class intended to simplify array operations such as finding an element, copying, adding, removing, searching etc.

Basic Usage

The following code examples show some basic ways to work with arrays.


          
           
int[] myArray = {2, 3, 4}; 

// To check if an array contains a given value 
boolean isContain = ArrayUtils.contains(myArray, 3); // returns true

// To add an element to the array 
int[] newArray = ArrayUtils.add(myArray, 5);   // returns [2, 3, 4, 5]

// To remove an element from the array 
int[] remArray = ArrayUtils.removeElement(myArray, 3);  // returns [2, 4]

// To shuffle the array 
ArrayUtils.shuffle(myArray); // returns a shuffled version of the array

Cloning Arrays

The 

          
           
clone()
method can be used to create a shallow copy of an array. 

          
           
// Create a copy of the original array
int[] copyArray = ArrayUtils.clone(myArray); // returns [2, 3, 4]

// Modifying the original array will not impact the cloned array
myArray[0] = 5; // myArray is now [5, 3, 4]
// copyArray remains unchanged - [2, 3, 4]

Resizing Arrays

The 

          
           
resize()
method is used to resize the specified array. The method takes a parameter that specifies the size of the new array. 

          
           
// Resize the array to have 6 elements
int[] resizedArray = ArrayUtils.resize(myArray, 6);  
// returns [2, 3, 4, 0, 0, 0]

Comparing Arrays

The 

          
           
isEquals()
method can be used to compare two arrays to determine if they contain the same elements in the same order. 

          
           
// Create a second array
int[] comparatorArray = {2, 3, 4};

// Check if the two arrays are equal
boolean isSame = ArrayUtils.isEquals(myArray, comparatorArray); // returns true

Unboxing Arrays

The 

          
           
toObject()
method can be used to unbox an array of primitive types into an array of objects. 

          
           
// Create an array of primitive types
int[] primitives = {1, 2, 3};

// Unbox the array
Integer[] objects = ArrayUtils.toObject(primitives); 
// returns [1, 2, 3]
