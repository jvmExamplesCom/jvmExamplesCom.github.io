java.sql.ResultSetMetaData

Java.sql.ResultSetMetaData is part of the Java programming language that allows a program to access information about the columns that make up a ResultSet object. This makes it easier for programs to query and process data from databases, by providing metadata about the columns that can be accessed in the same way that a column in a database table is accessed.

Creating a ResultSetMetaData Object

A ResultSetMetaData object is created from a ResultSet object by calling the getMetaData() method. This method returns an object of type ResultSetMetaData.


          
           
ResultSet rs = ...; // get a ResultSet from database connection
ResultSetMetaData rsmd = rs.getMetaData();

Getting Information from a ResultSetMetaData Object

A ResultSetMetaData object contains many methods for getting information about the columns in a ResultSet. Here are some of the more commonly used ones:

  • getColumnCount() - returns the number of columns in the ResultSet
  • getColumnName(int columnIndex) - returns the name of the specified column index
  • getColumnType(int columnIndex) - returns the type of the specified column index
  • getPrecision(int columnIndex) - returns the precision of the specified column index
  • getScale(int columnIndex) - returns the scale of the specified column index
  • getTableName(int columnIndex) - returns the table name of the specified column index

The following example shows how to use the getColumnCount() and getColumnName() methods to print out the names of all columns in a ResultSet:


          
           
// get a ResultSet from database connection
ResultSet rs = ...;

// create a ResultSetMetaData object from the ResultSet
ResultSetMetaData rsmd = rs.getMetaData();

// get the number of columns in the ResultSet
int columnCount = rsmd.getColumnCount();

// loop through each column and print its name
for (int i = 1; i <= columnCount; i++) {
    System.out.println("Column name: " + rsmd.getColumnName(i));
}
