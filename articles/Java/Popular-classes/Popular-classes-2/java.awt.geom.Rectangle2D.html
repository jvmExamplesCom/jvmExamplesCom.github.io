java.awt.geom.Rectangle2D

The java.awt.geom.Rectangle2D class represents a rectangle defined by a location (x,y) and dimension (width, height). It can be constructed to represent the smallest rectangle that encloses a set of points, or by providing the location and size.

Example 1: Creating a Rectangle2D


          
           
// Create a Rectangle2D with x=10, y=10, width=50, height=50
Rectangle2D r = new Rectangle2D(10, 10, 50, 50);

Example 2: Checking if two Rectangle2Ds intersect


          
           
// Create two Rectangle2Ds
Rectangle2D r1 = new Rectangle2D(0, 0, 100, 100);
Rectangle2D r2 = new Rectangle2D(50, 50, 150, 150);

// Check if the two Rectangle2Ds intersect
if(r1.intersects(r2)) {
    System.out.println("The Rectangle2Ds intersect!");
} else {
    System.out.println("The Rectangle2Ds do not intersect!");
}

Example 3: Checking if a point lies within a Rectangle2D


          
           
// Create a Rectangle2D
Rectangle2D r = new Rectangle2D(0, 0, 100, 100);

// Create a point
Point2D p = new Point2D(50, 50);

// Check if the point lies within the Rectangle2D
if(r.contains(p)) {
    System.out.println("The point lies within the Rectangle2D");
} else {
    System.out.println("The point does not lie within the Rectangle2D");
}

Example 4: Resizing a Rectangle2D


          
           
// Create a Rectangle2D
Rectangle2D r = new Rectangle2D(0, 0, 50, 50);

// Resize the Rectangle2D so that its height is doubled
r.setFrame(r.getX(), r.getY(), r.getWidth(), r.getHeight() * 2);

System.out.println("New height: " + r.getHeight());

Example 5: Getting the area of a Rectangle2D


          
           
// Create a Rectangle2D
Rectangle2D r = new Rectangle2D(0, 0, 10, 20);

// Calculate the area of the Rectangle2D
double area = r.getWidth() * r.getHeight();

System.out.println("Area = " + area);
