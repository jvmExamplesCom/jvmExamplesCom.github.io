android.support.annotation.RequiresApi is an Annotation used to indicate that the code using this annotation requires a specific API level to be able to function correctly. This is useful for libraries that need to use APIs that are not available on all versions of Android, or for apps that need to require a minimum API level for their core functionality.

Using android.support.annotation.RequiresApi

implementation 'com.android.support:support-annotations:27.0.2'
