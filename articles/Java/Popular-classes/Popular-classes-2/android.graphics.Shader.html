android.graphics.Shader

Android.graphics.Shader is a class in the Android graphics package that allows developers to apply a graphical shader to an object or surface. Shaders are programs that run on a GPU (graphics processing unit) and can be used to create texture, lighting and special effects. A shader takes in parameters such as size, color, position, etc., and changes the pixels of an image accordingly.

Creating a Shader

To create a shader, you must first define a subclass of android.graphics.Shader. Then, you must define the method 


          
           
public native void setShader(int programID, float[] params)
, where the parameters are shader-specific. Finally, you can call 
getProgram()
to get the shader's program ID and then use this ID with 
setShader()
to assign the shader to an object.

Example Code

Let's say we want to create a shader that will cause an item to shimmer when it moves. Below is an example of how to do this:


          
           
public class ShimmerShader extends Shader {
   private int programID;
   private float[] params; 

   public ShimmerShader(float[] params) {
      this.params = params; 
   } 

   @Override
   public void setShader(int programID, float[] params) {
      this.programID = programID;
      this.params = params; 
   }

   @Override
   public int getProgram() {
      return programID; 
   } 
}
