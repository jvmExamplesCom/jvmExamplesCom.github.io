javax.swing.JMenuItem

The javax.swing.JMenuItem class represents an item in a menu which can be selected by the user to perform an action. It provides methods to create and manipulate menus and their associated menu items, other than adding objects of type AbstractButton such as JButtons.

Creating a JMenuItem

To create a JMenuItem, use the following: 

          
           
JMenuItem myMenuItem = new JMenuItem("My Menu Item");

The first argument represents the text that will display on the menu item.

Adding an Action Listener to a JMenuItem

To add an action listener to the JMenuItem, use the following:


          
           
myMenuItem.addActionListener(new ActionListener() {
    public void actionPerformed(ActionEvent e) {
        //code to be executed when the menu item is clicked
    }
});

The actionPerformed() method contains the code that will be executed when the menu item is clicked by the user.

Adding a JMenuItem to a JMenu

To add a JMenuItem to a JMenu, use the following: 

          
           
JMenu myMenu = new JMenu("My Menu");
myMenu.add(myMenuItem);

This adds the myMenuItem JMenuItem to the myMenu JMenu.

Setting an Accelerator for a JMenuItem

An accelerator is a shortcut key combination that will trigger the same action as selecting the JMenuItem from the menu. To set an accelerator for a JMenuItem, use the following:


          
           
myMenuItem.setAccelerator(KeyStroke.getKeyStroke(KeyEvent.VK_S, InputEvent.CTRL_DOWN_MASK));

This sets the shortcut key combination to CTRL + S.

Enabling/Disabling a JMenuItem

The enabled state of a JMenuItem determines if it can be selected by the user or not. To enable a JMenuItem, use the following:


          
           
myMenuItem.setEnabled(true);
To disable a JMenuItem, use the following: 

          
           
myMenuItem.setEnabled(false);

Setting tool tips for a JMenuItem

Tool tips can be set for JMenuItems so that users can gain a better understanding of what the menu item is for. To set a tool tip for a JMenuItem, use the following:


          
           
myMenuItem.setToolTipText("This is the tool tip text");
