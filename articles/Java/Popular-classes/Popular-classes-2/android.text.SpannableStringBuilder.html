Introduction to Android Text SpannableStringBuilder

The Android Text SpannableStringBuilder provides an efficient way of creating and manipulating styled strings. It is an in-memory object that is used to store and manipulate a character sequence with underlying set of styling information. To create a new instance of the Android Text SpannableStringBuilder class, you must first get a CharSequence object. The CharSequence can be any string, including an empty one. CharSequence input = "Hello World!";
