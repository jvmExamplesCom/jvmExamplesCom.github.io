org.springframework.security.core.Authentication

The org.springframework.security.core.Authentication interface is a core interface in the Spring Security framework that represents an authentication request made by a user. It contains the principal (the user) and their credentials, as well as additional information necessary for the authentication request to be processed.

Overview

The Authentication interface is an abstraction for the authentication request created by the user. It contains two pieces of information: the principal - the identity of the user attempting to authenticate - and the credentials - the proof of the user's identity. The additional information can include details such as a list of granted authorities, user roles, session information, etc.

The Authentication object is also used by other parts of the Spring Security framework to access the details of the authenticated user. For example, the SecurityContextHolder class uses the Authentication object to retrieve the current user's details.

Example

The following code shows a simple example of how an Authentication object can be created:


          
           
UsernamePasswordAuthenticationToken authRequest = 
	new UsernamePasswordAuthenticationToken(username, password);
Authentication authentication = authenticationManager.authenticate(authRequest);
