android.content.pm.ResolveInfo

The 

          
           
android.content.pm.ResolveInfo
class contains information about an activity, service, or broadcast receiver that resolves to an intent filter and is used by an Intent to query the package manager for detailed information about a component.

Constructors


          
           
public ResolveInfo()
This constructor creates an empty ResolveInfo instance.

Methods


          
           
public CharSequence loadLabel(PackageManager pm)
This method returns the label associated with this resolve info. The label is retrieved from the application's resources. 

          
           
public Drawable loadIcon(PackageManager pm)
This method returns the icon associated with this resolve info. The icon is retrieved from the application's resources.

Example

The following example demonstrates how to use 

          
           
android.content.pm.ResolveInfo
. 

          
           
// Get a package manager
PackageManager pm = getPackageManager();

// Construct an intent
Intent intent = new Intent(Intent.ACTION_MAIN);
intent.addCategory(Intent.CATEGORY_LAUNCHER);

// Obtain a list of activities that can handle the intent
List<ResolveInfo> infos = pm.queryIntentActivities(intent, 0);

// Iterate over the list
for (ResolveInfo info : infos) {
    // Get label of activity
    CharSequence label = info.loadLabel(pm);
    // Get icon of activity
    Drawable icon = info.loadIcon(pm);

    // Do something with label and icon
}
