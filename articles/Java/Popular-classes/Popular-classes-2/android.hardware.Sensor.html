Sensor Types

The android.hardware.Sensor class is capable of accessing a variety of different types of sensors, including:

  • Accelerometers - Used to measure acceleration.
  • Gyroscopes - Used to measure rotational velocity and orientation.
  • Barometers - Used to measure atmospheric pressure.
  • Magnetometers - Used to measure magnetism.
