org.apache.http.client.config.RequestConfig

The 

          
           
org.apache.http.client.config.RequestConfig
class provides a way to configure HTTP request parameters. It helps establish the connection timeout, connection request timeout, socket timeout, redirects, and proxies.

Creating a RequestConfig Object

The 

          
           
RequestConfig.Builder
is used to create 
RequestConfigs
. The following example creates a 
RequestConfig
object with a connection timeout of 10 seconds, a connection request timeout of 30 seconds, and a socket timeout of 30 seconds. 

          
           
RequestConfig config = RequestConfig.custom()
    .setConnectionTimeout(10000)
    .setConnectTimeout(30000)
    .setSocketTimeout(30000)
    .build();

Configuring Redirects

The 
RequestConfig
enables you to configure how many redirects should be followed. You can set the maximum number as well as decide whether redirects should be relative or absolute. 

          
           
RequestConfig config = RequestConfig.custom()
    .setMaxRedirects(3)
    .setRelativeRedirectsAllowed(true)
    .build();

Configuring a Proxy

The 
RequestConfig
also supports setting up a proxy server. You can use the 
setProxy
method to set up a proxy. 

          
           
RequestConfig config = RequestConfig.custom()
    .setProxy(new HttpHost("localhost", 8080))
    .build();

You can also set the authentication scheme for a proxy. 


          
           
RequestConfig config = RequestConfig.custom()
    .setProxy(new HttpHost("localhost", 8080))
    .setProxyPreferredAuthSchemes(Arrays.asList(AuthSchemes.BASIC))
    .build();

Using the RequestConfig

Once a 
RequestConfig
object is created, it can be passed to an 
HttpClient
as shown below. 

          
           
HttpClient client = HttpClientBuilder.create().setDefaultRequestConfig(config).build();
It can also be passed as an argument to an 
HttpRequest
object when making an HTTP request. In this case, the 
RequestConfig
object will take precedence over any other settings. 

          
           
HttpGet request = new HttpGet("http://example.com");
request.setConfig(config);
