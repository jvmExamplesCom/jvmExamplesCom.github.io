android.graphics.PorterDuffXfermode

The 

          
           
android.graphics.PorterDuffXfermode
class is a Porter-Duff transfer mode that can be used to composite two images together. In Android, it is used for combining colors and effects in various drawing operations.

Overview

Porter-Duff transfer modes are a way of combining two images together by applying an operation to the pixels of both images. The result is a single image that has the combined effect of both images.

The 

          
           
android.graphics.PorterDuffXfermode
class allows you to specify a transfer mode when drawing on a 
Canvas
. This transfer mode will be applied to each pixel in the source image and the destination image when they are drawn together onto the canvas. There are several different transfer modes available: SRC, SRC_IN, SRC_OUT, SRC_ATOP, DST, DST_IN, DST_OUT, DST_ATOP, OVERLAY, XOR, DARKEN, LIGHTEN, and MULTIPLY.

Example Usage

Using the 

          
           
android.graphics.PorterDuffXfermode
class is simple. You simply create an instance of the class, passing in the desired transfer mode, and then set the instance as the 
Xfermode
on the 
Paint
object before drawing on the 
Canvas
. 

          
           
Paint paint = new Paint();
PorterDuffXfermode xfermode = new PorterDuffXfermode(PorterDuff.Mode.SRC_OUT);
paint.setXfermode(xfermode);
