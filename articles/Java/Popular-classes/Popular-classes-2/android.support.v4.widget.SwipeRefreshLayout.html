android.support.v4.widget.SwipeRefreshLayout

SwipeRefreshLayout is an Android Layout used for implementing the Swipe to Refresh pattern. It allows the user to enable pull-to-refresh functionality on the Views that are bound to it, such as a ListView or RecyclerView.

In order to use SwipeRefreshLayout, the layout must be wrapped in a ViewGroup. This can be done programmatically by creating an instance of the layout and setting its params.


          
           
LinearLayout viewGroup = new LinearLayout(context);
viewGroup.setLayoutParams(new LinearLayout.LayoutParams(
  ViewGroup.LayoutParams.MATCH_PARENT,
  ViewGroup.LayoutParams.MATCH_PARENT
));

// Create an instance of SwipeRefreshLayout
SwipeRefreshLayout refreshLayout = new SwipeRefreshLayout(context);
refreshLayout.setLayoutParams(new ViewGroup.LayoutParams(
  ViewGroup.LayoutParams.MATCH_PARENT,
  ViewGroup.LayoutParams.MATCH_PARENT
));

// Add SwipeRefreshLayout to the viewGroup
viewGroup.addView(refreshLayout);

refreshLayout.setOnRefreshListener(() -> {
  // Do some long running task in the background
});
