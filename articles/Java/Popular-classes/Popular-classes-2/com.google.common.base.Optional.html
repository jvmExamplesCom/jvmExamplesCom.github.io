What Is com.google.common.base.Optional?

com.google.common.base.Optional is a class in the Google Guava library that provides an alternative to using null values. It is used to represent objects that may or may not be present. Below is an example of how to use com.google.common.base.Optional. 

          
           
public class Example {
    public static void main(String[] args) {
        Integer value = null;
        Optional<Integer> optional = Optional.ofNullable(value);

        // Check if the value is present
        if(optional.isPresent()) {
            System.out.println("The value is present");
        } else {
            System.out.println("The value is not present");
        }

        // Get the value from the optional
        Integer val = optional.orElse(0);
        System.out.println("The value is " + val);
    }
}
