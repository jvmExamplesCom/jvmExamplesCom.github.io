org.springframework.security.core.userdetails.UserDetails

The org.springframework.security.core.userdetails.UserDetails is an interface that is used to provide user authentication and authorization services in applications built with the Spring Security framework. It provides an interface for retrieving and managing user information, as well as methods for validating user credentials and roles. This article will discuss the use of this interface and provide code examples of how to use it.

Overview of UserDetails

The UserDetails interface has two main methods: loadUserByUsername() and getAuthorities(). The loadUserByUsername() method takes a username string and returns a UserDetails object corresponding to the username. This object stores all the information about the user, including their username, password, and any assigned roles or authorities. This can be used to authenticate a user and check if they have access to certain views or functions. The getAuthorities() method returns a list of granted authorities, which can be used to limit what functions or views they have access to.

Basic Usage

Using the UserDetails interface is fairly straightforward. First, a class must be created which implements the UserDetails interface. This class must provide an implementation of the loadUserByUsername() and getAuthorities() methods. Here is an example of a basic implementation: 

          
           
public class MyUserDetail implements UserDetails {
 
    private String username;
    private String password;
    private List<GrantedAuthority> authorities;
 
    public MyUserDetail(String username, String password, List<GrantedAuthority> authorities) {
        this.username = username;
        this.password = password;
        this.authorities = authorities;
    }
 
    @Override
    public String getUsername() {
        return username;
    }
 
    @Override
    public String getPassword() {
        return password;
    }
 
    @Override
    public List<GrantedAuthority> getAuthorities() {
        return authorities;
    }
 
    @Override
    public boolean isAccountNonExpired() {
        return true;
    }
 
    @Override
    public boolean isAccountNonLocked() {
        return true;
    }
 
    @Override
    public boolean isCredentialsNonExpired() {
        return true;
    }
 
    @Override
    public boolean isEnabled() {
        return true;
    }
}
