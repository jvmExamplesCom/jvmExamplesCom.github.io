Overview of UserDetails

The UserDetails interface has two main methods: loadUserByUsername() and getAuthorities(). This object stores all the information about the user, including their username, password, and any assigned roles or authorities. The getAuthorities() method returns a list of granted authorities, which can be used to limit what functions or views they have access to.
