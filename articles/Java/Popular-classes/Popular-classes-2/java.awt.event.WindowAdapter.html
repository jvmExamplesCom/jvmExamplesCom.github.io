Java.awt.event.WindowAdapter

Java.awt.event.WindowAdapter provides a useful class for creating Java GUI applications by extending java.awt.Window. It adds some useful event-handling methods, such as windowClosing(), windowOpened(), and windowIconified(). It also defines some abstract methods such as windowClosing() and windowIconified() that can be overriden in order to customize the behaviour of a Window object.

Example 1: WindowAdapter Class

The following example demonstrates how to use the WindowAdapter class. Here, we create a WindowAdapter object and call the windowClosing() method when the user closes the window.


          
           
import java.awt.event.WindowAdapter;

public class MyWindowAdapter extends WindowAdapter {

  public void windowClosing(WindowEvent e) {
    System.out.println("Window closing...");
  }
}

In this example, we override the windowActivated() and windowDeactivated() methods of the WindowAdapter class.

  }

In this example, we create a Frame object and pass an instance of the WindowAdapter class to it.
