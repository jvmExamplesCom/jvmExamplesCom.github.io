android.widget.RadioGroup

The android.widget.RadioGroup is a class that allows the user to select one option from a set of options. The RadioGroup is made up of two or more RadioButton views, each one allowing the user to select one item from a list of items. To use this class, you must create a layout XML file that contains a RadioGroup element. You can also add RadioButtons as child elements of the RadioGroup.

Example Usage


          
           
<RadioGroup
    android:layout_width="match_parent"
    android:layout_height="wrap_content">
    <RadioButton
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="Option 1"/> 
    <RadioButton
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="Option 2"/> 
</RadioGroup>

The above example will create a RadioGroup that contains two RadioButtons, each one with the text “Option 1” and “Option 2” respectively. If a user selects one of the RadioButtons, they will be able to switch between the two.

Methods

The android.widget.RadioGroup class provides several methods which can be used to interact with the RadioGroup. These methods include:

  • clearCheck(): This method clears any selected radio button in the group.
  • check(int): This method sets the specified radio button in the group to be checked.
  • getCheckedRadioButtonId(): This method returns the id of the currently checked radio button.
  • isSelected(): This method returns true if the specified radio button is checked.
  • setOnCheckedChangeListener(): This method registers a callback to be invoked when the checked radio button changes.

Example

To demonstrate the usage of the RadioGroup class, we will create an application that allows a user to select one of two options. When a user selects one of the two options, a Toast message will appear on the screen indicating their choice.


          
           
public class MainActivity extends Activity {

    private RadioGroup mRadioGroup;

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);

        mRadioGroup = (RadioGroup) findViewById(R.id.radio_group);
        mRadioGroup.setOnCheckedChangeListener(new RadioGroup.OnCheckedChangeListener() {
            @Override
            public void onCheckedChanged(RadioGroup group, int checkedId) {
                RadioButton radioButton = (RadioButton) group.findViewById(checkedId);
                Toast.makeText(MainActivity.this, radioButton.getText(), Toast.LENGTH_SHORT).show();
            }
        });
    }
}

The above code creates an instance of the RadioGroup class and sets an OnCheckedChangeListener to it. This listener will be called whenever the user selects a different radio button. When the listener is triggered, it will show a Toast message with the text of the selected radio button.

The RadioGroup is a powerful and useful class which makes it easy for developers to create interfaces that allow users to select an option from a set of choices.
