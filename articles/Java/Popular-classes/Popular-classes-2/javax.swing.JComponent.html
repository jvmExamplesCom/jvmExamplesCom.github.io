javax.swing.JComponent

javax.swing.JComponent is a Swing GUI component that allows developers to create a rich user interface for applications. JComponent is the base class for all Swing components, and provides a wide range of functionality including event handling, layout management, and painting. It also provides a number of convenience methods and properties to help swing programmers implement their components.

Creating a JComponent

Creating a JComponent is easy with the javax.swing.JComponent class. All that is required to create a new instance is a constructor call. The following example creates a new instance of a JComponent:


          
           
JComponent jc = new JComponent();

In addition to the constructor, there are a few other methods that can be used when creating a JComponent. The setLayout() and setPreferredSize() methods can be used to set a layout and size for the JComponent. The add() method can be used to add a child component to the JComponent.


          
           
// Set a layout and preferred size
jc.setLayout(new FlowLayout());
jc.setPreferredSize(new Dimension(200, 100));
 
// Add a child component
JButton btn = new JButton("Click Me!");
jc.add(btn);

jc.addMouseListener(new MouseAdapter() {

Painting

protected void paintComponent(Graphics g) {
