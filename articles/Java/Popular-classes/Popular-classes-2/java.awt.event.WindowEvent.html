Java.awt.event.WindowEvent

Java.awt.event.WindowEvent is an event generated by a window object when their state changes. The events that can be generated by a window are instances of WindowEvent and contain the following fields:

  • WINDOW_ACTIVATED – activated event;
  • WINDOW_CLOSED – closed event;
  • WINDOW_CLOSING – closing event;
  • WINDOW_DEACTIVATED – deactivated event;
  • WINDOW_DEICONIFIED – deiconified event;
  • WINDOW_ICONIFIED – iconified event;
  • WINDOW_OPENED – opened event.

WindowEvent class has two methods for getting source, getSource() and getWindow(). getSource() returns the source of the event (in our case - window object), and getWindow() returns the window that generated the event.

Example code


          
           
WindowEvent event = new WindowEvent(window, WindowEvent.WINDOW_CLOSING);
window.dispatchEvent(event);

The above code example creates a WindowEvent object and then dispatches it using the window’s dispatchEvent() method. This will cause the window to close when the event is dispatched.
