Java.nio.CharBuffer

Java.nio.CharBuffer is a character buffer class in the Java programming language. This class provides an API for reading and writing characters to and from the underlying byte buffer. CharBuffer charBuffer = CharBuffer.allocate(1024);

resize()
method on the CharBuffer object.

get()
method can be used. This method takes a single parameter, which is the number of characters to read. For example: 

          
           
char[] data = new char[100];
charBuffer.get(data);

The above code reads 100 characters from the CharBuffer and stores them in the 

data

put()
