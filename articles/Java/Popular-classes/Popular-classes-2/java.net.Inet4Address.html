Java.net.Inet4Address

Java.net.Inet4Address is an abstract class that implements the IP address as defined in RFC 790, Inter Network Control Protocol (IP). It is used to represent IP version 4 Addresses. The Inet4Address class provides several methods to manipulate and access the underlying IP address. All of these methods are part of the java.net.InetAddress package.

Creating a new instance

To create a new instance of the Inet4Address class, you need to provide the IP address as a string. The new instance can be created using either the getByName() or getByAddress() method. For example: 

          
           
Inet4Address inet4 = (Inet4Address) Inet4Address.getByName("192.168.0.1");
Inet4Address inet4 = (Inet4Address) Inet4Address.getByAddress(new byte[] {(byte) 192, (byte) 168, 0, 1});

Methods

The Inet4Address class provides several methods to manipulate and access the underlying IP address. The most commonly used are listed below.

getAddress()


          
           
getAddress()
Returns the raw IP address of this InetAddress object. The returned address is in network byte order: the highest ordered byte of the address is in getAddress()[0].

getHostAddress()


          
           
getHostAddress()
Returns the IP address string in text form. This method returns a string such as "127.0.0.1".

getCanonicalHostName()


          
           
getCanonicalHostName()
Returns the fully qualified domain name for this IP address. If the IP address cannot be resolved, no hostname is returned.

getLocalHost()


          
           
getLocalHost()
Returns the InetAddress object for the local host. This is achieved by retrieving the name of the host from the system, then resolving that name into an InetAddress.

getByName()


          
           
getByName(String host)
Takes a string containing an IP address or hostname and returns an InetAddress object.

getAllByName()


          
           
getAllByName(String host)
Takes a string containing an IP address or hostname and returns an array of InetAddress objects. This is useful if a given host has multiple IP addresses associated with it.
