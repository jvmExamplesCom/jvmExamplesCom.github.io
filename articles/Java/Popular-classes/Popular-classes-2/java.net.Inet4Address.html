Java.net.Inet4Address

Java.net.Inet4Address is an abstract class that implements the IP address as defined in RFC 790, Inter Network Control Protocol (IP). It is used to represent IP version 4 Addresses. The Inet4Address class provides several methods to manipulate and access the underlying IP address. All of these methods are part of the java.net.InetAddress package.

Creating a new instance

To create a new instance of the Inet4Address class, you need to provide the IP address as a string. The new instance can be created using either the getByName() or getByAddress() method. For example: 

          
           
Inet4Address inet4 = (Inet4Address) Inet4Address.getByName("192.168.0.1");
Inet4Address inet4 = (Inet4Address) Inet4Address.getByAddress(new byte[] {(byte) 192, (byte) 168, 0, 1});
