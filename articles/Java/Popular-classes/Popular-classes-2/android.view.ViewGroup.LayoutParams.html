android.view.ViewGroup.LayoutParams

Android ViewGroup.LayoutParams is a class that enables developers to set parameters for views in a layout of an Android application. ViewGroup.LayoutParams params = new ViewGroup.LayoutParams(10dp, 10dp);

The constructors of ViewGroup.LayoutParams allow developers to set margins. This example sets a top margin and left margin. 

          
           
ViewGroup.LayoutParams params = new ViewGroup.LayoutParams(10dp, 10dp, 10dp, 10dp);

params.setGravity(Gravity.CENTER);
