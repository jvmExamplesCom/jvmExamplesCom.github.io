android.view.ViewGroup.LayoutParams

Android ViewGroup.LayoutParams is a class that enables developers to set parameters for views in a layout of an Android application. The ViewGroup.LayoutParams constructor can be used to create an instance of this class, passing in a width and height parameter. This example creates an instance with a width and height of 10dp. 

          
           
ViewGroup.LayoutParams params = new ViewGroup.LayoutParams(10dp, 10dp);

Setting Margins

The constructors of ViewGroup.LayoutParams allow developers to set margins. This example sets a top margin and left margin. 

          
           
ViewGroup.LayoutParams params = new ViewGroup.LayoutParams(10dp, 10dp, 10dp, 10dp);

params.setGravity(Gravity.CENTER);
