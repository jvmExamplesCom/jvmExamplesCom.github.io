java.time.ZonedDateTime

Java 8 introduced the 

          
           
java.time.ZonedDateTime
class, which provides a way of representing a date and time in respect to a certain load zone. These classes are immutable and thread safe, making them incredibly useful for avoiding race conditions when they are used in multi-threaded applications.

Example 1 - Create a ZonedDateTime

The following example shows how to create a 
ZonedDateTime
instance referencing the given time in the UTC time zone. 

          
           
ZonedDateTime zdt = ZonedDateTime.of(2020, Month.JULY, 10, 20, 15, 55, 0, ZoneOffset.UTC);
System.out.println(zdt);
The output of this example is: 

          
           
2020-07-10T20:15:55Z

Example 5 - Combine Date and Time with Zone to ZonedDateTime

The following example shows how to combine a 
LocalDate
and 
LocalTime
with a time zone to create a 
ZonedDateTime
: 

          
           
LocalDate date = LocalDate.of(2020, Month.JULY, 10);
