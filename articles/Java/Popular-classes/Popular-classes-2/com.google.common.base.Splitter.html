com.google.common.base.Splitter

The com.google.common.base.Splitter is a class from the Java library that allows a string to be split into tokens. It provides a simple way for a programmer to break up a string into token components that can then be used for further processing. This article will explain the different types of Splitter available and provide examples of how to use each type.

Simple Splitting

To split a string using the simple Splitter, simply create an instance of the Splitter with the desired delimiter. For example:


          
           
Splitter splitter = Splitter.on(" ");

This will create the Splitter object with a single space as the delimiter. To split a string, the 

split()
method must be called: 

          
           
List<String> strings = splitter.split("Hello World");

This will return a list of strings with each item containing one word. The output of this example is 

["Hello", "World"]
.

Regex Splitting

The Splitter class also supports splitting strings using regular expressions. To use this, the 

onPattern()
method must be called with the desired regex pattern: 

          
           
Splitter splitter = Splitter.onPattern("\\s+");

This will create the Splitter object with any number of whitespace characters as the delimiter. To split a string, the 

split()
method must still be called: 

          
           
List<String> strings = splitter.split("Hello  World");
This will return the same output as before: 
["Hello", "World"]
. Note that the regex version is more powerful and can match multiple whitespace characters.

Multiple Delimiters

The Splitter class also supports splitting strings on multiple delimiters. To do this, the 


          
           
withKeyValueSeparator()
method must be used. This method takes a regular expression as its argument. For example: 

          
           
Splitter splitter = Splitter.withKeyValueSeparator("\\s+|=");

This will create the Splitter object with whitespace or equal sign as the delimiters. To split a string, the 

split()
method must be called: 

          
           
Map<String, String> strings = splitter.split("a=1 b=2 c=3");

This will return a map of strings with each key-value pair separated by either whitespace or an equal sign. The output of this example is 


          
           
{"a": "1", "b": "2", "c": "3"}
.

Omitting Empty Tokens

By default, the Splitter class will include empty tokens when a string is split. To omit these empty tokens, the 

omitEmptyStrings()
method must be called: 

          
           
Splitter splitter = Splitter.on(",").omitEmptyStrings();

This will create the Splitter object with a comma as the delimiter, but it will omit any empty tokens. To split a string, the 

split()
method must be called: 

          
           
List<String> strings = splitter.split(",a,,b,,");

This will return the list of strings without any empty tokens. The output of this example is 

["a", "b"]
.

Trimming Tokens

The Splitter class also provides a mechanism for trimming the tokens it returns. To enable this, the 

trimResults()
method must be called: 

          
           
Splitter splitter = Splitter.on(" ").trimResults();

This will create the Splitter object with a single space as the delimiter, but it will trim all leading and trailing whitespace characters from each token. To split a string, the 

split()
method must be called: 

          
           
List<String> strings = splitter.split("Hello  World  ");

This will return the list of strings with all leading and trailing whitespace characters trimmed. The output of this example is 

["Hello", "World"]
.

Conclusion

The com.google.common.base.Splitter is a powerful utility class for splitting strings into tokens. It supports multiple delimiters, omitting empty tokens, and trimming tokens. With these features, the Splitter can be used to easily parse input strings into meaningful tokens.
