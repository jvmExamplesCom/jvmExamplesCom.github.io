android.text.InputType

Android.text.InputType is a class in the Android operating system that can be used to set the type of input expected when entering text into an EditText. It can also be used to allow certain types of autocomplete suggestions, such as contact names or phone numbers. This tutorial will discuss how to use android.text.InputType for various types of text inputs, as well as how to customize the autocomplete suggestions.

Using android.text.InputType

Using android.text.InputType is relatively straightforward. All that is needed is to set the type of input that is expected via the setInputType() method. This is done using one or more of the type constants defined in this class. The different types of inputs that can be set via this method are: 

          
           
TYPE_CLASS_TEXT;
TYPE_TEXT_VARIATION_EMAIL_ADDRESS;
TYPE_TEXT_VARIATION_PASSWORD;
TYPE_NUMBER_VARIATION_NORMAL;
TYPE_NUMBER_VARIATION_PASSWORD;
TYPE_TEXT_VARIATION_URI;
TYPE_TEXT_FLAG_MULTI_LINE.
The following example shows how to set these various types of inputs on an EditText view. 

          
           
EditText editText = (EditText) findViewById(R.id.editText);
// Set the type of input expected to be a email address
editText.setInputType(InputType.TYPE_CLASS_TEXT | InputType.TYPE_TEXT_VARIATION_EMAIL_ADDRESS);
// Set the type of input expected to be a password
editText.setInputType(InputType.TYPE_CLASS_TEXT | InputType.TYPE_TEXT_VARIATION_PASSWORD);
// Set the type of input expected to be a normal number
editText.setInputType(InputType.TYPE_CLASS_NUMBER | InputType.TYPE_NUMBER_VARIATION_NORMAL);
// Set the type of input expected to be a password number
editText.setInputType(InputType.TYPE_CLASS_NUMBER | InputType.TYPE_NUMBER_VARIATION_PASSWORD);
// Set the type of input expected to be a Uri
editText.setInputType(InputType.TYPE_CLASS_TEXT | InputType.TYPE_TEXT_VARIATION_URI);
// Set the type of input expected to be multi-line
editText.setInputType(InputType.TYPE_CLASS_TEXT | InputType.TYPE_FLAG_MULTI_LINE);
Once the input type has been set, the EditText will automatically display the correct type of keyboard input when it is displayed.

setAutocompleteHint()

String[] contactNames = {"John Smith","Jane Doe","John Doe"};
