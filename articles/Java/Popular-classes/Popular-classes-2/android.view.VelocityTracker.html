What is android.view.VelocityTracker?

Android.view.VelocityTracker is a class used to track the velocity of a finger or stylus which is in contact with a touch screen. This class can be used to compute the velocity of the user's finger or stylus quickly during each frame. It has various methods available that can be used to retrieve the velocity on both axes (horizontal and vertical).

Methods

  • clear()

computeCurrentVelocity(int units)
    : Computes the current velocity based on the points that have been collected. 
  • recycle()
    : Releases resources associated with the VelocityTracker object.
