Introduction to android.text.Spannable

android.text.Spannable is an object that is used to enable span-level styling of text within a CharSequence or Spannable. These spans can be used to apply formatting to make text look bold, italicised, underlined, crossed out or even give it a different colour. It can also be used to apply a clickable span to a string for use when building a UI or for text selection.

Creating a new Spannable Instance

The following code creates a new instance of the Spannable class: 

          
           
Spannable spannable = new SpannableString("This is a spannable string");

Setting a Span on a Substring

The following code sets a span on a substring of the given spannable instance: 

          
           
spannable.setSpan(new StyleSpan(Typeface.BOLD), 0, 6, Spannable.SPAN_EXCLUSIVE_EXCLUSIVE);
In this example, the span is a StyleSpan with a Typeface of BOLD, and it is applied to a substring that starts at index 0 and ends at index 6. The final parameter specifies how the span relates to other spans in the same spannable instance.

Retrieving Spans from a Spannable Instance

The following code retrieves the spans from a Spannable instance: 

          
           
Object[] spans = spannable.getSpans(0, spannable.length(), Object.class);
In this example, all spans are retrieved from the spannable, starting at index 0 and ending at the end of the spannable. The final parameter specifies which type of span should be retrieved.

Removing a Span from a Spannable Instance

The following code removes a span from a spannable instance: 

          
           
spannable.removeSpan(span);
In this example, the span is removed from the spannable instance.

Conclusion

android.text.Spannable is an object used to enable spans of text to have custom styling applied to them. These spans can be used to make text look bold, underline, italicised, crossed out and more. It can also be used to apply a clickable span to a string for UI interactions or for text selection.
