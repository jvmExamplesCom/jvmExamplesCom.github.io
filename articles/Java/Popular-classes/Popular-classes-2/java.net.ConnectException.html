java.net.ConnectException

The java.net.ConnectException is a subclass of the java.io.IOException class. It signals that an error has occurred while attempting to connect a socket to a remote address and port. A ConnectException is thrown when connection is refused or failed for some reason. This exception can also be thrown when a connection timeout occurs during the connection establishment.

Example 1 – ConnectException due to Invalid Network Setting


          
           
try
{
    // Create a URL and open a connection
    URL url = new URL("https://www.example.com");
    HttpURLConnection conn = (HttpURLConnection) url.openConnection();

    // Print response
    System.out.println(conn.getResponseCode());
}
catch (ConnectException ex) {
    System.out.println("A network error occurred: " + ex.getMessage());
}
catch (MalformedURLException ex) {
    System.out.println("Invalid URL provided: " + ex.getMessage());
}
catch (IOException ex) {
    System.out.println("IOException occurred: " + ex.getMessage());
}

The above code tries to create a connection to a URL and print the response code. If the connection fails due to an invalid network setting, a ConnectException is thrown with message "A network error occurred: Connection refused: connect" where "Connection refused: connect" is the detail message.

Example 2 – ConnectException due to Connection Timeout


          
           
try
{
    // Create a URL and open a connection
    URL url = new URL("https://www.example.com");
    HttpURLConnection conn = (HttpURLConnection) url.openConnection();

    // Set read timeout to 5 seconds
    conn.setReadTimeout(5000);

    // Print response
    System.out.println(conn.getResponseCode());
}
catch (ConnectException ex) {
    System.out.println("A connection error occurred: " + ex.getMessage());
}
catch (MalformedURLException ex) {
    System.out.println("Invalid URL provided: " + ex.getMessage());
}
catch (IOException ex) {
    System.out.println("IOException occurred: " + ex.getMessage());
}

The above code tries to create a connection to a URL and set the read timeout to 5 seconds. If this connection fails to establish within the given timeout period, a ConnectException is thrown with message "A connection error occurred: Connection timed out: connect" where "Connection timed out: connect" is the detail message.

Example 3 – How to Handle the ConnectException


          
           
// Create a URL and open a connection
URL url = new URL("https://www.example.com");
HttpURLConnection conn = (HttpURLConnection) url.openConnection();

// Try up to 5 times before giving up
for (int i = 0; i < 5; i++) {
    try {
        // Set read timeout to 5 seconds
        conn.setReadTimeout(5000);

        // Print response
        System.out.println(conn.getResponseCode());

        // Connection established, exit the loop
        break;
    } catch (ConnectException ex) {
        if (i == 4) {
            // Reached maximum retries, throw exception
            throw ex;
        } else {
            // Wait 1 second before retrying
            Thread.sleep(1000);
        }
    }
}

In the above code, we’ve used a try-catch block to handle any ConnectException that might occur while connecting to the remote server. We’ve also implemented a retry logic which will try to connect up to 5 times with a 1 second delay between each retry before giving up.
