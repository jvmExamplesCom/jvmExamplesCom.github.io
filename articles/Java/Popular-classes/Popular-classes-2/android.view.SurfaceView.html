android.view.SurfaceView

The android.view.SurfaceView class is a view that provides a dedicated drawing surface embedded inside of a view hierarchy. This is useful for drawing graphics that are not possible with the standard View system, such as animations, game graphics, or video playback. The surface can be associated with a SurfaceHolder object which provides access and control over the underlying surface.

Creating a Surface View

These methods will be executed when the surface is created or destroyed, and when it is changed. You can use them to handle these events.         super(context);

        getHolder().addCallback(this);

        // update size of drawing surface here
    }

        // destroy drawing surface here
    }
}
