android.view.SurfaceView

The android.view.SurfaceView class is a view that provides a dedicated drawing surface embedded inside of a view hierarchy. This is useful for drawing graphics that are not possible with the standard View system, such as animations, game graphics, or video playback. Creating a Surface View

        getHolder().addCallback(this);

}
