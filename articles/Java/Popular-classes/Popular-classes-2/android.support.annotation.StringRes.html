Java: android.support.annotation.StringRes

The 

          
           
android.support.annotation.StringRes
is a class in the Android framework used to represent resources that resolve to a string value. It is primarily used in the Android Studio code editor to indicate that an argument is of type string reference, allowing it to be autocompleted with resource identifiers such as @string/abc or R.string.abc.

Syntax


          
           
@StringRes 
int resId;

Parameters

  • resId
    : The resource ID of the string reference.

Examples

Example 1

To declare a resource that is of type string reference, use the 

@StringRes
annotation before the declaration: 

          
           
@StringRes
int stringRef;

Example 2

Then assign the resource identifier to the variable:


          
           
stringRef = R.string.hello_world;

Example 3

And display it on the app UI by passing the resource ID to the textview's setText() method:


          
           
TextView textView = findViewById(R.id.text);
textView.setText(stringRef);
