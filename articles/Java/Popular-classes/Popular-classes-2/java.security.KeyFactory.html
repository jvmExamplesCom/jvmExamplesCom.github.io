Java.security.KeyFactory

Java.security.KeyFactory is a class that enables applications to generate key instances from their encoded form, and also to convert keys into an encoded form suitable for maintaining the security of the key. This class acts as a central point of entry to access the functionality needed to create a KeyFactory object.

Creating a KeyFactory Object

The following code example shows how to create a KeyFactory Object: 

          
           
KeyFactory keyFactory = KeyFactory.getInstance("RSA");
The getInstance() method is used to specify the desired algorithm. In this case we are specifying the RSA algorithm.

Generating Public and Private Keys

Java.security.KeyFactory allows users to generate a public and a private key from the key specifications. The code snippet below shows how to do this using the generatePublic() and generatePrivate() methods: 

          
           
// Generate a public key
PublicKey pubKey = keyFactory.generatePublic(spec);

// Generate a private key
PrivateKey priKey = keyFactory.generatePrivate(spec);

Encoding and Decoding Keys

Java.security.KeyFactory also provides methods to encode and decode keys. The getEncoded() method can be used to encode a key in byte array format. The method is shown in the code snippet below: 

          
           
byte[] encodedPubKey = pubKey.getEncoded();
The getKeySpec() method can be used to decode a key. The code snippet below shows how to do this: 

          
           
RSAPublicKeySpec publicKeySpec = keyFactory.getKeySpec(pubKey, RSAPublicKeySpec.class);

The getKeySpec() method takes two parameters, the key being decoded and the type of key spec being used to decode the key. In this case we are using RSAPublicKeySpec and RSAPrivateKeySpec.
