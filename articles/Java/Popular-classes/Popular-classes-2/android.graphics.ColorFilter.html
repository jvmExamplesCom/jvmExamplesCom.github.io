What is java.android.graphics.ColorFilter?

How java.android.graphics.ColorFilter Works

The ColorFilter class contains several methods that allow developers to manipulate and process existing colors. Some of the available methods include getColorMatrix() and setColorFilter(). The getColorMatrix() method returns a matrix containing the current color filter values for the instance being used. The setColorFilter() method accepts a ColorFilter object and passes it to the canvas’s drawing operation wherein the color filter will be applied.
