android.media.MediaPlayer

Android's MediaPlayer class provides a way to control playback of audio/video files and streams. It can play audio and video both locally and over the internet. This article will discuss how to use this class to control your media, as well as provide code examples to demonstrate its use.

Setting Up

MediaPlayer can be instantiated in two ways. You can either pass in a source Uri or a descriptor of an existing media player. Here's an example for creating a MediaPlayer from a source Uri:


          
           
Uri sourceUri = Uri.parse("http://example.com/example.mp3");
MediaPlayer player = new MediaPlayer();
player.setDataSource(sourceUri);

You can also create a MediaPlayer from an existing media player descriptor by calling its 

create()
method: 

          
           
MediaPlayer player = MediaPlayer.create(context, R.raw.example);

Playing and Pausing Media

Once you have created your MediaPlayer, you can control its playback with the 

start()
, 
pause()
, and 
stop()
methods. Here are examples of each one: 

          
           
player.start();
player.pause();
player.stop();

Looping Audio

The MediaPlayer class provides a way to loop audio by setting the 

setLooping()
flag on the player. This will cause the audio to repeat indefinitely until stopped. Here is an example: 

          
           
player.setLooping(true);

Seeking Audio

The MediaPlayer class also provides a way to seek within a playing audio file or stream. To do this, you can call the 

seekTo()
method with a specified position in milliseconds. Here is an example: 

          
           
player.seekTo(5000); // Seek to 5 seconds

Releasing Resources

When you are done using the MediaPlayer instance, it is important to call its 

release()
method to free up its resources. Failure to do so can cause memory leaks in your application. Here is an example of the release method being called: 

          
           
player.release();

That's all there is to using the MediaPlayer class! With these few simple methods, you can easily control the playback of any audio/video files or streams.
