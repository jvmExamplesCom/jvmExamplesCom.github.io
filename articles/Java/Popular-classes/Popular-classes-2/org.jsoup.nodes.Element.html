org.jsoup.nodes.Element Overview

The org.jsoup.nodes.Element class provides a representation of an HTML element. It provides methods for accessing and manipulating the element, such as selecting elements by id or class, adding or removing attributes, or even setting the inner HTML of the element. This class is central to jsoup, representing both a parsed HTML element and an element that is being constructed. In this article, we will look at the various features and methods available on the Element class.

Creating an Element

An Element can be created either by parsing it from a HTML string, or by constructing it directly with the constructor. Here's an example of both methods:


          
           
// parse from HTML string
String htmlString = "<div id="myDiv">This is my div</div>";
Element myDiv = Jsoup.parse(htmlString);

// create from scratch
Element p = new Element("p");
p.attr("id", "myParagraph")
    .text("This is my paragraph");

The example shows two ways of creating an Element object. The first creates an Element by parsing a HTML string. The second creates an element by manually constructing it with its constructor.

Accessing Attributes

Elements have several attributes associated with them, such as their 

id
, 
class
, or 
style
. These attributes can be accessed with the 
attr()
method. Here’s an example of how to access an attribute and modify it: 

          
           
// get the element's id
String id = myDiv.attr("id");

// set the element's style
myDiv.attr("style", "background-color: #ff0");
The example accesses the 
id

attr()

style

Manipulating Content

Elements also allow you to manipulate their content, which includes their text and HTML. The 

text()
method allows you to access and modify the element's text content. Here's an example of getting and setting an element's text: 

          
           
// get the element's text
String text = myDiv.text();

// set the element's text
myDiv.text("This is my new text");
The example accesses the element's text with the 
text()
method, and sets a new text with the same method. You can also manipulate an element's HTML with the 
html()
method. Here's an example of getting and setting an element's HTML content: 

          
           
// get the element's html
String html = myDiv.html();

// set the element's html
myDiv.html("<span>This is my new html</span>");
The example accesses the element's HTML with the 
html()
method, and sets a new HTML with the same method. Note that both the 
text()
and 
html()
methods support setting and getting the element's content.

Conclusion

The org.jsoup.nodes.Element class provides a representation of an HTML element, with methods for accessing and manipulating the element.
