android.view.SurfaceHolder

The android.view.SurfaceHolder class is a mechanism to control the Android surface's properties such as its size and format, edit the pixels on the surface, and monitor notifications of changes to the surface. It is the owner of the surface and it gives access to the underlying surface through its various methods. A SurfaceHolder object can be created either manually by constructing it directly or as part of a view.

Example Usage

A basic usage example for the SurfaceHolder class looks like the following snippet, where a SurfaceView object is first created, then the SurfaceHolder object is retrieved from the view, and finally an addCallback() method is called to register a callback object that will be notified of changes to the surface:


          
           
SurfaceView mySurfaceView = new SurfaceView(context);
SurfaceHolder myHolder = mySurfaceView.getHolder();
myHolder.addCallback(myCallback);

Methods

The SurfaceHolder class provides methods to create a surface and its associated Canvas, which can then be used to draw on the screen. These methods are:

  • 
            
             
public void setFixedSize(int width, int height)
    : Sets the size of the canvas used to draw on the surface.
  • 
            
             
public void setFormat(int format)
    : Sets the pixel format of the canvas used to draw on the surface.
  • 
            
             
public Canvas lockCanvas()
    : Locks the canvas associated with this surface and returns a Canvas object that can be used to draw on the surface.
  • 
            
             
public void unlockCanvasAndPost(Canvas canvas)
    : Unlocks the canvas associated with this surface and posts it to the screen.
  • 
            
             
public void addCallback(SurfaceHolder.Callback cb)
    : Registers a callback object that will be notified of changes to the surface.

Example

Here is an example showing how to use the SurfaceHolder object to create a canvas and draw a circle on it:


          
           
//Create the surface
SurfaceView mySurfaceView = new SurfaceView(context);
SurfaceHolder myHolder = mySurfaceView.getHolder();

//Lock the canvas
Canvas myCanvas = myHolder.lockCanvas();

//Draw the circle
Paint myPaint = new Paint();
myPaint.setColor(Color.BLUE);
myCanvas.drawCircle(50, 50, 30, myPaint);

//Unlock the canvas and post it to the screen
myHolder.unlockCanvasAndPost(myCanvas);

This example should result in a blue circle being drawn on the screen at the coordinates (50,50) with a radius of 30.
