Introduction to Android Support V7 App ActionBarDrawerToggle

Android Support V7 App ActionBarDrawerToggle provides a convenient way for users to switch between the main navigation drawer and back. It provides an ActionBar-styled toggle at the top of the screen, allowing for easy switching between the main navigation drawer and its various other screens.

Using the ActionBarDrawerToggle

The ActionBarDrawerToggle can be used in any activity, as long as it is connected to a navigation drawer. To use the ActionBarDrawerToggle, you must first initialize it and set the appropriate callbacks. The following code snippet shows an example of such initialization: 

          
           
ActionBarDrawerToggle actionBarDrawerToggle = new ActionBarDrawerToggle(this, mDrawer, toolbar, R.string.open_drawer, R.string.close_drawer);
mDrawer.addDrawerListener(actionBarDrawerToggle);
actionBarDrawerToggle.syncState();
The above code initializes a new ActionBarDrawerToggle and adds it as a DrawerListener to the navigation drawer. Finally, it syncs the state of the ActionBarDrawerToggle so that it displays the correct icon on the toolbar.

Using the ActionBarDrawerToggle with the Toolbar

This allows for the user to switch between the main navigation drawer and the Toolbar without having to manually press the toggle button. To use the ActionBarDrawerToggle with a Toolbar, you must first initialize it, as shown in the following code snippet: 

          
           
ActionBarDrawerToggle actionBarDrawerToggle = new ActionBarDrawerToggle(this, mDrawer, toolbar, R.string.open_drawer, R.string.close_drawer);
actionBarDrawerToggle.syncState();
toolbar.setNavigationOnClickListener(new View.OnClickListener() {
    @Override

        }
    }
});
The above code snippet initializes a new ActionBarDrawerToggle and sets a Navigation OnClickListener to the Toolbar. This listener will open or close the navigation drawer when the Toolbar is pressed.

@Override
public boolean onOptionsItemSelected(MenuItem item) {

        case android.R.id.home:

            return true;

private void toggle() {
    if (mDrawer.isDrawerOpen(GravityCompat.START)) {
