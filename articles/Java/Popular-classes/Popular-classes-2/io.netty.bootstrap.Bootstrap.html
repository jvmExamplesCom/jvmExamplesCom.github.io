Introduction to io.netty.bootstrap.Bootstrap

What is Bootstrap?

Bootstrap is an asynchronous event-driven network library for rapid development of maintainable network applications, such as client and server applications. Bootstrap is highly configurable and extensible as well, allowing developers to customize the library to their specific requirements.

Bootstrap consists of two main components, namely the ClientBootstrap and ServerBootstrap. ClientBootstrap is used to initialize the client socket, while ServerBootstrap is used to initialize the server socket.
