Java.math.RoundingMode

The Java.math.RoundingMode class provides a set of predefined constants that define the rounding policy used to round different numerical operations. This class is designed to support operations such as exact equality comparisons, sorting, hashing and range checks. The values of these constants may be changed by the compiler when compiling code, so applications should not rely on the values.
