org.apache.commons.cli.Options

The Options class of the 

          
           
org.apache.commons.cli
package is a simple way to parse command line options and arguments. It enables users to define, parse and manipulate the options that could be provided when the application is run.

Options provides a way to manage command line options by describing them as name-value pairs with an allowed list of values. Each option can have either one or more values specified. The object's static methods are used to add and remove options, retrieve the current value of an option and parse strings containing the values of the options.

Creating and Adding Options

Options are created by simply passing in a name and a list of values that are acceptable for the option:


          
           
Options options = new Options();
options.addOption('h', "help", false, "Print help");
This code creates a new 
Options
object and adds an option named 
help
that takes no argument and has no values. For example, when the application is executed, it can accept the argument 
-h
or 
--help
to print help.

Retrieving Values

To retrieve the value of an option, one must call the Options' 

getOptionValue()
method: 

          
           
String value = options.getOptionValue("help");
This code returns the value set for the 
help
option (in this case, 
false
).

Parsing Strings

When an application is run, it can accept arguments and options in the form of strings. To parse these strings, one can use the Options' 

parse()
method: 

          
           
String[] args = {"--help"};
CommandLine cmdLine = options.parse(args);
System.out.println("help: " + cmdLine.hasOption("help")); // prints "help: true"
This code parses a string containing the option 
--help
. The result is then stored in a 
CommandLine
instance, which can be queried to determine if the option was present.

Conclusion

The 

          
           
org.apache.commons.cli.Options
class provides a simple way to parse and manage command line options. The API allows users to add and remove options, retrieve the value of an option and parse the values of options from a string. Understanding this API can greatly simplify the process of creating applications that can accept command line options and arguments.
