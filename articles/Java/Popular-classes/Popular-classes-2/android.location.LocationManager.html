android.location.LocationManager

The android.location.LocationManager class is responsible for providing access to location services on the Android device. It is a system service that provides an interface to use the location based services of the device. The LocationManager will allow the application to register location-based listeners, query the network and GPS for location data, acquire geocoded addresses, and manage the power usage of the location providers.

Using LocationManager

The Location Manager is accessed through the getSystemService() method. To use the LocationManager, you will need to pass in the Context.LOCATION_SERVICE constant as the argument to getSystemService():


          
           
LocationManager lm = (LocationManager) this.getSystemService(Context.LOCATION_SERVICE);

Once you have retrieved the LocationManager object, you can then register for location updates by calling requestLocationUpdates(), which takes in a location provider, minimum time interval between location updates, minimum distance between locations, and a LocationListener as parameters:


          
           
LocationListener locationListener = new MyLocationListener();
lm.requestLocationUpdates( LocationManager.GPS_PROVIDER, 0, 0, locationListener);

You can also use the getLastKnownLocation() method to get the last known location from the GPS or the Network, without registering for location updates:


          
           
Location location = lm.getLastKnownLocation(LocationManager.GPS_PROVIDER);

Creating a LocationListener

public abstract void onLocationChanged(Location location);
public abstract void onStatusChanged(String provider, int status, Bundle extras);
public abstract void onProviderEnabled(String provider);
public abstract void onProviderDisabled(String provider);
