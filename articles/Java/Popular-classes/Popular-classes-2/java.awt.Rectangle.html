java.awt.Rectangle

The java.awt.Rectangle class is a java class used to define rectangles and provides methods for computing the area, calculating the perimeter and manipulating the height and width of rectangles. This class is a subclass of the java.awt.geom.Rectangle2D class and implements the java.awt.Shape interface.

Creating a Rectangle Instance

To create a Rectangle instance we can call one of the constructors available on this class:


          
           
Rectangle r = new Rectangle(x, y, width, height);

This constructor creates an instance of the java.awt.Rectangle class with the specified parameters. The 

x
and 
y
parameters represent the coordinates of the upper left corner of the rectangle while the 
width
and 
height
parameters represent the width and height of the rectangle.

Calculating the Area and Perimeter

Once we have a java.awt.Rectangle instance, we can use its methods to calculate the area and perimeter of the rectangle:


          
           
double area = r.getArea();
double perimeter = r.getPerimeter();
The 
getArea()
method calculates the area of the rectangle while the 
getPerimeter()
method calculates the perimeter of the rectangle.

Manipulating Height and Width

To manipulate the height and width of a java.awt.Rectangle instance we can use the following methods:


          
           
r.setHeight(newHeight);
r.setWidth(newWidth);
The 
setHeight()
method sets the height of the rectangle to the given parameter while the 
setWidth()
method sets the width of the rectangle to the given parameter.

Example

Let's look at a simple example of using the java.awt.Rectangle class:


          
           
Rectangle r = new Rectangle(0, 0, 10, 20);

double area = r.getArea();
double perimeter = r.getPerimeter();

System.out.println("Area = " + area);
System.out.println("Perimeter = " + perimeter);

r.setHeight(50);
r.setWidth(20);

area = r.getArea();
perimeter = r.getPerimeter();

System.out.println("New Area = " + area);
System.out.println("New Perimeter = " + perimeter);

Area = 200.0

New Area = 1000.0
New Perimeter = 140.0
