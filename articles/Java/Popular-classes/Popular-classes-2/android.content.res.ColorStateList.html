android.content.res.ColorStateList

The android.content.res.ColorStateList class is a resource used to define a set of colors and their associated states that can be used by various widgets throughout an application. This color state list is defined in XML and can contain references to other resources, such as drawables. It’s most commonly used in the action bar for drawing a background with different colors for the various states (pressed,focused,normal, etc).

Creating a ColorStateList

Creating a ColorStateList is simple, and requires only a few lines of code. Here’s an example of creating a ColorStateList with two different colors, one for focused and one for normal: 

          
           
ColorStateList mylist = new ColorStateList(
    new int[][]{
        new int[] {android.R.attr.state_focused},
        new int[] {}
    },
    new int[]{
        Color.RED,
        Color.BLUE
    }
);
This example will create a ColorStateList where focused elements are colored red and normal elements are colored blue.

Using a ColorStateList

Once you have created a ColorStateList, you can use it in various places in your application. For example, if you wanted to set the background of a layout to use the ColorStateList you just created, you would do the following in the onCreate() method of your Activity or Fragment: 

          
           
LinearLayout myLayout = (LinearLayout) findViewById(R.id.my_layout);
myLayout.setBackgroundColor(mylist);
This will set the background of the LinearLayout to use the ColorStateList you created. It will then apply the appropriate color to each element depending on its state (focused, pressed, etc).
