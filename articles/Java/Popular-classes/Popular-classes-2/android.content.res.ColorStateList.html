android.content.res.ColorStateList

The android.content.res.ColorStateList class is a resource used to define a set of colors and their associated states that can be used by various widgets throughout an application. It’s most commonly used in the action bar for drawing a background with different colors for the various states (pressed,focused,normal, etc).

Creating a ColorStateList

ColorStateList mylist = new ColorStateList(
    new int[][]{

        Color.RED,

LinearLayout myLayout = (LinearLayout) findViewById(R.id.my_layout);
myLayout.setBackgroundColor(mylist);
