Java.util.jar.Manifest

Java.util.jar.Manifest is a set of attributes and entries that are used to provide the necessary information for a JAR (Java Archive) file. It is contained in a file named “MANIFEST.MF” located in the META-INF directory of the JAR file. Creating a Manifest

A Manifest is created using the Manifest class. The Manifest class contains a number of methods which allow you to construct a Manifest from a file, create a Manifest from scratch, or add entries and attributes to an existing Manifest.
