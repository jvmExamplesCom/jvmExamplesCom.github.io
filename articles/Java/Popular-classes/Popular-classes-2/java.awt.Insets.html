Java.awt.Insets

Java.awt.Insets is a class of the Java AWT (Abstract Window Toolkit) that can be used to create empty space between components (such as buttons, text fields and other objects); where it’s better known as “padding”. Instances of this class are created by using the constructor insets(top, bottom, left, right). This class is especially useful for creating UI layout in applications.

In the following example, we instantiate an instance of Insets with no parameters: 


          
           
Insets insets = new Insets();
