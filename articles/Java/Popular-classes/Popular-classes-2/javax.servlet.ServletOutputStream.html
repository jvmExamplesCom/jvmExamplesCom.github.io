javax.servlet.ServletOutputStream

The javax.servlet.ServletOutputStream class provides an output stream for the servlet container to write data to the client. It is used for sending binary data, such as images or other kinds of files, to the client. By default, data written to this stream is buffered and sent out at the end of the request. The javax.servlet.ServletOutputStream class is an abstract class that must be extended to create a concrete implementation.
