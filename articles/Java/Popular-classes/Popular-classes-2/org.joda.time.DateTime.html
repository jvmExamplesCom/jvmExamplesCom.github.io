Introduction to org.joda.time.DateTime in Java

The 

          
           
org.joda.time.DateTime
class is a powerful API for working with dates and times in the Java programming language. This class provides an easy way to manipulate, format, convert, calculate, and compare dates and times. In this article, we will look at how to use this API to perform some of these tasks.

Creating DateTime Objects


          
           
DateTime dt = new DateTime();
This code creates a new DateTime object set to the current date and time. We can also specify a specific date and time when creating the DateTime object. 

          
           
DateTime dt = new DateTime(2020, 7, 1, 12, 0, 0);
This code creates a DateTime object set to July 1, 2020 at 12:00. We can also specify more granular values, such as milliseconds or nanoseconds.

Formatting DateTime Objects

We can format DateTime objects using the 

          
           
DateTimeFormatter
class. For example, we can print out the time in a standard ISO 8601 format by using the 

          
           
ISODateTimeFormat.dateTime()
method. 

          
           
DateTimeFormatter fmt = ISODateTimeFormat.dateTime();
String str = fmt.print(dt);
System.out.println(str);
// prints "2020-07-01T12:00:00.000"
We can also specify a custom format by using 

          
           
DateTimeFormat.forPattern()
. For example, we can print out the date in a month/day/year format using the following code: 

          
           
DateTimeFormatter fmt = DateTimeFormat.forPattern("MM/dd/yyyy");
String str = fmt.print(dt);
System.out.println(str);
// prints "07/01/2020"

Calculating Date Differences

We can calculate the difference between two DateTime objects by using the 

          
           
getMillis()
and 

          
           
minus()
methods. For example, if we wanted to calculate the difference between two dates, one on January 1 and one on June 30, we could do so with the following code: 

          
           
DateTime dt1 = new DateTime(2020, 1, 1, 0, 0, 0);
DateTime dt2 = new DateTime(2020, 6, 30, 0, 0, 0);

long diff = dt2.getMillis() - dt1.getMillis();
System.out.println(diff);
// prints "1583230400000"
We can also use the 

          
           
minus()
method to subtract a specified period of time from a DateTime object. For example, if we wanted to subtract three months from our date on June 30, we could do so with the following code: 

          
           
DateTime dt = new DateTime(2020, 6, 30, 0, 0, 0);
DateTime dt3 = dt.minusMonths(3);
System.out.println(dt3);
// prints "2020-03-31T00:00:00.000"

Adding Time

We can also add time to a DateTime object by using the 

          
           
plus()
method. For example, if we wanted to add three weeks to our date on June 30, we could do so with the following code: 

          
           
DateTime dt = new DateTime(2020, 6, 30, 0, 0, 0);
DateTime dt3 = dt.plusWeeks(3);
System.out.println(dt3);
// prints "2020-07-21T00:00:00.000"

Comparing DateTimes

We can compare two DateTime objects by using the 

          
           
isBefore()
, 

          
           
isAfter()
, and 

          
           
isEqual()
methods. For example, if we wanted to know if our date on June 30 occurred after our date on January 1, we could do so with the following code: 

          
           
DateTime dt1 = new DateTime(2020, 1, 1, 0, 0, 0);
DateTime dt2 = new DateTime(2020, 6, 30, 0, 0, 0);

boolean isAfter = dt2.isAfter(dt1);
System.out.println(isAfter);
// prints "true"
We can also compare two DateTime objects by using the 

          
           
compareTo()
method. This method returns a negative integer if the first DateTime is before the second, a positive integer if the first DateTime is after the second, and zero if they are the same. 

          
           
DateTime dt1 = new DateTime(2020, 1, 1, 0, 0, 0);
DateTime dt2 = new DateTime(2020, 6, 30, 0, 0, 0);

int comparison = dt1.compareTo(dt2);
System.out.println(comparison);
// prints "-1"

Conclusion

In this article, we looked at how to use the 

          
           
org.joda.time.DateTime
class to work with dates and times in the Java programming language. We explored how to create DateTime objects, format them, calculate differences and similarities between them, and compare them. By understanding these concepts, we can take advantage of the many features of this powerful API.
