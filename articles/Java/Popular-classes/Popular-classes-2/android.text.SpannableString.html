android.text.SpannableString

The android.text.SpannableString class is a powerful class that can be used to create customized text views and manipulate text in an Android application. It provides a wide range of features, including styling, colorization, multiple fonts, and the ability to add images and icons.

Creating a SpannableString

A SpannableString can be created by simply calling its constructor: 

          
           
SpannableString spannableString = new SpannableString("Hello World!");

Applying Styles to a SpannableString

Once a SpannableString has been created, styles can be applied to it using the setSpan() method. This method takes three arguments: a start index, an end index, and an Object representing the style (e.g. a ForegroundColorSpan). 

          
           
// Create a SpannableString
SpannableString spannableString = new SpannableString("Hello World!");

// Apply a foreground color of red to the first 5 characters
spannableString.setSpan(new ForegroundColorSpan(Color.RED), 0, 5, 0);

The contents of a SpannableString can be accessed in several ways. To get the total length of the string, the length() method can be used: 

          
           
int len = spannableString.length();
To retrieve a single character, the charAt() method can be used:
