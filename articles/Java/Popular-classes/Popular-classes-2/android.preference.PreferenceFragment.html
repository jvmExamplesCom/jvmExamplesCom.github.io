android.preference.PreferenceFragment

The android.preference.PreferenceFragment class is a specialized Fragment class that provides a bridge between an Activity and the underlying data store of preferences managed by the PreferenceManager. This class also knows how to properly save, restore and display an application’s preference hierarchy.

Creating a Preference Fragment

In order to create a PreferenceFragment in your Activity, you must first define a subclass of PreferenceFragment and override the onCreate method: 

          
           
public class MyPreferenceFragment extends PreferenceFragment {

    @Override
    public void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);

        // Load the preferences from an XML resource
        addPreferencesFromResource(R.xml.my_preferences);
    }

}
Once you have defined your PreferenceFragment, you can then add it to your Activity by adding a FragmentTransaction to the Activity’s FragmentManager: 

          
           
FragmentManager fragmentManager = getFragmentManager();
FragmentTransaction fragmentTransaction = fragmentManager.beginTransaction();
MyPreferenceFragment myPreferenceFragment = new MyPreferenceFragment();
fragmentTransaction.add(android.R.id.content, myPreferenceFragment);
fragmentTransaction.commit();

Managing Preferences

Once you have created your PreferenceFragment, you can then begin managing your application’s preferences. To set a value for a preference, use the PreferenceManager’s setSharedPreferencesName method: 

          
           
PreferenceManager.getDefaultSharedPreferences(getActivity()).
        edit().putString("my_pref", "value").apply();
To get a value for a preference, use the getSharedPreferences method: 

          
           
String value = PreferenceManager.getDefaultSharedPreferences(getActivity()).
        getString("my_pref", "default value");

Listening for Changes

You can also register a listener to be notified when a preference changes: 

          
           
final SharedPreferences.OnSharedPreferenceChangeListener listener =
        new SharedPreferences.OnSharedPreferenceChangeListener() {
            @Override
            public void onSharedPreferenceChanged(
                    SharedPreferences sharedPreferences, String key) {
                // Do something when the preference changes
            }
        };

PreferenceManager.getDefaultSharedPreferences(getActivity())
        .registerOnSharedPreferenceChangeListener(listener);
Once you are done with the listener, you must unregister it to avoid memory leaks: 

          
           
PreferenceManager.getDefaultSharedPreferences(getActivity())
        .unregisterOnSharedPreferenceChangeListener(listener);
