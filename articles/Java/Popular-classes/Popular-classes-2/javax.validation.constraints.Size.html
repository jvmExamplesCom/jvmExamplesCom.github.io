Java javax.validation.constraints.Size

The javax.validation.constraints.Size is an annotation used for elements of a Java Bean that have a size restriction associated with them. This annotation can be applied to a java.lang.String, a java.util.Collection, a java.util.Map, or an array.

This example uses the annotation to validate a String field in a bean.

public class ExampleBean  {
