javax.swing.JButton

A JButton is a component of the Java Swing user interface library for creating a push button within an application’s user interface. This component lets users perform an action or trigger a process when they click it. It is one of the most commonly used components within Swing applications.

Creating a JButton

The simplest way to create a JButton is to use its constructor. The following example creates a JButton object with the text “Click Me”:


          
           
JButton button = new JButton('Click Me');

You can also specify an icon to be used instead of text. The following example uses an image named image.png as the button icon instead of text:


          
           
JButton button = new JButton(new ImageIcon('image.png'));

Adding Action Listeners

An action listener. is an object which can respond to certain events (in this case, a user clicking the button). To add an action listener to the JButton, use the addActionListener() method. The following example adds an action listener to the button created in the previous examples:


          
           
button.addActionListener(new ActionListener() {
    public void actionPerformed(ActionEvent e) {
        // code to be executed on click
    }
});

Setting the Button’s Text or Icon

To set the text of the JButton, use the setText() method. The following example sets the text of the button to “Press Me!”:


          
           
button.setText('Press Me!');

To set the icon of the JButton, use the setIcon() method. The following example sets the icon of the button to an image named button-icon.png:


          
           
button.setIcon(new ImageIcon('button-icon.png'));

Determining if the Button has been Clicked

To determine if the JButton has been clicked by a user, you can use the isSelected() method. This method will return true if the button has been clicked, and false otherwise. The following example checks if the button has been clicked:


          
           
if (button.isSelected()) {
    // code to be executed if the button is clicked
}
