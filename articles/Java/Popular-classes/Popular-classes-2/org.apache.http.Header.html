org.apache.http.Header

org.apache.http.Header
.

Methods of org.apache.http.Header

The Apache HTTP Client library provides several methods for working with request headers. The most commonly used are: 
  • getName()
    – Returns the name of the header.
  • getValue()
    – Returns the value of the header.
  • getElements()
    – This method returns an array of all the elements that the header contains.

org.apache.http.Header

Header

addHeader()

Header h = new Header("MyName", "MyValue");

getHeaders()

setValue()
