Java org.apache.http.Header

Apache HTTP Client library provides the 

          
           
org.apache.http.Header
class which allows developers to work with request and response headers. This class is useful for anything from parsing requests and responses to manipulating them or creating entirely new ones. In this article, we will take a look at some of the most common uses of 

          
           
org.apache.http.Header
.

Methods of org.apache.http.Header

The Apache HTTP Client library provides several methods for working with request headers. The most commonly used are: 
  • getName()
    – Returns the name of the header.
  • getValue()
    – Returns the value of the header.
  • getElements()
    – This method returns an array of all the elements that the header contains.

Creating a Request Header

Request headers can be created using the 

          
           
org.apache.http.Header
class. To do this, we need to first create a 

          
           
Header

addHeader()
method. 

          
           
Header h = new Header("MyName", "MyValue");
request.addHeader(h);

Parsing a Response Header

org.apache.http.Header

getHeaders()
method, which takes a name as a parameter and returns an array of all the matching headers. 

          
           
Header[] headers = response.getHeaders("MyName");

org.apache.http.Header

setValue()

addValue()
