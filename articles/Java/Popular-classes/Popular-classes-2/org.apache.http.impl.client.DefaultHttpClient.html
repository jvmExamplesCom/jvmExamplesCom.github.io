Overview of org.apache.http.impl.client.DefaultHttpClient

Connection Pool Management Strategy

The DefaultHttpClient also supports multiple connection pool management strategies, allowing developers to tailor their application's connections to the specific environment. The default connection pool management strategy is based on the concept of a “per-route” pool, which maintains separate pools for each host and port combination that it makes requests to.
