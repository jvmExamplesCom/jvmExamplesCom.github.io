javax.xml.bind.annotation.XmlRootElement

Introduction

javax.xml.bind.annotation.XmlRootElement is an annotation that indicates that a given class can be used as an XML root element. This annotation can be used to generate an XML schema from a Java Object model. It is part of the Java Architecture for XML Binding (JAXB) API.

Usage

To use the XmlRootElement annotation, the first step is to add a dependency to the jaxb-api library as follows: 

          
           
<dependency>
  <groupId>javax.xml.bind</groupId>
  <artifactId>jaxb-api</artifactId>
  <version>2.3.1</version>
</dependency>
Then, you can use the XmlRootElement annotation on the classes that should be mapped from and to XML. For example, the following class defines an XML root element “Student” and two child elements “name” and “age”: 

          
           
@XmlRootElement(name="Student")
public class Student {

  private String name;
  private int age;

  public String getName() {
    return name;
  }

  @XmlElement
  public void setName(String name) {
    this.name = name;
  }

  public int getAge() {
    return age;
  }

  @XmlElement
  public void setAge(int age) {
    this.age = age;
  }

}
The resulting XML looks like this: 

          
           
<Student>
  <name>John Doe</name>
  <age>30</age>
</Student>

Advanced Usage

XmlRootElement also supports some advanced configuration options.

Specifying namespace

You can specify an XML namespace for the root element by passing the namespace URI to the namespace parameter. For example: 

          
           
@XmlRootElement(name="Student", namespace="http://javax.xml.bind")
public class Student { ... }

Specifying Name Mapping

You can specify how the element name and attributes should be mapped from the Java object model to the XML schema by adding XmlType annotations. For example: 

          
           
@XmlRootElement
@XmlType(name="Student", propOrder={"name", "age"})
public class Student {

  private String name;
  private int age;

  public String getName() { ... }

  @XmlElement
  public void setName(String name) { ... }

  public int getAge() { ... }

  @XmlElement
  public void setAge(int age) { ... }

}
This will result in the following XML schema: 

          
           
<Student>
  <name>John Doe</name>
  <age>30</age>
</Student>

Specifying Namespace Prefix

You can also specify a namespace prefix for the root element by passing it to the nsPrefix parameter. For example: 

          
           
@XmlRootElement(name="Student", nsPrefix="st")
public class Student { ... }
This will result in the following XML schema: 

          
           
<st:Student>
  <st:name>John Doe</st:name>
  <st:age>30</st:age>
</st:Student>
