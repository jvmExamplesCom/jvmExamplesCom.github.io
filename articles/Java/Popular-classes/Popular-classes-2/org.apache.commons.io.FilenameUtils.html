org.apache.commons.io.FilenameUtils

The 

          
           
org.apache.commons.io.FilenameUtils
class provides utilities for working with file and directory names. It contains methods for splitting a file name into its components, normalizing paths, and creating temporary files. The class is suitable for use in both Java SE and Java EE applications.

getBaseName()

The 
getBaseName()
method returns the base name of a file without its extension. For instance, if the file name is 
file.txt
then the method will return 
file
. 

          
           
String baseName = FilenameUtils.getBaseName("file.txt");
System.out.println(baseName);
// Output: file

getExtension()

The 
getExtension()
method returns the file extension of a file. For instance, if the file name is 
file.txt
then the method will return 
txt
. 

          
           
String extension = FilenameUtils.getExtension("file.txt");
System.out.println(extension);
// Output: txt

getName()

The 
getName()
method returns the file name including the extension. For instance, if the file name is 
file.txt
then the method will return 
file.txt
. 

          
           
String name = FilenameUtils.getName("file.txt");
System.out.println(name);
// Output: file.txt

normalize()

The 
normalize()
method normalizes a file path by removing any redundant separators or parent paths. For example, a path of 
/home/foo///bar/..
would be normalized to 
/home/foo
. 

          
           
String normalisedPath = FilenameUtils.normalize("/home/foo///bar/..");
System.out.println(normalisedPath);
// Output: /home/foo

getTempFilePath()

The 
getTempFilePath()
method returns the path of a temporary file. The method takes two arguments - a prefix and a suffix - which are used to create a unique file name. 

          
           
String tempFilePath = FilenameUtils.getTempFilePath("myPrefix", "mySuffix");
System.out.println(tempFilePath);
// Output: /tmp/myPrefix145378768740046586.mySuffix
