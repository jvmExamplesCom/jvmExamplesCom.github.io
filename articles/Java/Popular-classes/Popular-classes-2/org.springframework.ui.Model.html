org.springframework.ui.Model

The org.springframework.ui.Model is a Java class that provides access to the servlet request to the view template. It gives the ability to manipulate the model data using a variety of methods.

The org.springframework.ui.Model class is part of the Spring Framework and is an interface that is used to bind model attributes to the view. It provides methods for creating, adding, and retrieving model attributes.

Creating Model Attributes

The org.springframework.ui.Model class provides several methods for creating model attributes. The most commonly used method is the addAttribute() method. This method takes two arguments - a name for the attribute and its value.


          
           
model.addAttribute("name", "John Doe");

Adding Model Attributes

The addAllAttributes() method of the org.springframework.ui.Model class can be used to add all the attributes of a Map or Collection instance to the model. This is useful when you have a large number of model attributes that should be added at once.


          
           
Map<String, Object> attributes = new HashMap<>(); 
attributes.put("name", "John Doe"); 
attributes.put("age", 30); 
model.addAllAttributes(attributes);

Retrieving Model Attributes

The org.springframework.ui.Model class provides the containsAttribute() method to check if an attribute exists in the model. This method takes a name for the attribute as an argument.


          
           
boolean contains = model.containsAttribute("name");

The getAttribute() method can be used to retrieve the value of a model attribute. This method takes the name of the attribute as an argument. If the attribute does not exist, it will return null.


          
           
Object value = model.getAttribute("name");
