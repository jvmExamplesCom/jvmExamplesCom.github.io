java.awt.Font

Java.awt.Font is a class that represents fonts, which are used to render text in a visible way. With it, you can set the font size, style, color and other attributes of text in your application. It is part of the Java Abstract Window Toolkit (AWT) library, which contains a set of core classes for creating graphical user interfaces (GUIs).

CreatingFont Objects

Creating a new Font object is simple. First, you need to specify the name of the font you want to use. The following code creates a new Font object using the default font of the system: 

          
           
Font myFont = new Font(null, Font.PLAIN, 12);
The first parameter is the name of the font, which can be either a physical font name or a logical font name. Here, the value is null, which means that this method will create a Font object that uses the default font of the system. The second parameter specifies the style for the font, which can be Font.PLAIN, Font.ITALIC, Font.BOLD or a combination of these. Lastly, the third parameter sets the size of the font.

Accessing Font Attributes

Once you create a Font object, you can access its attributes such as the font size, style and family. The following code snippet shows how to get and print out the font size: 

          
           
int fontSize = myFont.getSize();
System.out.println("Font size: " + fontSize); // Prints "Font size: 12"
You can also use the getStyle() and getFamily() methods to retrieve the font style and font family, respectively.

Using the Font Object in an Application

Now that we have a Font object, we can use it in our application. For instance, if you have a Swing application, you can add the Font object to a JLabel component like this: 

          
           
JLabel label = new JLabel("Hello World!");
label.setFont(myFont);
This will make the label display the text using the font specified by the myFont object. You can also use the Font object when drawing text using a graphics context. The following code snippet draws a string on the screen using the font created earlier: 

          
           
Graphics g = ...; // Get a Graphics object
g.setFont(myFont);
g.drawString("Hello World!", 10, 10);
The string will be drawn with the font specified by the myFont object at the specified coordinates (10, 10).

Conclusion

In this tutorial, we looked at the Java.awt.Font class and saw how to use it in an application. We saw how to create a Font object and access its attributes. We also saw how to use the Font object in a Swing application and when drawing text with a Graphics context.
