java.time.temporal.ChronoUnit
enum is a date-time class that represents the concept of a standard temporal unit such as days, weeks, hours, and so on. It provides access to units of time that are not covered by the java.time package and its subpackages.

Date-Time Unit Representation

ChronoUnit

ChronoUnit
