android.telephony.TelephonyManager

The android.telephony.TelephonyManager class provides information about the telephony services on a device such as the phone number, network type, SIM state, and subscriber ID. It also offers methods to access information like the software version of the GSM or CDMA phones.

Getting Device Telephony Information

To get the device’s TelephonyManager, use the getSystemService() method of the context object. The following code snippet shows how to get TelephonyManager: 

          
           
TelephonyManager tm = (TelephonyManager)getSystemService(Context.TELEPHONY_SERVICE);

Accessing Information from TelephonyManager

When you have an instance of TelephonyManager, you can access all the telephony-related information from it. The following are some of the useful methods of this class: 

          
           
getDeviceId() — Returns the unique device ID
getPhoneType() — Returns the type of the phone (e.g., GSM)
getNetworkOperatorName() — Returns the network operator name
getSimState() — Returns the state of the SIM card
getSubscriberId() — Returns the subscriber ID

Examples of using TelephonyManager

Example 1: Get Devices Phone Number

This example shows how to get the devices phone number using TelephonyManager. 

          
           
TelephonyManager tm = (TelephonyManager) getSystemService(Context.TELEPHONY_SERVICE);
String phoneNumber = tm.getLine1Number();

Example 2: Get Mobile Service Provider Name

This example demonstrates how to get the mobile service provider name from the device’s TelephonyManager. 

          
           
String operatorName = tm.getNetworkOperatorName();

Example 3: Get Device Software Version

The following code example shows how to get the software version of a device from its TelephonyManager.

Example 4: Get SIM State

The following example demonstrates how to get the SIM state from the device's TelephonyManager. 

          
           
int simState = tm.getSimState();
