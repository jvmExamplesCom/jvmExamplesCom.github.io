java.io.FilterInputStream

Java.io.FilterInputStream is an abstract subclass of InputStream that serves as the base class for all classes that read filtered byte streams. It wraps another stream and the FilteredInputStream object passes requests to the contained stream by overriding some of its methods. It is used to read data from a particular input stream, filter it in some specific way, and pass it on to the next entity in the chain.

Constructors

The java.io.FilterInputStream class has two constructors.

public BufferedInputStream(InputStream in,int bufferSize)

Example

The following example shows the usage of java.io.FilterInputStream.


          
           
import java.io.*;
 
public class Test
{
    public static void main(String args[])throws Exception
    {
        FileInputStream fin = new FileInputStream("C:\\file.txt");
        BufferedInputStream bin = new BufferedInputStream(fin);
        DataInputStream din = new DataInputStream(bin);
        System.out.println("Reading Characters from the file");
        while(din.available()!=0)
        {
            System.out.print((char)din.read() + "  ");
        }
        din.close();
        System.out.println("\nReading Characters from the file using FilterInputStream\n");
        FileInputStream finNew = new FileInputStream("C:\\fileNew.txt");
        FilterInputStream filter = new FilterInputStream(finNew);
        int k =0;
        while((k=filter.read())!=-1)
        {
            System.out.print((char)k + "  ");
        }
        finNew.close();
        filter.close();
    }
}
