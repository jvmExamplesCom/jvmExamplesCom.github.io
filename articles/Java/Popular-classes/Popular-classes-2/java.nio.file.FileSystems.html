java.nio.file.FileSystems

We can use the FileSystem instance to set file attributes. To do this, we can call its setAttribute() method, which takes a Path object as a parameter. Path file = Paths.get("/some/directory/file.txt");

Copying and Moving Files

We can use the FileSystem instance to copy and move files. To do this, we can call its copy() and move() methods, which take two Path objects as parameters.
