java.nio.file.FileSystems

Java.nio.file.FileSystems is a class used to access file systems in the Java programming language. It provides methods for interacting with the file system, such as viewing the contents of a directory, creating files and directories, setting permissions on files and directories, setting file attributes, and copying and moving files.

Creating a File System

The first step in working with the FileSystems class is to create an instance of it. This can be done by calling its static getDefault() method:


          
           
FileSystem fs = FileSystems.getDefault();

Creating Files and Directories

We can use the FileSystem instance to create files and directories. To do this, we can call its create() method, which takes a Path object as a parameter. The Path object points to the file or directory we want to create. For example:


          
           
Path file = Paths.get("/some/directory/file.txt");

    Files.createFile(file);

    Files.createDirectory(directory);

Setting File Attributes

We can use the FileSystem instance to set file attributes. To do this, we can call its setAttribute() method, which takes a Path object as a parameter. The Path object points to the file we want to set the attribute for. We also need to provide an attribute name and an attribute value. For example:


          
           
Path file = Paths.get("/some/directory/file.txt");
try {
    Files.setAttribute(file, "attrName", "attrValue");
} catch (IOException e) {
    e.printStackTrace();
}

This code will set the attribute attrName with the value attrValue to the file /some/directory/file.txt.

Copying and Moving Files

We can use the FileSystem instance to copy and move files. To do this, we can call its copy() and move() methods, which take two Path objects as parameters. The first Path object points to the source file and the second Path object points to the destination file. For example:


          
           
Path src = Paths.get("/some/directory/file.txt");
Path dest = Paths.get("/some/destination/file.txt");

    Files.copy(src, dest);

    e.printStackTrace();

    Files.move(src, dest);
} catch (IOException e) {
    e.printStackTrace();
}

This code will copy the file /some/directory/file.txt to /some/destination/file.txt, then move it afterwards.
