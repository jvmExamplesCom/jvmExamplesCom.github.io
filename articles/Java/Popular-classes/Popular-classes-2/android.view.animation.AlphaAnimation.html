android.view.animation.AlphaAnimation

Android AlphaAnimation is an animation that controls the alpha level of an object, making it appear or disappear. It can be used to fade out a view, for instance.

This constructor creates an AlphaAnimation object that starts from fromAlpha and ends at toAlpha. 

          
           
public AlphaAnimation (Context context, AttributeSet attrs)
This constructor creates an AlphaAnimation object from a set of attributes defined in an XML resource file.

This constructor creates an AlphaAnimation object from a Parcel. The AlphaAnimation class also has a few methods, including:

cancel()
This method cancels the animation. 

          
           
reset()
This method resets the animation, which can be used to replay the animation from the beginning. 

          
           
isStarted()
This method returns true if the animation has been started. 

          
           
setInterpolator()
This method sets the interpolator used to calculate the animation's rate of change.

Example Code

Here is an example of how to use AlphaAnimation:

fadeOutAnimation.setDuration(1000);
myView.startAnimation(fadeOutAnimation);
