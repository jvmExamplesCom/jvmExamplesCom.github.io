java.nio.FloatBuffer

The 
FloatBuffer
class is part of Java’s NIO (Non-blocking I/O) framework and represents a buffer of floats. It's used to transfer data between different channels of communication in an efficient manner, such as reading/writing from/to a file or a network connection.

Creating a FloatBuffer

To create a 
FloatBuffer
, you first need to decide how much storage you need. You can use the 
allocate()
method which takes an 
int
argument that specifies size, or you can specify the size when constructing a new 
FloatBuffer
instance. 

          
           
// Allocate 20 ints 
FloatBuffer buffer = FloatBuffer.allocate(20);

// Create a FloatBuffer of size 10
FloatBuffer buffer2 = new FloatBuffer(10);

Writing to a FloatBuffer

Once you have allocated a 
FloatBuffer
, you can write data to it using either the 
put(float)
method or the 
put(float[])
method. 

          
           
// Write a single float
buffer.put(42f);

// Write an array of floats
float[] array = new float[] {1, 2, 3, 4};
buffer.put(array);

Reading from a FloatBuffer

Reading from a 
FloatBuffer
is similar to writing. You can use the 
get()
method to read a single float, or the 
get(float[])
method to read an array of floats. 

          
           
// Read a single float

Flipping a FloatBuffer

The contents of a 
FloatBuffer
are not always immediately available for reading. Before you can do so, you must "flip" the buffer using the 
flip()

buffer.flip();
