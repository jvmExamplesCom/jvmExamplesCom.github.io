javax.ws.rs.GET

javax.ws.rs is an API for handling web requests through the use of Java technology. It provides a simple and consistent way of interacting with web services and performing operations such as GET, POST, PUT, and DELETE.

GET Method

The 
javax.ws.rs.GET
method is used to retrieve information from a server. This method is typically used when the request does not affect the state of the server and is simply to obtain information about the specified resource. For example, retrieving the contents of a website or a file stored on a server.

Example

The following code example illustrates how to use the 

javax.ws.rs.GET
method: 

          
           
// imports necessary packages
import javax.ws.rs.client.Client;
import javax.ws.rs.client.WebTarget;

// create client and target
Client client = ClientBuilder.newClient();
WebTarget target = client.target("http://example.com/resource");

// send GET request and receive response
Response response = target.request().get();
String responseBody = target.request().get(String.class);

The response object contains information about the HTTP response, such as the status code, headers, and entity body. The 


          
           
response.getStatus()
method can be used to get the status code, and the 

          
           
response.getHeaders()
method can be used to get the response headers.

Output

The output of the 
javax.ws.rs.GET
method is an instance of 

          
           
javax.ws.rs.core.Response
which contains information about the response. This includes the status code, headers, and entity body. 

          
           
Response response = target.request().get();
int status = response.getStatus(); // 200 OK
Map<String, List<Object>> headers = response.getHeaders(); // response headers
