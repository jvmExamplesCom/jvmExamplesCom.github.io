javax.management.ObjectName

javax.management.ObjectName is a Java class used in the JMX API that provides a reference to an object registered in the MBean server. This class is important for processing and managing the resources of the application being monitored. It uniquely identifies a managed resource, such as an MBean, within the MBean server. The ObjectName is composed of two parts: domain name and a key property list.

Example

The following example demonstrates the use of the ObjectName class.


          
           
public class ObjectNameExample {
    public static void main(String[] args) {
        try {
            // create MBean server
            MBeanServer mBeanServer = ManagementFactory.getPlatformMBeanServer();

            // create ObjectName
            ObjectName objName = new ObjectName("MBean:type=MyMBean");

            // create parameter
            String param = "Hello World!";

            // create new MBean
            mBeanServer.registerMBean(new MyMBean(param), objName);
            System.out.println("MBean registered");

            // get the value of the attribute
            Object result = mBeanServer.getAttribute(objName, "MyAttribute");
            System.out.println("Result: " + result);

            // unregister the MBean
            mBeanServer.unregisterMBean(objName);
            System.out.println("MBean unregistered");
        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}
