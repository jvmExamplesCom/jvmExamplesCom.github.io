What are Context Objects?

Context objects are objects that provide access to information regarding the current environment. For example, the javax.ws.rs.core.UriInfo object provides access to information such as the base URI of the request and the path parameters. Similarly, the javax.ws.rs.core.HttpHeaders object provides access to information regarding the request and response headers.
