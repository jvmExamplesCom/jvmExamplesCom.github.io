Introduction to com.fasterxml.jackson.annotation.JsonCreator

The 
JsonCreator

Using JsonCreator Annotation

To use the 
JsonCreator
annotation, it needs to be added to the constructor or static factory method that should act as deserialization factory. If no annotations are used, all fields and parameters are used. Example:
