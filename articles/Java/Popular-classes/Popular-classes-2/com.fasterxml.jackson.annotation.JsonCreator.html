Introduction to com.fasterxml.jackson.annotation.JsonCreator

The 
JsonCreator
annotation is used in Jackson JSON processing to define methods as creators for deserialization purposes. It defines a "deserialization factory" for an annotated class, which is used to find the right constructor or static factory method to use during deserialization process.

Using JsonCreator Annotation

To use the 
JsonCreator
annotation, it needs to be added to the constructor or static factory method that should act as deserialization factory. It can also be used at field level or parameter level, to indicate which properties or parameters are to be included in resulting object. If no annotations are used, all fields and parameters are used. Example: 

          
           
public class SampleBean {
    private final int id;
 
    @JsonCreator
    public SampleBean(@JsonProperty("id") int id) {
        this.id = id;
    }
}

Using Multiple Constructors or Factory Methods with JsonCreator

When there are multiple constructors or factory methods with the same set of parameters, the annotation 

JsonCreator
can be used to specify which one to use. The annotation can be placed on any of the constructors or factory methods, and only one such annotation can be used. If no annotations are used, the first such method or constructor is chosen by default. Example: 

          
           
public class SampleBean {
    private final int id;
    private final String name;
 
    @JsonCreator
    public SampleBean(@JsonProperty("id") int id, 
                      @JsonProperty("name") String name) {
        this.id = id;
        this.name = name;
    }
 
    public SampleBean(int id) {
        this(id, null);
    }

The annotation 
JsonCreator

String[]
