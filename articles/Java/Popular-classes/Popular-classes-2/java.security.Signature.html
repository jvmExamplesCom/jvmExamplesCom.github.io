Java Security Signature

java.security.Signature
class allows applications to generate and verify digital signatures. It provides a framework for generating, verifying and storing digital signatures based on cryptographic techniques. Signature signature = Signature.getInstance("RSA");

initSign()
or 
initVerify()
methods. The

Signing The Data

Verifying The Data
