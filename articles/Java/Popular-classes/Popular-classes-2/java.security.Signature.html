Java Security Signature

java.security.Signature
class allows applications to generate and verify digital signatures. It provides a framework for generating, verifying and storing digital signatures based on cryptographic techniques. getInstance()

Signature signature = Signature.getInstance("RSA");

initSign()
or 
initVerify()
methods. The

signature.initSign(privateKey);

Signing The Data
