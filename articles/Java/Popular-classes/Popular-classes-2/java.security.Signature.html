Java Security Signature

The 

          
           
java.security.Signature
class allows applications to generate and verify digital signatures. It provides a framework for generating, verifying and storing digital signatures based on cryptographic techniques. This class can be used to sign and verify data in digital applications. It is part of the Java Cryptography Architecture (JCA).

getInstance()

Signature signature = Signature.getInstance("RSA");

initSign()

initVerify()
methods. The

Signing The Data
