Java AWT Event MouseAdapter

The 

          
           
j ava.awt.event.MouseAdapter
class is an abstract class which implements the 
MouseListener
, 
MouseMotionListener
, and 
MouseWheelListener
interfaces. It is used to handle events in a graphical user interface where the mouse is the primary input tool and is commonly used when creating GUI applications. It contains methods that can be overridden to implement user interface behavior when the mouse is used. This article provides an overview of how to use the MouseAdapter class and includes example code.

Methods

The 
MouseAdapter
class has several methods which can be used to handle mouse events: 
  • mousePressed()
    : this method is called when the mouse button is pressed.
  • mouseReleased()
    : this method is called when the mouse button is released.
  • mouseEntered()
    : this method is called when the mouse enters the component.
  • mouseExited()
    : this method is called when the mouse exits the component.
  • mouseDragged()
    : this method is called when the mouse is dragged across the component.
  • mouseMoved()
    : this method is called when the mouse is moved over the component.
  • mouseWheelMoved()
    : this method is called when the mouse wheel is moved.

Examples

The following code examples illustrate how to use the 

MouseAdapter
class to create a simple application. For example, the following code demonstrates how to create a simple window with a custom 
MouseAdapter
implementation: 

          
           
import java.awt.EventQueue;
import java.awt.event.MouseAdapter;
import javax.swing.JFrame;

public class MyClass {
   public static void main(String[] args) {
        // Create a frame
        JFrame frame = new JFrame();
        frame.setSize(200, 200);

        // Create a mouse adapter
        MouseAdapter ma = new MouseAdapter() {
            @Override
            public void mouseClicked(MouseEvent e) {
                System.out.println("Mouse clicked!");
            }
        };

        // Set the mouse adapter
        frame.addMouseListener(ma);

        // Display the frame
        frame.setVisible(true);
    }
}
This code creates a simple window with a custom 
MouseAdapter
instance. The 
mouseClicked
method will be called whenever the mouse is clicked on the window. The output of the program will be "Mouse clicked!" when the mouse is clicked.

Another example is to create a frame with a button which prints out a message when the button is clicked. The following code illustrates how to do this using the 

MouseAdapter
class: 

          
           
import java.awt.EventQueue;
import java.awt.event.MouseAdapter;
import javax.swing.JButton;
import javax.swing.JFrame;

public class MyClass {
   public static void main(String[] args) {
        // Create a frame
        JFrame frame = new JFrame();
        frame.setSize(200, 200);

        // Create a button
        JButton button = new JButton("Click Me!");

        // Create a mouse adapter
        MouseAdapter ma = new MouseAdapter() {
            @Override
            public void mouseClicked(MouseEvent e) {
                System.out.println("Button clicked!");
            }
        };

        // Set the mouse adapter
        button.addMouseListener(ma);

        // Add the button to the frame
        frame.add(button);

        // Display the frame
        frame.setVisible(true);
    }
}
In this example, the 
mouseClicked
method will be called whenever the button is clicked. The output of the program will be "Button clicked!" when the button is clicked.
