javax.swing.JLabel

A JLabel is a display area for a short text, an image, or both. A JLabel object can display either text, an image, or both. You can use the setText method to change the text that a JLabel displays. You can use the setIcon method to change the image that a JLabel displays.

Example Usage

Here is an example of how to use a JLabel in a program: 

          
           
JLabel label = new JLabel("This is a JLabel");
label.setVisible(true);
The code above creates a new instance of a JLabel, sets its text to "This is a JLabel" and sets it to visible. The result is a label with the text "This is a JLabel" being displayed on the screen.

Properties

A JLabel can have various properties set. For example, the following code sets the font and color of a JLabel: 

          
           
JLabel label = new JLabel("This is a JLabel");
label.setFont(new Font("Arial", Font.BOLD, 16));
label.setForeground(Color.RED);
label.setVisible(true);
The code above sets the font of the JLabel to Arial, bold and size 16, and sets the foreground (text) color to red.

Events

A JLabel can respond to events such as mouse clicks. The following code adds a mouse listener to a JLabel: 

          
           
JLabel label = new JLabel("This is a JLabel");
label.addMouseListener(new MouseAdapter() {
    public void mouseClicked(MouseEvent e) {
        System.out.println("Label clicked");
    }
});
label.setVisible(true);
The code above adds a mouse listener to the JLabel. When the JLabel is clicked, it will print out "Label clicked" to the console.

Methods

A JLabel has many useful methods. The following are some of the most commonly used ones: 

          
           
setText(String): Sets the text displayed by the JLabel
setIcon(Icon): Sets the image displayed by the JLabel
setVisible(Boolean): Sets whether the JLabel is visible or not
addMouseListener(MouseListener): Adds a mouse listener to the JLabel

Conclusion

JLabels are a simple but powerful way to display text and images in Java applications. With their event handling capabilities, they can be used to create interactive user interfaces.
