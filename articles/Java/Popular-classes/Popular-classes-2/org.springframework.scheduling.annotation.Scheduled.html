org.springframework.scheduling.annotation.Scheduled

The org.springframework.scheduling.annotation.Scheduled annotation is used for scheduling tasks in the Spring Framework. It makes use of the TaskScheduler interface of the spring framework to schedule tasks. This annotation can be used to schedule cron jobs, timed tasks, and fixed-delay tasks in the Spring application.

Scheduled Annotation Syntax

The Scheduled annotation supports a number of attributes that let us customize the scheduling process. Below is the syntax of a Scheduled annotation.


          
           
@Scheduled(fixedRate = <number>)<br>@Scheduled(fixedDelay = <number>)<br>@Scheduled(cron=<String>)

fixedRate Attribute

The fixedRate attribute tells the scheduler to execute the task at a fixed interval. The interval specified is in milliseconds. For example, if we specify a fixedRate of 1000, the task will be executed every second.


          
           
@Scheduled(fixedRate = 1000)
public void printMessage(){
    System.out.println("This message is printed every second");
}

fixedDelay Attribute

The fixedDelay attribute tells the scheduler to execute the task at a fixed interval. The interval specified is in milliseconds. This attribute is similar to the fixedRate attribute. However, the difference between these two attributes is that the fixedDelay attribute takes into account the completion time of the task. For example, if the task takes 5 seconds to execute and we specify a fixedDelay of 1000, then the next invocation of the task will be scheduled 5 seconds after the completion of the previous task.


          
           
@Scheduled(fixedDelay = 1000)
public void printMessage(){
    System.out.println("This message is printed every second");
}

cronAttribute

The cron attribute tells the scheduler to execute the task according to a cron expression. A cron expression is a string with 6 or 7 fields separated by a space that represents a time schedule. The fields are in order: second, minute, hour, day of month, month, day of week, and optionally year.


          
           
@Scheduled(cron="0 0/1 * 1/1 * ?")
public void printMessage(){
    System.out.println("This message is printed every minute");
}

Conclusions

The org.springframework.scheduling.annotation.Scheduled annotation simplifies the scheduling process in the Spring Framework. We can use this annotation to schedule tasks at fixed intervals, fixed delay, or as per a cron expression. All we need to do is specify the appropriate attributes on the annotation.
