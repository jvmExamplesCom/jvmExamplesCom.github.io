androidx.recyclerview.widget.LinearLayoutManager

The androidx.recyclerview.widget.LinearLayoutManager is a ViewGroup subclass that is used to display items in a linear, horizontal or vertical arrangement. It is one of the most commonly used layout managers and is supported by the Android Support Library.

The LinearLayoutManager uses the view hierarchy to provide an efficient way of displaying items in an array-like structure. It's main features are:

  • It can be used for both horizontal and vertical display of items.
  • It can be used for both linear and staggered (grid) display of items.

In order to use the LinearLayoutManager, you need to include the following line in your build.gradle file:


          
           
implementation 'androidx.recyclerview:recyclerview-core:X.X.X'

Example 1: Simple Linear Layout Manager

Here is an example of using a LinearLayoutManager to show a list of items in a vertical list.We will use a RecyclerView; we will define a custom layout for each item in the list, and then bind the data to the view elements.


          
           
// Create the RecyclerView
RecyclerView recyclerView = findViewById(R.id.my_recycler_view);

// Set the LayoutManager
LinearLayoutManager layoutManager = new LinearLayoutManager(this);
recyclerView.setLayoutManager(layoutManager);

// Set the Adapter
MyAdapter myAdapter = new MyAdapter(myDataList);
recyclerView.setAdapter(myAdapter);
