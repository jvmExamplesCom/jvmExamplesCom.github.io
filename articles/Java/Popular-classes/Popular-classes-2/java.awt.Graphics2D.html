Introduction to Java.awt.Graphics2D

Java.awt.Graphics2D is a powerful class in the AWT (Abstract Window Toolkit) package of the Java language. g2d.setRenderingHint(RenderingHints.KEY_ANTIALIASING, RenderingHints.VALUE_ANTIALIAS_ON);
