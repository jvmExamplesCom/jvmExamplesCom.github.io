To create an object of this class use the following code:


          
           
UsernamePasswordAuthenticationToken authenticationToken = 
  new UsernamePasswordAuthenticationToken(username, password);

username

password

To authenticate the token you can use the following code:


          
           
Authentication auth = 
  authenticationManager.authenticate(authenticationToken);

authenticationManager
is an instance of an object implementing 

          
           
org.springframework.security.authentication.AuthenticationManager
. It authenticates the token by checking if the provided credentials are valid.
