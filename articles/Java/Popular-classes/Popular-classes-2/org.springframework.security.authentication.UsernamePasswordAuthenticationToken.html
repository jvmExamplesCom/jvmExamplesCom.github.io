org.springframework.security.authentication.UsernamePasswordAuthenticationToken

The org.springframework.security.authentication.UsernamePasswordAuthenticationToken is a class used to authenticate Java applications with Spring Security authentication framework.

Creating a UsernamePasswordAuthenticationToken Object

To create an object of this class use the following code:


          
           
UsernamePasswordAuthenticationToken authenticationToken = 
  new UsernamePasswordAuthenticationToken(username, password);
In the above example, 

          
           
username
and 

          
           
password
are the credentials to be authenticated.

Authenticating the Token

To authenticate the token you can use the following code:


          
           
Authentication auth = 
  authenticationManager.authenticate(authenticationToken);
The 

          
           
authenticationManager
is an instance of an object implementing 

          
           
org.springframework.security.authentication.AuthenticationManager
. It authenticates the token by checking if the provided credentials are valid.

Checking If Authentication Was Successful

To check if authentication was successful, the following code can be used:


          
           
if (auth.isAuthenticated()) {
    // authentication was successful
} else {
    // authentication failed
}

Retrieving the Authenticated Principal

Once authentication has been successful, the authenticated principal can be retrieved as follows:

principal
