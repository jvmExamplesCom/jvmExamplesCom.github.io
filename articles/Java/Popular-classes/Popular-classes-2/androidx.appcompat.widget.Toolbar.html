Introduction to AndroidX Appcompat Toolbar

AndroidX Appcompat Toolbar provides a user interface element used to provide navigation and quick access to common actions. It consists of a navigation button, an optional logo image, and several menu items that can be dynamically populated from code or from XML. This article will discuss the different components of the AndroidX AppCompat Toolbar and provide code examples for implementing each one.

Adding a Toolbar to Your Activity

The first step in using a Toolbar is to add it to your activity as part of the layout. To do this, use the androidx.appcompat.widget.Toolbar widget in your layout file, like so: 

          
           
<androidx.appcompat.widget.Toolbar
    android:id="@+id/toolbar"
    android:layout_width="match_parent"
    android:layout_height="wrap_content"
    android:minHeight="?attr/actionBarSize"
    android:background="?attr/colorPrimary" />
This code adds a Toolbar to your activity, setting its width to match the parent view, setting its height to the action bar size, and setting its background color to the primary color.

Customizing a Toolbar

Once the Toolbar is added, you can customize it further by setting its title, navigation icon, and custom views.

Setting the Title

To set the title in the Toolbar, use the setTitle() method, like so: 

          
           
toolbar.setTitle("My Activity");

Setting the Navigation Icon

To set the navigation icon in the Toolbar, use the setNavigationIcon() method, like so: 

          
           
toolbar.setNavigationIcon(R.drawable.ic_back);

Adding Custom Views

You can also add custom views to the Toolbar by using the addView() method, like so: 

          
           
TextView tv = new TextView(this);
tv.setText("Hello!");
toolbar.addView(tv);

Adding Menu Items

Finally, you can add menu items to the Toolbar by either directly adding them from code or from an XML menu resource file.

Adding Menu Items from Code

You can add a menu item to the Toolbar programmatically by using the add() method, like so: 

          
           
toolbar.add(0, 123, 0, "Menu Item");

Adding Menu Items from an XML Menu Resource File

You can also add menu items from an XML menu resource file by using the inflateMenu() method, like so: 

          
           
toolbar.inflateMenu(R.menu.toolbar_menu);
Where the 

          
           
R.menu.toolbar_menu
is a reference to the menu resource file.

Conclusion

The AndroidX Appcompat Toolbar provides a simple and effective way to add navigation and quick access to common actions. It can be customized further by setting its title, navigation icon, and custom views, and menu items can be added from code or from an XML menu resource file. In this article, we discussed the different components of the Toolbar and provided code examples for implementing each one.
