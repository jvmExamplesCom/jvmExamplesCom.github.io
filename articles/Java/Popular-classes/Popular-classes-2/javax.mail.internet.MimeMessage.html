javax.mail.internet.MimeMessage

javax.mail.internet.MimeMessage is an API that provides the ability to compose and parse MIME messages. It makes it easy to create and manipulate MIME messages, which are widely used in email communications. This API can be used to create both text and HTML messages, and is also used to attach files.

Creating a MimeMessage

The following example shows how to create a basic MimeMessage using the javax.mail.internet.MimeMessage API.


          
           
Session session = Session.getDefaultInstance(props);
MimeMessage message = new MimeMessage(session);

message.addRecipient(Message.RecipientType.TO, "recipient@example.com");
message.setSubject("My Subject");
message.setText("Hello world!");
message.saveChanges();

    for (int i = 0; i < multipart.getCount(); i++) {
