Introduction to org.springframework.web.bind.annotation.ControllerAdvice

Declaring @ControllerAdvice Classes

@ControllerAdvice
. This annotation can be used on both classes and methods in order to provide different levels of exception handling granularity. The following example shows how a controller advice could be declared: 

          
           
@ControllerAdvice 
public class MyControllerAdvice {

        // Do something
        return "error";
