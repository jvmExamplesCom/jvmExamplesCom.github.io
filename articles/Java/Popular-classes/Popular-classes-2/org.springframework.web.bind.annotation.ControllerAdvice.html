Introduction to org.springframework.web.bind.annotation.ControllerAdvice

org.springframework.web.bind.annotation.ControllerAdvice
is a controller-specific annotation that supports global handling of specific exceptions across multiple @RequestMapping handlers. It can be used as an alternative to writing WebApplicationInitializers and registering DispatcherServlet instances directly in web.xml files.

Declaring @ControllerAdvice Classes

@ControllerAdvice
. This annotation can be used on both classes and methods in order to provide different levels of exception handling granularity. The following example shows how a controller advice could be declared: 

          
           
@ControllerAdvice 
public class MyControllerAdvice {

    @ExceptionHandler(MyException.class)

        // Do something
        return "error";
    }
}
