javax.swing.JCheckBox

The javax.swing.JCheckBox class provides a graphical user interface element that allows the user to select one or more from a group of options. With a JCheckBox, the user can select and unselect the box. It can also be used to show and hide additional components when selected. This class consists of several constructors and methods allowing developers to create and manipulate check boxes.
