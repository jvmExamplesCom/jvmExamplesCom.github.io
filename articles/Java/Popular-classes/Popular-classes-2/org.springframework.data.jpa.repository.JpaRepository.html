org.springframework.data.jpa.repository.JpaRepository

The org.springframework.data.jpa.repository.JpaRepository is an interface in Java data access technology, part of the Spring Application Framework, that provides a powerful and flexible way to access and manage data stored in JPA (Java Persistence API) compliant databases. It provides several methods for retrieving, saving, and deleting entities, as well as allowing query methods to search for specific entities.

Overview

The org.springframework.data.jpa.repository.JpaRepository provides a number of methods to access and manipulate data stored in JPA-compliant databases. These methods can be broadly classified into four categories: query methods, CRUD methods, pagination methods, and miscellaneous methods.

Query Methods

Query methods provide the ability to retrieve one or more entities from a repository, based on parameter values provided by the client. These methods can come in two forms: finder methods, which return a single entity or null, and list methods, which return a collection of entities. The following example shows how to use the findById() method to retrieve a single Product by its primary key:


          
           
Product p = productRepository.findById(12345);

The following example shows how to use the findAll() method to retrieve all products from the repository:


          
           
List<Product> products = productRepository.findAll();

Product p = new Product(12345, "NewProduct");
productRepository.save(p);

The following example shows how to use the delete() method to remove a Product from the repository:


          
           
productRepository.delete(p);

Pagination methods allow clients to retrieve a subset of objects from a repository. These methods are useful for limiting the amount of data retrieved from a repository, as well as for navigation between multiple pages of results. The following example shows how to use the findAll() method with pagination parameters to retrieve a page of Products from the repository:


          
           
Page<Product> page = productRepository.findAll(PageRequest.of(0, 10));
