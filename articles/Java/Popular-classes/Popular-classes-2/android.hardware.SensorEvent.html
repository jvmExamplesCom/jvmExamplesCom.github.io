android.hardware.SensorEvent

The android.hardware.SensorEvent class represents a single event from a Sensor. This is used to report the sensor's data, type and accuracy.

Creating an Instance

To create a SensorEvent, you can use one of the following methods: 

          
           
SensorEvent(int[] values)
SensorEvent(float[] values)
SensorEvent(int accuracy, long timestamp, int[] values)
SensorEvent(int accuracy, long timestamp, float[] values)
The 

          
           
values
parameter contains the array of data that the sensor has produced. The 

          
           
accuracy
parameter indicates how accurate the data is, and the 

          
           
timestamp
parameter contains the time at which the event occurred.

Methods

The android.hardware.SensorEvent class provides the following methods for manipulating and accessing its data: 

          
           
getAccuracy()              // Returns the accuracy of this event
getValues()                 // Returns the array of values for this event
setValues(float[] values)   // Sets the array of values for this event
clone()                     // Creates a clone of this event
toString()                  // Returns a string representation of this event

Example Usage

The following example shows how to create a SensorEvent and access its data: 

          
           
// Create a SensorEvent with a sensor reading of 10
SensorEvent event = new SensorEvent(10);

// Get the array of values 
float[] values = event.getValues();

// Print the value
System.out.println("Value: " + values[0]);
