What is 

           
            
org.apache.http.client.methods.HttpUriRequest
Class?

The 

          
           
org.apache.http.client.methods.HttpUriRequest
is an interface in the Apache HttpClient library that defines methods to send HTTP requests for a URI or URL. This interface is used for representing a request, which provides methods to set various properties of the request such as headers, content etc. The implementing classes of this interface can be used to construct HTTP request methods such as POST and GET.

org.apache.http.client.methods.HttpUriRequest

Example 3: Custom Method

In this example, we will create a custom implementation of the 

HttpUriRequest
interface. 

          
           
class CustomHttpUriRequest implements HttpUriRequest {

    private URI uri;
    private Map<String, String> headers = new HashMap<>();
    private String method;
    private List<NameValuePair> parameters = new ArrayList<>();

    public CustomHttpUriRequest(URI uri, String method) {
        this.uri = uri;
        this.method = method;
    }

    @Override
    public URI getURI() {
        return uri;
    }

    @Override
    public String getMethod() {
        return method;
    }

    @Override
    public void addHeader(String name, String value) {
        headers.put(name, value);
    }

    @Override
    public void addParameter(String name, String value) {
        parameters.add(new BasicNameValuePair(name, value));
    }

    public Map<String, String> getHeaders() {
        return headers;
    }

    public List<NameValuePair> getParameters() {
        return parameters;
    }
}

In above example, we have implemented all the methods of the 

HttpUriRequest
interface. We can use this class to make a custom HTTP request with arbitrary parameters and headers.
