android.net.wifi.WifiManager

The android.net.wifi.WifiManager class is responsible for managing the Wi-Fi functionality of devices running Android. It provides access to the Wi-Fi hardware, as well as methods for scanning, connecting, and disconnecting from Wi-Fi access points and networks. This article examines the capabilities of the WifiManager and provides code examples demonstrating its use.

Creating a WifiManager Instance

A WifiManager instance can be created by using the getSystemService() method on an Activity or using the getApplicationContext() method in an Android Service. Here is an example of creating a WifiManager instance on an Activity:


          
           
//Activity
WifiManager wifiManager = (WifiManager) getSystemService(Context.WIFI_SERVICE);

Scanning Wi-Fi Networks

The startScan() method will begin scanning for available Wi-Fi networks. To receive the scan results, you must register a BroadcastReceiver with an intent filter of WifiManager.SCAN_RESULTS_AVAILABLE_ACTION. Here is an example of how to do this:


          
           
//Activity
BroadcastReceiver wifiBroadcastReceiver = new BroadcastReceiver() {
    @Override
    public void onReceive(Context context, Intent intent) {
        //handle the broadcast here
    }
};

IntentFilter intentFilter = new IntentFilter();
intentFilter.addAction(WifiManager.SCAN_RESULTS_AVAILABLE_ACTION);
registerReceiver(wifiBroadcastReceiver, intentFilter);

//in your activity or service
if (wifiManager.startScan()) {
    //scan was successful
} else {
    //scan failed
}

Retrieving Scan Results

The getScanResults() method will return a list of ScanResult objects that were discovered during a scan. Each ScanResult represents an access point that was discovered. Here is an example of how to retrieve the scan results after a scan has been performed:


          
           
List<ScanResult> scanResults = wifiManager.getScanResults();

Connecting to a Network

The connect() method will attempt to connect to a Wi-Fi network. The network is identified either by a WifiConfiguration object, or by providing the network's SSID and security type. Here is an example of how to connect to an open network:


          
           
//open network
WifiConfiguration wifiConfig = new WifiConfiguration();
wifiConfig.SSID = "\"" + ssid + "\"";
wifiConfig.allowedKeyManagement.set(WifiConfiguration.KeyMgmt.NONE);

int netId = wifiManager.addNetwork(wifiConfig);

if (netId != -1) {
    if (wifiManager.enableNetwork(netId, true)) {
        //connection was successful
    } else {
        //connection failed
    }
}

Disconnecting From a Network

The disconnect() method will disconnect from the currently connected network. Here is an example of how to disconnect from a network:


          
           
wifiManager.disconnect();
