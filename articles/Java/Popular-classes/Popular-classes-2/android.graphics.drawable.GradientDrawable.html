android.graphics.drawable.GradientDrawable

An Android GradientDrawable is a class used to draw gradient shapes to a UI component. It can be used to draw a Shape or a Drawable object in a variety of ways, including setting its color, opacity, size, shape and orientation. The GradientDrawable class provides methods for drawing simple shapes and gradients directly onto a Canvas.

Creating a GradientDrawable

Creating a GradientDrawable is simple, as it only requires two steps: creating a new instance of the GradientDrawable class, and then setting its properties.


          
           
GradientDrawable gd = new GradientDrawable();
gd.setColors(new int[] { Color.RED, Color.GREEN });

Properties and Methods

The GradientDrawable class provides several properties and methods that can be used to manipulate a GradientDrawable. Below are some of the most commonly used properties and methods.

setGradientType()

This method sets the type of gradient to be used by the GradientDrawable. The available values are LINEAR_GRADIENT, RADIAL_GRADIENT, and SWEEP_GRADIENT.


          
           
gd.setGradientType(GradientDrawable.LINEAR_GRADIENT);

setColors()

This method sets the colors used to draw the gradient. The colors should be specified as an array of integers.


          
           
gd.setColors(new int[] { Color.RED, Color.GREEN });

setOrientation()

This method sets the orientation of the gradient. The available values are TOP_BOTTOM, BOTTOM_TOP, LEFT_RIGHT, RIGHT_LEFT, TR_BL, and TL_BR.


          
           
gd.setOrientation(GradientDrawable.TOP_BOTTOM);

setGradientCenter()

This method sets the position of the center of the gradient. The center is specified as a fraction of the size of the view.


          
           
gd.setGradientCenter(0.5f, 0.5f);

setShape()

This method sets the shape of the GradientDrawable. The available shapes are OVAL, RECTANGLE, LINE, and RING.


          
           
gd.setShape(GradientDrawable.RECTANGLE);

setSize()

This method sets the size of the GradientDrawable. The size is specified in pixels as width, height.


          
           
gd.setSize(400, 200);

draw()

This method draws the GradientDrawable onto a specified Canvas.
