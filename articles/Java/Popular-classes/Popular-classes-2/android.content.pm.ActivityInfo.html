android.content.pm.ActivityInfo

Introduction

android.content.pm.ActivityInfo is an Android class representing a single activity component. It contains the description of an activity, including the package it belongs to, the description string, the label, the icon drawable, the class name, and other information. This class provides information that can be used to start an activity in a particular mode (standard, singleTop, singleTask).

Usage


          
           
ActivityInfo info = context.getPackageManager().getActivityInfo(componentName, 0);
The ActivityInfo is obtained using a PackageManager with a ComponentName as an argument. A parameter passed to the getActivityInfo() method specifies the flags to include in the returned ActivityInfo object.

Fields


          
           
public int descriptionRes;
This field stores a resource identifier of the description string. The description string is typically used in the application's user interface instead of the label. 

          
           
public String enabled;
This field stores a boolean value indicating whether the activity is enabled or not. 

          
           
public String exported;
This field stores a boolean value indicating whether the activity can be launched by components other than its own package. 

          
           
public String launchMode;
This field stores an integer value indicating the launching mode of the activity. The possible values are 

          
           
ActivityInfo.LAUNCH_MULTIPLE
, 

          
           
ActivityInfo.LAUNCH_SINGLE_TOP
, and 

          
           
ActivityInfo.LAUNCH_SINGLE_TASK
. 

          
           
public int labelRes;
This field stores a resource identifier of the label string. The label string is typically used in the application's user interface instead of the description.

Methods


          
           
public static ActivityInfo getActivityInfo(ComponentName component, int flags)
This method returns an ActivityInfo object containing information about the activity associated with the given ComponentName. 

          
           
public Intent makeLaunchIntent()
This method creates an Intent object suitable for launching the activity associated with this ActivityInfo.

Examples

Below is an example of how to obtain an ActivityInfo and use it to launch an activity. 

          
           
// Get the ActivityInfo for the activity
ActivityInfo info = context.getPackageManager().getActivityInfo(componentName, 0);

// Create the Intent object
Intent intent = info.makeLaunchIntent();

// Launch the activity
context.startActivity(intent);
In this example, the ActivityInfo is obtained from the PackageManager, then used to create an Intent object, which is then used to launch the activity.
