org.apache.ibatis.annotations.Param

The org.apache.ibatis.annotations.Param annotation is used to provide names for the parameters of a mapped statement in the MyBatis framework. It allows for a better design, readability and maintainability of the code.

Parameters

The annotation has two main parameters:
  • value which contains the name of the parameter
  • name which contains the name of the parameter

Example of Usage


          
           
@Select("SELECT * FROM users WHERE username = #{userName} and password = #{password}")
User findByNameAndPassword(@Param("userName") String username, @Param("password")String password);

In this example, two parameters are added to the query - "userName" and "password". The @Param annotation allows us to specify the exact name of the parameter in the query, and the value of the parameter is provided when the method is called.

Output

The output of the @Param annotation is the same as if the parameter was specified without the annotation.


          
           
SELECT * FROM users WHERE username = ? and password = ?

The only difference is that the code is more readable and maintainable.

Conclusion

The org.apache.ibatis.annotations.Param annotation is a useful tool for providing names for parameters in the MyBatis framework. It helps to improve the readability and maintainability of the code.
