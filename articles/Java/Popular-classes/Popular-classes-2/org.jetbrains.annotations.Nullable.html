org.jetbrains.annotations.Nullable

org.jetbrains.annotation.Nullable is a Java annotation type used to indicate that a parameter, return value, field, or variable may be null. This annotation is part of JetBrains' Annotations library, which makes it easier for developers to detect and address potential nullability issues in their code.

Code Examples

Below are some examples of the @Nullable annotation in action:


          
           
@Nullable String s = getString();
if (s != null) {
    System.out.println(s);
}

          

          
           
doSomething(null);

          

          
           
@Nullable String s = getString();
if (s == null) {
    return "{'status': 'error'}";
} else {
    return s;
}
