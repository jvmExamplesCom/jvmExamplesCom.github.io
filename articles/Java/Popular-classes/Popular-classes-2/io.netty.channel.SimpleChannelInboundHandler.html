Netty is a NIO (non-blocking input/output) client-server framework for developing network applications such as protocol servers and clients. The SimpleChannelInboundHandler class is a generic message handler which allows you to convert your incoming message into the type of your choice. It provides inbound event handling for the ChannelHandler to process the received messages from the other end of the Channel.

Basics of SimpleChannelInboundHandler

The SimpleChannelInboundHandler class implements both ChannelInboundHandler and ChannelInboundHandlerAdapter.
