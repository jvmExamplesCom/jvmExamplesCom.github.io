Using a Content Provider

To use a content provider, you must first request access to it from the Android system. You do this by declaring it in your application’s manifest file. Here’s an example of declaring a content provider: 

          
           
<manifest>
    <application>
        <provider android:name="MyContentProvider" 
            android:authorities="com.example.myapp.provider"/>
        ...
    </application>
</manifest>
