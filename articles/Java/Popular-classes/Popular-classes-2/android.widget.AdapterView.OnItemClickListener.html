Android's 

          
           
AdapterView.OnItemClickListener
is an interface used to define the behavior of a list item or grid item when it is clicked. It can be implemented in any Activity, Fragment, or View. When the user clicks on a particular item, the onItemClick method is invoked and passes the view, position, and id of the item that was clicked.

For example, if you have a ListView named listView in your layout, you could register the OnItemClickListener like this:


          
           
listView.setOnItemClickListener(new AdapterView.OnItemClickListener() {
    @Override
    public void onItemClick(AdapterView<!--?--> parent, View view, int position, long id) {
        // Handle item click here
    }
});

public void onItemClick(AdapterView<!--?--> parent, View view, int position, long id) {

    }
