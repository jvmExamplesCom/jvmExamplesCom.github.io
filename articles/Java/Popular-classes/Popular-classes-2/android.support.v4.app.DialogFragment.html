android.support.v4.app.DialogFragment

The android.support.v4.app.DialogFragment class is a subclass of the Fragment class. It is designed to create an interactive dialog window instead of an activity. It is part of the support library and is included in the Android Support Library v4 package.

DialogFragment manages the lifecycle of the dialog, and provides callbacks when the user interacts with it. It also displays the UI elements inside the dialog. DialogFragment can be used to display a form, pick a date, show a list of choices, or display a simple message.

Creating a DialogFragment

To create a DialogFragment, you will need to extend the DialogFragment class and implement the onCreateDialog() method. This method is responsible for creating the Dialog object that will be displayed. The onCreateDialog() method takes in a Bundle object which is used to store the state of the dialog.


          
           
public class MyDialogFragment extends DialogFragment {
    @Override
    public Dialog onCreateDialog(Bundle savedInstanceState) {
        AlertDialog.Builder builder = new AlertDialog.Builder(getActivity());
        builder.setTitle("My Dialog");
        builder.setMessage("This is my Dialog!");
        return builder.create();
    }
}

You can also create a custom class that extends DialogFragment and provide additional methods to configure the dialog. For instance, you could add a setMessage() and setTitle() method to the MyDialogFragment class above:
