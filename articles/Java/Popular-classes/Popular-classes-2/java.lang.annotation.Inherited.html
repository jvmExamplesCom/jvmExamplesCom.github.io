Java.lang.annotation.Inherited

The 
@Inherited
annotation applies to the annotation type and indicates that an annotation type may be inherited from superclasses. The 
@Inherited
meta-annotation is defined in the 

          
           
java.lang.annotation
package.

Syntax


          
           
@Inherited
public @interface AnnotationType {
    //members
}
The 
@Inherited
annotation itself has no members.

Example

Here is an example of the 

          
           
@Inherited
meta-annotation.

import java.lang.annotation.Retention;
import java.lang.annotation.RetentionPolicy;
import java.lang.annotation.Inherited;

@Retention(RetentionPolicy.RUNTIME)
@Inherited
@interface Marker{

}
The 
@Marker
annotation is annotated with the 
@Inherited

Inheriting Annotations

When an annotation type is annotated with the 
@Inherited
meta-annotation, any annotations of that type on a class will be inherited by its subclasses. This means that if an annotation type is annotated with 
@Inherited
, any annotation of that type on a superclass will be visible in its subclasses. For example, consider the following classes: 

          
           
@Marker
class Base {

}

class Sub extends Base {

}
Since the 
@Marker

@Inherited

It is possible to override inherited annotations by annotating the subclass with the same annotation type. For example, consider the following classes:

Inheriting Annotation Attributes

When an annotation type is inherited, only the annotation itself is inherited, not its attributes. For example, consider the following annotation type:
