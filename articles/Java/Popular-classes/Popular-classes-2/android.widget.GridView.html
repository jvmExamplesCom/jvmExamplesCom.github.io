android.widget.GridView

The 

          
           
android.widget.GridView
is an Android UI component used to display a collection of items in a scrollable, grid-style view. It can be used for lists, grids, image galleries and more. This widget is often used as part of larger user interfaces, such as the navigation drawer, where it can be used to show the user’s current location in the application.

Creating a GridView

Creating a GridView is done in two steps: first you create an instance of the GridView class, and then add the list of data to it. 


          
           
// Create an instance of the GridView
GridView gridView = new GridView(this);

// Set the data to be displayed in the GridView
gridView.setAdapter(new ArrayAdapter<String>(this, android.R.layout.simple_list_item_1, myData));

Customizing a GridView

The GridView component can be customized using different attributes. These can be set either in XML layout files or programmatically.

Most common attributes used are: 
  • android:numColumns
    , used to specify the number of columns in each row.
  • android:gravity
    , used to specify how elements are aligned within the rows.
  • android:stretchMode
    , used to specify how each element is stretched to fit the available space.

          
           
<GridView
    android:id="@+id/gridview"
    android:layout_width="match_parent"
    android:layout_height="wrap_content"
    android:numColumns="3"
    android:gravity="center"
    android:stretchMode="columnWidth" />

You can also register click listeners on the rows of the GridView. To do this, use the 


          
           
setOnItemClickListener
method like so: 

          
           
gridView.setOnItemClickListener(new OnItemClickListener() {
    @Override
    public void onItemClick(AdapterView<!--?--> parent, View view, int position, long id) {
        // Handle click event here
    }
});
The callback passed into the 
onItemClick
method will receive a reference to the GridView, the current item view, its position in the list, and its ID.

Conclusion

The GridView component is a powerful, yet simple way to display a scrollable collection of items in an Android app. The component can easily be customized using XML attributes and custom click listeners, making it a great choice for lists, grids, image galleries, and more.
