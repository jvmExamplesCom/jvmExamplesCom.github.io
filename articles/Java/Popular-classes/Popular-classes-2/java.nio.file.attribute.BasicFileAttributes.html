Introduction to java.nio.file.attribute.BasicFileAttributes

The java.nio.file.attribute.BasicFileAttributes class is a key class in the Java NIO package. This class provides access to the basic attributes of a file. It supports methods to get the size, creation time, and last modified time of a file. It also provides support for reading and writing file attributes, and determining if a file is a directory or a regular file.

Import Statement


          
           
import java.nio.file.attribute.BasicFileAttributes;

Code Example 1 - Getting File Size with BasicFileAttributes

The following code example prints the size of a file. The size is returned in bytes. 


          
           
Path filePath = Paths.get("data.txt");
BasicFileAttributes attr;
attr = Files.readAttributes(filePath, BasicFileAttributes.class);
System.out.println("Size: " + attr.size());

Code Example 2 - Checking if File is a Directory

The following code example checks if a file is a directory.


          
           
Path filePath = Paths.get("data.txt");
BasicFileAttributes attr;
attr = Files.readAttributes(filePath, BasicFileAttributes.class);
if (attr.isDirectory()) {
    System.out.println("This is a directory.");
} else {
    System.out.println("This is not a directory.");
}

Code Example 3 - Getting Time Attributes

The following code example prints out the creation time and last modified time of a file.


          
           
Path filePath = Paths.get("data.txt");
BasicFileAttributes attr;
attr = Files.readAttributes(filePath, BasicFileAttributes.class);
System.out.println("Creation time: " + attr.creationTime());
System.out.println("Last modified time: " + attr.lastModifiedTime());

Conclusion

The java.nio.file.attribute.BasicFileAttributes class provides access to the basic attributes of a file, such as its size, creation time, and last modified time. It also provides support for reading and writing file attributes and determining if a file is a directory or a regular file.
