Java.util.StringJoiner Overview

The Java.util.StringJoiner class is a convenience class for constructing strings from iterable objects such as collections and arrays. This class provides utility methods for handling delimiters, prefixes, suffixes, and creates a new StringJoiner from a collection of objects.

Constructors

The following constructors are available for creating a StringJoiner instance:


          
           
StringJoiner sj1 = new StringJoiner(","); //Delimiter
StringJoiner sj2 = new StringJoiner(",", "prefix-", "-suffix"); //Delimiter and Prefix-Suffix
StringJoiner sj3 = new StringJoiner(",", "[", "]"); //Delimiter and Brackets
StringJoiner sj4 = new StringJoiner(null, null, null); //No delimiter

Add() Method

The add() method is used to add elements to the existing StringJoiner instance. This method takes a CharSequence as a parameter and can be chained together.


          
           
StringJoiner sj1 = new StringJoiner(",");
sj1.add("A").add("B").add("C"); //"A,B,C"

Merge() Method

The merge() method is used to merge two StringJoiner instances into one. It takes an instance of StringJoiner as a parameter and adds it to the existing StringJoiner instance.


          
           
StringJoiner sj1 = new StringJoiner(",");
sj1.add("A").add("B").add("C");
StringJoiner sj2 = new StringJoiner(",");
sj2.add("D").add("E").add("F");

sj1.merge(sj2); //"A,B,C,D,E,F"

toString() Method

The toString() method is used to return a string representation of the StringJoiner instance. It returns the elements added to the StringJoiner, with the prefix and suffix present if provided during construction.


          
           
StringJoiner sj1 = new StringJoiner(",", "[", "]");
sj1.add("A").add("B").add("C");

System.out.println(sj1.toString()); //"[A,B,C]"
