Introduction to Java.util.ArrayList

The Java.util.ArrayList class is a collection of objects that grow dynamically as items are added and removed from the list. It provides various methods for accessing and manipulating the collection, such as retrieving objects, searching for objects, deleting objects, and sorting the elements. By leveraging the power of generics, ArrayLists can also be used to store any type of data.

Creating an ArrayList

An ArrayList can be created in several different ways:


          
           
// Create empty ArrayList
ArrayList<String> myList = new ArrayList<>();

// Create an ArrayList initialized with values
ArrayList<Integer> numbers = new ArrayList<>(Arrays.asList(1, 2, 3, 4, 5));

// Create an ArrayList from another collection
ArrayList<String> strings = new ArrayList<>(myList);

Once an ArrayList has been created, items can be added using the 

add()
method: 

          
           
// Add a single item
myList.add("Hello World");

// Add multiple items
numbers.addAll(Arrays.asList(6, 7, 8, 9, 10));
Items can be removed by index or value: 

          
           
// Remove item by index
myList.remove(0);

// Remove item by value
numbers.remove((Integer)5);

Accessing Elements of an ArrayList

Accessing elements by index is one of the most common operations on an ArrayList. It can be done using the 

get()
method: 

          
           
String element = myList.get(0);
The size of the list can be accessed using the 
size()
method: 

          
           
int size = myList.size();

Searching an ArrayList

The 
contains()
method can be used to search for an element in an ArrayList: 

          
           
boolean hasElement = myList.contains("Hello World");

The index of an element can be retrieved using the 

indexOf()
method: 

          
           
int index = myList.indexOf("Hello World");

Sorting an ArrayList

The elements of an ArrayList can be sorted using the 

sort()
method. This method requires a 
Comparator
which is responsible for defining the ordering of the elements: 

          
           
myList.sort(Comparator.naturalOrder());

Conclusion

In this tutorial, we have introduced the Java.util.ArrayList class and explored its various methods for accessing and manipulating the collection. We have also discussed how to create an ArrayList, add items to it, and remove them. Finally, we have seen how to search and sort an ArrayList.
