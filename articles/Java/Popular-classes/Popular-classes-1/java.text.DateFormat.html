java.text.DateFormat

The 

          
           
java.text.DateFormat
class is a one of the classes used in internationalization and localization in Java. It provides support for formatting and parsing dates and times. The DateFormat class also provides methods to be used to parse and format dates in any locale. This allows applications to work in multiple languages.

Formats a Date

The 

          
           
java.text.DateFormat
class provides the 
format(Date date)
method, which formats a date according to the given date format. For example, the following code formats a date in the default format: 

          
           
Date date = new Date();
DateFormat dateFormat = DateFormat.getInstance();
String output = dateFormat.format(date);
System.out.println(output);

This will print the date in the current locale's default format, such as 'MM-dd-yyyy':


          
           
03-18-2020

Parses a Date

The 

          
           
java.text.DateFormat
class also provides the 

          
           
parse(String source)
method, which parses a date string into a 
Date
object. For example, the following code parses a date string: 

          
           
String source = "03-18-2020";
DateFormat dateFormat = DateFormat.getInstance();
Date date = dateFormat.parse(source);
System.out.println(date);

This will print the parsed date object in the current locale's default format, such as the following:


          
           
Wed Mar 18 00:00:00 IST 2020

Time Zones

The 

          
           
java.text.DateFormat
class provides the 
setTimeZone()
method, which allows you to specify a time zone. This is useful when dealing with dates or times in different locales. For example, the following code sets the time zone to the Eastern Standard Time (EST) zone: 

          
           
TimeZone timeZone = TimeZone.getTimeZone("EST");
dateFormat.setTimeZone(timeZone);

java.text.DateFormat
class also provides support for message formatting. This is useful when displaying an internationalized message that contains a date string. The 
MessageFormat
