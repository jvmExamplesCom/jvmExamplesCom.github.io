java.util.Calendar

The 

          
           
java.util.Calendar
class is a useful tool for performing date and time calculations in Java. It has various methods and variables that allow developers to compare, format, and manipulate dates and times. This article will discuss how to use the Calendar class in Java and provide code examples to illustrate this.

Using Calendar

The 

          
           
Calendar
class can be used in several ways. It can be used to compare two dates or times, add or subtract days, find out the day of the week, or even format a date. All of this requires the use of the class's methods and variables.

Comparing Dates

The method 

          
           
calendar.compareTo()
can be used to compare two dates. It returns a negative number if the first date is earlier than the second date, zero if the two dates are equal, and a positive number if the first date is later than the second date. 

          
           
Calendar date1 = Calendar.getInstance();
Calendar date2 = Calendar.getInstance();
date2.add(Calendar.DATE, 10);   
int comparison =  date1.compareTo(date2);
System.out.println(comparison);
In the example above, 

          
           
date2
is 10 days later than 

          
           
date1
, so 

          
           
comparison
will be a positive number.

Adding and Subtracting Days

The 

          
           
add()
and 

          
           
subtract()
methods can be used to add or subtract days from a date. The first argument of the method is the field being modified (e.g. year, month, day) and the second argument is the amount of days being added or subtracted. 

          
           
Calendar date = Calendar.getInstance();
date.add(Calendar.DATE, 7);
System.out.println(date.toString());
date.subtract(Calendar.DATE, 3);
System.out.println(date.toString());
The example above adds seven days to 

          
           
date
and then subtracts three days from it. The resulting date will be four days later than the initial date.

Finding the Day of the Week

The 

          
           
get()
method can be used to find out the day of the week for a certain date. It requires one argument, which is the field being queried (day of the week). The method returns an integer from 0-6, where 0 is Sunday and 6 is Saturday. 

          
           
Calendar date = Calendar.getInstance();
int dayOfWeek = date.get(Calendar.DAY_OF_WEEK);
System.out.println(dayOfWeek);
The example above will print out the day of the week of the current date.

Formatting a Date

The 

          
           
format()
method can be used to format a date into a particular pattern. This method takes a DateFormat object as an argument, which is then used to determine how the date should be formatted. 

          
           
Calendar date = Calendar.getInstance();
SimpleDateFormat formatter = new SimpleDateFormat("MM/dd/yyyy");
String formattedDate = formatter.format(date.getTime());
System.out.println(formattedDate);
The example above uses the SimpleDateFormat class to create a DateFormat object which will format the date into the pattern "MM/dd/yyyy". The resulting String will contain the formatted date.

Conclusion

The 

          
           
java.util.Calendar
class is a powerful tool for working with dates and times in Java. It allows developers to easily compare dates, add or subtract days, find out the day of the week, and format dates into particular patterns. The code examples provided in this article should help developers get started using the Calendar class.
