java.util.Collection Overview

Java collection framework provides a set of interfaces and classes which help in storing and manipulating the group of objects. The Collection in java is a framework that provides an architecture to store and manipulate the group of objects. Java Collection framework provides many interfaces (Set, List, Queue, Deque) and classes (ArrayList, Vector, LinkedList, PriorityQueue), etc. to represent a group of objects as a single unit.

The Collection interface ( java.util.Collection ) is the root interface in the collection hierarchy. A collection represents a group of objects known as its elements. The Collection interface provides the most commonly used methods for working with collections such as size(), add(), remove(), contains(), etc. All such methods are defined in the Collection interface and can be accessed by any class which implements the Collection interface.

Syntax


          
           
public interface Collection<E>
    extends Iterable<E> {
    // Bulk Operations
    int size();
    boolean isEmpty();
    boolean contains(Object element);
    boolean add(E element);
    boolean remove(Object element);

    // Array Operations
    Object[] toArray();
    <T> T[] toArray(T[] a);
 
    // Iterators
    Iterator<E> iterator();
 
    // Bulk Operations
    boolean containsAll(Collection<?> c);
    boolean addAll(Collection<? extends E> c);
    boolean removeAll(Collection<?> c);
    boolean retainAll(Collection<?> c);
    void clear();
 
    // Comparison and hashing
    boolean equals(Object o);
    int hashCode();
}

Example

Let's see a simple example of Collection interface implementation where we are creating an instance of ArrayList arraylist, adding elements in it and iterating over it. 


          
           
import java.util.*;
 
public class CollectionDemo {
   public static void main(String args[]) {
      // Create a collection
      Collection<string> collection = new ArrayList<string>();
 
      // Add elements in the collection
      collection.add("Element 1");
      collection.add("Element 2");
      collection.add("Element 3");
      collection.add("Element 4");
      collection.add("Element 5");
  
      // Iterating over the collection
      Iterator iterator = collection.iterator();
      while (iterator.hasNext()) {
         System.out.println(iterator.next());
      }
   }
}
