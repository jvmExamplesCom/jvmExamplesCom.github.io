android.widget.TextView

TextView is a View in Android UI that displays text or other characters. It is a base class of EditText and this class is used to create text field. This is one of the most used views in Android applications mainly because it is used to display text to the user.

Creating TextView

To create a TextView you need to add the following lines of code inside your Activity. 

          
           
TextView textView = new TextView(this);

Customizing TextView

Android provides various attributes to customize a TextView like changing its font size, color, background, etc.

Font Color

You can setTextColor() method to change the text color. 

          
           
textView.setTextColor(Color.RED);

Font Size

You can use setTextSize() method to change font size. 

          
           
textView.setTextSize(20);

Font Style

You can also use setTypeface() method to change the text style. 

          
           
textView.setTypeface(Typeface.BOLD);

Background

Changing background color of TextView can be done using setBackgroundColor() method. 

          
           
textView.setBackgroundColor(Color.GREEN);

TextView onClick Event

Android provides onClickListener attribute to attach click events to TextView. To set onClickListener for TextView, use the following code. 

          
           
textView.setOnClickListener(new View.OnClickListener(){
     public void onClick(View view){
          // Your code here
     }
 });

TextView Properties

Following are some properties associated with TextView: 

          
           
maxLines – Maximum number of lines to be shown in TextView
minLines – Minimum number of lines to be shown in TextView
autoLink – If this attribute is set as “web”, links in the text are automatically found and converted to clickable links.
