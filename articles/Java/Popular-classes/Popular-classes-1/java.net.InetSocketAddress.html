java.net.InetSocketAddress

java.net.InetSocketAddress is a Java class that represents an Internet Protocol (IP) address and a port number combination. It is used to connect clients and servers on the same or different machines. This class provides methods for creating, parsing, and managing InetSocketAddress objects. It also allows you to set and retrieve IP addresses, socket timeouts, and other properties.

Creating an InetSocketAddress Object

An InetSocketAddress object can be created in several ways. The simplest way is with two parameters, an IP address and a port number. For example:


          
           
InetSocketAddress addr = new InetSocketAddress("127.0.0.1", 8080);

This will create an InetSocketAddress object with the IP address “127.0.0.1” and the port number “8080”. To create an InetSocketAddress object from an existing Socket object, you can use the following code:


          
           
InetSocketAddress addr = new InetSocketAddress(socket.getInetAddress(), socket.getPort());

This code creates an InetSocketAddress object from the IP address and port of an existing Socket object.

Parsing an InetSocketAddress

An InetSocketAddress can be parsed from a string by using the static method parse(). For example, the following code parses an IP address and port from a string:


          
           
InetSocketAddress addr = InetSocketAddress.parse("127.0.0.1:8080");

This will create an InetSocketAddress object with the IP address “127.0.0.1” and the port number “8080”.

Setting and Retrieving Properties

InetSocketAddress provides several methods for setting and retrieving various properties related to sockets and IP addresses. The following example shows how to set the socket timeout for an InetSocketAddress object:


          
           
addr.setSoTimeout(10000);

The above example sets the socket timeout to 10 seconds. To retrieve the current socket timeout, you can use the following code:


          
           
int timeout = addr.getSoTimeout();

The above code will return the current socket timeout in milliseconds.
