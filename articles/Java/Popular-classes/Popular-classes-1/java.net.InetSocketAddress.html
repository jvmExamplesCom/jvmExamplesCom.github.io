java.net.InetSocketAddress is a Java class that represents an Internet Protocol (IP) address and a port number combination. It is used to connect clients and servers on the same or different machines. This class provides methods for creating, parsing, and managing InetSocketAddress objects. An InetSocketAddress object can be created in several ways. The simplest way is with two parameters, an IP address and a port number. For example:


          
           
InetSocketAddress addr = new InetSocketAddress("127.0.0.1", 8080);

InetSocketAddress addr = new InetSocketAddress(socket.getInetAddress(), socket.getPort());

InetSocketAddress addr = InetSocketAddress.parse("127.0.0.1:8080");

This will create an InetSocketAddress object with the IP address “127.0.0.1” and the port number “8080”.

Setting and Retrieving Properties
