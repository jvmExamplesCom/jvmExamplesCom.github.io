java.net.InetSocketAddress is a Java class that represents an Internet Protocol (IP) address and a port number combination. An InetSocketAddress object can be created in several ways. The simplest way is with two parameters, an IP address and a port number. For example:


          
           
InetSocketAddress addr = new InetSocketAddress("127.0.0.1", 8080);

InetSocketAddress addr = InetSocketAddress.parse("127.0.0.1:8080");

Setting and Retrieving Properties
