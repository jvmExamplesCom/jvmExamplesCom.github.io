java.io.FileOutputStream

java.io.FileOutputStream class is used to write data to the file. It is part of the Java IO package and allows writing to binary files. This class opens a file or an OutputStream, depending on what is passed to it as an argument.

The following code snippet demonstrates how to create a FileOutputStream object:


          
           
FileOutputStream fos = new FileOutputStream("C:/example.txt");

This example creates a FileOutputStream object that writes to a file located at C:/example.txt.

Writing to a FileOutputStream Object

The FileOutputStream class provides several methods for writing data to the file. The most commonly used methods are write(), close(), and flush().

The following code snippet demonstrates how to write data to a FileOutputStream object:


          
           
fos.write("Hello World!".getBytes());

This example writes the string "Hello World!" to the file that was opened in the first example using the FileOutputStream object.

Closing the FileOutputStream Object

Once the FileOutputStream object is no longer needed, it must be closed by calling the close() method. This ensures that all data is properly written to the file before the object is closed. The following code snippet demonstrates how to close a FileOutputStream object:


          
           
fos.close();

This example closes the FileOutputStream object that was opened in the first example.

Flushing the FileOutputStream Object

The flush() method can be used to ensure that all data is written to the file. This is usually done when the file needs to be closed immediately and the contents of the file need to be written to disk. The following code snippet demonstrates how to flush a FileOutputStream object:


          
           
fos.flush();

This example flushes the FileOutputStream object that was opened in the first example.

Using the FileOutputStream Class with Try-with-Resources

The FileOutputStream class is also compatible with the try-with-resources statement. This statement allows the object to be opened, written to, and closed with one line of code. The following code snippet demonstrates how to use the FileOutputStream class with the try-with-resources statement:


          
           
try (FileOutputStream fos = new FileOutputStream("C:/example.txt")) {
    fos.write("Hello World!".getBytes());
}

This example creates a FileOutputStream object, writes the string "Hello World!" to the file, and closes the FileOutputStream object in one line of code.
