java.time.format.DateTimeFormatter

The java.time.format.DateTimeFormatter class is a useful class for formatting and parsing date/time objects in Java applications. It provides methods for formatting different types of dates and times, such as LocalDate, LocalTime, and DateTime. Creating a DateTimeFormatter Object

The DateTimeFormatter class has several static methods that can be used to create a DateTimeFormatter object. The simplest way to create a DateTimeFormatter object is to use the ofPattern() method. This method takes a String argument that specifies the pattern of the desired format. Here is an example of creating a DateTimeFormatter object with the ofPattern() method:


          
           
DateTimeFormatter formatter = DateTimeFormatter.ofPattern("yyyy-MM-dd");

In this example, we are creating a DateTimeFormatter object that will use the format of "yyyy-MM-dd" for formatting DateTime objects. In this case, the DateTime object will be formatted as a four-digit year, followed by a two-digit month, followed by a two-digit day.

Formatting a DateTime Object

Once you have created your DateTimeFormatter object, you can use it to easily format a DateTime object. LocalDate date = LocalDate.now();
String formattedDate = formatter.format(date);
System.out.println(formattedDate);

In this example, we are getting the current LocalDate and then formatting it with our DateTimeFormatter object. The output of this code will be a String representation of the current date in the format of "yyyy-MM-dd".

Parsing a DateTime Object
