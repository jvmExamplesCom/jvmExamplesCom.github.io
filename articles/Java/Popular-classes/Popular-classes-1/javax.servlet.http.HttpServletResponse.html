javax.servlet.http.HttpServletResponse

The 

          
           
javax.servlet.http.HttpServletResponse
class is used to encapsulate the response from a servlet that generates content for an HTTP request. This class provides methods for adding headers, setting status codes, writing data to the client, and other various tasks.

Creating Instance

The 
HttpServletResponse
class provides a constructor method that creates an instance of the class: 

          
           
HttpServletResponse response = new HttpServletResponse();
The constructor requires no parameters.

Adding Headers

The 
addHeader()
method is used to add a header to the response: 

          
           
response.addHeader("Content-Type", "application/json");

This method takes two parameters - the name of the header and the value for the header.

Setting Status Code

The 
setStatusCode()
method is used to set the status code for the response: 

          
           
response.setStatusCode(200);

This method takes one parameter - the HTTP status code.

PrintWriter

PrintWriter writer = response.getWriter();
This method takes no parameters and returns a

This method takes one parameter - the data to write to the client.
