java.text.SimpleDateFormat

java.text.SimpleDateFormat is a concrete class for formatting and parsing dates in a locale-sensitive manner. It allows for formatting (date → text), parsing (text → date), and normalization. SimpleDateFormat allows you to start by choosing any user-defined patterns for date-time formatting.

Formatting Dates

To format a date according to a specific pattern, use the format() method of SimpleDateFormat class:


          
           
SimpleDateFormat simpleDateFormat = new SimpleDateFormat("dd-MM-yyyy HH:mm:ss");
String dateString = simpleDateFormat.format(new Date());
System.out.println(dateString);
The above code prints out the current date in 
dd-MM-yyyy HH:mm:ss
format: 

          
           
20-09-2020 20:34:02
You can also specify custom formats: 

          
           
SimpleDateFormat simpleDateFormat = new SimpleDateFormat("EEEE dd MMMM yyyy");
String dateString = simpleDateFormat.format(new Date());
System.out.println(dateString);
The following code prints out the current date in 
EEEE dd MMMM yyyy
format: 

          
           
Tuesday 22 September 2020

Parsing Dates

To parse a string representing a date and time according to a specific pattern, use the parse() method of SimpleDateFormat class:


          
           
String dateString = "20-09-2020 20:34:02";
SimpleDateFormat simpleDateFormat = new SimpleDateFormat("dd-MM-yyyy HH:mm:ss");
Date date = simpleDateFormat.parse(dateString);
System.out.println(date);

The above code prints out the date from the given string:


          
           
Tue Sep 22 20:34:02 GMT 2020

Normalizing Dates

Normalizing is the process of converting a date that is localized according to the system's current locale to a UTC standard. To normalize a date, use the normalize() method in combination with the getTimeZone() method:


          
           
TimeZone gmtTimeZone = TimeZone.getTimeZone("GMT");
SimpleDateFormat simpleDateFormat = new SimpleDateFormat("EEEE dd MMMM yyyy");
simpleDateFormat.setTimeZone(gmtTimeZone);
Date date = simpleDateFormat.parse("Tuesday 22 September 2020");
Date dateNormalized = simpleDateFormat.normalize(date);
System.out.println(dateNormalized);

The code prints out the normalized date according to the given time zone:


          
           
Tue Sep 22 00:00:00 GMT 2020
