Introduction to android.support.design.widget.Snackbar

android.support.design.widget.Snackbar

Snackbars are widely used on Android for displaying short-lived messages, such as updating the user after performing a task, during idle times, or through error handling.

make()

The second parameter is the message string. If the message is longer than one line, it will be wrapped into multiple lines to fit the view.
