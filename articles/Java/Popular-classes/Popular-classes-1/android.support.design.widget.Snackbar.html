Introduction to android.support.design.widget.Snackbar

The 

          
           
android.support.design.widget.Snackbar
class is a lightweight message that provides quick feedback about an operation in the form of a simple message at the bottom of the user's device screen. This type of message supports both text and action components, allowing the user to make a contextual decision or initiate a workflow. Snackbars are widely used on Android for displaying short-lived messages, such as updating the user after performing a task, during idle times, or through error handling.

How to Create a Snackbar

Creating a Snackbar requires two components: a view and a message. To create a Snackbar, use the static method 

          
           
make()
which takes a View as its first parameter. The View is used to anchor the Snackbar within the application's view hierarchy, and is typically the root view of the activity or fragment. The second parameter is the message string. If the message is longer than one line, it will be wrapped into multiple lines to fit the view. 

          
           
Snackbar snackbar = Snackbar.make(rootView, "This is a snackbar");

Customizing the Snackbar

The Snackbar class has several methods that can be used to customize the look and feel of the Snackbar. For example, the 

          
           
setAction()
method can be used to add an action to the Snackbar. The action appears as a button below the message and can be used to initiate a workflow. 

          
           
snackbar.setAction("Dismiss", new View.OnClickListener() {
    @Override
    public void onClick(View v) {
        snackbar.dismiss();
    }
});
The Snackbar also has methods for changing the background color and text color of the Snackbar, setting a duration for how long the Snackbar is displayed on the screen, and adding a swipe-to-dismiss behavior.

Displaying the Snackbar

To display the Snackbar, use the 

          
           
show()
method. 

          
           
snackbar.show();
The Snackbar is displayed on the user's device for the length of time specified in the 

          
           
setDuration()
method, or until it is dismissed by the user with a swipe gesture.

Conclusion

The Snackbar is a simple, lightweight message intended to provide quick feedback to the user.
