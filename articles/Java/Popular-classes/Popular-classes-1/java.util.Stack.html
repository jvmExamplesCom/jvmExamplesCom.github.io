java.util.Stack

Java.util.Stack is a class in the Java Collections Framework. It is typically used for data structures such as stacks and queues, where elements are added and removed from the top of the stack. The class provides a number of useful methods for manipulating elements in a stack.

Stack Constructor

The Stack constructor takes an optional initial capacity and optional maximum capacity arguments:


          
           
Stack<Integer> myStack = new Stack<>(10, 20);

The maximum capacity argument is optional and can be omitted if the default maximum capacity for the stack is desired.

add() Method

The add() method adds an element to the top of the stack:


          
           
myStack.add(1);

The add() method returns true if the element was successfully added, and false otherwise.

peek() Method

The peek() method retrieves the top element of the stack without removing it:


          
           
int peekedElement = myStack.peek();

The peek() method returns null if the stack is empty.

pop() Method

The pop() method retrieves and removes the top element from the stack:


          
           
int poppedElement = myStack.pop();

The pop() method returns null if the stack is empty.

search() Method

The search() method searches for an element in the stack and returns its position in the stack (or -1 if the element is not in the stack):


          
           
int position = myStack.search(5);

isEmpty() Method

The isEmpty() method tests if the stack is empty and returns true if the stack is empty, false otherwise:


          
           
boolean isEmpty = myStack.isEmpty();

size() Method

The size() method returns the number of elements in the stack:


          
           
int size = myStack.size();
