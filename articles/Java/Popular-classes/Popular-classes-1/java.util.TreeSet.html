java.util.TreeSet

Java TreeSet is a powerful data structure that is used to store and manage data in a tree-like structure. In contrast to HashSet, TreeSet maintains the elements in ascending or descending order. It implements the NavigableSet interface along with the Set interface. TreeSet does not allow duplicate elements, so if you try to insert a duplicate element, add() method will return false.

Constructors

TreeSet offers four types of constructors: 
  • TreeSet()
    - This constructor builds an empty TreeSet that maintains its elements in their natural order.
  • 
            
             
TreeSet(Comparator<!--? super E--> comparator)
    - This constructor builds an empty TreeSet that maintains its elements based on the given Comparator.
  • 
            
             
TreeSet(Collection<!--? extends E--> c)
    - This constructor builds a TreeSet that contains all the elements of the given collection c, storing them in their natural order.
  • 
            
             
TreeSet(SortedSet ss)
    - This constructor builds a TreeSet that contains all the elements of the given sorted set ss, storing them in their natural order.

Methods

TreeSet also provides several methods for manipulating its elements, some of which are mentioned below:

  • add(E e)
    - This method adds the specified element e to the TreeSet if it is not already present.
  • remove(Object o)
    - This method removes the specified element o from the TreeSet.
  • contains(Object o)
    - This method returns true if the TreeSet contains the specified element o, otherwise it returns false.
  • iterator()
    - This method returns an iterator over the elements in this TreeSet in the natural order.
  • headSet(E e)
    - This method returns a view of the portion of this TreeSet whose elements are less than (or equal to, if inclusive is true) e.
  • tailSet(E e)
    - This method returns a view of the portion of this TreeSet whose elements are greater than (or equal to, if inclusive is true) e.
  • 
            
             
subSet(E fromElement, E toElement)
    - This method returns a view of the portion of this TreeSet whose elements range from fromElement, inclusive, to toElement, exclusive.
  • first()
    - This method returns the first (lowest) element currently in the TreeSet.
  • last()
    - This method returns the last (highest) element currently in the TreeSet.
  • isEmpty()
    - This method returns true if the TreeSet contains no elements, false otherwise.
  • size()
    - This method returns the number of elements in the TreeSet.

Example

Below is a simple example that demonstrates how to create a TreeSet, add and remove elements from it, and iterate over it. We create a TreeSet of strings, add three strings to it, and then use an iterator to iterate over the elements in the TreeSet.


          
           
import java.util.TreeSet;
import java.util.Iterator;

public class Test {
    public static void main(String[] args) {

        // Create a TreeSet
        TreeSet<String> set = new TreeSet<String>();

        // Add elements to the TreeSet
        set.add("John");
        set.add("Adam");
        set.add("Steve");

        // Iterate over the elements in the TreeSet
        Iterator<String> iterator = set.iterator();
        while(iterator.hasNext()){
            System.out.println(iterator.next());
        }
    }
}
The output of the above code is: 

          
           
Adam
John
Steve
