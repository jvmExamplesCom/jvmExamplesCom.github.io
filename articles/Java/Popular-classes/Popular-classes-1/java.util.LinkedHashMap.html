java.util.LinkedHashMap

The 

          
           
java.util.LinkedHashMap
is a class that implements the 
Map
interface and extends the 
HashMap
class. It creates an ordered map with a doubly-linked list between its entries, preserving the insertion order of the elements in the backing list. LinkedHashMaps are an ideal choice when you need to maintain the insertion order of your key-value pairs.

Constructors

Methods

The LinkedHashMap provides several methods to access or manipulate the data in the map. The most commonly used methods are listed here:

  • get()
    : Returns the value for the specified key.
  • put()
    : Adds or updates a key-value pair in the map.
  • remove()
    : Removes the specified key and its associated value from the map.
  • clear()
    : Removes all key-value pairs from the map.
  • containsKey()
    : Returns true if the map contains the specified key.
  • containsValue()
    : Returns true if the map contains the specified value.
  • entrySet()
    : Returns a set of all key-value pairs in the map.
  • keySet()
    : Returns a set of all keys in the map.
  • size()
    : Returns the number of key-value pairs in the map.
  • values()
    : Returns a Collection of all values in the map.

Example

Here is a simple example of using a LinkedHashMap:


          
           
LinkedHashMap<String, Integer> ages = new LinkedHashMap<>();
ages.put("John", 30);
ages.put("Jane", 28);
ages.put("Bob", 32);

for (String name : ages.keySet()) {
    System.out.println(name + " is " + ages.get(name) + " years old.");
}

// John is 30 years old.
// Jane is 28 years old.
// Bob is 32 years old.

LinkedHashMaps are widely used in Java applications since they provide a way to maintain the insertion order of the elements. This can be especially useful for maintaining the order of entries in a list or for keeping track of the sequence of operations.
