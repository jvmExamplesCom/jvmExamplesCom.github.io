java.net.URI

The java.net.URI class represents a Uniform Resource Identifier (URI) reference. With the two constructors, it can be easily created with a string representation of the URI.

Constructors

The 
java.net.URI
class has two constructors: 

          
           
public URI(String s) throws URISyntaxException
This constructor builds a URI from a string. 

          
           
public URI(String scheme, String ssp, String fragment) throws URISyntaxException

This constructor builds a URI from its components. The string argument ‘ssp’ is the scheme specific part. This constructor throws a URISyntaxException if any of the arguments are malformed or is null.

Example

To use the 
java.net.URI
class you first need to import the package: 

          
           
import java.net.URI;

Then you use the constructors to create the URI object:


          
           
String myURI = "http://www.example.com";
URI uri = new URI(myURI);

String scheme = "ftp";

String fragment = "download";

java.net.URI
class has several useful methods to get the parts of the URI: 

          
           
public String getScheme()
This method returns the scheme of the URI. 

          
           
public String getRawSchemeSpecificPart()

This method returns the scheme specific part of the URI.


          
           
public String getAuthority()

This method returns the authority part of the URI.
