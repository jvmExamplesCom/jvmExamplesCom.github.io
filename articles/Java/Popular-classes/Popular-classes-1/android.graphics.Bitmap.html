android.graphics.Bitmap

Android.graphics.Bitmap is a class in the Android Graphics package that provides methods for managing Bitmap objects. A Bitmap represents an image, either as a file on disk or an image in memory. The class allows manipulation of bitmap data, such as scaling, cropping and adding filters. It also contains methods for drawing the bitmap to a canvas, which can be used to create representations of the bitmap suitable for display.

Bitmap.create()
