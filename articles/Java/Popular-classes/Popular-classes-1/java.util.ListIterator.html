java.util.ListIterator

The ListIterator interface is a subinterface of Iterator found in the java.util package. It provides the ability to traverse the list in either direction, modify the list during iteration and obtain the index of the current element. It has 5 methods as follows:

  • hasNext() – returns true if this list iterator has more elements when traversing the list in the forward direction.
  • next() – returns the next element in the list and advances the cursor position.
  • hasPrevious() – returns true if this list iterator has more elements when traversing the list in the reverse direction.
  • previous() – returns the previous element in the list and moves the cursor position backwards.
  • nextIndex() – returns the index of the element that would be returned by a subsequent call to next()

Example 1

Let's create a simple example with a list and a list iterator, which we will use to iterate over the list in both directions:


          
           
List<String> list = new ArrayList<>();
list.add("a");
list.add("b");
list.add("c");

ListIterator<String> iterator = list.listIterator();

We can now use the list iterator to traverse the list in both directions. To use the ‘next’ method, we would do something like this:


          
           
while(iterator.hasNext()) {
   String element = iterator.next();
   System.out.println(element);
}

This will print out each element in the list, in order.

To use the ‘previous’ method, we would do something like this:


          
           
while(iterator.hasPrevious()) {
   String element = iterator.previous();
   System.out.println(element);
}

This will print out each element in the list, in reverse order.

Example 2

Let's take a look at another example, this time we will use the ‘nextIndex’ method to get the index of the current element:


          
           
List<Integer> list = new ArrayList<>();
list.add(1);
list.add(2);
list.add(3);

ListIterator<Integer> iterator = list.listIterator();

while(iterator.hasNext()) {
   Integer element = iterator.next();
   int index = iterator.nextIndex();
   System.out.println(element + " at index " + index);
}

This will print out each element in the list, along with its index in the list.

Example 3

Finally, let's take a look at an example where we modify the list while iterating over it:


          
           
List<String> list = new ArrayList<>();
list.add("a");
list.add("b");
list.add("c");

ListIterator<String> iterator = list.listIterator();

while(iterator.hasNext()) {
   String element = iterator.next();
   if(element.equals("b")) {
      iterator.set("d");
   }
}

System.out.println(list);

This will replace the element with the value ‘b’ with the value ‘d’ in the list, and the output will be [a, d, c].
