android.os.Handler

The 

          
           
android.os.Handler
class provides a mechanism for taking action after a given delay, or at regular intervals. It is an essential component of the Android SDK and is used to manage background tasks in any Android application. Basic Usage

To use an 

          
           
android.os.Handler
in your application, you must first define an instance of 

          
           
Handler
and then set up the code that will be executed by the handler. The following code example shows how to define a 

          
           
Handler

Runnable runnable = new Runnable() {
