android.os.Handler

The 

          
           
android.os.Handler
class provides a mechanism for taking action after a given delay, or at regular intervals. It is an essential component of the Android SDK and is used to manage background tasks in any Android application. Basic Usage

android.os.Handler

Handler

Handler
