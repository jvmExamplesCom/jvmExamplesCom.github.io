Introduction to Java.util.EnumSet

Java.util.EnumSet is an unmodifiable set of zero or more elements that are all of the same enum type. This powerful and convenient utility class allows you to group enums together as a collection and it provides efficient, specialized methods for manipulating these collections.

Creating an EnumSet

The simplest way to create an EnumSet is to call the static emptySet() method. This returns an empty set that is not connected to any particular enum. To add elements to the set, use the of() method. For example: 

          
           
EnumSet<DayOfTheWeek> days = EnumSet.of(DayOfTheWeek.MONDAY, DayOfTheWeek.FRIDAY);
This creates a set with Monday and Friday as the only elements. You can also use the allOf() method to create an EnumSet with every element from an enum included: 

          
           
EnumSet<DayOfTheWeek> allDays = EnumSet.allOf(DayOfTheWeek.class);
Similarly, you can use the noneOf() method to create an empty set: 

          
           
EnumSet<DayOfTheWeek> noDays = EnumSet.noneOf(DayOfTheWeek.class);

Adding and Removing Elements

Once you have created an EnumSet, you may want to add or remove elements from it. To do this, use the add() and remove() methods. For example: 

          
           
EnumSet<DayOfTheWeek> days = EnumSet.of(DayOfTheWeek.MONDAY, DayOfTheWeek.FRIDAY);
days.add(DayOfTheWeek.WEDNESDAY);
days.remove(DayOfTheWeek.MONDAY);
This adds Wednesday to the set and removes Monday.

Iterating Over Enums

Combining EnumSets

EnumSet<DayOfTheWeek> days = EnumSet.of(DayOfTheWeek.MONDAY, DayOfTheWeek.FRIDAY);
EnumSet<DayOfTheWeek> weekendDays = EnumSet.complementOf(days);

EnumSet<DayOfTheWeek> daysCopy = EnumSet.copyOf(days);
