java.util.Base64

Java.util.Base64 is a class in Java that provides Base64 encoding and decoding of strings and binary data. In this article, we'll explore the various methods used to encode and decode strings and binary data into the Base64 format.

Encoding Strings

The most common way to encode strings is to use the 


          
           
Base64.encodeBase64String()
method. This method requires one argument, the string to be encoded, and will return the Base64 encoded result. An example of this can be seen below: 

          
           
String str = "Hello World!";
String encodedStr = Base64.encodeBase64String(str.getBytes());
System.out.println(encodedStr);
// Output: SGVsbG8gV29ybGQh

Decoding Strings

To decode a Base64 encoded string back to its original form, we can use the 


          
           
Base64.decodeBase64()
method. This method requires one argument, the string to be decoded, and will return the decoded result. An example of this can be seen below: 

          
           
String encodedStr = "SGVsbG8gV29ybGQh";
byte[] decodedBytes = Base64.decodeBase64(encodedStr);
String decodedStr = new String(decodedBytes);
System.out.println(decodedStr);
// Output: Hello World!

Encoding Binary Data

Encoding binary data into the Base64 format is also possible using the 


          
           
Base64.encodeBase64String()
method. This method requires two arguments, the binary data to be encoded and the offset of the data within the array, and will return the Base64 encoded result. An example of this can be seen below: 

          
           
byte[] data = new byte[16];
String encodedData = Base64.encodeBase64String(data, 0);
System.out.println(encodedData);
// Output: AAAAAAAAAAAAAAA==

Decoding Binary Data

To decode a Base64 encoded data back to its original form, we can use the 


          
           
Base64.decodeBase64()
method. This method requires three arguments, the encoded data, the offset of the data within the array, and the length of the encoded data, and will return the decoded result. An example of this can be seen below: 

          
           
String encodedData = "AQIDBA==";
byte[] decodedBytes = Base64.decodeBase64(encodedData, 0, 8);

// Output: [1, 2, 3, 4]
