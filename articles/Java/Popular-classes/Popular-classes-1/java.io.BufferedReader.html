Java BufferedReader

The java.io.BufferedReader class reads text from a character-input stream, buffering characters so as to provide for the efficient reading of characters, arrays, and lines. The buffer size may be specified, or the default size may be used. The default is large enough for most purposes.

Creating a BufferedReader

To create a BufferedReader, you will need an InputStreamReader which converts bytes from an underlying stream into characters that can be read by a BufferedReader. You can then pass the InputStreamReader to the BufferedReader constructor:


          
           
InputStreamReader isr = new InputStreamReader(myInputStream);
BufferedReader br = new BufferedReader(isr);

You can also pass a buffer size to the constructor to set the buffer size:


          
           
BufferedReader br = new BufferedReader(isr, 1024);

Reading from a Stream

Once you have created a BufferedReader, you can start reading from the stream. The 

readLine()
method reads one line of text from the stream: 

          
           
String line = br.readLine();
The 
read()
method reads one character from the stream: 

          
           
int c = br.read();
The 
read(char[] buf)
method reads characters into an array: 

          
           
char[] buf = new char[1024];
int numCharsRead = br.read(buf);

Closing a Stream

When you have finished reading from the stream, you should close it using the 

close()
method. This will free up any resources associated with the stream: 

          
           
br.close();
