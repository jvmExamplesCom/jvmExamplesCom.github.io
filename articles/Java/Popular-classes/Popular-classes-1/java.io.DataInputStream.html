java.io.DataInputStream

The Java class 

          
           
java.io.DataInputStream
provides an interface to read primitive data types from a given 
InputStream
or 
byte
array. Primitive data types include byte, double, float, int, long, short, and boolean. It also includes UTF-8 encoded characters. This class is useful for recovering the internal state of an application from the stream.

Constructors

There are three constructors that can be used to create a new instance of 

DataInputStream
: 

          
           
// Create a DataInputStream from a given InputStream
DataInputStream(InputStream in)

// Create a DataInputStream from a given byte array
DataInputStream(byte[] in)

// Create a DataInputStream from a given String
DataInputStream(String in)

Methods

The class provides a number of methods for reading the primitive data types, as well as UTF-8 encoded Strings.


          
           
// Read a boolean from the stream
boolean readBoolean()

// Read a byte from the stream
byte readByte()

// Read a double from the stream
double readDouble()

// Read a float from the stream
float readFloat()

// Read an int from the stream
int readInt()

// Read a long from the stream
long readLong()

// Read a short from the stream
short readShort()

// Read an encoded String from the stream
String readUTF()

Example

The following example shows how to use 
DataInputStream
to read primitive data types from a file: 

          
           
try (DataInputStream dis = 
  new DataInputStream(new FileInputStream("test.dat"))) {

  // Read and print a boolean
  System.out.println(dis.readBoolean());

  // Read and print a double
  System.out.println(dis.readDouble());

  // Read and print a float
  System.out.println(dis.readFloat());

  // Read and print an int
  System.out.println(dis.readInt()); 

  // Read and print a long
  System.out.println(dis.readLong());

  // Read and print a short
  System.out.println(dis.readShort());

  // Read and print an encoded String
  System.out.println(dis.readUTF());
}
