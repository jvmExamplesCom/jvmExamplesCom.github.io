java.io.FilenameFilter

Java IO Filename filter is a class which can be used to define filtering rules for file listings. It consists of a single method called accept() which takes two arguments: the directory in which the file or directory resides, and the name of the file or directory. The method should return true if the file should be included in a given listing, and false otherwise. The following code example shows how to implement a simple subclass of FilenameFilter which filter's out all "hidden" files and directories, ignoring any beginning with a period character (.) 

          
           
class HiddenFileFilter implements FilenameFilter  {
    public boolean accept(File dir, String name){
        return !name.startsWith(".");
    }
}
Once defined, the filter can be used with various File objects, as in the following example: 

          
           
File myDir = new File("/path/to/my/dir");
String[] children = myDir.list(new HiddenFileFilter());
Alternatively, the filter may also be specified as an anonymous inner class while calling the list() method directly. 

          
           
String[] children = myDir.list(new FilenameFilter(){
    public boolean accept(File dir, String name){
        return !name.startsWith(".");
    }
});
Finally, since Java 8, FilenameFilter can also be expressed as a lambda expression, as follows: 

          
           
String[] children = myDir.list((dir, name) -> !name.startsWith(".") );
This makes it possible to define the filter without having to write a dedicated class first.

Using FilenameFilter with a File Class

Using FilenameFilter with a File class allows you to implement powerful file filtering capabilities. In the following example, we will filter out all text files (".txt") from a given directory. 

          
           
File dir = new File("/path/to/my/dir");

String[] txtFiles = dir.list(new FilenameFilter() {
    public boolean accept(File dir, String name) {
        return name.endsWith(".txt");
   }
});

for (String txtFile : txtFiles) {
    System.out.println(txtFile);
}
In this example, the accept() method checks if a given filename ends with ".txt", and if it does, then it is included in the listing returned by the list() method.

Using FilenameFilter with a DirectoryStream

Path dir = Paths.get("/path/to/my/dir"); 

try (DirectoryStream<Path> stream = Files.newDirectoryStream(dir, new FilenameFilter() {
     public boolean accept(File dir, String name) {

     }

     for (Path path : stream) {
         System.out.println(path.getFileName());
     }

    ex.printStackTrace();
}
In this example, the DirectoryStream will filter out any files or directories that begin with a period character as they are being iterated.
