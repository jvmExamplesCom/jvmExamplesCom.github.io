The android.app.ActivityManager is an Android system service that provides a range of tasks related to managing activities, services, and the application processes. It is primarily used by developers for debugging and to control application resource usage.

Basic Methods

The ActivityManager class provides basic methods for starting, stopping and getting details about activities, processes and services. 

          
           
ActivityManager activityManager = (ActivityManager) getSystemService(ACTIVITY_SERVICE);

Stopping an Activity
