javax.persistence.Entity

The 

          
           
javax.persistence.Entity
annotation is used to mark a class as an entity bean. This annotation is applied to the class and the class must be annotated with the 

          
           
@Entity
annotation and have an associated mapping file. It is also used to map the properties of an entity bean to the columns of a database table.

Usage

The 

          
           
@Entity
annotation is generally used in combination with other annotations, such as 

          
           
@Table
or 

          
           
@Column
, to specify the structure of the database table.

Code Examples


          
           
@Entity
public class User {

    @Id
    @GeneratedValue(strategy = GenerationType.AUTO)
    private Long id;

    private String name;

    // getters and setters...
}
The 

          
           
@Entity
annotation is used to mark the class as a JPA entity. The 

          
           
@Id
annotation is used to map the primary key of the entity bean and the 

          
           
@GeneratedValue
annotation is used to configure the generation strategy for the primary key. 

          
           
@Table
The 

          
           
@Table
annotation is used to map the entity bean to a database table. 

          
           
@Table(name = "users")
public class User {

    // ...
}
The 

          
           
@Table
annotation is used to specify the name of the database table which the entity will be mapped to. 

          
           
@Column
The 

          
           
@Column
annotation is used to map the properties of the entity bean to the columns of the database table. 

          
           
@Entity
@Table(name = "users")
public class User {

    @Id
    @GeneratedValue(strategy = GenerationType.AUTO)
    private Long id;

    @Column(name = "name")
    private String name;

    // getters and setters...
}
The 

          
           
@Column
annotation is used to specify the name of the column in the database which the property will be mapped to.

Conclusion

The 

          
           
javax.persistence.Entity
annotation is used to mark a class as an entity bean and to map the properties of an entity bean to the columns of a database table. This annotation is generally used in combination with other annotations, such as 

          
           
@Table
or 

          
           
@Column
, to specify the structure of the database table.
