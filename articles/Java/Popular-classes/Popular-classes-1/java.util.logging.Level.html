java.util.logging.Level

java.util.logging.Level is the java class used for configuring levels for logging messages in a java application. This class contains several static constants for setting and retrieving level settings, as well as convenience methods for setting and retrieving the current log level.

Static Level Constants

  • SEVERE (highest value)
  • WARNING
  • INFO
  • CONFIG
  • FINE
  • FINER
  • FINEST (lowest value)

Methods

The java.util.logging.Level class provides several useful methods for working with log levels:

  • getName(): Returns the name of the Level.
  • intValue(): Returns the integer value associated with this Level.
  • parse(): Parses a String for a Level value.
  • toString(): Returns a String representation of the Level's value.

Examples

Set the log level to WARN


          
           
Logger logger = Logger.getLogger(MyClass.class.getName());
logger.setLevel(Level.WARN);

Get the current log level


          
           
Logger logger = Logger.getLogger(MyClass.class.getName());
Level currLevel = logger.getLevel();
System.out.println("Current log level: "+currLevel);

Parse a String to a Level


          
           
String levelString = "FINE";
Level level = Level.parse(levelString);
System.out.println("Level for string: "+level);
