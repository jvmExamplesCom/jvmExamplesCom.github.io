com.google.gson.JsonObject

The com.google.gson.JsonObject class is a Java library used to parse and create JSON (JavaScript Object Notation) objects. It provides an easy way to convert between Java objects and their JSON representation. It is available as part of the Gson library, which is part of Google’s open-source suite of libraries.

Using com.google.gson.JsonObject

Using com.google.gson.JsonObject is straightforward. To get started, you need to create an instance of the JsonObject class. This can be done in two ways; either passing the JSON string into the constructor or by creating the JsonObject manually.

For example, to create a basic JsonObject from a JSON string:


          
           
String jsonString = "{'name':'John','age':25}";
JsonObject object = new JsonObject(jsonString);
And to create it manually: 

          
           
JsonObject object = new JsonObject();
object.addProperty("name", "John");
object.addProperty("age", 25);

Once you have created an instance of the JsonObject class, you can start to work with it. The JsonObject has several methods for accessing and manipulating the data held in it. These include the following:

  • get(String name): Returns the value associated with the key 'name'.
  • remove(String name): Removes the key/value pair associated with 'name'.
  • addProperty(String name, Object value): Adds a key/value pair with 'name' and 'value'.
  • has(String name): Checks if there is a key/value pair with 'name'.

Example

Here is an example of using the com.google.gson.JsonObject class:


          
           
String jsonString = "{'name':'John','age':25}";

// Create a JsonObject from the JSON string
JsonObject object = new JsonObject(jsonString);

// Get the value of the 'name' key
String name = object.get("name"); // This will return "John"
