java.util.StringTokenizer

The java.util.StringTokenizer class is used to break a string into tokens. It provides a way to tokenize strings in Java. The StringTokenizer class allows us to break a string into individual "tokens" (words, numbers, etc.) that can be easily manipulated. This class can be used to quickly break up a string into its individual components.

Using the StringTokenizer class, we can split a string by either one or more delimiters (e.g. whitespace, punctuation, etc.), or by a specific pattern. This can be useful for extracting data from text files, manipulating strings, and other string-related tasks.

Example 1: Single Delimiter

This example uses the StringTokenizer class to split a string by whitespace or commas.


          
           
String str = "Hello, World! How are you?";
StringTokenizer tokenizer = new StringTokenizer(str, " ,!");
 
while (tokenizer.hasMoreTokens()) {
   String token = tokenizer.nextToken();
   System.out.println(token);
}

Hello
World
How
are
you

This example uses the StringTokenizer class to split a string by multiple delimiters (whitespace, punctuation, etc.).

This example uses the StringTokenizer class to split a string based on a specific pattern.
