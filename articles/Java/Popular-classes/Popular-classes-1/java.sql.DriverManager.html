java.sql.DriverManager

Java's sql package contains the DriverManager class, which is an interface for connecting to a database. It is a static class that provides methods for registering, loading and unloading JDBC drivers. It is used to manage the JDBC drivers that are installed on a system, and also to connect to any given database. In this article, we will take a look at the various methods provided by the DriverManager class, and how to use them.

Registering Drivers

The most important method of the DriverManager class is the registerDriver() method, which can be used to register JDBC drivers with the DriverManager. This allows applications to access the driver without having to load it explicitly using Class.forName(). This method takes an instance of the driver as an argument, and returns an Object of type java.sql.Driver. 

          
           
Driver driver = new oracle.jdbc.driver.OracleDriver();
Object obj = DriverManager.registerDriver(driver);

Connecting to a Database

Once a driver is registered with the DriverManager, applications can use the getConnection() method to connect to a database. This method takes a few arguments such as a URL, a username and a password for the database. The URL specified must follow the format jdbc: : , where is the protocol of the driver, and is the database name. 

              
               
String url = "jdbc:oracle:thin:@localhost:1521:orcl";
String username = "username";
String password = "password";
Connection con = DriverManager.getConnection(url, username, password);

Loading Drivers

The DriverManager class also provides a method called loadDriver(), which can be used to explicitly load a driver. This method takes a String representing the fully qualified class name of the driver class as an argument, and returns an instance of the Driver class. 

              
               
String driverClassName = "oracle.jdbc.driver.OracleDriver";
Driver driver = DriverManager.loadDriver(driverClassName);

Unloading Drivers

Finally, the DriverManager class provides a method called unloadDriver(), which can be used to unload all the loaded drivers from the DriverManager. This method takes no arguments and does not return anything. 

              
               
DriverManager.unloadDriver();
