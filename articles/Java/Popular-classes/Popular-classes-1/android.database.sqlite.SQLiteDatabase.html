Overview of android.database.sqlite.SQLiteDatabase

SQLiteDatabase is an API class from the Android framework in the package android.database.sqlite, that provides methods to manage a SQLite database. The SQLiteDatabase class provides methods such as insert(), query(), update() and delete() that enable you to manipulate database records as if they were normal Java objects. The SQLiteDatabase also provides other utility methods, such as getPath() which returns the path of the database file, and getVersion() which returns the version of the currently active database schema. The class also provides several cursors to manipulate database records, such as RawQueryCursor which executes a raw SQL statement and returns the results in a Cursor, and QueryCursor which is used for selecting records from the database table.

Database Tables

In order to use the SQLiteDatabase class, you need to define the structure of your database tables. Each table represents a data collection, so it must have at least one column and one row. To define a database table, you create a SQLiteOpenHelper subclass and override two methods, onCreate() and onUpgrade(). OnCreate() defines the structure of the table, creating columns and specifying data types. OnUpgrade() is used to modify the structure of the table after it has been created. An example of the SQLiteOpenHelper subclass is given below: 

          
           
public class MyDatabaseHelper extends SQLiteOpenHelper {

    public static final String TABLE_NAME = "mytable";
    public static final String COLUMN_ID = "_id";
    public static final String COLUMN_NAME = "name";
    public static final String COLUMN_AGE = "age";

    private static final String DATABASE_NAME = "mydb.db";
    private static final int DATABASE_VERSION = 1;

    // SQL statement used to create the database
    private static final String DATABASE_CREATE = 
        "create table " + TABLE_NAME + "(" 
        + COLUMN_ID + " integer primary key autoincrement, " 
        + COLUMN_NAME + " text not null, " 
        + COLUMN_AGE + " integer" + ");";

    public MyDatabaseHelper(Context context) {
        super(context, DATABASE_NAME, null, DATABASE_VERSION);
    }

    @Override
    public void onCreate(SQLiteDatabase db) {
        db.execSQL(DATABASE_CREATE);
    }

    @Override
    public void onUpgrade(SQLiteDatabase db, int oldVersion, int newVersion) {
        db.execSQL("DROP TABLE IF EXISTS " + TABLE_NAME);
        onCreate(db);
    }
}
