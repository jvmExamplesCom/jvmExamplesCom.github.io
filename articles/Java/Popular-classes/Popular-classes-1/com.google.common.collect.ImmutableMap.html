Creating an ImmutableMap

An ImmutableMap can be created using the static factory method 


          
           
ImmutableMap.of()
. It requires either two or four parameters (two parameter version is for a single key-value pair). The two parameter version is used when we want to create a map with one element directly without any additional methods. Example: 

          
           
ImmutableMap<String, Integer> map = ImmutableMap.of("key", 5);

The four parameter version is used when we want to create a map with multiple elements. It will take two arrays of type key and value respectively as arguments.

ImmutableMap<String, Integer> map = ImmutableMap.of(keys, values);

Using an ImmutableMap

An ImmutableMap is like a regular Map but with some key differences. First, it cannot be modified once it is created, hence the name “Immutable”. Also, the ImmutableMap is thread-safe. This means that multiple threads can access the same ImmutableMap without any data corruption.
