java.util.concurrent.ScheduledFuture

The java.util.concurrent.ScheduledFuture interface represents a task which is scheduled to be executed periodically or once at a specific time in the future. It is part of the Java Concurrency package, and allows you to schedule tasks to be executed at specific times or intervals. The ScheduledFuture interface extends the Future interface, which enables the tasks to return a value when they are finished.

The ScheduledFuture interface provides several methods to control and manage the scheduled tasks including:

  • cancel()
    – Cancels execution of the task.
  • isCancelled()
    – Checks if the task has been cancelled.
  • getDelay()
    – Gets the remaining delay before the execution of the task.
  • isDone()
    – Checks if the task has been completed.

        ExecutorService executor = Executors.newSingleThreadExecutor();
 
        // Create a Runnable task
        Runnable task = () -> {
            try {
                TimeUnit.SECONDS.sleep(2);
                System.out.println("Running Task!");
            } catch (InterruptedException e) {
                System.err.println("Task interrupted");
            }
