android.app.PendingIntent is a special component that allows an application to control another application. It provides the capability of an application to send an intent to another application, and if the receiving application is currently in a different process, to have that intent delivered to the target application regardless of whether or not the receiving application is runningn.

The Context is the current application, the Intent is the component to be executed by the receiving application, and the request code is an integer that identifies the intent.

Using a PendingIntent

Once the PendingIntent object has been created, it can be used to send an Intent to the receiving application. The send() method of the PendingIntent class takes four parameters: the Context, request code, Intent, and flags which determines how the intent should be handled. 

          
           
pendingIntent.send(context, 0, intent, 0);
This will send the intent to the requesting application, even if the receiving application is in a different process. The flags parameter can be used to specify how the intent should be handled, for example, if the intent should be broadcast or launched in its own task.
