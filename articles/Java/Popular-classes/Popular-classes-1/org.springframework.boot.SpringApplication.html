The 
SpringApplication
class provides a convenient way to bootstrap a Spring application that is started from a Java main method. The primary purpose of the 
SpringApplication
class is to create an application context from one or more configuration sources, and then run the applications from the same application context.

Creating an Application Context

The 
SpringApplication
class can be used to create an application context from sources such as a Java class, XML configuration, or an annotation-driven configuration. The following example uses a Java configuration class to create an application context: 

          
           
public class MyConfiguration {
        @Bean
        public HelloWorld helloWorld() {
            return new HelloWorld();
        }
    }

public class MyApplication {
        public static void main(String[] args) {
            SpringApplication.run(MyConfiguration.class, args);
        }
    }
The 

          
           
SpringApplication.run
method creates an application context from the given configuration class, and then starts the application in the context. It is important to note that a single application context can only be started once.
