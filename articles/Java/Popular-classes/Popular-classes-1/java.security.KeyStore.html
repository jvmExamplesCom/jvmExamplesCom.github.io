Java Security KeyStore

The Java Security KeyStore is a platform independent secure storage facility that provides an encryption mechanism for storing sensitive information like private keys, passwords, and certificates. It can store both symmetric and asymmetric key pairs. Java KeyStore was introduced in Java 1.2 and is available in all versions of Java since then. It supports the storage of a wide range of cryptographic objects, and makes it easy to manage their lifecycle.

Creating a KeyStore

A new Java KeyStore can be created by using the 

          
           
KeyStore.getInstance()
method. This method will return an instance of KeyStore for the specified type. The following example creates a 
KeyStore
instance with the type of 
JKS
: 

          
           
KeyStore ks = KeyStore.getInstance("JKS");

Loading the KeyStore

The 
load()

InputStream

FileInputStream

FileInputStream fis = new FileInputStream("keystore.jks");
ks.load(fis, "password".toCharArray());
fis.close();

Saving the KeyStore

The

FileOutputStream fos = new FileOutputStream("keystore.jks");
ks.store(fos, "password".toCharArray());
fos.close();
