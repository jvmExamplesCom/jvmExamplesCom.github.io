javax.xml.parsers.ParserConfigurationException

The javax.xml.parsers.ParserConfigurationException is a checked exception that indicates an error in the configuration of an XML parser so that it is unable to understand the document being parsed. It is defined by the Java API for XML Processing (JAXP) 1.3 specification.

try {
  // parsing XML document
} catch (ParserConfigurationException e) {
  // handle exception
}
