java.time.Instant

The 
java.time.Instant
class is designed to store an immutable point on the temporal timeline. It represents an instant in time with nanosecond precision.

Creating Instances

Instances of this class can be created in a few ways. The simplest way to create an instance is to use the 

now()
method, which will return the current time according to the system clock: 

          
           
Instant now = Instant.now();
System.out.println(now);  // Output: 2020-10-13T16:49:26.271419400Z

Another way is to create an instance from the number of seconds since the epoch, which can be useful when converting from other time formats like Unix time:


          
           
Instant epochSeconds = Instant.ofEpochSecond(1602683048);
System.out.println(epochSeconds);  // Output: 2020-10-13T15:50:48Z

Finally, an instance can also be created by combining a year, month, day, hour, minute and second:


          
           
Instant specificTime = Instant.of(2020, 10, 13, 16, 49, 26);
System.out.println(specificTime);  // Output: 2020-10-13T16:49:26Z

Comparing Instants

Instants can be compared to determine which is larger or smaller using the Comparison methods including 


          
           
isBefore(), isAfter()
and 
isEqual()
. For example: 

          
           
Instant now = Instant.now();
Instant fiveMinutesFromNow = now.plus(Duration.ofMinutes(5));
System.out.println(fiveMinutesFromNow.isAfter(now));  // Output: true

Manipulating Instants

Instants can be manipulated by adding or subtracting a 

Duration
, which is a length of time represented by a number of seconds and nanoseconds. For example, to add five minutes to an instant: 

          
           
Instant now = Instant.now();
Instant fiveMinutesFromNow = now.plus(Duration.ofMinutes(5));
System.out.println(fiveMinutesFromNow);  // Output: 2020-10-13T16:54:40.071420700Z

Formatting Instants

Instants can be formatted using the 
format()
method, which takes an 
DateTimeFormatter
object as an argument. For example, to format an instant as a human-readable string: 

          
           
Instant now = Instant.now();
String formattedNow = now.format(DateTimeFormatter.ofLocalizedDateTime(FormatStyle.LONG));
System.out.println(formattedNow);  // Output: October 13, 2020 4:49:26 PM PDT
