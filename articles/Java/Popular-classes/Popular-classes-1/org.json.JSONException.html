org.json.JSONException

JSON stands for JavaScript Object Notation and is a language-independent data-interchange format that allows you to represent data structures in a standard way. JSONException is an exception class when something goes wrong while parsing or processing JSON content. It is thrown when an error occurs while accessing the token (e.g Invalid character, missing key).

Creating an Exception

JSONException can be created using several different constructors. Here are some of the most commonly used ones. 

          
           
// NumberFormatException constructor
try {
    Integer.parseInt("abc");
} catch (NumberFormatException e) {
    System.out.println("Error: " + e);
}
This will output "Error: java.lang.NumberFormatException: For input string: "abc"" 

          
           
// JSONException constructor
try {
    JSONObject jsonObject = new JSONObject("{\"name\":\"John\"}");
} catch (JSONException e) {
    System.out.println("Error: " + e);
}
This will output "Error: org.json.JSONException: A JSONObject text must begin with '{' at 1 [character 2 line 1]"

Methods

JSONException provides several useful methods. These include: 

          
           
getMessage()
This method returns a description of the exception which is generated by calling the toString() method. 

          
           
getCause()
Returns the cause of the exception or null if the cause is unknown. 

          
           
getStackTrace()
Returns an array of stack trace elements which can be useful for debugging purposes.

Example

Here's a basic example of how to use the JSONException class: 

          
           
try {
    JSONObject jsonObject = new JSONObject("{ \"name\": \"John\" }");
    System.out.println(jsonObject);
} catch (JSONException e) {
    System.out.println("Error: " + e);
}
This will output "{ "name": "John" }".
