java.android.support.v7.app.ActionBar

The Java android.support.v7.app.ActionBar provides developers with an API for creating custom action bars which can be used in all activities and fragments. Action bars allow users to quickly access important information as well as provide an aesthetic look to the application's UI. This library allows developers to create action bars that support multiple configurations such as title, navigation mode, actions, tab navigation, and custom views. Additionally, developers are able to easily apply custom styling to the action bar using styles and themes.

Creating an ActionBar

To create an action bar, the developer must first add the "appcompat-v7" library dependency to their project: 

          
           
dependencies {
    implementation 'com.android.support:appcompat-v7:28.0.0'
}
Once this has been added, the application can call the Activity.getSupportActionBar() method to get an instance of the ActionBar. This method can then be used to customize the action bar.

Customizing the Action Bar

The ActionBar class has numerous methods for customizing it, including setting the title, navigation mode, actions, tab navigation, and custom views.

Setting the Title

To set the title of the action bar, the setTitle() method should be called with the desired text string. 

          
           
actionBar.setTitle("My Action Bar");

Setting the Navigation Mode

Action bars can be set to either standard or tab navigation mode by calling the setNavigationMode() method: 

          
           
actionBar.setNavigationMode(ActionBar.NAVIGATION_MODE_STANDARD);

Adding Actions

Actions can be added to the action bar by calling the addAction() method: 

          
           
actionBar.addAction(action);
Where action is an object of type android.support.v7.app.ActionBar.Action.

Setting Tab Navigation

Action bars can be configured to use tab navigation with the setTabNavigation() method: 

          
           
actionBar.setTabNavigation(tabs);
Where tabs is an array of objects of type Tab.

Adding Custom Views

Custom views can be added to the action bar by calling the addCustomView() method: 

          
           
actionBar.addCustomView(view);
Where view is an object of type View.

Styling the Action Bar

The action bar can be styled by applying a style or theme to the activity or fragment. To apply a style to the action bar, use the following syntax: 

          
           
<style name="MyActionBarStyle" parent="@style/Widget.AppCompat.ActionBar">
    <item name="background">@color/myBackgroundColor</item>
    <item name="titleTextAppearance">@style/MyTitleTextAppearance</item>
</style>
To apply this style to the action bar, use the following syntax: 

          
           
<activity android:name=".MyActivity"
    android:theme="@style/MyActionBarStyle"/>
Alternatively, a theme can be applied to the action bar with the following syntax: 

          
           
<activity android:name=".MyActivity"
    android:theme="@style/MyActionBarTheme"/>

Conclusion

The java.android.support.v7.app.ActionBar provides developers with an API for creating custom action bars for their applications. It allows developers to easily customize the action bar, and apply custom styling to it using styles and themes.
