Introduction to android.util.Log

Android.util.Log is an Android logging class used to debug applications and log important information. It is used to log messages, errors and other information during the development phase of an application.

The android.util.Log class provides a log method for each type of message, such as verbose, debug, warning and error. You can also specify custom tags for your log messages which make it easier to filter logs by type or by specific keywords.

Creating Logs

The simplest way to create a log message is using the Log.v() method:


          
           
Log.v("MyTag", "This is my message");

You can also specify additional parameters for more detailed logging:


          
           
Log.v("MyTag", "This is my message", exception);
Where exception is an Exception object.

The following examples illustrate how to use the different log methods available in the android.util.Log class:


          
           
// verbose log
Log.v("MyTag", "This is my message");

// debug log
Log.d("MyTag", "This is my message");

// warning log
Log.w("MyTag", "This is my message");

// error log
Log.e("MyTag", "This is my message");

Log Formatting

The android.util.Log class supports two types of log formatting:

  • String formatting: this includes placeholders and escape characters to format the log message.
  • logger: this is a collection of pre-defined log formats that are used to log messages in a more structured way.

String formatting allows you to add placeholders, quotes and escape characters to log messages as needed. For example:


          
           
// format a log message with placeholders
Log.v("MyTag", "Message with %s and %d", "string", 5);

// format a log message with quotes
Log.v("MyTag", "Message with \"quotes\"");

// format a log message with escape character
Log.v("MyTag", "Message with \"new line
character\"");

The logger allows you to define a custom log format that can be used to format log messages in a consistent manner. For example:


          
           
LogLogger logger = new LogLogger(new SimpleDateFormat("yyyy-MM-dd HH:mm:ss"));

// format a log message with the logger
Log.v("MyTag", logger.format("This is a log message"));

Reading Logs

The android.util.Log class also provides a method for reading log messages from a file or from the console. The method takes a log tag as a parameter and returns a List of LogRecords:


          
           
List<LogRecord> logRecords = Log.getLogRecords("MyTag");

Each LogRecord contains information about the log message such as the timestamp, message, level and tag. For example:


          
           
for (LogRecord logRecord : logRecords) {
  System.out.println(logRecord.getTimestamp() + " - " + logRecord.getMessage());
}
