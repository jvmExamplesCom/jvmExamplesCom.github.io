Introduction to javax.servlet.http.HttpServlet

javax.servlet.http.HttpServlet is a servlet class provided in the Java Servlet API which enables the user to create web applications. It serves as a base class for HTTP servlets, implementing protocol-independent methods that allow subclasses to concentrate on the details of the particular protocol they are designed to support.
