java.lang.reflect.ParameterizedType

ParameterizedType is an interface of java.lang.reflect package that represents a parameterized type. It is used to retrieve the generic type information of a type variable or generic type.

Example 1: Class Example

In this example, we will see how to use ParameterizedType to get generic type information of a class. 

          
           
public class MyClass<T1, T2> { }

Class<MyClass> cls = MyClass.class;
ParameterizedType pType = (ParameterizedType) cls.getGenericSuperclass();

Type[] typeArguments = pType.getActualTypeArguments();
for(Type type : typeArguments) {
    System.out.println("Type Arguments: "+type);
}
Output: 

          
           
Type Arguments: T1
Type Arguments: T2

public class MyClass<T1, T2> {
    public void myMethod(T1 t1, T2 t2) { }
}

Method method = MyClass.class.getDeclaredMethod("myMethod", Object.class, Object.class);
