Java.util.function.Consumer

The 

          
           
java.util.function.Consumer
interface is a functional interface that is part of Java 8's new collections framework. It can be used to perform operations on objects stored in a collection, such as manipulating the data or printing out results. The interface has a single method that accepts a parameter and performs an operation on it.

Syntax


          
           
@FunctionalInterface
public interface Consumer<T> {
    void accept(T t);
}
The 
accept()
method takes a single argument and performs an operation on it.

Example

The following example uses a 
Consumer
to print out all the elements of a 
List
: 

          
           
import java.util.Arrays;
import java.util.List;
import java.util.function.Consumer;

public class Example {

    public static void main(String[] args) {
        List<Integer> list = Arrays.asList(1, 2, 3, 4, 5);

        // define a Consumer
        Consumer<Integer> consumer = (x) -> System.out.println(x);

        // iterate over the list and accept each element
        list.forEach(consumer);
    }

}
