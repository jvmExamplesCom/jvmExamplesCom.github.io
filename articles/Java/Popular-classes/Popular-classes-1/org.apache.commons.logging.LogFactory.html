org.apache.commons.logging.LogFactory

The 

          
           
org.apache.commons.logging.LogFactory
class is a factory for producing Log instances. It provides support for pluggable logging modules and allows applications to be configured with their logging framework of choice. This class is used to create Log objects and the logging environment is configured in the LogFactory.properties file.

Example 1: Creating a Log Object with LogFactory


          
           
Log log = LogFactory.getLog(MyClass.class);

This example code creates a Log object which can be used to log messages, warnings, errors and other types of logging events. The 

MyClass
argument should be the class object associated with the logger you want to use.

log.info("This is an info message");
log.warn("This is a warning message");
log.error("This is an error message");

This example code shows how to log messages with the LogFactory. The 

log.info()
, 
log.warn()

log.error()

log.error()
