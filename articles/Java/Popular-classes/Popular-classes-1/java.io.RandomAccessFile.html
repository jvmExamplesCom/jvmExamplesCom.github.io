java.io.RandomAccessFile

java.io.RandomAccessFile is a class used for reading and writing data on random access file streams. It supports random access of data by allowing the user to seek and read/write to a specific point within the file.

The java.io.RandomAccessFile provides two types of constructors, one for read access and one for write access. When creating an object for read operations, the constructor requires a File object as an argument. For write operations, the second argument indicates whether to open the file for writing (true) or not (false).

Code Examples


          
           
// Create a RandomAccessFile object
RandomAccessFile raf = new RandomAccessFile("example.txt", "rw");

// Set the file pointer to the beginning
raf.seek(0);

// Write some text to the file
raf.writeBytes("Some text to be written here");

// Read the first 10 bytes
byte[] bytes = new byte[10];
raf.read(bytes);
for (byte b : bytes) {
   System.out.print((char) b);
}
System.out.println();

// Reset the position of the file pointer
raf.seek(0);

// Read the first 10 bytes again
raf.read(bytes);
for (byte b : bytes) {
   System.out.print((char) b);
}
System.out.println();

The seek() method sets the file pointer to the specified position. The position is specified in bytes from the beginning of the file. To read data from the file, the read() method is used and requires a byte array as parameter which specifies the number of bytes to be read.

To write data to the RandomAccessFile the writeBytes() method can be used. This method will overwrite any existing data or append the data to the end of the file if the file pointer is at the end of the file.

Closing the RandomAccessFile Object

When using a RandomAccessFile object it is important to close the file after use. This can be done by calling the close() method which will release any resources associated with the file.


          
           
raf.close();
