Java NIO File Path

The java.nio.file.Path class is part of the Java NIO (New I/O) API for working with files and directories in Java. The Path class provides many useful methods for querying and manipulating a path, including getting the parent directory, getting a file name, determining if a file exists, etc.

Creating an Instance

To create an instance of the Path class, we can use the static Paths.get() method, which takes a single string argument. For example:


          
           
Path path = Paths.get("my_directory");

This will create a Path instance, pointing to a directory or file named my_directory. This directory or file must, of course, exist on disk.

Retrieving Information About a Path

The Path class provides several methods for retrieving information about a path. These include:

  • getFileName()
    : returns the file name of the path
  • getParent()
    : returns the parent directory of the path
  • getRoot()
    : returns the root component of the path
  • isAbsolute()
    : determines if the path is absolute
  • toAbsolutePath()
    : returns an absolute version of the path
  • toUri()
    : returns a java.net.URI object representing the path
Let's look at an example: 

          
           
Path path = Paths.get("my_directory");
String fileName = path.getFileName();
Path parentPath = path.getParent();
Path rootPath = path.getRoot();
boolean isAbsolute = path.isAbsolute();
Path absolutePath = path.toAbsolutePath();

In this code snippet, we first create a Path instance, pointing to the my_directory. Then we retrieve the file name, parent directory, root component, whether the path is absolute, and finally an absolute version of the path.

Checking if a Path Exists

The Path class also provides a method for checking if a path actually exists on disk. The method is called 

exists()
and it takes no arguments. It returns true if the path exists, and false otherwise. 

          
           
Path path = Paths.get("my_directory");
if (path.exists()) {
    System.out.println("Path exists!");
} else {
    System.out.println("Path does not exist.");
}

In this code snippet, we check if the path my_directory exists. If it does, then the code prints out "Path exists!". Otherwise, it prints "Path does not exist.".

Creating Directories or Files

The Path class also provides methods for creating directories or files. To create a directory, we can call the 

createDirectory()
method, which takes no arguments and returns a Path instance corresponding to the newly created directory. 

          
           
Path directory = Paths.get("new_directory");
directory.createDirectory();

This code creates a new directory at the given path.

To create a file, we can call the 
createFile()
method, which takes no arguments and returns a Path instance corresponding to the newly created file. 

          
           
Path file = Paths.get("new_file.txt");
file.createFile();
This code creates a new file at the given path.

Deleting Directories or Files

The Path class also provides methods for deleting directories or files. To delete a directory, we can call the 

deleteDirectory()
method, which takes no arguments and returns a boolean value indicating whether the directory was successfully deleted. 

          
           
Path directory = Paths.get("new_directory");
boolean wasDeleted = directory.deleteDirectory();

This code deletes the directory at the given path.

To delete a file, we can call the 
deleteFile()
method, which takes no arguments and returns a boolean value indicating whether the file was successfully deleted. 

          
           
Path file = Paths.get("new_file.txt");
boolean wasDeleted = file.deleteFile();
This code deletes the file at the given path.
