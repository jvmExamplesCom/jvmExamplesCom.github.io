java.io.StringReader

java.io.StringReader is a Java class that allows for the reading of a string as an input stream. It is part of the java.io package and can be used to read data from a String, allowing for simple conversion between different types of String readers and writers. The StringReader is useful for writing strings to files and other such tasks.

Converting a String to a StringReader

To convert a String to a StringReader, you need to first create a new StringReader object. You can do this by passing a String to the constructor method of the StringReader.


          
           
StringReader stringReader = new StringReader("This is a String");
The StringReader is now ready to use.

Reading a String with StringReader

Once the StringReader has been created, you can read data from the String using the read() method of the StringReader. This method reads one character at a time from the String and returns an integer value. A value of -1 is returned when the end of the String has been reached.


          
           
int c;
while((c = stringReader.read()) != -1) {
    System.out.print((char) c);
}
// Output: This is a String

Closing a StringReader

When you are done reading the String, you should close the StringReader by calling its close() method. This will free up any resources held by the StringReader.


          
           
stringReader.close();
