java.nio.charset.StandardCharsets

Java.nio.charset.StandardCharsets provides the basic set of character encodings which can be used to represent characters in a byte sequence in the Java platform. The StandardCharsets class contains four public static constants that represent the supported character encodings: UTF-8, ISO-8859-1, US-ASCII, and ISO-8859-15.

Example 1: Convert String To Byte Array Using StandardCharsets


          
           
public static void main(String[] args) {
    String text = "Hello World";
    byte[] bytes = text.getBytes(StandardCharsets.UTF_8);
    System.out.println(Arrays.toString(bytes));
}

Output: [72, 101, 108, 108, 111, 32, 87, 111, 114, 108, 100]

Example 2: Convert Byte Array To String Using StandardCharsets


          
           
public static void main(String[] args) {
    byte[] bytes = new byte[] { 72, 101, 108, 108, 111, 32, 87, 111, 114, 108, 100 };
    String text = new String(bytes, StandardCharsets.UFT_8);
    System.out.println(text);
}
Output: Hello World

Example 3: Default Charset With StandardCharsets


          
           
public static void main(String[] args) {
    Charset charset = StandardCharsets.defaultCharset();
    System.out.println(charset.name());
}
Output: UTF-8

Example 4: Charsets With StandardCharsets


          
           
public static void main(String[] args) {
    Charset charset = StandardCharsets.ISO_8859_1;
    System.out.println(charset.name());
}
Output: ISO-8859-1
