Android.Widget.CompoundButton

Android.Widget.CompoundButton is an abstract class that provides a foundation for a group of button controls with related behavior. It implements the Checkable interface and contains methods such as getButtonDrawable(), setButtonDrawable() and setOnCheckedChangeListener(). It provides a common API for working with buttons such as radio buttons, checkboxes, and toggle buttons.

Example 1 - Set Button Drawable

In this example, we will create a CheckBox and set a drawable to its button. 

          
           
public class MainActivity extends AppCompatActivity {

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);

        Drawable myDrawable = getResources().getDrawable(R.drawable.ic_check_blue);
        CheckBox checkBox = findViewById(R.id.checkbox);
        checkBox.setButtonDrawable(myDrawable);
    }
}

Example 2 - Set On Checked Change Listener

In this example, we will create a ToggleButton and set an OnCheckedChangeListener to it. 

          
           
public class MainActivity extends AppCompatActivity {

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);

        ToggleButton toggleButton = findViewById(R.id.toggleButton);
        toggleButton.setOnCheckedChangeListener(new CompoundButton.OnCheckedChangeListener() {
            @Override
            public void onCheckedChanged(CompoundButton buttonView, boolean isChecked) {
                if (isChecked) {
                    Log.d("TAG", "ToggleButton is checked");
                } else {
                    Log.d("TAG", "ToggleButton is unchecked");
                }
            }
        });
    }
}
