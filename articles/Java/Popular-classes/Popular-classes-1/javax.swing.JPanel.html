javax.swing.JPanel

A JPanel is an effective container class in Java that can group various components together into a single pane. It acts as a base class for other components, such as labels, text fields, or buttons, among others. It provides a flexible layout that can be adjusted and modified to suit the particular needs of the application.

Creating a JPanel

A JPanel can be created with the following code: 

          
           
JPanel panel = new JPanel();
A constructor can also be used to initialize the JPanel with a specific layout. For example, we can assign a 

          
           
FlowLayout
to the panel with the following code: 

          
           
JPanel panel = new JPanel(new FlowLayout());

Adding Components to a JPanel

Components can be added to a JPanel using the 

          
           
add()
method. This method takes a 

          
           
Component
object as an argument. An example of adding a button to a panel is shown below: 

          
           
JButton button = new JButton("Click Me!");
panel.add(button);

Setting a Layout Manager

A layout manager defines how the components on a JPanel are arranged and displayed. By default, a JPanel uses a 

          
           
FlowLayout
, which arranges the components in a row from left to right, top to bottom. However, a different layout manager can be set if desired. For example, the following code sets a 

          
           
BorderLayout
on the JPanel: 

          
           
panel.setLayout(new BorderLayout());
To learn more about layout managers, refer to the documentation.

Example

The following example creates a JPanel with four buttons arranged in a centered grid. 

          
           
import javax.swing.JFrame;
import javax.swing.JPanel;
import javax.swing.JButton;
import java.awt.GridLayout;

public class Main {
  public static void main(String[] args) {
    JFrame frame = new JFrame();
    frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);

    JPanel panel = new JPanel(new GridLayout(2, 2));

    JButton button1 = new JButton("Button 1");
    JButton button2 = new JButton("Button 2");
    JButton button3 = new JButton("Button 3");
    JButton button4 = new JButton("Button 4");

    panel.add(button1);
    panel.add(button2);
    panel.add(button3);
    panel.add(button4);

    frame.add(panel);
    frame.pack();
    frame.setVisible(true);
  }
}
