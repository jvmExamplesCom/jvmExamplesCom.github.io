Java.io.Writer

The Java.io.Writer class is a base class used to write character streams in the Java programming language. It is the abstract superclass of all classes provided by the java.io package for writing character streams. It provides methods for writing characters, strings and arrays, and for flushing the output stream. The subclasses of the Writer class can be used to write data to a file, the console, and any I/O stream.

Functions of Java.io.Writer Class

The functions of the Java.io.Writer class are as follows:

  • void write(int c)
    : This method writes the specified character to the writer.
  • 
            
             
void write(char[] cbuf)
    : This method writes the specified array of characters to the writer.
  • 
            
             
void write(String str)
    : This method writes a string to the writer.
  • void flush()
    : This method flushes the underlying stream.
  • void close()
    : This method closes the writer.

Example - Writing a String to the Output Stream

This example demonstrates how to write a string to the output stream using the 

write()
method of the Java.io.Writer class. 

          
           
import java.io.Writer;
import java.io.StringWriter;

public class Test {
  public static void main(String[] args) {
    try {
      // Create a new writer
      Writer out = new StringWriter();

      // Write a string to the output stream
      out.write("Hello World!");

      // Flush the output stream
      out.flush();

      // Print the result
      System.out.println(out.toString());
    } catch (Exception ex) {
      ex.printStackTrace();
    }
  }
}
