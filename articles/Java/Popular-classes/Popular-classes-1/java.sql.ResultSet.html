ResultSet is an interface in the Java DataBase Connectivity (JDBC) API. It provides access to the queried data from a database table, and also acts as an iterator object allowing the user to traverse through the data records returned from the query.

Creating a ResultSet Object

A new instance of a ResultSet can be created by passing an SQL query to a statement object and calling its executeQuery() method. The statement object can be built using the Connection class.
