java.util.SortedMap

Java.util.SortedMap is an interface that extends Map and defines a total ordering on its elements. It is used to store key-value pairs in a sorted order, usually based on the natural ordering of the keys. The SortedMap interface provides access and retrieval operations based on the element’s natural ordering, and additionally allows a range view of the entries.

Methods of java.util.SortedMap

The java.util.SortedMap extends the Map interface and contains several methods for manipulation and retrieval of data from the Map. Some of the major methods are:

  • comparator()
    : Returns the comparator used to order the elements in this map, or null if this map uses its elements' natural ordering.
  • firstKey()
    : Returns the first (lowest) key currently in the map.
  • headMap(K toKey)
    : Returns a view of the portion of this map whose keys are strictly less than the toKey argument.
  • lastKey()
    : Returns the last (highest) key currently in the map.

subMap(K fromKey, K toKey)
    : Returns a view of the portion of this map whose keys range from fromKey, inclusive, to toKey, exclusive.
  • tailMap(K fromKey)
    : Returns a view of the portion of this map whose keys are greater than or equal to fromKey.
