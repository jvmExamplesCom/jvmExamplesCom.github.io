Java.awt.event.ActionListener

Java.awt.event.ActionListener is an interface used in the data model of Java to permit the delegation of user interaction with a graphical user interface (GUI) element to a separate object. The ActionListener interface is typically used as a part of event-listening mechanisms and requires a class of type ActionListener to be registered with a graphical component prior to the component triggering an action event. This registration process generally takes place in a component constructor.

Example Code

The following example creates a frame that contains a JButton component and a JTextField component. When the button is clicked, the text in the text field is displayed in a dialog box:


          
           
import java.awt.event.ActionListener; 
import javax.swing.JButton; 
import javax.swing.JFrame; 
import javax.swing.JTextField; 
import javax.swing.JOptionPane;

public class ButtonEventListener { 
  public static void main(String[] args) { 
  
    // Create a new JFrame object 
    JFrame frame = new JFrame("Button Event Listener"); 
    frame.setSize(400, 300); 
    
    // Create a JButton component 
    JButton button = new JButton("Press Me!"); 
    frame.add(button); 
    
    // Create a JTextField component 
    JTextField textField = new JTextField(); 
    frame.add(textField); 
    
    // Register an anonymous ActionListener implementation to handle the button click event 
    button.addActionListener(new ActionListener() {
      public void actionPerformed(ActionEvent e) { 
        // Display the text from the text field in the dialog box when the button is clicked 
        JOptionPane.showMessageDialog(null, textField.getText()); 
      } 
    }); 
 
    frame.setVisible(true); 
  } 
}

When the code is executed, the button will display a text dialog box when clicked. This is because when the button is clicked, it triggers an ActionEvent, which is then caught by ActionListener, which then calls the actionPerformed() method, which displays the text in the text field in the dialog box.
