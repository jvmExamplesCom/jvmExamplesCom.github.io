com.google.common.base.Joiner

The Joiner class in the com.google.common.base package of Java is a powerful tool for joining strings together. It allows developers to create, manipulate and control string separators, such as commas, quotation marks or other symbols. This can be used to join elements of an array into a single string, or to chain multiple strings together with a specified delimiter. This article will explore the Joiner class and its various methods.

Methods Overview

The Joiner class is an abstract class, meaning it contains methods that must be overridden by a subclass before it can be used. The main methods of the Joiner abstract class are:

  • 
            
             
on(CharSequence delimiter)
    : Sets the delimiter for the Joiner
  • 
            
             
join(CharSequence... elements)
    : Joins the provided elements into a single string using the defined delimiter
  • skipNulls()
    : Skips all null elements in the input array
  • 
            
             
useForNull(String nullText)
    : Substitutes null values with the provided text

Basic Usage

To use the Joiner class, create a new instance and set a delimiter. The following example will create a Joiner with a comma as a delimiter:


          
           
Joiner joiner = Joiner.on(",");

Now that the Joiner is configured, you can use the join method to turn the provided string array into a single string:


          
           
String result = joiner.join("John", "Jane", "Jake");
The resulting string will look like this: 
"John,Jane,Jake"
.

Skipping Null Values

By default, Joiner will not skip any elements, meaning if there are any null values in the input array, they will be included in the output string. You can use the 

skipNulls()
method to prevent this from happening. The following example creates a Joiner that skips null values: 

          
           
Joiner joiner = Joiner.on(",").skipNulls();

Replacing Null Values

In some cases, you may want to replace null values with some other text. You can do this using the 


          
           
useForNull(String nullText)
method. The following example creates a Joiner that replaces null values with the word “Unknown”: 

          
           
Joiner joiner = Joiner.on(",").useForNull("Unknown");

The Joiner class in the com.google.common.base package of Java provides a powerful way to join strings together. This makes it an essential tool for working with strings in Java.
