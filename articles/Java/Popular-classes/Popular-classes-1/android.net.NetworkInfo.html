Android.net.NetworkInfo

Introduction

Android.net.NetworkInfo is a Java class that allows applications to access information about the networks that are connected to the device. The class provides information on what type of network is available (Wi-Fi, cell/mobile data, etc.), whether or not the network is currently connected, and other details about the connection. This class is useful for developing applications that require network connectivity.

Usage

To use this class, you first need to get an instance of the ConnectivityManager class. This is used to create instances of NetworkInfo. 

          
           
ConnectivityManager cm = (ConnectivityManager)context.getSystemService(Context.CONNECTIVITY_SERVICE);
Once you have an instance of ConnectivityManager, you can use it to get an instance of NetworkInfo. 

          
           
NetworkInfo ni = cm.getActiveNetworkInfo();

boolean isConnected = ni.isConnected(); // whether or not the network is connected

The type is one of the constants from ConnectivityManager, such as ConnectivityManager.TYPE_MOBILE or ConnectivityManager.TYPE_WIFI.
