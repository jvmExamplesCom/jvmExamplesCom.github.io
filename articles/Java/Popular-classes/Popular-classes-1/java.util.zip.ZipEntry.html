java.util.zip.ZipEntry

The java.util.zip.ZipEntry utility class provides an API for reading and writing Zip files. It allows developers to store compressed data in a file-like structure, which can be accessed and manipulated in a standard way.

Creating a ZipEntry Object

A new ZipEntry object can be instantiated by calling the constructor.


          
           
ZipEntry entry = new ZipEntry();

Alternatively, you can use one of the overloaded constructors which accepts a path argument.


          
           
ZipEntry entry = new ZipEntry("/path/to/file");

Accessing ZipEntry Properties

You can access various properties of a ZipEntry instance using the following methods:

  • getName()
    – returns the name of the entry
  • getSize()
    – returns the uncompressed size of the entry
  • getCompressedSize()
    – returns the compressed size of the entry
  • getCrc()
    – returns the CRC value of the entry
  • getComment()
    – returns the comment for the entry
  • isDirectory()
    – checks if the entry is a directory

Example Usage

Let's assume we have a 
ZipFile
instance called 
zipFile
. We can then list all the entries in the zip file and print out various information about them. 

          
           
Enumeration<? extends ZipEntry> entries = zipFile.entries();

while (entries.hasMoreElements()) {
    ZipEntry entry = entries.nextElement();

    System.out.println("Name: " + entry.getName());
    System.out.println("Size: " + entry.getSize());
    System.out.println("Compressed Size: " + entry.getCompressedSize());
    System.out.println("CRC: " + entry.getCrc());
    System.out.println("Comment: " + entry.getComment());
    System.out.println("Directory: " + entry.isDirectory());
}
