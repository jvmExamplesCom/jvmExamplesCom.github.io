Iterable Iterables.concat(Iterable ... iterables)

Iterables.concat() creates an iterable from two or more existing iterables. The newly created iterable will contain all of the elements from the iterables that were provided as input, in the same order.

  List<Integer> list2 = Arrays.asList(5, 6);
  Iterable<Integer> concat = Iterables.concat(list1, list2);
  
  for (Integer elem : concat) {
      System.out.println(elem);
  }

  // 3
  // 4
  // 5
  // 6
