Java.net.InetAddress

Java.net.InetAddress is a class in the java.net package that represents an Internet Protocol (IP) address. It provides methods to access information, such as host name, IP address, and subnet mask. The InetAddress class also provides methods to obtain a host name for a given IP address or reverse DNS lookup of an IP address. This is useful for implementing network communication applications.

Methods

The InetAddress class provides various methods which can be used to get information about an IP address, such as host name, IP address, and subnet mask. We'll look at some of the most commonly used methods here:

  • getLocalHost()
    - This method returns the InetAddress object of the local host.
  • 
            
             
getByName(String hostName)
    - This method returns the InetAddress object associated with the specified host name. If no host name is specified then it will return the local host.
  • 
            
             
getByAddress(byte[] ipAddress)
    - This method returns the InetAddress object associated with the specified IP address. The IP address must be in the form of a byte array.
  • getHostName()
    - This method returns the host name of the InetAddress object.
  • getHostAddress()
    - This method returns the IP address of the InetAddress object in the form of a String.

Example Usage

We'll now look at an example of how to use the InetAddress class to retrieve information about a host. We'll use the following code:


          
           
InetAddress addr = InetAddress.getByName("example.com");
String hostname = addr.getHostName();
String ipAddress = addr.getHostAddress();
