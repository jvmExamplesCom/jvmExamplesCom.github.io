Java.io.DataOutputStream is a class used to write primitive data types to an output stream, such as a file or network connection. It allows applications to write accurately formatted data with a variety of options, including buffering, format detection, and endianness. This class can be used in conjunction with other classes such as DataInputStream for reading data from the output stream.

    out.writeUTF("This is a test string");
    out.flush();

}
