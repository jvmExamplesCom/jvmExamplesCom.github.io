Overview of Java.util.SortedSet in Java

java.util.SortedSet

SortedSet

What is SortedSet?


          
           
SortedSet
is an interface that extends 

          
           
java.util.Set
and provides access to sorted data. It provides insertion order along with nearly constant-time performance for the basic operations like add, remove, and contains. The data stored in the set is sorted according to its natural ordering. If the elements are not Comparable, then a Comparator can be provided at the time of creation to specify the sorting order.

Creating an instance of SortedSet

The 

          
           
SortedSet
interface does not provide any direct implementation, and to create a SortedSet, we need to use some implementation like TreeSet or ConcurrentSkipListSet. Let's look at an example of creating a 

          
           
TreeSet
and adding elements to it: 

          
           
SortedSet<Integer> numbers = new TreeSet<>();
numbers.add(2);
numbers.add(3);
numbers.add(1);

TreeSet

: 

          
           
SortedSet<Integer> numbers = new TreeSet<>();
