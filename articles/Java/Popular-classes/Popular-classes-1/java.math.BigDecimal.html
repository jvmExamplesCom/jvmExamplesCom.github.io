Java Math BigDecimal

Java math offers a wide variety of powerful mathematical and statistical functions to explore more complex numbers. Among these is the Java.math.BigDecimal class, an immutable, arbitrary-precision signed decimal number. It is used when greater precision and accuracy is required than what binary floating points (such as the float and double classes) can offer.

Creating a BigDecimal Object

An instance of a BigDecimal object can be created in several ways. A BigDecimal object can be constructed with a string representation of a number, for example: 

          
           
BigDecimal bd = new BigDecimal("123.45");
Alternatively, a BigDecimal object can be created with a double or float variable with the use of the valueOf() method. 

          
           
BigDecimal bd = BigDecimal.valueOf(123.45);
It is also possible to create a BigDecimal object with the use of a long or int variable, again using the valueOf() method: 

          
           
BigDecimal bd = BigDecimal.valueOf(123);
Lastly, a BigDecimal object can be created directly with its own constructor, which takes two parameters; an unscaled value and the scale of the number. The unscaled value is a long type and the scale is an integer value. 

          
           
BigDecimal bd = new BigDecimal(123, 2); // 123.00

The BigDecimal class provides many useful methods and functions to manipulate and transform a BigDecimal object. 

          
           
add()

- scales a given BigDecimal object by an integer power of 10

- returns an integer value exactly without rounding 

          
           
doubleValue()
- returns a double value, rounding if necessary

- compares two BigDecimal objects and returns 0 if they are equal, 1 if the first is greater than the second, and -1 if the second is greater than the first 

          
           
equals()

- returns true if the given BigDecimal is equal to zero, false otherwise
