Java Security MessageDigest

The Java Security MessageDigest classes allow for creating a digital signature for a message, verifying the digital signature of a message, and creating a hash value for a message. MessageDigest is an abstract class that provides the functionality of a cryptographic hash function. A MessageDigest object starts out initialized. The data is processed through it using the update() method. At any point, calling the digest() method will reset the digest to its initialized state where it can process new data.

Overview of MessageDigest

A MessageDigest object is an instance of a cryptographic hash function that takes in data and produces a fixed size digest (also known as a message digest or fingerprint) that uniquely identifies the data. This digest is used to verify the integrity of the data, meaning that it has not been changed or tampered with. The MessageDigest class is part of the Java Security package and includes several algorithms for hashing data. These algorithms include SHA-1, SHA-256, SHA-384, SHA-512, MD5, and more. Each algorithm produces a different size digest and are used for different levels of security.

Creating a MessageDigest Instance

Before a MessageDigest object can be used, it must be initialized with an algorithm via the getInstance() static method from the MessageDigest class. The algorithm is specified as a String parameter, which is case-insensitive. For example, to initialize a SHA-256 message digest, we would use: 

          
           
MessageDigest sha256 = MessageDigest.getInstance("SHA-256");

Updating the MessageDigest Object

Once a MessageDigest object is initialized, the data to be hashed can be passed to it through the update() method. The update() method accepts a byte array, a byte, a ByteBuffer, and an InputStream as parameters. All of the data passed in through these methods are added to the data that is already in the MessageDigest object, and the data is then hashed into a digest. For example, to add a single byte to a MessageDigest object, you could use the following code:

Producing the Digest

Once all of the data has been added to the MessageDigest object, the digest() method can be called to produce the digest. This method will reset the MessageDigest object so that it can process new data. The digest() method returns a byte array containing the digest. For example, to produce the digest from our MessageDigest object, we could use the following code: 

          
           
byte[] digest = sha256.digest();

This concludes our overview of the Java Security MessageDigest classes.
