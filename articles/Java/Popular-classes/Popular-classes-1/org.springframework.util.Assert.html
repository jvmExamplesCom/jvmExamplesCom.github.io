org.springframework.util.Assert

org.springframework.util.Assert is a utility class that provides convenient methods for asserting conditions in code such as preconditions, postconditions, and invariants. The methods provided by org.springframework.util.Assert provide extra functionality to the standard Java assertions, allowing for more powerful expression of program logic. The following code examples demonstrate the various ways in which the methods of org.springframework.util.Assert can be used.

assertTrue()

The 
assertTrue()
method allows for a boolean expression to be tested. If the expression evaluates to false, an exception will be thrown. 

          
           
assertTrue("Expected true but was false", someBooleanExpression);

assertFalse()

Similar to 
assertTrue()
, the 
assertFalse()
method allows for a boolean expression to be tested. If the expression evaluates to true, an exception will be thrown. 

          
           
assertFalse("Expected false but was true", someBooleanExpression);

assertNull()

The 
assertNull()
methods allows for an object to be tested for null. If the object is not null, an exception will be thrown. 

          
           
assertNull("Expected null but was something else", someObject);

assertNotNull()

The opposite of 
assertNull()
, the 
assertNotNull()
methods allows for an object to be tested for non-null. If the object is null, an exception will be thrown. 

          
           
assertNotNull("Expected something but was null", someObject);

assertSame()

Object obj1 = getObject();
