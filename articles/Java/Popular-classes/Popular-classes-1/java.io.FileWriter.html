java.io.FileWriter

The java.io.FileWriter class is a convenient class for writing text files. It has several constructors which let you specify the file to write to, and how you want to open it. It also has methods for writing text to the file, as well as for closing the file when you're done.

Constructors

The most basic constructor takes a String parameter containing the path of the file to write to: 

          
           
FileWriter writer = new FileWriter("myfile.txt");
This will open the file with default options (append mode set to false and autoflush set to true). Other constructors take two additional parameters: a boolean append flag, and a boolean autoflush flag. The append flag determines whether data is appended to the file (if it exists) or overwritten. The autoflush flag determines whether the FileWriter should buffer data internally before writing it to the file. 

          
           
FileWriter writer = new FileWriter("myfile.txt", true, false);
This will open the file in append mode (true) and autoflush mode (false).
