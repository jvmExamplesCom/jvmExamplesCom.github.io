Introduction to Java.lang.management.ManagementFactory

The Java.lang.management.ManagementFactory class is a utility class of the java management extension (JMX) framework. It provides access to the platform management interface and acts as an entry point for accessing platform MXBean instances. Usage Overview

The getRuntimeMXBean() returns a RuntimeMXBean object which provides information about the runtime environment of the Java virtual machine. The getThreadMXBean() method returns a ThreadMXBean object which provides information about the threads running in the Java virtual machine. The getLoggingMXBean() method returns a LoggingMXBean object which provides methods to configure logging levels and handlers.
