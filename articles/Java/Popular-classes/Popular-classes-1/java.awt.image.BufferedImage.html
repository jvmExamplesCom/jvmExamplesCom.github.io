java.awt.image.BufferedImage

The java.awt.image.BufferedImage class is a subclass of Image class and is used for performing operations on images. It extends the RenderedImage class that encapsulates the long-term storage of a BufferedImage object. A BufferedImage object holds an image in memory. The object is created and then a Graphics2D context is created from it which allows operations such as drawing shapes and painting to be performed on the image.

Creating a BufferedImage object

A BufferedImage object can be created by calling its constructor as follows:


          
           
BufferedImage bufferedImage = new BufferedImage(width, height, type);

where width and height are the desired dimensions of the image while type is one of the following values:


          
           
TYPE_BYTE_GRAY 
TYPE_INT_RGB 
TYPE_INT_ARGB 
TYPE_3BYTE_BGR 
TYPE_4BYTE_ABGR

Drawing on the BufferedImage object

Once the BufferedImage object is created, a Graphics2D context can be obtained from it in order to perform various drawing operations on it. This can be done as follows:


          
           
Graphics2D g2d = (Graphics2D) bufferedImage.getGraphics();

Once a Graphics2D context is obtained, shapes and lines can be drawn on the image using methods like drawLine(), fillRect(), drawOval() etc. For example, the following code segment draws a red line on the image:


          
           
g2d.setColor(Color.RED);
g2d.drawLine(x1, y1, x2, y2);

Similarly, other shapes (like rectangles and ovals) and complex shapes can be drawn as well.

Filling a Shape

The fill() method of the Graphics2D class can be used to fill a particular shape on the image. For example, the following code segment fills a rectangle with yellow color:


          
           
g2d.setColor(Color.YELLOW);
g2d.fillRect(x, y, width, height);

Drawing an Image

An existing image can also be drawn on the BufferedImage object. This can be done using the drawImage() method of the Graphics2D class. For example, the following code segment draws an existing image onto the BufferedImage object:


          
           
g2d.drawImage(image, x, y, observer);

Displaying the image

Once the image is ready, it can be displayed on the screen or stored in an image file. To display the image on the screen, the getGraphics() method of the BufferedImage object can be used. For example, the following code segment displays the image on the screen:


          
           
Graphics g = bufferedImage.getGraphics();
g.drawImage(bufferedImage, 0, 0, null);

To save the image as an image file, the write() method of the ImageIO class can be used. This method takes two arguments, the BufferedImage object and the output file name. For example, the following code segment saves the image as a PNG file:


          
           
ImageIO.write(bufferedImage, "png", new File("myImage.png"));
