java.awt.image.BufferedImage

The java.awt.image.BufferedImage class is a subclass of Image class and is used for performing operations on images. It extends the RenderedImage class that encapsulates the long-term storage of a BufferedImage object. A BufferedImage object holds an image in memory. The object is created and then a Graphics2D context is created from it which allows operations such as drawing shapes and painting to be performed on the image.

Creating a BufferedImage object

A BufferedImage object can be created by calling its constructor as follows:


          
           
BufferedImage bufferedImage = new BufferedImage(width, height, type);

where width and height are the desired dimensions of the image while type is one of the following values:


          
           
TYPE_BYTE_GRAY 
TYPE_INT_RGB 
TYPE_INT_ARGB 
TYPE_3BYTE_BGR 
TYPE_4BYTE_ABGR
