java.util.concurrent.ConcurrentLinkedQueue

The java.util.concurrent.ConcurrentLinkedQueue is a thread-safe linked list based implementation of the java.util.Queue interface. It provides a high-performance, low-contention alternative to the traditional LinkedBlockingQueue and ArrayBlockingQueue implementations. The ConcurrentLinkedQueue uses an internal linked structure that can be efficiently traversed and manipulated without any external locking.
