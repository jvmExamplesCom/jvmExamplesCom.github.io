java.util.GregorianCalendar

The java.util.GregorianCalendar class is a concrete subclass of Calendar and provides the standard calendar system used by most of the world. This class implements a Greogrian calendar with which to manipulate dates and times. The GregorianCalendar includes methods for manipulating dates and times, as well as for computing the difference between two dates.

It is important to note that GregorianCalendar objects are not in real-time, they simply provide a way of manipulating date information. This means that when a date or time is changed, it is not reflected in the real world and must be manually updated.

Example

The following example demonstrates how to use the GregorianCalendar class to manipulate dates by adding days to a given date.


          
           
import java.util.Calendar;
import java.util.GregorianCalendar;
 
public class Example {
    public static void main(String[] args) {
        // Create a GregorianCalendar object set to today's date
        GregorianCalendar today = new GregorianCalendar();
