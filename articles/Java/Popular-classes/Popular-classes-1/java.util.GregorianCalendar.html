The java.util.GregorianCalendar class is a concrete subclass of Calendar and provides the standard calendar system used by most of the world. The GregorianCalendar includes methods for manipulating dates and times, as well as for computing the difference between two dates.

Example

import java.util.Calendar;
import java.util.GregorianCalendar;
 
public class Example {
    public static void main(String[] args) {
        // Create a GregorianCalendar object set to today's date
