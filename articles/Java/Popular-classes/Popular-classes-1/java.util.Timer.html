java.util.Timer

The 
java.util.Timer
class is a powerful tool used for scheduling tasks to run at a predetermined time or periodically. It is an essential tool in many Java applications and systems, providing timed events for code execution. This tutorial will provide a brief overview of the Timer class, followed by numerous code examples to demonstrate its use.

Overview

The Timer class follows the guidelines set out in the Java Thread class. It allows developers to schedule tasks to be executed at a later time. The main methods used are 

schedule()
, 

          
           
scheduleAtFixedRate()
, 
cancel()
, and 
purge()
. The 
schedule()
method will execute a task once at a predetermined time. The 

          
           
scheduleAtFixedRate()
method will execute a task repeatedly at a given rate, with the same delay between executions. The 
cancel()
method will cancel a previously scheduled task and the 
purge()
method purges all cancelled tasks from the timer’s queue.

The following example illustrates how to schedule a task to be executed at a future time using the 

schedule()
method. 

          
           
Timer timer = new Timer();

method.

// Schedules a task to be repeated every 10 seconds
timer.scheduleAtFixedRate(new Task(), 0, 10000);
