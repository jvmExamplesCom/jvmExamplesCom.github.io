java.sql.Statement

Overview

The java.sql.Statement interface is used to execute a static SQL statement and retrieve the results produced by it. It creates an object which represents a precompiled SQL statement that you can use to execute one or multiple times. It is part of the JDBC API and allows developers to create and execute SQL statements easily.

Creating the Statement Object

To create a Statement object, you need to use the createStatement() method of the Connection object. This method takes no arguments and returns a Statement object. 

          
           
Connection conn = DriverManager.getConnection("jdbc:mysql://localhost/database", "user", "password");
Statement stmt = conn.createStatement();

Using the Statement Object

Once you have created the statement object you can use it to execute various SQL statements. The general form for executing a statement is: 

          
           
stmt.execute(String sql);
where the sql argument is a String containing the SQL statement to be executed. For example, the following statement creates a table named "products": 

          
           
stmt.execute("CREATE TABLE products (name VARCHAR(20), price DECIMAL(9,2))");
You can execute other types of SQL statements as well, such as SELECT, UPDATE, and DELETE.

Retrieving Data using the Statement Object

When executing SELECT statements, the execute() method returns a ResultSet object which contains the data returned by the query. For example, the following statement retrieves all records from the "products" table: 

          
           
ResultSet rs = stmt.execute("SELECT * FROM products");
The ResultSet object can then be used to retrieve the data returned by the query. For example, the following code iterates through the ResultSet and prints out the values for each row: 

          
           
while (rs.next()) {
   String name = rs.getString("name");
   double price = rs.getDouble("price");

   System.out.println("Product: " + name + ", Price: " + price);
}
Using the Statement object, you can also execute INSERT, UPDATE and DELETE statements. These statements do not return any results and can be executed using the same execute() method. For example, the following statement inserts a new record into the "products" table: 

          
           
stmt.execute("INSERT INTO products VALUES ('mouse', 19.99)");

Closing the Statement Object

Once you have finished using the Statement object, it must be closed using the close() method. This ensures that any resources associated with the Statement object are released, such as database connections. 

          
           
stmt.close();
It is important to remember to always close the Statement object after you have finished using it. Failing to do so may lead to memory leaks and other problems.
