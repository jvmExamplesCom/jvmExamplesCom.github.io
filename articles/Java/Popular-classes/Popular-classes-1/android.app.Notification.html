android.app.Notification

The android.app.Notification class is an important part of the Android Framework since it allows applications to easily display their notifications to their users. It is used for many tasks such as alerting the user about a missed call, displaying an image in the status bar, or simply playing a sound. There are a few key methods and properties to understand in order to fully use and customize the android.app.Notification class.

Methods

The most important methods for the android.app.Notification class are create(), setTitle(), setContentText(), and addAction(). The create() method is responsible for creating the notification object using a builder pattern. The setTitle() and setContentText() methods are then used to set the title and content text of the notification respectively. Lastly the addAction() method is used to add an action, such as a button, that can be used with the notification.

Properties

The key properties for the android.app.Notification class are notificationPriority, visibility, and category. The notificationPriority defines how important this notification is compared to other notifications. The visibility defines the visibility of the notification, such as only visible to the sender or visible to everyone. Lastly the category defines the type of notification, such as a message or email.
