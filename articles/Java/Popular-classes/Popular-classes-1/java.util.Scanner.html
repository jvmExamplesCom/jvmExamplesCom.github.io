Skipping Tokens

If you have multiple tokens separated by a delimiter, you can use the skip() method to skip over certain tokens. For example, if you have a string of numbers separated by commas, you can use the skip() method to skip over the commas and read only the numbers.

Scanner scanner = new Scanner(numbers);
while (scanner.hasNext()) {

    int number = scanner.nextInt();
    System.out.println(number);
}
