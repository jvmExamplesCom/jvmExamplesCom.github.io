Creating a Scanner Object

The first step in using the Scanner class is to create an object. To create a Scanner object, you must pass in a java.lang.Readable object, which is typically a text file or the System.in object for console input. 

          
           
// Create a Scanner object for console input
Scanner scanner = new Scanner(System.in);
                
// Create a Scanner object for a text file

Reading User Input

Once you have created a Scanner object, you can begin reading data from the source. The Scanner class provides various methods for reading various types of data, such as nextInt(), nextDouble(), nextBoolean(), etc. 

          
           
// Read an int

// Read a double
double number = scanner.nextDouble();

Skipping Tokens

If you have multiple tokens separated by a delimiter, you can use the skip() method to skip over certain tokens. For example, if you have a string of numbers separated by commas, you can use the skip() method to skip over the commas and read only the numbers. 

          
           
// Skip over commas
String numbers = "1,2,3,4,5";
Scanner scanner = new Scanner(numbers);
while (scanner.hasNext()) {
    scanner.skip(",");
    int number = scanner.nextInt();
    System.out.println(number);
}

Using Regular Expressions

The Scanner class also supports the use of regular expressions when reading from the source. For example, you can use a regular expression to match any token that contains a number. 

          
           
// Match tokens containing a number
String numbers = "1, 2, one, two";
Scanner scanner = new Scanner(numbers);
while (scanner.hasNext()) {
    if (scanner.hasNext("\\d+")) {
        int number = scanner.nextInt();
        System.out.println(number);
    } else {
        scanner.next();
    }
}
