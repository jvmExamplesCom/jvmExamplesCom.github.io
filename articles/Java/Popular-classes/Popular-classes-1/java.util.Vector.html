Java Vector

Java Vector is a collection class in the Java utility package. It is used to store a sequence of objects, similar to an array. It is dynamic in size, meaning that it can grow and shrink as needed. Java Vector has several useful properties including storing multiple data types, keeping track of its size, and allowing access to elements in constant time.

In this article, we'll take a look at how a Java Vector works and how to use one to store and manipulate data.

Initializing a Java Vector

The most basic way to create a Java Vector is with the constructor:


          
           
Vector v = new Vector();

This will create an empty Vector with a size of 0. We can also specify the initial capacity of the Vector using the two-parameter constructor:


          
           
Vector v = new Vector(10);

This will create an empty Vector with an initial capacity of 10. Note that this does not specify the size of the Vector; it just defines how much space it has allocated for elements.

We can also specify both the initial capacity and the amount by which the Vector should grow when more space is needed using the three-parameter constructor:


          
           
Vector v = new Vector(10, 5);

This will create an empty Vector with an initial capacity of 10 and a growth factor of 5. When the Vector grows beyond its current capacity, the capacity will be increased by the growth factor.

Adding Elements to a Vector

Once we've initialized a Vector, we can add elements using the 

add()
method: 

          
           
v.add("Hello");
v.add("World");

This will add the strings "Hello" and "World" to the Vector. The Vector's size will be updated accordingly.

We can also add multiple elements at once using the 

addAll()
method: 

          
           
List<String> list = Arrays.asList("foo", "bar", "baz");
v.addAll(list);

This will add the strings "foo", "bar", and "baz" to the Vector. The Vector's size will be updated accordingly.

Removing Elements from a Vector

We can remove elements from a Vector using the 
remove()
method: 

          
           
v.remove("Hello");

This will remove the string "Hello" from the Vector. The Vector's size will be updated accordingly.

We can also remove multiple elements at once using the 

removeAll()
method: 

          
           
List<String> list = Arrays.asList("foo", "bar", "baz");
v.removeAll(list);

This will remove the strings "foo", "bar", and "baz" from the Vector. The Vector's size will be updated accordingly.

Accessing Elements from a Vector

We can access elements from a Vector using the 
get()
method: 

          
           
String s = (String) v.get(0);

This will return the element at index 0, which is the first element in the Vector. Note that the 

get()
method returns an 
Object
, so we have to cast it to the desired type.

Iterating Over a Vector

We can iterate over a Vector using an 
Iterator
: 

          
           
Iterator<String> iter = v.iterator();
while (iter.hasNext()) {
    String s = iter.next();
    System.out.println(s);
}

This will loop through each element in the Vector, printing out the value of each element.

Sorting a Vector

We can sort a Vector using the 
sort()
method: 

          
           
v.sort();

This will sort the elements in the Vector using the natural ordering of the elements. If the elements are objects, they must implement the 

Comparable
interface. We can also pass in a 
Comparator
to sort the elements by a custom comparison: 

          
           
v.sort(new Comparator<String>() {
    public int compare(String s1, String s2) {
        return s1.length() - s2.length();
    }
});

This will sort the elements in the Vector by their length.

Conclusion

In this article, we've looked at how to use a Java Vector to store and manipulate data. We've seen how to initialize a Vector, add and remove elements, access elements, iterate over a Vector, and sort a Vector. With these tools, we can easily work with collections of data in Java.
