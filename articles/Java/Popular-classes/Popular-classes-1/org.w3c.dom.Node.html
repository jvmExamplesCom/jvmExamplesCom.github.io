org.w3c.dom.Node

The org.w3c.dom.Node interface is a fundamental part of the Document Object Model (DOM) API. It provides a generic view of an XML document, allowing for traversal of nodes and modification of the DOM tree.

Creating & Accessing Nodes

The Node interface allows for the creation of new nodes and accessing existing nodes in the DOM tree.

Creating a new node

Creating a new node can be done using the createElement() or createTextNode() methods. 


          
           
Element myElement = document.createElement("myElement");
Text myText = document.createTextNode("My Text");

To add the newly created nodes to the DOM tree, they must be appended to an existing node.


          
           
Node parentNode = document.getElementById("parent-node");
parentNode.appendChild(myElement);
parentNode.appendChild(myText);

Accessing a Node

The getElementById() method can be used to access an existing node in the DOM tree. 


          
           
Node myNode = document.getElementById("my-node");

Modifying Nodes

Nodes can also be modified using the setAttribute() , removeChild() or replaceChild() methods.

Set Attributes

Attributes can be added or modified using the setAttribute() method.


          
           
myElement.setAttribute("class", "my-class");

Remove Child

To remove a child node, the removeChild() method can be used.


          
           
parentNode.removeChild(myElement);

Replace Child

An existing child node can also be replaced with a different node using the replaceChild() method.


          
           
Node newNode = document.createElement("new-node");
parentNode.replaceChild(newNode, myElement);

Conclusion

The org.w3c.dom.Node interface provides an essential part of the DOM API for creating, accessing, and modifying nodes in an XML document. It is a powerful tool for manipulating XML documents and provides a generic view of the document.
