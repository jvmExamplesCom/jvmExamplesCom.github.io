android.content.DialogInterface

Android’s DialogInterface is an interface in the Android SDK that allows developers to create different kinds of dialog boxes to be used in an Android application. These dialog boxes can be used to ask for user input, display alerts and messages, or display progress bars. This tutorial will cover the basics of using DialogInterface and how it can be used to improve the user experience of an Android application.

Creating Dialog Interfaces

Creating a DialogInterface is done through an Activity class. An Activity class is a Java class used to initiate the user interface of an Android application. To create a DialogInterface, an instance of the class must first be created. The following code creates a DialogInterface instance: 

          
           
DialogInterface d = new DialogInterface(this);
In the code above, the “this” refers to the Activity instance that is currently active.

Methods

The DialogInterface class has several methods that can be used to alter or retrieve information from the dialog box. Some of these methods are listed below. 

          
           
d.setTitle(String title);
This method sets the title of the dialog box. It takes a String parameter which is used as the title of the dialog box. 

          
           
d.setMessage(String message);
This method sets the message of the dialog box. It takes a String parameter which is used as the message of the dialog box. 

          
           
d.show();
This method displays the dialog box on the screen. It should be invoked once all other configuration settings have been made. 

          
           
d.dismiss();
This method closes the dialog box. It should be invoked after the user has interacted with the dialog box.

Example Usage

The following code example demonstrates the usage of DialogInterface in an Android application. 

          
           
public class MainActivity extends Activity {
    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);

        //Create DialogInterface instance
        DialogInterface d = new DialogInterface(this);

        //Set title and message
        d.setTitle("Please Confirm");
        d.setMessage("Are you sure you want to proceed?");

        //Show dialog box
        d.show();

        //Dismiss dialog box
        d.dismiss();
    }
}
In the code above, a DialogInterface instance is created in the onCreate() method of the main activity class. A title and message are set and the dialog box is then shown and dismissed.
