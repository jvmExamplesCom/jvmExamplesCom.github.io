What is java.io.ObjectOutputStream?

The java.io.ObjectOutputStream class enables you to write Java objects to an output stream. It is used to serialize and deserialize objects to and from byte streams. This is useful when you want to pass objects over the network or store them in a file.

Examples of Using ObjectOutputStream

The code below shows an example of writing a simple Person object to an output stream. 


          
           
public void writeObjectToStream(Person person) throws IOException { 
    try (ObjectOutputStream out = new ObjectOutputStream(new FileOutputStream("person.bin"))) { 
        out.writeObject(person); 
    } 
}

Here's an example of reading an object back from the stream:


          
           
public Person readObjectFromStream() throws IOException, ClassNotFoundException { 
    try(ObjectInputStream in = new ObjectInputStream(new FileInputStream("person.bin"))) { 
        return (Person)in.readObject(); 
    } 
}

The code below shows an example of writing a list of Person objects to an output stream. 


          
           
public void writeListToStream(List<Person> people) throws IOException {
