java.io.BufferedInputStream

The java.io.BufferedInputStream class is a subclass of InputStream, which allows for high-performance buffered input operations on streams. This means the data can be read from an input stream and stored in an internal buffer before being accessed at the application level. This has the effect of increasing the rate at which the data can be read from the underlying stream and subsequently processed, as the time taken to read from the buffer is generally much faster than reading from the input stream.

In order to use a BufferedInputStream, you must provide a reference to an existing InputStream. The data from this stream will then be read into the BufferedInputStream's internal buffer, making it available for subsequent access operations. The size of the buffer used by the BufferedInputStream can be configured when constructing the object.

Example of using BufferedInputStream

The following example shows how to construct and use a BufferedInputStream:


          
           
InputStream input = new FileInputStream("myfile.txt");
BufferedInputStream bis = new BufferedInputStream(input, 1024);

int c;
while((c = bis.read()) != -1) {
  System.out.println((char)c);
}

bis.close();

In the example above we create an InputStream from a file called "myfile.txt" and then construct a BufferedInputStream around it with a buffer size of 1024 bytes. We then loop through the stream, reading one byte at a time and outputting its character representation to the console. Finally, we close the BufferedInputStream, which will also close the backing InputStream.

Reading bytes from a BufferedInputStream

The BufferedInputStream also provides methods for reading arrays of bytes from the stream. For example, the read(byte[], int, int) method reads up to the specified length of bytes from the stream and stores them in the supplied byte array starting at the given offset.


          
           
InputStream input = new FileInputStream("myfile.txt");
BufferedInputStream bis = new BufferedInputStream(input, 1024);

byte[] data = new byte[1024];
int len = bis.read(data, 0, 1024);

System.out.println(Arrays.toString(data));

bis.close();

In this example, we begin by creating a BufferedInputStream around an existing InputStream and then allocate a byte array of size 1024 bytes. We then call the read() method on our BufferedInputStream, passing in the byte array and the number of bytes we want to read. This will cause the BufferedInputStream to fill the array with the specified number of bytes read from the backing stream. Finally, we print out the contents of the array to the console.

Wrapping up

The BufferedInputStream is an incredibly useful class for increasing the performance of input operations on streams. By providing your own buffer and managing the reading from the underlying stream yourself, you can greatly improve the data throughput and reduce the time taken to process the data.
