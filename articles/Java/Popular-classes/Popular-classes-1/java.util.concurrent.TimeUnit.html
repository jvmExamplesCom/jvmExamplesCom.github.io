java.util.concurrent.TimeUnit

Java provides the java.util.concurrent.TimeUnit class which encapsulates time durations in well-known units, including milliseconds, seconds, minutes, hours, days, and so on. It provides utility methods for conversion between different units, and for manipulation, such as truncating or rounding. In addition to these, it allows for the creation of a composite unit, for example hours and minutes.

The TimeUnit class represents an amount of time in a given unit of measurement. The available units of measurement are milliseconds, seconds, minutes, hours, days, weeks, months, and years. Each of these can be converted to any other unit using the convert method.

Example Code


          
           
// Get the current time in milliseconds
long millis = System.currentTimeMillis();

// Convert milliseconds to seconds
long seconds = TimeUnit.MILLISECONDS.toSeconds(millis);
 
// Convert milliseconds to minutes
long minutes = TimeUnit.MILLISECONDS.toMinutes(millis);

// Convert milliseconds to hours
long hours = TimeUnit.MILLISECONDS.toHours(millis);

// Convert milliseconds to days
long days = TimeUnit.MILLISECONDS.toDays(millis);

// Print the results
System.out.println("Milliseconds: " + millis);
System.out.println("Seconds: " + seconds);
System.out.println("Minutes: " + minutes);
System.out.println("Hours: " + hours);
System.out.println("Days: " + days);

Creating a Composite TimeUnit

TimeUnit also provides the ability to create a composite unit, that is a combination of two or more units. For example, if you want to represent a duration of 2 hours and 45 minutes you can do so by creating a composite unit of hours and minutes.

To create a composite unit, you need to first create a TimeUnit instance with the desired unit (for example hours) and then call the createComposite method with the desired value and the additional unit (minutes). The following example creates a composite unit representing 2 hours and 45 minutes:


          
           
TimeUnit hours = TimeUnit.HOURS;
TimeUnit compositeUnit = hours.createComposite(2, TimeUnit.MINUTES);

long millis = System.currentTimeMillis();
long compositeDuration = compositeUnit.convert(millis, TimeUnit.MILLISECONDS);

The result is the number of units of the composite TimeUnit (in this example hours and minutes) corresponding to the given duration in the given source unit (in this example milliseconds). The result can then be further broken down into its component parts using the toHours and toMinutes methods.

The result of the above code is that the hours and minutes variables will have the values 2 and 45 respectively.
