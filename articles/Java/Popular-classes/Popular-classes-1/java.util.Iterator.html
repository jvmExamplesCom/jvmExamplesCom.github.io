java.util.Iterator

The 
java.util.Iterator
interface is an important tool for working with collections of objects in Java. An iterator is an object that allows you to traverse a collection of objects in a linear fashion, either forwards or backwards. The Iterator interface has a variety of methods to help you manage the collection and move through it. In this article, we will discuss the various methods of the Iterator interface and provide examples.

Iterator Interface Methods

The Iterator interface provides five methods that are used to traverse a collection:

  • hasNext()
    – returns 
    true
    if there are more elements in the collection and 
    false
    otherwise.
  • next()
    – returns the next element in the collection.
  • remove()
    – removes the current element from the collection.
  • forEachRemaining()
    – performs an action for each remaining element in the collection.
  • spliterator()
    – returns a Spliterator for the given collection.

Creating an Iterator

To create an iterator, you must first create a collection. For this example, we will use an ArrayList:


          
           
List<String> list = new ArrayList<String>();
list.add("A");
list.add("B");
list.add("C");
We can then create an Iterator: 

          
           
Iterator<String> iterator = list.iterator();

next()

String element = iterator.next();

The

forEachRemaining()

The 
forEachRemaining()
method performs an action for each remaining element in the collection:

The 
spliterator()
method returns a 
Spliterator
for the given collection. The 
Spliterator
is an iterator-like object that provides additional methods for traversing and manipulating the collection: 

          
           
Spliterator<String> spliterator = list.spliterator();

Example

Let's look at an example of using the Iterator interface. We will create an ArrayList of integers and use the 

Iterator
to print each element in the list:
