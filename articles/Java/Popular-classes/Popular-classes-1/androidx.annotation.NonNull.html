What is androidx.annotation.NonNull?

The

When the annotation is applied to a method parameter, it implies that the value of the parameter should never be null when the method is invoked. Additionally, when the annotation is applied to a field, it implies that the field should never be null, even if the field has not been initialized. Additionally, the 

          
           
androidx.annotation.NonNull

@Nullable
