What is androidx.annotation.NonNull?

The 

          
           
androidx.annotation.NonNull
annotation is a type of meta-annotation which indicates that a method or field should never be null. This annotation can be applied to either a method parameter or to a field. When the annotation is applied to a method parameter, it implies that the value of the parameter should never be null when the method is invoked. Additionally, when the annotation is applied to a field, it implies that the field should never be null, even if the field has not been initialized. Additionally, the 

          
           
androidx.annotation.NonNull
annotation can also be used to indicate that a method should never return a null value. This annotation should be used in conjunction with other annotations such as 

          
           
@Nullable
to indicate the expected behavior of a method.
