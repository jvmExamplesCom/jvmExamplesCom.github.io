org.apache.commons.io.FileUtils

The Java class org.apache.commons.io.FileUtils provides several convenient static methods to access files and directories. It supports a wide variety of file operations such as creating, deleting, copying, and moving files and directories, reading and writing files, and creating symbolic links.

Examples

Let's look at some examples of using the FileUtils class.

deleteDirectory()
