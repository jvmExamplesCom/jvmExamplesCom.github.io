java.io.File Reader

The java.io.FileReader class is a utility for reading data from a file. It provides methods for reading a character, an array of characters, or a line of text from a file. In addition, it can be used to create a buffered reader, which can then be used to read data in chunks. The Java FileReader class enables you to access files in your computer's file system.

Creating an instance

To create an instance of the java.io.FileReader class, use the constructor that takes a String parameter representing the name of a file, or a File object which represents the file. Here are some examples:


          
           
FileReader fr = new FileReader("data.txt");
File f = new File("data.txt");
FileReader fr2 = new FileReader(f);

Reading from a file

Once we have an instance of the java.io.FileReader class, we can read data from a file. There are several methods available. Here are some examples:


          
           
int ch = fr.read();
int[] buffer = new int[100];
int count = fr.read(buffer, 0, 100);
String line = fr.readLine();
The 

          
           
read()
method reads a single character from a file and returns it as an integer. The 

          
           
read(int[], int, int)
method reads up to a specified number of characters from a file, starting at a given offset, and stores them in a character array. The 

          
           
readLine()
method reads a line of text from a file.

Closing a file

When you are finished reading from a file, it is important to close it. To do this, simply call the 


          
           
close()
method of the java.io.FileReader object: 

          
           
fr.close();

Creating a BufferedReader

The java.io.FileReader class can also be used to create a buffered reader. A buffered reader can be used to read from a file in chunks, rather than one character or one line at a time. To create a BufferedReader, use the following code:


          
           
BufferedReader br = new BufferedReader(fr);
The 

          
           
read()
and 

          
           
readLine()
methods of the BufferedReader class work in the same way as those of the FileReader class.
