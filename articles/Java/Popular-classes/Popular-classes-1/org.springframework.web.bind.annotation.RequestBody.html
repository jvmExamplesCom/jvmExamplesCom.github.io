org.springframework.web.bind.annotation.RequestBody

The 

          
           
org.springframework.web.bind.annotation.RequestBody
annotation indicates that a method parameter should be bound to the value of the HTTP request body. Used in conjunction with the 
@Controller
and 
@RequestMapping
annotations, this annotation can be used to bind the body of the HTTP request to your controller method. It can be used to receive both JSON and XML payloads, depending on the content type of the request.

Examples of Using RequestBody

Here is an example of how to use the 
@RequestBody
annotation to bind the JSON contents of an HTTP request to a method parameter: 

          
           
@RequestMapping("/adduser")
public User addUser(@RequestBody User user) {
	// save the user object to the database
	return user;
}
In this example, the 
addUser
method is mapped to an HTTP request with the URL "/adduser". This method will take the contents of the request body, convert it into a User object, and return it. The same technique can be used to handle an XML request body: 

          
           
@RequestMapping("/adduser")
public User addUser(@RequestBody XmlUser user) {
	// save the user object to the database
	return user;
}

In this example, the incoming request body is assumed to be XML and will be converted into an 

XmlUser
object. You can also annotate specific fields within a model object if you want to map them directly from the request body: 

          
           
@XmlRootElement
public class User {
	@RequestBody
	private String name;
	@RequestBody
	private int age;
	
	// getters and setters for name and age
}
In this example, the 
name
and 
age
fields will be automatically populated from the request body without the need for any additional mapping code.

Conclusion

The 

          
           
org.springframework.web.bind.annotation.RequestBody
annotation allows you to easily bind the contents of an HTTP request body to a method parameter in your controller. It supports both JSON and XML request bodies, and provides annotations that allow you to map individual fields of an object directly from the request body. With this annotation, you can quickly create powerful RESTful APIs that can consume complex data from the client.
