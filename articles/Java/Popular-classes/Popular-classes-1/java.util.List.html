java.util.List

Java provides various ways to store and manipulate data structures, one of which is the java.util.List interface. The List interface is part of the Java Collection framework and provides an ability to store a collection of objects in a certain order. It is a part of the java.util package.

Creating List

A list can be created in two ways. One is to use the static method 


          
           
List.of()
to create an immutable list and the other is to use the default constructor 

          
           
new ArrayList()
. 

          
           
List<String> list1 = List.of("Dave", "John", "Bill");
List<String> list2 = new ArrayList<>();
list2.add("Dave");
list2.add("John");
list2.add("Bill");

Adding Elements

Elements can be added to a list using the 

          
           
add(E e)
method or by using the 

          
           
addAll(Collection c)
method to add multiple elements at once. 

          
           
List<String> list1 = List.of("Dave", "John", "Bill");

list1.add("Mary");

List<String> list2 = new ArrayList<>();
list2.add("Dave");
list2.add("John");
list2.add("Bill");

list2.addAll(List.of("Mary", "Alice"));

Retrieving Elements

Elements can be retrieved from the list using two methods. The 


          
           
get(int index)
method retrieves an element at a specified index and the 

          
           
contains(Object o)
method checks whether the list contains a particular element or not. 

          
           
// Initialize the list
List<String> list = List.of("Dave", "John", "Bill", "Mary", "Alice");

// Retrieve an element at specified index
String name = list.get(3); // name == "Mary"

// Check if list contains an element
boolean isPresent = list.contains("Alice"); // isPresent == true

Removing Elements

Elements can be removed from the list using the 

          
           
remove(int index)
or 

          
           
remove(Object o)
methods. 

          
           
// Initialize the list
List<String> list = List.of("Dave", "John", "Bill", "Mary", "Alice");

// Remove an element at specified index
list.remove(2); // list = ["Dave", "John", "Mary", "Alice"]

// Remove a particular element
list.remove("Mary"); // list = ["Dave", "John", "Alice"]

Iterating Over List

The list can also be iterated over using the for-each loop or the for-loop with the iterator. 


          
           
// Initialize the list
List<String> list = List.of("Dave", "John", "Bill", "Mary", "Alice");

// Using for-each loop
for (String name : list) {
    System.out.println(name);
}

// Using Iterator
Iterator<String> iter = list.iterator();
while(iter.hasNext()) {
    String name = iter.next();
    System.out.println(name);
}

Conclusion

The Java List interface provides a powerful way to store and manipulate collections of data. It is an important part of the Java Collections framework and provides efficient ways to perform operations like add, remove, get, iterate and search.
