java.awt.Dimension

Java AWT Dimension class is a container class used to represent the size of a component or an area in terms of width and height. This class encapsulates the width and height of components and provides methods to calculate the area.

Constructors


          
           
Dimension d1 = new Dimension(); 
Dimension d2 = new Dimension(width, height); 
Dimension d3 = new Dimension(d);
The first constructor creates an instance with invalid values for width and height. The second constructor takes width and height as parameters and the third constructor takes an object of the same class as the parameter and assigns the values to this object.

Methods


          
           
getWidth(): Returns the width of this object.
getHeight(): Returns the height of this object. 
setSize(int width, int height): Sets the size of this object to specified width and height. 
getSize(): Returns an instance of Dimension which has the same width and height as this.

Example


          
           
import java.awt.Dimension;

public class DimensionDemo {
    public static void main(String[] args) {
       Dimension d1 = new Dimension(5, 10);

       System.out.println("Width: " + d1.getWidth());
       System.out.println("Height: " + d1.getHeight());
       System.out.println("Area: " + (d1.getWidth() * d1.getHeight()));

       d1.setSize(20, 30);
       System.out.println("New Width: " + d1.getWidth());
       System.out.println("New Height: " + d1.getHeight());
       System.out.println("New Area: " + (d1.getWidth() * d1.getHeight()));
    }
}
