Java.util.function.Supplier

Java.util.function.Supplier is a functional interface introduced in Java 8 as part of the java.util.function package. It is a functional interface and its single abstract method, get(), takes no arguments and returns a result of type T. This interface can be used in scenarios where you want to produce a result of type T without taking any arguments.

Simple Example

Let's take a look at an example of creating a Supplier and using it to retrieve a result. We will create a simple Supplier that returns a String. 

          
           
public class SupplierExample {

    public static void main(String[] args) {
        Supplier<string> supplier  = () -> "Hello World!";
        System.out.println(supplier.get());
    }
    
}

Hello World!
