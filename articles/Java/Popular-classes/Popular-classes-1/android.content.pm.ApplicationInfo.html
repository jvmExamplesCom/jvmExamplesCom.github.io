Android.content.pm.ApplicationInfo

Android's ApplicationInfo class provides information about an application package, such as its package name, version code, and flags.

Methods


          
           
public int getVersionCode()
Returns the version code of the application. 

          
           
public String getPackageName()
Returns the package name of the application. 

          
           
public int getFlags()
Returns a set of flags associated with the application. This includes indicates the defaults, if any, set by the package installer. The values could be FLAG_DEBUGGABLE, FLAG_SYSTEM, FLAG_EXTERNAL_STORAGE, FLAG_UPDATED_SYSTEM_APP, FLAG_SUPPORTS_SMALL_SCREENS, FLAG_SUPPORTS_NORMAL_SCREENS, FLAG_SUPPORTS_LARGE_SCREENS, FLAG_RESIZEABLE_FOR_SCREENS, FLAG_SUPPORTS_XLARGE_SCREENS, FLAG_INSTALLED, FLAG_IS_DATA_ONLY, FLAG_IS_GAME and FLAG_FACTORY_TEST.

Example

The following example retrieves the package name of the app, the version code and the flags: 

          
           
ApplicationInfo info = context.getPackageManager().getApplicationInfo(context.getPackageName(), 0);
String packageName = info.packageName;
int versionCode = info.versionCode;
int flags = info.flags;
System.out.println("packageName=" + packageName);
System.out.println("versionCode=" + versionCode);
System.out.println("flags=" + flags);
