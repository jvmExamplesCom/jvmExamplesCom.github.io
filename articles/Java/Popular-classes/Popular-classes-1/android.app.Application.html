android.app.Application

The android.app.Application class is the base class for all Android applications. It provides an environment in which different kinds of application components can cooperate and interact effectively. This class is responsible for managing the lifecycle of all activities, services, broadcasts receivers, and content providers.

Methods

The android.app.Application class provides several methods for its subclass to use:

  • onCreate()
    : This method is called when the application is first created.
  • onLowMemory()
    : This method is called when the application is running low on memory.
  • onTrimMemory()
    : This method is called when the application's memory usage needs to be trimmed.
  • onTerminate()
    : This method is called when the application is about to be terminated.

onConfigurationChanged()
    : This method is called when the configuration of the device changes.

onProvideAssistData()
    : This method is called when the user requests assist data for the application.
