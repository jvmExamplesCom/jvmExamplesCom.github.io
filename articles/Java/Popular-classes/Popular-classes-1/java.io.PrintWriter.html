Java.io.PrintWriter

The java.io.PrintWriter class is a powerful tool for writing character output to a file. It provides methods for printing formatted representations of objects to the file and flushing buffered output to the file. This class is used in a variety of applications, such as writing log files, reading and writing text files, and generating HTML files.

Creating a PrintWriter

A PrintWriter must be created before it can be used. A PrintWriter can be created from a File, a FileWriter, an OutputStream, or a Writer. The following code snippet shows how to create a PrintWriter from a File.


          
           
File file = new File("myFile.txt");
PrintWriter pw = new PrintWriter(file);

The following code snippets demonstrate how to create a PrintWriter from a FileWriter, an OutputStream, and a Writer respectively.


          
           
FileWriter fw = new FileWriter("myFile.txt");
PrintWriter pw = new PrintWriter(fw);

OutputStream os = new FileOutputStream("myFile.txt");
PrintWriter pw = new PrintWriter(os);

Writer writer = new OutputStreamWriter(System.out);
PrintWriter pw = new PrintWriter(writer);

Writing to a File

Once a PrintWriter has been created, it can be used to write character data to a file. The PrintWriter class provides several methods for writing to a file. The 

print()
method is used to write primitive types to the file and the 
println()
method is used to write strings and objects to the file. The following code snippet demonstrates how to use the 
println()
method to write a string to a file. 

          
           
PrintWriter pw = new PrintWriter("myFile.txt");
pw.println("This is a line of text.");
The 
printf()
method is used to write formatted strings to the file. The syntax of this method is similar to that of the 
System.out.printf()
method. The following code snippet demonstrates how to use the 
printf()
method to format and write a string to a file. 

          
           
PrintWriter pw = new PrintWriter("myFile.txt");
pw.printf("This is a %d line of text.", 1);

Flushing Buffered Output

When a PrintWriter is created, it uses an internal buffer to store the characters that are written to it. The internal buffer is flushed when one of the following conditions is met:

  • The internal buffer becomes full.
  • The 
    flush()
    method is called.
  • The 
    close()
    method is called.

The following code snippet demonstrates how to call the 

flush()
method to flush the internal buffer. 

          
           
PrintWriter pw = new PrintWriter("myFile.txt");

pw.println("This is the first line of text.");
pw.println("This is the second line of text.");

pw.flush();  // Flush the internal buffer

Closing a PrintWriter

It is important to close a PrintWriter when it is no longer needed. Closing a PrintWriter also flushes the internal buffer. The following code snippet demonstrates how to use the 

close()
method to close a PrintWriter. 

          
           
PrintWriter pw = new PrintWriter("myFile.txt");

pw.println("This is the first line of text.");
pw.println("This is the second line of text.");

pw.close();  // Close the PrintWriter
