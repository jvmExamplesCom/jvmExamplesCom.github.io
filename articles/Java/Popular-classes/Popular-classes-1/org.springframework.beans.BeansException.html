org.springframework.beans.BeansException

org.springframework.beans.BeansException is an exception thrown when a bean could not be created from the given definition in the XML configuration file or when a matching dependent bean could not be found. It is a sub-class of RuntimeException and is thrown by the Spring framework during bean creation.

Examples of org.springframework.beans.BeansException

        ApplicationContext context = new ClassPathXmlApplicationContext("beans.xml");

    }
}
