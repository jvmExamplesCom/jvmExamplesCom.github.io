Java.util.Enumeration

Java.util package provides an interface "Enumeration" which is used to represent an entity, such as a list or a set of elements, that can be accessed one at a time. The Enumeration interface provides a way to iterate through a collection of objects, allowing access to one element at a time.

Enumeration Methods

The Enumeration interface provides two methods, hasMoreElements() and nextElement(), that allow you to iterate through a collection of objects. The hasMoreElements() method returns true if there are more elements to be accessed in the collection, while the nextElement() method returns the next element in the collection.

Example 1: Retrieving an Element from an Enumeration


          
           
Enumeration<String> list = Arrays.asList("Monday", "Tuesday", "Wednesday").elements();
while (list.hasMoreElements()) {
  System.out.println(list.nextElement());
}
// Outputs: Monday
//           Tuesday
//           Wednesday

Example 2: Checking for the Existence of an Element in an Enumeration


          
           
Enumeration<String> list = Arrays.asList("Monday", "Tuesday", "Wednesday").elements();
String day = "Thursday";
while (list.hasMoreElements()) {
  if (day.equals(list.nextElement())) {
    System.out.println("Day is present in the Enumeration!");
  }
}
// Outputs: Day is present in the Enumeration!

Example 3: Iterating Through an Enumeration Without Modifying its Contents


          
           
ArrayList<String> list = new ArrayList<>(Arrays.asList("Monday", "Tuesday", "Wednesday"));
Enumeration<String> enumList = Collections.enumeration(list);
while (enumList.hasMoreElements()) {
   System.out.println(enumList.nextElement());
}
// Outputs: Monday
//           Tuesday
//           Wednesday

The java.util.Enumeration interface provides an easy way to iterate through a collection of objects, allowing access to one element at a time. Enumeration provides two main methods, hasMoreElements() and nextElement(), that can be used to retrieve elements from a collection.
