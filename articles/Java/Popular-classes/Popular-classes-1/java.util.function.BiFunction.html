Introduction to java.util.function.BiFunction

Java's 

          
           
java.util.function.BiFunction
provides a functional interface used to provide a function of two parameters that return a single result. It is used primarily to operate on two objects of the same type, or a combination of two different types. This article will discuss what BiFunction does and how it works. In addition, code examples will be provided for deeper understanding.

What Does BiFunction Do?

BiFunction takes in two parameters of any two given types, performs an operation on those parameters, and returns a single result that is of a given type. This can be useful for performing calculations, combining data from two sources, or any other operations that require two inputs.

Example

As an example, consider a 

          
           
BiFunction<Integer, Integer, Integer> adder
. The 

          
           
adder
BiFunction takes in two Integers, sums them together, and returns an Integer as the result. This could be implemented like so: 

          
           
BiFunction<Integer, Integer, Integer> adder = (x, y) -> x + y;
int sum = adder.apply(2, 3); // sum is 5
The apply method can be used to perform the operation on any two parameters of the proper types.

BiFunction<String, Double, String> concater = (s, d) -> s + Double.toString(d);
