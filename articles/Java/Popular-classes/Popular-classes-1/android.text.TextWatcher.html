android.text.TextWatcher

Android TextWatcher is an interface used to listen for text changes in an EditText. It can be useful for implementing features such as auto-complete, validation, formatting, spell-check and other text manipulation functions.

Overview

The TextWatcher interface contains three methods which are called when the text of an EditText is changed:
  • onTextChanged: Called every time the text of an EditText changes.
  • beforeTextChanged: Called before the text of an EditText changes.
  • afterTextChanged: Called after the text of an EditText changes.
These methods are passed a CharSequence containing the current text, a start and end position, and an optional count of new characters that have been added.

        // Your code here
    }

    public void afterTextChanged(Editable s) {
        // Your code here
    }
});
The above example adds a TextWatcher to the specified EditText. The onTextChanged, beforeTextChanged and afterTextChanged methods will then be called any time the text in the EditText changes.
