android.graphics.Matrix

The android.graphics.Matrix class is a powerful tool for performing 2D transformations on an image. It can be used for scaling, translating, and skewing an image. This article provides an overview of the Matrix class and some examples of how it is used.

Overview

The Matrix class consists of two components: a 4x4 matrix and a float array. The 4x4 matrix is used to store the transformation information while the float array stores the four elements of the matrix. The Matrix class has numerous functions that allow manipulation of the elements of the matrix and the resulting transformation.

Examples

The following code examples show how to use the Matrix class to perform various transformations.

Scaling


          
           
Matrix matrix = new Matrix(); 
matrix.setScale(2.0f, 2.0f);

This example creates a new Matrix and sets the scale to 2.0f (doubling the size of the image). This transformation will be applied to any subsequent drawing operations.

Translating


          
           
matrix.setTranslate(100f, 100f);

This example sets up a translation of 100f in both x and y directions. This will move all subsequent drawing operations by 100f in both directions.

Skewing


          
           
matrix.setSkew(20f, 20f);

This example sets up a skew of 20f in both x and y directions. This will cause all subsequent drawing operations to be skewed in both directions at a 20f angle.

Applying Transformations


          
           
canvas.setMatrix(matrix);

To apply the transformations, the matrix needs to be set on the Canvas object. This example sets the matrix on the canvas, so that all subsequent drawing operations will be affected by the transformations.

Conclusion

The Matrix class is a powerful tool for manipulating images in 2D space. It is capable of scaling, translating, and skewing images, as well as combining multiple transformations together. This article has provided an overview of the Matrix class and some examples of its usage.
