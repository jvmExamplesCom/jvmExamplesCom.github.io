javax.servlet.ServletException Overview

javax.servlet.ServletException is an exception that is thrown by the server when a servlet is not able to fulfill a request.

The servlet exception includes many subclasses, each with their own specific conditions, such as IOException and FileNotFoundException. This article will provide an overview of the servlet exception and its subclasses and provide some code examples of how to handle the exceptions in a web application.

Subclasses

javax.servlet.ServletException has several subclasses that are used for different purposes:

  • IOException is thrown when an input/output error occurs.

  • FileNotFoundException is thrown when a file is not found.

  • IllegalStateException is thrown when an illegal state is encountered in the servlet.

  • UnavailableException is thrown when a servlet is either being redeployed or destroyed.

  • ServletException is thrown when an unexpected condition occurs in the servlet.
