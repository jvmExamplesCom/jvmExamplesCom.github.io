android.widget.EditText

The android.widget.EditText class is a subclass of the TextView class in Android. It is a user interface element used for entering and modifying text. It is also used to display text that is read-only. EditText objects can be configured to accept different input types, such as plain text, numbers, phone numbers, dates, etc.

EditText is typically used in conjunction with other widgets and views, such as an AlertDialog, ListView, or Spinner, in order to provide a more complete user interface experience. For example, an EditText object could be used in an AlertDialog to allow the user to enter a message before sending it.

Example Usage

The following code example shows how to create an EditText object and set its properties:


          
           
EditText myEditText = new EditText(this);

// Set the EditText's input type
myEditText.setInputType(InputType.TYPE_CLASS_NUMBER);

// Allow the user to enter multiple lines of text
myEditText.setSingleLine(false);

// Set the hint text
myEditText.setHint("Type your message here...");

Once an EditText is created and configured, it can be added to an Activity or Fragment using the 


          
           
addView()
method.

Listeners

EditText objects can be configured to listen for events, such as when the user enters or changes text. The following code example shows how to create a TextWatcher object and attach it to an EditText:


          
           
myEditText.addTextChangedListener(new TextWatcher() {
    @Override
    public void onTextChanged(CharSequence s, int start, int before, int count) {
        // Do something each time the text is changed...
    }

    @Override
    public void beforeTextChanged(CharSequence s, int start, int count, int after) {
        // Do something before the text is changed...
    }

    @Override
    public void afterTextChanged(Editable s) {
        // Do something after the text is changed...
    }
});

The TextWatcher object allows the developer to listen for events and respond accordingly.

The EditText class is an important part of creating user interfaces in Android. It is used to allow the user to enter and modify text, as well as to display text in a read-only manner.
