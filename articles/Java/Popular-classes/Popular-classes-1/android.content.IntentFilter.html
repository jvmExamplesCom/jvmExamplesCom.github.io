android.content.IntentFilter

The 

          
           
android.content.IntentFilter
class is used to express an Intent that an Android application can receive, along with the action it can perform. The IntentFilter class can be used for registering broadcast receivers for system events or for custom broadcast intents.

Creating an Intent Filter

An Intent filter can be created by instantiating a new IntentFilter object with a particular action:


          
           
IntentFilter intentFilter = new IntentFilter("my_action");

The action can be anything, but usually it is something like “ACTION_VIEW” or “ACTION_EDIT” etc. Once the IntentFilter has been instantiated, other filters can be added to it using the addAction() and addDataPath() methods:


          
           
intentFilter.addAction(Intent.ACTION_VIEW);
intentFilter.addDataPath("/my/data/path", PatternMatcher.PATTERN_LITERAL);

The addAction() method adds an action specification to the filter, while the addDataPath() method adds a data path specification - a URI scheme and authority (optional) and a type of pattern matcher. These methods can be iterated over multiple times to add multiple filters. Once the filters have been added, the filter can be registered with the Android system using the registerReceiver() method:


          
           
registerReceiver(myReceiver, intentFilter);

Handling Intents

Once the broadcast receiver has been registered, it will receive all broadcasts that match the filter criteria. The onReceive() method of the broadcast receiver will be called each time an intent is received, and it is up to the broadcast receiver to handle that intent. The following example shows the basic code for handling an intent:


          
           
public void onReceive(Context context, Intent intent){

    String action = intent.getAction();

    if (Intent.ACTION_VIEW.equals(action)) {
        // handle the view action
    } else if (Intent.ACTION_EDIT.equals(action)) {
        // handle the edit action
    }
}

In this example, the onReceive() method first gets the action of the intent, and then compares it against various actions defined in the IntentFilter. If the action matches one of the actions specified in the IntentFilter, the appropriate action is handled by the broadcast receiver.
