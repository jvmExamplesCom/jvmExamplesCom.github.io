Introduction to org.apache.http.client.methods.HttpPost

The class org.apache.http.client.methods.HttpPost provides an implementation of the HTTP POST method. This class can be used to send data, such as query strings or file contents, to a server effortless. It also allows for customization of the request headers, if needed. In this article, we will go over some of the basics and methods of this class, as well as look at some code examples. Let's get started!

Constructors

The class provides two constructors to create a new HttpPost object: 

          
           
HttpPost() // Constructs a new HttpPost with no URL.
HttpPost(String url) // Constructs a new HttpPost with the specified URL.

Methods

This class provides several methods to configure different parts of the request. These are some of the most important ones: 

          
           
setAddHeader(String name, String value) // Adds a header to the request.
setEntity(HttpEntity entity) // Sets the body of the request.
addParameters(HttpParams params) // Adds parameters to the request.
addRequestHeader(String name, String value) // Adds a request header to the request.

Example Usage

Now let's take a look at a code example of how to construct and make a request using the HttpPost class. 

          
           
HttpPost post = new HttpPost("http://example.com");
post.addHeader("Content-Type", "application/json");
post.addRequestHeader("Accept", "application/json");
post.setEntity(new StringEntity("{\"message\": \"Hello World!\"}"));
CloseableHttpClient client = HttpClients.createDefault();
CloseableHttpResponse response = client.execute(post);
System.out.println("Response code: " + response.getStatusLine().getStatusCode());
In this example, we first create a new HttpPost object with the specified URL, then add headers to tell the server what content type and accepted content types the client supports. Next, we set the content of the request to a JSON string containing a message. Finally, we create an instance of the CloseableHttpClient class, make the request, and print out the response code.
