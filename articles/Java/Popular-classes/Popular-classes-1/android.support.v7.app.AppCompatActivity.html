It is designed to be used as the base class when creating activities in order to ensure compatibility across devices.

Overview

AppCompatActivity is a subclass of Activity and provides a range of features and methods to help developers build applications that are both backward compatible and up-to-date with the latest Android versions. Features include support for ActionBar, Material Design themes, initialization of core features like the Toolbar and NavigationDrawer, and more. It also acts as a bridge between the older API support libraries and AppCompat Library.
