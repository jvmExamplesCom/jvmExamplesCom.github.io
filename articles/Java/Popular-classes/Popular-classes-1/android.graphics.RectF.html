Introduction to Android.graphics.RectF

Android.graphics.RectF is an Android class used to define and manipulate rectangular coordinates. RectF is part of the Android graphics library and is used for creating rectangles in android UI. Creating a RectF Object

Creating a RectF object requires four float values as parameters representing the four points (clockwise order) of the rectangle. For example,

float top = 20.0f;

RectF myRectangle = new RectF(left, top, right, bottom);
