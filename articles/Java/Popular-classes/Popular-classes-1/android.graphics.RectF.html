Introduction to Android.graphics.RectF

Android.graphics.RectF is an Android class used to define and manipulate rectangular coordinates. It stores four float coordinates that represent the top-left, top-right, bottom-left and bottom-right corners of the rectangle. RectF is part of the Android graphics library and is used for creating rectangles in android UI. Creating a RectF Object

Creating a RectF object requires four float values as parameters representing the four points (clockwise order) of the rectangle. For example,

float top = 20.0f;
float right = 30.0f;
float bottom = 40.0f;

RectF myRectangle = new RectF(left, top, right, bottom);
