Introduction to Android.graphics.RectF

Android.graphics.RectF is an Android class used to define and manipulate rectangular coordinates. Creating a RectF object requires four float values as parameters representing the four points (clockwise order) of the rectangle. For example,

float top = 20.0f;

RectF myRectangle = new RectF(left, top, right, bottom);
