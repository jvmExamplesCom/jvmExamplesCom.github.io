Android.Graphics.Path

Android.Graphics.Path provides operations for creating and manipulating vector-based paths. It is the backbone of all drawing operations done in Android. The Path class can be used to draw lines, rectangles, circles, oval, arcs, and any other shape.

Creating a Path

A Path object is created by calling 

          
           
new Path()
. This is often done when creating the Paint object that will be used to draw with the Path.

Path Methods

The Path class includes a variety of methods for creating, manipulating, and drawing paths.

moveTo()

The 

          
           
moveTo()
method moves the current position of the Path to a new location without adding a line. This is useful for setting the starting point of a path.

lineTo()

The 

          
           
lineTo()
method adds a line from the current position to the specified point.

addRect()

The 

          
           
addRect()
method adds a rectangle to the Path.

addCircle()

The 

          
           
addCircle()
method adds a circle to the Path.

addOval()

The 

          
           
addOval()
method adds an oval to the Path.

addArc()

The 

          
           
addArc()
method adds an arc to the Path.

close()

The 

          
           
close()
method closes the Path by adding a line from the current position to the initial position.

quadTo()

The 

          
           
quadTo()
method adds a quadratic Bézier curve to the Path.

cubicTo()

The 

          
           
cubicTo()
method adds a cubic Bézier curve to the Path.

Drawing with a Path

Once a Path has been created, it can be drawn using the 

          
           
drawPath()
method of the Canvas class. The 

          
           
drawPath()
method takes two parameters, a Path object and a Paint object. The Paint object defines the color, style, and other drawing attributes.

Example

The following example shows how to create and draw a simple path. 

          
           
Path path = new Path();
path.moveTo(50, 50);
path.lineTo(100, 100);
path.addRect(150, 150, 200, 200, Path.Direction.CW);

Paint paint = new Paint();
paint.setStyle(Paint.Style.STROKE);
paint.setColor(Color.RED);

Canvas canvas = getHolder().lockCanvas();
canvas.drawPath(path, paint);
getHolder().unlockCanvasAndPost(canvas);
This example creates a Path with three elements: a line, a rectangle, and a closed path. The Paint object sets the style to a stroke and color to red. Finally, the Path is drawn on the canvas.

Conclusion

Android.Graphics.Path provides a powerful and flexible way to create and manipulate vector-based paths. With the various methods provided, it is possible to create complex shapes and drawings.
