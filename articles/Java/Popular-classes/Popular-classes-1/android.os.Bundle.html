android.os.Bundle

Android bundles are a type of container used to store and pass data between Android components. Bundles are usually passed to activities, services, or broadcast receivers as an argument in their onCreate() methods, as well as to Fragment constructors.

Creating a Bundle

A Bundle can be created and populated with data using the 

putXXX()
methods, where XXX is the type of the data put in the bundle. Some of the put methods include: 
  • 
            
             
putString(String key, String value)

            
  • 
            
             
putInt(String key, int value)

            
  • 
            
             
putLong(String key, long value)

            
  • 
            
             
putBoolean(String key, boolean value)

For example, the following code creates a Bundle and adds some strings to it:


          
           
Bundle myBundle = new Bundle();
myBundle.putString("name", "John Smith");
myBundle.putInt("age", 28);
myBundle.putLong("phone", 2125551212L);
myBundle.putBoolean("isMarried", false);

Retrieving Data from Bundle

To retrieve data from a Bundle, you can use the corresponding 

getXXX()
method. For example, to get the string value stored in the above Bundle, you can do the following: 

          
           
String name = myBundle.getString("name");

Other getXXX() methods for retrieving data from Bundle include:

  • 
            
             
int age = myBundle.getInt("age");

            
  • 
            
             
long phone = myBundle.getLong("phone");

            
  • 
            
             
boolean isMarried = myBundle.getBoolean("isMarried");

Passing Bundle Between Components

Once you have created a Bundle, you can pass it as an argument in calls to other components (activities, broadcast receivers, services, etc.).

For example, you can pass the previously created Bundle to an Activity using the startActivity() method:


          
           
Intent intent = new Intent(this, MyActivity.class);
intent.putExtras(myBundle);
startActivity(intent);

The Bundle object can then be retrieved in the Activity's onCreate() method:


          
           
@Override
protected void onCreate(Bundle savedInstanceState) {
    super.onCreate(savedInstanceState);
    Bundle data = getIntent().getExtras();
    ...
}

Saving Bundle States

Bundle objects are often used to save the state information of an activity or a fragment. You can use the onSaveInstanceState() method to save the Bundle:


          
           
@Override
public void onSaveInstanceState(Bundle outState) {
    outState.putString("name", "John Smith");
    outState.putInt("age", 28);
    outState.putLong("phone", 2125551212L);
    outState.putBoolean("isMarried", false);
    super.onSaveInstanceState(outState);
}

The Bundle can then be retrieved in the onCreate() method:


          
           
@Override
protected void onCreate(Bundle savedInstanceState) {
    super.onCreate(savedInstanceState);
    if (savedInstanceState != null) {
        String name = savedInstanceState.getString("name");
        int age = savedInstanceState.getInt("age");
        long phone = savedInstanceState.getLong("phone");
        boolean isMarried = savedInstanceState.getBoolean("isMarried");
    }
}
