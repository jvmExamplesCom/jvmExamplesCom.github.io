Java.lang.annotation.ElementType

Java.lang.annotation.ElementType is an enum type of Java that represents the possible target types for an annotation.

It consists of the following constants:
  • TYPE
  • FIELD
  • METHOD
  • PARAMETER
  • CONSTRUCTOR
  • LOCAL_VARIABLE
  • ANNOTATION_TYPE
  • PACKAGE
  • TYPE_PARAMETER
  • TYPE_USE

Examples

The ElementType enum can be used to indicate the type of a particular element that an annotation is attached to.

Example 1

In this example, we will use the ElementType enum to annotate a method.

@Target(ElementType.METHOD)
public @interface TestAnnotation {

}
