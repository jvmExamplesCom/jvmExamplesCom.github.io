org.apache.commons.logging.Log is a Java API which provides a generic, type-safe logging API that can be used to log messages to various logging frameworks like Log4j and Java Util Logging.

Using Log, the user can create loggers with different log levels and destinations, such as the console, files, or databases. Loggers created by Log have a set of methods to log messages, depending on the log level. For example, to log an error message, the user would use the 

error(String msg)
method. 

          
           
Log logger = LogFactory.getLog("My Logger");
logger.error("Error Message");

logger.error("An %s occurred: %s", errorType, errorMessage);
