android.widget.ImageButton

An android.widget.ImageButton is an Android Widget that provides an Image-based button UI element. It can be used to display an image, and when clicked it can perform an action. The ImageButton widget can be defined in XML within a layout file, or programmatically in code.

Defining an ImageButton in XML

The ImageButton can be defined in XML using the 

          
           
<ImageButton>
tag. This tag takes several attributes, the most important of which are 

          
           
android:src
, which defines the drawable or mipmap resource that will be displayed in the ImageButton, and 

          
           
android:onClick
, which defines a method in the Activity class that will be called when the ImageButton is clicked. Here is an example of an ImageButton defined in XML that displays a drawable stored in the app's 

          
           
res/drawable
folder, and calling the 

          
           
onImageButtonClicked()
method upon being clicked: 

          
           
<ImageButton
    android:id="@+id/image_button"
    android:layout_width="wrap_content"
    android:layout_height="wrap_content"
    android:src="@drawable/image_resource"
    android:onClick="onImageButtonClicked" />

Defining an ImageButton Programmatically

The ImageButton can also be defined programmatically in code. To do this, you first need to create an instance of the ImageButton using the 

          
           
new ImageButton()
constructor, and then set its properties using the various setters provided. Here is an example of creating an ImageButton in code and setting its drawable, width, height, and click listener: 

          
           
ImageButton imageButton = new ImageButton(this);
imageButton.setImageDrawable(getResources().getDrawable(R.drawable.image_resource));
imageButton.setLayoutParams(new LinearLayout.LayoutParams(ViewGroup.LayoutParams.WRAP_CONTENT, ViewGroup.LayoutParams.WRAP_CONTENT));
imageButton.setOnClickListener(new View.OnClickListener() {
    @Override
    public void onClick(View view) {
        onImageButtonClicked();
    }
});

Handling Clicks

When an ImageButton is clicked, the method defined in its 

          
           
android:onClick
attribute (or its 

          
           
setOnClickListener()
method if it is defined programmatically) will be called. This method should accept a 

          
           
View
as its only argument, and return void. Here is an example of such a method: 

          
           
public void onImageButtonClicked(View view) {
    // Handle ImageButton click here
}
In this method, you can handle the click event however you need. You can e.g. execute some code, show a Toast message, start a new Activity, etc.

Conclusion

The android.widget.ImageButton is an Android Widget that provides an Image-based button UI element. It can be defined in XML, or programmatically in code, and its click event can be handled through the 

          
           
android:onClick
attribute or the 

          
           
setOnClickListener()
