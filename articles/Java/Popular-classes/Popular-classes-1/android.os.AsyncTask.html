android.os.AsyncTask

AsyncTask is an easy to use method for performing background tasks in Android. It allows for easy creation of long running operations, such as downloads, computations and database access that can run in the background without blocking the UI thread. AsyncTask is designed to be used in short operations, usually 15 seconds or less. Longer operations should be handled with a Service.

basics

AsyncTasks are usually defined as a subclass of android.os.AsyncTask, and they must override at least one method, doInBackground(). An example of a basic AyncTask might look like this:


          
           
class MyAsyncTask extends AsyncTask<Params, Progress, Result> {

    protected void doInBackground(Params... params) {
        // perform background operations here
    }
}

An AsyncTask must be instantiated before use. This is done with:


          
           
MyAsyncTask myTask = new MyAsyncTask();

Once instantiated, the task can be executed using execute(), passing in any class or interface (or an array of them) as arguments. For example, to pass two Strings:


          
           
myTask.execute("string 1", "string 2");

Getting Results

To get the results of the task, Override the onPostExecute() method. This will receive the result of the doInBackground() task.


          
           
class MyAsyncTask extends AsyncTask<Params, Progress, Result> {

    protected void doInBackground(Params... params) {
        // perform background operations here
    }

    protected void onPostExecute(Result result) {
        // get the result of the background operation 
    }
}

The result of the doInBackground() method will be passed as an argument to onPostExecute(). If there is no result, then the argument will be null.

Canceling Tasks

AsyncTasks can be cancelled with the cancel() method. This will cause the task to finish immediately and calls onCancelled() instead of onPostExecute(). Note that if the task is already running when cancel() is called, it may not be able to stop immediately. To determine if a task has been cancelled, you can check if isCancelled() returns true.


          
           
class MyAsyncTask extends AsyncTask<Params, Progress, Result> {

    protected void doInBackground(Params... params) {
        if (isCancelled()) {
            return;
        }
        // perform background operations here
    }

    protected void onCancelled() {
        // perform clean up operations 
    }
}
