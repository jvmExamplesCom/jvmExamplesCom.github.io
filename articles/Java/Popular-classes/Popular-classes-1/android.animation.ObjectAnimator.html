android.animation.ObjectAnimator is a subclass of the Android Animator class. It is used to animate arbitrary properties on target objects. It facilitates creating custom animations for any data type by providing the ability to modify properties of target objects using reflection. This makes ObjectAnimator an extremely versatile animation tool.

Creating ObjectAnimator Objects

ofFloat()

ofInt()

ObjectAnimator animator = ObjectAnimator.ofFloat(imageView, "translationX", 100f);

The first parameter is the target object, in this case an ImageView. The second parameter is the property name, which can be any valid JavaBean property. The third parameter is the desired new value for the property.
