What is Java.net.HttpURLConnection?

Java.net.HttpURLConnection is a package that is part of the Java networking package that facilitates communications over HTTP. It is a low-level, class-based API designed for developers that provides functionality for remote communication and manipulation of client-server data transfers.

Examples of Java.net.HttpURLConnection

The following examples show how to use Java.net.HttpURLConnection to communicate over HTTP.


          
           
URL url = new URL("http://www.example.com");
HttpURLConnection conn = (HttpURLConnection) url.openConnection();
conn.setRequestMethod("GET");
conn.connect();

if (conn.getResponseCode() == 200) {
    // do something with the response
}

To send data back to the server, the following example shows how to construct an HTTP POST request.


          
           
URL url = new URL("http://www.example.com");
HttpURLConnection conn = (HttpURLConnection) url.openConnection();
conn.setRequestMethod("POST");
conn.setDoOutput(true);

OutputStream os = conn.getOutputStream();

os.close();

    // do something with the response

To set a query parameter on a connection, use the setRequestProperty() method as shown in the following example:


          
           
conn.setRequestProperty("parameterName", "value");

To add a custom header, use the setRequestProperty() method as follows:
