javax.imageio.ImageIO

javax.imageio.ImageIO is a Java library that provides an API for reading and writing images in various formats. It is mainly used to work with raster data, but can also be used to work with vector graphics. The package includes support for PNG, JPEG, BMP, and GIF formats.

Reading an Image

The first step is to read an image from a file. This can be done using the ImageIO.read() method. The following example shows how to read an image in PNG format:


          
           
BufferedImage image = ImageIO.read(new File("image.png"));

The read() method returns a BufferedImage object which represents the image in memory. The image can then be modified or used for further processing.

Writing an Image

Once an image has been read, it can be written to another file. This is done using the ImageIO.write() method. The following example shows how to write an image in PNG format:


          
           
ImageIO.write(image, "png", new File("image.png"));

The write() method takes three arguments - an image, the output format, and the output file. The list of available output formats can be found in the imageio.ImageIO class.

Converting an Image

ImageIO.write() can also be used to convert an image from one format to another. This is done by passing the desired output format as the second argument. The following example shows how to convert an image from PNG to JPEG:


          
           
ImageIO.write(image, "jpeg", new File("image.jpg"));
