Introduction to com.google.common.collect.sets

The 

          
           
com.google.common.collect.Sets
class represents a collection of elements in a set structure. It provides various utility methods for manipulating the contents of a set, such as adding or removing elements, creating combinations of sets, and so on. This article provides an overview of the different methods available in the Sets class and shows how they can be used in code examples.

Creating a Set

A Set can be created from any Iterable object, such as a List. The following example creates a Set from a list of strings:


          
           
List<String> list = Lists.newArrayList("hello", "world", "foo");
Set<String> set = Sets.newHashSet(list);

This example creates a Set with three elements: "hello", "world" and "foo".

Adding Elements to a Set

The 
add()
method can be used to add an element to a Set: 

          
           
Set<String> set = Sets.newHashSet();
set.add("hello");
set.add("world");

This example adds two elements, "hello" and "world", to the set.

Removing Elements from a Set

The 
remove()
method can be used to remove an element from a Set: 

          
           
Set<String> set = Sets.newHashSet("hello", "world");
// Remove the element
set.remove("hello");

This example removes the element "hello" from the set.

Combining Sets

The 
union()
, 
intersection()
, and 
difference()
methods can be used to combine sets in various ways. For example, to create the union of two sets, use the 
union()
method: 

          
           
Set<String> set1 = Sets.newHashSet("a", "b");
Set<String> set2 = Sets.newHashSet("b", "c");
Set<String> result = Sets.union(set1, set2);

This example creates a new set containing the elements from both sets (i.e. "a", "b" and "c"). The same technique can be used to create the intersection or difference of two sets.

Checking for Set Membership

The 
contains()
method can be used to check if an element is in a Set: 

          
           
Set<String> set = Sets.newHashSet("hello", "world");
boolean containsHello = set.contains("hello");

This example checks if the set contains the element "hello" and stores the result in a boolean variable.

Iterating Over a Set

The Sets class provides several methods for iterating over a Set. The most common method is the 

forEach()
method: 

          
           
Set<String> set = Sets.newHashSet("hello", "world");
set.forEach(e -> System.out.println(e));
This example prints out each element of the Set.
