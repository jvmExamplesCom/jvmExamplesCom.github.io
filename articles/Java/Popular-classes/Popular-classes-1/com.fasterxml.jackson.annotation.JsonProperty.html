Introduction to com.fasterxml.jackson.annotation.JsonProperty

The 

          
           
com.fasterxml.jackson.annotation.JsonProperty
annotation is used in Jackson to indicate a field or method parameter should be bound to a property in a JSON object. @JsonProperty
, its properties, and some examples of its implementation.

Using @JsonProperty

Using 

          
           
@JsonProperty
is easy. All you need to do is add the annotation to the field or method parameter you want to bind to a property in a JSON object: 

          
           
@JsonProperty
private String name;

value
attribute of the annotation:
