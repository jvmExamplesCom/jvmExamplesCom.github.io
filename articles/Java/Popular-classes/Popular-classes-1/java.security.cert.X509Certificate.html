Java Security Certificate X509Certificate

The 

          
           
java.security.cert.X509Certificate
class is a standard interface for representing an X.509 certificate, which is a widely used format for public key certificates. This interface provides access to its members, such as the subject name, issuer name, serial number of the certificate, and the validity period.

Creating an X509Certificate Object

Creating an instance of an X509Certificate is relatively easy. You just need to pass in the values into the constructor like this:


          
           
X509Certificate cert = new X509Certificate("/path/to/certificate", "password");

Once you have created the instance, you can access its properties using dedicated methods, such as 

getSubjectDN()
, 
getIssuerDN()
, 
getSerialNumber()
etc.

Checking if an X509Certificate is Valid

You can check if an X509Certificate is valid by using the 

checkValidity()
method: 

          
           
boolean valid = cert.checkValidity();

This method will return true if the certificate is currently valid, and false otherwise. This method also throws an exception if the certificate is invalid.

Getting the Public Key

You can access the public key associated with the certificate by using the 

getPublicKey()
method: 

          
           
PublicKey publicKey = cert.getPublicKey();

Once you have the public key, you can use it to encrypt data or verify digital signatures.

Serializing an X509Certificate

You can serialize an X509Certificate instance to a byte array by using the 

getEncoded()
method. This method returns a byte array containing the encoded certificate: 

          
           
byte[] encodedCert = cert.getEncoded();
