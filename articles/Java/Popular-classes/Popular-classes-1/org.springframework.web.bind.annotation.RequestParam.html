org.springframework.web.bind.annotation.RequestParam

The 
@RequestParam
annotation is used in Spring MVC framework for reading the request parameters from the web request. It has various attributes that can be used for binding the request parameters to that method argument.

Example 1


          
           
@RequestMapping(value="/vehicles")
public void vehicles(@RequestParam(value="pageNum", required=false, defaultValue="1") int pageNum){
   // More code here
}
In the above example, we have a method 
vehicles
with a parameter of type 
int
. This parameter is annotated with the 
@RequestParam
annotation.

The value attribute is used to specify the name of the request parameter. The required attribute specifies whether this parameter is mandatory or not. If it is set to true, then the request needs to have the parameter present. The defaultValue attribute is used to specify the default value for the parameter if the request does not have the parameter or if the parameter is empty.

Example 2


          
           
@RequestMapping(value="/vehicles")
public void vehicles(@RequestParam(value="pageNum") int pageNum){
   // More code here
}

In this example, the required and defaultValue attributes are not specified. This means that the 

pageNum
parameter is required and there is no default value.

Example 3


          
           
@RequestMapping(value="/vehicles")
public void vehicles(@RequestParam Map<String,String> params){
   // More code here
}
In this example, we have a 
Map
as the method argument. This can be used to read all the request parameters in one go. The map keys will be the parameter names and the values will be the parameter values.

Example 4

   // More code here
}
In this example, we have a 

          
           
MultiValueMap<String, String>
as the method argument. This can be used to read all the request parameters in one go. The map keys will be the parameter names and the values will be the parameter values. In this case, the same parameter name can have multiple values which will be stored as a list.
