Java Text MessageFormat

The Java Text MessageFormat provides a way to create and internationalize messages based on empty argument objects. This allows for the construction of strings that are customized for various locales. It provides a consistent format to use when constructing messages in any language.

Code Example 1: Simple MessageFormat with String Arguments


          
           
String pattern = "Hello {0}!";
Object[] args = { "World" };
MessageFormat mf = new MessageFormat(pattern);
String message = mf.format(args);
System.out.println(message); // Hello World!

Code Example 2: MessageFormat with Date Arguments


          
           
String pattern = "Today is {0,date,short}";
Object[] args = { new Date() };
MessageFormat mf = new MessageFormat(pattern);
String message = mf.format(args);
System.out.println(message); // Today is 6/25/20

Code Example 3: MessageFormat with Number Arguments


          
           
String pattern = "You have {0,number,integer} messages.";
Object[] args = { 10 };
MessageFormat mf = new MessageFormat(pattern);
String message = mf.format(args);
System.out.println(message); // You have 10 messages.

Code Example 4: MessageFormat with Choice Arguments


          
           
String pattern = "You have {0,choice,0#no messages|1#one message|1<{0,number, integer} messages}.";
Object[] args = { 10 };
MessageFormat mf = new MessageFormat(pattern);
String message = mf.format(args);
System.out.println(message); // You have 10 messages.
