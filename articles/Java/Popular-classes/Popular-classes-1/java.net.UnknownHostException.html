java.net.UnknownHostException

java.net.UnknownHostException is an exception that occurs when a computer attempts to resolve an IP address from a hostname and fails. This exception can occur due to a variety of reasons including network errors, DNS lookup failures, or incorrect hostnames.

Code Examples

Here are some code examples that demonstrate how to use java.net.UnknownHostException.


          
           
try {
    InetAddress ip = InetAddress.getByName("www.example.com");
} catch (UnknownHostException e) {
    System.out.println("Unable to resolve address of www.example.com");
}

The above code example attempts to resolve the hostname ‘www.example.com’, using the InetAddress.getByName() method. If the hostname cannot be resolved, a java.net.UnknownHostException will be thrown.


          
           
try {
    InetAddress ip = InetAddress.getByName("example.com");
    System.out.println("IP Address: "+ip.toString());
} catch (UnknownHostException e) {
    System.err.println("Cannot resolve address of example.com");
}

This code example is similar to the one above, however this time, if the hostname is resolved successfully, the IP address is printed out to the console.

    InetAddress[] ips = InetAddress.getAllByName("example.com");
    for (InetAddress addr : ips) {
        System.out.println("IP Address: "+addr.toString());
    }

    System.err.println("Cannot resolve address of example.com");

This code example attempts to resolve the hostname ‘example.com’ and returns an array of all the associated IP addresses. If the resolution fails, then a java.net.UnknownHostException will be thrown.
