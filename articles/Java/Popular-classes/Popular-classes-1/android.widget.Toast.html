android.widget.Toast

The Toast class is a part of Android's widget toolkit and provides a way to show notifications to the user. It can be used to display general information, error messages, warning messages, and more. Toasts are displayed on top of the activity and can be swiped off-screen if desired. This article will discuss the android.widget.Toast class and provide some code examples.

Creating a Toast

Creating a Toast object is a simple two-step process. First, you'll need to use the makeText() method to create a Toast instance:


          
           
Toast toast = Toast.makeText(context, "Hello!", Toast.LENGTH_SHORT);

The makeText() method takes three parameters: a context, a string message, and a duration for how long the Toast should be visible on screen. The context is usually the application's context or an activity context. The message can be any string, such as a string resource ID. The duration parameter can be either Toast.LENGTH_LONG or Toast.LENGTH_SHORT to indicate how long the Toast should stay on the screen.

Once the Toast instance is created, it can be shown on the screen by calling the show() method:


          
           
toast.show();

The Toast will now be visible on the screen until it is manually removed or until the duration (which was specified when creating the Toast) has expired.

Customizing a Toast

In some cases, you may want to customize a Toast beyond just the message and duration. For example, you may want to set the background color or position of the Toast. This can be done by getting the View from the Toast object and making modifications to it:


          
           
// Get the View from the Toast object
View view = toast.getView();

// Set the background color 
view.setBackgroundColor(Color.RED);

// Position the Toast
toast.setGravity(Gravity.TOP|Gravity.LEFT, xOffset, yOffset);

The View object can be used to modify the background color, position, and other properties of the Toast. Once the View object has been modified, the show() method can be called again to show the updated Toast.

Canceling a Toast

Toast objects will automatically be canceled once the duration has expired, but it is also possible to manually cancel a Toast before it expires. This can be done with the cancel() method:


          
           
toast.cancel();

This will immediately remove the Toast from the screen and free up any resources that were associated with it.

Conclusion

The android.widget.Toast class provides a simple way to show notifications to the user. It can be used to display general information, error messages, warnings, and more. Creating a Toast is a two-step process involving the makeText() method and the show() method. Additionally, it is possible to customize a Toast beyond its message and duration, as well as manually cancel a Toast before it expires.
