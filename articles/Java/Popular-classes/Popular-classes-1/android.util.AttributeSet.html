Introduction to android.util.AttributeSet

android.util.AttributeSet provides a mechanism for retrieving the attributes of an XML element. It is a set of attribute/value pairs, in which an attribute is identified by the namespace and the local name of the attribute.

Advanced Usage of AttributeSet

AttributeSet allows for more advanced usages than just retrieving attributes. For example, it can be used to parse out and retrieve style attributes. Style attributes are special attributes that can be declared inside XML elements and are applied as a group to the target element.

The TypedArray is initialized by passing in the AttributeSet and an array of resource identifiers of all the style attributes supported by the view. Conclusion

android.util.AttributeSet provides a convenient way for retrieving the attributes of an XML element. It can be used with XML pull parser to retrieve the attributes of an element one by one, or with the TypedArray to retrieve and apply style attributes to a view.
