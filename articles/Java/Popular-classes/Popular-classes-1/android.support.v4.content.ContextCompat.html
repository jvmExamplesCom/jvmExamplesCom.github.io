android.support.v4.content.ContextCompat

The android.support.v4.content.ContextCompat class provides support for accessing features in Context class which was introduced in Android API level 23 or above. This class helps in resolving issues related to accessing features which were only available since API level 23 or above from the app code which might be running on devices with earlier versions of the Android platform. By using this class, developers can check for the existence of specific features within the app’s runtime environment before actually using them, so that the app does not crash due to an unavailable feature. This class also provides simple methods for checking if a permission is granted or not. It also provides methods to acquire permissions at runtime in order to use certain features if required.
