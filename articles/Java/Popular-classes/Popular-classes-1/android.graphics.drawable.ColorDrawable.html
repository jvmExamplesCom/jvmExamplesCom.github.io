android.graphics.drawable.ColorDrawable

Android ColorDrawable provides a way to draw a color or an array of colors. It can be used as the background of Views or when drawing custom shapes. The class supports generating a random color, coloring a specific color, and translating between number and color representations.

Creating ColorDrawable

To create a ColorDrawable you can use the constructor:


          
           
ColorDrawable colorDrawable = new ColorDrawable(color);
Where 
color
is an int representing the color. Here are a few ways to get the color: 

          
           
// get a random color
int randomColor = Color.random();

// get a color from resources
int redColor = ResourcesCompat.getColor(context.getResources(), R.color.red, null);

// get a color from which Hex value
int blueColor = Color.parseColor("#0000FF");

Access and Manipulate Colors

Once you have a ColorDrawable object you can access the color with the getColor() method:


          
           
int color = colorDrawable.getColor();

You can also set a new color using the setColor() method:


          
           
colorDrawable.setColor(color);

You can also manipulate the color by adding or subtracting to the RGB values:


          
           
int newColor = Color.addRGB(color, rDelta, gDelta, bDelta);
Here 
rDelta
, 
gDelta
, and 
bDelta
are the amounts by which to adjust the Red, Green, and Blue values respectively.

Drawing Custom Shapes

You can also use a ColorDrawable to draw custom shapes. To do so you need to override the draw() method:


          
           
colorDrawable.draw(canvas);
Where 
canvas
is a Canvas object. This allows you to draw complex shapes using the ColorDrawable.
