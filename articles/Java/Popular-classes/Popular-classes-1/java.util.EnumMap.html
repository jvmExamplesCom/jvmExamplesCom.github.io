Introduction to Java.util.EnumMap

Java.util.EnumMap is a specialized Map implementation designed to store and retrieve key-value pairs associated with an enumerated type, or enum. It allows for efficient retrieval of the underlying enum values and provides better performance than the standard HashMap when used with enum keys. This article will provide an overview of how to use EnumMap, along with code examples and usage scenarios.

Creating an EnumMap

Creating an EnumMap is done by passing in the class of the Enumeration you're mapping to as a parameter: 

          
           
EnumMap<Fruit, Integer> enumMap = new EnumMap<>(Fruit.class);
In this case, Fruit is a user-defined enum class with associated values.

Adding Entries to an EnumMap

Adding entries to an EnumMap is done using the put() method: 

          
           
enumMap.put(Fruit.APPLE, 2);
enumMap.put(Fruit.ORANGE, 1);
enumMap.put(Fruit.BANANA, 3);
The keys in an EnumMap must be enum constants, and the values can be any type.

Retrieving Values from an EnumMap

Values can be retrieved from an EnumMap using the get() method: 

          
           
int appleValue = enumMap.get(Fruit.APPLE); // returns 2
If the requested key does not exist in the EnumMap, get() returns null.

Iterating Over an EnumMap

Using the entrySet() method, the contents of an EnumMap can be iterated over using the for-each loop construct: 

          
           
for (Map.Entry<Fruit, Integer> entry : enumMap.entrySet()) {
   System.out.println(entry.getKey() + ": " + entry.getValue());
}
The output of this code would be: 

          
           
Apple: 2
Orange: 1
Banana: 3

Merging Two EnumMaps

It's possible to merge two EnumMaps using the putAll() method: 

          
           
EnumMap<Fruit, Integer> secondEnumMap = new EnumMap<>(Fruit.class);
secondEnumMap.put(Fruit.PEACH, 4);
secondEnumMap.put(Fruit.STRAWBERRY, 5);

enumMap.putAll(secondEnumMap);
After merging, the contents of enumMap will be as follows: 

          
           
Apple: 2
Orange: 1
Banana: 3
Peach: 4
Strawberry: 5

Removing Entries from an EnumMap

Entries can be removed from an EnumMap using the remove() method: 

          
           
enumMap.remove(Fruit.ORANGE);
This will remove the Orange key-value pair from the map.

Conclusion

In conclusion, EnumMap is a useful Map implementation for storing key-value pairs associated with an enum. It provides better performance than the standard HashMap and its use is recommended in cases where the keys are enum constants. This article has provided an overview of how to use EnumMap, along with code examples and usage scenarios.
