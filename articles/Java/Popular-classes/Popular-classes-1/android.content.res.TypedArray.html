Using android.content.res.TypedArray in Java

android.content.res.TypedArray is part of the Android SDK and provides a way to parse and obtain attribute values from XML styleables. This class is especially powerful as it can handle all kinds of data types and attributes in one call. This makes using TypedArrays much more efficient than manual parsing, and makes it easier to code with.

Defining a TypedArray

In order to use a TypedArray, you must first define it with the attributes that you wish to parse. This can be done either in the application's AndroidManifest.xml file, or directly in the code. The example below shows how to define a TypedArray in the manifest:


          
           
<typed-array 
    xmlns:android="http://schemas.android.com/apk/res/android"
    android:name="my_typed_array">

    <item android:name="string_value" android:type="string" />
    <item android:name="int_value" android:type="integer" />
    <item android:name="float_value" android:type="float" />
    ...
</typed-array>

Once the TypedArray is defined, it can be used in your code like this:


          
           
TypedArray typedArray = getResources().obtainTypedArray(R.array.my_typed_array);
String stringValue = typedArray.getString(0);
int intValue = typedArray.getInt(1, 0);
float floatValue = typedArray.getFloat(2, 0.0f);
...

// Be sure to recycle the resources when you're done
typedArray.recycle();

The above code will parse the values from XML and assign them to variables "stringValue", "intValue" and "floatValue". Note that the values are parsed from the XML in the same order they are written in the TypedArray definition.

Accessing Custom Attributes

It is also possible to access custom attributes using TypedArray. This requires a bit more code, but still fairly straightforward. The example below shows how to define a custom attribute and access it using TypedArray:


          
           
// Define the custom attribute in res/values/attrs.xml
<?xml version="1.0" encoding="utf-8"?>
<resources>
  <declare-styleable name="MyStyles">
    <attr name="my_custom_attribute" format="string" />
  </declare-styleable>
</resources>

// Access the attribute value from code
TypedArray typedArray = getResources().obtainTypedArray(R.styleable.MyStyles);
String myCustomAttributeValue = typedArray.getString(R.styleable.MyStyles_my_custom_attribute);
...

// Be sure to recycle the resources when you're done
typedArray.recycle();

This code will parse the custom attribute "my_custom_attribute" from XML, and assign it to the String variable "myCustomAttributeValue".
