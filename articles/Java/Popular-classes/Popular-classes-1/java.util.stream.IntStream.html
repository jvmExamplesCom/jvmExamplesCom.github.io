Java.Util.Stream.IntStream

Java.util.stream.IntStream is a powerful utility for performing operations on sequences of primitive int-valued elements. It provides a rich and flexible set of intermediate and terminal operations such as mapping, filtering, reducing, sorting and many more. IntStream is a subclass of Stream, so all the operations that apply to Stream also apply to IntStream.

Creating IntStreams

There are several ways to create IntStreams. One way is by using static methods of IntStream class:


          
           
IntStream emptyStream = IntStream.empty(); 
IntStream oneElementStream = IntStream.of(10); 
IntStream tenElementsStream = IntStream.range(0, 10); 
IntStream betweenElementsStream = IntStream.rangeClosed(0, 10);

int[] array = {1, 2, 3}; 
IntStream streamOfArray = Arrays.stream(array);

Intermediate Operations

Intermediate operations are operations that return the same type of IntStream they are called on. Here are some examples of available intermediate operations:


          
           
IntStream mapStream = streamOfArray.map(x -> x * x);

IntStream distinctStream = streamOfArray.distinct();

Parallel Streams

Parallel streams are streams that use multiple threads for execution of operations. Creating a parallel stream is easy:


          
           
IntStream parallelStream = streamOfArray.parallel();

Using parallel streams may improve performance but it can also introduce race conditions into the code. It is recommended to use them with caution.
