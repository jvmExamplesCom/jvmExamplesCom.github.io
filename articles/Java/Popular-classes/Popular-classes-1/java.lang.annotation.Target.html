java.lang.annotation.Target

The java.lang.annotation.Target annotation is used to indicate the kinds of program elements on which an annotation type is applicable. It belongs to the java.lang.annotation package.

It is a meta-annotation; therefore, it can be applied to other annotations.It has no preventions with any classes.

Syntax of @Target Annotation


          
           
@Target(value={ElementType.ANNOTATION_TYPE, ElementType.CONSTRUCTOR, ElementType.FIELD, ElementType.LOCAL_VARIABLE, ElementType.METHOD, ElementType.PACKAGE, ElementType.PARAMETER, ElementType.TYPE})

ElementTypes of @Target annotation

The ElementType enum provides constants to specify the type of target. The values are as follows:

  • ANNOTATION_TYPE: Used to annotate another annotation.
  • CONSTRUCTOR: Used to annotate constructor.
  • FIELD: Used to annotate fields.
  • LOCAL_VARIABLE: Used to annotate local variables.
  • METHOD: Used to annotate method.
  • PACKAGE: Used to annotate packages.
  • PARAMETER: Used to annotate parameters.
  • TYPE: Used to annotate class, interface.

Example of @Target Annotation


          
           
//Annotation with @Target
@Target(value = { ElementType.METHOD, ElementType.TYPE })
@Retention(RetentionPolicy.RUNTIME)
public @interface MethodInfo {
   String author() default "Yogesh";
   String date();
   int revision() default 1;
   String comments();
}

In the above example , MethodInfo is a custom annotation that is annotated with @Target annotation.It specifies that this annotation can be applied over methods and classes.
