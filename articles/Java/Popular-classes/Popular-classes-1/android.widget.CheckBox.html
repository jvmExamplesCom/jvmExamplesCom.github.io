Android.Widget.CheckBox

A CheckBox is a type of two-state button that can be either checked or unchecked. Android provides a CheckBox class which inherits the CompoundButton class. It is used to get a boolean value from the user.

Syntax


          
           
checkBox = (CheckBox) findViewById(R.id.checkBox);

Description

A CheckBox is a type of two-state button that can be either checked or unchecked. The state of the CheckBox can be set either in the XML layout file or programmatically in the activity file.

Attributes

The following is a list of attributes available for this view: 
  • android:button
    : Defines a drawable that will be used as the background of the CheckBox.
  • android:checked
    : Defines the initial state of the CheckBox.
  • android:clickable
    : Defines whether the CheckBox is clickable.
  • android:text
    : Defines the text shown on the CheckBox.

Example

The following example demonstrates how to use a CheckBox in an Android application: 

          
           
<?xml version="1.0" encoding="utf-8"?>
<LinearLayout xmlns:android="http://schemas.android.com/apk/res/android"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    android:orientation="vertical" >

    <CheckBox
        android:id="@+id/checkBox"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:checked="true"
        android:clickable="true"
        android:text="Enable CheckBox" />

</LinearLayout>

Methods

The following is a list of methods available for this view: 
  • isChecked()
    : Returns true if the CheckBox is checked.
  • setChecked()
    : Sets the checked state of the CheckBox.
  • toggle()
    : Toggles the checked state of the CheckBox.
  • 
            
             
setOnCheckedChangeListener()
    : Registers a callback to be invoked when the checked state of the CheckBox changes.

Example Usage

The following example demonstrates how to use a CheckBox in an Android Activity: 

          
           
public class MainActivity extends AppCompatActivity {

    private CheckBox checkBox;

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);

        checkBox = (CheckBox) findViewById(R.id.checkbox);

        // Set the initial state of the CheckBox
        checkBox.setChecked(true);

        checkBox.setOnCheckedChangeListener(new CompoundButton.OnCheckedChangeListener() {
            @Override
            public void onCheckedChanged(CompoundButton buttonView, boolean isChecked) {
                if (isChecked) {
                    Log.d("CheckBox", "Checkbox is checked");
                } else {
                    Log.d("CheckBox", "Checkbox is unchecked");
                }
            }
        });

    }

}
