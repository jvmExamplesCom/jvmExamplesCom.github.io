Introduction to java.io.Reader

Java.io.Reader is an abstract class that provides methods for reading character data from a source, such as a file. It is the base class for classes which reads raw bytes stream of character data. Java.io.Reader class declares read and close methods and provides implementation of these methods. The following example demonstrates how to useReader to read character data from a file: 

          
           
FileReader fr = new FileReader("file.txt");
int c;
while ((c = fr.read()) != -1) {
   // Process character
}
fr.close();
The above code snippet reads character data from "file.txt". Once a character is read, a value will be stored in the variable "c". Once all characters are read, the read method will return a -1 signifying the end of the file. The close method should always be called when done to free up any resources.

Example of Reading Character Data

The following example demonstrates how to read character data from a file and then write it out to the console: 

          
           
FileReader fr = new FileReader("file.txt");
int c;
while ((c = fr.read()) != -1) {
   System.out.println((char)c);
}
fr.close();
This example is similar to the previous one, except instead of processing the character, we write it out to the console using the println() method.

Using BufferedReader for Improved Performance

In addition to the Reader class, Java also has a BufferedReader class which provides improved performance when reading character data. This is because it reads larger chunks of data at a time, reducing the number of disk accesses. The following example shows how to use BufferedReader to read character data from a file: 

          
           
BufferedReader br = new BufferedReader(new FileReader("file.txt"));
String s;
while ((s = br.readLine()) != null) {
   System.out.println(s);
}
br.close();
This example uses the readLine() method provided by the BufferedReader class to read an entire line of data at once. This is more efficient than reading individual characters as we did in the previous examples.

Reading from Other Sources

In addition to reading from files, java.io.Reader can also be used to read from non-file sources such as StringReader or InputStreamReader. The following example demonstrates how to read character data from a String using StringReader: 

          
           
StringReader sr = new StringReader("Hello World!");
int c;
while ((c = sr.read()) != -1) {
   System.out.print((char)c);
}
sr.close();
This example uses the StringReader class to wrap a String object and then read individual characters from the string. The following example demonstrates how to read character data from an InputStream using InputStreamReader: 

          
           
InputStream is = new FileInputStream("file.txt");
InputStreamReader isr = new InputStreamReader(is);
int c;
while ((c = isr.read()) != -1) {
   System.out.print((char)c);
}
isr.close();
This example uses the InputStream class to open a file and then passes it to the InputStreamReader constructor. We then read individual characters from the file and write them to the console.
