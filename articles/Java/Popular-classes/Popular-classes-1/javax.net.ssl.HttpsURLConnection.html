javax.net.ssl.HttpsURLConnection is a class in the Java networking package (java.net) that provides support for the HTTP over SSL/TLS protocol. To create an HttpsURLConnection object, we'll use the openConnection() method of the URL class. This method returns an instance of HttpsURLConnection, which can be used to make HTTPS requests.

HttpsURLConnection con = (HttpsURLConnection)url.openConnection();
