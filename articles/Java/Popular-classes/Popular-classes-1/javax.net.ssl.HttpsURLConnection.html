javax.net.ssl.HttpsURLConnection

javax.net.ssl.HttpsURLConnection is a class in the Java networking package (java.net) that provides support for the HTTP over SSL/TLS protocol. HttpsURLConnection con = (HttpsURLConnection)url.openConnection();
