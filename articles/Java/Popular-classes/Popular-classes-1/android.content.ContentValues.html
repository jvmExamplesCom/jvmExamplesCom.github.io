android.content.ContentValues

android.content.ContentValues is an Android class used to represent a set of values that the ContentResolver can process. It enables developers to store values of type Integer, Long, Float, and String as key-value pairs, so that it can easily be inserted and updated within a database.

Code Examples

The following code example shows how to use the 

          
           
android.content.ContentValues
class. We are using a simple map to store the values which we then add to a ContentValues object. We then print out the values to the console. 

          
           
Map<String, Object> map = new HashMap<>();
map.put("name", "John");
map.put("age", 30);

ContentValues values = new ContentValues();
values.putAll(map);

for (String key : values.keySet()) { 
  System.out.println(key + " : " + values.getAsInteger(key)); 
}
Output: 

          
           
name : John
age : 30
