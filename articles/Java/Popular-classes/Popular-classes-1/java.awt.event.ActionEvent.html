The 

          
           
java.awt.event.ActionEvent
class is a type of event that occurs when an action is performed on a specific component of the Graphical User Interface (GUI). It's a subclass of the superclass

Example

Here is a simple example of how to use 

          
           
ActionEvent
. We create two buttons, "OK" and "Cancel". When either of these buttons is pressed, an action event is generated, which is then handled by the action listener. 

          
           
JButton okButton = new JButton("OK");                    
okButton.addActionListener(new ActionListener() {
  @Override

  }
});

JButton cancelButton = new JButton("Cancel");
cancelButton.addActionListener(new ActionListener() {
  @Override
