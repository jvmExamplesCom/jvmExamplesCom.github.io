android.graphics.Canvas

The 

          
           
android.graphics.Canvas
class is used to draw onto a Surface or View object. It provides methods for drawing shapes like lines, rectangles, circles, and arcs. It also provides methods for drawing text and images. The Canvas class is essential for any Android application that uses graphics.

Creating a Canvas

A Canvas can be created from a View object by calling the 

          
           
getHolder()
method: 

          
           
View view = findViewById(R.id.myview);
Canvas canvas = view.getHolder().lockCanvas();
Alternatively, a Canvas can be created from a Bitmap object by calling the 

          
           
new Canvas(bitmap)
constructor.

Drawing on a Canvas

Once a Canvas has been created, it can be used to draw various shapes and objects by calling the appropriate methods. For example, to draw a line use the 

          
           
drawLine()
method: 

          
           
canvas.drawLine(x1, y1, x2, y2, paint);
Drawing a rectangle is done with the 

          
           
drawRect()
method: 

          
           
canvas.drawRect(left, top, right, bottom, paint);
The 

          
           
paint
argument represents a Paint object which controls how the shape is drawn. To draw a circle call the 

          
           
drawCircle()
method: 

          
           
canvas.drawCircle(x, y, radius, paint);
Finally, to draw an arc use the 

          
           
drawArc()
method: 

          
           
canvas.drawArc(rectF, startAngle, sweepAngle, useCenter, paint);

Drawing Text

Text can also be drawn on a Canvas using the 

          
           
drawText()
method. This method takes two arguments: a String and a Point where the text should be drawn. The Point argument is specified as a float array of length 2. 

          
           
String text = "Hello World";
float[] point = { 100, 200 };
canvas.drawText(text, point[0], point[1], paint);

Drawing Images

Images can be drawn on a Canvas using the 

          
           
drawBitmap()
method. This method takes two arguments: the bitmap to be drawn and the Point where the bitmap should be drawn: 

          
           
Bitmap bitmap = BitmapFactory.decodeResource(getResources(), 
        R.drawable.myimage);
float[] point = { 100, 200 };
canvas.drawBitmap(bitmap, point[0], point[1], paint);

Saving and Restoring State

The Canvas class provides methods for saving and restoring the state of the drawing surface. To save the state, call the 

          
           
save()
method: 

          
           
int count = canvas.save();
The 

          
           
count
variable will contain the number of save points. This number can be used to restore the state at a later time. For example, the following code will restore the state that was saved in the previous example: 

          
           
canvas.restoreToCount(count);
