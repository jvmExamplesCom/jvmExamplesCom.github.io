What is Java.util.Properties?

Java.util.Properties is a class which is part of the Java language library. It provides a means to define key-value pairs, where values can be strings, integers, etc. The class is primarily used for configuration settings, such as properties for an application, or system variables.

Creating a Properties Instance

A Properties instance can be created in a few different ways. One way is to create an empty Properties instance and then add elements to it. For example:


          
           
Properties props = new Properties();
props.put("name", "MyApp");
props.put("version", "1.0");

Another way to create a Properties instance is to read in existing key-value pairs from a file. This can be done using the load() method:


          
           
Properties props = new Properties();
props.load(new FileReader("myapp.properties"));

The load() method takes a Reader object as an argument, so any type of Reader can be used. For instance, a BufferedReader can be used if the data is coming from an InputStream.

Getting Values from a Properties Instance

Once a Properties instance has been populated with key-value pairs, values can be retrieved using the getProperty() method:


          
           
String version = props.getProperty("version");

If the specified key does not exist in the Properties instance, the getProperty() method will return null.

Saving a Properties Instance to a File

To save a Properties instance to a file, the store() method can be used:


          
           
props.store(new FileWriter("myapp.properties"), "My App Configuration");

The store() method has two arguments: a Writer object, and an optional string that will be written as a comment at the beginning of the file.

Using XML Instead of a File

The Properties class can also be used to read in XML documents. To use this feature, the loadFromXML() method needs to be used instead of the load() method:


          
           
props.loadFromXML(new FileInputStream("myapp.xml"));

And to write out an XML document, the storeToXML() method needs to be used instead of the store() method:


          
           
props.storeToXML(new FileOutputStream("myapp.xml"), "My App Configuration");
