javax.servlet.http.HttpServletRequest

The javax.servlet.http.HttpServletRequest class is a part of the Java Servlet API, which is used for customizing web applications. It provides methods for receiving HTTP-specific information about a particular request. This includes the type of request that was made, the path to the requested resource, and any associated parameters. Additionally, it enables access to the HTTP headers and cookies.

Methods

The following are some of the public methods of javax.servlet.http.HttpServletRequest:

  • 
            
             
public String getAuthType()

    This method returns a string corresponding to the authentication scheme used for the request.

  • 
            
             
public String getContextPath()

    This method retrieves the top-level context path of the application.

public Cookie[] getCookies()

public String getPathInfo()

    This method returns the extra path information associated with the request.

public String getRequestURI()

    This method returns the portion of the request URI associated with the servlet.

public StringBuffer getRequestURL()

    This method returns the full URL associated with the request.

public HttpSession getSession()

    This method returns the HttpSession object associated with the request.
