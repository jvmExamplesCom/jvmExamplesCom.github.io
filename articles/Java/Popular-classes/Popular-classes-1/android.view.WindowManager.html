android.view.WindowManager for Java

Android WindowManager is an API that provides the ability to manage the different windows that an Android device supports. This API provides the tools to create, manage and manipulate windows, including a window manager, display manager, and window composition system. The API is designed to provide a unified interface to the different types of windows available on the device.

Using WindowManager API

The WindowManager API can be used in many ways including adding views, changing background, controlling visibility, and moving windows. Here are some examples of using the API with Java.

Adding a View


          
           
WindowManager wm = (WindowManager) getSystemService(Context.WINDOW_SERVICE);

// Create a new view
View view = new View(this);

// Add the view to the window
wm.addView(view, params);

Changing Background


          
           
WindowManager wm = (WindowManager) getSystemService(Context.WINDOW_SERVICE);

// Get the current window
Window window = wm.getCurrentWindow();

// Set the background color
window.setBackgroundColor(Color.BLUE);

Controlling Visibility


          
           
WindowManager wm = (WindowManager) getSystemService(Context.WINDOW_SERVICE);

// Get the current window
Window window = wm.getCurrentWindow();

// Show/Hide the window
window.setVisible(true/false);

Moving Windows


          
           
WindowManager wm = (WindowManager) getSystemService(Context.WINDOW_SERVICE);

// Get the current window
Window window = wm.getCurrentWindow();

// Move the window
wm.moveWindow(window, xOffset, yOffset);
