Introduction to java.util.concurrent.ThreadLocalRandom

The java.util.concurrent.ThreadLocalRandom class is a utility class to generate random numbers backed by an instance of java.util.Random that is scoped to the current thread. This class can be used for applications like games, simulations, and security systems that need to generate random numbers without consuming additional system resources.

Creating Instances

Instances of ThreadLocalRandom are typically created using one of the factory methods provided in the class. There are two main ways to create instances:

  • The static method 
    
            
             
ThreadLocalRandom.current()
    will return the current instance of ThreadLocalRandom associated with the current thread.
  • The static method 
    
            
             
ThreadLocalRandom.setSeed(long seed)
    must be called prior to calling any of the other static factory methods to set the seed value for the random number generator.
