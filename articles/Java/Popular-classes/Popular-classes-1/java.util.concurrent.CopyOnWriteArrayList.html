Introduction to Java.util.concurrent.CopyOnWriteArrayList

Java.util.concurrent.CopyOnWriteArrayList is a concurrent utility class that provides thread-safe access to an ArrayList. CopyOnWriteArrayList<String> list = new CopyOnWriteArrayList<>();
list.add("foo");
