org.junit.Assert

The org.junit.Assert class provides a set of assertion methods useful for writing tests. These methods can be used directly: Assert.assertEquals(), however, they read better if they are referenced through static import: import static org.junit.Assert.assertEquals;.

assertEquals()

The assertEquals() method is the most widely used. It asserts that the two objects passed to it are equal, or the two primitive values passed to it are equal. The syntax of this method is:


          
           
assertEquals(expected, actual);
For example: 

          
           
assertEquals(true, Boolean.valueOf(true));

This example asserts that the expected value is true and the actual value is Boolean.valueOf(true).

assertTrue() and assertFalse()

The assertTrue() and assertFalse() methods are used to test boolean values. The syntax of these methods is:


          
           
assertTrue(boolean expressions); 
assertFalse(boolean expressions);
For example: 

          
           
assertTrue(i == 1 &amp;&amp; j == 2);
assertFalse(i == 2 || j == 1);

This example asserts that the expression i == 1 && j == 2 is true and the expression i == 2 || j == 1 is false.

assertNull() and assertNotNull()

The assertNull() and assertNotNull() methods are used to test whether an object is null or not. The syntax of these methods is:


          
           
assertNull(object);
assertNotNull(object);
For example: 

          
           
assertNull(null);
assertNotNull(new Object());

This example asserts that the object null is null and the object new Object() is not null.

assertSame() and assertNotSame()

The assertSame() and assertNotSame() methods are used to test whether two objects are the same or not. The syntax of these methods is:


          
           
assertSame(expected, actual);
assertNotSame(expected, actual);
For example: 

          
           
assertSame("abc", "abc");
assertNotSame("abc", "xyz");

This example asserts that the two strings “abc” are the same and the two strings “abc” and “xyz” are not the same.

assertArrayEquals()

The assertArrayEquals() method is used to test if two arrays are equal. The syntax of this method is:


          
           
assertArrayEquals(expected, actual);
For example: 

          
           
int[] arr1 = {1, 2, 3};
int[] arr2 = {1, 2, 3};
assertArrayEquals(arr1, arr2);

This example asserts that the two arrays arr1 and arr2 are equal.
