java.util.HashMap

The 
java.util.HashMap
class is a class that implements the Map interface. It stores key-value pairs and is unordered by default. It is the most commonly used implementation of the Map interface.

A HashMap can be initialized in a few different ways. The simplest way is to just create a new instance of the class:


          
           
HashMap<Integer, String> map = new HashMap<>();
This code will create an empty 
HashMap
with keys of type Integer and values of type String. Another way to initialize a 
HashMap
is to specify the initial capacity and load factor when creating a new instance. The initial capacity determines how many buckets there are in the 
HashMap
, and the load factor determines how full the 
HashMap
can be before it is resized. The higher the load factor, the less likely the 
HashMap
will need to be resized, but the more memory it will consume. 

          
           
HashMap<Integer, String> map = new HashMap<>(20, 0.75f);
As mentioned above, the 
HashMap
stores key-value pairs. To put values into the 
HashMap
, the 
put()
method can be used: 

          
           
map.put(1, "one");
map.put(2, "two");
map.put(3, "three");
To get a value from the 
HashMap
, the 
get()
method can be used: 

          
           
String value = map.get(2); // returns "two"

If a value does not exist for the specified key, the 

get()
method will return 
null
. To check if a key exists in the 
HashMap
, the 
containsKey()
method can be used: 

          
           
boolean hasKey = map.containsKey(3); // returns true
boolean hasKey = map.containsKey(4); // returns false
To remove a key-value pair from the 
HashMap
, the 
remove()
method can be used: 

          
           
map.remove(2);
If the specified key does not exist in the 
HashMap
, nothing will happen. The 
HashMap
also has methods to check the size of the 
HashMap
, get all the keys, or get all the values. To get the number of key-value pairs in the 
HashMap
, the 
size()
method can be used: 

          
           
int size = map.size(); // returns 2
To get all the keys in the 
HashMap
, the 
keySet()
method can be used: 

          
           
Set<Integer> keys = map.keySet(); // returns [1, 3]
To get all the values in the 
HashMap
, the 
values()
method can be used: 

          
           
Collection<String> values = map.values(); // returns ["one", "three"]

Iterating Through the HashMap

The 
HashMap
also provides an iterator that allows you to iterate through each key-value pair. There are two different ways to use this iterator. The first way is to use the 
entrySet()
method, which returns a 
Set
of 
Map.Entry
objects. An 
Map.Entry
object contains both the key and the value for each key-value pair. 

          
           
Set<Map.Entry<Integer, String>> entries = map.entrySet();

for (Map.Entry<Integer, String> entry : entries) {
    int key = entry.getKey();
    String value = entry.getValue();

    System.out.println("Key: " + key + ", Value: " + value);
}
The output of this code will be: 

          
           
Key: 1, Value: one 
Key: 3, Value: three
The second way to use the iterator is to use the 
forEach()
method: 

          
           
map.forEach((key, value) -> {
    System.out.println("Key: " + key + ", Value: " + value);
});

The output of this code will be the same as the previous example.

Conclusion

In conclusion, the 
java.util.HashMap
class is a powerful data structure for storing and retrieving key-value pairs. It has many useful methods for getting, putting, and removing values from the 
HashMap
. It also supports iteration so that you can easily process each key-value pair in the 
HashMap
.
