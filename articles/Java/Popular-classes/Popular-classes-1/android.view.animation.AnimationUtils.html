android.view.animation.AnimationUtils

Creating Animations with AnimationUtils

The simplest way to create an animation using AnimationUtils is by using the 

loadAnimation()
method. This method takes as its first argument the context of the Activity or Fragment in which the animation will be created. The second argument specifies the resource ID of the animation XML file that will be used to define the animation. For example: 

          
           
Animation animation = AnimationUtils.loadAnimation(
    getActivity(), R.anim.my_animation);

The animation XML file contains the details of the animation such as the duration, interpolator, start offset, and repeat count. For example, the following XML file defines an animation that runs for 1.5 seconds with a linear interpolator and no repeat:


          
           
<?xml version="1.0" encoding="utf-8"?>
<set xmlns:android="http://schemas.android.com/apk/res/android"
    android:duration="1500"
    android:interpolator="@android:anim/linear_interpolator"
    android:repeatCount="0" >
</set>
