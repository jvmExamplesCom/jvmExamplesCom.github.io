loadAnimation()
method. This method takes as its first argument the context of the Activity or Fragment in which the animation will be created. The second argument specifies the resource ID of the animation XML file that will be used to define the animation. For example:

<?xml version="1.0" encoding="utf-8"?>

    android:repeatCount="0" >
</set>
