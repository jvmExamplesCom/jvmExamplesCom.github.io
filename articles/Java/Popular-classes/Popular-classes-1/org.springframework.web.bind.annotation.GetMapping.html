org.springframework.web.bind.annotation.GetMapping

The org.springframework.web.bind.annotation.GetMapping annotation is used for mapping HTTP GET requests onto specific handler methods. It is a composed annotation that acts as a convenience for @RequestMapping(method = RequestMethod.GET). This annotation can be applied to classes as well as methods and can be used with or without parameters.

Example 1: Basic Usage

In this example, the method will handle GET requests at the specified URL "/hello". 


          
           
@GetMapping("/hello")
public String helloWorld() {
  return "Hello World!";
}

Example 2: With Parameters

In this example, the method will handle GET requests at URLs that begin with "/hello/param1" followed by an arbitrary value. 


          
           
@GetMapping("/hello/{param1}")
public String getParam1(@PathVariable String param1) {
  return "Param1 is: " + param1;
}

Example 3: With Multiple Parameters

In this example, the method will handle GET requests at URLs that begin with "/hello/param1/param2" followed by two arbitrary values. 


          
           
@GetMapping("/hello/{param1}/{param2}")
public String getParam1And2(@PathVariable String param1, @PathVariable String param2) {
  return "Param1 is: " + param1 + ", Param2 is: " + param2;
}

Example 4: With Request Parameters

This example demonstrates how to handle requests with query parameters. In this example, the method will handle GET requests at urls that begin with "/hello?param1=value1¶m2=value2", followed by two name-value pairs. 


          
           
@GetMapping("/hello")
public String getRequestParam(@RequestParam String param1, @RequestParam String param2) {
  return "Param1 is: " + param1 + ", Param2 is: " + param2;
}
