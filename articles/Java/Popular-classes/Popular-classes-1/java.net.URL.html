Java.net.URL

Java.net.URL enables applications to use Uniform Resource Locators (URLs) to locate resources on the Internet. This class, along with other classes provided in the java.net package, provide the core of the networking capabilities in Java. It provides support for network communication, file system access, and more. The URL class is the primary entry point for using the Network API, allowing developers to create and use URLs for web services, content, and other resources.

Creating a URL

To create a URL, you must pass in a valid string that specifies the location of the resource. The URL class provides several constructors to assist with this. 

          
           
String urlString = "https://www.example.com";
URL url = new URL(urlString);
The above example creates a URL object from a String. The String can be an absolute or relative path to the resource.

Using the URL Object

Once the URL object is created, there are several methods available for using it. These include getting the protocol, getting the hostname, and more. 

          
           
String protocol = url.getProtocol();
// returns "https"
String hostname = url.getHost();
// returns "www.example.com"
Additionally, you can use the URL object to open a connection to the resource and read the data: 

          
           
URLConnection conn = url.openConnection();
InputStream in = conn.getInputStream();
// Read data from the input stream...
in.close();

Conclusion

The java.net.URL class is the primary entry point to accessing network resources from Java applications. It provides methods for creating a URL object from a valid string and for accessing its fields.
