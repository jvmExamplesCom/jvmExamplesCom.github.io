The 
java.io.PrintStream
class is a subclass of OutputStream and provides methods for printing strings and data to output streams. It is used for writing primitive data types, objects and text strings to an output stream. The PrintStream class also provides methods for writing formatted output to an output stream.

Creating a PrintStream Object

To create a PrintStream object, you must pass an OutputStream or a File object to the constructor. This can be done using the following code:


          
           
PrintStream ps = new PrintStream(outputStream);

PrintStream ps = new PrintStream(new File("output.txt"));

The PrintStream class provides the methods 
print()

println()

PrintStream ps = new PrintStream(outputStream);
