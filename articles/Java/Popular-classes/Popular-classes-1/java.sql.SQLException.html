What is java.sql.SQLException?

java.sql.SQLException is a Java Exception class that is thrown by the various classes of thejava.sql package when there is an error accessing a database. It is used to indicate that a database operation has failed due to an incorrect SQL statement or other problem.

Examples of SQLExceptions

Let's look at some examples of SQLExceptions. The first example is a syntax error in an SQL statement: 

          
           
String query = "SELECT * FROM table WHERE column = ? ";
PreparedStatement stmt = connection.prepareStatement(query);
stmt.setInt(1, 123);
ResultSet results = stmt.executeQuery();
In this case, a SQLException will be thrown because the query is missing the semi-colon at the end. Another example is a column name that does not exist: 

          
           
String query = "SELECT col1, col2 FROM table";
Statement stmt = connection.createStatement();
ResultSet results = stmt.executeQuery(query);
In this case, a SQLException will be thrown because the column named 'col2' does not exist in the specified table. A third example is when a parameterized query tries to pass a value that is not of the correct type: 

          
           
String query = "SELECT * FROM table WHERE column = ? ";
PreparedStatement stmt = connection.prepareStatement(query);
stmt.setString(1, "abc");
ResultSet results = stmt.executeQuery();
In this case, a SQLException will be thrown because the column being queried is expecting an integer value, and a String value was passed instead.

Handling SQLExceptions

When a SQLException is thrown, it is important to handle it properly. This can be done by using a try/catch block to capture the exception and take corrective action if necessary. For example, if an invalid column name is used in an SQL statement, the following code can be used to handle the exception: 

          
           
try {
    String query = "SELECT col1, col2 FROM table";
    Statement stmt = connection.createStatement();
    ResultSet results = stmt.executeQuery(query);
} catch (SQLException e) { 
    System.out.println("Error: Invalid column name!"); 
}
In this case, the exception is caught and a message is printed to the console indicating that there was an error with the column name.

Conclusion

java.sql.SQLException is an important class in Java used to indicate errors when accessing a database. It is important to handle these exceptions properly in order to avoid any unexpected errors in the application.
