android.view.Gravity

The android.view.Gravity class is a standard Android View class used in the construction of graphical user interfaces. It provides methods for arranging, positioning and scaling graphical elements in an Android view. The class is often used to position views relative to each other and to their parent layout.

The Gravity class can be used to set a view's position by providing values for horizontal and vertical gravity. Horizontal gravity determines a view's position along the x-axis, while vertical gravity determines a view's position along the y-axis. The value passed into thegravity constants must be one of the constants defined in the Gravity class.

Example Usage


          
           
// Create a new LayoutParams object with a specified width, height and gravity
LayoutParams params = new LayoutParams(300, 400, Gravity.CENTER);
// Set the view's layoutParams
view.setLayoutParams(params);

This example sets the width to 300px, the height to 400px and sets the Gravity to CENTER. This will cause the View to be centered horizontally and vertically within its parent layout.

Gravity Types

The Gravity class contains several constants that can be used to set different types of gravity.


          
           
Gravity.TOP

Sets the view's vertical gravity to the top of the parent layout.


          
           
Gravity.BOTTOM

Sets the view's vertical gravity to the bottom of the parent layout.


          
           
Gravity.LEFT

Sets the view's horizontal gravity to the left of the parent layout.


          
           
Gravity.RIGHT

Sets the view's horizontal gravity to the right of the parent layout.


          
           
Gravity.CENTER

Sets the view's horizontal and vertical gravity to the center of the parent layout.


          
           
Gravity.START

Sets the view's horizontal gravity to the start of the parent layout.


          
           
Gravity.END

Sets the view's horizontal gravity to the end of the parent layout.

Example with Multiple Gravities

The Gravity class also allows you to set multiple gravities at once. This can be useful when you want to create views that span multiple sides of the parent layout.


          
           
// Create a new LayoutParams object with specified width, height and different horizontal and vertical gravity values
LayoutParams params = new LayoutParams(300, 400, Gravity.LEFT | Gravity.TOP);
// Set the view's layoutParams
