android.view.Gravity

The android.view.Gravity class is a standard Android View class used in the construction of graphical user interfaces. It provides methods for arranging, positioning and scaling graphical elements in an Android view. The class is often used to position views relative to each other and to their parent layout.

The Gravity class can be used to set a view's position by providing values for horizontal and vertical gravity. Horizontal gravity determines a view's position along the x-axis, while vertical gravity determines a view's position along the y-axis. The value passed into thegravity constants must be one of the constants defined in the Gravity class.
