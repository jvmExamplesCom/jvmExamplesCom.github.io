android.media.AudioManager

The 

          
           
android.media.AudioManager
class provides access to volume and ringer mode control. It can be used by applications to change the device's audio settings. This class also provides access to audio focus related APIs, which allow applications to selectively request audio focus in order to gain priority when playing audio.

Methods

setStreamVolume

This method sets specified stream volume. 

          
           
audioManager.setStreamVolume(
    AudioManager.STREAM_MUSIC,
    5,
    0
);

setRingerMode

This method sets the ringer mode. The valid ringer modes are 

          
           
RINGER_MODE_SILENT
, 

          
           
RINGER_MODE_VIBRATE
, and 

          
           
RINGER_MODE_NORMAL
. 

          
           
audioManager.setRingerMode(AudioManager.RINGER_MODE_SILENT);

adjustStreamVolume

This method adjusts the specified stream volume. 

          
           
audioManager.adjustStreamVolume(
    AudioManager.STREAM_MUSIC,
    AudioManager.ADJUST_RAISE,
    0
);

requestAudioFocus

This method requests audio focus for playback.

    myOnAudioFocusChangeListener,
    AudioManager.STREAM_MUSIC,
    AudioManager.AUDIOFOCUS_GAIN
);

abandonAudioFocus

This method releases audio focus. 

          
           
audioManager.abandonAudioFocus(myOnAudioFocusChangeListener);
