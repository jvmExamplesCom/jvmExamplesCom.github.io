java.util.regex.Matcher

The 

          
           
java.util.regex.Matcher
class is used for matching a given pattern against a text. It allows for finding text that matches a certain regular expression pattern, making it an important part of the Java language.

Basic Usage

The simplest form of use for the 
Matcher
class is to check if a string matches a given regular expression, as shown in the example below: 

          
           
String regex = "^[A-Z]\\w";
String inputStr = "Hello";
Matcher matcher = Pattern.compile(regex).matcher(inputStr);
boolean isMatch = matcher.matches(); //true

The above code checks whether the string Hello matches the regular expression ^[A-Z]\\w. The 

isMatch
boolean will be set to true if the pattern matches the input string.

Advanced Usage

The 
Matcher
class can also be used to extract parts of a string that match a given regular expression. This is accomplished through the 
find()
and 
group()
methods, as demonstrated below: 

          
           
String regex = "\\d+";
String inputStr = "There are 3 cats.";
Matcher matcher = Pattern.compile(regex).matcher(inputStr);
if(matcher.find()) {
    System.out.println(matcher.group()); //3
}
In the above example, the 
find()
method is used to search for any digits in the string There are 3 cats., while the 
group()
method will return the value of the first matched group (which in this case is 3).

Conclusion

The 

          
           
java.util.regex.Matcher
class is a powerful tool for searching and extracting data from strings that match a given regular expression pattern. Through its basic and advanced usage patterns, it is a valuable addition to the Java language.
