Introduction to Network Interface in Java

Network interface is a Java class which provides an application programming interface (API) to access the underlying networking configuration. It is present in the 

          
           
java.net
package and is part of the Java platform's core library. NetworkInterface provides network information about the system, such as details about all the installed network interfaces, IP addresses, broadcast addresses, and multicast addresses.
