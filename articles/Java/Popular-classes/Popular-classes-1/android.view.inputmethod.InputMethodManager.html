android.view.inputmethod.InputMethodManager

The android.view.inputmethod.InputMethodManager provides a system service that manages input methods. It is used by applications to request and interact with the current input method, as well as to monitor its state.

Using InputMethodManager to show/hide soft keyboard

To show or hide the software keyboard programmatically, we use the 

          
           
toggleSoftInput
method of 

          
           
InputMethodManager
. 

          
           
InputMethodManager imm = (InputMethodManager) getSystemService(Context.INPUT_METHOD_SERVICE);
imm.toggleSoftInput(InputMethodManager.SHOW_FORCED, 0);
imm.toggleSoftInput(InputMethodManager.HIDE_IMPLICIT_ONLY, 0);

The 

          
           
isActive
method of 

          
           
InputMethodManager

InputMethodManager imm = (InputMethodManager) getSystemService(Context.INPUT_METHOD_SERVICE);
boolean isSoftKeyboardVisible = imm.isActive();

Using InputMethodManager to detect when a software keyboard is shown/hidden

The 

          
           
InputMethodManager
class also provides a way to register a 

          
           
InputMethodManager.OnInputMethodChangeListener
listener to detect when the software keyboard is shown or hidden. The listener will be called when the software keyboard's visibility changes. 

          
           
InputMethodManager imm = (InputMethodManager) getSystemService(Context.INPUT_METHOD_SERVICE);
imm.registerOnInputMethodChangeListener(new InputMethodManager.OnInputMethodChangeListener() {
    @Override
    public void onInputMethodChanged(boolean active) {
        Log.d("InputMethodManager", "The software keyboard is now " + (active ? "active." : "inactive."));
    }
});
