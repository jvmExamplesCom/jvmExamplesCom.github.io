Android.animation.AnimatorListenerAdapter

Android.animation.AnimatorListenerAdapter provides an implementation of the Animator.AnimatorListener interface which can be overridden to implement only the methods of interest. It is suitable for use with ValueAnimator and ObjectAnimator.

Example

The following example shows how to use an AnimatorListenerAdapter to override the onAnimationEnd() method and log the message “Animation finished.” 

          
           
AnimatorListenerAdapter adapter = new AnimatorListenerAdapter() {
      @Override
      public void onAnimationEnd(Animator animation) {
        Log.d("animation", "Animation finished.");
      }
};

onAnimationCancel()

The onAnimationCancel() method is called when the animation is canceled either by a call to cancel() or end(). This method is called after both onAnimationStart() and onAnimationEnd().

Example

The following example shows how to use an AnimatorListenerAdapter to override the onAnimationCancel() method and log the message “Animation canceled.” 

          
           
AnimatorListenerAdapter adapter = new AnimatorListenerAdapter() {
      @Override
      public void onAnimationCancel(Animator animation) {
        Log.d("animation", "Animation canceled.");
      }
};

onAnimationRepeat()

The onAnimationRepeat() method is called when the animation reaches the end of a single loop. This method is called before the onAnimationEnd() method.

Example

The following example shows how to use an AnimatorListenerAdapter to override the onAnimationRepeat() method and log the message “Animation repeated.” 

          
           
AnimatorListenerAdapter adapter = new AnimatorListenerAdapter() {
      @Override
      public void onAnimationRepeat(Animator animation) {
        Log.d("animation", "Animation repeated.");
      }
};

onAnimationStart()

The onAnimationStart() method is called when the animation starts, either due to a call to start() or due to the auto-starting feature. This method is called before any of the other methods in the adapter are called.

Example

AnimatorListenerAdapter adapter = new AnimatorListenerAdapter() {
