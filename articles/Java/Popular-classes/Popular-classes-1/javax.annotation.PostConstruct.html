javax.annotation.PostConstruct

The javax.annotation.PostConstruct annotation is a specialized form of the Java annotation that is used to specify the method and invocation order of code blocks when an instance of a class is created. It is commonly used to perform necessary initialization tasks before the instance is used.

The PostConstruct annotation can be used on any public method of a class that takes no parameters, and returns void. The method must be declared within a class that follows the JavaBeans conventions, and must also not explicitly declare a throws clause.

setup

init
