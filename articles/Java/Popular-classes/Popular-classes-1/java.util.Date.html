java.util.Date

The java.util.Date class in Java is used to represent a specific instant in time, with millisecond precision. It encapsulates a long value representing the number of milliseconds since January 1, 1970 00:00:00 GMT (1970-01-01T00:00:00Z).

Creating a Date Instance

Creating a date object is straightforward. Here's an example: 

          
           
Date dateNow = new Date();
The code above creates a Date instance which will represent the current time – the moment when the line of code was executed.

Accessing Date Parts

The get methods of the Date class allow us to access parts of the date or time. For example, the following code snippet prints the month for the given Date instance. 

          
           
int month = dateNow.getMonth() + 1;
System.out.println(month);
Note: The getMonth() method returns the calendar month of the year, ranging from 0 (January) to 11 (December), so we have to add 1 to get the correct month number.

Formatting Date and Time

Since the Date class doesn't provide any formatting capabilities of its own, we need to use a third-party library such as SimpleDateFormat. Here's an example of how to format a date and time into a String representation: 

          
           
SimpleDateFormat dateFormat = new SimpleDateFormat("yyyy-MM-dd HH:mm:ss");
String dateTimeString = dateFormat.format(dateNow);
System.out.println(dateTimeString);
The code above prints the current date and time in the following format: “2020-10-25 19:37:17”

Comparing Dates and Times

The compareTo() method of the Date class can be used to compare two dates: 

          
           
Date date1 = new Date();
Thread.sleep(1000);
Date date2 = new Date();

// compare date1 and date2
int result = date1.compareTo(date2);

if (result > 0) {
  System.out.println("date1 is later than date2");
} else if (result == 0) {
  System.out.println("date1 is equal to date2");
} else {
  System.out.println("date1 is earlier than date2");
}
The code above will print “date1 is earlier than date2” since the first date was created before the second one. Note that we used the Thread.sleep() method to make sure that the two dates are different.

Converting Dates and Times

The Date class provides methods to convert a Date instance to and from other formats. For example, the following code snippet converts a Date instance to a Unix timestamp: 

          
           
long unixTime = dateNow.getTime()/1000L;
System.out.println(unixTime);
The code above prints the current time in seconds since the epoch (January 1, 1970 00:00:00 GMT).

Conclusion

In this article, we've discussed the java.util.Date class which is used to represent a specific instant in time. We looked at how to create a Date instance, access date and time parts, compare dates, convert dates to other formats, and more.
