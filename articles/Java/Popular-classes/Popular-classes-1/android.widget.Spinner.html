Android’s Spinner widget is a dropdown menu that allows users to make a single selection from a list of choices. It is similar to the

Creating a Spinner Using XML

The simplest way to create a Spinner is to use XML layout definitions. To create a Spinner, first you must define an array of possible values. This array should be stored in the resources file, and the array’s resource ID should be passed to the Spinner constructor. 

          
           
// Store possible values in the res/values/strings.xml file 
<string-array name="spinnerValues">
    <item>Value 1</item>
    <item>Value 2</item>
    <item>Value 3</item>
</string-array>

// Reference the string array in the layout file
<Spinner
    android:id="@+id/my_spinner"
    android:layout_width="wrap_content"
    android:layout_height="wrap_content"
    android:entries="@array/spinnerValues"/>
