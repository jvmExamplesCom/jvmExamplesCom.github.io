android.widget.SeekBar

android.widget.SeekBar is a user interface element that allows users to select a value from a range. It provides a sliding control that can be used to set a value or progress level. The SeekBar can also be used to adjust the volume, brightness, and other settings.

Usage

To use the SeekBar widget in your Android app, you must first add it to your layout file. This can be done with the following XML code:


          
           
<SeekBar
    android:id="@+id/seekbar"
    android:layout_width="match_parent"
    android:layout_height="wrap_content" />

You can then add functionality to the SeekBar in your Activity or Fragment with the following code:


          
           
SeekBar seekBar = (SeekBar) findViewById(R.id.seekbar);

seekBar.setOnSeekBarChangeListener(new OnSeekBarChangeListener() {
    @Override
    public void onProgressChanged(SeekBar seekBar, int progress, boolean fromUser) {
        // Do something with progress
    }

    @Override
    public void onStartTrackingTouch(SeekBar seekBar) {
        // Do something with touch
    }

    @Override
    public void onStopTrackingTouch(SeekBar seekBar) {
        // Do something after touch
    }
});

Customization

The SeekBar can be customized with a variety of attributes. For example, the android:thumb attribute can be used to set an image as the SeekBar's thumb. The android:progressDrawable attribute can be used to set a custom drawable as the SeekBar's progress indicator. 


          
           
<SeekBar
    android:id="@+id/seekbar"
    android:layout_width="match_parent"
    android:layout_height="wrap_content"
    android:thumb="@drawable/thumb"
    android:progressDrawable="@drawable/progress_indicator" />

In addition to these attributes, the SeekBar also has a few methods that provide more control over its behavior. The setMax() method can be used to set the maximum value of the SeekBar, while the getProgress() and setProgress() methods can be used to get and set the progress of the SeekBar, respectively.


          
           
// Get current progress
int progress = seekBar.getProgress();

// Set new progress
seekBar.setProgress(50);

// Set maximum value
seekBar.setMax(100);
