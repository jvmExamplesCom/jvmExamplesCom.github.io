java.awt.Color Class

The java.awt.Color class is one of the most important classes in the AWT package in Java. It provides various methods to deal with color-related functions like creating a color object, setting the color values, and getting individual color values.

Creating a Color Object

A color object can be created by using any of the following methods:

  • 
            
             
Color(int r, int g, int b)

            
  • Color(int rgb)
  • 
            
             
Color(float r, float g, float b)

            
  • Color(int argb)
  • Color(String str)

Setting Color Values

The setter methods available in Color class are used for setting the various color values. The setter methods are:

  • 
            
             
static void setRGB(int r, int g, int b)

            
  • 
            
             
void setRGB(int rgb)

            
  • 
            
             
static void setARGB(int argb)

Getting Color Values

We can get the individual color components using the getter methods provided in the Color class. The getter methods are:

  • int getRed()
  • int getGreen()
  • int getBlue()
  • int getRGB()
  • int getAlpha()

Examples

To demonstrate the usage of the java.awt.Color class, let us look at the following example:


          
           
//creating a Color object
Color color = new Color(255, 0, 0); //red color
 
//getting the color components
System.out.println("Red : " + color.getRed());
System.out.println("Green : " + color.getGreen());
System.out.println("Blue : " + color.getBlue());
System.out.println("RGB : " + color.getRGB());
System.out.println("Alpha : " + color.getAlpha());
 
//setting the color components
color.setRGB(0, 255, 0); //green color
System.out.println("Red : " + color.getRed());
System.out.println("Green : " + color.getGreen());
System.out.println("Blue : " + color.getBlue());
System.out.println("RGB : " + color.getRGB());
System.out.println("Alpha : " + color.getAlpha());
