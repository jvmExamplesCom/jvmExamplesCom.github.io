javax.xml.transform.TransformerFactory

The javax.xml.transform.TransformerFactory class provides an API for constructing new instances of javax.xml.transform.Transformer, a transformation engine which transforms source trees into result trees.

This class is not thread safe. Users of this class should take care to ensure that the Transformer created is only used within a single thread.

Creating a TransformerFactory

The TransformerFactory class provides a static method called newInstance() which can be used to create an instance of a TransformerFactory:


          
           
TransformerFactory factory = TransformerFactory.newInstance();

Creating a Transformer from a Source

Given a javax.xml.transform.Source object, a Transformer object can be created using the newTransformer() method:


          
           
Transformer transformer = factory.newTransformer(source);

Creating a Transformer from a StyleSheet

The transformers created using this method are optimized for performance when used with a specific style sheet. If a transformer is used multiple times with the same style sheet, it is usually faster to first use the newTemplates() method to obtain an immutable Templates object, and then use newTransformer() to create as many transformer objects as necessary.

Transformer transformer = templates.newTransformer();
