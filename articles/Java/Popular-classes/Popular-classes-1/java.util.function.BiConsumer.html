Java.util.function.BiConsumer

Java.util.function.BiConsumer is a functional interface that accepts two arguments and performs an operation on them. It is one of the predefined functional interfaces that is part of the Java 8 streams API, and can be used with both primitive types and reference types. BiConsumer allows for functions to be executed as part of a stream, such as performing an operation on two given values, or finding their difference.

Examples of BiConsumer Usage

Below are some examples of how Java.util.function.BiConsumer is used:


          
           
//Example 1 
BiConsumer<Integer, Integer> sum = (a, b) -> System.out.println(a + b); 
sum.accept(2, 3); // Outputs 5 

//Example 2 
BiConsumer<String, String> difference = (a, b) -> System.out.println(a.length() - b.length());
difference.accept("Hello", "Bye"); // Outputs 2

In the first example, a BiConsumer is created which takes two Integer values as arguments, and prints out the sum of both. In the second example, a BiConsumer is created which takes two strings, and prints out the difference between their lengths.
