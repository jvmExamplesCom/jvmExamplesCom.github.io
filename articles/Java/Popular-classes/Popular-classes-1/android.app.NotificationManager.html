Android App NotificationManager

Android App NotificationManager is a class designed for managing notifications in the Android framework. It provides a way for applications to send messages, display icons, and other visual indicators about events or activities.

Creating and Managing Notifications

Notifications can be created from within any Activity, or from a service or broadcast receiver. The NotificationManager class contains several methods that allow applications to create and manage notifications.

Creating a Notification

Creating a notification requires an instance of the Notification class. This class contains several parameters for configuring the type, title, message, and icon of the notification. 


          
           
// Create a Notification
Notification notification = new Notification();
notification.setTitle("My Application"); 
notification.setText("Hello World!"); 
notification.setIcon(R.drawable.my_icon);

Setting the Notification Flag

The notification flag is used to determine how a notification should behave when displayed. There are several flags available, such as 

FLAG_AUTO_CANCEL
, 
FLAG_INSISTENT
, and 
FLAG_NO_CLEAR
. 

          
           
// Set the notification flag
notification.flags = Notification.FLAG_AUTO_CANCEL;

Sending a Notification

Once the notification has been configured, it can be sent via the 

notify()
method of the NotificationManager class. The first parameter is a unique identifier, which can be used to cancel or update the notification later. The second parameter is the Notification object itself. 

          
           
// Get a NotificationManager instance
NotificationManager notificationManager = (NotificationManager) getSystemService(Context.NOTIFICATION_SERVICE);

// Send the notification
int notificationId = 1; 
notificationManager.notify(notificationId, notification);

Updating Notifications

Notifications can be updated after they have been displayed by using the 

notify()
method with the same notification identifier. This allows applications to modify existing notifications without creating a new one. 

          
           
// Update an existing notification
notification.setText("Updated Message");

// Send the new notification
notificationManager.notify(notificationId, notification);

Cancelling Notifications

Notifications can be cancelled using the 
cancel()
method of the NotificationManager class. This method takes a notification identifier as its parameter and will remove the notification from the system. 

          
           
// Cancel an existing notification
notificationManager.cancel(notificationId);

Conclusion

The Android App NotificationManager class provides an easy way to create and manage notifications in an Android application. Notifications can be created, updated, and cancelled using the provided methods.
