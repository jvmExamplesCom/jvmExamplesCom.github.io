Java.util.UUID

Java.util.UUID is a class that can be used to create, store and compare Universally Unique Identifiers (UUIDs). It is part of the Java platform that provides an API for generating and comparing UUIDs. This class also implements the Comparable interface for defining a natural ordering among UUIDs.

Generating UUIDs

UUIDs are 128-bit values that are generated by a random number generator. To generate a UUID, use the following code snippet:


          
           
UUID uuid = UUID.randomUUID();

The randomUUID() method will return an instance of UUID populated with a randomly generated UUID.

Comparing UUIDs

The compareTo() method of the UUID class can be used to compare two UUIDs for order. It returns a negative integer, 0 or a positive integer if the current UUID is less than, equal to, or greater than the provided UUID respectively.


          
           
int result = uuid1.compareTo(uuid2);
System.out.println(result);

The above code will print out a negative number, 0 or a positive number depending on the comparison of the two UUIDs.

Storing UUIDs

The toString() method of the UUID class can be used to convert a UUID into its String representation. Storing the String representation of a UUID instead of the actual UUID object can result in better performance.


          
           
UUID uuid = UUID.randomUUID();
String strUuid = uuid.toString();

Retrieving UUIDs from Strings

The fromString() method of the UUID class can be used to retrieve a UUID from its String representation. This method accepts a String representation of a UUID and will return an instance of the UUID class populated with the same value.
