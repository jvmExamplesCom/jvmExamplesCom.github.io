What is Java.lang.annotation.RetentionPolicy?

Java.lang.annotation.RetentionPolicy is an annotation type meta-annotated with the @Retention annotation that tells the compiler when and how the annotations should be retained. SOURCE Retention Policy

RUNTIME Retention Policy
