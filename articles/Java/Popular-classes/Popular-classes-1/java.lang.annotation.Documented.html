Java.lang.annotation.Documented

The Java.lang.annotation.Documented annotation is an annotation type used to denote that the annotated element should be documented by the javadoc tool. It is a meta-annotation, meaning it is an annotation used to annotate other annotations.

Example of Usage

The following example shows a simple use of the Documented annotation:


          
           
@Documented
@Target(ElementType.FIELD)
public @interface MyAnnotation {
  String value();
}

In this example, the @Documented annotation indicates that the @MyAnnotation should be documented by javadoc. The @Target(ElementType.FIELD) indicates that the @MyAnnotation can only be used for fields.

How Does It Work?
