Example

ServerSocket server = new ServerSocket(8080);
server.setSoTimeout(5000);
while (true) {
    System.out.println("Waiting for connection...");
    Socket connection = server.accept();
    System.out.println("Connection established.");

    // Send HTML page
    PrintWriter out = new PrintWriter(connection.getOutputStream());
    out.println("<html>");
    out.println("<head><title>My Web Server</title></head>");
    out.println("<body>Hello World!</body>");
    out.println("</html>");
    out.close();

    connection.close();
}

In this example, the server is set up to automatically close after 5 seconds of inactivity. This ensures that the server is not left running unnecessarily and wasting resources.
