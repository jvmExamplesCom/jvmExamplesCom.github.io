Overview of java.androidx.appcompat.app.AppCompatActivity

The java.androidx.appcompat.app.AppCompatActivity class is a subclass of the Activity class which provides compatibility with older versions of the Android platform. This allows developers to write code that will be compatible with both old and new versions of the platform.

Example Code


          
           
public class MyActivity extends AppCompatActivity
{
    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);
    }
}

In this example, we create a simple activity that sets the content view to a layout file.
