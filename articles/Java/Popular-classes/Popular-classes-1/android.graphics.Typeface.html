android.graphics.Typeface

android.graphics.Typeface is a class in Android that allows developers to easily apply custom fonts to their application. Typeface myTypeface = Typeface.createFromFile("myfont.ttf");

Typeface myTypeface = Typeface.createFromAsset(getAssets(), "myfont.ttf");

String fontFamily = myTypeface.getFontFamily();
