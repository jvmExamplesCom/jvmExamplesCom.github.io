android.graphics.Typeface is a class in Android that allows developers to easily apply custom fonts to their application. Creating and Retrieving Typefaces

The simplest way to create a Typeface is to use one of the static methods provided by android.graphics.Typeface. These methods allow you to create a Typeface from either a file or a resource. For example, to create a Typeface from a file: 

          
           
Typeface myTypeface = Typeface.createFromFile("myfont.ttf");

Typeface myTypeface = Typeface.createFromAsset(getAssets(), "myfont.ttf");

String fontFamily = myTypeface.getFontFamily();
