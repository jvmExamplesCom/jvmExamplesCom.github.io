javax.ws.rs.core.MediaType

javax.ws.rs.core.MediaType is an enumeration type that represents a standard MIME media type, with all its associated information, as defined in RFC 2045 and 2046. It is used in web services to define the type of content that is passed in a request or response.

The following example shows how to retrieve the type of a given file using the javax.ws.rs.core.MediaType class:

System.out.println("The MIME type of the file is: " + mimeType);
