Overview of android.support.v4.view.ViewPager

The android.support.v4.view.ViewPager class provides the functionality of a ViewPager which allows the user to flip through pages of data. ViewPagers are used in many popular applications, such as photo albums. ViewPager implements the concept of a current item position and uses the PagerAdapter to provide the data for pages. ViewPager objects can be used to display slideshows, tabbed content, or other paged data.

Creating a Simple ViewPager

To create a simple ViewPager, start by instantiating an instance of the ViewPager class: 

          
           
ViewPager viewPager = new ViewPager(context);
Once instantiated, the ViewPager's layout can be set to match its parent's layout. To do this, use the setLayoutParams() method on the ViewPager object. Next, create an adapter to provide the data for the ViewPager. This is done by creating a subclass of the PagerAdapter class. The adapter will be responsible for providing data to the ViewPager and connecting it to the views that will be displayed. For example, the following adapter creates four Fragment objects, each containing a TextView with a different piece of text: 

          
           
public class MyPagerAdapter extends PagerAdapter {

    private static final String[] texts = {
        "Text 1",
        "Text 2",
        "Text 3",
        "Text 4"
    };

    public Object instantiateItem(ViewGroup container, int position) {
        // Create fragment for the current position
        Fragment fragment = new MyFragment(); 
        Bundle args = new Bundle(); 
        args.putString("text", texts[position]);
        fragment.setArguments(args);

        // Add the fragment to the container
        container.addView(fragment);

        return fragment;
    }

    public void destroyItem(ViewGroup container, int position, Object view) {
        container.removeView((View) view);
    }

    public int getCount() {
        return texts.length;
    }

    public boolean isViewFromObject(View view, Object object) {
        return (view == (View) object);
    }
}
The instantiateItem() method is responsible for creating the fragments and connecting them to the views. In this case, the TextViews are each given a different text string from the texts array. Finally, connect the adapter to the ViewPager by calling the setAdapter() method on the ViewPager object: 

          
           
viewPager.setAdapter(new MyPagerAdapter());
Now, when the user swipes through the pages, the TextViews will change accordingly.

Customizing the ViewPager

The ViewPager can be customized in various ways. For example, the page margins, the page indicator, and the scroll duration can all be adjusted. To set the page margins, use the setPageMargin() method. The page margin is the space between the Pages. To set a 10-pixel margin, use the following code: 

          
           
viewPager.setPageMargin(10);
The page indicator is the small circle that appears at the bottom of the ViewPager to indicate the current page. To set the page indicator, use the setPageIndicator() method. The page indicator can be any drawable resource, such as an image or a shape. To set the page indicator to a circle, use the following code: 

          
           
viewPager.setPageIndicator(
    ContextCompat.getDrawable(this, R.drawable.circle_drawable));
The scroll duration is the time it takes for a page to scroll from one page to the next. To set the scroll duration, use the setScrollDuration() method. To set the scroll duration to 1 second, use the following code: 

          
           
viewPager.setScrollDuration(1000);
