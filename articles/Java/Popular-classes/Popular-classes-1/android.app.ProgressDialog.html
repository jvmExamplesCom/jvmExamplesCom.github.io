android.app.ProgressDialog

ProgressDialog is a class in the Android framework that provides a dialog that shows progress as a user action or task is being completed. This class is typically used to provide feedback to users that a task, such as uploading a file or downloading data, is being performed. ProgressDialog can show progress in the form of an indeterminate progress bar or a determinate progress bar.

Creating a ProgressDialog Object

A ProgressDialog object can be created using the 

          
           
ProgressDialog.show()
method. The method takes two parameters, a Context, and a String for the title of the dialog. 

          
           
context = new Context();
String title = "Downloading Data";
     
ProgressDialog progressDialog = ProgressDialog.show(context, title);

Setting Attributes

ProgressDialog objects have several attributes that can be set when creating the object to customize its appearance and behavior.

setIndeterminate(boolean)


          
           
progressDialog.setIndeterminate(true);
This method sets whether the progress indicator should be displayed as an indeterminate progress bar or a determinate progress bar. If set to true, an indeterminate progress bar will be shown. If set to false, a determinate progress bar will be shown.

setMax(int)


          
           
progressDialog.setMax(100);
This method sets the maximum progress value. It is only used when the progress indicator is set to be a determinate progress bar. The maximum progress value must be set to a value greater than 0.

setProgressStyle(int)


          
           
progressDialog.setProgressStyle(ProgressDialog.STYLE_HORIZONTAL);
This method sets the style of the progress indicator. The styles available are ProgressDialog.STYLE_SPINNER for an indeterminate progress indicator, and ProgressDialog.STYLE_HORIZONTAL for a determinate progress indicator.

Updating Progress

Once the ProgressDialog has been created and its attributes configured, the progress can be updated.

incrementProgressBy(int)


          
           
progressDialog.incrementProgressBy(10);
This method is used to increment the progress of the progress indicator. It is only used with a determinate progress indicator. The incrementProgressBy() method takes an int parameter that represents the amount to increment the progress.

setProgress(int)


          
           
progressDialog.setProgress(50);
This method is used to set the progress of the progress indicator. It is only used with a determinate progress indicator. The setProgress() method takes an int parameter that represents the progress to set the progress indicator to.

Dismissing the Dialog

When the task is complete and the progress indicator is no longer needed, the dialog can be dismissed using the 

          
           
progressDialog.dismiss();
method.

Conclusion

The android.app.ProgressDialog class is a valuable tool for displaying progress while a user task is being performed. The dialog can be customized by setting its various attributes, and by updating the progress with the incrementProgressBy() and setProgress() methods. When the task is complete, the dialog can be dismissed.
