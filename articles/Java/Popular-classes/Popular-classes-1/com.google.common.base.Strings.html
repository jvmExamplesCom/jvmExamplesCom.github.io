The com.google.common.base.Strings class provides a method for trimming whitespace from a string: 

          
           
trim(String str)

String str = "    example    ";

,

pad(String str, int minLength, char padChar)
. All of these methods take a string, an integer minimum length, and a char padding character, and return a new string of at least the minimum length with the given character as padding. Thus, to pad a string one can use the syntax:
