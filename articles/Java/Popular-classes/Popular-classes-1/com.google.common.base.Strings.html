isNullOrEmpty(String str)

if(Strings.isNullOrEmpty(str)) {

Trimming whitespace from a string

The com.google.common.base.Strings class provides a method for trimming whitespace from a string: 

          
           
trim(String str)
. This method takes a string and returns a new string with all leading and trailing whitespace removed. Thus, to trim a string one can use the syntax: 

          
           
String str = "    example    ";
String trimmedStr = Strings.trim(str);
System.out.println(trimmedStr);  // prints "example"

Padding a string to a given length

The com.google.common.base.Strings class provides three methods for padding a string to a given length: 

          
           
padEnd(String str, int minimumLength, char padChar)
, 

          
           
padStart(String str, int minimumLength, char padChar)
, and 

          
           
pad(String str, int minLength, char padChar)
. All of these methods take a string, an integer minimum length, and a char padding character, and return a new string of at least the minimum length with the given character as padding. Thus, to pad a string one can use the syntax: 

          
           
String str = "example";
int minLength = 10;
char padChar = '_';
String paddedStr = Strings.padStart(str, minLength, padChar);
System.out.println(paddedStr);  // prints "_example___"
