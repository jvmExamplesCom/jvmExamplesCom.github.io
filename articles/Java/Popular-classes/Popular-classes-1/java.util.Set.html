java.util.Set

Java.util.Set is an interface in the Java Collections Framework that extends the Collection Interface. It represents a collection of objects which generally does not contain duplicate elements. Set implementations in java are represented by HashSet, LinkedHashSet, TreeSet and SortedSet. This article will cover the following topics:

HashSet

A HashSet is a Set implementation backed by a HashTable (actually a HashMap instance). It makes no guarantees as to the iteration order; in particular, it does not guarantee that the order will remain constant over time.

Below is an example of how to declare a HashSet in Java: 


          
           
Set<String> myHashSet = new HashSet<String>();

Once the HashSet is declared, you can add elements to it using the add() method: 


          
           
myHashSet.add("Pizza");
myHashSet.add("Burger");
myHashSet.add("Fries");

LinkedHashSet

LinkedHashSet is a Set implementation backed by a LinkedHashMap. It maintains a doubly-linked list running through all of its entries. This allows insertion-order iteration over the set.

Below is an example of how to declare a LinkedHashSet in Java: 


          
           
Set<String> myLinkedHashSet = new LinkedHashSet<String>();

Once the LinkedHashSet is declared, you can add elements to it using the add() method:


          
           
myLinkedHashSet.add("Pizza");
myLinkedHashSet.add("Burger");
myLinkedHashSet.add("Fries");

TreeSet

TreeSet is a NavigableSet implementation backed by a TreeMap. It is sorted according to the natural ordering of its keys.

Below is an example of how to declare a TreeSet in Java: 


          
           
Set<String> myTreeSet = new TreeSet<String>();

Once the TreeSet is declared, you can add elements to it using the add() method:


          
           
myTreeSet.add("Pizza");
myTreeSet.add("Burger");
myTreeSet.add("Fries");

SortedSet

SortedSet is an interface that extends the Set interface. It provides the sorting of elements using the natural ordering of their keys or a comparator.

Below is an example of how to declare a SortedSet in Java:


          
           
Set<String> mySortedSet = new TreeSet<String>();

Once the SortedSet is declared, you can add elements to it using the add() method:


          
           
mySortedSet.add("Pizza");
mySortedSet.add("Burger");
mySortedSet.add("Fries");

These are the four different types of Set implementations in Java. All of them have their own advantages and disadvantages. You should decide which type of Set to use depending on your specific requirements.
