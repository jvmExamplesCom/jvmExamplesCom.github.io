Android Support Design Widget FloatingActionButton

The Android Support Design Widgets FloatingActionButton is a component used for quick access to commonly used features in an Android application. It is typically displayed as a small circle or square containing an icon. When clicked, it will open up an additional menu of options related to the icon displayed.

Syntax

To use the FloatingActionButton, you must first declare the dependency in your Gradle build file: 

          
           
implementation 'com.android.support:design:+'
Then you can add the FloatingActionButton to your layout XML code: 

          
           
<android.support.design.widget.FloatingActionButton
    android:id="@+id/floating_action_button"
    android:layout_width="wrap_content"
    android:layout_height="wrap_content"
    android:src="@drawable/my_icon"
    app:backgroundTint="#FFFFFF"
    app:rippleColor="#AAAAAA" 
    app:fabSize="normal"/>

Attributes

Here are some of the common attributes that you can use with a FloatingActionButton: 
  • android:id
    – Specifies an ID for the button.
  • android:src
    – Specifies the icon to display in the button.
  • app:backgroundTint
    – Sets the background color of the button.
  • app:rippleColor
    – Sets the color of the ripple effect when the button is clicked.
  • app:fabSize
    – Sets the size of the button. Options include mini, normal, and auto.

Events

The FloatingActionButton supports several events that you can handle in your code: 
  • onClick()
    – Triggered when the user clicks on the button.
  • onLongClick()
    – Triggered when the user holds down the button.
  • onTouch()
    – Triggered when the user interacts with the button.
  • onFocusChange()
    – Triggered when the button gains or loses focus.

Example

Here is an example of how to use a FloatingActionButton in an activity: 

          
           
public class MainActivity extends AppCompatActivity {

    FloatingActionButton fab;

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);

        fab = findViewById(R.id.floating_action_button);

        // Set a click listener for the FloatingActionButton
        fab.setOnClickListener(new View.OnClickListener() {
            @Override
            public void onClick(View view) {
                // Do something when the button is clicked
            }
        });
    }
}
In this example, we create a FloatingActionButton and assign it to the variable fab. We then set an onClick listener, so that when the button is clicked, a certain action is performed.

Conclusion

The Android Support Design Widgets FloatingActionButton is a convenient way to give users quick access to commonly used features in an application. With a few simple attributes and events, it is easy to customize the button's appearance and behavior to match the needs of your application.
