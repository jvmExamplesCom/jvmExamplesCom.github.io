Introduction to javax.xml.transform.stream.StreamResult

javax.xml.transform.stream.StreamResult is an abstract class that is a subclass of org.xml.transform.Result and serves as the destination for a transformation result.

The StreamResult class reads data into a byte-stream from a source and then creates a string representation of the data which can be parsed as XML.

This class is useful for transforming XML documents into other formats such as HTML, RTF, or XSLT.

Syntax of StreamResult Class


          
           
public abstract class StreamResult extends Result
