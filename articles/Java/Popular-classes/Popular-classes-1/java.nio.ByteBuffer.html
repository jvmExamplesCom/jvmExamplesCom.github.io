Java.nio.ByteBuffer

In Java, the 
java.nio.ByteBuffer
class is an abstraction for a sequence of bytes. It enables direct allocation of memory for a buffer and support arbitrary memory access. It is intended for use in network applications, as it provides an efficient way to allocate objects for buffering and manipulating data.

Creating a ByteBuffer

A 
ByteBuffer
can be created using the 
allocate()
method of the 
java.nio.ByteBuffer
class: 

          
           
ByteBuffer buffer = ByteBuffer.allocate(1024);

This creates a new buffer of size 1024 bytes. The buffer is considered "empty".

Writing to a ByteBuffer

Once a 
ByteBuffer
has been created, data can be written to it by calling the 
put()
method: 

          
           
int value = 123;
buffer.putInt(value);
The 
putInt()
method stores the specified integer value into the buffer at the current position (the position is advanced by four bytes).

Reading from a ByteBuffer

To read data from the buffer, the 
get()
method can be used: 

          
           
int value = buffer.getInt();
The 
getInt()
method retrieves the integer value at the current position and advances the position by four bytes.

Wrapping a ByteBuffer

It is also possible to wrap a 
ByteBuffer
around an existing byte array: 

          
           
byte[] array = new byte[1024];
ByteBuffer buffer = ByteBuffer.wrap(array);

This creates a new buffer that wraps the given byte array. Data can then be written to and read from the buffer as with any other 

ByteBuffer
.

Flipping a ByteBuffer

When writing data to a 
ByteBuffer
, it is necessary to call the 
flip()
method before attempting to read data from it: 

          
           
//write some data to the buffer
buffer.putInt(123);

//flip the buffer
buffer.flip();

//read the data
int value = buffer.getInt();
The 
flip()
method sets the limit to the current position and the position to zero, thus allowing the buffer to be read from.

Clearing a ByteBuffer

To clear a 
ByteBuffer
, the 
clear()
method can be used: 

          
           
buffer.clear();

This method sets the limit to the capacity and the position to zero, thus resetting the buffer for writing. Note that clearing the buffer does not clear the underlying byte array.

Releasing a ByteBuffer

To release a 
ByteBuffer
, the 
release()
method can be used: 

          
           
buffer.release();

This method releases the buffer and makes it available for garbage collection.
