Java.io.InputStream

The java.io.InputStream is an abstract class that provides a means for reading bytes from various sources in a Java program. It can be used to read data from different types of input streams such as files, network connections, and binary data.

Constructors

The java.io.InputStream has the following constructors:


          
           
public InputStream()

This is the default constructor for the InputStream class.


          
           
protected InputStream(Object lock) {};

This constructor takes an Object parameter which is used as a lock when reading from the InputStream.

Methods

The java.io.InputStream class provides the following methods:


          
           
public int read() throws IOException;

This method reads a single byte from the underlying input stream.


          
           
public int available() throws IOException;

This method returns the number of bytes available to be read from the underlying input stream.


          
           
public int skip(int n) throws IOException;

This method skips the specified number of bytes in the underlying input stream.


          
           
public void close() throws IOException;
This method closes the underlying input stream.

Example

The following example demonstrates how to use the java.io.InputStream class to read data from a file:


          
           
import java.io.*;

public class InputStreamExample {
    public static void main(String[] args) {
        try {
            FileInputStream fis = new FileInputStream("data.txt");

            int data = fis.read();
            while (data != -1) {
                System.out.println((char) data);
                data = fis.read();
            }
            fis.close();
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

In this example, we create a FileInputStream object and pass it a file name as an argument. We then use the read() method to read the contents of the file one byte at a time, printing each byte to the console. Finally, we call the close() method to close the stream.
