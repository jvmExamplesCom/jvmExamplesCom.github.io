android.graphics.CornerPathEffect

android.graphics.CornerPathEffect is a PathEffect subclass that adds rounded corners to drawn paths. The CornerPathEffect applies an offset to each corner of a path so that the sequence of line segments connected by these corners looks like it has been carved with a circular chisel.

Creating a CornerPathEffect

To create a CornerPathEffect, use one of the factory methods: 

          
           
CornerPathEffect cornerPathEffect = new CornerPathEffect(radius);
The 

          
           
radius
parameter specifies the size of the circular chisel to use when carving the corners of the path. To create a CornerPathEffect with a given radius and antialiasing, use: 

          
           
CornerPathEffect cornerPathEffect = new CornerPathEffect(radius, antialiasing);
The 

          
           
antialiasing
parameter specifies whether or not the corners of the Path will be antialiased.

Using a CornerPathEffect

Once you have created a CornerPathEffect, you can use it when drawing a path with a Paint object: 

          
           
Paint paint = new Paint();
paint.setPathEffect(cornerPathEffect);
canvas.drawPath(path, paint);
This will apply the CornerPathEffect to the path, resulting in a path with rounded corners.

Example

In this example, we create a Path with a series of lines and apply a CornerPathEffect with a radius of 10 to it. 

          
           
Path path = new Path();
path.moveTo(0, 0);
path.lineTo(50, 0);
path.lineTo(50, 50);
path.lineTo(0, 50);
path.lineTo(0, 0);

CornerPathEffect cornerPathEffect = new CornerPathEffect(10);

Paint paint = new Paint();
paint.setPathEffect(cornerPathEffect);

Canvas canvas = new Canvas();
canvas.drawPath(path, paint);
The resulting path will look like this: 

          
           
_______
|       |
|   o   |
|_______|
Where the "o" represents a rounded corner.
