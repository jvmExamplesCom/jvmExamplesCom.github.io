org.eclipse.swt.events.DisposeListener

The DisposeListener interface is used in the org.eclipse.swt package to detect events where a control has been disposed. This listener can be attached to a given Control object and will be triggered when its dispose() method is called. The org.eclipse.swt.events package provides implementations of this listener.

The DisposeListener interface contains one method: widgetDisposed(). When a control is disposed, this method will be called with an instance of DisposeEvent as an argument. The DisposeEvent class contains information about the control that was disposed.

Example 1

In this example, a DisposeListener is attached to a Shell object. When the shell is disposed, the listener's widgetDisposed() method will be called and print a statement to the console.


          
           
// Create the Shell
Shell shell = new Shell();

// Attach the DisposeListener
shell.addDisposeListener(new DisposeListener() {
    public void widgetDisposed(DisposeEvent evt) {
        System.out.println("Shell disposed");
    }
});

// Dispose the Shell
shell.dispose();

Example 2

In this example, a DisposeListener is attached to a Label object. When the label is disposed, the listener's widgetDisposed() method will be called and print the label's text to the console.


          
           
// Create the Label
Label label = new Label(shell, SWT.NONE);
label.setText("My Label");

// Attach the DisposeListener
label.addDisposeListener(new DisposeListener() {
    public void widgetDisposed(DisposeEvent evt) {
        Label source = (Label)evt.getSource();
        System.out.println("Label disposed: " + source.getText());
    }
});

// Dispose the Label
label.dispose();

Example 3

In this example, a DisposeListener is attached to a Table object. When the table is disposed, the listener's widgetDisposed() method will be called and print the number of rows in the table to the console.


          
           
// Create the Table
Table table = new Table(shell, SWT.NONE);

    }
});

// Dispose the Table
table.dispose();
