org.apache.http.impl.client.BasicResponseHandler

The org.apache.http.impl.client.BasicResponseHandler class is a utility class of the Apache HTTP Client library that simplifies handling of HTTP responses, allowing developers to concentrate on the content of the response itself. It implements the org.apache.http.client.ResponseHandler interface and provides an easy way to process the response of an HTTP request.

Creating an Instance of BasicResponseHandler

Creating an instance of BasicResponseHandler is quite simple. All that needs to be done is call the constructor with no parameters. For example:


          
           
BasicResponseHandler handler = new BasicResponseHandler();

Handling an HTTP Response

Once an instance of BasicResponseHandler has been created, it can be used to handle the response of an HTTP request. This is done using the 


          
           
handleResponse()
method, which takes an instance of HttpResponse as a parameter. For example: 

          
           
HttpResponse response = httpClient.execute(request);
String content = handler.handleResponse(response);
The 

          
           
handleResponse()
method will process the response and return the content of the response as a String. This content can then be used however it is necessary.

Example Usage

Using BasicResponseHandler is very straightforward, as the following example shows:


          
           
public static void main(String[] args) throws Exception {
   // Create an HttpClient
   DefaultHttpClient httpClient = new DefaultHttpClient();

   // Construct a get request and execute it
   HttpGet request = new HttpGet("http://www.example.com/resource");
   HttpResponse response = httpClient.execute(request);
