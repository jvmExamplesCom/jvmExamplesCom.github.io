java.security.UnrecoverableEntryException

The java.security.UnrecoverableEntryException is a checked exception thrown when an attempt to recover an entry from a keystore failed. It extends the KeyStoreException class and also implements the Serializable interface.

Example 1 - Throwing the UnrecoverableEntryException

Here is an example of code that throws the UnrecoverableEntryException. It tries to retrieve the "myalias" entry from the "mykeystore.jks" keystore, but it fails because it does not exist.


          
           
KeyStore ks = KeyStore.getInstance("JKS");
ks.load(new FileInputStream("mykeystore.jks"), "mypassword".toCharArray());

try {
    KeyStore.Entry entry = ks.getEntry("myalias", null);
} catch (NoSuchAlgorithmException | UnrecoverableEntryException | KeyStoreException e) {
    e.printStackTrace();
    throw new UnrecoverableEntryException();
}

Running this code will print out a stack trace and throw the UnrecoverableEntryException.

Example 2 - Recovering an Entry From a Keystore

Here is an example of code that recovers an entry from a keystore. It retrieves the "myalias" entry from the "mykeystore.jks" keystore.


          
           
KeyStore ks = KeyStore.getInstance("JKS");
ks.load(new FileInputStream("mykeystore.jks"), "mypassword".toCharArray());

try {
    KeyStore.Entry entry = ks.getEntry("myalias", null);
    System.out.println("Entry recovered successfully.");
} catch (NoSuchAlgorithmException | UnrecoverableEntryException | KeyStoreException e) {
    e.printStackTrace();
}

Running this code will print out 'Entry recovered successfully.' if it succeeds in recovering the entry from the keystore.
