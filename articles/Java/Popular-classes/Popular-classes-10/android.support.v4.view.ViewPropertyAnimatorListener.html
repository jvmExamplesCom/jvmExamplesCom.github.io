android.support.v4.view.ViewPropertyAnimatorListener

android.support.v4.view.ViewPropertyAnimatorListener is an interface which provides a callback mechanism for animation events. It is used to listen for changes in the state of an object's properties. It allows developers to be alerted when their object's properties have changed. This enables them to adjust the display accordingly or take other necessary action.

Methods

The ViewPropertyAnimatorListener interface contains three methods:

  • onAnimationStart(View view) - This method is called when the animation starts.
  • onAnimationEnd(View view) - This method is called when the animation ends.
  • onAnimationCancel(View view) - This method is called when the animation is cancelled.

Example

In the following example, a custom ViewPropertyAnimatorListener is created and set to an object. When the object's property changes, the listener is notified and can take appropriate action such as updating the object's display.


          
           
public class MyViewPropertyAnimatorListener implements ViewPropertyAnimatorListener {

  public void onAnimationStart(View view) {
    // Code to run when the animation starts
  }

  public void onAnimationEnd(View view) {
    // Code to run when the animation ends
  }

  public void onAnimationCancel(View view) {
    // Code to run when the animation is cancelled
  }
}

MyObject obj = new MyObject();
obj.setViewPropertyAnimatorListener(new MyViewPropertyAnimatorListener());

In this example, when the animation of the object's property starts, ends or is cancelled, the appropriate callback method will be called and the code inside the method will be executed.
