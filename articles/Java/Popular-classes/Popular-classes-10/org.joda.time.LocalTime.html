Introduction to org.joda.time.LocalTime

org.joda.time.LocalTime is a Java time library which provides classes and methods for manipulating and working with dates and times. It provides classes and methods which allow developers to accomplish tasks such as calculating the current time, formatting timezones, and adding/subtracting days and hours from the current time.
