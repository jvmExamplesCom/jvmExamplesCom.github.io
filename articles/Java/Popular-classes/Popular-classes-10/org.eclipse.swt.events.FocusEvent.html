org.eclipse.swt.events.FocusEvent

The org.eclipse.swt.events.FocusEvent class provides specific information about focus-related events that are fired when any widget obtains or loses keyboard focus in an application. This class provides subclasses of the FocusListener interface, allowing an application to attach a listener to any component and respond to focus-related events.

When a widget gets focus, the interface’s focusGained method is called, whereas when a widget loses focus, the interface's focusLost method is called.

Creating a FocusListener

Creating a FocusListener requires creating a subclass of the FocusListener interface that implements its two methods, focusGained and focusLost.


          
           
public class MyFocusListener extends FocusListener { 
 
  // Invoked when a widget gains keyboard focus 
  @Override 
  public void focusGained(FocusEvent e) { 
      // Code to be executed when a widget gains focus
  } 
  
  // Invoked when a widget loses keyboard focus
  @Override 
  public void focusLost(FocusEvent e) { 
      // Code to be executed when a widget loses focus
  } 
}

Adding the FocusListener to a Widget

To attach the FocusListener to a widget, you must use the addFocusListener method of the appropriate widget.


          
           
widget.addFocusListener(new MyFocusListener());

Handling Focus Gained Events

When a widget gains focus, the focusGained method of the FocusListener will be triggered. The following example code shows how to handle such an event:


          
           
// Invoked when a widget gains keyboard focus 
@Override 
public void focusGained(FocusEvent e) { 
    System.out.println("Widget gained focus!"); 
}

Handling Focus Lost Events

When a widget loses focus, the focusLost method of the FocusListener will be triggered. The following example code shows how to handle such an event:


          
           
// Invoked when a widget loses keyboard focus
@Override 
public void focusLost(FocusEvent e) { 
    System.out.println("Widget lost focus!");

               display.sleep(); 
             } 
       } 
 
       display.dispose(); 
  }

        new TestApp(); 
  }
