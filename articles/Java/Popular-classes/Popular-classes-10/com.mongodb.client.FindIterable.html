FindIterable in Java

Java's com.mongodb.client.FindIterable class provides an iterable interface to the MongoDB query cursor. This allows for iteration over the result set of a MongoDB query. Instead of manually setting up a loop and stepping through, it can be done more elegantly.

The FindIterable class has several methods which make it easier to manipulate the query result set. For example, it allows user to apply a filtering function to the result set, as well as sort and limit the result set.

Creating a FindIterator

FindIterable<Document> iterable = collection.find();

This will create an iterable object which will include all documents in the collection.

limit()
method, as shown below: 

          
           
FindIterable<Document> iterable = collection.find().limit(100);
