FindIterable in Java

Java's com.mongodb.client.FindIterable class provides an iterable interface to the MongoDB query cursor. This allows for iteration over the result set of a MongoDB query. Instead of manually setting up a loop and stepping through, it can be done more elegantly.

FindIterable<Document> iterable = collection.find();
