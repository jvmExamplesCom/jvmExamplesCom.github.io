org.springframework.format.FormatterRegistry

The FormatterRegistry class is a registry of field formatters from the Spring Framework, providing a mechanism for the application to register custom formatters and parse user input. With the help of this registry, developers can create and add formatter beans to their application context.

Creating Formatter Beans

Formatter beans are created by subclassing the abstract Formatter class and overriding the 

print
and 
parse
methods. Here's an example of a simple formatter bean that prints and parses strings representing US currency: 

          
           
public class CurrencyFormatter extends Formatter<Number> {

    @Override
    public String print(Number object, Locale locale) {
        return String.format("$%.2f", object.doubleValue());
    }

    @Override
    public Number parse(String text, Locale locale) throws ParseException {
        return new BigDecimal(text.substring(1));
    }

}

Once a formatter has been registered, it can be used anywhere in the application. For example, if a Spring MVC controller has a
