Java ProgressManager

The ProgressManager class is a part of the IntelliJ Open API, which provides an extensible framework for creating powerful development tools in Java. This class makes it possible to track the progress of any number of tasks in an application or library.

Creating a ProgressManager Object

Creating a 
ProgressManager
object is simple. All you need to do is create a new instance with no arguments: 

          
           
ProgressManager progressManager = new ProgressManager();

Using the ProgressManager Class

When you have created a ProgressManager object, you can use its methods to monitor the progress of tasks. For example, you can call the 

startBackgroundTask
method to begin tracking progress. This method takes two arguments; a String describing the task, and a 
Runnable
containing the code to be executed: 

          
           
progressManager.startBackgroundTask("Downloading files", fileDownloadRunnable);
The 
startBackgroundTask
method returns a 
ProgressIndicator
object that you can use to track the progress of the task. You can also add custom labels to your ProgressIndicator: 

          
           
ProgressIndicator indicator = progressManager.startBackgroundTask("Downloading files", fileDownloadRunnable);
indicator.setText("Fetching file list...");

Example - Downloading Files

Let's say you want to download a list of files from a remote server. You can use the 

ProgressManager
class to keep track of progress as the files are downloaded. Here's an example of how this could be done: 

          
           
public static void downloadFiles(List<String> fileList) {
    ProgressManager progressManager = new ProgressManager();
    ProgressIndicator indicator = progressManager.startBackgroundTask("downloading files", () -> {
        // loop through list of files and download each one
        for (String fileName : fileList) {
            indicator.setText("Downloading " + fileName + "...");
            downloadFile(fileName);
        }
    });
}
