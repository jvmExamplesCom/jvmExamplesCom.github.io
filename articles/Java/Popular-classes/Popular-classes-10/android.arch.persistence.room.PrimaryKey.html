Overview of android.arch.persistence.room.PrimaryKey

android.arch.persistence.room.PrimaryKey is an annotation that is used to designate a field as primary key in a Room database table. This annotation allows developers to mark a field, or collection of fields, as the primary key for a Room database table. It also allows them to customize the order of the fields that make up the key.

Using the PrimaryKey Annotation

The PrimaryKey annotation can be used by itself on a single field to designate it as the table's primary key. For example: 

          
           
@PrimaryKey
int id;

@PrimaryKey(order = 0) 
int id;

@PrimaryKey(order = 1)
String name;
