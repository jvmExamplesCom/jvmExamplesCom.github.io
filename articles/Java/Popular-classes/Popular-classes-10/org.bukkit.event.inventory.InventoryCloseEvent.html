org.bukkit.event.inventory.InventoryCloseEvent

The org.bukkit.event.inventory.InventoryCloseEvent is a specialized event that is fired when an inventory is closed by a player. This could occur when a player leaves the game, when they switch to a different inventory view, or when they are kicked from the server. This event is usually used to clean up resources associated with an inventory window, such as cursor stacks, potion effects, etc.
