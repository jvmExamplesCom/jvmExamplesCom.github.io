Introduction to javax.mail.Store

The 
javax.mail.Store
class is used to provide support for accessing the contents of a message store. It is an abstract class and provides the ability to read, delete, and write messages in a variety of formats. The implementation of the 
javax.mail.Store
class can be done in different ways, depending on the underlying mail server being used. The methods for accessing a mail store are largely dependent upon the type of store being used.

Methods of javax.mail.Store

The 
javax.mail.Store
class provides several methods for working with a message store. These include: 
  • connect()
    : This method is used to establish a connection to the mail store.
  • getFolder()
    : This method is used to access a particular folder in the store.
  • list()
    : This method is used to retrieve messages from the store.
  • create()
    : This method is used to create a new folder in the store.
  • delete()
    : This method is used to delete a folder in the store.
  • expunge()
    : This method is used to permanently delete messages from the store.
  • close()
    : This method is used to close the connection to the store.

Example of javax.mail.Store

To illustrate how the 
javax.mail.Store
class can be used, here is an example of a program that connects to a mail store, retrieves messages from it, and then prints out the subject lines of each message. 

          
           
import javax.mail.*;

public class MailStoreExample
{
    public static void main(String[] args)
    {
        try
        {
            // Create a Properties object to contain the connection
            // properties
            Properties props = new Properties();
            props.put("mail.store.protocol", "imaps");

            // Create a Session object with the given properties
            Session session = Session.getInstance(props);

            // Connect to the store
            URLName url = new URLName("imaps://user@host");
            Store store = session.getStore(url);
            store.connect();

            // Get all folders
            Folder[] folders = store.getDefaultFolder().list();

            // Loop through all folders
            for (Folder folder : folders)
            {
                // Print the folder name
                System.out.println("Folder Name: " + folder.getName());

                // Open folder
                folder.open(Folder.READ_ONLY);

                // Get all messages
                Message[] messages = folder.getMessages();

                // Loop through all messages
                for (Message message : messages)
                {
                    // Print the message subject
                    System.out.println("  Subject: " + message.getSubject());
                }

                // Close folder
                folder.close(false);
            }

            // Disconnect from the store
            store.close();
        }
        catch (Exception e)
        {
            e.printStackTrace();
        }
    }
}
