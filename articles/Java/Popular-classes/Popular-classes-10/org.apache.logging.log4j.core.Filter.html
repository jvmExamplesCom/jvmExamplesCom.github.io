Java org.apache.logging.log4j.core.Filter

The Java class org.apache.logging.log4j.core.Filter is an advanced filtering mechanism for log4j logging events. This class allows for all kinds of parameters to be set in order to filter, discard or store log events. It is a powerful tool that enables developers to customize the way their application logs records.

Methods and Properties

The org.apache.logging.log4j.core.Filter class contains several methods and properties used to filter log messages. Some of the key ones are:

filter(LogEvent event)
    : The main filter method, this should return either ACCEPT, DENY or NEUTRAL
  • getOnMatch()
    : Gets the Action to be taken when a match occurs

    : Gets the Action to be taken when no match occurs
  • isStarted()
    : Checks whether the filter has been started or not
