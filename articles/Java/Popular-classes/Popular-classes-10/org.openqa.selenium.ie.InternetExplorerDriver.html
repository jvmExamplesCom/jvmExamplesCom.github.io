Overview of org.openqa.selenium.ie.InternetExplorerDriver

It uses Microsoft's IEDriverServer to control Internet Explorer and is compliant with the WebDriver specification. This driver allows users to easily write automated tests that take advantage of the features of Internet Explorer, including Windows Integrated Authentication and automatic detection of proxy settings.
