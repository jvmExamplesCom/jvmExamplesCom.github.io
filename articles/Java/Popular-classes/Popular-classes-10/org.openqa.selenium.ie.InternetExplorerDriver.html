Overview of org.openqa.selenium.ie.InternetExplorerDriver

It uses Microsoft's IEDriverServer to control Internet Explorer and is compliant with the WebDriver specification. The driver communicates with a remote server, typically located on the same machine as the IE browser. This driver allows users to easily write automated tests that take advantage of the features of Internet Explorer, including Windows Integrated Authentication and automatic detection of proxy settings.

Once the IEDriverServer is installed, you can create a new instance of the InternetExplorerDriver. To do this, you will need to provide the path to the driver executable.
