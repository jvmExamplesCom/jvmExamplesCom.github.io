com.baomidou.mybatisplus.annotation.TableName

The com.baomidou.mybatisplus.annotation.TableName annotation is a MyBatisPlus annotation used to map the entity class to a database table. By using this annotation, it is possible to specify the table that the entity class would map to, as well as the schema name. In addition, it allows the user to specify what columns should be included in the table. This annotation facilitates the mapping of an entity class to a database table, making it easier to generate SQL query statements and enabling the application to talk to the database in a more consistent manner.

}
