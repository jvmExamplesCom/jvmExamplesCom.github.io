JavaFx Beans Observable

JavaFX beans provides a library of classes and interfaces that enable you to implement the Observer and Observable patterns. The observer pattern is used in an application when there is a one-to-many relationship between objects such as when one object is modified, its dependents are notified automatically. The observable interface defines methods for adding or removing observers and for notifying observers, as well as the state change.

Observable Interface

The Observable interface is part of the javafx.beans package and belongs to the JavaFX Beans Framework. The interface provides methods for registering and unregistering observers, as well as notifying them of state changes in the observed object. The implementation of an Observable class must notify observers when its state has changed, otherwise the observers will not see the change. The following code example shows the Observable interface: 

          
           
public interface Observable {
    public void addListener(InvalidationListener listener);
    public void removeListener(InvalidationListener listener);
    public void addListener(ChangeListener<?> listener);
    public void removeListener(ChangeListener<?> listener);
    public void notifyListeners();
}
The 

          
           
addListener()
method is used to register an observer. It takes an invalidation listener object and adds it to list of observers. When state changes occur, the observer will be notified via the 

          
           
notifyListeners()
method. The 

          
           
removeListener()
method removes an observer from the list of observers.

Observer Interface

The Observer interface is also part of the javafx.beans package and is the counterpart to the Observable interface. It provides methods to observe changes in an observable object. The Observer interface consists of the following methods: 

          
           
public void invalidated(Observable observable);
public void changed(ObservableValue<?> observable, Object oldValue, Object newValue);
The 

          
           
invalidated()
method is called when an observable object is invalidated, meaning that its state has changed. The 

          
           
changed()
method is called when an observable object changes its value.

Example

Let's take a look at a practical example of how we can use the observer pattern in a JavaFX application. In this example, we will create an observable object and an observer to watch for changes in the object’s state. First, we create a simple class called MyObservable which implements the Observable interface: 

          
           
public class MyObservable implements Observable {

    private int value;
    private List<InvalidationListener> listeners;

    public MyObservable() {
        this.listeners = new ArrayList<>();
    }

    @Override
    public void addListener(InvalidationListener listener) {
        this.listeners.add(listener);
    }

    @Override
    public void removeListener(InvalidationListener listener) {
        this.listeners.remove(listener);
    }

    @Override
    public void notifyListeners() {
        for (InvalidationListener listener : listeners) {
            listener.invalidated(this);
        }
    }

    public int getValue() {
        return value;
    }

    public void setValue(int value) {
        this.value = value;
        notifyListeners();
    }
}
In this class, we create a private field called 

          
           
value
, which represents the state of our observable object. We also create a list of invalidation listeners and create methods for adding/removing observers and for notifying observers when the state has changed. Next, we create an observer to watch for changes in our observable object: 

          
           
public class MyObserver implements InvalidationListener {

    @Override
    public void invalidated(Observable observable) {
        if (observable instanceof MyObservable) {
            MyObservable myObservableObject = (MyObservable) observable;
            System.out.println("Value has changed to: " + myObservableObject.getValue());
        }
    }
}
Finally, we create a main method to test out our application: 

          
           
public static void main(String[] args) {
    MyObserver observer = new MyObserver();
    MyObservable observable = new MyObservable();
    observable.addListener(observer);
    observable.setValue(10);
}
When we run this program, the output will be: 

          
           
Value has changed to: 10
This demonstrates the observer pattern and shows how we can use the javafx.beans package to create observable objects and observers in our applications.
