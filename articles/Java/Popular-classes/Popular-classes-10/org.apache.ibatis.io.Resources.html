org.apache.ibatis.io.Resources

The org.apache.ibatis.io.Resources class is a Java utility class in the open source Apache iBATIS project which provides several methods related to I/O operations. Specifically, it provides methods for loading resources such as SQL mapping files, logging configurations and mappers.

Loading Resources

The most commonly used method is 

          
           
Resource.getResourceAsStream()
, which takes a resource path and returns an input stream for the given resource. It attempts to locating the resource using the following methods in order:
  • Using the thread context class loader to load the resource
  • Using MyBatis or the class loader used to load the Resources class
  • Using the system class loader (i.e., the one obtained via ClassLoader.getSystemResourceAsStream())

Here is an example of how to use it to load a resource file:


          
           
InputStream inputStream = Resources.getResourceAsStream("path/to/file.properties");

Resource.classForName()

Resource.getResourceAsStream()

Resource.close()
