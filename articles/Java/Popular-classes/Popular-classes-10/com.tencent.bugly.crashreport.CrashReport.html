Class Overview

The CrashReport class provides various methods for reporting a crash to a remote server. The CrashReport class also allows developers to control the logging level and to add custom data to the report.

Methods

  • logCrash(), which logs the details of a crash to a remote server.
  • captureScreenshot(), which captures and stores a screenshot of the application at the time of the crash.

  • setLogLevel(), which sets the log level of the application.
