Introduction to android.graphics.Picture

The 

          
           
android.graphics.Picture
class is a special type of drawing object that records the drawing calls made by its associated Canvas. It can be used to store a set of drawing commands, and replay them later.

Picture pic = new Picture();

beginRecording()

Canvas canvas = pic.beginRecording(w, h);
Where 

          
           
w

endRecording()

drawPicture()

android.graphics.Picture
