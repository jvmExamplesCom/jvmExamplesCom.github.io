javax.mail.internet.MimeUtility

javax.mail.internet.MimeUtility class provides methods for encoding and decoding MIME (Multipurpose Internet Mail Extensions) headers. It parses the input string into its constituent tokens and encodes or decodes them as needed. MIME header strings are used in email messages to identify their content type, encoding, etc.

Encoding MIME Headers

To encode a MIME header string, you can use the following method of the MimeUtility class: 

          
           
public static String encodeWord(String s) throws UnsupportedEncodingException
This method encodes the given string using the specified charset. Following is an example that shows how to encode a MIME header using this method: 

          
           
String headerValue = "This is a sample header";

String encodedHeaderValue = MimeUtility.encodeWord(headerValue);

System.out.println("Encoded Header Value: " + encodedHeaderValue);

Encoded Header Value: =?utf-8?Q?This_is_a_sample_header?=

public static String decodeWord(String s) throws UnsupportedEncodingException
