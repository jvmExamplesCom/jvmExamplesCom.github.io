A Handle is a reference to a method or field in the generated bytecode for a specific Java class. Handles allow accessing methods and fields in the bytecode without using reflection, which can offer performance benefits. In this article we'll look at how to work with Handles and some of the example uses.

Creating a Handle

The basic interface for creating a Handle is to use the public static Handle create(Kind kind, String owner, String name, String descriptor) method. Handle handle = Handle.create(Kind.METHOD_HANDLE, "ownerClassName", "methodName", "descriptor");
