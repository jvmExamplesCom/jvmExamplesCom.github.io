org.bouncycastle.asn1.ASN1ObjectIdentifier

The org.bouncycastle.asn1.ASN1ObjectIdentifier class is an object that uniquely identifies an ASN.1 object and provides a means to create and manage them. It supports a wide variety of ASN.1 objects, including OIDs (Object Identifier), SETs, SEQUENCEs and STRING instances.

Constructors

This class provides two constructors for creating an ASN.1 object identifier:


          
           
public ASN1ObjectIdentifier(String oid)
public ASN1ObjectIdentifier(int[] arcNumbers)

The first constructor takes a String parameter which contains the dot-separated OID arc numbers. The second one takes an array of integers which are the OID arc numbers.
