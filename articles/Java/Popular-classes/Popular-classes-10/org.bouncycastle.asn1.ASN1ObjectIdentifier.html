org.bouncycastle.asn1.ASN1ObjectIdentifier

The org.bouncycastle.asn1.ASN1ObjectIdentifier class is an object that uniquely identifies an ASN.1 object and provides a means to create and manage them. It supports a wide variety of ASN.1 objects, including OIDs (Object Identifier), SETs, SEQUENCEs and STRING instances.

Constructors

This class provides two constructors for creating an ASN.1 object identifier:


          
           
public ASN1ObjectIdentifier(String oid)
public ASN1ObjectIdentifier(int[] arcNumbers)

The first constructor takes a String parameter which contains the dot-separated OID arc numbers. The second one takes an array of integers which are the OID arc numbers.

public boolean equals(Object obj)
public boolean startsWith(ASN1ObjectIdentifier oid)

public int getPart(int index)

public ASN1ObjectIdentifier toSubLevel(int distance)
public int size() 
public String getId()
public String toString()

The getArcNumbers() method returns the array of integers that represent the arc numbers in this ASN.1 object identifier. The equals() method compares two identifiers and returns true if they are equal. The startsWith() and on() methods check if the current identifier starts with, or is on the branch of, the given identifier. The getPart() method returns the value of the arc number at the specified index. The toBigInteger() method converts the identifier to a BigInteger number. The append() method attaches a new ASN.1 object to the end of the identifier. The branch() method creates a new branch off the current identifier. The ancestor() method returns the ancestor of the current identifier at the specified distance. The child() method adds a child to the current identifier. The toSubLevel() method creates a sublevel from the current identifier. The size() method returns the size of the identifier, i.e. the number of arc numbers that it consists of. The getId() method returns a string representation of the identifier. Finally, the toString() method returns the string representation of the identifier.
