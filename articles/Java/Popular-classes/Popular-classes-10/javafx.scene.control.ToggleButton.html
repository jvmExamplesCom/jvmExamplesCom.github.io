The 
ToggleButton
class is a part of the JavaFX API which provides a specialized button that can be toggled between two states, such as on and off. It is a type of button that is used when you want to give the user two distinct options, such as Yes and No, or On and Off. ToggleButtons are often used in forms, in preference to regular CheckBoxes, as they provide a cleaner and more consistent look. The 
ToggleButton
class provides several methods for controlling the behaviour of the button and getting information about it. These include methods for setting values of the button, such as its text and icon, as well as methods for setting event handlers for when the button is pressed or released.

Creating a ToggleButton

Creating a 
ToggleButton

ToggleButton
class and set its properties. For example, the following code creates a new 
ToggleButton

ToggleButton toggleButton = new ToggleButton("On/Off");

ToggleButton toggleButton = new ToggleButton("On/Off");
toggleButton.setSelected(true);
