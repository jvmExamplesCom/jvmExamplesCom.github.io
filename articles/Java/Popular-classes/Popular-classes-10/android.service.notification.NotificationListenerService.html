The 

          
           
android.service.notification.NotificationListenerService
class is a service that can be extended to receive notifications posted by applications. This class can also be used to post new notifications or cancel existing ones.

onListenerConnected()

		// Code to execute when the listener is connected

		// Code to execute when the listener is disconnected
	}
