android.service.notification.NotificationListenerService

It provides methods to register and unregister to receive notifications, as well as access to the data contained in the notification. This class can also be used to post new notifications or cancel existing ones.

		// Code to execute when the listener is connected

		// Code to execute when the listener is disconnected
	}
