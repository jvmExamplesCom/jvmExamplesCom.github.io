The 

          
           
android.service.notification.NotificationListenerService
class is a service that can be extended to receive notifications posted by applications. It provides methods to register and unregister to receive notifications, as well as access to the data contained in the notification. This class can also be used to post new notifications or cancel existing ones.

Creating the Service

To create the service, extend the NotificationListenerService class and override the 


          
           
onListenerConnected()
and 

          
           
onListenerDisconnected()
methods. Also, add the required permissions and register the service in your manifest. 

          
           
public class MyNotificationListener extends NotificationListenerService {
	@Override
	public void onListenerConnected() {
		// Code to execute when the listener is connected
	}

	@Override
	public void onListenerDisconnected() {
		// Code to execute when the listener is disconnected
	}
}

registerListener()
