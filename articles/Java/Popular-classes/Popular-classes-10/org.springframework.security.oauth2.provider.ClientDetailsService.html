org.springframework.security.oauth2.provider.ClientDetailsService

The org.springframework.security.oauth2.provider.ClientDetailsService interface allows an application to create, update and query OAuth2 Client details stored in the application's database. This interface provides methods for getting client details from various sources, such as from a database, from a properties file or from a custom source. It also provides methods for adding, updating and deleting client details.

Methods

The org.springframework.security.oauth2.provider.ClientDetailsService interface defines the following methods for managing OAuth2 Client details:

  • addClientDetails(ClientDetails): adds a ClientDetails object to the application's existing database. Takes a ClientDetails object as a parameter and returns a Boolean value indicating whether the addition was successful.
  • updateClientDetails(ClientDetails): updates an existing ClientDetails object in the application's existing database. Takes a ClientDetails object as a parameter and returns a Boolean value indicating whether the update was successful.
  • removeClientDetails(String): removes an existing ClientDetails object from the application's existing database. Takes a String object containing the Client ID and returns a Boolean value indicating whether the removal was successful.
  • loadClientByClientId(String): loads ClientDetails object from the application's existing database. Takes a String object containing the Client ID and returns a ClientDetails object from the application's existing database.
