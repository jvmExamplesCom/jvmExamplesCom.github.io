Introduction to org.eclipse.swt.graphics.ImageData

org.eclipse.swt.graphics.ImageData is a Java class which provides a way to access and manipulate image data. It stores information about image width, height, color depth, bits per pixel, and scan line stride. ImageData also provides basic editing operations such as scaling, cropping, rotating, and setting pixels.

Creating an ImageData Object

An ImageData object can be created from an image file, a source of raw image data, or a BufferedImage object.


          
           
File image_file = new File("image.jpg");
ImageData id = new ImageData(image_file.toString());

byte[] raw_data = ...;
ImageData id2 = new ImageData(raw_data);

BufferedImage bi = ...;
ImageData id3 = new ImageData(bi);

Accessing Image Data

ImageData provides several methods for accessing image data, such as 

getHeight()
and 
getWidth()
for getting image dimensions, 
getDepth()
and 
getBytesPerPixel()
for getting the color depth, and 
getScanlineStride()
for getting the scan line stride. 

          
           
int height = id.getHeight();
int width = id.getWidth();
int depth = id.getDepth();
int bytesPerPixel = id.getBytesPerPixel();
int scanlineStride = id.getScanlineStride();

Editing Image Data

ImageData provides several methods for editing image data. The following example rotates an image by 90 degrees.


          
           
ImageData rotatedImage = id.rotate(90);

The following example scales an image by a factor of 0.5 along both the x and y axes.


          
           
ImageData scaledImage = id.scale(0.5, 0.5);

The following example crops an image along the x-axis from the left side of the image to the middle of the image.


          
           
ImageData croppedImage = id.crop(0, 0, id.getWidth()/2, id.getHeight());

Setting Pixels
