Introduction to org.apache.hadoop.mapred.FileInputFormat

org.apache.hadoop.mapred.FileInputFormat is a Java class used in Hadoop that serves as the base class for input formats. It provides several methods for controlling how an application reads data from files in a distributed environment like Hadoop. The FileInputFormat class is a key component of Hadoop’s MapReduce framework and its associated components such as the InputSplit and RecordReader.

FileInputFormat provides methods to create an InputSplit object, which is a logical representation of data to process. It also provides methods to read the data from the InputSplit, convert it into records and feed that data to mappers. This allows the user to provide a single file or directory with data and have it processed as separate records by Hadoop.

FileInputFormat features

  • Provides a generic way to access data.
  • Handles the location of split points (splits the data into multiple pieces).
  • Uses RecordReader to provide an iterator over all records in an InputSplit.
  • Incorporates its own splittable data file format.
  • Integrates with Hadoop’s job configuration system.
  • Offers simple methods to read and write data.

FileInputFormat is the base class for reading data from a file. All other input formats in Hadoop extend from FileInputFormat. It provides a common set of functions such as splitting data into chunks, generating new InputSplits, and returning a Stream of Records from each InputSplit.

Example of FileInputFormat usage

To demonstrate how to use FileInputFormat, let's consider an example of a simple Hello World program that reads data from an HDFS file and prints it to the console.


          
           
import java.io.IOException;

import org.apache.hadoop.fs.Path;
import org.apache.hadoop.io.IntWritable;
import org.apache.hadoop.io.Text;
import org.apache.hadoop.mapreduce.InputSplit;
import org.apache.hadoop.mapreduce.JobContext;
import org.apache.hadoop.mapreduce.RecordReader;
import org.apache.hadoop.mapreduce.TaskAttemptContext;
import org.apache.hadoop.mapreduce.lib.input.FileInputFormat;

public class MyInputFormat extends FileInputFormat<intwritable, text> {

  @Override
  public RecordReader<intwritable, text> createRecordReader(InputSplit split,
      TaskAttemptContext context) throws IOException, InterruptedException {
    return new MyRecordReader();
  }
  
  @Override
  protected boolean isSplitable(JobContext context, Path filename) {
    return false;
  }

}

Configuration conf = new Configuration();
