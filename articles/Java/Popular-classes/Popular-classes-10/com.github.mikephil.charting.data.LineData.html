LineData in Java

LineData is a class in the com.github.mikephil.charting library that provides data in the form of lines to be plotted on a graph. The following code demonstrates how to create a LineData object: 

          
           
LineData data = new LineData();
We can then add a data set to it, for example: 

          
           
data.addDataSet(new LineDataSet(someDataList));
To create a graph that displays the data, we must use the ChartData object that references the LineData. This code sample creates a BarChart using the data stored in the LineData object: 

          
           
BarChart chart = new BarChart(data);
Now that the chart has been created, we can customize various attributes such as the label text, colors, and data markers. Here is a sample to change the background color to light gray: 

          
           
chart.setBackgroundColor(Color.LTGRAY);
We can now add our chart to a layout and show it on the screen. Here is an example of embedding it into a LinearLayout: 

          
           
LinearLayout layout = (LinearLayout) findViewById(R.id.my_layout);

data.draw(canvas);
