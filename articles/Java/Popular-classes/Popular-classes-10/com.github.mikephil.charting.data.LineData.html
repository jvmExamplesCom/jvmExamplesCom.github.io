LineData in Java

LineData is a class in the com.github.mikephil.charting library that provides data in the form of lines to be plotted on a graph. The following code demonstrates how to create a LineData object: 

          
           
LineData data = new LineData();

data.addDataSet(new LineDataSet(someDataList));

BarChart chart = new BarChart(data);

chart.setBackgroundColor(Color.LTGRAY);
