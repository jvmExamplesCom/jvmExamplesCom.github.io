EqualsVerifier is a java library, created by Jan Ouwens, that makes writing the equals, hashCode and toString methods easier and more reliable. It is available under the Apache 2 license, and is hosted on Google Code. EqualsVerifier helps to detect common problems with these methods, e.g. incorrect cardinality, symmetry and transitivity, and provides a robust collection of tests to detect problems like these.

To use EqualsVerifier, you must first add the dependencies to your project's build.gradle or maven files. You can then use it in your tests, by constructing an EqualsVerifier instance and then passing in the class or objects you wish to test.
