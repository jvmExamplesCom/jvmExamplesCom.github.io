javax.management.openmbean.TabularData is a Java class used to represent the underlying data in tabular form. It has two distinct properties, an internal structure which identifies the columns and their types and an array of data which provides the actual values. TabularData also defines methods to access or modify values.

Creating a TabularData Instance
