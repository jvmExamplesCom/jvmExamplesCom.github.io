javax.management.openmbean.TabularData

javax.management.openmbean.TabularData is a Java class used to represent the underlying data in tabular form. It has two distinct properties, an internal structure which identifies the columns and their types and an array of data which provides the actual values. TabularData also defines methods to access or modify values.

Creating a TabularData Instance

To create a TabularData instance, you first need to define its internal structure. This is done by providing an array of CompositeData objects representing each column. The array must be of even length and each item must be of type CompositeData. Once the columns have been defined, you can use them to create a new instance of TabularData.


          
           
CompositeData[] columns = new CompositeData[2];
columns[0] = new CompositeDataSupport(new String[]{"name", "age"}, 
                                      new Object[]{"John", 25});
columns[1] = new CompositeDataSupport(new String[]{"name", "occupation"}, 
                                      new Object[]{"Jane", "Web Developer"});

TabularData tabularData = new TabularDataSupport(columns);

Accessing Data

TabularData provides various get methods to retrieve data from the underlying array. The most commonly used method is getAll(), which returns all values as an array of CompositeData instances.


          
           
CompositeData[] data = tabularData.getAll();
for (CompositeData row : data) {
    System.out.println(row.get("name") + " - " + row.get("age"));
    System.out.println(row.get("name") + " - " + row.get("occupation"));
}

Modifying Data

TabularData also provides various methods to modify its values. To add a new row of data, use the put method, providing the CompositeData object representing the row.


          
           
CompositeData row = new CompositeDataSupport(new String[]{"name", "age"}, 
                                                    new Object[]{"Sam", 27});
tabularData.put(row);

To update a single field in a row, use the update method, providing the same fields again along with the modified value.
