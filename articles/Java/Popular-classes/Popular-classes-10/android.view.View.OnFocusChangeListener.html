Introduction to android.view.View.OnFocusChangeListener

android.view.View.OnFocusChangeListener is an interface used to report focus changes in Android applications. It is part of the Android SDK and provides developers with a way to respond to user interaction with items on the screen that have focus. The interface allows developers to receive callback events when there is a focus change for a target view or when the touch mode changes for a target view. This helps them to create a more interactive experience for their users.

Sample Code: Setting and Responding to Focus Changes

Let's look at the following code example, which shows how to set a focus change Listener on a TextView in an Android application and what should be done when output is received on that event.


          
           
TextView textview = (TextView) findViewById(R.id.myTextView);

textview.setOnFocusChangeListener(new OnFocusChangeListener(){
    @Override
    public void onFocusChange(View v, boolean hasFocus) {
        if (hasFocus) {
            // Do your stuff here
        }
    }
});

In this example, the textView variable has been declared and initialized by using the findViewById method. Then, the setOnFocusChangeListener method has been called in order to set the OnFocusChangeListener on the textView. The OnFocusChangeListener's onFocusChange method is then be called when the view gains or loses focus. In this example, the code will be executed when the view gains focus.

Output

When a TextView gains focus it will show an orange glow around it and the onFocusChange method will be called, but no output will be generated.
