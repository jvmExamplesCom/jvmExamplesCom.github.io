javax.xml.transform.sax.SAXResult

The javax.xml.transform.sax.SAXResult class is a Result implementation that is used to write an XML document to an instance of the org.xml.sax.ContentHandler interface. It functions as an adapter between a SAX2 parser and a destination for an XML document.

Creating Instance

public SAXResult(ContentHandler handler) { ... }
This creates a new SAXResult object with the specified ContentHandler.

  • public ContentHandler getHandler() - Gets the ContentHandler.
