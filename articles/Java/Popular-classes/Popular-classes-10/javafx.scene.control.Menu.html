Introduction to javafx.scene.control.Menu

javafx.scene.control.Menu is a class used in JavaFX to create a menu to provide user interaction with the application. It contains a list of different menus that can be populated and organized. This class provides the functionality to create a hierarchical menu structure, which can be used to show different options to the user. The javafx.scene.control.Menu class provides an easy way for developers to create simple GUI applications.

Creating a Menu

Here is an example code that shows how to create a simple menu in JavaFX:


          
           
MenuBar menuBar = new MenuBar();
 
Menu menuFile = new Menu("File");
MenuItem newFile = new MenuItem("New...");
MenuItem openFile = new MenuItem("Open...");
MenuItem saveFile = new MenuItem("Save...");
menuFile.getItems().addAll(newFile, openFile, saveFile);
 
 Menu menuEdit = new Menu("Edit");
 MenuItem cutSelection = new MenuItem("Cut...");
 MenuItem copySelection = new MenuItem("Copy...");
 MenuItem pasteSelection = new MenuItem("Paste...");
 menuEdit.getItems().addAll(cutSelection, copySelection, pasteSelection);
 
menuBar.getMenus().addAll(menuFile, menuEdit);

This code will result in the following menu structure: 


          
           
File
    New...
    Open...
    Save...
Edit
    Cut...
    Copy...
    Paste...

Adding Sub-Menu Items

Sub-menu items can be added to a menu using the same methods as above. For example, we can add a sub-menu item to the “File” menu by adding the following code: 


          
           
Menu openRecent = new Menu("Open Recent");
 MenuItem file1 = new MenuItem("File1");
 MenuItem file2 = new MenuItem("File2");
 openRecent.getItems().addAll(file1, file2);
 
menuFile.getItems().add(openRecent);

When we add this code, the “File” menu will now look like this: 


          
           
File
    New...
    Open...
    Save...
    Open Recent
        File1
        File2
Edit
    Cut...
    Copy...
    Paste...

Handling Menu Item Actions

Each menu item can have an action associated with it that is called when the item is clicked. To set an action, we can use the “setOnAction” method on a MenuItem object. For example, we can set an action to the “New…” item of the “File” menu like this: 


          
           
newFile.setOnAction(e -> {
    System.out.println("New File Action");
});

Now when the “New…” item is clicked, the action “New File Action” will be printed to the console.

Conclusion

The javafx.scene.control.Menu class provides an easy way to create hierarchical menu structures within JavaFX applications. From adding simple menu items to setting actions for menu items, javafx.scene.control.Menu makes it easy to add basic user interaction with your application.
