What is com.google.zxing.NotFoundException?

com.google.zxing.NotFoundException is a Java-based exception class that is thrown when an attempt to decode a barcode or QR code fails. It indicates that the supplied image or data stream does not contain any such code, or it cannot be successfully decoded. This is a checked exception which means that the code must handle it in order to compile.

Example Usage

The following sample code demonstrates how to catch com.google.zxing.NotFoundException. It attempts to read a barcode from an image file and prints out the text it encodes. If the barcode can't be found, it catches the NotFoundException and prints an appropriate message.


          
           
try {
    BufferedImage img = ImageIO.read(new File("barcode.jpg"));
    BarcodeReader reader = new BarcodeReader();
    String text = reader.decode(img);
    System.out.println("Barcode text: " + text);
} catch (NotFoundException e) {
    System.out.println("No barcode found!");
}
