What is com.google.zxing.NotFoundException?

com.google.zxing.NotFoundException is a Java-based exception class that is thrown when an attempt to decode a barcode or QR code fails. try {
    BufferedImage img = ImageIO.read(new File("barcode.jpg"));

}
