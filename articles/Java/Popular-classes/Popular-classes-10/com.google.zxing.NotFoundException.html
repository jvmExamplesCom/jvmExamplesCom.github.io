What is com.google.zxing.NotFoundException?

com.google.zxing.NotFoundException is a Java-based exception class that is thrown when an attempt to decode a barcode or QR code fails. This is a checked exception which means that the code must handle it in order to compile.

Example Usage

try {
    BufferedImage img = ImageIO.read(new File("barcode.jpg"));
    BarcodeReader reader = new BarcodeReader();

} catch (NotFoundException e) {
    System.out.println("No barcode found!");
}
