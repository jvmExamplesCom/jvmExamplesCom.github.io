When a selection is changed, the element that changed is passed to the listener's method: 


          
           
public void selectionChanged(SelectionChangedEvent event)

The selection changed event can then be used to query the source, get the selection and take action appropriate for your application.

selectionChanged()

   }
}

MySelectionChangedListener
