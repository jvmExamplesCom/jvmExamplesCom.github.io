The 

          
           
org.eclipse.jface.viewers.ISelectionChangedListener
interface is an important part of the Eclipse JFace framework. It provides a way to register a selection change event listener with a view part or viewer. The listener will receive notification whenever a selection changes in the view part or viewer.

When a selection is changed, the element that changed is passed to the listener's method: 


          
           
public void selectionChanged(SelectionChangedEvent event)

The selection changed event can then be used to query the source, get the selection and take action appropriate for your application.

Let's look at a simple example of how to respond to a selection change. We'll create a class that implements 


          
           
ISelectionChangedListener
and then register it with our view part or viewer. When a selection is made, the 
selectionChanged()
method will be called and we can take action as required. 

          
           
public class MySelectionChangedListener implements ISelectionChangedListener {

    public void selectionChanged(SelectionChangedEvent event) {

        IStructuredSelection selection = (IStructuredSelection) event.getSelection();

   }
}
To register the 

          
           
MySelectionChangedListener
with our view part or viewer, simply call the addSelectionChangedListener() method on the view part/viewer: 

          
           
viewer.addSelectionChangedListener(new MySelectionChangedListener());
