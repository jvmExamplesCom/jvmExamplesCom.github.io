javax.enterprise.inject.spi.Bean

Overview

The javax.enterprise.inject.spi.Bean class is part of the Java API for Contexts and Dependency Injection (CDI), which provides a type-safe way to access and interact with Java objects. This interface is used to represent beans that are managed by the container.

Methods

The javax.enterprise.inject.spi.Bean interface has several methods that can be used to interact with the bean instance. These methods include:
  • getTypes() – This method returns a set of types that represent the bean type of the bean.
  • getQualifiers() – This method returns a set of qualifiers that are used to distinguish the bean from other beans.
  • getScope() – This method returns the scope of the bean.
  • getName() – This method returns the name of the bean.
  • getStereotypes() – This method returns a set of stereotypes associated with the bean.
  • isAlternative() – This method returns a boolean value indicating if the bean is an alternative representation of the same type.
  • getBeanClass() – This method returns the class of the bean.
  • isNullable() – This method returns a boolean value indicating if the bean can be set to null.
