Creating a ReactInstanceManager

To create a ReactInstanceManager, the following code should be written within the activity that will be hosting the React Native application:


          
           
ReactInstanceManager mReactInstanceManager = ReactInstanceManager.builder()

    .setBundleAssetName("index.android.bundle")

The addPackage method adds a package to the list of packages that are available to the application. The useDeveloperSupport decides whether or not the development features should be enabled, usually set to true for debugging purposes. The setInitialLifecycleState sets the initial state of the application, usually set to “RESUMED”.
