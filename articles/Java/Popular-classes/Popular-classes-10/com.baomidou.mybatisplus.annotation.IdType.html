com.baomidou.mybatisplus.annotation.IdType

The 

          
           
com.baomidou.mybatisplus.annotation.IdType
is an enumeration type in MyBatisPlus, used to represent the data type of primary keys. It exists in MyBatisPlus 3.2.0 and higher.

Usage

The 
@TableId
annotation is used to indicate the data type of the primary key of a table for a Java model object. It can take one of the following four enums as a value: 
  • NONE
    : The primary key is not auto-generated.
  • AUTO
    : The primary key is auto-generated by an auto-incrementing algorithm.
  • INPUT
    : The primary key is input by you and won't be auto-generated.
  • ID_WORKER
    : The primary key uses IDWorker algorithm to generate its value.

For example, let's say we have a model object called 

User
, and its primary key is 
id
: 

          
           
public class User {
    @TableId(type = IdType.AUTO)
    private Long id;

    // other fields
}
This indicates that the primary key of the 
User
table will be an auto-generated integer.
