org.springframework.boot.test.context.runner.ApplicationContextRunner

The ApplicationContextRunner is a test utility class provided by the Spring Boot framework that helps in setting up and running a complete application context, including the configuration of an application context using the runner's configuration methods.

In general, an application context is the container that holds all the beans and objects that are used by a Spring-powered application. The ApplicationContextRunner helps to create an application context with all the configuration details provided in one place, effectively eliminating the need for having multiple configuration files or classes scattered across the project.

Example Usage

The following is a sample example of how to use the ApplicationContextRunner for setting up an application context.


          
           
ApplicationContextRunner contextRunner = new ApplicationContextRunner()
    .withConfiguration(AutoConfigurations.of(MyAutoConfiguration.class));

contextRunner.run(context -> {
   // code executed in the application context
});

In this example, the ApplicationContextRunner is configured with the AutoConfiguration of the MyAutoConfiguration class. Once the runner is configured, its 

run()
method is executed, and within it, the provided lambda expression will be executed in the application context.

Conclusion

The ApplicationContextRunner is a helpful utility class provided by the Spring Boot framework that helps in quickly setting up an application context for testing purposes.
