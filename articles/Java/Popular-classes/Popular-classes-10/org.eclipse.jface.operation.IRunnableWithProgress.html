Introduction to org.eclipse.jface.operation.IRunnableWithProgress

org.eclipse.jface.operation.IRunnableWithProgress is an interface for runnable classes, which support progress and cancellation. It is part of the Eclipse JFace framework and it is used for long running operations. The interface provides two methods, run() and createCanceledException().

The run() method is abstract and must be implemented by the class implementing this interface. It contains the actual code that will be executed by the application. The createCanceledException() is also abstract and must be implemented. This method is called when the user cancels the operation.

Example Code

The following example shows a simple implementation of IRunnableWithProgress. It is a long running task to count up to 1000.


          
           
public class RunnableExample implements IRunnableWithProgress {
    @Override
    public void run(IProgressMonitor monitor) throws InvocationTargetException, InterruptedException {
        int max = 1000;
        monitor.beginTask("Counting", max);
        for (int i = 0; i < max; i++) {
            if (monitor.isCanceled()) {
                throw new InterruptedException();
            }
            Thread.sleep(100); 
            monitor.worked(1);
        }
        monitor.done();
    } 

    @Override
    public IStatus createCanceledException() {
        // Create a status indicating the operation was canceled
        return new Status(IStatus.CANCEL, "org.example", 0, "Operation canceled", null);
    }
}

The above example code creates a RunnableExample that counts up to 1000, using a IProgressMonitor. The IProgressMonitor is used to track the progress of the task and the createCanceledException() method is called when the user cancels the task.
