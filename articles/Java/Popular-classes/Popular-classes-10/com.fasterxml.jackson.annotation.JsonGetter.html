com.fasterxml.jackson.annotation.JsonGetter

The com.fasterxml.jackson.annotation.JsonGetter is an annotation used to mark a method which will be used as a "getter" method for the purposes of serialization and deserialization of a Java object. This annotation can be placed on specific methods you want to use for serialization, such as getters and setters on a Java class, to explicitly tell Jackson to use that method for serialization. It is also used to indicate that a method should be used to deserialize a given key from an incoming JSON object.

Code Examples

Below are some examples of using the JsonGetter annotation. The code will be presented in HTML format.

Simple Class Example


          
           
@JsonIgnoreProperties(ignoreUnknown = true)
public class SimpleClass {

    private String name;
    private int age;

    public SimpleClass() { }

    public SimpleClass(String name, int age) {
        this.name = name;
        this.age = age;
    }

    @JsonGetter("name")
    public String getName() {
        return name;
    } 

    @JsonGetter("age")
    public int getAge() {
        return age;
    }

}

Serialization Example


          
           
// Create object to serialize
SimpleClass sc = new SimpleClass("John", 25);

// Serialize object to JSON string
ObjectMapper mapper = new ObjectMapper();
String jsonString = mapper.writeValueAsString(sc);

// Output
System.out.println(jsonString);
Output: 

          
           
{"name":"John","age":25}

Deserialization Example


          
           
// Create JSON string to deserialize
String jsonString = "{\"name\":\"John\",\"age\":25}";

// Deserialize JSON string
ObjectMapper mapper = new ObjectMapper();
SimpleClass sc = mapper.readValue(jsonString, SimpleClass.class);

// Output
System.out.println("Name: "+sc.getName());
System.out.println("Age: "+sc.getAge());
Output: 

          
           
Name: John
Age: 25
