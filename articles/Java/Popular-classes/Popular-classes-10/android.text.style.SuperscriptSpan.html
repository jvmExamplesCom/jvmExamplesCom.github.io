Introduction to android.text.style.SuperscriptSpan

The superscripted style can also be applied to existing text. The SuperscriptSpan class is associated with the Paint class and as such, it interacts closely with the Paint class to provide a rich visual experience.

The SuperscriptSpan class has several different constructors that allow the user to customize the style of the superscripted text as they see fit.
