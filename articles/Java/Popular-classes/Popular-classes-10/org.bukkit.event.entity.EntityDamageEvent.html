Introduction to org.bukkit.event.entity.EntityDamageEvent

org.bukkit.event.entity.EntityDamageEvent

EntityDamageEvent

EntityEvent

setDamage()
method. Lastly, this event also allows a plugin to cancel any damage received.

When is EntityDamageEvent Fired?

The 

          
           
EntityDamageEvent
