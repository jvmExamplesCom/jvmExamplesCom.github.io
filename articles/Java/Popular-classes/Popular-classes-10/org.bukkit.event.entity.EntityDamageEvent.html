Introduction to org.bukkit.event.entity.EntityDamageEvent

org.bukkit.event.entity.EntityDamageEvent

EntityDamageEvent

EntityEvent

setDamage()

When is EntityDamageEvent Fired?

The 

          
           
EntityDamageEvent
