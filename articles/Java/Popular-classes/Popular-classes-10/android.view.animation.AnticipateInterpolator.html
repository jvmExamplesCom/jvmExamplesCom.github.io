android.view.animation.AnticipateInterpolator

The android.view.animation.AnticipateInterpolator class is a subclass of the android.view.animation.TimeInterpolator abstract class. This class provides an interpolator in which the animation starts backward, and then flings forward. Specifically, it starts with an initial velocity that is negative, reaches zero, and then moves with a positive velocity.

Usage

This interpolator can be used to animate any object or property. To use, simply specify the AnticipateInterpolator when creating your animation object. For example:


          
           
ObjectAnimator anim = ObjectAnimator.ofFloat(myView, "x", 0, 100);
anim.setInterpolator(new AnticipateInterpolator());
anim.start();

This will create an animation that moves from x=0 to x=100, starting with a negative velocity, slowing down to a stop, and then speeding up again for the second half of the animation.

The anticipate interpolator also takes a tension parameter (tension > 0) that controls the amount of anticipation. The higher the tension, the more dramatic the effect will be.


          
           
ObjectAnimator anim = ObjectAnimator.ofFloat(myView, "x", 0, 100);
anim.setInterpolator(new AnticipateInterpolator(2.0f));
anim.start();
