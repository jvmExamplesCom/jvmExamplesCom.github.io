javax.management.IntrospectionException

Javax.management.IntrospectionException is a checked exception thrown by the methods of the DynamicMBean interface to indicate that the particular attribute or operation specified is not found. This exception type can also be thrown by the getAttributes and getAttribute methods of the javax.management.MBeanInfo interface when an attribute or class attribute specified is not found.

Example 1:

In the following example, we try to set the attribute "name" in a Person class object to a new value. Since there is no public setter method for this attribute, the code throws a javax.management.IntrospectionException.


          
           
public class Person{
    private String name;
    ...
    public String getName() {
        return name;
    }
    ...
}

Person person = new Person();

try {
     // Throws IntrospectionException:
     mBeanInfo.setAttribute(person, "name", "John"); 
} catch (javax.management.IntrospectionException e) {
     System.err.println("IntrospectionException caught: " + e.getMessage());
}
