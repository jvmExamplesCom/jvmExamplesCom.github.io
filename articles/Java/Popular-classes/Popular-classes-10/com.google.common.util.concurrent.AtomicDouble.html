Java com.google.common.util.concurrent.AtomicDouble

Java's com.google.common.util.concurrent.AtomicDouble class is a thread-safe, mutable double wrapper. It can be used to atomically update double values in multi-threaded environment. This class provides methods such as get(), set(), addAndGet(), compareAndSet(), decrementAndGet() and incrementAndGet() for manipulating values.

AtomicDouble Example

The following example shows how to use AtomicDouble to atomically update a double value stored in a variable. 


          
           
AtomicDouble atomicDouble = new AtomicDouble(0.0);

// Update the value atomically
atomicDouble.addAndGet(1.0);

// Get the value
double d = atomicDouble.get();
System.out.println("d = " + d); // prints 1.0

AtomicDouble Compare & Set

AtomicDouble also provides a compareAndSet() method which takes an expected value and a new value as parameters. It atomically sets the new value only if the expected value matches the current one. For example: 


          
           
AtomicDouble atomicDouble = new AtomicDouble(0.0);

if (atomicDouble.compareAndSet(0.0, 1.0)) {
    System.out.println("Value successfully updated");
}

The incrementAndGet() method increments the value by one and returns the resulting value. For example: 


          
           
AtomicDouble atomicDouble = new AtomicDouble(0.0);
