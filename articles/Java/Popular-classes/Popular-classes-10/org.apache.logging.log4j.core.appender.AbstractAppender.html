org.apache.logging.log4j.core.appender.AbstractAppender: Introduction

The org.apache.logging.log4j.core.appender.AbstractAppender class is a base abstract class for all appenders and provides the framework for all Appender implementations. This class provides basic support for layout, filters, error handling, and the option to suppress successive messages from the same logger. It can be used to extend existing Appenders or create new ones such as SyslogAppenders, HttpAppenders, etc.

To use the org.apache.logging.log4j.core.appender.AbstractAppender class, create a class which extends AbstractAppender and override the following methods: 

          
           
@Override
protected void append(LogEvent event) {
   // Code to log the event
}

@Override

    // Code to return true or false depending on whether the Appender requires a layout
}

    </MyAppender>
</Appenders>
where MyAppender is the name of the class extending the AbstractAppender.
