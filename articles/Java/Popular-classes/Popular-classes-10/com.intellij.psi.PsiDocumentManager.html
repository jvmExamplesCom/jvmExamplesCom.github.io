com.intellij.psi.PsiDocumentManager

The 

          
           
com.intellij.psi.PsiDocumentManager
is a very important class in the IntelliJ Platform API which provides an API to update and manage the state of documents and their associated PSI files. It is used to ensure that resources are kept in sync between the IntelliJ Platform and its plugins.

Methods

The following are some of the commonly used methods in the com.intellij.psi.PsiDocumentManager class:

  • getInstance()
    - returns the shared instance of the document manager.
  • 
            
             
commitAllDocuments()
    - commits all documents currently open in the IntelliJ Platform.
  • 
            
             
commitDocument(Document)
    - commits a particular document.
  • 
            
             
getCachedPsiFiles(VirtualFile)
    - returns the cached PsiFile instances associated with the given VirtualFile.
  • getCachedPsiFiles()
    - returns all cached PsiFile instances.
  • 
            
             
updateFromCachedPsi(Document,boolean)
    - updates the contents of the given document from the corresponding PsiFile instance.

com.intellij.psi.PsiDocumentManager
class to commit all documents open in the IntelliJ Platform:
