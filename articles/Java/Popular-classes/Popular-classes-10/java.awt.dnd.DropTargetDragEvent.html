Java awt.dnd.DropTargetDragEvent

Java AWT (Abstract Window Toolkit) provides a built-in Drag & Drop (DND) package namely 

          
           
java.awt.dnd.
for creating DND functionality. The 

          
           
DropTargetDragEvent
class is one of the important classes in the Java DND package. It extends from 

          
           
DropTargetEvent
and used to create objects that describe context of a Drag & Drop operation.

Constructor


          
           
public DropTargetDragEvent(DropTargetContext dtc, Point cursorLocn, int dropAction, int srcActions)
This constructor creates a 

          
           
DropTargetDragEvent
object with given DropTargetContext, location of cursor, user drop action, and source actions.

Methods

  • 
            
             
public void acceptDrag(int dragOp)
    This method calls acceptDrag() method on DropTargetContext associated with this DropTargetDragEvent. 
  • 
            
             
public void rejectDrag()
    This method calls rejectDrag() method on DropTargetContext associated with this DropTargetDragEvent. 
  • 
            
             
public Point getLocation()
    This method returns the location of the cursor during this event. 
  • 
            
             
public int getSourceActions()
    This method returns the source action of this event. 
  • 
            
             
public void dropComplete(boolean success)
    This method signals that the Drop Target has completed the Drop operation.

Example

In this example, a 

          
           
DropTarget
object is created. When an item is dragged and dropped on the component, it will trigger the 

          
           
dragEnter()
, 

          
           
dragOver()
, 

          
           
dropActionChanged()
and 

          
           
drop()
methods of 

          
           
DropTargetListener
. 

          
           
import java.awt.*;
import java.awt.dnd.*;
public class MyDropTarget extends Panel implements DropTargetListener {
   // Create the DropTarget and register 
   // it with the component 
   DropTarget dt = new DropTarget (this, DnDConstants.ACTION_COPY_OR_MOVE, this);
   public void dragEnter (DropTargetDragEvent dtde) {
      System.out.println ("Drag Enter");
   }
   public void dragExit (DropTargetEvent dte) {
      System.out.println ("Drag Exit");
   }
   public void dragOver (DropTargetDragEvent dtde) {
      System.out.println ("Drag Over");
   }
   public void dropActionChanged (DropTargetDragEvent dtde) {
      System.out.println ("Drop Action Changed");
   }
   public void drop (DropTargetDropEvent dtde) {
      System.out.println ("Drop");
   }
}
