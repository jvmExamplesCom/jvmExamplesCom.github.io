org.kohsuke.stapler.DataBoundSetter

Java annotation org.kohsuke.stapler.DataBoundSetter is used to indicate that a particular method defines a property setter in a data binding context. The method is marked with the @DataBoundSetter annotation, and takes a single parameter that corresponds to the property to be set.
