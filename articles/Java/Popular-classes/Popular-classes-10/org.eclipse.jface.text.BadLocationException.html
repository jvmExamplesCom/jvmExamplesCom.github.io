What is org.eclipse.jface.text.BadLocationException?

The org.eclipse.jface.text.BadLocationException is a checked exception thrown by classes in the 


          
           
org.eclipse.jface.text
package to indicate that an illegal position has been requested. An IllegalPositionRequest indicates that a request was made for a particular line or column number, which does not exist in the document.

IDocument document = new Document("Hello World!");
int offset = 5;

try {

BadLocationException
