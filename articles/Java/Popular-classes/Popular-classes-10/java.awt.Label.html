java.awt.Label

Java's AWT Label class is an implementation of a Label UI component that can be used to display a text label. Labels can be used to provide users with information, such as descriptions, warnings, or instructions.

Creating a Label

To create a Label, use the following code: 

          
           
Label myLabel = new Label("LabelText");
This creates a Label object with the text "LabelText".

Adding a Label to a Container

To add a Label to a Container object, use the add() method to add the Label to the Container, like this: 

          
           
myContainer.add(myLabel);
Where myContainer is the name of the Container object.

Setting the Label's Text

To set the Label's text, use the setText() method, like this: 

          
           
myLabel.setText("NewText");
This sets the Label's text to "NewText".

Getting the Label's Text

To get the Label's text, use the getText() method, like this: 

          
           
String labelText = myLabel.getText();
This gets the Label's text and stores it in the String variable "labelText".

Setting the Label's Alignment

To set the Label's alignment, use the setAlignment() method. The possible alignments are LEFT, RIGHT, and CENTER. To set it to left alignment, use this code: 

          
           
myLabel.setAlignment(Label.LEFT);
To set it to right alignment, use this code: 

          
           
myLabel.setAlignment(Label.RIGHT);
And to set it to center alignment, use this code: 

          
           
myLabel.setAlignment(Label.CENTER);

Setting the Label's Background Color

To set the Label's background color, use the setBackground() method, like this: 

          
           
myLabel.setBackground(Color.colorName);
Where colorName is the name of the Color you want to set it to, such as RED, GREEN, or BLUE.

Setting the Label's Foreground Color

To set the Label's foreground color, use the setForeground() method, like this: 

          
           
myLabel.setForeground(Color.colorName);
Once again, colorName is the name of the Color you want to set it to, such as RED, GREEN, or BLUE.

Using a Label with a Listener

If you want to respond to user actions on a Label, you will need to add a Listener to the Label. To do this, use the addActionListener() method, like this: 

          
           
myLabel.addActionListener(listener);
Where listener is the name of the ActionListener object you want to add.

Example

This example shows how to create a Label, set its text and colors, and add it to a Frame. It also shows how to add an ActionListener to the Label. 

          
           
import java.awt.*;
import java.awt.event.*;

public class LabelExample {
    public static void main(String[] args) {
        // Create a frame
        Frame frame = new Frame("Label Example");
        
        // Create a label
        Label label = new Label("Label Text");
        
        // Set label background color
        label.setBackground(Color.BLUE);
        
        // Set label foreground color
        label.setForeground(Color.YELLOW);
        
        // Set label alignment
        label.setAlignment(Label.CENTER);
        
        // Create a listener
        LabelListener listener = new LabelListener();
        
        // Add listener to label
        label.addActionListener(listener);
        
        // Add label to frame
        frame.add(label);
        
        // Set frame size
        frame.setSize(200, 200);
        
        // Show frame
        frame.setVisible(true);
    }
}

class LabelListener implements ActionListener {
    public void actionPerformed(ActionEvent e) {
        System.out.println("Label clicked!");
    }
}
When the user clicks the Label, the message "Label clicked!" is printed to the console.
