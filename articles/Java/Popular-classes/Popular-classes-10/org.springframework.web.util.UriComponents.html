org.springframework.web.util.UriComponents

The org.springframework.web.util.UriComponents class is a utility class in the Spring Framework Web module used to encode and decode Uniform Resource Identifiers (URIs) and their components. It provides components to both construct URIs from scratch and to parse existing URIs into their components. The components of URIs can be encoded or decoded with percent-encoding to ensure they are correct and well-formed.

Examples

The following examples will demonstrate how to use the UriComponents class in various ways.

Encoding a URI from Components

The following example shows how to use the UriComponentsBuilder to construct a URI using its components.


          
           
UriComponents uriComponents = UriComponentsBuilder
    .newInstance()
    .scheme("https")
    .host("example.com")
    .path("/somePath")
    .queryParam("query", "value")
    .build();
String uri = uriComponents.toUriString();
// uri = "https://example.com/somePath?query=value"

The above example produces a URI with the scheme “https”, the host “example.com”, the path “/somePath”, and the query parameter “query=value”. This can be useful when constructing URIs programmatically.

Decoding a URI into Components

The following example shows how to use the UriComponentsBuilder to decode an existing URI into its components.


          
           
String uri = "https://example.com/somePath?query=value";
UriComponents uriComponents = UriComponentsBuilder.fromUriString(uri).build();
String scheme = uriComponents.getScheme(); // scheme = "https"
String host = uriComponents.getHost(); // host = "example.com"
String path = uriComponents.getPath(); // path = "/somePath"
MultiValueMap<String, String> queryParams = uriComponents.getQueryParams(); 
// queryParams = {query=[value]}

The above example parses the given URI into its components. The scheme is https, the host is example.com, the path is /somePath, and the query parameter is query=value.

Conclusion

The org.springframework.web.util.UriComponents class is a powerful utility class for parsing and constructing URIs. It can be used to encode and decode URIs and their components and can be used to produce well-formed and correctly encoded URIs.
