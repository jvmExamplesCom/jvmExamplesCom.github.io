Introduction to androidx.annotation.RestrictTo

Overview of androidx.annotation.RestrictTo

  • SUBCLASSES
  • TESTS

The TESTS parameter marks a class or method as restricted to be accessible only by tests. This is useful for testing components that are not accessible outside of a test environment.
