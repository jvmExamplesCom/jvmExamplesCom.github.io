org.springframework.core.env.PropertiesPropertySource

The PropertiesPropertySource is an implementation of Spring's PropertySource interface that serves as a bridge between the abstracted environment of Spring and the underlying operating system or runtime container specific properties. Its main purpose is to allow for easy lookup and retrieval of environment-specific property values. This article will explain how to use it, what are its key features, and provide code examples.

Why Use PropertiesPropertySource?

PropertiesPropertySource provides an easy way to access the underlying underlying properties of the environment, such as operating system or container-specific properties. It also allows users to easily lookup and retrieve property values using a single source of truth. Furthermore, it eliminates the need to create multiple, separate configuration files for different environments.

How To Use PropertiesPropertySource

To use PropertiesPropertySource, you must first create a new instance. You can create a new instance by instantiating a PropertiesPropertySource with a String or Properties object.


          
           
// Create a new instance of PropertiesPropertySource using a String
PropertiesPropertySource source = new PropertiesPropertySource("mySource", "someString");

// Create a new instance of PropertiesPropertySource using a Properties object
Properties properties = new Properties();
properties.put("foo", "bar");
PropertiesPropertySource source = new PropertiesPropertySource("mySource", properties);

You can then retrieve values from the PropertiesPropertySource object by calling the getProperty() method, passing in the key for the desired property. For example:


          
           
String fooValue = source.getProperty("foo");

This will return the value of “foo” from the Properties object.

Example Usage

Below is an example of how to use PropertiesPropertySource to read environment specific properties. We will be using a Properties object, but you could also use a String.


          
           
Properties properties = new Properties();
properties.put("database.host", "127.0.0.1");
properties.put("database.port", "3306");
properties.put("database.user", "root");

PropertiesPropertySource source = new PropertiesPropertySource("mySource", properties);
String dbHost = source.getProperty("database.host");

Database User: root
