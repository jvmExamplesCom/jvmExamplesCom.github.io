Introduction to Java ASTNode Class

The 

          
           
com.intellij.lang.ASTNode
class is a fundamental class of the IntelliJ Platform PSI (Program Structure Interface). It represents a single node in an Abstract Syntax Tree (AST) and provides basic APIs for getting and setting the node's types, text, and tree structure. This class is used across the entire IntelliJ Platform and is fundamental to understanding the parsing and manipulation of IntelliJ Platform source code.

Creating and Accessing an ASTNode Object

An ASTNode object can be created in various ways in the IntelliJ Platform. The simplest way to create one is by calling the 


          
           
ASTNode.createLeafElement()
method. This method takes a 
IElementType
parameter which is used to create the ASTNode object. Here is an example of creating an ASTNode for a Java language identifier: 

          
           
ASTNode nameNode = ASTNode.createLeafElement(JavaTokenType.IDENTIFIER, "name");

Once an ASTNode is created, it can be accessed in several ways. The most common way of accessing an ASTNode object is through its parent ASTNode. Every ASTNode has exactly one parent ASTNode instance and all ASTNodes are descendants of a root ASTNode.

Types of ASTNodes

ASTNode objects can have one of four types: Leaf, Composite, WhiteSpace, and Error. Leaf ASTNodes contain a single token of information, such as a keyword or operator. Composite ASTNodes contain a combination of Leaf ASTNodes, WhiteSpace ASTNodes, and Error ASTNodes. WhiteSpace ASTNodes represent non-tokens such as indentation and newlines. Error ASTNodes represent Lexer errors. Here are examples of each type of ASTNode in the Java language:


          
           
// Leaf ASTNode
ASTNode nameNode = ASTNode.createLeafElement(JavaTokenType.IDENTIFIER, "name");

// Composite ASTNode
ASTNode assignStatement = new ASTNode(JavaElementType.ASSIGNMENT_EXPRESSION);
assignStatement.addChild(ASTNode.createLeafElement(JavaTokenType.IDENTIFIER, "x"));
assignStatement.addChild(ASTNode.createLeafElement(JavaTokenType.EQ, "="));
assignStatement.addChild(ASTNode.createLeafElement(JavaTokenType.INTEGER_LITERAL, "10"));

// WhiteSpace ASTNode
ASTNode whitespaceNode = ASTNode.createWhiteSpace("\n\t\t");

// Error ASTNode
ASTNode errorNode = new ASTNode(JavaElementType.ERROR);

Manipulating ASTNodes

Once an ASTNode object is created, its contents can be manipulated using the ASTNode API. For example, 

getFirstChild()
, 
getLastChild()
and 
findChildByType()
can be used to access the children of an ASTNode. To add or remove children, 
addChild()
, 
removeChild()
, and 
replaceChild()
can be used. Here is an example of modifying an ASTNode: 

          
           
ASTNode assignStatement = new ASTNode(JavaElementType.ASSIGNMENT_EXPRESSION);
assignStatement.addChild(ASTNode.createLeafElement(JavaTokenType.IDENTIFIER, "x"));
assignStatement.addChild(ASTNode.createLeafElement(JavaTokenType.EQ, "="));
assignStatement.addChild(ASTNode.createLeafElement(JavaTokenType.INTEGER_LITERAL, "10"));

// Replace integer literal with a string literal
assignStatement.replaceChild(assignStatement.findChildByType(JavaTokenType.INTEGER_LITERAL), ASTNode.createLeafElement(JavaTokenType.STRING_LITERAL, "\"hello\""));

Conclusion

The ASTNode class is a fundamental element of the IntelliJ Platform PSI. Understanding how to create and manipulate ASTNode objects is essential to understanding how the IntelliJ Platform parses and manipulates source code.
