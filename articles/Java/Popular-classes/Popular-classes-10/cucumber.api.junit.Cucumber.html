Writing Features

Now that your environment is setup, we can start writing our features in Gherkin syntax. A feature is made up of one or more scenarios, which describe the behavior of the system. The Given statements describe the initial state of the system, the When statements describe the action taken on the system and the Then statements describe the expected outcome.
