What is the OreDictionary?

The OreDictionary is a dictionary of all registered items in the game. It stores each item by its name, which is used when searching for the item. How Does the OreDictionary Work?

The OreDictionary works by keeping track of a set of ore-type items. When an item is registered to the OreDictionary, it's given a unique string ID, which is used to identify it when searching.
