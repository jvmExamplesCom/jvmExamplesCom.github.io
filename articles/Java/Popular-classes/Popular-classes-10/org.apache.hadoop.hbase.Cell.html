Introduction to org.apache.hadoop.hbase.Cell

The org.apache.hadoop.hbase.Cell class represents an HBase cell, a combination of a row key, column family, qualifier, timestamp, type and value, which is used by the Apache HBase database.   • type: The type of the cell.
