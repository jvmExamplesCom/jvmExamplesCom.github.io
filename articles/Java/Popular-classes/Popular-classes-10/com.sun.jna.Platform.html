com.sun.jna.Platform Overview

The com.sun.jna.Platform class provides features for identifying the underlying operating system and architecture of the platform it is currently running on. In addition, it is also possible to detect the given platform's current language, timezone, and environment settings. This can be useful when developing applications that need to respect user preferences and regional nuances.
