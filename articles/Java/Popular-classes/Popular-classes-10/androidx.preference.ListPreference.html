androidx.preference.ListPreference

AndroidX ListPreference is an Android library for displaying a list of items that can be selected by the user. It is part of the AndroidX Preference library and can be used to create a customizable list of selectable items. This is useful when building applications with user preferences or settings that need to be presented in a list format.

ListPreference utilizes the XML attributes, as well as the AndroidX PreferenceScreen attribute to define the layout and values of the list. To use ListPreference, you must include the following in your XML:


          
           
<PreferenceScreen xmlns:android="http://schemas.android.com/apk/res/android"
    android:key="list_preference" >
  <androidx.preference.ListPreference
      android:key="list_preference_key"
      android:title="Choose an Option" 
      android:summary="Pick one from the list" 
      android:entries="@array/list_preference_values" 
      android:entryValues="@array/list_preference_values_keys" />
</PreferenceScreen>

You can also set the values of the list items in an array resource and reference it in the XML. Here's an example of a list preference resource with two items:


          
           
<string-array name="list_preference_values">
    <item>Option 1</item>
    <item>Option 2</item>
</string-array>

<string-array name="list_preference_values_keys">
    <item>key1</item>
    <item>key2</item>
</string-array>

To retrieve the selected item's value, you can use the following code:


          
           
SharedPreferences sharedPreferences = PreferenceManager.getDefaultSharedPreferences(context);
String value = sharedPreferences.getString("list_preference_key", null);

The first parameter is the key of the ListPreference, and the second is the default value in case none is selected. If a user has selected an item from the list, the corresponding key of that item will be returned. Thus, if the user selects option 1, it will return "key1".

ListPreference also has several other methods that can be used to interact with the list. These methods include getEntries(), which will return the array of entries stored in the XML, and getEntryValues(), which will return the array of entry values stored in the XML. Both of these are useful for programmatically accessing the data stored in the ListPreference.
