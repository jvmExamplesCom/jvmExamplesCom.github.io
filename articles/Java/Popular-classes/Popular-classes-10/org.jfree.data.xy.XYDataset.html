Introduction to org.jfree.data.xy.XYDataset

org.jfree.data.xy.XYDataset is a Java class found within the JFreeChart chart library that provides an interface for representing a collection of (x, y) items. XYDataset provides methods for retrieving the minimum and maximum values of the x and y coordinates, as well as methods for obtaining a list of unique x-values and a list of unique y-values.

Code Examples

The following example shows how to use the XYDataset class to create a simple chart. The code creates an anonymous implementation of the XYDataset interface, which defines the data points that will be plotted on the chart.


          
           
XYDataset dataset = new XYDataset() {
    @Override
    public int getSeriesCount() {
        return 1;
    }
    @Override
    public int getItemCount(int series) {
        return 5;
    }
    @Override
    public Number getX(int series, int item) {
        return item;
    }
    @Override
    public Number getY(int series, int item) {
        return item * item;
    }
};

The code below creates a simple chart using the XYDataset, setting all chart properties such as chart title and axes labels.


          
           
JFreeChart chart = ChartFactory.createXYLineChart(
    "My Simple Chart",
    "X Axis",
    "Y Axis",
    dataset,
    PlotOrientation.VERTICAL,
    true,
    false,
    false);
