Understanding org.greenrobot.greendao.database.Database

DatabaseConfiguration configuration = new DatabaseConfiguration();   
Database database = Database.getInstance(configuration);
Once you have an instance of the Database class, you can open a connection to the desired database by calling the Database#open() method. String jdbcUrl = "jdbc:sqlite:my_database.db";
database.open(jdbcUrl);

Accessing a Database

This method takes a SQL query string as its argument and returns an Iterable object containing the results of the query. For example, to query all the records from a table called “users” in an SQLite database, you can use the following code: 

          
           
Iterable<Record> resultSet = database.query("SELECT * from users");
The result of the query will be stored in a Record object. The Record object contains the data for each column returned by the query. You can access the data from a Record object by calling the Record#get() method, providing the column name as the argument.

Updating a Database
