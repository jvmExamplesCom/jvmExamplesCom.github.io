Introduction to org.apache.http.impl.client.BasicAuthCache

The org.apache.http.impl.client.BasicAuthCache class is a utility class that provides the ability to cache authentication credentials for a given HTTP host in order to use those credentials when sending requests to the same host. The BasicAuthCache class can be used to store the credentials for a host in memory and use them when sending requests to the same host.
