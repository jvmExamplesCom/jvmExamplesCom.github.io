What is io.netty.handler.ssl.SslProvider?

io.netty.handler.ssl.SslProvider is an interface for providing secure socket layer (SSL) protocol implementations for use with Netty. This interface provides a single method for getting an instance of SSLEngine by passing in an SslContext object.
