Introduction to org.springframework.web.servlet.support.ServletUriComponentsBuilder

The org.springframework.web.servlet.support.ServletUriComponentsBuilder class provides an easy way to build URLs in a web application. It is part of the Spring Web MVC framework and enables the construction of URLs based on the request path, query parameters, and the “host” portion of the URL.

This class is typically used in conjunction with the RequestMapping annotation and is used to construct URLs for view controllers. This allows for dynamic URLs with customizable parameters.

Using ServletUriComponentsBuilder

ServletUriComponentsBuilder provides several methods for constructing URLs, from basic parameters that define the full URL all the way to advanced ones that allow for further customization. The most commonly used methods include 

fromContextPath()
, which allows you to set the context path of the URL; 

          
           
fromServletMapping()
, which allows you to set the servlet mapping of the URL; and 
queryParam()
, which allows you to add query parameters to the URL.

The following example shows how to use ServletUriComponentsBuilder to construct a URL with a context path, a servlet mapping, and a query parameter:


          
           
String url = ServletUriComponentsBuilder
        .fromContextPath("/myapp")
        .fromServletMapping("/servlet")
        .queryParam("param1", "value1")
        .build().toUriString();
This code generates the following URL string: 

          
           
/myapp/servlet?param1=value1
