Java.security.spec.ECPoint

Java.security.spec.ECPoint is a class in the Java security API that specifies an elliptic curve point on a given elliptic curve, using affine coordinates.

Overview of ECPoint

ECPoint is an immutable class that can be used to represent points on elliptic curves over finite fields. It has two coordinate fields: x and y, which represent the affine coordinates of the point on the elliptic curve.

The point may be normalized by setting an affine point on the curve, or it may be represented as an encoded byte array.
