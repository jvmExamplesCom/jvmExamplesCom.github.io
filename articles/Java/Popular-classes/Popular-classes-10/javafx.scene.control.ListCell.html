JavaFX Scene Control ListCell

JavaFX Scene Control ListCell is a class that provides the base class for list items in a ListView control. It provides basic cell functionality, including text and graphic content, as well as selection and focus behavior.

Create a ListCell

The following example illustrates how to create a simple ListCell with text content: 

          
           
ListCell<String> cell = new ListCell<>();
cell.setText("My Cell Text");
The following example illustrates how to create a ListCell with graphic content: 

          
           
ImageView imageView = new ImageView(new Image("myImage.png"));
ListCell<String> cell = new ListCell<>(imageView);
The following example illustrates how to create a ListCell with both text and graphic content: 

          
           
ImageView imageView = new ImageView(new Image("myImage.png"));
ListCell<String> cell = new ListCell<>("My Cell Text", imageView);

Update the Content of a ListCell

By default, when a ListCell is created, it will not update its contents automatically. To update the content of a ListCell, we must override the updateItem() method. The updateItem() method is called whenever there is a change to the ListCell's content, such as when a new item is added to the list. The following example illustrates how to override the updateItem() method to update the content of a ListCell: 

          
           
ListCell<String> cell = new ListCell<String>() {
    @Override
    protected void updateItem(String item, boolean empty) {
        super.updateItem(item, empty);

        if (item != null) {
            setText(item);
        } else {
            setText("");
        }
    }
};

Set the Selection and Focus of a ListCell

A ListCell can provide selection and focus behavior by setting the selection or focus property to true. The following example illustrates how to set the selection and focus of a ListCell: 

          
           
ListCell<String> cell = new ListCell<String>();
cell.setSelection(true);
cell.setFocus(true);
In the above example, the selection and focus properties of the ListCell are set to true, indicating that this ListCell should be considered selected and focused.

Disable a ListCell

A ListCell can be disabled by setting the disable property to true. The following example illustrates how to disable a ListCell: 

          
           
ListCell<String> cell = new ListCell<String>();
cell.setDisable(true);
In the above example, the disable property of the ListCell is set to true, indicating that this ListCell should be considered disabled.
