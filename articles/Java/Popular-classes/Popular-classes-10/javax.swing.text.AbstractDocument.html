javax.swing.text.AbstractDocument

The 

          
           
javax.swing.text.AbstractDocument
class is a Java class providing an abstract implementation of the Document interface. It provides a framework for managing text content, and editing capabilities, which can be extended by concrete subclasses. It contains methods that manage document structures, document views, and document filter chains, as well as those that manage document changes and the attributes of character content in the document.

Document Structures

The document structure of a document is determined by its content elements, which are organized in a tree structure. The root of the document structure is created with the 

createDefaultRoot()
method, which creates an element containing the entire document structure. By default, it is an element of type 

          
           
ElementSpec.ContentElementName
, but it can be any subclass of the 
ElementSpec
class. For each subsequent element in the document structure, create the element using the 
createLeafElement()
method. It takes three arguments: element name, attributes, and parent element.

Document Views

The views of a document are created by the 
createView()
method. It takes three arguments: an element to view, style information, and positioned location. The element is the starting point of the view; the style gives general formatting information to the view, such as font size, color, etc.; and the position is the start point, or end point, of the view. The view is then added to the document using the 
insertUpdate()
method.

Document Filters

Document filters are used to intercept changes made to documents and to filter out undesired changes. Filters are added to the document using the 

addDocumentFilter()
method. When a change is made to the document, the 
changedUpdate()
method is called and it checks the document against each filter. If the filter returns true, the change is allowed; otherwise, it is blocked.

Document Changes

Changes to the document are managed by the 
replace()
method and by notification calls to the 
insertUpdate()
and 
removeUpdate()
methods. The 
replace()
method allows a range of text within the document to be replaced by another string, whereas the 
insertUpdate()
and 
removeUpdate()
methods allow for single character insertion or deletion at a specified location.

Document Attributes

Character attributes are inspected with the 
getAttributeKeys()
method. This returns an array of 
AttributeSet
objects, which contain the name and value of each attributed character. To set attributes, use the 

          
           
setCharacterAttributes()
method.
