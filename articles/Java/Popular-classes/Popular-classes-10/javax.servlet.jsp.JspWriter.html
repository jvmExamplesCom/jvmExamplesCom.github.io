javax.servlet.jsp.JspWriter

The javax.servlet.jsp.JspWriter class provides the interface for writing dynamically generated content in a JSP page. The JspWriter object is initialized by the JSP container and is made available as an implicit variable named out. It is generally used for output content generation. For example, HTML tags or text can be written directly to the JSP page using the print() method.

Getting the Instance of javax.servlet.jsp.JspWriter

The JspWriter instance can be obtained from the JspContext instance by calling the getOut() method. The following code snippet shows how to get the JspWriter instance from the JspContext instance.


          
           
JspWriter out = pageContext.getOut();

Writing Content Using javax.servlet.jsp.JspWriter

The JspWriter class provides many methods to write the content to the response stream. The most commonly used methods are the print() and println() methods. The print() method writes the specified string to the response stream without any line terminator while the println() method writes the specified string to the response stream with a line terminator.


          
           
out.print("Hello Javatpoint..");  
out.println("How are you?");

The content written to the response stream can be HTML tags too. In the following code snippet, we have written two HTML tags to the response stream to be displayed on the browser.


          
           
out.print("<b>Hello Javatpoint</b>");  
out.println("<em>How are you?</em>");

Methods of javax.servlet.jsp.JspWriter Class

The javax.servlet.jsp.JspWriter class provides several methods to write content to the response stream. Some of the commonly used methods are listed below:

  • print(): This method is used to write the specified string to the response stream without any line terminator.
  • println(): This method is used to write the specified string to the response stream with a line terminator.
  • write(): This method is used to write the specified character array to the response stream.
  • newLine(): This method is used to write a line terminator to the response stream.
  • clear(): This method is used to clear the current buffer.

Example

In this example, we will be writing a simple HTML page using the JspWriter class. The below code snippet shows how to write an HTML page to the response stream. 


          
           
// Get the JspWriter instance  
JspWriter out = pageContext.getOut();  

// Writing the HTML page to the response stream  
out.println("<!DOCTYPE html>");  
out.println("<html>");  
out.println("<head>");  
out.println("<title>Sample Page</title>");  
out.println("</head>");  
out.println("<body>");  
out.println("Welcome to Javatpoint");  
out.println("</body>");  
out.println("</html>");

Closing the JspWriter Stream

The javax.servlet.jsp.JspWriter object must be closed after using it. It is important to note that we cannot close the JspWriter object manually. It will be automatically closed by the JSP container when the JSP page is destroyed.
