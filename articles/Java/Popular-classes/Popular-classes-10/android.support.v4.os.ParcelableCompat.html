What is the ParcelableCompat Class?

The ParcelableCompat class is part of the Android Support Library. It provides an API for objects that need to be parceled (converted to a byte array) for storage or transport. Parcelables are used to store and transfer data between activities, services, and processes on Android. The ParcelableCompat class allows apps to access Parcelable objects in a backwards compatible way, meaning they can be used by apps written for Android 5.0 (API level 21) and up.
