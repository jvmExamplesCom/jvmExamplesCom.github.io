org.bukkit.potion.PotionEffectType

The org.bukkit.potion.PotionEffectType class is a type of effect that can be applied to an entity through the use of potions. It defines all the possible effects that can be applied, and provides methods for manipulating them. This class is part of the Bukkit API.

Constructors


          
           
public PotionEffectType(String name)
Creates a PotionEffectType object from the specified name. The name should match one of the registered types in the registry. 

          
           
public PotionEffectType(int id, String name)
Creates a PotionEffectType object with the specified id and name. The name should match one of the registered types in the registry.

Methods


          
           
public static PotionEffectType getByName(String name)
Returns the potion effect type which matches the given name, or null if not found. 

          
           
public static Set<potioneffecttype> getAll() 
</potioneffecttype>

public String getName()
Returns the name of this potion effect type. 

          
           
public boolean isInstant()
Returns true if this potion effect type is an instant effect, or false if it is an ongoing effect. 

          
           
public int getId()
Returns the unique ID of this potion effect type.
