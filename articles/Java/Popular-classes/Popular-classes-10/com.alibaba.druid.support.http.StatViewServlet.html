Understanding com.alibaba.druid.support.http.StatViewServlet

The 

          
           
com.alibaba.druid.support.http.StatViewServlet
class is a servlet used to provide an administration interface for the Druid framework. It is used to monitor and manage the performance, data sources, threads, and memory of a web application. It provides an easy-to-use graphical interface that allows users to quickly view and manage the status of their web applications.

Getting Started with StatViewServlet

To get started with 
StatViewServlet
, simply add the following code to your web application's web.xml configuration file: 

          
           
<servlet>
  <servlet-name>DruidStatView</servlet-name>
  <servlet-class>com.alibaba.druid.support.http.StatViewServlet</servlet-class>
</servlet>
<servlet-mapping>
  <servlet-name>DruidStatView</servlet-name>
  <url-pattern>/druid/*</url-pattern>
</servlet-mapping>

Once the code is added, you can access the administration interface by navigating to http://localhost/druid/.

Configuration and Customization

The 
StatViewServlet
class can be customized using the following parameters: 
  • resetEnable
    – Enables or disables the reseting of Druid's internal statistics. Defaults to 
    false
    .
  • allow
    – A comma-separated list of IP addresses and subnets which are allowed to access the StatViewServlet. Defaults to 
    127.0.0.1
    . 
  • deny
    – A comma-separated list of IP addresses and subnets which are not allowed to access the StatViewServlet. Defaults to 
    null
    .
  • loginUsername
    – The username required to access the StatViewServlet. Defaults to 
    null
    .
  • loginPassword
    – The password required to access the StatViewServlet. Defaults to 
    null
    .
  • sessionStatMaxCount
    – The maximum number of sessions that will be tracked in the StatViewServlet. Defaults to 
    1000
    .
  • 
            
             
principalSessionName
    – The session attribute name containing the user principal. Defaults to 
    null
    .
  • profileEnable
    – Enables or disables profiling of the StatViewServlet. Defaults to 
    true
    .

The parameters can be set either in web.xml, or programmatically:


          
           
StatViewServlet druidStatView = new StatViewServlet();
druidStatView.setAllow("127.0.0.1");
druidStatView.setDeny(null);
druidStatView.setLoginUsername("username");
druidStatView.setLoginPassword("password");
druidStatView.setResetEnable(false);
druidStatView.setSessionStatMaxCount(1000);
druidStatView.setPrincipalSessionName(null);
druidStatView.setProfileEnable(true);
