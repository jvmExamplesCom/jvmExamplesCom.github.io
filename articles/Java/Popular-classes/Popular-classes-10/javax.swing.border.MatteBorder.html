The 

          
           
javax.swing.border.MatteBorder
class provides a simple decorative border consisting of an arbitrary number of line segments, each with a specified color and thickness. MatteBorder can be used to draw either a line or an empty border. It is a subclass of AbstractBorder and acts as an easy way to create a simple, yet attractive looking border that can be used to enhance the appearance of graphical user interfaces.

The MatteBorder constructor takes four arguments; top, left, bottom, and right. Each argument is an integer representing the width of the border in pixels. The following code example creates a new MatteBorder with a top, left, bottom, and right border each 30px wide:


          
           
MatteBorder myBorder = new MatteBorder(30, 30, 30, 30);

MatteBorder also has a two argument variant which takes an integer for the thickness of all borders, and one color. The following code example creates a new MatteBorder with a single border thickness of 30px and a black border color:


          
           
MatteBorder myBorder = new MatteBorder(30, Color.black);

MatteBorder also has a one argument variant which takes a single Color for the border. This creates a border with the default thickness of 1px. The following code example creates a new MatteBorder with a pink border color:


          
           
MatteBorder myBorder = new MatteBorder(Color.pink);

MatteBorders can also be combined with other kinds of borders. The following code example combines a MatteBorder and LineBorder to create a border with an inner white 10px line and a border color of black.
