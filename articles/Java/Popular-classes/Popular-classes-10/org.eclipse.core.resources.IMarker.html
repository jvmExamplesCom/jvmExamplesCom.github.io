Introduction to org.eclipse.core.resources.IMarker

The 

          
           
org.eclipse.core.resources.IMarker
interface is a marker interface used in the Eclipse platform. Markers are objects that represent annotations, bookmarks, breakpoints, or other similar objects that can be added to resources (files and folders) in the workspace. This interface allows the user to create, modify, and delete markers.

Creating a New Marker

Creating a new marker object is done using the 

          
           
MarkerResource.createMarker()
method. The following code example creates a new marker for a given resource name. 

          
           
String resourceName = "myFile.txt";
IMarker marker = MarkerResource.createMarker(resourceName);
If the marker was successfully created, then the 

          
           
marker
object will contain a reference to the newly created marker. If the marker failed to be created, then the 

          
           
marker
object will be 

          
           
null
.

Modifying an Existing Marker

MarkerResource.setAttribute()
method. This method takes two parameters: the attribute name and the attribute value. The following code example sets the "description" attribute of a given marker: 

          
           
String attributeName = "description";
String attributeValue = "This is a description of the marker";
marker.setAttribute(attributeName, attributeValue);

Deleting an Existing Marker

An existing marker can be deleted using the 

          
           
MarkerResource.deleteMarker()
method. This method takes one parameter: the marker object to be deleted. The following code example deletes a marker from the workspace:
