Introduction to org.eclipse.core.resources.IMarker

The 

          
           
org.eclipse.core.resources.IMarker
interface is a marker interface used in the Eclipse platform. Markers are objects that represent annotations, bookmarks, breakpoints, or other similar objects that can be added to resources (files and folders) in the workspace. This interface allows the user to create, modify, and delete markers.

Creating a New Marker

Creating a new marker object is done using the 

          
           
MarkerResource.createMarker()
method. The following code example creates a new marker for a given resource name. 

          
           
String resourceName = "myFile.txt";
IMarker marker = MarkerResource.createMarker(resourceName);

marker

marker
object will be 

          
           
null
.

Deleting an Existing Marker
