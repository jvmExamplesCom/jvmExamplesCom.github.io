The LookupElementBuilder class is a useful utility class that facilitates creating and configuring LookupElements, which are used with IntelliJ's code completion feature. create()

String
argument and returns a LookupElement for it. 

          
           
LookupElement element = LookupElementBuilder.create("myString");
