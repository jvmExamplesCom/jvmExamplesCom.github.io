The XYLineAndShapeRenderer class is a specialized renderer used to draw lines and shapes on a XYPlot. It can be used to create a range of different types of line-shaped plots, including 2D line plots, 3D line plots, scatter plots, and area plots. In addition, it supports customization of line and shape styles, as well as support for multiple data series.

Creating an XYLineAndShapeRenderer

Creating an XYLineAndShapeRenderer is straightforward. The following code creates a new instance of the renderer:


          
           
XYLineAndShapeRenderer renderer = new XYLineAndShapeRenderer();

Setting Line and Shape Styles

The XYLineAndShapeRenderer class provides a range of methods for setting line and shape styles. For example, the following code sets the line and shape styles for a series index 0:


          
           
renderer.setBaseLinesVisible(true, 0);
renderer.setBaseShapesVisible(true, 0);
renderer.setSeriesStroke(0, new BasicStroke(3.0f));

The above code set the series' lines visible, sets the shapes visible, and sets the stroke width to 3 pixels.

renderer.setBackgroundPaint(Color.white);

Adding Data Series

double[] xData = {1.0, 2.0, 3.0};

The above code adds a series with the label “My Series”, and three data points (with x values 1, 2, and 3, and y values 4, 5, and 6).
