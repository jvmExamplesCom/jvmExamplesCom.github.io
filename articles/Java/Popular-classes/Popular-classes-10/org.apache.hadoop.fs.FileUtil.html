org.apache.hadoop.fs.FileUtil

org.apache.hadoop.fs.FileUtil is a utility class providing static methods for interacting with files and directories in the Hadoop File System. This class provides basic file and directory manipulation functions such as creating and deleting files, listing files in a directory, and copying and moving files.

Creating a File or Directory

The FileUtil class provides two static methods for creating a new file or directory: create() and mkdir(). The create() method creates a new file and write data from an InputStream to it. It takes three arguments:


          
           
create(FileSystem fs, Path path, InputStream in, boolean overwrite, short permission, Progressable progress)

The mkdir() method creates a new directory. It takes three arguments:


          
           
mkdir(FileSystem fs, Path path, FsPermission permission, boolean createParent)

Deleting a File or Directory

The FileUtil class provides two static methods for deleting a file or directory: delete() and delete_recursive(). The delete() method deletes a file or an empty directory. It takes two arguments: 


          
           
delete(FileSystem fs, Path path, boolean recursive)

The delete_recursive() method deletes all the contents of a directory including subdirectories and files. It takes one argument: 


          
           
delete_recursive(FileSystem fs, Path path)

Copying Files and Directories

The FileUtil class provides two static methods for copying files and directories: copy() and copyMerge(). The copy() method copies a file or directory from one location to another. It takes three arguments: 


          
           
copy(FileSystem srcFS, Path src, FileSystem dstFS, Path dst, boolean deleteSource, Configuration conf)

The copyMerge() method copies multiple files from one location to another and merges them into a single file. It takes five arguments: 


          
           
copyMerge(FileSystem srcFS, Path srcDir, FileSystem dstFS, Path dstFile, boolean deleteSource, Configuration conf, String addString)

Moving Files and Directories

The FileUtil class provides two static methods for moving files and directories: move() and rename(). The move() method moves a file or directory from one location to another. It takes four arguments: 


          
           
move(FileSystem srcFS, Path src, FileSystem dstFS, Path dst, boolean deleteSource, Configuration conf)

The rename() method renames a file or directory. It takes three arguments: 


          
           
rename(FileSystem fs, Path src, Path dst, Options.Rename... options)

Listing File or Directory Contents

The FileUtil class provides a static method for listing a file or directory's contents: listFiles(). This method lists the contents of a directory. It takes four arguments: 


          
           
listFiles(FileSystem fs, Path path, boolean recursive)

The method optionally recursively lists files in all subdirectories.
