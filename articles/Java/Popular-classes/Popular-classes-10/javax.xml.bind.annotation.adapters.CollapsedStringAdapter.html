javax.xml.bind.annotation.adapters.CollapsedStringAdapter

javax.xml.bind.annotation.adapters.CollapsedStringAdapter is a class in the Java package javax.xml.bind.annotation.adapters. This class provides XML bindings for strings containing whitespace, such as spaces and tabs.

Example 1: Using CollapsedStringAdapter

The following is an example of using the CollapsedStringAdapter class to bind a string containing whitespace. 


          
           
@XmlJavaTypeAdapter(CollapsedStringAdapter.class)
String name = "John    Doe";

In this example, the CollapsedStringAdapter class is used to bind a String containing multiple whitespace characters. This will result in a String with only single whitespace characters.

Example 2: Customizing CollapsedStringAdapter

The CollapsedStringAdapter class can also be customized to change its behavior. The following example demonstrates how to customize the CollapsedStringAdapter class. 


          
           
@XmlJavaTypeAdapter(value=CollapsedStringAdapter.class, type=String.class, 
 									collapseWhitespace=true)
String name = "John    Doe";

In this example, the CollapsedStringAdapter class is customized to collapse all whitespace characters. This means that when the String is bound, it will be collapsed into a single whitespace character.
