Introduction to Java Interface IInventory

Java has a rich library of APIs, interfaces, classes and packages. One such interface is the IInventory. It is found in the net.minecraft.inventory package. This interface provides an object-oriented way of dealing with an inventory. The IInventory interface provides three key methods: getInventoryStackLimit(), isItemValidForSlot() and getInventoryName(). The getInventoryStackLimit() method returns the maximum number of items that can be held in a single inventory slot. For example:

The isItemValidForSlot() method takes an item as a parameter and returns true or false, depending on whether the item can be added to the given inventory slot. String invName = inventory.getInventoryName();
