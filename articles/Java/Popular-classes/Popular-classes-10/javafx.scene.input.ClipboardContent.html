Introduction to javafx.scene.input.ClipboardContent

The 

          
           
javafx.scene.input.ClipboardContent
class is a part of the JavaFX API, and it provides methods and fields for managing a data structure used by the system clipboard. The content is stored in a 

          
           
HashMap<DataFormat, Object>
and arranged according to the type of 

          
           
DataFormat
associated with each 

          
           
Object
.

javafx.scene.input.ClipboardContent
. Here are some of them:

put()

put()

        this.data.put(key, value);

key

DataFormat

value

get()

Object

DataFormat
.

        return this.data.get(key);
    }

clear()

clear()
