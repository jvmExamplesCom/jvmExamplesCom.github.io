org.apache.tools.ant.DirectoryScanner

The org.apache.tools.ant.DirectoryScanner is a class in the Apache Ant library used to scan a directory tree for files matching certain criteria. It can be used to provide an efficient mechanism for locating files matching specific criteria, such as file types or filenames.

Usage

The DirectoryScanner can be used on its own or with other Apache Ant classes, such as Java task. To use it, create an instance of the DirectoryScanner and set the start directory. For example:


          
           
DirectoryScanner ds = new DirectoryScanner();
ds.setStartDirectory("my/start/directory");
Or if initializing from a project, use: 

          
           
DirectoryScanner ds = new DirectoryScanner(project);
ds.setStartDirectory("my/start/directory");

Once the start directory is set, the files can be retrieved by using the 

getIncludedFiles()
method. This returns an array of 
String
s containing the relative paths of the files found. For example: 

          
           
String[] includedFiles = ds.getIncludedFiles();

Filters

The DirectoryScanner makes use of filters to restrict which files are returned. There are various filters available, such as 

NameFilter
, 
TypeFilter
and 
SizeFilter
. For example, to limit the results to those containing the pattern 
"*.txt"
, the following can be used: 

          
           
NameFilter nameFilter = new NameFilter("*.txt");
ds.addFilter(nameFilter);

For a full list of filters and usage examples, please refer to the Apache Ant Manual.

Examples

In this example, we'll create a basic 
DirectoryScanner
and make use of a 
TypeFilter
to filter the results to only include 
.java
files. First, initialize the 
DirectoryScanner
with a start directory and add the 
TypeFilter
. We can then call the 
getIncludedFiles()
method and loop through the results, printing each file path. 

          
           
DirectoryScanner ds = new DirectoryScanner();
ds.setStartDirectory("my/start/directory");

TypeFilter typeFilter = new TypeFilter("java");
ds.addFilter(typeFilter);

}
