Introduction to android.arch.persistence.room.Query

The android.arch.persistence.room.Query class provides methods to write SQL queries in a more convenient way and access databases within the Android platform. This class is part of a whole set of features provided by the Android Architecture Components library, a collection of libraries that have been designed to help improve the development of apps in the Android platform.

Querying Data with android.arch.persistence.room.Query

To query data with android.arch.persistence.room.Query, we first need to instantiate a RoomDatabase object and then use its query() method to create a Query object. We can then use this Query object to define the SQL query using the following methods: 

          
           
// Create a RoomDatabase object
RoomDatabase db = Room.databaseBuilder(context, MyDatabase.class, "db-name").build();

// Create a Query object
Query query = db.query();
We can then specify the SQL query using the following methods provided by the Query class:
  • select(): to specify the columns of the table to return,
  • from(): to specify the name of the table,
  • where(): to specify the conditions,
  • orderBy(): to specify the sorting order, and
  • limit(): to specify the maximum number of rows to return.
For example, if we want to get all records from the Employee table sorted by employee name in descending order, we can write the following query: 

          
           
Query query = db.query()
    .select("*")
    .from("Employee")
    .orderBy("name DESC");
To execute this query, we use the following method provided by the Query class: 

          
           
List<Employee> employees = query.execute();
The execute() method will return a List of objects with the results of the query.

Building Queries with Parameters

When building queries, we can also pass parameters to the query parameters to be used to specify conditions or filter data. These parameters must be passed as an array of Objects when building the query. For example, if we want to filter our query by employee name, we can use the following code: 

          
           
String employeeName = "John";

Query query = db.query()
    .select("*")
    .from("Employee")
    .where("name = ?")
    .orderBy("name DESC");

List<Employee> employees = query.execute(employeeName);
In this example, the value of the parameter employeeName is passed as an argument to the execute() method and it is used in the query as a placeholder (?) which will be replaced with the actual value.
