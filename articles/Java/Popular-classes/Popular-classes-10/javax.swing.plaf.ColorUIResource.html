javax.swing.plaf.ColorUIResource is a UI resource class that provides colors for use in the user interface of Java Swing programs. The class makes it easy to use consistent colors throughout the GUI, and also makes it easy to alter the colors of the UI elements at run time. Using ColorUIResource

ColorUIResource myColor = new ColorUIResource(0x80A0FF);

Once you have a ColorUIResource object, you can pass it as an argument wherever a Color object is needed. For example, if you want to set the background of a JButton to be the color created above, you can use the setBackground() method like this:


          
           
myButton.setBackground(myColor);

Dynamic Color Changes
