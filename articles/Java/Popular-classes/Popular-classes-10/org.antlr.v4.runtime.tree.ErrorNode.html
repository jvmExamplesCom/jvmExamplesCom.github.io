Introduction to org.antlr.v4.runtime.tree.ErrorNode

org.antlr.v4.runtime.tree.ErrorNode is a class from the ANTLR4 library that provides an implementation of the ParseTree interface for representing errors during parsing. It is used as a placeholder for an erroneous token or subtree in a parse tree.

Structure of org.antlr.v4.runtime.tree.ErrorNode

ErrorNode also implements the ParseTree interface, which provides access to the ParserRuleContext parent and the child array.
