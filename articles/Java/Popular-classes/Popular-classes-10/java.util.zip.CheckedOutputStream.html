java.util.zip.CheckedOutputStream

The 

          
           
java.util.zip.CheckedOutputStream
class implements an output stream filter that updates a checksum of the data written to a stream. This can be used to verify the integrity of the data written to a stream.

Constructor


          
           
public CheckedOutputStream(OutputStream out, Checksum cksum)

Creates a new output stream that updates the checksum with the written data. The checksum class must implement the 


          
           
java.util.zip.Checksum
interface.

Methods


          
           
java.util.zip.CheckedOutputStream
provides the following methods: 
  • 
            
             
public void write(int b):
    Writes the specified byte to the underlying output stream and updates the checksum.
  • 
            
             
public void write(byte[] buf, int off, int len):
    Writes 
    len
    bytes from the specified byte array starting at offset 
    off
    to the underlying output stream and updates the checksum.
  • 
            
             
public void close():
    Closes the underlying output stream and updates the checksum.
  • 
            
             
public Checksum getChecksum():
    Returns the checksum associated with the stream.

Example

In the following example, we create a 
CheckedOutputStream
using an 
Adler32
checksum and write some data to it. We then print out the checksum at the end. 

          
           
import java.util.zip.Adler32;
import java.util.zip.CheckedOutputStream;
import java.io.ByteArrayOutputStream;

public class CheckedOutputStreamExample {
     public static void main(String[] args) throws Exception {
        ByteArrayOutputStream bos = new ByteArrayOutputStream();
        Adler32 checksum = new Adler32();
        CheckedOutputStream cos = new CheckedOutputStream(bos, checksum);

        String data = "Hello World";
        byte[] bytes = data.getBytes("UTF-8");

        // Write the data to the stream
        cos.write(bytes, 0, bytes.length);
        cos.flush();
        cos.close();

        System.out.println("Checksum: " + checksum.getValue());
    }
}
