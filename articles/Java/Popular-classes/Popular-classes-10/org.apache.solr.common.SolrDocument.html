org.apache.solr.common.SolrDocument

The org.apache.solr.common.SolrDocument class provides a convenient way to store, create and query documents for Apache Solr. It is a JavaBean-like interface, which can be used to define document fields, set and get values, set and get boost values, retrieve stored fields from the index, and put and delete documents from the index.

Solr provides an API for retrieving stored fields from the index. The API does not require the user to know anything about how the index is structured, and it is possible to retrieve fields from any document in the index.

SolrDocument doc = new SolrDocument();
