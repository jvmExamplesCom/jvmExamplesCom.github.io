Introduction to com.nineoldandroids.animation.AnimatorSet

AnimatorSet is a class used in the NineOldAndroids library that can be used to play animations sequentially or at the same time as one another. AnimatorSet can also be used to pause, resume, or end running animations and listens for when an animation has ended.

Constructor 


          
           
AnimatorSet animatorSet = new AnimatorSet();

Methods

playTogether(): The playTogether() method allows multiple Animators to be played together, at the same time. This method takes any number of Animator objects which will then be set to play at the same time.


          
           
Animator animatorA = ObjectAnimator.ofFloat(view, "translationX", 0, 100);
Animator animatorB = ObjectAnimator.ofFloat(view, "translationY", 0, 100);
AnimatorSet.playTogether(animatorA, animatorB);

playSequentially(): The playSequentially() method allows multiple Animators to be played sequentially, one after another. This method takes any number of Animator objects which will then be set to play in the order they were specified.


          
           
Animator animatorA = ObjectAnimator.ofFloat(view, "translationX", 0, 100);
Animator animatorB = ObjectAnimator.ofFloat(view, "translationY", 0, 100);
AnimatorSet.playSequentially(animatorA, animatorB);

pause(): The pause() method pauses all the currently running Animators in the AnimatorSet.

resume(): The resume() method resumes all the currently paused Animators in the AnimatorSet.
