org.elasticsearch.action.update.UpdateRequest

The org.elasticsearch.action.update.UpdateRequest class is an UpdateRequest used by Elasticsearch to update an existing document in the cluster and replace it with a new one.

Example Usage

The following example demonstrates how to use this class to update an existing document in the cluster:


          
           
UpdateRequest updateRequest = new UpdateRequest("index", "type", "id");
updateRequest.doc("field", "newValue");
UpdateResponse updateResponse = client.update(updateRequest).get();

In this example, we create a new UpdateRequest object for the specified index, type, and id. We then set the new value of the field with the doc() method and send it to the cluster using the client's update() method. Finally, we get the UpdateResponse back from the update() method.

Methods

The UpdateRequest class has several methods that can be used to customize the update process:

  • 
            
             
doc(String field, Object value)
    : Sets the field and value of the new document to be indexed.
  • 
            
             
timeout(long timeout)
    : Sets the timeout of the request.
  • 
            
             
retryOnConflict(int retryOnConflict)
    : Sets the number of times to retry the request if there is a conflict.
