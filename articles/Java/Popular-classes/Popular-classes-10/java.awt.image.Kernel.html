Java.awt.image.Kernel is an class used in image filtering from the package java.awt.image. It is a class used for manipulating an image by passing a matrix filter to it, used for tasks such as sharpening, blurring or embossing.

Constructors

The Kernel class provides two constructors : Kernel(int width, int height, float[] data) and Kernel(int width, int height, int[] data). The first constructor allows you to pass a float array with the filter values, while the second one uses integer values.

Methods
