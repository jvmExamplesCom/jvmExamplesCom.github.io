What is Retrofit2.http.PUT?

Retrofit2.http.PUT is an open-source library for making RESTful calls in Java. It is designed to simplify the process of making http calls in Java by providing convenient annotations to define the API endpoints and the request type. Defining an API Endpoint

An API endpoint is defined using an annotation. An example of defining an API endpoint using Retrofit2.http.PUT is given below:
