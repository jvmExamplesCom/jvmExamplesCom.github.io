FIFO


          
           
QueueProcessingType fifo = QueueProcessingType.FIFO;

LIFO


          
           
QueueProcessingType lifo = QueueProcessingType.LIFO;
Last-in-first-out (LIFO) principle means that the last image in the queue will be processed first. This is useful when processing large numbers of images, since it may speed up the loading of images by reducing the amount of time spent on processing each image.
