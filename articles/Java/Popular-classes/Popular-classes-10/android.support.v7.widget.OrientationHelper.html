What is android.support.v7.widget.OrientationHelper?

android.support.v7.widget.OrientationHelper is a class in the Android Support Library v7 that provides helper methods and constants related to Orientation (the direction in which something is pointing or facing). It allows you to easily get the current orientation, as well as the starting point, end point, and middle point between two orientations. It also provides utility methods dealing with rotation and other related operations.

Examples of Usage

Getting the Current Orientation

The 

          
           
getOrientation(Context context)
method can be used to retrieve the current orientation of the device. The following code example demonstrates how to use this method to get the current orientation and then display it in a Toast message: 

          
           
int orientation = OrientationHelper.getOrientation(getApplicationContext());
String orientationString = null;
switch (orientation) {
    case OrientationHelper.ORIENTATION_PORTRAIT:
        orientationString = "Portrait";
        break;
    case OrientationHelper.ORIENTATION_LANDSCAPE:
        orientationString = "Landscape";
}
Toast.makeText(getApplicationContext(),
            orientationString + " orientation", Toast.LENGTH_SHORT).show();
