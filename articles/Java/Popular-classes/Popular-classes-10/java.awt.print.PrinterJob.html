java.awt.print.PrinterJob

The java.awt.print.PrinterJob class enables Java applications to query and control the attributes of a particular print job. It also allows an application to initiate a print job, which will be further processed by a printer driver. This class is generally used with the java.awt.print package.

Creating a PrinterJob

The PrinterJob class provides convenient methods to create an instance of a PrinterJob. The createPrintJob() method allows creating a PrinterJob instance.


          
           
PrinterJob job = PrinterJob.createPrintJob();

Configuring Print Job Attributes

The PrinterJob class allows the application to configure attributes of the Print Job such as page format and printable area using the PageFormat and Printable interfaces. It also allows configuring of printer resolution etc.


          
           
PageFormat pf = job.defaultPage();
job.setPrintable(printable, pf);

job.print(); //initiate the print job

Print Quality or Resolution

The PrinterJob object allows setting the resolution of the printer for the print job. The setResolution() method takes one of the predefined resolutions constants (eg. PRINTRESOLUTION_HIGH, PRINTRESOLUTION_LOW, etc.) as argument. The resolution can be queried using the getResolution() method.


          
           
job.setResolution(PrinterJob.PRINTRESOLUTION_HIGH);
int res = job.getResolution(); //res will contain the current resolution

Printing Multiple Copies

The PrinterJob class allows printing multiple copies of the same document. This is done by passing an integer value to the setCopies() method. This value dictates the number of copies that need to be printed.


          
           
job.setCopies(3);
int cop = job.getCopies(); //cop will contain the number of copies set to print

Print Dialog

The showPrintDialog() method launches a print dialog box from which the user can set additional parameters before printing. This method returns true if the user clicks the OK button in the print dialog else it returns false.


          
           
boolean result = job.showPrintDialog();
if (result) {
    job.print(); //print job after the user has accepted changes
} else {
    job.cancel(); //cancel the print job
}
