Java — com.intellij.psi.PsiReference

com.intellij.psi.PsiReference is part of the IntelliJ Java API and provides a way to programmatically reference elements in a Java source file. It allows code to be traced back to its origin and is beneficial for debugging, error checking, and other tasks. This article will explain how to use com.intellij.psi.PsiReference Java class for programming different tasks.

Creating a Reference Object

To create a reference from an element in a source file, you can use the following code:


          
           
PsiReference ref = element.createReference();

This code creates a PsiReference object from a PSI element (an element of the file).

Getting the Origin of a Reference

To get the origin of a reference, you can use the following code:


          
           
PsiElement origin = ref.getElement();

This code returns the PSI element which is the origin of the reference.

ResolveResult[] results = ref.multiResolve(false);

This code returns the parent element of the reference, or null if the reference has no parent.

Getting the Parent Class

PsiClass parentClass = PsiTreeUtil.getParentOfType(ref.getElement(), PsiClass.class);
