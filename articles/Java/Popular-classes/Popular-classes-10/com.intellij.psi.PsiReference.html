com.intellij.psi.PsiReference is part of the IntelliJ Java API and provides a way to programmatically reference elements in a Java source file. This article will explain how to use com.intellij.psi.PsiReference Java class for programming different tasks.

This code creates a PsiReference object from a PSI element (an element of the file).

Getting the Origin of a Reference

PsiElement origin = ref.getElement();

This code returns the PSI element which is the origin of the reference.
