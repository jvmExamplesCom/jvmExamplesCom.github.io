org.mozilla.javascript.Context

The org.mozilla.javascript.Context class is a part of the Mozilla Rhino project and is used to implement JavaScript execution context in Java. It provides methods for creating, binding, executing and manipulating the JavaScript context. It is also used for controlling how JavaScript is evaluated, including setting options such as whether script errors should be thrown or not. This class is essential for providing the infrastructure for creating and executing scripts written in JavaScript.

A new Context instance can be created using the 
Context.enter()
method, which takes an optional JavaScript scope argument. This scope argument sets the starting scope for evaluating JavaScript code.
