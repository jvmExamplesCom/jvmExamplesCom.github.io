org.springframework.web.socket.config.annotation.EnableWebSocket

The org.springframework.web.socket.config.annotation.EnableWebSocket annotation enables WebSocket support for a Spring application. It provides configuration for the creation of a WebSocket message broker that enables processing of messages from and to the WebSocket clients. This annotation can be combined with @Configuration classes in order to define the necessary beans for the WebSocket configuration.

Example

An example of using the EnableWebSocket annotation to configure a Spring application to support WebSockets looks like this:


          
           
@Configuration
@EnableWebSocket
public class WebSocketConfig {

    @Bean
    public ServerEndpointExporter serverEndpointExporter() {
        return new ServerEndpointExporter();
    }

}

In this example, we are creating a bean for ServerEndpointExporter that is used for exposing WebSocket endpoints without having to deploy them to a web server. The ServerEndpointExporter bean helps in configuring the endpoint automatically with the help of the @ServerEndpoint annotation.

Output

The output of the above example looks like this when run on a web application server:


          
           
WebSocket configured successfully
