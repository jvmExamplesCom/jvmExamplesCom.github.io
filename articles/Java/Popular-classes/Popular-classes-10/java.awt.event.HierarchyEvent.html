Java.awt.event.HierarchyEvent

Java.awt.event.HierarchyEvent is an event class used to notify AWT components when there has been a change with regards to its hierarchy such as a addition of component to the component's container or removal of a component from its container. This event type is used to notify theAWT components when they have been added to a container, moved within a container, removed from a container, or made the focus owner.
