org.apache.http.conn.routing.HttpRoute

The org.apache.http.conn.routing.HttpRoute class represents a route for routing messages between two points, typically a client and a server. A HttpRoute may be based on a set of proxy host addresses, which are used to direct the message to the intended destination host address.

Creating a HttpRoute

HttpRoute(InetAddress target)

HttpRoute route = new HttpRoute(HttpHost("example.com"));

route.setConnectTimeout(30000);

Routing Connections

HttpClient.route(HttpHost target, HttpRoute route, HttpContext context);
