JavaFx.scene.layout.Background

JavaFx.scene.layout.Background is an API in the JavaFX library used to improve the appearance of a scene by allowing the user to modify the background fill. This class consists of several classes which provide different methods of setting the background tone, color, and patterns. The following sections will provide a detailed overview of this API.

Creating a Background Instance

The Background instance can be created using the javafx.scene.layout.Background.Builder class. 

          
           
Background bg = new Background.Builder()
    .backgroundFill(Color.BLUE)
    .build();
This code example creates an instance of the Background class.

Setting the Background Color

The background color for the scene can be set with the setBackgroundFill method. 

          
           
bg.setBackgroundFill(Color.RED);
This code example lets us set the background of the scene to red.

Drawing Background Patterns

Patterns can be created and applied to the background of the scene by using the createLinearGradient and createRadialGradient methods. 

          
           
LinearGradient lg = new LinearGradient(0, 0, 1, 0, true, CycleMethod.NO_CYCLE, 
    new Stop(0, Color.WHITE), 
    new Stop(1, Color.BLACK));
bg.setBackgroundFill(lg);
This example draws a white to black linear gradient on the background.

Setting the Background Opacity

To set the level of background opacity, use the setOpacity method. 

          
           
bg.setOpacity(0.5);
This code example sets the background opacity to 50%.

Conclusion

JavaFx.scene.layout.Background is a powerful API within the JavaFX library that allows the user to conveniently set the background of a scene to any color, pattern, or opacity that they desire.
