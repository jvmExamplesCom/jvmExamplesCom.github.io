com.mongodb.MongoException

The com.mongodb.MongoException is an exception thrown when using the MongoDB Java driver. It is thrown upon attempting any operation that is not supported by the driver. Examples of operations that can result in a MongoException include: attempting to create a new collection when a collection with the same name already exists, attempting to view a collection that does not exist, attempting to insert data into a collection using an incorrect syntax, and attempting to set a query field to an empty array.
