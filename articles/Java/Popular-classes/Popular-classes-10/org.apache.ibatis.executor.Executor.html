org.apache.ibatis.executor.Executor

The org.apache.ibatis.executor.Executor interface is a powerful tool for executing mapped statements in MyBatis. It provides an abstraction layer for developer to execute methods, such as queries and insert, update, and delete operations. It also facilitates the reuse of existing mapped statement instances when the same mapped statement is executed multiple times.

Code Examples

The following is an example of how to use the Executor interface to execute a mapped statement:


          
           
Executor executor = configuration.getExecutorFactory().getExecutor(configuration);
MappedStatement statement = configuration.getMappedStatement("exampleStatement");
ResultSetHandler handler = new ExampleResultSetHandler();
List<Object> results = executor.query(statement, new HashMap<String, Object>(), handler);
