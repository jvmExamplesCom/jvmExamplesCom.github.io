Introduction to the org.apache.kafka.clients.CommonClientConfigs Class

Configuration Options Available from CommonClientConfigs

The CommonClientConfigs class provides options for configuring a range of different aspects of Kafka clients such as security, communication protocols, and message formats. Some of the available options include:
