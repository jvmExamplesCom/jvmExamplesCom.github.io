Introduction to org.elasticsearch.rest.RestStatus

org.elasticsearch.rest.RestStatus is a Java class that provides a set of HTTP status codes which can be used when responding from a REST service. These codes are made up of a 3 digit number and a textual description, and provide a consistent way to inform clients of the success of an operation.

Usage of RestStatus

A RestStatus instance is created by providing a valid HTTP status code and optionally an additional detail message. This instance can then be used to respond to clients by returning the corresponding HTTP status code and the optional detail message. 


          
           
RestStatus status = new RestStatus(404, "Not Found");

// Send a response including the status code and message
response.setStatus(status.value());
response.getWriter().println("Error:" + status.getMessage());
