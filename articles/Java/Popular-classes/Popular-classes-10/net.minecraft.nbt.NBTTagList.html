java.net.minecraft.nbt.NBTTagList Overview

Java's net.minecraft.nbt.NBTTagList is an interface which provides a way to store compound tags in a list. Compound tags are used in the NBT (Named Binary Tag) data format for Minecraft. The NBTTagList class is used to store multiple tags of the same type in one list. It provides methods to add, remove, and access tags stored in the list.

The NBTTagList class is part of the Java package net.minecraft.nbt, which contains classes to interact with the NBT data format. The NBTTagList class implements the NBTBase interface, which provides methods to access and manipulate the data stored in the tag list.

Methods

The following are common methods which can be used to work with an NBTTagList:

  • add(NBTBase tag)
    : Adds the given tag to the end of the tag list.
  • remove(int index)
    : Removes the tag at the given index from the list.
  • get(int index)
    : Returns the tag at the given index.
  • size()
    : Returns the number of tags in the list.

Sample Code

This code sample demonstrates how to create an NBTTagList object, add two tags to it, and then access the first tag:


          
           
// Create a new NBTTagList
NBTTagList list = new NBTTagList();

// Create two tags
NBTTagString tag1 = new NBTTagString("foo");
NBTTagLong tag2 = new NBTTagLong(42);

// Add the tags to the list
list.add(tag1);
list.add(tag2);

// Get the first tag from the list
NBTBase tag = list.get(0);
System.out.println(tag); // prints "foo"

The above code will print out "foo", because the first tag in the list is a NBTTagString with a value of "foo".
