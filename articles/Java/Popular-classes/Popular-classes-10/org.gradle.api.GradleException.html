org.gradle.api.GradleException

The org.gradle.api.GradleException class is used for exceptions thrown by Gradle. It is a type of RuntimeException and contains several methods for accessing details of the exception that was thrown. This exception can be thrown from any Gradle task or plugin.

Example 1 - Throwing an Exception

The following example shows how to throw a GradleException with a custom message:


          
           
task myTask {
    doFirst {
        throw new GradleException('My custom message')
    }
}

Example 2 - Accessing the Exception Message

The following example shows how to access the exception message from a GradleException instance:


          
           
try {
    gradle.taskGraph.whenReady() {
        throw new GradleException('My custom message')
    }
} catch (e: GradleException) {
    println 'Exception message: ' + e.getMessage()
}

Example 3 - Default Message

The following example shows how to access the default message from a GradleException instance:


          
           
try {
    gradle.taskGraph.whenReady() {
        throw new GradleException()
    }
} catch (e: GradleException) {
    println 'Default message: ' + e.getDefaultMessage()
}

Example 4 - Printing the Stack Trace

The following example shows how to print the stack trace from a GradleException instance:

    gradle.taskGraph.whenReady() {
