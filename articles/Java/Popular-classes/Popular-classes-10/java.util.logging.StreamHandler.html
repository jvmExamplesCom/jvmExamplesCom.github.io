java.util.logging.StreamHandler

The java.util.logging.StreamHandler class is a Handler object used to write events to a Java OutputStream. It has been part of the Java platform since Java 1.4.

Constructors

StreamHandler has three constructors for creating a StreamHandler object with desired settings.


          
           
public StreamHandler()
public StreamHandler(OutputStream output, Formatter formatter)
public StreamHandler(OutputStream output, Formatter formatter, Level level)

The first constructor creates a StreamHandler object that writes to standard output with a SimpleFormatter, and produces log records with no logging level restriction. The second and third constructor create a StreamHandler object that writes to the specified OutputStream, with the given Formatter and logging level, respectively.

Methods

The StreamHandler provides the following methods: 

          
           
public void close()
public void flush()
public Level getLevel()
public Filter getFilter()
public Formatter getFormatter()
public void publish(LogRecord record)
public void setLevel(Level newLevel)
public void setFilter(Filter newFilter)
public void setFormatter(Formatter newFormatter)

Example

To demonstrate use of the StreamHandler, we will create a simple logger that prints messages to the console:


          
           
import java.util.logging.ConsoleHandler;
import java.util.logging.Level;
import java.util.logging.Logger;
import java.util.logging.SimpleFormatter;

public class LoggerDemo { 
    public static void main(String[] args) { 
        Logger logger = Logger.getLogger("MyLogger"); 
        logger.setLevel(Level.INFO); 
        ConsoleHandler handler = new ConsoleHandler(); 
        handler.setLevel(Level.INFO); 
        handler.setFormatter(new SimpleFormatter()); 
        logger.addHandler(handler); 
        logger.info("This is an info message");
    }
}
