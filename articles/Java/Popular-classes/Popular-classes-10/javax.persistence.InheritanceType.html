javax.persistence.InheritanceType

The javax.persistence.InheritanceType enumeration determines how the inheritance of entities is declared in the entity hierarchy or class hierarchy. It is a type-safe way of configuring the entity inheritance strategy to be used for an entity hierarchy.

Types

  • TABLE_PER_CLASS - uses a separate table for each concrete entity class.
  • SINGLE_TABLE - uses a single table per class hierarchy.
  • JOINED - uses a table per class hierarchy, joined by foreign keys.
This enumeration has the following properties:
  • class - the class that is represented by this enumeration.
  • name - the name of the enumeration.
  • ordinal - the ordinal of the enumeration.
  • values - an array of all possible values of the enumeration.
  • valueOf(String name) - returns the enumeration value of the given name.

Example

An example of using the InheritanceType enumeration is as follows:


          
           
@Entity
@Inheritance(strategy=InheritanceType.TABLE_PER_CLASS)
public class Person {
    //...
}

The above code will configure the entity inheritance strategy to use a separate table for each concrete entity class.

Output

The output of the above code is as follows: 

          
           
@Inheritance(strategy=InheritanceType.TABLE_PER_CLASS)
