Introduction to org.eclipse.aether.resolution.ArtifactResult

org.eclipse.aether.resolution.ArtifactResult is a Java class used by the Eclipse Aether dependency resolution system. It represents the result of an artifact resolution request.

The class provides several methods that allow manipulation and retrieval of information related to the resolved artifact, such as its coordinates (e.g. its group id, artifact id, version), resolution exceptions, and its file.

Basic Usage Example


          
           
ArtifactResult result = resolver.resolveArtifact(request);

// get the resolved artifact's coordinates
String groupId = result.getArtifact().getGroupId();
String artifactId = result.getArtifact().getArtifactId();
Version version = result.getArtifact().getVersion();

// get the resolved artifact's file
File artifactFile = result.getArtifact().getFile();

// get resolution exceptions
List<Exception> exceptions = result.getExceptions();
