Introduction to Java Util Primitive Iterator

The 

          
           
java.util.PrimitiveIterator
interface is used for creating an iterator for the primitive data types like int, char, long and double. The various methods of this interface help in traversing through a collection of primitive elements and provides a convenient way to work with them. In this article, we will discuss the methods provided by this interface and explore the usage of one such method, the nextInt() method.

Methods in PrimitiveIterator Interface

The PrimitiveIterator interface provides the following methods: 
  • hasNext()
    - This method returns a boolean value true or false depending upon whether any more element is present in the underlying collection.
  • next()
    - This method returns the next element in the collection.
  • nextInt()
    - This method is specific to int elements in the collection and returns the next int element in the collection.
  • nextDouble()
    - This method is specific to double elements in the collection and returns the next double element in the collection.
  • nextLong()
    - This method is specific to long elements in the collection and returns the next long element in the collection.
  • nextChar()
    - This method is specific to char elements in the collection and returns the next char element in the collection.
  • forEachRemaining()
    - This method applies a given lambda expression on all the elements till the end of the collection.

nextInt() Method

The 

          
           
nextInt()
method is used to return the next element of type 

          
           
int
in the collection. It throws an exception if there are no more elements present in the collection.

    if (hasNext()) {
        return elements[index++];

nextInt()

public class PrimeIterator implements PrimitiveIterator<Integer, IntConsumer> {

    // integer array containing the list of prime numbers
    private int[] elements = {1, 3, 5, 7, 11, 13, 17, 19, 23, 29};

    // index pointing to next element
    private int index = 0;

    public int nextInt() {
        if (hasNext()) {
            return elements[index++];
        }
        throw new NoSuchElementException();
    }

    public boolean hasNext() {
        return index < elements.length;
    }
}

nextInt()

PrimeIterator
