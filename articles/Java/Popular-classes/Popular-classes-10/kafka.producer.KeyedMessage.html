Kafka Producer Keyed Message

Kafka Producer KeyedMessage is an API used to send messages to Kafka Topic, through a Kafka Producer. It provides the ability to specify a key for each message and helps in maintaining message order when sending to Kafka Topic.

Basic Usage

In order to use the Kafka Producer KeyedMessage, we need to create an instance of the class using the following code: 

          
           
KeyedMessage<String, String> message = new KeyedMessage<String, String>(topicName, key, value);
The first argument is the name of the topic to which the message should be sent. The second and third arguments are the key and value of the message respectively.

Example

Let us consider an example of a simple Kafka Producer sending a KeyedMessage to a specific Topic. 

          
           
public class ExampleProducer {

    public static void main(String[] args) {
        Properties props = new Properties();
        props.put("bootstrap.servers", "localhost:9092");
        props.put("key.serializer", "org.apache.kafka.common.serialization.StringSerializer");
        props.put("value.serializer", "org.apache.kafka.common.serialization.StringSerializer");

        //Create a Producer
        Producer<String, String> producer = new KafkaProducer <>(props);
        //Create a message with key and value
        KeyedMessage<String, String> message = new KeyedMessage<String, String>("my-topic", "my-key", "my-value");
        //Send the message
        producer.send(message);
    }
}
In this example, we have created a simple Kafka Producer that sends the KeyedMessage with a key “my-key” and a value “my-value” to the topic “my-topic”.

Conclusion

Kafka Producer KeyedMessage is a powerful API that provides the ability to send messages with an associated key. This allows us to maintain message order and helps to improve message processing speed and accuracy.
