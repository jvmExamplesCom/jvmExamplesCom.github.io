org.apache.commons.net.ftp.FTP

The org.apache.commons.net.ftp.FTP class is a high-level API that allows you to connect to an FTP server and perform basic operations like copying files and directories, downloading files, uploading files, and deleting files and directories. It also enables you to rename files, get file information, and retrieve a directory listing.

Connecting to an FTP Server

To connect to an FTP server, use the connect() method. This method takes the hostname or IP address of the FTP server as the parameter. The following code example shows how to connect to an FTP server.


          
           
FTPClient ftpClient = new FTPClient();
String server = "localhost";
int port = 21;
String user = "anonymous";
String pass = "anonymous@example.com";
ftpClient.connect(server, port);
ftpClient.login(user, pass);

To get the list of files and directories in the current working directory, use the listFiles() method. FTPFile[] files = ftpClient.listFiles();

To download a file from an FTP server, use the retrieveFile() method. This method takes two parameters, the remote file name and the local file name. The following code example shows how to download a file from the server and save it in the local filesystem.

File localFile = new File("/local/path/file.txt");
