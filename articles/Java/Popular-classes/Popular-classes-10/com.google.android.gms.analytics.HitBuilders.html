Introduction to com.google.android.gms.analytics.HitBuilders

This is a very useful feature, allowing developers to quickly and easily collect data on user's interactions with their Android applications.

Creating a HitBuilder Object

To create a HitBuilder object, you need to use the following code:


          
           
HitBuilders.EventBuilder hitBuilder = new HitBuilders.EventBuilder();

This will create a HitBuilder object, ready to be used.

Setting Hit Properties

hitBuilder.setHitType("event")
      .set("&amp;ec", "category")
      .set("&amp;el","label");

You can also set other properties such as action, interaction, viewport and region.

Sending A Hit

Once you have set all the properties of the Hit, you can send it using the following code:


          
           
tracker.send(hitBuilder.build());
