javax.naming.ldap.LdapContext

The javax.naming.ldap.LdapContext class provides a context for connecting, binding and communicating over an LDAP directory in the Java Naming and Directory Interface (JNDI) application programming interface (API). It extends the DirContext interface and defines additional methods for the LDAP communication protocol.

Creating an LdapContext

An LdapContext is created by providing a set of properties and a bind principal name to the InitialLdapContext constructor. The properties specify the details of the connection, such as the host name, port, authentication details and encrypted transport.

env.put(Context.SECURITY_CREDENTIALS, "SECRET");

LdapContext ctx = new InitialLdapContext(env, null);
