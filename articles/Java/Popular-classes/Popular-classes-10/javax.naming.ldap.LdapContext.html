javax.naming.ldap.LdapContext

The javax.naming.ldap.LdapContext class provides a context for connecting, binding and communicating over an LDAP directory in the Java Naming and Directory Interface (JNDI) application programming interface (API). It extends the DirContext interface and defines additional methods for the LDAP communication protocol.

Creating an LdapContext

An LdapContext is created by providing a set of properties and a bind principal name to the InitialLdapContext constructor. The properties specify the details of the connection, such as the host name, port, authentication details and encrypted transport.


          
           
Hashtable<String, String> env = new Hashtable<String, String>();
env.put(Context.INITIAL_CONTEXT_FACTORY, "com.sun.jndi.ldap.LdapCtxFactory");
env.put(Context.PROVIDER_URL, "ldap://example.com:389");
env.put(Context.SECURITY_AUTHENTICATION, "simple");
env.put(Context.SECURITY_PRINCIPAL, "cn=Manager,dc=example,dc=com");
env.put(Context.SECURITY_CREDENTIALS, "SECRET");

LdapContext ctx = new InitialLdapContext(env, null);

Binding to a Directory

The bind operation is used to authenticate the user and establish the session. The bind operation can be used with either the bind() method or the bindDn() method. The bind() method is used to authenticate a previously created user by providing the username and password, whereas the bindDn() method is used to authenticate a directory-level user with administrative privileges.

Searching a Directory

The search operation is used to retrieve objects from the directory. It can be used with the search() method or one of its variants, such as the searchOne() and searchSubtree() methods. The search() method allows you to set various options, such as the base name, scope and filter. The searchOne() and searchSubtree() methods allow you to set the base name, scope and filter in the one call.


          
           
SearchControls sc = new SearchControls();
sc.setSearchScope(SearchControls.SUBTREE_SCOPE);

String[] returningAttributes = {"cn", "mail", "givenname"};
sc.setReturningAttributes(returningAttributes);

String filter = "(&amp;(cn=John*)(mail=*))";

NamingEnumeration results = ctx.search("dc=example,dc=com", filter, sc);
