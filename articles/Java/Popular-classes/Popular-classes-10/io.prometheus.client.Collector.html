Overview of io.prometheus.client.Collector in Java

io.prometheus.client.Collector is an interface in java for performing Prometheus metrics collection. This can be useful for collecting custom metrics from application code, or from any other sources. The interface also provides a way to directly instrument the code, with methods such as start() and stop() allowing programmatic control over the metrics collection.

Collector collector = new Collector();
