android.text.ClipboardManager

The android.text.ClipboardManager class provides access to the system clipboard by allowing objects to be copied and pasted. It is part of the Android SDK. This class provides an interface for copying text, images, and other types of data to the clipboard, as well as retrieving that data.

Basic Usage

The android.text.ClipboardManager class is simple to use. To copy some data to the clipboard, create a new instance of ClipboardManager and set the data via the setText(), setPrimaryClip(), or setPrimaryClip() methods:


          
           
ClipboardManager clipboard = (ClipboardManager) getSystemService(CLIPBOARD_SERVICE);
clipboard.setText("Example text");

To retrieve the data from the clipboard, simply call getText() or getPrimaryClip():


          
           
String text = clipboard.getText();

It's also possible to add an OnPrimaryClipChangedListener, which triggers when the primary clip on the system clipboard is updated. This listener can then be used to detect when the clipboard is modified and act accordingly.

Examples

The following code example demonstrates how to copy a string to the clipboard, retrieves the contents of the clipboard, and displays it in a Toast:


          
           
// Copy a string to the clipboard
ClipboardManager clipboard = (ClipboardManager) getSystemService(CLIPBOARD_SERVICE);
clipboard.setText("Example text");

// Retrieve the contents of the clipboard
String text = clipboard.getText();

// Display the contents of the clipboard in a Toast
Toast.makeText(this, "Text: " + text, Toast.LENGTH_SHORT).show();

The following example demonstrates how to copy an image to the clipboard, retrieves the contents of the clipboard, and displays the image in an ImageView:


          
           
// Copy an image to the clipboard
ClipboardManager clipboard = (ClipboardManager) getSystemService(CLIPBOARD_SERVICE);
Bitmap bitmap = BitmapFactory.decodeResource(getResources(), R.drawable.example_image);
ClipData clipData = ClipData.newUri(getContentResolver(), "Example Image", Uri.parse("android.resource://com.example/drawable/" + R.drawable.example_image));
clipboard.setPrimaryClip(clipData);

// Retrieve the contents of the clipboard
ClipData clip = clipboard.getPrimaryClip();
if (clip != null &amp;&amp; clip.getItemCount() > 0) {
    Uri imageUri = clip.getItemAt(0).getUri();
    
    // Display the image in an ImageView
    ImageView imageView = findViewById(R.id.image_view);
    imageView.setImageURI(imageUri);
}
