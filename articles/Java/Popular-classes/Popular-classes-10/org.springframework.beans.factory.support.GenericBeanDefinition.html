org.springframework.beans.factory.support.GenericBeanDefinition Overview

This class provides support for bean definition inheritance, constructor-based argument resolution, and direct access to the bean factory. This flexibility allows the developer to define beans in a more intuitive and powerful way than the standard container configuration models.
