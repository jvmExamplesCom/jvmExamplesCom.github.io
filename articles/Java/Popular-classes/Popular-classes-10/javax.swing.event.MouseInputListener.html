What Is javax.swing.event.MouseInputListener?

javax.swing.event.MouseInputListener is an interface in the Java Abstract Window Toolkit (AWT) that is implemented to detect mouse motion and clicks on various components within a graphical user interface (GUI). It helps developers create events that can be used to respond to the user's mouse movements and clicks. This is useful for creating interactive applications that allow users to interact with components on the screen.

Methods of MouseInputListener

The MouseInputListener interface contains six abstract methods:
  • mouseClicked(MouseEvent e): triggered when a mouse button is clicked on a component
  • mousePressed(MouseEvent e): triggered when a mouse button is pressed on a component
  • mouseReleased(MouseEvent e): triggered when a mouse button is released on a component
  • mouseEntered(MouseEvent e): triggered when the mouse pointer enters a component
  • mouseExited(MouseEvent e): triggered when the mouse pointer exits a component
  • mouseDragged(MouseEvent e): triggered when the mouse pointer is dragged inside a component

Example of MouseInputListener Implementation


          
           
// class MyClass implements MouseInputListener
public class MyClass implements MouseInputListener {
    
    // override mouseClicked() method
    public void mouseClicked(MouseEvent e) {
        System.out.println("Mouse clicked");
    }
            
    // override mousePressed() method
    public void mousePressed(MouseEvent e) {
        System.out.println("Mouse pressed");
    }
            
    // override mouseReleased() method
    public void mouseReleased(MouseEvent e) {
        System.out.println("Mouse released");
    }
            
    // override mouseEntered() method
    public void mouseEntered(MouseEvent e) {
        System.out.println("Mouse entered");
    }
            
    // override mouseExited() method
    public void mouseExited(MouseEvent e) {
        System.out.println("Mouse exited");
    }
            
    // override mouseDragged() method
    public void mouseDragged(MouseEvent e) {
        System.out.println("Mouse dragged");
    }
}

Using MouseInputListener

The MouseInputListener interface is used to detect and respond to user mouse activity. To utilize the interface, it must be implemented by a class. Once the methods are implemented, they can be utilized with the adding of a mouse listener to the component the developer wishes to monitor: 

          
           
component.addMouseListener(new MouseInputListener());
When any of the mouse related activities occur (e.g. mouse clicked, mouse pressed, etc), the corresponding method will be triggered and can be used to respond accordingly.
