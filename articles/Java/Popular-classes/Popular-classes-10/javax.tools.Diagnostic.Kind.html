javax.tools.Diagnostic.Kind

javax.tools.Diagnostic.Kind is a class in the Java programming language that is part of the Java Compiler API (JSR 199). It represents the kinds of diagnostics that can be produced by a Java compiler. It is intended to simplify compiler-reported errors and warnings by providing a single type to cover them all.

The following code examples demonstrate how to use the javax.tools.Diagnostic.Kind class.

    System.out.println(kind.name());
}
