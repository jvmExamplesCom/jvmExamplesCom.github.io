JavaFX Scene Canvas GraphicsContext

JavaFX Scene Canvas GraphicsContext is a class that provides methods to draw various shapes, apply colors and fonts, create gradients, and even manipulate images. The class uses the GraphicsContext object to render different types of graphical components such as lines, arcs, rectangles, and ellipses.

Methods Available

The following methods are available in the GraphicsContext class:

  • draw(Shape s): Draws the specified shape.
  • fill(Shape s): Fills the specified shape with the current fill Paint.
  • clearRect(double x, double y, double width, double height): Clears the specified rectangle.
  • setStroke(Paint p): Sets the current stroke color.
  • setFill(Paint p): Sets the current fill color.
  • setFont(Font f): Sets the current font.
  • setTextAlign(TextAlignment alignment): Sets the current text alignment.
  • fillText(String t, double x, double y): Draws the specified text at the specified position.
  • createLinearGradient(double startX, double startY, double endX, double endY,Stop... stops): Creates a linear gradient.
  • createRadialGradient( double centerX, double centerY, double radiusX, double radiusY,Stop... stops): Creates a radial gradient.
  • drawImage(Image image, double x, double y): Draws the specified image at the specified position.
  • transform(Affine t): Transforms the current GraphicsContext.
