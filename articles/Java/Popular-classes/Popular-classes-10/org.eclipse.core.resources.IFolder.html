org.eclipse.core.resources.IFolder

The org.eclipse.core.resources.IFolder interface represents a folder resource in the workspace. It provides facilities for performing operations on folders, and for accessing the members of a folder.

Creating IFolder Instance

You can create an instance of an IFolder in your workspace:


          
           
IWorkspaceRoot root = ResourcesPlugin.getWorkspace().getRoot();
IFolder folder = root.getFolder(path);
if (!folder.exists()) {
    folder.create(true, true, null);
}

The above example creates a folder resource in the workspace if it does not already exist.

Manipulating the Contents of an IFolder

You can use the methods in the IFolder interface to manipulate the contents of a folder:


          
           
IFolder folder = ...;
folder.getFile("file.txt");  //returns the file resource, if it exists
folder.create(false, true, null);  //creates the folder, if it doesn't already exist
folder.delete(true, null);  //deletes the folder and its contents

The above example shows the basic operations that you can perform on an IFolder.

Accessing the Members of an IFolder

You can use the methods in the IFolder interface to access the members of a folder resource:


          
           
IFolder folder = ...;
IResource[] members = folder.members();  //returns the members of the folder
for (IResource member : members) {
    System.out.println(member.getName());
}

The above example prints the names of the members of a folder.
