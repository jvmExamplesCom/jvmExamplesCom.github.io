Introduction to com.jayway.jsonpath.Configuration in Java

The second line creates a configuration instance which an empty builder which allows for customizing the JsonPath library options. The isIncludeNullOnPathNotFound() method returns true if null values should be included when a JsonPath path is not found. The isPathLeafToNull() method returns true if leaves of a path should be returned as null values. The isPathDropEmptyArraysSet() method returns true if paths will drop empty arrays. The getPathDropEmptyArraysLimit() method returns the maximum number of empty arrays dropped by the path. The isGlobalSettingEnabled() method returns true if global settings should be enabled. The isPathsValidationDisabled() method returns true if path validation should be disabled.
