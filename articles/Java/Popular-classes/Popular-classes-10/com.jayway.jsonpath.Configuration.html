Introduction to com.jayway.jsonpath.Configuration in Java

The isIncludeNullOnPathNotFound() method returns true if null values should be included when a JsonPath path is not found. The getPathDropEmptyArraysLimit() method returns the maximum number of empty arrays dropped by the path. The isGlobalSettingEnabled() method returns true if global settings should be enabled. The isPathsValidationDisabled() method returns true if path validation should be disabled.
