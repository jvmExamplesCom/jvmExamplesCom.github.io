Introduction to com.jayway.jsonpath.Configuration in Java

The com.jayway.jsonpath.Configuration class provides different methods for configuring the JsonPath library. In this article, we will explore its different methods, code examples, and their output.

Constructors

com.jayway.jsonpath.Configuration has several constructors. The ones that are commonly used are listed below: 

          
           
Configuration conf = Configuration.defaultConfiguration();
Configuration userConf = Configuration.builder().build();
Configuration customConf = Configuration.builder().option(Option.DEFAULT_PATH_LEAF_TO_NULL, true).build();
The first line creates a configuration instance with the default settings from the library. The second line creates a configuration instance which an empty builder which allows for customizing the JsonPath library options. The third line creates a configuration instance with some customized options. In this case, we are setting the Option.DEFAULT_PATH_LEAF_TO_NULL to true.

The getOption() method is used to retrieve the specified configuration option. The isDefinitePathsOnly() method returns true if only definite paths will be evaluated. The isWriteCustomClassTreatedAsBean() method returns true if custom classes should be treated as beans when writing JSON with the JsonProvider. The isAllowUnquotedFieldNames() method returns true if unquoted field names are allowed. The isAllowSingleQuotes() method returns true if single quotes are allowed. The isCompressionEnabled() method returns true if the response from the JsonProvider should be compressed. The isCaseSensitive() method returns true if the JsonPath library is case sensitive. The isIncludeNullOnPathNotFound() method returns true if null values should be included when a JsonPath path is not found. The isCollectionLimitSet() method returns true if a collection limit has been set. The getCollectionLimit() method returns the collection limit if it has been set. The isPathLeafToNull() method returns true if leaves of a path should be returned as null values. The isPathDropEmptyArraysSet() method returns true if paths will drop empty arrays. The getPathDropEmptyArraysLimit() method returns the maximum number of empty arrays dropped by the path. The isGlobalSettingEnabled() method returns true if global settings should be enabled. The isPathsValidationDisabled() method returns true if path validation should be disabled.

Code Examples

Below are some code examples which demonstrate how to use the com.jayway.jsonpath.Configuration class: 

          
           
// create a configuration instance
Configuration conf = Configuration.defaultConfiguration();

// set a global setting
conf.setGlobalSetting("jsonp.max_depth", 20);

// check if global setting is enabled

// set the collection limit

Path validation is disabled with the setPathValidationDisabled() method. The collection limit is set with the setCollectionLimit() method and retrieved using the getCollectionLimit() method. The setPathLeafToNull() method sets the leafs of a path to either null or the value of the leaf.
