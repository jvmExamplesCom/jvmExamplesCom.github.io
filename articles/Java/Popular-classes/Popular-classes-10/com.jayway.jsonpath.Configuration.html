Introduction to com.jayway.jsonpath.Configuration in Java

The com.jayway.jsonpath.Configuration class provides different methods for configuring the JsonPath library. In this article, we will explore its different methods, code examples, and their output.

Constructors

com.jayway.jsonpath.Configuration has several constructors. The ones that are commonly used are listed below: 

          
           
Configuration conf = Configuration.defaultConfiguration();
Configuration userConf = Configuration.builder().build();
Configuration customConf = Configuration.builder().option(Option.DEFAULT_PATH_LEAF_TO_NULL, true).build();
The first line creates a configuration instance with the default settings from the library. The second line creates a configuration instance which an empty builder which allows for customizing the JsonPath library options. The third line creates a configuration instance with some customized options. In this case, we are setting the Option.DEFAULT_PATH_LEAF_TO_NULL to true.

The isAllowUnquotedFieldNames() method returns true if unquoted field names are allowed. The isAllowSingleQuotes() method returns true if single quotes are allowed. The isCompressionEnabled() method returns true if the response from the JsonProvider should be compressed. The isCaseSensitive() method returns true if the JsonPath library is case sensitive. The isIncludeNullOnPathNotFound() method returns true if null values should be included when a JsonPath path is not found. The isCollectionLimitSet() method returns true if a collection limit has been set. The getCollectionLimit() method returns the collection limit if it has been set. The isPathLeafToNull() method returns true if leaves of a path should be returned as null values. The isPathDropEmptyArraysSet() method returns true if paths will drop empty arrays. The getPathDropEmptyArraysLimit() method returns the maximum number of empty arrays dropped by the path. The isGlobalSettingEnabled() method returns true if global settings should be enabled. The isPathsValidationDisabled() method returns true if path validation should be disabled.

Code Examples

// create a configuration instance
