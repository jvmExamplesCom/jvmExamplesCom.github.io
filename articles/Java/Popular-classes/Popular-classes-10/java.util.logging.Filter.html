Java.util.logging.Filter

Java.util.logging.Filter is an abstract class which is used to filter logging messages. It can filter based on message level, source class or source method name. This class is used to control which log messages are written. It can either allow or deny a certain log message

Code Examples


          
           
Logger logger = Logger.getLogger("com.example");
logger.setFilter(new MyFilter());

public class MyFilter extends Filter {
    public boolean isLoggable(LogRecord record) {
        String msg = record.getMessage();
        if (msg.contains("password")) {
            return false;
        }
        return true;
    }
}

The above example filters out any message containing the string “password”.


          
           
Logger.setFilter(new MyFilter());
The above line sets the filter for all loggers. 

          
           
Logger logger = Logger.getLogger("com.example");
logger.setLevel(Level.FINEST);
logger.setFilter(new MyFilter());

public class MyFilter extends Filter {
    public boolean isLoggable(LogRecord record) {
        Level level = record.getLevel();
        if (level.intValue() < Level.INFO.intValue()) {
            return false;
        }
        return true;
    }
}

The above example filters out logs with levels lower than INFO.


          
           
logger.setFilter(null);

The above line removes the filter from the logger.
