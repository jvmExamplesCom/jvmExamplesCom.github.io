Introduction to org.apache.ibatis.annotations.Update

MyBatis is an open-source persistence framework for working with databases, mapping them to Java objects. The @Update annotation is one of several annotations available in MyBatis for mapping Java objects and databases.

The following example shows a MyBatis mapper interface using the @Update annotation to map an update statement: 

          
           
@Mapper
public interface PersonMapper {
