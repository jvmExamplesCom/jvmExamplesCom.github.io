Overview of android.opengl.EGLExt

The android.opengl.EGLExt package is organized into two main categories: feature-level extensions and implementation-specific extensions. Feature-level extensions are extensions to the core OpenGL ES API which provide new functionality. This includes features such as improved blending and depth/stencil testing, as well as new image formats like RGBA32F and RGBA16F. Implementation-specific extensions are extensions that are specific to certain GPU implementations, and are dependent on the vendor or platform.
