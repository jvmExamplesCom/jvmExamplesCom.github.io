Introduction to io.opentracing.Scope

The io.opentracing.Scope class is a central part of the OpenTracing API. It allows developers to define and manage scopes in their applications. A Scope represents an execution environment, such as the current thread, or a logical unit of work. The Scope class also provides access to the Span associated with the current Scope, as well as methods for ending a Scope.

The Scope Class

Creating a Scope

ThreadScope
