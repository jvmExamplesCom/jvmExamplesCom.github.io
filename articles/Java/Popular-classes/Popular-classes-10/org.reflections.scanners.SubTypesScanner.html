Overview of org.reflections.scanners.SubTypesScanner

org.reflections.scanners.SubTypesScanner

org.reflections.scanners.AbstractScanner

Reflections

How to use SubTypesScanner

The 

          
           
SubTypesScanner
class is typically used in combination with the 

          
           
Reflections
object, which provides convenient methods to perform scanning operations. To scan for all subtypes of a given type using the Reflections object, one would use the 

          
           
Reflections.getSubTypesOf(type)
method, where type is the class or interface whose subtypes should be scanned.
