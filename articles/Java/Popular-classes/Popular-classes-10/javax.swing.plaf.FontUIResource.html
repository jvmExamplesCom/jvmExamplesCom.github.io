javax.swing.plaf.FontUIResource

javax.swing.plaf.FontUIResource is a class in the Java Swing library that is used to represent a particular font as part of the Look and Feel (L&F) feature of the library. FontUIResource instances can be used to adjust the size, style, and other UI features of a particular font. It is often used when creating custom user interfaces or with components that have different behaviors depending on which font is used.

Using FontUIResource

The syntax for using FontUIResource is as follows:


          
           
FontUIResource font = new FontUIResource(Font font);
Where 
font
is an instance of a 
java.awt.Font
. This line of code will create a new FontUIResource instance with the specified font.

Once a FontUIResource instance is created, it can be used in a variety of ways. For example, it can be used to set the default font for all components within a certain container by calling the 

setDefaultFont()
method: 

          
           
container.setDefaultFont(font);

It can also be used to set the font for a specific component by calling the 

setFont()
method: 

          
           
component.setFont(font);

Examples

Here is a sample program that demonstrates how FontUIResource can be used to set the font of all components within a JPanel:


          
           
import javax.swing.*;
import java.awt.*;

public class Demo {
   public static void main(String[] args) {
      JFrame frame = new JFrame();
      frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
      
      // Create a panel to add components to
      JPanel panel = new JPanel();

      // Create a font to use for the components
      Font font = new Font("Verdana", Font.BOLD, 12);
      // Create a FontUIResource from the font
      FontUIResource fur = new FontUIResource(font);
      
      // Set the default font for all components in the panel
      panel.setDefaultFont(fur);
      
      // Add a label to the panel
      JLabel label = new JLabel("This is a label");
      panel.add(label);
      
      // Add a button to the panel
      JButton button = new JButton("Click Me!");
      panel.add(button);
      
      // Add the panel to the frame
      frame.add(panel);
      
      // Display the frame
      frame.setSize(300, 200);
      frame.setVisible(true);
   }
}
This example creates a 
FontUIResource
instance from a 
Font
instance and then sets it as the default font for all components within a panel. This ensures that all components will use the same font. Another example is creating a 
FontUIResource
from a different 
Font
instance and setting it as the font for a specific component. This example creates a button whose font will be different from the other components in a panel: 

          
           
import javax.swing.*;
import java.awt.*;

public class Demo {
   public static void main(String[] args) {
      JFrame frame = new JFrame();
      frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
      
      // Create a panel to add components to
      JPanel panel = new JPanel();
      // Set a default font for all components in the panel
      panel.setFont(new Font("Arial", Font.PLAIN, 12));
      
      // Create a font to use for the button
      Font buttonFont = new Font("Courier New", Font.ITALIC, 14);
      // Create a FontUIResource from the font
      FontUIResource fur = new FontUIResource(buttonFont);
      
      // Add a label to the panel
      JLabel label = new JLabel("This is a label");
      panel.add(label);
      
      // Add a button to the panel, with a different font
      JButton button = new JButton("Click Me!");
      button.setFont(fur);
      panel.add(button);
      
      // Add the panel to the frame
      frame.add(panel);
      
      // Display the frame
      frame.setSize(300, 200);
      frame.setVisible(true);
   }
}

In this example, a different font is used for the button, while all other components use the default font. This is a useful technique when creating user interfaces that require different fonts for different components.
