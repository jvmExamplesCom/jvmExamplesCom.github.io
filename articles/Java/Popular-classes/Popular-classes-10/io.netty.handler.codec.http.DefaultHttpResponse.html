io.netty.handler.codec.http.DefaultHttpResponse

The io.netty.handler.codec.http.DefaultHttpResponse class is used to represent an HTTP response by providing the response status and optional content. It is a concrete implementation of the HttpResponse interface. The DefaultHttpResponse class provides various methods for setting header fields, cookies, content and more.

Constructors

The following constructors are available in the class:

    , which creates an HttpResponse with the specified version and status.

    , which creates an HttpResponse with the specified version and status, and request headers.

    , which sets the HTTP version.

Example

The following example shows how to create a DefaultHttpResponse and set its content.


          
           
HttpVersion httpVersion = HttpVersion.HTTP_1_1;
HttpResponseStatus httpResponseStatus = HttpResponseStatus.OK;

HttpResponse response = new DefaultHttpResponse(httpVersion, httpResponseStatus);
ByteBuf content = Unpooled.copiedBuffer("Hello, World!", CharsetUtil.UTF_8);
