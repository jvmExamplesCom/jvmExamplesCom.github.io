javax.swing.event.MouseInputAdapter

javax.swing.event.MouseInputAdapter is a class in the Java Swing package that provides support for mouse events. It is an abstract adapter class for receiving mouse input events. The MouseInputAdapter class implements the MouseInputListener interface, which contains methods for responding to mouse events such as mouse clicks, mouse movement, and mouse dragging. The MouseInputAdapter class allows developers to easily create custom mouse event listeners by overriding the methods in the MouseInputListener interface.

Code Example 1


          
           
import javax.swing.event.MouseInputAdapter;

public class MyClass extends MouseInputAdapter {
  
  @Override
  public void mouseMoved(MouseEvent evt) {
    // Do something when the mouse moves
  }

}

In this code example, we create a subclass of the MouseInputAdapter class called MyClass. We override the mouseMoved() method from the MouseInputListener interface, which will be called whenever the mouse is moved. We can then add custom logic in the method implementation to respond to the mouse movement.

import javax.swing.JFrame;
import javax.swing.JLabel;
import javax.swing.event.MouseInputAdapter;

public class MyFrame extends JFrame {

  private JLabel label;

  public MyFrame() {
    label = new JLabel("Hello World!");
    label.addMouseListener(new LabelMouseInputAdapter());
    add(label);
  }

  private class LabelMouseInputAdapter extends MouseInputAdapter {

    @Override
    public void mouseEntered(MouseEvent evt) {
      label.setText("Mouse has entered!");
    }

    @Override
    public void mouseExited(MouseEvent evt) {
      label.setText("Mouse has exited!");
    }

  }

}

In this code example, we create a subclass of JFrame called MyFrame. We add a JLabel to the frame and create an inner class of the frame that extends MouseInputAdapter, allowing us to implement mouseEntered() and mouseExited() in order to respond to when the mouse enters and exits the label. We then add an instance of the inner class as a MouseListener to the label, so that whenever the user moves the mouse over the label, we can respond accordingly.
