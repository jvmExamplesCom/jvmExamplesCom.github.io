android.support.v7.preference.PreferenceFragmentCompat

Android's PreferenceFragmentCompat class provides an interface to manage settings within an Android app. This makes it easy to create settings with specific preferences, control the behavior of the settings, and access settings from outside the application. The PreferenceFragmentCompat is part of the support library and is supported all the way back to API level 4.

Creating a PreferenceFragmentCompat Instance

To create a PreferenceFragmentCompat instance, you need to create a class that extends PreferenceFragmentCompat. This class should implement onCreatePreferences() as follows:


          
           
public class MySettingsFragment extends PreferenceFragmentCompat {
  @Override
  public void onCreatePreferences(Bundle bundle, String s) {
    // Add preferences
  }
}

MySettingsFragment settingsFragment = new MySettingsFragment();

Adding Preferences

@Override
public void onCreatePreferences(Bundle bundle, String s) {
  // Add preferences
