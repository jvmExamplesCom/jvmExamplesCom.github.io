Introduction to org.xml.sax.helpers.AttributesImpl

It is used when an application needs to create Attribute objects with a set of name/value pairs for use in SAX event-based parsing. This implementation provides convenient methods for setting attributes and for retrieving values by name or by index.

AttributesImpl attrs = new AttributesImpl();

attrs.setAttribute("name", "John Doe");
