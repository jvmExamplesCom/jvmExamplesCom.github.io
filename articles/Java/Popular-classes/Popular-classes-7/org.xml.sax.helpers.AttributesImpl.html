Introduction to org.xml.sax.helpers.AttributesImpl

org.xml.sax.helpers.AttributesImpl is an implementation of the SAX2 Attributes interface. It is used when an application needs to create Attribute objects with a set of name/value pairs for use in SAX event-based parsing. This implementation provides convenient methods for setting attributes and for retrieving values by name or by index.

Creating AttributesImpl Objects

An AttributesImpl object can be created using its no-argument constructor:


          
           
AttributesImpl attrs = new AttributesImpl();

Setting Attributes

attrs.setAttribute("name", "John Doe");

Retrieving Attribute Values
