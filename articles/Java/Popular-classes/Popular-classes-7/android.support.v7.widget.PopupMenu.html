android.support.v7.widget.PopupMenu

PopupMenu is a powerful tool for Apps with Android Studio that enables developers to easily create and access menus from within their apps. It is part of the v7 support library, which is no longer maintained by Google, but is still widely used.

Creating a PopupMenu

To create a PopupMenu, the first step is to define a View (e.g. an ImageButton) that will trigger the PopupMenu when clicked: 

          
           
ImageButton imageButton = (ImageButton) findViewById(R.id.imageButton);
imageButton.setOnClickListener(new View.OnClickListener() {
   @Override
   public void onClick(View v) {
      // Create the PopupMenu here
   }
});
Once this is done, we can create the PopupMenu in the onClick() method: 

          
           
PopupMenu popupMenu = new PopupMenu(this, imageButton);
We can then inflate the menu from a resource file: 

          
           
getMenuInflater().inflate(R.menu.popup_menu, popupMenu.getMenu());
Where R.menu.popup_menu is an XML file containing the menu items like so: 

          
           
<menu xmlns:android="http://schemas.android.com/apk/res/android">
    <item
        android:id="@+id/menu_item_1"
        android:title="Item 1" />
    <item
        android:id="@+id/menu_item_2"
        android:title="Item 2" />
</menu>

PopupMenu Callbacks

We can listen for user interaction with the PopupMenu by setting an OnMenuItemClickListener: 

          
           
popupMenu.setOnMenuItemClickListener(
    new PopupMenu.OnMenuItemClickListener() {
        @Override
        public boolean onMenuItemClick(MenuItem item) {
            // Handle the click event here.
            return false;
        }
    });
This listener will be called each time a user clicks on a menu item, and it is up to the developer to decide what action should be taken based on the selected item.

Displaying the PopupMenu

Finally, we can display the PopupMenu by calling show() on the PopupMenu instance: 

          
           
popupMenu.show();
And that's it! The PopupMenu should now be displayed and can be interacted with.
