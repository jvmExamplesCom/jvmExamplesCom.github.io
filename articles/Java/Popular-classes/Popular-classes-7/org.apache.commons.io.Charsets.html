The org.apache.commons.io.Charsets class contains six constants that represent the various charsets available in the Java platform:

  • UTF_8
  • ISO_8859_1

  • UTF_16
  • UTF_16BE
  • UTF_16LE

These constants can be used to retrieve a Charset object using the static method
