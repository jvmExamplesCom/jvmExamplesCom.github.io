org.apache.commons.io.Charsets

The org.apache.commons.io.Charsets class provides a set of standard charsets that can be accessed in Java programs via their constants. This class is part of the Apache Commons IO library and allows developers to access charsets like UTF-8 or ISO-8859-1 easily, with a minimum of code.

Class Overview

The org.apache.commons.io.Charsets class contains six constants that represent the various charsets available in the Java platform:

  • UTF_8
  • ISO_8859_1
  • US_ASCII
  • UTF_16
  • UTF_16BE
  • UTF_16LE

These constants can be used to retrieve a Charset object using the static method 

Charset.forName()

Charset utf8 = Charset.forName(Charsets.UTF_8);

The Charset object returned by this method can then be used to encode and decode String objects as well as Raw Byte Arrays.
