androidx.appcompat.widget.AppCompatTextView

AndroidX AppCompatTextView is a subclass of the Android TextView and is used to display text in an application. It provides a platform-specific implementation of the TextView and adds many features that are not available in the normal TextView. AppCompatTextView is part of the Xamarin.Android Support Libraries and is designed to provide backward compatibility with versions of Android prior to API Level 21 (Lollipop).

Creating an AppCompatTextView Instance

To create an instance of AppCompatTextView, the following code can be used in the Android application:


          
           
AppCompatTextView textView = new AppCompatTextView(this); 
textView.setText("Hello World!"); 
textView.setTextSize(18); 
textView.setTextColor(Color.BLUE);

The code above creates an AppCompatTextView instance named textView with the initial text “Hello World!”, text size 18 and blue text color.

Using AppCompatTextView Attributes

AppCompatTextView provides a wide range of attributes that can be set using the setText...() methods. Some of the available attributes are:

  • setTextSize(int)
    : Sets the text size value.
  • setGravity(int)
    : Sets the text gravity to either top/center/bottom left/right.
  • setTextColor(int)
    : Sets the text color.
  • setTextStyle(int)
    : Sets the text style like bold, italic, normal etc.
  • 
            
             
setSingleLine(boolean)
    : Sets the text to be single or multiple line.
  • setMaxLines(int)
    : Sets the maximum number of lines allowed in the text view.

The below code sets the gravity to center, sets the text style to bold and it sets the maximum lines to 2.


          
           
textView.setGravity(Gravity.CENTER); 
textView.setTextStyle(Typeface.BOLD); 
textView.setMaxLines(2);

Using AppCompatTextView Listeners

AppCompatTextView provides a wide range of listeners which can be used to receive notifications when certain actions occur. These listeners are:

  • 
            
             
onFocusChangeListener
    : This listener is triggered when the focus changes.
  • 
            
             
onTextChangedListener
    : This listener is triggered when the text changes.
  • 
            
             
onSelectionChangedListener
    : This listener is triggered when the selection of text changes.

The below code sets an onFocusChangeListener which logs a message when the focus changes:


          
           
textView.setOnFocusChangeListener(new View.OnFocusChangeListener() {
