Overview of The java.nio.DoubleBuffer Class

The java.nio.DoubleBuffer class is a Java NIO buffer that stores double-precision (64-bit) floating-point values in memory. It supports fast, non-blocking bulk operations, with the ability to read and write individual elements. It also provides atomic operations on elements, which are used for thread safety purposes. The DoubleBuffer class is used in many applications, such as fast math computations and financial calculations.

Constructors

The DoubleBuffer class provides two constructors for creating instances:

  • 
            
             
DoubleBuffer(int capacity)
    – Creates a double buffer with a fixed capacity.
  • 
            
             
DoubleBuffer(double[] array)
    – Creates a double buffer from an existing array.

Methods

In addition to the standard buffer methods listed below for reading and writing data, the DoubleBuffer class also provides atomic operations, such as compareAndSet(), getAndSet(), and putIfAbsent(). 

  • double get()
    – Returns the element at the current position and then advances the position.
  • 
            
             
double put(double value)
    – Sets the element at the current position and then advances the position.
  • 
            
             
DoubleBuffer put(double[] src)
    – Copies the elements of the source array into this buffer starting at the current position and then advances the position.
  • DoubleBuffer flip()
    – Flips this buffer so that its limit is set to the current position and the position is set to zero.

Examples

Creating a DoubleBuffer

This example creates a DoubleBuffer with a capacity of 8 and loads it with a few numbers:


          
           
DoubleBuffer buf = DoubleBuffer.allocate(8);
buf.put(0.01);
buf.put(5.58);
buf.put(10.11);
buf.put(15.62);
// Capacity of buf is now 4

// Print out the contents of the buffer
while (buf.hasRemaining()) {
    System.out.println(buf.get());
}

// Output:
// 0.01
// 5.58
// 10.11
// 15.62

Flipping a DoubleBuffer

This example shows how to flip a DoubleBuffer from write mode to read mode:


          
           
DoubleBuffer buf = DoubleBuffer.allocate(8);
buf.put(1.23);
buf.put(4.56);
buf.put(7.89);
// Capacity of buf is now 3

// Flip the buffer
buf.flip();

// Print out the contents of the buffer
while (buf.hasRemaining()) {
    System.out.println(buf.get());
}

// Output:
// 1.23
// 4.56
// 7.89
