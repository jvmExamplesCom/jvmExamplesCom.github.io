What is org.antlr.v4.runtime.RecognitionException?

The org.antlr.v4.runtime.RecognitionException class is a subclass of the RecognitionException class in the Antlr 4 API, which is part of the Apache 2.0 License. It is used to signal when an error occurs while attempting to match a sequence of characters (or tokens) against a grammar or rule set.

Example

try {
    // match token against grammar
} catch (RecognitionException ex) {
    System.out.println("Failed to match token.");
}
