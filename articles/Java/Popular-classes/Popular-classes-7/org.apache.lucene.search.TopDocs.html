Introduction to org.apache.lucene.search.TopDocs

org.apache.lucene.search.TopDocs is a Java class used for retrieving search results from a Lucene search engine. It provides access to the documents that matched the query, their scores and the total number of documents that matched the query. This class is part of the Lucene Core API and can be used together with other Lucene components such as QueryParser, IndexReader and Searcher.

Methods of org.apache.lucene.search.TopDocs

The org.apache.lucene.search.TopDocs class has several methods that allow you to access and manipulate the data contained in it:

  • getMaxScore() – returns the maximum score of all documents returned by the query. It is useful for normalizing scores.
  • getScoreDocs() – returns an array of ScoreDoc objects that hold the documents matched by the query and their scores.
  • getTotalHits() – returns the total number of documents that matched the query.

Example Usage of TopDocs

The following example shows how to use the TopDocs class to retrieve the top 10 documents from a search query:


          
           
IndexSearcher searcher = new IndexSearcher(reader);
Query query = parser.parse(queryString);
TopDocs results = searcher.search(query, 10);

The code above creates a new IndexSearcher based on the provided IndexReader, creates a Query object from a query string and then performs a search on the IndexSearcher object, returning up to 10 results in a TopDocs object.

Accessing Document Data from TopDocs

Once you have the TopDocs object, you can access the individual ScoreDoc objects using the getScoreDocs() method. Each ScoreDoc contains the document ID and the score assigned to the document. You can use the document ID to retrieve the actual document from the index.


          
           
for (ScoreDoc sd : results.scoreDocs) {
    int docId = sd.doc;
    Document d = searcher.doc(docId);

In the code above, we iterate over the ScoreDoc array and for each of them we retrieve the document by its ID and perform some action on it.
