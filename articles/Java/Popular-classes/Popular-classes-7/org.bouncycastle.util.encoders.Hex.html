What is org.bouncycastle.util.encoders.Hex?

The Hex class provides methods for both encoding and decoding byte arrays, and also provides methods for translating byte arrays into hexadecimal strings and vice versa.

String hexEncodedData = Hex.toHexString(data);
System.out.println("Encoded: "+hexEncodedData);
// Outputs: Encoded: FFAB
