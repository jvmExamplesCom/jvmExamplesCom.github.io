Features and Usage

The FieldSpec class provides a way to easily create a field with a set of modifiers, type, and name. It also allows for annotations to be specified, which can be used to provide additional metadata to the field. 

          
           
FieldSpec field = FieldSpec.builder(int.class, "count")
