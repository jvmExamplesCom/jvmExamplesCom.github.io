java.nio.channels.SeekableByteChannel

java.nio.channels.SeekableByteChannel is an interface in the Java NIO package that defines a byte channel which supports seek operations as well as read and write operations. These seek operations allow the programmer to set a current position within the byte stream.

The SeekableByteChannel interface defines five methods for seeking, writing and closing operations. These methods are described in detail below:

Methods

  • position() - This method returns the current position within the byte stream.
  • position(long newPosition) - This method sets the current byte stream position to the value passed in as the argument.
  • read(ByteBuffer dst) - This method reads up to 
    dst
    length bytes from this channel into the given byte buffer.
  • write(ByteBuffer src) - This method writes all remaining bytes from the given byte buffer to this channel.
  • close() - This method closes the channel. All further operations on this channel will fail with an IOException.

Example Usage

Here is a code example demonstrating how to use SeekableByteChannel:


          
           
try (SeekableByteChannel channel = Files.newByteChannel()) {
    // get position of channel
    long position = channel.position();
    System.out.println("Current position: " + position);
  
    // set the position
    channel.position(5);
    System.out.println("Set position to 5.");

    ByteBuffer buf = ByteBuffer.allocate(10);
    int bytesRead = channel.read(buf);
    System.out.println("Number of bytes read: " + bytesRead);
  
    // write some bytes to the channel
    String text = "Hello world!";
    byte[] bytes = text.getBytes();
    channel.write(ByteBuffer.wrap(bytes));

    // close the channel
    channel.close();
} catch (IOException e) {
    e.printStackTrace();
}
