Java MongoDB DBObject

Basic usage

Creating a new DBObject instance is simple: 

          
           
DBObject obj = new BasicDBObject();
Since DBObject is an interface, the instance created here is actually an implementation of it—the BasicDBObject. This can be treated as any other DBObject instance. Now, we can start adding data to our new object. We do this with the put() method, which takes two arguments: the key (a string), and the value (any valid object type): 

          
           
obj.put("name", "John");
obj.put("age", 30);
The data is stored in the object and can be accessed with the get() method, which requires the key as an argument: 

          
           
String name = (String) obj.get("name");
int age = (int) obj.get("age");

obj.removeField("name");
