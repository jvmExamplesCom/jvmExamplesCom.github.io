javax.management.ObjectInstance

javax.management.ObjectInstance represents an MBean instance and is a part of the Java Management Extensions (JMX) API. It is used to represent an MBean instance and its associated ObjectName, and enables applications to maintain a list of MBeans.

ObjectInstance(String objectName, String className) 
ObjectInstance(ObjectName objectName)

ObjectName objectName = new ObjectName("com.example:type=Foo");
ObjectInstance objectInstance = new ObjectInstance(objectName);
