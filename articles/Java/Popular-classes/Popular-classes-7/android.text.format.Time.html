android.text.format.Time

android.text.format.Time is a Java class used to handle dates and times in Android applications. It provides various utility methods for manipulating date and time values, such as the ability to set time fields, retrieve time values, and compare dates. This article will explain how to use the android.text.format.Time class and provide numerous code examples to demonstrate its capabilities.

Construct Time Objects

To create a new Time object, you can use its default constructor, which sets the time to the current system time. 

          
           
Time curTime = new Time();
You can also create a new Time object from an existing calendar object, such as GregorianCalendar, or from another Time instance: 

          
           
GregorianCalendar gc = new GregorianCalendar();
Time gcTime = new Time(gc);
Time copyTime = new Time(curTime);
You can also create a new Time object by setting its fields explicitly: 

          
           
Time myTime = new Time();
myTime.set(12, 15, 2020); // set hour, minute and year

Set Time Fields

Time t1 = new Time();
t1.set(10, 0, 2020);

Time t2 = new Time();
t2.setToNow();

Get Time Values

Once you have set the time, you can get its values using the methods getHour(), getMinute(), getSecond(), and so on, as well as the formatted string representation of the time using the format() method.

Compare Dates

It returns a positive result if the first Time is greater than the second, a negative result if the first is less than the second, and zero if they are equal. 

          
           
Time earlierTime = new Time();
earlierTime.set(9, 0, 2020);
