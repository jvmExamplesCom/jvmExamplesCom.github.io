Jenkins.model.Jenkins

Jenkins.model.Jenkins is an object-oriented Java class that lets developers create, configure, and maintain Jenkins instances. It offers various tools, utilities, and plugins to help achieve a continuous integration and deployment environment. It allows developers to build, test, and deploy projects in an automated way, without manual intervention or manual configuration.

Methods

Jenkins.model.Jenkins offers many versatile methods that developers can use to manage Jenkins instances. Some of the most used methods are listed below:
  • getPluginManager(): Returns the PluginManager instance.
  • getNode(): Returns the Node instance.
  • getInstance(): Returns the Jenkins instance.
  • createNode(): Creates and returns a new Node instance.
  • init(): Initializes and sets up Jenkins for the first time.
  • save(): Saves the current configuration to file.
  • load(): Loads the configuration from file.
  • enableSecurity(): Enables security-related settings.
  • setUpSecurity(): Sets up the security settings.
  • addView(): Adds a new view to the instance.

// Create a new Jenkins instance
Jenkins jenkins = Jenkins.getInstance();

// Initialize the instance
jenkins.init();

// Save the configuration to file
