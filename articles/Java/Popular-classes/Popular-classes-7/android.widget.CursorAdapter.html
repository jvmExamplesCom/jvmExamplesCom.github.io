Overview of android.widget.CursorAdapter

The android.widget.CursorAdapter class provides a powerful tool for displaying data sourced from a content provider's database query in a ListView or a GridView. It offers a number of features, including sorting, filtering and more, to help you create an efficient, good-looking UI.

Using android.widget.CursorAdapter

To use the CursorAdapter, create a subclass that overrides the required methods: 

          
           
newView()
and 

          
           
bindView()
. In 

          
           
newView()
, inflate the layout and return a new View that binds to the data source. In 

          
           
bindView()
, populate the View with the data from the cursor. Finally, set the CursorAdapter on your ListView or GridView using the 

          
           
setAdapter()
method.

        super(context, c);

        // Inflate your layout
    }
 
    @Override
    public void bindView(View view, Context context, Cursor cursor) {
        // Populate your view with data from the cursor
    }
}
