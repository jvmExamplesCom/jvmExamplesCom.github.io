Introduction to org.springframework.web.multipart.MultipartHttpServletRequest

org.springframework.web.multipart.MultipartHttpServletRequest is an interface to handle a multipart request which is a representation of a form submitted with multipart/form-data encoding. It is a sub-interface of the HttpServletRequest interface and extends it by providing access to the parts of the multipart request. This interface also provides methods for accessing the parameters and headers of each part of the request.

Methods provided by org.springframework.web.multipart.MultipartHttpServletRequest

The MultipartHttpServletRequest interface provides several useful methods including:

  • getParts()
    : returns the array of Part objects contained in the request.
  • 
            
             
getPart(String partName)
    : returns the Part object with the given name.
  • getParameterNames()
    : returns the names of all parameters contained in the request.
  • 
            
             
getParameterValues(String name)
    : returns the array of values for the parameter with the given name.
  • 
            
             
getAttribute(String name)
    : returns the value of the attribute with the given name.
  • 
            
             
getHeader(String name)
    : returns the value of the header with the given name.

Example usage of org.springframework.web.multipart.MultipartHttpServletRequest

In the following example, we will use the MultipartHttpServletRequest to process a simple multipart form with two text fields. We will then print out the values of the parameters in the form.


          
           
protected void doPost(HttpServletRequest req, HttpServletResponse resp) {
  MultipartHttpServletRequest mreq = (MultipartHttpServletRequest) req;
  String[] values = mreq.getParameterValues("name");
  System.out.println("Name: " + values[0] + " - " + values[1]);
}

In this example, we first cast the HttpServletRequest to a MultipartHttpServletRequest. We then retrieve the parameter values for the field with name "name" and print them out.
