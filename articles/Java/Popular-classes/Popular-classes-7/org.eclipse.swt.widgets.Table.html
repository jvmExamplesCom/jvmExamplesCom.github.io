Introduction to org.eclipse.swt.widgets.Table

org.eclipse.swt.widgets.Table is an interface within the Eclipse Standard Widget Toolkit. It allows developers to create and mange tables within their application user interfaces. Table provides a two-dimensional grid that can be used to display and edit data, as well as provide menus and tooltips for functions or extra data.

Attributes of org.eclipse.swt.widgets.Table

Table can be used to store various types of data, and provide a set of functions for manipulating these values. These attributes include:
  • Set and modify the layout style of the table
  • Set the number of columns and rows
  • Retrieve and access individual cells in the table
  • Allow sorting of items in the table
  • Embed other widgets within the table

The following snippet of code creates a table with three columns and three rows:

table.setLinesVisible(true);

// Create table columns

column1.setText("Column 1");

column2.setText("Column 2");

TableColumn column3 = new TableColumn(table, SWT.NONE);
column3.setText("Column 3");

for (int i = 0; i < 3; i++) {
    TableItem item = new TableItem(table, SWT.NONE);

}
