java.awt.Composite

Java.awt.Composite is an abstract class that allows two or more graphical objects or images to be combined in a single graphical composition. It provides the basic framework for creating and manipulating complex graphical objects. This class defines the composite operations, which are the basic operations used to construct complex graphic objects.
