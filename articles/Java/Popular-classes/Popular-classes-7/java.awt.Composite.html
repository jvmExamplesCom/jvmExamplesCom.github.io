java.awt.Composite

Java.awt.Composite is an abstract class that allows two or more graphical objects or images to be combined in a single graphical composition. It provides the basic framework for creating and manipulating complex graphical objects. This class defines the composite operations, which are the basic operations used to construct complex graphic objects.

Constructors

public Composite()
This constructor creates a Composite object. 

          
           
public Composite(int rule)

  • AlphaComposite.CLEAR

  • AlphaComposite.DST_IN
  • AlphaComposite.DST_OUT
  • AlphaComposite.SRC_OVER
  • AlphaComposite.SRC_IN
  • AlphaComposite.SRC_OUT

This constructor creates a Composite from an integer representing a blending rule and a float representing an alpha value.
