com.sun.jersey.api.client.Client

The com.sun.jersey.api.client.Client class is a part of the Java API for RESTful Web Services (JAX-RS) and is used for invoking RESTful web services. This class provides a uniform way of invoking both synchronous and asynchronous requests from a Java application.

Creating a Client Instance

To create an instance of the Client class, we can use the ClientBuilder class:


          
           
Client client = ClientBuilder.newClient();

This will create a new Client object which can then be used to make HTTP requests.
