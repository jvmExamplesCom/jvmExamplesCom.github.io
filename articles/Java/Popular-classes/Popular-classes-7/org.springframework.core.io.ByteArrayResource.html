org.springframework.core.io.ByteArrayResource

The 

          
           
org.springframework.core.io.ByteArrayResource
is a specialized 

          
           
Resource
implementation for a given byte array. It creates a ByteArrayInputStream on the given byte array, and is seen as Resource interface in the Spring framework.

Create a ByteArrayResource

Creating a 

          
           
ByteArrayResource
requires a byte array which can be provided as an argument to the constructor. 

          
           
byte[] bytes = { 0x48, 0x65, 0x6c, 0x6c, 0x6f };

ByteArrayResource resource = new ByteArrayResource(bytes);

Accessing the Contents of a Byte Array Resource

Once a 

          
           
ByteArrayResource
has been created, it can be used to access the contents of the underlying byte array. There are several methods available, including 

          
           
getInputStream()
, 

          
           
getDescription()
, and 

          
           
getFilename()
. For example, to access the contents of a 

          
           
ByteArrayResource
using a 

          
           
BufferedReader
, we could use the following code:

    System.out.println(line);

Setting a Description and Filename for a Byte Array Resource

A 

          
           
ByteArrayResource
can also have a description and a filename set by using the 

          
           
setDescription()
and 

          
           
setFilename()
methods. The following example sets both the description and filename for a
