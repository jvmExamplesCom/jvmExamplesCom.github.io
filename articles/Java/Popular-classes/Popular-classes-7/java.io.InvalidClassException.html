Introduction

The java.io.InvalidClassException is an Exception that is thrown when an invalid class is used to deserialize an object. Deserialization is the process of taking the serialized data and reconstructing the object from it. The InvalidClassException is thrown when a class does not match with the class it was originally serialized from.
