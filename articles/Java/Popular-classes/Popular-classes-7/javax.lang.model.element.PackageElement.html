javax.lang.model.element.PackageElement

The javax.lang.model.element.PackageElement interface represents a package program element. It provides access to information about the package, including: its simple name, its fully qualified name, and, if available, its documentation.

Package elements are used in the process of resolving a fully qualified name. They can also be used for creating instances of TypeElements when a class name is not known.

Example


          
           
// Create PackageElement from package name
String packageName = "java.util";
Elements elements = (Elements) processingEnv.getElementUtils();
PackageElement packageElement = elements.getPackageElement(packageName);
if (packageElement == null) {
 throw new IllegalArgumentException("Invalid package name.");
}

In the example above, we create a PackageElement by providing a package name. We use the method getPackageElement of the Elements interface which is obtained from the ProcessingEnvironment. If the package name is invalid, an IllegalArgumentException will be thrown.

Methods

The accessors of PackageElement provide access to basic information about the package. Below are some of these methods:

    - Returns the fully qualified name of the package.
  • boolean isUnnamed()
    - Returns true if the package is unnamed.

    - Returns the enclosed Element instances for the package.

Finding Types within a Package

You can use a PackageElement to locate types within that package:


          
           
// Construct a name on the form "com.example.MyClass"
Name className = packageElement.getQualifiedName().append("MyClass");
TypeElement typeElement = elements.getTypeElement(className);
if (typeElement == null) {
 throw new IllegalArgumentException("Invalid type name.");
}

In the example above, we construct a name for a type within the package by appending "MyClass" to the fully qualified name of the package. Then we use the method 


          
           
getTypeElement
of the Elements interface to obtain a TypeElement associated with the given name.
