Introduction to org.apache.commons.net.ftp.FTPClient

The FTPClient class from the org.apache.commons.net.ftp package provides an easy-to-use interface for communicating with FTP servers and performing various file transfers. This class supports secure protocols such as FTP/S (SFTP), as well as proxy access and data compression. It is capable of downloading and uploading files, creating and deleting directories, and listing directory contents. In this article, we will explore some basic and advanced uses of the FTPClient class.

Creating an FTP Client and Establishing a Connection

Before we can begin interacting with a remote FTP server, we must create and configure an instance of the FTPClient class. This can be done as follows: 

          
           
FTPClient ftp = new FTPClient();
ftp.configure(new FTPClientConfig());
ftp.connect(hostname);
if (!ftp.login(username, password)) {
    System.err.println("Failed to log in!");
    return;
}
Once an FTPClient has been created and connected, we can check its connection status by calling the isConnected() method: 

          
           
boolean connected = ftp.isConnected();
We can also set the transfer mode to binary or ASCII using the setFileType() method, depending on the type of file being transferred: 

          
           
ftp.setFileType(FTP.BINARY_FILE_TYPE);

Downloading Files from a Remote Server

Once an FTPClient object has been configured and connected to a remote server, downloading files is fairly straightforward. The downloadFile() method of the FTPClient class takes two parameters: 

          
           
ftp.downloadFile(String remoteFile, OutputStream local);
The first parameter is the name of the remote file to be downloaded. The second parameter is a valid OutputStream object where the data will be written. Here is an example of downloading a file from a remote server and writing it to a local file: 

          
           
try (OutputStream outputStream = new FileOutputStream("local-file.txt")) {
    ftp.downloadFile("remote-file.txt", outputStream);
} catch (IOException e) {
   System.err.println("Error downloading file: " + e.getMessage());
}

Uploading Files to a Remote Server

Similarly, uploading files to a remote server is done using the uploadFile() method: 

          
           
ftp.uploadFile(InputStream local, String remoteFile);
This method takes two parameters: a valid InputStream object containing the data to be uploaded, and the name of the remote file to create. Here is an example of uploading a local file to a remote server: 

          
           
try (InputStream inputStream = new FileInputStream("local-file.txt")) {
    ftp.uploadFile(inputStream, "remote-file.txt");
} catch (IOException e) {
   System.err.println("Error uploading file: " + e.getMessage());
}

Deleting Files and Directories

The FTPClient class also provides methods to delete files and directories. The deleteFile() method takes the name of the remote file to delete and returns a boolean indicating whether the operation was successful. Similarly, the deleteDirectory() method takes the name of the remote directory to delete and returns a boolean indicating success or failure. 

          
           
boolean deleted = ftp.deleteFile("remote-file.txt");
boolean dirDeleted = ftp.deleteDirectory("remote-dir");

Listing Directory Contents

The listFiles() method of the FTPClient class returns a list of all the files in the specified remote directory. This method takes two parameters: the remote directory to list, and an optional filter that can be used to return only files matching a certain criteria. 

          
           
FTPFile[] files = ftp.listFiles("/home/user", FileFilter filter);
