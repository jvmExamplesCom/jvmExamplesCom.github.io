Introduction to WebArchive

WebArchive Basics

The WebArchive class extends the standard JarArchive class. It provides the same functionality for creating and manipulating an archive, with some extra methods specific to web archives. When you create a WebArchive, you can add files, directories, and web application resources such as servlets, filters, and web-fragments, as well as web configuration information such as web.xml and other configuration files.

The WebArchive class provides methods for adding these resources either directly or through a ShrinkWrap Descriptor file. This allows you to quickly and easily create a web archive that can be deployed to an application server.
