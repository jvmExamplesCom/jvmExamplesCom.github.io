javafx.scene.control.ScrollPane

The javafx.scene.control.ScrollPane class provides a scrollable area to which other JavaFX nodes can be added. It is useful when the user needs to view or modify a large amount of content. Nodes can be scrolled horizontally and vertically within the ScrollPane, allowing for a wide range of usage scenarios. In this article, we will look at the features and usage of the ScrollPane class.

Creating a ScrollPane

Creating a ScrollPane is easy. Just create a new ScrollPane object and add the content to it: 

          
           
ScrollPane scrollPane = new ScrollPane();
scrollPane.setContent(node);
Here, 

          
           
node
is the node to be added to the ScrollPane. This example creates an empty ScrollPane with no content.

Scrolling

By default, the ScrollPane will attempt to scroll the content to make it visible. If the content is larger than the viewport of the ScrollPane then the scrolling can be enabled by setting the 

          
           
vbarPolicy
and/or 

          
           
hbarPolicy
properties to 

          
           
ScrollBarPolicy.ALWAYS
: 

          
           
scrollPane.setVbarPolicy(ScrollBarPolicy.ALWAYS);
scrollPane.setHbarPolicy(ScrollBarPolicy.ALWAYS);
This will cause the content to be scrollable both horizontally and vertically.

Adding Content

The content of a ScrollPane can be any kind of Node, including layouts and controls. To add content to a ScrollPane, simply set the 

          
           
content
property of the ScrollPane: 

          
           
scrollPane.setContent(node);
Where 

          
           
node
is the Node to be added.

Styling

The ScrollPane can be styled using CSS. The following example sets the background color to green and adds some padding to the ScrollPane: 

          
           
scrollPane.setStyle("-fx-background-color: green; -fx-padding: 10px;");

Conclusion

In this article, we have explored the javafx.scene.control.ScrollPane class, which provides a scrollable area to which other JavaFX nodes can be added. We have seen how to create a ScrollPane, enable scrolling, add content and style the ScrollPane.
