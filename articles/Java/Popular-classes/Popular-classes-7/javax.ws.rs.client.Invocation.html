Creating a New Invocation

A new Invocation can be created by calling the 
newBuilder()

Invocation.Builder

In its simplest form, the code below creates an empty Invocation using the 

.build()
method: 

          
           
Invocation inv = Invocation.newBuilder().build();

The

		.uri("https://api.example.com/rest")

.method()

		.build();
