The 
Invocation
class provides a way to instantiate and build requests for invocation of external resources. It is part of the JAX-RS (Java API for RESTful Web Services) Client API, and is a powerful tool for making client-side HTTP or HTTPS requests.

Creating a New Invocation

A new Invocation can be created by calling the 
newBuilder()
method after instantiating the Invocation class. This will return an 
Invocation.Builder
object, which is used to construct the request.

In its simplest form, the code below creates an empty Invocation using the 

.build()
method: 

          
           
Invocation inv = Invocation.newBuilder().build();

Building An Invocation With Arguments

The 
.build()
method can be called with several optional arguments, including the request URI and HTTP method: 

          
           
Invocation inv = Invocation.newBuilder()
		.uri("https://api.example.com/rest")
		.method(HttpMethod.POST)
		.build();
The argument to the 
.uri()
method should be the base URL of the external resource you want to access. The argument to the 
.method()
method should be one of the HttpMethod enum values (e.g. GET, POST, PUT, DELETE).

Adding Request Headers

You can add request headers to your invocation using the 

.header()

Invocation inv = Invocation.newBuilder()

		.header("Content-Type", "application/json")
		.header("X-My-Header", "MyHeader-Value")
		.build();
