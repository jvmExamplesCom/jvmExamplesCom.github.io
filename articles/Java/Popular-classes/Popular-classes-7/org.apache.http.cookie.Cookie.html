org.apache.http.cookie.Cookie

The org.apache.http.cookie.Cookie class represents the server response to a client's request in the form of an HTTP cookie. It contains all the cookies that were set by the server in its response header, including name, value, domain, path, expiration date, etc.

The following example code shows how to create a simple org.apache.http.cookie.Cookie object:

    cookies.addAll(Cookie.parse(value));
}
