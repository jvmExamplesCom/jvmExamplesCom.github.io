Overview of Credentials

Credentials
class is part of the Apache HttpClient library, which provides an easy-to-use API for interacting with HTTP servers. The

Credentials

Credentials
class also provides a mechanism for creating “compound” credentials, which combine multiple sets of credentials into a single object.
