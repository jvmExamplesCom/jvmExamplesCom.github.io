Overview of Credentials

The 

          
           
Credentials
class is part of the Apache HttpClient library, which provides an easy-to-use API for interacting with HTTP servers. The 

          
           
Credentials
class provides an abstraction for user credentials, making it possible to access an authentication service without requiring that the client code know the details behind the authentication protocol.

Creating Credentials

Credentials

The 

          
           
Credentials
class also provides a mechanism for creating “compound” credentials, which combine multiple sets of credentials into a single object. This is useful if an application needs to authenticate with multiple services. For example, suppose you need to authenticate with an OpenID Connect service, a Kerberos service, and a SAML service. You could create three separate 

          
           
Credentials
