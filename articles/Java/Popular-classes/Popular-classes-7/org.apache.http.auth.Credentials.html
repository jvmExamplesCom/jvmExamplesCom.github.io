Overview of Credentials

The 

          
           
Credentials
class is part of the Apache HttpClient library, which provides an easy-to-use API for interacting with HTTP servers. The 

          
           
Credentials

Credentials

Credentials
class also provides a mechanism for creating “compound” credentials, which combine multiple sets of credentials into a single object. This is useful if an application needs to authenticate with multiple services. For example, suppose you need to authenticate with an OpenID Connect service, a Kerberos service, and a SAML service.
