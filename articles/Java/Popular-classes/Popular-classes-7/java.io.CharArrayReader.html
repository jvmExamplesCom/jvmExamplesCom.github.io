Java IO CharArrayReader

The 

          
           
java.io.CharArrayReader
class is a subclass of the Reader class and allows for the reading of a character array. This is an efficient way of reading data from an array of characters in a single operation. The data can then be read, modified, or written back to the array as desired.

Constructor

The constructor for this class can be used to create a new instance of a CharArrayReader object. 


          
           
CharArrayReader(char[] buf)
The 
buf
parameter is a char array which contains the data that will be read in.

Methods

A CharArrayReader object provides a number of useful methods for reading data from the underlying character array.

  • int read()
    - reads a single character from the buffer, or returns -1 if there is no data left to read.
  • 
            
             
int read(char[] cbuf, int offset, int length)
    - reads up to the specified number of characters into the given array starting at the specified offset.
  • long skip(long n)
    - skips over the specified number of characters in the buffer.
  • void reset()
    - resets the reader so that the next read starts at the beginning of the buffer.
  • void close()
    - closes the reader.

Example Usage

Here is an example program which uses the CharArrayReader to read characters from a character array and write them back out to another array.


          
           
import java.io.CharArrayReader;

public class CharArrayIO {
    public static void main(String[] args) throws Exception {
        char[] origData = { 'h', 'e', 'l', 'l', 'o' };
        char[] destData = new char[origData.length];

        CharArrayReader reader = new CharArrayReader(origData);
        reader.read(destData);

        System.out.println(new String(destData));
    }
}
