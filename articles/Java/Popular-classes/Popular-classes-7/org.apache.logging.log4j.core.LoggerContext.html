org.apache.logging.log4j.core.LoggerContext

The 

          
           
org.apache.logging.log4j.core.LoggerContext
is a Java logging framework used to create loggers and associated log files. It provides an interface for configuring loggers and their respective appenders. It also allows for the configuration of log formatters, filters and other components related to logging.

Getting Started with LoggerContext

To get started with the 
LoggerContext
, first create an instance of the class by providing the name of the configuration file: 

          
           
LoggerContext ctx = new LoggerContext("logging.config");
Now you can access the various components of the 
LoggerContext
. To set up a logger, begin by getting its name and configuring it with a 
Level
, e.g. 

          
           
Logger logger = ctx.getLogger("com.example.MyLogger");
logger.setLevel(Level.INFO);

Next, you need to set an appender for the logger. Appenders are responsible for formatting the log messages and writing them to a certain destination (e.g. file, console, etc.). For example, to write the log messages to a file, you have to specify the file name and the layout:


          
           
Appender appender = FileAppender.createAppender("/path/to/file.log", 
  "PatternLayout", "false", "MyTest");
appender.start();
logger.addAppender(appender);

Finally, you can start logging messages by calling the 

log()
method on the logger: 

          
           
logger.log(Level.INFO, "This is an info message");

Customizing Log Formatting

By default, the log messages will be printed in the format defined by the 

PatternLayout
. However, you can customize the formatting of the log messages using the 
Layout
class. For example, you can create a custom layout by extending the 

          
           
AbstractStringLayout
class and overriding the 
format()
method to define a custom format: 

          
           
public class CustomLayout extends AbstractStringLayout {

  // The format of the log messages
  private static final String FORMAT = "custom %c %m%n";

  public CustomLayout() {
    super(FORMAT);
  }

  @Override
  public String format(LogEvent event) {
    String message = String.format(FORMAT, 
        event.getLoggerName(), event.getMessage().getFormattedMessage());
    return message;
  }
}
Then use the 
CustomLayout
when configuring the appender: 

          
           
Appender appender = FileAppender.createAppender("/path/to/file.log", 
  "CustomLayout", "false", "MyTest");
appender.start();
logger.addAppender(appender);
