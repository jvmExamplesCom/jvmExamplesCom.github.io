Introduction to com.google.android.gms.maps.SupportMapFragment

The SupportMapFragment class is a specialized Fragment that displays a GoogleMap view and allows for user interaction. This article will discuss how to use the SupportMapFragment class in your apps with code examples.

Setting Up the SupportMapFragment

The first step is to set up the SupportMapFragment in your layout: 

          
           
<fragment
  android:id="@+id/map_fragment"
  class="com.google.android.gms.maps.SupportMapFragment"
  android:layout_width="match_parent"
  android:layout_height="match_parent"/>
When the layout is inflated, your Activity will get a reference to the SupportMapFragment with the following code: 

          
           
SupportMapFragment mapFragment = (SupportMapFragment)
