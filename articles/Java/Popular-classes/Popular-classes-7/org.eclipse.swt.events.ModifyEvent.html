Introduction to org.eclipse.swt.events.ModifyEvent

ModifyEvent is a subclass of Event and is part of the Standard Widget Toolkit (SWT) and the Eclipse platform. It is used to notify an application when a widget has been modified. This article details ModifyEvent, its available methods and provides examples in Java.
