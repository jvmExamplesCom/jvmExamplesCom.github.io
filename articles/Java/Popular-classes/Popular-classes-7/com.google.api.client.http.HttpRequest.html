com.google.api.client.http.HttpRequest

The com.google.api.client.http.HttpRequest class represents a request to be sent over an HTTP connection. It is part of the Google API Client Library for Java, and provides an easy way to construct, configure, and send an HTTP request.

A HttpRequest object is created with a given 
HttpTransport
instance and a particular type of request such as GET or POST. An optional content type and request body can also be provided. Once constructed, the request can be configured with additional headers, parameters, or a timeout. The request can then be sent and the response received.

Creating a Request

A HttpRequest is created with two parameters: a 
HttpTransport
instance, and a 
HttpMethod
enum value (for example, 
HttpMethod.GET
or 
HttpMethod.POST
). An optional content type and request body can also be given. 

          
           
HttpRequest request = 
  new HttpRequest(transport, HttpMethod.GET, "https://example.com/");
request.setContentType("application/json");
request.setRequestBody("{\"foo\":\"bar\"}");

Configuring a Request

Once a request is constructed, it can be configured with additional headers, parameters, or a timeout. Headers are added with the 


          
           
setHeader(String name, String value)

addParameter(String name, Object value)
method. Finally, a timeout can be set with the 

          
           
setTimeout(int timeout)
method. 

          
           
request.setHeader("Accept", "application/json");
request.addParameter("page", 1);
request.setTimeout(500);

Sending a Request

execute()

HttpResponse

HttpResponse response = request.execute();
