com.intellij.openapi.util.text.StringUtil

The com.intellij.openapi.util.text.StringUtil class provides several methods for manipulating strings in the IntelliJ IDEA platform. It is part of a set of common utilities in the IDE and is meant to reduce code size in plugins.

Methods

TheStringUtil class has the following methods:

Common Methods

  • concat - Concatenates two strings.
  • compare - Compares two strings. 1 if str1 > str2, 0 if equal, -1 if str1 < str2
  • equalsIgnoreCase - Checks the equality of two strings, ignoring case.
  • startsWithIgnoreCase - Checks if one string starts with another, ignoring case.
  • endsWithIgnoreCase - Checks if one string ends with another, ignoring case.

Regex Methods

  • matches - Checks whether the given string matches regex.
  • split - Splits the given string by the specified regex.
  • replaceAll - Replaces all occurrences of a pattern in the given string with a substitution.

Character Manipulation Methods

  • camelCaseToUpperCase - Converts a camel-case identifier to its upper-case counterpart.
  • camelCaseToSentence - Converts a camel-case identifier to a sentence.
  • sentenceToUpperCase - Converts a sentence to its upper-case counterpart.
  • escapeStringCharacters - Escapes special characters in the given string.

Common Methods


          
           
String str1 = "Hello";
String str2 = "World";
String concatStr = StringUtil.concat(str1, str2);
int compareResult = StringUtil.compare(str1, str2);
boolean equalsIgnoreCaseResult = StringUtil.equalsIgnoreCase("hello", "HELLO");
boolean startsWithIgnoreCaseResult = StringUtil.startsWithIgnoreCase("Hello World", "hello");
boolean endsWithIgnoreCaseResult = StringUtil.endsWithIgnoreCase("Hello World", "world");

System.out.println("Concatenated string: " + concatStr);
System.out.println("Compare result: " + compareResult);
System.out.println("Equals ignore case result: " + equalsIgnoreCaseResult);
System.out.println("Starts with ignore case result: " + startsWithIgnoreCaseResult);
System.out.println("Ends with ignore case result: " + endsWithIgnoreCaseResult);

Regex Methods


          
           
String str = "This is a test";
String[] splitResult = StringUtil.split(str, "\\s");
String replaceAllResult = StringUtil.replaceAll(str, "\\s", "-");
boolean matchesResult = StringUtil.matches("This is a test", "This.*test");

System.out.println("Split result: " + Arrays.toString(splitResult));
System.out.println("Replace all result: " + replaceAllResult);
System.out.println("Matches result: " + matchesResult);
