Introduction to org.apache.tools.ant.Task

The org.apache\.tools.ant.Task class is an abstract base class in Apache Ant, a Java-based build tool. It provides an interface for writing and running tasks that are used to execute complex build operations. Task objects are created and configured by Ant scripts, which are read at compilation time and then executed each time the script is invoked.

<target name="pack">
    <zipcompress sourceDir="src" zipFileName="archive.zip/>
    <renametask targetDir="archive.zip">
</target>
