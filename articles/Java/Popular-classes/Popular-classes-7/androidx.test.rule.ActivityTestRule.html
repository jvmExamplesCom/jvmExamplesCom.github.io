androidx.test.rule.ActivityTestRule

The androidx.test.rule.ActivityTestRule is a JUnit rule which provides functional testing of a single activity. It is designed to launch activities in isolation and gives the test writer access to the underlying activity instance. @Rule

MyActivity activity = activityRule.launchActivity(null);

Intent intent = new Intent();
