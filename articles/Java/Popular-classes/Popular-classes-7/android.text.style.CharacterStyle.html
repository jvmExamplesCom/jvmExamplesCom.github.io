android.text.style.CharacterStyle

android.text.style.CharacterStyle is a base class for character-level text styling objects. This class is used by the android.text.SpannableString class and its descendants to mark regions of the text to which it has been applied with a CharacterStyle object.

Examples


          
           
CharacterStyle style = new ForegroundColorSpan(Color.RED);
SpannableString string = new SpannableString("This text is red.");
string.setSpan(style, 0, string.length(), Spanned.SPAN_EXCLUSIVE_EXCLUSIVE);
System.out.println(string);
// Output: "This text is red."

          

          
           
TextPaint paint = new TextPaint();
paint.setColor(Color.BLUE);
CharacterStyle style = new CharacterStyle() {
    public void updateDrawState(TextPaint tp) {
        tp.setColor(paint.getColor());
    }
};
SpannableString string = new SpannableString("This text is blue.");
string.setSpan(style, 0, string.length(), Spanned.SPAN_EXCLUSIVE_EXCLUSIVE);
System.out.println(string);
// Output: "This text is blue."
