The OnFailureListener interface enables developers to specify an action to take when an asynchronous task fails. The OnFailureListener will be triggered by the task object's fail() function.

Using OnFailureListener

Using OnFailureListener requires implementing the onFailure() method. The following code snippet illustrates basic usage of the OnFailureListener:


          
           
// Create instance of OnFailureListener
OnFailureListener onFailureListener = new OnFailureListener() {
  @Override
  public void onFailure(@NonNull Exception e) {
    // Handle the failure
  }
};

// Attach it to the task
MyTask task = new MyTask();
task.addOnFailureListener(onFailureListener);

The above code snippet defines an OnFailureListener and then assigns it to an existing task. Once the task fails, the OnFailureListener's onFailure() method will be invoked.
