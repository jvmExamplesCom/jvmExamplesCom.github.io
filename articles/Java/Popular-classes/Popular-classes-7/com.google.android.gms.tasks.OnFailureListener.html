Using OnFailureListener

Using OnFailureListener requires implementing the onFailure() method. The following code snippet illustrates basic usage of the OnFailureListener:


          
           
// Create instance of OnFailureListener
OnFailureListener onFailureListener = new OnFailureListener() {
  @Override

    // Handle the failure
  }
};

// Attach it to the task
MyTask task = new MyTask();
task.addOnFailureListener(onFailureListener);
