Introduction to org.springframework.context.i18n.LocaleContextHolder

org.springframework.context.i18n.LocaleContextHolder is a utility class provided by the Spring Framework. It is used to determine the Locale currently set in a thread-bound context. The LocaleContextHolder can be used to inform other parts of the application about the current Locale that is set.
