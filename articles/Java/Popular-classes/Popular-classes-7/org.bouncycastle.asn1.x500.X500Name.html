A Brief Overview of X.500 Distinguished Name

The X.500 standard is part of the OSI (Open System Interconnection) suite of protocols. The X.500 directory service provides a distributed platform for locating people, organizations and other resources across networks. A distinguished name is a unique identifier used in X.500 to locate an object within the directory service. It is comprised of relative distinguished names (RDNs) which are composed of attribute-value pairs.
