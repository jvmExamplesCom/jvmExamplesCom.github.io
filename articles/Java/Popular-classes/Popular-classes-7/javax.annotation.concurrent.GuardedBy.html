javax.annotation.concurrent.GuardedBy

javax.annotation.concurrent.GuardedBy is an annotation type used for fields, methods or types for specifying the lock that should be used for accessing instances of the annotated element. This annotation is used to specify a thread-safe program. It states the lock that should be held in order to access the annotated information.

Example 1: Field Annotation


          
           
@GuardedBy("this")
private int sum = 0;

public void add(int value) {
    synchronized (this) {
        sum += value;
    }
}
In the above example, 
@GuardedBy("this")
annotation indicates that the field 
sum
should only be accessed while having a lock on the instance of the enclosing object.

}

lock
object.
