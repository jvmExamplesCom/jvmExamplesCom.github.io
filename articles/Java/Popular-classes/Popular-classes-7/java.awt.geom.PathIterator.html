Java.awt.geom.PathIterator

Java.awt.geom.PathIterator is an interface in the Java programming language that allows for iteration over the segments of a path. It provides methods for retrieving the coordinates and type of the current path segment. It also allows for the alteration of the iteration by skipping points or returning only portions of the path.

Constructors

A PathIterator does not have any public constructors, instead it must be instantiated using static methods.


          
           
public static PathIterator create(Shape shape);

Instance Methods

The following methods can be used on a PathIterator instance.


          
           
public boolean isDone();
public int currentSegment(float[] coords);
public int currentSegment(double[] coords);
public int getWindingRule();
public void next();

Examples

The following example shows how to use a PathIterator to iterate over a path and print out the coordinates.


          
           
PathIterator pi = myPath.getPathIterator();
while(!pi.isDone()) {
    float[] points = new float[6];
    int type = pi.currentSegment(points);
    System.out.println("Type:" + type + " Points:" + Arrays.toString(points));
    pi.next();
}

PathIterator pi = myPath.getPathIterator();
while(!pi.isDone()) {
    // Skip type 1, curve segments
    if (pi.currentSegment(new float[6]) != 1) {
        // Do something with this segment
        // ...

        // Go the next segment
        pi.next();
    }
}

PathIterator pi = myPath.getPathIterator();
int windingRule = pi.getWindingRule();
