Java com.intellij.openapi.editor.Document Introduction

com.intellij.openapi.editor.Document
in Java is an interface that provides access to text documents in the IntelliJ IDE and allows developers to access, modify, and add text. Methods for Manipulating Text

com.intellij.openapi.editor.Document
interface provides methods for manipulating text. One of the most basic operations is to insert text into a document at a certain location.

int startOffset = 100;
String text = "Hello World!";
doc.insertString(startOffset, text);

Searching in Documents

com.intellij.openapi.editor.Document
interface also allows us to search for a given text string in a document. To do this, we can use the 
findString()
method.
