Introduction to org.springframework.data.mongodb.core.query.Query

The 

          
           
org.springframework.data.mongodb.core.query.Query
class helps developers create query objects that can be used to perform queries against a MongoDB database. This class provides object-oriented methods for creating, executing and managing MongoDB queries. In this article, we'll go over the basics of using the Query class and provide some practical examples for creating and executing queries.

Creating Queries with the Query Class

The most basic way to create a MongoDB query is to use one of the static 

          
           
createQuery()
methods. This method takes an array of criteria as arguments and returns a 

          
           
Query
object. For example, consider the following query that finds documents with a "name" field equal to "John": 

          
           
Query query = Query.createQuery(Criteria.where("name").is("John"));
It is also possible to mix and match different criteria in the same query. For example, this query retrieves all documents where the "name" field equals "John" and the "age" field is greater than 18: 

          
           
Query query = Query.createQuery(
    Criteria.where("name").is("John"), 
    Criteria.where("age").gt(18)
);

Person person = mongoTemplate.findOne(query, Person.class);
