Introduction to android.support.multidex.MultiDexApplication

Android supports using multiple dex files in its applications by using the MultiDexApplication class. This class allows developers to create an application that can handle more than 64K references when being compiled into a DEX file. What is android.support.multidex.MultiDexApplication?

android.support.multidex.MultiDexApplication is an abstract subclass of android.app.Application. It adds support for the Android runtime's multidex feature which enables multiple DEX files to be used in a single application. This allows developers to access more than 64K references in their application.

Why Use android.support.multidex.MultiDexApplication?

When developing Android applications, the Android runtime limits the amount of references that can be included in a single DEX file. This limitation can be problematic for larger projects that need to access more than 64K references. The MultiDexApplication class allows developers to bypass this limitation by enabling them to use multiple DEX files in their application.
