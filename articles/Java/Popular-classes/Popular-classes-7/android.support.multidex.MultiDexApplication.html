What is android.support.multidex.MultiDexApplication?

android.support.multidex.MultiDexApplication is an abstract subclass of android.app.Application. Why Use android.support.multidex.MultiDexApplication?

When developing Android applications, the Android runtime limits the amount of references that can be included in a single DEX file. The MultiDexApplication class allows developers to bypass this limitation by enabling them to use multiple DEX files in their application.
