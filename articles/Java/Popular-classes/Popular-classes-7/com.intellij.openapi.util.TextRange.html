The Meaning of com.intellij.openapi.util.TextRange

com.intellij.openapi.util.TextRange is an interface used to represent a range or span of text within a larger body of text. It has various utility methods designed to facilitate the editing and management of ranges. This article will delve into the use and meaning of TextRange, as well as provide code examples.

Creating a TextRange Instance

A TextRange instance can be created in a variety of ways. In its most basic form, it is created with a start and end index:


          
           
TextRange myRange = new TextRange(int startIndex, int endIndex);

The start index and end index can also be specified as TextPointers, which indicate an offset from the start of the text:


          
           
TextPointer startPointer = new TextPointer(int characterOffset);
TextPointer endPointer = new TextPointer(int characterOffset);
TextRange anotherRange = new TextRange(startPointer, endPointer);

Utility Methods

TextRange provides various utility methods designed to make the manipulation of text simpler. These include methods to shift a range by a certain amount, get the length of the range, or check if the range is valid within the current text:


          
           
TextRange shiftedRange = myRange.shiftLeft(int amountToShift);
int rangeLength = myRange.getLength();
boolean isValidRange = myRange.isValidIn(String text);

Examples

One practical use for the TextRange class is to highlight a certain portion of text. Here is an example of how one might go about using TextRange and some built-in functionality to do this:


          
           
//Create TextRange object
TextRange highlightRange = new TextRange(int startIndex, int endIndex);

//Find text to be highlighted
String text = getSomeText();

//Highlight text in TextRange
String modifiedText = highlightRange.applyTo(text, (range, stringBuilder) -> { 
    //Inserts opening &amp; closing highlight tags around TextRange
    stringBuilder.insert("<span style="background-color: yellow">", range.getStartOffset()); 
    stringBuilder.insert("</span>", range.getEndOffset());
});

//Output highlighted text
System.out.println(modifiedText);

This code should result in the desired output of the text with the appropriate range highlighted in yellow.

com.intellij.openapi.util.TextRange is a powerful tool for manipulating text, allowing for the easy creation, shifting, and validation of text ranges. With its utility methods, the TextRange class makes the task of managing text much easier.
