Introduction to org.springframework.util.ResourceUtils

org.springframework.util.ResourceUtils is a Java utility class that provides various convenience methods for working with external resource locations such as classpath, file system, etc. The utility class contains methods such as createRelative(), getFile(), getURL(), isJarURL() and others. It is part of the Spring framework's core library and is included in the spring-core jar.

Methods of ResourceUtils Class

CreateRelative()

The createRelative() method creates a new Resource instance from a given parameters. The syntax is:


          
           
public static Resource createRelative(Resource relativeTo, String relativePath) throws IOException

It takes two parameters: a source Resource instance, and a String for the relative path. For example, if we want to create a new Resource relative to a given file system path we can use:


          
           
Resource relative = ResourceUtils.createRelative(new FileSystemResource("/path/to/file"), "relative/path");

The createRelative() method throws IOException if an error occurred while reading or writing the resource.

GetFile()

The getFile() method returns a File instance corresponding to the given Resource. The syntax is:


          
           
public static File getFile(Resource resource) throws IOException

It takes a Resource instance as parameter and returns the File resource it implements. For example, if we have a FileSystemResource:


          
           
Resource resource = new FileSystemResource("/path/to/file");

We can get a File instance for it using the getFile() method:


          
           
File file = ResourceUtils.getFile(resource);

The getFile() method throws an IOException if an error occurs while reading or writing the resource.

GetURL()

The getURL() method returns a URL instance corresponding to the given Resource. The syntax is:


          
           
public static URL getURL(Resource resource) throws IOException

It takes a Resource instance as parameter and returns the URL it implements. For example, if we have a ClassPathResource:


          
           
Resource resource = new ClassPathResource("/path/to/file");

We can get a URL instance for it using the getURL() method:


          
           
URL url = ResourceUtils.getURL(resource);

The getURL() method throws an IOException if an error occurs while reading or writing the resource.

IsJarURL()

The isJarURL() method returns whether the given URL is a JAR URL or not. The syntax is:


          
           
public static boolean isJarURL(URL url)

It takes a URL object as parameter and returns true if the URL is a JAR URL i.e. if it has the form jar:Something!. For example, if we have a URL:


          
           
URL url = new URL("jar:http://example.com/foo.jar!/bar.txt");

We can check if it is a JAR URL or not using the isJarURL() method:


          
           
boolean isJarUrl = ResourceUtils.isJarURL(url);
