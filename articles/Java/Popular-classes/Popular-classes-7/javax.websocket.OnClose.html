javax.websocket.OnClose

javax.websocket.OnClose is an annotation for callback methods, used to set the closure of a websocket connection. It can be used to determine when the session ends and to perform any needed cleanup or closing operations.

Syntax

The syntax for javax.websocket.OnClose is as follows:


          
           
@OnClose
public void callbackMethod() {
//Your code goes here
}

Example

Here is an example of how this annotation can be used in practice:


          
           
@OnClose
public void closeConnection(Session session) {
    System.out.println("Connection closed");
    try {
        session.close();
    } catch (IOException e) {
        e.printStackTrace();
    }
}

In this example, we have defined a callback method which is invoked when the connection is closed. When the connection is closed, it prints out "Connection closed" and then attempts to close the session.

Output

The output of this code will simply be the message "Connection closed" when the connection closes.
