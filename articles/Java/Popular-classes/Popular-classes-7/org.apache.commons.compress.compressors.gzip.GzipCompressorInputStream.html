org.apache.commons.compress.compressors.gzip.GzipCompressorInputStream Explained

The 

          
           
GzipCompressorInputStream
class is a Java class within the library of Apache Commons Compress. The goal of the class is to provide an easy way to read data compressed with GZIP by making it read like any other input stream. It is usually used when files or data available over the net are being compressed using GZIP.

API Overview

The 

          
           
GzipCompressorInputStream
class provides quite a few useful methods. Here we give a quick overview of some of the more useful ones: 
  • 
            
             
public int read(byte[] b)
    : reads up to 
    b.length
    uncompressed bytes into an array. Returns number of bytes actually read.
  • public void close()
    : closes the inputstream and releases system resources.
  • 
            
             
public void mark(int readlimit)
    : marks the current position in the inputstream for resetting purposes.
  • public void reset()
    : resets the inputstream back to the marked position.
  • 
            
             
public boolean markSupported()
    : determines whether the inputstream supports mark and reset.
