Using org.eclipse.ui.PlatformUI

The 

          
           
org.eclipse.ui.PlatformUI
class provides the primary access point to the Eclipse user interface. It includes methods for creating, running, and disposing the user interface components. It also provides information about the current workbench page, active workbench window, active part or selection.

Creating a Workbench Window

The following code example shows how to create a new workbench window:


          
           
IWorkbenchWindow window = PlatformUI.createAndRunWorkbench(display, new MyWorkbenchAdvisor());
The 
display
parameter should be an instance of 
Display
, and the 
MyWorkbenchAdvisor
must implement the 
WorkbenchAdvisor
interface. The returned value is an 
IWorkbenchWindow
instance.

Running the Workbench

The following code example shows how to run the workbench. It invokes the 

run
method of the 
PlatformUI
class: 

          
           
PlatformUI.getWorkbench().run(false);

The above code will run the workbench in the normal mode, i.e. it will show the workbench windows. The 

false
parameter indicates that no additional workbench windows should be opened.

Disposing the Workbench Window

The following code example shows how to dispose a workbench window:


          
           
window.getShell().dispose();
The 
window
parameter should be an instance of 
IWorkbenchWindow
. The above code will dispose the specified window. After the invocation of the 
dispose
method, the specified window will no longer exist.

Getting the Active Workbench Window

The following code example shows how to get the active workbench window:


          
           
IWorkbenchWindow window = PlatformUI.getWorkbench().getActiveWorkbenchWindow();
The returned value is an 
IWorkbenchWindow
instance. This window is the currently active window.

Getting the Active Part

The following code example shows how to get the active part in the current workbench window:


          
           
IWorkbenchPart part = window.getActivePage().getActivePart();
The 
window
parameter should be an instance of 
IWorkbenchWindow
. The returned value is an 
IWorkbenchPart
instance. This part is the currently active part in the specified workbench window.

Getting the Current Selection

The following code example shows how to get the current selection in the current workbench window:


          
           
ISelection selection = window.getSelectionService().getSelection();
The 
window
parameter should be an instance of 
IWorkbenchWindow
. The returned value is an 
ISelection
instance. This selection represents the current selection in the specified workbench window.
