Java.net.HttpCookie

Java.net.HttpCookie is part of the Java SE (Standard Edition) package and provides a way to represent HTTP state management. It represents an individual cookie that can be stored in the browser’s memory. Java.net.HttpCookie objects can be created, manipulated, and stored in a client’s browser.

Date expires = new Date(now + 1000 * 60 * 60 * 24 * 7); //7 days
cookie.setMaxAge(expires.getTime());

          

          
           
// Set the domain

// Set the path for the cookie
