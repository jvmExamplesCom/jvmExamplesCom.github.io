javax.ws.rs.core.MultivaluedHashMap

The javax.ws.rs.core.MultivaluedHashMap class is an implementation of the MultivaluedMap interface which provides access to a set of key-value pairs. It stores elements in an unordered collection and allows read and write access. The keys are unique but values can be duplicated. This class provides a general-purpose associative array that can be used to store and retrieve arbitrary data.

Creating a MultivaluedHashMap

Use the constructor to create a new instance of the MultivaluedHashMap class. A parameterless constructor creates an empty map, or provide a Map object as an argument to populate the new instance with the provided elements.


          
           
// Create an empty map
MultivaluedHashMap<String, Object> map = new MultivaluedHashMap<>();

// Create a map from existing Map object
Map<String, Object> source = new HashMap<>();
source.put("key1", "value1");
source.put("key2", "value2");
MultivaluedHashMap<String, Object> map = new MultivaluedHashMap<>(source);

Storing Values

Use the put method to store a new element in the map. This method takes two parameters: the key and the value. The key must be unique; attempting to store an element with a duplicate key will overwrite the previous value associated with the key.


          
           
map.put("key1", "value1");
map.put("key2", "value2");

Reading Values

Use the get method to read a value from the map. This method takes one parameter, the key whose associated value should be read.


          
           
Object value1 = map.get("key1");
Object value2 = map.get("key2");

Removing Values

Use the remove method to delete an element from the map. This method takes one parameter, the key of the element to be removed.


          
           
map.remove("key1");
