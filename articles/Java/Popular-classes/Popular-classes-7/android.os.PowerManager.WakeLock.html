Android.os.PowerManager.WakeLock

The Android.os.PowerManager.WakeLock class provides access to device power management features from within an Android application. Power Management provides a way for applications to take control of the device's power levels and conserve energy by temporarily reducing the device's power consumption when not actively used. The API allows applications to acquire and release a WakeLock, which is a mechanism for controlling the device's power usage. Each WakeLock is represented by an object of type WakeLock. Before an application can acquire or release a WakeLock, it must first obtain a reference to a PowerManager instance.

Acquiring a WakeLock

To acquire a WakeLock, call the acquire() method on your PowerManager object. This will return a WakeLock object, or throw an illegal state exception if no resource could be acquired. When you have a WakeLock object, you can use it to control the device's power levels.

Releasing a WakeLock

When you are finished using a WakeLock, you should call the release() method on it. This will release the WakeLock and allow other applications to use it. It is important to remember to call this method when you are done with the WakeLock, as otherwise resources may be held un-necessarily.
