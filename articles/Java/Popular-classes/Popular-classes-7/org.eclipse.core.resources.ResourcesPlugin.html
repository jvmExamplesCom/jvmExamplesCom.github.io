org.eclipse.core.resources.ResourcesPlugin

The 

          
           
org.eclipse.core.resources.ResourcesPlugin
class is a major component of the Eclipse platform. It provides many of the core capabilities for managing projects, working sets, and the underlying workspace. This class exposes methods for updating the resources in a workspace, for creating new resources, for manipulating the workspace tree, for listening for workspace events, and for accessing the current workspace.

Creating a New Resource

A new resource can be created using the 
create
method. This method requires 4 parameters: the parent folder, the name of the resource, the resource type, and the update flag. 

          
           
IWorkspaceRoot workspaceRoot = ResourcesPlugin.getWorkspace().getRoot();
IFolder folder = workspaceRoot.getFolder("myProject/src/main/java");
IResource resource = ResourcesPlugin.create(folder, "MyClass.java", IResource.FILE, true);

This example creates a new file named "MyClass.java" in the "myProject/src/main/java" folder. Once created, the resource will be flagged to be updated during the next workspace synchronization.

Listening for Workspace Changes

The Eclipse platform provides a way to listen for changes in the workspace by registering a workspace listener. The listener can observe when resources are created, deleted, or moved.


          
           
WorkspaceListener myListener = new WorkspaceListener() {
    @Override
    public void resourceChanged(IResourceChangeEvent e) {
        String message = String.format("Resource %s was %s",
            e.getResource(),
            e.getType() == IResourceChangeEvent.POST_CHANGE ? "changed" : "created/deleted");
        
        System.out.println(message);
    }
};

ResourcesPlugin.getWorkspace().addResourceChangeListener(myListener);

This example creates a workspace listener that will print a message to the console whenever a resource is created, changed, or deleted. The listener can then be registered with the workspace.

Accessing the Workspace Tree

The 

          
           
org.eclipse.core.resources.ResourcesPlugin
class also provides methods for accessing the workspace's tree structure. The 
getRoot
method returns the workspace root, from which individual projects and resources can be obtained. 

          
           
IWorkspaceRoot workspaceRoot = ResourcesPlugin.getWorkspace().getRoot();
IProject[] projects = workspaceRoot.getProjects();
for (IProject project : projects) {
    System.out.println(project.getName());
}

This example prints the names of all the projects in the workspace.
