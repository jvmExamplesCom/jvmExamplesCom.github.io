org.apache.commons.lang3.ClassUtils

The org.apache.commons.lang3.ClassUtils class provides several useful utilities to deal with Java classes. It allows to identify, convert and manipulate java classes in various ways. For example, it can convert a given class name into a Class or primitive type, or check if the given class implements an interface. It also provides various methods to get the inner class, get the superclass, get the all interfaces of a class and much more.

Methods and Examples

This section discuss some of the commonly used methods in org.apache.commons.lang3.ClassUtils. With these examples you will be able to understand how this class works and how you can use it for your own applications.

Creating Classes from a Name: forName() and forNameWithInitialization()

The 
forName()
and 

          
           
forNameWithInitialization()
methods allow you to create a class from a string representing a class name. This can be useful when you want to instantiate a class at runtime. 

          
           
String className = "java.lang.Integer"; 
Class<?> clazz = ClassUtils.forName(className);
The 

          
           
forNameWithInitialization()
method is similar to 
forName()
but it additionally initializes the class. This can be useful when you want to ensure that all static blocks of the class have been executed. 

          
           
String className = "com.journaldev.MyClass"; 
Class<?> clazz = ClassUtils.forNameWithInitialization(className);

Is the Class a Primitive Type: isPrimitiveWrapper(), isPrimitiveOrWrapper() and isPrimitive()

The 

          
           
isPrimitiveWrapper()
, 

          
           
isPrimitiveOrWrapper()
and 
isPrimitive()
methods allow you to check if a class is a primitive type, a wrapper of a primitive type or both respectively. 

          
           
Class<?> intClazz = Integer.class;
boolean isPrimitiveWrapper = ClassUtils.isPrimitiveWrapper(intClazz); // true 
boolean isPrimitiveOrWrapper = ClassUtils.isPrimitiveOrWrapper(intClazz); // true 
boolean isPrimitive = ClassUtils.isPrimitive(intClazz); // false

The 
getAllInterfaces()
and 

          
           
getAllSuperclasses()
methods allow you to get all the interfaces and superclasses of a class. These methods can be useful when you want to introspect a class and find out its relationship with other classes. 

          
           
public class MyClass { // ... } 
Class<MyClass> clazz = MyClass.class;
Class<?>[] interfaces = ClassUtils.getAllInterfaces(clazz); 
Class<?>[] superclasses = ClassUtils.getAllSuperclasses(clazz);

You can also get the direct superclass of a class using the 

getSuperclass()
