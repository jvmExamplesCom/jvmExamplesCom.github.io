org.apache.commons.lang3.ClassUtils

The org.apache.commons.lang3.ClassUtils class provides several useful utilities to deal with Java classes. It allows to identify, convert and manipulate java classes in various ways. For example, it can convert a given class name into a Class or primitive type, or check if the given class implements an interface. It also provides various methods to get the inner class, get the superclass, get the all interfaces of a class and much more.

Methods and Examples

This section discuss some of the commonly used methods in org.apache.commons.lang3.ClassUtils. With these examples you will be able to understand how this class works and how you can use it for your own applications.
