What is com.alibaba.fastjson.TypeReference?

com.alibaba.fastjson.TypeReference is a powerful feature of the Alibaba FastJSON library for Java, which enables it to parse and serialize Java classes with type parameters, such as generic collections. It helps developers create more flexible JSON processing codes and allows to operate on raw types instead of relying on type casts.

Example of Use

Let's look at an example of how TypeReference can be used in practice. Suppose we have a simple POJO class, called Person: 

          
           
public class Person {
    private String name;
    private int age;
    // getters / setters omitted for brevity
}
Now, suppose we want to parse the JSON string 

          
           
{"name": "John", "age": 20}
into an instance of Person. Here's how this can be done with FastJSON: 

          
           
String jsonString = "{\"name\": \"John\", \"age\": 20}";
Person person = JSON.parseObject(jsonString, new TypeReference<Person>() {});
Notice here how we used the TypeReference in order to tell FastJSON that the JSON String should be parsed into an instance of Person.

Real-World Applications

The most common use of TypeReference is in parsing of generic types. Let's look at an example. Suppose we want to parse a JSON string like 

          
           
{"data":[1, 2, 3, 4, 5]}
into a generic type List . This can be done using the following code: 

           
            
String jsonString = "{\"data\":[1, 2, 3, 4, 5]}";
List<Integer> data = JSON.parseObject(jsonString, new TypeReference<List<Integer>>() {});
Here, we used the TypeReference to tell FastJSON that the JSON string will be parsed into type List .
