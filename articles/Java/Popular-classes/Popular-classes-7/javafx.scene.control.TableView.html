JavaFX TableView

The JavaFX TableView is a UI component that presents data in two dimensional form. It allows for the sorting, grouping, and filtering of data to be done quickly and efficiently. The TableView also has an associated data model which contains the data used in the table. This data model can be manipulated to provide different views of the data. In this article we will discuss how to use the TableView, as well as its associated data model.

Creating a TableView

To create a TableView in JavaFX, first create a TableView instance:


          
           
TableView<Object> tableView = new TableView<>();

The type parameter is the type of the elements in the TableView's associated data model. Once the TableView is created, columns can be added to it:


          
           
TableColumn<Object, String> column = new TableColumn<>("Name");

In this example, we create a TableColumn that displays Strings. Other types of objects can be displayed as well. Then, we add the column to the TableView:


          
           
tableView.getColumns().add(column);

Populating the TableView

Once the TableView has been created, we need to populate it with data. To do this, we need to create a data model associated with the TableView. We can use the following code to create a simple data model:


          
           
ObservableList<Object> data = FXCollections.observableArrayList();

We can then add objects to the data model as needed. Once the data model is populated, we need to set it as the TableView's data model:


          
           
tableView.setItems(data);

Sorting

The TableView allows for the sorting of data. To enable sorting for a particular column, set the column's sortType to SortType.ASCENDING or SortType.DESCENDING:


          
           
column.setSortType(SortType.ASCENDING);

The TableView will now sort the data when the user clicks on the column's header. The order of the sorting can be changed by clicking on the column's header multiple times. By default, the sorting will be done in ascending order. To change the order, simply pass SortType.DESCENDING instead.

Grouping

The TableView also supports the grouping of data. To group data, first create a List of Groups:


          
           
List<Group> groups = new ArrayList<>();

The Group class is a special class provided by JavaFX which defines a group of items. Then, add each group to the list as needed:


          
           
Group g1 = new Group("Group 1");
g1.getItems().add("Item 1");
g1.getItems().add("Item 2");
groups.add(g1);

Group g2 = new Group("Group 2");
g2.getItems().add("Item 3");
g2.getItems().add("Item 4");
groups.add(g2);

Finally, set the list of groups as the TableView's grouping model:


          
           
tableView.setGroups(groups);

Filtering

The TableView also supports the filtering of data. To filter the data, provide a Predicate to the TableView which returns a boolean value indicating whether or not the data should be included in the view:


          
           
tableView.setPredicate(predicate);

The predicate can be used to filter the data based on any criteria. For example, the following predicate will only include data with values greater than 5:


          
           
tableView.setPredicate(value -> {
    return Integer.parseInt(value) > 5;
});

Once the predicate is provided, the TableView will only show data that matches the criteria.

Example

Let's take a look at a complete example that creates a TableView and populates it with data:


          
           
TableView<Object> tableView = new TableView<>();

TableColumn<Object, String> column = new TableColumn<>("Name");
tableView.getColumns().add(column);

ObservableList<Object> data = FXCollections.observableArrayList();
data.add("John Doe");
data.add("Jane Doe");

tableView.setItems(data);

This example creates a TableView and adds a column to it. It then creates a data model and populates it with data. Finally, it sets the data model as the TableView's data model. The result is a TableView that displays the data.

Conclusion

In this article, we looked at how to use the JavaFX TableView. We discussed how to create a TableView, how to populate it with data, and how to enable sorting, grouping, and filtering. Finally, we saw a complete example that put all of these concepts together. With the information from this article, you should be able to create powerful TableViews with JavaFX.
