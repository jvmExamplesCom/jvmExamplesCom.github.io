Introduction to com.google.android.gms.ads.AdRequest

By creating an AdRequest object and setting its properties, developers can target their application’s audience, customize the ad request process, and specify which types of ads should appear in the application.

Using AdRequest

AdRequest request = new AdRequest.Builder().build();

Once the request has been created, it can be customized by setting various properties. For example, to set the gender of the target audience for the ad request, developers can use the .setGender(int) method:


          
           
request.setGender(AdRequest.GENDER_FEMALE);

Once the AdRequest is configured, it can be sent to the AdMob network. This can be done with the .loadAd(AdRequest) method:
