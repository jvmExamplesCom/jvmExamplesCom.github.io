javax.naming.Name

The 

          
           
javax.naming.Name
interface represents a generic name -- an ordered sequence of components. For example, one might represent the name of a person in terms of the attributes given name and family name: 

          
           
GivenName = John
FamilyName = Smith
FullName = John Smith
In this case, the Name would have two components, the string "John" and the string "Smith", in that order.

Creating a Name Object

You create a Name object by passing name components to the context's 

          
           
Context.createName()
method. This method creates an instance of a class that implements the Name interface. For example: 

          
           
Context ctx = new InitialContext();
Name n = ctx.createName( "John", "Smith" );
The components of the name can also be passed as an array of strings: 

          
           
String[] components = { "John", "Smith" };
Name n = ctx.createName( components );

Getting Components of a Name

The components of a Name can be obtained through the 

          
           
Name.getAll()
, 

          
           
Name.getPrefix()
, and 

          
           
Name.getSuffix()
methods. The 

          
           
Name.getAll()
method returns an array of strings containing all of the components in the name. For example: 

          
           
String[] components = n.getAll();
for ( int i = 0; i < components.length; i++ )
{
   System.out.println( components[i] );
}
This will print out each component of the Name in order. The 

          
           
Name.getPrefix()
method can be used to obtain the components of a Name up to a specified index. For example: 

          
           
String[] prefixComponents = n.getPrefix(1);
for ( int i = 0; i < prefixComponents.length; i++ )

   System.out.println( prefixComponents[i] );
}

Name.getSuffix()
method can be used to obtain the components of a Name starting at a specified index. For example: 

          
           
String[] suffixComponents = n.getSuffix(1);
for ( int i = 0; i < suffixComponents.length; i++ )
{
   System.out.println( suffixComponents[i] );
}
This will print out the last two components of the Name.

Adding and Removing Components from a Name

The components of a Name can be added and removed using the 

          
           
Name.add()

Name.remove()

Name.add()

The 

          
           
Name.remove()

n.remove();

Name.equals()
