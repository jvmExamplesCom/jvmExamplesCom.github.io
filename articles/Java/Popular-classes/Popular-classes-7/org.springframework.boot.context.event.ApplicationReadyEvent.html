org.springframework.boot.context.event.ApplicationReadyEvent

The 

          
           
ApplicationReadyEvent
is an event that is sent when a Spring Boot application has started and is ready to service requests. This event fires after the ApplicationContext is fully initialized and refreshed, and is useful for performing any final startup tasks in the application before it begins servicing requests.

Event listeners can be registered to listen for this event using the @EventListener annotation. In this example, we create a simple listener that prints out a message when the application is ready:


          
           
@Component
public class MyListener {
 
    @EventListener
	public void handleReadyEvent(ApplicationReadyEvent event) {
		System.out.println("My application is now ready to service requests!");
	}

}

When the application starts and is ready, the following will be printed out:


          
           
My application is now ready to service requests!

Alternatively, the ApplicationReadyEvent can be used to execute the necessary code to wire up a Java-based IoC container and configure the application once the application context is initialized. In this example, we use the ApplicationReadyEvent to register a bean with the Spring application context and print out a message once the application is ready:


          
           
@Component
public class MyConfig {

	@EventListener
	public void handleReadyEvent(ApplicationReadyEvent event) {
		System.out.println("Registering myBean");
		event.getApplicationContext()
    		.registerBean("myBean", MyBean.class);
		System.out.println("My application is now ready to service requests!");
	}

}

When the application starts, the following will be printed out:


          
           
Registering myBean
My application is now ready to service requests!

Conclusion

The ApplicationReadyEvent is a powerful tool that enables developers to customize their Spring Boot applications by executing code at startup or tying into existing infrastructure that isn’t managed by Spring Boot. Using the @EventListener annotation, developers can easily register event listeners that respond to the ApplicationReadyEvent and perform any necessary operations.
