javafx.animation.KeyFrame

JavaFX provides the 

          
           
KeyFrame
class, which is part of the javafx.animation package, for creating animation KeyFrames to be used in a timeline. A 

          
           
KeyFrame
contains the following properties: 
  • time
    (the time at which this frame occurs, in seconds)
  • value
    (the new value to be applied in this frame)
  • interpolator
    (the interpolator that defines the function used to calculate the value of the frame)
  • action
    (an optional action to be triggered when this frame occurs)
The 

          
           
KeyFrame
class is used to define the various stages in an animation timeline. The following example shows how to create a 

          
           
KeyFrame
with the specified values and an action: 

          
           
KeyFrame keyFrame = new KeyFrame(
  Duration.seconds(1.0), // time
  new Rectangle(0, 0, 10, 50), // value
  Interpolator.LINEAR, // interpolator
  event -> System.out.println("Action!") // action
);
In this example, a 

          
           
KeyFrame
is created that will occur after one second, which will set the value of a 

          
           
Rectangle
to a specified size, with a linear interpolator, and which will trigger the action of printing "Action!" to the console.

Using a Timeline

The 

          
           
Timeline
class is used to play an animation sequence consisting of one or more 

          
           
KeyFrame
s. The following example shows how to create and play a 

          
           
Timeline
containing three 

          
           
KeyFrame
s: 

          
           
// create timeline
Timeline timeline = new Timeline(
  new KeyFrame(Duration.ZERO, e -> System.out.println("Start!")), // frame 1
  new KeyFrame(Duration.seconds(1.0), e -> System.out.println("Middle!")), // frame 2
  new KeyFrame(Duration.seconds(2.0), e -> System.out.println("End!")) // frame 3
);

// play timeline
timeline.play();
In this example, a 

          
           
Timeline
containing three 

          
           
KeyFrame
s is created, with each frame triggering a different action when it occurs. When the 

          
           
play()
method is called, the 

          
           
Timeline
will execute each of its 

          
           
KeyFrame
s in succession, resulting in the output "Start!", "Middle!", and "End!" being printed to the console.

KeyFrame
