Java javax.lang.model.element.AnnotationValue

Java javax.lang.model.element.AnnotationValue is an interface that extends the AnnotatedConstruct interface and provides access to the value of an annotation type element. It represents a single element of an annotation type.

AnnotationValue Usage

The AnnotationValue class provides an API for getting and setting the value of an element in an annotation type. The Value can be an array of any type including basic primitives.

Using AnnotationValue

An AnnotationValue instance is created by calling getValue() on a given AnnotationMirror. To set the value of the AnnotationValue, use the setValue() method.

AnnotationValue Example

Consider the following example: We have an AnnotationMirror representing the MyAnnotation annotation. The annotation has one attribute, a boolean named "flag". We want to get and set the value of the flag attribute using AnnotationValue. 


          
           
AnnotationMirror am = ... // Get the AnnotationMirror for MyAnnotation
AnnotationValue av = am.getValue("flag");
boolean flag = (boolean) av.getValue();
av.setValue(!flag); // Set the new value of the flag attribute

In this example, we retrieve the AnnotationValue representing the flag attribute and store it in the variable av. We then call getValue() to get the current value of the flag attribute. We then call setValue() to set the value to the opposite of what it was.
