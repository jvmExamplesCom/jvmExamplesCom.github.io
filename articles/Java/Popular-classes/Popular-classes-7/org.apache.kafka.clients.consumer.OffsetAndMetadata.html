org.apache.kafka.clients.consumer.OffsetAndMetadata

OffsetAndMetadata is a class in the org.apache.kafka.clients.consumer package of the Apache Kafka Java client library. It is used to store the offset and metadata associated with a consumer group. OffsetAndMetadata holds the current position (offset) of the consumer in a topic partition. It also contains any additionnal metadata associated with this reading position, such as the timestamp when it was set and the commit-id.
