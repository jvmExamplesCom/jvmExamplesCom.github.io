javax.management.Attribute

The 

          
           
javax.management.Attribute
class is a Java management extension (JMX) that provides an application programming interface (API) for representing and manipulating access control lists (ACLs). The Attribute class provides a way to represent the attributes of a resource, such as the key and value pairs associated with the resource.

javax.management.Attribute

String
