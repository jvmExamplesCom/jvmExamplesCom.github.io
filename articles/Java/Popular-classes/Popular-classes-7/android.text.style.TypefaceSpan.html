android.text.style.TypefaceSpan

android.text.style.TypefaceSpan is a class that extends the CharacterStyle class and enables styling of a span of Text with a custom Typeface. It can be used to customize the font family of a text in a span, across different versions of android OS.

Code Examples

Here is an example of how to set up a TypefaceSpan for a SpannableString by passing a Typeface:


          
           
Typeface typeface = Typeface.createFromAsset(getContext().getAssets(), "fonts/Roboto-Light.ttf");
SpannableString string = new SpannableString("This is a text with a custom font");
string.setSpan(new TypefaceSpan(typeface), 0, string.length(), Spanned.SPAN_EXCLUSIVE_EXCLUSIVE);

And here is an example of setting a TypefaceSpan using the Constructor with a specified typeface:


          
           
Typeface typeface = Typeface.createFromAsset(getContext().getAssets(), "fonts/Roboto-Regular.ttf");
SpannableString string = new SpannableString("This is a text with a custom font");
string.setSpan(new TypefaceSpan(typeface), 0, string.length(), Spanned.SPAN_EXCLUSIVE_EXCLUSIVE);

The following example shows how to update the font family of a part of text using a TypefaceSpan:


          
           
Typeface typeface = Typeface.createFromAsset(getContext().getAssets(), "fonts/Roboto-Medium.ttf");
SpannableString string = new SpannableString("This is a text with a custom font"); 
int startIndex = 10;
int endIndex = 22;
string.setSpan(new TypefaceSpan(typeface),startIndex,endIndex, Spanned.SPAN_EXCLUSIVE_EXCLUSIVE);

TextView textView = (TextView) findViewById(R.id.text_view);
