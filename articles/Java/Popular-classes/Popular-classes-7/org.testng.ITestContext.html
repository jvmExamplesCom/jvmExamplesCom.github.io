Overview of org.testng.ITestContext

The org.testng.ITestContext interface provides the context of the current test run in TestNG. It is used to provide access to configuration methods, factory methods, and other information related to a TestNG test suite.

Methods of org.testng.ITestContext

The org.testng.ITestContext interface has 8 methods that are used to get different kinds of information. They are as follows:

  • getAttribute(String name): It is used to get the object stored in the test context for the given name.
  • getName(): It is used to get the name of the current test suite.
  • getSuite(): It is used to get the instance of the current TestNG suite.
  • getStartDate(): It is used to get the start date of the current test suite.
  • getEndDate(): It is used to get the end date of the current test suite.
  • getExcludedGroups(): It is used to get the list of excluded groups of tests.
  • getIncludedGroups(): It is used to get the list of included groups of tests.
  • getParameters(): It is used to get the list of parameters passed to the current test suite.
