android.text.style.AbsoluteSizeSpan

The android.text.style.AbsoluteSizeSpan class represents a span of text whose font size is set to an absolute size, independent of the text size of its parent view or the text it is in. It is used to change the size of text relative to the base text size.

Example Usage

The following example code shows how to use the AbsoluteSizeSpan class in Android. 

          
           
// Create a new AbsoluteSizeSpan with a font size of 20sp
AbsoluteSizeSpan absoluteSizeSpan = new AbsoluteSizeSpan(20);

// Create a new SpannableString object
SpannableString spannableString = new SpannableString("This is some text");

// Apply the AbsoluteSizeSpan to the spannableString
spannableString.setSpan(absoluteSizeSpan, 0, spannableString.length(), 0);

// Get the TextView and apply the spannableString
TextView textView = (TextView) findViewById(R.id.text_view);
textView.setText(spannableString);
Another example usage is shown below: 

          
           
// Create a new AbsoluteSizeSpan with a font size of 40sp
AbsoluteSizeSpan absoluteSizeSpan = new AbsoluteSizeSpan(40);

SpannableStringBuilder spannableStringBuilder = new SpannableStringBuilder("Some text");

// Get the TextView and apply the spannableStringBuilder
TextView textView = (TextView) findViewById(R.id.text_view);
textView.setText(spannableStringBuilder);

Properties of the AbsoluteSizeSpan

The following properties of the AbsoluteSizeSpan can be accessed and modified: 
  • int size
    - The size of the font in sp.
  • boolean dip
    - If set to true, the size refers to dip units, otherwise, it is in sp.
