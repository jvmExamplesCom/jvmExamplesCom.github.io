Introduction to org.eclipse.swt.events.SelectionListener

The org.eclipse.swt.events.SelectionListener is an interface that defines the methods for responding to the selection events. It consists of two methods; widgetSelected and widgetDefaultSelected. This interface is used to detect a selection in the widget and determine what should happen when the selection occurs.
