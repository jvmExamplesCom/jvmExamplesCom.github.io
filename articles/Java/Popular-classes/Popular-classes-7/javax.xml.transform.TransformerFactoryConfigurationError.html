Error

TransformerFactory tf = TransformerFactory.newInstance();
Transformer transformer = tf.newTransformer();

TransformerFactory

TransformerFactoryConfigurationError
is thrown. This error can be caught and handled as follows: 

          
           
try {

    // Handle the error here
}

TransformerFactory

interface and should be included in the classpath for applications making use of it. Alternatively, an implementation can be provided as part of a specific JDK version, or included within the application itself. Once an implementation is available, the
