javax.net.ServerSocketFactory

The javax.net.ServerSocketFactory is a Java foundation class used for creating server-side sockets used for receiving data from clients. The ServerSocketFactory establishes the protocol used for communication, creates the socket and configures it for the communication. It can also create both secure and unsecure server sockets.

Creating a ServerSocket Factory

To create a ServerSocketFactory, the simplest approach is to use the static method 

getDefault()
from the ServerSocketFactory class like this:

Once the ServerSocketFactory object is created, a socket can be created with the 

createSocket()

ServerSocket socket = factory.createSocket(8000, 10000);
