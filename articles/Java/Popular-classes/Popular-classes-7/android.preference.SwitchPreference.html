SwitchPreference is an extension of the Preference class in the Android frameworks. It provides a two-state toggleable option, represented as a Switch widget on the UI. It is typically used to enable/disable a setting or feature.

android:fragment
attribute: 

          
           
<SwitchPreference android:key="my_switch_pref" 
android:title="My Switch Preference"
android:summary="Some summary"
android:defaultValue="false"
android:fragment="com.example.MySettingsFragment"/>

SwitchPreference switchPref = (SwitchPreference) findPreference("my_switch_pref");
