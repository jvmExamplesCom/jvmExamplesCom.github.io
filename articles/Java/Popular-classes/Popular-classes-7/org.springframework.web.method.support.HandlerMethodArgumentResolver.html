Basics

HandlerMethodArgumentResolver is a strategy interface that defines a contract for resolving method parameters in request-handling components. A HandlerMethodArgumentResolver implementation can be registered with the Spring MVC DispatcherServlet in order to add custom argument resolution logic to handler methods.
