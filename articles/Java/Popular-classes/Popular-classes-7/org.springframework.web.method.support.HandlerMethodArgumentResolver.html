Basics

HandlerMethodArgumentResolver is a strategy interface that defines a contract for resolving method parameters in request-handling components. A HandlerMethodArgumentResolver implementation can be registered with the Spring MVC DispatcherServlet in order to add custom argument resolution logic to handler methods. The HandlerMethodArgumentResolver is called when Spring MVC encounters a parameter in a request-handling component. The HandlerMethodArgumentResolver implementation then evaluates the parameter and returns either a value for the parameter (if it is resolved successfully) or null (if it is not).
