JavaFX Stage Modality

JavaFX Stage Modality allows users to create a pop-up window that blocks interaction of the main window. This type of window is useful when a user needs to make multiple choices without any interruption. The following code example shows how to set up Modality for a JavaFX stage.


          
           
Stage primaryStage = new Stage();
primaryStage.initModality(Modality.APPLICATION_MODAL);
primaryStage.setTitle("Modality Example");
primaryStage.show();

Stage dialogStage = new Stage();
dialogStage.initModality(Modality.APPLICATION_MODAL);
dialogStage.setTitle("Modal Dialog");

DialogPane pane = new DialogPane();
pane.getButtonTypes().addAll(ButtonType.YES, ButtonType.NO);

dialogStage.showAndWait();

Stage blockingStage = new Stage();

blockingStage.setTitle("Modal Blocked Window");

blockingStage.showAndWait();
