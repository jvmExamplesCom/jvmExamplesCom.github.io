The MessageBuilder class provides an interface for creating messages in Spring. This class can be used to create messages from scratch, or to convert existing objects into messages.

Creating Messages

To create a message, use the static builder method on the MessageBuilder class. The builder method takes two arguments: the payload of the message and the headers of the message. The payload is the content of the message, while the headers are metadata that can be used to control how the message is processed.

For example, the following code creates a message with an integer payload and an “alert” header:


          
           
Message<Integer> message = MessageBuilder.withPayload(1)
    .setHeader("alert", "true")
    .build();

Message<Map<String, Object>> message = MessageBuilder.withPayload(data).build();
