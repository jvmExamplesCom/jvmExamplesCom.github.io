org.eclipse.swt.layout.FillLayout

The org.eclipse.swt.layout.FillLayout is a type of layout used in the Standard Widget Toolkit (SWT) to arrange other widgets in a container such as a Composite or window. It arranges its children in a single row or column, with each one taking up all the available space. It is useful for creating simple user interfaces that need to fill out the entire window.

Constructor

The following code instantiates a new FillLayout: 

          
           
FillLayout layout = new FillLayout();

The constructor has several parameters which can be used to customize the FillLayout:


          
           
FillLayout(int style, int marginWidth, int marginHeight)<br> 
 FillLayout(int style, int spacing)
The 
style
parameter is used to specify how the children will be laid out - either vertically or horizontally. The 
marginWidth

marginHeight
parameters are used to set the margin sizes for the layout. Lastly, the 
spacing
parameter is used to set the spacing between the children.

Example

The following example shows how to use a FillLayout to arrange two buttons in a window:


          
           
// Create a Shell and a Composite
 Shell shell = new Shell(display);
 Composite parent = new Composite(shell, SWT.NONE);
 
 // Set the layout to FillLayout
 FillLayout fillLayout = new FillLayout();
 fillLayout.setMarginHeight(20);
 parent.setLayout(fillLayout);
 
 // Create two buttons
 Button b1 = new Button(parent, SWT.PUSH);
 b1.setText("Button 1");
 Button b2 = new Button(parent, SWT.PUSH);
 b2.setText("Button 2");
 
 // Open the Shell
 shell.pack();
 shell.open();

When the above code is executed, the two buttons will be arranged in a single column with 20 pixels of margin height between them.
