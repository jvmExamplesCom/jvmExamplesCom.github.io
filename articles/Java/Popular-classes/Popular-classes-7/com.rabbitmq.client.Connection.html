com.rabbitmq.client.Connection

The com.rabbitmq.client.Connection class provides the functionality necessary to establish and manage a connection to a RabbitMQ server. Once connected, users can send messages, receive messages, and monitor the state of the connection.

Creating a Connection

To create a connection, use the static factory method ConnectionFactory.newConnection(). This method accepts two arguments: a connection configuration object, and an IFrameHandler object.


          
           
Connection connection = ConnectionFactory.newConnection(
    new ConnectionConfig(), 
    new IFrameHandler()
);

The ConnectionConfig object contains configuration settings for the connection, such as the hostname and port of the RabbitMQ server.


          
           
ConnectionConfig config = new ConnectionConfig();
config.host("192.168.0.1");
config.port(5672);

The IFrameHandler object is used to handle incoming frames. Incoming frames must be handled in order for messages to be received. For more information on working with frames, see the javadocs.

Channel channel = connection.createChannel()

channel.basicPublish(

Receiving Messages
