JavaFX Animation Timeline

JavaFX is a powerful Java tool for creating visually rich user interfaces. One of its most useful features is the built-in Animation Timeline, which allows users to create animated sequences in a visual timeline. In this article, we'll explore the basics of the Timeline class, along with some code examples to demonstrate how powerful it can be.

Overview

The Timeline class provides an interface for representing animations in a graphical timeline. It consists of an array of Animation objects that are applied sequentially over a specified duration. Animations can be used in a variety of ways, including transitions between different states and effects on UI elements.

Creating a Timeline

The Timeline constructor takes two parameters: a Duration and an array of Animation objects. The Duration specifies how long the timeline will take to complete, in milliseconds. For example: 

          
           
Timeline timeline = new Timeline(Duration.millis(3000),
    new FadeTransition(Duration.millis(1000), node));
This creates a timeline that will run for 3 seconds and apply a FadeTransition animation to the given Node.

Adding Animations to a Timeline

Animations can be added to a timeline using the add() method. The following example adds an animation to a timeline that fades a node from 0 opacity to 1 in 1 second: 

          
           
timeline.getChildren().add(new FadeTransition(Duration.millis(1000),
    node, 0, 1));
You can also add multiple animations at once by passing an array of Animations to the add() method. For example: 

          
           
timeline.getChildren().addAll(
    new TranslateTransition(Duration.millis(1000), node),
    new RotateTransition(Duration.millis(1000), node),
    new ScaleTransition(Duration.millis(1000), node));
This will add three animations to the timeline, which will move, rotate, and scale the node over a period of 1 second.

Playing and Pausing a Timeline

Once a timeline has been created and populated with animations, it can be played and paused using the play() and pause() methods. You can also access the current state of the timeline using the isPlaying() method. 

          
           
timeline.play();
if (timeline.isPlaying()) {
    timeline.pause();
}

Seeking a Timeline

It's possible to seek forward or backward in a timeline using the seek() method. This method takes a single parameter which is the time, in milliseconds, to seek to. For example: 

          
           
timeline.seek(2000); // Skip to 2 seconds into the timeline

Looping a Timeline

Timelines can be made to loop (play repeatedly) by setting the loopCount property to -1. This makes the timeline loop indefinitely until it is stopped. 

          
           
timeline.setCycleCount(-1); // Loop indefintely
