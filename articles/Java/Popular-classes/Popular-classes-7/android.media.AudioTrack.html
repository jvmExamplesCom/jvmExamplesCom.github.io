android.media.AudioTrack

android.media.AudioTrack is an audio track resource which allows users to play back audio files on their Android device. This class provides access to the hardware-specific components that are needed for playback, such as a microphone, mixer, and audio output devices. It also provides APIs for controlling the audio playback, including changing volume, playing sound effects, and more.

Code Examples

The following code examples demonstrate how to use the android.media.AudioTrack class.


          
           
// Create an instance of AudioTrack
AudioTrack track = new AudioTrack(audioType, sampleRate, channelConfig, audioFormat, bufferSize, mode); 

// Set the playback volume
track.setVolume(1.0f);  

// Play the audio track
track.play();

// Get the current playback volume
float volume = track.getVolume();

// Pause the audio track
track.pause(); 

// Release the audio track
track.release();
