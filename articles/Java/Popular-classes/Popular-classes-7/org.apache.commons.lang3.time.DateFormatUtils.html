Introduction to org.apache.commons.lang3.time.DateFormatUtils

The org.apache.commons.lang3.time.DateFormatUtils class is a utility class that provides fast and convenient methods for formatting dates and times as Strings. It can be used to format both java.util.Date and java.util.Calendar objects. The class is part of the Apache Commons Lang library, and it is designed to provide additional functionality not available in the core Java libraries.

Methods

This class contains several static methods for formatting date and time strings. The methods return a formatted String based on the supplied date or time object and the specified format pattern. Here are some of the most commonly used methods:

  •

    - Formats a Date object using the given pattern.
  •

    - Formats the given milliseconds timestamp to the given pattern, UTC-style.

Examples

Date date = new Date();
