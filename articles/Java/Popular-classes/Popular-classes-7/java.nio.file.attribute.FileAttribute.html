java.nio.file.attribute.FileAttribute

java.nio.file.attribute.FileAttribute is an interface that provides methods to retrieve and set the file attributes of a file or directory in a file system. It is used to capture the state of a file or directory at the time it is created. The following are some of the FileAttributes supported by FileAttribute:

FileOwnerAttributeView

The FileOwnerAttributeView interface provides methods to get and set the owner of a file/directory. Here is an example:


          
           
Path path = Paths.get("C:/myFile");
FileOwnerAttributeView
    view = Files.getFileAttributeView(path, FileOwnerAttributeView.class);
UserPrincipal owner = view.getOwner();
System.out.println("Owner is " + owner.getName());
Output: 

          
           
Owner is John Doe

PosixFileAttributeView

The PosixFileAttributeView interface provides methods to get and set file attributes on files/directories that support the POSIX family of operating systems, such as Linux and BSD. Here is an example:


          
           
Path path = Paths.get("C:/myFile");
PosixFileAttributeView
    view = Files.getFileAttributeView(path, PosixFileAttributeView.class);
Set<PosixFilePermission> permissions =
    view.getPosixFilePermissions();
System.out.println("The current permissions are: " + permissions);
Output: 

          
           
The current permissions are: [OWNER_READ, OWNER_WRITE, GROUP_READ, OTHERS_READ]

DosFileAttributeView

The DosFileAttributeView interface provides methods to get and set DOS (Disk Operating System) specific file attributes on files/directories that support the DOS operating system. Here is an example:


          
           
Path path = Paths.get("C:/myFile");
DosFileAttributeView
    view = Files.getFileAttributeView(path, DosFileAttributeView.class);
view.setHidden(true);
boolean isHidden = view.isHidden();
System.out.println("Is hidden? " + isHidden);
Output: 

          
           
Is hidden? true

AclFileAttributeView

The AclFileAttributeView interface provides methods to get and set Access Control Lists (ACLs) on files/directories. Here is an example:


          
           
Path path = Paths.get("C:/myFile");
AclFileAttributeView
    view = Files.getFileAttributeView(path, AclFileAttributeView.class);
List<AclEntry> list = view.getAcl();
System.out.println("ACL entries: " + list);
Output: 

          
           
ACL entries: [user:FULL_CONTROL, group:READ, others:NONE]
