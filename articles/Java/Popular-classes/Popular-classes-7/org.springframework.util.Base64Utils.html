Java org.springframework.util.Base64Utils

Base64 is a common encoding scheme used to represent binary data in an ASCII string format. Java provides a class called org.springframework.util.Base64Utils which provides utility methods for encoding and decoding data using the Base 64 encoding scheme.

Example 1: Encode a String

The following code example shows how to encode a string using the Base64Utils class:


          
           
String originalStr = "hello world";
String encodedStr = Base64Utils.encodeToString(originalStr.getBytes());

// Output: aGVsbG8gd29ybGQ=

Example 2: Decode a String

The following code example shows how to decode a string using the Base64Utils class:


          
           
String encodedStr = "aGVsbG8gd29ybGQ=";
byte[] decodedBytes = Base64Utils.decodeFromString(encodedStr);
String decodedStr = new String(decodedBytes);

// Output: hello world
