InetAddresses

Overview of InetAddresses

The 
InetAddresses
class is a utility class. It has several static methods which allow you to work with IP addresses. It can be used to check if an IP address is valid, format it as a renderable string, get the subnet mask of an IP address, and more.

Using InetAddresses

In this section, we'll demonstrate the use of the 
InetAddresses
class with code examples.

isInetAddress(String)
