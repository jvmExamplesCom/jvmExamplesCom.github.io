What is javafx.stage.StageStyle?

javafx.stage.StageStyle is an enumeration class in the JavaFX API that defines the possible styles for a JavaFX Stage. A StageStyle is used to define the look and feel of the application window. There are 8 different options available for the stage style: UNDECORATED, TRANSPARENT, UTILITY, UNIFIED, DECORATED, TITLED, POPUP, and UNIFIED_SEPARATOR.

UNDECORATED Style

The UNDECORATED style provides no decorations on the window. The window will have no title, borders, or system menu. This style is useful for creating a borderless window. 


          
           
Stage primaryStage = new Stage();
primaryStage.initStyle(StageStyle.UNDECORATED);

TRANSPARENT Style

The TRANSPARENT style provides a transparent background for the window. The contents of the window will be visible but the window will not receive user input events. This style can be used for adding special effects to the application window. 


          
           
Stage primaryStage = new Stage();
primaryStage.initStyle(StageStyle.TRANSPARENT);

UTILITY Style

The UTILITY style provides a minimal window with a close button. The window has no title, menu, or resize handle. This style is useful for windows that should perform short-term tasks such as printing or file browsing. 


          
           
Stage primaryStage = new Stage();
primaryStage.initStyle(StageStyle.UTILITY);

UNIFIED Style

The UNIFIED style provides a unified window with a single menu bar. The menu bar contains the standard system menu items such as Edit, File, Window, etc. This style is useful for applications that have a single menu bar or have multiple windows that share a single menu bar. 


          
           
Stage primaryStage = new Stage();
primaryStage.initStyle(StageStyle.UNIFIED);

DECORATED Style

The DECORATED style provides a window with a title bar, system menu, and window controls. The window decorations can be customized using the CSS styling language. 


          
           
Stage primaryStage = new Stage();
primaryStage.initStyle(StageStyle.DECORATED);

TITLED Style

The TITLED style provides a window with a title bar but no system menu or window controls. The window decorations can be customized using the CSS styling language. 


          
           
Stage primaryStage = new Stage();
primaryStage.initStyle(StageStyle.TITLED);

POPUP Style

The POPUP style provides a window with no decorations whatsoever. The window cannot be moved, resized, or brought to the front. This style is useful for creating popup windows. 


          
           
Stage primaryStage = new Stage();
primaryStage.initStyle(StageStyle.POPUP);

UNIFIED_SEPARATOR Style

The UNIFIED_SEPARATOR style provides a window with a separate menu bar for each window. Each window's menu bar contains the standard system menu items such as Edit, File, Window, etc. This style is useful for applications that have multiple windows with different menu bars. 


          
           
Stage primaryStage = new Stage();
primaryStage.initStyle(StageStyle.UNIFIED_SEPARATOR);
