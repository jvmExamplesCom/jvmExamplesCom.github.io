android.bluetooth.BluetoothSocket

android.bluetooth.BluetoothSocket is a class used for establishing a connection between two devices over Bluetooth. This class, along with its underlying protocol, allows for data exchange between the two devices in an efficient and secure manner. The API allows developers to access core Bluetooth functionalities such as service discovery, connection management, and data transmission.

Overview

The android.bluetooth.BluetoothSocket class provides the necessary functions to establish a Bluetooth connection between two devices. Connections are established using either a server-client or peer-to-peer architecture. The BluetoothSocket object acts as the interface for transmitting and receiving data over the established connection. Once connected, data is exchanged via InputStream and OutputStream objects.

The BluetoothSocket class also provides methods for managing the connection state and for discovering services available on the device. It also provides callbacks for events such as new data arrival and disconnection.

socket.connect();

The BluetoothSocket class provides methods for managing the connection state. To close a connection, use the close() method:

The close() method will terminate the connection and release all associated resources.
