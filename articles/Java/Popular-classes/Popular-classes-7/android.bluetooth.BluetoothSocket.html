android.bluetooth.BluetoothSocket

android.bluetooth.BluetoothSocket is a class used for establishing a connection between two devices over Bluetooth. Overview

The android.bluetooth.BluetoothSocket class provides the necessary functions to establish a Bluetooth connection between two devices. Connections are established using either a server-client or peer-to-peer architecture. The BluetoothSocket class also provides methods for managing the connection state and for discovering services available on the device. It also provides callbacks for events such as new data arrival and disconnection.
