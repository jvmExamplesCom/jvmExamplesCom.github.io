org.apache.poi.ss.usermodel.CellStyle

Overview

The 

          
           
org.apache.poi.ss.usermodel.CellStyle
class is used to define the style and formatting of an excel cell in a spreadsheet. It allows for the setting of font, colour, borders and background as well as alignment and more for a cell. The CellStyle object can then be set to a cell using the 

          
           
Cell.setCellStyle(CellStyle)
method.

Creating a CellStyle Object

The 

          
           
org.apache.poi.ss.usermodel.Workbook
class has a 

          
           
createCellStyle()
method which returns a new 

          
           
CellStyle
instance. 

          
           
Workbook workbook = new XSSFWorkbook();
CellStyle cellStyle = workbook.createCellStyle();

Setting Styles and Formatting

Once you have a 

          
           
CellStyle
instance, it can be configured using the various set methods.

Font

The cell's font can be set using the 

          
           
setFont(Font)
method. You must create a an instance of a 

          
           
org.apache.poi.ss.usermodel.Font
first. 

          
           
Font font = workbook.createFont();
font.setFontName("Arial");
font.setFontHeightInPoints((short) 14);
cellStyle.setFont(font);

Alignment

You can set the horizontal and vertical alignment of a cell using the 

          
           
setAlignment(HorizontalAlignment)
and 

          
           
setVerticalAlignment(VerticalAlignment)
methods. 

          
           
cellStyle.setAlignment(HorizontalAlignment.CENTER);
cellStyle.setVerticalAlignment(VerticalAlignment.CENTER);

Colour

You can set both the foreground and background colors of the cell text by making use of the 

          
           
setFillForegroundColor(short)
, 

          
           
setFillBackgroundColor(short)
and 

          
           
setFillPattern(FillPatternType)
methods. 

          
           
Workbook wb = new XSSFWorkbook(); 
        CellStyle cs = wb.createCellStyle();
        cs.setFillForegroundColor(IndexedColors.YELLOW.getIndex());
        cs.setFillBackgroundColor(IndexedColors.BLUE.getIndex());
        cs.setFillPattern(FillPatternType.SOLID_FOREGROUND);
The available colors are defined in the 

          
           
org.apache.poi.ss.usermodel.IndexedColors
enum.

Border

You can set the four borders of a cell using the 

          
           
setBorderTop(BorderStyle)
, 

          
           
setBorderRight(BorderStyle)
, 

          
           
setBorderBottom(BorderStyle)
and 

          
           
setBorderLeft(BorderStyle)
methods. 

          
           
cellStyle.setBorderTop(BorderStyle.THIN);
cellStyle.setBorderRight(BorderStyle.THIN);
cellStyle.setBorderBottom(BorderStyle.THIN);
cellStyle.setBorderLeft(BorderStyle.THIN);
The available border styles are defined in the 

          
           
org.apache.poi.ss.usermodel.BorderStyle
enum.

Applying a CellStyle

Once you have defined a style, you can apply it to a cell using the 

          
           
Cell.setCellStyle(CellStyle)
method. 

          
           
Cell cell = sheet.getRow(rowNum).getCell(colNum);
cell.setCellStyle(cellStyle);

Conclusion

The 

          
           
org.apache.poi.ss.usermodel.CellStyle
class provides the ability to define and set the style and formatting of an excel cell in a spreadsheet. You can set the font, color, alignment and borders as well as many other settings. Once the style is defined, it can be applied to a cell using the 

          
           
Cell.setCellStyle(CellStyle)
method.
