org.eclipse.core.resources.IResource

IResource is a type of resource available in the Eclipse platform. It is an interface used for representing different types of resources such as projects, folders, and files in the workspace. IResource allows for working with different types of resources in the same manner and provides access to their properties such as name, absolute path and last modified dates. Additionally, it also provides methods for creating, deleting, moving, and copying resources.

The following code snippet demonstrates the creation of a new folder using the IResource interface:


          
           
IFolder folder = project.getFolder("newFolder");
folder.create(true, true, null); // Create the folder

Get Workspace Resources

The IResource interface also provides methods for getting resources from the workspace. The getResource() method returns a resource from a given path. This method can be used to get resources for a specific project, folder or file.


          
           
IWorkspaceRoot root = ResourcesPlugin.getWorkspace().getRoot();
IProject project = root.getProject("MyProject");
IProject[] projects = project.members(false);

The above code snippet gets all the projects in a workspace. Similarly, it is possible to get a specific file or folder from the workspace using the getFile() and getFolder() methods respectively.


          
           
IResource file = project.getFile("myFile.txt");
IResource folder = project.getFolder("myFolder");

Access Properties

IResource provides methods for accessing the properties of resources such as name, absolute path, and last modified dates. The getName() method returns the name of the resource. It can be used to get the name of a project, folder or file.


          
           
String name = resource.getName();

Similarly, the getLocation() method returns the absolute path of the resource. It can be used to get the absolute path of a project, folder or file.


          
           
IPath location = resource.getLocation();

The getModificationStamp() method returns the last modified date of a resource. This can be used to get the last modified date of a project, folder or file.


          
           
long timestamp = resource.getModificationStamp();

IResource provides methods for creating, deleting, moving, and copying resources. The create() method can be used to create a new resource. IFile file = project.getFile("myFile.txt");

The delete() method deletes a resource. It takes one argument: boolean flag indicating whether to delete sub-resources recursively.

The move() method moves a resource to a specified destination. It takes three arguments: an IResource object indicating the destination, a boolean flag indicating whether to overwrite existing resources, and an instance of IProgressMonitor for tracking progress.


          
           
IFolder folder = project.getFolder("myFolder");
