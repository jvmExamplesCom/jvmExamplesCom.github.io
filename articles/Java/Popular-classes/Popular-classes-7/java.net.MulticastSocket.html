Java.net.MulticastSocket

Java's multicast socket class java.net.MulticastSocket enables applications to send and receive data to and from multiple hosts simultaneously over network. MulticastSocket is a subclass of DatagramSocket, and thus extends its features. This class is most commonly used for broadcasting messages across LAN or WAN networks.

To create a MulticastSocket, we must specify a port number on which to bind the socket. To join a multicast group, the application must use the joinGroup() method, specifying a multicast address. To leave a multicast group, the application must use the leaveGroup() method. The setTimeToLive(int ttl) method allows the application to specify the Time-To-Live (TTL) value for multicast packets. This is the number of hops (routers) the packet will travel until it expires.

Let's look at an example of code. 

          
           
import java.net.*;

public class MulticastSend {
   public static void main(String[] args) throws Exception {
      System.out.println("Sending Packet");
      InetAddress group = InetAddress.getByName("230.0.0.1");
      MulticastSocket s = new MulticastSocket();
      s.setTimeToLive(1);
      DatagramPacket hi = new DatagramPacket("Hello".getBytes(), "Hello".length(), group, 4446);
      s.send(hi);
      s.close();
   }
}

In this example, we create a MulticastSocket and bind it to port 4446. We then set a TTL value of 1, indicating that the packets will travel through a maximum of 1 router hop. We create a DatagramPacket with the message "Hello" and specify the group address 230.0.0.1 as the destination. Finally, we send the packet and close the socket.

We can also receive packets from groups through a MulticastSocket. The following code example shows how to receive packets from a multicast group.


          
           
import java.net.*;

public class MulticastReceive {
   public static void main(String[] args) throws Exception {
      System.out.println("Receiving Packet");
      InetAddress group = InetAddress.getByName("230.0.0.1");
      MulticastSocket s = new MulticastSocket(4446);
      s.joinGroup(group);
      byte[] buf = new byte[1000];
      DatagramPacket recv = new DatagramPacket(buf, buf.length);
      s.receive(recv);
      String msg = new String(recv.getData(), 0, recv.getLength());
      System.out.println("Received: " + msg);
      s.leaveGroup(group);
      s.close();
   }
}

In this example, we create a MulticastSocket and bind it to port 4446. We then join the group address 230.0.0.1. We create a buffer and DatagramPacket to receive packets, and then use the receive() method to receive a packet. Finally, we print out the received message and leave the group before closing the socket.

MulticastSocket makes it easy to send and receive data between multiple hosts. It allows applications to broadcast data to a large number of recipients without requiring direct connections between each sender and receiver, making it ideal for large-scale data distribution.
