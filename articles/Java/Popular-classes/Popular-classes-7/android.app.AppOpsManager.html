Android.app.AppOpsManager

Android AppOpsManager API is a powerful tool that enables developers to manage how the system manages application operations. It provides an API with which developers can control or set permissions for specific system operations such as camera usage, Bluetooth access, and more. This API provides system administrators with a way to control and restrict applications’ operations including managing data access and controlling other operations.

How to use Android AppOpsManager API

Using the Android AppOpsManager API requires a few steps. First, you need to include the appropriate dependencies in your app. Then, request permission to use the AppOpsManager API by adding the 

          
           
android.permission.APP_OPS_MANAGE
permission to your app's 

          
           
AndroidManifest.xml
file. Next, instantiate an 

          
           
AppOpsManager
object with the application context. Finally, call the appropriate methods to perform the desired operations.

Examples of using Android AppOpsManager API

The following code snippets show a few examples of using the Android AppOpsManager API.

Check if a permission is granted


          
           
// Get an instance of AppOpsManager
AppOpsManager appOpsManager = (AppOpsManager)getSystemService(Context.APP_OPS_SERVICE);

// Check whether a permission is granted
int result = appOpsManager.checkOpNoThrow(
    AppOpsManager.OPSTR_READ_SMS, Process.myUid(), getPackageName());

if (result == AppOpsManager.MODE_ALLOWED) {
    // Permission is granted
} else {
    // Permission is not granted
}

Change a permission state


          
           
// Get an instance of AppOpsManager
AppOpsManager appOpsManager = (AppOpsManager)getSystemService(Context.APP_OPS_SERVICE);

// Change the permission state
appOpsManager.setMode(
    AppOpsManager.OPSTR_WRITE_EXTERNAL_STORAGE, 
    Process.myUid(), 
    getPackageName(), 
    AppOpsManager.MODE_ALLOWED);

Check if a given set of permissions are granted


          
           
// Get an instance of AppOpsManager
AppOpsManager appOpsManager = (AppOpsManager)getSystemService(Context.APP_OPS_SERVICE);

// Create an array with the permissions to check
String[] permissions = {
    AppOpsManager.OPSTR_CAMERA,
    AppOpsManager.OPSTR_READ_SMS
};

// Check if the permissions are granted
int[] results = appOpsManager.checkOpNoThrow(permissions, Process.myUid(), getPackageName());

for (int i = 0; i < results.length; i++) {
    if (results[i] == AppOpsManager.MODE_ALLOWED) {
        // Permission is granted
    } else {
        // Permission is not granted
    }
}
