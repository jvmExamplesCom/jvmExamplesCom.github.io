org.openqa.selenium.JavascriptExecutor

Introduction

org.openqa.selenium.JavascriptExecutor is an interface that helps to execute JavaScript in the context of the currently selected frame or window in Selenium. It provides several methods for manipulating the browser window, such as resizing it, getting its position, changing its position, and so on.

Methods

The org.openqa.selenium.JavascriptExecutor has several methods that can be used to interact with the browser window:

  • 
            
             
executeScript(String script, Object... args): Object
    - Executes a given JavaScript code snippet in the context of the currently selected frame or window. It returns the result of the evaluated code.
  • 
            
             
executeAsyncScript(String script, Object... args): Object
    - Executes an asynchronous JavaScript code snippet in the context of the currently selected frame or window. It returns the result of the evaluated code.
  • 
            
             
getWindowPosition(): Point
    - Returns the position of the browser window.
  • 
            
             
setWindowPosition(int x, int y): void
    - Sets the position of the browser window.
  • 
            
             
getWindowSize(): Dimension
    - Returns the size of the browser window.
  • 
            
             
setWindowSize(int width, int height): void
    - Sets the size of the browser window.
  • 
            
             
getScreenOrientation(): String
    - Returns the screen orientation of the browser window.
