Introduction to org.springframework.web.servlet.mvc.support.RedirectAttributes

The org.springframework.web.servlet.mvc.support.RedirectAttributes class is a helper class used in Spring MVC to store and retrieve URL variables in-between redirects. It helps the process of passing attributes between different requests such as form submission and result request. This tutorial provides an introduction to the RedirectAttributes class, showing how it can be used to save data between requests.

Using RedirectAttributes

To use the RedirectAttributes class, you first need to create an instance of it:


          
           
RedirectAttributes redirectAttributes = new RedirectAttributes();

Once you have an instance of RedirectAttributes, you can add attributes to it using its 

addAttribute()
method. This method takes two arguments, the name of the attribute and its value. For example, the following code adds an attribute called 
message
with a value of 
Hello World!
: 

          
           
redirectAttributes.addAttribute("message", "Hello World!");

You can then access the attribute's value by calling the

String message = redirectAttributes.getAttribute("message");

To save the attribute for later use, you can call the 

save()
method. This method takes two arguments, the name of the attribute and its value. For example, the following code saves the 
message
attribute with its current value: 

          
           
redirectAttributes.save("message", message);

Example

Let's look at a simple example to demonstrate the use of RedirectAttributes. We'll create a simple web page with a form that takes input from the user and then passes it on for further processing. We'll use the RedirectAttributes class to save the input from the form so that we don't have to re-enter it on each request.

First, let's create a FormController class to handle the form submission. This class will take the input from the user and store it in the RedirectAttributes instance. Here's the code for the FormController class:


          
           
@Controller
public class FormController {

	@RequestMapping(value="/form")
	public String submitForm(@RequestParam(value="name") String name, RedirectAttributes redirectAttributes) {
		String message = "Hello " + name + "!";
		
		redirectAttributes.addAttribute("message", message);
		redirectAttributes.save("message", message);
		
		return "redirect:/result";
	}

}

public class ResultController {

	@RequestMapping(value="/result")

		String message = redirectAttributes.getSavedAttribute("message");
		
		model.addAttribute("message", message);
		
		return "result";
	}

}
