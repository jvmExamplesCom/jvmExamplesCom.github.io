Introduction to org.springframework.web.servlet.mvc.support.RedirectAttributes

The org.springframework.web.servlet.mvc.support.RedirectAttributes class is a helper class used in Spring MVC to store and retrieve URL variables in-between redirects. This tutorial provides an introduction to the RedirectAttributes class, showing how it can be used to save data between requests.

RedirectAttributes redirectAttributes = new RedirectAttributes();

redirectAttributes.addAttribute("message", "Hello World!");
