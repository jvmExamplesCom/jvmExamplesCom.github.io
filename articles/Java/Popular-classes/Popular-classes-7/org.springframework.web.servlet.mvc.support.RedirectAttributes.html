Introduction to org.springframework.web.servlet.mvc.support.RedirectAttributes

The org.springframework.web.servlet.mvc.support.RedirectAttributes class is a helper class used in Spring MVC to store and retrieve URL variables in-between redirects. It helps the process of passing attributes between different requests such as form submission and result request. This tutorial provides an introduction to the RedirectAttributes class, showing how it can be used to save data between requests.

Using RedirectAttributes

RedirectAttributes redirectAttributes = new RedirectAttributes();

addAttribute()

redirectAttributes.addAttribute("message", "Hello World!");

message

redirectAttributes.save("message", message);

Example

@Controller
public class FormController {
