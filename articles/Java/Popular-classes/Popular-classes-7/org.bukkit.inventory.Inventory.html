The org.bukkit.inventory.Inventory Java Class

Java's org.bukkit.inventory.Inventory class is used to manage and store ItemStacks associated with a player. This class provides access to methods such as addItem(), getItem(), removeItem(), and so on. It can also work with a variety of different types, such as Menu and AnvilInventory.

Constructors

The org.bukkit.inventory.Inventory class has three constructors. The first constructor creates an empty inventory with a specific size, type, and title:


          
           
public Inventory(int size, InventoryType type, String title)

The second constructor creates an inventory with a specific size, type, and pre-filled ItemStacks:


          
           
public Inventory(int size, InventoryType type, ItemStack[] contents)

The third constructor creates an inventory with a specific size and type, but no title or ItemStacks:


          
           
public Inventory(int size, InventoryType type)

Methods

The org.bukkit.inventory.Inventory class has a number of useful methods for working with ItemStacks. These include the following:


          
           
addItem(ItemStack... items):  Adds the specified item(s) to this inventory.

          

          
           
getItem(int index):           Retrieves the item at the given index.

          

          
           
removeItem(ItemStack... items): Removes the specified item(s) from this inventory.

          

          
           
clear():                      Clears all items from this inventory.

          

          
           
contains(ItemStack item):     Returns true if this inventory contains the given item.

          

          
           
isEmpty():                    Returns true if this inventory is empty.

          

          
           
getSize():                    Returns the size of this inventory.

          

          
           
getType():                    Returns the type of this inventory.

          

          
           
getTitle():                   Returns the title of this inventory.

          

          
           
setItem(int index, ItemStack item): Sets the given item at the given index in this inventory.

          

          
           
setMaxStackSize(int size):    Sets the maximum number of items that can be stacked in this inventory.

          

          
           
getMaxStackSize():            Gets the maximum number of items that can be stacked in this inventory.

Examples

The following examples demonstrate how to use the org.bukkit.inventory.Inventory class in Java. All code must be escaped accordingly.

Creating a new Inventory with a specific size and type:


          
           
Inventory inv = new Inventory(27, InventoryType.CHEST);

Adding an ItemStack to the Inventory:


          
           
inv.addItem(new ItemStack(Material.DIAMOND));

Retrieving an ItemStack from the Inventory:


          
           
ItemStack item = inv.getItem(0);

Removing an ItemStack from the Inventory:


          
           
inv.removeItem(item);

Clearing the Inventory:


          
           
inv.clear();

Checking if the Inventory contains a specific item:


          
           
inv.contains(new ItemStack(Material.DIAMOND));
