Android Support v4 View ConfigurationCompat

The Android Support Library v4 ViewConfigurationCompat class provides access to the system-level features of the view configuration. This class contains fields and methods that are useful for accessing various view-related settings. It can be used to optimize user experience by detecting any resource constraints that affect the UI, such as screen size or input mode.
