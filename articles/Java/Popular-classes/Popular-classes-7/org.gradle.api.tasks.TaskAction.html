Introduction to org.gradle.api.tasks.TaskAction

org.gradle.api.tasks.TaskAction is an interface that provides action-like methods for all tasks. Any task class implementing this interface will have its own set of task actions. A task action is a publicly visible method in the task class which can be used by external code to access the task's state or use the actual task instance to execute various tasks.

When a task is executed, the task actions are responsible for performing the actual work. This means that in order to define custom behavior for a task, you must implement or override the applicable task actions.

Methods of org.gradle.api.tasks.TaskAction

The interface has these three methods:
  • execute(Object): This method is used to execute the task. It takes one argument – an object of type Object. This argument is used to pass additional information to the task while it is being executed.
  • getName(): This method returns the name of the task action.
  • getDescription(): This method returns the description of the task action.

Examples of Using org.gradle.api.tasks.TaskAction

Let's look at some simple examples of how to use the org.gradle.api.tasks.TaskAction interface.

Example 1: Simple Task

In this example, we'll create a simple Task class that implements TaskAction. The Task class will have one method – execute(), which will just print out a message.


          
           
import org.gradle.api.tasks.TaskAction;

public class Task implements TaskAction {
    public void execute(Object obj) {
        System.out.println("Executing task...");
    }

    public String getName() {
        return "My Task";
    }

    public String getDescription() {
        return "This is a simple task.";
    }
}

Example 2: Counting Task

In this example, we'll create a task that counts the number of elements in an array. The execute() method will accept an Object that contains the array of data.


          
           
import org.gradle.api.tasks.TaskAction;

public class CountTask implements TaskAction {
    public void execute(Object obj) {
        if (obj instanceof int[]) {
            int[] array = (int[]) obj;
            int count = array.length;
            System.out.println("The number of elements in the array is: " + count);
        }
    }

    public String getName() {
        return "Count Task";
    }

    public String getDescription() {
        return "This task counts the number of elements in an array.";
    }
}

Example 3: Summing Task

In this example, we'll create a task that sums up the elements of an array. The execute() method will accept an Object that contains the array of data.


          
           
import org.gradle.api.tasks.TaskAction;

public class SumTask implements TaskAction {
    public void execute(Object obj) {
        if (obj instanceof int[]) {
            int[] array = (int[]) obj;
            int sum = 0;
            for (int i = 0; i < array.length; i++) {
                sum += array[i];
            }
            System.out.println("The sum of the elements in the array is: " + sum);
        }
    }

    public String getName() {
        return "Sum Task";
    }

    public String getDescription() {
        return "This task sums up the elements of an array.";
    }
}
