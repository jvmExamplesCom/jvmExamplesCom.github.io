org.eclipse.swt.widgets.Combo

The org.eclipse.swt.widgets.Combo class is part of the Eclipse Standard Widget Toolkit (SWT). It is a powerful GUI component to easily let users select one option from many possible choices. The Combo widget is particularly useful when there are too many options or when the number of options are very dynamic.

Creating a Combo Widget

A Combo widget can be created in code by instantiating the Combo class. The following code example creates a Combo with its constructor: 

          
           
Combo combo = new Combo(parent, SWT.NONE);
The parent argument is the Composite widget that will contain the Combo, and the second argument corresponds to the style for the Combo. In this case, as no special style is needed, SWT.NONE is used.

Adding Options to the Combo

Options in the Combo can be added using the add() method. Each option has a string label and its associated data. This data can be any Java object. The following example adds three options to a Combo, each with its associated data: 

          
           
String[] labels = new String[] {"One", "Two", "Three"};
Object[] data = new Object[] {new Object(), new Integer(42), new Double(3.141592)};
for (int i = 0; i < labels.length; i++) {
  combo.add(labels[i], data[i]);
}
Alternatively, the add() method can take a single string as argument, in which case the label and data are the same: 

          
           
combo.add("Hello World");

Getting the Currently Selected Option

The currently selected Option in the Combo can be retrieved with getText() and getData(). The getText() method returns a string with the label of the selected Option, while the getData() method retrieves the associated data. The following example prints the value of the currently selected Option in the console: 

          
           
String label = combo.getText();
Object data = combo.getData();
System.out.println("Selected Option: " + label + " (" + data + ")");
It also possible to retrieve only the index of the currently selected Option using the getSelectionIndex() method. This returns an integer representing the zero-based index of the selected item.

Setting the Current Selection

The currently selected Option in the Combo can be changed with the select() method. This method takes an integer as argument, which corresponds to the zero-based index of the desired selection. The following example sets the second option as the current selection: 

          
           
combo.select(1);
It is also possible to select an Option using its label using the setText() method. The following example selects the "Two" option in the Combo: 

          
           
combo.setText("Two");
