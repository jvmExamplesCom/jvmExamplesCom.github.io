org.springframework.data.jpa.repository.JpaSpecificationExecutor

org.springframework.data.jpa.repository.JpaSpecificationExecutor is a Java interface that provides methods for executing JPA queries based on Specifications. This interface extends the Repository interface and provides methods for executing JPA Specifications as Query and for counting entities that match the criteria of a specification. It also provides methods for pagination and sorting of the query results.

Basic Example

In this example, we are going to illustrate how to use the JpaSpecificationExecutor interface in a simple case. We will create a repository interface, define a Specification and then execute the specified query.


          
           
public interface UserRepository extends JpaRepository<User, Long>, JpaSpecificationExecutor<User> { 

}

Specification<User> spec = (root, query, cb) -> cb.equal(root.get("name"), "John");

List<User> users = userRepository.findAll(spec);

The JpaSpecificationExecutor interface also provides methods for pagination and sorting of the query results. To paginate the results, we can use the following method:


          
           
Page<User> page = userRepository.findAll(spec, PageRequest.of(0, 10));

To sort the results we can use the following method:
