org.eclipse.ui.IWorkbench

The org.eclipse.ui.IWorkbench class provides access to the workbench. It provides access to the various parts of the workbench, including views and editors. It also provides access to the window management system, perspective management, and other miscellaneous services. The IWorkbench class itself does not manage any of the windows or perspectives, but instead passes requests on to the window manager.

Code Examples

The following example shows how to access the workbench:


          
           
IWorkbench workbench = PlatformUI.getWorkbench();

The following example shows how to get a reference to the active window:


          
           
IWorkbenchWindow window = workbench.getActiveWorkbenchWindow();

The following example shows how to create a new window:


          
           
IWorkbenchWindow window = workbench.openWorkbenchWindow(configurer);

The following example shows how to open an editor:
