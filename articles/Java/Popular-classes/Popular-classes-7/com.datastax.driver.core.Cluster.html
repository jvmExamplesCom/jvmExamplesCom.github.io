com.datastax.driver.core.Cluster

The com.datastax.driver.core.Cluster class is a Java driver that allows applications to connect and interact with Apache Cassandra™ databases. It provides an intuitive and easy-to-learn interface, allowing developers to quickly write concise and efficient code to interact with Cassandra clusters.

Example Usage

The following example shows how to connect to a Cassandra cluster and execute a simple query.


          
           
Cluster cluster = Cluster.builder()
    .addContactPoint("127.0.0.1")
    .withPort(9042)
    .build();
Session session = cluster.connect();

String cqlStatement = "SELECT * FROM users";
ResultSet resultSet = session.execute(cqlStatement);

for (Row row : resultSet) {

  int age = row.getInt("age");

  System.out.println("Age: " + age);
}

session.close();
cluster.close();
