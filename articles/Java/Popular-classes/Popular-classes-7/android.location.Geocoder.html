Geocoder is a class which enables applications to access geocoding services in the Android platform, which translates between geographic coordinates and human-readable addresses. This can be done either by directly instantiating the Geocoder object or by requesting an instance through a Context. For example:


          
           
Geocoder geocoder = new Geocoder(context);

getFromLocation()
