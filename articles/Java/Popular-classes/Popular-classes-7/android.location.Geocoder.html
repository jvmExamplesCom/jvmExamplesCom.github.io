Geocoder is a class which enables applications to access geocoding services in the Android platform, which translates between geographic coordinates and human-readable addresses. It can serve as an essential tool for applications that require location features to work. This article will explore the Geocoder class in detail, providing code examples.

Geocoder Basics

The Geocoder class enables developers to access the system’s underlying geocoding service. To use this class, you must first obtain an instance of it. This can be done either by directly instantiating the Geocoder object or by requesting an instance through a Context. For example:


          
           
Geocoder geocoder = new Geocoder(context);

Once an instance of the Geocoder class has been created, you can use its various methods to access the service. For example, the 

getFromLocation()
method can be used to retrieve a list of addresses from latitude and longitude coordinates. This code snippet demonstrates how to use this method: 

          
           
List<Address> addresses = geocoder.getFromLocation(latitude, longitude, maxResults);

getFromLocationName()
method is used to obtain address information from a human-readable search query. This code snippet demonstrates how to use this method: 

          
           
List<Address> addresses = geocoder.getFromLocationName(locationName, maxResults);

List

  • The address’s latitude and longitude coordinates.

  • The postal code.

These properties can be accessed through instance methods of the 

Address

Example Usage
