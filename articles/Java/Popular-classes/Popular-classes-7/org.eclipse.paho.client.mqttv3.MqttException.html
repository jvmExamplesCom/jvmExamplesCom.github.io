org.eclipse.paho.client.mqttv3.MqttException

Constructors

The MqttException class has three constructors: 

          
           
MqttException(int reasonCode)
MqttException(int reasonCode, Throwable cause)
MqttException(Throwable cause)

All three constructors throw an MqttException. The first two constructors accept an int parameter—the reasonCode. This integer argument gives more context to the exception and can be one of the following values:


          
           
REASON_CODE_SUCCESS                   = 0;

REASON_CODE_CLIENT_EXCEPTION            = 60007;
REASON_CODE_CLIENT_CONNECTION_LOST      = 60008;
REASON_CODE_CLIENT_EXCEPTION_THROWN     = 60009;

The second constructor additionally takes a Throwable object which may give further details for the exception. The third constructor takes only a Throwable object.
