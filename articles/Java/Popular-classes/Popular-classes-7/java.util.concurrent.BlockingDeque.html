java.util.concurrent.BlockingDeque

Java.util.concurrent.BlockingDeque is a thread-safe class from JDK 5.0 which allows threads to wait for items to be added or removed from a deque. It implements the BlockingQueue interface and provides a deque-based implementation of a blocking queue. This class enables multiple threads to access a single queue in an efficient manner.

The following example shows how to use a BlockingDeque to implement a producer-consumer pattern.

  public static void main(String[] args) {
    // Create a blocking deque with capacity 10
    BlockingDeque<Integer> deque = new LinkedBlockingDeque<>(10);
