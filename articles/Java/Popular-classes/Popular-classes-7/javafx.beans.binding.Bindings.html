javafx.beans.binding.Bindings

javafx.beans.binding.Bindings is a powerful API that provides easy access to observable elements and allows developers to bind properties and values together, allowing for dynamic updates whenever their values change. This API can be used to create binding between properties, monitor the state of properties, and even react to those changes. In this article, we will discuss how to use the API, what the different functions do, and give code examples.

How to Use Bindings

Using Bindings is fairly straightforward. The majority of the Bindings API exists in the 

          
           
javafx.beans.binding
package and the majority of the available methods can be found under the Bindings class. All you need to do is instantiate a new instance of the Bindings object and pass in the properties and values that you want to bind together. You can then use the 

          
           
bind()
or 

          
           
bindBidirectional()
to create a binding between the two. 

          
           
//Instantiate the Bindings object
Bindings bindObj = new Bindings();

//Bind two properties together
int x = 5;
int y = 6;
bindObj.bind(x, y);

//Watch the binding status
boolean isBound = bindObj.isBound();
System.out.println(isBound); //True

Binding Functions

The primary function of the Bindings API is to create bindings between properties. This is done using the 

          
           
bind()
and the 

          
           
bindBidirectional()
, which both accept two properties as parameters and generates a binding between them. The difference between the two functions is that with the 

          
           
bind()
function, any changes made to the source property will be reflected in the target property, whereas with 

          
           
bindBidirectional()
any changes made to either property will be reflected in the other. 

          
           
//Create new Bindings object
Bindings bindObj = new Bindings();

int x = 5;
int y = 6;

//Only changes to x will be reflected in y:
bindObj.bind(x, y);

//Changes to either x or y will be reflected in the other
bindObj.bindBidirectional(x, y);
The 

          
           
bind()
and 

          
           
bindBidirectional()
methods are also capable of accepting a third parameter, which is an expression that can be used to manipulate the value of a property before it is set on the target property. This is useful for performing calculations or conversions on the source property before it is set on the target property. 

          
           
//Create new Bindings object
Bindings bindObj = new Bindings();

int x = 5;
int y = 6;

//The third parameter is an expression that is applied to x before setting the value on y
bindObj.bind(x, y, x -> x * 10);

//x is now equal to 50 and y is equal to 500
System.out.println(x); //50
System.out.println(y); //500
The Bindings API also provides several other functions for monitoring the state of the bindings. The 

          
           
isBound()
method can be used to check the status of a binding, the 

          
           
unbind()
method can be used to remove a binding, and the 

          
           
removeListener()
method can be used to remove a listener that was previously added to a binding. 

          
           
//Create new Bindings object
Bindings bindObj = new Bindings();

int x = 5;
int y = 6;

//Bind two properties together
bindObj.bind(x, y);

//Check the state of the binding
boolean isBound = bindObj.isBound();
System.out.println(isBound); //True

//Unbind the two properties
bindObj.unbind();

//Check the state of the binding
isBound = bindObj.isBound();
System.out.println(isBound); //False
Finally, the Bindings API also supports adding listeners to bindings, allowing developers to react to changes in the properties that are bound together. This is done by passing a ChangeListener instance to the 

          
           
addListener()
method. This listener will be invoked whenever the value of a property changes and will provide the previous and current values of the property. 

          
           
//Create new Bindings object
Bindings bindObj = new Bindings();

int x = 5;
int y = 6;

//Bind two properties together
bindObj.bind(x, y);

//Add a listener to the binding
bindObj.addListener((observable, oldValue, newValue) -> {
    //Print the old and new values
    System.out.println("Old Value: " + oldValue);
    System.out.println("New Value: " + newValue);
});

//Change the value of x
x = 10;

Prints: 
Old Value: 5
New Value: 10
