org.apache.commons.lang.math.NumberUtils

The org.apache.commons.lang.math.NumberUtils package is a set of utility methods for working with numbers in Java.

Usage

This package is designed to provide a number of convenience methods for working with numbers, such as converting strings to numbers or comparing two different numbers.

Example

The following example shows how to use the NumberUtils class to convert a string to an integer:


          
           
String input = "123";
int output = NumberUtils.toInt(input);
The output of this code would be the integer 123.

The following example shows how to use the NumberUtils class to compare two doubles:


          
           
double firstNumber = 0.5;
double secondNumber = 0.8;
boolean isLessThan = NumberUtils.compare(firstNumber, secondNumber) < 0;
In this case, 
isLessThan
would be set to 
true

Limitations

The NumberUtils package does not support operations like addition and subtraction. For these operations, the Java Math package will need to be used.
