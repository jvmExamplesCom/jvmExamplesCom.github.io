org.apache.commons.lang3.time.StopWatch

The org.apache.commons.lang3.time.StopWatch is a utility class in the Apache Commons Lang library that is used to measure elapsed time. It is designed to be as lightweight and simple to use as possible, while providing accurate measurements of elapsed time. The StopWatch class can measure elapsed time up to one-millisecond accuracy and it supports start/stop, pause/resume, and resetting of the timer. The StopWatch class has several useful methods for measuring elapsed time, including getStartTime(), getStopTime(), getElapsedTime(), and reset(). The getStartTime() method returns the start time of the StopWatch in milliseconds. The getStopTime() method returns the stop time of the StopWatch in milliseconds. The getElapsedTime() method returns the total elapsed time in milliseconds. The reset() method resets the StopWatch and sets all the accumulated times back to zero.
