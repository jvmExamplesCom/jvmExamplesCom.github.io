org.apache.catalina.Context

The org.apache.catalina.Context class is a fundamental component of the Java Servlet Container, Tomcat. It is responsible for providing the web application with an environment in which to run. Context context = new StandardContext();

context.addServlet("myservlet", MyServlet.class);

FilterDef filterDef = new FilterDef();

context.addFilterDef(filterDef);

context.addApplicationListener(MyListener.class);
