Java and Animation with Android.AnimatorInflater

Using Android.AnimatorInflater, developers can easily apply and reuse existing animation sets, as well as create new ones from scratch. Getting Started with Android.AnimatorInflater

To use the library, developers must first include the Android.AnimatorInflater dependency in their project. This can be done via Maven or Gradle. 

          
           
compile 'com.android:android.animation:2.2.0'
Once this is done, animations can be defined in the xml layout file. For example, the following animation set defines a simple translation animation that moves the view by 100dp in the x direction over the course of 4 seconds. 

          
           
<animator
    android:name="translate"
    android:duration="4000"

    android:interpolator="@android:anim/accelerate_decelerate_interpolator"

The AnimatorInflater class can then be used to load the animation set from the xml layout file. 

          
           
AnimatorInflater.loadAnimator(context, R.layout.example_animation)

Animator animator = AnimatorInflater.loadAnimator(context, R.layout.example_animation);
