javax.websocket.OnMessage

The javax.websocket.OnMessage annotation is a type-level annotation which allows developers to define methods to be called when an inbound message is received from the other endpoint of a WebSocket connection. The annotation enables users to map any message types that are supported by the Java API for WebSocket (JSR 356) to an onMessage method and also enables to intercept incoming messages for the endpoints.

Syntax


          
           
@OnMessage
public void onMessage(InputStream is, boolean last)

Parameters

  • InputStream is - This is the input stream of the message.
  • boolean last - This parameter determines whether this is the last fragment of the message coming in.

Examples

This example shows how to use the javax.websocket.OnMessage annotation for an incoming message. In this example, the incoming message is a text message.


          
           
@OnMessage
public void onTextMessage(String message,Session session) { 
    System.out.println("Received message: "+message);
}

This example shows how to use the javax.websocket.OnMessage annotation for an incoming message. In this example, the incoming message is a binary message.


          
           
@OnMessage
public void onBinaryMessage(ByteBuffer message, Session session) {
    while (message.hasRemaining()) {
        System.out.println("Received message: "+message.get());
    }
}

The next example shows how to use the javax.websocket.OnMessage annotation for an incoming message. In this example, the incoming message is a streaming message. The streaming message is read from the input stream.


          
           
@OnMessage
public void onStreamMessage(InputStream is,Session session) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(is));
        String line = null;
        StringBuilder sb = new StringBuilder();

        //Read bytes from the input stream and construct a String
        while ((line = br.readLine()) != null) {
            sb.append(line);
        }
        System.out.println("Received message: "+sb.toString());
    }
}
