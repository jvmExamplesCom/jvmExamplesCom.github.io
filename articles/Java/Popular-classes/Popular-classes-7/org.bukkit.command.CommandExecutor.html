CommandExecutor

Using CommandExecutor

To use 
CommandExecutor

onCommand()

CommandSender
object and the second is the 
String
command that was sent. The 
onCommand()
method must return a boolean which indicates whether or not the command was successfully handled. 

          
           
public class MyCommandExecutor implements CommandExecutor {
    public boolean onCommand(CommandSender sender, String cmd, String[] args) {
        // Handle command here
        return true; // Return true if command was handled successfully
    }
}
