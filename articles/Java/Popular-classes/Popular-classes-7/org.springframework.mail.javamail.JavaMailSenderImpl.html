org.springframework.mail.javamail.JavaMailSenderImpl

org.springframework.mail.javamail.JavaMailSenderImpl is a JavaMail interface implementation that is used to send emails via an SMTP server. It provides all the basic operations needed to send emails, including simple MIME messages as well as attachments.

Creating a JavaMailSenderImpl Instance

To create an instance of JavaMailSenderImpl we will need to pass in an implementation of the JavaMailSender interface, which should be provided by the framework you are using (e.g. Spring). For example, in Spring we can use the SimpleMailMessage class:


          
           
SimpleMailMessage message = new SimpleMailMessage();
message.setFrom("example@example.com");
message.setTo("recipient@example.com");
message.setSubject("Hello world!");
message.setText("This is a test message.");

JavaMailSenderImpl sender = new JavaMailSenderImpl(message);

We now have an instance of JavaMailSenderImpl that we can use to send our emails.

Sending a Simple Message

send
method: 

          
           
sender.send(message);

Adding Attachments

If we want to add attachments to our messages, we can use the 

addAttachment

// Create an attachment

addCc

sender.addBcc(message, "bcc@example.com");

sender.send(message);

Setting Timeouts

The JavaMailSenderImpl also allows us to set timeouts for the underlying SMTP connection. This can be done by setting the 

timeout
field before sending the message:

In this example we've used a timeout of 5 seconds, but this can be adjusted to whatever value makes sense for your application.
