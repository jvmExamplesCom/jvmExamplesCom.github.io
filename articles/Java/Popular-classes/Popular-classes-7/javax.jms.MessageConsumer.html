javax.jms.MessageConsumer

Description

The javax.jms.MessageConsumer interface is used to create objects for receiving messages from a queue or topic. It provides an object for consuming messages from a JMS provider.

Methods

MessageConsumer provides following methods:
  • public void close() throws JMSException: Closes the MessageConsumer.
  • public MessageListener getMessageListener(): Gets the message listener for this MessageConsumer.
  • public String getMessageSelector(): Gets this MessageConsumer's message selector expression.
  • public Message receive() throws JMSException: Receives the next message that arrives within the specified timeout interval.
  • public Message receive(long timeout) throws JMSException: Receives the next message that arrives within the specified timeout interval.
  • public Message receiveNoWait(): Receives the next message if one is immediately available.
  • public void setMessageListener(MessageListener listener) throws JMSException: Sets the message listener for this MessageConsumer.

Example


          
           
QueueConnectionFactory qcf = new ActiveMQConnectionFactory("tcp://localhost:61616");
QueueConnection qc = qcf.createQueueConnection();
QueueSession qs = qc.createQueueSession(false, Session.AUTO_ACKNOWLEDGE);
Queue q = qs.createQueue("TestQueue");
MessageConsumer mc = qs.createConsumer(q);
qc.start();

Message msg = mc.receive();
System.out.println("Received message: " + msg);

mc.close();
qc.close();
