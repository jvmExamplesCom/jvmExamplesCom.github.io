org.bukkit.entity.EntityType

The org.bukkit.entity.EntityType class represents the kinds of entities that can be spawned in a Minecraft world. It is an enum that contains all the possible entity types that a server can spawn via a plugin. This includes monsters, NPCs, animals, and various other creatures. The EntityType class provides an easy way to create new entities, as well as modify existing ones. In this article, we will explore the different methods and properties available in the EntityType class.

Methods

  • create() – Create a new entity based on the given type.
  • setName(String name) – Sets the entity's display name.
  • getName() – Returns the entity's display name.
  • setHealth(Integer health) – Sets the entity's health.
  • getHealth() – Returns the entity's current health.

  • health – The current health of the entity.
  • maxHealth – The maximum health of the entity.
