org.bukkit.entity.EntityType

The org.bukkit.entity.EntityType class represents the kinds of entities that can be spawned in a Minecraft world. It is an enum that contains all the possible entity types that a server can spawn via a plugin. This includes monsters, NPCs, animals, and various other creatures. The EntityType class provides an easy way to create new entities, as well as modify existing ones.   • setHealth(Integer health) – Sets the entity's health.
  • getHealth() – Returns the entity's current health.
