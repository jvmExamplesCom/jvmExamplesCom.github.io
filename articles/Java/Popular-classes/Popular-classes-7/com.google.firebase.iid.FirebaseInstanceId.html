FirebaseInstanceId

The FirebaseInstanceId class gives access to the registration token of Instance ID, an instance of a Firebase project for a mobile app. From the Instance ID token it is possible to manage the app instance created and access data associated with it, such as the IID Token and notifications.

What is FirebaseInstanceId?

FirebaseInstanceId is an implementation part of the Google Firebase Cloud Messaging system. It provides a unique token for devices that register with a Firebase project. With this token, devices can receive messages from the server, either from a Firebase project or from third-party applications. This makes it possible to send push notifications to users on their device.

How to Use FirebaseInstanceId

To use Firebase InstanceId, you must first create a Firebase project. Once you have done so, you will be able to obtain a token for each device that registers with the project. To do this, you must use the FirebaseInstanceId.getInstance().getToken() method. The token obtained from this method is then used to register the device with the Firebase project. Once registered, the device will be able to receive notifications sent to it by the server or by other applications.
