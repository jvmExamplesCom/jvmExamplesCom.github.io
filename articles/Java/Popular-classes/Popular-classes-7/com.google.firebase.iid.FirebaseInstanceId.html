What is FirebaseInstanceId?

FirebaseInstanceId is an implementation part of the Google Firebase Cloud Messaging system. It provides a unique token for devices that register with a Firebase project. With this token, devices can receive messages from the server, either from a Firebase project or from third-party applications. This makes it possible to send push notifications to users on their device.
