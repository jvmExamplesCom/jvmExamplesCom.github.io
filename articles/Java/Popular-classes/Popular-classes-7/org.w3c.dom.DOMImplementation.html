Introduction to org.w3c.dom.DOMImplementation

The 

          
           
org.w3c.dom.DOMImplementation

It is essentially an abstract factory for producing DOM documents from their constituent objects, such as elements, attributes, and text nodes. The DOMImplementation interface is part of the W3C Document Object Model (DOM), and provides a way to define "abstract" objects that are able to interact with concrete DOM implementations.
