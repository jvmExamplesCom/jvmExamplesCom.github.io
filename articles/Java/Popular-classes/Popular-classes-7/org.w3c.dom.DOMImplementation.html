Introduction to org.w3c.dom.DOMImplementation

The 

          
           
org.w3c.dom.DOMImplementation
interface describes a set of methods which can be used to create new DOM trees. It is essentially an abstract factory for producing DOM documents from their constituent objects, such as elements, attributes, and text nodes. The DOMImplementation interface is part of the W3C Document Object Model (DOM), and provides a way to define "abstract" objects that are able to interact with concrete DOM implementations.

Creating a DOMImplementation Object

To create a DOMImplementation object, use the static method 

          
           
createDOMImplementation()
provided by the class 

          
           
org.w3c.dom.DOMImplementationRegistry
. 

          
           
// Create a DOMImplementation object
  DOMImplementation domImpl = DOMImplementationRegistry.createDOMImplementation();
