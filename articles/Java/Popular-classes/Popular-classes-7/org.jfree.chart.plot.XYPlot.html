Introduction to org.jfree.chart.plot.XYPlot

org.jfree.chart.plot.XYPlot is a Plot class of the JFreeChart library which is used to draw XY plots like scatter, line, area and others. It is mainly used for graphs which provides data in form of two dimensional arrays.

Usage of the class

To use the org.jfree.chart.plot.XYPlot class, we first need to create an instance of it. This can be done using its constructor. Following is an example of creating a new x-y plot.


          
           
XYPlot plot = new XYPlot();

Once we have created the plot, we can add various elements to it like dataset, renderer and annotations. We will discuss about them in detail later.

Methods of the class

The org.jfree.chart.plot.XYPlot class provides some useful methods which can be used to configure the plot. Following is a list of some of the commonly used methods:

  • getDataset(): Used to return the dataset associated with the plot.
  • setRenderer(XYItemRenderer renderer): Used to set the renderer for the plot.
  • addAnnotation(XYAnnotation annotation): Used to add an annotation to the plot.
  • setRangeGridlinesVisible(boolean flag): Used to set the visibility of the range gridlines for the plot.

Example

Let us now look at an example of how to use the org.jfree.chart.plot.XYPlot class to create a simple scatter plot. We will first create a dataset consisting of two XYSeries objects:


          
           
XYSeries series1 = new XYSeries("Series 1");
series1.add(1, 2);
series1.add(2, 4);
series1.add(3, 6);

XYSeries series2 = new XYSeries("Series 2");
series2.add(2, 3);
series2.add(4, 5);
series2.add(6, 7);

XYSeriesCollection dataset = new XYSeriesCollection();
dataset.addSeries(series1);
dataset.addSeries(series2);

We will now create an instance of the XYPlot class and set the dataset on it:


          
           
XYPlot plot = new XYPlot();
plot.setDataset(dataset);

We will now add a renderer to create a scatter plot. For this we will use the XYLineAndShapeRenderer class:


          
           
XYLineAndShapeRenderer renderer = new XYLineAndShapeRenderer();
renderer.setSeriesShapesVisible(0, true);
renderer.setSeriesShapesVisible(1, true);
plot.setRenderer(renderer);

We can now visualize the plot by adding it to a chart and displaying it on the screen:


          
           
JFreeChart chart = new JFreeChart(plot);
ChartFrame frame = new ChartFrame("Scatter Plot", chart);
frame.pack();
frame.setVisible(true);
