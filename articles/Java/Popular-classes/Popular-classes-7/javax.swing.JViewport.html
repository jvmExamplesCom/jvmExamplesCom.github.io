javax.swing.JViewport

The javax.swing.JViewport class provides a scrollable view of a portion of an underlying component. It can be used to implement the scrolling capabilities in applications that need to manage the display of a larger component in a smaller space. The viewport is a rectangular area with the visible part of the underlying component drawn inside it. It can be used for example to view a large image within a smaller window.

Example 1: Creating a JViewport

In this example, we create a JViewport and its associated ViewportUI class. First, we create the JViewPort object, using the constructor specifying the viewport's desired size:


          
           
JViewport viewport = new JViewport(50, 50);

The ViewportUI class is then created, using the constructor specifying the desired viewport to use with the user interface:


          
           
ViewportUI ui = new ViewportUI(viewport);

Once the ViewportUI class has been created, the viewport can be added to a JScrollPane, to provide scrolling capabilities for the underlying component within the viewport:


          
           
JScrollPane scrollPane = new JScrollPane(viewport);

Example 2: Setting view position

The JViewport class provides methods for setting the current view position of the underlying component within the viewport. In this example, we set the view position to move the scroll bar of the underlying component to the right by 10 pixels:


          
           
viewport.setViewPosition(new Point(10,0));

Example 3: Adding a Component to the Viewport

Finally, the JViewport class provides methods for adding a component to the viewport. In this example, we add a JTextArea to the viewport:
