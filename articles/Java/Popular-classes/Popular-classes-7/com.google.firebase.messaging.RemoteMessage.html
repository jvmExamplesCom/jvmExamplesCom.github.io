Introduction to com.google.firebase.messaging.RemoteMessage

Com.google.firebase.messaging.RemoteMessage is a class in the Firebase Messaging SDK that allows developers to send messages to their apps on Android, iOS, and the web. This class provides methods for setting various message parameters such as title, body, data payload, and notification settings.

RemoteMessage objects can be sent using Firebase Messaging or through HTTP connections with the Firebase Cloud Messaging (FCM) web API. It is not recommended to construct RemoteMessage objects manually, but instead use the FirebaseMessaging#send() method available in the Firebase Messaging SDK.

Creating a RemoteMessage Object

The RemoteMessage object is created by calling the constructor. This requires two arguments: the sender id and the message id.


          
           
RemoteMessage remoteMessage = new RemoteMessage(senderId, messageId);

The sender id identifies the server that is sending the message and the message id should be a unique identifier for the message.

Sending a Message

Once the RemoteMessage object has been created, it can be sent using the FirebaseMessaging#send() method. This requires three arguments: the message, the target, and the message options.


          
           
RemoteMessage message = ...;
Target target = ...;
MessageOptions options = ...;

FirebaseMessaging.getInstance().send(message, target, options);

The message is the RemoteMessage object, the target indicates where to send the message, and the message options provide additional settings.

Setting Message Options

The MessageOptions class is used to set additional options for the message. These options include setting the priority, for example whether the message should be delivered immediately or with a certain delay. Other options include setting the time to live, which specifies how long the message should be kept in the server if the device is offline, and setting the collapse key to group related messages together.

Setting Notification Settings

The notification settings allow you to specify whether the message should be displayed as a notification on the device, or if it should only be delivered silently in the background. The setNotificationSettings() method takes a NotificationSettings object which can be used to specify these settings. For example, to set the message as a high priority notification:

    new NotificationSettings.Builder()
