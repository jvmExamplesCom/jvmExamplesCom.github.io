Introduction to com.google.android.gms.maps.model.Marker

com.google.android.gms.maps.model.Marker is a class within the Android Google Maps API that allows developers to easily add markers to a map object. Markers are essentially objects that are used on a Google Map to represent a point of interest (POI). They can be used to show an area of interest, or indicate a certain point on the map, such as a road junction, business location, or user-defined location. Developers can add multiple markers to a single map, and customize the appearance and behavior of each marker. This article provides an overview of the functionality provided by com.google.android.gms.maps.model.Marker and provides code examples to illustrate the various ways in which markers can be used.

Creating a Marker

A marker is created with the 

          
           
newMarker()
method. This method takes a 

          
           
LatLng
object, which represents a precise latitude and longitude, as a parameter. This method also takes an optional 

          
           
MarkerOptions
object, which can be used to customize the appearance and behavior of the marker. 

          
           
Marker m = googleMap.addMarker(new MarkerOptions().position(latLng));
The 

          
           
LatLng
object can be created from a pair of double values representing latitude and longitude. 

          
           
double lat = 25.003;
double lng = 45.789;
LatLng latLng = new Latlng(lat, lng);
The optional 

          
           
MarkerOptions
object can be used to customize the appearance and behavior of the marker. The following example creates an options object with a particular title and color. 

          
           
MarkerOptions options = new MarkerOptions()
    .title("My Marker")
    .icon(BitmapDescriptorFactory.defaultMarker(BitmapDescriptorFactory.HUE_GREEN));
This options object can then be passed to the 

          
           
addMarker()
method along with the 

          
           
LatLng
object. 

          
           
Marker m = googleMap.addMarker(options.position(latLng));

Manipulating Markers

Once a marker has been added to a map object, it can be manipulated in various ways. For example, the 

          
           
setPosition()
method can be used to change the position of a marker. 

          
           
m.setPosition(new LatLng(25.123, 45.890));
Similarly, the 

          
           
setTitle()
method can be used to change the title of a marker. 

          
           
m.setTitle("My Updated Marker");
The 

          
           
remove()
method can be used to remove a marker from a map. 

          
           
m.remove();

Conclusion

The com.google.android.gms.maps.model.Marker class provides an easy way for developers to add markers to a map object. Markers can be customized with the MarkerOptions object, and manipulated with various methods. In this article, we have seen how to create and manipulate markers using the Google Maps Android API.
