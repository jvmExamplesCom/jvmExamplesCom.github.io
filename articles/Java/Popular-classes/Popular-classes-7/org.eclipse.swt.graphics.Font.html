org.eclipse.swt.graphics.Font

Creating a Font Object

To create a Font object, you need to call the constructor and provide a FontData instance as the argument. The FontData class is explained in more detail below:


          
           
FontData fontData = new FontData("arial", 10, SWT.BOLD);
Font font = new Font(Display.getCurrent(), fontData);

The first argument for the FontData constructor is a string representing the font family name. The second argument is an integer specifying the font size, and the third argument is an SWT constant that determines if the font should be drawn in bold or italic.

FontData Class
