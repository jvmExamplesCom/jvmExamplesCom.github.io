org.springframework.amqp.core.Queue

The org.springframework.amqp.core.Queue is an interface in the Spring AMQP project that represents a queue in the messaging system. It provides methods to get and set various configuration properties of the queue. This includes name, durability, exclusive, auto-delete, and others.

Creating a Queue

A Queue is created with the QueueBuilder which has several default configuration settings. To create a queue with all default settings:


          
           
Queue queue = QueueBuilder.durable("my-queue").build();

The QueueBuilder has many methods for setting the configuration properties of the queue. For example, to create a queue with specific durability and auto-delete settings:


          
           
Queue queue = QueueBuilder.durable("my-queue")
    .withDurability(true)
    .withAutoDelete(true)
    .build();

Getting Configuration Properties

The Queue interface provides methods to get the configuration properties of the queue. For example, to get the name of the queue:


          
           
String queueName = queue.getName();

Setting Configuration Properties

The Queue interface also provides methods to set the configuration properties of the queue. For example, to set the exclusive flag of the queue:


          
           
queue.setExclusive(true);

Deleting a Queue

A queue can be deleted from the messaging system by calling the delete() method on the Queue:


          
           
queue.delete();
