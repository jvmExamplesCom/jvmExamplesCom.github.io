com.intellij.openapi.ui.Messages

com.intellij.openapi.ui.Messages is an IntelliJ code library that provides a platform-independent and customizable way of creating message dialogs with options for the user to select from.

Creating Message Dialogs

Creating message dialogs with com.intellij.openapi.ui.Messages is easy. The primary method to use is showMessageDialog(Component parentComponent, Object message, String title, int messageType). All arguments are required.

The first argument specifies the parent component for this dialog. The second argument is the message that will be displayed. The third argument is the title for the dialog. And finally, the fourth argument specifies the type of dialog message, e.g., an information or warning message.

Example: 

          
           
Messages.showMessageDialog(null, "This is a message", "Message Title", Messages.getInformationIcon());

Example: 

          
           
Messages.getInformationIcon()

Creating Confirmation Dialogs
