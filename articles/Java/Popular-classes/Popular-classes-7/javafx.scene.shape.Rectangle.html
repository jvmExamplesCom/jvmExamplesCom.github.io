JavaFX Scene Shape Rectangle

JavaFX Scene Shape Rectangle is a basic shape class that can be used to create rectangles. It inherits from Shape class, and provides a constructor taking two parameters (x and y coordinates) specifying the top left corner of the rectangle, in addition to width and height.

Creating a Rectangle

The following code will create a simple rectangle at position (100, 100) with a size of 300 x 150:


          
           
Rectangle rect = new Rectangle(100, 100, 300, 150);

Setting Color and Opacity

To set the color of a Rectangle , use the setFill method. The following code will create a red rectangle:


          
           
rect.setFill(Color.RED);

To set the opacity (transparency) of a Rectangle, use the setOpacity method. The following code will create a rectangle with 50% opacity:


          
           
rect.setOpacity(0.5);

Setting Stroke

To set the stroke (outline) of a Rectangle , use the setStroke method. The following code will create a black stroke with a width of 10:


          
           
rect.setStroke(Color.BLACK);
rect.setStrokeWidth(10);

Adding to Scene

To add a Rectangle to a scene, use the add method of the Pane class. The following code will add a rectangle to the scene:


          
           
Pane pane = new Pane();
pane.getChildren().add(rect);
