javafx.scene.input.MouseButton

Overview

The javafx.scene.input.MouseButton class allows developers to define which mouse button was pressed. It contains constants that represent the different mouse buttons: PRIMARY, MIDDLE, SECONDARY and NONE.

Constructor

The MouseButton class does not have a constructor.

Methods

The MouseButton class has three methods: 
  • isPrimary()
    : returns true if the button is PRIMARY.
  • isMiddle()
    : returns true if the button is MIDDLE.
  • isSecondary()
    : returns true if the button is SECONDARY.

Example


          
           
import javafx.scene.input.MouseButton;

public class MyClass {

    public static void main(String[] args) {

        MouseButton button = MouseButton.PRIMARY;

        if (button.isPrimary()) {
            System.out.println("Button is primary");
        } else if (button.isMiddle()) {
            System.out.println("Button is middle");
        } else if (button.isSecondary()) {
            System.out.println("Button is secondary");
        } else {
            System.out.println("Unknown button");
        }
    }
}
In this example, we created a MouseButton instance and then checked which button it is using the isPrimary(), isMiddle() and isSecondary() methods.
