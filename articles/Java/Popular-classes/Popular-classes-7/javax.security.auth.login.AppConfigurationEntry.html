javax.security.auth.login.AppConfigurationEntry

The javax.security.auth.login.AppConfigurationEntry class represents a component of a LoginModule configuration. It contains the name of a LoginModule class and a set of options configured for that LoginModule.

The AppConfigurationEntry class is used to store information about configured LoginModules used by an application. It provides for the definition of a module-specific set of options, each of which is identified by a key and stores a value. The set of options for each LoginModule may vary depending on the module's implementation.

Let's consider a simple example in which we configure a LoginModule called “MyLoginModule” which requires two options: “opt1” and “opt2”.


          
           
AppConfigurationEntry entry = new AppConfigurationEntry(
  "MyLoginModule",
  AppConfigurationEntry.LoginModuleControlFlag.REQUIRED,
  new HashMap<String, String>() {
    {
      put("opt1", "value1"); 
      put("opt2", "value2"); 
    }
  }
);

The first parameter of the constructor is the name of the LoginModule which in this case is “MyLoginModule”.

The second parameter is the control flag which specifies how the LoginModule should be treated within the overall authentication process. The available flags are REQUIRED, REQUISITE, SUFFICIENT, and OPTIONAL.

The third parameter is a map of options which is specific to the LoginModule being configured. In this example, two options “opt1” and “opt2” are configured with the corresponding values “value1” and “value2”.

Usage Example

Let's consider a simple example in which we create a Configuration object using a single AppConfigurationEntry. 


          
           
AppConfigurationEntry entry = new AppConfigurationEntry(
  "MyLoginModule",
  AppConfigurationEntry.LoginModuleControlFlag.REQUIRED,
  new HashMap<String, String>() {
   {
     put("opt1", "value1"); 
     put("opt2", "value2"); 
   }
  }
);

Configuration config = new Configuration() {
  public AppConfigurationEntry[] getAppConfigurationEntry(String name) {
    return new AppConfigurationEntry[]{entry};
  }
};

The Configuration object can then be used to obtain a LoginContext for authentication:


          
           
LoginContext loginContext = new LoginContext(
  "MyLoginContext",
  null,
  null,
  config
);

loginContext.login();

In this example, the LoginContext will use the MyLoginModule configured with the two options “opt1” and “opt2”.
