javax.swing.border.CompoundBorder

The javax.swing.border.CompoundBorder class enables developers to combine multiple borders into a single combined border. This is useful when wanting to add a combination of features to a given border, such as adding a beveled border around a line border. The class can also be used to create overlapping borders.

Creating a CompoundBorder

A CompoundBorder is created using the CompoundBorder(Border outer, Border inner) constructor, which takes two Border objects as arguments representing the outer and inner borders respectively. The following example creates a CompoundBorder with a beveled border as the outer border and a line border as the inner border.


          
           
CompoundBorder border = new CompoundBorder(
    new BevelBorder(BevelBorder.RAISED), 
    new LineBorder(Color.BLACK));

Once created, the CompoundBorder should be set on a Component by calling the 

setBorder()
method: 

          
           
component.setBorder(border);

Methods

The CompoundBorder class contains a 
getInsideBorder()
and 
getOutsideBorder()
methods which return the inner and outer borders respectively. The following example displays the classes of both the outside and inside borders: 

          
           
System.out.println("Outer border: "+border.getOutsideBorder().getClass().getName());
System.out.println("Inner border: "+border.getInsideBorder().getClass().getName());
which produces the following output: 

          
           
Outer border: javax.swing.border.BevelBorder
Inner border: javax.swing.border.LineBorder
