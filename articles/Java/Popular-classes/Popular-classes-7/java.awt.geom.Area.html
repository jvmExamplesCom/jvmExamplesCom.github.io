Introduction to Java.awt.geom.Area

Java.awt.geom.Area is a class in the java.awt.geom package, which provides a means of representing an area of 2-dimensional geometry. This class enables construction and manipulation of individual area objects and allows for operations on those areas such as union, intersection and exclusive or. In this article, we will explore the fundamentals of this class and provide code examples demonstrating how to use it.

Creating a New Area Object

A new Area object can be created using the constructor provided by the java.awt.geom.Area class. The constructor takes an instance of one of the two Shape subclasses Rectangle2D or Path2D as an argument. Here is an example of how to do this:


          
           
Rectangle2D rect = new Rectangle2D.Double(10, 10, 40, 40);
 Area area = new Area(rect);

This creates a new Area object with a rectangular shape defined by the provided rectangle object.

Union Operations

An existing Area object can be combined with another Shape object using the add() method, like so:


          
           
Rectangle2D rect2 = new Rectangle2D.Double(50, 50, 40, 40);
area.add(new Area(rect2));

This adds the newly created Area object, which consists of the second Rectangle2D instance, to the existing Area object. As a result of this operation, the Area object now contains both rectangles.

Intersection Operations

Similarly, an existing Area object can be combined with another Shape object using the intersect() method. For example:


          
           
Rectangle2D rect3 = new Rectangle2D.Double(25, 25, 20, 20);
area.intersect(new Area(rect3));

This combines the existing Area object with the newly created Area object, which consists of the third Rectangle2D instance, by taking the intersection of the two areas. As a result of this operation, the Area object now contains only the portion of the original rectangles that intersect the third one.

Exclusive Or Operations

Finally, an existing Area object can be combined with another Shape object using the exclusiveOr() method. For example:


          
           
Rectangle2D rect4 = new Rectangle2D.Double(35, 35, 20, 20);
area.exclusiveOr(new Area(rect4));
