Java: android.view.FocusFinder

The android.view.FocusFinder class provides APIs for finding views inside of a view hierarchy that can be focused. It offers the ability to focus on a given view, defining boundaries where focus can move, and more. This class is mainly used by implementations of the ViewGroup class and its subclasses such as LinearLayout, FrameLayout, and RelativeLayout.

Methods

The android.view.FocusFinder class contains several methods that allow developers to manage the focus flow inside a ViewGroup. These methods are listed below.

findNextFocus()

The findNextFocus() method takes in a ViewGroup and a View and returns the next view in the ViewGroup which can receive focus or null if no such view exists. 

          
           
View nextView = FocusFinder.findNextFocus(viewGroup, view);

findNextFocusFromRect()

The findNextFocusFromRect() method takes a Rect defining the area from which to start searching for the next view that can receive focus, a direction, and an array of Views representing the focusables in the specified rect. The method returns the next view that can receive focus or null if no such view exists. 

          
           
View nextView = FocusFinder.findNextFocusFromRect(rect, direction, focusables);

The getMaxScrollAmount() method takes a View and returns the maximum amount that the view can be scrolled in either the x- or y-direction.
