Example Usage of javax.crypto.CipherOutputStream

The following code shows an example usage of javax.crypto.CipherOutputStream for encrypting a text file. 

          
           
try {
    // Create a Cipher object
    Cipher cipher = Cipher.getInstance("AES/CBC/PKCS5Padding");

    // Create CipherInputStream and CipherOutputStream objects
    CipherOutputStream cos = new CipherOutputStream(new FileOutputStream("encrypted_file.txt"), cipher);
    CipherInputStream cis = new CipherInputStream(new FileInputStream("input_file.txt"), cipher);

    // Read input file
    byte[] buffer = new byte[1024];
    int nread;
    while((nread = cis.read(buffer)) != -1) {
        cos.write(buffer, 0, nread);
    }

    // Close streams
    cis.close();
    cos.close();

} catch (Exception e) {
    System.out.println("Error: " + e.getMessage());
}
In the above example, a Cipher object is created using the AES/CBC/PKCS5Padding encryption algorithm. The CipherInputStream and CipherOutputStream objects are then created. Finally, both streams are closed, releasing all resources associated with them.
