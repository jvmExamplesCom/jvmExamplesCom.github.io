org.eclipse.jetty.util.thread.QueuedThreadPool

The org.eclipse.jetty.util.thread.QueuedThreadPool is an implementation of the ThreadPool interface that utilizes a queue for request handling. It is part of the Jetty open source server project. Basic Usage

A QueuedThreadPool can be created with a simple constructor call that takes the maximum pool size and thread timeout parameters.


          
           
QueuedThreadPool qtp = new QueuedThreadPool(5, 2);

This will create a thread pool with a maximum of 5 threads and a timeout of 2 seconds. After this point the thread pool will start to automatically manage the threads to accommodate incoming requests.
