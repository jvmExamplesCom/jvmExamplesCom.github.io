Java Reflection FieldUtils Intro

Java Reflection FieldUtils is part of the org.apache.commons.lang3 library and it's mainly used to get fields of a class, including those of superclasses.

Getting All Fields

The easiest way to get all of the fields from a class is to call the 

getAllFields()
method and pass in the desired type. This will include all fields from the given class, as well as any declared fields from parent classes. 

          
           
Field[] fields = FieldUtils.getAllFields(MyClass.class);

Getting Accessible Fields

If you need to filter out any private, protected or package-private fields, you can use the 


          
           
getAccessibleFields()
method: 

          
           
Field[] fields = FieldUtils.getAccessibleFields(MyClass.class);

Getting Declared Field

If you need to retrieve the declared fields only, use the 

getDeclaredFields()
method. This will only get the fields that are declared in the given class (not inherited): 

          
           
Field[] fields = FieldUtils.getDeclaredFields(MyClass.class);

Getting Fields with a Specific Modifier

You can also get fields that have a specific modifier, such as public or private, using the 


          
           
getFieldsWithModifier()
method. The first parameter is the type to search, and the second is the modifier. 

          
           
Field[] fields = FieldUtils.getFieldsWithModifier(MyClass.class, Modifier.PRIVATE);

Getting Field By Name

If you already know the name of the field you want to retrieve, you can use the 

getField()
method. This will search the class hierarchy for the specified field, returning it if found: 

          
           
Field field = FieldUtils.getField(MyClass.class, "name");

Getting Field Value

Once you have a Field instance, you can use the 
getValue()
method to get the value of the field. This takes an instance of the containing object as the second parameter: 

          
           
Object value = FieldUtils.getValue(field, obj);

Setting Field Value

Finally, you can use the 
setValue()
method to set the value of a field. Again, this takes the instance of the containing object as the third parameter: 

          
           
FieldUtils.setValue(field, obj, value);
