Overview of javax.ws.rs.core.GenericType

The javax.ws.rs.core package contains APIs for developing Representational State Transfer (REST) web services in the Java programming language. One of the important classes it contains is the Generic Type API. This API provides a generic type that can be used to obtain specific types of objects from an interface.

Creating a Generic Type

A Generic Type consists of two parts: the base type and the type parameter.

The base type is the class or interface of the object you want to obtain from the interface. It can be any type that implements the interface. For example, if you wanted to obtain a List of integers, then the base type would be java.util.List.

The type parameter is what will actually be passed into the interface. It must be compatible with the base type. For instance, if the base type is java.util.List, then the type parameter must be an Integer. The following code shows how to create a Generic Type for a List of integers:


          
           
GenericType<List<Integer>> myGenericType = new GenericType<>(List.class, Integer.class);

public class MyInterface { 
  public List<Integer> getListOfIntegers(GenericType<List<Integer>> genericType) { 
    return genericType.get(); 
  } 
}

In this example, the Generic Type is passed into the interface as an argument. The interface then uses this Generic Type to obtain a List of integers. The result is a List of integers that can be used within the interface.

Conclusion

The javax.ws.rs.core.GenericType API provides a convenient way to obtain specific types of objects from an interface.
