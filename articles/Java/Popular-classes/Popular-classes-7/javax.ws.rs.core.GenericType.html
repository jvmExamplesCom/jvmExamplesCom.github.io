Overview of javax.ws.rs.core.GenericType

The javax.ws.rs.core package contains APIs for developing Representational State Transfer (REST) web services in the Java programming language. Creating a Generic Type

A Generic Type consists of two parts: the base type and the type parameter.

The base type is the class or interface of the object you want to obtain from the interface. It can be any type that implements the interface. For example, if you wanted to obtain a List of integers, then the base type would be java.util.List.

The type parameter is what will actually be passed into the interface. It must be compatible with the base type. GenericType<List<Integer>> myGenericType = new GenericType<>(List.class, Integer.class);
