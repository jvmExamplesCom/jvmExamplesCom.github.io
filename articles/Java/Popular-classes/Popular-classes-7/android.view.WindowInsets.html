android.view.WindowInsets

The android.view.WindowInsets class is a subclass of Insets which describes the insets for a given Window. It is used to specify where various elements of the window should be placed, such as the status bar, navigation bar, and more. Creating Instances

There are three ways to get instances of WindowInsets: 

          
           
WindowInsets insets = new WindowInsets(int left, int top, int right, int bottom);

// OR 

WindowInsets insets = new WindowInsets(Rectangle rect);

// OR 

WindowInsets insets = new WindowInsets();
These methods will create a new instance of WindowInsets with the specified insets.

Getting Insets

WindowInsets provides several methods to get the insets that were provided when the instance was created. The most common method is getInsets(), which returns a Rectangle containing the left, top, right, and bottom insets. 

          
           
WindowInsets insets = new WindowInsets(10, 5, 20, 15);
Rectangle rect = insets.getInsets();
In this example, rect would contain the values 10, 5, 20, and 15.

Setting Insets

WindowInsets also provides methods to set individual insets. To set all four insets at once, you can use setInsets().

insets.setInsets(10, 5, 20, 15);
You can also set individual insets using setLeft(), setTop(), setRight(), or setBottom(), or set several of these at once using setInsets(int left, int top, int right, int bottom).
