android.view.WindowInsets

The android.view.WindowInsets class is a subclass of Insets which describes the insets for a given Window. WindowInsets insets = new WindowInsets();

Getting Insets

WindowInsets insets = new WindowInsets(10, 5, 20, 15);
Rectangle rect = insets.getInsets();
In this example, rect would contain the values 10, 5, 20, and 15.

Setting Insets
