Introduction to Apache Commons CLI and OptionBuilder

Apache Commons CLI (Command Line Interface) is a library from the Apache Software Foundation that provides an API for easily processing command-line arguments and options. The library is available under the Apache 2 license, making it free to use in any Java application. OptionBuilder is a utility class within the Commons CLI library that can be used to easily create Options.

Overview of OptionBuilder

OptionBuilder is a ‘fluent’ builder-style class used to create options for the Commons CLI library. It allows for easy creation of Options, which are essentially data holders for individual command-line arguments and/or options. OptionBuilder provides various methods for setting various properties of an Option, such as its long name, short name, description and other properties. A single Option object can then be passed to the appropriate CommandLineParser instance in order to be parsed and acted upon.

Creating an Option with OptionBuilder

OptionBuilder can be used to quickly create an Option object: 

          
           
Option option = OptionBuilder.withLongOpt("longname")
    .withDescription("Description")
    .create();
In this example, an Option is created with a long name (i.e. the parameter after “–longname”) of “longname”, and a description of “Description”. This Option can then be passed to the appropriate CommandLineParser instance in order to be parsed and acted upon.

Setting Additional Properties for an Option

OptionBuilder also allows for the setting of additional properties for an Option. For example, the hasArg() method can be used to indicate if the option requires an argument: 

          
           
Option option = OptionBuilder.withLongOpt("longname")
    .hasArg()
    .withDescription("Description")
    .create();
In this example, the Option requires an argument, meaning that it must be followed by a value (e.g. “–longname=value”). The hasArgs() method can also be used to indicate if the option accepts multiple arguments: 

          
           
Option option = OptionBuilder.withLongOpt("longname")
    .hasArgs()
    .withDescription("Description")
    .create();
In this example, the Option accepts multiple arguments, meaning that it can be followed by multiple values (e.g. “–longname=value1,value2,value3”). The setRequired() method can be used to indicate if the option is required or not: 

          
           
Option option = OptionBuilder.withLongOpt("longname")
    .setRequired()
    .withDescription("Description")
    .create();
In this example, the Option is required, meaning that the command-line must contain the option in order for it to be parsed successfully.

Conclusion

OptionBuilder is an extremely useful utility class within the Apache Commons CLI library that can be used to easily create Options.
