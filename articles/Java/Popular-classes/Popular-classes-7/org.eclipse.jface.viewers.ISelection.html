org.eclipse.jface.viewers.ISelection

The org.eclipse.jface.viewers.ISelection interface is part of the JFace framework. It allows a user to maintain the current selection in a viewer or editor. The ISelection interface includes methods to access the selection, check if the selection is empty, and clear the selection.
