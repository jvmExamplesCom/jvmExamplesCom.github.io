Android.Widget.Tab Host

A TabHost is a layout container that allows the developers to add multiple tabs and widgets. It works as a container for tabbed window view enabling the user to move from tab to tab, or from one window or view to another. As such, it provides a structure for organizing the user interaction and data consumption. The TabHost contains two components: the TabWidget and the TabContent.

Using TabHost

To use TabHost in an Android application, you need an XML layout file similar to this one:


          
           
<?xml version="1.0" encoding="utf-8"?>
<TabHost 
    xmlns:android="http://schemas.android.com/apk/res/android"
    android:id="@android:id/tabhost" 
    android:layout_width="fill_parent" 
    android:layout_height="fill_parent">
<LinearLayout 
    android:orientation="vertical"
    android:layout_width="fill_parent"
    android:layout_height="fill_parent"
    >
    <TabWidget
        android:id="@android:id/tabs"
        android:layout_width="fill_parent"
        android:layout_height="wrap_content"/>
    <FrameLayout
        android:id="@android:id/tabcontent"
        android:layout_width="fill_parent"
        android:layout_height="fill_parent"/>
</LinearLayout>
</TabHost>

The TabHost needs to have an ID, a FrameLayout, as well as a TabWidget. You can also set other parameters, such as size, orientation, padding and gravity. The TabWidget handles the display of the tabs, while the FrameLayout handles the switching between the tabs.

Adding TabItems

TabHost tHost = (TabHost)findViewById(R.id.tabhost);
TabSpec firstTabSpec = tHost.newTabSpec("tab_id1"); 
TabSpec secondTabSpec = tHost.newTabSpec("tab_id2");

Next, you will want to set the label and content associated with each tab. To do this, use the setIndicator() and setContent() methods. The setIndicator() method sets the label or icon for the tab. The setContent() method sets the content to be displayed when the tab is selected. You can also set a background image here.


          
           
firstTabSpec.setIndicator("Tab 1");
firstTabSpec.setContent(R.id.tab_1);

secondTabSpec.setIndicator("Tab 2");
secondTabSpec.setContent(R.id.tab_2);

After setting the label and content, add the tab to the TabHost using addTab().


          
           
tHost.addTab(firstTabSpec);
tHost.addTab(secondTabSpec);

To handle changes in tab selection, you need to implement the OnTabChangeListener interface in your activity. This is usually done inside the onCreate() method. Once you implement the interface, your activity receives a callback whenever a tab is changed. The callback method receives two arguments: the currently selected tab id and the previously selected tab id.

   }

With its ability to contain and manage multiple tabs, it enables developers to create more interactive experiences for their users.
