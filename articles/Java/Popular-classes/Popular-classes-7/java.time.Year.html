Java.time.Year

Java.time.Year is a class within the Java SE 8 date-time API which provides date and time support. This class represents a year in the ISO-8601 calendar system, such as 2007. Year has no public constructor thereby forcing all its subclasses to implement the same set of methods.

Methods available


          
           
public boolean isLeap()
This method determines if this year is a leap year in the ISO calendar system. The following example returns true if given year is a leap year, false otherwise: 

          
           
class YearExample {

  private int year;

  public YearExample(int y) {
    this.year = y;
  }

  public boolean checkLeapYear(){
    return Year.of(year).isLeap();
  }

  public static void main(String[] args){
    YearExample y = new YearExample(2016);
    System.out.println("Is the given year leap? "+ y.checkLeapYear()); 
    // output: Is the given year leap? true
  }
}
