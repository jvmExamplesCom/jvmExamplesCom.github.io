Introduction to com.auth0.jwt.JWT

com.auth0.jwt.JWT is a Java library that supports creating, signing and verifying signed JSON Web Tokens (JWTs). This tutorial will provide an in-depth guide on how to use the library to work with JWTs.

Creating a JWT

First, let's take a look at how to create a JWT. The following example creates a JWT and signs it using an RSA key:


          
           
String jwt = JWT.create()
    .withIssuer("MyApp")
    .withSubject("Jimmy Jones")
    .sign(Algorithm.RSA256(publicKey, privateKey));

The JWT created in this example has an issuer of “MyApp” and a subject of “Jimmy Jones”. The JWT will be signed using an RSA algorithm with the given public and private keys.

Adding Claims to a JWT

It is possible to add additional claims to a JWT. Claims are statements about the token itself, such as its expiration time or the audience it is intended for. The following example adds an expiration time and an audience claim to the previously created JWT:


          
           
String jwt = JWT.create()
    .withIssuer("MyApp")
    .withSubject("Jimmy Jones")
    .withExpiresAt(expirationTime)
    .withAudience("MyAppUsers")
    .sign(Algorithm.RSA256(publicKey, privateKey));

The JWT now includes an expiration time and an audience claim. It is also possible to add custom claims, such as a user’s role or the version of the application.

Verifying a JWT

In addition to creating JWTs, com.auth0.jwt.JWT can also be used to verify that a given JWT is valid. The following example verifies the previously created JWT:


          
           
JWTVerifier verifier = JWT.require(Algorithm.RSA256(publicKey, privateKey))
    .withIssuer("MyApp")
    .build();

DecodedJWT decodedJwt = verifier.verify(jwt);
The 
JWTVerifier
instance is used to verify that the given JWT is signed using the given RSA key and has an issuer of “MyApp”. If the JWT is valid, the 
DecodedJWT
instance is returned. From the 
DecodedJWT
instance, it is possible to get access to the claims of the JWT, including the subject, expiration time and audience.

Conclusion

com.auth0.jwt.JWT is a powerful library for working with JWTs in Java. This tutorial provided an overview of how to use the library to create, sign and verify JWTs.
