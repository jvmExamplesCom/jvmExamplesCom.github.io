Introduction to android.text.method.PasswordTransformationMethod

The 

          
           
android.text.method.PasswordTransformationMethod
class is an Android-specific class that is used for concealing user input in EditText widgets when entering passwords or other sensitive information. This class provides a clean and secure interface for the user when entering passwords, or any other text input.

Using android.text.method.PasswordTransformationMethod

To use the 

          
           
PasswordTransformationMethod
class, you simply need to set it as the transformation method of an 

          
           
EditText
: 

          
           
EditText editText = (EditText) findViewById(R.id.my_edit_text);
editText.setTransformationMethod(new PasswordTransformationMethod());
This will replace the content of the 

          
           
EditText
with asterisks (*) to conceal it from view. The actual content of the 

          
           
EditText
can still be retrieved by calling its 

          
           
getText()
method, but will not be visible to the user.

Caveats

It is important to note that the 

          
           
PasswordTransformationMethod
does not actually encrypt the data that is entered into an 

          
           
EditText
. It only obscures it from view by replacing it with asterisks (*). This is fine for concealing passwords and other sensitive data from the user, but should not be relied upon for encryption.

PasswordTransformationMethod
class: 

          
           
public class MainActivity extends AppCompatActivity {

    @Override
    protected void onCreate(Bundle savedInstanceState) {

    }
