org.gradle.api.DefaultTask

The org.gradle.api.DefaultTask class is the default task implementation in Gradle. It provides an easy way to write tasks— the user only needs to override the execute() method. This class implements the Task interface, which is a marker interface for Gradle tasks.

Example Usage:

    @TaskAction
    public void doStuff() {
        // do something
    }

}

@TaskAction
