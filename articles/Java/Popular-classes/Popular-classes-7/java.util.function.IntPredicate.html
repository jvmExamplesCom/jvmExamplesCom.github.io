java.util.function.IntPredicate

java.util.function.IntPredicate is an interface in the Java programming language that specifies an abstract method that tests a given int-valued argument against some criterion and returns a boolean result. IntPredicate is most commonly used as a functional interface, meaning it can be used as the assignment target for a lambda expression or method reference. This interface has one abstract method, test, which takes an int-valued argument and returns a boolean result.

Example Usage

List<Integer> numbers = Arrays.asList(1, 2, 3, 4, 5, 6, 7, 8, 9);
 
IntPredicate evenNums = n -> n % 2 == 0;
