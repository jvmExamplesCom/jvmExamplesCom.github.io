Java androidx.preference.Preference

The 

          
           
androidx.preference.Preference
class is the base class of all preferences used in Android projects. By extending this class, you can add new preferences to your project. androidx.preference.Preference

int resId = R.xml.pref_enabled;
addPreferencesFromResource(resId);

boolean enabled = getValue("enabled", false);
