The File OpenOption interface specifies methods for determining the permissions and other property settings that should be applied when opening a file. This interface provides developers with a set of options that can be used to configure operations such as file copy, move, delete and open.

How to use Java NIO File OpenOption

Then, an invocation of the Files.open() method is used to create a new File object that represents the file that is to be opened.
