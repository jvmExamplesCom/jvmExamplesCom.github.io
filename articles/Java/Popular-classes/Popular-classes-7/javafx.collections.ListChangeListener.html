JavaFx.Collections.ListChangeListener

JavaFx.Collections.ListChangeListener is an interface used in the JavaFx platform to monitor changes to an observable list. The observable list is a list which has been modified to be observed and listened to by an object implementing the listener interface.

The ListChangeListener interface defines a method, onChanged(), which is called every time the list changes and passes in a ListChangeListener.Change object. The change object contains information about the list changes that have occured, such as which elements were added or removed and the indices of where these changes occurred.

Example Usage

To demonstrate usage, let's create a simple observable list and call a ListChangeListener upon it. Our list will contain strings.


          
           
ObservableList<String> list = FXCollections.observableArrayList();
list.addAll("a", "b", "c");

list.addListener(new ListChangeListener<String>() {
	public void onChanged(Change<? extends String> c) {
		System.out.println("Change detected!");
		while (c.next()) {
			if (c.wasAdded()) {
				System.out.println("Elements added: " + c.getAddedSubList());
				System.out.println("Indexes of added element: " + c.getFrom();
			} else if (c.wasRemoved()) {
				System.out.println("Elements removed: " + c.getRemoved());
				System.out.println("Indexes of removed element: " + c.getFrom());
			}
		}
	}
});

In the example above, we create an observable list containing a, b and c strings and we add a ListChangeListener to it. Our ListChangeListener will print out a message whenever a change is detected in the list and will also print out details about which elements were added/removed and their indices.

Now let's make some changes to our list to see how our ListChangeListener responds.


          
           
list.addAll("d", "e");
list.remove("a");

When these changes are made, our ListChangeListener will respond with the following output.


          
           
Change detected!
Elements added: [d, e]
Indexes of added element: 3
Change detected!
Elements removed: [a]
Indexes of removed element: 0

As we can see, our ListChangeListener accurately reported the changes that were made to the list.
