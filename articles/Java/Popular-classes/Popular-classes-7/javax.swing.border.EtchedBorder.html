javax.swing.border.EtchedBorder

The 
EtchedBorder
class, part of the 
javax.swing.border
package, provides a border with an etched effect. The actual contents of the etched border can be defined as either beveled or relieved. This can provide a nice visual effect to any application or website.

Constructors

The 
EtchedBorder
class exposes the following constructors: 

          
           
EtchedBorder()
EtchedBorder(int etchType)
EtchedBorder(int etchType, Color highlight, Color shadow)
EtchedBorder(Color highlight, Color shadow)
EtchedBorder(BevelBorder bevelBorder)

Parameters

The 
etchType
parameter can take on one of two different values: 
LOWERED
or 
RAISED
. This denotes which type of etched effect is desired, beveled or relieved. The 
highlight
and 
shadow
parameters are used to specify the colors of the etched effect. Typically, the 
highlight
color is chosen one shade lighter than the background color, while the 
shadow
color is chosen one shade darker than the background color.

Example

Here is an example of how to use the 
EtchedBorder
class: 

          
           
import javax.swing.border.EtchedBorder;
import java.awt.Color;

EtchedBorder myBorder = new EtchedBorder(EtchedBorder.RAISED, Color.YELLOW, Color.GREEN);

EtchedBorder
class with an etch type of 
RAISED
, and with a 
highlight
color of yellow and a
