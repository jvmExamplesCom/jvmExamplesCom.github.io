What is Java io.realm.RealmObject?

Java 

          
           
io.realm.RealmObject
is a class used to create data objects which can be stored in a Realm database. These objects represent persistent data which can be queried using the powerful RealmQuery language. The RealmObject class is an abstract class which must be extended by other classes to define the structure of the data object.

A Simple Example of Java io.realm.RealmObject

Let's create a simple data object for a Person: 

          
           
public class Person extends RealmObject {
  private String name;
  private int age;

  public String getName() {
     return name;
  }

  public void setName(String name) {
     this.name = name;
  }

  public int getAge() {
     return age;
  }

  public void setAge(int age) {
     this.age = age;
  }
}

This simple Person class defines two properties: name and age. These properties are mappable to a Realm database, so you can store and query them easily.

Creating a Realm Object Instance

You can create a RealmObject instance by using the createObject() method from the Realm class:


          
           
Person person = realm.createObject(Person.class);
person.setName("John Doe");
person.setAge(30);

The above code creates a Person object and sets the name and age properties. Note that the createObject() method will only persist the object when a Realm transaction has been committed.

Querying a Realm Object

You can use the powerful RealmQuery language to query a RealmObject instance:


          
           
RealmQuery<Person> query = realm.where(Person.class);
Person person = query
    .equalTo("name", "John Doe")
    .findFirst();

The above code will find the Person object with the name "John Doe". If no such object is found, then the query will return null.

Updating a Realm Object

You can update a RealmObject with the update() method from the Realm class:


          
           
person.setName("Jane Doe");
realm.update(person);

The above code will update the Person object with the new name value. Note that this will only take effect after a Realm transaction is committed.

Deleting a Realm Object

You can delete a RealmObject with the delete() method from the Realm class:


          
           
realm.delete(person);

The above code will delete the Person object. Note that this only takes effect after a Realm transaction is committed.
