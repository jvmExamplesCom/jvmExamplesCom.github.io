javax.swing.JScrollBar

The javax.swing.JScrollBar class is a component that provides the user with the ability to view and interact with content that is larger than the visible area of a display. It allows the user to scroll content by dragging the handle, or by clicking on the arrows at either end of the scroll bar. It can also be used as an input device for selecting values within a range.

The JScrollBar class provides a couple of methods that allow developers to control the behaviour of the scrollbar, like setUnitIncrement() and setBlockIncrement(). Similarly, the setBlockIncrement() method determines the step size when the user drags the handle of the bar. The default value is 100 pixels.

Creating a JScrollBar

The following example shows how to create a basic JScrollBar:


          
           
JScrollBar scrollBar = new JScrollBar();

scrollBar.setPreferredSize(new Dimension(200, 20));
