javax.swing.JScrollBar

The javax.swing.JScrollBar class is a component that provides the user with the ability to view and interact with content that is larger than the visible area of a display. Similarly, the setBlockIncrement() method determines the step size when the user drags the handle of the bar. The default value is 100 pixels.

Creating a JScrollBar

The following example shows how to create a basic JScrollBar:


          
           
JScrollBar scrollBar = new JScrollBar();
