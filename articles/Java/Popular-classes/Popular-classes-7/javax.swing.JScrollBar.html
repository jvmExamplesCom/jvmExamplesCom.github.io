javax.swing.JScrollBar

The javax.swing.JScrollBar class is a component that provides the user with the ability to view and interact with content that is larger than the visible area of a display. It allows the user to scroll content by dragging the handle, or by clicking on the arrows at either end of the scroll bar. It can also be used as an input device for selecting values within a range.

The JScrollBar class provides a couple of methods that allow developers to control the behaviour of the scrollbar, like setUnitIncrement() and setBlockIncrement(). The setUnitIncrement() method is used to determine the step size of the scrollbar when the user clicks on one of the arrows at the end of the bar. The default value is 10 pixels. Similarly, the setBlockIncrement() method determines the step size when the user drags the handle of the bar. The default value is 100 pixels.

Creating a JScrollBar

The following example shows how to create a basic JScrollBar:


          
           
JScrollBar scrollBar = new JScrollBar();
scrollBar.setOrientation(JScrollBar.HORIZONTAL);
scrollBar.setPreferredSize(new Dimension(200, 20));

We can also specify the minimum and maximum values that the scrollbar can take. This is done using the setMinimum() and setMaximum() methods:


          
           
scrollBar.setMinimum(0); 
scrollBar.setMaximum(100);
