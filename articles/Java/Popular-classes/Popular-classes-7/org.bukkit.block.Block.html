org.bukkit.block.Block

The org.bukkit.block.Block class is a core class in the Bukkit API, and provides a wide range of methods and functions that enable developers to easily interact with the various blocks and tiles found in the game of Minecraft. Material blockType = block.getType();

We can also set the type of a block using the setType() method. This example sets the block to a dirt block:


          
           
block.setType(Material.DIRT);
