org.bukkit.block.Block

The org.bukkit.block.Block class is a core class in the Bukkit API, and provides a wide range of methods and functions that enable developers to easily interact with the various blocks and tiles found in the game of Minecraft. Get and Set Type

The getType() method allows us to retrieve the type of the block. For example, if we wanted to find out the type of a particular block, we could do so like this:


          
           
Block block = world.getBlockAt(x, y, z);
Material blockType = block.getType();

We can also set the type of a block using the setType() method. This example sets the block to a dirt block:


          
           
block.setType(Material.DIRT);

byte data = block.getData();
