org.springframework.data.annotation.CreatedDate

The org.springframework.data.annotation.CreatedDate annotation is used to mark a field as containing the date when an entity was created in the database. The CreatedDate annotation provides us with a way to track when a particular entity was created.

Example Usage

To demonstrate how the CreatedDate annotation can be used, let's take the following example: we have a simple Customer entity with two fields, an id and a name.


          
           
public class Customer {

    private Long id;

    private String name;

}

To add a CreatedDate field to this entity, we first need to add the annotation to the field:


          
           
public class Customer {

    private Long id;

    private String name;

    @CreatedDate
    private LocalDateTime createdAt;

}

This tells Spring Data to use the current timestamp as the value for the createdAt field. We can also specify a different timestamp if we wish, by providing a value for the annotation:


          
           
@CreatedDate(value = "2020-01-01 00:00:00")
    private LocalDateTime createdAt;

In this case, the timestamp provided in the annotation will be used instead of the current timestamp.

Conclusion

The org.springframework.data.annotation.CreatedDate annotation is a useful way to keep track of when specific entities were created in the database. It is simple to use and provides us with an easy way to store the creation date of entities.
