android.content.ContentProviderClient

Overview

ContentProviderClient is a way to access a remote ContentProvider in an easy and efficient manner. It allows you to make calls to the ContentProvider without having to pass through the binder framework. You can also use it to manage transactions and reads. Additionally, ContentProviderClient is a replacement for managedQuery and other query methods that were deprecated in API level 11 (Android 3.0).
