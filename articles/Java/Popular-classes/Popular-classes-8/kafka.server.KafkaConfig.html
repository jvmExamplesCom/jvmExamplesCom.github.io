The 

          
           
KafkaConfig
class is part of the Apache Kafka package 

          
           
org.apache.kafka.server
. It is responsible for setting up a Kafka server node. It provides several pieces of data which are used by a client-facing Kafka broker to create topics, configure logfiles, and customize its behavior. The first part of the 

          
           
KafkaConfig
class is the 

          
           
config
field. This is a 

          
           
Map<String, Object>

Map<String, Object> config = new HashMap<>();
    config.put("broker.id", "1");
    config.put("port", 9092);
    config.put("advertised.host.name", "example.com");

KafkaConfig
class are setters and getters for various configurable options. For example, the

Properties props = new Properties();
