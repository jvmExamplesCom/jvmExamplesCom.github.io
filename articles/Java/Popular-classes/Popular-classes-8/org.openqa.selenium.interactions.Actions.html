org.openqa.selenium.interactions.Actions

Creating and Running Actions

The action can be passed a WebElement object as an argument, which allows the action to be performed on an element. When all the actions are added, they can be executed with the perform() method, which will execute all the chained actions in the order they were added. For example, the following code creates a chain of three actions, and executes them when the perform() method is called:


          
           
Actions action = new Actions(webDriver); 
action.moveToElement(element).click().perform();
