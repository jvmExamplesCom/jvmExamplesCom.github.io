android.preference.Preference.OnPreferenceClickListener

The android.preference.Preference.OnPreferenceClickListener interface is used to receive click events on a Preference which is a view in an Android application. This interface provides an easy way to respond to user clicks and is commonly used to open a new Activity or display a dialog when a particular option is clicked.

Implementation

The following code shows how to implement the OnPreferenceClickListener interface: 

          
           
public class MyActivity extends AppCompatActivity implements OnPreferenceClickListener {

   @Override
   protected void onCreate(Bundle savedInstanceState) {
      super.onCreate(savedInstanceState);
      addPreferencesFromResource(R.xml.preferences);
      // get the Preference object for the "My Preference"
      Preference myPreference = findPreference("my_pref");
      // set the click listener
      myPreference.setOnPreferenceClickListener(this);
   }

   @Override
   public boolean onPreferenceClick(Preference preference) {
      // do something when the preference is clicked
      return true;
   }
}

Usage

Once the OnPreferenceClickListener interface is implemented, it can be used to handle click events on a Preference. The following example shows how to use the listener: 

          
           
// get the Preference object for the "My Preference"
Preference myPreference = findPreference("my_pref");
// set the click listener
myPreference.setOnPreferenceClickListener(new OnPreferenceClickListener() {
   @Override
   public boolean onPreferenceClick(Preference preference) {
      // do something when the preference is clicked
      return true;
   }
});
When the user clicks the Preference, the onPreferenceClick method will be called and the application can then perform any appropriate action.
