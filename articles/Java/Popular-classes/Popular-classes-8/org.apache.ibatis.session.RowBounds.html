org.apache.ibatis.session.RowBounds

RowBounds is a class in the org.apache.ibatis.session package that allows for the configuration of a “paging” query in a database. It provides methods to set the number of records to skip and the maximum number of records to retrieve from the database.

Overview

The RowBounds class is used to set limits on the number of records retrieved from the database. That is, it specifies an offset and a limit within the total number of rows, allowing a subset of rows to be retrieved. The main advantage of using this class is that, rather than executing two queries (one to count the total number of rows and one to retrieve the subset of rows), we can execute a single query that returns exactly the desired data. The RowBounds class also allows developers to page through large result sets easily, since each page can contain a subset of the total number of rows returned by the query.

How to Use it

RowBounds rowBounds = new RowBounds(offset, limit);

SQLQuery query = ...;
List<Object> results = session.selectList(query, rowBounds);

Now the query will return only the records specified by the rowBounds object, i.e. the offset and the limit set by the RowBounds object.
