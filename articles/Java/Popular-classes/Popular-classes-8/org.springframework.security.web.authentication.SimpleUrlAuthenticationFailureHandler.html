org.springframework.security.web.authentication.SimpleUrlAuthenticationFailureHandler

The SimpleUrlAuthenticationFailureHandler class is a part of the Spring Security web package that helps developers handle authentication failures through the use of customizable methods. This class implements the interface AuthenticationFailureHandler, and allows developers to pass a number of parameters into the constructor.

The SimpleUrlAuthenticationFailureHandler constructor takes in three arguments: a URL used for unsuccessful login attempts, a flag for determining whether or not to broadcast an event if the authentication fails, and a set of query parameters to be sent with the redirect.

Usage

The SimpleUrlAuthenticationFailureHandler can be used as part of a basic authentication system. First, an authentication provider is created.


          
           
AuthenticationProvider authProvider = new CustomAuthenticationProvider();

The authentication provider is then set up with a failure handler:


          
           
authProvider.setAuthenticationFailureHandler(new SimpleUrlAuthenticationFailureHandler("/login/failure", true, ParameterMap.of("username", "%s")));
We can now set up the security filter chain: 

          
           
http.authorizeRequests()

  .and()
  .addFilterBefore(new CustomFilter(), BasicAuthenticationFilter.class)
  .formLogin().and()
  .logout();

Finally, we can provide our authentication provider to the authentication manager:


          
           
Auth.authenticationProvider(authProvider);

Example

Let's take a look at a simple example. We'll use the Spring Boot Starter Security package to create a basic login system. We will also set up our CustomAuthenticationProvider to use the SimpleUrlAuthenticationFailureHandler.


          
           
@SpringBootApplication
public class ExampleApplication {
     
    public static void main(String[] args) {
        SpringApplication.run(ExampleApplication.class, args);
    }
    
    @Bean
    public AuthenticationProvider authProvider() {
        CustomAuthenticationProvider authProvider = new CustomAuthenticationProvider();
        authProvider.setAuthenticationFailureHandler(new SimpleUrlAuthenticationFailureHandler("/login/failure", true, ParameterMap.of("username", "%s")));
        return authProvider;
    }
    
    @Bean
    public WebSecurityConfigurerAdapter securityConfigurerAdapter() {
        return new WebSecurityConfigurerAdapter() {

            @Override
            protected void configure(HttpSecurity http) throws Exception {
                http.authorizeRequests()
                        .anyRequest().authenticated()
                        .and()
                        .addFilterBefore(new CustomFilter(), BasicAuthenticationFilter.class)
                        .formLogin().and()
                        .logout();
            }
        };
    }
    
}

In this example, we've set up a simple authentication system using our CustomAuthenticationProvider. We've used the SimpleUrlAuthenticationFailureHandler to redirect the user to a page when authentication fails. The URL is set to "/login/failure", and the query parameter "username" will be passed with the redirect.
