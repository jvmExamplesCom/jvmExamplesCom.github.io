The IDialogConstants interface provides a basic set of button labels, dialog messages, and other constants which can be used to present a consistent set of dialog boxes across all eclipse-based applications. It also allows developers to customize the dialog according to their needs.

Using IDialogConstants

IDialogConstants.YES_LABEL
to assign the label “Yes” to a button:
