The Eclipse JFace IDialogConstants

The package org.eclipse.jface.dialogs.IDialogConstants is an interface in the Eclipse graphical framework that can be used to provide user dialogs with a range of features and control. The IDialogConstants interface provides a basic set of button labels, dialog messages, and other constants which can be used to present a consistent set of dialog boxes across all eclipse-based applications. It also allows developers to customize the dialog according to their needs.

Using IDialogConstants

One of the primary uses of IDialogConstants is to define the labels or captions of the buttons shown in a dialog box. For example, the following code uses the constant 


          
           
IDialogConstants.YES_LABEL
to assign the label “Yes” to a button: 

          
           
Button btn = new Button(parent, SWT.PUSH); 
btn.setText(IDialogConstants.YES_LABEL);

In addition to the common YES/NO/OK/CANCEL buttons, IDialogConstants also provides constants for more specialized scenarios. For example, the 


          
           
IDialogConstants.ABORT_LABEL
constant will display a button with the label “Abort”, while 

          
           
IDialogConstants.RETRY_LABEL
will produce a “Retry” button.

The IDialogConstants interface also provides constants that allow developers to assign dialog titles, as well as message strings, to any given dialog. For example, the following code assigns the title “Warning” to a dialog, and displays the message “Are you sure?”:
