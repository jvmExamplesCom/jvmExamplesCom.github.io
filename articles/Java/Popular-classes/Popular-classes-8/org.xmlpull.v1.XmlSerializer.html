Introduction to org.xmlpull.v1.XmlSerializer

Methods of XmlSerializer

XmlSerializer serializer = new XmlSerializer();

serializer.writeTag("namespace", "tagName", attributes);

The writeText() method adds text content to the document. It can be used to write plain text or CDATA sections. It takes a single argument, the text to be written.


          
           
serializer.writeText("text content");

The writeEndTag() method closes a tag that has been previously opened with writeTag(). It takes two arguments: a namespace and a tag name.


          
           
serializer.writeEndTag("namespace","tagName");
