Introduction to org.thymeleaf.context.Context

The org.thymeleaf.context.Context class is an essential part of the Thymeleaf template engine for Java web applications. It is used to store context-specific objects and variables, which are scoped for the duration of an HTML page. The Context class provides the programmer with an easy way to access and modify values within a context-specific scope. In this article we will explore how to use the Context class, and provide some code examples to illustrate the usage.

Creating a Context Object

Creating a new Context object is easy, and can be done with the following code: 

          
           
Context context = new Context();
Once a Context object has been created, values can be set using the setVariable() method. This method takes two parameters; the name of the variable to be set, and the value of the variable. 

          
           
context.setVariable("name", "John Doe");
context.setVariable("age", 32);
The setVariable() method can also take a third parameter, which is used to denote the scope of the variable. The default value for this parameter is Context.LOCAL_SCOPE, and other valid values are Context.GLOBAL_SCOPE and Context.PAGE_SCOPE. The following example sets a variable in the global scope: 

          
           
context.setVariable("email", "john.doe@example.com", Context.GLOBAL_SCOPE);

Retrieving a Variable from the Context

To retrieve a variable from the Context, the getVariable() method can be used. This method takes a single parameter, the name of the variable, and returns the associated value. 

          
           
String name = context.getVariable("name");
int age = context.getVariable("age");
String email = context.getVariable("email");

Removing Variables from the Context

Variables can be removed from the Context with the removeVariable() method, which takes two parameters; the name of the variable and its scope. To remove a variable from the page scope, the default scope can be omitted, as shown in the following example: 

          
           
context.removeVariable("name");
context.removeVariable("email", Context.GLOBAL_SCOPE);
