android.Manifest.permission

Android permissions are necessary for apps to access certain features on a user’s phone. These permissions are requested by the app during installation and must be granted by the user. The android.Manifest.permission class contains methods that are used to check the permission status of an app and request permission from the user.

Checking Permission Status

The 

          
           
checkSelfPermission()
method is used to check the permission status of an app. This method takes a parameter of Permission and returns a result code. The result code will be either PERMISSION_GRANTED or PERMISSION_DENIED. For example, to check if an app has the permission to access the camera: 

          
           
int result = ContextCompat.checkSelfPermission(this, Manifest.permission.CAMERA);

if (result == PackageManager.PERMISSION_GRANTED){
    // App has the permission
} else {
    // App does not have the permission
}

Requesting Permission

The 

          
           
requestPermissions()
method is used to request permission from the user. This method takes an array of permissions along with a request code as parameters. The request code will be used when the user responds to the request. For example, to request permission to access the camera: 

          
           
ActivityCompat.requestPermissions(this, new String[]{Manifest.permission.CAMERA}, CAMERA_REQUEST_CODE);
The user will then be presented with a dialog asking them to grant permission to the app. When the user responds, the 

          
           
onRequestPermissionsResult()
method will be called with the request code and the result of the request. For example: 

          
           
public void onRequestPermissionsResult(int requestCode, @NonNull String[] permissions, @NonNull int[] results){
    if (requestCode == CAMERA_REQUEST_CODE){
        if (results.length > 0 &amp;&amp; results[0] == PackageManager.PERMISSION_GRANTED){
            // Permission granted
        } else {
            // Permission denied
        }
    }
}

Conclusion

The android.Manifest.permission class provides methods that can be used to check and request permissions on an Android device. These methods should be used in all apps to ensure that the user is aware of the permissions that are being requested and that the app only accesses those features that it needs to.
