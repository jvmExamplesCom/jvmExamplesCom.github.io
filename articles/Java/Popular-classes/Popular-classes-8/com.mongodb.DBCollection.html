Overview of com.mongodb.DBCollection

Introduction

The com.mongodb.DBCollection class is part of the MongoDB Java Driver version 3.0 and is a basic object-oriented programming interface for interacting with MongoDB, an open-source document-oriented NoSQL database. DBCollection objects represent a collection in a MongoDB database and allow for operations such as inserting, updating, querying, and deleting documents. Additionally, these objects can be used to create indexes, authenticate users, manage access control lists, and more. In this article we will cover the basics of the com.mongodb.DBCollection class and provide example code demonstrating each supported operation.

Creating a DBCollection

Before a DBCollection object can be used, it must first be created. To initiate a new collection, you may use either the 

          
           
MongoClient#getDatabase()
or 

          
           
MongoClient#getCollection()
methods. The 

          
           
getDatabase()
method returns a 

          
           
com.mongodb.client.MongoDatabase
, which provides access to the collections via its 

          
           
MongoDatabase#getCollection()
method. The following example shows how to create a DBCollection from a MongoClient: 

          
           
MongoClient mongoClient = new MongoClient();
DBCollection collection = mongoClient.getDatabase("test").getCollection("myCollection");
Alternatively, you may also create a DBCollection directly from the MongoClient by passing the collection's name as an argument to the 

          
           
MongoClient#getCollection()
method: 

          
           
MongoClient mongoClient = new MongoClient();
DBCollection collection = mongoClient.getCollection("myCollection");

DBCollection#insert()
method. This method can accept either a single 

          
           
com.mongodb.BasicDBObject

BasicDBObject doc = new BasicDBObject("name", "John")
                            .append("age", 30)

                                                       .append("city", "Boston")

DBCollection#insert()

Querying Documents

To query the documents in a DBCollection, you may use the 

          
           
DBCollection#find()
method. This method accepts a 

          
           
com.mongodb.BasicDBObject
containing the query parameters and returns a 

          
           
com.mongodb.DBCursor
which provides an iterator over all of the documents matching the query. The following example shows how to query the documents in a collection and iterate over the results: 

          
           
// Create a query
BasicDBObject query = new BasicDBObject("name", "John");
// Execute the query
DBCursor cursor = collection.find(query);
// Iterate over the results
while (cursor.hasNext()) {
    System.out.println(cursor.next());

Updating Documents

You may also update documents in a collection using the 

          
           
DBCollection#update()
method. This method accepts two arguments: a 

          
           
com.mongodb.BasicDBObject
containing the query parameters used to identify the target documents, and a 

          
           
com.mongodb.BasicDBObject
containing the document data to be updated. The following example shows how to update a single document in a collection: 

          
           
// Create a query
BasicDBObject query = new BasicDBObject("name", "John");
// Create the document data to be updated
BasicDBObject updateDoc = new BasicDBObject("$set", new BasicDBObject("age", 31));
// Execute the update operation
collection.update(query, updateDoc);

Deleting Documents

Finally, you may delete documents from a collection using the 

          
           
DBCollection#remove()
method. This method accepts a 

          
           
com.mongodb.BasicDBObject

collection.remove(query);
