org.bukkit.event.inventory.InventoryClickEvent

The 

          
           
org.bukkit.event.inventory.InventoryClickEvent
is a Bukkit event that is fired when an inventory is clicked. This event is part of the Bukkit Inventory API and can be used to create custom inventory systems for your game.

Overview

The 

          
           
org.bukkit.event.inventory.InventoryClickEvent
is triggered when a player clicks on an inventory item. It allows developers to know what slot in the inventory was clicked, and the type of click (right-click, left-click, middle-click). The event also fires when an item is moved within an inventory, or when an item is taken out of an inventory.

Usage

Using the 

          
           
org.bukkit.event.inventory.InventoryClickEvent
requires registering a listener to listen for the event. The following code shows how to register a listener for the event: 

          
           
@EventHandler
public void onInventoryClick(InventoryClickEvent event) {
    // handle inventory click event
}
After registering the listener, the event will fire every time a player clicks on an inventory item. The following code shows how to get the slot number of the item that was clicked: 

          
           
int slotNumber = event.getSlot();
The following code shows how to get the type of click (right-click, left-click, middle-click): 

          
           
ClickType clickType = event.getClick();
The following code shows how to get the item that was clicked: 

          
           
ItemStack clickedItem = event.getCurrentItem();

Example

The following example shows how to create a custom inventory that displays the numbers from 0 to 9. When the player clicks on a number, the number is displayed in the chat: 

          
           
public class MyPlugin extends JavaPlugin {

    @Override
    public void onEnable() {
        // Register the event listener
        getServer().getPluginManager().registerEvents(new MyListener(), this);
    }

    // Event listener class
    public class MyListener implements Listener {

        @EventHandler
        public void onInventoryClick(InventoryClickEvent event) {
            // Get the clicked item
            ItemStack clickedItem = event.getCurrentItem();

            // Check if the clicked item is a number
            if (clickedItem.getType() == Material.CLAY_BALL) {
                // Get the number from the item name
                int number = Integer.parseInt(clickedItem.getItemMeta().getDisplayName());

                // Send message to player
                Player player = (Player) event.getWhoClicked();
                player.sendMessage("You clicked the number " + number);
            }
        }
    }
}
In order for the example to work, you need to create an inventory with the numbers from 0 to 9, and set their display names to the numbers.
