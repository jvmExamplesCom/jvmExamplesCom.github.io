javax.activation.MimetypesFileTypeMap

The javax.activation.MimetypesFileTypeMap class provides an implementation of the javax.activation.FileTypeMap interface that determines the MIME type of files based on their filename extensions. This class allows applications to use the same file type determination mechanism as the Java Activation Framework (JAF).

Creating a MimetypesFileTypeMap Object

Creating a MimetypesFileTypeMap object is simple. The following code creates and initializes a MimetypesFileTypeMap object:


          
           
MimetypesFileTypeMap mft = new MimetypesFileTypeMap();
mft.addMimeTypes(“text/plain txt”);
mft.addMimeTypes(“image/gif gif”);
mft.addMimeTypes(“image/jpeg jpg jpeg”);

Getting the MIME Type of a File

Once the MimetypesFileTypeMap object is initialized, you can use it to get the MIME type of a file. This is done by calling the MimetypesFileTypeMap.getContentType() method, passing in the name of the file to examine. For example:


          
           
File f = new File(“test.jpg”);
String mimeType = mft.getContentType(f);   // mimeType is “image/jpeg”

You can also pass a filename to the getContentType() method, rather than a File object. The following code is equivalent to the preceding:


          
           
String mimeType = mft.getContentType(“test.jpg”);   // mimeType is “image/jpeg”

MimetypesFileTypeMap Properties File

By default, the MimetypesFileTypeMap uses the following file to determine MIME types:


          
           
lib/mime.types

This file is usually located in the JRE's lib directory. You can also specify an alternative file to use if desired. To do so, set the following system property before creating any MimetypesFileTypeMap objects:


          
           
System.setProperty(“java.activation.mimetypesfiletypemap.mimetypes”, “/path/to/alt/mime.types”);

The MIME types file consists of one or more lines where each line specifies a mapping between a MIME type and a list of associated filename extensions. For example:


          
           
text/plain    txt
image/gif      gif
image/jpeg     jpg jpeg

Conclusion

The javax.activation.MimetypesFileTypeMap class provides a convenient way to determine the MIME type of a file based on its filename extension. By default, it reads its information from the system-provided mime.types file; but it can also be configured to use a custom file instead.
