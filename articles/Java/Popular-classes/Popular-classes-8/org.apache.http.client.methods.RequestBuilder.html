org.apache.http.client.methods.RequestBuilder

The org.apache.http.client.methods.RequestBuilder class is used to create an HttpUriRequest instance, which represents a request to an HTTP server. It uses a chain-style API in which each method returns an identical new instance allowing for the convenient chaining of calls. For example:


          
           
HttpUriRequest request = RequestBuilder.get()
     .setUri("https://www.example.com")
     .addHeader("Accept", "application/json")
     .build();

The RequestBuilder class can also be used to create POST, PUT or DELETE requests.

Creating a POST Request

To create a POST request with a body, we use the post() method, followed by the setEntity() method with a StringEntity containing the body of the request. For example:


          
           
HttpUriRequest request = RequestBuilder.post()
     .setUri("https://www.example.com")
     .addHeader("Content-Type", "application/json")
     .setEntity(new StringEntity("{\"name\":\"Foo\"}"))
     .build();

Creating a PUT Request

To create a PUT request with a body, we use the put() method, followed by the setEntity() method with a StringEntity containing the body of the request. For example:


          
           
HttpUriRequest request = RequestBuilder.put()
     .setUri("https://www.example.com")
     .addHeader("Content-Type", "application/json")
     .setEntity(new StringEntity("{\"name\":\"Foo\"}"))
     .build();

Creating a DELETE Request
