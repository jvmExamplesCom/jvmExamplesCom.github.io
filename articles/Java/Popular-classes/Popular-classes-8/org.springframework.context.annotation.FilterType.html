Java org.springframework.context.annotation.FilterType

Java org.springframework.context.annotation.FilterType is an interface in the Spring Framework that provides different types of filtering options for components in the application context.

Code Example 1

This code example shows how to configure a filter type:


          
           
@FilterType(
    type = FilterType.Type.ANNOTATION, 
    classes = {MyAnnotation.class})
public class MyComponent {
    ...
}

Code Example 2

This code example shows how to configure multiple filter types:


          
           
@FilterTypes({
    @FilterType(
        type = FilterType.Type.ANNOTATION, 
        classes = {MyAnnotation1.class}),
    @FilterType(
        type = FilterType.Type.ASSIGNABLE_TYPE, 
        classes = {MyInterface.class})
})
public class MyComponent {
    ...
}

Code Example 3

This code example shows how to configure a filter type with multiple classes:


          
           
@FilterType(
    type = FilterType.Type.ANNOTATION, 
    classes = {MyAnnotation1.class, MyAnnotation2.class})
public class MyComponent {
    ...
}

Code Example 4

This code example shows how to configure a filter type with custom filters:


          
           
@FilterType(
    type = FilterType.Type.CUSTOM, 
    classes = {MyFilter1.class, MyFilter2.class})
public class MyComponent {
    ...
}

The output of this configuration is a list of components that match the given filter criteria.
