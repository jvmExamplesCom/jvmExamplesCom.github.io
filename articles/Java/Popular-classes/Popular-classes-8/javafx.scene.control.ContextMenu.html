Introduction to JavaFX Scene Control ContextMenu

The 

          
           
javafx.scene.control.ContextMenu
class provides a contextual menu which appears when the user right-clicks on the application window. This menu typically contains items that are relevant or related to the current context. The ContextMenu class is derived from javafx.scene.control.Menu class, and also it is similar to javafx.scene.control.PopupMenu. Moreover, the ContextMenu class has various methods to manage its content and implementation.

Creating a ContextMenu Component in JavaFX

In this section, we’ll learn how to create a ContextMenu component in JavaFX with a few code examples.

The ContextMenu class constructor creates an empty context menu instance. The 

MenuItem
object can be added to a 
ContextMenu
through the 
getItems()
method. Here is an example of creating a ContextMenu: 

          
           
ContextMenu contextMenu = new ContextMenu();
MenuItem item1 = new MenuItem("Item 1");
MenuItem item2 = new MenuItem("Item 2");
contextMenu.getItems().addAll(item1, item2);
In the above code, we first created a 
ContextMenu
instance named 
contextMenu
. Then, we created two 
MenuItem
objects and added them to the 
contextMenu
using the 
getItems()
method.

We can also create the ContextMenu with menu items added directly to the constructor. Here is an example of creating a ContextMenu with menu items added directly to the constructor:


          
           
ContextMenu contextMenu = new ContextMenu(
    new MenuItem("Item 1"),
    new MenuItem("Item 2")
);
In the above code, we created a 
ContextMenu
instance and added two menu items directly to the constructor.

To add a 
ContextMenu
to a node in JavaFX, we use the 
setContextMenu()
