io.grpc.Channel

The io.grpc.Channel class provides an interface for communication between clients and services using gRPC. It encapsulates all of the functionality required to establish and maintain a secure gRPC connection. The primary use of a Channel is to create a stub, which can then be used to issue RPCs.

Creating a Channel

A Channel can be created in several ways: 

          
           
ManagedChannel channel = ManagedChannelBuilder
  .forAddress("localhost", 8080)
  .build();
or 

          
           
ManagedChannel channel = InProcessChannelBuilder
  .forName("myChannel")
  .usePlaintext()
  .build();

Using a Channel

Once a Channel has been created, it can be used to create a stub. This stub is used to make RPC calls on the service.


          
           
HelloServiceGrpc.HelloServiceBlockingStub stub = HelloServiceGrpc.newBlockingStub(channel);

You can also create an asynchronous stub for non-blocking calls:


          
           
HelloServiceGrpc.HelloServiceFutureStub futureStub = HelloServiceGrpc.newFutureStub(channel);

Closing a Channel

When you are done with the Channel, it should be shutdown to free any resources associated with it:


          
           
channel.shutdown();
