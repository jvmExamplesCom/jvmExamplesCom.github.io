org.apache.hadoop.hbase.HColumnDescriptor

The org.apache.hadoop.hbase.HColumnDescriptor is a class in the Apache HBase library. It is used to describe the column families of an HBase table and they are specific to HBase. The HColumnDescriptor class provides accessors and mutators to manipulate the contents of a column family.

Example Usage

Below is an example of how to create an HColumnDescriptor, setting all its values.


          
           
HTableDescriptor htd = new HTableDescriptor();
HColumnDescriptor hcd = new HColumnDescriptor(
  "cf".getBytes(),        // column family name
  1,                      // max version (1 for HBase 0.94)
  Compression.Algorithm.GZ,// compression algorithm
  true,                   // in-memory
  true,                   // block cache enabled
  8 * 1024 * 1024,        // blocksize
  false,                  // bloomfilter
  0,                      // time to live
  null                    // bloom filter parameters
);

htd.addFamily(hcd);
