The 
XmlMapper
class is part of the Jackson XML data format module that provides the capability to read and write JSON as XML. It acts like a bridge between the document object model (DOM) objects and streaming data formats such as XML, GZip, and Avro. It provides an easy mechanism for converting between XML and JSON.

The XmlMapper class can be used to easily create an instance of it by passing in a Java ObjectMapper object. This makes it easy to use the same configuration settings for both the XML and JSON formats.


          
           
ObjectMapper objectMapper = new ObjectMapper();
XmlMapper xmlMapper = new XmlMapper(objectMapper);

If needed, you can also pass in additional configurations, such as changing the character encoding, changing the tag name, or setting up how the root element should look like.

writeValue()
