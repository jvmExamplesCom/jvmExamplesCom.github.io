It acts like a bridge between the document object model (DOM) objects and streaming data formats such as XML, GZip, and Avro. It provides an easy mechanism for converting between XML and JSON.

ObjectMapper objectMapper = new ObjectMapper();
XmlMapper xmlMapper = new XmlMapper(objectMapper);
