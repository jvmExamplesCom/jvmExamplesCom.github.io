Using OutputFormat

Using org.dom4j.io.OutputFormat is straightforward. Here’s an example of how to use it to format a DOM4J Document.

String indent = "  ";
boolean omitDeclaration = false;
boolean omitEncoding = false;
String lineSeparator = System.getProperty("line.separator");
boolean trimText = false;

Document doc = new Document();
Element root = new Element("root");
doc.add(root);
root.setText("Hello World!");

// Create OutputFormat
OutputFormat outputFormat = new OutputFormat(encoding, indent, omitDeclaration, omitEncoding, lineSeparator, trimText);

// Set OutputFormat
OutputFormat.setDocument(doc);

// Serializer

xmlSerializer.write(doc);
