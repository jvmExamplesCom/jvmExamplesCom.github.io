Java.io.UTFDataFormatException

The Java.io.UTFDataFormatException is a checked exception which occurs when the data received does not follow the UTF-8 format or any of its subsets, such as UTF-16 or UTF-32.

It extends the IOException class, which indicates that an I/O related error has occurred. In order to handle this exception, the calling function must use try-catch blocks within the code.

Code Examples

Below are some examples of how to use the UTFDataFormatException.

    // some code here
} catch (IOException | UTFDataFormatException e) {
    e.printStackTrace();
}
