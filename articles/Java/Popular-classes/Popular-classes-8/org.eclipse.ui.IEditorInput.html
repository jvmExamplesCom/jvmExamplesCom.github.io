org.eclipse.ui.IEditorInput

getName
and 
getToolTipText
. The 
getName
method returns a name that will be displayed inside the editor. The 
getToolTipText
method returns a tooltip text that will be displayed if the mouse hovers over the editor's title tab.

The interface also defines a method for querying the type of the input that can be either one of the predefined types like 

FileEditorInput
, 
StringEditorInput

getEditorInputType

getPath
