Java.awt.KeyboardFocusManager

This class provides a mechanism for managing which Component gains the focus when the user uses the Tab key to traverse through the components of a Java application. It also provides a way for different focus traversals over the components of an application, depending on various conditions.

Types of Focus Traversal

The KeyboardFocusManager class supports two types of focus traversal: default and explicit. Default focus traversal is based on the order components are added to the application's window or container, and the explicit traversal is based on an explicit order of components specified by the developer.

Example of Default Focus Traversal


          
           
// create a new frame 
Frame frame = new Frame(); 
 
// add two buttons to the frame 
Button b1 = new Button("Button 1"); 
frame.add(b1); 
Button b2 = new Button("Button 2"); 
frame.add(b2); 
 
// set the frame visible 
frame.setVisible(true); 
 
// get the KeyboardFocusManager 
KeyboardFocusManager manager = KeyboardFocusManager.getCurrentKeyboardFocusManager(); 
 
// print out the initial focus owner 
System.out.println("Initial Focus Owner: " + manager.getFocusOwner()); 
 
// press the tab key 
manager.focusNextComponent(); 
 
// print out the new focus owner 
System.out.println("New Focus Owner: " + manager.getFocusOwner());
In this example, a frame with two buttons is created, and then the tab key is pressed to move the focus from one button to the other. The output of this code would be: 

          
           
Initial Focus Owner: Button 1
New Focus Owner: Button 2

Example of Explicit Focus Traversal

Explicit focus traversal allows the developer to specify an order of components for the focus to traverse. This is done by creating an array of Components. In the example below, an array of three components is created, and then an instance of the KeyboardFocusManager is configured to use the array for focus traversal. 

          
           
// create a new frame 
Frame frame = new Frame(); 
 
// add three buttons to the frame 
Button b1 = new Button("Button 1"); 
frame.add(b1); 
Button b2 = new Button("Button 2"); 
frame.add(b2); 
Button b3 = new Button("Button 3"); 
frame.add(b3); 
 
// set the frame visible 
frame.setVisible(true); 
 
// create an array of components 
Component[] components = new Component[]{b1, b2, b3}; 
 
// get the KeyboardFocusManager 
KeyboardFocusManager manager = KeyboardFocusManager.getCurrentKeyboardFocusManager(); 
 
// set the explicit focus traversal policy 
manager.setExplicitFocusTraversal(components); 
 
// print out the initial focus owner 
System.out.println("Initial Focus Owner: " + manager.getFocusOwner()); 
 
// press the tab key 
manager.focusNextComponent(); 
 
// print out the new focus owner 
System.out.println("New Focus Owner: " + manager.getFocusOwner());
The output of this code would be: 

          
           
Initial Focus Owner: Button 1
New Focus Owner: Button 2

Conclusion

The Java.awt.KeyboardFocusManager class provides a way for developers to manage the focus of components in a Java application. It supports two types of focus traversal: default and explicit, and can be used to set up custom focus traversal policies.
