org.eclipse.swt.events.Key Event

The org.eclipse.swt.events.KeyEvent class is an event that describes a key pressed or released by the user. It is part of the SWT (Standard Widget Toolkit) package. The org.eclipse.swt.events.KeyEvent class allows developers to capture and respond to keyboard input in Java applications.

The code below shows how to set up keyboard shortcuts for a user interface element. 


          
           
Button myButton = new Button(shell, SWT.PUSH);
myButton.addKeyListener(new KeyAdapter() {
   public void keyPressed(KeyEvent e) {
        if (e.character == 'a') {

        }
   }
});

Example: Capturing Keyboard Events

The following example shows how to capture and respond to a user pressing a key. 


          
           
public void keyPressed(KeyEvent e) {
    switch (e.keyCode) {
        case SWT.ARROW_DOWN:
            System.out.println("arrow down pressed");
            break;
        case SWT.ARROW_UP:
            System.out.println("arrow up pressed");
            break;
    }
}

    }
