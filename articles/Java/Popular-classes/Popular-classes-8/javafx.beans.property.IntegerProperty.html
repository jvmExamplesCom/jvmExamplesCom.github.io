JavaFX Beans Property IntegerProperty

JavaFX provides a set of classes for dealing with properties and binding. The javafx.beans.property.IntegerProperty class is part of this package. IntegerProperty is a subclass of ReadOnlyIntegerProperty that provides methods to modify the value like setValue(). An IntegerProperty can be bound to other properties or be invalidated if the value changes in any way.

Creating an IntegerProperty

An IntegerProperty can be created using the classes constructor. 

          
           
IntegerProperty intProp = new IntegerProperty();
Alternatively, you can use the static factory method: 

          
           
IntegerProperty intProp = IntegerProperty.integerProperty(initialValue);
Where initialValue is the initial value of the IntegerProperty.

Setting the value of an IntegerProperty

The setValue() method sets the value of the IntegerProperty. 

          
           
intProp.setValue(newValue);
Where newValue is the new value of the IntegerProperty.

Binding an IntegerProperty to another Property

The bind() method binds the IntegerProperty to another property. 

          
           
intProp.bind(otherProp);
Where otherProp is the property to which the IntegerProperty will be bound.

Unbinding an IntegerProperty from another Property

The unbind() method unbinds the IntegerProperty from another property. 

          
           
intProp.unbind();

Invalidating an IntegerProperty

The invalidate() method invalidates the IntegerProperty. 

          
           
intProp.invalidate();
This causes any bound properties to become unbound and have their current value evaluated.
