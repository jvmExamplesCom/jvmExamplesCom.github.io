Introduction to android.widget.SectionIndexer

Android’s SectionIndexer class is a helpful tool for developers when developing UI elements for Android applications. It provides an interface between the application’s UI and its underlying data structure. This allows the developer to easily create a view that displays the contents of a list of items with ease. The SectionIndexer class is heavily used in ListViews and helps developers quickly create easily browsed lists.

Example Code

The following example code shows how to use an instance of SectionIndexer to create a ListView showing categories cities:


          
           
//Get a section indexer class instance
SectionIndexer sectionIndexer = new SectionIndexer();

//Add the cities to the indexer
sectionIndexer.addCity("New York", "NY");
sectionIndexer.addCity("Los Angeles", "LA");
sectionIndexer.addCity("Chicago", "IL");

//Create an adapter for the ListView
ArrayAdapter adapter = new ArrayAdapter(this, android.R.layout.simple_list_item_1, sectionIndexer.getCities());

//Set the adapter for the ListView
listview.setAdapter(adapter);

The above example code will create a ListView filled with the cities added to the SectionIndexer instance. Any changes made to the collection of cities will be automatically reflected in the ListView.
