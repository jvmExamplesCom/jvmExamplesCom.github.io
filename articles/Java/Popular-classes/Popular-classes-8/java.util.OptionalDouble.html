java.util.OptionalDouble

Java.util.OptionalDouble is a class of the Java API used to represent a double-type value that may or may not be present. It allows developers to work with optional values instead of the regular primitive or wrapper type, providing an alternative for working with nulls.

Constructors

The OptionalDouble class has two different constructors that allow its user to create instances. 


          
           
public OptionalDouble()
public static OptionalDouble of(double val)

The first constructor creates an empty OptionalDouble instance in which its value is absent, while the second one allows creating an OptionalDouble instance initialized with the value provided in its argument.

Methods

The class provides several methods allowing users to manipulate OptionalDouble instances. 


          
           
public boolean isPresent()
public double getAsDouble()
public double orElse(double other)
public void ifPresent(DoubleConsumer block)

The 


          
           
<strong>isPresent()</strong>
method is used to check whether the value is present or not. If it is present, it returns 

          
           
true
. Otherwise, it returns 

          
           
false
.

The 


          
           
<strong>getAsDouble()</strong>
method is used to retrieve the value from the OptionalDouble instance if it is present. If not, it throws a

.

The 


          
           
<strong>orElse(double other)</strong>
method is used to obtain the value from the OptionalDouble instance if it is present. Otherwise, it returns the 

          
           
other
value provided as an argument.

Finally, the 


          
           
<strong>ifPresent(DoubleConsumer block)</strong>
method is used to execute a piece of code, given as an argument, if the value is present. Otherwise, nothing happens.

Examples

Let’s see some examples of how to use all these methods.


          
           
public static void main(String[] args) {

    // Create an OptionalDouble
    OptionalDouble opt = OptionalDouble.of(10); 

    // Check if the value is present
    if (opt.isPresent()) { 
        System.out.println("Value is present!"); 
    } 

    // Get the value
    System.out.println(opt.getAsDouble()); 

    // Get the value or else use a default one
    System.out.println(opt.orElse(5.0)); 

    // Execute a block of code

}
