Eclipse SWT Widgets Menu

The org.eclipse.swt.widgets.Menu class is part of the Eclipse Standard Widget Toolkit (SWT), and its purpose is to create menus for the user interface of an application.

A menu is a hierarchical list of items, where each item can be associated with a command that is executed when it is selected. A menu can contain one or more sub-menus, which are called cascade menus. There can also be multiple menus open at once on a given window.

Adding a Cascade Menu

To add a cascade menu to the menu, you need to create a 

Menu
object and then set it as the parent of the menu item that will act as the cascade menu. Here is an example of creating a cascade menu with two items:

MenuItem item3 = new MenuItem(cascadeMenu, SWT.PUSH);
item3.setText("Item 3");
MenuItem item4 = new MenuItem(cascadeMenu, SWT.PUSH);
