com.google.auth.oauth2.GoogleCredentials

GoogleCredentials is a Google Client Library for Java that handles the authorization flow and consent dialogue needed to access user data using a variety of services, including Google Cloud Platform, Google APIs, and other third-party APIs.

Getting Started

To use the GoogleCredentials library, you will need to first set up a project in the Google Developer Console to get an OAuth2 Client ID, which is used to grant access to your application's resources.

Once you have created a project in the Google Developer Console, you can add credentials for it by navigating to APIs & Services > Credentials.

Once you have obtained your OAuth2 Client ID, you can begin using the GoogleCredentials library in your Java application.

Using GoogleCredentials in Your Application

The GoogleCredentials class is the main class used to provide authentication credentials when making requests to Google APIs. It makes use of the OAuth2ClientCredential class to store the Client ID and secret associated with the application.


          
           
GoogleCredentials credentials = GoogleCredentials.getApplicationDefault();
OAuth2ClientCredentials clientCredentials = new OAuth2ClientCredentials(
    "YOUR_CLIENT_ID",
    "YOUR_CLIENT_SECRET";
credentials.setClientSecret(clientCredentials);

The API request can then be made with the authenticated credentials:


          
           
HttpRequestFactory requestFactory = 
credentials.getHttpTransport().createRequestFactory();
GenericUrl genericUrl = new GenericUrl("https://www.googleapis.com/urlshortener/v1/url");
HttpRequest request = requestFactory.buildGetRequest(genericUrl);
request.setParser(new com.google.api.client.json.jackson2.JacksonFactory().createJsonObjectParser());
request.getHeaders().setAuthorization("Bearer " + credentials.getAccessToken());
HttpResponse response = request.execute();

The response object contains the data returned from the request.

Conclusion

The GoogleCredentials library makes it easy to add authentication to your Java applications and make requests to Google APIs. With just a few lines of code, you can get up and running quickly with OAuth2 and Google APIs.
