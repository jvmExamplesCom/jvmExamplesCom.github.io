org.apache.hadoop.mapreduce.RecordWriter

org.apache.hadoop.mapreduce.RecordWriter is an important class in the MapReduce API of Hadoop. It's responsible for writing key-value pairs to an OutputFormat. It defines an abstract method, write(K key, V value), which must be implemented by subclasses in order to specify an output format.

Example Code:

The following example shows how to use org.apache.hadoop.mapreduce.RecordWriter to write a record to a text file:


          
           
Configuration conf = new Configuration(); 
Job job = new Job(conf); 
job.setOutputFormatClass(TextOutputFormat.class); 
FileOutputFormat.setOutputPath(job, outputPath); 
RecordWriter<K, V> writer = job.getOutputFormat().getRecordWriter(context); 
writer.write(key, value); 
writer.close();

This example writes the key-value pair to the output file specified in the configuration. The key and value are written using the TextOutputFormat. The RecordWriter is obtained by calling the getRecordWriter() method on the OutputFormat instance.
