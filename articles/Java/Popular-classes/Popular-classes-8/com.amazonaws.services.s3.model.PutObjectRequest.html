com.amazonaws.services.s3.model.PutObjectRequest

The 

          
           
com.amazonaws.services.s3.model.PutObjectRequest
class is used to upload an object to an Amazon S3 bucket. It contains methods for setting the object's metadata, server-side encryption, ACL, and other properties.

Code Examples

The following code examples demonstrate common use cases of the 

PutObjectRequest
class.

Uploading an Object from a File


          
           
AmazonS3 s3 = AmazonS3ClientBuilder.standard().build();

String bucketName = "my-bucket";
File objectToUpload = new File("my-object.txt");
String key = "my-object.txt";

PutObjectRequest request = new PutObjectRequest(bucketName, key, objectToUpload);
s3.putObject(request);

Setting Metadata with PutObjectRequest


          
           
PutObjectRequest request = new PutObjectRequest(bucketName, key, objectToUpload)
        .withMetadata(ImmutableMap.of("my-key", "my-value"));
s3.putObject(request);

Setting Server-Side Encryption with PutObjectRequest


          
           
PutObjectRequest request = new PutObjectRequest(bucketName, key, objectToUpload)
        .withSSEAlgorithm(ObjectMetadata.AES_256_SERVER_SIDE_ENCRYPTION);
s3.putObject(request);

Setting Access Control List with PutObjectRequest


          
           
AccessControlList acl = new AccessControlList();
acl.grantPermission(new CanonicalGrantee("userId"), Permission.FullControl);
PutObjectRequest request = new PutObjectRequest(bucketName, key, objectToUpload)
        .withAccessControlList(acl);
s3.putObject(request);

Setting Bucket Tagging with PutObjectRequest


          
           
PutObjectRequest request = new PutObjectRequest(bucketName, key, objectToUpload)
        .withTagging(TagSet.fromTags(new Tag("my-tag", "my-value")));
s3.putObject(request);
