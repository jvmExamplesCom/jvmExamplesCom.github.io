com.baomidou.mybatisplus.core.mapper.BaseMapper

The com.baomidou.mybatisplus.core.mapper package provides a BaseMapper interface which is extremely useful for efficient object mapping in Java applications. With the help of this interface, developers can easily perform data manipulation tasks such as saving, updating, deleting, and querying objects.

BaseMapper Interface

The BaseMapper interface provides several important methods to facilitate the mapping of objects to the database. These include:

  • save(): This method saves or inserts an entity into the database.
  • update(): This method updates an entity in the database.
  • delete(): This method deletes an entity from the database.
  • findById(): This method queries an entity by its primary key.
  • findList(): This method queries multiple entities based on criteria.

user.setAge(25);
