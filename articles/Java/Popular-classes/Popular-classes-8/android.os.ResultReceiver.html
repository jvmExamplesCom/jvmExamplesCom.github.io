Introduction to android.os.ResultReceiver

The android.os.ResultReceiver class is part of the Android software development kit (SDK), and provides a means to handle asynchronous results from communication with remote services. It receives data in response to an intent request, and can be used to communicate between activities, services, and other components within an application.

Basic Usage of ResultReceiver

The following code shows the basic usage of the ResultReceiver class. Here, we are using an intent to start a service and sending a result back to the activity.


          
           
Intent intent = new Intent(this, MyService.class);
intent.putExtra("resultReceiver", new ResultReceiver(new Handler()) {
    @Override
    protected void onReceiveResult(int resultCode, Bundle resultData) {
        if (resultCode == 1) {
            String result = resultData.getString("result");
            // do something with the result
        }
    }
});
startService(intent);

In the above example, we have defined an instance of ResultReceiver that receives the result code 1 and the associated data. The result code and the data are passed as arguments to the 

onReceiveResult()
method.
