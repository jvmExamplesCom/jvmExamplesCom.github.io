javax.script.CompiledScript

The 

          
           
javax.script.CompiledScript
is a class in the Java Scripting API that allows scripts to be compiled and executed as one atomic operation. It provides a means for scripts to be pre-compiled, then executed multiple times without recompiling. This eliminates the need to compile and link each time a script is run, thus providing performance benefits.

Usage

To use the 

          
           
CompiledScript
class, the user must first create an instance of it. This can be done by calling the 

          
           
ScriptEngine.compile()
method on the appropriate 

          
           
ScriptEngine
instance. The compiler will then create the CompiledScript instance which can be used to execute the script multiple times.

Example Code


          
           
ScriptEngine engine = new ScriptEngineManager().getEngineByName("javascript");
CompiledScript script = (CompiledScript) engine.compile("print('Hello World!');");
script.eval();
The above code creates a new JavaScript 

          
           
ScriptEngine
, compiles a script which prints out the message “Hello World!” and then runs it using the 

          
           
eval()
method.
