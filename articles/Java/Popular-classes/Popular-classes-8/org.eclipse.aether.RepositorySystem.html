org.eclipse.aether.RepositorySystem
facilitates the *resolution* of artifacts and their transitive dependencies from a set of repositories. It allows Java developers to easily and reliably use external libraries in their projects. The repository system is plugin-based, allowing custom implementations to be plugged in as needed.

org.eclipse.aether.RepositorySystem
, you must first add a dependency to your project. Add the following dependency to your project's

org.eclipse.aether.RepositorySystem
