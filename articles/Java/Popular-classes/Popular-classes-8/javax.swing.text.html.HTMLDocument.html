javax.swing.text.html.HTMLDocument

javax.swing.text.html.HTMLDocument class is the HTML parser and lexer implementation in Java Swing. It is used to parse strings containing HTML content and create a document tree (DOM) of elements, which can then be manipulated. This class is intended to be used when loading HTML documents from the network or from the file system. It has the capability of creating a DOM structure of the parsed string and provides API for manipulating it further. The API includes methods to insert and remove elements, add attributes to elements, update the document with new content, etc.

Code Examples

Here are some example code snippets that demonstrate how to use javax.swing.text.html.HTMLDocument.

Reading HTML from File


          
           
try {
  File file = new File("sample.html");
  URL url = file.toURI().toURL();
  HTMLDocument doc = (HTMLDocument) new HTMLReader(url).read();
} catch (Exception e) {
  System.out.println(e);
}

Insert New Element


          
           
Element element = doc.createElement("div");
doc.getElementsByTagName("body").item(0).appendChild(element);

Insert Attribute


          
           
Element element = doc.getElementsByTagName("div").item(0);
element.setAttribute("id", "myDiv");

Update Content


          
           
Element element = doc.getElementsByTagName("div").item(0);
element.setTextContent("This is my div content");
