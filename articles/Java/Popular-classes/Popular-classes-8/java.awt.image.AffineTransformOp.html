java.awt.image.AffineTransformOp

The java.awt.image.AffineTransformOp class is a filter that performs an affine transformation on an image. An affine transformation is a linear mapping from one space to another, preserving points, straight lines, and planes. It allows scaling, rotation, translation, and shear of an image.
