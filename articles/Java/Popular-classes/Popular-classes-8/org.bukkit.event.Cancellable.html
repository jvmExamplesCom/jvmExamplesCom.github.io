org.bukkit.event.Cancellable

The org.bukkit.event.Cancellable interface defines an event that can be cancelled by its listeners. This means that when a Cancellable is fired, all the listeners who are registered to it will be called in turn and given the opportunity to cancel the event.

Example 1 - Basic Usage of Cancellable

Here is an example of how to create a Cancellable event:


          
           
MyCancellableEvent event = new MyCancellableEvent();
Bukkit.getPluginManager().callEvent(event);
if (!event.isCancelled()) {
    // Do something if the event isn't cancelled
}

The above code creates a new instance of the MyCancellableEvent class, which implements the Cancellable interface. Then the event is fired with the Bukkit plugin manager, and if it isn't cancelled, we can do something with it.

Example 2 - Registering Listeners to a Cancellable

We can register listeners to a Cancellable either inside the class that implements it, or outside. Here is an example of how to register a listener inside the class: 


          
           
public class MyCancellableEvent implements Cancellable {

    private boolean cancelled = false;

    @Override
    public void setCancelled(boolean cancelled) {
        this.cancelled = cancelled;
    }

    @Override
    public boolean isCancelled() {
        return cancelled;
    }

    public void registerListener() {
        Bukkit.getPluginManager().registerEvent(this, new Listener() {
            @EventHandler
            public void onCancellableEvent(MyCancellableEvent event) {
                // Do something when the event is fired
                event.setCancelled(true);
            }
        }, EventPriority.NORMAL, null, Bukkit.getPluginManager().getPlugin("myPlugin"));
    }
}

The above code shows how to register a listener for a Cancellable inside the class that implements it. We can also register listeners outside the class like this: 


          
           
MyCancellableEvent event = new MyCancellableEvent();
Bukkit.getPluginManager().registerEvent(event, new Listener() {
    @EventHandler
    public void onCancellableEvent(MyCancellableEvent event) {
        // Do something when the event is fired
        event.setCancelled(true);
    }
}, EventPriority.NORMAL, null, Bukkit.getPluginManager().getPlugin("myPlugin"));

Example 3 - Checking if the Event is Cancelled

Once our listeners have been registered, we can check if the event has been cancelled. This can be done by calling the isCancelled() method. Here is an example: 


          
           
MyCancellableEvent event = new MyCancellableEvent();
if (!event.isCancelled()) {
    // Do something if the event isn't cancelled
}

In the example above, if any of the registered listeners for the event have called the setCancelled() method, then the isCancelled() method will return true and the code inside the if statement will not be executed.
