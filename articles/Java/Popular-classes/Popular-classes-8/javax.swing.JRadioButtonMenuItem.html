javax.swing.JRadioButtonMenuItem

The 

          
           
javax.swing.JRadioButtonMenuItem
is a specialized type of a menu item in Java Swing that has the same capabilities as a regular menu item, but also provides an option for the user to select from a list or multiple choices. This type of menu item is similar to a radio button control in HTML, where a group of radio buttons can be activated or deactivated depending on the user's preference. The 

          
           
javax.swing.JRadioButtonMenuItem
provides the same functionality as a regular menu item, including being able to display text, icons, and other components. However, it also adds the ability to select only one item out of a list.

Creating a JRadioButtonMenuItem

The 

          
           
JRadioButtonMenuItem
class can be used to create a menu item with a list of choices by providing the text for each option and adding them as items to the menu. The following example creates a menu with three options: "A", "B" and "C": 

          
           
JRadioButtonMenuItem itemA = new JRadioButtonMenuItem("A");
JRadioButtonMenuItem itemB = new JRadioButtonMenuItem("B");
JRadioButtonMenuItem itemC = new JRadioButtonMenuItem("C");

JMenu menu = new JMenu("Options");
menu.add(itemA);
menu.add(itemB);
menu.add(itemC);

The menu can then be added to a UI component such as a menu bar or a popup menu in order to be displayed and used by the user.

Grouping Radio Buttons

In order to ensure that only one option can be selected at any given time, the radio buttons must be grouped together. This is done by creating a 

ButtonGroup
object and adding all of the radio buttons to it. For example, the previous example could be modified as follows: 

          
           
ButtonGroup group = new ButtonGroup();

JRadioButtonMenuItem itemA = new JRadioButtonMenuItem("A");
JRadioButtonMenuItem itemB = new JRadioButtonMenuItem("B");
JRadioButtonMenuItem itemC = new JRadioButtonMenuItem("C");

group.add(itemA);

group.add(itemC);

JMenu menu = new JMenu("Options");
menu.add(itemA);
menu.add(itemB);
menu.add(itemC);

Now only one of the options can be selected at any given time; if a different option is selected, the previously selected option will become deselected.

Setting the Default Selection

The default selection of a 

          
           
JRadioButtonMenuItem
can be changed by calling the 
setSelected()
method on the item which should be selected by default. For example, the following code will select the "B" option as the default: 

          
           
itemB.setSelected(true);

Listening for Selection Changes

In order to detect when a selection has been made by the user, a 

ChangeListener
can be added to the 

          
           
JRadioButtonMenuItem
. The change listener will be invoked whenever the selection is changed by the user. The following example adds a change listener to the previous example in order to print out a message when the selection is changed. 

          
           
itemA.addChangeListener(e -> System.out.println("A was selected"));
itemB.addChangeListener(e -> System.out.println("B was selected"));

When the user selects one of the options, a message will be printed to the console indicating which option was selected.
