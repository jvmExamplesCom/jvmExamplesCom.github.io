Overview of org.eclipse.aether.artifact.DefaultArtifact

org.eclipse.aether.artifact.DefaultArtifact is an embedded class in the Eclipse Aether project which provides a complete implementation of the Artifact interface, an abstraction for files and other entities that can be used as part of a build process. This class handles the parsing and manipulation of a Maven-style artifact coordinates from a text string and provides methods to access the components of the artifact.

DefaultArtifact artifact = new DefaultArtifact(artifactCoordinates);
String groupId = artifact.getGroupId(); // groupId = "org.apache.maven"
