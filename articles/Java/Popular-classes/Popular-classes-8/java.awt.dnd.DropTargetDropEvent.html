java.awt.dnd.DropTargetDropEvent

The java.awt.dnd.DropTargetDropEvent class provides an event which is generated whenever a drop occurs over the component where the DragGestureRecognizer has detected a drag gesture. This type of event is sent regardless of whether a drop was accepted or not.

Constructors

  • 
            
             
DropTargetDropEvent(DropTargetContext dtc)
    : Creates a DropTargetDropEvent given the specified DropTargetContext.
  • 
            
             
DropTargetDropEvent(DropTargetContext dtc, Point cursorLocn, int dropAction, int srcActions)
    : Create a DropTargetDropEvent given the specified DropTargetContext, the location of the "drag" cursor's hotspot within the component's coordinates and the currently selected user drop action and source drop actions.

Methods

  • 
            
             
acceptDrop(int dropAction)
    : Accepts the drop which is currently being performed and sets the current drop action.
  • getDropAction()
    : Returns the currently selected drop action.
  • getLocation()
    : Returns the current location of the mouse cursor.
  • getSourceActions()
    : Returns the source drop actions.
  • rejectDrop()
    : Rejects the drop which is currently being performed.
  • 
            
             
dropComplete(boolean success)
    : Notifies the peer that the drop transfer is completed.

Example


          
           
import java.awt.datatransfer.DataFlavor;
import java.awt.datatransfer.Transferable;
import java.awt.dnd.DnDConstants;
import java.awt.dnd.DropTarget;
import java.awt.dnd.DropTargetDragEvent;
import java.awt.dnd.DropTargetDropEvent;
import java.awt.dnd.DropTargetEvent;
import java.awt.dnd.DropTargetListener;

public class SimpleDropTargetListener implements DropTargetListener { 

    public void dragEnter(DropTargetDragEvent dtde) { 
        System.out.println("Drop Target Drag Entered");
    } 

    public void dragOver(DropTargetDragEvent dtde) { 
        System.out.println("Drop Target Drag Over");
    } 

    public void dropActionChanged(DropTargetDragEvent dtde) { 
        System.out.println("Drop Target Action Changed");
    } 

    public void dragExit(DropTargetEvent dte) { 
        System.out.println("Drop Target Exit");
    } 

    public void drop(DropTargetDropEvent dtde) { 
        System.out.println("Drop");
        try { 
            Transferable transferable = dtde.getTransferable();

            if (transferable.isDataFlavorSupported(DataFlavor.stringFlavor)) { 
                dtde.acceptDrop(DnDConstants.ACTION_COPY); 
                String data = (String) transferable.getTransferData(DataFlavor.stringFlavor); 
                System.out.println("Data is " + data); 
            } 
            else { 
                dtde.rejectDrop();
            } 
        } 
        catch (Exception e) { 
            e.printStackTrace(); 
            dtde.rejectDrop();
        }
    } 
}

In the above example, we are creating a class named SimpleDropTargetListener which implements the DropTargetListener interface. This class contains several methods which handle events such as 

dragEnter()
, 
dragOver()
, 
dropActionChanged()
, 
dragExit()
, and 
drop()
. The 
drop()
method is used to check the type of the data which is being transferred and process it accordingly. The 
acceptDrop()
and 
rejectDrop()
methods are used to accept or reject the drop transfer. Finally, the 
dropComplete()
method is used to notify the peer that the drop transfer is completed successfully.
