javax.sound.sampled.DataLine

javax.sound.sampled.DataLine is a class in the Java Sound API that supports capturing and playing back audio data. It is an abstract class, meaning it provides a basic implementation of many methods to be implemented by subclasses.

Example

Here's an example of using DataLine to capture and play back audio data:


          
           
import javax.sound.sampled.AudioFormat;
import javax.sound.sampled.AudioSystem;
import javax.sound.sampled.DataLine;
import javax.sound.sampled.LineUnavailableException;
import javax.sound.sampled.SourceDataLine;
import javax.sound.sampled.TargetDataLine;

public class AudioCapture {
    public static void main(String[] args) {
        try {
            AudioFormat format = new AudioFormat(44100.0f, 16, 1, true, true);
            DataLine.Info targetInfo = new DataLine.Info(TargetDataLine.class, format);
            TargetDataLine targetLine = (TargetDataLine) AudioSystem.getLine(targetInfo);
 
            DataLine.Info sourceInfo = new DataLine.Info(SourceDataLine.class, format);
            SourceDataLine sourceLine = (SourceDataLine) AudioSystem.getLine(sourceInfo);
 
            targetLine.open(format);
            targetLine.start();
 
            sourceLine.open(format);
            sourceLine.start();
 
            int bytesRead;
            byte[] targetData = new byte[targetLine.getBufferSize() / 5];
 
            while (true) {
                // read data from the target data line
                bytesRead = targetLine.read(targetData, 0, targetData.length);
                if (bytesRead == -1) {
                    break;
                }
 
                // write data to the source data line
                sourceLine.write(targetData, 0, bytesRead);
            }
 
            targetLine.close();
            sourceLine.drain();
            sourceLine.close();
        } catch (LineUnavailableException e) {
            e.printStackTrace();
        }
    }
}

The example opens a TargetDataLine and a SourceDataLine. The TargetDataLine is used to capture audio data which is then written to the SourceDataLine to be played back. The application will then loop indefinitely, reading data from the TargetDataLine and writing it to the SourceDataLine.

The DataLine class provides a convenient way to capture and play back audio data. It offers several methods for retrieving information about lines and controlling audio capture and playback.
