org.jboss.shrinkwrap.api.spec.JavaArchive

The org.jboss.shrinkwrap.api.spec.JavaArchive is a powerful API that can be used to create Java archives (JARs) easily and quickly. It is part of the JBoss ShrinkWrap project, and is designed to help developers streamline the creation of Java Archives for deployment in any environment.

Creating a Simple Java Archive

Creating a basic archive is an easy task, one that requires just a few lines of code:


          
           
JavaArchive archive = ShrinkWrap.create(JavaArchive.class);
archive.addClass(MyClass.class);
archive.addAsManifestResource("my-config.xml");

The first line will create a new instance of JavaArchive, which is the main entry point into this API. The second line adds the MyClass class to the archive, while the third line adds the my-config.xml file as a manifest resource.

Building an Executable Jar

To build an executable Jar with main class, you can use the following code:


          
           
JavaArchive archive = ShrinkWrap.create(JavaArchive.class);
archive.addClass(MyClass.class);
archive.addAsManifestResource("my-config.xml");
archive.addAsManifestResource(EmptyAsset.INSTANCE, "main-class", "Main-Class: MyClass");

Working With Resources

archive.addAsResource("my-file.txt");

Finally, you can add an entire directory of resources with the addAsResources() method:
