Creating a Simple Java Archive

JavaArchive archive = ShrinkWrap.create(JavaArchive.class);
archive.addClass(MyClass.class);
archive.addAsManifestResource("my-config.xml");

The first line will create a new instance of JavaArchive, which is the main entry point into this API. The second line adds the MyClass class to the archive, while the third line adds the my-config.xml file as a manifest resource.

JavaArchive archive = ShrinkWrap.create(JavaArchive.class);
