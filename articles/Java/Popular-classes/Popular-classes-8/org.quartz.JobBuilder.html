org.quartz.JobBuilder

The 

          
           
org.quartz.JobBuilder
class provides APIs for constructing a 

          
           
JobDetail

Creating a JobDetail Object

The following code shows how to use the 

          
           
JobBuilder
to create a 

          
           
JobDetail
instance with default settings: 

          
           
JobDetail job = JobBuilder.newJob(MyJob.class)
    .withIdentity("myJob")
    .build();
In this example, 

          
           
MyJob
is a class that implements the 

          
           
Job
interface. The 

          
           
withIdentity
method sets the job's unique identity.

usingJobData

JobDetail

JobDetail job = JobBuilder.newJob(MyJob.class)

JobDetail job = JobBuilder.newJob(MyJob.class)
