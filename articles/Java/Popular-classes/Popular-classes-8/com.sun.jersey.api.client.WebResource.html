com.sun.jersey.api.client.WebResource

The com.sun.jersey.api.client.WebResource class represents a resource at a given URL that can be used to interact with the server. It allows clients to interact with the server-side methods in different ways depending on their needs. The class provides multiple methods for sending HTTP requests, retrieving and manipulating data, and parsing the responses.

Creation of WebResource Instance:

A WebResource instance is created from a Client instance by invoking the 


          
           
resource(String uri)
method. This method takes a URI as parameter and returns a new WebResource instance for the specified resource identifier. 

          
           
Client client = Client.create();
WebResource wr = client.resource("http://example.com");

Making Requests and Retrieving Response

Once the instance of WebResource is created, a request can be made to the server by invoking any of the 


          
           
request()
methods of the WebResource class. Following are some of the commonly used methods. 
  • 
            
             
get(Class responseType)
    - This method makes a GET request to the server and returns a response object of the specified type.
  • 
            
             
post(Object requestEntity)
    - This method makes a POST request by sending the request entity as the body of the request message.
  • 
            
             
put(Object requestEntity)
    - This method makes a PUT request by sending the request entity as the body of the request message.
  • delete()
    - This method makes a DELETE request to the server.

Parsing Response

The response of the request can be parsed using one of the available 


          
           
Accept

XML xml = wr.accept("application/xml").get(XML.class);

header()

wr.header("Content-Type", "application/json");
