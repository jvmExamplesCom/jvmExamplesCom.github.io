Introduction to android.widget.ViewSwitcher

android.widget.ViewSwitcher is an Android View that allows users to quickly switch between two views. It is commonly used in applications that require a user to switch between two views, for example: switching between a list view, and an image view. This article will explain how to use the ViewSwitcher, with code examples to demonstrate each step.

Creating a ViewSwitcher

To create a ViewSwitcher, you must first set up a Layout. The Layout will contain the two views (Views 1 and 2) that you wish to switch between. Here is an example of setting up a simple ViewSwitcher using a FrameLayout:


          
           
<FrameLayout 
    android:layout_width="match_parent"
    android:layout_height="match_parent" >

    <View1 
        android:layout_width="match_parent"
        android:layout_height="match_parent" />

    <View2 
        android:layout_width="match_parent"
        android:layout_height="match_parent" />

</FrameLayout>

Once you have set up the Layout, you can then create the ViewSwitcher. When creating the ViewSwitcher, you need to provide two parameters: the two views you want to switch between (Views 1 and 2) and an initial view (View 1 by default). 


          
           
ViewSwitcher switcher = new ViewSwitcher(this, view1, view2);

Switching Views

Now that you have created the ViewSwitcher, you can start switching between the two views. To do this, you use the showNext() and showPrevious() methods.


          
           
switcher.showNext();     // Shows View 2
switcher.showPrevious(); // Shows View 1

These methods switch to the next (or previous) view in the ViewSwitcher. You can also use the showPrevious() method with a parameter to switch to any specific view in the ViewSwitcher.


          
           
switcher.showPrevious(view2); //Show View 2

Conclusion
