org.jdom.Document

The org.jdom.Document class is a representation of an XML document in Java. It serves as the root element for a JDOM document, and it provides access to the document’s content, including its elements, attributes, namespaces, and entities.

Creating a Document

A JDOM document can be created from scratch by instantiating a new Document object: 

          
           
org.jdom.Document document = new org.jdom.Document();
It can also be created from a file, an InputStream, or a Reader: 

          
           
org.jdom.Document document = org.jdom.input.SAXBuilder().build(new File("document.xml"));
org.jdom.Document document = org.jdom.input.SAXBuilder().build(new InputStream("document.xml"));
org.jdom.Document document = org.jdom.input.SAXBuilder().build(new Reader("document.xml"));

Adding Elements, Attributes, and Namespaces

Elements, attributes, and namespaces can be added to a JDOM document using the setRootElement(), addContent(), addContent(), setAttribute(), and setNamespace() methods. For example: 

          
           
org.jdom.Element element = new org.jdom.Element("root");
document.setRootElement(element);

org.jdom.Element child = new org.jdom.Element("child");
child.addContent("Hello World!");
element.addContent(child);

org.jdom.Attribute attribute = new org.jdom.Attribute("foo", "bar");
child.setAttribute(attribute);

org.jdom.Namespace namespace = org.jdom.Namespace.getNamespace("prefix", "uri");
child.setNamespace(namespace);

Serializing the Document

The document can be serialized to a file, an OutputStream, or a Writer using the XMLOutputter class. 

          
           
org.jdom.output.XMLOutputter outputter = new org.jdom.output.XMLOutputter();
outputter.output(document, new FileWriter("document.xml"));
outputter.output(document, new OutputStreamWriter(System.out));
It can also be converted to a String using the toString() method: 

          
           
String xmlString = outputter.outputString(document);

Conclusion

The org.jdom.Document class is a powerful tool for manipulating XML documents in Java. It allows developers to create, modify, and serialize XML documents with ease.
