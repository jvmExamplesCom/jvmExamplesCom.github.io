org.jdom.Document

The org.jdom.Document class is a representation of an XML document in Java. It serves as the root element for a JDOM document, and it provides access to the document’s content, including its elements, attributes, namespaces, and entities.

Creating a Document

A JDOM document can be created from scratch by instantiating a new Document object: 

          
           
org.jdom.Document document = new org.jdom.Document();
It can also be created from a file, an InputStream, or a Reader: 

          
           
org.jdom.Document document = org.jdom.input.SAXBuilder().build(new File("document.xml"));
org.jdom.Document document = org.jdom.input.SAXBuilder().build(new InputStream("document.xml"));
org.jdom.Document document = org.jdom.input.SAXBuilder().build(new Reader("document.xml"));

org.jdom.Element element = new org.jdom.Element("root");

child.setAttribute(attribute);
