org.xml.sax.SAXNotSupportedException

The org.xml.sax.SAXNotSupportedException is an exception that is thrown when a feature or property is not supported by the SAX implementation. This typically happens when the requested feature or property is not supported by any of the underlying XML processors involved in the parsing. The SAXNotSupportedException is a subclass of SAXException, which is the parent class for all types of SAX-related exceptions.

Example 1: Parse XML Document With Unsupported Feature

In this example we'll try to parse an XML document that uses an unsupported feature. This will result in an org.xml.sax.SAXNotSupportedException being thrown.


          
           
import java.io.File;

import javax.xml.parsers.SAXParser;
import javax.xml.parsers.SAXParserFactory;

import org.xml.sax.InputSource;

public class SaxNotSupportedExceptionExample {

    public static void main(String[] args) throws Exception {
        SAXParserFactory factory = SAXParserFactory.newInstance();
        // attempt to use an unsupported feature

        SAXParser parser = factory.newSAXParser();
  
        File xmlFile = new File("students.xml");
        InputSource inputSource = new InputSource(xmlFile.toURI().toString());
        parser.parse(inputSource, new MySaxHandler());
    }

}

The above code will result in an org.xml.sax.SAXNotSupportedException being thrown due to the unsupported feature being set on the SAXParserFactory.

In order to avoid the org.xml.sax.SAXNotSupportedException, we can use an implementation of the XmlReader interface instead of the SAXParser. The XmlReader is more flexible than the SAXParser and supports more features. In this example we'll use the XmlReader to parse an XML document that uses an unsupported feature.


          
           
import java.io.File;

import javax.xml.parsers.SAXParserFactory;
import javax.xml.stream.XMLInputFactory;
import javax.xml.stream.XMLStreamReader;

import org.xml.sax.InputSource;
import org.xml.sax.XMLReader;

public class XmlReaderExample {

    public static void main(String[] args) throws Exception {
        XMLInputFactory xmlInputFactory = XMLInputFactory.newInstance();
        XMLStreamReader xmlStreamReader = xmlInputFactory.createXMLStreamReader(new File("students.xml"));
  
        SAXParserFactory saxParserFactory = SAXParserFactory.newInstance();
        XMLReader xmlReader = saxParserFactory.newSAXParser().getXMLReader();
        xmlReader.setContentHandler(new MySaxHandler());
        xmlReader.parse(new InputSource(xmlStreamReader));
    }

}

In this example we create a XMLStreamReader from the XML document and then create an XMLReader from a SAXParser and set the content handler to our custom MySaxHandler. We then parse the document with the XMLReader and it will not throw the org.xml.sax.SAXNotSupportedException.
