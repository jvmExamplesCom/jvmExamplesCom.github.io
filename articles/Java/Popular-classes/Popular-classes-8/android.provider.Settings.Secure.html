Android.provider.Settings.Secure

Android.provider.Settings.Secure is an Android class that contains secure system settings. Settings that are stored in the Android Secure class are restrictive and can only be changed by an administrator, or by a system app with the right permissions. The Android Secure APIs provide access to system settings related to preferences for secure features such as encryption, screen pinning and user access control.

        Settings.Secure.LOCK_SCREEN_CREDENTIAL_TYPE);

if (lockScreenCredentialType.equals(Settings.Secure.LOCK_SCREEN_CREDENTIAL_TYPE_NONE))

    // Do something
}

Example 2: Setting Secure Settings


          
           
// Set the Secure settings to system
ContentResolver cr = getContentResolver();
