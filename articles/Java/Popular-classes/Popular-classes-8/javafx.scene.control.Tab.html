JavaFx Scene Control Tab

JavaFX Scene Control Tab is an extension of the JavaFX Scene class, and provides a mechanism for grouping together different Scenes in one UI with a single root container. It provides a convenient way to switch between different parts of an application by clicking on tabs.

Creating a Tab

A tab can be created with the following code: 

          
           
Tab tab = new Tab();
tab.setText("My Tab");

Once the tab is constructed, it must be added to a TabPane. A TabPane can be created like so:


          
           
TabPane tabPane = new TabPane();

Then the newly created tab can be added to the TabPane:


          
           
tabPane.getTabs().add(tab);
Now, the tab will be visible in the UI.

Adding Content to the Tab

Content can be added to the tab in the form of nodes. A node is a type of UI element (such as a label, button, text field, etc.) that can be used to enter data and interact with. To add a node to the tab, it must first be created, like so:


          
           
Label label = new Label("My Label");
Then it can be added to the tab’s content area: 

          
           
tab.setContent(label);

Now, when the user switches to that tab, the label will be visible.

Setting the Tab's Icon

The icon used to represent the tab can be set with the 

setGraphic()
method. This method takes an object of type ImageView, which can be constructed with an image file like so: 

          
           
Image image = new Image("/images/myimage.png");
ImageView imageView = new ImageView(image);

Now, this image view can be set as the tab’s icon:


          
           
tab.setGraphic(imageView);

Adding an Event Handler

Event handlers can be added to the tab to perform an action when it is selected or deselected. An event handler is an instance of the EventHandler interface, and requires an implementation of its 

handle()
method. This method takes two arguments: the tab itself and an event object, which can be used to determine why the tab was selected or deselected. Here is an example: 

          
           
EventHandler<Tab> handler = (tab, event) -> {
    if (event.getEventType() == TabSelectionEvent.SELECTED) {
        // code to run when tab is selected
    }
    else if (event.getEventType() == TabSelectionEvent.DESELECTED) {
        // code to run when tab is deselected
    }
};

This event handler can then be set on the tab with the 


          
           
setOnSelectionChanged
method: 

          
           
tab.setOnSelectionChanged(handler);

Now, the code in the event handler will be executed whenever the tab is selected or deselected.

Conclusion

In this article, we discussed how to construct and use JavaFX Scene Control Tabs. We covered how to create a tab, add content to it, set an icon, and add an event handler. With the techniques discussed here, you should now be able to build effective tab-based user interfaces.
