org.apache.commons.lang.SystemUtils

The org.apache.commons.lang.SystemUtils class provides a way to access and identify operating system, Java virtual machine and other environment information. It works as a convenient way of getting access to system properties, with public static methods of the SystemUtils class allowing easy access to frequently-used system properties.

Getting Operating System Information

The org.apache.commons.lang.SystemUtils class provides convenient way to access operating system information. The following code example shows how to get the name of the current operating system:


          
           
String osName = SystemUtils.OS_NAME;
System.out.println(osName);

This code will print out the name of the current operating system. For example, on a Windows 10 system it will print “Windows 10”.

Getting Java Virtual Machine Information

To get information about the Java virtual machine running within the application, the 

SystemUtils
class provides access to all system properties exposed by the JVM. To get the name of the version of the JVM running, you can use the following code: 

          
           
String jvmName = System.getProperty("java.vm.name");
System.out.println(jvmName);

This code will print out the name of the current JVM running, such as “OpenJDK 64-Bit Server VM”.

Getting Other System Properties

The 
SystemUtils
class also provides access to other commonly used system properties. For example, to get the name of the user running the application, you can use the following code: 

          
           
String userName = System.getProperty("user.name");
System.out.println(userName);

This code will print out the name of the user running the application. For example, on a Windows 10 system it will print “admin”.

The org.apache.commons.lang.SystemUtils class provides a convenient way to access and identify operating system, Java virtual machine and other environment information. This makes it very useful for applications that need to adapt their behavior according to the underlying environment.
