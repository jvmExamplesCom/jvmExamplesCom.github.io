javax.swing.JWindow

The javax.swing.JWindow class is a top-level window that does not have any borders or menu bar. It displays a blank region of the specified size. It doesn't have any title bar, window decorations or close button. The JWindow class is usually used as a container for the other Swing components.

Example 1: Creating a Simple JWindow


          
           
JWindow window = new JWindow();
window.setSize(400, 300);
window.setLocationRelativeTo(null);
window.setVisible(true);

Example 2: Creating a JWindow with Background Color


          
           
JWindow window = new JWindow();
window.getContentPane().setBackground(Color.ORANGE);
window.setSize(400, 300);
window.setLocationRelativeTo(null);
window.setVisible(true);

Example 3: Adding Components to JWindow


          
           
JWindow window = new JWindow();
window.getContentPane().add(new JLabel("Hello World"));
window.setSize(400, 300);
window.setLocationRelativeTo(null);
window.setVisible(true);

The above program creates a window with a label "Hello World" in the center. To add more components inside the window, we can use the 


          
           
window.getContentPane().add()
method.

Example 4: Creating a Custom Layout

We can also create a custom layout for the window by creating an object of the 

java.awt.GridLayout
class and setting it as the layout of the content pane. 

          
           
JWindow window = new JWindow();
window.getContentPane().setLayout(new GridLayout(2, 2));
window.getContentPane().add(new JLabel("Hello World"));
window.getContentPane().add(new JTextField());
window.getContentPane().add(new JLabel("Goodbye World"));
window.getContentPane().add(new JButton("Click Here"));
window.setSize(400, 300);
window.setLocationRelativeTo(null);
window.setVisible(true);

The above program creates a window with four components arranged in a 

2x2
grid. It has two labels and two controls.

Example 5: Setting Window's Title and Icon

The 
setTitle()
method is used to set the title of the window and the 
setIconImage()
method is used to set the icon of the window. 

          
           
JWindow window = new JWindow();
window.setTitle("My Window");
window.setIconImage(new ImageIcon("icon.png").getImage());
window.getContentPane().add(new JLabel("Hello World"));
window.setSize(400, 300);
window.setLocationRelativeTo(null);
window.setVisible(true);

In the above example, we have set a title for the window and an icon for the window.

Example 6: Closing the Window

The 
dispose()
method is used to close the window and free up its resources. 

          
           
JWindow window = new JWindow();
window.getContentPane().add(new JLabel("Hello World"));
window.setSize(400, 300);
window.setLocationRelativeTo(null);
window.setVisible(true);
window.dispose();

Example 7: Handling Window Closing Event

The 
addWindowListener()
method is used to add a window listener which listens to the window closing event. 

          
           
JWindow window = new JWindow();
window.addWindowListener(new WindowAdapter() {
  public void windowClosing(WindowEvent e) {
      System.out.println("Window Closed");
  }
});
window.getContentPane().add(new JLabel("Hello World"));
window.setSize(400, 300);
window.setLocationRelativeTo(null);
window.setVisible(true);

In the above example, when we close the window, it will print "Window Closed" to the console.
