java.util.function.LongSupplier

The java.util.function.LongSupplier interface is a functional interface that enables the user to get and return a long-type result without taking any argument. This makes it a perfect choice for when you want to get a result from a supplier but don't need to pass in any arguments. The LongSupplier interface is part of the Java 8 SDK and was released with the version 1.8.

Using LongSupplier

To make use of the LongSupplier interface, we have to create a class that implements this interface and then override its single method, getAsLong(). Inside this method, we can specify whatever logic is necessary to return a value. Here's an example of a simple implementation of the LongSupplier interface:


          
           
public class RandomLongSupplier implements LongSupplier {
    @Override
    public long getAsLong() {
        long min = -100;
        long max = 100;
        return min + (long)(Math.random() * (max - min));
    }
}

Once our class is implemented, we can use it as a Lambda expression like this:

long result = supplier.getAsLong();

The above code will return a random number between -100 and 100.
