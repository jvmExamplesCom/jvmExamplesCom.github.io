Using LongSupplier

Inside this method, we can specify whatever logic is necessary to return a value. Here's an example of a simple implementation of the LongSupplier interface:

    @Override
    public long getAsLong() {

        long max = 100;
        return min + (long)(Math.random() * (max - min));
