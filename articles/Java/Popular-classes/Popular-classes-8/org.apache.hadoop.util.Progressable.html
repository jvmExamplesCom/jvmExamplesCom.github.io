Progressable in Apache Hadoop

Apache Hadoop is an open-source software platform for distributed storage and distributed processing of very large datasets on computer clusters. The org.apache.hadoop.util package provides a Progressable interface which is used to give feedback of process progress to the user. In this article, we will explore the Progressable interface and discuss code examples of its usage.

What is Progressable?

The Progressable interface is used to monitor the progress of the computation task. It is used by processes such as MapReduce, HDFS, YARN. Progressable can report progress in several ways, such as printing messages, logging, writing to a stream. Additionally, it provides hooks for progress tracking and other performance related metrics.

Syntax and Methods

The Progressable interface is part of the org.apache.hadoop.util package. There are two methods associated with the Progressable interface: 
  • void progress()
    : This method is called periodically by the task tracker to indicate the progress of the current task.
  • 
            
             
void setTaskId(TaskId taskId)
    : This method is called when the task tracker assigns a taskId to the task. This can be used to identify the task and track its progress.

Usage Example

Here is an example of how to use the Progressable interface. This example prints out the progress of the task for every 5 seconds. 

          
           
public class Foo implements Progressable {
	public void progress() {
		System.out.println("Task is in progress...");
	}
	public void setTaskId(TaskId taskId) {
		System.out.println("TaskId: " + taskId);
	}
	public static void main(String[] args) throws Exception {
		Foo foo = new Foo();
		while (true) {
			foo.progress();
			Thread.sleep(5000);
		}
	}
}

Conclusion

The Progressable interface is a useful tool for providing feedback about the progress of a task. It can be used to track and measure performance related metrics. By implementing the Progressable interface in your application you can easily monitor the progress of a long-running task.
