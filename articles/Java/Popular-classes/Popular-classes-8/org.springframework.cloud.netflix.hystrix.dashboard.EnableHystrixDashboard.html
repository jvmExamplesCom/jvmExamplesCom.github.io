Analysis of org.springframework.cloud.netflix.hystrix.dashboard.EnableHystrixDashboard in Java

This annotation provides developers with real-time information about system health, including numeric values such as latency and errors.

This annotation is used to enable Hystrix Dashboard for a given application. To do this, add the @EnableHystrixDashboard annotation to the main application configuration class.
