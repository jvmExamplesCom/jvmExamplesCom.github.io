The Dimension class in Selenium is an object of type org.openqa.selenium.Dimension. This object helps in the manipulation of size and position of a browser window.

The 

          
           
org.openqa.selenium.Dimension
class has the following three methods: 
  • getWidth()
    – which returns the width of the browser window.
  • getHeight()
    – which returns the height of the browser window.
  • 
            
             
manage().window().setSize()
    – which sets the size of the browser window.

The following code example shows how to get the width and height of a browser window:


          
           
Dimension d = driver.manage().window().getSize();
int width = d.getWidth();
int height = d.getHeight();

System.out.println(“Height of the window: ” + height);

Width of the window: 1920

The following code example shows how to set the size of a browser window:


          
           
Dimension d = new Dimension(400, 600);
driver.manage().window().setSize(d);
