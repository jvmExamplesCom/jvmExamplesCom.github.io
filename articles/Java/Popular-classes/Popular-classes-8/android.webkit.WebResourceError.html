android.webkit.WebResourceError

android.webkit.WebResourceError is an error class that describes errors generated in the network layer when loading resources such as web pages. It contains information about the error code and error description associated with the error. This class is used in conjunction with android.webkit.WebViewClient to provide error info during asynchronous page loading.

Example usage

The following example shows how to use android.webkit.WebResourceError to detect and handle errors in a WebView application.

        if (error != null) {

            // handle error according to error code
        } 
     }
});
