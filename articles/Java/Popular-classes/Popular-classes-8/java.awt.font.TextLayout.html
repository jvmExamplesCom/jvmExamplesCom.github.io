java.awt.font.TextLayout

The Java AWT font TextLayout class provides various methods to draw text. It takes a FontRenderContext instance and a String as parameters and returns an instance of TextLayout. The returned instance can be manipulated and then rendered with the given FontRenderContext. This class is part of the java.awt package, which provides the public API for interacting with the graphical user interface components of Java.

Methods

The TextLayout class includes the following methods:

  • draw(Graphics2D g2, float x, float y): draws the text at the specified location using the given Graphics2D renderer.
  • getBounds(): returns a Rectangle2D object containing the bounds of the text.
  • getCaretInfo(int index): returns a CaretInfo object at the specified index.
  • getGlyphVector(): returns a GlyphVector object containing the glyphs of the text.
  • hitTestChar(float x, float y): returns the character index at the specified location.
  • hitTestPoint(float x, float y): returns true if the point is contained in the layout.
  • getVisibleAdvance(): returns the visible width of the text.
