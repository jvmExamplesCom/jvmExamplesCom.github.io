Introduction to the TarArchiveOutputStream Class

The TarArchiveOutputStream class is part of the org.apache.commons.compress.archivers package, and it allows you to create tar files from Java programs. Tar files are a common archive format and there are many compression algorithms available for them. Tar is usually the most basic of these, but several more efficient algorithms can be used to further compress the data. The TarArchiveOutputStream class provides access to these algorithms and makes it easy to write tar files from your Java code.

TarArchiveOutputStream out = new TarArchiveOutputStream(new FileOutputStream("example.tar"));
