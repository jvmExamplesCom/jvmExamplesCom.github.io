What is com.amazonaws.services.s3.model.ObjectListing?

com.amazonaws.services.s3.model.ObjectListing is an interface which represents a list of objects stored in Amazon S3. It provides methods to access the list and details of each item listed.

Creating an ObjectListing

ObjectListing can be created by calling the 
listObjects()
method of the AmazonS3Client class. This method accepts several parameters allowing users to customize the list returned: 

          
           
AmazonS3Client s3 = new AmazonS3Client(); 
ObjectListing objectListing = s3.listObjects();

Accessing the List

getObjectSummaries()
method of the ObjectListing class provides access to the objects and their properties. 

          
           
List<s3objectsummary> summaries = objectListing.getObjectSummaries();

ObjectListing contains several other useful methods, including:

    : Returns the name of the bucket containing the objects.

  • getMarker()
