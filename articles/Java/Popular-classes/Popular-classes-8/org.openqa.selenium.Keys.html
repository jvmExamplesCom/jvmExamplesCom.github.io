The org.openqa.selenium.Keys Class

The org.openqa.selenium.Keys class is a Java utility class used to implement various keys used in various languages and special characters when interacting with web elements using the Selenium WebDriver. This class is an essential part of the Selenium API, providing access to a variety of keyboard-centric functions that enable users to select elements on the screen as well as press and control various keyboard keys.

Keyboard Keys & Functions

The most common keyboard key functions implemented by the org.openqa.selenium.Keys class are the following:

  • Enter/Return key: When interacting with web elements, pressing the Return or Enter key provides a way to submit form data and force completion of a series of tasks.
  • Shift Key: Used to toggle upper and lower case letters. This is particularly useful when selecting elements in capitalization-sensitive programs.
  • Control/Ctrl Key: Used to create shortcuts such as copy and paste or to open menus or dialog boxes.
  • Tab Key: Used to switch between various form or web element fields.
  • Backspace/Delete Keys: Used to delete data from form or web element fields.
  • Arrow Keys: Used to navigate menu items and selections.
  • Function Keys: Used to trigger shortcuts and special functions.
  • Alt Key: Used to trigger special menu functions.
  • Escape Key: Used to exit out of certain types of forms or web elements.

Example Code

driver.findElement(By.name("q")).sendKeys(Keys.ENTER);
