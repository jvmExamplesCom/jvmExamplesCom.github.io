Overview

The 

          
           
uk.co.senab.photoview.PhotoViewAttacher

This library provides an easy way to add Zoomable and Panning functionality to Views, such as ImageViews and others. It also supports Fling gestures and can work with both normal and matrix ScaleTypes.

uk.co.senab.photoview.PhotoViewAttacher
, you’ll first need to add the library to your project. To do this, simply add the following Maven dependency to your project’s pom.xml:
