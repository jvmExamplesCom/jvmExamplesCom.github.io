Java Library: uk.co.senab.photoview.PhotoViewAttacher

Overview

The 

          
           
uk.co.senab.photoview.PhotoViewAttacher
is a Java library designed by Chris Banes for the Android platform to allow for application developers to easily attach a PhotoView to any view, allowing simple zooming and scaling between two views. This library provides an easy way to add Zoomable and Panning functionality to Views, such as ImageViews and others. It also supports Fling gestures and can work with both normal and matrix ScaleTypes.

uk.co.senab.photoview.PhotoViewAttacher
, you’ll first need to add the library to your project. To do this, simply add the following Maven dependency to your project’s pom.xml: 

          
           
<dependency>
  <groupId>uk.co.senab</groupId>
  <artifactId>photoview</artifactId>
  <version>1.4.4</version>
</dependency>

PhotoViewAttacher
. The easiest way to do this is to simply create a new instance of the class and pass in the View that you wish to attach it to. For example, if you have an 

          
           
ImageView

PhotoViewAttacher

PhotoViewAttacher mAttacher = new PhotoViewAttacher(imageView);

PhotoViewAttacher
, you can then use its various methods to modify the View. For example, you can enable or disable the zoom functionality using the
