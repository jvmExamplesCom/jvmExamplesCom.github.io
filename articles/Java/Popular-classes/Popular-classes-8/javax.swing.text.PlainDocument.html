Introduction to javax.swing.text.PlainDocument

javax.swing.text.PlainDocument is a class in the Java Swing GUI library that represents a document with text content. It provides methods for accessing and manipulating the text data, such as searching for words, replacing text, and formatting text. The PlainDocument class is part of the javax.swing.text package, which also includes other classes that provide support for documents with different types of characters and text formatting, such as javax.swing.text.StyledDocument, javax.swing.text.AbstractDocument, and javax.swing.text.DefaultStyledDocument.

Creating a PlainDocument Object

A PlainDocument object is created by passing a DocumentFilter object to the constructor. The DocumentFilter is used to filter the text when it is inserted into the document. 

          
           
DocumentFilter filter = new MyDocumentFilter();
PlainDocument doc = new PlainDocument(filter);

Inserting Text into a PlainDocument Object

Text can be inserted into a PlainDocument object using the insertString() method. The method takes two parameters — an offset and a string — and inserts the string into the document at the specified offset. If the offset is out of bounds, an IllegalArgumentException is thrown. 

          
           
doc.insertString(0, "Hello world!");
// inserts "Hello world!" at the beginning of the document

doc.insertString(5, " there,");
// inserts "there," at offset 5

Getting Text from a PlainDocument Object

The getText() method can be used to retrieve the text from a PlainDocument object. The method takes one parameter — an array to store the text — and returns the number of characters stored in the array. The text is stored as a char array and is null-terminated.

Searching a PlainDocument Object

The search() method can be used to search for a string in a PlainDocument object. The method takes two parameters — a string and an offset — and returns the position of the string in the document. If the string is not found, -1 is returned.

Replacing Text in a PlainDocument Object

The replace() method can be used to replace a string in a PlainDocument object. The method takes three parameters — a string, an offset and a length — and replaces the string in the document starting at the specified offset and ending at the offset+length.

// prints out the updated contents of the document
