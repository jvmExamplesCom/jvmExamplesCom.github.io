Introduction to Java MongoCredential

MongoCredential is an interface used in the Java driver for MongoDB to store credentials used for authentication when connecting to a MongoDB server. It is part of the com.mongodb package and is capable of authenticating with a username, password, and database name as well as authenticating against MongoDB’s SCRAM-SHA-1 protocol. This article covers the basics of MongoCredential and provides some examples of how to use it.

Creating a MongoCredential Object

The first step to using MongoCredential is creating a MongoCredential object. There are three different methods that can be used to create a MongoCredential object.

Using Username and Password

The simplest way to create a MongoCredential object is to provide a username and password. The following example creates a MongoCredential object using the username testuser and the password mypassword: 

          
           
MongoCredential credential = MongoCredential.createCredential("testuser", "mydatabase", "mypassword".toCharArray());

Using SCRAM-SHA-1 Authentication

If you want to use the SCRAM-SHA-1 authentication protocol then you can use the createScramSha1Credential method. This example shows how to create a MongoCredential object with a username, database name, and secret: 

          
           
MongoCredential credential = MongoCredential.createScramSha1Credential("testuser", "mydatabase", "mysecret".toCharArray());

Using X.509 Authentication

If you are setting up X.509 authentication then you use the createMongoX509Credential method. This example shows how to create a MongoCredential object using an X.509 certificate: 

          
           
MongoCredential credential = MongoCredential.createMongoX509Credential("CN=testuser,O=MyOrg");

Connecting to MongoDB with MongoCredential

MongoClient mongoClient = new MongoClient(new ServerAddress("localhost", 27017), Arrays.asList(credential));
