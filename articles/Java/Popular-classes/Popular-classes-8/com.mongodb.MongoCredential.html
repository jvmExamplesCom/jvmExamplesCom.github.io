Introduction to Java MongoCredential

MongoCredential is an interface used in the Java driver for MongoDB to store credentials used for authentication when connecting to a MongoDB server. It is part of the com.mongodb package and is capable of authenticating with a username, password, and database name as well as authenticating against MongoDB’s SCRAM-SHA-1 protocol. Creating a MongoCredential Object

The first step to using MongoCredential is creating a MongoCredential object. There are three different methods that can be used to create a MongoCredential object.

Using Username and Password

The simplest way to create a MongoCredential object is to provide a username and password. The following example creates a MongoCredential object using the username testuser and the password mypassword: 

          
           
MongoCredential credential = MongoCredential.createCredential("testuser", "mydatabase", "mypassword".toCharArray());

If you want to use the SCRAM-SHA-1 authentication protocol then you can use the createScramSha1Credential method. This example shows how to create a MongoCredential object with a username, database name, and secret:

MongoCredential credential = MongoCredential.createMongoX509Credential("CN=testuser,O=MyOrg");
