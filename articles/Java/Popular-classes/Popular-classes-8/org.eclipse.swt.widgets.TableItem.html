What is org.eclipse.swt.widgets.TableItem?

org.eclipse.swt.widgets.TableItem is an item in a Table widget which can be set to display other items, such as text or images. TableItem typically contains an array of strings which are used to represent the data contained in the item.

Example Usage

It is possible to create a TableItem instance using the following code: 

          
           
TableItem tableItem = new TableItem(table, SWT.NONE);
This will create a new TableItem instance, associated with the provided Table widget.

String[] itemData = {"Item 1", "Item 2", "Item 3"};
tableItem.setText(itemData);

The data stored in a TableItem can also be retrieved. The following code snippet shows an example of how this can be done: 

          
           
String[] itemData = tableItem.getText();
