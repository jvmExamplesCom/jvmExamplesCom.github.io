What is org.eclipse.swt.widgets.TableItem?

Example Usage

It is possible to create a TableItem instance using the following code: 

          
           
TableItem tableItem = new TableItem(table, SWT.NONE);
This will create a new TableItem instance, associated with the provided Table widget.

String[] itemData = {"Item 1", "Item 2", "Item 3"};
tableItem.setText(itemData);

String[] itemData = tableItem.getText();
