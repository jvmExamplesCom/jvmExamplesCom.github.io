java.awt.dnd.DnDConstants

The java.awt.dnd.DnDConstants class provides constants used during drag and drop operations. It encompasses two kinds of constants: drag and drop actions, and drop location types. Drag and drop actions correspond to the standard drag and drop operation modes, while the drop location types correspond to the different ways the data can be dropped.

Drag and Drop Actions

The following are constants corresponding to the possible drag and drop actions:


          
           
public static final int ACTION_NONE = 0x0;
public static final int ACTION_COPY = 0x1;
public static final int ACTION_MOVE = 0x2;
public static final int ACTION_COPY_OR_MOVE = ACTION_COPY | ACTION_MOVE;
public static final int ACTION_LINK = 0x40000000;

This example prints out the value of ACTION_COPY_OR_MOVE:


          
           
System.out.println(DnDConstants.ACTION_COPY_OR_MOVE); //prints out 3

Drop Location Types

The following are constants corresponding to the different types of drop locations which can be specified. These correspond to the different targets specified in the target drop action:

public static final int DROP_COPY = 0x1;
public static final int DROP_MOVE = 0x2;
