android.accessibilityservice.AccessibilityService

The android.accessibilityservice.AccessibilityService class is a core class in the Android Accessibility framework that enables accessibility services such as screen readers, switch access, and dictation services to detect and observe events across the system, deliver user interface feedback, and control device behaviors.

Overview

An AccessibilityService runs in the background and receives callbacks by the system when AccessibilityEvents are fired. Such events denote some state transition in the user interface, for example, the focus has changed, a button has been clicked, etc. Such a service can optionally request the capability for querying the content of the active window. Development of an accessibility service requires extending this class and implementing its abstract methods. An AccessibilityService can optionally specify a set of Capabilities defining additional features it supports - queries about the contents of the active window, keyboard shortcut capture, etc.
