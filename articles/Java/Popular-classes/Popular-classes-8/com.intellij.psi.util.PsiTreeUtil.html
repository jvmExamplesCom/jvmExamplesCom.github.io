com.intellij.psi.util.PsiTreeUtil

It is designed to provide basic features such as finding elements by type, and traversing parent, child, and sibling nodes within a tree.

Element element1 = ...;
Element element2 = ...;
Element element3 = ...;

root.addChild(element1);
root.addChild(element2);
