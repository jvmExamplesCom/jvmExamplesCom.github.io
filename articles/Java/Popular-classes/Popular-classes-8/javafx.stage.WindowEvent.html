WindowEvent
is an event that is dispatched by a JavaFX window. It represents the action of closing, hiding, or showing a window. It’s usually specified within the Window class, which represents the physical window for a JavaFX application. WindowEvents can be monitored to detect when the user interacts with the window, and allows application developers to apply different actions based on these events.

public class WindowEvent extends Event

Example

    // Perform action when window is closed
});
