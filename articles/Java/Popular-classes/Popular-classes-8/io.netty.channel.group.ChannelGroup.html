Overview of io.netty.channel.group.ChannelGroup

io.netty.channel.group.ChannelGroup is an implementation of a set of open channels that allows for easy iteration and manipulation of channels. It is an extension of java.util.Set, making the API and functionality very familiar to standard Java programming.
