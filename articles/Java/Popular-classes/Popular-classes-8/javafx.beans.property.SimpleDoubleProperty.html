JavaFX Beans Property SimpleDoubleProperty

javafx.beans.property.SimpleDoubleProperty

double

SimpleDoubleProperty

, you must first import the class as follows: 

          
           
import javafx.beans.property.SimpleDoubleProperty;

double
: 

          
           
SimpleDoubleProperty myDouble = new SimpleDoubleProperty(5.0);

double
