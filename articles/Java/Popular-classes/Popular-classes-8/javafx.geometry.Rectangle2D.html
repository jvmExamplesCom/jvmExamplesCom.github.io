Defining a Rectangle2D Object

To create a Rectangle2D object, you must define the x, y coordinates of the upper left corner, the width, and the height of the rectangle. This is done through the Rectangle2D constructor, which takes four parameters: x, y, width, and height. Rectangle2D rect = new Rectangle2D(10, 20, 50, 40);

rect.setWidth(70);
