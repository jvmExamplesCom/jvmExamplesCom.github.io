What is org.joda.time.Instant?

Creating an Instant

Instances of Instant can be created from either epoch-milli or epoch-micro values. Epoch-milli values represent the number of milliseconds since the Unix epoch, whereas epoch-micro values represent the number of microseconds since the Unix epoch. Both values can be used to create an instance of Instant, which can then be used to work with that specific point in time.
