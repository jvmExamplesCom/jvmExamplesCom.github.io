javax.validation.ValidationException

A javax.validation.ValidationException is an unchecked exception thrown to indicate a failed validation. This class is the base class of exceptions thrown when a validation fails, and can be thrown directly, or as the cause in another exception.

When this exception is thrown, it generally indicates something wrong with the data being validated. The cause can include incorrect business logic, invalid user input, or data sent from an external source. In any case, attempting to continue with the operation that caused the exception throws an exception.

Example 1: Simple Validation Example


          
           
public static void validateInput(String input) {
    try {
        if (input == null || input.isEmpty()) {
            throw new javax.validation.ValidationException("Input cannot be empty");
        }
    } catch (javax.validation.ValidationException e) {
        e.printStackTrace();
    }
}

public static void checkForConstraintViolations(Person person)

    ValidatorFactory factory = Validation.buildDefaultValidatorFactory();

@Target(ElementType.TYPE)
@Retention(RetentionPolicy.RUNTIME)
@Constraint(validatedBy = MinAgeValidator.class)
public @interface MinAge
{
   int value();
   String message() default "Age must be at least {value}";
   Class<?>[] groups() default {};
   Class<? extends Payload>[] payload() default {};
 }

Once the constraint is created, it can be applied to the Person class.

To enforce the constraint, the MinAgeValidator class must be implemented. This class checks that the age is greater than or equal to 18. If not, a

public class MinAgeValidator implements ConstraintValidator<MinAge, Integer>
{
    private int minAge;

    public void initialize(MinAge annotation)
    {
        minAge = annotation.value();
    }

    public boolean isValid(Integer age, ConstraintValidatorContext context)
    {
        if (age < minAge) {
            throw new javax.validation.ValidationException("Age must be at least "+minAge);
        }
        return true;
    }
}
