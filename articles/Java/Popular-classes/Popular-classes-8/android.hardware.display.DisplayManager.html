Android Hardware Display Manager - Java

Android Hardware Display Manager is a collection of APIs that allow you to interact with the system-level display hardware. You can use it to query the display properties, control attributes like brightness, or create and manage display output devices in the system. This article will discuss how to use Android Hardware Display Manager in Java.

Overview of DisplayManager Features

The DisplayManager provides several capabilities for controlling the display:
  • Querying the display properties, such as its size and resolution.
  • Controlling display attributes, such as brightness and contrast.
  • Creating and managing display output devices.
  • Listing available displays and getting their display IDs.
  • Creating displays from device ID and setting their resolution.
  • Enabling and disabling displays.

Using DisplayManager in Java

To use the DisplayManager in Java, you will need to first import the necessary classes: 

          
           
import android.hardware.display.DisplayManager;
import android.view.Display;
Then, you can get access to the DisplayManager object using the following code: 

          
           
DisplayManager dm = (DisplayManager)getSystemService(Context.DISPLAY_SERVICE);
From there, you can start to use the various methods included in the DisplayManager class.

Querying Display Properties

To query the display properties, you can use the getDisplayInfo() method of the DisplayManager. This method takes in a display ID as a parameter, and returns a Display.Info object containing information about the display. The Display.Info object contains information such as the display name, size, resolution, and so on. Here is an example of how to use the getDisplayInfo() method: 

          
           
Display.Info di = dm.getDisplayInfo(displayId);
String name = di.name;
int width = di.getNaturalWidth();
int height = di.getNaturalHeight();

Controlling Display Attributes

You can control display attributes such as brightness and contrast using the setBrightness() and setContrast() methods of the DisplayManager. These methods take in a display ID and a float value as parameters, and set the corresponding attribute to the specified value. Here is an example of how to use the setBrightness() method: 

          
           
dm.setBrightness(displayId, 0.5f);

Creating and Managing Display Output Devices

You can create and manage display output devices using the createVirtualDisplay() and release() methods of the DisplayManager. The createVirtualDisplay() method takes in a display ID, a name for the display, the size of the display, and the format of the image. The release() method takes in a display ID and releases the associated display output device. Here is an example of how to use these methods: 

          
           
DisplayManager.VirtualDisplay vd = dm.createVirtualDisplay(displayId, "My Display", 1024, 768,
    ImageFormat.RGBA_8888);
vd.release();

Conclusion

The Android Hardware Display Manager provides an easy way to interact with system-level display hardware. With its various capabilities, you can easily query and control the display properties and create and manage display output devices. With the APIs provided, you can now easily include this functionality in your apps written in Java.
