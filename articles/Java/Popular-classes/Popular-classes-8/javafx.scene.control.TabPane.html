Introduction

The javafx.scene.control.TabPane class is a JavaFX control that displays two or more content areas in a tabbed window format, allowing the user to switch between them with ease. Each tab consists of a label and an associated content area. Tabs can also be closed, so that the user can make more room for their content. This tutorial covers the basics of how to create, edit and use the TabPane control.

Creating a TabPane

To create a new TabPane object, use the following code:


          
           
TabPane pane = new TabPane();

This will create an empty TabPane with no tabs. To add tabs to the TabPane, you can create each tab and then add it to the pane.

Creating a Tab

A tab consists of a label and its associated content. The label is a String, while the content can be any valid JavaFX Node object. To create a tab, use the following code:


          
           
Tab tab = new Tab("Tab Label", content);

Adding a Tab

Once you have created the tab, you can add it to the TabPane using the following code:


          
           
pane.getTabs().add(tab);

Configuring the TabPane

You can also configure the TabPane by setting some properties. For example, you can set whether or not the tabs are closable, the tab orientation (horizontal or vertical), and the currently selected tab. To do this, use the following code:


          
           
pane.setClosable(true);
pane.setTabPlacement(TabPane.TabPlacement.HORIZONTAL);
pane.getSelectionModel().select(tab);

Conclusion

This tutorial has covered the basics of how to create, edit, and use the javafx.scene.control.TabPane control in JavaFX. With this simple control, you can easily create tabbed windows to store different pieces of content.
