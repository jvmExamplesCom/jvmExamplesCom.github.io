javax.json.JsonObjectBuilder

javax.json.JsonObjectBuilder is a class in the Java API used to create JSON objects. It provides an interface for constructing a JSON object model from scratch using an API. This class is closely related to javax.json.JsonArrayBuilder and javax.json.JsonValue; users of this class are able to access these classes for some additional features.

JSON objects are composed of key/value pairs, where the keys must be strings and the values can be any valid JSON value. The JsonObjectBuilder class provides method calls to add key/value pairs to the JSON object being constructed.

builder.add("name", "John Doe");
builder.add("age", 25);
JsonObject data = builder.build();

build()
