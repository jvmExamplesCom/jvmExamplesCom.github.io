java.text.AttributedString

The java.text.AttributedString class is a character sequence that can be used to associate attributes with individual characters. This class provides a way to do the following: represent text with character attributes, such as font and color; apply multiple styles of text to a single string; and produce high-level text manipulation effects, such as justification.

Creating an AttributedString

An AttributedString is created by calling its constructor with a character sequence parameter. An AttributedCharacterIterator can also be used to create an AttributedString.


          
           
AttributedString str = new AttributedString("My Text");

Adding Attributes to an AttributedString

Attributes are added to an AttributedString using the addAttribute() method. Attributes are applied to a range of characters in the string and can be used to specify font, color, style, etc. The addAttribute() method takes two parameters: an Attribute (defined by the AttributedCharacterIterator interface) and a value.


          
           
str.addAttribute(TextAttribute.FONT, Font.BOLD);

The above example adds the Bold font attribute to all characters in the AttributedString.

Applying Multiple Styles of Text to an AttributedString

The addAttribute() method can be used to add multiple attributes to a single AttributedString. Attributes can be applied to specific ranges of characters in the string by specifying the start index and end index of the range.


          
           
str.addAttribute(TextAttribute.FONT, Font.ITALIC, 5, 10);

The above example applies the Italic font attribute to characters 5 to 10 in the AttributedString.

Producing High-Level Text Manipulation Effects with an AttributedString

An AttributedString can be used to produce justification and other high-level text effects. The setJustification() method takes a single parameter: a float value between 0 and 1. A value of 0 indicates left-aligned text, and a value of 1 indicates fully-justified text.


          
           
str.setJustification(0.75f);

The example above sets the AttributedString's justification to 75% (3/4).
