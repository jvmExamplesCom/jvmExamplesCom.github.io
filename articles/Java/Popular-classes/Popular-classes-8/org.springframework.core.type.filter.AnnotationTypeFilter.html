Introduction to AnnotationTypeFilter

org.springframework.core.type.filter.AnnotationTypeFilter is a filter that can be used for matching a particular annotation type. In its most basic form, we can provide the class name of a single annotation to the constructor of this filter and then use it within a TypeFilter to detect if a type has that annotation or not.

Creating an AnnotationTypeFilter

Creating an AnnotationTypeFilter is pretty simple. All we need to do is to create a new AnnotationTypeFilter instance providing the required annotation class, as shown below:


          
           
AnnotationTypeFilter filter = new AnnotationTypeFilter(MyAnnotation.class);

Using AnnotationTypeFilter with TypeFilter

Once we have an AnnotationTypeFilter instance, we can use it within a TypeFilter to match a given type on the presence of the annotation or its absence:


          
           
TypeFilter filter = new TypeFilter() {
     public boolean match(MetadataReader metadataReader, MetadataReaderFactory metadataReaderFactory) {
         return annotationTypeFilter.match(metadataReader, metadataReaderFactory);
     }
};

Example

Now, let's see how we can use AnnotationTypeFilter and TypeFilter to match all classes that have a particular annotation. For example, let's say that we have an @MyAnnotation and we want to find all the classes in the classpath that are annotated with that annotation:


          
           
AnnotationTypeFilter annotationTypeFilter = new AnnotationTypeFilter(MyAnnotation.class);
ClassPathScanningCandidateComponentProvider scanner = 
     new ClassPathScanningCandidateComponentProvider(false);
scanner.addIncludeFilter(annotationTypeFilter);

for (BeanDefinition component : components) {
     Class<?> clazz = Class.forName(component.getBeanClassName());
     // do something with the class
}
