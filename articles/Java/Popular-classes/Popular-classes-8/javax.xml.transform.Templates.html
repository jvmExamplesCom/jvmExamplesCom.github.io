javax.xml.transform.Templates

Templates

TransformerFactory factory = TransformerFactory.newInstance();
Templates templates = factory.newTemplates(new StreamSource("filename.xsl"));

templates()

Applying Templates to XML Documents

Once a Templates object is created, it can then be applied to any syntactically matching XML document. This is done by using the 

          
           
templates.getTransformer()

Transformer transformer = templates.getTransformer();
transformer.transform(new StreamSource("data.xml"), new StreamResult("data.html"));
