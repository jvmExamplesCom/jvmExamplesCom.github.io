javax.xml.transform.Templates

The 

          
           
javax.xml.transform.Templates
object is an interface that provides a way to store and access objects related to XML transformations. This enables developers to create templates from the source document that can be applied to any syntactically matching XML document.

Creating a Templates Object

javax.xml.transform.TransformerFactory

Templates
object. This is an example of how to create such a Templates object: 

          
           
TransformerFactory factory = TransformerFactory.newInstance();
Templates templates = factory.newTemplates(new StreamSource("filename.xsl"));
This object will then provide the 

          
           
templates()
method which returns the Templates object.

Applying Templates to XML Documents

Once a Templates object is created, it can then be applied to any syntactically matching XML document. This is done by using the 

          
           
templates.getTransformer()
method. For example: 

          
           
Transformer transformer = templates.getTransformer();
transformer.transform(new StreamSource("data.xml"), new StreamResult("data.html"));
This code example transforms the XML document referenced in the StreamSource (data.xml) and saves the result of the transformation to a file named data.html.

templates.setParameter("method", "html");
Transformer transformer = templates.getTransformer();

Using Templates for Caching

The Templates object can also be used for caching purposes. Once a Templates object is created, that same Templates object can be reused for subsequent transformations. // Create the Templates object
