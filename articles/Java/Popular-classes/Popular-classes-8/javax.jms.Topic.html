javax.jms.Topic

The 
javax.jms.Topic
class is a special type of destination in Java Message Service (JMS). It is used to represent a particular jms topic and is an object-oriented representation of a jms topic in a Java program.

A topic provides a facility for subscribers and publishers to communicate so that when a publisher publishes an object, all the subscribed consumers will receive it. If there are no subscribed consumers, the message is lost.

Topics are most commonly used when application needs to have asychronous communications. In this case, one application will publish a message to a topic and any other applications that are interested in that message can subscribe to the same topic and receive it.

We can create a topic by using

method.

method.
