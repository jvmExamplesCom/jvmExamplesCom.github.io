The com.squareup.otto.Bus is a powerful tool for simplifying communication between components in any Android application. This article will explore how to use the Bus to send messages between components in an Android application.

The first step when using the Bus is creating an instance of it, which can be done by simply calling the constructor:


          
           
Bus bus = new Bus();

Next, any component which needs to be able to send or receive messages must register itself as a listener on the Bus. onRegister()

bus.register(listener);

bus.post(message);
