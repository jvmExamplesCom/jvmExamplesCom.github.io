org.apache.commons.pool2.impl.DefaultPooledObject

Overview

org.apache.commons.pool2.impl.DefaultPooledObject is an implementation of the PooledObject interface. It provides a common setup for pooling implementations, allowing for objects to be wrapped, synchronized, and tracked within a pool.
