org.springframework.web.bind.MissingServletRequestParameterException

The org.springframework.web.bind.MissingServletRequestParameterException is a Runtime exception that occurs when an expected request parameter is missing from the servlet request.

Example 1

In the below code example, if the request parameter named “name” is not present in the servlet request, then an instance of org.springframework.web.bind.MissingServletRequestParameterException will be thrown:


          
           
if (request.getParameter("name") == null) {
    throw new MissingServletRequestParameterException("name", "String");
}

In the below example, a MissingServletRequestParameterException is thrown when a request parameter “pageNumber” is missing in the request:


          
           
int pageNumber = 0;
try {
    pageNumber = Integer.parseInt(request.getParameter("pageNumber"));

    throw new MissingServletRequestParameterException("pageNumber", "Integer");
}
