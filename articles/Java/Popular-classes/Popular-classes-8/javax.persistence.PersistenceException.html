javax.persistence.PersistenceException

A javax.persistence.PersistenceException is an unchecked exception that is thrown when an error is encountered when attempting to access an entity manager or associated query. This exception is typically caused by either an incorrect configuration of the persistence unit or a database related communication issue.

  // Handle exception
}
