org.apache.http.impl.cookie.BasicClientCookie Overview

org.apache.http.impl.cookie.BasicClientCookie is a class for representing a basic cookie. It implements the ClientCookie interface and defines methods for managing attributes of a cookie such as domain, comment, version, etc.

Creating a BasicClientCookie Object

A BasicClientCookie object is created with a name and a value:


          
           
BasicClientCookie cookie = new BasicClientCookie("name", "value");

Setting Cookie Attributes

Once we have created a BasicClientCookie, we can set various attributes of the cookie such as domain, path, expiration date, etc. Following code shows how to set the domain and path of the cookie:


          
           
String domain = "example.com";
String path = "/";
cookie.setDomain(domain);
cookie.setPath(path);

Date expiryDate = new Date();
cookie.setExpiryDate(expiryDate);

Retrieving Cookie Attributes

We can get the value of a cookie attribute using the appropriate getXXX() method. For example, to get the domain of the cookie, we can use the getDomain() method:
