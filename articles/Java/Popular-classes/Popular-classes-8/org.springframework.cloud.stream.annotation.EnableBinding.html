Introduction to org.springframework.cloud.stream.annotation.EnableBinding

The org.springframework.cloud.stream.annotation.EnableBinding annotation is used to enable binding of the components of a message producer or consumer to the interfaces declared in the application context. This enables the application to bind to any number of interfaces, such as Kafka, AMQP, and file-based sources, and publish or consume messages. By default, all interfaces are bound to the same channel, though this may be overridden by passing arguments to the annotation.

Code Examples


          
           
@EnableBinding(MyBinder.class)
public class MyApp {
}

This code example binds the interface MyBinder to the application context, allowing it to bind to any number of interfaces.


          
           
@EnableBinding({MyBinder1.class, MyBinder2.class})

This code example binds two interfaces to the application.


          
           
@EnableBinding({MyBinder1.class, MyBinder2.class}, 
		channel="channelname")

This code example binds two interfaces to the application and overrides the default channel.
