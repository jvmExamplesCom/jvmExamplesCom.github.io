GlobalSearchScope in Java

GlobalSearchScope is an interface class in the com.intellij.psi.search package of the IntelliJ Platform API, used for accessing elements of a project's source code within the framework of global search.

Description

The GlobalSearchScope interface provides access to all the elements of a project's source code that are visible and accessible to the user, either through the project pane or via searches. This includes classes, methods, fields and constants declared directly in the source code as well as imports, libraries and packages. The interface also provides an API to navigate the source code by searching in various scopes. Scope defines the context within which the source code element is located.

Using GlobalSearchScope

The GlobalSearchScope API can be used to perform various source code operations such as finding all classes, methods, fields and constants in a project, and retrieving their declarations. It also enables users to search for specific elements by name or type, using different types of search scopes.

Methods

The GlobalSearchScope interface provides methods for creating and manipulating search scopes, as well as for finding elements in a project's source code. Some of the methods included in the interface are:


          
           
public static GlobalSearchScope getScopeRestrictedByFileTypes(GlobalSearchScope scope, FileType[] fileTypes)
public boolean contains(VirtualFile file)
public boolean accept(PsiFileSystemItem element)
public boolean acceptsLocalInnerClasses()
public GlobalSearchScope intersectWith(GlobalSearchScope other)
public String getDisplayName()
public String getId()
public GlobalSearchScope union(GlobalSearchScope other)

Examples

Here are some examples of how to use the GlobalSearchScope interface in Java code:


          
           
// Create search scope restricted to certain file types
GlobalSearchScope fileTypeScope = GlobalSearchScope.getScopeRestrictedByFileTypes(GlobalSearchScope.allScope(myProject), new FileType[] {JavaFileType.INSTANCE});

// Check if a search scope contains a given file
boolean containsFile = fileTypeScope.contains(myFile);

// Accepts an element in a search scope
boolean acceptsElement = fileTypeScope.accept(myPsiElement);

// Checks if a search scope includes local inner classes

// Retrieve display name for a search scope
