org.apache.commons.io.LineIterator

LineIterator

Iterator
interface inherited from Java's core 
Collection

. It provides the functionality to read lines from the readers.

Methods of LineIterator class

The 
LineIterator
class provides several methods for easy manipulation of the reader. The most important ones are listed below: 
  • hasNext()
    - checks if there is another line to read
  • nextLine()
    - retrieves the next line from the reader
  • close()
    - closes the reader

while (itr.hasNext()) {
    String line = itr.nextLine();
    System.out.println(line);
}

LineIterator
