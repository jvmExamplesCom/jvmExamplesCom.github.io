Introduction to Java com.google.common.collect.Queues

Queues are often used in multithreaded applications, where tasks are enqueued, executed and then dequeued after completion. myQueue.add(myElement1);
myQueue.offer(myElement2);
These methods differ in terms of how they handle errors. The `add` method will throw an exception if the queue is full, while the `offer` method will simply return `false`.
