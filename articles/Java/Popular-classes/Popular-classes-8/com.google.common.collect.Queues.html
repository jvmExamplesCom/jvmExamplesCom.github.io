Introduction to Java com.google.common.collect.Queues

Java com.google.common.collect.Queues is a set of utility classes for dealing with queues in java code. It provides support for both bounded and unbounded queues, as well as delayed and sorted queues. Additionally, it provides various implementations of queues such as the ArrayDeque and LinkedList classes. Queues are often used in multithreaded applications, where tasks are enqueued, executed and then dequeued after completion. This article will provide an overview of the Queues API and demonstrate how it can be used in Java applications.

Basic Usage

Queue<MyType> myQueue = Queues.newLinkedList();

Queue<MyType> myQueue = Queues.newLinkedList(10);
You can then add elements to the queue using the `add` or `offer` methods: 

          
           
myQueue.add(myElement1);
myQueue.offer(myElement2);
These methods differ in terms of how they handle errors. The `add` method will throw an exception if the queue is full, while the `offer` method will simply return `false`. You can remove elements from the queue using the `remove` or `poll` methods. These methods behave similarly to their counterparts when adding elements, with `remove` throwing an exception when the queue is empty and `poll` returning `null`.

int size = myQueue.size();
