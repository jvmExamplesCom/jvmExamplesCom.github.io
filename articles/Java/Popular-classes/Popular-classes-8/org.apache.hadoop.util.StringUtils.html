Introduction to org.apache.hadoop.util.StringUtils

The org.apache.hadoop.util.StringUtils class is a collection of many utility methods for the manipulation of Java strings. It contains all the standard string-handling functions such as trim, replace, and substring; along with more advanced methods such as escaping and unescaping HTML entities, converting case, and computing hashcodes for strings.

Examples of org.apache.hadoop.util.StringUtils Methods

Below are some examples of the StringUtils class methods in action.

trim()

The 
trim()
method will remove leading and trailing whitespace from a String. 

          
           
String s = "  Hello World  ";
String sTrimmed = StringUtils.trim(s);  //sTrimmed will equal "Hello World"

replace()

The 
replace()
method can be used to find and replace strings inside of another string. 

          
           
String s = "Hello World";
String sReplaced = StringUtils.replace(s, "World", "Universe");  //sReplaced will equal "Hello Universe"

isAlpha()

The 
isAlphabet()
method will check if a given string is composed only of alphabetic characters. 

          
           
String s = "HelloWorld";
boolean isAlphabet = StringUtils.isAlpha(s);  //isAlphabet will equal false

Conclusion

The org.apache.hadoop.util.StringUtils class is an invaluable tool for manipulating strings within the context of Java programming. By utilizing its many methods, tasks such as trimming, replacing strings within other strings, and converting to different cases can be done quickly and easily.
