Introduction to IO.ReactiveX.Functions.Predicate

IO.ReactiveX.Functions.Predicate is a form of higher order function that allows developers to express complex logic in concise and elegant code. It returns true or false, depending upon the state of the object or data passed as an argument. Predicates are particularly useful in filtering collections of objects or data in Java, as they allow developers to define a condition once, avoiding code repetition.

Java syntax for Predicate

The Syntax for creating a predicate in Java is: 

          
           
Predicate<Type> predicateName = (parameters) -> (logic expression);

Where Type is the type of argument passed to the function, and the expression is a boolean expression.

Example of Predicate in Java

Let's look at an example of how to use a Predicate in Java. We will create a Predicate to determine if an integer is even or odd, and then loop through an array of integers, printing out whether each integer is even or odd. Here is our code:


          
           
Predicate<Integer> isEven = x -> x % 2 == 0;

int[] numbers = {1, 2, 3, 4, 5, 6};

for (int number : numbers) {
  if (isEven.test(number)) {
    System.out.println(number + " is even");
  } else {
    System.out.println(number + " is odd");
  }
}

When we run this code, it produces the following output:


          
           
1 is odd
2 is even
3 is odd
4 is even
5 is odd
6 is even

Using Predicates for Filtering Collections

List<Person> allPersons = ...;  //List with Person objects

Predicate<Person> isOver30 = person -> person.getAge() > 30;

Here, we used the Stream API method 
.filter()
