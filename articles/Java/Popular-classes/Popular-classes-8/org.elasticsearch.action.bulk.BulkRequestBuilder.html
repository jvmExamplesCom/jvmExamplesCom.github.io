Java org.elasticsearch.action.bulk.BulkRequestBuilder

Java org.elasticsearch.action.bulk.BulkRequestBuilder is a powerful tool to send multiple requests in a single request to Elasticsearch in an efficient manner. It helps reduce the number of requests made, thereby improving performance and reducing network latency. It makes life easier for Java developers when working with Elasticsearch.

Overview

The BulkRequestBuilder is responsible for creating the bulk requests, which can contain up to 1024 different requests at once. These requests are converted into a JSON format and sent to the Elasticsearch server in a single request. The response from the server is then parsed and converted into the appropriate response objects.

Example Code

Here is an example of how to use the BulkRequestBuilder to add documents to an index: 

          
           
BulkRequestBuilder bulkRequest = new BulkRequestBuilder();

// Add a document to the index
Map<String, Object> document1 = new HashMap<>();
document1.put("title", "my title");
document1.put("content", "my content");

IndexRequest indexRequest = new IndexRequest("index_name", "type_name", "1")
    .source(document1);

bulkRequest.add(indexRequest);

// Add another document to the index
Map<String, Object> document2 = new HashMap<>();
document2.put("title", "another title");
document2.put("content", "another content");

IndexRequest indexRequest2 = new IndexRequest("index_name", "type_name", "2")
    .source(document2);

bulkRequest.add(indexRequest2);

// Execute the bulk request
BulkResponse bulkResponse = bulkRequest.execute().actionGet();
if (bulkResponse.hasFailures()) {
    // handle failures
}
