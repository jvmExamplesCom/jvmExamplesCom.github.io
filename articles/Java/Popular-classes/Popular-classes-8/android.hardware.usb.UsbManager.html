android.hardware.usb.UsbManager

The android.hardware.usb.UsbManager class provides access to USB devices connected to an Android device. It enables applications to discover available USB devices and request permission from the user to access them.

Getting Started

To get started with the UsbManager class, you will need to get a handle of the UsbManager object by getting a reference from the Android application context. This can be done by calling the getSystemService() method with the parameter “usb”.


          
           
UsbManager manager = (UsbManager) context.getSystemService(Context.USB_SERVICE);

Accessing Attached USB devices

Once you have the UsbManager instance, you can access a list of all attached devices with the getDeviceList() method. This method returns a HashMap containing UsbDevice instances which represent the attached usb devices.


          
           
HashMap<String, UsbDevice> deviceList = manager.getDeviceList();

You can access each UsbDevice instance in the deviceList HashMap using the get() method.

Before your application is able to communicate with a USB device, it must first acquire permission from the user. The UsbManager class provides this functionality with the requestPermission() method. This method takes 2 parameters, the UsbDevice instance that you wish to access, and a PendingIntent which will be used to notify the caller of the result of the permission request.


          
           
PendingIntent pendingIntent = PendingIntent.getBroadcast(context, 0, new Intent(""), 0);
