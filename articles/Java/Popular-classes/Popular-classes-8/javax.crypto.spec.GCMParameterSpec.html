javax.crypto.spec.GCMParameterSpec

The javax.crypto.spec.GCMParameterSpec class represents the parameter set used with the Galois/Counter Mode (GCM) mode of operation for block ciphers. It is an one-time authenticated data encryption method for providing confidentiality, integrity, and authenticity for transmitted messages. GCM uses a variable length authentication tag and supports data expansion, making it ideal for use in applications that require authenticated encryption with associated data (AEAD).

Example 1: Encryption using GCMParameterSpec


          
           
SecretKey secretKey = new SecretKeySpec("MySecretKey".getBytes(), "AES");
Cipher cipher = Cipher.getInstance("AES/GCM/NoPadding");

// Initialize Cipher for ENCRYPT_MODE
cipher.init(Cipher.ENCRYPT_MODE, secretKey);

// Set IV and GCMParameterSpec
byte[] iv = ...; // Generate random IV
GCMParameterSpec gcmParams = new GCMParameterSpec(16 * Byte.SIZE, iv);
cipher.init(Cipher.ENCRYPT_MODE, secretKey, gcmParams);

// Perform Encryption
byte[] cipherText = cipher.doFinal("My Plaintext".getBytes());

Example 2: Decryption using GCMParameterSpec


          
           
SecretKey secretKey = new SecretKeySpec("MySecretKey".getBytes(), "AES");
Cipher cipher = Cipher.getInstance("AES/GCM/NoPadding");

// Initialize Cipher for DECRYPT_MODE
cipher.init(Cipher.DECRYPT_MODE, secretKey);

// Set IV and GCMParameterSpec
byte[] iv = ...; // Obtain the same IV used for encryption
GCMParameterSpec gcmParams = new GCMParameterSpec(16 * Byte.SIZE, iv);
cipher.init(Cipher.DECRYPT_MODE, secretKey, gcmParams);

// Perform Decryption
byte[] plainText = cipher.doFinal(cipherText);
