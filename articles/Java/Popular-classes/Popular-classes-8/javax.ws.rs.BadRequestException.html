javax.ws.rs.BadRequestException

The javax.ws.rs.BadRequestException is an exception class derived from the WebApplicationException class and is thrown when the request sent to the resource is invalid. It serves as an indication that the actual request sent by the client was not in proper form, and should be corrected before sending it to the server. This can be used to indicate that a particular parameter is missing, incorrect, or is of an invalid data type.

    conn.setRequestMethod("GET");
    conn.connect();
	
    //Check if request was successful
    int responseCode = conn.getResponseCode();
    if (responseCode != 200) {
        throw new BadRequestException("Bad Request");
