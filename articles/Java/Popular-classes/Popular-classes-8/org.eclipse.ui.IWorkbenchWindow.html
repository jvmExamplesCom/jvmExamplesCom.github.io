org.eclipse.ui.IWorkbenchWindow

The org.eclipse.ui.IWorkbenchWindow interface defines the methods for accessing workbench windows. It provides access to the basic controls for a workbench window, including its size, position, and display state.

Creating a Window

A new window can be created using the 

          
           
IWorkbench.createWorkbenchWindow
method. This method takes an IAdaptable object as a parameter which can contain the initial settings for the window. 

          
           
IWorkbenchWindow window =
  PlatformUI.getWorkbench().createWorkbenchWindow(null);

Configuring a Window

The window can be configured by setting the size, position, and other properties. The position of the window is defined by a Rectangle object, with the x and y coordinates defining the top-left corner of the window.


          
           
Rectangle rect = new Rectangle(100, 100, 500, 400);
window.setBounds(rect);
The size of the window can also be set using the 

          
           
IWorkbenchWindow.setSize
method. This method takes two parameters - the width and height of the window. For example: 

          
           
window.setSize(800, 600);

Other properties such as the title of the window can be set using the 


          
           
IWorkbenchWindow.setTitle
method: 

          
           
window.setTitle("My WorkbenchWindow");

Showing a Window

Once the window has been configured, it can be shown on the screen by calling the 


          
           
IWorkbenchWindow.open
method. This method takes an optional parameter which is a boolean flag indicating whether the window should be visible when it is opened. 

          
           
window.open(true);

Closing a Window

A window can be closed by calling the 

          
           
IWorkbenchWindow.close
method. This method takes a boolean flag as a parameter which indicates if the window should be closed immediately or if it should be closed after all the open windows in the workbench session have been closed. 

          
           
window.close(true);
