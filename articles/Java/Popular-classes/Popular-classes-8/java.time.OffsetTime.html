Java.time.OffsetTime Explained

Java.time.OffsetTime is a class in the Java Time API that represents a time with an offset-from-UTC/Greenwich, intended primarily for use in date-time calculations. It stores the time as a combination of a LocalTime and an offset from UTC/Greenwich. For example, 10pm in Paris, France is 9:00pm GMT, represented as 21:00+01:00. This is easily represented by OffsetTime. An OffsetTime can be constructed from the following parameters:
  • LocalTime - the local time part
  • ZoneOffset - the offset part
  • OffsetDateTime - combines a LocalDate and OffsetTime

Methods

OffsetTime has various methods, such as:
  • now – creates an instance for the current moment with the offset specified.
  • getHour – Returns the hour-of-day field.
  • getMinute – Returns the minute-of-hour field.
  • getSecond – Returns the second-of-minute field.
  • plusHours – Creates an OffsetTime with the specified hour added.
  • minusHours – Creates an OffsetTime with the specified hour removed.
  • toString – Outputs a string representation of the time.

Conclusion

OffsetTime is a useful class to have in your toolkit when working with date-times in Java. It allows you to represent a point in time within a given offset.
