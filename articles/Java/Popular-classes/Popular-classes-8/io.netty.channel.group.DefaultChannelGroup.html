DefaultChannelGroup Overview

DefaultChannelGroup is a specialized ChannelGroup implementation that stores the members of the group in a thread-safe Set. In addition to managing the collection, it also provides several utility methods for broadcast operations. The following example shows how to create and use a DefaultChannelGroup: 

          
           
// Create a new DefaultChannelGroup
DefaultChannelGroup channelGroup = new DefaultChannelGroup();

// Add Channels to the group
Channel ch1 = ...;
channelGroup.add(ch1);

// Broadcast a message
channelGroup.writeAndFlush("Hello World!");

// Close the Group when done
channelGroup.close();
The DefaultChannelGroup also provides methods for querying the size of the group, checking for membership, and iterating over each member of the group.
