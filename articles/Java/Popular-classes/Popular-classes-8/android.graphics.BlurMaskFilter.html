android.graphics.BlurMaskFilter

The android.graphics.BlurMaskFilter is a class in the Android API that can be used to create a blur effect by using a mask filter. This class can be used with Paint objects and is designed for blurring bitmap images.

Understanding BlurMaskFilter

The BlurMaskFilter class works by creating a set of kernels for each pixel in an image. The kernel is then the sum of all the surrounding pixels times a weighting parameter. The resulting pixel value is the sum of the kernel. For instance, if you had a kernel of [0.2, 0.2, 0.2], the pixel at (x, y) would be the sum of the values at (x-1, y-1), (x, y-1) and (x+1, y-1) multiplied by 0.2. This can be done with any pixel in the image and it will produce a blur effect.

Using BlurMaskFilter

Using BlurMaskFilter is simple and requires only a few lines of code. First, you need to create an instance of the BlurMaskFilter class: 

          
           
BlurMaskFilter blurMaskFilter = new BlurMaskFilter(radius, BlurMaskFilter.Blur.NORMAL);
The 

          
           
radius
parameter determines how much blurring should occur. Then you need to set your Paint object to use the BlurMaskFilter: 

          
           
Paint paint = new Paint();
paint.setMaskFilter(blurMaskFilter);
Finally, you need to draw the image with this Paint object to get the desired blur effect:
