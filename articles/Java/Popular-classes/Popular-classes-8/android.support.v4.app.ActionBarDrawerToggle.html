The android.support.v4.app.ActionBarDrawerToggle class is used to create a toggleable, sliding drawer menu in Android apps. The ActionBarDrawerToggle class works with both the ActionBar and Toolbar classes to provide a consistent navigation experience across multiple Android versions.

mDrawerLayout.addDrawerListener(toggle);
