android.support.v4.app.ActionBarDrawerToggle

The android.support.v4.app.ActionBarDrawerToggle class is used to create a toggleable, sliding drawer menu in Android apps. This user interface element provides a convenient way for app users to navigate between different sections of an app without needing to use a back button or remember where they are. The ActionBarDrawerToggle class works with both the ActionBar and Toolbar classes to provide a consistent navigation experience across multiple Android versions.

Creating an ActionBarDrawerToggle Instance

The first step in using ActionBarDrawerToggle is to create an instance of the class. This is done with the following code: 

          
           
ActionBarDrawerToggle toggle = new ActionBarDrawerToggle(this, mDrawerLayout, toolbar, R.string.open_drawer, R.string.close_drawer);
mDrawerLayout.addDrawerListener(toggle);
toggle.syncState();
