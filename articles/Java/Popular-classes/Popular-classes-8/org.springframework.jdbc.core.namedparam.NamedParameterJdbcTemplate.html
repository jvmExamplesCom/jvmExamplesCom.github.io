Using Named Parameters

The main benefit of NamedParameterJdbcTemplate is that it allows you to use named parameters in your SQL statements. These names correspond to properties in your Java beans. String sql = "SELECT * FROM my_table WHERE first_name = :firstName AND last_name = :lastName";
