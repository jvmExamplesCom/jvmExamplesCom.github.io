net.minecraft.entity.EntityLivingBase

Java class net.minecraft.entity.EntityLivingBase is the base class for all living entities in Minecraft. It provides basic properties and functions that are common to all living entities in the game, such as health and movement. This article will provide a deep dive into the class and its associated functions.

Properties

The EntityLivingBase class contains several properties that are used to provide data about the entity:

  • worldObj
    : A reference to the World containing the entity.
  • posX
    , 
    posY
    , 
    posZ
    : The coordinates of the entity in world coordinates.
  • prevPosX
    , 
    prevPosY
    , 
    prevPosZ
    : The coordinates of the entity in the previous tick.
  • motionX
    , 
    motionY
    , 
    motionZ
    : The velocity of the entity in each direction.
  • health
    : The current health of the entity.
  • maxHealth
    : The maximum health of the entity.

Methods

The EntityLivingBase class provides several methods that are useful for manipulating the entity:

  • 
            
             
setPosition(double x, double y, double z)
    : Sets the position of the entity to the specified coordinates.
  • 
            
             
setVelocity(double x, double y, double z)
    : Sets the velocity of the entity to the specified velocity.
  • 
            
             
damageEntity(DamageSource source, float amount)
    : Inflicts damage to the entity from the specified source with the specified amount.
  • heal(float amount)
    : Restores health to the entity with the specified amount.
  • isAlive()
    : Returns true if the entity is alive (health greater than 0), false otherwise.
  • onUpdate()
    : Called every tick to update the entity's state.
