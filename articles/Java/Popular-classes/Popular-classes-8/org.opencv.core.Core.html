A Comprehensive Overview of the Java org.opencv.core.Core Library

In Java programming, the openCV Core library (org.opencv.core.Core) provides a wide range of functionalities that allow developers to manipulate, detect and recognize images within the code. This article will provide an overview of the features included in this library and demonstrate them with examples.

Importing Core Library

To make use of the Core library in a Java program, we must first import it.


          
           
import org.opencv.core.Core;

Mat Functions

The Mat() function is a constructor used to create a new matrix object in OpenCV. It can take three parameters:

  • Number of Rows
  • Number of Columns
  • Data type
The data type must be one of the following:
  • CV_8U – 8-bit unsigned integers ( 0..255 )
  • CV_8S – 8-bit signed integers ( -128..127 )
  • CV_16U – 16-bit unsigned integers ( 0..65535 )
  • CV_16S – 16-bit signed integers ( -32768..32767 )
  • CV_32S – 32-bit signed integers ( -2147483648..2147483647 )
  • CV_32F – 32-bit floating-point numbers ( -FLT_MAX..FLT_MAX, INF, NAN )
  • CV_64F – 64-bit floacomparing numbers ( -DBL_MAX..DBL_MAX, INF, NAN )

          
           
Mat m = new Mat(4,4, CV_8U);

The Mat object can also take in an array of doubles.


          
           
double[] vals = {1.0,2.0,3.0,4.0};
Mat m = new Mat(3,3,CV_64F, vals);

Split Method

The split() method can be used to split a single matrix into separate channels. The method takes in the destination channels to store the split data. The number of destination channels must match the number of channels in the source matrix.

Threshold Method

The threshold() method is used to apply a threshold to a matrix. The method takes in three parameters: the source matrix, the destination matrix, and the threshold value.

src.threshold(dest, 128); // Apply threshold to source image

The default threshold type is THRESH_BINARY, which sets all the values above the threshold to 255 (white) and all the values below the threshold to 0 (black). There are other types of thresholding, such as THRESH_BINARY_INV, which sets all the values above the threshold to 0 and all the values below the threshold to 255.
