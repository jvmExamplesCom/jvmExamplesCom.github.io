java.time.YearMonth

Java.time.YearMonth is a date-time API that provides classes representing year, month, and its combination. It allows applications to get information such as the total numbers of days in a month or year, adding or subtracting years and months from a particular instance of YearMonth, etc.

Creating a YearMonth Instance:

The YearMonth has two underlying systems, one using ISO-8601 calendar system and other using another calendar system called Proleptic Gregorian calendar. We can create an instance of YearMonth using either of the calendar systems. The default calendar system displayed is the ISO-8601 which stores but does not display negative years.


          
           
//creating a YearMonth instance using ISO-8601 calendar system
YearMonth yearMonth = YearMonth.of(2019, Month.JULY);
 
//creating a YearMonth instance using Proleptic Gregorian calendar system
YearMonth yearMonth2 = YearMonth.of(1944, Month.AUGUST);
System.out.println(yearMonth2); // 1944-08

Getting Information About a YearMonth Instance:

We can use the 
lengthOfMonth()
and 
lengthOfYear()
methods provided by the API to get the total number of days in a month or year for a given YearMonth instance. 

          
           
int daysInMonth = yearMonth.lengthOfMonth();
int daysInYear = yearMonth.lengthOfYear();
System.out.println(daysInMonth); // 31
System.out.println(daysInYear); // 364

We can also get the current month or year as a string using the 

toString()
method. 

          
           
String strMonth = yearMonth.toString();
System.out.println(strMonth); // 2019-07

Adding or Subtracting Months/Years:

We can add or subtract months or years to a YearMonth instance by using the 

plusMonths()
and 
minusMonths()
methods. 

          
           
//adding 3 months
YearMonth plusMonths = yearMonth.plusMonths(3);
System.out.println(plusMonths); // 2019-10
 
//subtracting 4 years
YearMonth minusYears = yearMonth.minusYears(4);
System.out.println(minusYears); // 2015-07

Comparing YearMonth Instances:

We can compare YearMonth instances using the 
isBefore()
, 
isAfter()
and 
equals()
methods provided by the API. These methods return true if one instance is respectively before, after or equal to the other one. 

          
           
YearMonth otherYearMonth = YearMonth.of(2020, Month.AUGUST);

boolean isBefore = yearMonth.isBefore(otherYearMonth);
boolean isAfter = yearMonth.isAfter(otherYearMonth);
boolean isEqual = yearMonth.equals(otherYearMonth);

System.out.println(isBefore); // true

Java.time.YearMonth provides useful methods for obtaining information about a date-time, adding/subtracting months and years, and comparing YearMonth instances. It is a powerful API that allows applications to easily manipulate date-time data.
