Introduction to Java and DefaultPrettyPrinter

Java is an object-oriented programming language developed by Sun Microsystems in 1995. It allows developers to create cross-platform applications that can work on any operating system such as Windows, Mac, Linux, etc. Java is used in a wide variety of domains, from web and enterprise applications to mobile applications and scientific applications. One important feature of Java is its rich set of libraries and packages. These libraries and packages allow developers to quickly develop powerful applications without having to write many lines of code. One such library is the Jackson JSON library. This library contains a utility class called DefaultPrettyPrinter which is used for printing out nicely formatted JSON text.

What is DefaultPrettyPrinter?

DefaultPrettyPrinter is a utility class that is a part of the Jackson library. This class contains methods for formatting of JSON text. It provides the ability to format JSON text into a user-defined layout. This makes it easier to read and understand the output. Using the DefaultPrettyPrinter class, developers can specify the indentation size, line separator, number of spaces per indentation and so on. The printed JSON text can be customized as per the specific needs of the application.

Example Code

Below is an example code to demonstrate the use of the DefaultPrettyPrinter class. This code creates an instance of the class and then prints out some JSON text in a custom format. 

          
           
ObjectMapper objectMapper = new ObjectMapper();
DefaultPrettyPrinter prettyPrinter = new DefaultPrettyPrinter();
prettyPrinter.indentArraysWith(DefaultIndenter.SYSTEM_LINEFEED_INSTANCE);
objectMapper.setDefaultPrettyPrinter(prettyPrinter);

String json = "{\"name\": \"Bob\", \"age\": 25}";

try {
    JsonNode rootNode = objectMapper.readTree(json);
    String formattedJson = objectMapper.writerWithDefaultPrettyPrinter().writeValueAsString(rootNode);
    System.out.println(formattedJson);
} catch (IOException e){
    e.printStackTrace();
}
The output of the code above will be: 

          
           
{
  "name": "Bob",
  "age": 25
}
In the above code, the indentation size is specified using the “indentArraysWith” method. This method takes in a DefaultIndenter instance which can be configured to use different indent sizes. In this example, the “SYSTEM_LINEFEED_INSTANCE” has been used to set the indentation size to 4 spaces. Other available options are SINGLE_SPACE_INSTANCE (1 space) and TWO_SPACES_INSTANCE (2 spaces).
