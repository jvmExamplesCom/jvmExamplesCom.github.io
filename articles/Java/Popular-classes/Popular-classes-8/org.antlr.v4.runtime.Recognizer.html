org.antlr.v4.runtime.Recognizer

The Recognizer class is an abstract base class which provides the basic functionality used by all ANTLR parsers. It provides the ability to recognize a TokenStream, to build a parse tree from the recognized symbols and to generate a parse tree in response to a parse request.

Methods

The Recognizer class contains the following methods:

  • getInputStream(): Returns the TokenStream currently being parsed.
  • setInputStream(TokenStream input): Sets the TokenStream to be parsed.
  • match(int ttype): Matches the next token against the specified token type.
  • consume(): Consumes the next token from the TokenStream, effectively removing it from the stream.
  • getCurrentToken(): Returns the current token in the TokenStream.
  • getRuleIndex(String ruleName): Returns the rule index for the specified rule name.
  • getRuleInvocationStack(): Returns the stack of rule invocations.
  • getSourceName(): Returns the source name associated with the parser.

import org.antlr.v4.runtime.TokenStream;

        this.tokens = tokens;
    }

        return tokens;
    }

    @Override
    public void setInputStream(TokenStream input) {
        this.tokens = input;
    }

    @Override
    public MyTokenType match(int ttype) throws Exception {
        //implement custom matching logic
    }

    @Override
    public void consume() {
        tokens.consume();
    }
}
