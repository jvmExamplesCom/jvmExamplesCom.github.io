Introduction to org.joda.time.LocalDateTime

org.joda.time.LocalDateTime is a Java class used to represent a date and time. It stores a combination of year, month, day, hour, minute, second, time zone offset, and daylight saving time offset. LocalDateTime provides numerous methods for getting, setting, and manipulating individual fields from the date and time. This class also provides methods for comparison, formatting, and parsing of ISO 8601 dates. This article will provide an introduction to this class and demonstrate how to use its various methods.

Creating a LocalDateTime Object

The most basic way to create a LocalDateTime object is to set it to the current system time:


          
           
LocalDateTime now = new LocalDateTime();

Alternatively, you can pass in a specific date and time when creating the object:


          
           
LocalDateTime dateTime = new LocalDateTime(2018, 7, 3, 12, 45);

The LocalDateTime class also contains several static factory methods that can be used to construct a LocalDateTime object based on different parameters:

// Create a LocalDateTime from a text string in ISO8601 format
