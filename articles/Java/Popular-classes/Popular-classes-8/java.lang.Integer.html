Java.lang.Integer

The Java.lang.Integer class wraps a value of the primitive type int in an object. An object of type Integer contains a single field whose type is int. In addition, this class provides several methods for converting an int to a String and a String to an int, as well as other constants and methods useful when dealing with an int.

Constructors

The Java.lang.Integer class has two constructors: 
  • 
            
             
public Integer(int value)
    : creates an Integer object initialized with the specified int value.
  • 
            
             
public Integer(String s)
    : creates an Integer object initialized with the value of the specified string.

Methods

The Java.lang.Integer class has a number of methods for manipulating integers:

  • 
            
             
public static int parseInt(String s)
    : converts the specified string to an int.
  • 
            
             
public static String toString(int i)
    : converts the specified int to a String.
  • 
            
             
public static int compare(int x, int y)
    : compares the two specified int values.
  • 
            
             
public static int min(int a, int b)
    : returns the smaller of the two specified int values.
  • 
            
             
public static int max(int a, int b)
    : returns the larger of the two specified int values.
  • 
            
             
public static int abs(int a)
    : returns the absolute value of the specified int value.

Example

Let's look at an example that demonstrates how to use the Java.lang.Integer class.


          
           
public class Example {
    public static void main(String[] args) {
