java.android.app.WallpaperManager

Java's android.app.WallpaperManager class provides methods to manage device wallpapers. It allows you to set and get the current system wallpaper, set different wallpapers for certain devices and apply effects on them.

Setting Wallpapers

To set a wallpaper, use the setBitmap() method. This takes a Bitmap object representing the image you want to set as the wallpaper. 

          
           
Bitmap bitmap = BitmapFactory.decodeResource(context.getResources(), R.drawable.my_image);
WallpaperManager wallpaperManager = WallpaperManager.getInstance(context);
wallpaperManager.setBitmap(bitmap);
You can also specify a Uri object to point to an image file. 

          
           
WallpaperManager wallpaperManager = WallpaperManager.getInstance(context);
Uri myImageUri = Uri.parse("file:///sdcard/images/image.jpg");
wallpaperManager.setUri(myImageUri);

Getting Wallpapers

The getWallpaperFile() method returns the Uri of the currently set wallpaper. 

          
           
WallpaperManager wallpaperManager = WallpaperManager.getInstance(context);
Uri currentWallpaperUri = wallpaperManager.getWallpaperFile();
The getBitmap() method returns the currently set wallpaper as a Bitmap object. 

          
           
WallpaperManager wallpaperManager = WallpaperManager.getInstance(context);
Bitmap currentWallpaperBitmap = wallpaperManager.getBitmap();

Applying Effects on Wallpapers

You can apply effects to an existing wallpaper using the setWallpaperOffsets() method. This method takes two parameters: x and y, where you can set the amount of offset in either direction. 

          
           
float x = 0.1f;
float y = 0.2f;
WallpaperManager wallpaperManager = WallpaperManager.getInstance(context);
wallpaperManager.setWallpaperOffsets(x, y);
You can also scale the wallpaper image by setting its width and height. 

          
           
int width = 500;
int height = 1000;
WallpaperManager wallpaperManager = WallpaperManager.getInstance(context);
wallpaperManager.suggestDesiredDimensions(width, height);
