Overview of org.apache.hadoop.hbase.HTableDescriptor

The HTableDescriptor includes such information as the name of the table, its column families, and their various attributes. It also provides methods for accessing and changing table properties as well as methods for copying, cloning or serializing it.

The second constructor takes an additional parameter which is the list of column family names that the table should contain.
