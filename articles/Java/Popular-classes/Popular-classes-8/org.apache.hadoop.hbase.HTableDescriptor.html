Overview of org.apache.hadoop.hbase.HTableDescriptor

org.apache.hadoop.hbase.HTableDescriptor is a class that provides an interface for creating and manipulating HBase tables in an HBase cluster. The HTableDescriptor includes such information as the name of the table, its column families, and their various attributes. It also provides methods for accessing and changing table properties as well as methods for copying, cloning or serializing it.

Constructors

The second constructor takes an additional parameter which is the list of column family names that the table should contain.
