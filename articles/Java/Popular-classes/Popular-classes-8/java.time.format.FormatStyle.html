Overview of java.time.format.FormatStyle

The java.time.format.FormatStyle class defines two constants that control the style of date and time formatting. It is part of the Java Date/Time API introduced with Java 8.

Usage

The FormatStyle enum is used to determine the format of a date or time instance when converting it to a string. It is used in the DateTimeFormatter class, which provides various formatting options.

Constants

The FormatStyle class has two constants: FULL and LONG. These constants define the length of the format.

FULL

The FULL constant specifies the full length format, including month names, day names, and days of the week. This is the default option for the DateTimeFormatter when no other format is specified.


          
           
DateTimeFormatter formatter = DateTimeFormatter.ofLocalizedDateTime(FormatStyle.FULL);
LocalDateTime dateTime = LocalDateTime.now();
String formattedDateTime = dateTime.format(formatter);
System.out.println(formattedDateTime); // Wednesday, April 15, 2020 9:00 PM

LONG

The LONG constant specifies a shorter format, including numeric month and day values, and abbreviated day names. This is useful for displaying dates in a more concise format.


          
           
DateTimeFormatter formatter = DateTimeFormatter.ofLocalizedDateTime(FormatStyle.LONG);
LocalDateTime dateTime = LocalDateTime.now();
String formattedDateTime = dateTime.format(formatter);
System.out.println(formattedDateTime); // Apr 15, 2020 9:00 PM

Conclusion

The java.time.format.FormatStyle class provides a convenient way to control the formatting of dates and times. With just two constants, developers can easily specify the desired formatting of date and time values.
