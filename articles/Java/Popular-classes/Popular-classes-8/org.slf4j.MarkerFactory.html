Java provides an org.slf4j.MarkerFactory class which allows for the creation of marker objects. The exists method is used to check if a Marker with the specified name exists. Let's look at a few code examples to see how these methods work. To get started, we will create a new Marker object: 

          
           
Marker myMarker = MarkerFactory.getMarker("my_marker");

LOGGER.debug(myMarker, "This is my message");
