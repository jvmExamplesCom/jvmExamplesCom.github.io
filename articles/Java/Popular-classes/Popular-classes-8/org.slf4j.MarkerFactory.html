org.slf4j.MarkerFactory

Java provides an org.slf4j.MarkerFactory class which allows for the creation of marker objects. A marker is a data structure that can be used to filter, categorize, and organize Java logging messages. Markers are hierarchical, meaning they may contain other markers as children. The MarkerFactory class provides two methods that can be used to create and manage Marker objects: getMarker and exists. The getMarker method is used to check if a Marker with the specified name already exists. If it does not exist, it will create it. The exists method is used to check if a Marker with the specified name exists. Let's look at a few code examples to see how these methods work. To get started, we will create a new Marker object: 

          
           
Marker myMarker = MarkerFactory.getMarker("my_marker");

LOGGER.debug(myMarker, "This is my message");

Boolean markerExists = MarkerFactory.exists("my_marker");
