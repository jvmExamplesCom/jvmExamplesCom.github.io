<dependency>

  <version>2.2.9.RELEASE</version>
</dependency>

KafkaTemplate<String, String> kafkaTemplate = new KafkaTemplate<>(producerFactory);

KafkaTemplate<String, String> kafkaTemplate = new KafkaTemplate<>(consumerFactory);

When creating a KafkaTemplate, you need to provide it with a ProducerFactory or ConsumerFactory, depending on whether you want to use it to send or receive messages. These factories provide the template with the necessary configuration to communicate with the Apache Kafka broker.
