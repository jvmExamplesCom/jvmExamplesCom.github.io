org.springframework.kafka.core.KafkaTemplate

KafkaTemplate is a utility class that simplifies the process of sending and receiving messages in Apache Kafka through Spring. It is part of the org.springframework.kafka package and provides convenient methods for sending and receiving messages to and from the Apache Kafka broker.

To get started with KafkaTemplate, first you need to add the necessary dependencies to your project. The following example uses the latest version of the spring-kafka library: 

          
           
<dependency>
  <groupId>org.springframework.kafka</groupId>
  <artifactId>spring-kafka</artifactId>
  <version>2.2.9.RELEASE</version>
</dependency>

KafkaTemplate<String, String> kafkaTemplate = new KafkaTemplate<>(producerFactory);

KafkaTemplate<String, String> kafkaTemplate = new KafkaTemplate<>(consumerFactory);

When creating a KafkaTemplate, you need to provide it with a ProducerFactory or ConsumerFactory, depending on whether you want to use it to send or receive messages. These factories provide the template with the necessary configuration to communicate with the Apache Kafka broker. The following example shows how to create a ProducerFactory using the Spring Boot auto-configuration:
