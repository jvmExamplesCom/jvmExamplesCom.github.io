An Introduction to org.jfree.chart.axis.NumberAxis

org.jfree.chart.axis.NumberAxis is a class in the JFreeChart library that provides a number axis to a chart. It supports auto ranging, which finds a suitable range for the data it contains. It can also handle negative and positive numbers, provide tick units and an associated label generator. This article will explore some of its features and provide code examples.

Constructors

The main constructor of the NumberAxis class takes a String parameter, which is the name of the axis (e.g. the x-axis or y-axis). It also has several alternate constructors which allow you to specify the range and tick unit.


          
           
NumberAxis axis = new NumberAxis("Axis Name");
NumberAxis axis = new NumberAxis("Axis Name", 0, 10, 2);
NumberAxis axis = new NumberAxis("Axis Name", -10, 10, 5);

Auto Ranging

Auto ranging allows the number axis to automatically set the range based on the data that it contains. This can be enabled or disabled by calling the setAutoRange() method on the NumberAxis.


          
           
NumberAxis axis = new NumberAxis("Axis Name");
axis.setAutoRange(true); // enable auto ranging
axis.setAutoRange(false); // disable auto ranging

Negative Numbers

The number axis can also handle negative numbers. To do this, the lower bound of the range must be less than zero. This can be done using one of the alternative constructors which takes a range as a parameter, or by explicitly setting the range using the setRange() method.


          
           
NumberAxis axis = new NumberAxis("Axis Name", -10, 10);
axis.setRange(-20, 20);

Tick Units

The number axis allows for different tick units, which determine the increments between each tick mark. Tick units can be specified when constructing the NumberAxis, or they can be set later with the setTickUnit() method.


          
           
NumberAxis axis = new NumberAxis("Axis Name", 0, 10, 2); //tick unit of 2
NumberAxis axis = new NumberAxis("Axis Name");
axis.setTickUnit(3); //tick unit of 3

Label Generator

A label generator can be specified for the number axis, which determines how the tick marks are labeled. The DefaultNumberAxisLabelGenerator is provided for convenience, which generates labels in the form of integers. Custom label generators can also be provided which generate labels in any format.


          
           
NumberAxis axis = new NumberAxis("Axis Name");
axis.setLabelGenerator(new DefaultNumberAxisLabelGenerator());
axis.setLabelGenerator(new CustomLabelGenerator());

Conclusion

The NumberAxis class provides powerful features which allow you to customize the appearance and behavior of your chart's number axis. It supports auto ranging, negative numbers, custom tick units and label generators. With a few simple lines of code, you can easily take advantage of all these features.
