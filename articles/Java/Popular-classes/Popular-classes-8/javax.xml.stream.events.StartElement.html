javax.xml.stream.events.StartElement

javax.xml.stream.events.StartElement is a class that represents an XML start element. It provides methods to get the tag name, namespace and attributes associated with the start element. It also provides methods to add or remove attributes.

Usage

StartElement can be used to create an XML document by adding elements and attributes. For example, a start element representing a person can be created as follows:


          
           
StartElement personElement = factory.createStartElement("", "", "person");

where factory is a javax.xml.stream.XMLOutputFactory instance.

The following code demonstrates how to add an attribute to the person element:


          
           
Attribute ageAttr = factory.createAttribute("age", "32");
personElement.addAttribute(ageAttr);

To add a child element to the person element, one can use the following pattern:


          
           
StartElement nameElement = factory.createStartElement("", "", "name");
personElement.addChildElement(nameElement);
