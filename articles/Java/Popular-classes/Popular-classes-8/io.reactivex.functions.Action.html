Introduction to the io.reactivex.functions.Action Class in Java

The io.reactivex.functions.Action class is a functional interface that allows you to execute a set of instructions when it is called upon. It is part of the ReactiveX library, which is an implementation of the asynchronous data stream programming model. This class is useful when you need to perform a set of operations when an event occurs or when a condition is met. In this article, we will take a look at the basics of working with the io.reactivex.functions.Action class and provide some code examples.

What is io.reactivex.functions.Action?

The io.reactivex.functions.Action class is a functional interface that implements the Function Type from the ReactiveX library. It can be used to define a set of instructions to be performed when it is called upon, usually when an event or condition is met. It does not conform to any particular generic type, nor does it return a value. Instead, it simply performs the tasks specified within its body.

Creating an Action Object

An Action object can be created by defining a lambda expression inside the invocation of the Action constructor. This lambda expression will contain the instructions to be executed when the Action object is called. For example, the following code creates an Action object that prints “Hello World!” on the console window: 

          
           
Action action = new Action(() -> System.out.println("Hello World!"));
Once created, the Action object can be executed by calling its action() method. This will result in the instructions defined within the lambda expression being performed.

Example Usage

The Action class can be used in various scenarios where it is necessary to execute a set of instructions when a certain event or condition is met. This can include situations such as an observable emitting a new value, or when a user clicks on a button.

Observable Emission Example

In this example, we will create an observer that observes an Observable and prints out the items emitted by the Observable. We will then use an Action object to perform an operation when the Observable completes emitting items. 

          
           
Observable<String> observable = Observable.just("Item 1", "Item 2", "Item 3");

observable.subscribe(new Observer<String>() {
    @Override
    public void onNext(String item) {
        System.out.println("Item emitted: " + item);
    }

    @Override
    public void onComplete() {
        System.out.println("Observable completed");
    }
});

Action completionAction = new Action(() -> System.out.println("Action executed"));
observable.doOnComplete(completionAction).subscribe();
In the above code, we have first created an Observable that emits three strings. We have then subscribed to the Observable using an anonymous Observer implementation. This implementation prints out the item emitted by the Observable. Next, we have created an Action object that prints out “Action executed” to the console window. Finally, we have attached this Action to the Observable using the doOnComplete() operator. This will result in the Action being executed once the Observable completes emitting items. When this code is executed, the following output will be printed to the console window: 

          
           
Item emitted: Item 1
Item emitted: Item 2
Item emitted: Item 3
Observable completed
Action executed
As can be seen from the output, the Action was performed successfully after the Observable completed emitting items.

Button Click Example

In this example, we will create a Button and attach an Action object to it. This Action object will be executed when the Button is clicked. 

          
           
final Button button = new Button("Click Me!");
Action clickAction = new Action(() -> System.out.println("Button clicked"));
button.setOnAction(clickAction);
In the code above, we have first created a Button with the text “Click Me!” Then, we have created an Action object that prints out “Button clicked” to the console window. Finally, we have attached this Action to the Button using the setOnAction() method. This will result in the Action being executed when the Button is clicked.
