org.springframework.util.concurrent.ListenableFuture

The org.springframework.util.concurrent.ListenableFuture interface is part of the Spring Framework and provides an asynchronous alternative to Java's Future interface. It enables components to register to receive a callback when the result is available. The interface supports standard methods such as get() and isDone(), as well as an addListener() method to register a ListenableFutureCallback.

    } catch (InterruptedException | ExecutionException e) {
        e.printStackTrace();
    }
});
