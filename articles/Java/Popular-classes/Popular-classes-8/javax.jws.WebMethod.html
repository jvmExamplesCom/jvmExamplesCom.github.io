javax.jws.WebMethod

The 
javax.jws.WebMethod
annotation is used in the development of Java Web Services to mark methods that should be exposed as web services. It is part of the Java EE platform's Java API for XML-Based Web Services (JAX-WS) specification.

Usage

The 
@WebMethod
annotation is used on a service endpoint interface (SEI) or service endpoint implementation class (SEIC). It can also be used to mark methods that are not exposed as web services, this allows for methods to be excluded from exposure as web services. The default setting is for all public methods of a SEI or SEIC to be exposed as web services.

Syntax


          
           
@WebMethod()

Example

The following example shows how the 
@WebMethod
annotation is used on a SEI. 

          
           
@WebService
public interface HelloWorldService {

&nbsp; &nbsp; @WebMethod
    String sayHello(String name);

}

In this example, the exposed web service method is the 

sayHello()
method. This method will be exposed as a web service. All other public methods of the SEI will also be exposed unless they are annotated with the 
@WebMethod
annotation with the 
exclude=true
attribute.

Attributes

The 
@WebMethod
annotation supports the following attributes that can be used to customize the behavior of the exposed web service. 
  • operationName
    : Customizes the name of the operation exposed by the web service.
  • action
    : Specifies the action associated with the web service operation.
  • exclude
    : If set to 
    true
    , it excludes the annotated method from being exposed as a web service operation.

Examples

The following example shows how the 
operationName
and 
action
attributes can be used: 

          
           
@WebService
public interface HelloWorldService {

&nbsp; &nbsp; @WebMethod(operationName="SayGoodbye", action="SayGoodbyeAction")
    String sayHello(String name);

}

In this example, the exposed web service operation will be named 

SayGoodbye
with an associated action of 
SayGoodbyeAction
.

It supports several attributes that can be used to customize the behavior of the exposed web service.
