The 

          
           
javax.swing.event.ListDataListener
interface is a part of the Java Swing library and is used to provide notification of changes to lists. It is implemented by Classes that need to track changes in a list data model.

The ListDataListener interface contains 3 methods; 


          
           
intervalAdded(ListDataEvent e)
, 

          
           
intervalRemoved(ListDataEvent e)
and 

          
           
contentsChanged(ListDataEvent e)
. These methods will be called on any modification of the underlying list data model.

The IntervalAdded method will be called whenever new elements are added to the list data model. It will take a ListDataEvent as the parameter which contains details about the range of indices that have changed and the type of change (add or remove).
