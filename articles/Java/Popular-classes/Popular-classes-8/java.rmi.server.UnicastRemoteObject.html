class is an abstract class which provides an implementation of the java.rmi.Remote interface. It is a part of the Java Remote Method Invocation (RMI) framework and allows client applications to remotely invoke methods on a server object.

Its subclasses must implement the abstract methods defined in the java.rmi.server.UnicastRemoteObject class.
