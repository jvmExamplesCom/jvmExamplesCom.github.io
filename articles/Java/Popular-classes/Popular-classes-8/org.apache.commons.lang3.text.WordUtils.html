org.apache.commons.lang3.text.WordUtils

The org.apache.commons.lang3.text.WordUtils class from the Apache Commons Lang library provides a number of useful methods to work with text, including capitalization, truncation, and word wrapping.

WordUtils provides several methods for capitalizing words. The capitalize() method capitalizes the first letter of a String, while the capitalizeFully() method capitalizes all words in a String.


          
           
String capitalized = WordUtils.capitalize("hello world");
// Hello World

String fullCapitalized = WordUtils.capitalizeFully("hello world");
// Hello World

String titleCase = WordUtils.capitalizeFully("hello world", '.', '!');
// Hello.World!
