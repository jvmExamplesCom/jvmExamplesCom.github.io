Introduction to org.springframework.mock.web.MockServletContext

The 

          
           
org.springframework.mock.web.MockServletContext
class provides a mock implementation of the javax.servlet.ServletContext interface. It is commonly used in unit tests in order to provide a testing environment that simulates a web container. It can also be used as a base class when developing custom servlet containers.

MockServletContext Constructors

The MockServletContext class has two constructors. The most commonly used constructor is:


          
           
public MockServletContext(String resourceBasePath)

String
