Introduction to javax.swing.AbstractListModel

javax.swing.AbstractListModel is a class that provides a basic implementation of the ListModel interface. It maintains a list of objects and an optional array of listeners. The list can be set using the setList() method, which allows a single list to be shared among multiple models.

The methods of this class allow an application to add, insert, remove and replace elements in the list, as well as to query the size of the list and retrieve elements from it. When manipulating the list, any registered ListDataListeners will be notified of the changes.
