Introduction to androidx.recyclerview.widget.DividerItemDecoration

AndroidX RecyclerView widget DividerItemDecoration is an ItemDecoration that can be used as a divider between items of a LinearLayoutManager. This library helps make it easier for developers to draw dividers between items in the RecyclerView. This can help differentiate one item from another, and gives a more organized and visually appealing look. In this article, we will discuss how to use the DividerItemDecoration and explore some of the methods and code examples associated with it.

Getting Started with DividerItemDecoration

To begin using the DividerItemDecoration library, first add the following line to your app's build.gradle file: 

          
           
dependencies {
    implementation 'androidx.recyclerview:recyclerview:1.1.0'
}
Next, create an instance of the DividerItemDecoration class. For example, the code below would create a divider with a height of 16dp between items and draw the divider on top of each item: 

          
           
DividerItemDecoration dividerItemDecoration = new DividerItemDecoration(context, LinearLayoutManager.VERTICAL);
dividerItemDecoration.setDrawable(ContextCompat.getDrawable(context, R.drawable.divider));
dividerItemDecoration.setHeight(16dp);
recyclerView.addItemDecoration(dividerItemDecoration);
The DividerItemDecoration accepts three parameters in its constructor; a Context, the Layout orientation, and whether or not to draw the divider on top of the item. The two orientation constants are LinearLayoutManager.HORIZONTAL and LinearLayoutManager.VERTICAL.

Methods

The DividerItemDecoration class has several methods associated with it.

Set Height

The setHeight() method sets the height of the divider line. The method takes one parameter, an int specifying the desired height in pixels. This value must be greater than 0. 

          
           
dividerItemDecoration.setHeight(16dp);

Set Drawable

The setDrawable() method sets the drawable used for the divider. This method takes one parameter, a Drawable object. It can be any type of drawable, such as a ColorDrawable or a ShapeDrawable. For example, the following code creates a thin white line as the divider: 

          
           
dividerItemDecoration.setDrawable(new ColorDrawable(Color.WHITE));

Set Margins

The setMargins() method sets the margins of the divider. The method takes four parameters, all ints specifying the left, right, top, and bottom margins for the divider, respectively. The values must all be greater than 0. 

          
           
dividerItemDecoration.setMargins(16dp, 16dp, 16dp, 16dp);

onDraw() and onDrawOver()

The onDraw() and onDrawOver() methods are used to draw the divider on the canvas. They both take a Canvas object as a parameter, but beyond that, the two methods work differently. The onDraw() method is called before the items are drawn on the canvas, while the onDrawOver() method is called after the items have been drawn. Therefore, if you want to draw a divider over the items, you should use the onDrawOver() method.

Example

The following is a full example of how to use the DividerItemDecoration. In this example, we create a thin gray line as a divider between the items, with a height of 1px and a 16dp margin on all sides. 

          
           
// Create the DividerItemDecoration instance
DividerItemDecoration dividerItemDecoration = new DividerItemDecoration(context, LinearLayoutManager.VERTICAL);

// Set the divider height to 1px
dividerItemDecoration.setHeight(1px);

// Set the divider drawable to a gray line
dividerItemDecoration.setDrawable(new ColorDrawable(Color.GRAY));

// Set the margins to be 16dp
dividerItemDecoration.setMargins(16dp, 16dp, 16dp, 16dp);

// Add the divider to the RecyclerView
recyclerView.addItemDecoration(dividerItemDecoration);
