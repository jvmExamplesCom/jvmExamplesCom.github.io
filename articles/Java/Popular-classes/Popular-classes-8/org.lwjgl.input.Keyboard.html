Introduction to org.lwjgl.input.Keyboard

org.lwjgl.input.Keyboard is a class of the LWJGL library which provides an interface to the keyboard. It offers features such as key press states, key constants, and more. This article will discuss the basics of this class including its usage, properties, and methods. We will also walk through several code examples to help illustrate how to use it.

Usage

In order to use org.lwjgl.input.Keyboard in your program, you will need to include the LWJGL library in your project. This can usually be done by adding a dependency in your build.gradle or pom.xml files if you are working with a Java project.

Properties

org.lwjgl.input.Keyboard has several properties that can be used to configure its behavior. These include:
  • KEY_COUNT - The total number of keys on the keyboard.
  • KEY_ESCAPE - The key constant for the escape key.
  • KEY_BACK - The key constant for the backspace key.
  • KEY_CAPITAL - The key constant for the caps lock key.
  • KEY_SHIFT - The key constant for the shift key.

      • Methods

      org.lwjgl.input.Keyboard also provides several useful methods that allow you to get information about the current state of the keyboard. These include:
      • isKeyDown - Returns true if the specified key is currently down.
      • getEventKeyState - Returns the current state of the specified key.
      • next - Returns the next event from the keyboard.
      • poll - Updates the internal state of the keyboard based on the current events.
