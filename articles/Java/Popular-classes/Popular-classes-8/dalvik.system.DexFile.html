Introduction to Dalvik System - DexFile

The Dalvik.system.DexFile class provides access to the collection of Dalvik Executable (.dex) files that have been loaded into a process. This class allows for interaction with dex files, including basic information about the dex files and locating specific classes within the dex files.

To create an instance of a DexFile, you can use either of these two functions: 

          
           
static DexFile loadDex(String sourcePathName, String outputPathName, int flags)
static DexFile load(String sourcePathName, int flags)

sourcePathName

int

0x0000 (DEFAULT)
