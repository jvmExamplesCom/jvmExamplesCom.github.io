org.springframework.core.io.support.PathMatchingResourcePatternResolver

The org.springframework.core.io.support.PathMatchingResourcePatternResolver class is an implementation of the ResourcePatternResolver interface that supports the Ant-style path patterns to find resources in the file system and classpath.

Constructors


          
           
public PathMatchingResourcePatternResolver() 
public PathMatchingResourcePatternResolver(ClassLoader classLoader) 
public PathMatchingResourcePatternResolver(ResourceLoader resourceLoader) 
public PathMatchingResourcePatternResolver(ClassLoader classLoader, ResourceLoader resourceLoader)

Methods


          
           
public Resource[] getResources(String locationPattern) throws IOException
This method searches for the resources relative to the given location pattern. 

          
           
public Resource getResource(String location)
This method loads the resource with the given location.

Example

Let's look at a simple example: 

          
           
public static void main(String[] args) throws Exception {
    String locationPattern = "classpath*:org/springframework/**/*.xml";
    Resource[] resources = new PathMatchingResourcePatternResolver().getResources(locationPattern);
    for (Resource resource : resources) {
        System.out.println(resource.getFilename());
    }
}
The above example prints the filenames of all the XML files in the org/springframework directory and its sub-directories. The output of the program would look something like this: 

          
           
applicationContext-customer.xml
applicationContext-order.xml
dataSource.xml
log4j.xml
mvc-config.xml
properties.xml
security.xml
