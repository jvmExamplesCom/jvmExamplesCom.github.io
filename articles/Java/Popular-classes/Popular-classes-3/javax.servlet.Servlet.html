javax.servlet.Servlet

The javax.servlet API is a simple, portable, open source and easy-to-use API for creating, deploying and managing web applications. This API supports the development of web services and applications, using Java technology and related frameworks such as JavaServer Pages (JSP) and Java Database Connectivity (JDBC).

Implementing a Servlet

To create a servlet, one must implement the javax.servlet.Servlet interface. This interface has two methods that must be implemented - init() and service(). The init() method is called when the servlet is instantiated and is used to set up any resources or database connections that the servlet needs. The service() method handles each incoming client request. The servlet will receive the request, process it, and then return a response to the client.


          
           
public class MyServlet implements javax.servlet.Servlet {
    public void init(){
        // set up resources or database connections
    }

    public void service(ServletRequest req, ServletResponse res){
        // handle the incoming request and return response
    }
}

The servlet must then be registered in the web application’s deployment descriptor (web.xml). This tells the servlet container which servlets should be started when the web application is loaded.


          
           
<web-app>
    ...
    <servlet>
        <servlet-name>MyServlet</servlet-name>
        <servlet-class>MyServlet</servlet-class>
    </servlet>
    ...
</web-app>
