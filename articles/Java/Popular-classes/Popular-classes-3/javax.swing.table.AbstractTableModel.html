javax.swing.table.AbstractTableModel

The javax.swing.table.AbstractTableModel class provides a facade for the basic table functions that are necessary to populate a JTable component with data from an underlying custom model. It provides methods for configuring table column names, sorting and filtering, and querying and modifying table data.

TableColumnModel

The TableColumnModel class is used to set up and maintain references to the table's columns. This includes setting the column name, width, and other properties for each column in the table.

Sorting and Filtering

The AbstractTableModel class provides methods for sorting and filtering the table data. This includes methods for sorting and filtering by column, as well as methods for getting the currently sorted/filtered column list and for setting a sort order.

Querying and Modifying the Table Data

The AbstractTableModel class provides methods for querying the table data, including methods for getting the value of a cell, the number of rows and columns in the table, and the row and column of a given cell. It also provides methods for modifying the table data, such as adding or removing rows and columns, changing the value of a cell, and clearing the table.
