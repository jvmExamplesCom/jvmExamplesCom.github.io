javax.ws.rs.client.WebTarget

The javax.ws.rs.client.WebTarget class is the core Java class for initiating HTTP invocations over an established connection.

Creating WebTarget instances

A WebTarget can be created from an existing Client instance. Here is an example:


          
           
Client client = ClientBuilder.newClient();
WebTarget target = client.target("http://example.com/customers");

The target string provided as a parameter defines the request target - in this case http://example.com/customers. This can include query parameters and other variables embedded in the URL.

Request Method

A WebTarget represents the client side view of a particular web resource, requests can be submitted to this resource. The request method specifies the kind of request (e.g., GET, POST, PUT, etc.). Here is an example:


          
           
Invocation.Builder invocationBuilder = target.request();
Response response = invocationBuilder.get();

In this example, the get() method is used to submit a GET request to the web resource represented by the WebTarget.

Query Parameters

WebTarget also supports adding query parameters to the request. Here is an example:


          
           
WebTarget targetWithQuery = target.queryParam("name", "value");

In this example, the queryParam() method is used to add a query parameter with name "name" and value "value" to the request. Multiple query parameters can be added if needed.

Headers

Headers can also be added to the request using the header() method. Here is an example:


          
           
Invocation.Builder invocationBuilder = target.request().header("X-My-Header", "Value");

In this example, the header() method is used to add a header with name "X-My-Header" and value "Value" to the request.

Request Entities

WebTarget allows you to set data that will be sent in the request body. Here is an example:


          
           
Entity entity = Entity.json("{\"name\":\"value\"}");
Invocation.Builder invocationBuilder = target.request().entity(entity);

In this example, the entity() method is used to set a JSON string as the request body data. The entity() method can take any Java object as a parameter, which will automatically be converted to the appropriate format for the request.

Response Processing

When the response is received, it is processed based on the data type specified in the request. Here is an example:


          
           
Response response = invocationBuilder.get(String.class);
String responseString = response.readEntity(String.class);

In this example, the get() method is used to submit a GET request to the web resource and the readEntity() method is used to extract the response data as a String.
