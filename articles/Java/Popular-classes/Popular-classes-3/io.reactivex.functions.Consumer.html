Introduction to Java io.reactivex.functions.Consumer

Java io.reactivex.functions.Consumer is a functional interface that establishes a contract for an operation that accepts a single argument and returns no result. It is a part of the Java ReactiveX library, and can be used to define a unit of work that must execute for each item that is emitted by a reactive stream. This interface is primarily used in operations such as filtering, mapping, transforming, and doing side-effects.

Syntax of io.reactivex.functions.Consumer

The syntax for implementing io.reactivex.functions.Consumer looks like this: 

          
           
@FunctionalInterface
public interface Consumer<T> {
    void accept(T t);
}
It takes a single argument of type T and returns void.

Examples of io.reactivex.functions.Consumer

Here are some examples of how io.reactivex.functions.Consumer can be used in Java code:

Example 1: Filter Operation

The following example uses the filter operation to filter out even numbers from a reactive stream: 

          
           
Flux<Integer> flux = Flux.range(1, 10);
flux
  .filter(i -> i % 2 == 0)
  .subscribe(new Consumer<Integer>() {
      @Override
      public void accept(Integer t) {
          System.out.println(t);
      }
  });
The output of this code will be the following numbers: 2, 4, 6, 8, 10.

Example 2: Map Operation

The following example uses the map operation to convert a stream of strings to a stream of integers: 

          
           
Flux<String> flux = Flux.just("1", "2", "3", "4");
flux
  .map(Integer::parseInt)
  .subscribe(new Consumer<Integer>() {
      @Override
      public void accept(Integer t) {
          System.out.println(t);
      }
  });
The output of this code will be the following numbers: 1, 2, 3, 4.

Example 3: Transform Operation

The following example uses the transform operation to convert a stream of strings to a stream of integers and converts the stream back to a stream of strings: 

          
           
Flux<String> flux = Flux.just("1", "2", "3", "4");
flux
  .transform(n -> n.map(Integer::parseInt).map(String::valueOf))
  .subscribe(new Consumer<String>() {
      @Override
      public void accept(String t) {
          System.out.println(t);
      }
  });
The output of this code will be the following strings: "1", "2", "3", "4".

Example 4: Do Side-Effects

The following example uses the doOnNext() operation to log each item as it is emitted by the reactive stream: 

          
           
Flux<String> flux = Flux.just("a", "b", "c");
flux
  .doOnNext(new Consumer<String>() {
      @Override
      public void accept(String t) {
          System.out.println("Logging: " + t);
      }
  })
