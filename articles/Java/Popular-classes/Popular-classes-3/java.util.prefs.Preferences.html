java.util.prefs.Preferences Overview

The java.util.prefs.Preferences class provides a mechanism for storing application and user preferences in the form of key-value pairs. This class enables applications to store and query data in an operating system independent way.

Creating Preferences Instances

The java.util.prefs.Preferences class is a singleton, meaning there is only one instance of it. To get a reference to the singleton instance, use the 

getSystemRoot()
or 
getUserRoot()
methods: 

          
           
// Get system preferences instance
Preferences systemPrefs = Preferences.systemRoot();

// Get user preferences instance
Preferences userPrefs = Preferences.userRoot();

Storing and Retrieving Preferences

Preferences are stored as key-value pairs. Keys are strings, and values can be of any type supported by the 

putValue()
method, including strings, numbers, and boolean values. To store a preference, use the 
putValue()
method. For example, here we store a string value with the key 
"username"
: 

          
           
userPrefs.putValue("username", "jdoe");
To retrieve a stored preference, use the 
getValue()
method. For example, here we retrieve the value with key 
"username"
: 

          
           
String username = userPrefs.getValue("username", "");
The second argument to 
getValue()
is an optional default value. If no value is found for the specified key, this default value will be returned instead.

Removing Preferences

To remove a stored preference, use the 
removeValue()
method. For example, here we remove the value with the key 
"username"
: 

          
           
userPrefs.removeValue("username");

Saving Changes

Changes to stored preferences are not automatically saved. To save changes, use the 

flush()
method. Note that this method will only save changes made since the last time you used the 
flush()
method: 

          
           
userPrefs.flush();

Conclusion

The java.util.prefs.Preferences class provides an easy way to store application and user preferences in an operating system independent way. It is simple to save and retrieve preferences, as well as remove unwanted preferences. However, remember to use the 

flush()
method to save your changes.
