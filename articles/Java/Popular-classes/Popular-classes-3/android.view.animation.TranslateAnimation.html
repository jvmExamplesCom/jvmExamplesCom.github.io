android.view.animation.TranslateAnimation

The android.view.animation.TranslateAnimation class is a powerful and versatile class for creating animations in Android. It allows developers to create animations which move views from one position to another in either an X or Y direction. This class can be used to animate arbitrary objects, such as Views or Drawables, making it a key tool in complex UI design. This article will provide an overview of the TranslateAnimation class as well as example code snippets and sample output.

Overview

The primary purpose of the TranslateAnimation class is to allow developers to create animations which move objects along a path from one position to the other. The class constructor takes four parameters, two for each axis: the starting position and the ending position. The positions are specified as absolute coordinates within the View or Drawable objects. Additionally, the class also has several optional parameters, including the duration and interpolator type.

Example Code

The following code snippet demonstrates how to create a simple animation using TranslateAnimation. 

          
           
TranslateAnimation animation = new TranslateAnimation(
    0, // From x coordinate
    100, // To x coordinate
    0, // From y coordinate
    200); // To y coordinate

animation.setDuration(1000);
animation.setInterpolator(new LinearInterpolator());
animation.setFillAfter(true);

view.startAnimation(animation);
This code creates an animation which moves a View object a distance of 100 units in the X-direction and 200 units in the Y-direction. The animation lasts 1000 milliseconds and uses a linear interpolator. The setFillAfter() method ensures that the View remains at the final position after the animation has finished.
