Android.graphics.drawable.StateListDrawable

StateListDrawable is an Android drawable that manages a number of different drawables and can switch between them depending on the state of the View. It can be used to create custom states for a Button or other Views, and is more efficient than creating a new drawable for each state change.

What is StateListDrawable?

StateListDrawable is an Android drawable that can manage a number of different states and switch to the appropriate drawable when the state of the View changes. A StateListDrawable can contain multiple different drawables, each associated with a particular state of the associated View. For example, a StateListDrawable could have a different image for a Button when it is pressed, selected, or enabled.

Creating a StateListDrawable

A StateListDrawable is created using XML. The root element is 

          
           
<selector>
, which contains multiple 

          
           
<item>
elements which define the drawable to use for each state. The following example shows how to create a StateListDrawable with two drawables, one for when the associated View is pressed and one for when it is not pressed: 

          
           
<selector xmlns:android="http://schemas.android.com/apk/res/android">
    <item android:state_pressed="true" android:drawable="@drawable/button_pressed"/>
    <item android:drawable="@drawable/button_normal"/>
</selector>
As you can see, each 

          
           
<item>
element has an attribute, 

          
           
android:state_pressed
, which specifies whether the associated View is pressed or not. If it is 

          
           
true
, then the drawable specified in the 

          
           
android:drawable
attribute will be used. Otherwise, the drawable specified in the second 

          
           
<item>
element will be used.

The list of available states for a StateListDrawable is defined in the Android SDK. Here is a list of some of the most commonly used states:

Each

Example Usage

Here is an example of a StateListDrawable being used to create a custom button. When the button is pressed, it will use an image of a yellow star, and when it is not pressed, it will use an image of a blue star.

    android:layout_width="wrap_content"
    android:layout_height="wrap_content"

StateListDrawable is a powerful tool for creating custom states for Buttons and other Views in Android. With StateListDrawable, you can enable different Drawables for different states of the associated View, making it much easier to create custom visuals.
