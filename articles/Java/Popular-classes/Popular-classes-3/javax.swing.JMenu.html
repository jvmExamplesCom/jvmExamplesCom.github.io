javax.swing.JMenu

The javax.swing.JMenu class is used to create and manipulate menus in a Java application. It works with the JMenuBar and JPopupMenu classes to create popup menus and menu bars. The JMenu class provides methods for setting up menus, adding items, setting icons and mnemonics, and handling events. It also provides methods for capturing user selection and making selections programmatically.

JMenu menu = new JMenu("Menu");
menuBar.add(menu);
