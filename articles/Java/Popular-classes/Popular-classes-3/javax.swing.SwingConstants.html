SwingConstants

  • VERTICAL
    - indicates that a component should be oriented vertically.

You can use these constants when creating a new instance of a component like this:


          
           
JLabel label = new JLabel("Hello world", SwingConstants.HORIZONTAL);

  • LEFT

  • CENTER
    - centers the component in its container.

label.setAlignment(SwingConstants.CENTER);
