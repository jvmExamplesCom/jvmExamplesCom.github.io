javax.ws.rs.core.HttpHeaders

The javax.ws.rs.core.HttpHeaders class defines the set of HTTP headers that are used by JAX-RS and its subclasses. This class provides utility methods for parsing, serializing, and manipulating the various parts of a request or response header. It also provides methods for setting headers in the application context.

Methods

String acceptLanguageHeader = httpHeaders.getHeaderString("Accept-Language");
