javax.ws.rs.core.HttpHeaders

The javax.ws.rs.core.HttpHeaders class defines the set of HTTP headers that are used by JAX-RS and its subclasses. This class provides utility methods for parsing, serializing, and manipulating the various parts of a request or response header. It also provides methods for setting headers in the application context.

Methods

getHeaderString

getHeaderString()
method is used to get a single header from the request or response. The header value can be either a String or a List, depending on if there are multiple headers with the same name. The following example returns the value of the "Accept-Language" header: 

          
           
String acceptLanguageHeader = httpHeaders.getHeaderString("Accept-Language");

The 

          
           
setHeader()
method is used to set the value of a single header in the application context. The following example sets the "Content-Type" header to "application/json": 

          
           
httpHeaders.setHeader("Content-Type", "application/json");

httpHeaders.filter(h -> h.startsWith("x-"));

The
