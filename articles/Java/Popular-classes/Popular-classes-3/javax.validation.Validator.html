javax.validation.Validator

The javax.validation.Validator class provides an interface for validating the contents of objects according to a set of rules. It is part of Java's bean validation API and allows developers to validate objects according to their own custom rules. @NotNull
@Length(min = 3, max = 20)
public String myStringField;

The complete list of available constraints can be found in the Javadocs.
