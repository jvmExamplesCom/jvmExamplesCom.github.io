javax.swing.BoxLayout

Java Swing's BoxLayout is a powerful tool to use when creating user interfaces. It allows for an easy way to layout components, by putting them in boxes, with the components all lined up one after another in the same direction. The components can be nested and aligned, making it simple to create complex user interfaces. This article will explain how to use BoxLayout and provide code examples.

Constructors

BoxLayout has two constructors available: 

          
           
public BoxLayout(Container target, int axis) 

public BoxLayout(Container target, int axis, int gap)
The first argument, 
target
, is the 
Container
that the BoxLayout will be applied to. The second argument, 
axis
, is an integer constant that specifies in which direction the components will be laid out. The constants are: 

          
           
BoxLayout.X_AXIS - Components are laid out horizontally
BoxLayout.Y_AXIS - Components are laid out vertically
The third argument, 
gap
, is an integer that specifies the size of the gap in pixels between each component. This argument is optional and can be omitted if no gap is wanted.

Adding Components

Components can be added to the 
Container
either directly or through the 
BoxLayout
instance. To add components directly to the 
Container
, simply use the 
add(Component)
method. For example: 

          
           
JFrame frame = new JFrame("My Frame");
frame.setLayout(new BoxLayout(frame, BoxLayout.X_AXIS));
frame.add(new JLabel("Label 1"));
frame.add(new JButton("Button 1"));
frame.add(new JTextField("Text Field 1"));
To add components through the 
BoxLayout
instance, use the 

          
           
addLayoutComponent(Component, Object)
method. The first argument is the 
Component
to be added, and the second argument is a 
String
, which specifies where in the layout the 
Component
should be added. The possible values are: 

          
           
BoxLayout.LINE_START - Adds the Component to the beginning of the line
BoxLayout.LINE_END - Adds the Component to the end of the line
BoxLayout.PAGE_START - Adds the Component to the beginning of the page
BoxLayout.PAGE_END - Adds the Component to the end of the page
BoxLayout.CENTER - Adds the Component to the center of the layout
For example: 

          
           
JFrame frame = new JFrame("My Frame");
BoxLayout layout = new BoxLayout(frame, BoxLayout.X_AXIS);
layout.addLayoutComponent(new JLabel("Label 1"), BoxLayout.LINE_START);
layout.addLayoutComponent(new JButton("Button 1"), BoxLayout.LINE_END);
layout.addLayoutComponent(new JTextField("Text Field 1"), BoxLayout.PAGE_END);
frame.setLayout(layout);

Examples

BoxLayout
, let's look at a few examples of how this can be used. Here is an example of how to create a simple user interface with two columns and two rows: 

          
           
JFrame frame = new JFrame("My Frame"); 
frame.setLayout(new BoxLayout(frame, BoxLayout.X_AXIS)); 
frame.add(new JLabel("Label 1"));
