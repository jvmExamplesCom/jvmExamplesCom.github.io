What is com.squareup.okhttp.OkHttpClient?

The com.squareup.okhttp.OkHttpClient library is a set of classes that makes it easy to make HTTP requests in Java. It also allows developers to configure custom timeouts and read or write headers and cookies, among others. The library is open source and is actively maintained by the Square Engineering team.

import com.squareup.okhttp.OkHttpClient;
