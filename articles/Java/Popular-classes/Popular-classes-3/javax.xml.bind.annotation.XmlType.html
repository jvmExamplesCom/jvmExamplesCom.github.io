javax.xml.bind.annotation.XmlType

javax.xml.bind.annotation.XmlType is an annotation type used to map a class or an enum type to the XML representation. It can be used to customize the name and the ordering of the elements and attributes in the XML representation.

Example 1: Customizing Class Name, Element Names, and Order

In this example, we will customize the class name, element names, and order using XmlType. We will use a Customer class with fields for customerName and address.


          
           
@XmlType(propOrder = {"customerName", "address"})
public class Customer {
    private String customerName;
    private String address;

    public String getCustomerName() {
        return customerName;
    }
    public void setCustomerName(String customerName) {
        this.customerName = customerName;
    }
    public String getAddress() {
        return address;
    }
    public void setAddress(String address) {
        this.address = address;
    }
}
