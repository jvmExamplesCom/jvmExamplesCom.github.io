What is org.junit.rules.TestName?

org.junit.rules.TestName is a JUnit rule that provides access to the current test name in a test class. This allows the test class to make use of the test name for logging and other similar purposes.

Features

Example

@Rule
public TestName testName = new TestName();

@Test

// ... 
}
