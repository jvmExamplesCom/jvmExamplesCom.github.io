What is org.junit.rules.TestName?

org.junit.rules.TestName is a JUnit rule that provides access to the current test name in a test class. This allows the test class to make use of the test name for logging and other similar purposes.

Features

org.junit.rules.TestName provides the test name as a String in all test methods. It also provides a getName() method that returns the test name as an Object. This makes it easy to store, process, and manipulate the test name.

Example

The following code example shows how to use org.junit.rules.TestName to print the test name in the test class: 

          
           
@Rule
public TestName testName = new TestName();

@Test
public void testMyMethod() {
System.out.println("Test name: " + testName.getName());
// ... 
}
