ExceptionUtils in org.apache.commons.lang3.exception

ExceptionUtils is a class in the org.apache.commons.lang3.exception package of the Apache Commons Lang library. It provides static utility methods for handling and processing Java Exceptions. In this article, we'll explore how this class works and the various ways in which it can be used.

Creating an ExceptionUtils Object

The ExceptionUtils class cannot be instantiated as there exists no constructor within it. All the methods provided by this class are static and can therefore be invoked directly using the Class name.


          
           
ExceptionUtils.getStackTrace(new Exception("This is an example exception"));

Methods Provided by ExceptionUtils

The ExceptionUtils class provides several static methods for handling and processing Java Exceptions. Some of the commonly used methods are as follows:

  • getRootCause(): This method returns the root cause of the given Throwable object. The root cause is the underlying cause of the given Throwable object.
  • getStackTrace(): This method returns the stack trace of the given Throwable object. The stack trace is an array of StackTraceElement objects representing each line in the stack trace.
  • toString(): This method returns the detail message string of the given Throwable object. The detail message string is a human-readable string providing details about the exception.
  • getMessage(): This method returns the localized message of the given Throwable object. The localized message is a string designed to be read by a person, possibly in a different language.
  • isRethrown(): This method checks if the given Throwable object is rethrown. A Throwable object is rethrown if it was caught and rethrown by a try-catch block.

Example Usage of ExceptionUtils

Let us try to understand the usage of ExceptionUtils class by taking an example. Consider the following code which throws an exception and then prints its stack trace using the getStackTrace() method of the ExceptionUtils class:


          
           
try {
    int result = 1 / 0;
} catch (ArithmeticException e) {
    String stackTrace = ExceptionUtils.getStackTrace(e);
    System.out.println(stackTrace);
}
The output of this code will be:

As can be seen, the getStackTrace() method returned the stack trace of the exception thrown in the try block.
