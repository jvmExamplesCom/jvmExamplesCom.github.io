javax.swing.JTable

The 
javax.swing.JTable
class provides a powerful and versatile means of displaying, organizing, and manipulating data. A JTable is a type of container which can hold multiple columns of data, including column headers, and allows for sorting, searching, and editing of the data. The JTable is one of the most popular user interface components in Java applications, as it provides an intuitive and visually appealing way to display data.

Creating a JTable

Creating a 
JTable
is fairly straightforward. First, an instance of the table can be created with no parameters: 

          
           
JTable table = new JTable();
Alternatively, you can provide with an array of 
Object
s that are used as the column headers: 

          
           
String[] columnHeaders = {"First Name", "Last Name", "Age"};
JTable table = new JTable(columnHeaders);
You can also pass in a two-dimensional array of 
Object
s to the 
JTable
constructor, which will create a 
JTable
with the specified data. This is useful if you already have the data that needs to be displayed: 

          
           
Object[][] dataArray = { 
    {"John", "Smith", 30},  
    {"Amy", "Jones", 27},  
    {"Bill", "Brown", 32} 
};
JTable table = new JTable(dataArray);

Adding and Removing Data

Once the 
JTable
has been instantiated, you can add and remove data from it. You can add rows to the table using the 
addRow
method, which takes an array of 
Object
s that represent the data for the row: 

          
           
Object[] data = {"Tom", "Taylor", 26};
table.addRow(data);
You can also remove rows from the table using the 
removeRow
method, which takes the index of the row to be removed: 

          
           
table.removeRow(2);

Sorting and Searching

The 
JTable
class provides several methods that can be used to sort the table's data. The 
sort
method allows you to sort the table based on the contents of a specific column by providing the index of the column. For example, to sort the table by the “Last Name” column: 

          
           
table.sort(1);
The 
JTable
also provides a 
search
method, which allows you to search for a particular piece of data within the table. The 
search
method takes two parameters - the index of the column to be searched, and the value that should be searched for: 

          
           
table.search(1, "Taylor");

This will search the “Last Name” column for the value “Taylor” and return the index of the row containing that value. If the value is not found, the method will return 

-1
.

javax.swing.JTable
