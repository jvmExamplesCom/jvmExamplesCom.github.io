java.util.function.UnaryOperator

The 

          
           
java.util.function.UnaryOperator
is an interface that represents an operation on a single operand that produces a result of the same type as its operand. It's a specialization of 
Function
for the case where the operand and result are of the same type. This is a functional interface whose functional method is 
apply(Object)
.

Example Usage

The 
UnaryOperator
interface can be used to perform various operations on a single input. Here's an example of how it can be used to increment (add one) to an integer: 

          
           
UnaryOperator<Integer> increment = e -> e + 1;
int x = 10;
int y = increment.apply(x); // y = 11

And, here's an example of using it to multiply a number by two:


          
           
UnaryOperator<Integer> multiplyByTwo = e -> e * 2;
int x = 10;
int y = multiplyByTwo.apply(x); // y = 20

UnaryOperator
interface can be used with Java 8 Streams to apply an operation to each element of a stream. Here's an example of using the 
UnaryOperator
to triple each element in a stream: 

          
           
UnaryOperator<Integer> triple = e -> e * 3;
