java.security.Permission

Java.security.Permission provides the permissions that are required to provide access control in the Java programming language. It is an abstract class that encapsulates a permission name and associated actions.

A Permission consists of two components: the name of the permission (for instance, java.io.FilePermission) and the associated action list. The action list is a set of one or more action strings that allow access to a specified resource.
