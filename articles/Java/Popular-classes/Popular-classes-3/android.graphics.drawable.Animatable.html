android.graphics.drawable.Animatable

The android.graphics.drawable.Animatable interface is an extension of Drawable which allows a drawable to start and stop animation. It's typically used when a Drawable is animated, such as with AnimatedVectorDrawables.

Methods

The Animatable interface defines two methods: 
start()
and 
stop()
. These methods are used to respectively start and stop the animation associated with the drawable.

Example Usage

Here is an example of using the Animatable interface to start and stop a drawable animation:


          
           
Animatable animatable = (Animatable) drawable;
animatable.start();
animatable.stop();

  • BitmapDrawable
