java.nio.channels.WritableByteChannel

The java.nio.channels.WritableByteChannel interface provides a channel to write bytes to. A WritableByteChannel represents an open connection to an entity such as a file that is capable of writing bytes. It is obtained from an appropriate type of channel.

Methods

WritableByteChannel contains the following methods:

  • write(ByteBuffer src): Writes a sequence of bytes to this channel from the given buffer.
  • isOpen(): Tells whether or not this channel is open.

Example

The following example shows how to use WritableByteChannel:


          
           
import java.nio.ByteBuffer;
import java.nio.channels.WritableByteChannel;

public class MyClass {

  public static void main(String[] args) {

    WritableByteChannel channel = ... ; // get the channel
    ByteBuffer buf = ByteBuffer.wrap("Hello World".getBytes());

    // write data from buffer to the channel
    channel.write(buf);

    // check if the channel is still open
    System.out.println(channel.isOpen());
  }

}
