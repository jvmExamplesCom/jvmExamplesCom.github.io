Introduction to com.google.android.material.snackbar.Snackbar

com.google.android.material.snackbar.Snackbar is an Android Library that allows developers to easily create snack bars. A snack bar is a short prompt at the bottom of the screen that briefly notifies users about a process or task that has been completed. The library uses the Material Design specification from Google and provides APIs to work with snack bars.

Creating a Snackbar

Creating a Snackbar requires a few simple steps. First, we need to create a new 

Snackbar
instance. We can do this by calling the 
make
method on the 
Snackbar
class: 

          
           
Snackbar snackbar = Snackbar.make(view, "This is a Snackbar", Snackbar.LENGTH_SHORT);

The first argument is the view to which we want to attach the snackbar, the second is the text to display, and the third is the duration for which the snackbar should be visible.

Once we have our 
Snackbar
instance, we need to make it visible. To do this we call the 
show
method on our instance: 

          
           
snackbar.show();
That's all there is to creating a basic Snackbar.

Adding Action Buttons

In addition to displaying a text message, Snackbars can also include an action button. To add an action button we use the 

setAction
method on our 
Snackbar
instance: 

          
           
snackbar.setAction("Undo", new View.OnClickListener() {
    @Override
    public void onClick(View view) {
        // Perform some action...
    }
});

The first argument is the text to display on the button and the second argument is a 


          
           
View.OnClickListener
that will be triggered when the button is clicked.

Setting the Background Color

We can set the background color of the snackbar using the 

getView
method on our 
Snackbar
instance. This will return a 
View
object that we can use to set the background color: 

          
           
View sbView = snackbar.getView();
sbView.setBackgroundColor(ContextCompat.getColor(this, R.color.my_color));
The first argument to the 
getColor
method is the context from which the color should be resolved and the second argument is the color resource id.

Displaying Images

In addition to text and buttons, we can also display images within our snackbars. To do this we need to use a custom layout for our snackbar. We can create a custom layout in an XML file, then inflate it using the 

LayoutInflater
class. Once we have the layout, we can pass it to the 
setView
method on our 
Snackbar
instance: 

          
           
LayoutInflater inflater = (LayoutInflater) getSystemService(Context.LAYOUT_INFLATER_SERVICE);
View customLayout = inflater.inflate(R.layout.my_custom_layout, null);
snackbar.setView(customLayout);
We can then add an 
ImageView
to the layout and set the drawable we wish to display: 

          
           
ImageView imageView = (ImageView) customLayout.findViewById(R.id.my_image_view);
imageView.setImageDrawable(ContextCompat.getDrawable(this, R.drawable.my_image));

Configuring the Animation

We can configure the animation used when displaying and hiding the snackbar using the 

setAnimationMode
method. This takes one of two constants, either 

          
           
Snackbar.ANIMATION_MODE_SLIDE
or 

          
           
Snackbar.ANIMATION_MODE_FADE
, as an argument: 

          
           
snackbar.setAnimationMode(Snackbar.ANIMATION_MODE_SLIDE);
