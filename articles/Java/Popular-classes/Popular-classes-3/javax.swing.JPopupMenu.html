javax.swing.JPopupMenu

The javax.swing.JPopupMenu object is a component that provides the user with a context menu when they right-click on a component. It provides an easy way to add interactive options to your application and can be customized to provide any functionality you need. This article will give an overview of JPopupMenu and provide code examples to assist in its implementation.

What is JPopupMenu?

JPopupMenu is a class in the Java Swing library which provides a way to create popup menus. This is useful when you need to provide immediate access to certain actions within a component, such as a list or a table. When the user right-clicks on the component, the popup menu will be displayed, allowing them to select from the available options.

Creating a JPopupMenu

Creating a JPopupMenu is easy and only requires a few lines of code. The following example creates a new JPopupMenu that has two options: ‘Add’ and ‘Delete’:


          
           
JPopupMenu popupMenu = new JPopupMenu();
JMenuItem addItem = new JMenuItem("Add");
JMenuItem deleteItem = new JMenuItem("Delete");
popupMenu.add(addItem);
popupMenu.add(deleteItem);

Showing the JPopupMenu

Once the JPopupMenu has been created, it needs to be displayed on the screen. The following code example shows how to display the JPopupMenu in response to a mouse event:


          
           
public void mousePressed(MouseEvent e) {
    if (e.isPopupTrigger()) {
        popupMenu.show(e.getComponent(), e.getX(), e.getY());
    }
}
