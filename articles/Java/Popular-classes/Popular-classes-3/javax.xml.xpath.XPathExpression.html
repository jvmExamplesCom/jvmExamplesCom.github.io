javax.xml.xpath.XPathExpression

javax.xml.xpath.XPathExpression is an interface that represents a compiled XPath expression. It provides access to the underlying executable XPath object, allowing the evaluation of the XPath expression in an XML document.

Example Usage

The following example shows how to use XPathExpression to evaluate an XPath expression. 

          
           
//Create XPathFactory object
XPathFactory xpathFactory = XPathFactory.newInstance();

//Create XPath object
XPath xpath = xpathFactory.newXPath();

//Compile the XPath expression
XPathExpression expr = xpath.compile("/catalog/book[price > 30]/title");

//Evaluate expression result on XML document
Object result = expr.evaluate(doc, XPathConstants.NODESET);

//Print out the result
NodeList nodes = (NodeList) result;
for (int i = 0; i < nodes.getLength(); i++) {
  System.out.println(nodes.item(i).getNodeValue());
}
