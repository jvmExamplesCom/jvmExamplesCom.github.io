java.awt.event.KeyAdapter

Java.awt.event.KeyAdapter is a class in the Java AWT (Abstract Window Toolkit) package. It is used to handle keyboard events such as key presses and key releases. A KeyAdapter object can be added to a window or a component, and the adapter's methods are invoked whenever the user presses a key.

Methods

The KeyAdapter class has two abstract methods that must be implemented by a subclass:

  • 
            
             
keyPressed(KeyEvent)
    : Invoked when a key is pressed down.
  • 
            
             
keyReleased(KeyEvent)
    : Invoked when a key is released.

Example Usage

This example uses a KeyAdapter to detect when a user presses the "space" key and prints a message to the console.


          
           
public class SpaceListener extends KeyAdapter {
    public void keyPressed(KeyEvent e) {
        if(e.getKeyCode() == KeyEvent.VK_SPACE) {
            System.out.println("Space Bar Pressed!");
        }
    }
}

We can then add the listener to a window or a component:


          
           
Window window = new Window();
window.addKeyListener(new SpaceListener());
