javax.xml.bind.annotation.XmlAttribute

javax.xml.bind.annotation.XmlAttribute is an annotation used to map an attribute of an XML element with the member of a class. It can be used with the following java data types: boolean, byte, char, double, float, int, long, short, String, and any type that has either a public no-args constructor or it needs to have a static factory method to create its instance.

Usage Examples

Consider the following class: 

          
           
public class Student {
    @XmlAttribute
    private String name;

    @XmlAttribute
    private int age;
}

The above class can be mapped to the following XML:


          
           
<Student name="John Doe" age="20"/>

Marshalling

In order to marshal this class to XML we need to use JAXB. We create a 

JAXBContext
instance, create a Marshaller and then invoke the 
marshall()
method passing our object as a parameter. 

          
           
JAXBContext jaxbContext = JAXBContext.newInstance(Student.class);
Marshaller jaxbMarshaller = jaxbContext.createMarshaller();

// output pretty printed
jaxbMarshaller.setProperty(Marshaller.JAXB_FORMATTED_OUTPUT, true);

jaxbMarshaller.marshal(stud, System.out);

Unmarshalling

To unmarshal a XML document to Java objects we can use the same JAXB `JAXBContext` instance and then invoke the `unmarshal()` method. This will return a `JAXBElement` that contains our object.


          
           
JAXBContext jaxbContext = JAXBContext.newInstance(Student.class);
Unmarshaller jaxbUnmarshaller = jaxbContext.createUnmarshaller();

JAXBElement<Student> studentElement = jaxbUnmarshaller.unmarshal(new StreamSource(new File("student.xml")), Student.class);
Student student = studentElement.getValue();
