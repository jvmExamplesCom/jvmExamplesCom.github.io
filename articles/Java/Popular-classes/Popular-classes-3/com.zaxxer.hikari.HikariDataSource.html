Overview of com.zaxxer.hikari.HikariDataSource

The 

          
           
com.zaxxer.hikari.HikariDataSource
class is a robust and performant JDBC connection pool for Java applications. It is a high-performance replacement for existing connection pools such as Apache Commons DBCP and C3P0. It is designed to scale to many concurrent connections from clients on a production system.

Features of com.zaxxer.hikari.HikariDataSource

  • High performance.
  • Thread safe.
  • Concurrent access optimized with the help of Connection Pool.
  • Allows for multiple configurations at the same time for different modules.
  • Ability to support both read/write and read-only operations.
  • Supports asynchronous operation.
  • Uses the latest JDBC drivers from vendors.

com.zaxxer.hikari.HikariDataSource

public HikariDataSource(String jdbcUrl, String username, String password, Properties props)

Example of com.zaxxer.hikari.HikariDataSource Usage

A simple example of how the 

          
           
com.zaxxer.hikari.HikariDataSource
can be used to create a data source for a database is shown below:

dataSource.setJdbcUrl("jdbc:h2:mem:test");
