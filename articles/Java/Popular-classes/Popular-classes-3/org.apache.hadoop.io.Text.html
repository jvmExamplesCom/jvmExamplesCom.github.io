org.apache.hadoop.io.Text Class in Java

The org.apache.hadoop.io.Text class is an important utility class used in Apache Hadoop for text manipulation. It is a subclass of org.apache.hadoop.io.WritableComparable and holds UTF-8 encoded character sequence data. The Text class provides methods to convert the text sequence into different forms, for example, converting it to binary form, reading or writing text to or from the filesystem, and so on.

Text Class Constructors

The org.apache.hadoop.io.Text class provides multiple constructors depending on the type of data to be manipulated. To create an empty Text object, you can use the default constructor: 

          
           
Text txt = new Text();
You can also construct a new Text object to wrap a given String: 

          
           
String str = "This is a test string.";
Text txt = new Text(str);
And, you can construct a new Text object to wrap a given byte array: 

          
           
byte[] b = {'a','b','c','d','e','f'};
Text txt = new Text(b);
