Introduction to org.reactivestreams.Publisher

The 

          
           
org.reactivestreams.Publisher
interface in Java is a core component of the Reactive Streams library, providing an application programming interface (API) for asynchronous stream processing with non-blocking back pressure. The Publisher is an abstraction of the data producer, and can be used to facilitate the production of data elements in a wide variety of ways. It operates like a push-based system, and is useful for one-to-many communication between components of an application.
