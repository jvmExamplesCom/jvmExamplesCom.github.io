java.io.LineNumberReader

The java.io.LineNumberReader class is a subclass of the BufferedReader class that allows reading lines from an external source like a file, a string, etc. It also provides methods for marking and resetting the current read position in the stream, allowing for efficient random access. The LineNumberReader keeps track of the current line number, which can be useful for debugging or reporting error messages.

The following code example shows how to create an instance of the java.io.LineNumberReader class:


          
           
LineNumberReader reader = new LineNumberReader(
        new FileReader("myfile.txt")
    );

Reading lines with the LineNumberReader

Once the LineNumberReader is created, you can start reading lines from the input source. Here's an example:


          
           
String line;
    while((line = reader.readLine()) != null) {
        System.out.println("Line #" + reader.getLineNumber() + ": " + line);
    }

This code will print out all the lines from the source, along with the corresponding line numbers.

Marking and Resetting

The LineNumberReader also provides methods for setting and resetting markers in the stream. This is useful if you need to read certain parts of the input more than once, or if you want to save the current position so that you can return to it later. Here's an example of how to use the mark() and reset() methods:


          
           
// Set a marker at the current position
    reader.mark(1000);

    // Read some lines from the stream
    String line = reader.readLine();
    System.out.println("Line #1: " + line);

    // Reset the marker back to the beginning
    reader.reset();

    // Read another line
    line = reader.readLine();
    System.out.println("Line #1 (again): " + line);

The above code will print out the same line twice, since the marker was reset after the first read.
