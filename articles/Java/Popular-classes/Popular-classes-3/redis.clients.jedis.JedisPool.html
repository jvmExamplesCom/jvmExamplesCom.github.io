What is Redis - Java.clients.jedis.JedisPool?

How to Use Redis Java.clients.jedis.JedisPool

JedisPool jedisPool = new JedisPool(new JedisPoolConfig(), "localhost");

The JedisPoolConfig class provides various parameters to control the size and behavior of the pool.
