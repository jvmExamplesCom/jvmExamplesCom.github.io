javax.swing.JTabbedPane

javax.swing.JTabbedPane is a powerful and versatile container class used to create tabbed layouts in Java. It can be used to create an attractive panel of several different components with each component occupying a distinct page in the tabbed pane. The components can be of any type, from text fields or images to complete applications and applets.

To create a JTabbedPane object, you must provide two arguments, the tabbed pane's orientation and the initial tab index. The orientation determines how the tabs are positioned on the tabbed pane. This can be either horizontal (from left to right) or vertical (from top to bottom). The initial tab index is the index of the tab that will be initially selected when the tabbed pane is created.

Code Examples

Here is a simple example of creating a horizontal JTabbedPane with three tabs:


          
           
tabbedPane = new JTabbedPane(JTabbedPane.HORIZONTAL);
tabbedPane.add("Tab 1", new JPanel());
tabbedPane.add("Tab 2", new JPanel());
tabbedPane.add("Tab 3", new JPanel());

The following code example demonstrates how to add a ChangeListener to the JTabbedPane to be notified when the selection of tabs changes:


          
           
tabbedPane.addChangeListener(new ChangeListener() {
    public void stateChanged(ChangeEvent e) {
        System.out.println("Tab Selection Changed");
    }
});

Here is how you can set or change the title of a tab in the tabbed pane:


          
           
tabbedPane.setTitleAt(int index, String title);

You can remove tab by specifying the index of the tab to be removed:


          
           
tabbedPane.remove(int index);

The following code example demonstrates how to get the index of the currently selected tab in the tabbed pane:


          
           
tabbedPane.getSelectedIndex();

And finally, here is how to set the specified tab as the currently selected tab:


          
           
tabbedPane.setSelectedIndex(int index);
