org.apache.hadoop.fs.FileSystem

The 

          
           
org.apache.hadoop.fs.FileSystem
class is a Java-based implementation of the Hadoop Distributed File System (HDFS). It provides an abstract view of the HDFS filesystem. This class defines the basic operations to be performed on the file system, like listing and reading directory contents, writing to files, creating directories or deleting files.

Getting Started with Hadoop File System

The first step to get started with HDFS is to create a new instance of the 

          
           
org.apache.hadoop.fs.FileSystem
class. This can be done in two ways:

Using the FileSystem API

Using the 

          
           
FileSystem
API, you can access the HDFS in the same manner as when using local file systems. You will need to pass the URI of the hdfs to the API which will return a 

          
           
FileSystem
object. For example: 

          
           
FileSystem fs = FileSystem.get(new URI("hdfs://localhost:9000"), config);
The returned 

          
           
FileSystem
object can then be used for performing various operations on the HDFS such as listing the directory contents and writing files.

Using the FileContext API

The 

          
           
FileContext
API provides an object-oriented interface for accessing HDFS. This API makes it easier to work with HDFS since it provides access to both local and HDFS filesystems in a unified way. To use the 

          
           
FileContext
API, you will need to create an instance of the 

          
           
FileContext
class and pass in the configuration object. For example: 

          
           
Configuration config = new Configuration();
FileContext fc = FileContext.getFileContext(config);
This 

          
           
FileContext
object can then be used for performing various operations on the HDFS such as listing the directory contents and writing files.

Basic Operations on the FileSystem

Once you have obtained a 

          
           
FileSystem
or 

          
           
FileContext
instance, you can perform some basic operations on the HDFS. These operations include listing directory contents, reading from files, writing to files, creating directories, and deleting files.

Listing Directory Contents

To list the contents of a directory, you can use the 

          
           
listStatus()
method. This method takes a 

          
           
Path
object for the directory and returns an array of 

          
           
FileStatus
objects. Each 

          
           
FileStatus
object contains information about the file or directory, such as the path, size, permission, block size, etc. For example: 

          
           
Path path = new Path("/user/hadoop/data");
FileStatus[] status = fs.listStatus(path);
for (int i = 0; i < status.length; i++) {
    System.out.println(status[i].getPath());
}

Reading from Files

To read data from a file, you need to use the 

          
           
open()
method. This method takes a 

          
           
Path
object for the file and returns an 

          
           
FSDataInputStream
. The stream can then be used to read data from the file. For example: 

          
           
Path path = new Path("/user/hadoop/data/test.txt");
FSDataInputStream inputStream = fs.open(path);
byte[] buffer = new byte[1024];
int bytesRead = inputStream.read(buffer);

Writing to Files

To write data to a file, you need to use the 

          
           
create()
method. This method takes a 

          
           
Path
object for the file and returns an 

          
           
FSDataOutputStream
. The stream can then be used to write data to the file. For example: 

          
           
Path path = new Path("/user/hadoop/data/test.txt");
FSDataOutputStream outputStream = fs.create(path);
byte[] buffer = new byte[1024];
int bytesWritten = outputStream.write(buffer);

Creating Directories

The 

          
           
mkdirs()
method can be used to create a directory. This method takes a 

          
           
Path
object for the directory to be created and returns a boolean indicating whether the directory was successfully created. For example: 

          
           
Path path = new Path("/user/hadoop/data/foo");
boolean created = fs.mkdirs(path);

Deleting Files

The 

          
           
delete()
method can be used to delete a file. This method takes a 

          
           
Path
object for the file to be deleted and returns a boolean indicating whether the file was successfully deleted. For example: 

          
           
Path path = new Path("/user/hadoop/data/test.txt");
boolean deleted = fs.delete(path);
