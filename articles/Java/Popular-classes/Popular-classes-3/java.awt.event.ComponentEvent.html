java.awt.event.ComponentEvent

java.awt.event.ComponentEvent is an event which indicates that a component has been changed in some way, such as being moved, resized, hidden, or shown. This class can be subclassed to create more specialized events. The ComponentEvent class is used both for providing information about changes to an object, and for inter-component communication.

Example 1

The following example illustrates how to create and use a ComponentEvent. In the example, a window (created by Window class) is created and it registers a listener for ComponentEvents (implemented by MyComponentListener). When the window is moved or resized, a ComponentEvent is fired which is handled by the MyComponentListener which prints out a message.


          
           
import java.awt.Window;
import java.awt.event.ComponentEvent;
import java.awt.event.ComponentListener;
 
public class Example {
 
    public static void main(String[] args) {
        // Create the window
        Window window = new Window();
 
        // Register a listener for Component Events
        window.addComponentListener(new MyComponentListener());
    }
 
    static class MyComponentListener implements ComponentListener {
 
        public void componentResized(ComponentEvent event) {
            System.out.println("Window is resized.");
        }
 
        public void componentMoved(ComponentEvent event) {
            System.out.println("Window is moved.");
        }
 
        public void componentShown(ComponentEvent event) {
            System.out.println("Window is shown.");
        }
 
        public void componentHidden(ComponentEvent event) {
            System.out.println("Window is hidden.");
        }
    }
}

Example 2

In this example, an applet (created by Applet class) is created and it registers an adapter (implemented by MyComponentAdapter) for ComponentEvents. When the applet is moved or resized, a ComponentEvent is fired which is handled by the MyComponentAdapter which prints out a message.


          
           
import java.applet.Applet;
import java.awt.event.ComponentEvent;
import java.awt.event.ComponentAdapter;
 
public class Example extends Applet {
 
    public void init() {
        // Register an adapter for ComponentEvents
        addComponentListener(new MyComponentAdapter());
    }
 
    class MyComponentAdapter extends ComponentAdapter {
        public void componentResized(ComponentEvent event) {
            System.out.println("Applet is resized.");
        }
 
        public void componentMoved(ComponentEvent event) {
            System.out.println("Applet is moved.");
        }
    }
}
