javax.swing.border.EmptyBorder Overview

The javax.swing.border.EmptyBorder class provides an empty border which takes up no space. It is a part of the Java Foundation Classes (JFC), a suite of packages for developing graphical user interfaces (GUIs). EmptyBorder is used to create a visual margin around components, such as text fields or buttons. It can also be used as a decorative element by itself.

Creating an EmptyBorder Object

An EmptyBorder object can be created with one of the following constructor methods: 

          
           
//Constructs an EmptyBorder with the specified margins
public EmptyBorder(int top, int left, int bottom, int right)

//Constructs an EmptyBorder with the specified margins and an Insets object
public EmptyBorder(Insets insets)

//Constructs an EmptyBorder with the specified margins and a MatteBorder color
public EmptyBorder(int top, int left, int bottom, int right, Color matteColor)

Using an EmptyBorder

An EmptyBorder can be applied to any component by using the setBorder() method, as shown in the following example: 

          
           
JLabel label = new JLabel("Text");
label.setBorder(new EmptyBorder(10, 10, 10, 10));
//Creates an EmptyBorder with 10px margins around the label

Example

The following example creates a GUI frame containing a label and button. The frame has a 

          
           
BorderLayout
layout, with the label and button positioned at the north and south sides respectively. The background of the label is set to orange and the label is assigned an EmptyBorder with 10px margins. 

          
           
import javax.swing.*;
import javax.swing.border.EmptyBorder;
import java.awt.*;

public class EmptyBorderExample {
  public static void main(String[] args) {
    //Create a window
    JFrame window = new JFrame("EmptyBorder Example");
    window.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);

    //Create label with orange background
    JLabel label = new JLabel("Label", SwingConstants.CENTER);
    label.setBackground(new Color(0xFF9900));
    label.setOpaque(true);
    //Apply EmptyBorder to label with 10px margins
    label.setBorder(new EmptyBorder(10, 10, 10, 10));

    //Create button
    JButton button = new JButton("Button");

    //Set border layout
    window.setLayout(new BorderLayout());
    //Position label and button in north and south
    window.add(label, BorderLayout.NORTH);
    window.add(button, BorderLayout.SOUTH);

    //Display window
    window.pack();
    window.setVisible(true);
  }
}
