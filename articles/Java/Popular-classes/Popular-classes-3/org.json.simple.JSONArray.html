org.json.simple.JSONArray

The org.json.simple.JSONArray class is a powerful utility used to create and modify arrays in the Java programming language. This class allows developers to parse and build JSON (JavaScript Object Notation) code quickly and with ease. It also offers a number of convenience methods that make creating and manipulating JSON code much simpler. In this article, we will look at some of the features of this class and see how it can be used to improve your code.

Creating JSONArrays

The org.json.simple.JSONArray class provides several different ways for developers to create a new array. One way is to pass an existing array as argument to the JSONArray constructor, like so:


          
           
JSONArray array = new JSONArray(existingArray);

Another way is to use the static method from() to directly parse a String into a JSONArray, like so:


          
           
JSONArray array = JSONArray.from("[\"foo\", \"bar\", \"baz\"]");

It is also possible to add elements one by one to a JSONArray, like so:


          
           
JSONArray array = new JSONArray();
array.add("foo");
array.add("bar");
array.add("baz");

int size = array.size();

To get the value of an element in a JSONArray, use the get() method:


          
           
String element = (String)array.get(0); // element is "foo"

Modifying JSON Arrays

The org.json.simple.JSONArray class also provides several methods for modifying existing arrays, including methods for adding, removing, and replacing elements. To add an element to a JSONArray, use the add() method, like so:


          
           
array.add("qux"); // array is now ["foo", "bar", "baz", "qux"]

To remove an element from a JSONArray, use the remove() method, like so:
