Creating JSONArrays

The org.json.simple.JSONArray class provides several different ways for developers to create a new array. One way is to pass an existing array as argument to the JSONArray constructor, like so:


          
           
JSONArray array = new JSONArray(existingArray);

Another way is to use the static method from() to directly parse a String into a JSONArray, like so:


          
           
JSONArray array = JSONArray.from("[\"foo\", \"bar\", \"baz\"]");

JSONArray array = new JSONArray();
array.add("foo");

array.add("baz");
