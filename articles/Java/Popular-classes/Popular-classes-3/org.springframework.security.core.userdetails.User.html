org.springframework.security.core.userdetails.User

The org.springframework.security.core.userdetails.User class is an implementation of the org.springframework.security.core.userdetails.UserDetails interface. It provides basic functionality for representing a user in the security system. It holds the username, password and authorities granted to the user.

        Collection<!--? extends GrantedAuthority--> authorities)
This constructor is a shortcut version, which enables the user by default, and sets the account, credentials and account-locked flags appropriately.

Methods


          
           
public boolean isAccountNonExpired()
This method returns true if the user's account has not expired, false otherwise. 

          
           
public boolean isAccountNonLocked()
This method returns true if the user's account is not locked, false otherwise. 

          
           
public boolean isCredentialsNonExpired()
This method returns true if the user's credentials have not expired, false otherwise. 

          
           
public boolean isEnabled()
This method returns true if the user is enabled (active), false otherwise.
