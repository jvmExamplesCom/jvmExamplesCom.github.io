Java.awt.FlowLayout

Java.awt.FlowLayout is a layout class used to arrange components in a specific manner. It arranges components in a sequence, one after the other, and aligns them accordingly with the container’s width. The flow layout provides an easy way to manage the components in the interface that can be dynamically changed as per the needs of the user.

FlowLayout flowLayout = new FlowLayout();

The FlowLayout class constructor takes four optional arguments. The first argument defines the horizontal alignment of the components within the layout, which can be either LEFT, RIGHT, CENTER or LEADING. The default alignment is centered.


          
           
FlowLayout flowLayout = new FlowLayout(FlowLayout.LEFT);

The second argument sets the horizontal gap between components. By default, it is set to 5 pixels.


          
           
FlowLayout flowLayout = new FlowLayout(FlowLayout.LEFT, 10);

The fourth argument sets whether components are resized to fit the size of the layout. By default, it is set to true.
