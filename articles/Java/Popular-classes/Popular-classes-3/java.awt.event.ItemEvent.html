java.awt.event.ItemEvent

Java.awt.event.ItemEvent is a class that handles events generated by items in ItemSelectable objects, such as Checkboxes and List components.

Methods

  • public Object getItem()
    Returns the selection item effected by the ItemEvent.
  • public StateChangeEvent getStateChangeEvent()
    Returns the state change event associated with the ItemEvent.
  • public int getStateChange()
    Returns an integer value indicating which type of change occurred in the selection event.

Code Examples

Here are some code examples demonstrating how to utilize the java.awt.event.ItemEvent class.


          
           
// Create a Checkbox
Checkbox check = new Checkbox("Check Me");

// Add an ItemListener to the checkbox
check.addItemListener(new ItemListener(){
    public void itemStateChanged(ItemEvent e){
        if(e.getStateChange() == ItemEvent.SELECTED){
            System.out.println("Checkbox checked");
        } else {
            System.out.println("Checkbox unchecked");
        }
    }
});

          

          
           
// Create a list
String[] listItems = {"Apple", "Banana", "Orange"};
List list = new List(listItems);

// Add an ItemListener to the list
list.addItemListener(new ItemListener(){
    public void itemStateChanged(ItemEvent e){
        if(e.getStateChange() == ItemEvent.SELECTED){
            System.out.println("Selected item: " + e.getItem());
        }
    }
});
