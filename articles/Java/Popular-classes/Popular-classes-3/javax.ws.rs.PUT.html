Annotation

When used as an annotation, the javax.ws.rs.PUT annotation is used to provide metadata about a particular web service request. This includes information such as the URI path of the web service, any applicable query parameters and headers, and the representation of the request data.

@Path("/user/{id}")
public Response updateUser(@PathParam("id") int id, User user) {
    //...
}
