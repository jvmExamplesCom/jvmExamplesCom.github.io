Examples of Using DateTimeFormat

Formatting a DateTime Object as a String

format(DateTime dateTime)
method can be used to format a DateTime object as a string. For example, the following code will format a DateTime object as a string in the format MM/dd/yyyy:

Parsing a String into a DateTime Object

The 

          
           
parseDateTime(String str)
method can be used to parse a string into a DateTime object. For example, the following code will parse a string in the format MM/dd/yyyy into a DateTime object:

DateTimeFormatter formatter = DateTimeFormat.forPattern("MM/dd/yyyy");
DateTime dateTime = formatter.parseDateTime(dateTimeStr);
System.out.println(dateTime); // Outputs "2019-12-03T00:00:00.000Z"

Formatting a DateTime Object using a Locale

withLocale(Locale locale)
method can be used to specify the locale that should be used when formatting a DateTime object as a string. For example, the following code will format a DateTime object as a string in the US locale: 

          
           
DateTimeFormatter formatter = DateTimeFormat.forPattern("MM/dd/yyyy")
