Introduction to org.openqa.selenium.WebElement

The org.openqa.selenium.WebElement class represents an HTML element. It provides methods for interacting with DOM elements in a browser. This class enables users to locate and manipulate web elements such as buttons, links, and text boxes. This class is indispensable when writing automated tests with Selenium WebDriver.

Instantiating WebElement Objects

WebElement objects can be instantiated by locating the element using Selenium WebDriver. The following example uses the findElement method to locate a button on a web page and create a WebElement object for it. 

          
           
WebElement element = driver.findElement(By.id("button-id"));

Interacting with WebElement Objects

Once an object is created, you can interact with it. The following example clicks on the button located in the previous example. 

          
           
element.click();
You can also use the WebElement class to get and set values of elements. For example, to set the value of a text input: 

          
           
element.sendKeys("input text");
And to get the value: 

          
           
String value = element.getAttribute("value");

Getting Information from WebElement Objects

In addition to getting and setting values, the WebElement class provides methods for getting information about the element. getting a boolean value indicating if the element is enabled: 

          
           
boolean isEnabled = element.isEnabled();
getting the element’s tag name: 

          
           
String tagName = element.getTagName();
getting the element’s text content: 

          
           
String textContent = element.getText();

Conclusion

The org.openqa.selenium.WebElement class provides a powerful way of working with web elements. It enables users to locate and interact with web elements, get and set values, and get information about the element. With the help of this class, users can easily write automated tests with Selenium WebDriver.
