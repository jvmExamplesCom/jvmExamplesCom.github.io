Java .awt.FontMetrics

The 

          
           
java.awt.FontMetrics
class provides information about the dimensions of characters and strings when they are rendered in a particular font. It also provides methods for getting the standard height, ascent, descent, leading, and average character width of a font. The following example demonstrates how FontMetrics can be used to measure the length of a string: 

          
           
String str = "This is a string";
FontMetrics fm = new FontMetrics();
int width = fm.stringWidth(str);
System.out.println("Width of String: " + width);
The output of this code is 
Width of String: 73
, which measures the length of the string in pixels.

Methods


          
           
java.awt.FontMetrics
provides several useful methods, the most commonly used ones are listed below: 
  • getAscent()
    : Returns the maximum distance characters extend above the baseline.
  • getDescent()
    : Returns the maximum distance characters extend below the baseline.
  • getHeight()
    : Returns the total height of a line of text.
  • getLeading()
    : Returns the amount of space between lines of text.
  • 
            
             
stringWidth(String str)
    : Returns the width of the specified string in pixels.
  • getMaxAdvance()
    : Returns the maximum width of any character in the specified font.
  • 
            
             
getAverageCharacterWidth()
    : Returns the average width of all characters in the specified font.

Example

The following example demonstrates how to use different 

FontMetrics
methods: 

          
           
Font font = new Font("Arial", Font.PLAIN, 12);
FontMetrics fm = getFontMetrics(font);

int ascent = fm.getAscent();
System.out.println("Ascent: " + ascent);

int descent = fm.getDescent();
System.out.println("Descent: " + descent);

int height = fm.getHeight();
System.out.println("Height: " + height);

int leading = fm.getLeading();
System.out.println("Leading: " + leading);

String str = "This is a string";
int width = fm.stringWidth(str);
System.out.println("Width of String: " + width);

int maxAdv = fm.getMaxAdvance();
System.out.println("Max Advance: " + maxAdv);

int avgCharWidth = fm.getAverageCharacterWidth();
System.out.println("Average Character Width: " + avgCharWidth);
The output of this code is: 

          
           
Ascent: 10
Descent: 4
Height: 14
Leading: 0
Width of String: 73
Max Advance: 7
Average Character Width: 5

Conclusion

The 

          
           
java.awt.FontMetrics
class provides methods for measuring the dimensions of strings and characters when rendered in a particular font. This can be useful for creating graphical interfaces that require precise measurements of text.
