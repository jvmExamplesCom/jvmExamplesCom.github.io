Introduction to android.view.TextureView

Android.view.TextureView enables developers to display a content stream within an application. This class is especially useful when content streaming must be integrated with UI elements, or with hardware acceleration. Additionally, android.view.TextureView can be used to efficiently display several content streams simultaneously.

<TextureView

    android:layout_height="match_parent"/>

The following code example illustrates how to initialize the TextureView within a Java activity:


          
           
TextureView textureView = (TextureView) findViewById(R.id.texture_view);
