Introduction to java.util.EventObject

Java.util.EventObject is a class that provides functionality for notification when an event happens. It is part of the java.util package, which provides common utility functionalities for developing Java programs. EventObject is typically used in application development for handling events, such as user actions. In this article, we will explore EventObject and its methods.

Constructors

EventObject has two constructors you can use: 

          
           
public EventObject(Object source) 

public EventObject(Object source, String name)

The first constructor creates an event object with the specified source object. The second overrides the first one and also enables passing a custom name for the event.

Methods

EventObject provides the following methods you can use: 


          
           
public Object getSource()

public boolean equals(Object obj)

public int hashCode()

public String toString()
