javax.swing.event.ListSelectionEvent

javax.swing.event.ListSelectionEvent
is a class used in the Java Swing library to generate list selection events to be used by classes that implement 

          
           
javax.swing.event.ListSelectionListener
. This event is triggered when a single item, multiple items, or a range of items are selected in the list. When this occurs, the selection event is generated and sent to the relevant listeners, who can then take appropriate action.

Example Usage

We can create a new 
ListSelectionEvent
with the following code: 

          
           
ListSelectionEvent e = new ListSelectionEvent(this, firstIndex, lastIndex, 	isAdjusting);

this
is the source of the event, 
firstIndex
and 
lastIndex
are the start and end indices of the selection respectively, and 
isAdjusting
is a boolean that determines whether the selection is in a state of flux while this event is handled.

ListSelectionEvent

e.addListSelectionListener(listener);
