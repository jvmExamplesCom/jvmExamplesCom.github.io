Java com.google.common.io.BaseEncoding

Java com.google.common.io.BaseEncoding provides a set of convenient methods for encoding and decoding byte arrays, using both base64 and Base32 encodings. This class handles all the details of padding out byte arrays and converting between bytes and characters in various character sets.

Encoding an array of bytes with Base64

Base64 is a popular encoding for binary data, and in Java it is provided by the class com.google.common.io.BaseEncoding. Encoding an array of bytes with Base64 is simple:


          
           
byte[] data = new byte[] { 0x01, 0x02, 0x03, 0x04 };
String encoded = BaseEncoding.base64().encode(data);
System.out.println(encoded);

The output of this code will be the encoded string 

AQIDBA==
.

Decoding a Base64 string

You can easily decode a Base64 string back into its original byte array, using the 

decode()
method of the BaseEncoding class: 

          
           
String encoded = "AQIDBA==";
byte[] decoded = BaseEncoding.base64().decode(encoded);
for (byte b : decoded) {
    System.out.printf("%02x ", b);
}
This code will output 
01 02 03 04
, which is the original byte array that was encoded.

Encoding an array of bytes with Base32

You can also use the BaseEncoding class to encode an array of bytes with the Base32 encoding. The process is similar to Base64, just use the 

base32()
method instead: 

          
           
byte[] data = new byte[] { 0x01, 0x02, 0x03, 0x04 };
String encoded = BaseEncoding.base32().encode(data);
System.out.println(encoded);

The output of this code will be the encoded string 

AEBGCF==
.

Decoding a Base32 string

You can easily decode a Base32 string back into its original byte array, using the same 

decode()
method of the BaseEncoding class: 

          
           
String encoded = "AEBGCF==";
byte[] decoded = BaseEncoding.base32().decode(encoded);
for (byte b : decoded) {
    System.out.printf("%02x ", b);
}
This code will again output 
01 02 03 04
