What is org.junit.rules.ExternalResource?

org.junit.rules.ExternalResource is a base class for Rules (similar to a Test rule) used for lightweight setup and teardown of external resources in a JUnit test suite. Examples of such external resources include databases, servers, and other long-living resources used in the entire test suite.

Code Examples and Output

We can use org.junit.rules.ExternalResource in our JUnit tests like this:


          
           
@Rule 
public ExternalResource resource = new ExternalResource() { 

    protected void before() throws Throwable { 
        //Initially set up environment 
    }; 

    protected void after() { 
        //Tear down environment  
    }; 
};

before()
method is used to setup the environment, and the 
after()
method is used to tear down the environment. We can also override the 

          
           
protected void starting()

protected void finished()

Conclusion

The org.junit.rules.ExternalResource class provides an easy way to manage external resources in a JUnit test suite. We can use it to set up and tear down resources such as databases before and after each test, ensuring that the environment is always in a known state before and after each test run.
