org.apache.http.client.methods.HttpRequestBase

The org.apache.http.client.methods.HttpRequestBase class is a base class for all HTTP request methods in the Apache HttpClient library, providing common functionality and utilities for implementation classes. HttpRequestBase provides abstractions for priority, entity, headers, and other commonly used request parameters. This class is used in the construction of HTTP requests for use with the HttpClient. It does not actually initiate any communication with the target server, but rather provides immutable objects for use in the creation of requests.
