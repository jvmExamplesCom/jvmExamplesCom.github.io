java.util.logging.Handler

java.util.logging.Handler

java.util.logging.Handler
class, first it must be instantiated with a logger object. A logger object is an instance of the 

          
           
java.util.logging.Logger

Logger logger = Logger.getLogger("MyLogger");

logHandler.publish(myLogRecord);

logHandler.flush();
