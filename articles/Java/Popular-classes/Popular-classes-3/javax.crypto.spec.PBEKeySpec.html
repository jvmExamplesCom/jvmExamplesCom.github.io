javax.crypto.spec.PBEKeySpec

The PBEKeySpec class provides a way to generate and retrieve passwords or keys for use in Password Based Encryption (PBE) algorithms. This class provides support for generating secure, random passwords with varying levels of strength, as well as retrieving keys from a securely stored key specification.

Overview

The javax.crypto.spec.PBEKeySpec class provides functions for the generation of passwords and key retrieval from a Password Based Encryption (PBE) algorithm. It is typically used to create passwords for encryption, decryption, and message authentication codes (MACs).

Example Code

The following example code shows how to generate a secure, random password using PBEKeySpec. 


          
           
import javax.crypto.SecretKeyFactory;
import javax.crypto.spec.PBEKeySpec;

// Generate a 32-byte random salt
byte[] salt = new byte[32];
SecureRandom random = SecureRandom.getInstanceStrong();
random.nextBytes(salt);

int iterationCount = 1000; // number of hashing iterations

// Generate a secure 32-byte random password 
char[] password = new char[32];
random.nextBytes(password);

PBEKeySpec keySpec = new PBEKeySpec(password, salt, iterationCount);

SecretKeyFactory keyFactory =
    SecretKeyFactory.getInstance("PBKDF2WithHmacSHA256");

byte[] keyBytes = keyFactory.generateSecret(keySpec).getEncoded();

// Print the derived key in hexadecimal format
System.out.println(DatatypeConverter.printHexBinary(keyBytes));
