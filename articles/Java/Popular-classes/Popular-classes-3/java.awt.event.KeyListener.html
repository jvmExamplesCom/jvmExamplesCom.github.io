What is Java.awt.event.KeyListener?

Java.awt.event.KeyListener is an interface in the Java AWT (Abstract Window Toolkit) package that informs an object when a key on the keyboard has been pressed or released. It is the most commonly used method for capturing user input from the keyboard in Java Applications. This article will explain how to use this listener in your own applications.

What does KeyListener do?

The KeyListener interface contains three methods that are invoked when a key is pressed, released, or both. The three methods are: 

          
           
public void keyPressed(KeyEvent e);    //invoked when a key is pressed
public void keyReleased(KeyEvent e);   //invoked when a key is released
public void keyTyped(KeyEvent e);      //invoked when a key is typed
The KeyEvent parameter passed to these methods contains information about which key was pressed or released, as well as the current state of the keyboard at the time of the event.

Example of KeyListener Usage

In order to use the KeyListener to capture user input, we'll first need to create an instance of it. We can do this by implementing the KeyListener interface into our class: 

          
           
public class MyClass implements KeyListener {...}
Now that our class is set up to receive key events, we can implement the KeyListener methods to respond to user inputs. In the following example, we'll print out a message whenever the space bar is pressed on the keyboard. 

          
           
public class MyClass implements KeyListener {

  public void keyPressed(KeyEvent e) {
    if (e.getKeyCode()== KeyEvent.VK_SPACE)
      System.out.println("Space bar pressed!");
  }

  public void keyReleased(KeyEvent e) {
    // Empty implementation
  }

  public void keyTyped(KeyEvent e) {
    // Empty implementation
  }

}
In the code above, we first check to see which key was pressed using the getKeyCode() method from the KeyEvent class. If the space bar was pressed, then we print out a message.

Adding the Listener to a Component

In order for the KeyListener to work, it needs to be added to a component in our application. The most common way to do this is through the addKeyListener() method of the component class. For example, if we wanted our KeyListener to listen for key presses on a JTextField, we could add it like this: 

          
           
MyClass myClass = new MyClass();
JTextField textField = new JTextField();
textField.addKeyListener(myClass);
Now, every time a key is pressed on the text field, we'll be notified through the keyPressed() method of our class.

Conclusion

The KeyListener is a powerful tool for capturing user input in Java applications. By implementing the KeyListener interface and adding it to a component, you can easily capture the user's keystrokes and respond accordingly.
