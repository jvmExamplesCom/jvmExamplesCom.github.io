java.nio.channels.Selector

The java.nio.channels.Selector class provides the capability of non-blocking multiplexed I/O operations. This class is included in the Java NIO package and allows applications to use fewer threads to perform I/O operations. Selectors are created with the open() method of the Selector class.

Creating a Selector

The following code example shows how to create a new instance of the Selector class:


          
           
Selector selector = Selector.open();

SelectionKey key = channel.register(selector, SelectionKey.OP_READ);

The register() method returns a SelectionKey object. This object encapsulates the state of the connection between the channel and the selector and also provides access to the set of operations that the channel is interested in.
