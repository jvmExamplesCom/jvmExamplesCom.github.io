android.view.animation.RotateAnimation

Android's RotateAnimation class enables developers to easily create rotation effects for their applications. The animation can be defined in XML or programmatically. It is a subclass of Animation and the parent class for any other rotational animation, such as Rotate3dAnimation and ScaleAnimation.

Creating a RotateAnimation in XML

To create a RotateAnimation in XML, you'll need to define the "fromDegrees", "toDegrees" and "pivotX" and "pivotY" properties. The "fromDegrees" and "toDegrees" properties set the starting and ending angle of the rotation. The "pivotX" and "pivotY" properties define the point at which the rotation will take place. The values must be given as percentages of the view's width and height respectively.

    android:duration="500"
    android:fromDegrees="0"
    android:toDegrees="90"

    android:pivotY="50%" />

Creating a RotateAnimation Programmatically

You can also create a RotateAnimation programmatically. First you'll need to create an instance of the RotateAnimation class. You can then set the duration of the animation by calling the setDuration() method and set the rotation angle by calling the setFromDegrees() and setToDegrees() methods. Lastly, you'll need to set the pivot point of the rotation by calling the setPivotX() and setPivotY() methods.
