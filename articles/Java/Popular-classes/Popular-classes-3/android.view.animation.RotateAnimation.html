To create a RotateAnimation in XML, you'll need to define the "fromDegrees", "toDegrees" and "pivotX" and "pivotY" properties. The "fromDegrees" and "toDegrees" properties set the starting and ending angle of the rotation. The "pivotX" and "pivotY" properties define the point at which the rotation will take place. Creating a RotateAnimation Programmatically
