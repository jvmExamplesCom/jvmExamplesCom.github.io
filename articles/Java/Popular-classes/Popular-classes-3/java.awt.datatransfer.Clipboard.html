Java.awt.datatransfer.Clipboard

The java.awt.datatransfer.Clipboard class provides access to the system clipboard, including a way of transferring objects across applications. This object allows applications to share data by using the common clipboard facility provided by the underlying platform, such as the X window system.

Creating a Clipboard Object

A Clipboard object can be created like this: 

          
           
Clipboard clip = new Clipboard("myClip");

Once created, the clipboard can be used for various operations such as get and set.

Getting Objects from the Clipboard

To get an object from the clipboard, the following code can be used:


          
           
Object data = clip.getData(DataFlavor.stringFlavor);

The above code snippet will get the data stored on the clipboard in the given data flavor. The value returned will be of the type specified by the data flavor.

Setting Objects on the Clipboard

To set an object on the clipboard, the following code can be used:


          
           
clip.setContents(new StringSelection("Hello World!"), null);

The above code snippet sets a string value on the clipboard. The first argument is the data to be set, and the second argument is a transferable object that contains the data.

Transferring Objects

To transfer objects across applications, a Transferable object needs to be created. The code below creates a transferable object which can contain a String object:


          
           
Transferable trans = new StringTransferable("Hello World!");

The Transferable object can then be added to the clipboard using the following code:


          
           
clip.setContents(trans, null);

The Transferable object can be retrieved from the clipboard using the getData() method:


          
           
String data = (String) clip.getData(DataFlavor.stringFlavor);

The retrieved value will be of the same type as the one set on the clipboard.

Conclusion

The java.awt.datatransfer.Clipboard class provides an easy way to transfer objects between applications. By creating a Clipboard object, data can be retrieved and set on the clipboard, and objects can be transferred using the Transferable interface.
