org.apache.http.auth.AuthScope

Introduction

org.apache.http.auth.AuthScope is an authentication scope class, which encapsulates a set of authentication parameters associated to a certain authentication scope. It serves as an argument to the credentials provider for indicating what authentication parameters need to be used for supplying credentials for given set of target host, port and realm.
