javax.xml.xpath.XPathExpressionException

javax.xml.xpath.XPathExpressionException is an exception that is thrown when a problem occurs while parsing an XPath expression. It typically occurs when a malformed or invalid XPath expression is used in the code. This exception is a subclass of the org.xml.sax.SAXException and it extends the standard Java exception class java.lang.Exception.

In this example, we will try to evaluate an invalid XPath expression using the javax.xml.xpath.XPath API. 

          
           
import javax.xml.xpath.XPath;
import javax.xml.xpath.XPathConstants;
import javax.xml.xpath.XPathExpression;
import javax.xml.xpath.XPathExpressionException;
import javax.xml.xpath.XPathFactory;

public class App 
{
    public static void main( String[] args )
    {
        try 
		{
            XPathFactory factory = XPathFactory.newInstance();
            XPath xPath = factory.newXPath();

            // invalid XPath expression
            String expression = "//*[@id='@id']";

            // compile the XPath expression
            XPathExpression xPathExpression = xPath.compile(expression);

            System.out.println("Error: " + e.getMessage());

Example 2: Using isValid() Method to Avoid Throwing javax.xml.xpath.XPathExpressionException

In this example, we will use the isValid() method to check if the XPath expression is valid before evaluating it. 

          
           
import javax.xml.xpath.XPath;
import javax.xml.xpath.XPathFactory;

public class App 
{
    public static void main( String[] args )
    {
        XPathFactory factory = XPathFactory.newInstance();
        XPath xPath = factory.newXPath();

        // invalid XPath expression
        String expression = "//*[@id='@id']";

        // check if the expression is valid
        boolean isValid = xPath.isValid(expression);

        if(isValid) 
		{
            // compile the XPath expression
            XPathExpression xPathExpression = xPath.compile(expression);

            // evaluate the expression
            Object result = xPathExpression.evaluate(document, XPathConstants.NODESET);
        } 
		else 
		{
            System.out.println("Invalid XPath expression");
        }
    }
}
The output will be: 

          
           
Invalid XPath expression
In this example, we have used the isValid() method to check if the given XPath expression is valid or not. If the expression is invalid, then javax.xml.xpath.XPathExpressionException is avoided.
