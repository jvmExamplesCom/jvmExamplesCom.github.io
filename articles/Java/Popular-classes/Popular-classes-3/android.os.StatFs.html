android.os.StatFs

android.os.StatFs is a class that provides methods for accessing information about the system's filesystems. It can be used to query the status of the filesystem, get the size and number of blocks, calculate the available space, and even determine the maximum filename length. This class is particularly useful for accessing system-level information, such as the total amount of memory available on the device.

Methods

getAvailableBytes()

The 

          
           
getAvailableBytes()
method is used to retrieve the number of available bytes on the file system. It takes no arguments and returns a long integer representing the byte count.

getBlockSize()

The 

          
           
getBlockSize()
method returns the size of each data block in bytes. It takes no arguments and returns an int.

getFreeBlocks()

The 

          
           
getFreeBlocks()
method returns the number of unallocated blocks available on the file system. It takes no arguments and returns a long integer representing the block count.

getMaxFilenameLength()

The 

          
           
getMaxFilenameLength()
method returns the maximum length of filenames allowed on the file system. It takes no arguments and returns an int representing the maximum filename length.

getTotalBytes()

The 

          
           
getTotalBytes()
method returns the total number of bytes on the file system. It takes no arguments and returns a long integer representing the total byte count.

restat()

The 

          
           
restat()
method can be used to refresh the file system data stored by the StatFs instance. It takes no arguments and does not return a value.

Code Examples

The following code examples demonstrate how to use the android.os.StatFs class.

Retrieve the Amount of Available Memory on the Device


          
           
// Create new instance of StatFs
StatFs stat = new StatFs(Environment.getRootDirectory().getAbsolutePath());

// Retrieve the available memory
long availableMemory = stat.getAvailableBytes();

Calculate the Size and Number of Data Blocks


          
           
// Create new instance of StatFs
StatFs stat = new StatFs(Environment.getRootDirectory().getAbsolutePath());

// Retrieve the size of each data block
int blockSize = stat.getBlockSize();

// Calculate the number of data blocks
long blockCount = stat.getAvailableBytes() / blockSize;

Determine the Maximum Filename Length on the File System


          
           
// Create new instance of StatFs

// Retrieve the maximum filename length
int maxFilenameLength = stat.getMaxFilenameLength();
