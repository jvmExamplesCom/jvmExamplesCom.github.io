JsonSerializationContext

JsonSerializationContext is a Java class that enables easy serialization of arbitrary Java objects into JSON formatted text or strings. The class can also be used to serialize primitive types and java.util.Date objects into compatible JSON. It was introduced in JAVA 7 and is part of the com.google.gson package. It provides several useful methods to transform Java classes into JSON formats.

Method Overview

The 

          
           
JsonSerializationContext
class provides several methods for serializing various data types into JSON. The most commonly used methods are listed below: 
  • 
            
             
toJsonObject(Object obj)
    - serializes an Object into a JSONObject.
  • 
            
             
toJsonArray(Object[] obj)
    - serializes an Object array into a JSONArray.
  • 
            
             
toJsonPrimitive(Object value)
    - serializes a primitive value (i.e. boolean, int, float, char, etc.) into a JSON primitive.
  • 
            
             
toJsonString(Object obj)
    - serializes a String into a JSON string.
  • 
            
             
toJsonDate(java.util.Date date)
    - serializes a java.util.Date into a JSON string.

Example Usage : Serializing an Object

Let's assume we have an 

          
           
Employee
class which has fields like 

          
           
name, age and address
. We can use the 

          
           
JsonSerializationContext
class to serialize this object into JSON format with the 

          
           
toJsonObject()
method: 

          
           
JsonSerializationContext context = new JsonSerializationContext();
Employee employee = new Employee("John Doe", 32, "123 Main St.");

JsonObject employeeJson = context.toJsonObject(employee);
System.out.println(employeeJson.toString());
This will output the following JSON string: 

          
           
{
  "name": "John Doe",
  "age": 32,
  "address": "123 Main St."
}

Let's now assume we have an array of 

          
           
Employee

toJsonArray()

JsonSerializationContext context = new JsonSerializationContext();
Employee[] employees = {
    new Employee("John Doe", 32, "123 Main St."),
    new Employee("Jane Doe", 30, "456 Main St.")
