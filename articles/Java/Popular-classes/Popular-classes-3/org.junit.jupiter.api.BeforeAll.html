org.junit.jupiter.api.BeforeAll

org.junit.jupiter.api.BeforeAll is an annotation used to signal that the annotated method should be executed before all tests in a given test class. This annotation is semantically equivalent to JUnit 4’s @BeforeClass annotation. This annotation can be used on methods declared in non-static inner classes.

Code Examples


          
           
@BeforeAll
public static void setupTestSession() {
   // Initialization logic
}

The annotated method can be declared with any access modifier (public, protected, package private or private). Furthermore, it must not be declared as static. Only one method in a given test class may be annotated with @BeforeAll.
