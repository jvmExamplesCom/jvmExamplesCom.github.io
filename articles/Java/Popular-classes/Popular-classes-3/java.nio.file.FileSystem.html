Introduction to Java.nio.file.FileSystem

Java.nio.file.FileSystem is an I/O (Input/Output) based API which is used to access files and other related resources that are outside the scope of the Java Virtual Machine (JVM). This makes it suitable for applications that need to access such external resources without being restricted by the JVM. It provides a way for applications to interact with the underlying file system in a strong-typed manner, making it easy to work with files and directories.

Creating a FileSystem Object

The FileSystem class provides several factory methods to create a FileSystem instance. The most common method is to use the getFileSystem method. This takes the URI of the file system to be accessed as an argument, and returns a FileSystem instance for accessing the file system. For example: 

          
           
FileSystem fs = FileSystems.getFileSystem(URI.create("file:///"));
