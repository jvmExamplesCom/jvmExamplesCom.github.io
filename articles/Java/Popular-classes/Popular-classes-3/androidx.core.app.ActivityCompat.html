Using androidx.core.app.ActivityCompat

AndroidX Core App ActivityCompat is an open-source library that provides backward compatibility for applications using AndroidX Core App components and allows developers to more easily access components such as the camera, contacts list and location services. This library provides a convenient way for developers to access important system functionality without having to worry about compatibility with older versions of Android.

Import and Activate

Before using the library, you must first import it into your application by adding the following line to your app's build.gradle file:


          
           
implementation "androidx.core:core:${version}"

Once you've imported the library, you must also activate it in your code by adding the following line before calling any of its methods:


          
           
ActivityCompat.requestPermissions(this, new String[]{Manifest.permission.CAMERA}, 0);

Common Use Cases

The most common use cases for the AndroidX Core App ActivityCompat library include:

  • Accessing the Camera
  • Accessing the Contacts List
  • Accessing Location Services

Camera Access

if(ActivityCompat.checkSelfPermission(this, Manifest.permission.CAMERA) != PackageManager.PERMISSION_GRANTED) {
