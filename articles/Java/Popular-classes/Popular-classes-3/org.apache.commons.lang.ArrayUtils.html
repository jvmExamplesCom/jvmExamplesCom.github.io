ArrayUtils in Java: Introduction

ArrayUtils is a utility class in the package org.apache.commons.lang of the Apache Commons Lang library introduced in version 1.0 for manipulating Java arrays. This utility class offers methods to manipulate arrays such as sorting, searching, transforming, copying, etc. It provides a convenient alternative to the array syntax available in the Java language since it has been designed to simplify and add functionality to many of the operations that one might need to do on an array. In this article, we will learn about the various methods provided by this utility class and go through some code examples.

Using ArrayUtils Class

Before we look into the methods provided by this utility class, let us take a look at how we can use the ArrayUtils class. The ArrayUtils class is an abstract class, so we cannot create its instance. To use its methods, we need to make sure that we have the Apache Commons Lang library installed in our project. We can do this by adding this dependency to our Gradle/Maven build file. Once we have the Apache Commons Lang library installed in our project, we can import the ArrayUtils class into our project with the following statement: 

          
           
import org.apache.commons.lang.ArrayUtils;
We can then use the various methods provided by this utility class.

Methods in ArrayUtils

Let us now look into some of the most commonly used methods in the ArrayUtils class.

add()

The add() method is used to add an element at the end of an array. 

          
           
String[] names = {"John", "Jane", "Adam"};
String[] newNames = ArrayUtils.add(names, "Jack");
In the above code example, we have an array of strings containing 3 elements. The add() method is used to add one more element - "Jack" to the array. The new array will contain 4 elements.

addAll()

The addAll() method is used to add all the elements of an array to another array. 

          
           
String[] names1 = {"John", "Jane", "Adam"};
String[] names2 = {"Max", "Hannah"};
String[] newNames = ArrayUtils.addAll(names1, names2);
In the above code example, we have two arrays - names1 and names2. The addAll() method is used to add all the elements of the names2 array to the names1 array and create a new array with the result. The new array will contain 5 elements.

contains()

The contains() method is used to check if an array contains a certain element. 

          
           
String[] names = {"John", "Jane", "Adam"};
boolean result = ArrayUtils.contains(names, "Adam"); //returns true
In the above code example, we have an array of strings containing 3 elements. The contains() method is used to check if the array contains the string "Adam". The method will return true if the array contains the element, and false otherwise.

indexOf()

The indexOf() method is used to get the index of an element in an array. 

          
           
String[] names = {"John", "Jane", "Adam"};
int index = ArrayUtils.indexOf(names, "Adam"); //returns 2
In the above code example, we have an array of strings containing 3 elements. The indexOf() method is used to get the index of the string "Adam" in the array. The method will return 2, which is the index of the element in the array.

sort()

The sort() method is used to sort an array in ascending or descending order. 

          
           
String[] names = {"John", "Jane", "Adam"};
String[] sortedNames = ArrayUtils.sort(names);
In the above code example, we have an array of strings containing 3 elements. The sort() method is used to sort the array in ascending order, so the resulting array will be ["Adam", "Jane", "John"].
