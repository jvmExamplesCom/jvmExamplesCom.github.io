javax.swing.Icon

The 
javax.swing.Icon
class provides an abstract class to manage icons in 
swing
, which represent graphical objects that can be used for representional purposes in a GUI. The 
javax.swing.Icon

java.lang.Object

javax.swing

getIconHeight
, 
getIconWidth
, and so on.

javax.swing.Icon
. It creates an ImageIcon with a specified image file, and then uses the

getIconHeight()
