android.webkit.MimeTypeMap Overview

The android.webkit.MimeTypeMap class provides access to an instance of the MIME type map associated with the system. This class is used to determine the type of a MIME type string. It parses MIME types and determines their standard extensions as well as MIME type of a file based on its filename.
