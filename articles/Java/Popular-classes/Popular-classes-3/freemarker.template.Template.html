Introduction to Freemarker.template.Template in Java

Freemarker.template.Template is an open source template engine for Java-based web applications. It provides developers with a powerful yet simple way to create dynamic web pages and applications, as well as a wide range of other document types. In this article, we'll take a look at the basic principles behind Freemarker.template.Template, how it works, and some examples of how to use it.

What is Freemarker.template.Template?

Freemarker.template.Template is a template engine written in Java. It is used to turn text documents into HTML, XML and other output formats using the Freemarker language. Freemarker was designed with model-view-controller (MVC) architecture in mind and is used widely in MVC frameworks like Spring MVC and JavaServer Faces. It is also widely used for generating emails, code snippets, and more.

How Does Freemarker.template.Template Work?

At its core, Freemarker.template.template works by parsing text documents and replacing specific fields with values from a data model. For example, if the text document contains the field ${name}, Freemarker will replace it with the value of the name property in the data model. Once all the fields have been replaced, the document is rendered as HTML or other output formats.

Example of Using Freemarker.template.Template

Below is an example of a simple Freemarker.template.Template template that renders a greeting card: 

          
           
<div class="greeting-card">
    <h1>Hello, ${name}!</h1>
    <p>It's nice to see you again! Have a great day!</p>
</div>
The template can be rendered using the following code snippet: 

          
           
Map<String, Object> model = new HashMap<>();
model.put("name", "John");

Configuration cfg = new Configuration(Configuration.VERSION_2_3_30);
cfg.setClassForTemplateLoading(FreemarkerDemo.class, "/templates");

Template template = cfg.getTemplate("greeting.ftl");

Writer out = new OutputStreamWriter(System.out);
template.process(model, out);
out.flush();
In this example, we first create a HashMap object that contains the model data for our template. We then set up the configuration for the template engine, and load our template file. Finally, we process the template using the model data, and write the output to the console. The resulting output will look something like this: 

          
           
<div class="greeting-card">
    <h1>Hello, John!</h1>
    <p>It's nice to see you again! Have a great day!</p>
</div>

Conclusion

Freemarker.template.Template is a powerful, easy-to-use template engine that can be used to generate a wide variety of documents. By defining templates in the Freemarker language, you can quickly and easily create dynamic documents such as emails, webpages, and more.
