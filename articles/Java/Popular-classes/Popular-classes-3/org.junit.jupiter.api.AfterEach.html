org.junit.jupiter.api.AfterEach

The org.junit.jupiter.api.AfterEach annotation is used to denote a method as an after-each test method that will be invoked after each test method of a test class. It is part of the JUnit 5 API and is provided by JUnit 5. It can be used to ensure that all tests are executed properly, and to perform any necessary clean up after each test.
