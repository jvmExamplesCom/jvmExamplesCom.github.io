Java.net.Inet6Address

The Java.net.Inet6Address class provides an implementation for Internet Protocol version 6 (IPv6), as described in RFC 2373. This class represents an IP address with a 128 bit address, in the standard IPv6 textual representation format.

Creating an Inet6Address Object

To create an Inet6Address object, use the getByName() or getAllByName() methods of the Inet6Address class. The syntax is:


          
           
Inet6Address address = (Inet6Address) Inet6Address.getByName( hostname );
Inet6Address[] addresses = (Inet6Address[]) Inet6Address.getAllByName( hostname );
Where 
hostname
is a string representing the host name (e.g. "www.example.com").

Checking Whether an Address is an IPv6 Address

To check whether an address is an IPv6 address, use the isIPv4Mapped() and isIPv4Compatible() methods of the Inet6Address class. The syntax is:


          
           
boolean isIPv4Mapped = address.isIPv4Mapped();
boolean isIPv4Compatible = address.isIPv4Compatible();
Where 
address
is a valid Inet6Address object.

Getting the IPv6 Address of an Inet6Address Instance

To get the address of an Inet6Address object, use the getAddress() method. The syntax is:


          
           
byte[] address = address.getAddress();

Getting the Host Name of an Inet6Address Instance

To get the host name of an Inet6Address object, use the getHostName() method. The syntax is:
