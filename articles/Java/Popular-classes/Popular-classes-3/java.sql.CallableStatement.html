java.sql.CallableStatement

The java.sql.CallableStatement class is a type of PreparedStatement used for executing SQL stored procedures. This interface provides methods for setting the IN and OUT parameters of the stored procedure and for executing the stored procedure which can return either an update count or a ResultSet object.
