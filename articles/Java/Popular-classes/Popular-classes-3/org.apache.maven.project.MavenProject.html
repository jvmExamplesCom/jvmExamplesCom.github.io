Introduction to Apache Maven Project

Apache Maven is a powerful build automation tool. It helps developers create Java applications faster and better than ever before. Maven's main purpose is to provide an easy, clear way to handle dependencies, configuration and structure for projects.

It is possible to modify the contents of the MavenProject after it has been built. To do this, you'll use one of the available methods on the project object. For example, to add a dependency to the project, you would use the addDependency method.
