What is java.util.NavigableMap?

java.util.NavigableMap is an interface that extends the SortedMap interface, providing the possibility to access elements by navigational operations like lower and higher entry and floor and ceiling entry.

Example of NavigableMap in Java

Here is an example of a NavigableMap in Java: 

          
           
NavigableMap<String, String> navigableMap = new TreeMap<>(); 
 
navigableMap.put("A", "Apple"); 
navigableMap.put("B", "Banana"); 
navigableMap.put("C", "Cat"); 
navigableMap.put("D", "Dog"); 
navigableMap.put("E", "Elephant"); 
 
System.out.println("The original navigable map : " + navigableMap); 
 
System.out.println("The lower entry for D : " + 
                    navigableMap.lowerEntry("D")); 
 
System.out.println("The higher entry for B : " + 
                    navigableMap.higherEntry("B")); 
 
System.out.println("The floor entry for E : " + 
                    navigableMap.floorEntry("E")); 
 
System.out.println("The ceiling entry for A : " + 
                    navigableMap.ceilingKey("A"));
This will produce the following output: 

          
           
The original navigable map : {A=Apple, B=Banana, C=Cat, D=Dog, E=Elephant}
The lower entry for D : C=Cat
The higher entry for B : C=Cat
The floor entry for E : E=Elephant
The ceiling entry for A : Apple

Methods of NavigableMap

NavigableMap provides you the several methods that you can use to navigate through the elements in the map.

  • ceilingEntry(K key)
    - returns the least element which is greater than or equal to the given key.
  • ceilingKey(K key)
    - returns the least key which is greater than or equal to the given key.
  • firstEntry()
    - returns the first (lowest) entry.
  • firstKey()
    - returns the first (lowest) key.
  • floorEntry(K key)
    - returns the greatest element which is lesser than or equal to the given key.
  • floorKey(K key)
    - returns the greatest key which is lesser than or equal to the given key.
  • higherEntry(K key)
    - returns the least element which is greater than the given key.
  • higherKey(K key)
    - returns the least key which is greater than the given key.
  • lastEntry()
    - returns the last (highest) entry.
  • lastKey()
    - returns the last (highest) key.
  • lowerEntry(K key)
    - returns the greatest element which is lesser than the given key.
  • lowerKey(K key)
    - returns the greatest key which is lesser than the given key.
  • pollFirstEntry()
    - removes and returns the least element.
  • pollLastEntry()
    - removes and returns the greatest element.

Conclusion

NavigableMap is a great addition to the Java collections API as it allows you to easily perform navigation operations on maps. It is important to remember that all methods that traverse the map return the nearest element or key to the one that was specified. All methods are also slightly different than those provided by the Map interface.
