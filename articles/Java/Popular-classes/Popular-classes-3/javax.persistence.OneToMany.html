javax.persistence.OneToMany

The javax.persistence.OneToMany is an annotation from the Java Persistence API (JPA). It defines a one-to-many relationship between two entities, and is used to express that one entity has ownership over the other. For example, a customer has many orders, so Customer is the “one” side and Order is the “many” side of the relationship.

Example

Consider the following example, which models a Customer entity, with a OneToMany association to Order:


          
           
@Entity
public class Customer {

    @Id
    @GeneratedValue
    private long id;

    private String name;

    @OneToMany(mappedBy="customer")
    private List<Order> orders;

}

The mappedBy attribute specifies that the Customer entity owns the relationship with Order. The associated Order entity looks like this:


          
           
@Entity
public class Order {

    @Id
    @GeneratedValue
    private long id;

    private BigDecimal totalAmount;

    @ManyToOne
    private Customer customer;

}

The ManyToOne attribute in the Order entity tells JPA that the Order entity is part of a one-to-many relationship with Customer.

CascadeType Attribute

The CascadeType attribute can be specified as part of the @OneToMany annotation on the “one” side of the relationship. It specifies what operations will be cascaded from the “many” side to the “one” side. For example, if CascadeType.ALL is specified, then any changes made to the “many” side will be automatically synchronized with the “one” side. 


          
           
@OneToMany(mappedBy="customer", cascade=CascadeType.ALL)
private List<Order> orders;

FetchType Attribute

The FetchType attribute can also be specified as part of the @OneToMany annotation. It specifies when data from the “many” side of the relationship should be loaded. By default, the data is fetched lazily (that is, when it is actually needed). This can be overridden by specifying FetchType.EAGER, which will cause the data to be loaded in one go. This can be useful in certain scenarios, such as when displaying a list of related entities on a web page. 


          
           
@OneToMany(mappedBy="customer", fetch=FetchType.EAGER)
private List<Order> orders;
