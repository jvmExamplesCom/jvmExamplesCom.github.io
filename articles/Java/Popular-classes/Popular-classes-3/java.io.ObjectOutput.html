What is Serialization?

Serialization is the process of converting an object’s state into a byte stream. This byte stream can then be used for transmission or storage of the object. In order to serialize an object, it must first be converted into a byte stream. This byte stream consists of information about the object, including its type, data, and any other attributes. Once an object’s byte stream has been created, it can then be written to a file or transmitted across a network. Java.io.ObjectOutput offers a straightforward way to both write and read an object to and from a file in a serialized fashion.
