Using org.apache.http.client.methods.HttpPut class

The org.apache.http.client.methods.HttpPut class is a part of the Apache HttpComponents library and provides support for sending PUT requests to the server. This class can be used in conjunction with other classes within the library to create powerful applications that can interact with web services or APIs.

HttpPut httpput = new HttpPut("https://www.googleapis.com/api/data");
