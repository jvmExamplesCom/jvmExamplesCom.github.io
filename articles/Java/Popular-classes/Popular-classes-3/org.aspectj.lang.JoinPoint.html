What is Java org.aspectj.lang.JoinPoint?

Java org.aspectj.lang.JoinPoint is a type of pointcut expression in AspectJ used to denote a particular join point, i.e., a point in code where an aspect can be applied. It allows us to capture execution points at the time of executing a program. For example, when a method is called, when an exception is thrown, when a constructor is executed, when an object is initialized and so on.

public class MyClass { 
  public void myMethod() { 
    System.out.println("This is myMethod()"); 
  } 
}

myMethod()

@Pointcut("execution(* MyClass.myMethod())") 
public void myMethodPointCut() {}

@Before("myMethodPointCut()") 
public void beforeMyMethodAdvice() { 
  System.out.println("Before myMethod() is called"); 
}

myClass.myMethod(); // beforeMyMethodAdvice() will be called first
