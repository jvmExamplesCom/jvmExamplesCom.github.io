Introduction to org.apache.http.impl.client.BasicCredentialsProvider

org.apache.http.impl.client.BasicCredentialsProvider is a class from the Apache HttpComponents project that was designed to provide credentials for an HTTP request. It can be used in combination with a HttpClient instance to make authorization and authentication requests. This class provides an easy way to manage credentials for multiple domains.

Using BasicCredentialsProvider

The example below shows how to create an instance of BasicCredentialsProvider and set it up with a username and password for an HTTP request.


          
           
String userName = "User";
String password = "Password";

CredentialsProvider credentialsProvider = new BasicCredentialsProvider();
credentialsProvider.setCredentials(AuthScope.ANY,
    new UsernamePasswordCredentials(userName , password));

Once the credentials provider has been created, it can be set on an HttpClient instance. The following example shows how to set the credentials provider on an instance of HttpClientBuilder:


          
           
HttpClientBuilder clientBuilder = HttpClientBuilder.create();
clientBuilder.setDefaultCredentialsProvider(credentialsProvider);

HttpClient httpClient = clientBuilder.build();

Once the credentials provider has been set, the httpClient instance can be used to make requests that require authentication.

String userName = "User";
