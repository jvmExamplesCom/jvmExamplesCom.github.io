android.graphics.LinearGradient

android.graphics.LinearGradient is a class used create a linear gradient between two or more colors in an Android application. It is part of the android.graphics package, which provides classes for drawing shapes, lines, and other graphics onto the canvas of an Android device.

How to Use

Using the LinearGradient class is easy. The general syntax is as follows: 

          
           
LinearGradient(float x0, float y0, float x1, float y1, int[] colors, float[] positions, TileMode tile)
Where x0 and y0 are the coordinates of the start point of the gradient line, and x1 and y1 are the coordinates of the end point of the gradient line. The colors parameter is an int array containing the colors (in order) to use for the gradient, and the positions parameter is a float array that defines where each color should begin and end in the gradient. The tile parameter is an enum that defines how the edges of the gradient should be repeated.

Code Examples

To create a simple two-color gradient from red to yellow, you would use the following code: 

          
           
LinearGradient gradient = new LinearGradient(0f, 0f, 1f, 1f, 
    new int[] {Color.RED, Color.YELLOW}, null,
    TileMode.CLAMP);
To create a more complex gradient with three colors, you could do the following: 

          
           
LinearGradient gradient = new LinearGradient(0f, 0f, 1f, 1f, 
    new int[] {Color.RED, Color.GREEN, Color.BLUE}, 
    new float[] {0f, 0.3f, 1f},
    TileMode.REPEAT);
In this example, the green color will only appear from 0.3 to 1 of the gradient, and the edges of the gradient will be repeated (tiled). The TileMode can be set to CLAMP or MIRROR for different effects.

Conclusion

android.graphics.LinearGradient is a powerful class for creating color gradients in an Android application. With the ability to specify multiple colors and the range for each color, you can create dynamic and beautiful gradients with ease.
