Java.awt.datatransfer.StringSelection

Java.awt.datatransfer.StringSelection is a class in Java which allows developers to transfer a string of characters from one java program to another, by adding it to the system clipboard. It is implemented in the Abstract Windowing Toolkit, or AWT, and provides a simple way of using the clipboard for inter-program communication.

Usage

StringSelection ss = new StringSelection("Hello World!");
Once the StringSelection has been created, it can be added to the system clipboard by calling the Toolkit.getDefaultToolkit().getSystemClipboard().setContents(StringSelection, null) method, like so: 

          
           
Toolkit.getDefaultToolkit().getSystemClipboard().setContents(ss, null);

Object o = Toolkit.getDefaultToolkit().getSystemClipboard().getContents(null);
The returned Object can then be cast to a StringSelection, which allows us to access the data stored inside, like so: 

          
           
StringSelection ss = (StringSelection) o;
