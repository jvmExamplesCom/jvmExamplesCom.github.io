Java.awt.datatransfer.StringSelection

Java.awt.datatransfer.StringSelection is a class in Java which allows developers to transfer a string of characters from one java program to another, by adding it to the system clipboard. Object o = Toolkit.getDefaultToolkit().getSystemClipboard().getContents(null);
