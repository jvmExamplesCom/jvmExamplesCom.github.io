android.util.SparseIntArray

The 

          
           
android.util.SparseIntArray
class is an efficient way to store and retrieve data in the form of ints on Android devices. Unlike traditional hashmaps, a 
SparseIntArray
does not store any other types such as Objects, making it ideal for use cases where only integers are needed. This article will discuss how to use the 
SparseIntArray
, with several code examples included.

Creating a SparseIntArray

Creating a 
SparseIntArray
is a simple process. All one must do is create a new empty instance: 

          
           
SparseIntArray myArray = new SparseIntArray();

Putting and Getting Values

One can then insert values into the array using the 


          
           
put(int key, int value)
method. This method takes two integer parameters: a key and a value. Both values must be ints, and when the same key is put twice, the newer value will overwrite the older value. For example: 

          
           
myArray.put(1, 10);
myArray.put(2, 20);
myArray.put(3, 30);
myArray.put(1, 40);

This adds the values 10, 20, and 30 with their respective keys 1, 2, 3. When we put 1 with the value 40, this overwrites the previous value, 10. We can then view values stored in the array using the 

get(int key)
method. This method takes an integer parameter, the key, and returns the corresponding value. For example: 

          
           
int x = myArray.get(1);
x // 40

Here the value associated with key 1 is retrieved and stored in the variable x. We can also determine if a given key exists in the 

SparseIntArray
using the 

          
           
containsKey(int key)
method. This method takes an integer parameter, the key, and returns a boolean. For example: 

          
           
boolean exists = myArray.containsKey(3);
exists // true

This statement checks if the key 3 exists in the array, which it does. Thus, 

exists
is set to true.

Size and Clear

One can retrieve the size of the array with the 
size()
method, which returns an integer representing the number of entries in the 
SparseIntArray
. Additionally, we can clear the entire array using the 
clear()
method. For example: 

          
           
int size = myArray.size();
size // 3

myArray.clear();

int newSize = myArray.size();
newSize // 0

Iterating Through Entries

Another useful method provided by the 
SparseIntArray
class is 
keyAt(int index)
. This method takes an int parameter, the index of the desired key to retrieve, and returns an int, the key at the specified index. The entries in the array are ordered, starting from the lowest index ranging up. For example, if we created the 
SparseIntArray
above and added another entry: 

          
           
myArray.put(4, 50);

This would add the value 50 with the key 4, and the entries stored in the array would be ordered as follows:


          
           
[1, 40]
[2, 20]
[3, 30]
[4, 50]
With this information in mind, we can now use the 
keyAt()
method to iterate through each entry in the array. For example: 

          
           
for (int i = 0; i < myArray.size(); i++) {
    int key = myArray.keyAt(i);
    int value = myArray.get(key);

    System.out.println("Key: " + key + " , Value: " + value);
}

This code iterates through each entry in the array, retrieves the key and value associated with each entry, and prints them out. The resulting output would be:


          
           
Key: 1 , Value: 40
Key: 2 , Value: 20
Key: 3 , Value: 30
Key: 4 , Value: 50
