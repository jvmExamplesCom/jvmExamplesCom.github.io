javax.ws.rs.client.Client

javax.ws.rs.client.Client
class provides a Java-based mechanism for processing RESTful (REpresentational State Transfer) web services. It provides methods for making requests to endpoints and for handling responses. Ambit

Client client = new Client();

WebTarget
