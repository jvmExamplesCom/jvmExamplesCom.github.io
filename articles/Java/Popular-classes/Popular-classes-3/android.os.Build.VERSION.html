android.os.Build.VERSION

The 

          
           
android.os.Build.VERSION
class contains information about the current Android framework version. This class contains several constants for the version codes of Android, including SDK_INT and RELEASE. This class can be used to find out the version of Android running on your device.

Example Usage

The following code will print out the version of Android running on your device:


          
           
Log.i("Version","Android Version: "+android.os.Build.VERSION.RELEASE); 
Log.i("SDK","Android SDK: "+android.os.Build.VERSION.SDK_INT);

The output of the above code should look something like this: 


          
           
I/Version: Android Version: 8.0.0
I/SDK: Android SDK: 26

You can also check for specific versions of Android by comparing the SDK_INT constant to a specific version code. For example, you could check if the current device is running Android 4.4 or higher with the following code: 


          
           
int currentApiVersion = android.os.Build.VERSION.SDK_INT;

if (currentApiVersion >= android.os.Build.VERSION_CODES.KITKAT) {
    // Do something for KitKat and above versions
} else {
    // Do something for lower than KitKat versions
}

You can check out the full list of Android version codes in the Android documentation here: Android Build.VERSION_CODES.
