javax.swing.KeyStroke

javax.swing.KeyStroke is a class used in Swing applications for accessing keyboard shortcuts. It provides a way to access platform-independent keystrokes. Keystrokes can be easily mapped to actions. This class is the basis for input maps and action maps. A KeyStroke object encapsulates a particular keystroke, consisting of a combination of a specific key and one or more modifiers, such as SHIFT, CTRL, or ALT.

Example

The following example creates a KeyStroke object for the key P and the control (CTRL) modifier: 


          
           
KeyStroke keyStroke = KeyStroke.getKeyStroke(KeyEvent.VK_P, InputEvent.CTRL_MASK);

This KeyStroke object can then be used to create an input map that maps this keystroke to an action:


          
           
InputMap inputMap = new InputMap();
inputMap.put(keyStroke, "actionName");

This input map can then be used to create an action map which is used by the application to handle the action when the keystroke is pressed:


          
           
ActionMap actionMap = new ActionMap();
actionMap.put("actionName", myAction);

When the key is pressed, the Action's actionPerformed method is called with the action event. The Action can then perform the desired action.
