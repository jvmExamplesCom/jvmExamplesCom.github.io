javax.net.ssl.SSLSocket

The javax.net.ssl.SSLSocket is a java.net API class used in the SSL/TLS handshake protocol when establishing a secure connection, as per the Transport Layer Security (TLS) and Secure Socket Layer (SSL) protocols. In its simplest form all data exchanged between a client and a server is transmitted through a single socket. The SSLSocket class encapsulates this socket, adding an additional layer of security and encryption to ensure that all data exchanged is secure and encrypted. The following sections will provide a detailed overview of the SSLSocket class, including its various methods and attributes.

Creating an SSLSocket Instance

Creating an instance of the SSLSocket class requires two parameters - the host address and port number to connect to. For example:


          
           
String host = "127.0.0.1";
int port = 2000;
SSLSocket socket = new SSLSocket(host, port);

Establishing a Secure Connection

Once an SSLSocket instance has been created, the connection must be established in order to send and receive data. This can be done by calling the 

startHandshake()
method, e.g.: 

          
           
socket.startHandshake();
The 
startHandshake()
method performs a series of steps to establish a secure connection between the client and the server, such as verifying the authenticity of the server's credentials. If any of these steps fail, a 
SSLException
is thrown.

Sending and Receiving Data

Once a secure connection has been established, data can be sent and received using the same methods as with a plain socket. To send data, you can use the 

write()
and 
flush()
methods. For example: 

          
           
byte[] data = "Hello World!".getBytes();
socket.getOutputStream().write(data);
socket.getOutputStream().flush();
To receive data, you can use the 
read()
method. For example: 

          
           
byte[] data = new byte[1024];
int numBytesRead = socket.getInputStream().read(data);

Closing the Connection

Once all data has been sent and/or received, it is important to close the connection with the 

close()
method. For example: 

          
           
socket.close();
