Introduction To android.graphics.drawable.ShapeDrawable

The android.graphics.drawable.ShapeDrawable class is a part of the Android Library that is used to create customizable shapes and graphics. It can be used to draw shapes such as circles, rectangles, ovals, and even more complicated ones such as stars or polygons. The ShapeDrawable class also allows for setting various properties to customize each shape, such as color, shadow, gradient, or padding. This class is an efficient way to draw custom shapes and graphics on the Android environment.

Creating an android.graphics.drawable.ShapeDrawable Object

Creating a ShapeDrawable object is simple. First, a shape must be declared as either a RectShape or an OvalShape. Below is an example of a ShapeDrawable object being declared: 

          
           
ShapeDrawable shapeDrawable = new ShapeDrawable(new RectShape());
