org.springframework.scheduling.annotation.Async

org.springframework.scheduling.annotation.Async is a convenience annotation used to automatically apply the necessary Spring configuration beans to enable asynchronous processing of @Async annotated methods within a Spring application. This annotation can be applied to void methods as well as methods that return a value.
