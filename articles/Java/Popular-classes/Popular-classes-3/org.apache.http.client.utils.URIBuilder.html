org.apache.http.client.utils.URIBuilder

The org.apache.http.client.utils.URIBuilder class is a utility class for constructing URIs in a safe manner, providing advanced features such as encoding and automatic addition of query parameters.

Usage

URIBuilder uriBuilder = new URIBuilder(baseUri);

// Create a UriBuilder from a complete Uri string
String uriString = "https://example.com/?param1=value1";
URIBuilder uriBuilder = new URIBuilder(uriString);
