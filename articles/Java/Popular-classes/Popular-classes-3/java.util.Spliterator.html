The Java Spliterator is an interface from the java.util package that is used to visit elements of a source in a bidirectional manner. Using the Spliterator Interface

Object[] dataSource = {1, 2, 3, 4, 5};
Spliterator<Object> spliterator = Arrays.spliterator(dataSource);
