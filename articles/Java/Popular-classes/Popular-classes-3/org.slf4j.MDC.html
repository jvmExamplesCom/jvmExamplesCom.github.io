What is MDC?

The Mapped Diagnostic Context (MDC) is a tool used in the logging process of many programming languages. It allows developers to add more context to their log entries so that it can be better analyzed during debugging or monitoring. As part of SLF4J, MDCs are available for many languages including Java. By adding this extra layer of information to log entries, it becomes much easier to identify and debug problems.
