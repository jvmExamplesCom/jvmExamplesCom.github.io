javax.swing.JMenuBar

The javax.swing.JMenuBar class is a component that implements a menu bar which lets a user access different functions and options within an application. It is typically used in conjunction with a JFrame, but can be used independent of one if desired. The JMenuBar class is part of the Java Swing library and is used to create menus and submenus.

Creating a JMenuBar

Creating a JMenuBar is very straightforward and easy. The first step is to create an instance of the JMenuBar class: 

          
           
JMenuBar menuBar = new JMenuBar();
Next, you can add menus to the JMenuBar by calling the 

          
           
add()
method. Each menu added will appear as a drop-down list when the user clicks on it. For example, to add a "File" and "Edit" menu to the JMenuBar, you could call the 

          
           
add()
method twice, like this: 

          
           
menuBar.add(new JMenu("File"));
menuBar.add(new JMenu("Edit"));

Adding Menus and Items

Now that we have a JMenuBar created with two menus, we can use the 

          
           
getMenu()
method to get a reference to a particular menu and add items to it. In order to add items to a menu, we must first create a JMenuItem object: 

          
           
JMenuItem item = new JMenuItem("Save");
Then we can add the item to the desired menu like this: 

          
           
menuBar.getMenu(0).add(item);
This adds the "Save" menu item to the "File" menu. We can also create submenus and add them to the menu. For example, to add a "New" submenu to the "File" menu, you could code something like this: 

          
           
JMenu newMenu = new JMenu("New");
menuBar.getMenu(0).add(newMenu);

Adding Action Listeners

The final step is to add action listeners to the JMenuItems so that something happens when a user clicks on them. To do this, we must first register an object as a listener with the JMenuItem: 

          
           
item.addActionListener(listenerObject);
Then, the listener object must implement the 

          
           
ActionListener
interface and provide an 

          
           
actionPerformed()
method. This method will be called whenever the user clicks on the JMenuItem. The following code shows how this could be done for the "Save" item we created earlier: 

          
           
public class MyListener implements ActionListener {
   public void actionPerformed(ActionEvent e) { 
      // do something... 
   }
}

MyListener listenerObject = new MyListener();
item.addActionListener(listenerObject);
