java.lang.String

The Java language class 
java.lang.String
is the most widely used class when dealing with Strings in Java programming. It provides various methods to manipulate Strings, and is one of the most popular classes due to its ease-of-use and powerful features.

Basics

A String object is a sequence of characters that can be manipulated and stored within a program. Strings are immutable objects, meaning that the internal contents cannot be changed once the object is created. All operations performed on a String object will return a new String object with the result of the operation.

Creating a String object is simple: 

          
           
String str = "Hello, World!";

When accessing a String's contents, the charAt() method can be used; for example: 


          
           
char ch = str.charAt(0); // ch = 'H'

This retrieves the character at the specified index in the String. When looping through the characters in a String, the length() method can be used to get its length - which should be done prior to using charAt(), as an exception will be thrown if an incorrect index is passed in.


          
           
int len = str.length();
for (int i = 0; i < len; i++) {
  char ch = str.charAt(i);
  System.out.println("Character at index " + i + " is " + ch);
}

This loop will print out each character from the String.

Comparing Strings

Comparing strings is also easy with Java's String class. The equals() method can be used to compare two Strings by their value, returning 

true
if they are equal, or 
false
if not. 

          
           
String str1 = "abc";
String str2 = "abc";
boolean isEqual = str1.equals(str2); // isEqual = true

The compareTo() method can be used to compare two Strings lexicographically, returning a negative number if str1 comes before str2, 0 if they are equal, and a positive number if str2 comes after str1. 


          
           
String str1 = "abc";
String str2 = "abcd";
int comparisonResult = str1.compareTo(str2); // comparisonResult = -1

Manipulating Strings

The String class provides various methods for manipulating Strings, such as concatenation, trimming, and searching.

To combine two Strings, the concat() method can be used; for example:


          
           
String str1 = "hello";
String str2 = "world";
String combinedStr = str1.concat(str2); // combinedStr = "helloworld"

The trim() method can be used to remove leading and trailing whitespace from a String:


          
           
String str = "  hello  ";
String trimmedStr = str.trim(); // trimmedStr = "hello"

Finally, the indexOf() method can be used to search for a substring within a String, returning the position where it was first found (or -1 if it wasn't found):


          
           
String str = "Hello World";
int index = str.indexOf("World"); // index = 6

Conclusion

The Java language class 
java.lang.String
is an incredibly useful and popular class for working with Strings in Java. It provides basic methods for creating, comparing, and manipulating Strings, while also being relatively simple to use.
