android.support.v4.content.LocalBroadcastManager

Android's LocalBroadcastManager is a utility class for sending and receiving broadcasts for local components.

Installation

As part of the Android support library, the class is available as a Maven dependency and can be included in the project with:


          
           
dependencies {
  implementation "androidx.localbroadcastmanager:localbroadcastmanager:1.0.0"
}

Creating an Instance

An instance of LocalBroadcastManager can be created by calling its static getInstance method, which expects a Context as an argument.


          
           
LocalBroadcastManager localBroadcastManager = LocalBroadcastManager.getInstance(context);

Sending Broadcasts

A broadcast can be sent using the sendBroadcast method, which requires an Intent as an argument. The Intent should contain information about the action and any extra data that should be sent to the receiver.


          
           
Intent intent = new Intent("my.action");
intent.putExtra("some_data", "my data");
localBroadcastManager.sendBroadcast(intent);

Receiving Broadcasts

In order to receive broadcasts, an instance of BroadcastReceiver needs to be registered with LocalBroadcastManager using the registerReceiver method. This should be done during initialization and can optionally include a filter to specify which Intents the receiver should handle.


          
           
BroadcastReceiver receiver = new BroadcastReceiver() {
  @Override
  public void onReceive(Context context, Intent intent) {
    // Handle broadcast
  }
};
localBroadcastManager.registerReceiver(receiver, new IntentFilter("my.action"));
