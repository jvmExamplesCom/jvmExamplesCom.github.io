javax.xml.bind.Marshaller is a key element of the Java Architecture for XML Binding (JAXB). It is an API used for marshalling and unmarshalling Java objects to and from XML documents. Marshaller marshaller = jaxbContext.createMarshaller();

marshaller.marshal(obj, System.out); // marshalling to System.out
