javax.xml.bind.Marshaller

javax.xml.bind.Marshaller is a key element of the Java Architecture for XML Binding (JAXB). It is an API used for marshalling and unmarshalling Java objects to and from XML documents. In other words, JAXB provides a mechanism both for serializing and deserializing Java objects into XML documents, and vice versa. The Marshaller is a specific implementation of this process, which takes a JAXB-annotated Java object instance and generates the corresponding XML document, or “marshals” it.

JAXBContext jaxbContext = JAXBContext.newInstance();
Marshaller marshaller = jaxbContext.createMarshaller();

Marshalling

MyObject obj = ...;
marshaller.marshal(obj, System.out); // marshalling to System.out
