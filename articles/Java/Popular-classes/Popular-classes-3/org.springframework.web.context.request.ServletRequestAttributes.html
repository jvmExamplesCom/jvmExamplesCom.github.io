What is org.springframework.web.context.request.ServletRequestAttributes?

org.springframework.web.context.request.ServletRequestAttributes is a class included in Spring framework. It contains methods to store, retrieve and manipulate servlet request attributes. It serves as a bridge between the HttpServletRequest and the application context outside of a web application.

Basic Usage

To use the org.springframework.web.context.request.ServletRequestAttributes class, one must first have a servlet request object available. This can be done by calling the static method getCurrentRequest() from the WebUtils class.


          
           
HttpServletRequest request = WebUtils.getCurrentRequest();

Once we have the servlet request object, we can use it to create an instance of ServletRequestAttributes:


          
           
ServletRequestAttributes servletRequestAttributes = new ServletRequestAttributes(request);

Methods

ServletRequestAttributes has several methods that are useful for manipulating servlet request attributes. These methods include:

  • setAttribute(String name, Object value): This method sets an attribute to the given name and value on the servlet request.
  • getAttribute(String name): This method gets an attribute from the servlet request by name.
  • removeAttribute(String name): This method removes an attribute from the servlet request by name.
  • getAttributeNames(): This method returns an Enumeration of all attributes names on the servlet request.
  • hasAttribute(String name): This method checks if an attribute exists on the servlet request by name.

Example

We can use the above methods to create a simple example of setting and retrieving an attribute from a servlet request.


          
           
// Create servlet request
HttpServletRequest request = WebUtils.getCurrentRequest();

// Create ServletRequestAttributes
ServletRequestAttributes servletRequestAttributes = new ServletRequestAttributes(request);

// Set attribute on servlet request
servletRequestAttributes.setAttribute("myAttribute", "My Attribute Value");

// Get attribute from servlet request
Object myAttributeValue = servletRequestAttributes.getAttribute("myAttribute");

System.out.println("My Attribute Value: " + myAttributeValue);

The output of the code above would look like this:


          
           
My Attribute Value: My Attribute Value
