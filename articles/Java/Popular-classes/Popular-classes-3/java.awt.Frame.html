java.awt.Frame

java.awt.Frame is a component of the Java GUI (Graphical User Interface) toolkit, which is part of the Java Foundation Classes. It provides a basic window with a title bar and borders, allowing applications to be displayed as windows on the screen.

Creating A Frame

The following example shows how to create a simple frame with a title:


          
           
import java.awt.*;
public class MyFrame extends Frame {
   public MyFrame(String title) {
       super(title);
       setSize(300,200);
       setVisible(true);
   }
   public static void main(String[] args) {
       new MyFrame("My Frame Example");
   }
}

frame.setSize(400, 300);

You can also set the frame's location by specifying its coordinates on the screen. For example, the following code will set the frame's location to the upper left corner of the screen:


          
           
frame.setLocation(0, 0);

frame.setVisible(true);

frame.setBackground(Color.white);
