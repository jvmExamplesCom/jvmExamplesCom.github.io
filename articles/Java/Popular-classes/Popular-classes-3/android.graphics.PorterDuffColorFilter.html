android.graphics.PorterDuffColorFilter

The 

          
           
PorterDuffColorFilter
class is used to apply color filters to Drawable objects in the Android framework. A color filter is a way to modify the colors of a Drawable's content, either by changing their hue, saturation, brightness, or changing the alpha value. The PorterDuffColorFilter class provides an easy way to apply color filters to a Drawable.

Creating a PorterDuffColorFilter

The PorterDuffColorFilter class provides the following constructors:


          
           
PorterDuffColorFilter(int color, PorterDuff.Mode mode)  
PorterDuffColorFilter(ColorStateList colorStateList, PorterDuff.Mode mode)  
PorterDuffColorFilter(int color, PorterDuff.Mode mode, PorterDuffColorFilter source)

The first constructor takes an integer representing the color to use for the color filter and a PorterDuff.Mode enum value that specifies the blending mode to use when applying the color filter. This blending mode determines how the original colors of the Drawable are mixed with the new specified color.

The second constructor takes a ColorStateList object, which can store multiple colors for different states such as "pressed" or "focused," and a PorterDuff.Mode enum value. This constructor allows the color filter to automatically use a different color depending on the current state of the Drawable.

The third constructor is similar to the first one but also takes a PorterDuffColorFilter object as an argument. This allows the color filter to be composed of multiple color filters.

Applying a PorterDuffColorFilter

Once a PorterDuffColorFilter has been created, it can be applied to a Drawable via the Drawable's setColorFilter() method. This method takes a PorterDuffColorFilter as an argument. For example:


          
           
Drawable drawable = getResources().getDrawable(R.drawable.my_icon);
PorterDuffColorFilter filter = new PorterDuffColorFilter(
  Color.RED, PorterDuff.Mode.SRC_ATOP);
drawable.setColorFilter(filter);

This code creates a PorterDuffColorFilter that applies the color red using the SRC_ATOP blending mode, and then applies it to a Drawable object.

Conclusion

The PorterDuffColorFilter class provides an easy way to apply color filters to Drawable objects in the Android framework. It can be used to modify the hue, saturation, brightness, alpha, or create composites of multiple color filters. The PorterDuffColorFilter class provides constructors for creating it, and the Drawable class provides a setColorFilter() method for applying it.
