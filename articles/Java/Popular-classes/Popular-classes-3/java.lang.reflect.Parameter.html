java.lang.reflect.Parameter

Java.lang.reflect.Parameter is a class provided in the Java Reflection API which represents an individual parameter of a method or constructor. It provides methods to access information about the parameter such as its name, modifiers, annotations and type. This class can be instantiated from the class Method or Constructor.

Creating a Parameter Object

A Parameter object can only be created from a Method or Constructor object. The first step is to obtain a Method object, followed by calling its getParameters() method. This will return an array of parameters which can be used to create a Parameter object.


          
           
Method method = MyClass.class.getDeclaredMethod("myMethod", String.class);
Parameter[] parameters = method.getParameters();
Parameter param = parameters[0];

It is also possible to use the getParameter(String name) method of the Method class to obtain a specific parameter.


          
           
Parameter param = method.getParameter("paramName");

Getting Information about the Parameter

Once a Parameter object has been obtained, information about the parameter can be obtained using the following methods.

  • getName()
    - Returns the name of the parameter
  • getModifiers()
    - Returns a set of modifier flags associated with the parameter
  • getAnnotations()
    - Returns an array of Annotation objects associated with the parameter
  • getType()
    - Returns the Class object representing the type of the parameter

Example

The following example demonstrates how to use the Parameter class to obtain information about a method's parameters.


          
           
public void myMethod(@MyAnnotation String paramName) {
    // Some code...
}

// Get the method
Method method = MyClass.class.getDeclaredMethod("myMethod", String.class);

// Get the parameters
Parameter[] parameters = method.getParameters();

// Iterate over each parameter
for (Parameter parameter : parameters) {

    // Print parameter name
    System.out.println("Parameter Name: " + parameter.getName());

    // Print modifiers
    System.out.println("Modifiers: " + parameter.getModifiers());

    // Print annotations
    Annotation[] annotations = parameter.getAnnotations();
    for (Annotation annotation : annotations) {
        System.out.println("Annotation: " + annotation);
    }

    // Print type
    System.out.println("Type: " + parameter.getType());
}

Parameter Name: paramName
Modifiers: 0
Annotation: @MyAnnotation
Type: class java.lang.String
