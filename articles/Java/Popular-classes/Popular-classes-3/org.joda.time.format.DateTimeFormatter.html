Overview of org.joda.time.format.DateTimeFormatter

org.joda.time.format.DateTimeFormatter

Using DateTimeFormatter for Formatting and Parsing Dates

The DateTimeFormatter class supports both formatting and parsing of date-time strings according to the ISO8601 standard. The following code example illustrates how to create a DateTimeFormatter instance and use it to format a java.util.Date object as a date-time string. 


          
           
DateTimeFormatter formatter = DateTimeFormatter.ISO_LOCAL_DATE_TIME;
LocalDateTime now = LocalDateTime.now();
String formattedDateTime = now.format(formatter);
System.out.println(formattedDateTime);
The above example produces the following output: 

          
           
2019-11-01T11:47:50.322

The same DateTimeFormatter instance can be used to parse the same string back into a LocalDateTime instance, as demonstrated in the following code example. 


          
           
LocalDateTime parsedDateTime 
  = formatter.parse(formattedDateTime, LocalDateTime::from); 
System.out.println(parsedDateTime);
This produces the following output: 

          
           
2019-11-01T11:47:50.322

DateTimeFormatter formatter
  = DateTimeFormatter.ofPattern("dd/MM/yyyy hh:mm a");

The formatter can then be used to format and parse date-time strings in this custom format.
