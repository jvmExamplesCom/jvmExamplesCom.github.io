javax.xml.stream.XMLStreamReader

The javax.xml.stream.XMLStreamReader is an interface used to read XML documents written in the StAX (Streaming API for XML) format. This interface enables a developer to parse and navigate an XML document while processing only small chunks of the document at a time, allowing for efficient memory and CPU usage. The XMLStreamReader is used to both parse and validate XML content and can be used with any XML 1.0 or 1.1 document.

Implementations

The XMLStreamReader interface is implemented by several classes available in the java.xml.stream package, including the following:

  • StAX Parser: This implementation provides basic parsing support for well-formed XML documents.
  • XMLInputFactory: This class provides factory methods for creating XMLStreamReader instances.
  • XMLEventReader: This class provides a cursor-style API for reading XML documents using XMLEvents.
  • XMLStreamWriter: This class provides a writer-style API for writing XML documents using XMLStreamWriter.

XMLStreamReader reader = inputFactory.createXMLStreamReader(new FileInputStream("data.xml"));

while (reader.hasNext()) {
  int eventType = reader.next();

  if (eventType == XMLStreamConstants.START_ELEMENT) {
    String localName = reader.getLocalName();

    if (localName.equals("person")) {
      System.out.println("Found a person");
    } else if (localName.equals("name")) {
      System.out.println("Found a name");
      System.out.println("value: " + reader.getElementText());
    }
  }
