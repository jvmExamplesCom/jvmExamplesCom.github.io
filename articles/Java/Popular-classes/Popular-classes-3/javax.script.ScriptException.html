javax.script.ScriptException

The javax.script.ScriptException class is an unchecked exception which signals an error occurred in the underlying script engine. This class is the parent class for all script engine-related exceptions thrown by the Java Scripting API.

Example 1: Simple ScriptEngine Error

The following example shows a simple script engine error, where an invalid operator symbol is used in the script:


          
           
String script = "2+";
ScriptEngineManager mgr = new ScriptEngineManager();
ScriptEngine engine = mgr.getEngineByName("JavaScript");
try {
   engine.eval(script);
} catch (ScriptException e) {
   System.out.println("Error: "+ e.getMessage());
}
// Output: Error: sun.org.mozilla.javascript.internal.EvaluatorException: missing ) after argument list

Example 2: Compilation Errors

The following example shows how a compilation errors can be caught using the javax.script.ScriptException:


          
           
String script = "var x = 100; x++;";
ScriptEngineManager mgr = new ScriptEngineManager();
ScriptEngine engine = mgr.getEngineByName("JavaScript");
try {
   engine.compile(script).eval();
} catch (ScriptException e) {
   System.out.println("Error: "+ e.getMessage());
}
// Output: Error: sun.org.mozilla.javascript.internal.EvaluatorException: missing ) after argument list

Example 3: Catching Exceptions Thrown From Scripts

The following example shows how to catch exceptions thrown from scripts:


          
           
String script = "throw new Error('Custom exception');";
ScriptEngineManager mgr = new ScriptEngineManager();
ScriptEngine engine = mgr.getEngineByName("JavaScript");
try {
   engine.eval(script);
} catch (ScriptException e) {
   System.out.println("Error: "+ e.getMessage());
}
// Output: Error: Custom exception

Example 4: Catching Runtime Exceptions From Scripts

The following example shows how to catch runtime exceptions thrown from scripts:


          
           
String script = "var x = function() { throw new TypeError('Something went wrong!'); }; x();";
ScriptEngineManager mgr = new ScriptEngineManager();
ScriptEngine engine = mgr.getEngineByName("JavaScript");
try {
   engine.eval(script);
} catch (ScriptException e) {
   System.out.println("Error: "+ e.getMessage());
}
// Output: Error: sun.org.mozilla.javascript.internal.EcmaError: TypeError: Something went wrong!
