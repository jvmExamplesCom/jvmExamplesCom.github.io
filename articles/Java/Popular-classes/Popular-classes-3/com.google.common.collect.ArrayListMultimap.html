Introduction to Java ArrayListMultimap

The Java ArrayListMultimap is a collection that allows storing multiple values against a key. This is different from a regular Map, since it allows multiple values to be associated with a single key. It can store duplicate keys and values, and requires only the same amount of memory as an ordinary hashmap. It is part of the Google Guava library, making it a very useful and powerful tool for any Java application.

Creating and Using ArrayListMultimap

Creating an ArrayListMultimap is simple. Use the static create() method on the ArrayListMultimap object, which will return a new empty instance. To add key-value pairs, use the put(K,V) method, where K is the key and V is the value: 


          
           
ArrayListMultimap<String, Integer> myMap = ArrayListMultimap.create();
myMap.put("apple", 1);
myMap.put("orange", 2);
myMap.put("apple", 3);

The above code creates an ArrayListMultimap containing two entries: the first is “apple” with the value 1, and the second is “orange” with the value 2. Since the same key (“apple”) was used twice, the second put() call adds the value 3 to the existing entry for “apple”, resulting in a list of values [1, 3] associated with the key.

To retrieve a list of values associated with a particular key, use the get(K) method, which returns a List of values: 


          
           
List<Integer> list = myMap.get("apple");
// list contains [1, 3]

Removing Entries from ArrayListMultimap

Entries in an ArrayListMultimap can be removed in two ways: by removing all values associated with a particular key, or by removing a single value associated with a key.

To remove all values associated with a particular key, use the removeAll(K) method, where K is the key: 


          
           
myMap.removeAll("apple");
//myMap now does not contain "apple" key

To remove a single value associated with a key, use the remove(K, V) method, where K is the key and V is the value:
