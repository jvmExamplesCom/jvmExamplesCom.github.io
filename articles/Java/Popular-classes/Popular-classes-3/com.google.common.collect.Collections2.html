Exploring Google’s com.google.common.collect.Collections2 Class

Google’s 

          
           
com.google.common.collect.Collections2
class provides several methods for creating an immutable collection out of a provided list. The purpose of this article is to explain how the 
Collections2
class can be used in Java code.

Creating an Immutable Collection with Collections2.transform()

The 

          
           
Collections2.transform()
method allows us to create an immutable version of our original collection by applying a transformation function to every element in the collection. This transformation function is defined as a 
Function
object containing a 
apply
method, which takes an object of type 
F
and returns an object of type 
T
. Here is an example of using the 

          
           
Collections2.transform()
method to create an immutable collection: 

          
           
// Create a list of strings
List<String> myList = Arrays.asList("foo", "bar", "baz");

// Create a function to uppercase each element in the list
Function<String, String> toUpperCase = new Function<String, String>() {
    public String apply(String input) {
        return input.toUpperCase();
    }
};

// Create an immutable version of the list
Collection<String> immutableCollection = Collections2.transform(myList, toUpperCase);

// Print the elements of the immutable collection
for (String s : immutableCollection) {
    System.out.println(s);
}

// Output:
// FOO
// BAR
// BAZ
In the example, we create a 
List
and a 

          
           
Function<String, String>
that uppercases each element in the list. We then use the 

          
           
Collections2.transform()
method to create an immutable version of the list by applying the 
toUpperCase
function to each element in the original list.

Creating an Immutable Collection with Collections2.filter()

The 

          
           
Collections2.filter()
method allows us to create an immutable collection by filtering out elements that do not meet a certain criteria. This criteria is defined as a 
Predicate
object containing a 
apply
method, which takes and object of type 
T
and returns 
true
or 
false
based on the criteria. Here is an example of using the 

          
           
Collections2.filter()
method to create an immutable collection: 

          
           
// Create a list of numbers
List<Integer> myList = Arrays.asList(1, 2, 3, 4, 5);

// Create a predicate to filter out odd numbers
Predicate<Integer> isEven = new Predicate<Integer>() {
    public boolean apply(Integer input) {
        return input % 2 == 0;
    }
};

// Create an immutable version of the list
Collection<Integer> immutableCollection = Collections2.filter(myList, isEven);

// Print the elements of the immutable collection
for (Integer i : immutableCollection) {
    System.out.println(i);
}

// Output:
// 2
// 4
In the example, we create a 
List

Collections2.filter()

isEven
