java.awt.event.ItemListener

The java.awt.event.ItemListener interface is used to detect the user's selection of an item from a Choice, Checkbox, List, or Scrollbar component. It defines only one method: 

itemStateChanged
, which is called when an item is selected or deselected by the user. This interface can be added to a component using the component's 
addItemListener
method.

Example

The following example shows the use of 
ItemListener
in an applet. It uses four check boxes, labeled “Option A”, “Option B”, “Option C”, and “Option D”. When a user clicks on each checkbox, it displays a message indicating which checkbox was checked or unchecked. 

          
           
import java.awt.*;
import java.awt.event.*;
import java.applet.*;

public class ItemListenerExample extends Applet implements ItemListener {
   Checkbox cb1, cb2, cb3, cb4;

   public void init() {
      cb1 = new Checkbox("Option A");
      cb2 = new Checkbox("Option B");
      cb3 = new Checkbox("Option C");
      cb4 = new Checkbox("Option D");
      add(cb1);
      add(cb2);
      add(cb3);
      add(cb4);
      cb1.addItemListener(this);
      cb2.addItemListener(this);
      cb3.addItemListener(this);
      cb4.addItemListener(this);
   }

   public void itemStateChanged(ItemEvent ie) {
      repaint();  
   }

   public void paint(Graphics g) {
      String str;
      g.drawString("Current state : ", 10, 50);
      if (cb1.getState()) {
         str = "Option A is Selected.";
         g.drawString(str, 10, 70);
      }
      if (cb2.getState()) {
         str = "Option B is Selected.";
         g.drawString(str, 10, 90);
      }
      if (cb3.getState()) {
         str = "Option C is Selected.";
         g.drawString(str, 10, 110);
      }
      if (cb4.getState()) {
         str = "Option D is Selected.";
         g.drawString(str, 10, 130);
      }
   }
}

The itemStateChanged Method

The 
itemStateChanged
method is called whenever the state of an item is changed by the user. It has the following signature: 

          
           
public void itemStateChanged(ItemEvent e)
It accepts an instance of 
ItemEvent
as its argument and has no return value. The 
ItemEvent
object contains the source of the event and the state of the item (selected or deselected).

Example Usage

In the example code, the 
itemStateChanged
method is used to update the state of each checkbox every time the user changes it. It also calls the 
repaint
method to make sure the current state is updated in the GUI. The 
repaint
method calls the 
paint
method, which then checks the state of each checkbox and displays a message accordingly.
