Introduction to javafx.application.Platform

JavaFX Platform is a platform for creating rich internet applications, or RIA. It provides a set of libraries for developing graphical user interfaces in Java. javafx.application.Platform is one of the core components of this platform.

What Does javafx.application.Platform Do?

javafx.application.Platform is responsible for loading the application, and provides access to a variety of utility functions, such as running tasks on the UI thread, setting up and running Thread groups, setting up and running Runnable tasks and more. In order to use it, you must create an instance of javafx.application.Platform, which is usually done via the static method Platform.runLater().

Example Usage of javafx.application.Platform

Let's consider an example in which we'll be using javafx.application.Platform to run a simple Runnable task on the UI thread.


          
           
Runnable task = () -> {
    System.out.println("Task is running on UI thread.");
};

// Run the task on the UI thread.
Platform.runLater(task);

Platform.runLater()
