Mockito Captor is an annotation-driven framework that enables Java developers to create powerful unit tests for their applications. The Mockito Captor allows developers to write automated tests by utilizing the power of annotations.   <artifactId>mockito-captor</artifactId>
  <version>1.0.1</version>
</dependency>

Matchers
