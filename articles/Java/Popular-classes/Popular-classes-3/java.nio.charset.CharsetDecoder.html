java.nio.charset.CharsetDecoder

The java.nio.charset.CharsetDecoder class is a part of the Java NIO Charset API that defines how to decode a sequence of bytes into a characters using a given charset. It is an abstract class which provides methods to convert bytes into characters.

Constructors

There are three constructors provided by this class:


          
           
public CharsetDecoder()
public CharsetDecoder(float averageCharsPerByte, float maxCharsPerByte)
public CharsetDecoder(Charset cs, float averageCharsPerByte, float maxCharsPerByte)

The first constructor creates a new decoder for the given charset. The second constructor creates a new decoder for the given charset with the given averages and maximums for chars per byte. The third constructor creates a new decoder with the given charset, averages, and maximums.

Methods

The CharsetDecoder class provides many methods to decode a sequence of bytes into characters using a given charset. Here are some of the methods provided by this class:


          
           
public CoderResult decode(ByteBuffer in, CharBuffer out, boolean endOfInput)
public final Charset detectedCharset()
public CodingErrorAction malformedInputAction()
public CharsetDecoder onMalformedInput(CodingErrorAction newAction)
public CodingErrorAction unmappableCharacterAction()
public CharsetDecoder onUnmappableCharacter(CodingErrorAction newAction)
The 

          
           
decode()
method is used to decode a sequence of bytes from the given ByteBuffer into characters in the given CharBuffer. The

method is used to retrieve the charset detected during the decoding process. The 

          
           
malformedInputAction()

onMalformedInput()

unmappableCharacterAction()

onUnmappableCharacter()

Example

Here is an example demonstrating the usage of this class:


          
           
// Get the charset decoder
CharsetDecoder decoder = Charset.forName("UTF-8").newDecoder(); 

// Set the malformed input action
decoder.onMalformedInput(CodingErrorAction.REPLACE); 

// Set the unmappable character action
decoder.onUnmappableCharacter(CodingErrorAction.IGNORE); 

// Decode the byte array
