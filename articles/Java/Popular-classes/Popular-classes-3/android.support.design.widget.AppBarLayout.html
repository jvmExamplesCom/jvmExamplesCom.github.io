Introduction to android.support.design.widget.AppBarLayout

android.support.design.widget.AppBarLayout is a vertical linear layout which implements the material design principles of motion and scrolling structure. It is used to provide consistent material design user experience with navigation, view manipulation, and transition effects. The AppBarLayout also supports customization of its appearance and behavior, using XML attributes and custom views.

Attributes

The AppBarLayout provides several attributes which allow it to be customized and tailored to any design. These include: 
  • 
            
             
app:layout_scrim_start
    – Specifies the start color of the scrim displayed when content is scrolled.
  • 
            
             
app:layout_scrim_end
    – Specifies the end color of the scrim displayed when content is scrolled.
  • 
            
             
app:layout_collapseMode
    – Determines the scroll behavior of the AppBarLayout when content is scrolled.
  • 
            
             
app:layout_collapseParallaxMultiplier
    – Specifies the amount of parallax applied when content is scrolled.
  • app:layout_behavior
    – Determines the AppBarLayout's behavior when content is scrolled.

Example Usage

Here is a basic example of how to use an AppBarLayout in your app: 

          
           
<?xml version="1.0" encoding="utf-8"?>
<android.support.design.widget.CoordinatorLayout
    xmlns:android="http://schemas.android.com/apk/res/android"
    xmlns:app="http://schemas.android.com/apk/res-auto"
    android:layout_width="match_parent"
    android:layout_height="match_parent">

    <android.support.design.widget.AppBarLayout
        android:id="@+id/appbar_layout"
        android:layout_width="match_parent"
        android:layout_height="wrap_content"
        app:layout_scrollFlags="scroll|enterAlways"
        app:layout_collapseMode="pin">

        ...

    </android.support.design.widget.AppBarLayout>

    ...

</android.support.design.widget.CoordinatorLayout>
This code will create a vertical linear layout that is pinned to the top of the screen when content is scrolled, and will scroll offscreen when content is entered.

Conclusion

In conclusion, AppBarLayout is a powerful tool for providing consistent material design experiences in Android applications. By using the AppBarLayout's attributes, you can customize the look and behavior of your application's navigation, views, and transitions.
