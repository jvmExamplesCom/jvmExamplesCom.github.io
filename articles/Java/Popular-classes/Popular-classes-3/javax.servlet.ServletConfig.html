javax.servlet.ServletConfig

The 

          
           
javax.servlet.ServletConfig
interface is an object that contains initialization parameters and startup information for a servlet. Its primary purpose is to provide access to web application-specific context data, such as initialization parameters, resource paths, and configuration data. It also provides a way of acquiring a reference to the 
ServletContext
, which is the environment of the servlet.

How to Create a ServletConfig Object

Each servlet must have an associated 
ServletConfig
object, which is created when the servlet is initialized by the container. The settings for each servlet are specified in the deployment descriptor. When the servlet is loaded, the container creates a 
ServletConfig
object and passes it to the servlet's 
init()
method.

Methods of ServletConfig

The 
ServletConfig
interface defines the following methods: 
  • getServletName()
    : Returns the name of the servlet
  • getServletContext()
    : Returns a reference to the 
    ServletContext
  • 
            
             
getInitParameter(String name)
    : Returns the value of the specified initialization parameter
  • 
            
             
getInitParameterNames()
    : Returns an 
    Enumeration
    of all the initialization parameter names

MyServlet
class:

  private ServletConfig config;

    this.config = config;
  }

}
