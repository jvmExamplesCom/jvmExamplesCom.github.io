io.netty.channel.ChannelHandler

It defines a interface for writing asynchronous event-driven network applications with complete control over how network data is transmitted. This makes it easier to develop high-performance network applications in Java. The ChannelHandler class is an abstract class that serves as the base class for all channel handler implementations.
