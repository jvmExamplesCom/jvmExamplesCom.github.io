org.apache.maven.plugins.annotations.Parameter

The org.apache.maven.plugins.annotations.Parameter annotation is used to indicate a parameter of a Maven plugin. It is one of the annotations that can be used to simplify the implementation of a plugin by providing an alternative to configuring the plugin in the pom.xml file. Parameters marked with this annotation can be configured via the project configuration or from the command line.

Annotation Syntax

This annotation follows the syntax described in the javadoc for the annotation. The full syntax for the annotation is as below.

    String description() default "";
