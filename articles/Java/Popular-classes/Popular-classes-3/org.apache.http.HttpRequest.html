What is org.apache.http.HttpRequest?

org.apache.http.HttpRequest is a class in the Apache HttpComponents Core library that provides an abstract inteface for HTTP request messages. The HttpRequest interface defines the basic contract for HTTP request messages which includes methods for accessing message headers and the request target as well as convenience methods for convenient access to commonly used aspects of HTTP requests.
