java.io.PipedOutputStream

Java.io.PipedOutputStream is a part of the Java I/O system which provides an output stream for bytes that are written to a piped input stream. A PipedOutputStream can be connected to a PipedInputStream to create a communications pipe between two threads. The PipedOutputStream class is used to write data to a target piped input stream.

Creating a PipedOutputStream

To create a PipedOutputStream object, use one of the constructors: 


          
           
PipedOutputStream(PipedInputStream snk) 
PipedOutputStream()

The first constructor creates a PipedOutputStream connected to the piped input stream snk.
The second one creates an unconnected PipedOutputStream, which must be connected to a PipedInputStream before being used.

Writing to a PipedOutputStream

After a PipedOutputStream is created, data can be written to it by using the inherited write() methods from OutputStream.


          
           
// write a single byte 
void write(int b)

This method writes the specified byte to the piped output stream. 


          
           
// write bytes from an array 
void write(byte buf[], int off, int len)

This method writes len bytes from the specified array to the piped output stream starting at offset off.

Closing a PipedOutputStream

When a PipedOutputStream is no longer needed, it should be closed by invoking its inherited close() method.


          
           
void close() throws IOException

This method closes the pipe, flushes the pipe, and releases any system resources associated with the pipe.
