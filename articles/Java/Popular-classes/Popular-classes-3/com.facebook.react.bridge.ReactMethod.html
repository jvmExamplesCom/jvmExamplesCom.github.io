com.facebook.react.bridge.ReactMethod

The 

          
           
com.facebook.react.bridge.ReactMethod
interface defines methods which allow the React Native bridge to interface with native modules on both Android and iOS platforms. Native modules can implement methods which can be invoked from JavaScript code running in the React Native environment. It allows your application to access platform specific function calls and bridge them to JavaScript.

Examples

To demonstrate, let's look at a very simple example of a native module. Here, we'll create a module called 

MyModule
: 

          
           
public class MyModule extends ReactContextBaseJavaModule {
  @ReactMethod
  public void showAlert(String message) {
    AlertDialog.Builder alertDialog = new AlertDialog.Builder(getCurrentActivity());
    alertDialog.setMessage(message);
    alertDialog.show();
  }
  // Other methods omitted
}
This module has a single method called 
showAlert
, which shows an alert dialog with a given message on the screen. This method is annotated with the 
@ReactMethod
annotation, which tells React Native that this method should be exposed to JavaScript.

Once the module is defined, we can invoke it from JavaScript:


          
           
var MyModule = require("NativeModules").MyModule;
MyModule.showAlert("Hello from JavaScript!");
When the 
showAlert
method is invoked, the alert dialog will be displayed on the screen.

Conclusion

The 

          
           
com.facebook.react.bridge.ReactMethod
interface is a powerful tool for bridging native code and React Native. By using this interface, you can access platform-specific APIs and functions from your React Native code.
