javax.mail.Message

javax.mail.Message is a class from the javax.mail package which provides a simple API for accessing email message data. It is designed to give applications an easy way to access and manipulate email messages, including retrieving, creating, sending, and deleting messages.

Creating a Message

The simplest way to create a Message instance is by providing a source URL. The source should point to a file containing the contents of the message in RFC 822 format.


          
           
String sourceUrl = "http://example.com/message.eml";
Message message = new Message(sourceUrl);

Alternatively, the message can be created from a javax.mail.Session instance.


          
           
Session session = Session.getDefaultInstance(System.getProperties());
Message message = new Message(session);

Retrieving Message Data

Once the Message instance has been created, the data contained within it can be accessed using the various getters provided. These getters provide access to the message's subject, date sent, sender, recipients, and content.


          
           
String subject = message.getSubject();

Object content = message.getContent();

Sending Messages
