JavaFX.scene.control.Label is an important class in JavaFX that provides access to the Label control for GUI applications. Labels provide text display for information, for example, displaying statuses of operations, directions, or other messages. Labels can also be used as the source of text for user input, with some limitations such as they are not editable.

Label label = new Label("My Label");

label.setFont(font);
