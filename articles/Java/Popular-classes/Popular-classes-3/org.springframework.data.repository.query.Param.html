org.springframework.data.repository.query.Param

What is org.springframework.data.repository.query.Param?

org.springframework.data.repository.query.Param is a class in the Spring Data Repository package that allows controllers to bind request parameters to named parameters in queries. This can help reduce the number of hard-coded query strings used in the application.

How to Use org.springframework.data.repository.query.Param

The first step is to create a 

          
           
@Param
annotation on the method parameter. This will bind the request parameter to the parameter in the query. Then, you can use 

          
           
@Query("select p from Product p where p.name = :name")
in your repository methods. This way, the value of the "name" parameter will be substituted into the query when you call the repository's find methods.

Example of org.springframework.data.repository.query.Param

Here’s an example of using 

          
           
@Param
in a Spring REST controller. 

          
           
@RestController
public class ProductController {

    @Autowired
    private ProductRepository productRepository;

    @GetMapping("/product")
    public List<Product> getProduct(@RequestParam String name) {
        return productRepository.findByName(name);
    }
}
Next, let’s look at the repository interface for this controller: 

          
           
public interface ProductRepository extends CrudRepository<Product, Long> {

   @Query("SELECT p FROM Product p WHERE p.name = :name")
   List<Product> findByName(@Param("name") String name);

}
The 

          
           
@Query
annotation defines a query for the 

          
           
findByName()
method. The 

          
           
@Param
annotation binds a method parameter to the query named parameter. Now, when we call the 

          
           
getProduct()
endpoint in our REST controller, it will execute the query and bind the value of the "name" request parameter to the named parameter in the query.

Conclusion

In this article, we looked at how to use org.springframework.data.repository.query.Param to bind request parameters to named parameters in queries.
