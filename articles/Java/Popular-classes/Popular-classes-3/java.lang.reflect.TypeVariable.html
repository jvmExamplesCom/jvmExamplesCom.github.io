java.lang.reflect.TypeVariable

The class 
TypeVariable
 in the  
java.lang.reflect
 package represents a type variable, which is an unbound generic type that declares a generic type. It is used to create instances of generic types or obtain type information of generic types.

It is important to note that this class is not a type itself but rather a holder for type information. It is also used to obtain information about the bounds of a type variable, the naming conventions associated with it, and the lower bounds associated with the type.

Example Usage

The following example code shows how to use the 
TypeVariable
class. 

          
           
import java.lang.reflect.TypeVariable;
//...

//Create a type variable called 'T'
TypeVariable<?> t = new TypeVariable<>("T");

//Set the bounds of 'T'
t.setBounds(new Type[] { Integer.class, Long.class });

//Set the lower bounds of 'T'
t.setLowerBounds(new Type[] { Integer.class });

//Print the name of the type variable
System.out.println("Name of type variable: " + t.getName());

//Print the upper bounds of the type variable
System.out.println("Upper bounds of type variable: ");
for (Type bound : t.getBounds()) {
    System.out.println("- " + bound.getTypeName());
}

//Print the lower bounds of the type variable
System.out.println("Lower bounds of type variable: ");
for (Type bound : t.getLowerBounds()) {
    System.out.println("- " + bound.getTypeName());
}
