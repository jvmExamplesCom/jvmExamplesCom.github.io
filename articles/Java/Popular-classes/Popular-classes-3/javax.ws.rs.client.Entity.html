javax.ws.rs.client.Entity

javax.ws.rs.client.Entity is a specific type of javax.ws.rs.client, which is an Extensible Client-side Java framework used to access resources that are exposed in a HTTP protocol. It provides helper classes and methods to easily create requests and manipulate responses.

Creating an Entity

An Entity can be created using the 
Entity.json(Object)
or 
Entity.xml(Object)
methods. This will create an object with the specified content-type with the given entity. 

          
           
Entity myEntity = Entity.json(jsonObject);

Using an Entity

An Entity can be used for creating requests, for example 

post(URI, Entity)
, or for receiving responses, for example 
readEntity(Class)
. 

          
           
String response = target.request().post(myEntity).readEntity(String.class);

Examples

Here are some examples of how to use javax.ws.rs.client.Entity.

Create a POST request with XML Entity


          
           
Entity xmlEntity = Entity.xml(xmlObject);
Response response = target.request().post(xmlEntity);

Convert a response from a POST request to a JSON object


          
           
JsonObject jsonObject = target.request().post(myEntity).readEntity(JsonObject.class);
