javax.ws.rs.client.ClientBuilder

javax.ws.rs.client.ClientBuilder is a class of the Java API for RESTful Web Services (JAX-RS, JSR-311). This class provides a singeton instance of Client that can be used to access the Web resources by utilizing the fluent API.

Methods

The methods available in ClientBuilder class are as follows:

  • accept(): It sets an accept header to indicate acceptable media types in the request.
  • connectTimeout(): It sets the connect timeout value.
  • cookie(): It adds a cookie to the request.
  • entity(): It sets an entity object to be sent in the request.
  • followRedirects(): It sets a flag determining whether redirects are followed.
  • hostnameVerifier(): It sets a HostnameVerifier implementation to be used to verify if the hostname matches the expected one.
  • userAgent(): It sets the user agent for the request.
  • sslContext(): It sets the SSLContext to use for secure connections.
  • register(): It registers the given class or instance.
  • build(): It builds the Client instance.

Example Usage

Below is a example demonstrating how to use the ClientBuilder class to build a Client instance and make a request:


          
           
Client client = ClientBuilder.newClient();

Response response = client.target("http://www.example.com")
                          .request(MediaType.TEXT_PLAIN)
                          .get();

String result = response.readEntity(String.class);

System.out.println(result);

In this example, we first create a Client instance with the ClientBuilder. Then, we make a request to the specified URL, setting the response type to TEXT_PLAIN. Finally, we read the response body and output it to the console.
