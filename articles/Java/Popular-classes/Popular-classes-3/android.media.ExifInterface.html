android.media.ExifInterface

The android.media.ExifInterface class is an Android class that can read and write Exif tags in a JPEG file or a RAW image file. Exif (Exchangeable image file format) is a standard that specifies the formats for images, sound, and ancillary tags used by digital cameras (including smartphones), scanners and other systems handling image and sound files recorded by digital cameras.

Getting Started

To use the android.media.ExifInterface, first initialize it with a file path and call its getAttribute method to retrieve the Exif tags from the file.


          
           
ExifInterface exif = new ExifInterface("image.jpg");
String attribute = exif.getAttribute(ExifInterface.TAG_MAKE);
The 

          
           
getAttribute
method takes a string argument representing the name of the Exif tag. It returns the value associated with the tag or null if the tag does not exist in the file.

Writing Tags

To write Exif tags, first create a ContentValues instance with the tag name and value. Call the setAttribute method of the android.media.ExifInterface instance to write the tag.


          
           
ContentValues values = new ContentValues();
values.put(ExifInterface.TAG_MAKE, "My camera");
exif.setAttribute(ExifInterface.TAG_MAKE, values);

          

          
           
setAttribute
takes two string arguments - the tag name and the value. The attribute will be written to the file after the call to the 

          
           
saveAttributes
method. 

          
           
exif.saveAttributes();

Example

To demonstrate writing and reading Exif tags, we will create a simple app that reads and writes Exif tags to JPEG and RAW image files. First, create a new project in Android Studio with a blank Activity.

Next, add the following permissions to the AndroidManifest.xml file:


          
           
<uses-permission android:name="android.permission.READ_EXTERNAL_STORAGE" />
<uses-permission android:name="android.permission.WRITE_EXTERNAL_STORAGE" />

These permissions are needed to access the external storage where our images are located. Now we need to write code to access the Exif tags of an image.

First, create a method to read the Exif tags from an image file. This method takes a path to an image file as an argument and returns a Map of the Exif tags and their corresponding values.


          
           
public static Map<String, String> readExifTags(String path) {
    Map<String, String> exifData = new HashMap<>();

    try {
        ExifInterface exif = new ExifInterface(path);
        for (String tagName : exif.getAttributeTable().keySet()) {
            exifData.put(tagName, exif.getAttribute(tagName));
        }
    } catch (IOException e) {
        e.printStackTrace();
    }

    return exifData;
}

Now we can write a method to write Exif tags to an image file. This method takes a path to an image file, a Map of tag names and values, and an optional boolean flag. If the boolean flag is set to true, existing tags in the image file will be overwritten. Otherwise, only tags that do not already exist in the file will be written.


          
           
public static void writeExifTags(String path, Map<String, String> exifData, boolean overwrite) {
    try {
        ExifInterface exif = new ExifInterface(path);

        for (Map.Entry<String, String> entry : exifData.entrySet()) {
            String tagName = entry.getKey();
            String value = entry.getValue();

            if (overwrite || !exif.hasAttribute(tagName)) {
                exif.setAttribute(tagName, value);
            }
        }

        exif.saveAttributes();
    } catch (IOException e) {
        e.printStackTrace();
    }
}

Finally, we create a method to read and write Exif tags. This method takes a path to an image file, a Map of Exif tags and values, and a boolean flag. The boolean flag indicates whether existing tags should be overwritten.


          
           
public static void readWriteExifTags(String path, Map<String, String> exifData, boolean overwrite) {
    Map<String, String> readData = readExifTags(path);
    readData.putAll(exifData);

    writeExifTags(path, readData, overwrite);
}

Now we can use the readWriteExifTags method to read and write Exif tags. We can also use the readExifTags and writeExifTags methods individually to read or write tags.

And that's how you can read and write Exif tags using the android.media.ExifInterface class.
