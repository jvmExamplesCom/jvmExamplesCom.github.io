javax.swing.border.Border

The javax.swing.border.Border class is a Java Swing library class that provides a way of defining the boundaries of components like Panels, Buttons, etc., within a graphical user interface (GUI). It is an abstract class and its subclasses are used to create specific kinds of borders, such as Line Border, Titled Border, Bevel Border, and many others. A Border can be set on any component by invoking the setBorder() method. Besides providing aesthetics, borders also help users get more familiar with the component's purpose.

Creating Instances of Border Subclasses

To use a border, one must first create an instance of the desired subclass. To do this, you must use a constructor that takes one or two arguments. The following code creates an instance of the javax.swing.border.LineBorder class:


          
           
LineBorder lb = new LineBorder(Color.BLUE, 5);

This creates a LineBorder object with a blue color and a thickness of 5 pixels. After creating a border, you can then use it to set the desired effect on a component by invoking the setBorder() method.


          
           
Button btn = new Button;
btn.setBorder(lb);

The above code sets the line border created at the beginning to a button component.

Other Border Subclasses

As explained earlier, there are other subclasses for defining Borders. These are TitledBorder, EmptyBorder, CompoundBorder, EtchedBorder, and MatteBorder. For example, the following code creates an instance of the javax.swing.border.TitledBorder class: 


          
           
TitledBorder tb = new TitledBorder("My Title");

This creates a TitledBorder object with the title "My Title". Other attributes like color, font, position, etc. can be specified while creating the TitledBorder object. After creating a TitledBorder object, you can then use it to set the desired effect on a component by invoking the setBorder() method.


          
           
Button btn = new Button;
btn.setBorder(tb);

The above code sets the titled border created at the beginning to a button component.

Using CreateCompoundBorder()

The javax.swing.border.Border class provides a createCompoundBorder() method which can be used to combine two or more border objects. For example, the following code creates a compound border using a LineBorder and a TitledBorder:


          
           
Border cb = BorderFactory.createCompoundBorder(lb, tb);

Now the compound border created at the beginning can be used to set the desired effects on a component by invoking the setBorder() method.


          
           
Button btn = new Button;
btn.setBorder(cb);

The above code sets the compound border created at the beginning to a button component.
