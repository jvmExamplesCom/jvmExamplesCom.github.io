java.nio.file.DirectoryStream

The 

          
           
java.nio.file.DirectoryStream
is an interface that provides an efficient way to access sets of files, primarily for the purpose of searching for specific directory entries. It is a subinterface of 
java.nio.file.Path
, which provides access to path-based operations. The DirectoryStream interface has several methods which can be used to search for and iterate over the contents of a given directory.

Usage Example:

Let's take a look at a simple example of how to use the DirectoryStream interface in practice. We'll be accessing the contents of a given directory and looking for files with a given name.


          
           
try (DirectoryStream<Path> dirStream = Files.newDirectoryStream(Paths.get("/my_dir"))) {
    for (Path p : dirStream) {
        if (p.getFileName().toString().equals("my_file.txt")) {
            System.out.println(p);
        }
    }
} catch (IOException e) {
    e.printStackTrace();
}

In this example, we are opening a DirectoryStream from the directory located at 

/my_dir
. We then loop through all of the files in the directory and check if the file name is equal to 
my_file.txt
. If it is, we output the file's path to the console.

Filter Interface:

While the example above only finds a single file, the DirectoryStream interface allows us to apply filters to our searches. This enables us to locate multiple files which satisfy a certain criteria. Let's take a look at how this works in practice.


          
           
try (DirectoryStream<Path> dirStream = Files.newDirectoryStream(Paths.get("/my_dir"),
    p -> p.toString().endsWith(".txt"))) {
    for (Path p : dirStream) {
        System.out.println(p);
    }
} catch (IOException e) {
    e.printStackTrace();
}

In this example, we pass a filter to the DirectoryStream when we open it. The filter is a lambda expression which tests if the file's path ends with the extension 

.txt
. For each file which passes the filter, we are outputting its path to the console. This allows us to easily search for multiple files with different characteristics.

Conclusion:

In this article, we have explored the basics of the DirectoryStream interface. We have looked at how to use it in order to search for specific files and apply filters to our searches.
