javax.management.MalformedObjectNameException

javax.management.MalformedObjectNameException is a Java exception that is thrown when an invalid ObjectName is passed to the JMX server. The object name must be formatted according to the rules set out in the JMX specification in order to be accepted by the server. try {

domain:property=value
