android.provider.ContactsContract

Android.provider.ContactsContract is a content provider class in the Android framework containing contact information, including the names and phone numbers of contacts associated with the user or device. It is a part of the Contacts Provider, which manages all contact data, including raw contacts, aggregate contacts, and metadata. The ContactsContract class stores its information in a SQLite database located at /data/data/com.android.provider.contacts/databases.

Features of android.provider.ContactsContract

  • It provides APIs to read and write the contact data from and to the device storage.
  • It provides APIs to query for contact information matching certain criteria, including name, phone numbers, emails, etc.
  • It provides API to manipulate the contact data, such as creating, deleting, updating, and searching.
  • It provides APIs to support communication via different methods, such as SMS, mail, etc.

// Add the contact phone number

                ContactsContract.CommonDataKinds.Phone.CONTENT_ITEM_TYPE)
        .withValue(ContactsContract.CommonDataKinds.Phone.NUMBER, "1234567890")
        .withValue(ContactsContract.CommonDataKinds.Phone.TYPE,

// Add the contact email
ops.add(ContentProviderOperation.newInsert(ContactsContract.Data.CONTENT_URI)

                ContactsContract.CommonDataKinds.Email.CONTENT_ITEM_TYPE)
        .withValue(ContactsContract.CommonDataKinds.Email.DATA, "john@doe.com")
        .withValue(ContactsContract.CommonDataKinds.Email.TYPE,
