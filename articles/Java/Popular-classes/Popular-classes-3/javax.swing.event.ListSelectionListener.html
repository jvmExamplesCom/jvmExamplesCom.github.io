javax.swing.event.ListSelectionListener

ListSelectionListener is an interface in the javax.swing package that is used to monitor changes in the user's selection in a List component, such as a JList. It has a single method, valueChanged(), which is invoked when the selection is changed.

Code Example 1


          
           
DefaultListModel listModel = new DefaultListModel();
listModel.addElement("Item 1");
listModel.addElement("Item 2");
listModel.addElement("Item 3");

JList list = new JList(listModel);
list.addListSelectionListener(new ListSelectionListener() {
  public void valueChanged(ListSelectionEvent event) {
    // Handle the event
  }
});
