java.util.jar.JarOutputStream

Java.util.jar is a collection of classes used to manipulate files and data in the JAR (Java ARchive) file format, which is based on the traditional ZIP file format. The java.util.jar package provides classes for reading and writing the JAR (Java ARchive) file format which is based on the standard ZIP file format.

The JarOutputStream class is a subclass of OutputStream which provides an easy way to write data and resources to a JAR-format output stream. A JarOutputStream instance is used to write a JAR file and is normally constructed on top of an existing OutputStream.

Example


          
           
//Create a new JarOutputStream and wrap it with a BufferedOutputStream 
JarOutputStream jarOutputStream = new JarOutputStream(new BufferedOutputStream(new FileOutputStream("myJarFile.jar")));
//Write bytes to the JarOutputStream
jarOutputStream.write("Hello World".getBytes());
//Close the JarOutputStream
jarOutputStream.close();

Methods

JarOutputStream class provides the following methods for writing to the JAR file:

  • writeEntry()
    - This method writes a JAR file entry to the JAR output stream.
  • putNextEntry()
    - This method creates a new JAR file entry and positions the stream to the start of the entry data.
  • closeEntry()
    - This method closes the current JAR file entry and positions the stream to write the next entry.
  • setComment()
    - This method sets the comment string for the JAR file.
  • finish()
    - This method finishes writing the contents of the JAR output stream without closing the underlying stream.

Example

In the following example, we will use JarOutputStream to create a JAR file containing two files: "file1.txt" and "file2.txt".


          
           
// Create a new JarOutputStream 
JarOutputStream jarOutputStream = new JarOutputStream(new FileOutputStream("myJarFile.jar"));
// Create a FileInputStream for each of the files to be added to the ZIP file
FileInputStream fis1 = new FileInputStream("file1.txt");
FileInputStream fis2 = new FileInputStream("file2.txt");
// Create a new JarEntry for each file and add it to the JarOutputStream
JarEntry entry1 = new JarEntry("file1.txt");
jarOutputStream.putNextEntry(entry1);
JarEntry entry2 = new JarEntry("file2.txt");
jarOutputStream.putNextEntry(entry2);
// Write the content of the InputStream to the JarOutputStream
byte[] buffer = new byte[1024];
int length;
while((length = fis1.read(buffer)) > 0) {
    jarOutputStream.write(buffer, 0, length);
}
while((length = fis2.read(buffer)) > 0) {
    jarOutputStream.write(buffer, 0, length);
}
// Close the JarEntries
jarOutputStream.closeEntry();
jarOutputStream.closeEntry();
// Close the JarOutputStream
jarOutputStream.close();

The above code will create a JAR file named "myJarFile.jar" containing two files: "file1.txt" and "file2.txt"
