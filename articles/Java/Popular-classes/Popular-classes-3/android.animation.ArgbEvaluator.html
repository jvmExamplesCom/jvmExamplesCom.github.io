android.animation.ArgbEvaluator

android.animation.ArgbEvaluator is a utility class used to calculate an Integer type color value which is evaluated by two color values and the fraction from each of the color components. In order to calculate the fraction of the value, the value will be multiplied by the fraction and added to the start value. This class can be used with ObjectAnimator or ValueAnimator to perform animation calculations on color values.

Examples

The following example shows how to use the ArgbEvaluator class to calculate a color value between two colors: 

          
           
// Create the ArgbEvaluator
ArgbEvaluator argbEvaluator = new ArgbEvaluator();

// Evaluate the color value between two colors
int evaluatedColor = (int) argbEvaluator.evaluate(fraction, startValue, endValue);
In this example, the fraction argument represents the amount of time that has elapsed in the animation, the startValue argument represents the starting color and the endValue argument represents the ending color.

Output

The output of this example would be a calculated Integer type color value between the start and end color of the animation. The result of the calculation can then be used as the color property value for the animation target.

Conclusion

The android.animation.ArgbEvaluator class provides a convenient utility for calculating an Integer type color value between two colors based on a fraction value. This class can be used with ObjectAnimator or ValueAnimator to perform animation calculations on color values for Android applications.
