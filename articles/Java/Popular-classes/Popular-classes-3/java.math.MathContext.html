Java.math.MathContext

Java.math.MathContext enables precision math operations on arbitrary-precision numbers by encapsulating a given precision setting. It also lets you create arbitrary MathContexts with whatever settings you provide.

Overview

A Java MathContext object consists of the following parameters:
  • Precision - The number of digits used for the result of an operation.
  • Rounding Mode - The rounding algorithm used for operations that produce fractional results.
  • Form - The form in which arithmetic results are expressed.
The MathContext class is immutable. Once created, the properties cannot be modified.

Examples
