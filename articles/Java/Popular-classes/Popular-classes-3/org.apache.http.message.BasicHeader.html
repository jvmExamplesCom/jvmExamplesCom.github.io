Introduction to org.apache.http.message.BasicHeader in Java

BasicHeader h = new BasicHeader("Content-Type", "text/html");

BasicHeader h1 = new BasicHeader("Content-Type", "text/html");
BasicHeader h2 = new BasicHeader(h1);
