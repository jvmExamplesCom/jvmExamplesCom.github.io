Supported Formats

ImageFormat supports eight standard image formats available on the Android platform:
  • JPEG: Support static and progressive JPEG with both 8 bits and 12 bits.

  • PNG: Portable Network Graphics with 8 bits, 16 bits or 24 bits color depth.
  • WEBP: With both single and multiple frames, as well as animation.
