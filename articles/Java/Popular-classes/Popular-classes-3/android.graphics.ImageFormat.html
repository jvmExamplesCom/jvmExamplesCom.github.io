java.android.graphics.ImageFormat

The ImageFormat class is used to define the mapping from pixel formats to Java image types, providing functionality for the Android platform’s system-wide image format. It provides a number of constants that define the various image formats available in the API, as well as support for querying and manipulating images through Java classes.

Supported Formats

ImageFormat supports eight standard image formats available on the Android platform:
  • JPEG: Support static and progressive JPEG with both 8 bits and 12 bits.
  • GIF: Single frame and multi-frame GIF, and animated GIF encoding.
  • PNG: Portable Network Graphics with 8 bits, 16 bits or 24 bits color depth.
  • WEBP: With both single and multiple frames, as well as animation.
  • BMP: Bitmap with up to 24 bits per pixel.
  • HEIF: High Efficiency Image Format with support for still and motion images.
  • RAW_RGB: Raw RGB image data.
  • RAW_YUV: Raw YUV image data.

Example of ImageFormat Class

This example shows how to use the ImageFormat class to retrieve information about an image file. 

          
           
// Create an ImageFormat instance
ImageFormat imageFormat = new ImageFormat();

// Get the image format
int imageFormatType = imageFormat.getFormat();

// Determine the type of the image
if (imageFormatType == ImageFormat.JPEG) {
    System.out.println("Image file type: JPEG");
} else if (imageFormatType == ImageFormat.PNG) {
    System.out.println("Image file type: PNG");
}

Methods of ImageFormat Class

The ImageFormat class contains a number of methods used to query and manipulate images.
  • getFormat(): Returns the image format type.
  • getWidth(): Returns the width of the image.
  • getHeight(): Returns the height of the image.
  • getMimeType(): Returns the MIME type of the image.
  • isRGB(): Returns true if the image is in RGB format.
  • setColorDepth(): Sets the color depth of the image.
  • resize(): Resizes the image to the specified width and height.
  • rotate(): Rotates the image to the specified angle.
  • crop(): Crops the image to the specified dimensions.

ImageFormat is a powerful and convenient way to store and manipulate images on the Android platform.
