Reactor Core Publisher Mono

Reactor Core Publisher Mono is a reactive programming library for Java, written on the Reactive Streams specification. Mono is a component of the Reactor project, a collection of libraries for building high-performance, low-latency reactive systems.

Mono provides an efficient and lightweight asynchronous programming model for writing reactive Java applications. It is designed to support non-blocking, concurrent I/O operations, allowing developers to write asynchronous applications without having to learn complex threading models. Mono can also be used to write distributed computing applications that can scale across multiple cores and machines.

Creating a Mono

The simplest way to create a Mono is to use the factory method 

Mono.just()
: 

          
           
Mono<String> mono = Mono.just("Hello World!");

This code creates a Mono that emits a single item, namely the String "Hello World!".

It is also possible to create a Mono from a Publisher using the 

from()
method: 

          
           
Publisher<String> publisher = Flux.just("foo", "bar");
Mono<String> mono = Mono.from(publisher);

This code creates a Mono from a Flux containing two Strings, "foo" and "bar".

Transforming a Mono

Mono supports a wide range of transformation operations. These operations can be used to modify the emissions of a Mono in some way. For example, the 

map()
operator can be used to transform each item emitted by the Mono: 

          
           
Mono<String> mono = Mono.just("Hello World!");
Mono<String> transformedMono = mono.map(s -> s.toUpperCase());

This code creates a new Mono whose emissions are uppercase versions of the emissions of the original Mono. The transformation is applied lazily, i.e. only when the transformed Mono is subscribed to.

Subscribing to a Mono

Once a Mono has been created or transformed, it must be subscribed to in order for its emissions to be processed. This is done using the 

subscribe()
method: 

          
           
Mono<String> mono = Mono.just("Hello World!");
Mono<String> transformedMono = mono.map(s -> s.toUpperCase());
transformedMono.subscribe(s -> System.out.println(s));

This code will print the uppercase version of the String "Hello World!" to the console.
