Introduction to the org.apache.http.conn.ssl.SSLConnectionSocketFactory Class

The org.apache.http.conn.ssl.SSLConnectionSocketFactory class is a utility class similar to javax.net.ssl.SSLSocketFactory but for use with Apache HttpComponents client to create secure connections. It provides more control over the way connections are established and is useful when external libraries are used, such as when using HTTP with Google App Engine. This class wraps an SSL socket factory to enable secure connections using the Secure Sockets Layer (SSL) or Transport Layer Security (TLS).

Creating an SSLConnectionSocketFactory Instance

The static factory methods of this class allow one to obtain standard SocketFactory implementations that implement the specified secure communication protocols.
