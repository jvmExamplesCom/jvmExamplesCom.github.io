Introduction to Java.nio.charset.CharsetEncoder

Java.nio.charset.CharsetEncoder is an interface in the java.nio.charset package that enables developers to create encoders for character sets. This makes it possible to convert characters into bytes in different encoded formats. It is used to encode strings, chars and byte arrays.

Methods

The CharsetEncoder class contains several methods which allow for encoding data into a specific format. These methods are as follows:

  • encode(CharBuffer)
    : This method takes a CharBuffer object as an argument and returns a ByteBuffer object with the encoded contents of the CharBuffer.
  • flush()
    : This method flushes any remaining data from the charset encoder.
  • 
            
             
onMalformedInput(CodingErrorAction action)
    : This method specifies what action should be taken when a malformed input is encountered. By default, it throws a MalformedInputException.
  • 
            
             
onUnmappableCharacter(CodingErrorAction action)
    : This method specifies what action should be taken when an unmappable character is encountered. By default, it throws an UnmappableCharacterException.
  • reinitialize()
    : This method resets the charset encoder. It clears any internal state and resets any pending output.

Example

Here is an example that shows how to use the CharsetEncoder class. This example will encode a string called "Hello World!" in UTF-8 format.


          
           
String inputString = "Hello World!";

try {
    // Create a charset encoder object for UTF-8
    CharsetEncoder encoder = Charset.forName("UTF-8").newEncoder();

    // Wrap the input string in a CharBuffer
    CharBuffer charBuffer = CharBuffer.wrap(inputString);

    // Encode the CharBuffer and get the ByteBuffer
    ByteBuffer byteBuffer = encoder.encode(charBuffer);

    // Print out the encoded ByteBuffer
    System.out.println(byteBuffer);
} catch (Exception ex) {
    System.err.println(ex.getMessage());
}

The output of this code will be a ByteBuffer object containing the encoded version of "Hello World!" in UTF-8 format.
