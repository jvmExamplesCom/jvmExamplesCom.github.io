java.io.PipedInputStream

The java.io.PipedInputStream class is a subclass of InputStream and allows the reading of data from a stream through one pipe. This type of implementation is useful for communicating between threads, where one thread reads from a PipedInputStream and another thread writes to a PipedOutputStream.

Constructors

The java.io.PipedInputStream class has two constructors:

  • 
            
             
public PipedInputStream ()
    - This constructor creates an unconnected PipedInputStream object.
  • 
            
             
public PipedInputStream (PipedOutputStream src) throws IOException
    - This constructor creates a connected PipedInputStream object and connects it to the given PipedOutputStream.

Methods

The java.io.PipedInputStream class has several methods which can be used to read and manage data from the stream:

  • available()
    - This method returns the number of bytes that can be read from the stream without blocking.
  • close()
    - This method closes the input stream and releases any resources associated with the stream.
  • 
            
             
connect(PipedOutputStream src)
    - This method connects this piped input stream to a piped output stream.
  • read()
    - This method reads the next byte of data from the input stream.
  • read(byte[] b)
    - This method reads up to b.length bytes of data from the input stream into an array of bytes.
  • 
            
             
read(byte[] b, int off, int len)
    - This method reads up to len bytes of data from the input stream into an array of bytes starting at offset off in the array.
  • skip(long n)
    - This method skips n bytes of input from the stream.

Example Usage

The following example demonstrates how to use the PipedInputStream class to read data from a PipedOutputStream:


          
           
import java.io.*;

public class Main {
  public static void main(String[] args) {
    try {
      // Create a PipedInputStream
      PipedInputStream pis = new PipedInputStream();

      // Create a PipedOutputStream and connect it to the PipedInputStream
      PipedOutputStream pos = new PipedOutputStream(pis);

      // Write some data to the PipedOutputStream
      pos.write("Hello World!".getBytes());

      // Read the data from the PipedInputStream
      byte[] data = new byte[128];
      int length = pis.read(data);

      // Output the data that was read from the PipedInputStream
      System.out.println(new String(data, 0, length));

      // Clean up
      pos.close();
      pis.close();
    } catch (IOException e) {
      e.printStackTrace();
    }
  }
}
