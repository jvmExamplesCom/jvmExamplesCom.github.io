Understanding Java.time.format.DateTimeParseException

Java.time.format.DateTimeParseException is a subclass of DateTimeException and signals that an error has been encountered while parsing the date-time string. This is most often seen when trying to parse a date-time string which does not adhere to the defined format, or in cases where the specified temporal field is invalid.

String input = "2020-11-29";
try {
    LocalDate.parse(input);
