android.bluetooth.BluetoothDevice

Android.bluetooth.BluetoothDevice is a class that provides Bluetooth functionality to the Android platform. It enables developers to easily add Bluetooth connectivity to their applications. This article examines the various methods and properties that are part of the BluetoothDevice class.

Methods

The following methods are available from the BluetoothDevice class:

  • createBond()
    - Creates a connection between two Bluetooth devices.
  • connectGatt()
    - Connects to a GATT profile on a given Bluetooth device.
  • close()
    - Closes the connection to the Bluetooth device.
  • getAddress()
    - Gets the address of the Bluetooth device.
  • getBondState()
    - Returns the bond state of the Bluetooth device.
  • getName()
    - Returns the name of the Bluetooth device.
  • getUuids()
    - Returns an array of UUIDs supported by the Bluetooth device.
  • setPin()
    - Sets the PIN for a Bluetooth connection.

Properties

The following properties are available from the BluetoothDevice class:

  • TYPE_CLASSIC
    - Indicates that the device is a classic Bluetooth device.
  • TYPE_LE
    - Indicates that the device is a low energy Bluetooth device.
  • BOND_BONDED
    - Indicates that the device has been bonded.
  • BOND_NONE
    - Indicates that no bonding has taken place.
  • TRANSPORT_AUTO
    - Indicates that the transport type should be auto-negotiated.
  • TRANSPORT_BREDR
    - Indicates that the transport type should be Classic Bluetooth.
  • TRANSPORT_LE
    - Indicates that the transport type should be Low Energy Bluetooth.

Example

The following example shows how to use the BluetoothDevice class to discover nearby Bluetooth devices and display their names:


          
           
BluetoothAdapter adapter = BluetoothAdapter.getDefaultAdapter();
if (adapter != null &amp;&amp; adapter.isEnabled()) {
    Set<BluetoothDevice> devices = adapter.getBondedDevices();
    if (devices != null) {
        for (BluetoothDevice device : devices) {
            Log.d("BT", "Device Name: " + device.getName());
        }
    }
}

This example will print the names of any Bluetooth devices paired with the Android device. If no devices are paired, then no output will be printed.
