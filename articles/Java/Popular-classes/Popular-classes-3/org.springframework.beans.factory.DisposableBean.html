org.springframework.beans.factory.DisposableBean

DisposableBean is an interface from the Spring Framework's org.springframework.beans.factory package which provides a callback for resources that need to be released when no longer needed. Many BeanFactory implementations can automatically detect beans that implement this interface and invoke the necessary destroy callbacks.

Example Usage

The following example shows how to implement the DisposableBean in a simple class:


          
           
public class MyBean implements DisposableBean {
    @Override
    public void destroy() throws Exception {
        // Destroy resources here
    }
}

Once the bean is created, the Spring container will automatically invoke the destroy method. Depending on the use case, the method may be used to release resources, close database connections, etc.

Usage with Annotation

The DisposableBean can also be used with annotations as follows:


          
           
@PreDestroy
public void destroy() {
   // Release resources here
}
The 

          
           
@PreDestroy
annotation allows the destroy method to be automatically invoked by the Spring container.
