Introduction to org.xml.sax.ErrorHandler

org.xml.sax.ErrorHandler is an interface used for reporting errors during the process of parsing XML documents. This interface is implemented by applications that use the SAX parser, and when an error is encountered, it will be reported through the implementation of this interface. In this article, we will look at how to use this interface and the methods it contains.

Using ErrorHandler Interface

To use ErrorHandler interface, all we need to do is create a class implementing this interface. Let's demonstrate this by creating a class called MyErrorHandler which implements the ErrorHandler interface: 

          
           
public class MyErrorHandler implements ErrorHandler {
   // Implementation of ErrorHandler interface goes here     
}
Once we have created our custom ErrorHandler class, we can register it with the parser so that when a parsing error occurs, our custom ErrorHandler will be used to handle the error. The following code shows how to do this: 

          
           
SAXParserFactory parserFactory = SAXParserFactory.newInstance();
SAXParser saxParser = parserFactory.newSAXParser();
MyErrorHandler handler = new MyErrorHandler();
saxParser.setErrorHandler(handler);

public class MyErrorHandler implements ErrorHandler {

   public void warning(SAXParseException e) {
