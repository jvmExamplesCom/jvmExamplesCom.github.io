java.awt.BasicStroke

The java.awt.BasicStroke class provides the capabilities of drawing lines with a variety of stroke widths and end-cap styles and line join styles. It implements the Stroke interface, which allows it to be used directly in many other Java packages, including in the Java2D API. The BasicStroke class has several constructors that allow you to customize the stroke width, end-cap style, and line join style.

Constructors

The first constructor takes a single parameter, width, which is a float value specifying the width of the stroke. The default end-cap style is CAP_BUTT and the default line join style is JOIN_MITER.

The second constructor takes three parameters: width, cap, and join. width is a float value specifying the width of the stroke, as before. cap is an integer constant specifying the end-cap style, which can be one of three values: CAP_BUTT, CAP_ROUND, or CAP_SQUARE. join is an integer constant specifying the line join style, which can be one of three values: JOIN_MITER, JOIN_ROUND, or JOIN_BEVEL.

The third constructor takes seven parameters: width, cap, join, miterlimit, dash, dash_phase, and dash_array. The first three parameters are the same as for the second constructor. miterlimit is a float value specifying the limit to trim the miter when two lines join at a sharp angle. The last three parameters are used to create dashed lines. dash is an array of floats specifying the lengths of dashes and gaps. dash_phase is a float value specifying the offset into the dash array. dash_array is an array of floats specifying the lengths of dashes and gaps in between each dash.
