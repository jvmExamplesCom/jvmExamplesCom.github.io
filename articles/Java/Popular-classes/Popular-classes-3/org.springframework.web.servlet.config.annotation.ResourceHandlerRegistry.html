org.springframework.web.servlet.config.annotation.ResourceHandlerRegistry

The 

          
           
org.springframework.web.servlet.config.annotation.ResourceHandlerRegistry
class is a part of the Spring Framework. This class provides a registry for resource handlers and allows developers to register resource handlers using a registry instance. Resource handlers are used for managing resources such as JavaScript, HTML and CSS files. This registry allows developers to configure URL paths and file locations for resources. It also allows developers to specify parameters like cache control, content type and character encoding.

Example 1: Registering a Resource Handler with ResourceHandlerRegistry

In this example, we will register a resource handler with a 


          
           
ResourceHandlerRegistry
instance. We will register a resource handler that uses the "/resources" URL path and points to the "src/main/webapp/resources" folder. 

          
           
ResourceHandlerRegistry registry = new ResourceHandlerRegistry();
registry.addResourceHandler("/resources/**").addResourceLocations("classpath:/src/main/webapp/resources/");

Example 2: Configuring Cache Control with ResourceHandlerRegistry

In this example, we will configure cache control for a resource handler registered with a 


          
           
ResourceHandlerRegistry
instance. We will configure cache control for the "/resources" URL path and set it to 1 hour. 

          
           
ResourceHandlerRegistry registry = new ResourceHandlerRegistry();
registry.addResourceHandler("/resources/**")
        .addResourceLocations("classpath:/src/main/webapp/resources/")
        .setCacheControl(CacheControl.maxAge(1, TimeUnit.HOURS));

Example 3: Configuring Character Encoding with ResourceHandlerRegistry

In this example, we will configure character encoding for a resource handler registered with a 


          
           
ResourceHandlerRegistry
instance. We will configure character encoding for the "/resources" URL path and set it to UTF-8. 

          
           
ResourceHandlerRegistry registry = new ResourceHandlerRegistry();
registry.addResourceHandler("/resources/**")
        .addResourceLocations("classpath:/src/main/webapp/resources/")
        .setCharacterEncoding("UTF-8");
