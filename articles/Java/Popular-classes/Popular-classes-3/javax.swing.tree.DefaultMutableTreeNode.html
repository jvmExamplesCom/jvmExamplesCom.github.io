javax.swing.tree.DefaultMutableTreeNode

javax.swing.tree.DefaultMutableTreeNode

Creating a Simple Tree Node

Creating a new 

          
           
DefaultMutableTreeNode
is a simple matter of passing a string to its constructor: 

          
           
DefaultMutableTreeNode rootNode = new DefaultMutableTreeNode("Root");

Adding and Accessing Child Nodes

To add a child node to an existing node, call the 

          
           
add()
method: 

          
           
DefaultMutableTreeNode child1 = new DefaultMutableTreeNode("Child 1");
rootNode.add(child1);
The

insert()
