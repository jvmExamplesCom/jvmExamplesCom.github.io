Introduction to androidx.annotation.ColorInt

The androidx.annotation.ColorInt annotation allows developers to indicate that a color value is expected to be an int value instead of a color resource id. Using androidx.annotation.ColorInt

To use the androidx.annotation.ColorInt, you can add it to the top of variables which hold a color value.

@ColorInt
