Introduction to javax.ws.rs.ext.ExceptionMapper

The javax.ws.rs.ext.ExceptionMapper class is an extension of the JAX-RS API. It provides a mechanism for handling exceptions thrown by an application written in Java and ensuring that the appropriate response is sent back to the client. ExceptionMapper can be used to catch any unchecked exception such as NullPointerException, IllegalArgumentException, etc. and map it to an appropriate response.
