Introduction to javax.ws.rs.ext.ExceptionMapper

The javax.ws.rs.ext.ExceptionMapper class is an extension of the JAX-RS API. It provides a mechanism for handling exceptions thrown by an application written in Java and ensuring that the appropriate response is sent back to the client. ExceptionMapper can be used to catch any unchecked exception such as NullPointerException, IllegalArgumentException, etc. and map it to an appropriate response.

Creating a Custom ExceptionMapper

To create a custom ExceptionMapper, create a class that implements the ExceptionMapper interface. The following example creates a custom ExceptionMapper that maps a NullPointerException to a response with status code 500. 


          
           
public class NullPointerExceptionMapper implements ExceptionMapper {

	@Override
	public Response toResponse(NullPointerException exception) {
		return Response.status(Status.INTERNAL_SERVER_ERROR).build();
	}

}
