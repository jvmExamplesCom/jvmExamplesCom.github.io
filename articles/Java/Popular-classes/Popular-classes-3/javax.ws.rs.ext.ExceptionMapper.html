Introduction to javax.ws.rs.ext.ExceptionMapper

The javax.ws.rs.ext.ExceptionMapper class is an extension of the JAX-RS API. ExceptionMapper can be used to catch any unchecked exception such as NullPointerException, IllegalArgumentException, etc. and map it to an appropriate response.
