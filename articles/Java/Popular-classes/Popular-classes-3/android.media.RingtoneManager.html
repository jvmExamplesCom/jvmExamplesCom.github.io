android.media.RingtoneManager

The android.media.RingtoneManager is a class used to create and manage ringtones, the sounds that notify the user about incoming calls and other events. This class allows applications to access the device's list of known ringtones and to query and set the default or current ringtone. It also provides methods for building an intent to launch the default ringtone picker.
