Introduction to org.apache.kafka.clients.producer.ProducerRecord

The org.apache.kafka.clients.producer.ProducerRecord is a Java API that enables developers to interact with the Apache Kafka message broker for the purpose of sending messages. To create a ProducerRecord, an application must specify the topic, partition key, value, headers and other optional parameters. In the following example, a new ProducerRecord is created with a topic name and a message value.


          
           
ProducerRecord<String, String> producerRecord = 
new ProducerRecord<String, String>("topicName", "messageValue");
