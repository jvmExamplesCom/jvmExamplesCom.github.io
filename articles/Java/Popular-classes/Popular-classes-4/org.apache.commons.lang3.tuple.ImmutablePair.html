Introduction

org.apache.commons.lang3.tuple.ImmutablePair is a class from the Java library which provides the functionality to create a basic immutable pair object from two values. A pair consists of two values, commonly referred to as left and right, which can be of any type. This class is useful in situations where users need to store a pair of related objects.

Code Examples

Creating an ImmutablePair


          
           
ImmutablePair<Integer, String> pair = new ImmutablePair<>(1, "One");

The code above creates an immutable pair where the left element is an Integer and the right element is a String.

Accessing the Left and Right Element of a Pair


          
           
int leftElement = pair.getLeft();
String rightElement = pair.getRight();

The code above assigns the left element of the pair to the leftElement variable, and the right element of the pair to the rightElement variable.

Creating a New Pair from Existing Pair


          
           
ImmutablePair<Integer, String> newPair = pair.withLeft(2);
System.out.println(newPair); // Output: (2, One)

The code above creates a new pair with the same right element as the existing pair, but with a different left element. In this case, the new left element is an Integer with value 2.

Comparing Two Pairs


          
           
boolean equals = pair.equals(newPair);
System.out.println(equals); // Output: false

The code above compares the two pairs and assigns the result to the equals variable. The result is false, since the left elements of the two pairs are different (1 and 2).

Conclusion

org.apache.commons.lang3.tuple.ImmutablePair is a helpful class which allows users to create basic immutable pair objects from two values. It provides methods to access the left and right elements of a pair, create a new pair from an existing pair, and compare two pairs.
