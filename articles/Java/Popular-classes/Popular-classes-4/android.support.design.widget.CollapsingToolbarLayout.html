android.support.design.widget.CollapsingToolbarLayout

The android.support.design.widget.CollapsingToolbarLayout is a ViewGroup that implements many of the visual pattern from Google's Material Design specification. This class provides a handy way to add a compound view of title and floating action buttons, along with an optional expanded content area.

Basic Usage

In order to use the CollapsingToolbarLayout, you must include the following dependencies in your project: 

          
           
implementation 'com.android.support:design:27.+'
implementation 'com.android.support:appcompat-v7:27.+'
The CollapsingToolbarLayout should be placed inside the AppBarLayout, which is itself a direct child of the CoordinatorLayout, like so:

    android:id="@+id/main_content"
    android:layout_width="match_parent"
    android:layout_height="match_parent">

Attributes

The CollapsingToolbarLayout has several attributes that can be set in order to customize the behavior and appearance of the view.

app:layout_scrollFlags

This attribute sets the scrolling behavior when the user scrolls up or down. By setting the scroll flags, you can control whether the view should collapse or expand when the user scrolls. The available values are:
  • scroll: the view should scroll offscreen
  • exitUntilCollapsed: the view should exit until it is completely collapsed
  • enterAlways: the view should enter even if the user scrolls up
  • enterAlwaysCollapsed: the view should enter only if the user scrolls up and the view is completely collapsed
For example: 

          
           
android:layout_scrollFlags="scroll|exitUntilCollapsed"

app:layout_collapseMode

This attribute defines how the view should collapse when the user scrolls up or down. The available values are:
  • pin: the view should remain pinned at the top of the screen
  • parallax: the view should collapse with a parallax effect
For example: 

          
           
android:layout_collapseMode="parallax"

app:layout_collapseParallaxMultiplier

This attribute sets the multiplier for the parallax effect. The default value is 0.5, which means the view should collapse at half the speed of the user's scrolling. The value should be between 0 and 1. For example: 

          
           
android:layout_collapseParallaxMultiplier="0.75"

setExpandedTitleTextAppearance(int resId)

This method sets the text appearance of the title in the expanded toolbar. The text appearance should be provided as a resource id.
