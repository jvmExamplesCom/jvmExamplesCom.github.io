Java.nio.ShortBuffer

Java.nio.ShortBuffer is a class in the Java programming language that provides an allocation of short values from a short array, direct or non-direct ByteBuffer. It allows read/write access to a short array and can be used as an array view of a short array, as well as to perform operations on the individual elements of the array.

Creating a Short Buffer

A ShortBuffer can be created either via a 

          
           
allocate(int capacity)
method, passing the capacity of the buffer to allocate a new one, or by wrapping an existing short array: 

          
           
short[] array = {1, 2, 3};
ShortBuffer buffer = ShortBuffer.wrap(array);

Reading and Writing Data

Data can be read and written from a ShortBuffer using various methods such as 


          
           
put(short)
for writing single shorts, 

          
           
put(short[], int, int)
for writing an array of shorts, or 

          
           
get()
for reading single shorts. The data can also be retrieved from the short array view of the ShortBuffer with 

          
           
array()
. 

          
           
ShortBuffer buffer = ShortBuffer.allocate(5);
buffer.put(0, (short) 1);
buffer.put((short) 2);
buffer.put(new short[] {3, 4, 5}, 0, 3);
short[] result = buffer.array();
// result is equal to [1, 2, 3, 4, 5]

Limiting the Buffer

The number of bytes that can be read/written from/to the ShortBuffer can be limited using the 


          
           
limit(int)
method. This can be used to set a maximum length for the ShortBuffer, which will cause any read/write operations beyond the limit to fail. The limit can also be reset using the 

          
           
reset()
method. 

          
           
ShortBuffer buffer = ShortBuffer.allocate(10);
buffer.put(new short[] {1, 2, 3}, 0, 3);
buffer.limit(3);
// buffer now has a maximum size of 3

Flipping the Buffer

The ShortBuffer can be flipped using the 

          
           
flip()
method. This sets the limit to the current position and resets the position to 0, allowing the buffer to be read from the beginning. 

          
           
ShortBuffer buffer = ShortBuffer.allocate(10);
buffer.put(new short[] {1, 2, 3}, 0, 3);
buffer.flip();
// buffer.limit is equal to 3 and buffer.position is equal to 0

Clearing the Buffer

The ShortBuffer can be cleared using the 

          
           
clear()
method. This sets the position to 0 and the limit to the capacity of the buffer, allowing data to be written from the beginning of the buffer. 

          
           
ShortBuffer buffer = ShortBuffer.allocate(10);
buffer.put(new short[] {1, 2, 3}, 0, 3);
buffer.clear();
// buffer.limit is equal to 10 and buffer.position is equal to 0
