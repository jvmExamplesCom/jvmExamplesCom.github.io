Introduction to com.google.common.base.Supplier

The Java 

          
           
com.google.common.base.Supplier

com.google.common.base.Supplier
and demonstrate its use with code examples.

The Supplier Interface

The 

          
           
com.google.common.base.Supplier
interface is a simple functional interface that consists of the following method: 

          
           
T get()

It takes no arguments and is expected to return a result of type 

T
. This method is intended to be used to produce or retrieve objects on demand, such as a thread pool connection or a network resource.

com.google.common.base.Supplier<T>

Supplier
