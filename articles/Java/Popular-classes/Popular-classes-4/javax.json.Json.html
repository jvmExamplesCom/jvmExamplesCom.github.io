javax.json.Json

Java provides a library to work with JSON data. The javax.json.Json class allows you to read, write, and manipulate JSON data. This article will explore the features of the javax.json.Json class and provide code examples to illustrate how they can be used.

Creating a Json Object

A Json object can be created from a string using the static method: 

          
           
JsonObject jsonObject = Json.createObjectBuilder().build();
The above code will create an empty Json object.

Adding Values to a Json Object

Values can be added to a Json object using the add() method. The following example adds two keys and values to an existing Json object: 

          
           
jsonObject.add("name","John");
jsonObject.add("age", 30);

Reading Values from a Json Object

The getString() method is used to retrieve a value from a Json object. The following example retrieves the name from the Json object created earlier: 

          
           
String name = jsonObject.getString("name");

Iterating through a Json Object

The keySet() method returns a set of all the key names in a Json object. This set can then be iterated over to access each key value pair. The following example demonstrates how to iterate over a Json object: 

          
           
Set<String> keys = jsonObject.keySet();
for (String key: keys) {
    String value = jsonObject.getString(key);
    System.out.println(key + " : " + value);
}

Converting a Json Object to a String

The toString() method can be used to convert a Json object to a string. The following example will print the Json object as a string:
