org.apache.http.config.RegistryBuilder

The org.apache.http.config.RegistryBuilder is a class in the Apache HTTP Client Library which enables applications to define their own custom registries of supported protocol schemes, socket factories and credentials provider implementations.

Creating a RegistryBuilder

A RegistryBuilder can be instantiated as follows: 

          
           
RegistryBuilder<SchemeIOSessionStrategy> builder = 
    RegistryBuilder.create();

Example 1 - Adding Protocol Schemes and Socket Factories

The following code example adds the "http" and "https" protocol schemes with their respective socket factories:


          
           
builder.register("http", PlainConnectionSocketFactory.getSocketFactory());
builder.register("https", SSLConnectionSocketFactory.getSocketFactory());

Example 2 - Adding Credential Providers

The following code example adds a basic credential provider to the registry:


          
           
builder.register(AuthSchemes.BASIC, new BasicCredentialsProvider());

Building the Registry

Once all the desired components have been registered, the registry can be built with the following statement:


          
           
Registry<SchemeIOSessionStrategy> registry = builder.build();

Conclusion

The org.apache.http.config.RegistryBuilder class enables developers to easily create their own custom registries of supported protocol schemes, socket factories and credentials providers for use with the Apache HTTP Client library.
