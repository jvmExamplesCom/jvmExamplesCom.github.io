Java com.google.common.base.CharMatcher

The 

          
           
com.google.common.base.CharMatcher

The CharMatcher class provides various methods to detect and replace characters that match certain criteria. It can be used to eliminate undesirable characters, remove non-ASCII characters, or find common patterns in strings.
