Java.net.URLStreamHandler

Java.net.URLStreamHandler is a generic abstract class used to define a protocol specific stream handler. It provides a connection to the source of the data on a specific server.   • setURL() - This method is used to set a new URL.
  • equals() - This method checks if two URLs are equal.
