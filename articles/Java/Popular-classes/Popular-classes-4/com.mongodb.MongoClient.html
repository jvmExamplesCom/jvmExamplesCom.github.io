Introduction to com.mongodb.MongoClient

MongoClient is a Java library on the MongoDB driver that enables applications to connect to MongoDB servers and perform CRUD (Create, Read, Update and Delete) operations on documents in MongoDB databases. It provides a convenient interface to perform all kinds of operations with MongoDB. This tutorial will cover the basics of using the MongoClient library. We will go through how to connect to a MongoDB server, what operations can be performed using the MongoClient library, and the various ways of configuring the library.

Connecting to a MongoDB Server with MongoClient

To connect to a MongoDB server using MongoClient, you must first create a MongoClient instance. The following example shows how to create a MongoClient instance using the default connection settings. This will connect to a MongoDB server running on the default port, localhost:27017.


          
           
MongoClient mongoClient = new MongoClient();

You can also specify the database server's address and port in the constructor of the MongoClient class. For example, to connect to a MongoDB server running on 192.168.1.2 and port 1234, use the following code:


          
           
MongoClient mongoClient = new MongoClient("192.168.1.2", 1234);

Once you have created a MongoClient instance, you can use it to perform operations on the server.

Operations with MongoClient

The MongoClient class provides methods for performing various operations with MongoDB. Most of these methods are wrappers around the methods exposed by the MongoDatabase and MongoCollection classes. For example, the getDatabase() method is used to get an instance of MongoDatabase from the MongoClient instance. Once you have obtained a MongoDatabase instance, you can then use its methods to perform operations on the database.

Here is an example of how to get a MongoDatabase instance from a MongoClient instance, and then use it to list all collections in the database:


          
           
MongoClient mongoClient = new MongoClient();
MongoDatabase database = mongoClient.getDatabase("myDatabase");
MongoIterable<String> collections = database.listCollectionNames();
for (String collectionName : collections) {
    System.out.println(collectionName);
}

The MongoClient class also provides methods to directly access a collection in a database. The following example shows how to use the getCollection() method to get a MongoCollection instance to interact with a collection in a database: 


          
           
MongoClient mongoClient = new MongoClient();
MongoCollection<Document> collection = mongoClient.getDatabase("myDatabase").getCollection("myCollection");

Once you have obtained a MongoCollection instance, you can use its methods to perform various operations on the collection.

Configuring MongoClient

The MongoClient class also provides methods that allow you to configure the library. These configuration options can be used to modify the library's behaviour and enable features such as connection pooling and logging. Here is an example of how to use the MongoClientOptions class to configure a MongoClient instance:
