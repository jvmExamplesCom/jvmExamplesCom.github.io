javax.xml.validation.Schema

The javax.xml.validation.Schema class is used to load, parse and validate XML Schema documents. It is part of the Java API for XML Processing (JAXP) which provides a set of interfaces and classes that allow applications to access data stored in the Extensible Markup Language (XML) format. SchemaFactory factory = SchemaFactory.newInstance(XMLConstants.W3C_XML_SCHEMA_NS_URI);
