javax.xml.validation.Schema

The javax.xml.validation.Schema class is used to load, parse and validate XML Schema documents. It is part of the Java API for XML Processing (JAXP) which provides a set of interfaces and classes that allow applications to access data stored in the Extensible Markup Language (XML) format. This class allows an application to construct an XML Schema from scratch.

To create a Schema object, we can use the SchemaFactory class. The following code shows an example of using the newInstance method to create a SchemaFactory object, which is then used to create a Schema object:


          
           
SchemaFactory factory = SchemaFactory.newInstance(XMLConstants.W3C_XML_SCHEMA_NS_URI);
Schema schema = factory.newSchema();
