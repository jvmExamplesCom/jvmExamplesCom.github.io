javax.swing.JRadioButton

The javax.swing.JRadioButton class is a Java Swing component that displays two states: selected or unselected, and allows the user to switch between them. It can be used as a way to provide "radio buttons", which are mutually exclusive options that the user can pick from. When using multiple radio buttons, it is important to ensure that only one in the group can be selected at once.

To create a JRadioButton, you must use a constructor that requires a String argument for display text. You can also pass a boolean argument to determine if the button should be initially selected or not. The code snippet below creates a JRadioButton with the text 'optionA' and sets it to be initially selected.


          
           
JRadioButton optionA = new JRadioButton("optionA", true);

Once the radio button is created, you can add it to a container like any other Swing component. In the code example below, we add the radio button to a panel and then add the panel to a frame.


          
           
JPanel panel = new JPanel();
panel.add(optionA);

JFrame frame = new JFrame("My Frame");
frame.add(panel);
frame.pack();
frame.setVisible(true);

If multiple radio buttons are added to the same panel, they should be added to a ButtonGroup. This ensures that only one of the buttons can be selected at a given time. The code example below adds two radio buttons to a ButtonGroup and a panel.


          
           
JPanel panel = new JPanel();

ButtonGroup bg = new ButtonGroup();

The state of a JRadioButton can be changed programmatically by calling the setSelected() method. The code example below sets the optionA radio button to be selected.


          
           
optionA.setSelected(true);
