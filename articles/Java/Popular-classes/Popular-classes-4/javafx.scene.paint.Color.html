javafx.scene.paint.Color

The javafx.scene.paint.Color class is a JavaFX class that represents colors in the Scene graph. This class makes it easy to manipulate colors within the Scene graph by providing methods for constructing colors from different color spaces, converting colors to and from different color spaces, and determining colors in relation to other colors.

A color can be constructed from an ARGB value. An ARGB value is an 8-bit integer in the format 0xAARRGGBB, where AA is the alpha (or transparency) level, RR is the red level, GG is the green level, and BB is the blue level.


          
           
Color c = new Color(0xFF0000); // red

Or a color can be constructed from red, green, and blue components of a color. The components range from 0.0 to 1.0.


          
           
Color c = Color.rgb(1.0, 0.0, 0.0); // red

The Color class can also be used to convert colors between different color spaces. For instance, a color in RGB space can be converted to HSB space, and vice versa.


          
           
Color rgb = Color.rgb(1.0, 0.0, 0.0); // red

// convert to HSB 
HSBColor hsb = rgb.getHSBColor();

// convert back to RGB 
Color rgb2 = Color.hsb(hsb.getHue(), hsb.getSaturation(), hsb.getBrightness());
