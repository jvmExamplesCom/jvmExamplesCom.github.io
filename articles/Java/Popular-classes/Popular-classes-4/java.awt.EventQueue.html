java.awt.EventQueue

Java.awt.EventQueue is a class that handles the events generated by user actions and invokes a callback method to process the event. The Event Queue is part of the Java Abstract Window Toolkit (AWT). It consists of a queue of events that have been sent by the operating system or other user input devices such as mouse and keyboard. The events can be in different types such as MouseEvent, KeyEvent, PaintEvent and FocusEvent.

Creating an EventQueue Object

Creating an EventQueue object is simple. You just need to provide a constructor with no arguments.


          
           
EventQueue eventQueue = new EventQueue();

Dispatched Events

Events dispatched by the EventQueue are handled by AWT Event Dispatcher Thread. The events are put on the EventQueue when they come from the user. To process the events, the Event Dispatcher Thread calls the 

dispatchEvent()
method of the EventQueue. 

          
           
public void dispatchEvent(AWTEvent event)

The event can be processed by calling the processEvent() method of the object registered as a listener for that particular event type.


          
           
public void processEvent(AWTEvent event)

Pushing Events onto the Queue

Events can be pushed onto the queue by calling the 

postEvent()
method. 

          
           
public void postEvent(AWTEvent event)

The posted event will be added to the end of the event queue, and when the EventDispatcherThread is not busy, it will pick up the event and process it.

Processing Events

To process an event, you can call the processEvent() method of its corresponding class. For example, for MouseEvent you can use the following code snippet:


          
           
MouseEvent event = (MouseEvent) eventQueue.getNextEvent();
event.processEvent();

The processEvent() method will then call the appropriate methods of the listeners registered for that particular event type.

Removing Events

Events can also be removed from the queue by calling the 

removeEvent()
method. 

          
           
public void removeEvent(AWTEvent event)

This method takes an instance of the AWTEvent as its parameter and removes it from the queue.
