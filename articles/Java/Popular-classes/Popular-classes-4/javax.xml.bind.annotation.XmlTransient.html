Introduction to javax.xml.bind.annotation.XmlTransient

javax.xml.bind.annotation.XmlTransient is an annotation used to mark a class, field or method as being ignored by the JAXB processor. This annotation can be useful when dealing with objects that cannot be directly annotated with JAXB and must be excluded from the serialized XML result.

Code Examples


          
           
@XmlTransient
public class MyClass {
    // ...
}

This code example marks the class MyClass as being ignored by the JAXB processor.


          
           
@XmlTransient
private String myField;

The above code example marks the field myField as being ignored by the JAXB processor.
