Introduction to com.amazonaws.auth.BasicAWSCredentials

To create a 
BasicAWSCredentials
object, you need to provide your access key ID and secret access key to the constructor as strings. The access key ID and secret access key are provided by AWS when you create an IAM user. Here is an example of creating a 
BasicAWSCredentials
object using the constructor: 

          
           
String accessKeyId = "myAccessKeyId";
String secretAccessKey = "mySecretAccessKey";

BasicAWSCredentials credentials = new BasicAWSCredentials(accessKeyId, secretAccessKey);

credentials

List<Bucket> myBuckets = s3.listBuckets();
