java.security.PrivilegedActionException

A java.security.PrivilegedActionException is a runtime exception thrown when an application attempts to perform an action with insufficient security privileges. It can be thrown from within the code of an application or from within the Java EE security manager. The exception provides detailed information about the security violation, including the originator of the request, the action being performed, and necessary conditions required to allow the action. The exception also provides methods to obtain the originator's identity and to inspect the state of the security manager.
