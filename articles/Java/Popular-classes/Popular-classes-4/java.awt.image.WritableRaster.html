Java.awt.image.WritableRaster

Java.awt.image.WritableRaster is a class which enables the manipulation of image data within a specified rectangular region. This class can be used to read pixel values and to set new pixel values in the specified region.

Methods

The following methods are available in Java.awt.image.WritableRaster: 

  • 
            
             
getPixel(int x, int y, int[] iArray)
    - Returns the pixel color at the given coordinates.
  • 
            
             
setPixel(int x, int y, int[] iArray)
    - Sets the pixel color at the given coordinates.
  • 
            
             
getPixels(int x, int y, int w, int h, int[] iArray)
    - Gets the pixels from the given rectangle.
  • 
            
             
setPixels(int x, int y, int w, int h, int[] iArray)
    - Sets the pixels in the given rectangle.

Example Usage

The following code example shows how to use the WritableRaster class to get and set pixel values in a given 2D array: 


          
           
// Create an 8x8 WritableRaster 
WritableRaster raster = 
    Raster.createInterleavedRaster(DataBuffer.TYPE_BYTE, 8, 8, 1, null); 

// Create an array to store pixel data 
int[] pixelValues = new int[64]; 

// Fill the array with random values 
Random rand = new Random(); 
for (int i = 0; i < 64; i++) 
    pixelValues[i] = rand.nextInt(); 

// Get the pixel at 7,7 
int[] pixelAt77 = new int[1]; 
raster.getPixel(7, 7, pixelAt77); 
System.out.println("Pixel at 7,7: " + pixelAt77[0]); 

// Set the pixel at 6,6 
int[] newPixel66 = {1234}; 
raster.setPixel(6, 6, newPixel66); 

// Get the pixels from 5,5 to 7,7 
int[] pixels = new int[9]; 
raster.getPixels(5, 5, 3, 3, pixels); 
System.out.println("Pixels:"); 
for (int i = 0; i < 9; i++) 
    System.out.println("\t" + pixels[i]); 

// Set the pixels from 3,3 to 5,5 
int[] newPixels = {9999, 8888, 7777, 6666, 5555, 4444, 3333, 2222, 1111}; 
raster.setPixels(3, 3, 3, 3, newPixels);
