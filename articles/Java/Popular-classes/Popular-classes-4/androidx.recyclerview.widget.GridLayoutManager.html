androidx.recyclerview.widget.GridLayoutManager

The 

          
           
androidx.recyclerview.widget.GridLayoutManager
class is a LayoutManager implementation that lays out items in a grid. It supports both vertical and horizontal scrolling as well as multiple spans of items arranged in rows, columns, or both.

androidx.recyclerview.widget.GridLayoutManager
class has two constructors: one that takes a 
Context
object as an argument and another that takes a 
Context
and an integer indicating the number of columns or rows in the grid. 

          
           
GridLayoutManager(Context context) 
GridLayoutManager(Context context, int spanCount)

androidx.recyclerview.widget.GridLayoutManager
class has several methods that can be used to set up and manipulate the layout.

getSpanCount - Returns the number of spans in the grid.


          
           
int getSpanCount()
