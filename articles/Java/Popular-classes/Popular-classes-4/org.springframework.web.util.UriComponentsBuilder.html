org.springframework.web.util.UriComponentsBuilder

The org.springframework.web.util.UriComponentsBuilder Java class is a utility class used to create and manipulate uniform resource identifiers (URIs). It can be used to easily build URI components in a safe manner and supports many of the same techniques used by the popular Apache HttpClient project. In addition, it allows one to easily construct relative URIs and custom query strings.

Instantiating UriComponentsBuilder

The most basic way to use UriComponentsBuilder is to instantiate it with a single argument of type java.net.URI or java.net.URL. This will create an immutable version of the builder:


          
           
URI uri = new URI("http://example.com/foo?bar=baz");
UriComponentsBuilder builder = UriComponentsBuilder.fromUri(uri);

In this example, the builder is initialized with the given URI. The builder can then be used to build additional path segments and query parameters. For example:


          
           
UriComponents result = builder.pathSegment("bar")
        .queryParam("foo", "qux")
        .build();

builder.setConfiguration(DynamicConfiguration.ENABLE_ADDITIONAL_PATH_SEGMENTS, false);

Building a URI

UriComponentsBuilder builder = UriComponentsBuilder.fromUri(uri)
        .pathSegment("bar")
        .queryParam("foo", "qux")
        .build();

The UriComponentsBuilder also provides methods to set the scheme, host, port, username, password, path and fragment:

        .scheme("http")

        .build();

UriComponents result = UriComponentsBuilder.fromUri(baseUri)
