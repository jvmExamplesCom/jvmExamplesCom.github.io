Introduction to com.esotericsoftware.kryo.io.Output

Creating a Kryo Output Stream

To create a Kryo Output stream, you will need to create an instance of Kryo, define a serializer and then create an instance of Output like this:


          
           
Kryo kryo = new Kryo();
Serializer serializer = new MySerializer();
Output output = new Output(new FileOutputStream("file.txt"));

Writing Objects

To write an object to the stream, you can call 

          
           
kryo.writeObject(output, myObject)
This will serialize an object myObject into the stream using a given serializer. Any changes to the object after this point will not be reflected in the written result.

Closing the Stream

When you are done writing objects to the stream, you should call


          
           
output.close()
to ensure that all data is written correctly and any resources are released.
