Introduction to com.esotericsoftware.kryo.io.Output

Kryo kryo = new Kryo();
Serializer serializer = new MySerializer();
Output output = new Output(new FileOutputStream("file.txt"));

kryo.writeObject(output, myObject)

Closing the Stream

When you are done writing objects to the stream, you should call
