java.beans.PropertyChangeSupport

java.beans.PropertyChangeSupport is a java class that provides support for firing property change events and registering listeners for the same. It allows beans to have properties that can be changed and its change notified to other registered classes. The basic idea is that any class that wants to support property changes should extend java.beans.PropertyChangeSupport, and then it can fire property change events and register listeners to receive those events.

How It Works

java.beans.PropertyChangeSupport works by providing a public registerPropertyChangeListener method to register a PropertyChangeListener instance with the class. Whenever a property is changed in the class, the firePropertyChange method will be called on the PropertyChangeSupport instance. This will cause an event to be sent to all registered listeners, notifying them that a property has changed.

Example

Let's look at a simple example of how to use java.beans.PropertyChangeSupport. We will create a simple class called MyBean which extends java.beans.PropertyChangeSupport. This class will have two properties: foo and bar, and when either of these properties changes, a PropertyChangeEvent will be fired so that all registered listeners will be notified. 

          
           
public class MyBean extends java.beans.PropertyChangeSupport {

    private int foo;
    private int bar;

    public MyBean() {
        super(MyBean.class);
    }

    public int getFoo() {
        return this.foo;
    }

    public void setFoo(int foo) {

        return this.bar;
    }

        this.bar = bar;
        firePropertyChange("bar", oldBar, this.bar);
    }
}
