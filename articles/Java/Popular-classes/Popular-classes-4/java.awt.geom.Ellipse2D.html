Java.awt.geom.Ellipse2D

Java.awt.geom.Ellipse2D is a class used to construct and manipulate ellipses in Java. It allows creating circles, oval shapes, and also allows elliptic arcs. The class is compatible with java.awt.Shape, and thus allows us to use it for various operations such as drawing, filling or checking for containment.

Usage

The main operations offered by this class are the creation of ellipses from the provided parameters, getting the properties of the created ellipse, as well as clipping and transforming it.

Creating an Ellipse2D

Creating an Ellipse2D is done using one of the static methods from the class. There are several static methods available, each of them receiving different parameters.

  • 
            
             
Ellipse2D.Double(double x, double y, double w, double h)
    - Creates an ellipse with the specified coordinates (x,y), width (w), and height (h).
  • 
            
             
Ellipse2D.Float(float x, float y, float w, float h)
    - Creates an ellipse with the specified coordinates (x,y), width (w), and height (h).
  • 
            
             
Ellipse2D.Double(Rectangle2D rect)
    - Creates an ellipse from the given Rectangle2D.
  • 
            
             
Ellipse2D.Float(Rectangle2D rect)
    - Creates an ellipse from the given Rectangle2D.
  • 
            
             
Ellipse2D.getEllipse(Point2D center, double radiusX, double radiusY)
    - Creates an ellipse from the given center and the two radii.
  • 
            
             
Ellipse2D.getEllipse(Rectangle2D rect)
    - Creates an ellipse from the given Rectangle2D.

Getting properties of the Ellipse2D

There are several methods available to get information about the Ellipse2D instance.

  • getCenterX()
    - Returns the X coordinate of the ellipse's center.
  • getCenterY()
    - Returns the Y coordinate of the ellipse's center.
  • getWidth()
    - Returns the width of the ellipse.
  • getHeight()
    - Returns the height of the ellipse.
  • getRadiusX()
    - Returns the X radius of the ellipse.
  • getRadiusY()
    - Returns the Y radius of the ellipse.

Clipping and Transforming an Ellipse2D

The Ellipse2D class provides the following methods to clip and transform the ellipse:

  • 
            
             
clip(Rectangle2D rect)
    - Clips the ellipse to the given rectangle.
  • 
            
             
transform(AffineTransform trans)
    - Transforms the ellipse with the given AffineTransform.

Example

Let's create an example of using the Ellipse2D class.


          
           
// Create an Ellipse2D
Ellipse2D ellipse = new Ellipse2D.Double(0, 0, 10, 10);

// Get its center
Point2D center = ellipse.getCenter();
System.out.println("Center: " + center);

// Get the width and height
double width = ellipse.getWidth();
double height = ellipse.getHeight();
System.out.println("Width: " + width + ", Height: " + height);

// Clip the ellipse
Rectangle2D rect = new Rectangle2D.Double(-10, -10, 20, 20);
ellipse.clip(rect);

// Transform the ellipse
AffineTransform trans = AffineTransform.getTranslateInstance(10, 10);
ellipse.transform(trans);

Width: 10.0, Height: 10.0
