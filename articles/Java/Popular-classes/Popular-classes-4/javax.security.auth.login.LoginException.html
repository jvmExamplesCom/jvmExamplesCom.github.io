Understanding javax.security.auth.login.LoginException in Java

The javax.security.auth.login.LoginException class is a subclass of the java.lang.SecurityException class and is thrown by the login context when authentication fails. It usually indicates a wrong username or password. It is an unchecked exception, so it is not necessary to declare it in a method or handle it using a try-catch block.

Example

Following is an example that throws a LoginException when the username and password are incorrect:


          
           
try {
    //authentication code goes here
if (!authenticateUser(username, password)) 
    throw new LoginException("Username or password incorrect");
} catch (LoginException e) {
    e.printStackTrace();
}

In this example, if the username and password are incorrect, the authenticateUser() method will throw a LoginException with the message “Username or password incorrect”.
