javax.mail.internet.MimeBodyPart

javax.mail.internet.MimeBodyPart is a Java Standard Extension package which provides an API for creating MIME message bodies in multipart internet mail formats.

Overview

MimeBodyPart provides a class to represent body parts of a MIME message. It can create, parse and access the contents of a MIME body part. In addition, this class supports the work with the headers that contain additional meta-data about the MIME body part.

The javax.mail.internet.MimeBodyPart class provides an API for creating and managing MIME message body parts. This class provides methods to create, parse and access the contents of the body part and manipulate the header meta-data associated with it.
