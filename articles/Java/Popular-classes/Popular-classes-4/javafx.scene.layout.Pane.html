javafx.scene.layout.Pane

The 

          
           
javafx.scene.layout.Pane
class provides a general purpose layout for JavaFX applications. It contains two subclasses: 

          
           
javafx.scene.layout.Region
and 

          
           
javafx.scene.layout.GridPane
. It is a container that can contain other nodes (such as images, text, or controls).

Overview

The 

          
           
javafx.scene.layout.Pane
class is used to create a general purpose parent node for displaying JavaFX content. A pane contains two subclasses: 

          
           
javafx.scene.layout.Region
and 

          
           
javafx.scene.layout.GridPane
. Region subclasses are for application-specific layout designs (such as a game board), while GridPane subclasses are for creating a uniform grid of items. The Pane has various methods that can be used to add nodes to it, as well as methods for sizing and positioning the nodes.

Example


          
           
Pane pane = new Pane();
pane.setPrefSize(400, 200);

Text text = new Text("Hello World!");
text.setLayoutX(50);
text.setLayoutY(50);

pane.getChildren().add(text);

Scene scene = new Scene(pane);
primaryStage.setScene(scene);
This example creates a new Pane with a size of 400x200 and adds a Text element to it. The Text element is positioned at coordinates 50,50 within the Pane. Finally, the Pane is added to a new Scene which is then set to the primary stage.

Methods

The 

          
           
javafx.scene.layout.Pane
class contains several useful methods: 

          
           
add(Node): Adds a node to the Pane.

          

          
           
getChildren(): Returns a list of nodes in the Pane.

          

          
           
setPrefSize(double, double): Sets the preferred size of the Pane.

          

          
           
setLayoutX(double): Sets the x-position of a node within the Pane.

          

          
           
setLayoutY(double): Sets the y-position of a node within the Pane.
These methods allow developers to easily add and manipulate nodes within the Pane.

Conclusion

The 

          
           
javafx.scene.layout.Pane
class is a powerful tool for creating general purpose layouts in JavaFX applications. Through its two subclasses, developers can create either custom layouts or uniform grids of nodes. Additionally, several useful methods are available for manipulating the nodes within the Pane.
