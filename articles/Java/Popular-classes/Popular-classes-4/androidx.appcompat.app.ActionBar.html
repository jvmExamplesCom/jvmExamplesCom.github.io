androidx.appcompat.app.ActionBar

The androidx.appcompat.app.ActionBar class is used to provide an interface for users to interact with Android applications. It enables users to access a range of features and functions related to the application. This class provides a way for an application to control the behavior of its action bar and for an application to respond to user interaction events.

Features and Functions

The androidx.appcompat.app.ActionBar class provides several features and functions to android applications. These features and functions include:

  • Controls the appearance of the action bar.
  • Allows the user to navigate up and down within the application.
  • Supports action items, which are clickable images that can be used to perform certain actions within the application.
  • Provides support for the overflow menu.
  • Provides support for contextual action modes.
  • Enables the user to search within the application.

Creating an Action Bar

The following code example demonstrates how to create an ActionBar in an android application:


          
           
ActionBar actionBar = getSupportActionBar();
if (actionBar != null) {
    actionBar.setDisplayHomeAsUpEnabled(true);
}

In this code example, the getSupportActionBar() method is used to get the ActionBar object associated with the current activity. The setDisplayHomeAsUpEnabled() method is then used to enable the icon of the app on the action bar. This will allow the user to navigate up and down within the application.

Adding an Action Item

The following code example demonstrates how to add an action item to an ActionBar:


          
           
actionBar.setDisplayHomeAsUpEnabled(true);
actionBar.setHomeButtonEnabled(true); 
actionBar.addActionItem(R.drawable.ic_action_save, 
				"Save", 
				new View.OnClickListener(){
					@Override
					public void onClick(View view){
						// Save logic here
					}
				});

In this code example, the setDisplayHomeAsUpEnabled() and setHomeButtonEnabled() methods are used to enable the app icon on the action bar. Then, the addActionItem() method is used to add an action item to the action bar. This method takes the drawable resource id for the action item icon, a title for the action item, and a View.OnClickListener to handle clicks on the action item.
