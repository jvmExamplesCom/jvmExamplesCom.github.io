java.nio.charset.CodingErrorAction

The java.nio.charset.CodingErrorAction class defines action strategies for dealing with coding errors encountered while decoding or encoding a character stream. The strategies specified by the constants REPORT, REPLACE and IGNORE can be used to handle coding errors encountered when a charset is used to decode or encode a byte stream.
