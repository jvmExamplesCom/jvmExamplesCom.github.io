org.apache.ibatis.session.SqlSessionFactory

org.apache.ibatis.session.SqlSessionFactory is an interface in the Apache iBatis Java persistence framework which manages SQL Sessions. It is the central point of working with data persisted in a relational database. It creates a session, using which all the database related operations can be performed. This article will explain what SqlSessionFactory is and how to create and use it.

Creating a SqlSessionFactory

Creating a SqlSessionFactory requires an instance of the SqlSessionFactoryBuilder class. This in turn requires an instance of the Reader class and the environment. Once we’ve obtained the instances of the Builder, Reader, and environment objects, we can call the builder’s build() method, passing in the reader, environment, and a reference to the mapped statements.


          
           
Reader reader = Resources.getResourceAsReader("mybatis-config.xml");
SqlSessionFactoryBuilder builder = new SqlSessionFactoryBuilder();
Environment environment = new Environment("development", new JdbcTransactionFactory(), dataSource);
Configuration configuration = new Configuration(environment);
SqlSessionFactory factory = builder.build(reader, configuration, mappedStatement);

Using the SqlSessionFactory

The SqlSession factory provides two methods for creating a Session. The first is the openSession() method, which takes no parameters and returns a new Session object.


          
           
SqlSession sqlSession = factory.openSession();

The other method is the openSession(boolean autoCommit) method. This method accepts a boolean parameter that indicates whether or not the Session should be automatically committed. If the value is true, the Session will be automatically committed after every operation; if the value is false, it will not be.


          
           
SqlSession sqlSession = factory.openSession(false);

Closing a Session

When you are done working with the Session, you must close it. To do this, simply call the close() method on the Session object.


          
           
sqlSession.close();
