org.w3c.dom.Attr

The org.w3c.dom.Attr interface represents an attribute in an element. It is an inherited interface from Node, and provides methods for retrieving the name and value of an attribute, as well as other characteristics.

Getting Attribute Information

The main purpose of the org.w3c.dom.Attr interface is to facilitate access to attribute information. Here are a few methods available for retrieving information about attributes.

getName()

The getName() method returns the name of the attribute.


          
           
String attName = attr.getName();

getValue()

The getValue() method returns the value of the attribute.


          
           
String attValue = attr.getValue();

hasSpecified()

The hasSpecified() method returns true if the attribute was explicitly specified on the element.


          
           
boolean isSpecified = attr.hasSpecified();

getNodeType()

The getNodeType() method returns the node type, which is always equal to ATTRIBUTE_NODE (2).


          
           
int nodeType = attr.getNodeType();
