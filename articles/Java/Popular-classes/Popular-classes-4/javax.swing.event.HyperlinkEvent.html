javax.swing.event.HyperlinkEvent

The javax.swing.event.HyperlinkEvent class is an abstract class that provides support for events corresponding to user actions on hyperlinks in a document or a web page. A hyperlink is a reference in a document or a web page to another part of the document or another document or web page, and is typically displayed as colored, underlined text.

Hyperlink events are generated in response to user actions such as clicking or pressing Enter on a hyperlink. They can be used to trigger an action, such as loading a new document or web page. Hyperlink events are sent to objects that implement the HyperlinkListener interface.

Creating a Hyperlink Event

The javax.swing.event.HyperlinkEvent class provides a constructor that takes a URL object and an event type as parameters:


          
           
HyperlinkEvent(URL url, HyperlinkEvent.EventType type)

The URL object specifies the resource associated with the event, and the type parameter can have one of the following values:


          
           
ENTERED
EXITED
ACTIVATED

The last two values correspond to the mouse entering or exiting a hyperlink, or the user clicking or pressing Enter on it. The ENTERED value is only used when the mouse enters a hyperlink.

Example

The following example creates a HyperlinkEvent object with a URL and an EVENT_TYPE of ACTIVATED:


          
           
try {
    URL url = new URL("https://www.example.com");
    HyperlinkEvent event = new HyperlinkEvent(url, HyperlinkEvent.EventType.ACTIVATED);
