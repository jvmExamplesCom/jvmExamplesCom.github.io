org.aspectj.lang.annotation.Before

The org.aspectj.lang.annotation.Before annotation is used to create a pointcut that signals the execution of a method before the actual method call. This is useful for performing pre-execution activities such as security and data validation.

Example Code


          
           
@Before("execution(* *.*(..))")
  public void beforeAdvice() {
      System.out.println("Before advice called");
  }
In the above example, the 
@Before
annotation is applied to the 
beforeAdvice()
method. This method will be called before any other method in the application.

Output


          
           
Before advice called

Example with Join Point Arguments

The 
@Before
annotation also supports join point arguments. The join point arguments provide contextual information about the method being called. 

          
           
@Before("execution(* *.*(..))")
 public void beforeAdviceWithArgument(JoinPoint joinpoint) {
     System.out.println("Before advice called with "+joinpoint.getSignature().getName());
 }

Output


          
           
Before advice called with <method name>
