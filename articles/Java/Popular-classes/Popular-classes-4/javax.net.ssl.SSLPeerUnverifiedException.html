About javax.net.ssl.SSLPeerUnverifiedException

javax.net.ssl.SSLPeerUnverifiedException is a subclass of the Java SSL-related exception class that is thrown when an SSL handshake fails due to the server not being able to authenticate the client. This exception will be thrown if the client is unable to verify the identity of the server based on the available certificates, or if the server is not trusted by the client.
