io.reactivex.disposables.CompositeDisposable

The 

          
           
io.reactivex.disposables.CompositeDisposable
is an object that allows multiple Disposables to be managed together. It is particularly useful when you have multiple Disposables within a method and want to dispose them all with just a single call. This can make code more efficient and organized.

A CompositeDisposable is a container that can hold multiple Disposables. The Disposables can be added, removed, and disposed all in one go. The CompositeDisposable is the main class used in RxJava for managing the lifecycle of multiple Disposables.

create()
method: 

          
           
CompositeDisposable compositeDisposable = new CompositeDisposable();
