javax.security.auth.Subject

Java provides an implementation of the Subject class, found in the javax.security.auth package. The Subject class is used to represent a user or entity that has authentication credentials and can access resources on a system. A Subject is composed of Principals and public credentials. Principals are objects that represent a user's identity, such as a username or email address. Public credentials are objects that contain information about a user, such as an X.509 certificate or a Kerberos ticket. A Subject can also be associated with one or more security-related objects such as Access Control Lists (ACLs) and Principal-Based Access Control (PBAC).

The Subject class provides several static methods for performing authentication operations such as login and logout. These methods are passed a Subject object and authentication information such as a username, password, or public credentials. Upon successful authentication, the Subject object is populated with the user's principals and public credentials.
