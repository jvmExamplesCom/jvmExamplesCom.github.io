javax.security.auth.Subject

Java provides an implementation of the Subject class, found in the javax.security.auth package. The Subject class is used to represent a user or entity that has authentication credentials and can access resources on a system. A Subject is composed of Principals and public credentials. Public credentials are objects that contain information about a user, such as an X.509 certificate or a Kerberos ticket. A Subject can also be associated with one or more security-related objects such as Access Control Lists (ACLs) and Principal-Based Access Control (PBAC).
