java.util.AbstractMap.SimpleEntry

The java.util.AbstractMap.SimpleEntry class provides a skeletal implementation of the Map.Entry interface. This class simplifies the task for creating custom Entry objects.

A SimpleEntry object has two components: a key and a value. The key can be any object, while the value must be an object that implements the Map.Entry interface. The following example shows how to create a SimpleEntry:


          
           
SimpleEntry entry = new SimpleEntry(key, value);

Methods

The SimpleEntry class is defined by the following methods:

  • getKey()
    - Returns the key of the entry.
  • getValue()
    - Returns the value of the entry.
  • setKey(Object key)
    - Sets the key of the entry.

setValue(Object value)

  • equals(Object obj)

    - Returns a String representation of the entry.

The following example shows how to use the SimpleEntry class:


          
           
import java.util.AbstractMap; 
import java.util.Map; 

public class SimpleEntryExample { 

   public static void main(String[] args) { 

      // create an entry 
      AbstractMap.SimpleEntry<Integer, String> entry = 
              new AbstractMap.SimpleEntry<Integer, String>(1, "One"); 

      // get the key 
      System.out.println("Key: " + entry.getKey()); 

      // get the value 
      System.out.println("Value: " + entry.getValue()); 

      // set the value
