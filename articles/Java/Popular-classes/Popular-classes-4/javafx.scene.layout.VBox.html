JavaFX Scene Layout VBox

The JavaFX Scene Layout VBox is a component used to create a vertical layout for UI elements. It lays out its children within a single row that allows for a flexible way to space and align elements on the screen. With this component, you can arrange components and objects like text, images, and other UI elements in a column-like structure.

Example Usage

The following example demonstrates how to use the VBox component:


          
           
VBox vbox = new VBox();
vbox.setPadding(new Insets(10));
vbox.setSpacing(8);

Label title = new Label("VBox Example");
title.setFont(Font.font("Arial", FontWeight.BOLD, 14));
vbox.getChildren().add(title);

The VBox component has two layout constraints that affect how it sizes and positions its children:

  • fillWidth: When set to true, the child will stretch to fill the width of the VBox.
