javax.xml.transform.Result

javax.xml.transform.Result is an abstract class in Java, which defines a Destination for the data generated by a Transformer that can be a DOM node, an OutputStream, a java.io.Writer or even a filename. It allows applications to specify where Result objects are sent, and allows Transformer implementation to switch between different output methods.
