What is android.graphics.Bitmap.CompressFormat

android.graphics.Bitmap.CompressFormat is an enum which specifies the format of the image file to be compressed, such as PNG or JPEG. It can be used when creating a Bitmap to specify the type of compression to use when saving image data.

Examples of Usage

To create a Bitmap and set its compression format, you can use the following code:


          
           
Bitmap bmp = Bitmap.create(width, height, Bitmap.Config.ARGB_8888); 
bmp.compress(Bitmap.CompressFormat.JPEG, quality, outStream);

In the above example, we are creating a Bitmap with the desired configuration and then compressing it using the JPEG format with a given quality setting.

Other Options for Compression Format

In addition to the JPEG format, there are other options available for compression formats:

  • PNG - Portable Network Graphics format
  • WEBP - WebP format
  • GIF - Graphics Interchange Format

These additional formats can be used by replacing the 'JPEG' parameter in the example code with one of the other format names.

Conclusion

android.graphics.Bitmap.CompressFormat allows developers to easily specify the file format of a Bitmap before its data is saved, making it easier to store images in different formats without having to manually convert them.
