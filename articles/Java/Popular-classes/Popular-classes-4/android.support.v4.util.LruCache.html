It is a Least Recently Used (LRU) cache, meaning the objects it stores are ordered by the time they were last used and the least recently used objects are evicted first. To retrieve and item from the cache, use the get() method. It takes a key as an argument and returns the value associated with it. If the key does not exist in the cache, the method returns null:

When to Use android.support.v4.util.LruCache

android.support.v4.util.LruCache is most useful when an application needs to manage its memory usage carefully.
