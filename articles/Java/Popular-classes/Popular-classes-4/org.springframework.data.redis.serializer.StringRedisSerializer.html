Creating a StringRedisSerializer instance

To create an instance of a StringRedisSerializer, simply call the empty constructor: 

          
           
StringRedisSerializer stringRedisSerializer = new StringRedisSerializer();

Using StringRedisSerializer

redisTemplate.setValueSerializer(stringRedisSerializer);

String someString = "Some String value";
byte[] serializedString = stringRedisSerializer.serialize(someString);
redisTemplate.opsForValue().set("key", serializedString);
