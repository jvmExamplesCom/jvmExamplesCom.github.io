Downloading Files with Android's Download Manager

Android's Download Manager provides an efficient, reliable way to download files within your app. In this article, we'll look at how to use the Download Manager for downloading files in Android. We'll cover all the basics, as well as give code examples to help get your download running.

Getting Started

Using the Download Manager requires the following steps: 1. Create a Download Manager instance 2. Create a download request for a given URL 3. Execute the download request 4. Monitor the download progress

Creating a Download Manager Instance

To create an instance of the Download Manager, use the getSystemService() method from the Context class. A basic example is shown below: 

          
           
DownloadManager dm = (DownloadManager) getSystemService(Context.DOWNLOAD_SERVICE);

Creating a Download Request

To create a download request, you must provide the URL and destination folder where the file should be downloaded to. A basic example is shown below: 

          
           
DownloadManager.Request request = new DownloadManager.Request(url);
request.setDestinationInExternalPublicDir("/downloads", "filename");

Executing the Download Request

Once the download request is created, you can execute it using the enqueue() method. This method returns a long value which is the download ID. This ID is used to track the download progress. An example is shown below: 

          
           
long downloadID = dm.enqueue(request);

Monitoring the Download Progress

Once the download is executed, you can monitor the progress of the download using the DownloadManager class's query() method. This method takes a DownloadManager.Query object which specifies the columns you'd like to get back from the query. An example of this is shown below: 

          
           
DownloadManager.Query q = new DownloadManager.Query();
q.setFilterById(downloadID);

Cursor c = dm.query(q);
if (c.moveToFirst()) {
    //download progress
}
The query will return a number of columns including the following: 

          
           
STATUS_PENDING
STATUS_RUNNING
STATUS_PAUSED 
STATUS_SUCCESSFUL 
STATUS_FAILED
Each of these represents the current state of the download and can be used to check the progress of the download and determine what action needs to be taken.

Conclusion

In this article, we looked at how to use the Download Manager to download files in Android. We saw that it was relatively easy to setup and use, and that monitoring the progress of the download was also easy using the query() method.
