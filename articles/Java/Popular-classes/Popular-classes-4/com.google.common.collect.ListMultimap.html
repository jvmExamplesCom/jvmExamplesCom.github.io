Introduction to com.google.common.collect.ListMultimap

A ListMultimap is a specialized generic collection that allows a key to be associated with one or more values. It is provided as a part of the Google Guava library, and offers a convenient alternative to using a Map of Lists. In this article, we'll explore how to use ListMultimap in Java.

Creating a ListMultimap

The simplest way to create a ListMultimap instance is to use the factory method, 


          
           
ListMultimap<String,Integer> listMultimap = MultimapBuilder.treeListMultimap().build();
This will create a new ListMultimap instance with the specified types and parameters.

Adding and Removing Values

Adding values to a ListMultimap is similar to adding them to a regular Map. We can use the 


          
           
put()
method to add a key-value pair. For example: 

          
           
listMultimap.put("Fruits", "Apple");
listMultimap.put("Fruits", "Banana");
listMultimap.put("Vegetables", "Carrot");
We can also use the 

          
           
remove()
method to remove key-value pairs. For example: 

          
           
listMultimap.remove("Fruits", "Banana");

Retrieving Values

We can retrieve the list of values associated with a given key by calling the 


          
           
get()
method. For example: 

          
           
List<String> fruits = listMultimap.get("Fruits");

We can also retrieve all the values from the ListMultimap by calling the 


          
           
values()
method. This will return a Collection containing all the values in the ListMultimap. For example: 

          
           
Collection<String> allValues = listMultimap.values();

Iterating Over Entries

We can iterate over all the entries in the ListMultimap by calling the 


          
           
entries()
method. This will return a set of entries in the multimap, which we can then iterate over. For example: 

          
           
Set<Map.Entry<String,String>> entries = listMultimap.entries();
for (Map.Entry<String,String> entry : entries) {
    System.out.println(entry.getKey() + ": " + entry.getValue());
}

This will print out each key-value pair in the ListMultimap.

Conclusion
