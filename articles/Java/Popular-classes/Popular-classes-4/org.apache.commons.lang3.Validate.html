org.apache.commons.lang3.Validate

The 

          
           
org.apache.commons.lang3.Validate
class is a set of static utility methods which provides useful methods for validating object references, parameters and arguments in a program. Validate helps evaluate preconditions of a program, providing developers with a mechanism to enforce good programming practice and clear error messages in case of failure.

Methods

The Validation class provides multiple methods to assist with verification and validation of elements, including primitives (boolean, int, char, etc.), objects, collections, arrays, strings, enums and boolean expressions.

Validate.notNull()

Validate.notNull(Object obj) tests if the given object reference is not null. If the argument is null, an IllegalArgumentException is thrown. For example:


          
           
Validate.notNull(str, "String str must not be null");
Will throw an 

          
           
IllegalArgumentException
with the exception message provided as an argument. This is particularly helpful when debugging where the precise reason for the exception can be highlighted.

Validate.isTrue()

Validate.isTrue(boolean expression, String msg) checks if the given expression is true. If the expression is false, an 


          
           
IllegalArgumentException
is thrown with the given message. 

          
           
Validate.isTrue(num > 0, "Number must be greater than 0");

If num is less than or equal to 0, an IllegalArgumentException will be thrown with the message given as argument.

Validate.noNullElements()

Validate.noNullElements(Object[] array, String msg) checks if none of the elements of the given array are null. If any element is found to be null, an 


          
           
IllegalArgumentException
is thrown with the given message. 

          
           
Validate.noNullElements(strArray, "Array cannot contain null elements");

If any of the elements in strArray are null, an IllegalArgumentException will be thrown with the message given as argument.

Validate.allElementsOfType()

Validate.allElementsOfType(Collection collection, Class type, String msg) checks if all of the elements in the given collection are of the specified type. If any element is not of the specified type, an IllegalArgumentException is thrown with the given message.


          
           
Validate.allElementsOfType(list, Integer.class, "List must only contain integers");

If any of the elements in list are not of type Integer, an IllegalArgumentException will be thrown with the message given as argument.

Validate.inclusiveBetween()

Validate.inclusiveBetween(long start, long end, long value, String msg) checks if the value given is within the specified range (inclusive). If not within the range, an IllegalArgumentException is thrown with the given message.
