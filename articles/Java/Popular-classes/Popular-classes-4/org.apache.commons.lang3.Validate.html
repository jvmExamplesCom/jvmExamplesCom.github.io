org.apache.commons.lang3.Validate

The 

          
           
org.apache.commons.lang3.Validate
class is a set of static utility methods which provides useful methods for validating object references, parameters and arguments in a program. Methods

Validate.notNull(str, "String str must not be null");

with the exception message provided as an argument. This is particularly helpful when debugging where the precise reason for the exception can be highlighted.

Validate.isTrue(boolean expression, String msg) checks if the given expression is true. If the expression is false, an 


          
           
IllegalArgumentException
is thrown with the given message. 

          
           
Validate.isTrue(num > 0, "Number must be greater than 0");

Validate.noNullElements()

Validate.noNullElements(Object[] array, String msg) checks if none of the elements of the given array are null. If any element is found to be null, an 


          
           
IllegalArgumentException
