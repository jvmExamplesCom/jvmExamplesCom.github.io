Introduction to org.apache.shiro.subject.PrincipalCollection

It allows applications to easily and securely store and manage user data internally, without needing to rely on external database or storage systems.

A PrincipalCollection can be used to represent a user's identity, such as a username or email address, as well as custom information about the user such as an stored preferences. Applications can also use PrincipalCollection to store data associated with the user such as their role information.

PrincipalCollection principals = new SimplePrincipalCollection();
