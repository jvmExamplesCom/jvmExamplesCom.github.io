android.appwidget.AppWidgetProvider

android.appwidget.AppWidgetProvider is the base class for implementing an App Widget provider. AppWidgetProviders are responsible for the creation, update, and deletion of App Widgets. It provides several methods which can be overridden to receive updates on changes in state that happen asynchronously and there is also a method to handle broadcasts issued by the host app.
