What is javax.annotation.processing.Filer?

javax.annotation.processing.Filer is a core package in the Java Platform, Standard Edition (Java SE) that supports the creation of abstract representations of file and directory objects. The abstract representation of a file or directory can be used by the annotation processing tool to abstractly create, read and edit files and directories.

Classes in javax.annotation.processing.Filer

The javax.annotation.processing.Filer contains several classes for creating and manipulating abstract representations of files and directories:

  • FileObject: A representation of a file object.
  • JavaFileObject: A representation of a Java source file object.
  • Filer: A factory class for creating FileObjects and JavaFileObjects.

Examples

Creating a FileObject

The following example demonstrates how to create a FileObject using the Filer class:


          
           
Filer filer = Filer.create();
FileObject fileObject = filer.createObjectFromPath("/path/to/file");

Creating a JavaFileObject

The following example demonstrates how to create a JavaFileObject using the Filer class:


          
           
Filer filer = Filer.create();
JavaFileObject javaFileObject = filer.createJavaObjectFromPath("/path/to/java/file");

Reading a FileObject

The following example demonstrates how to read a FileObject using the Filer class:


          
           
Filer filer = Filer.create();
FileObject fileObject = filer.createObjectFromPath("/path/to/file");
String contents = fileObject.getContents();

Writing to a FileObject

The following example demonstrates how to write to a FileObject using the Filer class:


          
           
Filer filer = Filer.create();
FileObject fileObject = filer.createObjectFromPath("/path/to/file");
fileObject.write("Hello, world!");
