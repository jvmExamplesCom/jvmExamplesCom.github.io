Introduction to android.graphics.ColorMatrix

The android.graphics.ColorMatrix class is a class used to store and manipulate color matrix transformations in the Android operating system. It is used to create effects such as color tinting, saturation changes, brightness adjustments, contrast adjustments, and more. It can also be used to combine multiple transformations into a single matrix. Defining ColorMatrix Objects

A ColorMatrix object can be defined using a 4x4 matrix of float values. To create a new instance of it, use the following constructor: 

          
           
ColorMatrix cm = new ColorMatrix();
or the following constructor with arguments: 

          
           
ColorMatrix cm = new ColorMatrix(float[] matrix);
Where the matrix argument is a 4x4 array of floats representing the matrix's values.

Working with ColorMatrix Objects
