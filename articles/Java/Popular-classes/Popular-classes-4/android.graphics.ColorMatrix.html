Introduction to android.graphics.ColorMatrix

Defining ColorMatrix Objects

ColorMatrix cm = new ColorMatrix();
or the following constructor with arguments: 

          
           
ColorMatrix cm = new ColorMatrix(float[] matrix);
Where the matrix argument is a 4x4 array of floats representing the matrix's values.
