Introduction to com.sun.net.httpserver.HttpServer in Java

Java developers use the class 

          
           
com.sun.net.httpserver.HttpServer
when creating a simple lightweight HTTP server. It can be used for testing web applications and services that are implemented using Java technology, as well as for serving application data over the web. This tutorial will explain how to set up and use com.sun.net.httpserver.HttpServer.

Installing and Setting Up HttpServer

To utilize com.sun.net.httpserver.HttpServer, you must have the Java Development Kit (JDK) installed on your machine. In many cases, downloading and installing the JDK is necessary. The namespace to import for the HttpServer class is 

          
           
java.net.*
. An example of how to import this is: 

          
           
import java.net.*;

Using HttpServer in Code

To create an instance of the HttpServer class, you must specify the port and host address. You can also assign a server name to the instance. An example of creating an instance of HttpServer is 

          
           
HttpServer server = HttpServer.create(new InetSocketAddress("127.0.0.1", 8888), 0);
You can add handlers to your server to handle requests and responses. To add a handler, use the 

          
           
server.createContext(String path, HttpHandler handler)
method. The first parameter is the URL path to call the handler, and the second parameter is the handler instance. An example of adding a handler is: 

          
           
server.createContext("/", new TestHandler());
The final step is to start the server: 

          
           
server.start();

Example of a Simple HttpServer With a Handler

Below is an example of a simple program that creates a server and will respond to requests with a "Hello World" message. 

          
           
import java.io.*;
import java.net.*;
import com.sun.net.httpserver.*;  

public class HttpServerExample {
  public static void main(String[] args) throws IOException {

    // Create an HttpServer and set the port to 8888
    HttpServer server = HttpServer.create(new InetSocketAddress(8888), 0);

    // Create a context for the server on "/"
    server.createContext("/", new TestHandler());

    // Start listening to requests
    server.start();
    System.out.println("Server started on port 8888.");
  }
}

class TestHandler implements HttpHandler {
  @Override
  public void handle(HttpExchange exchange) throws IOException {

    // Set response header
    Headers responseHeaders = exchange.getResponseHeaders();
    responseHeaders.set("Content-Type", "text/plain");

    // Write response to the client
    exchange.sendResponseHeaders(200, 0);
    OutputStream response = exchange.getResponseBody();
    response.write("Hello World!".getBytes());

    // Close the connection
    response.close();
  }
}
When you run this code, it will start a server on port 8888 and any request made to that port will result in a response of “Hello World!”.
