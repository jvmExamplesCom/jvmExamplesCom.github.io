Overview

org.apache.zookeeper.KeeperException is an exception that is thrown when an error occurs in the ZooKeeper framework. It provides an enumeration for some of the errors that can occur and a getCode() method to access the actual error type. The exception is also used as a base class when additional specific error types are added.
