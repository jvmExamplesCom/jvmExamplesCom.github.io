Introduction to org.dom4j.io.SAXReader

org.dom4j.io.SAXReader is an API in the DOM4J library for reading various XML related formats like SAX, StAX, DOM4J, and JDOM. This API allows for parsing of XML documents and obtaining content on a particular node or element of the document. In this article, we will take a look at how to use SAXReader to parse XML documents and obtain content.

Prerequisites

Before proceeding further, make sure that you have basic knowledge of XML and its components like XML tags, elements, attributes, etc. Also, you need to have a good understanding of Java, as all code examples are written in Java.

Creating a SAXReader Object

We can create a SAXReader object with the 

          
           
new SAXReader()
syntax.

Parsing an XML Document

Once we create a SAXReader object, we can use it to parse an XML document. This can be done using the 


          
           
read(InputStream inputStream)
method. This method takes an 

          
           
InputStream
object as an argument and returns a 

          
           
Document
object representing the XML document.

Obtaining Content from an Element

Once we obtain a 

          
           
Document

getStringValue()
method which takes an 

          
           
Element
object as an argument and returns the content of the element as a 

          
           
String
.

Example

Let's take a look at an example to understand how SAXReader works. Consider the following XML document:


          
           
<root>
  <person>
    <name>John</name>
    <age>25</age>
  </person>
  <person>
    <name>Jane</name>
    <age>30</age>
  </person>
</root>

getStringValue()
method to obtain the name of the person. The code for this would look like this: 

          
           
SAXReader reader = new SAXReader();

SAXReader

name
