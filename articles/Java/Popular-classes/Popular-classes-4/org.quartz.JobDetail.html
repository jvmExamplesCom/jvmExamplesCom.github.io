org.quartz.JobDetail

The org.quartz.JobDetail class is a Java class that provide the ability to define jobs and their associated detail information. The class allows for Jobs to be written in Java, rather than relying on configuration files. This makes for a more programmatic approach to developing Job definitions. The class also has a built in mechanism for scheduling of jobs.

The core of the org.quartz.JobDetail class is the definition of the Job itself. This can be done by implementing the org.quartz.Job interface, or by extending the org.quartz.JobDetail class and overriding the execute() method.

JobDetail jobDetail = new JobDetail("myJob", MyJob.class);
