Introduction to the com.intellij.openapi.project.Project in Java

The 

          
           
com.intellij.openapi.project.Project

It is the entry point for any plugin that wants to access or extend information associated with the project, and it also contains the majority of methods needed to work with the project structure.

Project
in an action: 

          
           
public class MyAction extends AnAction {
   public void actionPerformed(AnActionEvent e) {
      Project project = e.getProject();
   }
}
