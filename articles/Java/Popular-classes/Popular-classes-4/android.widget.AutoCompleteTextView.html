android.widget.AutoCompleteTextView

The android.widget.AutoCompleteTextView class is a view that provides auto-completion capability to text entry. It displays a list of suggestions while the user is typing, and allows the user to select an item from the list. It can be used to provide autocomplete functionality in many different applications, including web search and contact lists.

Implementation

To use an AutoCompleteTextView, first add it to a layout file:


          
           
<android.widget.AutoCompleteTextView
  android:id="@+id/autocomplete_view"
  android:layout_width="match_parent"
  android:layout_height="wrap_content" />

The AutoCompleteTextView should then be initialized in the activityâ€™s onCreate method:


          
           
AutoCompleteTextView autocompleteView = (AutoCompleteTextView) findViewById(R.id.autocomplete_view);

The data source to be used for the auto complete list must then be set. This can be an array of strings, or a Cursor:


          
           
String[] data = new String[] { "Foo", "Bar", "Baz" };
ArrayAdapter<String> adapter =
    new ArrayAdapter<String>(this, android.R.layout.simple_list_item_1, data);
autocompleteView.setAdapter(adapter);

The above code sets the data source to an array of strings. If a Cursor is used instead, the data for each suggestion will come from different columns in the cursor. For example, if the Cursor contains columns for name, address and phone number, the auto complete list might show a name and address for each suggestion.

Optionally, a filter can be set up to control which items appear in the auto complete list. For example, a filter could be used to ensure that only contacts with an email address are shown in the list:


          
           
CursorFilter filter = new CursorFilter() {

  }
};

Finally, a listener can be set up to be notified when a selection is made from the auto complete list:


          
           
autocompleteView.setOnItemClickListener(new OnItemClickListener() {
  @Override
  public void onItemClick(AdapterView<?> parent, View view, int position, long id) {
    // Handle the selection
  }
});
