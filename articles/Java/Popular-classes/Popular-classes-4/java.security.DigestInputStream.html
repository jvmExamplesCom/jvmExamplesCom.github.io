java.security.DigestInputStream

Java.security.DigestInputStream is an abstract class that provides applications the functionality of a message digest algorithm, such as MD5 or SHA-1, on an InputStream. It provides a mechanism for calculating a Message Digest for data read from an input stream, in order to verify the data integrity. It does so by calculating the message digest value over the incoming data and comparing it against the specified digest value.

public DigestInputStream(InputStream is, MessageDigest md)
public DigestInputStream(InputStream is, MessageDigest md, boolean skipCheck)
The first constructor creates a DigestInputStream and binds it to the specified InputStream and MessageDigest object. The second constructor creates a DigestInputStream and binds it to the specified InputStream and MessageDigest object with the option to skip the check.

The "isOn" method returns true if the digest calculation is enabled, false otherwise. The "getDigest" method returns the current digest value.
