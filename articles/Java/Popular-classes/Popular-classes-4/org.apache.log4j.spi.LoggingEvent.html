org.apache.log4j.spi.LoggingEvent

The 

          
           
org.apache.log4j.spi.LoggingEvent
class is a Java class that is used to represent a logging event in the Log4j 2 framework. It provides information about the level, logger name, and message associated with a given logging event. The logging event also provides access to any parameters of the log statement as well as the time stamp and thread name of the logging event.

Creating a LoggingEvent Instance

A LoggingEvent instance can be created by calling the 


          
           
LoggingEvent(String fqcn, Logger logger, Level level, Message data, Throwable t)
constructor of the LoggingEvent class. The constructor takes five arguments: 
  • fqcn
    - Fully qualified class name of the category of the LoggingEvent.
  • logger
    - Logger associated with the event.
  • level
    - Level of the event.
  • data
    - The message associated with the logging event.
  • t
    - Throwable associated with the logging event (if any).

Methods of the LoggingEvent Class

The LoggingEvent class has several methods for getting information about the logging event. These are the main methods:

  • getFQCN()
    - Returns the fully qualified class name of the category of the LoggingEvent.
  • getLevel()

    Level

Example Usage

// create a LoggingEvent instance
