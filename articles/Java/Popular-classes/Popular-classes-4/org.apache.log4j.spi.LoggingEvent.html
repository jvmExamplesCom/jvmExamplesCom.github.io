org.apache.log4j.spi.LoggingEvent

The 

          
           
org.apache.log4j.spi.LoggingEvent
class is a Java class that is used to represent a logging event in the Log4j 2 framework. It provides information about the level, logger name, and message associated with a given logging event. The logging event also provides access to any parameters of the log statement as well as the time stamp and thread name of the logging event.

Creating a LoggingEvent Instance

A LoggingEvent instance can be created by calling the 


          
           
LoggingEvent(String fqcn, Logger logger, Level level, Message data, Throwable t)
constructor of the LoggingEvent class. The constructor takes five arguments: 
  • fqcn
    - Fully qualified class name of the category of the LoggingEvent.
  • logger
    - Logger associated with the event.
  • level
    - Level of the event.
  • data
    - The message associated with the logging event.
  • t
    - Throwable associated with the logging event (if any).

Methods of the LoggingEvent Class

The LoggingEvent class has several methods for getting information about the logging event. These are the main methods:

  • getFQCN()
    - Returns the fully qualified class name of the category of the LoggingEvent.
  • getLevel()
    - Returns the 
    Level
    of the logging event.
  • getLoggerName()
    - Returns the name of the logger associated with the logging event.
  • getMessage()
    - Returns the message associated with the logging event.
  • getParameters()
    - Returns an array of 
    Object
    s that were passed into the log statement as parameters.
  • getTimeStamp()
    - Returns the timestamp of the logging event as a 
    long
    .
  • getThreadName()
    - Returns the name of the thread that executed the logging statement.
  • getThrowable()
    - Returns the 
    Throwable
    associated with the logging event (if any).

Example Usage

Let's look at a simple example that demonstrates how the LoggingEvent class can be used. First, we'll create an instance of the LoggingEvent class with some sample data:


          
           
// create a LoggingEvent instance
LoggingEvent event = new LoggingEvent("org.example.MyClass", 
                                      LogManager.getLogger("example"), 
                                      Level.INFO, 
                                      "This is my message",
                                      null);

Now, let's get some information from the LoggingEvent instance:


          
           
// get the fully qualified class name
String className = event.getFQCN();
System.out.println("Class name: " + className);

// get the logger name
String loggerName = event.getLoggerName();
System.out.println("Logger name: " + loggerName);

// get the level
Level level = event.getLevel();
System.out.println("Level: " + level);

// get the message
String message = event.getMessage();
System.out.println("Message: " + message);

// get the timestamp
long timestamp = event.getTimeStamp();
System.out.println("Timestamp: " + timestamp);

// get the thread name
String threadName = event.getThreadName();
System.out.println("Thread Name: " + threadName);

// get the throwable (if any)
Throwable t = event.getThrowable();
System.out.println("Throwable: " + t);
