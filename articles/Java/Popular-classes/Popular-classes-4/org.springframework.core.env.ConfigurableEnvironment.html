ConfigurableEnvironment is a specialized implementation of the Environment interface that adds support for resolving placeholders in property values, profile-specific property sources, and unit testing support.

To create an instance of a ConfigurableEnvironment, you can use the new StandardEnvironment() constructor or the AbstractEnvironment() constructor. Both of these constructors accept configuration options as a parameter:


          
           
ConfigurableEnvironment env = new StandardEnvironment();
env.setActiveProfiles("development");

env.getProperty("my.property.name");

String requiredPropertyValue = env.getRequiredProperty("my.property.name");
