What is Java?

Java is an object-oriented programming language designed by James Gosling and developed at Sun Microsystems (a part of Oracle Corporation). It was released in 1995 as a core component of Sun Microsystems' Java platform. Java is considered to be one of the most popular programming languages for development of software applications for computers, web servers and mobile devices.

Java Libraries and Frameworks

Java has a large library of packages and classes that you can use to develop complex applications.
