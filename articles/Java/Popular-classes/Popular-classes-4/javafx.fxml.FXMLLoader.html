JavaFX.fxml.FXMLLoader

JavaFX.fxml.FXMLLoader is a powerful class in JavaFX that helps developers create complex graphical user interfaces (GUIs) with minimum effort and in no time. It allows developers to easily create JavaFX applications by loading FXML files, which are XML files that contain the layout of the application.

Creating an FXML Loader

The first step to using the FXMLLoader is to create it:


          
           
FXMLLoader loader = new FXMLLoader();

Once the loader is created, it can be used to load an FXML file.

Loading an FXML File

The load() method is used to load the FXML file. It takes a single parameter, which is a URL object pointing to the FXML file:


          
           
URL url = getClass().getResource("mylayout.fxml");
loader.load(url);

Once the FXML file is loaded, the controller associated with the FXML can be accessed.

Accessing the Controller

The FXMLLoader.getController() method can be used to access the controller associated with an FXML file. This method returns an object of type Object, so it must be casted to the appropriate type:


          
           
MyController controller = (MyController)loader.getController();

Once the controller is accessed, its methods and fields can be used to control the behavior of the application.

example


          
           
public static void main(String[] args) {
    FXMLLoader loader = new FXMLLoader();
    URL url = MyApp.class.getResource("mylayout.fxml");
    loader.load(url);

    MyController controller = (MyController) loader.getController();
    controller.doSomething();
}
