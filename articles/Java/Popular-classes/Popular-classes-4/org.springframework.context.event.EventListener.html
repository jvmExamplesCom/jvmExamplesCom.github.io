org.springframework.context.event.EventListener
annotation is a powerful tool that enables developers to subscribe to application events using event listener methods. In addition, the annotation provides support for implementing asynchronous event processing, such as scheduling of the event and preparation of a message.

EventListener
annotation, an interface must be defined and then implemented by the listening object. The interface should define all the parameters expected by the event listener method.
