java.io.CharArrayWriter

Java.io.CharArrayWriter is a class that enables writing to a character array buffer. The buffer increases as data is written to it. When the buffer is full, its size is automatically increased in order to accommodate more data. It also provides various methods for handling output such as toString(), reset(), close(), writeTo(), flush() and many more.

Methods

toString()

The toString() method is used to convert the contents of a CharArrayWriter to a String. The following example illustrates this:


          
           
CharArrayWriter writer = new CharArrayWriter(); 
writer.write("Hello World!"); 
String str = writer.toString(); 
System.out.println(str); 

// Output: Hello World!

reset()

The reset() method is used to reset the internal buffer, which empties the contents of CharArrayWriter and can be used to start writing again from the begining.


          
           
CharArrayWriter writer = new CharArrayWriter(); 
writer.write("Hello World!"); 
writer.reset(); 
writer.write("Hello Universe!"); 
String str = writer.toString(); 
System.out.println(str); 

// Output: Hello Universe!

close()

The close() method is used to close the stream, which will release any resources associated with it. This method does not have any effect on the internal buffer, which can still be used for writing.


          
           
CharArrayWriter writer = new CharArrayWriter(); 
writer.write("Hello World!"); 
writer.close(); 
String str = writer.toString(); 
System.out.println(str); 

// Output: Hello World!

writeTo()

The writeTo() method is used to write the contents of CharArrayWriter to another stream. This method accepts a Writer object as an argument, which can be either a FileWriter or BufferedWriter object. Following example illustrates this:


          
           
try { 
	CharArrayWriter writer = new CharArrayWriter(); 
	writer.write("Hello World!"); 
	writer.writeTo(new FileWriter("test.txt")); 
	System.out.println("Data written to File"); 
} catch (Exception e) { 
	System.out.println(e); 
} 

// Output: Data written to File

flush()

The flush() method is used to flush the internal buffer. It does not do anything, since the internal buffer is automatically flushed when its size exceeds the capacity.


          
           
CharArrayWriter writer = new CharArrayWriter(); 
writer.write("Hello World!"); 
writer.flush(); 
String str = writer.toString(); 
System.out.println(str); 

// Output: Hello World!
