java.text.Format

Java.text.Format is a class in the Java programming language which provides a general-purpose formatter for displaying values of type java.lang.Object. The format method returns an instance of java.text.Formatter which can be used to parse, format and convert values. It also provides methods for formatting currency, dates, times and numbers. This class also provides support for localized patterns.

Code Examples

The following examples illustrate the use of some of the Java.text.Format methods:


          
           
String pattern = "##0.00";

// Format a number
DecimalFormat myFormatter = new DecimalFormat(pattern);
String output = myFormatter.format(123456.789);

System.out.println(output); // Outputs "123456.79"

          

          
           
// Format a date
SimpleDateFormat dateFormatter = new SimpleDateFormat("yyyy-MM-dd");
String dateString = dateFormatter.format(new Date());

System.out.println(dateString); // Outputs "2021-05-02"

          

          
           
// Format a currency
NumberFormat currencyFormatter = NumberFormat.getCurrencyInstance();
String currencyString = currencyFormatter.format(12345.67);

System.out.println(currencyString); // Outputs "$12,345.67"

          

          
           
// Format a time
DateFormat timeFormatter = new SimpleDateFormat("hh:mm:ss a");
String timeString = timeFormatter.format(new Date());

System.out.println(timeString); // Outputs "08:44:38 AM"
