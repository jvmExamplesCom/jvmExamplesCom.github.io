java.io.PushbackInputStream

The java.io.PushbackInputStream class is a subclass of FilterInputStream, providing the ability to "push back" or "unread" one byte to the stream. It allows the containing application to read and process the byte, and then push it back so that subsequent reads will retrieve it again.

Example 1

To create a PushbackInputStream, usually an existing InputStream must be passed into the constructor. 


          
           
InputStream in = new FileInputStream(file);
PushbackInputStream pbIn = new PushbackInputStream(in);

InputStream in = new FileInputStream(file);
PushbackInputStream pbIn = new PushbackInputStream(in, bufferSize);

pbIn.unread(c);
