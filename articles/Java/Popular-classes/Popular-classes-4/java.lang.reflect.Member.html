Introduction to java.lang.reflect.Member

Java's reflection API enables the programmer to obtain information about classes and objects, and perform operations on them dynamically. This includes the ability to access fields, methods, and constructors of a class, and obtain information such as parameter names, generic type parameters, and modifiers. The 


          
           
java.lang.reflect.Member
is an abstract class that serves as the superclass of the classes 
Field
, 
Method
, and 
Constructor
. It contains several useful methods for obtaining and accessing members of the Java language.

Methods of the java.lang.reflect.Member Class

  • getModifiers(): This method returns an integer that contains the access modifier flags corresponding to the declared elements (field, method, or constructor). The value of these flags is either 0 (public) , 1 (private), 2 (protected), 4 (static), 8 (final), 16 (synchronized), 32 (volatile) or 64 (transient).
  • getName(): This method returns the name of the declared element (field, method or constructor) as a 
    String
    .
  • isSynthetic(): This method returns 
    true
    if the declared element was synthesized by the compiler; otherwise it returns 
    false
    .
  • getDeclaringClass(): This method returns the 
    Class
    object representing the class that declares this member.

Example Usage of java.lang.reflect.Member

The following example demonstrates how to use the 

          
           
java.lang.reflect.Member
class to obtain information about a given 
Method
: 

          
           
import java.lang.reflect.Method;

public class Example {

    public static void main(String[] args) throws Exception {
        Method m = Example.class.getMethod("add", int.class, int.class);

        System.out.println("The name of the declared Method is: " + m.getName());
        System.out.println("The modifiers of the declared Method are: " + m.getModifiers());
        System.out.println("The declaring Class of the declared Method is: " + m.getDeclaringClass());
        System.out.println("The declared Method is synthetic: " + m.isSynthetic());
    }

    public static int add(int x, int y) {
        return x + y;
    }
}
