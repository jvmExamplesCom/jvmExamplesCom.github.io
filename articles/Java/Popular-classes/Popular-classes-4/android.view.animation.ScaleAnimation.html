android.view.animation.ScaleAnimation

android.view.animation.ScaleAnimation is a subclass of android.view.animation.Animation that performs a scale animation, which can enlarge or shrink the size of an object. This class lets you specify the start and end size of the object, along with the duration of the animation, either in the constructor or using the setters.

Creating an Animation Instance

To create an instance of ScaleAnimation you need to provide the following parameters:
  • fromXScale – The initial X-axis scaling factor.
  • toXScale – The final X-axis scaling factor.
  • fromYScale – The initial Y-axis scaling factor.
  • toYScale – The final Y-axis scaling factor.
  • pivotX – The X-coordinate of the point about which the object is scaled, specified as an absolute number where 0 is the left edge.
  • pivotY – The Y-coordinate of the point about which the object is scaled, specified as an absolute number where 0 is the top edge.
For example: 

          
           
ScaleAnimation scaleAnimation = new ScaleAnimation(
    1.0f, 0.5f, // fromXScale, toXScale
    1.0f, 0.5f, // fromYScale, toYScale
    0.0f, 0.0f // pivotX, pivotY 
 );
This would create a ScaleAnimation that shrinks an object by half on both the X-axis and the Y-axis, relative to the origin point (0,0).

Setting the Duration

By default, the duration of the animation is set to 0 milliseconds, so you will likely want to change this in order to get the desired effect. You can do this either in the constructor or via the setDuration(long) method. For example: 

          
           
scaleAnimation.setDuration(2000); // Animates for 2 seconds

Starting the Animation

Once you have created the ScaleAnimation, you will need to start it in order to see the animation in action. To do this, call the start() method on the animation object. For example: 

          
           
scaleAnimation.start();

Example

The following example demonstrates how to use the ScaleAnimation class to fade an image in and out over 2 seconds: 

          
           
ImageView imageView = findViewById(R.id.image);

float fromScale = 0.0f;
float toScale = 1.0f;

ScaleAnimation scaleAnimation = new ScaleAnimation(
    fromScale, toScale, // fromXScale, toXScale
    fromScale, toScale, // fromYScale, toYScale
    0.0f, 0.0f // pivotX, pivotY 
);
scaleAnimation.setDuration(2000);
imageView.startAnimation(scaleAnimation);
When run, the above code will cause the image to enlarge from 0% to 100% of its original size over a period of 2 seconds.
