java.util.NavigableSet

Java provides the NavigableSet interface to represent ordered sets, which is a subtype of the Set interface. NavigableSet extends SortedSet with methods for efficiently navigating elements within a sorted set structure. In this article, we will explore the usage of the NavigableSet API and delve into its implementation details.

Creating a NavigableSet Instance

NavigableSet is an interface, so it provides factory methods to create new instances. These factory methods are provided by the TreeSet class, which implements NavigableSet and uses a red-black tree to maintain elements in sorted order. The following code creates a new TreeSet instance:


          
           
TreeSet<Integer> set = new TreeSet<>();

We can also provide a comparator for elements that do not implement Comparable:


          
           
Comparator<Integer> comparator = (x, y) -> Integer.compare(x, y);
TreeSet<Integer> set = new TreeSet<>(comparator);

Navigation Methods

The NavigableSet interface provides several methods for navigating elements in the sorted set. The simplest of these methods are first and last, which return the smallest and largest element in the set respectively. The following example demonstrates their usage:


          
           
TreeSet<Integer> set = new TreeSet<>();
set.add(3);
set.add(1);
set.add(4);

// Smallest element in the set
System.out.println(set.first()); // 1

// Largest element in the set
System.out.println(set.last()); // 4

The higher and lower methods return the smallest element greater than or equal to a given element, and the largest element less than or equal to a given element, respectively. The following example demonstrates their usage:


          
           
TreeSet<Integer> set = new TreeSet<>();
set.add(3);
set.add(1);
set.add(4);

// Returns 4
System.out.println(set.higher(3));

// Returns 3
System.out.println(set.lower(4));

The ceiling and floor methods are similar to the higher and lower methods but they return the given element if it exists in the set. The following example demonstrates their usage:


          
           
TreeSet<Integer> set = new TreeSet<>();

The pollFirst and pollLast methods are similar to the first and last methods but they also remove the returned element from the set. The following example demonstrates their usage:

Finally, the descendingSet method returns a NavigableSet instance with the elements in reverse order. The following example demonstrates its usage:
