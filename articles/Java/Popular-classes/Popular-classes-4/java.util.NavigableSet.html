Java provides the NavigableSet interface to represent ordered sets, which is a subtype of the Set interface. NavigableSet extends SortedSet with methods for efficiently navigating elements within a sorted set structure. Creating a NavigableSet Instance

NavigableSet is an interface, so it provides factory methods to create new instances. These factory methods are provided by the TreeSet class, which implements NavigableSet and uses a red-black tree to maintain elements in sorted order. TreeSet<Integer> set = new TreeSet<>();

Comparator<Integer> comparator = (x, y) -> Integer.compare(x, y);

Navigation Methods

The NavigableSet interface provides several methods for navigating elements in the sorted set. The simplest of these methods are first and last, which return the smallest and largest element in the set respectively. The following example demonstrates their usage:


          
           
TreeSet<Integer> set = new TreeSet<>();
set.add(3);
set.add(1);

// Smallest element in the set
System.out.println(set.first()); // 1

// Largest element in the set
System.out.println(set.last()); // 4
