Object invoke(MethodInvocation invocation) throws Throwable;
The first parameter is a MethodInvocation object, which provides access to the method being invoked. The second and third parameter are the parameters being passed to the method. Once the MethodInterceptor.invoke() method has been completed, the call returns to the original method as usual.
