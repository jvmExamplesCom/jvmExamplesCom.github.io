javax.persistence.Enumerated

javax.persistence.Enumerated is an annotation used to specify how the database should represent an enumerated type in the database. It is part of the Java Persistence API (JPA) and can be used to define the mapping between an enumerated type and the corresponding database type. This annotation can be applied to a field or property that holds the enumerated type.

Usage

The @Enumerated annotation can be used when defining an entity class. It can be used to define the mapping for an enumerated type. The following code example demonstrates how to use this annotation: 

          
           
@Entity
public class Employee {

    @Id
    private Long id;

    @Enumerated(EnumType.STRING)
    private EmploymentStatus status;
    
    //...
}
In this example, the field "status" is of type EmploymentStatus , which is an enumerated type. The @Enumerated annotation is used to specify how the value of the field should be mapped to the database. The EnumType.STRING argument specifies that the value of the field should be stored as a String in the database. The EnumType.ORDINAL argument can also be used, which stores the value of the field as an ordinal number (an integer). This is the default behavior if no argument is specified.
