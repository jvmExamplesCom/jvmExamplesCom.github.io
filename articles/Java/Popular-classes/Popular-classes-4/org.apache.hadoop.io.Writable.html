org.apache.hadoop.io.Writable

The org.apache.hadoop.io.Writable class is a Java library from Apache Hadoop that provides an interface for objects that are serializable and deserializable. It is the base interface for Hadoop's serialization framework, which means that all serialization and deserialization classes must implement Writable. This framework is designed to offer a scalable and efficient way of storing data in a distributed environment.

The Writable interface contains two important methods: write() and readFields(). The write() method is used to serialize an object, while the readFields() method is used to deserialize it. The following code example shows how to implement the Writable interface on a custom Person class, which contains a name (String) and an age (int):


          
           
public class Person implements Writable {
    private String name;
    private int age;

    public Person(){}

    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }

    @Override
    public void write(DataOutput out) throws IOException {
        out.writeUTF(name);
        out.writeInt(age);
    }

    @Override
    public void readFields(DataInput in) throws IOException {
        name = in.readUTF();
        age = in.readInt();
    }
}

The write() method writes the object's fields to the DataOutput stream, while the readFields() method reads them from the DataInput stream. Additionally, we have added two constructors to the class. You should also note that the write() and readFields() methods should be declared public, otherwise they will not be properly accessed by the Hadoop framework.
