javax.swing.border.TitledBorder

TitledBorder is a class of Java Swing package which provides an implementation for border with a title along with it, providing visual separation and decoration to components. The tiles of TitledBorder can have different fonts, alignment, etc. and can be constructed using several constructors.

Constructors

The constructors to create TitledBorder are as follows:

TitledBorder(Border border, String title)
TitledBorder(Border border, String title, int titleJustification, int titlePosition)

Example

The following code creates an instance of TitledBorder which has a BeveledBorder and the title “My Title” with left justified alignment and is placed above the component:


          
           
Border b = new BeveledBorder();
