org.springframework.context.MessageSource

org.springframework.context.MessageSource is a powerful interface that provides access to message resources. It is used to retrieve internationalized messages in a programmatic way using the locale of the current user. The MessageSource also allows for resolving arguments such as variables in messages as well as supports formatting techniques like string formatting, number and date formatting.

Example Code

This example code shows how you can use the MessageSource to get an internationalized message from a resource bundle file:


          
           
// get the message source
MessageSource messageSource = getMessageSource();

// get the internationalized message text
String messageText = messageSource.getMessage("greeting.message", null, Locale.US);

// print out the text
System.out.println(messageText);

The result of this code will be the internationalized greeting message with its text determined by the 

Locale.US
parameter.

Resolving Arguments

The MessageSource also allows for resolving arguments within the internationalized messages. For example, you can insert a variable into a message that will be resolved with its value when the message is retrieved:


          
           
// get the message source
MessageSource messageSource = getMessageSource();

// get the internationalized message text
Object[] args = {"John"};
String messageText = messageSource.getMessage("name.message", args, Locale.US);

// print out the text
System.out.println(messageText);

The result of this code will be the internationalized greeting message with its text including the name of the person passed as the argument in the 

args
variable.

Formatting Messages

The MessageSource also supports other formatting options including string formatting, number and date formatting. For example, you can use the same message sourcing method to format a date in a specific locale:


          
           
// get the message source
MessageSource messageSource = getMessageSource();

// set the date to format
Date date = new Date();

// format the date using the US locale
Object[] args = {date};
String messageText = messageSource.getMessage("date.message", args, Locale.US);

// print out the text
System.out.println(messageText);

The result of this code will be the formatted date according to the 

Locale.US
.
