org.apache.zookeeper.data.Stat

The org.apache.zookeeper.data.Stat class is a lightweight way to access basic information about a particular node in ZooKeeper. Stat objects are immutable and contain data such as the nodeâ€™s version, creation time, last modified time, etc.

Constructors

The Stat class contains two main constructors:
  • Stat(int version, long ctime, long mtime, int cversion, int dataLength, long numChildren, long pzxid)
  • Stat()

The first constructor takes in all the data necessary to construct a Stat object, while the second constructor creates an empty object.

Methods

The following are some of the methods available for a Stat object:

  • getVersion(): Gets the version of the data contained in the Zookeeper node.
  • getCtime(): Gets the creation time of the Zookeeper node.
  • getMtime(): Gets the time of the last modification made to the Zookeeper node.
  • getCversion(): Gets the current version number of the node.
  • getDataLength(): Gets the length of the data stored in the Zookeeper node.
  • getNumChildren(): Gets the number of children that the Zookeeper node has.
  • getPzxid(): Gets the last transaction identifier associated with the Zookeeper node.

System.out.println("Version: " + stat.getVersion());
System.out.println("Creation Time: " + stat.getCtime());
System.out.println("Last Modification Time: " + stat.getMtime());
