Java AWT Event FocusListener

The java.awt.event.FocusListener is an interface that handles events related to the focus of Component objects. It contains two methods, one for when a Component gains focus and the other for when a Component loses focus.

How To Create A FocusListener

If you want to create a FocusListener you have to define a class that implements the FocusListener interface.


          
           
public class MyFocusListener implements FocusListener {

@Override
public void focusGained(FocusEvent e) {
    //What to do when a Component gains focus
}

@Override
public void focusLost(FocusEvent e) {
    //What to do when a Component loses focus
}
}

How To Use FocusListener

Once implemented, it is necessary to create an instance of the class to use the FocusListener. This instance can then be attached to a Component object.


          
           
//Initialize Components
JTextField myTextField = new JTextField();
JButton myButton = new Button();

//Create MyFocusListener object
MyFocusListener myFocusListener = new MyFocusListener();

//Attach FocusListener to Components
myTextField.addFocusListener(myFocusListener);
myButton.addFocusListener(myFocusListener);

Methods of FocusListener

The FocusListener interface contains two methods, both of which are invoked before any changes are made to the Component. This allows the developer to check the state of the Component before making any changes.

focusGained()

The focusGained() method is called when a Component gains focus. The following code example prints out a message in the console whenever this occurs.

The focusLost() method is called when a Component loses focus. The following code example prints out a message in the console whenever this occurs.
