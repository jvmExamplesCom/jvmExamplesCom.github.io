javafx.beans.value.ChangeListener
interface is a part of the JavaFX API that is used when dealing with object properties, particularly when they are changed. ChangeListener<String> listener = new ChangeListener<String>() {

    public void changed(ObservableValue<? extends String> source, String oldValue, String newValue) {
