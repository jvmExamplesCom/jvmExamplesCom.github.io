org.glassfish.jersey.server.ResourceConfig

The org.glassfish.jersey.server.ResourceConfig class is a Java configuration class used to register components (such as resources, providers and features) to be deployed to web applications. This class provides access to the underlying Jersey runtime environment and acts as a bridge between the Java world and the web application. It also allows for customizations of the Jersey runtime for different web applications.

ResourceConfig config = new ResourceConfig();

ResourceConfig config = ResourceConfig.builder()
        .build();

ResourceConfig config = ResourceConfig.forEnvironment(env);

Resource resource = new Resource();
