android.database.ContentObserver

The android.database.ContentObserver class is a listener for data changes in content provider URIs. This class is used to watch for changes that are made to content URIs, and can be used to get notified when the content of the URI is changed. ContentObservers can be registered by calling the registerContentObserver() method on a ContentResolver instance. The ContentObserver will then be notified each time the content at the specified URI changes.

Example

ContentResolver contentResolver = getContentResolver(); 
contentResolver.registerContentObserver(userUri, true, myObserver);
