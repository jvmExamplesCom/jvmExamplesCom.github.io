Java.time.format.DateTimeFormatterBuilder

Java.time.format.DateTimeFormatterBuilder is a utility class that can be used to create complex DateTimeFormatters for formatting and parsing dates and times. It provides an API to build a DateTimeFormatter from simple building blocks consisting mainly of DateTimeFormat symbols, such as day-of-week, month-of-year and others. DateTimeFormatters built with this class are thread-safe and immutable, and can be used in multi-threaded applications without synchronization.

Creating a DateTimeFormatter

The DateTimeFormatterBuilder class provides a fluent API for creating DateTimeFormatters. The following code example shows how to create a DateTimeFormatter that formats date and time using ISO-8601 format:


          
           
DateTimeFormatter formatter = new DateTimeFormatterBuilder()
        .appendValue(ChronoField.YEAR)
        .appendLiteral('-')
        .appendValue(ChronoField.MONTH_OF_YEAR)
        .appendLiteral('-')
        .appendValue(ChronoField.DAY_OF_MONTH)
        .appendLiteral('T')
        .appendValue(ChronoField.HOUR_OF_DAY)
        .appendLiteral(':')
        .appendValue(ChronoField.MINUTE_OF_HOUR)
        .appendLiteral(':')
        .appendValue(ChronoField.SECOND_OF_MINUTE)
        .appendLiteral('.')
        .appendValue(ChronoField.MILLI_OF_SECOND)
        .toFormatter();

Formatting with DateTimeFormatter

Once a DateTimeFormatter has been created, it can be used for formatting LocalDate, LocalTime and LocalDateTime objects:


          
           
LocalDate date = LocalDate.now();
String dateFormatted = date.format(formatter);

LocalTime time = LocalTime.now();
String timeFormatted = time.format(formatter);

LocalDateTime dateTime = LocalDateTime.now();
String dateTimeFormatted = dateTime.format(formatter);

The code above will output something like the following:


          
           
2020-07-14T23:48:04.022

Parsing with DateTimeFormatter

A DateTimeFormatter can also be used to parse strings into LocalDate, LocalTime and LocalDateTime objects:


          
           
String dateStr = "2020-07-14";
LocalDate date = LocalDate.parse(dateStr, formatter);

String timeStr = "23:48:04";
LocalTime time = LocalTime.parse(timeStr, formatter);

String dateTimeStr = "2020-07-14T23:48:04.022";
LocalDateTime dateTime = LocalDateTime.parse(dateTimeStr, formatter);
