javax.swing.filechooser.FileNameExtensionFilter

The 

          
           
javax.swing.filechooser.FileNameExtensionFilter
class is a useful tool for filtering files in a 

          
           
JFileChooser
to allow the user to select files with certain extensions. This class provides a way to easily select files with specific types of extensions, such as 

          
           
.txt
, 

          
           
.jpg

.pdf
.

Constructors

The 

          
           
FileNameExtensionFilter
class contains two constructors: 

          
           
public FileNameExtensionFilter(String description, String... extensions)
This constructor takes in a description of the filter, as well as a varargs of extensions (i.e. 

          
           
.txt
, 

          
           
.jpg
). 

          
           
public FileNameExtensionFilter(String description, String[] extensions)
This constructor takes in a description of the filter, as well as an array of extensions (i.e. 

          
           
.txt
, 

          
           
.jpg
).

Methods

The 

          
           
FileNameExtensionFilter
class contains a few useful methods: 

          
           
getDescription()

accept(File f)
This method returns true if the file passed into it has one of the extensions specified in the constructor. 

          
           
getExtensions()

Examples

FileNameExtensionFilter
class:
