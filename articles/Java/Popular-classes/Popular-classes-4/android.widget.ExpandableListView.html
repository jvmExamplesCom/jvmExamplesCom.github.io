android.widget.ExpandableListView

Overview

The android.widget.ExpandableListView class is a type of view used to present information in a hierarchical structure. It allows users to expand and collapse items in the list view to view their contents.

Usage

To use the ExpandableListView, first create an adapter that will provide data to the view: 

          
           
MyAdapter adapter = new MyAdapter(context);
ExpandableListView listView = (ExpandableListView) findViewById(R.id.listview);
listView.setAdapter(adapter);
The adapter should implement the ExpandableListAdapter interface, which requires two methods for retrieving the list data: 

          
           
public List<?> getChild(int groupPosition, int childPosition);

public long getGroupId(int groupPosition);
Once this is done, the view can be populated with data. The following example shows how to add an item to the list: 

          
           
listView.addItem(groupPosition, childPosition, "Item Title", "Item Description");
The ExpandableListView also has several callback methods for reacting to user interaction with the view, such as when an item is expanded or collapsed: 

          
           
listView.setOnGroupExpandListener(new OnGroupExpandListener() {
  @Override
  public void onGroupExpand(int groupPosition) {
    // Do something when the group is expanded
  }
});

listView.setOnGroupCollapseListener(new OnGroupCollapseListener() {
  @Override
  public void onGroupCollapse(int groupPosition) {
    // Do something when the group is collapsed
  }
});

Example Usage

The following is a simple example of how to use an ExpandableListView: 

          
           
ExpandableListView listView = (ExpandableListView) findViewById(R.id.listview);

// Create the adapter
MyAdapter adapter = new MyAdapter(context);
listView.setAdapter(adapter);

// Set listeners for expanding and collapsing groups
listView.setOnGroupExpandListener(new OnGroupExpandListener() {
  @Override
  public void onGroupExpand(int groupPosition) {
    // Do something when the group is expanded
  }
});

listView.setOnGroupCollapseListener(new OnGroupCollapseListener() {
  @Override
  public void onGroupCollapse(int groupPosition) {
    // Do something when the group is collapsed
  }
});

// Add items to the list
listView.addItem(0, 0, "Item 1", "Description 1");
listView.addItem(0, 1, "Item 2", "Description 2");
listView.addItem(0, 2, "Item 3", "Description 3");

Conclusion

The android.widget.ExpandableListView is a versatile UI element that allows users to explore hierarchical information. By creating an adapter that implements the ExpandableListAdapter interface, it's relatively easy to populate the view with data and react to user interactions.
