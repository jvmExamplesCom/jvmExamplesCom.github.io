Introduction to org.apache.http.params.HttpParams

The HttpParams class defines methods for setting and retrieving parameters. For example, you can get a parameter value using the getParameter() method:


          
           
String paramValue = httpParams.getParameter("paramName");

You can also set parameter values using the setParameter() method:


          
           
httpParams.setParameter("paramName", "paramValue");
