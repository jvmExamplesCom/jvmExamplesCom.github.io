org.apache.log4j.LogManager

org.apache.log4j.LogManager is a Java Class which provides the basic support for managing a logging system. It is a sub package of Log4j, used for logging events in application. LogManager class helps in creating an instance of Logger, it also holds the global hierarchy of loggers, configures the loggers based on userâ€™s configuration code and Shutdown the logging system when required.

Features of LogManager

  • The LogManager class defines one method each for different operations like getting Logger instance, setting configuration codes, shutdown logging system etc.
  • It is a singleton class which means that there will ever be only one instance of the LogManager class.
  • LogManager uses Configuration instances to initialize the logging system.
  • It has global hierarchy of loggers (the global tree). All the loggers created using LogManager will automatically be added in this global hierarchy.

Methods

Following are the list of popular methods of LogManager class -

  • getLogger(): It is used to get an instance of logger with the name specified in argument. If no argument is passed then it return root logger with the name "ROOT". 
    
            
             
Logger logger = LogManager.getLogger("MyClass");
  • shutdown(): It is used to shutdown the logging system. After shutting down all the logging resources are released. 
    
            
             
LogManager.shutdown();
  • readConfiguration(): It is used to read a Configuration object from the input stream and configure the logging system accordingly. 
    
            
             
InputStream config = new FileInputStream("log4j2.xml");
LogManager.readConfiguration(config);

Example

Following example demonstrates LogManager's getLogger() method -


          
           
import org.apache.log4j.LogManager;
import org.apache.log4j.Logger;

public class MyClass {
    private static final Logger LOGGER = LogManager.getLogger(MyClass.class);
    
    public static void main(String[] args) {
        LOGGER.info("Started execution of main method.");
        System.out.println("Hello World!");
        LOGGER.info("Completed execution of main method.");
   }
}
