com.facebook.react.bridge.WritableMap

The com.facebook.react.bridge.WritableMap class provides an interface for writing data to a JavaScript object. It is used in conjunction with the ReadableMap class to provide serialization of native objects to and from JavaScript objects. It can be used to send data from a React Native app to a WebView, or to pass data between different React Native components.

method writes a boolean value to the object, identified by its key. The 

          
           
putDouble(String key, double value)
method does the same for a double value, and the

for an integer value.

merge(ReadableMap source)
method which allows merging the source into the target map. This is useful when combining multiple objects into one map, or when mapping an existing native object to a WritableMap.
