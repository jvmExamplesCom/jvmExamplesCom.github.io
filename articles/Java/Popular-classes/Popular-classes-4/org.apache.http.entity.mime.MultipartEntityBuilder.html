org.apache.http.entity.mime.MultipartEntityBuilder

The org.apache.http.entity.mime.MultipartEntityBuilder class provides a modular approach for constructing multipart entities. It simplifies the process of creating MIME messages by allowing developers to construct parts and attach them to the message in a more natural way.

Creating an Instance

You can create an instance of this class with the following code:


          
           
MultipartEntityBuilder builder = MultipartEntityBuilder.create();

Building a Multipart Entity

You can add parts to the entity with the 
addPart()
method: 

          
           
File file = new File("myfile.txt");
builder.addPart("file", new FileBody(file));

You can also add parameters to the entity with the 

addTextBody()
method: 

          
           
builder.addTextBody("param1", "val1");
builder.addTextBody("param2", "val2");

Generating the Entity

Once you have added all the parts and parameters, you can generate the entity with the 

build()
method: 

          
           
HttpEntity entity = builder.build();

Example

The following example demonstrates how to use MultipartEntityBuilder to create an entity with a file and two parameters:


          
           
MultipartEntityBuilder builder = MultipartEntityBuilder.create();
File file = new File("myfile.txt");
builder.addPart("file", new FileBody(file));
builder.addTextBody("param1", "val1");
builder.addTextBody("param2", "val2");
