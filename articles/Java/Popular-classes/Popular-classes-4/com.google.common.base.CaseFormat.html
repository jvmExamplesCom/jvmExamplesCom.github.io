Introduction to com.google.common.base.CaseFormat

The com.google.common.base.CaseFormat is a class from the Google Guava library that provides utility methods for converting between various common case formats.

It can be used to convert between camelCase, UPPER_CAMEL_CASE, lowerCamelCase, snake_case, dotted.case, and other types of case conventions.

The CaseFormat class contains static methods that accept strings and return strings converted to the specified case convention.

Basic Usage

To convert any string to a desired case format, you can use one of the static methods in the CaseFormat class, e.g.:


          
           
String stringInCamelCase = "thisIsAStringInCamelCase";
String stringInUpperCamelCase = CaseFormat.UPPER_CAMEL.to(CaseFormat.CAMEL, stringInCamelCase);
// thisIsAStringInCamelCase

The above code converts the string to UPPER_CAMEL_CASE (i.e. ThisIsAStringInCamelCase).

You can also convert from one case format to another, e.g.:


          
           
String stringInSnakeCase = "this_is_a_string_in_snake_case";
String stringInUpperCamelCase = CaseFormat.SNAKE_CASE.to(CaseFormat.UPPER_CAMEL, stringInSnakeCase);
// ThisIsAStringInSnakeCase

The above code converts the string to UPPER_CAMEL_CASE (i.e. ThisIsAStringInSnakeCase).

Advanced Usage
