android.preference.CheckBoxPreference

android.preference.CheckBoxPreference is a class designed to provide a boolean type preference. It allows users to check and uncheck a box in an Android application’s preferences, allowing the user to toggle an option on or off. It can be used to enable or disable a feature or behavior within an application. This class is part of the android.preference package.

Usage & Code Examples

Creating a CheckBoxPreference consists of two steps - creating the layout XML file and adding the Preference to the Activity. First, create an XML layout file containing the CheckBoxPreference. This file should reside in the res/xml directory. Here is an example: 

          
           
<CheckBoxPreference
        android:key="pref_checkbox"
        android:title="CheckBox Preference Example"
        android:summary="This preference allows for a boolean value to be selected"
        android:defaultValue="false" />
The above code creates a simple CheckBoxPreference, with a title, summary and default value. Once this layout has been created, it needs to be added to an Activity in order to be displayed to the user. The following code adds the CheckBoxPreference to an Activity: 

          
           
public class MainActivity extends AppCompatActivity {
   @Override
   protected void onCreate(Bundle savedInstanceState) {
      super.onCreate(savedInstanceState);
      addPreferencesFromResource(R.xml.checkbox_preference);
   }
}
The above code adds the CheckBoxPreference to the Activity by referencing the XML layout file. Once added, the CheckBoxPreference will be displayed when the user navigates to the respective Activity. To access the value of the CheckBoxPreference, use the PreferenceManager class. The getSharedPreferences() method can be used to return an instance of SharedPreferences which can then be used to access the value of the CheckBoxPreference. Here is an example: 

          
           
SharedPreferences sharedPreferences = PreferenceManager.getDefaultSharedPreferences(this);
boolean isChecked = sharedPreferences.getBoolean("pref_checkbox", false);
The above code returns an instance of SharedPreferences, which can then be used to access the value of the CheckBoxPreference. Note that in order to access the value, the key specified in the layout XML file must be passed as a parameter.
