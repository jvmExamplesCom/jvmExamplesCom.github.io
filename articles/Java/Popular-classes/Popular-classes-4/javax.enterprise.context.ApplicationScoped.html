javax.enterprise.context.ApplicationScoped

Java EE applications often need a way to provide access to data over multiple requests and sessions. The 

ApplicationScoped
annotation allows developers to create such scoped beans, in which their data is accessible and usable within the application, while allowing them to specify expiration and other methods of handling the data.

Usage

The `ApplicationScoped` annotation can be used in any Java EE application, and is most often used for long-term data storage solutions. It is important to note that the scope does not apply to the session or request, but only applies to the application. Data stored in an `ApplicationScoped` bean will only be available within the application.

Code Examples 


          
           
@ApplicationScoped 
public class MyBean { 
 
    private String myData;
 
   public void setMyData(String myData) {
      this.myData = myData;
   }
 
   public String getMyData() {
      return this.myData;
   }
}

          

          
           
@ApplicationScoped

This code example showcases a simple example of an 

ApplicationScoped
bean. It contains two simple methods, 
setMyData
and 
getMyData
, that can be used by the application to store and retrieve data from within this bean.

Expiration

The 
ApplicationScoped
annotation does not inherently provide any expiration capabilities for its beans. However, when combined with the appropriate frameworks, such as JCache, developers can easily provide expiration support for their beans. 

          
           
@ApplicationScoped
@CacheConfig(expiryTime=30)
public class MyBean { 
 
   private String myData;
 
   public void setMyData(String myData) {
      this.myData = myData;
   }
 
   public String getMyData() {
      return this.myData;
   }
}
In this example, the 
MyBean
class is given the 
@CacheConfig
annotation, with an expiration period of 30 seconds. This means that after the data has been stored in this bean for more than 30 seconds, it will be automatically expired and no longer accessible.

Conclusion

The 
ApplicationScoped
annotation provides developers with a convenient way to store and access application-wide data. With the help of frameworks like JCache, developers can even provide expiration support to their beans, allowing them to clean up stale data and keep their applications running efficiently.
