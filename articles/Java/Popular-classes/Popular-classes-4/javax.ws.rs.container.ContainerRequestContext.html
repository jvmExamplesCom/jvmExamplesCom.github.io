Javax.ws.rs.container.ContainerRequestContext

Javax.ws.rs.container.ContainerRequestContext is an interface that provides access to request-specific information about the current request, such as headers and cookies. The following example shows a basic usage of ContainerRequestContext:

ContainerRequestContext requestContext = new ContainerRequestContext();

// Access a specific header

// Set a custom status code
requestContext.setStatus(400);

// Set a custom header
