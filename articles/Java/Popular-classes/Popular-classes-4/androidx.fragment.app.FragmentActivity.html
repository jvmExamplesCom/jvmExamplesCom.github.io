androidx.fragment.app.FragmentActivity

The androidx.fragment.app.FragmentActivity class is used to manage the interaction between Fragment and Activity components in the Android Support Library. It is an extension of the regular Android Activity class, designed to be used with Fragment objects. The primary purpose of this class is to help developers write code that interacts between an Activity and a Fragment.

This class provides several methods that allow you to manage Fragment-specific operations, such as adding a Fragment to the activity, removing a Fragment from the activity, setting the activity’s content view, and so on. Additionally, this class also provides access to the FragmentManager, which can be used to manage the Fragments associated with the Activity.

Example Usage

To use the androidx.fragment.app.FragmentActivity class, you need to subclass it and then implement any desired logic:


          
           
public class MyFragmentActivity extends FragmentActivity {

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);

        // Get the FragmentManager and add a new Fragment
        FragmentManager fragmentManager = getSupportFragmentManager();
        FragmentTransaction fragmentTransaction = fragmentManager.beginTransaction();
        MyFragment myFragment = new MyFragment();
        fragmentTransaction.add(R.id.container, myFragment);
        fragmentTransaction.commit();
    }
}

The above code instantiates a new MyFragment instance and adds it to the activity’s View hierarchy (using the add() method).

You can also use the supportFragmentManager to retrieve existing Fragments associated with the Activity:


          
           
// Get the FragmentManager
FragmentManager fragmentManager = getSupportFragmentManager();

// Get a list of all Fragments associated with the Activity
List<Fragment> fragmentList = fragmentManager.getFragments();

// Iterate over the list, and do something with each Fragment
for (Fragment f : fragmentList) {
    ...
}

Finally, you can use the supportFragmentManager to remove a Fragment from the Activity:


          
           
// Get the FragmentManager
FragmentManager fragmentManager = getSupportFragmentManager();

// Find the Fragment to be removed
MyFragment myFragment = (MyFragment) fragmentManager.findFragmentById(R.id.my_fragment);

// Remove the Fragment from the Activity
FragmentTransaction fragmentTransaction = fragmentManager.beginTransaction();
fragmentTransaction.remove(myFragment);
fragmentTransaction.commit();

That’s all there is to it! As you can see, the androidx.fragment.app.FragmentActivity class makes it easy to manage the interaction between an Activity and its Fragments.
