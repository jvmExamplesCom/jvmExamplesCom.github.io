javax.lang.model.element.VariableElement

javax.lang.model.element.VariableElement is an interface in the Java Model Element Interface which represents a field, enum constant, method or constructor parameter, local variable, resource variable, or exception parameter. It provides access to the information about the variable, including its modifiers and type.

Usage

The javax.lang.model.element.VariableElement class is typically used to retrieve information about a variable found during annotations processing. The methods of this class can be used to inspect a variable’s modifiers and type, retrieve the value assigned to the variable at compile-time, and so on.

Methods

    : Returns the simple name of this variable.
  • asType()
    : Returns the type of this variable as a TypeMirror.
  • getConstantValue()
    : Returns the value of this variable if it is a compile-time constant.
