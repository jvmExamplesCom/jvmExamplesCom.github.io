org.junit.runners.Suite.SuiteClasses

The 

          
           
org.junit.runners.Suite.SuiteClasses
annotation is used to identify classes that should be run as part of a test suite. This annotation is used in conjunction with the 

          
           
@RunWith(Suite.class)
annotation in order to create a test suite.

Example Setup

A typical setup of a test suite using this annotation would look something like this:


          
           
@RunWith(Suite.class)
@SuiteClasses({
   TestClassOne.class, 
   TestClassTwo.class
})
public class TestSuite { }
