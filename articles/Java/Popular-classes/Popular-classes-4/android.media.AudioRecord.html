android.media.AudioRecord

The android.media.AudioRecord class is a powerful tool for applications that want to work directly with raw audio data. It allows an application to record audio from the device microphone, or from other supported sources, and then manipulate that data in various ways. With the AudioRecord class, developers can access the audio hardware for both recording and playback purposes.

In this article, we will explore the basics of the android.media.Audiorecord class and how to use it to capture audio from the device. We will also look at some of its advanced features and discuss situations where it might be useful.

Creating an AudioRecord Instance

The first step to working with the AudioRecord class is to create an instance. This is done using the AudioRecord constructor, which takes several parameters. These parameters include the audio source, sample rate, number of channels, encoding format and buffer size.


          
           
AudioRecord recorder = new AudioRecord(
    MediaRecorder.AudioSource.MIC,
    44100,
    AudioFormat.CHANNEL_IN_STEREO,
    AudioFormat.ENCODING_PCM_16BIT,
    4096
);

The audio source (MediaRecorder.AudioSource) denotes the source of audio data- either internal microphone or other input devices supported.

The sample rate (44100) specifies the sampling rate of the audio signal in hertz. It is important to choose a high enough sample rate so that the recorded audio does not sound distorted.

The number of channels (AudioFormat.CHANNEL_IN_STEREO) specifies the number of audio channels to be used in the recording- mono or stereo.

The encoding format (AudioFormat.ENCODING_PCM_16BIT) specifies the encoding format of the audio data.

The buffer size (4096) specifies the size of the buffer used to store the recorded audio data.

Recording Audio

Once the instance has been created, it can be used to start recording audio. This is done using the 

startRecording
method, which takes no parameters. 

          
           
recorder.startRecording();

The recorded audio data is now stored in a buffer. To read this data, you can use the 

read
method, which takes two parameters: an array of bytes to store the data and the offset from which to start reading. 

          
           
byte[] buffer = new byte[4096];
int offset = 0;

// reads the audio data into the buffer
int read = recorder.read(buffer, offset);
The 
read
method returns the number of bytes read, or -1 if an error occurred. Once the data has been read, you can process it or write it to a file.

Stopping Recording

When you are finished recording, you can use the 
stop
method to stop the recording. This will also release any resources allocated by the AudioRecord instance. 

          
           
recorder.stop();
