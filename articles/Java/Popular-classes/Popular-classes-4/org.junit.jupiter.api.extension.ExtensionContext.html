Introduction to JUnit Jupiter ExtensionContext

The org.junit.jupiter.api.extension.ExtensionContext is an interface in the JUnit 5 extension model. ExtensionContext provides the context in which an extension can be used. Using the ExtensionContext Interface

The ExtensionContext interface allows extensions to obtain information about the current test instance and the container, as well as a map of records that may have been created by other extensions.
