org.apache.commons.collections4.CollectionUtils is a collection of utility classes for working with Java collections. It provides useful methods for manipulating and transforming collections, such as filtering, sorting, and performing set-operations like union and intersection. Filtering Collections

List<String> list = Arrays.asList("foo", "bar", "baz");

Collection<String> results = CollectionUtils.select(list,
    new Predicate<String>() {

        }
    });
