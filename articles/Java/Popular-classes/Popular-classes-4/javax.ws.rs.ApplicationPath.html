What is javax.ws.rs.ApplicationPath?

javax.ws.rs.ApplicationPath is an annotation used in Java to specify the root mapping path for resources and restful web services. It is part of the Java API for RESTful Web Services (JAX-RS). By applying this annotation to a class, the mapper is specified to the web services for easy organization of the services.

Using javax.ws.rs.ApplicationPath

The javax.ws.rs.ApplicationPath annotation is applied to the subclasses of javax.ws.rs.core.Application. The subclass can use the annotation to set the application path like this:


          
           
@ApplicationPath("/myapp")
public class MyApp extends Application {
   //rest of the code
}

This annotation specifies that all the services are going to be mapped under /myapp. It is important to note that all the services published by the application must be using this path as the base path.
