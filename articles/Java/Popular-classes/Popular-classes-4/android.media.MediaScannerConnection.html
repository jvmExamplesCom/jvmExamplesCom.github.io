android.media.MediaScannerConnection

The android.media.MediaScannerConnection class is used to provide an interface for the UI to interact with the media scanner service. Overview

The MediaScannerConnection class provides a simple way for the application to initiate a scan of the specified file or directory. It does this by creating a background thread that communicates with the media scanner service and waits for the scan to complete. The result is a list of MediaScanResult objects, which can then be used to access information about the scanned files.
