org.apache.commons.csv.CSVFormat

The 

          
           
org.apache.commons.csv.CSVFormat
class is an important part of the Apache Commons CSV library for reading and writing comma-separated values (CSV) files. It provides a convenient way to define the format of a CSV file, including field delimiter, record separator, etc. The CSVFormat class can be used to create custom formatters for reading or writing, as well as to retrieve pre-defined formats like RFC-4180 and Excel. In this article, we'll take a look at how to use the CSVFormat class in Java.

Creating a Custom CSVFormat

To start, let's create a simple CSVFormat for our use case. We'll need to specify the field delimiter, which we'll set to a comma (,). This will cause the CSV file to look something like this:


          
           
field_1,field_2,field_3
value_1,value_2,value_3
value_4,value_5,value_6

We'll also need to set the record separator, which we'll set to a line feed character, "\n". To create this custom CSVFormat, you can use the following code:


          
           
CSVFormat csvFormat = CSVFormat.DEFAULT.withDelimiter(',').withRecordSeparator("\n");

Now that we've created our custom CSVFormat, let's take a look at how we can use it when reading or writing a CSV file.

Reading a CSV File with CSVFormat

Using the CSVFormat created above, we can read a CSV file using the following code:


          
           
FileReader fileReader = new FileReader("path/to/file.csv");
CSVParser csvParser = new CSVParser(fileReader, csvFormat);
List<CSVRecord> records = csvParser.getRecords();
for (CSVRecord record : records) {
    String field1 = record.get(0);
    String field2 = record.get(1);
    String field3 = record.get(2);

    // Do something with the fields here
}
csvParser.close();

The code above will open the file at "path/to/file.csv", then parse the contents using our custom CSVFormat. It will then loop through each record in the file, retrieving the values of the first, second, and third fields. Finally, it will close the parser.

Writing a CSV File with CSVFormat

Using the same CSVFormat from before, we can also write a CSV file using the following code:


          
           
FileWriter fileWriter = new FileWriter("path/to/file.csv");
CSVPrinter csvPrinter = new CSVPrinter(fileWriter, csvFormat);
for (int i=0; i<records.size(); i++) {
    csvPrinter.printRecord(records.get(i));
}
csvPrinter.flush();
csvPrinter.close();

The code above will open the file at "path/to/file.csv" and use our custom CSVFormat to write to the file. It will then loop through each record, writing it to the file. Finally, it will flush the printer and close the file.
