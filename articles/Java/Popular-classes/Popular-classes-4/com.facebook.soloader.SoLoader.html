com.facebook.soloader.SoLoader

The com.facebook.soloader.SoLoader class provides an API for the loading of shared libraries from Java code and the associated native methods.

What Does SoLoader Do?

SoLoader enables dynamic loading of a .so file from within Java, eliminating the need to manually include the .so library into a project directory or Android’s lib/ folder. It also allows Java classes to interact with native methods from the loaded .so library.

How to Use SoLoader

First, initialize SoLoader in your Activity's onCreate method:


          
           
@Override
protected void onCreate(Bundle savedInstanceState) {
    super.onCreate(savedInstanceState);
    SoLoader.init(this, false);
}

This enables the use of SoLoader within our application. To actually load the library, use the following call:


          
           
SoLoader.loadLibrary("library-name");

Using Native Methods From Loaded Libraries

Once a library is loaded, any classes of the same package can access its native methods through the following method signature:


          
           
public static native int nativeMethod(int value);

To call the native method, use the following syntax within the same package that contains the native method:


          
           
int x = SoLoader.nativeMethod(4);

Conclusion

SoLoader is a powerful and convenient tool for loading .so libraries from within Java applications. It simplifies the process of including native code and makes it easier to use native methods from within a Java class.
