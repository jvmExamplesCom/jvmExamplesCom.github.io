The

org.junit.runners.model

What does FrameworkMethod do?

The 
FrameworkMethod
allows developers to use a method as part of a unit test, which is often referred to as a "test case". This method wraps the code being tested and allows us to configure options for how to execute the test, as well as report the results. It also enables us to reuse code for testing across multiple tests.

Creating a FrameworkMethod

FrameworkMethod
requires a 
@Test
annotation to mark the method as a "test" to be run by the JUnit framework. Other annotations such as 
@Before

public void testAddition() {
    Assert.assertEquals(5, add(2, 3));
}

   return x + y;
