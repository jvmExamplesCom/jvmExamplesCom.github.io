What does FrameworkMethod do?

The 
FrameworkMethod

This method wraps the code being tested and allows us to configure options for how to execute the test, as well as report the results. It also enables us to reuse code for testing across multiple tests.

Creating a FrameworkMethod

FrameworkMethod
requires a 
@Test

@Before

    Assert.assertEquals(5, add(2, 3));
