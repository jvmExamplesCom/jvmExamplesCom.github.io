Android.accounts.AccountManager

Android.accounts.AccountManager is an Android API that provides the ability to create and manage the user profile on the device. It gives the apps the capability to securely store the user's information such as username, password, email address, etc. This API is available since Android 2.1 (API level 7).

Creating an Account Manager

To create an instance of the AccountManager, we can use the following code: 

          
           
AccountManager accountManager = AccountManager.get(context);
where context is an instance of the android.content.Context.

Adding a New Account

To add a new account, we can use the following code: 

          
           
AccountManagerFuture<Bundle> future = accountManager.addAccount(
    accountType,
    authTokenType,
    requiredFeatures,
    options,
    activity,
    callback,
    handler
);
The parameters for this function are:
  • accountType - string specifying the type of account
  • authTokenType - string specifying the authentication token type
  • requiredFeatures - array of strings which define the features the account must have
  • options - an optional bundle of data to be passed along to the authenticator
  • activity - an optional Activity context (which must be used if calling from a Service)
  • callback - optional AccountManagerCallback object which will receive the result
  • handler - optional Handler object which will be used to post the result

Listing Accounts

To list all existing accounts, we can use the following code: 

          
           
Account[] accounts = accountManager.getAccountsByType(accountType);
where accountType is a string specifying the type of account to list.

Retrieving Account Information

To retrieve information associated with a particular account, we can use the following code: 

          
           
AccountManagerFuture<Bundle> future = accountManager.getAuthToken(
    account,
    authTokenType,
    null,
    activity,
    callback,
    handler
);
The parameters for this function are:
  • account - Account object for the account to retrieve information from
  • authTokenType - string specifying the authentication token type
  • activity - an optional Activity context (which must be used if calling from a Service)
  • callback - optional AccountManagerCallback object which will receive the result
  • handler - optional Handler object which will be used to post the result

Using Authentication Tokens

To use authentication tokens, we can use the following code: 

          
           
String authToken = accountManager.blockingGetAuthToken(
    account,
    authTokenType,
    notifyAuthFailure
);
The parameters for this function are:
  • account - Account object for the account to retrieve the token from
  • authTokenType - string specifying the authentication token type
  • notifyAuthFailure - boolean indicating whether to notify the user of a failed authentication

Removing an Account

To remove an account, we can use the following code: 

          
           
accountManager.removeAccount(
    account,
    activity,
    callback,
    handler
);
The parameters for this function are:
  • account - Account object for the account to remove
  • activity - an optional Activity context (which must be used if calling from a Service)
  • callback - optional AccountManagerCallback object which will receive the result
  • handler - optional Handler object which will be used to post the result

Updating an Account

To update the information associated with an account, we can use the following code: 

          
           
accountManager.setAuthToken(
    account,
    authTokenType,
    authToken
);
The parameters for this function are:
  • account - Account object for the account to update
  • authTokenType - string specifying the authentication token type
  • authToken - string containing the authentication token

Invalidating an Authorization Token

To invalidate an authorization token, we can use the following code: 

          
           
accountManager.invalidateAuthToken(
    authTokenType,
    authToken
);
The parameters for this function are:
  • authTokenType - string specifying the authentication token type
  • authToken - string containing the authentication token
