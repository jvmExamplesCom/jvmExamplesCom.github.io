android.support.v7.widget.AppCompatTextView

Android’s AppCompat text view provides a simple interface for displaying text on the screen. The class is a subclass of TextView and inherits all of its properties and methods. It can be used to display simple text content or complex formatted text with embedded images and styled text. Android’s AppCompat text view is part of the Android Support Library, which contains reusable code for multiple versions of the Android platform.

Creating a Text View

To create an AppCompat Text View, instantiate the class and pass in attributes such as layout_width, layout_height, and textSize. For example: 

          
           
<AppCompatTextView
    android:layout_width="wrap_content"
    android:layout_height="wrap_content"
    android:textSize="20sp"
/>

Setting Text Content

After instantiating the text view, call setText() to add text content. This can be done directly, with a string resource, or from a formatted string. For example, here’s how to set a string from a resource: 

          
           
AppCompatTextView textView = new AppCompatTextView(context);
textView.setText(R.string.example_string);
Or set text directly: 

          
           
AppCompatTextView textView = new AppCompatTextView(context);
textView.setText("Example Text");

Formatting Text Content

The AppCompat text view supports HTML-like formatting tags. These include bold (), italic (), and underline () elements. These can be applied to any string by calling Html.fromHtml(). For example: 

             
              
String htmlString = "This is a <b>bold</b> example";
AppCompatTextView textView = new AppCompatTextView(context);
textView.setText(Html.fromHtml(htmlString));

Customizing the View

The AppCompat text view provides several ways to customize the appearance. For example, setting the text color or adding a background drawable. To set the text color: 

             
              
AppCompatTextView textView = new AppCompatTextView(context);
textView.setTextColor(Color.BLUE);
To add a background drawable: 

             
              
AppCompatTextView textView = new AppCompatTextView(context);
textView.setBackgroundResource(R.drawable.my_background);
