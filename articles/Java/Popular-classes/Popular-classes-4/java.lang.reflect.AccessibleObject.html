Java.lang.reflect.AccessibleObject

The Java.lang.reflect.AccessibleObject class is a utility for determining accessibility of a given object's fields and methods. The AccessibleObject class has two main functions: determining the accessibility of a field or method and setting the accessibility of the same. The setAccessible() function sets the accessibility of a field or method to either public or private and optionally adds additional security measures as defined by the SecurityManager. The getAccessible() function returns the accessibility status of the field or method.


          
           
AccessibleObject aObj = new AccessibleObject();
aObj.setAccessible(true);
boolean isAccessible = aObj.getAccessible();
