Introduction to Java org.apache.poi.ss.usermodel.Workbook

Java provides a rich library known as org.apache.poi.ss.usermodel.Workbook, which offers an easy way to access the all-important Microsoft Office documents such as Excel and PowerPoint. This library is part of the Apache POI project and enables Java programs to read and write Office documents in multiple formats. In this article, we will explore how the org.apache.poi.ss.usermodel.Workbook library can be used to create and manipulate Microsoft Office documents. We will discuss how to create a new document, open an existing document, read/write data and save the document. We will also look at several code examples to demonstrate the usage of the library.

Creating a New Workbook

The following code example shows how to create a new Workbook instance: 

          
           
Workbook wb = new XSSFWorkbook();
The above code creates a new empty Workbook instance by instantiating an XSSFWorkbook class. The XSSFWorkbook class is part of the org.apache.poi.xssf.usermodel package and is used for manipulating Microsoft Excel documents in the OpenXML format.

Opening an Existing Workbook

We can also open an existing Excel document using the org.apache.poi.ss.usermodel.Workbook library. The following code example shows how to open an Excel workbook from a file: 

          
           
FileInputStream fis = new FileInputStream("test.xlsx"); 
 Workbook wb = WorkbookFactory.create(fis);
The above code first creates a java.io.FileInputStream object to open the specified file. It then calls the static create() method of the WorkbookFactory class to open the Excel workbook.

Reading Data from the Workbook

Once the Excel document has been opened, data can be read from it using the org.apache.poi.ss.usermodel.Workbook library. The following code example shows how to read data from an Excel sheet: 

          
           
Sheet sheet = wb.getSheetAt(0); 
for (Row row : sheet) { 
   for (Cell cell : row) { 
      switch (cell.getCellType()) { 
         case Cell.CELL_TYPE_NUMERIC: 
            System.out.print(cell.getNumericCellValue() + "\t");
             break; 
         case Cell.CELL_TYPE_STRING: 
            System.out.print(cell.getStringCellValue() + "\t"); 
            break; 
      } 
   } 
   System.out.println(); 
}
The above code iterates through all the rows and cells in the sheet and prints their contents to the console. Note that we are using the getCellType() and getNumericCellValue() / getStringCellValue() methods of the Cell class to read the cell value.

Writing Data to the Workbook

We can also write data to an Excel document using the org.apache.poi.ss.usermodel.Workbook library. The following code example shows how to write data to an Excel sheet: 

          
           
Sheet sheet = wb.getSheetAt(0); 
Row row = sheet.createRow(0); 
Cell cell = row.createCell(0); 
cell.setCellValue("Hello World");
The above code creates a new Row and Cell objects and sets the cell value to "Hello World". We can similarly create and write other types of data such as numbers and formulas.

Saving the Workbook

The last step is to save the Excel document to a file. The following code example shows how to save the Workbook to a file: 

          
           
FileOutputStream fos = new FileOutputStream("test.xlsx");
wb.write(fos); 
fos.close();
The above code creates a FileOutputStream object and uses the write() method of the Workbook class to save the document to a file.
