org.springframework.core.convert.converter.Converter

The org.springframework.core.convert.converter.Converter is an interface in the Spring framework that enables developers to convert between different types of objects. This interface provides a flexible and powerful way to perform data conversions from one object type to another.

Usage

The Converter interface contains only one method - 

convert
. This method takes two parameters - the source value and the target type. It returns an object of type specified in the target type. 

          
           
public interface Converter<S, T> {
  T convert(S source);
}

Example

The following example shows a simple usage of the 
convert
method. It converts a 
String
value to an 
Integer
. 

          
           
String source = "123";
Converter<String, Integer> converter = new Converter<String, Integer>() {
    public Integer convert(String source) {
        return Integer.parseInt(source);
    }
};
Integer target = converter.convert(source);
The output of the above code will be: 

          
           
target = 123
The following example shows usage of the 
convert
method to convert a 
List
to a 
Set
. 

          
           
List<Integer> source = Arrays.asList(1, 2, 3);
Converter<List<Integer>, Set<String>> converter = new Converter<List<Integer>, Set<String>>() {
    public Set<String> convert(List<Integer> source) {
        Set<String> target = new HashSet<String>();
        for (Integer i : source) {
            target.add(i.toString());
        }
        return target;
    }
};
Set<String> target = converter.convert(source);
The output of the above code will be: 

          
           
target = [1, 2, 3]

Conclusion

The org.springframework.core.convert.converter.Converter interface provides developers with an easy and powerful way to convert between different types of objects. By implementing the convert method, developers can quickly create custom converters to perform any kind of data conversion.
