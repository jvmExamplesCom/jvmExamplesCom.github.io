io.netty.handler.codec.http.HttpResponseStatus

io.netty.handler.codec.http.HttpResponseStatus is a class in Java that provides access to HTTP response status codes and their associated texts. It allows for the creation of custom status codes and responses as well as offering functionality for retrieving the code from an incoming response, or setting a code to an outgoing response.

Creating Custom Response Status

The following code example shows how to create a custom response status: 

          
           
HttpResponseStatus status = new HttpResponseStatus(902, "My Custom HTTP Status");
