Java.awt.event.MouseMotionListener

java.awt.event.MouseMotionListener
interface is for objects that want to be notified when the mouse is being moved or dragged. This interface is part of the java.awt.event package, which contains all the events used by AWT components. Methods in MouseMotionListener Interface

@Override
