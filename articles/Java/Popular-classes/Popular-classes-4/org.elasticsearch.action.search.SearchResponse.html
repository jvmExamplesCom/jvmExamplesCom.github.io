Introduction to org.elasticsearch.action.search.SearchResponse

The org.elasticsearch.action.search.SearchResponse class is a public Java API for querying an Elasticsearch index or documents stored in an Elasticsearch cluster. The class provides methods for constructing, executing and retrieving the results of a search query. This article will provide an overview of how to use the SearchResponse class, as well as example code snippets to demonstrate its usage.

Constructing a SearchRequest

The first step to using the SearchResponse class is to construct a SearchRequest object which will contain the parameters of the search query. This is done by calling the searchRequest() method on the client object with the appropriate parameters. For example: 

          
           
SearchRequest searchRequest = client.searchRequest()
   .indices("posts")
   .types("doc")
   .source(new SearchSourceBuilder()
        .query(QueryBuilders.matchAllQuery())
        .size(10)
   );
The above code creates a search request that searches the posts index and doc type, with a simple match all query, returning a maximum of 10 results.

Executing a Search Request

Once the SearchRequest object has been constructed, it can be executed using the search method on the client object. This will return a SearchResponse object which contains the results of the search query. For example: 

          
           
SearchResponse searchResponse = client.search(searchRequest);

Retrieving the Results

The SearchResponse object can be used to retrieve the results of the search query in various ways. The getHits() method returns a list of SearchHit objects, each representing an individual document that matched the search query. The following example demonstrates how to iterate over these hits and access the fields of each document: 

          
           
for (SearchHit hit : searchResponse.getHits()) {
   String title = (String) hit.getSource().get("title");
   String author = (String) hit.getSource().get("author");
   System.out.println(title + " - " + author);
}
The getSourceAsString() method returns the entire source document as a String, which can then be parsed or transformed as needed. For example, the following code snippet shows how to transform the source String into a JSON object: 

          
           
JSONObject sourceJson = new JSONObject(searchResponse.getSourceAsString());
The SearchResponse object also provides other useful methods such as getTotalHits(), getTookInMillis(), getScrollId(), getAggregations() and getShardStats(). These methods allow for more detailed analysis of the query results.
