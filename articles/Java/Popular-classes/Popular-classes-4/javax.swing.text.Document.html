Introduction to javax.swing.text.Document

The 

          
           
javax.swing.text.Document
class provides a representation of an editable text document. It is the basis of Swing's text components and serves as the model for which they display. This class is made up of three parts: it defines an interface between the text component and any type of text storage model, defines an implementation of the interface, and finally defines a set of support classes that make text editing more efficient.

Implementing the Interface

The 
Document
class defines the interface for any type of text storage that can be used by text components. The basic methods provided in the interface allow the text component to access the document's content and modify it as necessary. Implementations of the interface should ensure that the changes made to the document are reflected in the text component.

Along with the main 
Document
