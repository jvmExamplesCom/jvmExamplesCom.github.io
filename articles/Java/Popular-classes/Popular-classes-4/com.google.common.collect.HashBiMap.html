Introduction to Java com.google.common.collect.HashBiMap

In other words, both the key and the value are uniquely identified in a bi-directional map, allowing for bi-directional lookups.

The com.google.common.collec.HashBiMap is an efficient implementation of a bi-directional map that allows developers to quickly store, retrieve, and modify data.
