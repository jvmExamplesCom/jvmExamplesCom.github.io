The org.junit.jupiter.api.Disabled annotation is an annotation type used to disable a test, test class, or test factory for execution. It is primarily used to indicate that a particular test or tests should be skipped and not ran during execution. This annotation has no parameters.

Using Disabled Annotation

Using the disabled annotation is quite simple and straightforward. All you need to do is put the annotation before the test method or class like this:


          
           
@Disabled
public void testMethod() {
    //test code
}

Why Use Disabled Annotation?

The Disabled annotation is useful if you have a test that is currently failing, but you don't want to delete it from the suite.
