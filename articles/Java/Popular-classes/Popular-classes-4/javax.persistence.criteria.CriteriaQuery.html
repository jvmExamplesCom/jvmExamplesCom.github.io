Introduction to javax.persistence.criteria.CriteriaQuery

javax.persistence.criteria.CriteriaQuery is an API that enables developers to create and execute queries on Java Persistence query language (JPQL) which is a language used to perform database operations.

Using the CriteriaQuery API, developers can build a type-safe query programmatically without the need for string manipulation or SQL. This makes it a powerful tool for developing robust, secure, and maintainable applications.

Creating a CriteriaQuery

The first step in using the CriteriaQuery API is to create an instance of the CriteriaQuery class. This is done by calling the CriteriaBuilder#createQuery() method, passing in the type of the query object as a parameter:


          
           
CriteriaBuilder cb = em.getCriteriaBuilder();
CriteriaQuery<Product> cq = cb.createQuery(Product.class);

Specifying the Query Root

A query root is required to define the entity on which the query will be executed. This is done by calling the CriteriaQuery#from() method, passing in the type of the root entity:


          
           
Root<Product> product = cq.from(Product.class);

Adding Predicates

Predicates are used to filter the query results. Predicates are created by calling the CriteriaBuilder#equal() method, passing in the root entity and the value to compare it against: 


          
           
Predicate priceEqualTo100 = cb.equal(product.get("price"), 100);

Additional predicates can be added to the query by calling the CriteriaQuery#where() method and passing in the predicate:


          
           
cq.where(priceEqualTo100);

Executing the Query

Finally, the query is executed by calling the EntityManager#createQuery() method, passing in the criteria query object as a parameter:


          
           
TypedQuery<Product> query = em.createQuery(cq);
List<Product> products = query.getResultList();

The query can also be executed with parameters by calling the EntityManager#createQuery() method and passing in the criteria query object as a parameter, along with a Map containing the parameters:


          
           
Map<String, Object> parameters = new HashMap<>();
parameters.put("price", 100);
TypedQuery<Product> query = em.createQuery(cq, parameters);
List<Product> products = query.getResultList();
