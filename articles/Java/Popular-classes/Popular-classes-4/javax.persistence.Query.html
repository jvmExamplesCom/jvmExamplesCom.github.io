javax.persistence.Query Basics

javax.persistence.Query is an interface used to control query execution. It is part of the Java Persistence API (JPA). This article covers the basics of using Query, such as constructing a query, executing a query, and getting query results.

Constructing a Query

The primary way of constructing a Query is by calling the createQuery method on the EntityManager object. For example:


          
           
EntityManager em = entityManagerFactory.createEntityManager();
Query q = em.createQuery("SELECT e FROM Employee e");

Executing a Query

  • getSingleResult - executes a SELECT query and returns a single result.
