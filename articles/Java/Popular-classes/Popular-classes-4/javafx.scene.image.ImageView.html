JavaFX: ImageView

The JavaFX 
ImageView
is a class that is used to display an image on the screen. It can be used for simple graphics or for more complicated images, such as photographs. 
ImageView
makes it easy to display an image inside of a 
Scene
. A 
Scene
is the main container for a JavaFX application.

Using ImageView

To use ImageView, one first needs to create an appropriate 

Image
object that can be loaded and displayed. This can be done with the 

          
           
Image.constructor(String url)
method, where the string is the url of an image file. Once an appropriate 
Image
object has been created, it can be set to an 
ImageView
with its 

          
           
setImage(Image image)
method. Finally, the 
ImageView
can be added to the 
Scene
with the 
addNode(Node node)
method. 

          
           
// create an Image object from a file
Image myImage = new Image("myFile.jpg");

// create an ImageView object
ImageView myImageView = new ImageView();

// set the image of the ImageView
myImageView.setImage(myImage);

// add the ImageView to the Scene
myScene.addNode(myImageView);

Example

Below is a complete example of how to create and add an 

ImageView
to a 
Scene
. 

          
           
// create an Image object from a file
Image myImage = new Image("myFile.jpg");

// create an ImageView object
ImageView myImageView = new ImageView();

// set the image of the ImageView
myImageView.setImage(myImage);

// create a Scene
Stage myStage = new Stage();
Scene myScene = new Scene();

// add the ImageView to the Scene
myScene.addNode(myImageView);

// show the Stage
myStage.setScene(myScene);
myStage.show();
