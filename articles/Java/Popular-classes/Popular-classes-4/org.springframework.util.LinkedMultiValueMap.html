Introduction to org.springframework.util.LinkedMultiValueMap

org.springframework.util.LinkedMultiValueMap is a HashMap implementation that stores multiple values in its entries. The LinkedMultiValueMap maintains insertion orders for the key-value pairs and makes it easy to access and update all the values associated with a single key.
