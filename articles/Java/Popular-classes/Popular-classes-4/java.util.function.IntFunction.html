Introduction to Java.util.function.IntFunction

An IntFunction is an interface that takes an int as argument and returns any value.

Example of IntFunction

In the example, we declared an IntFunction called intToString that takes an int argument and returns a String. We then applied the intToString function to the 5 integer using the apply() method, and it returned the String “5”.
