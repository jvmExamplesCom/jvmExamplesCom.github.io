org.bukkit.ChatColor

The 

          
           
org.bukkit.ChatColor
class provides a set of static fields that represent available colors in the Minecraft chat. These colors can be used to change the color of text sent to players or NPCs. The static fields are: 

          
           
public static final ChatColor BLACK, DARK_BLUE, DARK_GREEN, DARK_AQUA, DARK_RED, DARK_PURPLE, GOLD, GRAY, DARK_GRAY, BLUE, GREEN, AQUA, RED, LIGHT_PURPLE, YELLOW, WHITE;

Using them is easy, just call the desired field like so:


          
           
String message = ChatColor.RED + "This message is red!";

It's also possible to combine multiple colors together, for example:


          
           
String message = ChatColor.BLUE + ChatColor.BOLD + "This message is blue and bold!";

Custom Text Color

Sometimes it's necessary to use a color that's different from the standard set provided by the 

ChatColor
class. This can be done by using a custom hex code prefixed with an ampersand 
&
