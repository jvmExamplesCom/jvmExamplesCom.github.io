javax.swing.Box

The javax.swing.Box class is a lightweight container used to create a box layout for components such as buttons, text fields and other GUI elements. It provides a default layout manager that arranges the components in a horizontal or vertical sequence, depending on the orientation specified.

Constructors

The Box class has two constructors: one to create a Box with the specified orientation and another to create a Box with the specified alignment.


          
           
// Creates a Box with the specified orientation
Box box = new Box(orientation);

// Creates a Box with the specified alignment
Box box = new Box(alignment);

Adding Components

Box allows you to add components to it by calling the 

add()
method. The components can be either a Component object or another Box object. 

          
           
// Add a component
box.add(component);

// Add a another box
box.add(otherBox);

Commonly Used Methods

The Box class has several commonly used methods as listed below:


          
           
// Create a rigid area of specified size
box.createRigidArea(size);

// Set the alignment of the Box
box.setAlignment(alignment);

// Set the orientation of the Box
box.setOrientation(orientation);

// Create a horizontal glue
box.createHorizontalGlue();

// Create a vertical glue
box.createVerticalGlue();

Example

The following example creates a Box object with the orientation set to HORIZONTAL, adds two components to it and sets the alignment of the Box as CENTER.


          
           
// Create a Box with the orientation set to HORIZONTAL
Box box = new Box(BoxLayout.X_AXIS);

// Add components to the Box
box.add(component1);
box.add(component2);

// Set the alignment of the Box
box.setAlignment(Box.CENTER);
