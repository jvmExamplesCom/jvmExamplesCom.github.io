org.bson.Document

The org.bson.Document class is a part of the MongoDB Java driver. It's a data structure that maps keys to values, allowing developers to store and manipulate JSON data.

Documents are composed of field-value pairs and have the following characteristics:

  • Field names must be strings
  • Field values may be any BSON data type
  • Fields can be nested documents and arrays

Using Documents, developers can easily construct complex data models in MongoDB while taking advantage of its flexible schema. They can perform CRUD operations, query and index documents, aggregate data, and more.

The simplest way to create a Document is to pass in a 

String

Document()
constructor. The provided string will be used as the key-value pairs within the document. The basic syntax for creating a new 
Document

Document doc = new Document(String keyValuePairs);
For example: 

          
           
Document student = new Document("name", "John Doe").append("age", 21).append("gender", "male");

name

age

gender

Documents can also be nested inside other documents. To create a nested document, use the 

append()

Document

Document address = new Document("city", "New York").append("state", "NY");

The org.bson.Document is a powerful data structure for constructing and manipulating documents in MongoDB. It provides developers with a simple and efficient way to store and retrieve data from MongoDB databases.
