What is android.text.StaticLayout?

android.text.StaticLayout is a class in the Android API that provides a base layout for text that will not be edited by the user. It is used to render text onto a canvas, with support for bidirectional text, and advanced features such as line breaking, indents and tabs. The StaticLayout class can be used to create and manipulate text objects that are rendered in multiple lines.

Creating a StaticLayout Object

To create a StaticLayout object, an application needs to provide the text, width, Alignment and TextDirection. The easiest way to construct a StaticLayout is using the builder pattern. Here is an example of how to construct a StaticLayout object: 

          
           
String value = "The quick brown fox jumps over the lazy dog."; 
float width = 500.0f;
StaticLayout.Builder builder = StaticLayout.Builder.obtain(value, 0, value.length(), new TextPaint(), width);
Alignment alignment = Alignment.ALIGN_NORMAL; 
TextDirection textDirection = TextDirection.FIRST_STRONG_LTR; 
builder.setAlignment(alignment); 
builder.setTextDirection(textDirection); 
StaticLayout staticLayout = builder.build();

Methods available in StaticLayout

The StaticLayout class contains several methods which can be used to manipulate the text rendered in the layout. Here are some of the most commonly used methods:

getLineCount()


          
           
int lineCount = staticLayout.getLineCount();
This method returns the number of lines in the layout.

getEllipsisCount()


          
           
int ellipsisCount = staticLayout.getEllipsisCount(lineIndex);
This method returns the number of ellipses used in the line specified by the lineIndex parameter. An ellipsis is used when a line is too long and needs to be truncated.

getParagraphDirection()


          
           
int direction = staticLayout.getParagraphDirection(lineIndex);
This method returns the direction of a particular line. It can be either LTR or RTL.

getLineStart()

This method returns the start index for a line.

getLineEnd()


          
           
int endIndex = staticLayout.getLineEnd(lineIndex);
This method returns the end index for a line.

Using StaticLayout in an Android App

The StaticLayout class can be used to create custom text views in an Android app. To do this, an application needs to create a subclass of the View class, which will use the StaticLayout class to render the text. Here is an example of how to create a custom text view class: 

          
           
public class CustomTextView extends View { 
   private StaticLayout mLayout;

       super(context); 
   }

      int width = MeasureSpec.getSize(widthMeasureSpec);

      mLayout.draw(canvas); 
      canvas.restore(); 
   }
}
