org.apache.commons.lang3.SystemUtils

The SystemUtils class from the Apache commons-lang3 library provides various static methods to help applications know and interact with the system they are running on. This class is useful for ensuring that a program is running on the correct platform, checking system environment variables, and querying operating system characteristics so that an application can be adapted to different platforms.

Checking Platforms with isOSX/isWindows/isLinux/etc

One of the most useful features of SystemUtils is the ability to check which platform is currently running the program with simple boolean values. The following example shows how to check if a program is running on Linux:


          
           
boolean isLinux = SystemUtils.isLinux();
if(isLinux) {
    System.out.println("Program is running on Linux!");
}
In addition to the 
isLinux()
function, SystemUtils also provides other platform-checking functions such as 
isMac()
, 
isWindows()
, and 
isUnix()
.

String Property Accessors

The SystemUtils class also provides accessors for various string-based system properties. For example, to get the Java version the application is running on:

System.out.println("Java version: " + javaVersion);

Environment Variables
