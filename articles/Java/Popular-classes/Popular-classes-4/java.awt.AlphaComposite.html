Introduction to java.awt.AlphaComposite

java.awt.AlphaComposite is a class that provides methods for creating and manipulating basic cross-platform alpha-blending objects. It enables alpha values to be encapsulated and then applied to multiple graphics primitives well known in Java 2D rendering, including AlphaComposite objects. In essence, it allows developers to create intensities of color, enhancing the level of detail and realism of their projects.

Creating an AlphaComposite Object

The general form of a method used to create an AlphaComposite object is shown here:


          
           
static ALPHA_COMPOSITE AlphaComposite.getInstance(int rule, float alpha)

Here, rule specifies the type of compositing operation to be performed, such as AlphaComposite.SRC_OVER, and alpha specifies the alpha value. To learn more about other available rules, refer to the specification for AlphaComposite.

Applying an AlphaComposite Object

To apply an AlphaComposite object to a graphics primitive, use the setComposite( ) method defined by Graphics2D, as shown here:


          
           
void setComposite(AlphaComposite comp)
Here, comp is the AlphaComposite object.

Example Program

The following program demonstrates how to create an AlphaComposite object and apply it to a graphics primitive:


          
           
import java.awt.*;
import java.awt.image.*;
import java.awt.geom.*;

public class AlphaTest {
    public static void main (String [] args) {
        // Create an image of 16 X 16 pixels
        BufferedImage bim = new BufferedImage(16, 16,
            BufferedImage.TYPE_INT_ARGB);

        // Get its graphics context
        Graphics2D g2d = bim.createGraphics();

        // Set the background color
        g2d.setBackground (Color.white);
        g2d.clearRect(0, 0, 16, 16);

        // Draw a filled circle
        g2d.setPaint(Color.black);
        Ellipse2D circle = new Ellipse2D.Float(3, 3, 10, 10);
        g2d.fill (circle);

        // Create a composite with 0.5 alpha
        AlphaComposite comp = AlphaComposite.getInstance(
            AlphaComposite.SRC_OVER, 0.5f);

        // Set the composite to the Graphics2d context
        g2d.setComposite(comp);

        // Fill rectangle
        g2d.setPaint (Color.blue);
        g2d.fillRect (7, 7, 6, 6);
    }
}
