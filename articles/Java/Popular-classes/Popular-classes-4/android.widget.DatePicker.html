Android.Widget.DatePicker

The Android.Widget.DatePicker allows users to select a date from a pre-populated calendar. It is typically used to display the current date and time, but can be configured to display any date or time. This widget is designed to be simple, intuitive, and easy to use.

The following example illustrates how to initialize and configure an Android.Widget.DatePicker:


          
           
DatePicker datePicker = (DatePicker) findViewById(R.id.date_picker);
datePicker.setMinDate(System.currentTimeMillis() - 10000);
datePicker.setMaxDate(System.currentTimeMillis() + 10000);
datePicker.setCalendarViewShown(false);

This code snippet initializes a DatePicker widget, sets the minimum and maximum dates, and disables the calendar view. Note that the System.currentTimeMillis() method returns the current epoch time in milliseconds.

A DatePicker can also be configured to display specific dates and times by setting the display day, month, and year values:


          
           
int day = 3;
int month = Calendar.JANUARY;
int year = 2017;

datePicker.updateDate(year, month, day);

The above code snippet sets the DatePicker to display the 3rd of January 2017. The Calendar class provides several useful constants for representing different months.

It is also possible to set a listener to detect when the selected date has changed:


          
           
datePicker.setOnDateChangedListener(new OnDateChangedListener() {
    @Override
    public void onDateChanged(DatePicker view, int year, int monthOfYear, int dayOfMonth) {
        // Handle date change event
    }
});

The onDateChanged() callback method will be invoked whenever the user changes the selected date. The year, month, and day values can then be used to further process the date selection.

Conclusion

In this article, we have discussed the Android.Widget.DatePicker widget and its various features. We have seen how to initialize and configure the DatePicker, set specific dates and times, and detect changes to the currently selected date. With these abilities, the DatePicker widget makes it simple for developers to add a powerful calendar-based selection system to their Android applications.
