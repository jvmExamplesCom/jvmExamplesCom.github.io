An Introduction to com.intellij.openapi.actionSystem.AnActionEvent

com.intellij.openapi.actionSystem.AnActionEvent is a class from the IntelliJ platform that is used to pass data to an action in the IntelliJ user interface. It contains information about the context in which the action is being invoked, such as the project and the editor.

Creating AnActionEvent Object

The following code example shows how to create an AnActionEvent object:


          
           
AnActionEvent event = AnActionEvent.createFromAnAction(
    anAction, null, ActionPlaces.UNKNOWN, dataContext
);
In this example, 
anAction
is the instance of the action being invoked, 
dataContext
is the data context associated with the action's invocation, and 

          
           
ActionPlaces.UNKNOWN
is the place where the action is being invoked.

Usage

AnActionEvent can be used to get various pieces of information about the context in which an action is invoked. For example, the following code gets the project associated with the current action invocation:


          
           
Project project = event.getProject();

Similarly, the following code gets the editor associated with the current action invocation:


          
           
Editor editor = event.getData(CommonDataKeys.EDITOR);

AnActionEvent can also be used to present messages to the user. For example, the following code shows a message dialog containing the text "Hello, World!":


          
           
Messages.showMessageDialog(
    project,
    "Hello, World!",
    "",

);
