The 

          
           
android.database.MatrixCursor
is a class from the Android Open Source Project which provides an implementation of the abstract cursor interface for accessing data in the form of a two-dimensional array. It is typically used when creating dynamic cursors out of static data.

Creating a MatrixCursor

A 
MatrixCursor
can be created by passing an array of column names and an initial size to the constructor: 

          
           
String[] columnNames = {"id", "name", "age"};
MatrixCursor matrixCursor = new MatrixCursor(columnNames, 10);

addRow()

matrixCursor.addRow(new Object[]{1, "John", 24});
matrixCursor.addRow(new Object[]{2, "Jane", 28});
The 
addRow()
