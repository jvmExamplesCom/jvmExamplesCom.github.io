The ResourceLoader interface is a core part of the Spring Framework and provides a simple API to access application resources by relative path. It extends the org.springframework.core.io.Resource interface, which provides basic functions to manipulate resource objects such as files, directories, URL connections and more. The ResourceLoader interface also provides the convenience API for loading externalized configuration files and working with resource bundles.

ResourceLoader API
