Introduction to org.apache.velocity.app.VelocityEngine

org.apache.velocity.app.VelocityEngine is an open source Java template engine used to quickly and easily generate dynamic content on the web. VelocityEngine makes it easier and faster to develop, debug, and deploy web applications that are designed to scale.
