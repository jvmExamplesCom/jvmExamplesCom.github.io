org.objectweb.asm.MethodVisitor

The org.objectweb.asm.MethodVisitor class is an abstract class and part of the Objectweb ASM library, a toolkit for manipulating Java class files. It can be used to add new methods to existing classes and remove existing ones.

Usage

The MethodVisitor class provides several methods that can be used to modify the contents of a Java class. Developers can create an instance of the class and then use it to modify existing methods and create new ones.

For example, the following code uses the invokespecial method to add a new instance method to a class:


          
           
MethodVisitor mv = ...;
mv.visitCode();
mv.visitMethodInsn("INVOKESPECIAL", "MyClass", "myMethod", "(I)V");
mv.visitEnd();

This code will add a new instance method to the MyClass class, with a method signature of myMethod (int).

Methods

In addition, the following methods can be used to read information from a class file:
