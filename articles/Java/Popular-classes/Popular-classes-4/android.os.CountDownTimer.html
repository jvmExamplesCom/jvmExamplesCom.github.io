android.os.CountDownTimer in Java

android.os.CountDownTimer is a class that is used to implement a simple timer in Java. It allows users to schedule, start and stop the timer, as well as creating a callback that can be fired off when the timer has completed.

Example Usage

The following example shows how to use android.os.CountDownTimer in Java. The example will create a CountDownTimer of 1000 milliseconds (1 second), then set it to start after 500 milliseconds. After starting, the timer will execute a callback every 250 milliseconds while active.


          
           
// Create a CountDownTimer of 1000 millisecond
CountDownTimer myTimer = new CountDownTimer(1000, 250) {
    // Execute on tick event
    @Override
    public void onTick(long millisUntilFinished) {
        System.out.println("onTick: " + millisUntilFinished);
    }

    // Execute when timer ends
    @Override
    public void onFinish() {
        System.out.println("onFinish");
    }
};

// Start the timer
myTimer.startAfter(500);
