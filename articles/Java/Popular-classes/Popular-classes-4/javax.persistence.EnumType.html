javax.persistence.EnumType

javax.persistence.EnumType is an enumerated type that is part of the Java Persistence API (JPA). It allows developers to store and query enum values in databases. The EnumType class provides a set of constants that can be used to declare the enum column in a mapping file. There are two types of Enums, ORDINAL and STRING.

ORDINAL Enum Usage:

When using the ORDINAL enum type, an enum value is stored as an integer in the database. This is the default type if no type is specified during the mapping process.


          
           
enum Gender {
  MALE,
  FEMALE
}

@Entity
public class Person {
  @EnumType(EnumType.Ordinal)
  private Gender gender;
  // ...
}

In this example, the value of the Gender enum is stored and retrieved as an integer in the database.

STRING Enum Usage:

When using the STRING enum type, an enum value is stored as a String in the database. This type must be specified explicitly during the mapping.


          
           
enum Gender {
  MALE,
  FEMALE
}

@Entity
public class Person {
  @EnumType(EnumType.String)
  private Gender gender;
  // ...
}

In this example, the value of the Gender enum is stored and retrieved as a String in the database.

Using EnumType in Queries:
