Android SoundEffectConstants

Android's SoundEffectConstants interface provides access to the audio playback effects available on Android devices.     : This produces a click sound effect when triggered.
  • SOUND_EFFECT_DIALOG
    : This is used to provide an audible notification when an activity or dialog is opened.
  • SOUND_EFFECT_TICK
    : This creates a ticking sound when used.
  • SOUND_EFFECT_TIP
    : This adds a hint audio effect when triggered.
