javax.validation.ConstraintValidatorContext

javax.validation.ConstraintValidatorContext is an interface used to define an API for customizing validation messages and obtaining metadata about the root bean, property or class-level constraints that are being validated.

Implementation Example

The following example demonstrates how to create a custom validator that checks if a value is greater than a given threshold:


          
           
public class GreaterThanValidator implements ConstraintValidator<GreaterThan, Number> {
    private int minValue;

    public void initialize(GreaterThan constraintAnnotation) {
        this.minValue = constraintAnnotation.value();
    }

    public boolean isValid(Number value, ConstraintValidatorContext context) {
        if (value == null) {
            return true;
        }
        return value.intValue() > minValue;
    }
}
