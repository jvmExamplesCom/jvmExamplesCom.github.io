android.app.KeyguardManager

android.app.KeyguardManager is an Android class used to access keyguard related functionality, such as creating and managing lockscreens and unlocking devices. KeyguardManager can also be used as a way to store sensitive data, as it is wrapped by hardware backed security services. This tutorial will show how to use the KeyguardManager class to lock and unlock a device, as well as how to store and retrieve data using it.

Creating a KeyguardManager Instance

To create an instance of KeyguardManager the first step is to add a reference to it in your app's AndroidManifest.xml file. 

          
           
<uses-permission android:name="android.permission.USE_KEYGUARD_SERVICE" />
With this permission granted, the following code can be used to create an instance of KeyguardManager: 

          
           
KeyguardManager keyguardManager = 
     (KeyguardManager) context.getSystemService(Context.KEYGUARD_SERVICE);

keyguardManager.lockNow();
If the device is already locked, this method will cause the device to move directly to the unlock screen. This is a great way to secure the device when it is not in use, or to switch users quickly. Unlocking the device from the lock screen can be achieved by using the
