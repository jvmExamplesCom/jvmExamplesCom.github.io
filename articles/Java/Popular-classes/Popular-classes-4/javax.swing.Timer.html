Introduction to javax.swing.Timer

The javax.swing.Timer class is a Java utility class that makes it easier for developers to build animations, delay specific actions, and create periodic events. The Timer class allows the execution of code that is delayed by a certain amount of time or runs regularly at a certain frequency. This class is part of the Swing library, so if you want to use the Timer class, you'll need to make sure the Swing library is included in your project.

Creating a Timer

To create a Timer, you simply call the constructor like this:


          
           
Timer timer = new Timer(delay, actionListener);

The arguments for the constructor are the delay (in milliseconds) and an ActionListener. The ActionListener should be an object that implements the ActionListener interface. When the timer is started, it will execute the ActionListener's actionPerformed method after the specified delay. Here is an example of a simple ActionListener class:


          
           
class MyActionListener implements ActionListener {
    public void actionPerformed(ActionEvent e) {
        System.out.println("Action performed!");
    }
}

Now, we can create a Timer object using our ActionListener:


          
           
Timer timer = new Timer(1000, new MyActionListener());

This will create a timer with a delay of 1 second (1000 milliseconds), which will execute the actionPerformed method of our ActionListener after 1 second.

Starting and Stopping the Timer

Once the Timer is created, you can start it with the start() method:


          
           
timer.start();

This will start the timer and it will execute the ActionListener's actionPerformed method every time the delay has passed. If you want to stop the timer, you can just call the stop() method:


          
           
timer.stop();

Examples

Let's look at some simple examples of Timer usage. Here is an example that prints out "Hello World!" every 5 seconds:


          
           
Timer timer = new Timer(5000, new ActionListener() {
    public void actionPerformed(ActionEvent e) {
        System.out.println("Hello World!");
    }
});

timer.start();

Here is an example of a Timer that counts from 0 to 10, printing out the current number every second:


          
           
int count = 0;
Timer timer = new Timer(1000, new ActionListener() {
    public void actionPerformed(ActionEvent e) {
        System.out.println(count);
        count++;
        if (count > 10) {
            timer.stop();
        }
    }
});

timer.start();

Conclusion

The javax.swing.Timer class is a great utility class that makes it easy to create animations, delays, and periodically run code. It is an important part of the Swing library and should be familiar to any Java developer.
