Introduction to javax.lang.model.util.Elements

javax.lang.model.util.Elements is an API introduced in Java 6 that provides methods for getting information about elements of a class. It enables retrieval of information regarding constructors, fields and methods, annotations, and other related elements. Elements can also be tested for the presence of modifers and other useful utility methods like those for obtaining type information.

Element Types

The Element interface has various subtypes that provide information about the various kinds of elements. These include:

  • PackageElement
    - Describes a package
  • TypeElement
    - Describes a type (class, interface, enum)
  • VariableElement
    - Describes a field or a parameter
  • ExecutableElement
    - Describes a constructor or a method
  • 
            
             
TypeParameterElement
    - Describes a type parameter
  • AnnotationMirror
    - Describes an annotation

Example 2: Checking Modifiers


          
           
Elements elements = processingEnv.getElementUtils();
TypeElement fooClass = elements.getTypeElement("FooClass");
for (Element member : elements.getAllMembers(fooClass)) {
  Set<Modifier> modifiers = member.getModifiers();
  if (modifiers.contains(Modifier.PUBLIC)) {
    System.out.println(member + " is public");
  }
}

Elements elements = processingEnv.getElementUtils();

  AnnotationMirror annotation =
