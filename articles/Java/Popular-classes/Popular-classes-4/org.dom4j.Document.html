org.dom4j.Document

The org.dom4j.Document class is a Java XML manipulation API used to represent an XML document. It contains methods for reading, writing, parsing and validating XML documents. Document myDoc = new Document();

String xmlContent = "<root><child>content</child></root>";
Document fromString = DocumentHelper.parseText(xmlContent);

Once a document object has been created, we can read, write, parse and validate the XML content using the various methods provided by the Document class.
