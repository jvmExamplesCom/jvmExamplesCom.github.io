Using javafx.geometry.Pos for Positioning in JavaFX

The javafx.geometry.Pos class provides constants for the different positions of a Node in the user interface. This class is from the javafx.geometry package which many other classes and properties related to geometrical calculations.

The javafx.geometry.Pos Enum

The Pos enum is an enumeration type which provides three constants for position.


          
           
public enum Pos {
   TOP_LEFT,
   CENTER,
   BOTTOM_RIGHT
}

These constants are used to define the position of the node in its parent pane.

Usage of Pos in JavaFX

Pos is used in various JavaFX methods to define the position of a Node. For example, the 


          
           
setAlignment()
method of the GridPane class sets the alignment of the grid-nodes. 

          
           
GridPane.setAlignment(Pos.CENTER);

This sets the position of all nodes in the grid to the center of their parent pane. Similarly, some other positioning methods take the Pos enum as an argument such as:
