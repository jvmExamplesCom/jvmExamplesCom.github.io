java.security.spec.KeySpec

Java.security.spec.KeySpec is an interface used for key and public/private key specifications. It serves as an abstraction for a variety of keys, both symmetric and asymmetric, which are meant to be used with the JCE (Java Cryptographic Extension).

Creating KeySpec Instance
