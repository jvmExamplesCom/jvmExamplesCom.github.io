io.jsonwebtoken.Claims

The io.jsonwebtoken.Claims class provides a way to persist and manage JWTs (JSON Web Tokens) during authentication. The Claims class is an object representation of a JSON Web Token (JWT). It can be used to represent a set of claims that have been signed into a JWT.

Creating a Claims Object

Claims.Builder()
factory method. This method takes a Map of claims as an argument and creates a Claims object from it. The following example shows how to create a Claims object from a Map of claims: 

          
           
Map<String, Object> claims = new HashMap<>();
claims.put("iss", "example.com");
claims.put("expiresIn", 3600);

Claims jwtClaims = Claims.Builder().withClaims(claims).build();

String token = jwtClaims.sign();
