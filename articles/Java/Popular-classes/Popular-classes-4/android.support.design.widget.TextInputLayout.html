android.support.design.widget.TextInputLayout

The android.support.design.widget.TextInputLayout is a layout that wraps an EditText (or descendant) to show a floating label when the hint is hidden while the user inputs text. It also includes native touch handling and focus tracking. The TextInputLayout can be used as an editable form field which can validate the input value and can show an error message.

Code Example 1


          
           
TextInputLayout textInputLayout = new TextInputLayout(this);
textInputLayout.setHint("Enter your name");

EditText editText = new EditText(this);
textInputLayout.addView(editText);
