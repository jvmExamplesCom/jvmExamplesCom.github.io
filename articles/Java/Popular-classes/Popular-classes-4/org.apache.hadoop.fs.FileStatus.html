Introduction to org.apache.hadoop.fs.FileStatus

Java 

          
           
org.apache.hadoop.fs.FileStatus
class is an abstract base class for representing the basic attributes of a file. It is used by Hadoop FileSystem to provide File System related information such as file modification time, file owner, group-owner, block size etc. It contains the information about files and directories in distributed file systems and supports access control based on user/group permissions. In this article, we will be discussing the features and functionalities of org.apache.hadoop.fs.FileStatus.

Features of org.apache.hadoop.fs.FileStatus

The following are some of the key features of org.apache.hadoop.fs.FileStatus:
  • Efficient Way for Storing Information: FileStatus provides an efficient way for storing the information about the file system. It stores the information about the file/directory in memory, so it can be accessed quickly.
  • Support for User/Group Access Control: It supports access control based on user/group permissions. Every file or directory has a set of Access Control Entries (ACEs) associated with it, which defines who can access that particular entity, and what kind of operations they can perform on it.
  • Support for Link Target Lookup: FileStatus provides a method to lookup the target of a symbolic link. This is useful when attempting to open a symbolic link and reaching its target file or directory.
  • Simplifies File or Directory Operations: FileStatus provides a simpler way to perform file or directory operations such as rename, copy, delete, create, etc. Using the FileStatus instance, these operations can be performed more easily.

public static void main(String[] args) {
  Configuration conf = new Configuration();
  FileSystem fs = FileSystem.get(conf);
  Path path = new Path("/tmp/hadoop_file");
  try {
    FileStatus status = fs.getFileStatus(path);
    // Get the file size
    long size = status.getLen();
    // Get the owner
    String owner = status.getOwner();
    // Get the group
    String group = status.getGroup();
    // Get the modification time
    long mtime = status.getModificationTime();
    // Get the access time
    long atime = status.getAccessTime();
    // Get the replication factor
    short replication = status.getReplication();
    // Get the block size
    long blocksize = status.getBlockSize();
    // Check if the path is a directory
    boolean isDir = status.isDirectory();
    // Check if the path is a file
    boolean isFile = status.isFile();
    // Check if the path is a symbolic link
    boolean isLink = status.isLink();
    // Get the target path of the symbolic link
    Path symlink = status.getSymlink();
    // Get the permission
    FsPermission perm = status.getPermission();
  } catch (IOException e) {
    e.printStackTrace();
  }
}
