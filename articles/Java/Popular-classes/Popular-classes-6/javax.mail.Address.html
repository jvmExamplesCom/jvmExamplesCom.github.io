javax.mail.Address

The 
javax.mail.Address
class provides a representation of an email address. It is an abstract class so it cannot be instantiated directly; instead, subclasses are used to represent specific types of addresses. For example, there are subclasses for Internet addresses and UUCP-style addresses. This class also provides methods for converting an address to a string and vice versa.
