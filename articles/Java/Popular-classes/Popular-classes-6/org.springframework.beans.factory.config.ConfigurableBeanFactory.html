ConfigurableBeanFactory in Java

The 

          
           
org.springframework.beans.factory.config.ConfigurableBeanFactory
is a Java interface providing functionality for configuring a org.springframework.beans.factory.BeanFactory. This is an important part of the Spring Framework, as it allows developers to customize their application's bean factories.

public class MyBeanFactory implements ConfigurableBeanFactory {
