org.springframework.dao.EmptyResultDataAccessException

The org.springframework.dao.EmptyResultDataAccessException is an exception thrown by the Spring framework when a requested database operation returns no results. It is thrown when there is an attempt to retrieve data from the database and no results are found. This exception can also be thrown when using the queryForObject() method of JdbcTemplate .

Let's take a look at a simple example. In this example, we will execute a query which returns no results. 

          
           
// Create an instance of JdbcTemplate
JdbcTemplate jdbcTemplate = new JdbcTemplate();

// Execute the query
try {
  String sql = "SELECT * FROM users WHERE id = ?";
  jdbcTemplate.queryForObject(sql, new Object[] { 5 }, User.class);
} catch (EmptyResultDataAccessException e) {
