Introduction to retrofit2.http.Field

Retrofit2.http.Field is an annotation used for mapping a field name to the URL query string parameter of an HTTP request. Using retrofit2.http.Field

The retrofit2.http.Field annotation is used to map a field name to the URL query string parameter of an HTTP request. For example:


          
           
@Retrofit2.http.Field("name") 
String myField;
