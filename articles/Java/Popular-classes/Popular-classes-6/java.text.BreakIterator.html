java.text.BreakIterator

The java.text.BreakIterator is class used for locating the boundaries of text for a given locale. It basically tokenizes a piece of text and provides information about where the boundaries of the words, sentences and other pieces of text are located. It is used in many applications including natural language processing and machine translation.

Example Usage

Below is an example of using the BreakIterator class to locate the boundaries of a sentence in a sample sentence.


          
           
String sentence = "This is a sample sentence.";
BreakIterator bi = BreakIterator.getSentenceInstance();
bi.setText(sentence);
int start = bi.first();
for (int end = bi.next(); end != BreakIterator.DONE; start = end, end = bi.next()) {
  System.out.println(sentence.substring(start, end));
}
