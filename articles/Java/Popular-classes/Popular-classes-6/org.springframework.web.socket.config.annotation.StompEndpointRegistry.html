org.springframework.web.socket.config.annotation.StompEndpointRegistry

Setting Up StompEndpointRegistry

Before we can start using the StompEndpointRegistry, we need to first set it up. To do this, we must first create an instance of the StompEndpointRegistry in our code. This can be done with the following code: 

          
           
StompEndpointRegistry registry = new StompEndpointRegistry();

registry.addEndpoint("/myendpoint");
