Java.nio.file.attribute.PosixFilePermissions

Java.nio.file.attribute.PosixFilePermissions is a class used in the Java platform that provides access to file permissions and attributes in a POSIX-compliant environment. This example shows how to use the PosixFilePermissions class to get, set, and modify the permissions of a file using the PosixFileAttributes class.

Attributes

The PosixFilePermissions class also has three useful attributes; the owner, group and others permissions.
