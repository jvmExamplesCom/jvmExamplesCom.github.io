org.elasticsearch.action.index.IndexRequest

The org.elasticsearch.action.index.IndexRequest class is an ActionRequest class used to store data in an Elasticsearch index. It encapsulates the data itself as well as the metadata associated with it, such as the type of data being stored, the index name, and various other settings. This class is typically used when adding new documents to an Elasticsearch index.

Constructing an Index Request

To construct an IndexRequest, you must provide two elements: the JSON document to be indexed, and the index name. The index name can either be the name of an existing Elasticsearch index, or a new one. The document should be serialized in JSON format.


          
           
IndexRequest request = new IndexRequest("my_index")
    .source(myJsonDocument);

        .id("document_id");
