javax.naming.directory.DirContext

The DirContext class is a Java Naming and Directory Interface (JNDI) interface that provides methods for reading and writing entries in a directory. It provides the ability to create, delete, and update entries, as well as search the directory for entries. It is the primary interface used in the JNDI API for accessing directory data.

Creating a DirContext

InitialContext initialContext = new InitialContext();

env.put(Context.PROVIDER_URL, "ldap://ldap.example.com:389");

The env variables specify the Context Factory, Provider URL, security principal and security credentials required to connect to the LDAP server.

Reading an Entry from a Directory

Once a DirContext object has been created, the read operations can be performed. The following code reads an entry from the directory: 

          
           
String dn = "uid=jsmith,ou=people,dc=example,dc=com";
Attributes attrs = ctx.getAttributes(dn);
In the example above, the getAttributes() method is invoked on the DirContext object and the DN of the entry to be read is passed as an argument. The method returns an Attributes object containing the attributes of the entry.

The DirContext class also provides methods for writing entries to the directory. The following code adds an entry to the directory: 

          
           
String dn = "uid=jsmith,ou=people,dc=example,dc=com";

In the example above, the createSubcontext() method is invoked on the DirContext object. The DN of the entry to be added is passed as the first argument and an Attributes object containing the attributes of the entry as the second argument.

Searching a Directory

The DirContext class also provides methods for searching the directory. The following code searches the directory for all entries whose cn (Common Name) attribute has the value 'John': 

          
           
String filter = "(&amp;(objectClass=person)(cn=John))";

In the example above, the search() method is invoked on the DirContext object. The arguments passed to the method are the base DN of the search, the filter expression and the search controls. The filter expression specifies the criteria for the search, and the search controls determine the attributes to be returned by the query. The method returns a NamingEnumeration containing the search results.
