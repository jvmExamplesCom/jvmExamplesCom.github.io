javax.naming.directory.DirContext

The DirContext class is a Java Naming and Directory Interface (JNDI) interface that provides methods for reading and writing entries in a directory. It provides the ability to create, delete, and update entries, as well as search the directory for entries. It is the primary interface used in the JNDI API for accessing directory data.
