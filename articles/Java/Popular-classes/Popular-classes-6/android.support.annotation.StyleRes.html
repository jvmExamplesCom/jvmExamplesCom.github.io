android.support.annotation.StyleRes

The android.support.annotation.StyleRes is an annotation that indicates the resource ID should be a style resource. It can be used to enforce consistent style usage across Android components, including the support library.

Example 1


          
           
@StyleRes
int myThemeStyle = R.style.MyCustomTheme;
