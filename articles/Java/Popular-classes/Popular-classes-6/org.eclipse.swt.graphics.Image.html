Introduction to org.eclipse.swt.graphics.Image

The org.eclipse.swt.graphics.Image class is part of the Standard Widget Toolkit (SWT) of the Eclipse software platform and provides a high-level graphics programming interface for developers creating graphical user interfaces (GUIs). Creating an Image Object

The most basic way of creating an Image object is done by using the Image constructor: 

          
           
Image image = new Image(display, "myimage.gif");
This creates an Image object with the width and height of the image defined by the myimage.gif file. This can be used to simply load a GIF file into the program, however, the following code samples show some more advanced ways of creating an Image object. 

          
           
int[] data = new int[64*64]; // Create array of 64x64 pixels
ImageData imageData = new ImageData(64,64,1,new PaletteData(0xFF, 0xFF00, 0xFF0000),32,data);

The final parameter of 32 specifies the number of bits of each pixel. Once this is created, the resulting ImageData object can be used as an argument for the Image constructor: 

          
           
Image image = new Image(display, imageData);
This creates an Image object from the ImageData object which can then be used further in the program. Another way to create an Image object is by using two parameters of type InputStream and Display. An InputStream object contains all the data required to define an image, while the Display object serves as the host device for the Image: 

          
           
InputStream inputStream = new FileInputStream("myimage.gif");
Image image = new Image(display, inputStream);
Once again, this creates an Image object which can be used further in the program.

Disposing an Image Object

Once an Image object has been created, it must eventually be disposed using the dispose method: 

          
           
image.dispose();
