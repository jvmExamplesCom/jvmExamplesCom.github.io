Introduction to org.eclipse.swt.graphics.Image

Creating an Image Object

The most basic way of creating an Image object is done by using the Image constructor: 

          
           
Image image = new Image(display, "myimage.gif");

Once this is created, the resulting ImageData object can be used as an argument for the Image constructor: 

          
           
Image image = new Image(display, imageData);
This creates an Image object from the ImageData object which can then be used further in the program. Another way to create an Image object is by using two parameters of type InputStream and Display.
