Introduction to Java org.apache.shiro.authc.AuthenticationInfo

org.apache.shiro.authc.AuthenticationInfo is an interface from the Apache Shiro security library that provides an abstraction to represent valid user authentication information acquired by an AuthenticationRealm during an authentication attempt. AuthenticationInfo encapsulates the AuthenticationToken submitted by the user and any additional associated user data required by an application.

AuthenticationInfo Interface

The following are the methods of the AuthenticationInfo Interface.


          
           
public Object getCredentials()
public PrincipalCollection getPrincipals()
public void merge(AuthenticationInfo otherInfo)
public void setCredentials(Object credentials)
public void setPrincipals(PrincipalCollection principals)

The example class below implements the AuthenticationInfo interface.

    private PrincipalCollection principals;

    }
}
