android.graphics.drawable.TransitionDrawable

The android.graphics.drawable.TransitionDrawable class is an implementation of the Drawable class which can be used to draw a transition animation between two drawables. This class is used to implement a fading effect while transitioning between drawables.

Creating a TransitionDrawable

You can create a TransitionDrawable simply by providing two drawables as parameters to its constructor:


          
           
Drawable[] drawables = new Drawable[] {
    getResources().getDrawable(R.drawable.image_a), 
    getResources().getDrawable(R.drawable.image_b)
};

TransitionDrawable transitionDrawable = new TransitionDrawable(drawables);
