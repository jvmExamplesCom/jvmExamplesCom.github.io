androidx.core.graphics.drawable.DrawableCompat

DrawableCompat is a utility class for manipulating Drawables in Android. It allows for the manipulation of Drawables according to Drawable APIs since API level 21.

Creating A Drawable

In order to create a Drawable, you must first create a Bitmap and use its constructor to create an instance. Below is an example:


          
           
Bitmap bitmap = BitmapFactory.decodeResource(getResources(), R.drawable.my_drawable);
Drawable drawable = new BitmapDrawable(bitmap);

Extending Your Drawable

The DrawableCompat library provides a utility method that can be used to extend the existing drawable with additional properties. To extend a given drawable, simply use the 


          
           
DrawableCompat.wrap()
utility method with the following syntax: 

          
           
Drawable wrappedDrawable = DrawableCompat.wrap(drawable);

Once the drawable is wrapped, it can be manipulated with various methods that are part of the DrawableCompat APIs.

DrawableCompat.setTint()

DrawableCompat.setTintMode()

Drawable Transformation
