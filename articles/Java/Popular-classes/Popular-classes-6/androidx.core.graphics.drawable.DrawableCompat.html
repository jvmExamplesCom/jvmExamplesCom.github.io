androidx.core.graphics.drawable.DrawableCompat

DrawableCompat is a utility class for manipulating Drawables in Android. It allows for the manipulation of Drawables according to Drawable APIs since API level 21.

Creating A Drawable

In order to create a Drawable, you must first create a Bitmap and use its constructor to create an instance. Below is an example:


          
           
Bitmap bitmap = BitmapFactory.decodeResource(getResources(), R.drawable.my_drawable);
Drawable drawable = new BitmapDrawable(bitmap);

Extending Your Drawable

The DrawableCompat library provides a utility method that can be used to extend the existing drawable with additional properties. To extend a given drawable, simply use the 


          
           
DrawableCompat.wrap()
utility method with the following syntax: 

          
           
Drawable wrappedDrawable = DrawableCompat.wrap(drawable);

Once the drawable is wrapped, it can be manipulated with various methods that are part of the DrawableCompat APIs.

Changing Colors of a Drawable

With DrawableCompat, you can easily change the color of a Drawable using the 


          
           
DrawableCompat.setTint()
method. This method accepts a ColorStateList object as a parameter which defines the colors to apply to each state of the Drawable. An example usage of this method is provided below: 

          
           
ColorStateList colorStateList = ColorStateList.valueOf(ContextCompat.getColor(context, R.color.my_color));
DrawableCompat.setTintList(drawable, colorStateList);

DrawableCompat.setTintMode()
method. This method takes a PorterDuff.Mode enum value as a parameter which defines the tinting mode to apply to the Drawable. An example usage of this method is provided below: 

          
           
DrawableCompat.setTintMode(drawable, PorterDuff.Mode.MULTIPLY);

Drawable Transformation

The DrawableCompat library also provides a method for transforming a Drawable using a transformation matrix. This method, called 


          
           
DrawableCompat.setMatrix()
, takes a Matrix object as a parameter which defines the transformation applied to the Drawable. An example use of this method is provided below: 

          
           
Matrix matrix = new Matrix();
matrix.postRotate(45);
DrawableCompat.setMatrix(drawable, matrix);
