org.elasticsearch.index.query.QueryBuilder

Java Query Builder is a java library which assists in building complex queries for the Elasticsearch engine. It supports various operations like creating, updating and deleting indices, searching documents within an index, adding new documents and managing existing documents in an index. This library also provides support for query construction using the query DSL (Domain Specific Language) of the engine.

Objectives

The objective of Java Query Builder is to provide an easy way for developers to create complex query structures for Elasticsearch. The query language enables users to search documents that match specified conditions. Users can also create, update and delete indices and documents.

Features

Java Query Builder offers the following features:
  • Support for creating, updating and deleting indices.
  • Searching documents within an index by providing a query DSL.
  • Adding and managing documents within a given index.
  • Multi-index query support.
  • Cluster health monitoring.
  • Customizable search API.
  • Real-time analytics.
  • Full-text search.

Creating an Index

An index is created with the 
createIndex()
method. This method requires two parameters - the name of the index and the settings for the index. The following code example shows how to create an index with a single shard and a replication factor of 1. 

          
           
CreateIndexRequest request = new CreateIndexRequest("my_index"); 
request.settings(Settings.builder() 
        .put("index.number_of_shards", 1) 
        .put("index.number_of_replicas", 1)); 
CreateIndexResponse response = client.indices().create(request, RequestOptions.DEFAULT);

Adding Documents

Documents can be added to an index with the 
index()
method. This method requires two parameters - a document object, and an optional set of options. The following code example shows how to add a document to an index. 

          
           
Map<String, String> jsonMap = new HashMap<>(); 
jsonMap.put("name", "John Doe"); 
jsonMap.put("age", "30"); 
 
IndexRequest request = new IndexRequest("my_index") 
    .id("1") 
    .source(jsonMap); 
IndexResponse response = client.index(request, RequestOptions.DEFAULT);

Updating Documents

Documents can be updated with the 
update()
method. This method requires three parameters - the document object, the document id, and an optional set of options. The following code example shows how to update a document in an index. 

          
           
Map<String, Object> jsonMap = new HashMap<>(); 
jsonMap.put("name", "John Smith"); 
 
UpdateRequest request = new UpdateRequest("my_index", "1") 
    .doc(jsonMap); 
UpdateResponse response = client.update(request, RequestOptions.DEFAULT);

Deleting Documents

Documents can be deleted with the 
delete()
method. This method requires two parameters - the document id and an optional set of options. The following code example shows how to delete a document from an index. 

          
           
DeleteRequest request = new DeleteRequest("my_index", "1"); 
DeleteResponse response = client.delete(request, RequestOptions.DEFAULT);
