RxJava Backpressure Strategy

RxJava is a popular Java library for building reactive programs. It provides a powerful set of operators and utilities to create responsive, asynchronous applications. An important part of RxJava is dealing with backpressure. Backpressure is when your data rate is higher than your processing rate, and the system needs to adapt to it. Buffer Strategy

The Buffer strategy stores items in a buffer until the downstream consumer can process them. Items are added to the buffer up to a certain capacity, then discarded or overwritten if the buffer is already full.
