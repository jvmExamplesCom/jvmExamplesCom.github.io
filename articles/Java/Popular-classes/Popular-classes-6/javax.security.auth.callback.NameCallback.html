javax.security.auth.callback.NameCallback

The 

          
           
javax.security.auth.callback.NameCallback
class is a Callback that requests a name from an underlying security services module. The module may use this name to authorize the request, or to look up additional information.

The NameCallback class provides the following methods:

  • 
            
             
public NameCallback(String prompt)
    – Constructs a NameCallback with a prompt.
  • 
            
             
public String getPrompt()
    – Gets the prompt.
  • 
            
             
public void setName(String name)
    – Sets the name.
  • 
            
             
public String getName()
    – Gets the name.

Example

This example illustrates how to use NameCallback to prompt for a name and display it back to the user.


          
           
NameCallback nameCallback = new NameCallback("What is your name?");
nameCallback.setName("John Doe");
System.out.println("You entered: " + nameCallback.getName());
