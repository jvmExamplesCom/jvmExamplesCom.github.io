The 

          
           
javax.security.auth.callback.NameCallback

    – Constructs a NameCallback with a prompt.
  • 
            
             
public String getPrompt()
    – Gets the prompt.
  •

    – Sets the name.
  • 
            
             
public String getName()
    – Gets the name.
