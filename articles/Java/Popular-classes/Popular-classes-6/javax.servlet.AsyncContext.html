javax.servlet.AsyncContext

Java Servlet AsyncContext is an interface in the javax.servlet package that enables asynchronous communication between clients and servers in Java-based web applications. It provides methods to initiate, manage, and complete asynchronous activity such as timeouts, completions, and errors. AsyncContext simplifies the development of applications that require asynchronous processing.

Methods Overview

The javax.servlet.AsyncContext class provides the following methods:

  • start(Runnable)
    – starts the asynchronous process by running the specified Runnable.
  • 
            
             
addListener(AsyncListener)
    – adds an AsyncListener to this context.
  • complete()
    – completes the asynchronous process.
  • dispatch(String)
    – dispatches this request to a different resource.
  • getRequest()
    – returns the request for the current asynchronous process.
  • 
            
             
hasOriginalRequestAndResponse()
    – returns true if the original request and response are present, false otherwise.
  • getTimeout()
    – returns the timeout value for this context.
  • setTimeout(long)
    – sets the timeout value for this context.

HttpServletRequest request = ...; // get the Request
HttpServletResponse response = ...; //  get the Response
AsyncContext asyncContext = request.startAsync(request, response);
