Creating a MenuItem

To create a 
MenuItem
, you first need to create a new instance of the class with a Label, as seen below: 

          
           
MenuItem myMenuItem = new MenuItem("My Menu Item");

MenuItem

Image icon = new Image("myIcon.png");
MenuItem myMenuItem = new MenuItem("My Menu Item", new ImageView(icon));

MenuItem

KeyCombination keyCombination = new KeyCodeCombination(KeyCode.S, KeyCombination.CONTROL_DOWN);
myMenuItem.setAccelerator(keyCombination);
You can also set a tooltip for the 
MenuItem
. This is displayed when the user hovers their mouse over it: 

          
           
myMenuItem.setTooltip(new Tooltip("My Tooltip"));

Adding MenuItems to Menus

You can add 
MenuItems

Menu

ContextMenu
with the following code: 

          
           
menu.getItems().add(myMenuItem);
contextMenu.getItems().add(myMenuItem);

MenuItem

menu.getItems().addAll(myMenuItem);
contextMenu.getItems().addAll(myMenuItem);

MenuItem
