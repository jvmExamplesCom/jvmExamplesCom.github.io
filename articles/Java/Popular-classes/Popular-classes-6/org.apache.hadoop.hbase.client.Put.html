org.apache.hadoop.hbase.client.Put

org.apache.hadoop.hbase.client.Put is a Java class used to add data to an existing HBase table. It provides an API for specifying the content to be written and commit methods for writing it to either a single row or multiple rows within a single table. It is the easiest way to write data into HBase.

Basic Usage

The following example illustrates how to use the Put class to write data to a specific row in HBase.


          
           
Put put = new Put('rowkey');
put.addColumn('columnFamilyName', 'columnQualifier', 'columnValue');
table.put(put);

Writing Multiple Cells with a Single Put

It is also possible to write multiple columns with a single put operation by adding the various columns to the same Put instance:


          
           
Put put = new Put('rowkey');
put.addColumn('columnFamilyName', 'columnQualifier1', 'columnValue1');
put.addColumn('columnFamilyName', 'columnQualifier2', 'columnValue2');
...
put.addColumn('columnFamilyName', 'columnQualifierN', 'columnValueN');
table.put(put);

Writing Multiple Rows

The 

          
           
Table.put(List<Put>)
method allows you to write multiple Put operations with a single call. It takes a list of Put instances and performs the corresponding put operations on all of them atomically. This is useful for batch operations. 

          
           
List<Put> puts = new ArrayList<>();
Put put1 = new Put('rowkey1');
put1.addColumn('columnFamilyName', 'columnQualifier1', 'columnValue1');
puts.add(put1);
Put put2 = new Put('rowkey2');
put2.addColumn('columnFamilyName', 'columnQualifier2', 'columnValue2');
puts.add(put2);
...
Put putN = new Put('rowkeyN');
putN.addColumn('columnFamilyName', 'columnQualifierN', 'columnValueN');
puts.add(putN);
table.put(puts);

Check and Put Operations

The 
CheckAndPut
method checks for the presence of a specific value for a given column before performing a put operation. If the value does not match what is expected the put operation is not performed. This can be useful for managing data with multiple concurrent writers. 

          
           
Put put = new Put('rowkey');
put.addColumn('columnFamilyName', 'columnQualifier', 'columnValue');
boolean result = table.checkAndPut('rowkey',
                                   'columnFamilyName',
                                   'columnQualifier',
                                   'expectedValue',
                                   put);
The 
result
variable will be 
true
if the put operation was performed and 
false
if not.
