org.apache.lucene.index.IndexReader

IndexReader is an abstract class that provides access to the data associated with a Lucene index. It is used for reading data from a Lucene index. While most of the methods provided by this class serve as an interface to the underlying index, some of them provide useful utility functions. IndexReader is thread-safe and can be used in multi-threaded applications.

Methods

The following are the important methods provided by IndexReader:

  • Document: Gets the Document object for a given document ID. It returns null if the document is not found.
  • Terms: Returns a Terms object containing all terms in the given field.
  • TermDocs: This method returns the list of all documents which contain the given term.
  • TermPositions: Returns the positions at which the given term is present in the specified document.
  • reopen(): This method reopens the IndexReader, if it is already open. It may be used to refresh the state of the reader when changes have been made to the index.
  • docFreq(): This method is used to return the number of documents containing a given term.

Code Examples

Getting the Document object


          
           
IndexReader reader = IndexReader.open("index");
Document doc = reader.document(1);

Getting the list of all documents which contain the given term


          
           
IndexReader reader = IndexReader.open("index");
TermDocs termDocs = reader.termDocs("fieldName", "term");
while (termDocs.next()) {
    int docId = termDocs.doc();
    Document doc = reader.document(docId);
}

Reopening the IndexReader


          
           
IndexReader reader = IndexReader.open("index");
reader.reopen();

Getting the number of documents containing a given term


          
           
IndexReader reader = IndexReader.open("index");
int numDocs = reader.docFreq("fieldName", "term");
