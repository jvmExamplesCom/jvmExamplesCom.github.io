Introduction to AmazonServiceException in Java

The AmazonServiceException class is a type of Exception that is thrown when Amazon Web Services encounters an error response from an AWS service. In this article, we will discuss the AmazonServiceException class in Java, including its properties, methods, and usage scenarios.

Properties of AmazonServiceException in Java

The AmazonServiceException class has the following properties:

  • errorCode
    : This is the error code returned by the AWS service.
  • errorType
    : This is the type of error (e.g., Client Error or Server Error).
  • message
    : This is the error message returned by the AWS service.
  • serviceName
    : This is the name of the service that returned the error response.
  • requestId
    : This is the request ID associated with the error response.
  • statusCode
    : This is the HTTP status code returned by the AWS service.

Methods of AmazonServiceException in Java

The AmazonServiceException class has the following methods:

  • getErrorCode()
    : This method returns the error code associated with the exception.
  • getErrorType()
    : This method returns the type of error associated with the exception.
  • getMessage()
    : This method returns the error message associated with the exception.
  • getServiceName()
    : This method returns the name of the service associated with the exception.
  • getRequestId()
    : This method returns the request ID associated with the exception.
  • getStatusCode()
    : This method returns the HTTP status code associated with the exception.

Example of using AmazonServiceException in Java

The following example shows how to use the AmazonServiceException class in Java:


          
           
try {
    // Call AWS service
    // ...
} catch (AmazonServiceException e) {
    System.out.println("Error Code: " + e.getErrorCode());
    System.out.println("Error Type: " + e.getErrorType());
    System.out.println("Message: " + e.getMessage());
    System.out.println("Service Name: " + e.getServiceName());
    System.out.println("Request ID: " + e.getRequestId());
    System.out.println("Status Code: " + e.getStatusCode());
}

In this example, we are catching an AmazonServiceException and printing out the five properties associated with it. The output of this code would look something like this:

Message: The parameter is invalid
Service Name: Amazon EC2
