Java com.amazonaws.services.s3.model.ObjectMetadata

Java provides a way to store and manage object metadata via the com.amazonaws.services.s3.model.ObjectMetadata class. This class provides a convenient way to store, retrieve and manipulate S3 object metadata.

Creating an ObjectMetadata Instance

Creating an instance of ObjectMetadata is easy. It requires only two parameters:


          
           
ObjectMetadata metadata = new ObjectMetadata(String bucketName, String key);

Where the bucketName is the name of the S3 bucket where the object is stored, and the key is the unique identifier for the that object within the S3 bucket.

Accessing Object Metadata

Once an ObjectMetadata instance is created, it can be used to access various information about the stored object, such as its size, content type, content encoding, cache control, user metadata and more. Here's an example of how to get the object's size:


          
           
long size = metadata.getContentLength();

In this example, the size variable will contain the size of the object in bytes.

Manipulating Object Metadata

The ObjectMetadata class also provides methods for manipulating object metadata. For example, you can set the object's content type using the following code:


          
           
metadata.setContentType("text/plain");

You can also set any other custom metadata, such as cache control or user-defined metadata:


          
           
metadata.setCacheControl("max-age=3600");
metadata.addUserMetadata("description", "This is a plain text file.");

These changes can then be saved to the S3 bucket with a call to the save() method:


          
           
metadata.save();
