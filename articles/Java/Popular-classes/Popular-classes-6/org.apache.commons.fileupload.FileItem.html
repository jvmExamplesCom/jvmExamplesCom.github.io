Overview of org.apache.commons.fileupload.FileItem

org.apache.commons.fileupload.FileItem is a Java class that enables efficient handling of file uploads in applications. It provides an abstraction over FormFields and file items. With FileItem, users can manage multiple file uploads and access the data without having to move them to disk first.

FileItem encapsulates a single uploaded file, including the file data and its associated meta-data such as filename, content type, and field name.

Using FileItem

FileItem requires two steps to use: instantiating the class and using its methods.

Instantiation

To instantiate the FileItem class, use the following syntax:


          
           
<fileitem fileitem="new" fileitem();></fileitem>

Methods

The methods associated with FileItem allow users to manipulate files and access the data they contain. The available methods include:

  • getName()
    : Returns the name of the file item.
  • getFieldName()
    : Returns the name of the form field for the file item.
  • getString()
    : Returns the contents of the file item as a String.
  • getContentType()
    : Returns the content type of the file item.
  • getSize()
    : Returns the size of the file item, in bytes.
  • getInputStream()
    : Returns the file item's contents as an InputStream.
  • write()
    : Writes the contents of the file item to a file.

Example

The following example demonstrates how to use FileItem to read and write files. This example reads the contents of a file, displays the contents to the screen, and then writes the contents back to a file:


          
           
FileItem fileItem = new FileItem();

// Read contents of file into a String
String contents = fileItem.getString();

// Print contents to screen
System.out.println(contents);

// Write contents of file back to a file
fileItem.write("output.txt");

The output.txt file should now contain the same contents as the original file.
