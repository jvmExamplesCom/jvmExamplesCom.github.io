Understanding AsyncResult

At its core, the 

          
           
AsyncResult

Asynchronous operations indicate that the application does not have to wait for the operation to complete before continuing with other tasks. This means that the application can be more responsive and can better utilize available resources. The 

          
           
AsyncResult
object is a type of Future. It encapsulates the result of the operation and provides methods for accessing that result. Depending on the outcome of the operation, the 

          
           
AsyncResult
