Overview

Creating a Camera Instance

Camera camera = new Camera();

To capture an image, use the takePicture() method. The following example captures an image in JPEG format: 

          
           
Camera.Parameters parameters = new Camera.Parameters();

camera.takePicture(null, null, parameters);

camera.setCrop(10, 10, 5);

Scaling Images

camera.setScale(20, 20);

Rotating Images

camera.setRotation(45);
