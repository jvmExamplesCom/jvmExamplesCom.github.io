Java: net.minecraft.block.Block

The net.minecraft.block.Block class is the main class in the Minecraft modding API that represents blocks in the game. It provides an interface for creating and manipulating blocks, as well as providing access to a variety of different properties such as block type, material, metadata, etc. As part of the modding API, the Block class can be used to implement custom blocks, or modify existing ones, as well as manipulate their properties.

Creating a new block

To create a new block, you must first define a new subclass of Block. This can be done using the following code example: 

          
           
public class MyBlock extends Block {
    public MyBlock() {
        super(Properties.create(Material.ROCK)
            .hardnessAndResistance(2.0f)
            .lightValue(15));
    }
}
The MyBlock class extends the Block class, and passes in a Properties object defining the material, hardness, resistance, and light value of the block.

Manipulating Blocks
