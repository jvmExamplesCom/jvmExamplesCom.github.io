javax.enterprise.event.Observes

The 

          
           
javax.enterprise.event.Observes
annotation is a Java Enterprise Edition (EE) annotation used for observing an event with the 
@Observes
annotation. The annotation is typically used to observe events in Java EE applications and can be used in conjunction with CDI beans that are responsible for events.

Basic Example

The following code example illustrates how to observe an event with the 

@Observes
annotation: 

          
           
public class EventHandler {
    public void onEvent(@Observes MyEvent event) {
        // Do something with event object
    }
}

Notification Scenario

The 
@Observes
annotation can also be used to observe events in a notification scenario. In this example, an event is fired when an order is placed. An observer notification is sent to the administrator. 

          
           
public class OrderController {

    @Inject
    private Event<OrderPlaced> orderPlacedEvent;

    public void placeOrder(Order order) {
        OrderPlaced orderPlaced = new OrderPlaced(order);
        orderPlacedEvent.fire(orderPlaced);
    }
}

public class AdministratorNotification {
    public void notifyOrderPlaced(@Observes OrderPlaced orderPlaced) {
        // Send email or other notification to administrator
    }
}

onMyOtherEvent
