android.database.sqlite.SQLiteQueryBuilder

The SQLiteQueryBuilder class is an essential tool for applications using an SQLite database on the Android platform. It is used to construct an SQL query and then execute it against a SQLiteDatabase. The resulting data cursor can then be used to access the rows of data returned by the query. This class makes it easy to create complex SQL queries in a few lines of code without having to manually write each part of the query.

Using SQLiteQueryBuilder

Using SQLiteQueryBuilder is a simple process. First, create a new instance of the class and assign it to a variable:


          
           
SQLiteQueryBuilder qb = new SQLiteQueryBuilder();

Next, call the setTables method to specify which table(s) you wish to query. This method takes a string argument with the names of the tables separated by commas. For example:


          
           
qb.setTables("users, user_info");

Next, you can use the appendWhere and appendOrderBy methods to specify the criteria for selecting rows and how the results should be sorted:


          
           
qb.appendWhere("user_info.name = users.name");
qb.appendOrderBy("users.name ASC");

Finally, you can call the buildQuery method to generate the SQL query and execute it using a SQLiteDatabase instance:


          
           
String sqlQuery = qb.buildQuery();
Cursor cursor = db.rawQuery(sqlQuery,null);

The resulting Cursor can then be used to access the result set returned by the query.

Example

Here is an example of a complete query constructed with SQLiteQueryBuilder:


          
           
SQLiteQueryBuilder qb = new SQLiteQueryBuilder();
qb.setTables("users, user_info");
qb.appendWhere("user_info.name = users.name");
qb.appendOrderBy("users.name ASC");
String sqlQuery = qb.buildQuery();
Cursor cursor = db.rawQuery(sqlQuery,null);

This query selects all rows from the ‘users’ and ‘user_info’ tables where the name field in the user_info table matches the name field in the users table, and sorts the results by the name field in the users table in alphabetical order.
