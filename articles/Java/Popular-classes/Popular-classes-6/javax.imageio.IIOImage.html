javax.imageio.IIOImage

The javax.imageio.IIOImage class is a lightweight container for image data and image metadata including thumbnails, which allows for deferred reading and writing of images. IIOImage is designed to work with the Java Image I/O Framework and provides an easy way to manipulate images and their associated metadata. This class can be used to represent a buffered image, a set of thumbnails, or a combination of both.
