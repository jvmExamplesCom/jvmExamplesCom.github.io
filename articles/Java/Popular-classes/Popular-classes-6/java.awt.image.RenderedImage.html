Java.awt.image.RenderedImage

Java.awt.image.RenderedImage is an interface from java.awt.image package which provides an image representation created from a rendering (or producing) process. It is a sub-interface of java.awt.image.renderable.RenderableImage, and it extends the interface java.awt.image.BufferedImage.

RenderedImage objects are used to represent images that may be either stored in a file or stored in memory. The RenderedImage class supports a variety of raster data formats. Images are either produced or created by a Rendering Process. This process can be implemented as a program or application written in Java or as a native program written in another language.
