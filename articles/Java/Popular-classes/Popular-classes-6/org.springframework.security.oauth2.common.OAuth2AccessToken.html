Introduction to org.springframework.security.oauth2.common.OAuth2AccessToken

OAuth 2.0 is a widely used authorization standard that allows users to securely grant access to third party applications. OAuth 2.0 relies on Access Tokens which are generated by an authorization server and then passed to the application for authorization.
The org.springframework.security.oauth2.common.OAuth2AccessToken class is the Java representation of an OAuth 2.0 Access Token. This class provides several utility methods to manipulate OAuth 2.0 Access Tokens as well as several static factory methods to parse and generate valid Access Tokens.
