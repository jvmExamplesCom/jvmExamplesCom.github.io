What is java.io.StreamCorruptedException?

java.io.StreamCorruptedException is an exception that is thrown when control information in a stream is corrupted. It is raised when a Stream object encounters corrupted data, indicating that the stream is no longer valid and cannot be used for further operations. It extends the IOException class.

    //do something with the stream
} catch (IOException e) {
    System.out.println("IO Exception was thrown. Message: "+e.getMessage());
} catch (StreamCorruptedException sce) {
    System.out.println("StreamCorruptedException was thrown. Message: "+sce.getMessage());
