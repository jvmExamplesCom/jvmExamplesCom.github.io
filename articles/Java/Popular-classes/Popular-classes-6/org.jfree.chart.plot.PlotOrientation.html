org.jfree.chart.plot.PlotOrientation

org.jfree.chart.plot.PlotOrientation is a Java class that is used to define different types of chart orientation. There are two static variables available: 

          
           
VERTICAL
and 

          
           
HORIZONTAL
. These variables are of the type org.jfree.chart.axis.CategoryAxis, which allows for different orientations for a given chart. This makes it possible to show both horizontal and vertical charts within the same application.

Example 1: Create a Vertical Chart

In this example, we will create a simple XY line chart with a 

          
           
VERTICAL
orientation. 

          
           
public static void main(String[] args) {
    // Create a dataset
    DefaultCategoryDataset dataset = new DefaultCategoryDataset();

    // Add data to the set
    dataset.addValue(1.2, "Series 1", "Row 1");
    dataset.addValue(3.4, "Series 1", "Row 2");
    dataset.addValue(5.6, "Series 1", "Row 3");

    // Create the chart
    JFreeChart chart = ChartFactory.createLineChart("My Chart", 
        "X Axis Label", "Y Axis Label", dataset, 
        PlotOrientation.VERTICAL, false, false, false);

    // Create the frame
    ChartFrame frame = new ChartFrame("My Chart Frame", chart);
    frame.pack();
    frame.setVisible(true);

}
