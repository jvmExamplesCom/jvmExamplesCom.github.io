android.text.style.UnderlineSpan

Android api's UnderlineSpan class is a subclass of Character Style and MetricAffectingSpan class that allows you to set an underline to a specified region in a text. It underlines a part of the text or the entire text depending on the parameter you have passed while initializing UnderlineSpan instance.

Example

Here is an example demonstrating usage of the UnderlineSpan class: 

          
           
SpannableString str= new SpannableString("Example for UnderlineSpan");
  UnderlineSpan underlineSpan = new UnderlineSpan();
  str.setSpan(underlineSpan, 0, str.length(), 0);
  TextView textView = findViewById(R.id.text);
  textView.setText(str);
In the above example, we have created a SpannableString and initialized an UnderlineSpan object. Then we have set the span using setSpan method of the SpannableString. This applies the underline style to the entire string.

Usage

UnderlineSpan can be used to underline any part of text by passing the starting index and end index of the part that needs to be underlined. For example: 

          
           
SpannableString str= new SpannableString("Example for UnderlineSpan");
  UnderlineSpan underlineSpan = new UnderlineSpan();
  str.setSpan(underlineSpan, 0, 5, 0);
  TextView textView = findViewById(R.id.text);
  textView.setText(str);
In the above example, only the substring "Example" will be underlined as we passed the start index as 0 and end index as 5. 

          
           
It is also possible to set a color to the underline. Here is an example:

          

          
           
SpannableString str= new SpannableString("Example for UnderlineSpan");
  UnderlineSpan underlineSpan = new UnderlineSpan(Color.GREEN);
  str.setSpan(underlineSpan, 0, str.length(), 0);
  TextView textView = findViewById(R.id.text);
  textView.setText(str);
In this example, the underline will be in green color.

Conclusion

UnderlineSpan class provides an easy way to set the underline style to a part or whole of a text. This can come in handy when you want to highlight some specific part of the text in a textview.
