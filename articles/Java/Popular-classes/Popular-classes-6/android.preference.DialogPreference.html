The DialogPreference class allows the developer to specify a custom layout for the dialog box. The layout can contain any type of UI elements, such as text fields, radio buttons, checkboxes, etc. In addition to the layout, DialogPreference also allows the developer to specify an OnClickListener. This listener will be fired each time the user changes a preference in the dialog box.
