android.widget.SearchView

Android consists of a rich and comprehensive set of widget classes that allow you to build sophisticated, interactive user interfaces. One such widget class is android.widget.SearchView. This class provides an interface for text entry and search submission that looks like a search box. It also allows for adding extra features such as auto-completion, voice search, and much more.

Overview

The SearchView widget is a TextView that supports voice input, auto-complete suggestions, and user text entry. The SearchView widget can accept up to two parameters, a search query and optional arguments to configure the search bar. SearchView offers several features such as a text field for entering search text, a "go" button for submitting the search query and a drop-down menu for providing a list of recent searches.

<SearchView 
    android:layout_width="match_parent"
    android:layout_height="wrap_content" />
