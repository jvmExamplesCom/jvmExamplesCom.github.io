android.widget.SearchView

Android consists of a rich and comprehensive set of widget classes that allow you to build sophisticated, interactive user interfaces. One such widget class is android.widget.SearchView. This class provides an interface for text entry and search submission that looks like a search box. It also allows for adding extra features such as auto-completion, voice search, and much more.

Overview

The SearchView widget is a TextView that supports voice input, auto-complete suggestions, and user text entry. The SearchView widget can accept up to two parameters, a search query and optional arguments to configure the search bar. SearchView offers several features such as a text field for entering search text, a "go" button for submitting the search query and a drop-down menu for providing a list of recent searches.

Usage

To create a SearchView widget in your layout, you need to include the following XML code: 

          
           
<SearchView 
    android:layout_width="match_parent"
    android:layout_height="wrap_content" />
This will create a basic SearchView widget with no additional features enabled. To start using the SearchView widget, set a listener on the widget to listen for user interaction. This can be done by setting a OnQueryTextListener. 

          
           
SearchView searchView = findViewById(R.id.search_view);
searchView.setOnQueryTextListener(onQueryTextListener);
The OnQueryTextListener will be called when the user submits their text or selects a suggestion from the drop-down menu. 

          
           
searchView.setOnQueryTextListener(new SearchView.OnQueryTextListener() {
    @Override
    public boolean onQueryTextSubmit(String query) {
        // handle query
        return true;
    }

    @Override
    public boolean onQueryTextChange(String newText) {
        // handle new text 
        return false;
    }
});

searchView.setOnSuggestionListener(new SearchView.OnSuggestionListener() {
    @Override
