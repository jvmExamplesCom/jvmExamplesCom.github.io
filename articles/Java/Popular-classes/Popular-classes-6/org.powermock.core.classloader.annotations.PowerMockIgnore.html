The 
PowerMockIgnore
annotation is a powerful class from the PowerMock library which provides the ability to ignore certain classes when running tests. It allows tests to be run without dealing with issues caused by the presence of certain classes, such as those from a third-party library or framework. PowerMockIgnore
to an entire test class:

}
