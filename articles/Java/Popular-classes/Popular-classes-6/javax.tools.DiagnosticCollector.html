javax.tools.DiagnosticCollector

The javax.tools.DiagnosticCollector is a utility class that can be used to collect diagnostics reported by the Java compiler. It provides access to all of the compiler’s diagnostic messages and the source code locations in which they occur. The DiagnosticCollector can be used in conjunction with the JavaCompiler to compile Java programs and detect any errors or warnings that may exist.

Code Examples

The following code examples demonstrate how to use the DiagnosticCollector in a program.


          
           
// Create the DiagnosticCollector
DiagnosticCollector<JavaFileObject> diagnostics = new DiagnosticCollector<JavaFileObject>();

// Pass it to the JavaCompiler
JavaCompiler compiler = ToolProvider.getSystemJavaCompiler();
CompilationTask task = compiler.getTask(null, null, diagnostics, null, null, compilationUnits);

// Run the compilation task
task.call();

// Iterate over the diagnostics to print out the messages
for (Diagnostic<? extends JavaFileObject> diagnostic : diagnostics.getDiagnostics()) {
   System.out.println(diagnostic.getMessage(null));
}
