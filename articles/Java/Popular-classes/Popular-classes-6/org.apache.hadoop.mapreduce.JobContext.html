org.apache.hadoop.mapreduce.JobContext

The org.apache.hadoop.mapreduce.JobContext class is a key component of the Apache Hadoop MapReduce framework. It provides access to data shared between all of the tasks in a job, as well as job-wide configuration parameters.

Creating a JobContext Instance

A JobContext instance can be created using the static 

          
           
JobContext.getInstance()
method. The method takes two parameters: a Configuration instance and a JobID, both of which are required for creating a JobContext instance.

Methods in JobContext

The org.apache.hadoop.mapreduce.JobContext class provides the following methods.

getConfiguration()

This method returns the Configuration object passed in during the JobContext instance's creation.

getJobID()

This method returns the JobID of the JobContext instance.

getInputFormatClass()

This method returns the InputFormat class associated with the job. The InputFormat class is responsible for extracting data from an input source and creating Key-Value pairs, which are then processed by the Mapper.

getMapperClass()

This method returns the Mapper class associated with the job. The Mapper class is responsible for processing each pair of Key-Value created by the InputFormat.

getReducerClass()

This method returns the Reducer class associated with the job. The Reducer class is responsible for combining the output of the Mapper and generating the job's final output.

getOutputFormatClass()

This method returns the OutputFormat class associated with the job. The OutputFormat class is responsible for taking the job's final output and writing it to an output source.

Example Usage

To demonstrate how to use the org.apache.hadoop.mapreduce.JobContext class, we will create a sample application that prints out the configuration parameters for a job. 

          
           
// Create a new JobContext
JobContext jobContext = JobContext.getInstance(configuration, jobId);

// Get the configuration parameters
Configuration conf = jobContext.getConfiguration();

// Print out the configuration parameters
for (Map.Entry<String, String> entry : conf) {
    System.out.println(entry.getKey() + " = " + entry.getValue());
}
