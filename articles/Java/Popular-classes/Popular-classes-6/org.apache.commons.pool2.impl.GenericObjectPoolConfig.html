org.apache.commons.pool2.impl.GenericObjectPoolConfig

The org.apache.commons.pool2.impl.GenericObjectPoolConfig class is a configuration class that is used to configure a GenericObjectPool. Configuration Options

The org.apache.commons.pool2.impl.GenericObjectPoolConfig class has several configuration options that can be set to customize the behavior of the GenericObjectPool.

maxIdle:

This sets the maximum number of objects that can remain idle in the pool without being evicted. If this is set too low, then the pool may run out of idle objects and have to create new instances. If this is set too high, then the pool may contain too many idle objects and waste resources.

minIdle:

This sets the minimum number of objects that should remain idle in the pool. This will allow the pool to maintain a minimum level of idle objects even if there are few requests for new instances.

numTestsPerEvictionRun:

minEvictableIdleTimeMillis:
