Introduction to javax.net.ssl.SSLServerSocketFactory

SSLServerSocket

SSLServerSocketFactory
class can be used to customize the settings used for SSL connections.

Creating an SSLServerSocketFactory Instance

Creating an instance of 

          
           
SSLServerSocketFactory

getDefault()
static method. 

          
           
SSLServerSocketFactory ssf = SSLServerSocketFactory.getDefault();
