Overview of org.springframework.context.support.PropertySourcesPlaceholderConfigurer

The org.springframework.context.support.PropertySourcesPlaceholderConfigurer class is a configurer for property sources that resolves placeholder values from property sources in the Spring Environment and its owned property sources. This configurer is most often used in conjunction with PropertySourcesPlaceholderConfigurer to resolve properties from multiple sources including system properties, environment variables, and JavaBean-style properties files.

Creating a PropertySourcesPlaceholderConfigurer Object

To create a PropertySourcesPlaceholderConfigurer object, use the following code:


          
           
PropertySourcesPlaceholderConfigurer configurer = 
  new PropertySourcesPlaceholderConfigurer();

Configuring Property Sources

The PropertySourcesPlaceholderConfigurer class has several methods for configuring property sources such as setLocation() and addPropertySource(). To configure the location of a property source, use the setLocation() method:


          
           
configurer.setLocation(
    "classpath:com/mycompany/myapp/resources/properties.xml");

PropertySource myPropertySource = 
  new PropertySource("myPropertySource", 
                     new ClassPathResource("my.properties"));

Resolving Placeholder Values
