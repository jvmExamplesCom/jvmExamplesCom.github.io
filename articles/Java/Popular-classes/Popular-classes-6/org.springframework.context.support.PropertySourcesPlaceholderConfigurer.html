Overview of org.springframework.context.support.PropertySourcesPlaceholderConfigurer

The org.springframework.context.support.PropertySourcesPlaceholderConfigurer class is a configurer for property sources that resolves placeholder values from property sources in the Spring Environment and its owned property sources. PropertySourcesPlaceholderConfigurer configurer = 
  new PropertySourcesPlaceholderConfigurer();
