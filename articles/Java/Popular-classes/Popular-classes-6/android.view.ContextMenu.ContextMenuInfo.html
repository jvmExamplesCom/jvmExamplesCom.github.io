Overview of android.view.ContextMenu.ContextMenuInfo

The ContextMenuInfo class is a part of the Android framework and provides an interface for getting contextual information from a specific context menu. It contains two methods: getPosition() and getMenuInfo(). The getMenuInfo() method returns additional information about the context menu item.
