CircleImageView in Java

The CircleImageView is a specialized view available in the de.hdodenhof.circleimageview.CircleImageView package used to create circular images that can be displayed on the Android platform. It can be used to display profile pictures and thumbnails or any other image resource. The CircleImageView class provides the following features:

  • Support for circular images with any size.
  • Adjustable corner radius.
  • Border color, width and dash support.
  • Elevation shadow support.
  • Animatable between states.

Using the CircleImageView

To use the CircleImageView, first add a dependency to your app's build.gradle file:


          
           
implementation 'de.hdodenhof:circleimageview:3.1.0'

The CircleImageView can then be added in the layout xml file as follows:


          
           
<de.hdodenhof.circleimageview.CircleImageView
  android:id="@+id/profile_picture"
  android:layout_width="100dp"
  android:layout_height="100dp"/>

In the activity Java code, a background image can be set as follows:


          
           
CircleImageView profilePicture = findViewById(R.id.profile_picture);

profilePicture.setImageResource(R.drawable.background_image);

We can also adjust the corner radius of the image with the following code:


          
           
profilePicture.setCornerRadius(16f);

Styling the CircleImageView

The CircleImageView can be further styled using various attributes as follows:

  • Setting the border color with 
    
            
             
android:civ_border_color="@color/border_color"
  • Setting the border width with 
    
            
             
android:civ_border_width="4dp"
  • Setting the border dash width with 
    
            
             
android:civ_border_dash_width="4dp"
  • Disabling elevation shadows with 
    
            
             
android:elevation="0dp"

These attributes can also be set in the Java code using the corresponding methods.

Animating the CircleImageView

The CircleImageView supports animating between two states by overriding the animateToState() method. To animate from the default state to the pressed state, the following code can be used:


          
           
profilePicture.animateToState(CircleImageView.State.PRESSED);

The animation duration and scale factor can also be specified when animating. This can be done in XML using the civ_animation_duration and civ_animation_scale attributes, respectively, or in the Java code using the corresponding methods.

With its support for circular images, border customization and animation, the CircleImageView makes it easy to add stylish, dynamic images to Android applications.
