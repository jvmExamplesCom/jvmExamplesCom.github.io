Introduction to org.lwjgl.opengl.GL11.

org.lwjgl.opengl.GL11 is a class from the Lightweight Java Game Library (LWJGL) that enables access to the OpenGL API from Java programs. It provides a collection of methods for 2D and 3D graphics processing by using OpenGL, which is the industry standard for rendering interactive 3D graphics.

Methods

The GL11 class provides many methods for scaling, translating, drawing textures, quads, and other meshes, as well as methods for setting the viewport, controlling lighting, and creating shaders.

Examples

Here are a few examples of methods in the GL11 class:


          
           
GL11.glClear(GL11.GL_COLOR_BUFFER_BIT);
This method clears the color buffer bitpath. 

          
           
GL11.glViewport(0, 0, width, height);

This method sets the viewport for containing the rendered image.


          
           
GL11.glBegin(GL11.GL_QUADS);
This method begins drawing quads.

This method sets the matrix mode to projection.
