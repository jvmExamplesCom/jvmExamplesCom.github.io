An Introduction to com.google.android.gms.maps.OnMapReadyCallback

The 

          
           
com.google.android.gms.maps.OnMapReadyCallback

What Does the OnMapReadyCallback Interface Do?

The 

          
           
OnMapReadyCallback
interface provides a callback to detect when a Google Map has been successfully loaded and is ready to be used. The interface requires the 

          
           
onMapReady
method to be implemented, which is called once the map is ready. This method takes an argument of type 

          
           
GoogleMap
. 

          
           
public class MyActivity extends Activity implements OnMapReadyCallback {
    GoogleMap mMap;

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);

        // Obtain the SupportMapFragment and get notified when the map is ready to be used.
        SupportMapFragment mapFragment = (SupportMapFragment) getSupportFragmentManager()
                .findFragmentById(R.id.map);
        mapFragment.getMapAsync(this);
    }

    @Override
    public void onMapReady(GoogleMap googleMap) {
        mMap = googleMap;
    }
}
