org.springframework.web.client.RestClientException

The org.springframework.web.client.RestClientException is an unchecked exception that is thrown when a Rest Client fails due to a request or response processing failure. It is thrown from remote resources when the status code is not successful in the client-side application. This exception can be used to catch early any issues with the communications between the client and the server.
