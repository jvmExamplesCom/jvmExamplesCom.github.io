android.media.ThumbnailUtils

The 

          
           
android.media.ThumbnailUtils
is an utility class which provides methods for creating and manipulating images, specifically thumbnails of various sizes. It is used to create thumbnail images from larger images, including video files. This class can also be used to crop existing thumbnails or create a new one from a raw image data stream.

Methods

createImage(Bitmap, Configuration)


          
           
Bitmap thumbnail = ThumbnailUtils.createImage(myBitmap, config);
This method creates a thumbnail image from an existing Bitmap and specified Configuration object. The resulting image is a scaled-down version of the original bitmap. The configuration information determines the size and quality of the resulting image.

createThumbnail(ImageData, int)


          
           
Bitmap thumbnail = ThumbnailUtils.createThumbnail(imageData, size);
This method creates a thumbnail image from raw image data in the form of an ImageData object, and the specified size. The resulting image is a scaled-down version of the original image data.

decodeFile(String, Options)


          
           
Bitmap thumbnail = ThumbnailUtils.decodeFile(filePath, options);
This method decodes a file located at the specified path using options provided as part of a Options object. The resulting image is a scaled-down version of the original image stored in the specified file.

extractThumbnail(Bitmap, int, int)


          
           
Bitmap thumbnail = ThumbnailUtils.extractThumbnail(bitmap, width, height);
This method extracts a portion of an existing Bitmap, resulting in a new image which has been scaled down to the specified width and height. The portion of the image extracted is determined by the position of the original bitmap, as well as the width and height parameters.

getRotation(int)


          
           
int rotation = ThumbnailUtils.getRotation(orientation);
This method returns the rotation angle for an image with the specified EXIF orientation value. This value can be used to rotate the image when displaying it.

Example

The following example shows how to create a thumbnail image from a larger image stored in a file on the device's external storage. 

          
           
String filePath = "/sdcard/example.jpg";
 Options options = new Options();
 options.inSampleSize = 8;
 Bitmap myBitmap = ThumbnailUtils.decodeFile(filePath, options);
 int width = 100;
 int height = 100;
 Bitmap thumbnail = ThumbnailUtils.extractThumbnail(myBitmap, width, height);
In this example, the filePath variable contains the path of the larger image stored in the device's external storage. The options object is created with an inSampleSize value of 8. This means that the image will be downscaled by a factor of 8x before being decoded into a bitmap object. The decodeFile method is then used to decode the image from the file path into a Bitmap object. Finally, the extractThumbnail method is used to extract a thumbnail image from the Bitmap object, with a width and height of 100 pixels. The resulting thumbnail image is stored in the thumbnail variable.
