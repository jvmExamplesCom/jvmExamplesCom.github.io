org.bukkit.event.player.PlayerJoinEvent

The org.bukkit.event.player.PlayerJoinEvent is a Bukkit event that is fired when a player joins the server. This event can be used to perform additional tasks upon login, such as setting up the player's environment or updating their state.

When this event is fired, it contains information about the player and the type of connection they are using, so it can be used to detect when a player is joining from a specific IP or range, for example.

Usage Examples

We can make use of the PlayerJoinEvent to perform various tasks when a player logs in. Here are some examples:

Logging Player Login

We can use the PlayerJoinEvent to log the players who join the server. This can help us keep track of who is playing on the server.

public void onPlayerJoin(PlayerJoinEvent event) {
    Player player = event.getPlayer();
    String playerName = player.getName();
    String ipAddress = player.getAddress().getAddress().toString();

}
