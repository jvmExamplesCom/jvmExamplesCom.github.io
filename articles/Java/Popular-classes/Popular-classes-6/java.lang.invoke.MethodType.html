java.lang.invoke.MethodType Overview

Java.lang.invoke.MethodType is a class in the java.lang.invoke package that represents instances of Java method types. It provides access to methods and constructors, along with other related information.

MethodType objects are immutable, so once they are created they cannot be changed. They can be used to access methods and constructors, and invoke them or get their signature.

Creating a MethodType Object

A MethodType object can be created using the static method 


          
           
MethodType.methodType(Class<?> rtype, Class<?>... ptypes)
. The first parameter specifies the return type of the method/constructor, and the array of second parameters specifies the method/constructor argument types. 

          
           
MethodType mt = MethodType.methodType(String.class, int.class);

Invoking Methods and Constructors

Once a MethodType object has been created, it can be used to access and invoke a method or constructor. The 

findStatic()
and 
findVirtual()
methods can be used to find a method given its name and type, and 
invoke()
can be used to invoke a found method. Similarly, 
findConstructor()
can be used to find a constructor, and 

          
           
invokeWithArguments()
can be used to invoke a constructor. 

          
           
// Invoke a static method
MethodHandle mh = MethodHandles.lookup().findStatic(String.class, "valueOf", mt);
String result = (String) mh.invoke(1);
System.out.println(result); // Outputs "1"

// Invoke a constructor
MethodHandle mh = MethodHandles.lookup().findConstructor(String.class, mt);
String result = (String) mh.invokeWithArguments(1);
System.out.println(result); // Outputs "1"

Getting the Signature of a Method or Constructor

The static 
type()
method can be used to get the signature of a method or constructor. It takes the return type and argument types as parameters. For example: 

          
           
MethodType mt = MethodType.methodType(String.class, int.class);

The signature of this method type can be retrieved using the following code:


          
           
String signature = mt.type().toString();
System.out.println(signature); // Outputs "String(int)"
