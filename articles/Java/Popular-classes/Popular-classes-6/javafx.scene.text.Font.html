This class provides methods for setting the font family and size of text elements in a Scene, as well as a variety of options for applying font styles such as bold, italic, and underline.

Creating a Font Object

The simplest way to create a new font object is using the static Font.font() method. This method takes two parameters: the font family name, and the font size. For example, the following code creates a font object with the family name “Verdana” and the size of 24 points: 

          
           
Font font = Font.font("Verdana", 24);
Alternatively, you can also specify a font style when creating a font object. The following example creates a font object with the font family “Verdana”, font size of 24 points, and the bold style applied: 

          
           
Font font = Font.font("Verdana", FontWeight.BOLD, 24);

Applying a Font to a Text Element

Once you’ve created a font object, you can apply it to any text elements in a scene. For example, the following code applies the previously created font to a Text object: 

          
           
Text text = new Text("This is my text");
text.setFont(font);
You can also apply a font to a TextFlow object, or any other node that contains text.

Using Named Fonts

In addition to creating font objects manually, you can also use named fonts that are installed on the system. To do this, you can use the Font.getFont() method. This method takes a single parameter: the font family name. For example, the following code creates a font object with the family name “Courier”: 

          
           
Font font = Font.getFont("Courier");
You can also specify a font size and style when using named fonts. For example, the following code creates a font object with the family name “Courier”, font size of 24 points, and the bold style applied: 

          
           
Font font = Font.getFont("Courier", FontWeight.BOLD, 24);

Getting the Default Font

If you don’t specify a font when creating a font object, the default font will be used. You can get the default font using the Font.getDefault() method. For example: 

          
           
Font font = Font.getDefault();
