javax.net.ssl.SSLParameters

The javax.net.ssl.SSLParameters class is a Java API used to configure secure socket layers (SSL) through an established connection. It is a part of the Java Secure Socket Extension (JSSE) and is used to set parameters that support different types of cipher suites and protocols. The following is an example of using the SSLParameters API to configure a secure socket layer connection.

String[] cipher_suites = {"TLS_ECDHE_ECDSA_WITH_AES_128_GCM_SHA256"};
String[] protocols = {"TLSv1.2"};
SSLParameters params = new SSLParameters(cipher_suites);

// Close the connection
