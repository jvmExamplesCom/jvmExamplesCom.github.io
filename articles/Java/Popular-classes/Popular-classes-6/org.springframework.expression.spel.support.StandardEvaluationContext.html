org.springframework.expression.spel.support.StandardEvaluationContext

The StandardEvaluationContext class is a powerful and versatile Java object used in the Spring Framework for evaluating expressions written in SpEL (Spring Expression Language). It provides a set of methods for setting context variables that can be used during evaluation of the expression. This allows for the flexibility to create different contexts depending on the requirement of the application.

Creating a StandardEvaluationContext Object

A StandardEvaluationContext object can be created by calling the constructor of the org.springframework.expression.spel.support.StandardEvaluationContext class:


          
           
StandardEvaluationContext context = new StandardEvaluationContext();

Setting Variables in the Context

Using the context created above, variables can be set in the context by calling the setVariable() method. The setVariable() method takes two arguments, the first being the name of the variable to set and the second being the value of the variable.


          
           
context.setVariable("price", 10);

The setVariable() method can also be used to define properties on a bean by specifying the name of the bean followed by the property name and the value:


          
           
context.setVariable("product.cost", 20);
