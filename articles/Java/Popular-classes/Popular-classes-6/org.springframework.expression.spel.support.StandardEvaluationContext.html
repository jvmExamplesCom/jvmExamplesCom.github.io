org.springframework.expression.spel.support.StandardEvaluationContext

The StandardEvaluationContext class is a powerful and versatile Java object used in the Spring Framework for evaluating expressions written in SpEL (Spring Expression Language). A StandardEvaluationContext object can be created by calling the constructor of the org.springframework.expression.spel.support.StandardEvaluationContext class:


          
           
StandardEvaluationContext context = new StandardEvaluationContext();

context.setVariable("price", 10);

The setVariable() method can also be used to define properties on a bean by specifying the name of the bean followed by the property name and the value:
