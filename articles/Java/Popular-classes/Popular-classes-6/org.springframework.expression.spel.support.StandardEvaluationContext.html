org.springframework.expression.spel.support.StandardEvaluationContext

The StandardEvaluationContext class is a powerful and versatile Java object used in the Spring Framework for evaluating expressions written in SpEL (Spring Expression Language). StandardEvaluationContext context = new StandardEvaluationContext();

context.setVariable("price", 10);
