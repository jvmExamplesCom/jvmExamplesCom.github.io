android.text.format.Formatter

Formatting Bytes

The 

          
           
android.text.format.Formatter
class has methods that allow you to convert bytes into a human-readable format. For example, the following code snippet converts a byte count of 1048576 (1 megabyte) into a human-readable string format:

Formatting Dates and Times

android.text.format.Formatter
