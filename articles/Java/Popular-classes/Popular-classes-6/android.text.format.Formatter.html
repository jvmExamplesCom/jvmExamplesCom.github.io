android.text.format.Formatter

Android's 

          
           
android.text.format.Formatter
class provides utility methods to convert various data types, such as bytes, kilobytes, megabytes, and gigabytes, into a human-readable format. It also contains various methods for formatting dates and times into strings.

Formatting Bytes

The 

          
           
android.text.format.Formatter
class has methods that allow you to convert bytes into a human-readable format. For example, the following code snippet converts a byte count of 1048576 (1 megabyte) into a human-readable string format: 

          
           
String formattedSize = Formatter.formatShortFileSize(context, 1048576);
System.out.println(formattedSize);
// Output: 1.0 MB
Similarly, you can convert a kilobyte count into a human-readable string using the 

          
           
formatShortFileSize()
method. For example, the following code snippet converts a kilobyte count of 1024 (1 kilobyte) into a human-readable string format: 

          
           
String formattedSize = Formatter.formatShortFileSize(context, 1024);
System.out.println(formattedSize);
// Output: 1.0 KB
You can also use the 

          
           
formatFileSize()
method to format a byte count into a human-readable format with a more detailed result. For example, the following code snippet converts a byte count of 1048576 (1 megabyte) into a human-readable string format: 

          
           
String formattedSize = Formatter.formatFileSize(context, 1048576);
System.out.println(formattedSize);
// Output: 1.00 MB

Formatting Dates and Times

The 

          
           
android.text.format.Formatter
class also has methods that allow you to format dates and times into strings. For example, the following code snippet formats a date into a human-readable string format: 

          
           
Date date = new Date();
String formattedDate = Formatter.formatDateTime(context, date.getTime(), Formatter.SHOW_DATE | Formatter.SHOW_YEAR);
System.out.println(formattedDate);
// Output: Dec 13, 2020

formatTime()
method to format a time into a human-readable string format. For example, the following code snippet formats a time into a 12-hour format with AM/PM suffix: 

          
           
Time time = new Time();
String formattedTime = Formatter.formatTime(context, time);
System.out.println(formattedTime);

formatDateTime()
method. For example, the following code snippet formats both the date and time into a human-readable string format: 

          
           
Date date = new Date();
Time time = new Time();
String formattedDateTime = Formatter.formatDateTime(context, date.getTime(), time, Formatter.SHOW_DATE | Formatter.SHOW_YEAR | Formatter.SHOW_TIME);
