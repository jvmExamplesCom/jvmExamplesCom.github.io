Introduction to org.apache.zookeeper.ZooDefs

org.apache.zookeeper.ZooDefs is an interface used by Apache ZooKeeper that provides static constants and static methods is used to set node operations parameters. It is used to provide support for the ZooKeeper data model. This interface contains all the core methods and constructors which help in creating, updating and deleting a node in ZooKeeper. ZooDefs also includes authentication, ACLs, connection timeouts, etc.
