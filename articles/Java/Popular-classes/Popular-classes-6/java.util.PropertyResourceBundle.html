java.util.PropertyResourceBundle

Java has a feature called Resource Bundles which is a way of packaging and organizing localized objects. The PropertyResourceBundle class is part of this feature, and it allows the user to access resource bundles in a file with a name ending in “.properties”. By using this class, the user can access different languages and text based resources.

Basics

The syntax for creating a PropertyResourceBundle object is as follows: 

          
           
PropertyResourceBundle bundle= new PropertyResourceBundle(InputStream);
The above statement creates the bundle object which will read resources from the specified InputStream. The InputStream parameter can be set to any file or classpath resource. The input stream typically contains keys and values in a form familiar from Java Properties files.

Example

We can create and access a bundle using the following code: 

          
           
//Create a property resource bundle
PropertyResourceBundle bundle = new PropertyResourceBundle(new FileInputStream("mybundle.properties"));

//Access a value from the bundle
String welcomeMsg = bundle.getString("welcomeMsg");
System.out.println(welcomeMsg);
If the file mybundle.properties contains the following content: 

          
           
welcomeMsg = Welcome to my bundle!
Then the output of the above code will be: 

          
           
Welcome to my bundle!

Methods

getKeys()

This method returns an Enumeration of all the keys contained in the bundle. This is useful when the user wants to get all the keys in the bundle or loop through them.

getString(String key)

This method takes a String key and returns its associated value from the resource bundle. If the key does not exist then it will return null.

getObject(String key)

This method takes a String key and returns its associated object from the resource bundle. If the key does not exist then it will return null.

Conclusion

java.util.PropertyResourceBundle is a useful class for accessing localized objects in a bundle. It has several methods which can be used to access or loop through the keys in the bundle. It also has methods which allow the user to get values or objects associated with a specific key.
