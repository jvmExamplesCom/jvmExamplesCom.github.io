Understanding CameraCharacteristics

The CameraCharacteristics object contains information on the camera's capabilities such as its sensor size, orientation, and supported output sizes.

Accessing CameraCharacteristics in Java Android

CameraManager camManager = (CameraManager) getSystemService(Context.CAMERA_SERVICE);
