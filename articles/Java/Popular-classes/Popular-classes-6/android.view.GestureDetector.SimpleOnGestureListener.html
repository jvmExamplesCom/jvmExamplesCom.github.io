android.view.GestureDetector.SimpleOnGestureListener

It has several callback methods that are triggered when the corresponding gesture is detected. This listener can be used to detect single-tap, double-tap, long press, scroll, and fling gestures.

        // Handle the down event
        return true;

        // Handle the long press event
