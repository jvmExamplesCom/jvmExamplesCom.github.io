javax.xml.bind.annotation.XmlElementDecl

The javax.xml.bind.annotation.XmlElementDecl class is a java annotation type used for specifying the structure of an XML element in a document. It helps developers to easily create an element in the XML schema and keep the values in sync.

@XmlElementDecl(name = "book-store", scope = MyObject.class)
public JAXBElement<BookStore> createBookStore(BookStore value) {
    return new JAXBElement<BookStore>(new QName("http://example.com/", "book-store"), BookStore.class, null, value);
}

@XmlElementDecl
. The first parameter 
name

scope
determines the scope of the element and it should be set to the object where the annotation is defined. The last parameter is a 
JAXBElement

BookStore
object containing information about the books in the store.
