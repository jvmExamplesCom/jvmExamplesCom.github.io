org.apache.log4j.Appender

The org.apache.log4j.Appender class is part of the Apache Log4j Java library and provides an abstract class used by other logging components to output log messages. It is responsible for writing actual log events to the underlying logging system. It is typically used in conjunction with some form of Layout and Filter classes to configure a complete logging system.

Example

The following example demonstrates how to use the Appender class to output log messages to the console: 

          
           
// create a Console Appender
ConsoleAppender consoleAppender = new ConsoleAppender();

// specify layout format
String pattern = "%d [%p] %C{1}: %m%n";
consoleAppender.setLayout(new PatternLayout(pattern));

// specify logging level
consoleAppender.setThreshold(Level.INFO);

// add the appender to the logger
Logger.getRootLogger().addAppender(consoleAppender);

// output log messages
Logger.getLogger(MyClass.class).info("Log message");
