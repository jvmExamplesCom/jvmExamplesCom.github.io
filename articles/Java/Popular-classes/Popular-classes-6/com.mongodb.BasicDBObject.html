com.mongodb.BasicDBObject in Java

In Java, com.mongodb.BasicDBObject is a basic object that represents a document in MongoDB. It is used to create and manipulate documents within a MongoDB collection. The BasicDBObject class provides an easy way to create, retrieve, update and delete documents from a MongoDB collection.

A BasicDBObject can be created using either the constructor or the put() method. To create an empty object, the constructor can be used.


          
           
BasicDBObject obj = new BasicDBObject();

Map<String, Object> pairs = new HashMap<>();
pairs.put("name", "John");
pairs.put("age", 30);
pairs.put("address", "123 Main St.");

BasicDBObject obj = new BasicDBObject(pairs);

Alternatively, the put() method can be used to add individual name-value pairs to an empty BasicDBObject.


          
           
BasicDBObject obj = new BasicDBObject();

obj.put("name", "John");
