Basics of Hashing

Hashing is a process where data is converted into a fixed length value using a hashing algorithm. The most common hashing algorithms used are MD5 and SHA-1. There are some differences between the two. MD5 produces a 128-bit hash value, while SHA-1 produces a 160-bit hash value. The SHA-1 algorithm is more secure as it is resistant to brute-force attacks.

The process of hashing involves taking a string of characters and converting it into a fixed length code based on certain variables, such as the type of hashing algorithm used, the length of the hash value and the way the characters are encoded into the hash value. The resulting code, the hash value, is used to verify the authenticity of the data.
