Java.sql.Struct

Java.sql.Struct is a part of the Java Database Connectivity API, which works in conjunction with other API parts to provide a uniform way of accessing databases. The Java.sql.Struct class provides the methods and constants required to store data in its structured form in the database. It is an abstract class that needs to be implemented by the various database vendors.

Methods Available in Java.sql.Struct

The Java.sql.Struct class provides four methods for use in interacting with structured data:
  • getAttributes(): This method returns an array of objects that represent the attributes of the structured type instance.
  • getSQLTypeName(): This method returns the name of the SQL structured type that this Struct object represents.
  • toString(): This method returns a String object representing the data in the Struct object.
  • equals(): This method determines if the given Struct object is logically equal to the current Struct object.

Example Usage of Java.sql.Struct

Below is a sample example of how to use the Java.sql.Struct class to store structured data in a database. 

          
           
import java.sql.Connection;
import java.sql.DriverManager;
import java.sql.PreparedStatement;
import java.sql.ResultSet;
import java.sql.Struct;

public class JavaSQLStructExample {
  public static void main(String[] args) {
    // Create a connection to the database
    Connection conn = DriverManager.getConnection("jdbc:mariadb://localhost:3306/mydatabase");
    PreparedStatement stmt = conn.prepareStatement("SELECT * FROM users WHERE id = ?");
 
    // Set the values for the statement
    stmt.setInt(1, 1);
    ResultSet rs = stmt.executeQuery();
 
    // Get the Struct from the result set
    Struct userData = rs.getStruct("data");
 
    // Print the attributes of the Struct
    Object[] attributes = userData.getAttributes();
    for (Object o : attributes) {
      System.out.println(o);
    }
  }
}
In this example, we have used the Java.sql.Struct class to retrieve structured data from a table in a MariaDB database. The Struct object can then be used to get the attributes of the structured data that are stored in the database. These attributes can be printed out to view the data stored in the Struct object.
