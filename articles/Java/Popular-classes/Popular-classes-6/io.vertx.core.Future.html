The io.vertx.core.Future Class

The 

          
           
io.vertx.core.Future

Future
instance can either hold a result of a completed operation, or it can be in one of several pending states. The 
Future
instance provides methods for setting a result, failing with an exception, checking the result's status, registering a callback to be called when the operation is completed, and canceling the operation.

Future
class:

// Do some asynchronous stuff...

  // Get the result of the operation.
  String result = future.result();
