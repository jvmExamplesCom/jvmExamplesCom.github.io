java.time.DayOfWeek

Java.time.DayOfWeek is the class that represents a single day of the week in a calendar system. The seven days of the week are represented as constants in DayOfWeek: MONDAY, TUESDAY, WEDNESDAY, THURSDAY, FRIDAY, SATURDAY, and SUNDAY. The enum constant occurs exactly once in a single week, making it a great choice when dealing with day of week calculations.

Constructors

The two public constructors available to create a DayOfWeek instance are DayOfWeek.of(int) and DayOfWeek.valueOf(String). The first constructor takes an integer that corresponds to the numerical representation of the day of the week (Sunday=1, Monday=2, …). The second constructor uses the String representations of the days of the week (SUNDAY, MONDAY, …).

Retrieving days of the week

There are many useful methods available to retrieve information from the DayOfWeek instance. The simplest way to access the data contained in the instance is to use the getValue() and getDisplayName() methods. The former returns the numerical value for the day of the week (Sunday=1, Monday=2, …), while the latter returns the String representation of the day of the week (SUNDAY, MONDAY, …).


          
           
public static void main(String[] args) {
    DayOfWeek day = DayOfWeek.MONDAY;
    
    int value = day.getValue();
    String name = day.getDisplayName();
