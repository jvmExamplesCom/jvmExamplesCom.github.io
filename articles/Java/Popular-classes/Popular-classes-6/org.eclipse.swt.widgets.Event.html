org.eclipse.swt.widgets.Event

The org.eclipse.swt.widgets.Event class is a useful Java tool for creating and managing events. Events are essential in programming user interfaces as they allow the application to respond to user input. Events are typically used for tasks such as mouse clicks, key presses, and window activations. The org.eclipse.swt.widgets.Event class provides an easy way to create and manage these events.

Event() // Creates an empty Event object

Event(Event e) // Copies the specified Event object

The type of an event can be set and retrieved using the setType() and getType() methods, respectively. The available types are defined by the SWT constant fields listed below:

SWT.MouseWheel,

SWT.MouseExit

Additionally, all methods for setting and getting event parameters such as time, character, state, item, string, x, and y are provided. Finally, theorg.eclipse.swt.widgets.Event class provides a method called notifyListeners() which can be used to inform any listeners that the event has occurred.

Text text = new Text(shell, SWT.SINGLE);
text.addListener(SWT.MouseDown, new Listener() {
    @Override
    public void handleEvent(Event e) {
