BackgroundColorSpan in Java Android

The 

          
           
android.text.style.BackgroundColorSpan
class is a type of Android 
CharacterStyle
that provides formatting for a text string. This class gives control over the background color of the text. Applying a 
BackgroundColorSpan
is done by using the 

          
           
SpannableStringBuilder
class which supports chaining multiple spans together. An example implementation is provided below. 

          
           
SpannableStringBuilder sb = new SpannableStringBuilder("Text with colored Background");
sb.setSpan(new BackgroundColorSpan(Color.RED), 0, 10, Spanned.SPAN_EXCLUSIVE_EXCLUSIVE);
// Text with colored Background
The above example shows how to apply a 
BackgroundColorSpan
to a 

          
           
SpannableStringBuilder
. The parameters for the span are the color, the starting position and the end position of the range of text to be covered with the span. The last parameter takes a flag (in this case 

          
           
Spanned.SPAN_EXCLUSIVE_EXCLUSIVE
) which defines how the range overlaps with other spans.

In order to make sure the color of the text is also updated, you need to call the 

setColor()
method of the 
TextView
with the same color you used for the 
BackgroundColorSpan
, like so: 

          
           
textview.setTextColor(Color.RED);
You can specify the alpha value of a color with 
Color.argb()
to make it partially transparent: 

          
           
SpannableStringBuilder sb = new SpannableStringBuilder("Text with semi-transparent Background");
sb.setSpan(new BackgroundColorSpan(Color.argb(128, 255, 0, 0)), 0, 15, Spanned.SPAN_EXCLUSIVE_EXCLUSIVE);
// Text with semi-transparent Background
You can also combine multiple 
BackgroundColorSpan
s to achieve more complex effects. For example, you could use two 

          
           
BackgroundColorSpans
to create an alternating pattern of two colors: 

          
           
SpannableStringBuilder sb = new SpannableStringBuilder("Alternate Colors");
sb.setSpan(new BackgroundColorSpan(Color.RED), 0, 5, Spanned.SPAN_EXCLUSIVE_EXCLUSIVE);
sb.setSpan(new BackgroundColorSpan(Color.BLUE), 6, 11, Spanned.SPAN_EXCLUSIVE_EXCLUSIVE);
// Alternate Colors

It's also possible to set different styles (e.g. bold or italic) for each part of a text, similiar to the 

BackgroundColorSpan
: 

          
           
SpannableStringBuilder sb = new SpannableStringBuilder("Mixed Styles");
sb.setSpan(new StyleSpan(Typeface.BOLD), 0, 3, Spanned.SPAN_EXCLUSIVE_EXCLUSIVE);
sb.setSpan(new StyleSpan(Typeface.ITALIC), 4, 7, Spanned.SPAN_EXCLUSIVE_EXCLUSIVE);
// Mixed Styles
