javax.swing.JCheckBoxMenuItem

The javax.swing.JCheckBoxMenuItem class is a type of checkbox that can be placed in a menu component. It provides the ability to check and uncheck the box when it is selected, allowing the user to toggle between two states.

Creating a JCheckBoxMenuItem

To create a JCheckBoxMenuItem, you can use the constructor 

JCheckBoxMenuItem()
. This creates an empty checkbox item. You can also pass a string argument to set the text displayed next to the checkbox item. For example: 

          
           
JCheckBoxMenuItem myCheckBox = new JCheckBoxMenuItem("My checkbox");

Adding an ActionListener

Once a JCheckBoxMenuItem has been created, you need to add an ActionListener so that when the item is checked or unchecked, the ActionPerformed() method can be called. To do this, you can use the 

addActionListener()
method. An example of this is shown below: 

          
           
myCheckBox.addActionListener(new ActionListener(){
    public void actionPerformed(ActionEvent e){
        // perform some action
    }
});

Setting the Checked State

To set the current checked state of the JCheckBoxMenuItem, you can use the 

setSelected()
method. This takes a boolean parameter which is used to indicate whether the item should be checked or not. For example: 

          
           
myCheckBox.setSelected(true);

Getting the Checked State

To retrieve the current checked state of the JCheckBoxMenuItem, you can use the 

isSelected()
method. This returns a boolean value indicating whether the item is currently checked or not. For example: 

          
           
boolean isChecked = myCheckBox.isSelected();

Example Program

The following program shows an example of a JCheckBoxMenuItem in action. When the item is checked, a message is printed to the console. When it is unchecked, another message is printed instead.


          
           
import javax.swing.*;
import java.awt.*;

public class CheckBoxMenuItemExample {
    public static void main(String[] args){
        JFrame frame = new JFrame("Checkbox Menu Item Example");
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);

        JMenuBar menuBar = new JMenuBar();
        JMenu menu = new JMenu("My Menu");
        JCheckBoxMenuItem checkBox = new JCheckBoxMenuItem("My checkbox");

        checkBox.addActionListener(new ActionListener(){
            public void actionPerformed(ActionEvent e){
                if(checkBox.isSelected())
                    System.out.println("Checkbox is now checked");
                else
                    System.out.println("Checkbox is now unchecked");
            }
        });

        menu.add(checkBox);
        menuBar.add(menu);
        frame.setJMenuBar(menuBar);
        frame.setSize(400, 300);
        frame.setVisible(true);
    }
}
