javax.security.auth.callback.PasswordCallback

The javax.security.auth.callback.PasswordCallback class allows an application to collect a password from the caller. This is typically used by a JAAS authentication mechanism to acquire a user's private credentials.

Constructors

The PasswordCallback class provides two constructors:

  • 
            
             
public PasswordCallback(String prompt, boolean echoOn)
    – Creates a PasswordCallback with a prompt and setting whether or not the password is displayed as it is typed.
  • 
            
             
public PasswordCallback(String prompt, boolean echoOn, int promptLimit)
    – Creates a PasswordCallback with a prompt, setting whether or not the password is displayed as it is typed, and limit of number of characters it can contain.

Methods

The PasswordCallback class provides several methods:

  • 
            
             
public boolean getEchoOn()
    – Returns true if the password should be echoed while it is being typed, false otherwise.
  • 
            
             
public void setEchoOn(boolean echoOn)
    – Sets the echoOn property which determines whether the password is echoed while it is being typed.
  • 
            
             
public char[] getPassword()
    – Returns the password specified by the caller.
  • 
            
             
public void setPassword(char[] password)
    – Sets the specified password.
  • 
            
             
public int getPromptLimit()
    – Returns the maximum number of characters allowed in the password.
  • 
            
             
public void setPrompt(String prompt)
    – Sets the prompt used to request the password from the caller.

Example

Let's look at a simple example which requests a password from the user and prints out the result:


          
           
import javax.security.auth.callback.*;

public class PasswordCallbackExample {
    public static void main(String[] args) {
        // Create a PasswordCallback object
        PasswordCallback callback = new PasswordCallback("Enter Password: ",
                                                        true, 5);

        // Request the password
        try {
            Callback[] callbacks = new Callback[]{callback};
            CallbackHandler handler = new MyCallbackHandler();
            handler.handle(callbacks);
        } catch (Exception e) {
            e.printStackTrace();
        }

        // Print out the result
        System.out.println("Thank you, your password is: " + 
            new String(callback.getPassword()));
    }
}

class MyCallbackHandler implements CallbackHandler {

    public void handle(Callback[] callbacks) throws IOException {
        for (int i = 0; i < callbacks.length; i++) {
            if (callbacks[i] instanceof PasswordCallback) {
                PasswordCallback pc = (PasswordCallback)callbacks[i];
                BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
                System.out.print(pc.getPrompt());
                System.out.flush();
                // Read the password
                char[] tmp = br.readLine().toCharArray();
                // Set the password
                pc.setPassword(tmp);
            } else {
                throw new UnsupportedCallbackException(callbacks[i], 
                    "Unrecognized Callback");
            }
        }
    }
}
