javax.json.JsonReader Overview

Javax.json.JsonReader allows for the creation and manipulation of JSON data in Java applications. It allows for reading and writing of JSON documents from sources such as files, strings, and URL locations. This article will provide an overview of how to create and use a JsonReader in a Java application.

Creating a JsonReader Object

A JsonReader object can be created by passing in a Reader object:


          
           
String jsonString = "{\"name\": \"John\", \"age\": 20}";

StringReader reader = new StringReader(jsonString);
JsonReader jsonReader = Json.createReader(reader);
It can also be created from an InputStream: 

          
           
InputStream inputStream = ...;
JsonReader jsonReader = Json.createReader(inputStream);
Or from a URL: 

          
           
URL url = new URL("http://example.com/data.json");
JsonReader jsonReader = Json.createReader(url.openStream());

Reading JSON Data

Once a JsonReader is created, it can be used to read in JSON data. The JsonReader.readObject() method returns a JsonObject, which can then be used to access the data:


          
           
JsonObject jsonObject = jsonReader.readObject();
String name = jsonObject.getString("name");
int age = jsonObject.getInt("age");

The JsonObject can also be converted back into a string by using the JsonObject.toString() method:


          
           
String jsonString = jsonObject.toString();

Writing JSON Data

The JsonReader.writeObject() method can be used to write JSON data to a stream:


          
           
StringWriter writer = new StringWriter();
JsonWriter jsonWriter = Json.createWriter(writer);

JsonObject jsonObject = Json.createObjectBuilder().add("name", "John").add("age", 20).build();

jsonWriter.writeObject(jsonObject);
String jsonString = writer.toString();

{"name":"John","age":20}
