Introduction to Java.awt.image.ImageObserver

Java.awt.image.ImageObserver is an interface which is used to monitor the changes in an image or to receive notification of the availability of an image. It is mainly used for loading images that are produced asynchronously and for monitoring the loading state of an image. This interface can also be used for obtaining information about an image's properties.

Implementing ImageObserver

The ImageObserver interface must be implemented by classes that wish to be notified when an image has been changed, created, or added. The implementing class must provide an implementation of the imageUpdate() method. The most basic implementation looks like this: 

          
           
public boolean imageUpdate(Image img, int flags, int x, int y, int w, int h)
The imageUpdate() method returns a boolean value that indicates whether or not more updates will be required. The parameters provided allow the implementer to check the status of the image, its dimensions, and its properties.
