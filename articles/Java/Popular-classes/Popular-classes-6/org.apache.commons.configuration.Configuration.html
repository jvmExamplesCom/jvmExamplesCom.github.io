Introduction to org.apache.commons.configuration.Configuration

org.apache.commons.configuration.Configuration is a Java library that provides convenient functionality for reading and writing configuration data. It supports multiple formats such as XML, JSON, and YAML. With this library, developers can quickly and easily load configuration files into their applications, manipulate the properties within these files, and write the changes out to persistent storage.

Constructor

To construct a Configuration object, use either a constructor or the factory method createConfiguration().


          
           
Configuration config = new Configuration();
Configuration config = Configuration.createConfiguration();

Methods

This library provides many methods for retrieving and manipulating configuration values.

getProperty()


          
           
String value = config.getProperty("key");
Returns the value of the given property. If the property is not present in the configuration, it will return null.

setProperty()


          
           
config.setProperty("key", "value");
Sets the given property to the given value. If the property is already present, it will be overwritten with the new value.

addProperty()


          
           
config.addProperty("key", "value");
Adds the given property to the configuration. If the property is already present, the existing value will be preserved.

removeProperty()


          
           
config.removeProperty("key");
Removes the given property from the configuration.

Writing the Configuration to File

To write the configuration to a file, use the save() method.


          
           
config.save("filename.json");
The filename argument can have any supported extension. The library will automatically determine the format of the output based on the extension.
