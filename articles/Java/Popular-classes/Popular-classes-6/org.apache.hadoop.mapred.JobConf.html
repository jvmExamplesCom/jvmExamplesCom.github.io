Introduction To org.apache.hadoop.mapred.JobConf

org.apache.hadoop.mapred.JobConf is a Java API used to configure the Hadoop MapReduce framework. The JobConf class allows the user to set up configuration parameters that will be used by other components of the Hadoop MapReduce job, such as the Mapper and Reducer classes. This article covers the basics of using the JobConf API to set up a Hadoop MapReduce job.

How To Use JobConf

Using JobConf is simple and straightforward. First, you need to create an instance of the JobConf class, passing in the name of the job as an argument. The following example instantiates a JobConf object with the name “wordcount-job”: 

          
           
JobConf jobConf = new JobConf("wordcount-job");

jobConf.setInputFormat(TextInputFormat.class);
Similarly, you can use the getOutputFormat() method to specify the type of the output file format (e.g. TextOutputFormat or SequenceFileOutputFormat): 

          
           
jobConf.setOutputFormat(TextOutputFormat.class);

Once the required configuration parameters have been set up, you can submit the Hadoop MapReduce job by calling the JobClient.runJob() method.
