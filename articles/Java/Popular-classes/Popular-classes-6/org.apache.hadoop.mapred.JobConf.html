Introduction To org.apache.hadoop.mapred.JobConf

The JobConf class allows the user to set up configuration parameters that will be used by other components of the Hadoop MapReduce job, such as the Mapper and Reducer classes. This article covers the basics of using the JobConf API to set up a Hadoop MapReduce job.

JobConf jobConf = new JobConf("wordcount-job");

jobConf.setInputFormat(TextInputFormat.class);
