android.util.FloatMath is a Java class used to do mathematical calculations on float values. It contains static methods that are used to perform various calculations, such as square root, absolute value, and trigonometric functions.

Usage

import android.util.FloatMath;
Once imported, you can then use any of the static methods available in FloatMath. Here are some examples: 

          
           
sqrt(float f):
Returns the square root of the input float value. 

          
           
float result = FloatMath.sqrt(10f);

float result = FloatMath.abs(-3.14f);
System.out.println("Result = " +  result);
// Output: Result = 3.14
