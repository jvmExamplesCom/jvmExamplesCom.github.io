android.support.v4.app.TaskStackBuilder

Android’s TaskStackBuilder class is a utility class used to construct synthetic back stacks for activities which are managed by the system and not directly by the application. It allows your application to construct an artificial back stack of activities and helps you maintain backward compatibility when using navigation patterns like Up navigation.

Creating a TaskStackBuilder

The TaskStackBuilder class provides a static factory method called create() which accepts a Context and returns TaskStackBuilder object. This method should be used when creating TaskStackBuilder object:


          
           
TaskStackBuilder stackBuilder = TaskStackBuilder.create(context);

Adding intents to the TaskStackBuilder

The addNextIntentWithParentStack() is used to add intents to the TaskStackBuilder. This method takes the intent and flags as its parameters. The flag option is used to set whether the activity should be included in the task or not:


          
           
// Add intent without parent stack
stackBuilder.addNextIntent(intent);

// Add intent with parent stack
stackBuilder.addNextIntentWithParentStack(intent);

Starting the task stack

Once the intents are added to the TaskStackBuilder, you can use the startActivities() method to start the activities in the task stack. This method requires ActivityOptions object as its optional parameter:


          
           
// Start the task
stackBuilder.startActivities();

// Start the task with ActivityOptions 
stackBuilder.startActivities(activityOptions);

Example

Let's assume you want to add two activities to the task stack. The first activity will be the parent activity, and the second activity will be launched without the parent activity, and it will be the root activity of the task stack.


          
           
// Create TaskStackBuilder
TaskStackBuilder stackBuilder = TaskStackBuilder.create(context);

// Add parent activity
Intent parentIntent = new Intent(context, ParentActivity.class);
stackBuilder.addNextIntentWithParentStack(parentIntent);

// Add root activity
Intent rootIntent = new Intent(context, RootActivity.class);
stackBuilder.addNextIntent(rootIntent);

// Start the task
stackBuilder.startActivities();
