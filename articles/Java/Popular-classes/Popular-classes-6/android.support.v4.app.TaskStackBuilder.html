Creating a TaskStackBuilder

The TaskStackBuilder class provides a static factory method called create() which accepts a Context and returns TaskStackBuilder object. This method should be used when creating TaskStackBuilder object:


          
           
TaskStackBuilder stackBuilder = TaskStackBuilder.create(context);

The addNextIntentWithParentStack() is used to add intents to the TaskStackBuilder. This method takes the intent and flags as its parameters. The flag option is used to set whether the activity should be included in the task or not:


          
           
// Add intent without parent stack

// Add intent with parent stack
stackBuilder.addNextIntentWithParentStack(intent);

Starting the task stack
