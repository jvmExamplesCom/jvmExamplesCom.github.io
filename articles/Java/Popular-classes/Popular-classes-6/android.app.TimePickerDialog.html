What is TimePickerDialog?

Android.app.TimePickerDialog is an Android widget that allows users to select a time from a dialog. It displays a top-level dialog window with two options: the hour and minute. The user can choose from both, or just one of these options. The TimePickerDialog then sends the selected time to the activity that created it.

TimePickerDialog is part of the Android API and can be used in any Android application. It is typically used in date-picking, alarms, to set reminders, and other common use cases.

Creating a TimePickerDialog


          
           
// Get current system time
Calendar calendar = Calendar.getInstance();

// Create TimePickerDialog
TimePickerDialog timePickerDialog = new TimePickerDialog(
    context, 
    new TimePickerDialog.OnTimeSetListener() {
        @Override 
        public void onTimeSet(TimePicker view, int hourOfDay, int minute) {
            // Do something when time is selected
        }
    }, 
    calendar.get(Calendar.HOUR_OF_DAY),
    calendar.get(Calendar.MINUTE),
    true);

// Show the dialog
timePickerDialog.show();

This code creates a TimePickerDialog. First, it gets the current system time using the Calendar class. Then, it creates a TimePickerDialog using the context, an OnTimeSetListener, and the hour and minute from the Calendar. Finally, it shows the dialog.

OnTimeSetListener

When the user selects a time, the OnTimeSetListener is triggered. The method onTimeSet is called, and it passes the view (the TimePickerDialog), the hour (as an int from 0-23) and the minute (as an int from 0-59). This can be used to perform some custom action when the time is set by the user.

Customizing the TimePickerDialog

The TimePickerDialog can be customized to fit the needs of an application. The following code shows an example of customizing a TimePickerDialog.


          
           
TimePickerDialog timePickerDialog = new TimePickerDialog(
    context,
    new TimePickerDialog.OnTimeSetListener() {
        @Override
        public void onTimeSet(TimePicker view, int hourOfDay, int minute) {
            // Do something when time is selected
        }
    },
    calendar.get(Calendar.HOUR_OF_DAY),
    calendar.get(Calendar.MINUTE),
    false);

// Set the title
timePickerDialog.setTitle(“Select a Time”);
timePickerDialog.setCancelable(true);

// Add the buttons
timePickerDialog.setButton(
    DialogInterface.BUTTON_POSITIVE, 
    “Done”, 
    new DialogInterface.OnClickListener() {
        public void onClick(DialogInterface dialog, int whichButton) {
            // Do something when 'Done' is clicked  
        }
    });

timePickerDialog.setButton(
    DialogInterface.BUTTON_NEGATIVE, 
    “Go Back”, 
    new DialogInterface.OnClickListener() {
        public void onClick(DialogInterface dialog, int whichButton) {
            // Do something when 'Go back' is clicked  
        }
    });

// Show the dialog
timePickerDialog.show();

This code creates a TimePickerDialog that is similar to the previous example, but with some customization. The dialog is given a title and set to be cancelable. Two buttons (Done and Go Back) are added and listeners are set to do something when each button is clicked. Finally, the dialog is shown.

Conclusion

The android.app.TimePickerDialog is a useful Android widget for selecting a time. It can be customized to fit the needs of an application, and it is easy to use. If you need to add time selection functionality to your Android app, the TimePickerDialog is a great choice.
