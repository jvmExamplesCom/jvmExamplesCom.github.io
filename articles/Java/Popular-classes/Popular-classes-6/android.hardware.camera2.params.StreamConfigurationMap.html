Introduction to Android.hardware.camera2.params.StreamConfigurationMap

Android.hardware.camera2.params.StreamConfigurationMap is an API introduced by Google in Android 5.0 (Lollipop) for constructing stream configurations for camera devices. It consists of a class which holds a collection of sizes and formats that define the supported output streams from a given physical camera device, as well as their capabilities.

Overview of Android.hardware.camera2.params.StreamConfigurationMap

The android.hardware.camera2.params.StreamConfigurationMap class provides information about the different stream configurations of a camera device such as supported output sizes and formats, cropping type, maximum stall duration, ect. It's a useful API for applications that rely on taking captures efficiently and accurately.

Methods available in the StreamConfigurationMap Class

The following list of methods are available to the StreamConfigurationMap class:
  • getOutputSizes(int): Returns a list of supported output sizes for a given image format.
  • getOutputFormats(): Returns a list of supported output formats.
  • getMinFrameDuration(int, Size): Returns the minimum frame duration for a given output size and format.

  • isOutputSupportedFor(int): Returns true if the specified format is supported as an output.
