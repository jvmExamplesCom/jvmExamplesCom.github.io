Introduction to Android.hardware.camera2.params.StreamConfigurationMap

Android.hardware.camera2.params.StreamConfigurationMap is an API introduced by Google in Android 5.0 (Lollipop) for constructing stream configurations for camera devices. It consists of a class which holds a collection of sizes and formats that define the supported output streams from a given physical camera device, as well as their capabilities.

Overview of Android.hardware.camera2.params.StreamConfigurationMap

The android.hardware.camera2.params.StreamConfigurationMap class provides information about the different stream configurations of a camera device such as supported output sizes and formats, cropping type, maximum stall duration, ect. It's a useful API for applications that rely on taking captures efficiently and accurately.

Methods available in the StreamConfigurationMap Class

The following list of methods are available to the StreamConfigurationMap class:
  • getOutputSizes(int): Returns a list of supported output sizes for a given image format.
  • getOutputFormats(): Returns a list of supported output formats.
  • getMinFrameDuration(int, Size): Returns the minimum frame duration for a given output size and format.
  • getMaxFrameDuration(int, Size): Returns the maximum frame duration for a given output size and format.
  • getHighResolutionOutputSizes(int): Returns a list of high resolution output sizes for a given format.
  • isOutputSupportedFor(int): Returns true if the specified format is supported as an output.
  • getValidOutputFormatsForInput(int): Returns a list of valid output formats for a given input format.
  • getInputFormats(): Returns a list of supported input formats.
  • getInputSizes(int): Returns a list of supported input sizes a given input format.
  • getDepthStallDurations(): Returns the stall durations of a depth output.
  • getDepthMaxFrameDuration(Size): Returns the maximum frame duration for a given depth output size.
  • getSupportedCropRectangles(): Returns the list of cropping rectangles that are supported by the camera device.
  • getDepthMinFrameDuration(Size): Returns the minimum frame duration for a given depth output size.

Examples of using StreamConfigurationMap

To demonstrate usage of StreamConfigurationMap in practice, let's work through a short example that prints the available output formats and sizes. 

          
           
// Create a StreamConfigurationMap object
StreamConfigurationMap config = new StreamConfigurationMap();

// Get the supported output formats
int[] formats = config.getOutputFormats();

System.out.println("Supported output formats:");
for (int i= 0; i < formats.length; i++) {
    System.out.println("Format " + i + ": " + formats[i]);
}

// Get the supported output sizes
Size[] sizes = config.getOutputSizes(ImageFormat.JPEG);

System.out.println("\nSupported output sizes for ImageFormat.JPEG");
