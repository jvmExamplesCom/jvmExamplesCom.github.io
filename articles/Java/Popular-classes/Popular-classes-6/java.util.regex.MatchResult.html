java.util.regex.MatchResult

java.util.regex.MatchResult is a class used for storing the results of regex matching and capturing in Java. It provides methods for accessing the individual groups that were matched, the beginning and end of each group, and other useful methods. The MatchResult object is obtained from the matcher by calling the matches() or find() methods.

Code Examples

Let's take a look at how to work with MatchResult in code. The following examples show how to match a string against a regular expression and access the results using a MatchResult object.


          
           
String str = "Hello, World!";
Pattern pattern = Pattern.compile("\\w+");
Matcher matcher = pattern.matcher(str);

if(matcher.find()) {
    MatchResult result = matcher.toMatchResult();
    System.out.println("Matched group: " + result.group());
    System.out.println("Starting Index: " + result.start());
    System.out.println("Ending Index: " + result.end());
}
The output of the above code will be: 

          
           
Matched group: Hello
Starting Index: 0
Ending Index: 5

We can also use MatchResult to retrieve an array of all the matched groups. This is done using the groups() method:


          
           
String str = "Hello, World!";
Pattern pattern = Pattern.compile("\\w+");
Matcher matcher = pattern.matcher(str);

if(matcher.find()) {
    MatchResult result = matcher.toMatchResult();
    String[] groups = result.groups();
    System.out.println("Groups: " + Arrays.toString(groups));
}
The output of the above code will be: 

          
           
Groups: [Hello, World]

Finally, we can use MatchResult to check whether a particular group was matched or not. This is done using the hasGroup() method:


          
           
String str = "Hello, World!";
Pattern pattern = Pattern.compile("\\w+");
Matcher matcher = pattern.matcher(str);

if(matcher.find()) {
    MatchResult result = matcher.toMatchResult();
    boolean hasGroup = result.hasGroup("World");
    System.out.println("Has Group \"World\": " + hasGroup);
}
The output of the above code will be: 

          
           
Has Group "World": true
