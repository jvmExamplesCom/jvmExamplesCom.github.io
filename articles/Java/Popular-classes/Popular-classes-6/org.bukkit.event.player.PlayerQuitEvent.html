Event - PlayerQuitEvent

The org.bukkit.event.player.PlayerQuitEvent is a Bukkit event which is triggered when a player leaves the server.

Code Example 1


          
           
public class MyPlayerQuitListener implements Listener {
  
    @EventHandler
    public void onPlayerQuit(PlayerQuitEvent e) {
        // Handle player quitting
    }

}

Code Example 2


          
           
// In onEnable() of your plugin
getServer().getPluginManager().registerEvents(new MyPlayerQuitListener(), this);

Code Example 3


          
           
@EventHandler
public void onPlayerQuit(PlayerQuitEvent e) {
    Player player = e.getPlayer();
    String quitMessage = "Goodbye, " + player.getName() + "!";
    e.setQuitMessage(quitMessage);
}
