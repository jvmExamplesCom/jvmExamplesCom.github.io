android.animation.TypeEvaluator

The 

          
           
android.animation.TypeEvaluator
class is a utility class used in the animation framework of Android. It enables developers to create animations that transition between two values of a given type. It is an abstract class, so it cannot be directly instantiated. Instead, it must be extended and its evaluate() method overriden.

    @Override

    }
}

evaluate()

  • fraction

  • startValue

  • endValue

The
