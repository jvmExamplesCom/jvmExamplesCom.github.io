org.elasticsearch.common.transport.InetSocketTransportAddress

The 

          
           
org.elasticsearch.common.transport.InetSocketTransportAddress
class is used to represent an address to be used when establishing a transport connection such as an HTTP client or server. It encapsulates the host name, port and optionally a scheme, such as https. This class requires the Elasticsearch client library to be installed as a dependency.

Constructors

InetSocketTransportAddress
class has two primary constructors. The first constructor takes two parameters - an Internet Protocol (IP) address and a port. The second constructor takes three parameters - an IP address, a port and an optional scheme name. The scheme name is used for indicating which protocol the transport connection should use.
