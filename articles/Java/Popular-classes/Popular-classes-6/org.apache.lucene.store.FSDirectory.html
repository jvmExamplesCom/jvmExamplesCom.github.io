org.apache.lucene.store.FSDirectory

The org.apache.lucene.store.FSDirectory class is a powerful tool for creating, manipulating and reading a Lucene index from a file system. It is the default Lucene index store used by the Lucene search engine and provides an efficient and reliable way to manage an index over multiple files.

Creating a FSDirectory

A FSDirectory can be created using a java.io.File object pointing to the directory in which the Lucene index should be stored. The FSDirectory class also provides a static method, open(Path path) to create an FSDirectory instance. 


          
           
File dir = new File("indexdir");
    FSDirectory fsdir = FSDirectory.open(dir);

Reading from and Writing to a FSDirectory

The FSDirectory class provides several read/write methods. The openInput(String name) allows for the reading of a file from the index. Similarly, the openOutput(String name, IOContext context) method allows for the writing of files to the index.


          
           
// Read from the index
    IndexInput input = fsdir.openInput("file_to_read.txt");
    byte[] data = new byte[input.length()];
    input.readBytes(data, 0, data.length);
    
    // Write to the index
    IndexOutput output = fsdir.openOutput("file_to_write.txt", 
                                         IOContext.DEFAULT);
    output.writeBytes(data, 0, data.length);

Deleting Files from a FSDirectory

The deleteFile(String name) method of the FSDirectory class allows for the deletion of files from the Lucene index. 


          
           
fsdir.deleteFile("file_to_delete.txt");

Closing a FSDirectory

Finally, when done working with a FSDirectory instance, it must be closed in order to release any resources it holds. This can be accomplished through the close() method of the FSDirectory class.


          
           
fsdir.close();
