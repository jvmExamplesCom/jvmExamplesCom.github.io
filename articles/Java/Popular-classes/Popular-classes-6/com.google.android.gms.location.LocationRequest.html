Overview of the com.google.android.gms.location.LocationRequest Class

Using the LocationRequest Class

The LocationRequest class provides several methods for creating and configuring a location request. To create a request, you must specify the desired accuracy and duration of the request, as well as any additional parameters such as proximity alerting.

LocationRequest Parameters

The LocationRequest class has a number of parameters that can be used to customize the behavior of the request. The following parameters are available:

  • Fastest Interval - Sets the fastest interval between location updates. This can be used to avoid draining the battery too quickly.
