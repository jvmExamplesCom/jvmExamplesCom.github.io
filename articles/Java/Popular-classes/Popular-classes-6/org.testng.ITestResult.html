org.testng.ITestResult

The org.testng.ITestResult interface in TestNG provides information about the test result of a test method. It is an interface that represents an instance of a test execution. There are various methods like getStatus(), getName(), getThrowable() using which the result of a particular test method can be obtained.

Methods of ITestResult

  • getStatus(): This method returns the Result. This is an enum of type org.testng.ITestResult. The values that are returned are SUCCESS, SKIP, FAILURE, STARTED.
  • getName(): This method returns the test name.
  • getThrowable(): This method returns the throwable object if any exception occurs during the test execution.
  • getParameters(): This method returns an array of parameters passed to the test method.
  • getStartMillis(): This method returns the start time of the test in milliseconds.
  • getEndMillis(): This method returns the end time of the test in milliseconds.
  • getHost(): This method returns the hostname of the system on which the test has been executed.
  • getTestClass(): This method returns the class where the test method is present.
  • getInstanceName(): This method returns the name of the instance for which the test has been executed.
  • getTestName(): This method returns the qualified name of the test method.
  • getTestContext(): This method returns the TestContext object associated with the test.
  • getAttribute(): This method returns the value of the specified attribute.
  • getMethod(): This method returns the ITestNGMethod object associated with the test.
  • wasRetried(): This method returns true if test was retried else false.

Example

In the following example, we have a test case which invokes a test method. After the test method execution is complete, the test result is obtained using the ITestResult interface.


          
           
@Test
public void testResultExample() {
    System.out.println("This is a test method");
}

@AfterMethod
public void afterMethod(ITestResult result) {
    System.out.println("Test status: " + result.getStatus());
    System.out.println("Test name: " + result.getName());
}

When the above test case is executed, the following output will be displayed on the console:


          
           
This is a test method
Test status: SUCCESS
Test name: testResultExample
