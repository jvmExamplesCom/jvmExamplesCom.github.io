JavaFX - AnchorPane

AnchorPane is a container in JavaFX that supports resizing and repositioning of the content within it. Its primary use is to define the exact layout of components on a window, while allowing them to resize and move freely. The AnchorPane class implements a layout container with resizable edges. The top, left, right, and bottom properties can be used to move and size the content, thus providing a better user experience when building GUI applications.

Example 1: Creating an AnchorPane


          
           
AnchorPane anchorPane = new AnchorPane();
anchorPane.setPrefSize(400, 400);
Scene scene = new Scene(anchorPane);

Example 2: Adding a Node to an AnchorPane


          
           
AnchorPane anchorPane = new AnchorPane();
anchorPane.setPrefSize(400, 400);
Node node = new Button("Hello World");
anchorPane.getChildren().add(node);

Example 3: Setting Position and Size


          
           
AnchorPane.setLeftAnchor(node, 10.0);
AnchorPane.setTopAnchor(node, 10.0);
AnchorPane.setRightAnchor(node, 20.0);
AnchorPane.setBottomAnchor(node, 20.0);

Example 4: Using Constraints


          
           
AnchorPane.setLeftAnchor(node, 0.0);
AnchorPane.setTopAnchor(node, 0.0);
AnchorPane.setRightAnchor(node, AnchorPane.getWidth());
AnchorPane.setBottomAnchor(node, AnchorPane.getHeight());
