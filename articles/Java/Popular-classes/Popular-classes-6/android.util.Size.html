Introduction to android.util.Size

Android.util.Size is a utility class for working with width and height sizes. It allows you to store size objects in an array, or to compare sizes, or just to handle sizes more easily and efficiently than with regular width and height variables. In this article, we'll look at all the ways you can use Size and provide some code examples to help you get started.

Creating a Size Object

The easiest way to create a Size object is to use the static method 

          
           
Size.parseSize(String size);
This will take a string containing a width and height separated by an 'x' character and return a new Size object with the given width and height. For example: 

          
           
Size mySize = Size.parseSize("100x200");

Comparing Sizes

You can also compare two sizes using the 

          
           
equals()
method. This will return true if the width and heights of the two sizes are equal. You can also use the static method 

          
           
Size.compare(Size s1, Size s2)
which will return an integer indicating whether the two sizes given are equal, s1 is smaller than s2, or s2 is bigger than s1.

Extracting Dimensions
