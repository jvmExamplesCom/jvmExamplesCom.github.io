Introduction to android.util.Size

Android.util.Size is a utility class for working with width and height sizes. It allows you to store size objects in an array, or to compare sizes, or just to handle sizes more easily and efficiently than with regular width and height variables. Size.parseSize(String size);

Size mySize = Size.parseSize("100x200");

Comparing Sizes

equals()
method. This will return true if the width and heights of the two sizes are equal. You can also use the static method 

          
           
Size.compare(Size s1, Size s2)
