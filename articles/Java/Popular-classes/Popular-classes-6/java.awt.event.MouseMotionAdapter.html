Java awt.event.MouseMotionAdapter

Java awt.event.MouseMotionAdapter is a class used to handle events relating to mouse movement and positioning. It is part of the Abstract Window Toolkit (AWT) and is used to detect mouse movement, retrieve the coordinates of the cursor, and monitor changes in mouse position. This class can be used to identify drag and drop operations and for creating simple user interfaces.

Example Usage

The following example demonstrates how to use MouseMotionAdapter in an application. In this example, a Java class extending the MouseMotionAdapter class is created, and an instance of the class is used to register an event handler for mouse movement. The event handler function prints out the details of the mouse movement to the console. 

          
           
import java.awt.event.MouseMotionAdapter;
public class MyMotionListener extends MouseMotionAdapter{
    public void mouseMoved(MouseEvent e){
        System.out.println("Mouse moved at (" + e.getX() + ", " + e.getY() + ")");
    }
}	

MyMotionListener listener = new MyMotionListener();	
addMouseMotionListener(listener);
The above code snippet creates an instance of MyMotionListener, which extends the MouseMotionAdapter class, and registers an event handler for mouse movement. When the mouse moves, the event handler will print the (x,y) coordinates of the cursor to the console.

Example: Drawing with Mouse Drags

The following example demonstrates how to use the MouseMotionAdapter class to create a simple drawing application. In this example, a Java class extending the MouseMotionAdapter class is created, and an instance of the class is used to register an event handler for mouse dragging. The event handler then draws a line on the screen based on the mouse's current position. 

          
           
import java.awt.*;
import java.awt.event.MouseMotionAdapter;

public class DrawLineApp extends MouseMotionAdapter {
   private Point startPosition;
   private Color lineColor;

   public DrawLineApp(Color color) {
      this.lineColor = color;
   }

   public void mouseDragged(MouseEvent e) {
      Graphics2D graphics2d = (Graphics2D) e.getComponent().getGraphics();
      graphics2d.setColor(lineColor);
      if (startPosition != null) {
         int x = Math.min(startPosition.x, e.getX());
         int y = Math.min(startPosition.y, e.getY());
         int width = Math.abs(startPosition.x - e.getX());
         int height = Math.abs(startPosition.y - e.getY());
         graphics2d.drawLine(startPosition.x, startPosition.y, e.getX(), e.getY());
         graphics2d.fillRect(x, y, width, height);
      } 
      startPosition = e.getPoint();
   }

}
