io.swagger.annotations.ApiResponse

The io.swagger.annotations.ApiResponse annotation is a Java class used to declare the response that an API endpoint will return. It is used in conjunction with the io.swagger.annotations.Api annotation to create a definition of the responses that an endpoint may return. By using these annotations, developers can easily generate and share detailed documentation for their APIs, specifying the exact responses that each endpoint will return.

Syntax

The syntax for the io.swagger.annotations.ApiResponse annotation is as follows: 

          
           
@ApiResponse(code = <responsecode>, message = <message>, response = <responseclass>)
</responseclass></message></responsecode>

Parameters

  • code: The HTTP status code that this response status will return.
  • message: The response message that will be returned with the response.
  • response: The expected response object.

Examples

Example 1: Successful Response

In this example, we are creating a successful response, with a status code of 200. 

          
           
@ApiResponse(code = 200, message = "Success", response = User.class)

Example 2: Bad Request Response

In this example, we are creating a bad request response, with a status code of 400. 

          
           
@ApiResponse(code = 400, message = "Bad Request", response = Error.class)

In this example, we are creating an internal server error response, with a status code of 500.
