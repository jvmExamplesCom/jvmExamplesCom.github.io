Using the javafx.beans.property.StringProperty Class

javafx.beans.property.StringProperty

java.lang.String

Creating a StringProperty

Creating a 

          
           
StringProperty
is straightforward. To create one, simply use the default constructor: 

          
           
StringProperty strProp = new StringProperty();
The initial value of the property will be populated with an empty string. If you would like to specify an initial value, you can do so by using the constructor that takes a string parameter: 

          
           
StringProperty strProp = new StringProperty("Hello World!");
The above code creates a 

          
           
StringProperty
with an initial value of "Hello World!".

StringProperty
can be accessed or modified using the following methods: 
  • getValue()
    - Returns the current value of the property.

strProp.setValue("Goodbye World!");

intProp.bind(strProp);
