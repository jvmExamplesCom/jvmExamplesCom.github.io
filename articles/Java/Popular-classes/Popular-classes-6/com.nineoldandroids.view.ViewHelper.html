ViewHelper - com.nineoldandroids.view

ViewHelper is a utility class that offers easy and access to common view-related operations such as translating, rotating, and scaling views. It is a part of the nineoldandroids library, which is a collection of backport methods from Android ICS.

Importing ViewHelper

In order to use ViewHelper, you must include it in your project by adding the nineoldandroid JAR as a dependency. The easiest way to do this is through Gradle:


          
           
dependencies {
  compile 'com.nineoldandroids:library:2.4.0'
}

Common View Helper Methods

Below are the methods that are available in ViewHelper. Each of these methods have several overloads and additional parameters.

  • setTranslationX(float) - Sets the view's X translation.
  • setTranslationY(float) - Sets the view's Y translation.
  • setTranslation(float, float) - Sets the view's X and Y translations.
  • setRotation(float) - Sets the view's rotation.
  • setRotationX(float) - Sets the view's X rotation.
  • setRotationY(float) - Sets the view's Y rotation.
  • setScaleX(float) - Sets the view's X scale.
  • setScaleY(float) - Sets the view's Y scale.
  • setScale(float, float) - Sets the view's X and Y scales.
  • setPivotX(float) - Sets the view's X pivot.
  • setPivotY(float) - Sets the view's Y pivot.
  • setAlpha(float) - Sets the view's alpha value.

Code Examples

Below is a code example of simple usage of ViewHelper. This example sets the translation on a text view.


          
           
TextView textView = (TextView) findViewById(R.id.textView);
ViewHelper.setTranslationX(textView, 100.0f);

The code example below shows a more complex usage of ViewHelper. This example sets the scale, rotation, and pivot on an image view.


          
           
ImageView imageView = (ImageView) findViewById(R.id.imageView);
ViewHelper.setPivotX(imageView, 200.0f);
ViewHelper.setPivotY(imageView, 400.0f);
ViewHelper.setScaleX(imageView, 2.0f);
ViewHelper.setScaleY(imageView, 2.0f);
ViewHelper.setRotationX(imageView, 45.0f);

Using Animations

ViewHelper is often used in conjunction with Animation in order to achieve complex effects. The following code example animates a view's alpha, scale, and translation.


          
           
Animation animation = AnimationUtils.loadAnimation(this, R.anim.my_animation);
animation.setFillAfter(true);

ViewHelper.setScaleX(view, 2);

ViewHelper provides an easy way to perform common view operations. By using ViewHelper and Animation together, developers can create complex visual effects in their applications.
