javax.naming.directory.Attributes

The javax.naming.directory.Attributes class represents a collection of attributes associated with a directory object. It contains methods to add, modify and delete attributes. This article explains the basics of this class and some examples.

Constructors

The Attributes class provides two constructors: 

          
           
public Attributes()
public Attributes(int capacity)

The first one constructs an empty Attributes object. The second has an optional parameter for setting the initial capacity of the attribute list.

Methods

The Attributes class contains various methods to manipulate attributes. Here are some of them:


          
           
public void put(Attribute attribute);
public Attribute remove(String attrID);
public Attribute get(String attrID);
public int size();
public NamingEnumeration<attribute> getAll();
</attribute>
The 

          
           
put()
method adds a new attribute to the attribute list. The 

          
           
remove()
and 

          
           
get()
methods perform their respective operations. The 

          
           
size()
method returns the number of attributes in the list and the 

          
           
getAll()
method returns an enumerated list of all attributes.

Example 1

In this example, we will create a new Attributes object and add some attributes. First, we create the object:


          
           
Attributes attrs = new Attributes();

Then we create three new attribute objects and add them to the list:


          
           
Attribute attr1 = new BasicAttribute("name", "John");
Attribute attr2 = new BasicAttribute("age", 30);
Attribute attr3 = new BasicAttribute("city", "New York");

attrs.put(attr1);
attrs.put(attr2);
attrs.put(attr3);
We can now print out the list of attributes: 

          
           
NamingEnumeration<Attribute> e = attrs.getAll();
while (e.hasMore()) {    
    Attribute attr = (Attribute) e.next();
    System.out.println(attr);
}
The output should be something like this: 

          
           
name: John
age: 30
city: New York

Example 2

In this example, we will use the 

          
           
remove()
and 

          
           
get()
methods to remove and retrieve attributes. First, we create an attribute object and add it to the list: 

          
           
Attribute attr = new BasicAttribute("name", "John");
attrs.put(attr);

We can now attempt to remove and retrieve the attribute:


          
           
Attribute removedAttr = attrs.remove("name");
System.out.println(removedAttr);

Attribute retrievedAttr = attrs.get("name");
System.out.println(retrievedAttr);
The output should be something like this: 

          
           
name: John
null
This shows that the 

          
           
remove()
method was able to successfully remove the attribute from the list, while the 

          
           
get()
method returned null, indicating that the attribute is no longer present.
