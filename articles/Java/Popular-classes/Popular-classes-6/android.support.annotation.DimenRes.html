android.support.annotation.DimenRes

Overview

The 

          
           
android.support.annotation.DimenRes
annotation is used to indicate that an integer argument, field or method return value is expected to be a resource reference to a dimension resource. This annotation is intended to prevent errors like passing in a layout resource id instead of a dimension resource id.

Example Usage

In the example below, 
getDimension()
takes a 
@DimenRes
argument which indicates the id of a dimension resource. 

          
           
public float getDimension(@DimenRes int dimenId) {
    // ...
}
The following example shows using the 
@DimenRes
annotation on a field. 

          
           
@DimenRes 
private int minHeight;

Alternatives

For arguments, fields and method return values that can take any type of resource, you can use the 

@AnyRes
annotation.
