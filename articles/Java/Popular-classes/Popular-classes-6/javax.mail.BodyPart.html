javax.mail.BodyPart

javax.mail.BodyPart is an abstract class that holds a body part of a MIME message and implements the Part interface. It is defined in the javax.mail package.

Methods

The javax.mail.BodyPart class has several methods, including:

  • getContent()
    - This method retrieves the content of this body part.
  • 
            
             
setDataHandler(DataHandler dh)
    - This method associates a DataHandler object with this body part.
  • getSize()
    - This method returns the size of this body part.
  • getFileName()
    - This method returns the file name of this body part.

getHeader(String name)
    - This method returns all the headers for the specified header name.
