javax.mail.BodyPart

javax.mail.BodyPart is an abstract class that holds a body part of a MIME message and implements the Part interface. It is defined in the javax.mail package.

Methods

The javax.mail.BodyPart class has several methods, including:

  • getContent()
    - This method retrieves the content of this body part.

  • getFileName()

getHeader(String name)
