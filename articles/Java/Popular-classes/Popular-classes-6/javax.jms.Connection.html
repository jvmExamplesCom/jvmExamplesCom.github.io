javax.jms.Connection

Java Message Service (JMS) is an important part of Enterprise JavaBeans (EJB). JMS provides a framework that facilitates message-oriented communication. The javax.jms.Connection interface is used to create a connection between the JMS provider and the client application.

What is the javax.jms.Connection Interface?

The javax.jms.Connection interface is an abstract class that is used to create a connection between the JMS provider and the client application. This interface provides methods for creating Connection objects, setting connection parameters, and consuming messages using MessageListeners. It also provides methods for establishing transactions and for sending messages.

Methods of javax.jms.Connection Interface

The javax.jms.Connection interface has several methods which can be used to create, configure, and close connections. Some of these methods are as follows:
  • createConnection: This method is used to create a connection to the JMS provider.
  • close: This method is used to close the connection.
  • setClientID: This method is used to set the Client ID for the connection.
  • setExceptionListener: This method is used to register an ExceptionListener on the connection.
  • createSession: This method is used to create a Session object.
  • commit: This method is used to commit a transaction.
  • rollback: This method is used to rollback a transaction.
  • createQueueBrowser: This method is used to create a QueueBrowser object.
  • createDurableConnectionConsumer: This method is used to create a DurableConnectionConsumer object.

Example

In this example, we will create a JMS connection and use it to send a message. 

          
           
import javax.jms.Connection;
import javax.jms.ConnectionFactory;
import javax.jms.JMSException;
import javax.jms.MessageProducer;
import javax.jms.Session;
import javax.jms.TextMessage;

public class JMSConnectionExample {

    public static void main(String[] args) throws JMSException {
        ConnectionFactory connFactory = new com.sun.messaging.ConnectionFactory();  
        Connection connection = connFactory.createConnection();  
        Session session = connection.createSession(false, Session.AUTO_ACKNOWLEDGE);   
        MessageProducer messageProducer = session.createProducer(null);  
        TextMessage textMessage = session.createTextMessage("This is a test message");
        messageProducer.send(textMessage);
        System.out.println("Message sent successfully!");
    }
}

Message sent successfully!
