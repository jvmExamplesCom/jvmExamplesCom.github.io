javax.imageio.ImageWriteParam

The javax.imageio.ImageWriteParam class is an abstract class used to describe how an ImageWriter should write out an image as part of the Java Image I/O API. It allows for things such as controlling compression and encoding parameters of images, and even allowing the user to set which type of encoding to use.
