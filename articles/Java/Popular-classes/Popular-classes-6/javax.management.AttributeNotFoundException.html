javax.management.AttributeNotFoundException

javax.management.AttributeNotFoundException is a subclass of the 


          
           
javax.management.MBeanException
class which represents an exception thrown during an attribute access in the Java Model Management API (JMX). This exception occurs when a requested attribute of an MBean does not exist or cannot be retrieved. It is thrown either by the getter method of the MBean or by the MBean server’s 
getAttribute
method.
