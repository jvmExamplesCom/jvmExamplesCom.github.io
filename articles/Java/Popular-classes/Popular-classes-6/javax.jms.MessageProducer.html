javax.jms.MessageProducer

The javax.jms.MessageProducer interface is an important part of the Java Message Service (JMS) API and is responsible for the sending of messages from a JMS client to a JMS provider. It provides control over message delivery options, such as delivery mode and time-to-live.

The javax.jms.MessageProducer interface provides three methods: send(), setDeliveryMode() and setTimeToLive(). The send() method is used to actually send the message, while the setDeliveryMode() and setTimeToLive() methods are used to configure the delivery options.

Example 1: Create a producer and send a message.


          
           
Connection connection = ConnectionFactory.createConnection();
Session session = connection.createSession(false, Session.AUTO_ACKNOWLEDGE);
Topic newsTopic = session.createTopic("news");
MessageProducer producer = session.createProducer(newsTopic);
TextMessage message = session.createTextMessage("Latest News");
producer.send(message);

In this example, we create a connection and session to a JMS provider, create a topic and producer, create a TextMessage and then send it using the producer's send() method. The message sent is "Latest News".

Example 2: Set delivery mode and time-to-live.


          
           
MessageProducer producer = session.createProducer(newsTopic);
producer.setDeliveryMode(DeliveryMode.NON_PERSISTENT);
producer.setTimeToLive(10000); // 10 seconds

In this example, we set the delivery mode of the producer to non-persistent and the time-to-live to 10 seconds. This will affect all messages sent by this producer until it is changed again.

Example 3: Send multiple messages.


          
           
for (int i = 0; i < 5; i++) {
    TextMessage msg = session.createTextMessage("Message " + i);
    producer.send(msg);
}

In this example, we send 5 messages in a loop using the same producer. The messages sent are "Message 0" to "Message 4".
