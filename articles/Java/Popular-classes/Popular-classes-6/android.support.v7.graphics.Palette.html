Introduction to android.support.v7.graphics.Palette

The 

          
           
android.support.v7.graphics.Palette
class provides an API for retrieving colors from a given bitmap. It can be used to create color palettes based on the colors extracted from images, allowing developers to tailor their user interface to closely match what they see in the image. The Palette class works by extracting prominent colors from the source image, and making them available as a ColorStateList or as a set of RGB colors. These colors can then be used to create custom views, or to set the background color for a view. In this article, we will go over how to use the Palette API to create a color palette from an image. We will also cover how to create a view with colors from the generated palette.

Getting Started

First, you will need to include the library in your build.gradle file. 

          
           
dependencies {
    implementation 'com.android.support:Palette-v7:27.1.1'
}
Once you have included the library in your build.gradle file, you can start using it in your code.

Bitmap

Bitmap

Bitmap bitmap = BitmapFactory.decodeResource(context.getResources(), R.drawable.image);
