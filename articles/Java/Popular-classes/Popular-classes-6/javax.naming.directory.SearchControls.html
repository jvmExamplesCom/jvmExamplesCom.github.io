javax.naming.directory.SearchControls in Java

The javax.naming.directory.SearchControls class provides a way to control the search of a directory service, such as LDAP or NAM. It is generally used with the javax.naming.directory package which provides an API for searching and manipulating directory data.

The SearchControls class has several methods and properties which can be used to control the search of a directory service. This includes setting limits on the number of results returned, the time limit of the search, and the types of objects that can be searched for.

SearchControls Constructors

searchScope Property

The searchScope property defines the scope of the search. It is an integer value which can have one of three values:

SearchControls.OBJECT_SCOPE
    - searches only the object itself. This is the default value.
  • 
            
             
SearchControls.ONELEVEL_SCOPE
    - searches only objects in the same level as the starting object.
  • 
            
             
SearchControls.SUBTREE_SCOPE
    - searches for objects in all levels of the tree structure.

countLimit Property

The countLimit property sets a limit on the number of objects returned by the search. It is an integer value which can be set to any positive number. If the limit is set to zero (0) then no limit is applied. By default the limit is set to zero (0).

timeLimit Property

The timeLimit property sets a limit on the time taken to execute the search. It is an integer value which can be set to any positive number. The time is specified in milliseconds. If the limit is set to zero (0) then no limit is applied. By default the limit is set to zero (0).

attributesToReturn Property

The attributesToReturn property defines what object attributes the search should return. It is an array of strings which specify the names of the attributes that should be returned. If the array is empty or null then all attributes will be returned. By default, all attributes will be returned.

returnObjFlag Property

The returnObjFlag property defines whether the search should return objects or just their attributes. It is a boolean value which can be either true or false. If set to true then objects will be returned, if set to false then only the requested attributes will be returned. By default the flag is set to true.

derefAliases Property

The derefAliases property defines whether aliases in the directory should be resolved. It is a boolean value which can be either true or false. If set to true then aliases will be resolved, if set to false then aliases will not be resolved. By default the flag is set to true.
