org.skyscreamer.jsonassert.JSONAssert

JSONAssert is a powerful library for validating JSON data in Java. It enables developers to perform structural and value-based assertions on a JSON document or string without the need to first convert the JSON document into an object. JSONAssert supports a number of features that enable developers to control the assertion process including strict ordering, ignoring elements, custom matcher functions and more.

JSONAssert.assertEquals("{\"name\":\"John\", \"age\":30}",

This code asserts that two JSON documents are equal regarding their structure and values, regardless of their order. The third parameter (true) indicates that ordering is not important
