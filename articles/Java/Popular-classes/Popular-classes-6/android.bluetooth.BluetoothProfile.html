android.bluetooth.BluetoothProfile

The android.bluetooth.BluetoothProfile class provides basic functions that are necessary to implement a Bluetooth profile. Bluetooth profiles are standard protocols and applications that provide specific functionality and services on top of the Bluetooth radio. A Bluetooth profile is associated with a particular Bluetooth interface and a particular use case.

Methods

The android.bluetooth.BluetoothProfile class provides several methods for implementing a Bluetooth profile. These methods include:

  • 
            
             
connect(BluetoothDevice)
    - Connects to a remote Bluetooth device.
  • 
            
             
disconnect(BluetoothDevice)
    - Disconnects from a remote Bluetooth device.
  • 
            
             
getConnectionState(BluetoothDevice)
    - Returns the current connection state of the specified Bluetooth device.
  • 
            
             
getConnectedDevices()
    - Returns a list of currently connected Bluetooth devices.
  • 
            
             
listenUsingRfcommWithServiceRecord(String, UUID )
    - Creates an RFCOMM server socket listening on the specified service name and UUID.
  • close()
    - Closes an RFCOMM server socket.

Example

The following example code snippet illustrates how to use the BluetoothProfile class to implement a Bluetooth profile.


          
           
// Create an instance of the BluetoothProfile class
BluetoothProfile profile = new BluetoothProfile();

// Connect to a remote Bluetooth device
profile.connect(RemoteDevice);

// Get the current connection state of the remote device
int state = profile.getConnectionState(RemoteDevice);

// List devices currently connected to this profile
Set<BluetoothDevice> connectedDevices = profile.getConnectedDevices();

// Disconnect from the remote device
profile.disconnect(RemoteDevice);

// Create an RFCOMM server socket
String serviceName = "sample_server";
UUID uuid = UUID.fromString("99999999-8888-7777-6666-555555555555");
BluetoothServerSocket serverSocket = profile.listenUsingRfcommWithServiceRecord(serviceName, uuid);

// Close the server socket
serverSocket.close();
