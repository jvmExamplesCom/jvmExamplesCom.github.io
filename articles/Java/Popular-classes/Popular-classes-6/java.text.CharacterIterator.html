java.text.CharacterIterator

Java CharacterIterator is a class used to iterate through collections of characters. It provides a convenient way to perform operations such as counting characters, extracting substrings, and retrieving specific values. It can be used to quickly parse text and extract words and other information. CharacterIterator objects are created by invoking the invocation constructor java.text.CharacterIterator.create().

Example


          
           
//Create a new CharacterIterator from a string 
String str = "Hello World"; 
CharacterIterator iter = CharacterIterator.create(str); 

// Use character iterator to get the next character 
while (iter.hasNext()) { 
  char c = iter.next(); 
  System.out.print(c); 
}
Output: 

          
           
Hello World

Methods

The CharacterIterator class provides several useful methods for navigating and manipulating the character collection it was created with. These include: 
  • hasNext()
    : Returns true if there are any more characters in the collection.
  • next()
    : Returns the next character in the collection.
  • getIndex()
    : Returns the current index of the character in the collection.
  • setIndex()
    : Sets the current index of the character in the collection.
  • reset()
    : Resets the CharacterIterator to its initial state.
  • length()
    : Returns the length of the character collection.
  • 
            
             
substring(int startIndex, int endIndex)
    : Returns a substring of the character collection from the specified start and end indices.
  • charAt(int index)
    : Returns the character at the specified index in the character collection.
