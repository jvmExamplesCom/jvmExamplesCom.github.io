io.reactivex.ObservableSource

The 

          
           
io.reactivex.ObservableSource
is a base class for observable sources, which can be subscribed to for push-based notifications. It allows for efficient and concurrency-safe operations on a stream of values, such as mapping, filtering, folding, or scanning.

Creating an Observable

An observable can be created from different sources. For example, it can be created from another observable or a list, or by creating an empty one with no elements.

For example, an observable can be created from a list of items, where each item is represented as a string:


          
           
List<String> items = new ArrayList<>();
items.add("Item 1");
items.add("Item 2");
Observable<String> observable = Observable.fromIterable(items);

An empty observable can be created with the following code:


          
           
Observable<String> observable = Observable.empty();

Subscribing to an Observable

Once an observable has been created, it can be subscribed to and will start emitting values. This is done using the 

subscribe()
method, which takes a 
Subscriber
object as a parameter.

When subscribing to an observable, the subscriber will receive all of the values emitted by the observable. In the example below, each value received by the subscriber will be printed in the console:


          
           
observable.subscribe(new Subscriber<String>() {
    @Override
    public void onNext(String item) {
        System.out.println(item);
    }

    @Override
    public void onError(Throwable e) {
        e.printStackTrace();
    }

    @Override
    public void onComplete() {
        System.out.println("Done!");
    }
});

Operators

Operators are functions that can be applied to an observable. They allow you to manipulate and transform the values emitted by an observable. For example, the 

map()
operator can be used to transform each value emitted by an observable into a new value.

In the example below, each string is mapped to its length:


          
           
Observable<Integer> mappedObservable = observable.map(String::length);
The 
filter()
operator can be used to filter out values that do not satisfy a certain condition, for example: 

          
           
Observable<String> filteredObservable = observable.filter(s -> s.length() > 3);

Conclusion

The 

          
           
io.reactivex.ObservableSource
is a powerful tool for managing asynchronous streams of data. It provides a simple and efficient way to observe, filter, transform, and subscribe to data.
