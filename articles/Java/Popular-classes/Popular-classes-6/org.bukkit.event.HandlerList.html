org.bukkit.event.HandlerList: Overview

org.bukkit.event.HandlerList is a class used in the Bukkit API for Java to handle events in Minecraft. This class is responsible for maintaining lists of event handlers and invoking them when an event is fired. It is also responsible for unregistering handlers and clearing lists.
