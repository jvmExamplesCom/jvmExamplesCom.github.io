Working with JSONArray in Java

JSONArray is a class from the Vertx library for working with JSON arrays. It provides methods for adding and retrieving data from an array, as well as allowing manipulation of the array such as sorting and filtering. In this article, we'll take a look at how to use the methods in JSONArray to perform different tasks.

Creating a JSONArray

Before you can start manipulating an array, you need to create it. You can do this by passing in a JSON encoded array as a string, or by creating a new JSONArray object. First, let's look at how to create a new instance of JSONArray using the static method 

          
           
JSONArray.parse(String)
. This method takes a JSON encoded string and will return a new JSONArray instance. For example: 

          
           
String jsonString = "[1,2,3,4,5]";
JSONArray array = JSONArray.parse(jsonString);
You can also create a new empty array object like so: 

          
           
JSONArray array = new JSONArray();

Adding Elements to a JSONArray

Once you have created an array, you can add elements to it. To add an element to an array, use the 

          
           
add(Object)
method. This method takes an Object as its argument and adds it to the end of the array. For example: 

          
           
array.add(6);
The 

          
           
add
method can add any type of Object, including primitive objects, Strings, JSONObjects, and other JSONArrays. For example: 

          
           
String str = "Hello World";
JSONObject obj = new JSONObject();
JSONArray subArray = new JSONArray();

array.add(str);
array.add(obj);
array.add(subArray);
Note that if you are adding a primitive object, such as an int, you must use the wrapper class such as Integer. 

          
           
// This is invalid
array.add(7);

// This is valid
array.add(Integer.valueOf(7));

Retrieving Elements from a JSONArray

If you need to retrieve an element from an array, you can do so using the 

          
           
get(int)
method. This method takes an int argument, which is the index of the element you wish to retrieve. For example: 

          
           
Object firstElement = array.get(0);
The 

          
           
get
method returns an Object, so you should cast the object to the appropriate type to use it. For example: 

          
           
Integer secondElement = (Integer) array.get(1);

Sorting a JSONArray

The Vertx library also provides an easy way to sort a JSONArray. The 

          
           
sort(Comparator)
method takes a Comparator as its argument, which defines the comparison rules used to sort the elements of the array. For example, if we wanted to sort an array of integers in ascending order: 

          
           
JSONArray array = JSONArray.parse("[3,2,5,1,4]");

array.sort(Comparator.naturalOrder());
After this code is executed, the array will be sorted in ascending order, with 1 being the first element, 2 being the second, and so on.

Filtering a JSONArray

In addition to sorting, the Vertx library also provides a way to filter the elements of a JSONArray. The 

          
           
filter(Predicate)
method takes a Predicate as its argument, which defines the criteria used to filter the elements of the array. For example, if we wanted to filter an array to only keep elements that are greater than 3: 

          
           
JSONArray array = JSONArray.parse("[3,2,5,1,4]");

array.filter(item -> (int) item > 3);
After this code is executed, the array will contain only the elements 5 and 4, since they are the only elements greater than 3.

Conclusion

In this article we looked at how to work with JSONArray in Java using the Vertx library. We looked at how to create a new array, add elements to it, retrieve elements from it, sort it, and filter it. We hope this article has been helpful in getting you started with manipulating JSON arrays in Java.
