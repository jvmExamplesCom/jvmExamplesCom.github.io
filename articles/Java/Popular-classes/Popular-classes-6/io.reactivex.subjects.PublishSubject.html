io.reactivex.subjects.PublishSubject

What is PublishSubject?

io.reactivex.subjects.PublishSubject is a type of Subject from the ReactiveX library that emits items to its subscribers. This is a variant of ReplaySubject, except it does not emit any items stored in its history. It will only emit items after a subscriber has subscribed. This can be used to bridge between two different threads and allow communication between them.

Code Examples

The following code example creates a PublishSubject and then subscribes to it. 

          
           
PublishSubject<String> subject = PublishSubject.create();
 
subject.subscribe(value -> System.out.println("Subscriber 1 got value:" + value));
The following code example adds an item to the PublishSubject. 

          
           
subject.onNext("Hello World!");
The following code example prints out the item added to the PublishSubject. 

          
           
Subscriber 1 got value: Hello World!

Conclusion

io.reactivex.subjects.PublishSubject is a useful tool for bridging between different threads and allowing communication between them. It is easy to use and allows for a simple way of passing messages between different threads.
