Introduction to org.apache.commons.text.StringEscapeUtils

Code Examples

Now let’s take a look at some code examples that show how we can use the 

          
           
StringEscapeUtils
class in practice. We’ll start with an example of escaping and unescaping an HTML String. 

          
           
String htmlString = "<html>Hello World</html>";

// Escaping the String
String escapedHTMLString = StringEscapeUtils.escapeHtml4(htmlString);
System.out.println("Escaped HTML String: "+ escapedHTMLString); // &amp;lt;html&amp;gt;Hello World&amp;lt;/html&amp;gt;

// Unescaping the String
String unescapedHTMLString = StringEscapeUtils.unescapeHtml4(escapedHTMLString);
System.out.println("Unescaped HTML String: "+ unescapedHTMLString); // <html>Hello World</html>

String jsonString = "{\"name\":\"John Doe\", \"age\":30}";
