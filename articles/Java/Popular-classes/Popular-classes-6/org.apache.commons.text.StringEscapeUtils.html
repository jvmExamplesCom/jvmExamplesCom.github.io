Introduction to org.apache.commons.text.StringEscapeUtils

The 

          
           
org.apache.commons.text.StringEscapeUtils
is a Java utility class from the Apache Commons project. It provides a set of static methods for escaping, unescaping, and translating String objects to and from HTML, XML, JSON, Java, and other formats. In this article, we will explore the different methods available in this class and discuss some code examples showing how they can be used in practice.

StringEscapeUtils

Code Examples

Now let’s take a look at some code examples that show how we can use the 

          
           
StringEscapeUtils
class in practice. We’ll start with an example of escaping and unescaping an HTML String. 

          
           
String htmlString = "<html>Hello World</html>";

// Escaping the String
String escapedHTMLString = StringEscapeUtils.escapeHtml4(htmlString);
System.out.println("Escaped HTML String: "+ escapedHTMLString); // &amp;lt;html&amp;gt;Hello World&amp;lt;/html&amp;gt;

// Unescaping the String
String unescapedHTMLString = StringEscapeUtils.unescapeHtml4(escapedHTMLString);
System.out.println("Unescaped HTML String: "+ unescapedHTMLString); // <html>Hello World</html>
Next, let’s see how we can escape and unescape a JSON String. 

          
           
String jsonString = "{\"name\":\"John Doe\", \"age\":30}";

// Escaping the String
String escapedJSONString = StringEscapeUtils.escapeJson(jsonString);
System.out.println("Escaped JSON String: "+ escapedJSONString); // \u0022name\u0022:\u0022John Doe\u0022, \u0022age\u0022:30

// Unescaping the String
String unescapedJSONString = StringEscapeUtils.unescapeJson(escapedJSONString);
System.out.println("Unescaped JSON String: "+ unescapedJSONString); // {"name":"John Doe", "age":30}

String javaString = "\"Hello World!";

String unescapedJavaString = StringEscapeUtils.unescapeJava(escapedJavaString);
System.out.println("Unescaped Java String: "+ unescapedJavaString); // "Hello World!
