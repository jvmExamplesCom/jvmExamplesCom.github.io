android.graphics.PathMeasure

The android.graphics.PathMeasure class is a utility class used to measure the length of a path, determine whether a path is closed or not, find the position and tangent angle of a path at an arbitrary distance, and measure the location of successive points along a path. It can be used in conjunction with canvas drawing commands for complex shapes, for example when drawing text on a curved path.

Constructors

PathMeasure has two constructors, one which takes a Path object and another which takes a Path and a boolean flag indicating whether the Path should be treated as a closed contour: 

          
           
PathMeasure(Path path)
PathMeasure(Path path, boolean forceClosed)
The forceClosed flag can be used to override the automatic detection of closed contours by PathMeasure.

Methods

PathMeasure has several methods available to query the path, measure distances along it and to find the position and tangent at a particular point: 

          
           
float getLength()
boolean isClosed()
boolean getPosTan(float distance, float[] pos, float[] tan)
boolean getMatrix(float distance, Matrix matrix, int flags)
boolean nextContour()
The getLength() method returns the length of the path, in pixels. The isClosed() method can be used to check if the path is closed. The getPosTan() method is used to get the position and tangent of the path at a given distance from the start of the path. This is useful for calculating the position of points along a curved path. The getMatrix() method can be used to transform points along the path according to a given transformation matrix. Finally, the nextContour() method can be used to move to the next sub-path in the path, when multiple sub-paths are used.

Examples

Let's look at some examples of using PathMeasure. Here's an example of measuring the length of a path: 

          
           
Path path = new Path();
// draw a heart shape
path.moveTo(200, 200);
path.cubicTo(250, 50, 350, 50, 400, 200);
