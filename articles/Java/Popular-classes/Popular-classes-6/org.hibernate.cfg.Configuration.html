The

class is an important part of the Hibernate configuration. This class provides methods to configure the internal settings of Hibernate, and acts as the main entry point for configuring a Hibernate session factory.

To create a Configuration object, we use the 

          
           
org.hibernate.cfg.Configuration
class's static 
configure()
method, passing in a Java configuration file as an argument: 

          
           
Configuration cfg = Configuration.configure("hibernate.cfg.xml");

Adding Mapping Files

Configuring Properties

cfg.setProperty("hibernate.dialect", "org.hibernate.dialect.PostgreSQLDialect");
