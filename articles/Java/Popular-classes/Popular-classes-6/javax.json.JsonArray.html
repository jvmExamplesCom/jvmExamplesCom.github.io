javax.json.JsonArray

javax.json.JsonArray is a class used for creating and manipulating JSON arrays in Java. It provides methods for working with the data, including add, remove, get values, and so on. In this article, we'll take a look at how to use JsonArray and its methods.

Creating a JsonArray Instance

A JsonArray instance can be created from a JSON text string, a byte array, or an input stream. To create a JsonArray instance from a JSON text string, use the static factory method Json.createArrayBuilder() as shown in the following example:


          
           
JsonArray jsonArray = Json.createArrayBuilder()
      .add("John")
      .add("Peter")
      .add("Thomas")
      .build();

The above code creates a JsonArray instance and adds three strings (John, Peter, and Thomas).

Adding Elements to a JsonArray

Elements can be added to a JsonArray instance using the add() and addAll() methods. The add() method takes one parameter, which can be either a string, number, or Boolean value. For example:


          
           
jsonArray.add("John");
jsonArray.add(false);
jsonArray.add(1000);

The addAll() method takes an array, list, or set of values and adds them to the JsonArray, as shown in the following example:


          
           
Object[] values = { "John", false, 1000 };
jsonArray.addAll(values);

Removing Elements from a JsonArray

Elements can be removed from a JsonArray instance using the remove() and removeAll() methods. The remove() method takes one parameter, which is the index of the element to be removed. For example:


          
           
jsonArray.remove(2);

This will remove the third element (index 2) from the JsonArray. The removeAll() method takes an array, list, or set of values and removes them from the JsonArray, as shown in the following example:


          
           
Object[] values = { "John", false, 1000 };
jsonArray.removeAll(values);

Getting Values from a JsonArray

Values can be retrieved from a JsonArray instance using the get() method. This method takes one parameter, which is the index of the value to be retrieved. For example:


          
           
String name = jsonArray.get(0); // returns "John"

Iterating Through a JsonArray

A JsonArray instance can be iterated through using the forEach() method. This method takes a consumer as a parameter and applies it to each element in the JsonArray. For example:


          
           
jsonArray.forEach(element -> System.out.println(element));

In the above example, each element in the JsonArray will be printed out to the console.
