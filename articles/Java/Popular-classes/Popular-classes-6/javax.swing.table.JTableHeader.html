javax.swing.table.JTableHeader

The JTableHeader class is an extension of the JComponent class. It allows for the customization of the table header, including the ability to add headers to multiple columns. It also provides a built-in listener for the user's mouse events, allowing the user to interact with the table.

Usage

To use JTableHeader, instantiate an instance of it and add it to a JTable instance like this:


          
           
JTable table = new JTable(data, columnNames);
JTableHeader tableHeader = new JTableHeader(table.getColumnModel());
table.setTableHeader(tableHeader);

Once an instance of JTableHeader is added to a JTable, any changes made to the data in the table will affect the JTableHeader, such as when the table is resized or its contents are changed.

Example

This example shows how to set the default renderer for the JTableHeader instance, and also adds a mouse listener to show the columIndex of the column the user is hovering over.


          
           
JTable table = new JTable(data, columnNames);
JTableHeader header = table.getTableHeader();
header.setDefaultRenderer(new DefaultTableCellRenderer(){
    public Component getTableCellRendererComponent(JTable table, Object value, boolean isSelected, boolean hasFocus, int row, int column){
        JLabel label = new JLabel(value.toString());
        label.setHorizontalAlignment(JLabel.CENTER);
        label.setFont(label.getFont().deriveFont(Font.BOLD));
        return label;
    }
});
header.addMouseListener(new MouseAdapter(){
    public void mouseMoved(MouseEvent e){
        int col = header.columnAtPoint(e.getPoint());
        System.out.println("You are hovering over column " + col);
    }
});

By setting the default renderer and adding a mouse listener, we have created a custom table header that interacts with the user.
