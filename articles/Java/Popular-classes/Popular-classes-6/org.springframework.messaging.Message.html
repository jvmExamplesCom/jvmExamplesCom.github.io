Introduction to org.springframework.messaging.Message

The org.springframework.messaging.Message class is part of the Spring Framework for Java and is used to send messages between components that are participating in the messaging system.

The Message class supports various message headers, payloads and methods to create, manipulate and read messages. It supports headers of type String, byte[], Map and Object, and payloads can be any Object. As well as this, it also supports a default encoding scheme, envelope addressing and priority.

Creating Messages

Creating a message is relatively simple. Here is an example of how to create a basic message:


          
           
Message<String> helloMessage = 
    MessageBuilder.withPayload("Hello World").build();

In this example we have created a message with a payload of type String. This can be customized using various other methods.

Messaging Headers

Messaging Headers can be added to a message when creating it or after the message has been created. Here is an example of adding a header after the message has been created:


          
           
helloMessage.getHeaders().put("messageType", "greeting");

This adds a header called "messageType" to the message with a value of "greeting". The header type is either a String, byte[], Map or Object.

Envelope Addressing

The Message class also supports envelope addressing, which can be used to route messages between components. Envelopes are created by calling the Message.envelope() method and can contain a MessageId, CorrelationId, and ReplyTo address.


          
           
MessageEnvelope envelope = helloMessage.envelope();

The MessageId is an identifier assigned to the message by the sender. The CorrelationId is a way of relating this message to a previous message. Finally, the ReplyTo address is an address where the sender expects to receive replies to this message.

Priority

The Message class also supports the ability to set a priority on a message. This is done using the Message.setPriority() method and can be a value from 0 (lowest) to 10 (highest).


          
           
helloMessage.setPriority(5);

The priority level can then be retrieved using the Message.getPriority() method.

Default Encoding Scheme

The Message class also supports a default encoding scheme which is used to convert the payload of the message into a byte array if it is not already a byte array. The default encoding scheme is UTF-8 and can be changed by setting the System property "spring.messaging.default-encoding".


          
           
System.setProperty("spring.messaging.default-encoding", "ISO-8859-1");

Conclusion

The org.springframework.messaging.Message class is a powerful tool for sending messages between components in the Spring Framework for Java. It supports custom headers, envelope addressing and setting a priority on messages. It also supports a default encoding scheme and allows for the manipulation and reading of messages.
