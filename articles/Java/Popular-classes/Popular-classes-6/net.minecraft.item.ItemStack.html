ItemStack is an important class in the Java library for creating and manipulating items in the Minecraft game. Creating Items

createStack()

Item

ItemStack stack = ItemStack.createStack(Item.DIAMOND_PICKAXE);

ItemStack

getStackSize()

int size = stack.getStackSize();
