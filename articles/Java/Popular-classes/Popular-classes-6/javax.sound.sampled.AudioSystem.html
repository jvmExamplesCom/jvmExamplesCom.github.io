javax.sound.sampled.AudioSystem

The javax.sound.sampled.AudioSystem class is a part of the Java Sound API which provides access to audio system resources, such as lines and mixers, for the purposes of playing, recording, or converting audio data. This class also provides methods for querying properties of the system audio resources.

Constructors

The javax.sound.sampled.AudioSystem class has two constructors:


          
           
AudioSystem()
AudioSystem( AudioInputStream stream )

The first constructor is used to create an instance of AudioSystem. The second is used to create an AudioSystem from an AudioInputStream.

Methods

The javax.sound.sampled.AudioSystem class provides multiple methods for interacting with audio system resources.


          
           
getAudioInputStream( URL url )

This method creates an AudioInputStream object from the specified URL.


          
           
getMixerInfo()

This method retrieves information regarding the audio mixer devices installed on the system.


          
           
getLine( Line.Info lineInfo )

This method attempts to obtain a line having the specified characteristics.


          
           
getClip()

This method creates a Clip object which provides access to a sound clip resource.

Examples

The following code example demonstrates how to create an AudioSystem instance and use it to get a mixer info:


          
           
AudioSystem audioSystem = new AudioSystem();
Mixer.Info[] mixerInfos = audioSystem.getMixerInfo();
for ( Mixer.Info info : mixerInfos ) {
    System.out.println( info.getName() );
}

URL url = new URL( "https://example.com/audio.mp3" );
AudioInputStream audioInputStream = AudioSystem.getAudioInputStream( url );
AudioSystem audioSystem = new AudioSystem( audioInputStream );
