What is the MongoCollection class?

The MongoCollection class is a java.mongodb.client class that is used to interact with MongoDB collections. The MongoCollection class can be used to perform operations like creating, updating, and deleting documents. It also provides methods to query and aggregate collections.

MongoCollection<Document> customers = db.getCollection("customers");
