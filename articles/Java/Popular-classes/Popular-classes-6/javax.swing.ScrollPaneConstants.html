The ScrollPaneConstants class can be used in the creation of scroll panes. The following example creates a basic scroll pane with default constants applied:


          
           
JScrollPane scrollPane = new JScrollPane();

VERTICAL_SCROLLBAR_ALWAYS

VIEWPORT_FILL_VIEW
