javax.swing.ScrollPaneConstants is a class in the Java Swing package which provides many constants which are useful for working with scroll panes. The constants enable developers to configure the properties of scroll panes in order to create user friendly, robust applications.

Creating a Scroll Pane

The ScrollPaneConstants class can be used in the creation of scroll panes. The following example creates a basic scroll pane with default constants applied:


          
           
JScrollPane scrollPane = new JScrollPane();

Constant Values

HORIZONTAL_SCROLLBAR_ALWAYS
VERTICAL_SCROLLBAR_ALWAYS
VIEWPORT_ALWAYS_VISIBLE
VIEWPORT_FILL_VIEW

scrollPane.setVerticalScrollBarPolicy(ScrollPaneConstants.VERTICAL_SCROLLBAR_ALWAYS);
scrollPane.setHorizontalScrollBarPolicy(ScrollPaneConstants.HORIZONTAL_SCROLLBAR_ALWAYS);
