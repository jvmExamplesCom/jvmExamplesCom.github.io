org.apache.maven.model.Model

The org.apache.maven.model.Model class is an important part of the Apache Maven build system. It is a data structure that represents the POM or Project Object Model, which is a model of a project or module. Model model = new Model();
 model.setGroupId("org.apache");
 model.setArtifactId("maven.model");
 model.setVersion("1.0");
