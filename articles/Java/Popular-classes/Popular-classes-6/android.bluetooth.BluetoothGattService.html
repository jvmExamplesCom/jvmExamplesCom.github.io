android.bluetooth.BluetoothGattService

The android.bluetooth.BluetoothGattService class represents a GATT service that the Bluetooth device offers, containing a collection of BluetoothGattCharacteristic objects that define the characteristics and any related descriptors, and a list of all included BluetoothGattService objects for any child services.

A BluetoothGattService is a collection of characteristics and relationships to other services that encapsulate the behavior of part of a device. This class is used to create and expose an interface to a local GATT server, which a remote central can connect to and interact with.

Methods

This class provides the following methods:
  • addCharacteristic(): Adds a characteristic to the service.
  • addService(): Adds a service to the service.
  • getCharacteristic(): Gets a characteristic with a given UUID.
  • getCharacteristicCount(): Gets the number of characteristics in this service.
  • getIncludedServices(): Gets a list of included services.
  • getType(): Gets the type of this service.
  • getUuid(): Gets the UUID of this service.
  • setCharacteristicNotification(): Enables or disables notification/indication for a given characteristic.
  • setType(): Sets the type of this service.

Example Code

The following example code shows how to use the BluetoothGattService class:


          
           
// Create a GATT Service with a given UUID
BluetoothGattService service = new BluetoothGattService(UUID.fromString("00001820-0000-1000-8000-00805f9b34fb"),
                                           BluetoothGattService.SERVICE_TYPE_PRIMARY);

// Create a Characteristic
BluetoothGattCharacteristic characteristic = new BluetoothGattCharacteristic(UUID.fromString("6e400001-b5a3-f393-e0a9-e50e24dcca9e");

// Set Characteristic Properties
characteristic.setPermissions(BluetoothGattCharacteristic.PERMISSION_READ | BluetoothGattCharacteristic.PERMISSION_WRITE);

// Add the Characteristic to the Service
service.addCharacteristic(characteristic);

          

          
           
// Add the Service to the GATT Server
mBluetoothGattServer.addService(service);
