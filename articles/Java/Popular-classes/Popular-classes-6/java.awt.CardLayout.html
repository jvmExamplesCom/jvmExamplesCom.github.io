java.awt.CardLayout

The java.awt.CardLayout class is a concrete implementation of the LayoutManager2 interface, which is used to layout components in a container. It organizes components into a stack with components arranged in a row and columns, each component occupying one cell. A CardLayout object can manage several rows and columns of components, as well as providing support for navigating between components as needed. It is easy to use and allows for complex user interfaces to be created.

Example 1: Create a basic CardLayout

The following code example creates a new CardLayout object with two columns. Then, it adds three components to the object, with the first being set to the first column and the other two being set to the second column.


          
           
CardLayout layout = new CardLayout(2, 0);

JComponent c1 = new JLabel("Component 1");
JComponent c2 = new JLabel("Component 2");
JComponent c3 = new JLabel("Component 3");

layout.addLayoutComponent(c1, "1");
layout.addLayoutComponent(c2, "2");
layout.addLayoutComponent(c3, "3");

Example 2: Navigate through the cards

The following code example demonstrates how to navigate through the cards using the CardLayout.show() method. The code starts by calling the show() method on the first card, which results in the first card being displayed. Then, the next and previous methods are used to move forward and backward through the cards respectively.


          
           
//Show the first card
layout.show(container, "1");

//Display the next card
layout.next(container);

//Display the previous card
layout.previous(container);
