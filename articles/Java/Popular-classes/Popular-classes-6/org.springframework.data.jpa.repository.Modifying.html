Introduction to org.springframework.data.jpa.repository.Modifying

The org.springframework.data.jpa.repository.Modifying annotation is an annotation that is used to mark a method or class as an entity manager’s updating query and is part of the Java Persistence API (JPA). This annotation is used to indicate that the methods of the annotated type, or the method itself, will be used to modify existing data in the database. It is used in combination with the @Query annotation to execute native UPDATE or DELETE queries.

Example 1: Update Query

In this example, we use the @Modifying annotation to indicate that the query will be an update query and update the name field of an Employee entity.


          
           
@Modifying
@Query("UPDATE Employee e SET e.name = :name WHERE e.id = :id")
int updateName(@Param("id") Long id, @Param("name")String name);

In this example, we use the @Modifying annotation to indicate that the query will be a delete query and delete the data from the Employee table.


          
           
@Modifying
@Query("DELETE FROM Employee e WHERE e.id = :id")
int deleteEmployee(@Param("id") Long id);
