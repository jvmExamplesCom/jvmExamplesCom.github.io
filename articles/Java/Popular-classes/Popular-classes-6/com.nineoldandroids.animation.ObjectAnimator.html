Exploring Java com.nineoldandroids.animation ObjectAnimator

Java's com.nineoldandroids.animation ObjectAnimator is an android library used to animate objects. It allows developers to animate property values of any object. It works with any property that has a getter and setter method. Its main purpose is to manage the animation of property values over time.

Creating Instance of ObjectAnimator

For creating an instance of ObjectAnimator, the ofFloat() or ofInt() method needs to be used depending on the type of property that needs to be animated. 


          
           
ObjectAnimator animator = ObjectAnimator.ofFloat(targetObject, "propertyName", valueStart, valueEnd);
ObjectAnimator animator2 = ObjectAnimator.ofInt(targetObject, "propertyName", valueStart,valueEnd);

Here, the targetObject is the Object whose property needs to be changed, "propertyName" is the name of the property which will be animation, and valueStart is the initial value and valueEnd is the final value.

Setting Duration

animator.setDuration(1000);

Starting Animation

animator.start();

animator.setRepeatCount(-1);
