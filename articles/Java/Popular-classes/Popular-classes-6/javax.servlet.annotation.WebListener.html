javax.servlet.annotation.WebListener

The javax.servlet.annotation.WebListener annotation is used to mark a class as an event listener in the web application. It is part of the Java Servlet API. When the application is deployed, various event listeners are registered with the container to properly handle events in the servlet context.

Annotation Syntax

The following is the syntax for the WebListener annotation:


          
           
@Retention(RetentionPolicy.RUNTIME)
@Target(ElementType.TYPE)
public @interface WebListener {
	String value() default "";
}
where:
  • The 
    value
    attribute represents the listener name.

Example

We can use WebListener annotation to declare a listener class as follows:


          
           
@WebListener
public class MyListener implements ServletContextListener {
    public void contextInitialized(ServletContextEvent sce) {
        // Initialization logic here
    }

    public void contextDestroyed(ServletContextEvent sce) {
         // shutdown logic here
    }
}
In this example, we declared a class 
MyListener
as a listener class using 
@WebListener
annotation. The class implements 

          
           
ServletContextListener
interface and it provides implementation of 

          
           
contextInitialized()
and 
contextDestroyed()
methods. When the application is deployed, the 
MyListener
class will be registered with the container as a listener class.
