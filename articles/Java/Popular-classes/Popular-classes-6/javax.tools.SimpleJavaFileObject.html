javax.tools.SimpleJavaFileObject

The javax.tools.SimpleJavaFileObject is an abstract class that is part of the Java Platform Standard Edition, providing an abstract representation of a Java file. This class is used to represent a source file or other artifacts associated with compiling and running Java programs. SimpleJavaFileObject fileObject = new SimpleJavaFileObject(filePath);
