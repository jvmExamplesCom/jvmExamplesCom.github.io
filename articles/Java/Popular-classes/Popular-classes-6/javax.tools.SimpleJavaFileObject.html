javax.tools.SimpleJavaFileObject

The javax.tools.SimpleJavaFileObject is an abstract class that is part of the Java Platform Standard Edition, providing an abstract representation of a Java file. This class is used to represent a source file or other artifacts associated with compiling and running Java programs. The SimpleJavaFileObject provides a representation of a regular file, memory-mapped file, or zip entries. This class also provides access to the encoding information associated with a source file.

Constructors

The SimpleJavaFileObject class has two different constructors: one that accepts a URI, and one that accepts a URI and the Kind of the file. The first constructor takes just a single parameter: a URI representing the path of a file. Here’s an example of this constructor in use: 

          
           
URI filePath = new URI("path/to/file.java");
SimpleJavaFileObject fileObject = new SimpleJavaFileObject(filePath);
The second constructor takes two parameters: a URI representing the path of a file and a Kind enum representing the kind of file. Here’s an example of this constructor in use: 

          
           
URI filePath = new URI("path/to/file.java");
Kind fileKind = Kind.SOURCE;
SimpleJavaFileObject fileObject = new SimpleJavaFileObject(filePath, fileKind);

  • getAccessLevel(): Returns the access level of the file.
  • getCharContent(boolean ignoreEncodingErrors): Returns a CharSequence with the contents of the file, optionally ignoring any encoding errors.
  • getName(): Returns the name of the file.
  • openInputStream(): Opens an InputStream to the file.
  • openOutputStream(): Opens an OutputStream to the file.
