javax.tools.SimpleJavaFileObject

The javax.tools.SimpleJavaFileObject is an abstract class that is part of the Java Platform Standard Edition, providing an abstract representation of a Java file. This class is used to represent a source file or other artifacts associated with compiling and running Java programs. The SimpleJavaFileObject provides a representation of a regular file, memory-mapped file, or zip entries. This class also provides access to the encoding information associated with a source file.

The SimpleJavaFileObject class has two different constructors: one that accepts a URI, and one that accepts a URI and the Kind of the file. URI filePath = new URI("path/to/file.java");
SimpleJavaFileObject fileObject = new SimpleJavaFileObject(filePath);

Kind fileKind = Kind.SOURCE;
SimpleJavaFileObject fileObject = new SimpleJavaFileObject(filePath, fileKind);

  • openInputStream(): Opens an InputStream to the file.
