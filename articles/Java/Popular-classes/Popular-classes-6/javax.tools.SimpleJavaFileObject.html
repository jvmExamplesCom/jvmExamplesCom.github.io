javax.tools.SimpleJavaFileObject

The javax.tools.SimpleJavaFileObject is an abstract class that is part of the Java Platform Standard Edition, providing an abstract representation of a Java file. SimpleJavaFileObject fileObject = new SimpleJavaFileObject(filePath);
