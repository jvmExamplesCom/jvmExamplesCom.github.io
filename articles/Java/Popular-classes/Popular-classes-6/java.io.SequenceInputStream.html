java.io.SequenceInputStream

The java.io.SequenceInputStream class of the Java IO API allows a number of InputStreams to be read sequentially. This is useful when reading data from multiple sources, such as files and network sockets. This class allows InputStreams to be chained together, with the output of one InputStream becoming the input of the next InputStream.

Usage

To use SequenceInputStream, create an array or Vector of InputStreams that you want to read from. Then, create a SequenceInputStream object by passing this array or Vector to the constructor.


          
           
InputStream [] streams = {
    new FileInputStream("fileA.txt"),
    new FileInputStream("fileB.txt"),
    new FileInputStream("fileC.txt"),
};
SequenceInputStream sis = new SequenceInputStream(streams);
