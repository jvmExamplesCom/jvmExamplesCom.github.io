java.nio.file.StandardWatchEventKinds

Java.nio.file.StandardWatchEventKinds is a class that encapsulates the types of events that can be monitored for changes in a directory. This includes create, delete, modify, and access events. This class provides an efficient way to monitor changes within a specified directory.

Create Event

The create event will be triggered when a new file or directory is created within the observed directory. The following code snippet demonstrates how this event can be monitored: 


          
           
Path path = Paths.get("testdir");
WatchService watchService = FileSystems.getDefault().newWatchService();
path.register(watchService, StandardWatchEventKinds.ENTRY_CREATE);
WatchKey key = watchService.take();
for (WatchEvent<?> event : key.pollEvents()) {
    if (event.kind() == StandardWatchEventKinds.ENTRY_CREATE) {
        System.out.println("New file or directory created!");
    }
}
