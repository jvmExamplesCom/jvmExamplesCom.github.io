java.util.concurrent.atomic.AtomicReferenceArray

The java.util.concurrent.atomic.AtomicReferenceArray is a high-performance thread-safe modification of the Java array that provides atomic operations for read, write and CAS (compare and set) operations. This class allows to safely perform operations on elements of Java arrays without needing external synchronization.

Methods of AtomicReferenceArray

The java.util.concurrent.atomic.AtomicReferenceArray class provides the following methods −
  • get() − retrieves the current value at the given index position.
  • set() − sets the given value at the given index position.
  • lazySet() − sets the given value as the new value without any memory visibility guarantees.
  • getAndSet() − atomically sets the given value at the given index position and returns the previous value.
  • compareAndSet() − atomically sets the given value at the given index position if the expected value matches the current value.
  • weakCompareAndSet() − atomically sets the given value at the given index position if the expected value matches the current value without any memory visibility guarantees.
  • length() − returns the length of the array.
