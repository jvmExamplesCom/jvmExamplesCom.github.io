Determine if File or Directory Exists in Java

Determining whether a file or directory exists is an essential part of programming, allowing programs to take different paths depending on the existence or nonexistence of files and directories. In this article, we'll explore various ways to determine if a file or directory exists in Java.

Using java.io.File

The 

          
           
java.io.File
class provides several methods that can be used to determine file or directory existence. The 

          
           
exists()
method checks if the file or directory specified in the 

          
           
File
constructor exists. It returns 

          
           
true
if the path exists, and 

          
           
false
otherwise. 

          
           
File myDir = new File("/Users/person/directory");
if (myDir.exists()) {
    System.out.println("Directory exists!");
} else {
    System.out.println("Directory does not exist!");
}
The 

          
           
isFile()
and 

          
           
isDirectory()
methods can be used to check the type of a file or directory. These methods return 

          
           
true
if the 

          
           
File
instance is a file or a directory respectively. 

          
           
File myFile = new File("/Users/person/file.txt");
if (myFile.isFile()) {
    System.out.println("It's a file!");
} else {
    System.out.println("It's not a file!");
}

File myDir = new File("/Users/person/directory");
if (myDir.isDirectory()) {
    System.out.println("It's a directory!");
} else {
    System.out.println("It's not a directory!");
}

Using java.nio.file.Files

The 

          
           
java.nio.file.Files
class provides several static utility methods for working with files and directories. The 

          
           
exists()
method checks if the specified path exists. 

          
           
Path path = Paths.get("/Users/person/file.txt");
if (Files.exists(path)) {
    System.out.println("File exists!");
} else {
    System.out.println("File does not exist!");
}
The 

          
           
isDirectory()
and 

          
           
isRegularFile()
methods can be used to check the type of a file or directory. These methods return 

          
           
true
if the specified path is a directory or a regular file respectively. 

          
           
Path path = Paths.get("/Users/person/file.txt");
if (Files.isRegularFile(path)) {
    System.out.println("It's a file!");
} else {
    System.out.println("It's not a file!");
}

Path path = Paths.get("/Users/person/directory");
if (Files.isDirectory(path)) {
    System.out.println("It's a directory!");
} else {
    System.out.println("It's not a directory!");
}

java.io.File
and 

          
           
java.nio.file.Files
classes, and looked at several methods for checking the existence and type of files and directories.
