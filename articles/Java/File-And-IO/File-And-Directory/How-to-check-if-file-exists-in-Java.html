How to Check if File Exists in Java

Java provides numerous classes and methods that can be used to check if a file exists on a specified path. In this article, we will look through the different techniques that can be used in order to check if a file exists in Java.

Using java.io.File Class

The java.io.File class provides a static method 

          
           
exists()
which accepts a file or directory path and returns true if the file or directory exists at the specified path. Otherwise, it returns false. 

          
           
File file = new File("/path/to/file");
boolean result = file.exists();
System.out.println(result);

// Output: true

Using java.nio.file.Files Class

The 

          
           
java.nio.file.Files
class provides two static methods, 

          
           
exists()
and 

          
           
notExists()
, to check if a file exists on a given path. The 

          
           
exists()
method returns 

          
           
true
if the file exists, otherwise it returns 

          
           
false
. Similarly, the 

          
           
notExists()
method returns 

          
           
true
if the file does not exist, otherwise it returns 

          
           
false
. 

          
           
Path path = Paths.get("/path/to/file");
boolean result1 = Files.exists(path);
boolean result2 = Files.notExists(path);
System.out.println(result1);
System.out.println(result2);

// Output: true
//         false

Using java.nio.file.Path Interface

The 

          
           
java.nio.file.Path
interface provides an overloaded method 

          
           
exists()
to check whether a file exists at a given path or not. This method returns 

          
           
true
if the file exists, otherwise it returns 

          
           
false
. 

          
           
Path path = Paths.get("/path/to/file");
boolean result = path.toFile().exists();
System.out.println(result);

// Output: true

Using java.nio.file.PathMatcher Interface

The 

          
           
java.nio.file.PathMatcher
interface provides a static method 

          
           
matches()
which is used to match a given path against a given glob pattern. This method returns 

          
           
true
if the file exists at the given path, otherwise it returns 

          
           
false
. 

          
           
Path path = Paths.get("/path/to/file");
PathMatcher matcher = FileSystems.getDefault().getPathMatcher("glob:**/*.txt");
boolean result = matcher.matches(path);
System.out.println(result);

// Output: true
