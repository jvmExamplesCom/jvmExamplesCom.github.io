List Filesystem Roots in Java

Java provides several ways to list the filesystem roots on a machine. In this article, we will discuss how to list filesystem roots using the 

          
           
FileSystems
class and the methods it provides.

Using the 
FileSystems
Class

The easiest way to list filesystem roots in Java is by using the 

          
           
FileSystems
class. This class contains a static method called 

          
           
getDefault
that returns a 

          
           
FileSystem
object. The 

          
           
FileSystem
object has a 

          
           
getRootDirectories
method that returns a list of 

          
           
Path
objects, each one corresponding to a root of the filesystem. 

          
           
FileSystem fs = FileSystems.getDefault();
Iterable<Path> roots = fs.getRootDirectories();
for (Path root: roots) {
  System.out.println(root);
}
The above code will print out all the root directories of the filesystem.

Using the 
Path
Class

Another way to list filesystem roots is by using the 

          
           
Path
class. The 

          
           
Path
class contains a static method called 

          
           
toAbsolutePath
which takes a 

          
           
String
as an argument and returns a 

          
           
Path
. The 

          
           
String
should be a path to a file or directory. When the 

          
           
String
points to a directory, the absolute path of the directory will be returned. When the 

          
           
String
points to a file, the absolute path of the directory containing the file will be returned. This can then be used to list the filesystem roots. 

          
           
Path path = Paths.toAbsolutePath("/");
System.out.println(path);
The above code will print out the absolute path of the root directory.

Conclusion

In this article, we discussed how to list filesystem roots in Java using the 

          
           
FileSystems
class and the 

          
           
Path
class. Both methods are relatively straightforward and easy to use.
