Setting Last Modified Time of a File or Directory in Java

Java provides a couple of different ways for changing the last modified time of file or directory. Depending on the desired use case, there are different approaches to setting the last modified time of a file or directory.

Using the File class

The 

          
           
java.io.File
class provides several methods that can be used to interact with the file system. This includes the 

          
           
setLastModified
method, which allows for setting the last modified time for a given file or directory. The following code example shows how to set the last modified time of a given file using the 

          
           
setLastModified
method: 

          
           
File file = new File("myfile.txt");
file.setLastModified(System.currentTimeMillis());

Using the Files class

java.nio.file.Files
class provides an API for performing various operations on files and directories, including the ability to set the last modified time for a given file or directory. The following code example shows how to set the last modified time of a given file using the 

          
           
setLastModifiedTime
method:
