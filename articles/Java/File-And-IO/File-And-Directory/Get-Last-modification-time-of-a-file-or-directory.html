Get Last Modification Time of a File or Directory In Java

In Java, it is possible to get last modified time of a file or directory. Last modified time is the timestamp when the file or directory was last modified or changed. This article explains how to retrieve last modified time with code examples.

Using getLastModified() Method

The 
File
class in Java provides the 
getLastModified()
method. This method returns a 
long
value representing the time the file or directory was modified, measured in milliseconds since the epoch (January 1 1970 00:00:00 GMT). 

          
           
// Get file object
File file = new File("myfile.txt");

// Get last modification time in milliseconds
long lastModifiedMillis = file.lastModified();

// Get date from milliseconds
Date date = new Date(lastModifiedMillis);

// Print last modification time
System.out.println("Last modified: " + date);
// Output: Last modified: Sun May 10 03:17:07 UTC 2020

Timestamp to Date Format

It is often necessary to convert the 
long
value returned by the 
getLastModified()
method to a 
Date
format. For example, if you want to display the last modified time as a human readable string. The 
Date
class can be used for this conversion. 

          
           
long lastModifiedMillis = file.lastModified();
Date date = new Date(lastModifiedMillis);

Convert Timestamp to Custom Format

The above approach converts the timestamp to the default format (example: Sun May 10 03:17:07 UTC 2020). It is also possible to convert to a custom format with the help of SimpleDateFormat class.


          
           
long lastModifiedMillis = file.lastModified();

// Create SimpleDateFormat object with desired date format
SimpleDateFormat dateFormat = new SimpleDateFormat("dd/MM/yyyy HH:mm:ss");

// Get Date from milliseconds
Date date = new Date(lastModifiedMillis);

// Format the date using the format specified
String formatedDate = dateFormat.format(date);

// Print last modification time
System.out.println("Last modified: " + formatedDate);
// Output: Last modified: 10/05/2020 03:17:07

Get Last Modified Time Of A Directory

The 
getLastModified()
method can also be used to get the last modified time of a directory. The method works exactly the same way and returns the last modified time in milliseconds. 

          
           
// Get directory object
File dir = new File("mydir");

// Get last modification time in milliseconds
long lastModifiedMillis = dir.lastModified();

// Get date from milliseconds
Date date = new Date(lastModifiedMillis);

// Print last modification time
System.out.println("Last modified: " + date);
// Output: Last modified: Sun May 10 03:17:07 UTC 2020
