Creating a Temporary File in Java

A temporary file is a file that only exists temporarily in a computer’s memory. It is usually deleted once the program using it is finished. In this article we’ll look at how to create a temporary file in Java.

Code Snippet 


          
           
Path tempFile = Files.createTempFile("testFile", ".tmp");
System.out.println("Path of the temporary file: " + tempFile);

// The temp file will be deleted when the program exits
tempFile.toFile().deleteOnExit();
In the example above, we use the 

          
           
createTempFile()
method to create a 

          
           
Path
object representing the newly created temporary file. The first argument of the 

          
           
createTempFile()
method is the prefix of the name of the temporary file, while the second argument is the suffix of the name of the temporary file. We then print out the 

          
           
Path
object representing the newly created temporary file. Finally, we call the 

          
           
deleteOnExit()
method on the 

          
           
File
object returned by the 

          
           
toFile()
method to make sure that the temporary file is deleted when the program exits.

The output demonstrates that a temporary file has been successfully created.
