Working with File objects in Java

Using the java.io.File class you can work with file system paths and files. One common operation is to get the parent directory of a given path as a 

          
           
File
object.

One Parent Directory

If the path only contains one directory level, then the parent of that path is the root path. This is a special case, since the parent path is just an empty string. To account for this situation in our code we must check for it first. 

          
           
String path = "/my/path/";
File file = new File(path);
if (path.endsWith("/")) {
    // Use an empty string as parent path
    file = new File("");
} else {
    // Otherwise get the parent path from file object
    file = file.getParentFile();
}
You can also use the 

          
           
File.getParent()
method to achieve the same result: 

          
           
String path = "/my/path/";
File file = new File(path);
String parentPath = file.getParent();
if (parentPath == null) {
    // Use an empty string as parent path
    parentPath = "";
}
file = new File(parentPath);

Multiple Parent Directories

If the path contains multiple levels of directories then you can use the 

          
           
File.getParentFile()
method to traverse up the tree one directory at a time: 

          
           
String path = "/my/long/path/";
File file = new File(path);
File parent = file.getParentFile();
while (parent != null) {
    file = parent;
    parent = parent.getParentFile();
}
You can also use the 

          
           
File.getParent()
method to achieve the same result: 

          
           
String path = "/my/long/path/";
File file = new File(path);
String parentPath = file.getParent();
while (parentPath != null) {
    file = new File(parentPath);
    parentPath = file.getParent();
}
In both cases, the 

          
           
file
variable at the end will hold the root path, which is the parent of the given path.
