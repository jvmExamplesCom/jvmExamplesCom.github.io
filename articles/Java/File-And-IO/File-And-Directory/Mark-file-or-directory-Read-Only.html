Mark File or Directory Read Only in Java

Java provides an easy way to mark a file or directory as read-only. This can be done with the java.nio.file.attribute.PosixFilePermission class. To mark a file or directory for read-only permissions, you will need to use the following code: 

          
           
Path path = Paths.get("/path/to/file");
Set<posixfilepermission> perms = PosixFilePermissions.fromString("r--r--r--");
Files.setPosixFilePermissions(path, perms);
</posixfilepermission>
In this example, the 

          
           
r--r--r--
are the read-only permissions; it is the same as running the command 

          
           
chmod 444
. The above code sets the file owner's permissions to 

          
           
r--
, the group's permissions to 

          
           
r--
, and the others permissions to 

          
           
r--
. Once this code has been executed, the file or directory is marked as read-only.

Test the Permissions

To verify that the permissions have been applied correctly, you can use the following code: 

          
           
Path path = Paths.get("/path/to/file");
Set<PosixFilePermission> perms = Files.getPosixFilePermissions(path);
This code retrieves the permissions for the file or directory and prints them out.

Example Output

The output of the above code would be something like this: 

          
           
r--r--r--
Which indicates that the file or directory has read-only permissions.
