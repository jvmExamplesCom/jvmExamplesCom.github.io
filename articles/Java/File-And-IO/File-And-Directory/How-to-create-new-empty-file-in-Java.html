How to Create New Empty File in Java

Creating a new empty file in Java is fairly simple and can be done using the 

          
           
File
class in the Java I/O API. To illustrate how it's done, let's look at an example of creating a file for a text file called 

          
           
myfile.txt
.

The Code


          
           
// Create a file object with name myfile.txt
File myFile = new File("myfile.txt");

// Create the file if it doesn't exist
if (!myFile.exists()) {
  myFile.createNewFile();
}
In this code, we first create a 

          
           
File
object and pass the desired filename as parameter. Next, we check whether the file exists using the 

          
           
exists()
method of the 

          
           
File
class. If it doesn't exist, then we create a new file using the 

          
           
createNewFile()
method of the 

          
           
File
class. There are some cases when the file exists but is empty, so to make sure that the contents of the file are cleared, you can use the following code:

The Code (with File Clearing)


          
           
// Create a file object with name myfile.txt
File myFile = new File("myfile.txt");

// Create the file if it doesn't exist
if (!myFile.exists()) {
  myFile.createNewFile();
} else {
  // Clear any existing content
  PrintWriter writer = new PrintWriter(myFile);
  writer.print("");
  writer.close();
}
In this code, if the file exists, we open it as a 

          
           
PrintWriter
, write an empty string, and close it. And that's it! You now have a way of creating a new, empty file with Java.
