Get Absolute Path of a File in Java

Java offers various ways how to get the absolute path of a file. This article provides an overview of all the available methods and explains how to get the absolute path of a file in Java.

Using java.io.File

The 

          
           
java.io.File
class provides two methods to get the absolute path of a file: 
  • 
            
             
public String getAbsolutePath()

            
  • public URI toURI()
Example 1: 

          
           
File file = new File("test.txt");
String absolutePath = file.getAbsolutePath();
System.out.println(absolutePath); // Outputs: /home/user/Desktop/test.txt
Example 2: 

          
           
File file = new File("test.txt");
URI uri = file.toURI();
System.out.println(uri.getPath()); // Outputs: /home/user/Desktop/test.txt

Using java.nio.file.Path

Since Java 7, the 

          
           
java.nio.file.Path
class has been introduced. This class provides two methods to get the absolute path of a file: 
  • 
            
             
public Path toAbsolutePath()

            
  • 
            
             
public Path toRealPath(LinkOption... options)
Example 3: 

          
           
Path path = Paths.get("test.txt");
Path absolutePath = path.toAbsolutePath();
System.out.println(absolutePath); // Outputs: /home/user/Desktop/test.txt
Example 4: 

          
           
Path path = Paths.get("test.txt");
Path realPath = path.toRealPath();
System.out.println(realPath); // Outputs: /home/user/Desktop/test.txt

Using java.net.URL

The 

          
           
java.net.URL
class is another way to get the absolute path of a file. It provides the method: 
  • 
            
             
public String toExternalForm()
Example 5: 

          
           
URL url = new URL("file:///home/user/Desktop/test.txt");
String externalForm = url.toExternalForm();
System.out.println(externalForm); // Outputs: file:///home/user/Desktop/test.txt
This concludes our overview on how to get the absolute path of a file in Java.
