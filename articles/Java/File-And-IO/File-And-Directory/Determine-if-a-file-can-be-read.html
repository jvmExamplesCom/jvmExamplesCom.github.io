Determine If a File Can be Read Using Java

Java provides multiple ways to check if a file can be read. The following sections provide code examples of the different methods that can be used.

Using Accessible() Method

The accessible() method checks whether the file is accessible in the file system or not. This method returns a boolean value true or false. 

          
           
File f1 = new File("test.txt"); 

if(f1.canRead()) 
{
   System.out.println("The file is readable");
} else {
   System.out.println("The file isn't readable");
}

Using IsReadable() Method

The isReadable() method checks whether the file can be read or not. This method also returns a boolean value true or false. 

          
           
File f2 = new File("test.txt"); 

if(f2.isReadable()) 
{
   System.out.println("The file is readable");
} else {
   System.out.println("The file isn't readable");
}

Using Exists() Method

The exists() method checks whether the file exists in the file system or not. It also returns a boolean value true or false. 

          
           
File f3 = new File("test.txt"); 

if(f3.exists()) 
{
   System.out.println("The file exists");
} else {
   System.out.println("The file doesn't exist");
}

Using CanExecute() Method

The canExecute() method checks whether the file is executable or not. It also returns a boolean value true or false. 

          
           
File f4 = new File("test.txt"); 

if(f4.canExecute()) 
{
   System.out.println("The file is executable");
} else {
   System.out.println("The file isn't executable");
}

Using IsFile() Method

The isFile() method checks whether the file is a normal file or not. It also returns a boolean value true or false. 

          
           
File f5 = new File("test.txt"); 

if(f5.isFile()) 
{
   System.out.println("The file is a normal file");
} else {
   System.out.println("The file isn't a normal file");
}

Using CanWrite() Method

The canWrite() method checks whether the file can be written or not. It also returns a boolean value true or false. 

          
           
File f6 = new File("test.txt"); 

if(f6.canWrite()) 
{
   System.out.println("The file is writable");
} else {
   System.out.println("The file isn't writable");
}
