How to Get File Extension in Java

Java provides a few methods for retrieving the file extension of a file. It is important to understand that when a file name is given, the extension is actually part of the file name itself. The extension can have multiple parts and can be considered as a suffix of the file name. This article will provide an overview of different methods for obtaining the file extension of a file in Java and present working code examples for each method.

Using the Path Class

The recommended way of obtaining the file extension in Java is by using the 

Path
class in the 
java.nio.file
package. This class contains a method named 
getFileName()
which returns a 
Path
object representing just the file name. To obtain the file extension, we can call the 
getFileName()
method and then call the 
toString()
method on the returned 
Path
object. We can then split this String representation on the '.' character and take the last element of the resulting array as the file extension: 

          
           
Path path = Paths.get("/path/to/my/file.ext");
String fileName = path.getFileName().toString();
String fileExtension = fileName.split("\\.")[1];

System.out.println(fileExtension); // prints "ext"

Using String Manipulation

It is also possible to obtain the file extension by manually manipulating the String representation of the file name:


          
           
String fileName = "/path/to/my/file.ext";
String fileExtension = fileName.substring(fileName.lastIndexOf(".") + 1);

System.out.println(fileExtension); // prints "ext"

This approach is much less efficient than using the 

Path
class but works just as well. It should, however, be avoided if possible since it requires more code and is more prone to errors.

Conclusion

In this article, we have looked at how to obtain the file extension of a file in Java. We have seen two approaches: one using the 

Path
class and another using simple String manipulation. In most cases, the 
Path
class method should be preferred since it is more efficient and easier to read.
