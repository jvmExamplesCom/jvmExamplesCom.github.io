Rename File or Directory

Renaming a file or directory in Java is very simple. The 

          
           
java.io.File
class contains methods for renaming files and directories. The two main methods for renaming files are 

          
           
renameTo()
and 

          
           
setName()
.

renameTo() Method

The 

          
           
renameTo()
method takes a single parameter, which is the new filename. It returns a boolean value indicating whether the renaming operation was successful or not. Here's a simple example of using the 

          
           
renameTo()
method: 

          
           
File oldFile = new File("oldfile.txt");
File newFile = new File("newfile.txt");
boolean success = oldFile.renameTo(newFile);
if (success) {
    System.out.println("File renamed");
} else {
    System.out.println("File not renamed");
}

setName() Method

The 

          
           
setName()
method is similar to the 

          
           
renameTo()
method, but it does not take any parameters. Instead, it changes the name of the file to be the same as the name of the object that called the method. Here's an example of using the 

          
           
setName()
method: 

          
           
File oldFile = new File("oldfile.txt");
oldFile.setName("newfile.txt");
