Java: Getting the Parent Directory

Introduction

Java is a popular programming language used to create all kinds of applications and software. Consequently, file directories are an important part of writing code with Java. This article will discuss how to get the name of the parent directory in Java.

Using the File Class

In order to get the parent directory of a given file, we can use the 

          
           
File
class which is a part of the java.io package. The 

          
           
getParent()
method can be used to return the absolute pathname of the directory containing the associated file. For example: 

          
           
String filePath = "C:/User/Me/Example/file.txt";  
File file = new File(filePath);  
String parentDir = file.getParent();  
System.out.println(parentDir);
In this example, the output would be 

          
           
C:/User/Me/Example
.

Using Paths

Another way to get the parent directory of a file is to use the 

          
           
Paths
class which is a part of the 

          
           
java.nio.file
package. We can use the 

          
           
getParent()
method of the 

          
           
Paths
class to achieve the same result as using the 

          
           
File
class. For example: 

          
           
String filePath = "C:/User/Me/Example/file.txt";  
Path parentPath = Paths.get(filePath).getParent();  
System.out.println(parentPath);
Again, the output of this example would be 

          
           
C:/User/Me/Example
.

Conclusion

In this article, we discussed how to get the parent directory of a file in Java. We explored two approaches of getting the parent directory- using the 

          
           
File
class from the 

          
           
java.io
package, and using the 

          
           
Paths
class from the 

          
           
java.nio.file
package. Each approach can be used to achieve the same result.
