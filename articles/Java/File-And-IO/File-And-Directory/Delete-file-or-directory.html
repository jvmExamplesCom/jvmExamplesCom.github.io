Delete file or directory in Java

Java is a powerful language and allows for easy manipulation of files and directories, including the ability to delete them when needed. In this article, we will discuss how to delete a file or directory in the Java language.

Delete File in Java

In order to delete a file in Java, we can use the 

          
           
Files.delete()
method provided by the 

          
           
java.nio.file.Files
class. The syntax of this method is as follows: 

          
           
Files.delete(Path path)
where, 
  • path
    is the path of the file to be deleted.
This method returns a 

          
           
boolean
value which is 

          
           
true
if the file is deleted successfully else it returns 

          
           
false
. Let's take a look at a simple example which demonstrates how to delete a file in Java: 

          
           
// Create a Path object
Path path = Paths.get("/tmp/test.txt");

// Try to delete the file
try {
    boolean isFileDeleted = Files.delete(path);
    System.out.println("isFileDeleted ? "+ isFileDeleted);
} catch (NoSuchFileException e) {
    System.err.println("File doesn't exist!");
} catch (IOException e) {
    System.err.println("Error occurred while deleting the file!");
    e.printStackTrace();
}
In the above example, we first create a 

          
           
Path
object with the path of the file that we want to delete. Then, we try to delete the file using the 

          
           
Files.delete()
method. If the file is successfully deleted, then it will return 

          
           
true
, else it will throw an exception. We have handled two exceptions in the catch block, 

          
           
NoSuchFileException
and 

          
           
IOException
.

Delete Directory in Java

Similar to deleting a file, we can also delete a directory in Java using the same 

          
           
Files.delete()
method. However, there are some points to note while deleting a directory:
  • The directory to be deleted must be empty. If it's not empty, the method will return 
    false
    .
  • If the directory doesn't exist, it will throw a 
    NoSuchFileException
    .
Let's take a look at an example which demonstrates how to delete a directory in Java: 

          
           
// Create a Path object
Path path = Paths.get("/tmp/test_dir");

// Try to delete the directory
try {
    boolean isDirDeleted = Files.delete(path);
    System.out.println("isDirDeleted ? "+ isDirDeleted);
} catch (NoSuchFileException e) {
    System.err.println("Directory doesn't exist!");
} catch (DirectoryNotEmptyException e) {
    System.err.println("Directory is not empty!");
} catch (IOException e) {
    System.err.println("Error occurred while deleting the directory!");
    e.printStackTrace();
}
In the above example, we first create a 

          
           
Path
object with the path of the directory that we want to delete. Then we try to delete the directory using the 

          
           
Files.delete()
method. If the directory is successfully deleted, then it will return 

          
           
true
, else it will throw an exception. We have handled three exceptions in the catch block, 

          
           
NoSuchFileException
, 

          
           
DirectoryNotEmptyException
and 

          
           
IOException
.

Conclusion

In this article, we discussed how to delete a file or directory in Java. We saw how to delete a file using the 

          
           
Files.delete()
method and also how to delete a directory. It is important to note that if the directory is not empty, then it cannot be deleted using the 

          
           
Files.delete()
method and an appropriate exception is thrown.
