Introduction to Java

Java is a programming language and computing platform first released by Sun Microsystems in 1995. It is one of the most popular programming languages in use, particularly for client-server web applications, with a reported 10 million users. Java is an object-oriented language and its syntax is similar to C and C++. It can be used to create applications that run on a single computer or distributed across multiple computers.

Creating Directories with Java

In Java, you can create directories along with any required parent directories using the 

          
           
mkdirs()
method. This method is used to create a directory along with any nonexistent parent directories. The syntax for this method is as follows: 

          
           
boolean success =  new File(String pathname).mkdirs();
The pathname argument is the name of the directory to be created, including any nonexistent parent directories. When the method is called, it will return a boolean value depending on whether the directory and its parent directories were successfully created.

Example 1: Creating a Directory

In the following example we create a directory called "files" in the current directory. 

          
           
File f = new File("files");
boolean success = f.mkdirs();

if (success) {
    System.out.println("Directory created successfully");
} else {
    System.out.println("Directory creation failed");
}
In this example, if the directory files does not already exist, its parent directories will also be created.

Example 2: Creating Multiple Directories

In the following example we create multiple directories in one operation. 

          
           
File f = new File("files1/files2/files3/files4");
boolean success = f.mkdirs();

if (success) {
    System.out.println("Directories created successfully");
} else {
    System.out.println("Directories creation failed");
}
In this example, all of the directories in the pathname are created if they don't already exist.

Conclusion

mkdirs()
method in Java. We also saw two examples of how this method can be used. With this knowledge, you should be able to create directories in your Java applications.
