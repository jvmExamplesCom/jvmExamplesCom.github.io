How to get size of file in Java

Java provides multiple ways for getting the size of a File. Depending on your needs, you can use any of the available methods and techniques to determine the size of a file. In this article, we'll discuss these methods and show examples of their usage.

Using java.io.File.length()

One of the simplest ways to get the size of a file is to use the 

          
           
length()
method of the 

          
           
java.io.File
class. This method returns, as a 

          
           
long
, the size of the file in bytes. Consider the following example: 

          
           
import java.io.File;
 
public class Main {
 
    public static void main(String[] args) {
        File file = new File("sample.txt");
        long fileSize = file.length();
        System.out.println("File size in bytes: " + fileSize);
    }
}
In this example, we create a 

          
           
File
object pointing to a file named 

          
           
sample.txt
. We then call its 

          
           
length()
method and print the returned value to the console.

Using java.nio.file.Files.size()

The 

          
           
java.nio.file.Files
class also offers a method for determining the size of a file. It's called 

          
           
size()
and it works similarly to the 

          
           
length()
method discussed in the previous section - it returns the size of a file, but this time as a 

          
           
long
. Here's an example of its usage: 

          
           
Path path = Paths.get("sample.txt");
long fileSize = Files.size(path);
System.out.println("File size in bytes: " + fileSize);
In this example, we use the 

          
           
Paths.get()
method to create a 

          
           
Path
object pointing to our 

          
           
sample.txt
file. We then pass that object to the 

          
           
Files.size()
method and print the returned result to the console.

Using java.io.InputStream.available()

You can also use the 

          
           
available()
method of the 

          
           
InputStream
class to get the size of the file. The 

          
           
available()
method returns the number of bytes that can be read from the stream without blocking. Here's an example of its usage: 

          
           
FileInputStream inputStream = new FileInputStream("sample.txt");
int fileSize = inputStream.available();
System.out.println("File size in bytes: " + fileSize);
In this example, we create a 

          
           
FileInputStream
object pointing to our 

          
           
sample.txt
file. We then call the 

          
           
available()
method on the stream and print the returned result to the console.

Conclusion

In this article, we discussed how to get the size of a file in Java. We saw three different approaches for achieving this: using 

          
           
length()
or 

          
           
size()
methods of the 

          
           
java.io.File
or 

          
           
java.nio.file.Files
classes respectively, or using the 

          
           
available()
method of the 

          
           
InputStream
class. We also showed example code snippets for each of the approaches.
