How to Convert InputStream to String in Java

Converting an InputStream to a String is a common task when working with Java file streams. In this article, we will look at how to convert an InputStream to a String using various approaches.

Using BufferedReader

The most common approach for converting an InputStream to a String is to use the BufferedReader class. The code below shows an example of how to use BufferedReader to convert an InputStream to a String. 

          
           
InputStream inputStream = ...;
BufferedReader bufferedReader = new BufferedReader(
        new InputStreamReader(inputStream));
StringBuilder result = new StringBuilder();
String line;
while ( (line = bufferedReader.readLine()) != null) {
    result.append(line);
}
String stringResult = result.toString();
