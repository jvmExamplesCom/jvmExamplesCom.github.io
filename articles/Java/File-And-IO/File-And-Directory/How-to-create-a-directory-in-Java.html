How to Create a Directory in Java

Creating a directory in Java is a fairly straightforward process, and can be done simply by using the 

          
           
java.io.File
class. This class has a static method called 

          
           
mkdir()
, which can be used to create a new directory. It accepts a single argument, which should be the path of the directory to be created.

Examples

The following examples demonstrate how to use the 
java.io.File
class to create a directory in Java. 

          
           
//Create a directory named "example"
File dir = new File("example");
dir.mkdir();

          

          
           
//Create a directory named "example" in the current directory
File dir = new File("./example");
dir.mkdir();

          

          
           
//Create a directory named "example" in the specified parent directory
File dir = new File("/path/to/parent/example");
dir.mkdir();

Conclusion

Creating a directory in Java is a simple and straightforward process, and can be done by using the 

java.io.File
class and its 
mkdir()
method. It's important to remember to specify the full path of the directory that is being created.
