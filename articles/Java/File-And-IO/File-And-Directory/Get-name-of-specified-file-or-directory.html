Working with Files and Directories in Java

Working with files and directories in Java can be done in many different ways. In this article, we'll discuss how to get the name of a specified file or directory using different methods. We will also look at some example code and output it as HTML with proper escaping and tags.

Getting the Name of the Specified File or Directory

The easiest way to get the name of a specified file or directory is to use the 

          
           
getName()
method from the 

          
           
java.io.File class
. This method returns the last name of the file or directory pathname. Here's an example of how to use this method: 

          
           
File f = new File("C:\\test\\example.txt");
String name = f.getName();  // returns "example.txt"
The 

          
           
getName()
method can also be used to get the name of a directory. For example: 

          
           
File f = new File("C:\\test\\");
String name = f.getName();  // returns "test"
Another way to get the name of a file or directory is to use the 

          
           
getPath()
method from the 

          
           
java.io.File class
. This method returns the full pathname of the file or directory. Here's an example of how to use this method: 

          
           
File f = new File("C:\\test\\example.txt");
String name = f.getPath();  // returns "C:\test\example.txt"
The 

          
           
getPath()
method can also be used to get the path of a directory. For example: 

          
           
File f = new File("C:\\test\\");
String name = f.getPath();  // returns "C:\test\"

getName()

getPath()
