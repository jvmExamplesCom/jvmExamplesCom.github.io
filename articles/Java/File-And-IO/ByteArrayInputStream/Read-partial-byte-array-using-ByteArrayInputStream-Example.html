Read Partial Byte Array using ByteArrayInputStream Example

Java's ByteArrayInputStream class allows developers to read in a byte array as an input stream. This is useful for reading files, images, and binary data from a byte array rather than from a file system. In this article, we will show how to read partial byte arrays using the ByteArrayInputStream class.

Creating a ByteArrayInputStream Object

The first step to use a ByteArrayInputStream is to create an object. This involves passing in a byte array as a parameter to the constructor. For example: 

          
           
byte[] bytes = {1,2,3,4,5};
ByteArrayInputStream bais = new ByteArrayInputStream(bytes);
The above code creates a ByteArrayInputStream object called bais that can be used to read the byte array {1,2,3,4,5}.

Reading from a ByteArrayInputStream Object

Once a ByteArrayInputStream object is created, one of several methods from the InputStream class can be called on it to read from the byte array. One of the most commonly used methods is the read() method which reads a single byte from the stream and returns an int value. This method can be called in a loop until the end of the stream is reached. For example: 

          
           
int bytesRead;
while((bytesRead = bais.read()) != -1){
    System.out.print((byte)bytesRead);
}
The above code will print out all of the values in the byte array that was passed into the constructor.

Reading Partial Byte Array Using ByteArrayInputStream

The ByteArrayInputStream class also has a variant of the read() method that allows for reading only a portion of a byte array. This is useful for when you do not want to read in the entire byte array, but instead want to read in only a certain number of bytes at a time. To use this variant of the read() method, you must pass in two parameters: a byte array and the offset at which to start reading. The offset specifies the index in the byte array that the read should start at and is zero-based. For example: 

          
           
byte[] bytes = {1,2,3,4,5};
int startOffset = 1;

byte[] readBytes = new byte[3];
int bytesReadCount = bais.read(readBytes, startOffset, 3);
The above code will read a total of 3 bytes, starting at index 1 in the original byte array, and store them in the new byte array readBytes. The return value of the read() method is the actual number of bytes read, which could be less than the number of bytes requested if the end of the stream is reached.
