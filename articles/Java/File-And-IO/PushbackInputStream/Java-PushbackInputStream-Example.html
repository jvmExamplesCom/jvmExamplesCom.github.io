Java PushbackInputStream Example

PushbackInputStream class is a subclass of FilterInputStream class. It provides functionality to unread one byte of data that has been read from the stream. The PushbackInputStream class contains an internal buffer to store data that has been read from the stream but not yet consumed by the application program.

Creating a PushbackInputStream Object

We can create a PushbackInputStream object by passing an InputStream object to its constructor. 

          
           
PushbackInputStream pushbackstream = 
    new PushbackInputStream(new FileInputStream("C:\\demo.txt"));

Unreading Data from the Stream

The basic function of unreading data from the PushbackInputStream is provided by the unread() method. The following example shows how to use this method. 

          
           
// Reading one byte
int i = pushbackstream.read();

// Unreading the byte using the unread() method
pushbackstream.unread(i);

Closing the Stream

We can close any instance of PushbackInputStream by calling the close() method. The following example shows how to use this method. 

          
           
pushbackstream.close();

Example of PushbackInputStream

In the following example, we are reading characters from a file and writing them to the System console. 

          
           
try {
    FileInputStream fis = new FileInputStream("C:\\demo.txt");
    PushbackInputStream pbis = new PushbackInputStream(fis);

    int ch;
    while ((ch = pbis.read()) != -1) {
        System.out.println("Read Char: " + (char)ch);
    }
    
    pbis.close();
} catch (IOException e) {
    e.printStackTrace();
}
In the above example, all characters read from the file are printed to the console.

Example of PushbackInputStream with unreading feature

In the following example, we are using the unreading feature of PushbackInputStream. 

          
           
try {
    FileInputStream fis = new FileInputStream("C:\\demo.txt");
    PushbackInputStream pbis = new PushbackInputStream(fis);

    int ch;
    while ((ch = pbis.read()) != -1) {
        System.out.println("Read Char: " + (char)ch);
        pbis.unread(ch);
    }

    pbis.close();
} catch (IOException e) {
    e.printStackTrace();
}
In the above example, all characters read from the file are unread and printed to the console again.
