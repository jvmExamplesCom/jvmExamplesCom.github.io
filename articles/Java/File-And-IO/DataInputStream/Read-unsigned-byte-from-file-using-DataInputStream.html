Reading Unsigned Bytes from a File using DataInputStream

Java has the 

          
           
DataInputStream
class, which allows you to read in primitive data types (such as bytes) from an InputStream. This can be useful when reading in binary data from a file, as the 

          
           
DataInputStream
class provides methods to easily read in data as primitive data types without having to manually convert it to the desired type. In this article, we will discuss how to use the 

          
           
DataInputStream

Creating a DataInputStream

In order to read from a file using a 

          
           
DataInputStream
, we must first create a 

          
           
DataInputStream
object. To do this, we can use the following code: 

          
           
FileInputStream fis = new FileInputStream("myfile.bin");
DataInputStream dis = new DataInputStream(fis);
This creates a 

          
           
DataInputStream
object that is connected to the

DataInputStream
to read in bytes from the file.

DataInputStream
class provides two methods for reading in bytes from the stream: 

          
           
readByte()
and 

          
           
readUnsignedByte()
. The first method, 

          
           
readByte()

readUnsignedByte()

int b = dis.readUnsignedByte();
