Read Boolean from File using DataInputStream

Java offers different ways for reading a file. One of them is using the DataInputStream class, which allows you to read boolean values from the file. This tutorial explains how to read boolean values from a file using DataInputStream.

Example 1: Reading a boolean value with DataInputStream

In this example, we’ll read a file that contains one boolean value. The execution of the code will display the value in the console.


          
           
DataInputStream dis = new DataInputStream (new FileInputStream("file.dat"));
boolean b = dis.readBoolean();
System.out.println(b);

Example 2: Reading multiple Boolean values with DataInputStream

In this example, we’ll read a file containing three boolean values. The execution of the code will display the values in the console.


          
           
DataInputStream dis = new DataInputStream (new FileInputStream("file.dat"));
boolean b1 = dis.readBoolean();
boolean b2 = dis.readBoolean();
boolean b3 = dis.readBoolean();
System.out.println("Boolean 1: "+b1);
System.out.println("Boolean 2: "+b2);
System.out.println("Boolean 3: "+b3);

Example 3: Reading a boolean value from a specific position within a file

In this example, we’ll read the second boolean value from a file containing three boolean values. The execution of the code will display the value in the console.


          
           
DataInputStream dis = new DataInputStream (new FileInputStream("file.dat"));
//skip the first Boolean value
dis.readBoolean(); 
boolean b2 = dis.readBoolean();
System.out.println("Second Boolean value: "+b2);

That’s it! This tutorial demonstrated different ways to read boolean values from a file using DataInputStream. You can use these techniques to read boolean values from other sources as well, such as network connections.
