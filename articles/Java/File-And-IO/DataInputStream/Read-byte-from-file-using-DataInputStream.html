Read byte from file using DataInputStream

Introduction

DataInputStream is a class in Java that is used to read a data stream from an input source. It is a convenient way to read bytes from a file since it allows us to read multiple data types, such as integers, strings and booleans. In this article, we will learn about how to read bytes from a file using the DataInputStream class in Java.

Reading a file

The first step in reading bytes from a file is to create a FileInputStream object. This object is used to read bytes from a file. We can then create a DataInputStream object by passing the FileInputStream object as a parameter to its constructor. This is done as follows: 

          
           
FileInputStream fis = new FileInputStream("myfile.txt");
DataInputStream dis = new DataInputStream(fis);
Once we have created our DataInputStream object, we can use it to read bytes from the file. The method that we use to read bytes depends on the type of data that we want to read.

Reading a single byte

If we want to read a single byte from the file, we can use the readByte() method. This method returns the byte as an integer. We can then cast it to a byte if necessary. Here is an example: 

          
           
int b = dis.readByte();
byte c = (byte) b;

Reading multiple bytes

We can also use the DataInputStream class to read multiple bytes from the file at once. To do this, we use the readBytes() method. This method takes two parameters: an array of bytes, and the number of bytes to read. Here is an example: 

          
           
byte[] myBytes = new byte[100];
int numBytesRead = dis.readBytes(myBytes, 100);
The code above reads up to 100 bytes from the file, and stores them in the array myBytes. The variable numBytesRead contains the number of bytes actually read.

Closing the streams

Once we are finished reading bytes from the file, we should close the streams. This can be done by calling the close() method on each of the stream objects. Here is an example: 

          
           
dis.close();
fis.close();

Conclusion

In this article, we have learned about how to read bytes from a file using the DataInputStream class in Java. We have seen how to read a single byte, and how to read multiple bytes at once. We have also seen how to close the streams when we are finished.
