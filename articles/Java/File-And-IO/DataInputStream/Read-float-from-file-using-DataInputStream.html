Reading Float values From Files Using DataInputStream

DataInputStream is a class in Java that allows reading of primitive data types such as float values from an InputStream or a File. In this article, we will see how to use DataInputStream to read float values from a file using an example program.

Example Program

Letâ€™s write a simple example program to read a float value from a file using DataInputStream.


          
           
import java.io.DataInputStream;
import java.io.FileInputStream;
import java.io.IOException;

public class DataInputStreamExample {

    public static void main(String[] args) {
        try {
            // create a new DataInputStream object
            DataInputStream dis = new DataInputStream(new FileInputStream("float.dat"));
            // read float value from the stream
            float value = dis.readFloat();
            System.out.println("Float value: " + value);
            // close the stream
            dis.close();

        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}
In the above code, we have used the 
DataInputStream
constructor which takes an object of type 
FileInputStream
, which we have initialized with the file 
float.dat
. The 
readFloat()
method reads a float value from the input stream and stores it in a variable called 
value
. Finally, the stream is closed using the 
close()
method. When you run the program, the output will be: 

          
           
Float value: 3.14159265f
