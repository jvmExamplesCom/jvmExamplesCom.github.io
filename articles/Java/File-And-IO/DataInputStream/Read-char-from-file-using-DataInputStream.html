Reading Characters from a File with DataInputStream in Java

Java has a built-in utility to read characters from a file, called the DataInputStream. This stream is particularly useful when working with data formats that are not character based, like binary files or primitive types. In this tutorial, we'll learn how to use DataInputStream to read characters from a file in Java.

Creating a DataInputStream Object

The first step in reading characters from a file with DataInputStream is to create a DataInputStream object. To do this, first open a FileInputStream for the desired file: 

          
           
FileInputStream fis = new FileInputStream("myFile.txt");
Now, create a DataInputStream from the FileInputStream, using the following code: 

          
           
DataInputStream dis = new DataInputStream(fis);

Reading Characters with DataInputStream

Once the DataInputStream is created, we can start reading characters from the file. To do this, simply call the readChar() method on the DataInputStream instance. The method will return the next character in the file as a char primitive type. 

          
           
char c = dis.readChar();
It is important to note that calling this method multiple times will cause it to read each character one at a time in the same order they appear in the file.

Using DataInputStream to Read Strings

DataInputStream also allows us to read strings from a file. We can do this by calling the readUTF() method on the DataInputStream instance. This will return a String representing the next string in the file. 

          
           
String s = dis.readUTF();
It is important to note that calling this method multiple times will cause it to read each string one at a time in the same order they appear in the file.

Closing a DataInputStream

When you are done using the DataInputStream, it is important to close it. To do this, simply call the close() method on the DataInputStream instance. 

          
           
dis.close();
This will close both the DataInputStream and its associated FileInputStream.

Summary

In this tutorial, we learned how to use DataInputStream to read characters from a file in Java. We saw how to create a DataInputStream object and read characters from it, as well as how to use it to read strings from the file. Finally, we discussed the importance of closing a DataInputStream when you are done with it.
