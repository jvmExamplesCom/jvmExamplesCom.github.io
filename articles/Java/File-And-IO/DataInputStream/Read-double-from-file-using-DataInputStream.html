Reading a Double from File Using DataInputStream

Using the DataInputStream class, it is possible to read files containing double values. This can be useful for applications that require double data types in order to process certain tasks. This tutorial will explain how to read double values from a file using the DataInputStream class.

Creating the DataInputStream Object

The first step is to create a DataInputStream object. This object is used to read data types from a file. An example of creating a DataInputStream object is given below.


          
           
DataInputStream reader = new DataInputStream(new FileInputStream("example.txt"));

The above code creates a DataInputStream object called 

reader
that reads from the file located at the path 
example.txt
.

Reading a Double from a File

To read a double from the file, the 
readDouble()
method of the 
DataInputStream
object is used. The 
readDouble()
method returns a double value read from the file. An example of reading a double from a file is given below. 

          
           
double d = reader.readDouble();
The variable 
d
now contains the double value that was read from the file.

Closing the DataInputStream Object

Once all of the data has been read from the file, the 

DataInputStream
object must be closed. This can be done using the 
close()
method of the 
DataInputStream
object. An example of closing the 
DataInputStream
object is given below. 

          
           
reader.close();

Conclusion

In this tutorial, we have explained how to read double values from a file using the DataInputStream class. We have discussed how to create a DataInputStream object and how to use the 

readDouble()
and 
close()
methods.
