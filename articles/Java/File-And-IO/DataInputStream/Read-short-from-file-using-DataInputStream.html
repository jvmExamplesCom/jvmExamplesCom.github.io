Read Short From File Using DataInputStream

The 

          
           
DataInputStream
class is a Java standard class found in the 

          
           
java.io
package. It can be used to read data from a file in a stream-like, byte-oriented fashion. This article will explain how to read shorter pieces of data from a file using a 

          
           
DataInputStream
.

Creating a DataInputStream

Before attempting to read any data, it is necessary to create a 

          
           
DataInputStream
object. There are several ways to do this. The first option is to create a 

          
           
FileInputStream
and then pass that 

          
           
FileInputStream
to the constructor of the 

          
           
DataInputStream
. 

          
           
FileInputStream fis = new FileInputStream("data.dat");
DataInputStream dis = new DataInputStream(fis);
Another option is to directly create a 

          
           
DataInputStream
and pass a 

          
           
File
object to the constructor. 

          
           
File f = new File("data.dat");
DataInputStream dis = new DataInputStream(f);
Finally, a 

          
           
DataInputStream
can also be created by passing an existing 

          
           
InputStream
to the constructor. 

          
           
InputStream is = new URL("http://example.com/data.dat").openStream();
DataInputStream dis = new DataInputStream(is);

Reading Short Pieces of Data

Once a 

          
           
DataInputStream
has been created, it can be used to read data from the file. The 

          
           
DataInputStream
provides several methods for reading different types of data. For example, to read a single byte from the file, the 

          
           
readByte()
method can be used. 

          
           
int b = dis.readByte();
For more complex data types, such as 

          
           
int
, 

          
           
long
or 

          
           
String
, the 

          
           
readInt()
, 

          
           
readLong()
and 

          
           
readUTF()
methods can be used. 

          
           
int x = dis.readInt();
long y = dis.readLong();
String s = dis.readUTF();
It should be noted that the 

          
           
DataInputStream
only reads data in the same order it was written to the file. Therefore, care must be taken when writing and reading data.

Closing the Stream

Once all the data has been read from the file, the 

          
           
DataInputStream
should be closed. This can be done by calling the 

          
           
close()
method on the stream. 

          
           
dis.close();
It is also important to remember to close the underlying streams, such as the 

          
           
FileInputStream
, if they were manually created. 

          
           
fis.close();

Conclusion

In this article, we have seen how to read shorter pieces of data from a file using a 

          
           
DataInputStream
. We have seen how to create a 

          
           
DataInputStream
and how to use it to read data from the file. Finally, we have seen how to correctly close the stream once all the data has been read.
