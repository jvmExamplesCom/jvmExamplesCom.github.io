Writing a File with FileOutputStream in Java

In Java, writing a file is extremely easy thanks to the 

FileOutputStream
class. This class allows us to write any type of file and store it in an efficient way.

Creating a File using FileOutputStream

The first step to writing a file is to create the 
FileOutputStream
instance. This can be done using the constructor 

          
           
new FileOutputStream(String path)
. Here, we need to pass the path of the file that we want to create as the argument. 

          
           
FileOutputStream fos = new FileOutputStream("myfile.txt");

Now we have created a file called “myfile.txt” in the current directory. We can start writing data to this file now.

Writing Data with FileOutputStream

Once a 
FileOutputStream
instance has been created, we can start writing data to the file. This is done using the 
write
method. This method takes an 
int
as input, which represents the byte that needs to be written to the file. 

          
           
fos.write(0x41); // this writes the 'A' character (ASCII value 0x41)

We can also write an array of bytes instead of a single byte. This is done using the 

write
method, which takes an 
byte[]
argument representing the array of bytes to be written to the file. 

          
           
byte[] data = new byte[] {0x41, 0x42, 0x43}; // An array of 3 bytes
fos.write(data); // Writes the 3 bytes to the file

Closing the FileOutputStream

Once we are done writing data to the file, we need to close the 

FileOutputStream
instance. This is done using the 
close
method. This method will release any resources associated with the file and make sure that all the data is written to the destination file. 

          
           
fos.close();

Writing Text with FileOutputStream

Till now, we have seen how to write an array of bytes to the file. However, most of the times, we will be writing text to the file instead of bytes. For this, we need the help of a 

Writer
. The 
FileOutputStream
class takes a 
Writer
, which can be used to write characters to the file instead of bytes. 

          
           
Writer writer = new OutputStreamWriter(fos);
writer.write("This is a sample line of text");
writer.close();
fos.close();
Here, we create a new 
OutputStreamWriter
, which wraps around the 
FileOutputStream
instance. This allows us to use the 
write
method of the 
Writer
to write text to the file. Once we are done writing data, both the 
Writer
and 
FileOutputStream
instances need to be closed.

Conclusion

In this tutorial, we saw how to write a file using the 

FileOutputStream
class in Java. We also saw how to write an array of bytes and text using the 
write
method of the 
FileOutputStream
and 
Writer
class respectively. Once we are done writing the file, we need to make sure that both the 
FileOutputStream
and the 
Writer
instances are closed properly.
