Create FileOutputStream Object from String File Path

The Java class FileOutputStream is used to write data to a file. The FileOutputStream object can be created from a String file path, which specifies the location of the file on the computer. This article will explore how to create a FileOutputStream object from a given file path, as well as provide some example code.

Creating the FileOutputStream Object

The FileOutputStream constructor takes a single argument, which is the file path as a String. The example below shows how to create a FileOutputStream object with a file path argument.


          
           
String filePath = "C:\\myDirectory\\myFile.txt";
FileOutputStream fos = new FileOutputStream(filePath);

It is important to note that the given file path should be valid. If the file path provided is not valid, the FileOutputStream constructor will throw a 


          
           
FileNotFoundException
.

Writing to the File

Once the FileOutputStream object has been created, data can be written to the file it represents. To write data to the file represented by the FileOutputStream object, the 

write()
method can be used. The 
write()
method takes a single argument, which should be an array of bytes. An example is shown below. 

          
           
byte[] data = { 0x41, 0x42, 0x43 };
fos.write(data);

It is important to remember to also close the FileOutputStream object once the writing is finished. This can be done by calling the 

close()
method on the FileOutputStream object.

Putting It All Together

The following example shows how to create a FileOutputStream object, write data to it, and then close the object.


          
           
String filePath = "C:\\myDirectory\\myFile.txt";
FileOutputStream fos = new FileOutputStream(filePath);
byte[] data = { 0x41, 0x42, 0x43 };
fos.write(data);
fos.close();

In this example, the FileOutputStream object is created from the file path provided as a String. Then, an array of bytes is written to the file using the 

write()
method. Finally, the FileOutputStream object is closed by calling the 
close()
method.
