Copy Binary File Using Streams in Java

Java provides a powerful mechanism for copying and manipulating binary files. Using streams, you can read binary data using the 

InputStream
class, and you can write binary data using the 
OutputStream
class. In this article, we will discuss how to copy binary files using streams.

Creating Input and Output Streams

Before we can copy a binary file, we need to create an 

InputStream
and an 
OutputStream
. For example, if we want to copy a file from the file system, we can create a 
FileInputStream
and a 
FileOutputStream
. We can also create streams from URLs, memory buffers, databases and more. 

          
           
// Create input stream
InputStream inputStream = new FileInputStream("input.bin");

// Create output stream
OutputStream outputStream = new FileOutputStream("output.bin");

Copying the Data

Once we have created the streams, we can copy the data from one stream to another. The 

InputStream
and 
OutputStream
classes provide methods that allow us to copy data from one stream to another. For example, we can use the 
read()
and 
write()
methods to copy byte by byte, or we can use the 
readFully()
and 
writeFully()
methods to copy the entire contents of the stream: 

          
           
// Copy byte-by-byte
int b;
while((b = inputStream.read()) != -1) {
    outputStream.write(b);
}

// Copy entire stream
byte[] bytes = new byte[inputStream.available()];
inputStream.readFully(bytes);
outputStream.writeFully(bytes);

Closing the Streams

Once we have copied the data, we need to close the streams. The 

InputStream
and 
OutputStream
classes provide a 
close()
method that we can use to close the streams. It's important to close the streams after we have finished copying the data, as this will free up any resources that were being used by the streams. 

          
           
// Close streams
inputStream.close();
outputStream.close();

Example Program

Let's put all of this together in an example program that copies a binary file from one location to another:


          
           
import java.io.*;

public class BinaryFileCopy {
  public static void main(String[] args) throws IOException {
    // Create input stream
    InputStream inputStream = new FileInputStream("input.bin");

    // Create output stream
    OutputStream outputStream = new FileOutputStream("output.bin");

    // Copy byte-by-byte
    int b;
    while((b = inputStream.read()) != -1) {
        outputStream.write(b);
    }

    // Close streams
    inputStream.close();
    outputStream.close();
  }
}
In this example, we create an 
InputStream
and an 
OutputStream
, and then copy the contents of one stream to the other. Finally, we close the streams.

This example program shows the basics of copying binary files using streams. However, it is usually more efficient to copy large amounts of data using larger chunks. We can do this using the 

read()
and 
write()
methods, which allow us to read and write a block of data rather than just a single byte. For more information on how to do this, see the documentation for the 
InputStream
and 
OutputStream
classes.
