Append Output to File using FileOutputStream

In Java, writing to a file using the 
FileOutputStream
class is a quick and simple process. By appending output to an existing file, it is possible to add data to the end of the file without having to rewrite the entire file. The following code example provides a demonstration of how to append output to a file with 
FileOutputStream
: 

          
           
// Create out FileOutputStream object 
FileOutputStream fos = new FileOutputStream("myfile.txt", true);

// Write data to the end of the file 
String data = "This is a new line of text "; 
fos.write(data.getBytes()); 

// Close the stream 
fos.close();
The 
true
parameter in the 
FileOutputStream
constructor is key to enabling file appending. This parameter specifies whether or not the existing contents should be overwritten by the newly written data. When set to 
true
, any existing data will remain and the new data will be added after it.

Using a BufferedWriter

The above example required the creation of a 
FileOutputStream
object for each write to the file. A more efficient method is to use a 
BufferedWriter
. The following code example demonstrates how to append output to a file using a 
BufferedWriter
. 

          
           
// Create out BufferedWriter object 
BufferedWriter bw = new BufferedWriter(new FileWriter("myfile.txt", true)); 

// Write data to the end of the file 
String data = "This is a new line of text "; 
bw.write(data); 

// Close the stream 
bw.close();
By using a 
BufferedWriter
object, you can write multiple lines of text to the end of the file in one go, rather than having to open and close the file for each write operation as with the 
FileOutputStream
approach.

Conclusion

Whether you choose to use the 
FileOutputStream
or 
BufferedWriter
approach, appending output to a file in Java is relatively straightforward. There are several benefits of appending to a file, such as the ability to add new data at the end of the file without having to rewrite its entire contents.
