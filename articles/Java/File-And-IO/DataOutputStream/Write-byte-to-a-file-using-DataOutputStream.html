Write Byte to a File using DataOutputStream

Writing a byte to a file using 

          
           
DataOutputStream
class requires the use of methods 

          
           
writeByte()
and 

          
           
close()
. The following program demonstrates how this can be achieved: 

          
           
import java.io.*;

public class DataOutputStreamTest {
  public static void main(String args[]) throws IOException {
    FileOutputStream fos = new FileOutputStream("byte.txt"); 
    DataOutputStream dos = new DataOutputStream(fos);

    dos.writeByte(65);  
    dos.close();
  }
}
The program starts by creating a 

          
           
FileOutputStream
object which creates an output stream to write data to a file. Then a 

          
           
DataOutputStream
object is created that uses the 

          
           
FileOutputStream
object to write the data. Next, the program calls the 

          
           
writeByte()
method to write the given byte to the file. Finally the program calls the 

          
           
close()
method to close the 

          
           
DataOutputStream
and the associated 

          
           
FileOutputStream
. This program will create a file called ‘byte.txt’ in the current directory with a single byte ‘A’ written in it.

Using DataOutputStream to Write Other Data Types to File

Apart from writing bytes to a file, it is also possible to write other data types like short, int, float, double, char etc. to a file. To do this, the program needs to use the appropriate 

          
           
write
method for the type of data being written into the file. For example, the program can use 

          
           
writeInt()
method to write an integer, 

          
           
writeFloat()
method to write a float, 

          
           
writeDouble()
method to write a double and so on. The following program demonstrates how to use 

          
           
DataOutputStream
class to write some integers to a file: 

          
           
import java.io.*;

public class DataOutputStreamTest {
  public static void main(String args[]) throws IOException {
    FileOutputStream fos = new FileOutputStream("integers.txt"); 
    DataOutputStream dos = new DataOutputStream(fos);

    dos.writeInt(10);
    dos.writeInt(20);
    dos.writeInt(30);

    dos.close();
  }
}
This program creates a file called ‘integers.txt’ with three integers - 10, 20 and 30.

Reading Data Written with DataOutputStream

The data written using 

          
           
DataOutputStream
can also be read using a 

          
           
DataInputStream
object. The program should create a 

          
           
DataInputStream
object and call the appropriate 

          
           
read
method for the type of data to be read from the file. For example, the program can use 

          
           
readByte()
method to read a byte, 

          
           
readInt()
method to read an integer, 

          
           
readFloat()
method to read a float, and so on. The following program demonstrates how to use a 

          
           
DataInputStream
object to read the data that was previously written to a file using 

          
           
DataOutputStream
: 

          
           
import java.io.*;

public class DataInputStreamTest {
  public static void main(String args[]) throws IOException {
    FileInputStream fis = new FileInputStream("integers.txt");
    DataInputStream dis = new DataInputStream(fis);

    int n1 = dis.readInt();
    int n2 = dis.readInt();
    int n3 = dis.readInt();

    System.out.println(n1 + " " + n2 + " " + n3);
    dis.close();
  }
}
This program reads the three integers that were written to the file ‘integers.txt’ and prints them to the console. In summary, the 

          
           
DataOutputStream
class provides different methods to write primitive data types to a file. These methods can be used to write bytes, short, int, float, double, char etc. to a file. The data written to the file can then be read back using the 

          
           
DataInputStream
class.
