Writing Integer to File using DataOutputStream

DataOutputStream is one of the most powerful classes for writing data to a file in Java. It provides methods for writing primitive data types like int, float and double. The data written by the DataOutputStream class can be read by the DataInputStream class. In this article, we take a look at how to write an integer value to file using DataOutputStream.

Write Integer to File with DataOutputStream

The following example code demonstrates how to write an integer to a file using DataOutputStream.


          
           
DataOutputStream out = new DataOutputStream(new FileOutputStream("file.dat"));
out.writeInt(42);
out.close();
Above, we create a 
DataOutputStream
object with a 
FileOutputStream
. Then, we call the 
writeInt()
method to write the integer value 42 to the file. Lastly, we close the output stream.

Read Integer from File with DataInputStream

To read back the integer data from the file, we can use the 

DataInputStream
class. The following example code demonstrates how to read an integer from a file. 

          
           
DataInputStream in = new DataInputStream(
    new FileInputStream("file.dat")); 
int value = in.readInt();
in.close();
System.out.println("Integer value: " + value);
Here, we create a 
DataInputStream
object with a 
FileInputStream
. Then, we call the 
readInt()
method to read the integer value from the file. Afterwards, we close the input stream and print out the read integer.

We can run the above example code and it should print out the following output.


          
           
Integer value: 42

Conclusion

In this article, we looked at how to write an integer to a file using DataOutputStream and how to read the written data back using DataInputStream. Writing and reading primitive data types to a file is an easy task, which can be done with the help of these two classes.
