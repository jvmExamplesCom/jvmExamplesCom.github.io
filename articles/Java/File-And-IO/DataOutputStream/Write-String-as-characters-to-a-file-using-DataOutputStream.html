Writing a String as Characters to a File using DataOutputStream in Java

The 

          
           
DataOutputStream
class is a subclass of 

          
           
OutputStream
and provides methods for writing data to a file. It allows the programmer to write Java primitive data types such as 

          
           
int
, 

          
           
char
or 

          
           
String
as characters to a file. In this article, we will discuss how to write a String as characters to a file using the 

          
           
DataOutputStream
class in Java.

Creating a DataOutputStream Object

Before writing a String in a file using 

          
           
DataOutputStream
, we need to create an object of the 

          
           
DataOutputStream
class. We can do so by passing a FileOutputStream object to the DataOutputStream constructor. The FileOutputStream object should contain the path and name of the file. The following code creates a DataOutputStream object and passes it the path of the file "test.txt": 

          
           
FileOutputStream fos = new FileOutputStream("test.txt");
DataOutputStream dos = new DataOutputStream(fos);

Writing a String to a File using writeBytes() Method

Once the DataOutputStream object has been created, we can call its 

          
           
writeBytes()
method to write a String to the file. This method requires a String as an argument which is to be written in the file. The following code writes the String "Hello!" to the file "test.txt": 

          
           
dos.writeBytes("Hello!");
The 

          
           
writeBytes()
method can also be used to write bytes of data that is not a String to the file. It takes an array of bytes as an argument and writes them to the file. For example, the following code writes bytes of data containing a 3 and a 5 to the file: 

          
           
byte[] bytes = {3, 5}; 
dos.writeBytes(bytes);

Closing the DataOutputStream

Finally, we need to close the DataOutputStream after writing the String. This is done by calling the 

          
           
close()
method on the DataOutputStream object. The following code closes the DataOutputStream: 

          
           
dos.close();

Example Program

The following program demonstrates how to write a String to a file using the DataOutputStream class in Java: 

          
           
import java.io.DataOutputStream;
import java.io.FileOutputStream;
import java.io.IOException;

public class Example {
    public static void main(String[] args) {
        try {
            // Create a DataOutputStream object
            FileOutputStream fos = new FileOutputStream("test.txt");
            DataOutputStream dos = new DataOutputStream(fos);

            // Write a String to the file
            dos.writeBytes("Hello!");

            // Close the DataOutputStream
            dos.close();

            System.out.println("String written to file successfully!");
        }
        catch (IOException e) {
            e.printStackTrace();
        }
    }
}

Conclusion

In this article, we discussed how to write a String as characters to a file using the DataOutputStream class in Java. We created a DataOutputStream object and passed it a FileOutputStream object containing the path and name of the file we wanted to write. We then used the 

          
           
writeBytes()
method to write a String to the file. Finally, we closed the DataOutputStream when we were done.
