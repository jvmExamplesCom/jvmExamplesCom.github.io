Read Character from Console using InputStream

The InputStream class is a concrete class that is used to read data from a variety of sources, including the console. Here we will discuss how to read characters from the console using an InputStream.

Creating an InputStream Object

To read input data such as characters from the console, create an InputStream object. Here is an example of creating an InputStream object that reads from System.in:


          
           
InputStream input = System.in;

Reading Characters from the Console

Once you have created an input stream you can use the 

read()
method to read individual characters from the console. This method returns an 
int
containing the value of the character that was read. Here is an example of the 
read()
method being used to read a character from the console: 

          
           
int c = input.read();

Displaying Characters as Output

To display the character that was read to the console, you can use the 

println()
method with typecasting the 
int
returned by read to a 
char
. Here is an example of displaying a character on the console: 

          
           
System.out.println((char) c);

Example Program

Here is an example program that reads characters from the console and displays them as output:


          
           
import java.io.InputStream;

public class ReadChar {

  public static void main(String[] args) throws Exception {
    // Create an input stream
    InputStream input = System.in;

    // Read character from console
    int c = input.read();

    //Display character
    System.out.println((char) c);
  }
}
