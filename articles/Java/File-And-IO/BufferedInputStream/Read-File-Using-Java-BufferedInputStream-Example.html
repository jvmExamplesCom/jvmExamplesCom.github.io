Read File Using Java BufferedInputStream Example

In this tutorial, we'll learn how to read a file using the 

BufferedInputStream
class in Java.

Using BufferedInputStream to Read a File

The 
BufferedInputStream
class provides a convenient way to read bytes from a file. It buffers the data internally before reading it from the file. To use the 
BufferedInputStream
class, we need to create an instance of it with a 
FileInputStream
object. Then, we can call the 
read()
method to read a single byte from the file. 

          
           
BufferedInputStream bis = new BufferedInputStream(new FileInputStream("file.txt"));
int b = bis.read();
We can also call the 

          
           
read(byte[] b, int off, int len)
method to read multiple bytes into a byte array. 

          
           
byte[] buf = new byte[1024];
int len = bis.read(buf, 0, 1024);

Example Program

Let's look at an example program that reads a text file and prints out the contents.


          
           
import java.io.BufferedInputStream;
import java.io.FileInputStream;
import java.io.IOException;

public class Main {

    public static void main(String[] args) {
        try (BufferedInputStream bis = new BufferedInputStream(new FileInputStream("file.txt"))) {
            byte[] buf = new byte[1024];
            int len;
            
            while ((len = bis.read(buf)) != -1) {
                String s = new String(buf, 0, len);
                System.out.println(s);
            }
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}
In the above example, we create a 
BufferedInputStream
with a 
FileInputStream
. We then read the content of the file in chunks of 1024 bytes and print it out.

class to read a file in Java. We looked at an example program that read a text file and printed out the contents.
