Read File using FileInputStream

Java provides various classes for reading and writing data from a file. FileInputStream is a class used for reading raw bytes from a file. This class is used for reading the primitive data types, such as int, long, char, etc. It also provides methods to read the contents of a file into a byte array.

Create a FileInputStream Instance

A FileInputStream can be created in two ways. We can provide the path name of the file or the File object.


          
           
FileInputStream fin = new FileInputStream("sample.txt");  // using path

File file = new File("sample.txt");
FileInputStream fin = new FileInputStream(file);   // using File Object

We can also supply a FileDescriptor object for creating a FileInputStream.


          
           
FileDescriptor fd = //get from somewhere
FileInputStream fin = new FileInputStream(fd);

Reading Data from a FileInputStream

The read() method of FileInputStream class reads one byte at a time from a given file.


          
           
int ch;
while ((ch = fin.read()) != -1) {
    System.out.print((char)ch);
}

The read(byte[] b) method of FileInputStream class reads multiple bytes from a given file.


          
           
byte[] data = new byte[1024];
int length = fin.read(data);

The read(byte[] b, int off, int len) method of FileInputStream class reads multiple bytes from a given file.


          
           
int length = fin.read(data, 0, 1024);

Closing a FileInputStream

Once the work of a FileInputStream is finished, we need to close the stream using the close() method.


          
           
fin.close();
