Skip n bytes while Reading the file using FileInputStream

Java provides several ways to read files, but in this article we'll focus on reading a file using 

          
           
FileInputStream
. This stream allows reading bytes from a file; however, during the reading process, it's necessary to skip a certain number of bytes, so not all the content is read.

Creating an Input Stream

The creation of an 

          
           
InputStream
requires the file path as a parameter, for example: 

          
           
String filePath = "C:/example.txt";
FileInputStream fis = new FileInputStream(filePath);

Skip n Bytes

The 

          
           
skip()
method of 

          
           
FileInputStream
allows you to skip a certain number of bytes while reading the file. It takes one parameter, which is the number of bytes that should be skipped: 

          
           
fis.skip(3);
This method can be used multiple times in a row if needed. However, this method returns the number of bytes skipped, which can be used to control the reading process.

Read the file content

After calling 

          
           
skip()
, you can use the same stream to read the file content. The method 

          
           
read()
returns an integer between 0 and 255 that represents one byte from the file: 

          
           
int content = fis.read();
System.out.println(content); // Prints the byte read

Complete Example

The following example shows how to read a file, skip five bytes and then read the rest of the file: 

          
           
String filePath = "C:/example.txt";
FileInputStream fis = new FileInputStream(filePath);

// Skip the first five bytes
fis.skip(5);

// Read the rest of the file
int content;
while ((content = fis.read()) != -1) {
    System.out.print((char) content);
}

fis.close();
In the example, we check if the byte read is -1, which means the end of the file has been reached. Finally, it's important to remember to close the stream after its usage: 

          
           
fis.close();
