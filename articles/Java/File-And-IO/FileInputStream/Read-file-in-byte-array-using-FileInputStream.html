How to Read a File in a Byte Array using FileInputStream

Java has multiple ways of reading files, one popular way is to read the file in a byte array using FileInputStream. In this article, we will be looking at how to read the file in a byte array using the FileInputStream class.

What is FileInputStream?

FileInputStream is an input stream used to read data from a file. It is part of the Java IO package and it can be used to read data from any type of file. It supports both binary and text files and can be used to read any type of data.

Creating a FileInputStream Object

The first step to read a file in a byte array is to create a FileInputStream object. This object can be created by passing the path to the file as an argument to the constructor. Here is an example of how to create a FileInputStream object: 

          
           
FileInputStream fileInputStream = new FileInputStream("/path/to/file");
Once the FileInputStream object is created, it can be used to read the file.

Reading the File in a Byte Array

Now that we have created a FileInputStream object, we can read the file in a byte array by using the read() method of the FileInputStream class. This method will read the contents of the file and store it in a byte array. Here is an example of how to use the read() method: 

          
           
byte[] buffer = new byte[1024];
int bytesRead = fileInputStream.read(buffer);
The read() method will read up to 1024 bytes from the file and store it in the buffer array. The number of bytes read is stored in the bytesRead variable.

Closing the FileInputStream Object

When we are done reading the file, we need to close the FileInputStream object which we created earlier. To do so, we can use the close() method of the FileInputStream class. Here is an example of how to use the close() method: 

          
           
fileInputStream.close();
Once the FileInputStream object is closed, we can no longer read the file.

Conclusion
