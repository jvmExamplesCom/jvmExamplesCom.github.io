Writing Bytes to File Using BufferedOutputStream in Java

In this article, we will take a look at how to write bytes to a file using the BufferedOutputStream class in Java.

What is BufferedOutputStream?

BufferedOutputStream is a type of output stream in Java which allows writing of bytes to an underlying output stream in a buffered manner. The main advantage of using a BufferedOutputStream is that it reduces I/O by writing data in chunks to the output stream as opposed to writing one byte at a time.

Example Usage of BufferedOutputStream

In order to use BufferedOutputStream, we need to first create an instance of it using the constructor.


          
           
BufferedOutputStream bos = new BufferedOutputStream(new FileOutputStream("file.txt"));

The above code creates an instance of BufferedOutputStream with a given FileOutputStream instance as its parameter. Note that BufferedOutputStream can also be used with other types of output streams such as PipedOutputStream.

Once we have created the instance of BufferedOutputStream, we can now write bytes to the output stream.


          
           
byte[] bytes = {1,2,3};
bos.write(bytes);

The above code writes the given array of bytes to the output stream using the write() method.

It is important to note that BufferedOutputStream does not write the bytes to the underlying output stream immediately. Instead, it caches them in memory and periodically flushes the cached data to the output stream. This helps in improving the write speed by reducing the number of I/O operations.

Finally, once we are done writing bytes to the output stream, we need to close the stream.


          
           
bos.close();

The above code closes the BufferedOutputStream and flushes any remaining bytes to the underlying output stream.

Conclusion

In this article, we have seen how to write bytes to a file using the BufferedOutputStream class in Java. We have also seen an example of how to use the write() and close() methods of BufferedOutputStream.
