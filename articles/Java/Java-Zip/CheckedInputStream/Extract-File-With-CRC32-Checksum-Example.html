Extract File With CRC32 Checksum Example

In this article we will be looking into how to extract files using CRC32 checksum in Java. We will go over the necessary code, demonstrate and explain the example, as well as provide an output of the example.

Necessary Code

The necessary code for extracting a file with CRC32 checksum in Java is as follows:


          
           
FileInputStream fis = new FileInputStream("file.txt");
byte[] data = new byte[(int) fis.available()];
fis.read(data);
fis.close();

CRC32 crc32 = new CRC32();
crc32.update(data);
System.out.println("CRC32 Checksum : " + Long.toHexString(crc32.getValue());

Demonstration and Explanation of Example

This example is used to extract the contents of a file and then generate the CRC32 checksum of the extracted data. The first line of code creates a FileInputStream object and assigns it to the variable fis.


          
           
FileInputStream fis = new FileInputStream("file.txt");

The second line of code creates a new byte array object which is used to store the data from the file. The size of the array is made equal to the number of bytes that can be read from the file by using the available() method of fis.


          
           
byte[] data = new byte[(int) fis.available()];

The third line of code reads the content of the file into the byte array. This is done using the read() method of fis which takes the byte array as a parameter.


          
           
fis.read(data);

The fourth line of code closes the FileInputStream object.


          
           
fis.close();

The fifth line of code creates a CRC32 object which is used to generate the checksum of the data.


          
           
CRC32 crc32 = new CRC32();

The sixth line of code updates the checksum with the data read from the file.


          
           
crc32.update(data);

The seventh line of code prints out the generated checksum by converting it to hexadecimal string format.


          
           
System.out.println("CRC32 Checksum : " + Long.toHexString(crc32.getValue()));

Output of the Example

The output of the example is as follows: 

          
           
CRC32 Checksum : 89aefbf6
