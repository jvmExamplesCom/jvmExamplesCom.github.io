Generate Adler32 Checksum For Byte Array Example

This example shows how to generate an Adler32 checksum for a byte array using Java programming language. Adler32 is a 32-bit checksum algorithm which is almost as reliable as CRC32, but can be computed much more quickly.

Example

Let's create a demo example that generates an Adler32 checksum for a byte array. We will declare a byte array, populate it with some data and then calculate the Adler32 checksum. 

          
           
// Create a byte array
byte[] data = { 0x001, 0x002, 0x003 };

// Create Adler32 checksum
Adler32 adler32 = new Adler32();
adler32.update(data);
long checksum = adler32.getValue();

// Print Checksum
System.out.println("Checksum: " + checksum);
If we run the above program, it should print the following output: 

          
           
Checksum: 67108864
The Adler32 checksum for the given data is 67108864.
