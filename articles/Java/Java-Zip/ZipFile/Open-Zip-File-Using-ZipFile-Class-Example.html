Open Zip File Using ZipFile Class Example

The 
ZipFile
class in Java provides an API to read, create and modify ZIP files. It is part of the java.util.zip package. In this article we will discuss how to open ZIP files using 
ZipFile
class.

Example 1: Open a ZIP File Using Path

In this example, we will use Java's 
Path
class to open a ZIP file located in our local file system. 

          
           
Path path = Paths.get("C:\\Users\\User\\Documents\\zipfile.zip");
ZipFile zipFile = ZipFile.newZipFile(path);
System.out.println("Zip file opened successfully.");

Example 2: Open a ZIP File from an Input Stream

In this example, we will use a 
java.io.InputStream
object to open a ZIP file. 

          
           
InputStream is = new FileInputStream("C:\\Users\\User\\Documents\\zipfile.zip");
ZipFile zipFile = ZipFile.newZipFile(is);
System.out.println("Zip file opened successfully.");

Example 3: Open a ZIP File from a Byte Array

In this example, we will use a byte array to open a ZIP file.


          
           
byte[] data = Files.readAllBytes("C:\\Users\\User\\Documents\\zipfile.zip");
ZipFile zipFile = ZipFile.newZipFile(data);
System.out.println("Zip file opened successfully.");

Example 4: Open a ZIP File from a String

In this example, we will use a 
String
object to open a ZIP file. 

          
           
String str = new String(Files.readAllBytes("C:\\Users\\User\\Documents\\zipfile.zip"), StandardCharsets.UTF_8);
ZipFile zipFile = ZipFile.newZipFile(str);
System.out.println("Zip file opened successfully.");

Example 5: Close a ZIP File

Once we are done with a ZIP file, we should close it. Closing a ZIP file is simple. We just need to call the 

close()
method on the 
ZipFile
object. 

          
           
zipFile.close();
System.out.println("Zip file closed successfully.");
