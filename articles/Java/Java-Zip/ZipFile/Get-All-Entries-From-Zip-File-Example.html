Unzip a File in Java

We can also use the Java zip package to unzip a file. The following example consists of a method that takes in a File object representing the zip file to be extracted and extracts the content to the same folder as the zip file:


          
           
public void unzipFile(File zipFile) throws IOException {
        // Create a buffer for reading the files
        byte[] buffer = new byte[1024];

        // Create the output directory if it doesn't exist
        File folder = zipFile.getParentFile();
        if (!folder.exists()) {
            folder.mkdir();
        }

        // Get the zipped file entries
        ZipInputStream zis = new ZipInputStream(new FileInputStream(zipFile));
        ZipEntry entry = zis.getNextEntry();

        while (entry != null) {
            String fileName = entry.getName();
            File newFile = new File(folder.getAbsolutePath() + File.separator + fileName);

            // Create the new file
            new File(newFile.getParent()).mkdirs();
            FileOutputStream fos = new FileOutputStream(newFile);

            int len;
            while ((len = zis.read(buffer)) > 0) {
                fos.write(buffer, 0, len);
            }

            fos.close();
            entry = zis.getNextEntry();
        }

        zis.closeEntry();
        zis.close();
    }

In this code example, we create a ZipInputStream object which is used to read data from the zip file. We then loop through all the entries in the zip file and extract them to the specified folder. Once all the entries are extracted, we close the ZipInputStream object.

Get All Entries From Zip File

We can use the Java zip package to get all the entries from a zip file. The following example consists of a method that takes in a File object representing the zip file and returns a list of String objects containing the entries:


          
           
public List<String> getEntries(File zipFile) throws IOException {
        // Create a List to store the entries
        List<String> entries = new ArrayList<>();

        // Get the zipped file entries
        ZipInputStream zis = new ZipInputStream(new FileInputStream(zipFile));
        ZipEntry entry = zis.getNextEntry();

        while (entry != null) {
            String entryName = entry.getName();
            entries.add(entryName);
            entry = zis.getNextEntry();
        }

        zis.closeEntry();
        zis.close();

        return entries;
    }

In this code example, we create a ZipInputStream object which is used to read data from the zip file. We then loop through all the entries and add them to the entries list. Once all the entries are added, we close the ZipInputStream object.
