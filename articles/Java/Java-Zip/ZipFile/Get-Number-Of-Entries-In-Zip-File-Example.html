Get Number of Entries in Zip File Example

This example demonstrates how to get a count of the entries in a zip file using Java.

Example Code


          
           
// create a new ZipFile object 
ZipFile zipFile = new ZipFile("path/to/zipfile.zip"); 
  
// get the list of entries from the zip file 
Enumeration<? extends ZipEntry> entries = zipFile.entries(); 
  
int count = 0; 
  
// count the number of entries 
while (entries.hasMoreElements()) { 
    count++; 
    entries.nextElement(); 
} 

// print the number of entries 
System.out.println("Number of entries in zipfile = " + count);

Example Output


          
           
Number of entries in zipfile = 3
