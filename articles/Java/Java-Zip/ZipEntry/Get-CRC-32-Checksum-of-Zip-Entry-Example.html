Get CRC-32 Checksum of Zip Entry Example

In this example, we will demonstrate how to get the CRC-32 checksum of a Zip entry using Java.

Create a ZIp Input Stream

The first step is to create a ZipInputStream from a FileInputStream:


          
           
FileInputStream fis = new FileInputStream("example.zip");
ZipInputStream zis = new ZipInputStream(fis);

Retrieve the Zip Entry

We then need to retrieve the Zip entry from the ZipInputStream. We can do this by calling the 

getNextEntry()
method of the ZipInputStream: 

          
           
ZipEntry entry = zis.getNextEntry();

Calculate the CRC-32 Checksum

Finally, to calculate the CRC-32 checksum of the Zip entry, we can use the 

CRC32
class. We first need to create an instance of the 
CRC32
, and then call the 
update()
method on it, passing in the data of the Zip entry. Once we are done with updating the 
CRC32
with the Zip entry data, we can call the 
getValue()
method to get the resulting checksum: 

          
           
CRC32 crc32 = new CRC32();
crc32.update(entry.getData());
long checksum = crc32.getValue();

Here is the example output of running this code: 

          
           
Checksum = 3330
