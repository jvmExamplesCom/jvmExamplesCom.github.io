Get Compressed Size of Zip Entry Example

Java's zip classes provide the ability to work with ZIP files. The ZipEntry class has a method called getCompressedSize which will give you the size of the compressed file stored in the ZIP file. This example shows how to use the getCompressedSize method to find the size of the compressed file in a ZIP file.

Example

The following example creates a 

          
           
ZipFile
from a given file path, and retrieves the 

          
           
ZipEntry
. Once the 

          
           
ZipEntry
is obtained, the 

          
           
getCompressedSize()
method is used to get the compressed size of the given ZIP file. 

          
           
//Create a ZipFile from the supplied file path
String filePath = "C:/Users/Dan/sample.zip";
ZipFile zipFile = new ZipFile(filePath);

//Retrieve the ZipEntry
String entryName = "file1.txt";
ZipEntry entry = zipFile.getEntry(entryName);

// obtain the compressed size of the file
long compressedSize = entry.getCompressedSize();
System.out.println("Compressed size of the file: "+compressedSize);
The above code will print the following output: 

          
           
Compressed size of the file: 11244
