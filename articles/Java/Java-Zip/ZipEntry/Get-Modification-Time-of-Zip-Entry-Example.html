Get Modification Time of Zip Entry Example

Java provides the ZipEntry class to represent each entry of a ZIP file. In this example we will learn how to get the modification time of an entry of a ZIP file. To do this, we will use the 

getTime()
method of the ZipEntry class.

Creating a ZipInputStream

The first step for our example is to create a 
ZipInputStream
object with the file path of the ZIP file. We will then use this object to read each entry in the ZIP file. 

          
           
String filePath = "/path/to/file.zip";
ZipInputStream zis = new ZipInputStream(new FileInputStream(filePath));

Iterating Through Entries

Now that we have our 
ZipInputStream
, we can loop through each entry in the ZIP file using the 
getNextEntry()
method. We can call the 
getTime()
method on the 
ZipEntry
object to get the modification time. 

          
           
ZipEntry zipEntry;
long modTime;
while ((zipEntry = zis.getNextEntry()) != null) {
    modTime = zipEntry.getTime();
    System.out.println("Modification Time: " + modTime);
}

Formatting the Modification Time

We can use the 
SimpleDateFormat
class to format the modification time into a human-readable format. We can create a 
SimpleDateFormat
instance and set the format that we want to use. Then we can pass the modification time long to the 
format()
method to get the formatted date string. 

          
           
SimpleDateFormat sdf = new SimpleDateFormat("MM/dd/yyyy HH:mm:ss");
String modTimeString = sdf.format(modTime);
System.out.println("Formatted Modification Time: " + modTimeString);

Closing the ZipInputStream

After we have iterated through all the entries in the ZIP file, we need to close the 

ZipInputStream
. We can do this by calling the 
close()
method on the 
ZipInputStream
. 

          
           
zis.close();
