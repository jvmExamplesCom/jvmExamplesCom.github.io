Java: Get ZIP Entry Example

The java.util.zip package provides the ability to read, create and modify ZIP and GZIP file formats. In this article, we will discuss how to use the getZipEntry() method of the ZipFile class to retrieve a specific ZipEntry from a ZipFile.

Example 1

In this example, we use the getZipEntry() method to get the ZipEntry for a specified file name. 

          
           
import java.io.File;
import java.io.IOException;
import java.util.zip.ZipEntry;
import java.util.zip.ZipFile;

public class GetZipEntryExample {

    public static void main(String[] args) throws IOException {
        File zipFile = new File("test.zip");
        ZipFile zf = new ZipFile(zipFile);
        ZipEntry entry = zf.getZipEntry("file.txt");
        System.out.println("Found ZipEntry: " + entry);  
    }
}
