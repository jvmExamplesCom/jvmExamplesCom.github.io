Extract File From Zip File Using Command Line Arguments Example

Example 1: Using the jar command to extract a file from a zip archive


          
           
jar xf myzip.zip filename.txt

The above example extracts the file "filename.txt" from the zip archive "myzip.zip", using the jar command.

Example 2: Using the unzip command to extract a file from a zip archive


          
           
unzip -p myzip.zip filename.txt

The above example extracts the file "filename.txt" from the zip archive "myzip.zip", using the unzip command.

Example 3: Extracting multiple files from a zip using command line arguments


          
           
unzip myzip.zip *.txt

The above example extracts all of the text files in the zip archive "myzip.zip", using the unzip command.

Example 4: Extract a specific file with a specific path from a zip archive


          
           
unzip myzip.zip path/to/filename.txt

The above example extracts the file "filename.txt" from the zip archive "myzip.zip" and places it in the "path/to" directory, using the unzip command.

Example 5: Extract a file with command line arguments and redirect output


          
           
unzip myzip.zip filename.txt > myfile.txt

The above example extracts the file "filename.txt" from the zip archive "myzip.zip" and redirects the output to a new file called "myfile.txt", using the unzip command.
