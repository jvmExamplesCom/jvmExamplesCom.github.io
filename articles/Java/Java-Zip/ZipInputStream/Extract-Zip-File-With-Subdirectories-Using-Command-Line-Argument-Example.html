Extract Zip File With Subdirectories Using Command Line Argument Example

Command line arguments can be used to extract a zip file, with subdirectories, from the command line. This guide will demonstrate how to extract a zip file, with subdirectories, using a command line argument example.

Prerequisites

To complete this guide, you will need access to a command-line interface with a Java Development Kit installed.

Step 1 — Download a Zip File With Subdirectories

First, you will download a zip file that contains subdirectories. You can find plenty of zip files online, or you can create your own. For the purposes of this guide, we will download a sample zip file from here. Once the file is downloaded, you should be able to view the subdirectories in the file. On most systems, this will be accomplished through the unzip command.

Step 2 — Compile the Java Program

Now, we will create a Java program that is responsible for extracting the zip file, with subdirectories, from the command line. Create a file called ExtractZipFileWithSubdirectories.java with the following contents: 

          
           
import java.util.*; 
import java.io.*; 
import java.nio.file.*; 

public class ExtractZipFileWithSubdirectories { 
    public static void main (String[] args) throws IOException { 
        Path source = Paths.get(args[0]); 
        Path destination = Paths.get(args[1]); 
        try (ZipInputStream zipInputStream = new ZipInputStream(new FileInputStream(source.toString()))) { 
            ZipEntry entry = zipInputStream.getNextEntry(); 
            while (entry != null) { 
                File file = new File(destination + "/" + entry.getName()); 
                if (!file.exists()) { 
                    Files.createDirectories(file.getParentFile().toPath()); 
                    Files.createFile(file.toPath()); 
                } 
                Files.copy(zipInputStream, file.toPath(), StandardCopyOption.REPLACE_EXISTING); 
                entry = zipInputStream.getNextEntry(); 
            } 
        } 
    } 
}
The ExtractZipFileWithSubdirectories.java file contains the necessary logic for extracting the zip file and its subdirectories. This program takes two command line arguments—the source (the zip file to extract) and the destination (the directory where the zip file and its contents will be extracted). Compile the program with the following command: 

          
           
javac ExtractZipFileWithSubdirectories.java

Step 3 — Run the Java Program

Now, we will execute the ExtractZipFileWithSubdirectories program with the following command: 

          
           
java ExtractZipFileWithSubdirectories <source> <destination></destination>
Replace and with the path to the zip file and the destination directory respectively. For example, if the zip file is at /home/user/sample.zip and the desired destination is /home/user, then the command would be: 

           
            
java ExtractZipFileWithSubdirectories /home/user/sample.zip /home/user
Once the program has been executed, the zip file and its subdirectories will be extracted to the given destination.

Conclusion

In this guide, we have demonstrated how to extract a zip file, with subdirectories, from the command line. We first downloaded a zip file containing subdirectories, then created the ExtractZipFileWithSubdirectories program and compiled it. Finally, we executed the program using the command line argument examples listed above.
