Create Zip File From Multiple Files using ZipOutputStream Example

Zip files are used to store multiple files in a single file and transport it between systems. In this article, we will learn how to create a zip file from multiple files using the ZipOutputStream class in Java.

Creating a Zip File in Java

The ZipOutputStream class is used to write files to a ZIP file. The following example shows how to create a zip file from multiple files in Java. 

          
           
import java.io.File;
import java.io.FileInputStream;
import java.io.FileOutputStream;
import java.io.IOException;
import java.util.zip.ZipEntry;
import java.util.zip.ZipOutputStream;

public class CreateZip {
    public static void main(String[] args) throws IOException {
        // Source folder which you want to zip
        String srcFolder = "C:/Users/ai/Desktop/myfolder";
        // Output ZIP file location
        String outZipFile = "C:/Users/ai/Desktop/myfiles.zip";

        // Create a new ZipOutputStream
        try (ZipOutputStream zos = new ZipOutputStream(new FileOutputStream(outZipFile))) {
            // Get files to be zipped
            File[] files = new File(srcFolder).listFiles();
            for (File file : files) {
                if (!file.isDirectory()) {
                    // Add ZIP entry to output stream
                    zos.putNextEntry(new ZipEntry(file.getName()));
                    try (FileInputStream fis = new FileInputStream(file)) {
                        // Transfer bytes from the file to the ZIP file
                        byte[] buf = new byte[1024];
                        int len;
                        while ((len = fis.read(buf)) > 0) {
                            zos.write(buf, 0, len);
                        }
                    }
                    // Complete the entry
                    zos.closeEntry();
                }
            }
        }
    }
}
In the above example, we use the ZipOutputStream to create a zip file from multiple files. First, we get the list of files from the source folder which we want to zip. Then, for each file, we create a new ZipEntry and write it to the output stream. Finally, we close the entry when complete.

Conclusion

In this article, we learned how to create a zip file from multiple files using the ZipOutputStream class in Java. We also saw an example of how to use the class to zip up multiple files.
