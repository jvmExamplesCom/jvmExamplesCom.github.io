Create Zip File From Directory Recursively Using ZipOutputStream Example

In this article, we are going to see how to create a zip file from a directory recursively using ZipOutputStream and Java. We will also go through an example to demonstrate the usage of this API.

What is ZipOutputStream?

ZipOutputStream is an API that is used to write zip files. This API provides a convenient way to compress files and directories into a single zip file. It can be used to write multiple entries into the same zip file and can also be used to write data into the zip file.

Create Zip File From Directory Recursively

In order to create a zip file from a directory recursively, we need to traverse the directory tree and add each file into the ZipOutputStream. The following code snippet demonstrates how to do this using the ZipOutputStream API: 

          
           
public void createZipFromDir(String sourceDir, String zipFile) throws IOException {
        try (ZipOutputStream out = new ZipOutputStream(new FileOutputStream(zipFile))) {
            Path dirPath = Paths.get(sourceDir);

            Files.walk(dirPath).filter(Files::isRegularFile).forEach(file-> {
                String zipFileName = dirPath.relativize(file).toString();
                try  {
                    ZipEntry zipEntry = new ZipEntry(zipFileName);
                    out.putNextEntry(zipEntry);

                    Files.copy(file, out);
                    out.closeEntry();
                } catch (IOException e) {
                    System.err.println(e.getMessage());
                }
            });
        }
    }
In the above code snippet, we are doing the following:
  • We are creating a ZipOutputStream object with a FileOutputStream.
  • We are creating a Path object from the source directory.
  • We are using the walk() method to traverse the directory tree recursively.
  • We are using the filter() method to filter out only the regular files.
  • For each file, we are creating a ZipEntry object with the file's relative path as the entry name.
  • We are writing the file contents into the ZipEntry using the copy() method.
  • We are closing the ZipEntry after writing the file data.

    }
}
