Compress Byte Array Using Deflater Example

In this tutorial we will look at how to compress a byte array using the Deflater class in Java. We will look at some example code and explain what is happening at each step.

Example Code:


          
           
public static void compress() {
    //create a byte array with our input data.
    byte[] data = getInputData();
 
    // Compress the bytes
    byte[] output = new byte[100];
    Deflater compresser = new Deflater();
    compresser.setInput(data);
    compresser.finish();
    int compressedDataLength = compresser.deflate(output);
 
    // Decompress the bytes
    Inflater decompresser = new Inflater();
    decompresser.setInput(output, 0, compressedDataLength);
    byte[] result = new byte[100];
    int resultLength = decompresser.inflate(result);
    decompresser.end();
 
    // Decode the bytes into a String
    String outputString = new String(result, 0, resultLength, "UTF-8");
    System.out.println(outputString);
}

Explanation of the Code:

The first step is to create a byte array with the 

          
           
getInputData()
method. This is where we store the input data which we want to compress. Next, we use the Deflater class to compress the byte array. We create an instance of the Deflater class, set the input data, then call the 

          
           
finish()
and 

          
           
deflate()
methods. The deflate method takes an output byte array as a parameter and stores the compressed data in it. The compressed data length is returned by this method. We then use the Inflater class to decompress the bytes. We create an instance of the Inflater class, set the input data, then call the 

          
           
inflate()
method. This method also takes an output byte array as a parameter, which is used to store the decompressed data. The length of the decompressed data is returned by this method. Finally, we decode the bytes into a String by creating a 

          
           
String
object with the decompressed byte array, the result length and the charset "UTF-8". The output String is then printed to the console.
