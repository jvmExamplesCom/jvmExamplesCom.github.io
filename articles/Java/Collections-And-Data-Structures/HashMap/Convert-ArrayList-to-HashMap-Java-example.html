Converting ArrayList to HashMap in Java Example

In this article, we will discuss how to convert an ArrayList to a HashMap in Java. The ArrayList class provides a method called toArray() which returns an array containing all of the elements in the list in proper sequence (from first to last element). On the other hand, the HashMap class provides a method called putAll() which takes a map and copies all of its mappings into the target map.

We can leverage these two methods together to convert an ArrayList to a HashMap. Here’s an example code snippet that demonstrates the conversion:


          
           
// create an ArrayList 
ArrayList<String> list = new ArrayList<>(); 
list.add("Apple"); 
list.add("Banana"); 
list.add("Mango"); 

// create a HashMap 
HashMap<Integer, String> map = new HashMap<>(); 

// convert the ArrayList to a HashMap 
map.putAll(list.stream().collect(Collectors.toMap(i -> i, String::valueOf))); 

// print out the contents of the HashMap 
System.out.println(map);
The output of the above code is: 

          
           
{Apple=Apple, Banana=Banana, Mango=Mango}

The code works by leveraging the toArray() method of the ArrayList class and the putAll() method of the HashMap class. First, the toArray() method is used to convert the ArrayList contents to a Java array. Then the putAll() method is used to copy the elements of the array into the HashMap.
