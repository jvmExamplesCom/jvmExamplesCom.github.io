Comparing Two HashMap Objects in Java

HashMap is a popular data structure used in Java for mapping keys to values. It provides constant time performance for insertion, removal and search operations. In this article, we will look at how to compare two HashMap objects in Java. We will go over the different ways to do this, including using the equals() or the compareTo() method, as well as using simple iteration. We will also look at some examples of code and output for each approach.

Using the equals() Method

One way to compare two HashMap objects is by using the equals() method. This method is used for telling whether two maps are equal or not. It will return true if the two maps contain the same keys with the same associated values, false otherwise. The following example shows how to use the equals() method to compare the contents of two HashMap objects: 

          
           
HashMap<Integer, String> map1 = new HashMap<>();
 map1.put(1, "Apple");
 map1.put(2, "Banana");
 map1.put(3, "Orange");

 HashMap<Integer, String> map2 = new HashMap<>();
 map2.put(1, "Apple");
 map2.put(2, "Banana");
 map2.put(3, "Orange");

 boolean result = map1.equals(map2);
 System.out.println("Result: "+result);
This code will print out the following result: 

          
           
Result: true

Using the compareTo() Method

Another way to compare two HashMap objects is by using the compareTo() method. This method compares two maps based on the size of the map and the order of the entries in the maps. If the two maps are the same size, then it will compare the entries in the maps one by one and return 0 if all entries are equal, -1 if the entry in the first map is less than the entry in the second map, or 1 if the entry in the first map is greater than the entry in the second map. For example, the following code uses the compareTo() method to compare two HashMap objects: 

          
           
HashMap<Integer, String> map1 = new HashMap<>();
 map1.put(1, "Apple");
 map1.put(2, "Banana");
 map1.put(3, "Orange");

 HashMap<Integer, String> map2 = new HashMap<>();
 map2.put(1, "Apple");
 map2.put(2, "Banana");
 map2.put(4, "Kiwi");

 int result = map1.compareTo(map2);
 System.out.println("Result: "+result);
This code will print out the following result: 

          
           
Result: -1

Using Simple Iteration

You can also compare two HashMap objects by simply iterating over the maps and comparing the entries one by one. This is the most basic approach and is usually the most straightforward. The following example shows how to use simple iteration to compare two HashMap objects: 

          
           
HashMap<Integer, String> map1 = new HashMap<>();
 map1.put(1, "Apple");
 map1.put(2, "Banana");
 map1.put(3, "Orange");

 HashMap<Integer, String> map2 = new HashMap<>();
 map2.put(1, "Apple");
 map2.put(2, "Banana");
 map2.put(4, "Kiwi");

 boolean result = true;

 for(Map.Entry<Integer, String> entry : map1.entrySet()) {
      if(!map2.containsKey(entry.getKey()) || !map2.get(entry.getKey()).equals(entry.getValue())) {
           result = false;
           break;
        }
 }

 System.out.println("Result: "+result);
This code will print out the following result: 

          
           
Result: false
As you can see, the simple iteration approach is useful for comparing two HashMap objects.
