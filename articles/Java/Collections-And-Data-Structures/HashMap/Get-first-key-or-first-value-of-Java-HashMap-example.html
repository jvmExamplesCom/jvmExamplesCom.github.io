Get First Key or First Value of Java HashMap Example

Java HashMap allows us to store key-value pairs in the data structure. It can be useful for storing information or objects that are related to each other. We can retrieve the first key or the first value from a Java HashMap using some specific methods. In this article, we will look at examples of how to get the first key or first value from a HashMap instance.

Getting the First Key

We can get the first key from a HashMap instance using the entrySet() method. The entrySet() method returns a Set view of the mappings contained in the HashMap. To get the first key, we can use the iterator returned by the entrySet() to get the first element in the set. Below is an example: 

          
           
HashMap<Integer, String> map = new HashMap<>();
 	map.put(1, "one");
 	map.put(2, "two");
 	map.put(3, "three");
 
 	Iterator<Map.Entry<Integer, String>> it = map.entrySet().iterator();
 	System.out.println("First Key : " + it.next().getKey());
In the above example, we first create a HashMap instance and add three entries to it. Then, we get an iterator from the entrySet() method. Finally, we use the iterator’s next() method to get the first key from the HashMap. The output of the above code is: 

          
           
First Key: 1

Getting the First Value

We can also get the first value from a HashMap instance using the values() method. The values() method returns a Collection view of the values contained in the HashMap. To get the first value, we can use the iterator returned by the values() to get the first element in the collection. Below is an example: 

          
           
HashMap<Integer, String> map = new HashMap<>();
 	map.put(1, "one");
 	map.put(2, "two");
 	map.put(3, "three");
 
 	Iterator<String> it = map.values().iterator();
 	System.out.println("First Value : " + it.next());
In the above example, we first create a HashMap instance and add three entries to it. Then, we get an iterator from the values() method. Finally, we use the iterator’s next() method to get the first value from the HashMap. The output of the above code is: 

          
           
First Value : one

Conclusion

In this article, we looked at how to get the first key and first value from a Java HashMap. We used the entrySet() to get the first key and the values() method to get the first value.
