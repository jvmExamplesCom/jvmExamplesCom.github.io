Java HashMap Get Method Example

Using the HashMap class of Java, it is possible to store data in a key/value pair format. The following example will demonstrate how to use the get() method to retrieve a value from a HashMap with a specified key.

Creating the HashMap

In order to use the get() method, it is necessary to first create a HashMap. This can be done using the following syntax: 

          
           
HashMap<String, String> hashmap = new HashMap<>();
This creates an empty HashMap with String as the key and String as the value.

Adding Pairs to the HashMap

Once the HashMap has been created, it is possible to add entries to it. This can be done by using the following syntax: 

          
           
hashmap.put("key1", "value1");
This adds a key/value pair to the HashMap, with key1 mapped to value1. This can be repeated for any number of pairs that need to be added.

Getting a Value from the HashMap

Once all the entries have been added to the HashMap, it is possible to retrieve them by using the get() method. This is done by providing the key associated with the value, which returns the associated value. The following code demonstrates how this is done: 

          
           
String value = hashmap.get("key1");
This returns the value "value1", as this was the value associated with the key "key1" when the entry was added to the HashMap.

Example Usage

The following code is an example of a complete program that creates a HashMap, adds some values to it, and retrieves them using the get() method: 

          
           
import java.util.HashMap;

public class GetExample {
    public static void main(String[] args) {
        //Create the HashMap
        HashMap<String, String> hashmap = new HashMap<>();

        //Add keys and values to the HashMap
        hashmap.put("key1", "value1");
        hashmap.put("key2", "value2");
        hashmap.put("key3", "value3");

        //Get a value from the HashMap
        String value = hashmap.get("key2");

        //Print out the value
        System.out.println(value);
    }
}
This program creates a HashMap, adds three key/value pairs to it, and then retrieves the value associated with key2. Running this program will output the following: 

          
           
value2
