Sorting a HashMap by Keys in Java

Java provides a powerful data structure called a HashMap, which maps keys to values. In this article we will explore how to use the HashMap class to sort keys and values according to specific criteria.

Creating a HashMap

A HashMap can be created with the following code: 

          
           
HashMap<String, Integer> hashMap = new HashMap<>();

This creates an empty HashMap that is able to hold Strings as keys and Integers as values.

Adding Data to a HashMap

Data can be added to the HashMap using the put method. For example, the following code adds 3 key-value pairs to the HashMap: 


          
           
hashMap.put("John", 10);
hashMap.put("Mary", 15);
hashMap.put("David", 20);

Sorting a HashMap by Keys

Once the HashMap has been populated with data, it can be sorted by keys using the TreeMap class. The following code creates a TreeMap from the HashMap and stores the sorted values in a new TreeMap object: 


          
           
TreeMap<String, Integer> treeMap = new TreeMap<>(hashMap);

The TreeMap will sort the keys in alphabetical order, thus giving us a sorted HashMap. To access the sorted values we can use the get method on the TreeMap object: 


          
           
System.out.println(treeMap.get("John")); // prints 10
System.out.println(treeMap.get("Mary")); // prints 15
System.out.println(treeMap.get("David")); // prints 20

Sorting a HashMap by Values

In order to sort a HashMap by values we need to create a custom comparator. We can do this by creating a Comparator object and defining the compare method. The following code creates a Comparator object that compares two Integers and returns the result of the comparison: 


          
           
Comparator<Integer> comparator = (x, y) -> x.compareTo(y);

Once the Comparator object is created, we can use it to sort the HashMap by values. The following code creates a new LinkedHashMap object and stores the sorted values in it: 


          
           
Map<String, Integer> linkedHashMap = 
    hashMap.entrySet()
           .stream()
           .sorted(Map.Entry.comparingByValue(comparator))
           .collect(Collectors.toMap(Map.Entry::getKey, Map.Entry::getValue,
                                     (e1, e2) -> e2, LinkedHashMap::new));

The LinkedHashMap object will contain the sorted values from the HashMap, which can be accessed with the get method: 


          
           
System.out.println(linkedHashMap.get("John")); // prints 10
System.out.println(linkedHashMap.get("Mary")); // prints 15
System.out.println(linkedHashMap.get("David")); // prints 20

Conclusion

In this article we explored how to sort a HashMap by keys and values using the TreeMap and Comparator classes. By using these classes, we can easily sort data stored in a HashMap.
