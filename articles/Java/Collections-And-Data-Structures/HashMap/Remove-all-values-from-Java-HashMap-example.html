Removing All Values from Java HashMap Examples

In this article, we will learn how to remove all values from a Java 

HashMap
example. We'll look at various code examples and the output generated by each. Let's get started!

Using the clear() Method

The easiest way to remove all the values from a 
HashMap
is to use the 
clear()
method. This removes all entries in the map and sets the size of the map to zero. Here's an example. 

          
           
HashMap<String, Integer> myMap = new HashMap<>();
myMap.put("A", 10);
myMap.put("B", 20);
myMap.put("C", 30);
System.out.println(myMap); // Outputs {A=10, B=20, C=30}

myMap.clear();
System.out.println(myMap); // Outputs {}

This approach might be the simplest, but it doesn't provide any control over what values are removed or how they are removed.

Using the remove() Method

If you need more control over which entries are removed from the 

HashMap
, then you can use the 
remove()
method. This method takes a key as an argument, and it removes the mapping of this key from the 
HashMap
. Here's an example: 

          
           
HashMap<String, Integer> myMap = new HashMap<>();
myMap.put("A", 10);
myMap.put("B", 20);
myMap.put("C", 30);
System.out.println(myMap); // Outputs {A=10, B=20, C=30}

myMap.remove("B");
System.out.println(myMap); // Outputs {A=10, C=30}

Using the remove() method, you can selectively remove only certain entries from the 

HashMap
, while leaving other entries untouched.

Using the entrySet() Method

Finally, if you need to iterate through all the entries in the 

HashMap
and remove them one by one, then you can use the 
entrySet()
method. This returns a 
Set
object containing all the entries in the 
HashMap
. Here's an example: 

          
           
HashMap<String, Integer> myMap = new HashMap<>();
myMap.put("A", 10);
myMap.put("B", 20);
myMap.put("C", 30);
System.out.println(myMap); // Outputs {A=10, B=20, C=30}

Set<Map.Entry<String, Integer>> entries = myMap.entrySet();
for (Map.Entry<String, Integer> entry : entries) {
    myMap.remove(entry.getKey());
}
System.out.println(myMap); // Outputs {}
Using this approach, you can iterate through the 
HashMap
and remove all entries one by one.

clear()

remove()

entrySet()
