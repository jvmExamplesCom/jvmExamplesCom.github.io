Iterate Java HashMap Using forEach or for Loop

Java HashMap is a powerful data structure for storing key-value pairs. It has many useful methods for manipulating key-value pairs and offers better performance than an array or linked list. In this article, we will discuss how to iterate over a HashMap using forEach or for loop in Java.

Using forEach()

The forEach() method of the HashMap interface is used to loop through all the key-value pairs in the map. This method can be used with either a Lambda expression or a method reference. The following example shows how to use the forEach() method with a Lambda expression: 

          
           
HashMap<Integer, String> map = new HashMap<>();
map.put(1, "One");
map.put(2, "Two");
map.put(3, "Three");

map.forEach((key, value) -> System.out.println(key + " : " + value));
The output of this example is: 

          
           
1 : One
2 : Two
3 : Three
The forEach() method can also be used with a method reference as shown in the following example: 

          
           
map.forEach(System.out::println);
The output of this example is the same as the previous example.

Using for Loop

The for loop is another way to iterate over a HashMap in Java. The following example shows how to use the for loop to iterate over the elements of a HashMap: 

          
           
for (Map.Entry<Integer, String> entry : map.entrySet()) {
    System.out.println(entry.getKey() + " : " + entry.getValue());
}
The output of this example is the same as the previous two examples.

Conclusion

In this article, we discussed how to iterate over a HashMap using forEach or for loop in Java. We looked at examples of both methods and saw that they produce the same output.
