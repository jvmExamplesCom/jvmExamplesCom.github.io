Delete All Mappings or Clear HashMap Example

Knowing how to clear a HashMap in Java is an important practice for any Java programmer. In this article, we'll show you how to delete all mappings from a HashMap and provide code examples for doing so.

Syntax for Clearing a HashMap in Java

If you're familiar with the syntax for deleting a single mapping from a HashMap in Java, then you'll notice that the syntax to clear all mappings is quite similar. Here's the syntax that you should use to clear all mappings from a HashMap in Java:


          
           
hashMapObject.clear();

The code above will remove all mappings stored in the given HashMap object. Let's take a look at an example.

Example of Clearing a HashMap in Java

In the following example, we'll create a HashMap that stores student names and test scores. We'll also use the 

clear()
method to delete all of the mappings in the HashMap. Here's the code for our example: 

          
           
import java.util.HashMap;

public class ClearHashMap {
  public static void main(String[] args) {
    // Create a HashMap 
    HashMap<String, Integer> studentGrades = new HashMap<>();
    
    // Add mappings to HashMap
    studentGrades.put("John", 80);
    studentGrades.put("Jane", 90);
    studentGrades.put("Adam", 70);
    studentGrades.put("Tom", 85);
    
    // Print size of HashMap 
    System.out.println("HashMap Size before clearing: " + studentGrades.size());
    
    // Clear all mappings of HashMap 
    studentGrades.clear();
    
    // Print size of HashMap 
    System.out.println("HashMap Size after clearing: " + studentGrades.size());
  }
}
And here's the output of our program: 

          
           
HashMap Size before clearing: 4
HashMap Size after clearing: 0

As you can see from the output, the size of our HashMap went from 4 to 0 after we used the 

clear()
method to delete all of the mappings.
