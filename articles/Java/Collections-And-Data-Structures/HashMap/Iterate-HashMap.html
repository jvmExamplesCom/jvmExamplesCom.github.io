Iterating a HashMap in Java

Java's 

          
           
HashMap
class provides a useful structure for storing key-value pairs. It is a frequently used data structure in many applications, and it can sometimes be necessary to iterate through the elements of a 

          
           
HashMap
. In this article, we will describe how to iterate through a 

          
           
HashMap
in Java, including various examples.

Using a For Each Loop to Iterate Through a HashMap

One way to iterate through a 

          
           
HashMap
is to use a 

          
           
for
each loop. We can loop through both the keys and the values separately by using two separate 

          
           
for
each loops. An example of this is shown in the following code: 

          
           
HashMap<String, Integer> map = new HashMap<>();
map.put("one", 1);
map.put("two", 2);

//Loop through keys
for (String key : map.keySet()) {
    System.out.println(key);
}

//Loop through values
for (Integer value : map.values()) {
    System.out.println(value);
}
In this example, we create a 

          
           
HashMap
with two key-value pairs and two 

          
           
for
each loops to iterate through the keys and values.

Using an Iterator to Iterate Through a HashMap

Another way to iterate through a 

          
           
HashMap
is to use an 

          
           
Iterator
. This is done by first obtaining an 

          
           
Iterator
object from the 

          
           
HashMap
, then looping through the elements of the 

          
           
Iterator
. An example of this is shown in the following code: 

          
           
HashMap<String, Integer> map = new HashMap<>();
map.put("one", 1);
map.put("two", 2);

Iterator<Map.Entry<String, Integer>> it = map.entrySet().iterator();
while (it.hasNext()) {
    Map.Entry<String, Integer> pair = it.next();
    System.out.println(pair.getKey() + " = " + pair.getValue());
}
In this example, we create a 

          
           
HashMap
and obtain an 

          
           
Iterator
object from the 

          
           
entrySet()
method. We then loop through the 

          
           
Iterator
, printing out each key-value pair.

Using a Stream to Iterate Through a HashMap

The third method of iterating through a 

          
           
HashMap
is to use a stream. This is done by using the 

          
           
map.entrySet().stream()
method to obtain a stream of 

          
           
Map.Entry
objects and the 

          
           
forEach()
method to iterate through the stream. An example of this is shown in the following code: 

          
           
HashMap<String, Integer> map = new HashMap<>();
map.put("one", 1);
map.put("two", 2);

map.entrySet().stream()
    .forEach(e -> System.out.println(e.getKey() + " = " + e.getValue()));
In this example, we create a 

          
           
HashMap
and obtain a stream of 

          
           
Map.Entry
objects by using the 

          
           
entrySet().stream()
method. We then iterate through the stream, printing out each key-value pair.

Conclusion

In this article, we have described how to iterate through a 

          
           
HashMap
in Java. We have seen three different methods that can be used for this purpose: using a 

          
           
for
each loop, using an 

          
           
Iterator
, or using a stream. The appropriate method to use depends on the situation, but all three are useful techniques for iterating through a 

          
           
HashMap
.
