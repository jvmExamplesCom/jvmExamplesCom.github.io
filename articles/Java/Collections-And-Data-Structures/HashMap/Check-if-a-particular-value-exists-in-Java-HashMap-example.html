Check if a Particular Value Exists in Java HashMap

Java HashMap is a widely used data structure for storing key-value pairs. It offers a number of convenient facilities for searching and manipulating the keys and values within the HashMap. In this article, we will explore how to check if a particular value exists in a Java HashMap.

Using the containsValue() Method

The most straightforward way to check if a particular value exists in a HashMap is by using the 

          
           
containsValue()
method. This method takes a single parameter of type 

          
           
Object
, which is the value to look for in the HashMap. It returns 

          
           
true
if the value is found in the HashMap, and 

          
           
false
otherwise. 

          
           
Map<String, Integer> map = new HashMap<String, Integer>(); 
 map.put("A", 1); 
 map.put("B", 2); 
 map.put("C", 3); 
 
 System.out.println(map.containsValue(2)); // prints true 
 System.out.println(map.containsValue(4)); // prints false

Using the get() Method

Another approach to check if a particular value exists in a HashMap is by using the 

          
           
get()
method. This method takes a single parameter of type 

          
           
Object
, which is the key associated with the value in the HashMap. It returns the value associated with the key if it is found, or 

          
           
null
if it is not found. 

          
           
Map<String, Integer> map = new HashMap<String, Integer>(); 
 map.put("A", 1); 
 map.put("B", 2); 
 map.put("C", 3); 
 
 Integer valA = map.get("A"); 
 Integer valD = map.get("D"); 
 
 System.out.println(valA != null); // prints true 
 System.out.println(valD != null); // prints false

Using the forEach() Method

Finally, another way to check if a particular value exists in a HashMap is by using the 

          
           
forEach()
method. This method takes a 

          
           
BiConsumer
as a parameter, which is a function that takes two arguments, the key and the value, and performs some action on them. In this case, we can use it to search the HashMap for a particular value and break out of the loop when it is found. 

          
           
Map<String, Integer> map = new HashMap<String, Integer>(); 
 map.put("A", 1); 
 map.put("B", 2); 
 map.put("C", 3); 
 
 AtomicBoolean found = new AtomicBoolean(false); 
 
 map.forEach((k, v) -> { 
     if (v == 2) { 
         found.set(true); 
     } 
 }); 
 
 System.out.println(found.get()); // prints true
We have now seen three different ways to check if a particular value exists in a Java HashMap. Each approach has its own advantages and disadvantages, depending on the context.
