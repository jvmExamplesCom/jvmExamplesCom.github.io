Getting HashMap Size in Java

Java provides numerous powerful data structures to easily store and manipulate data. The HashMap is one of the most commonly used data structures in Java, used to store key-value pairs.

How to get the Size of a HashMap

In order to retrieve the size of a HashMap, the size() method can be used. This method returns the number of key-value pairs stored in the map.

Example Usage

Below is an example of how to use the size() method to retrieve the size of a HashMap instance: 

          
           
HashMap<String, String> testMap = new HashMap<>();
testMap.put("Key1", "Value1");
testMap.put("Key2", "Value2");
testMap.put("Key3", "Value3");

// Get the size of the HashMap
int size = testMap.size();
System.out.println("Size of the HashMap: " + size);
This code will print out the following: 

          
           
Size of the HashMap: 3
