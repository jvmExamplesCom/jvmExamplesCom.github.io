Sorting HashMap by Values in Java

When dealing with maps in Java, sorting them according to their values is an often used operation. In this article, we will discuss how to sort a HashMap by its values in Java.

Using TreeMap to Sort the HashMap by Values

One of the ways to sort a HashMap by values is to create a TreeMap from it and then use the descendingMap() method to obtain a reverse ordering view of the keys. Here is an example:


          
           
HashMap<String, Integer> map = new HashMap<>();
  map.put("A", 1);
  map.put("B", 5);
  map.put("C", 3);  
  map.put("D", 8);

// Create a TreeMap from the HashMap
TreeMap<String, Integer> treeMap = new TreeMap<>(map);
// Use descendingMap() to sort the TreeMap by values
Map<String, Integer> sortedMapByValues = treeMap.descendingMap();
The output of the above code is: 

          
           
{D=8, B=5, C=3, A=1}

Using Stream API to Sort HashMap by Values

Another way to sort the HashMap by its values is to use the Java Stream API. With the stream API, you can sort the entries in the map by their values using the Collectors.toMap() method. Here is an example:


          
           
HashMap<String, Integer> map = new HashMap<>();
  map.put("A", 1);
  map.put("B", 5);
  map.put("C", 3);  
  map.put("D", 8);

// Use Collectors.toMap() to sort the HashMap by values
Map<String, Integer> sortedMapByValues = 
  map.entrySet().stream()
    .sorted(Map.Entry.comparingByValue(Comparator.reverseOrder()))
    .collect(Collectors.toMap(
      Map.Entry::getKey, 
      Map.Entry::getValue, 
      (e1, e2) -> e2, 
      LinkedHashMap::new
    ));
The output of the above code is: 

          
           
{D=8, B=5, C=3, A=1}

In this article, we discussed how to sort a HashMap by its values in Java. We looked at two ways to do this using either the TreeMap class or the Java Stream API. Both methods are fairly straightforward and easy to use.
