Java HashMap put Method Example

Java has a 
HashMap
class that implements the 
Map
interface. This class provides an efficient way to store key-value pairs in which keys are unique within the collection. The 
put()
method of this class inserts a new element in the Map, or if an element with that key already exists, it updates the value associated with the key.

Example 1: Basic Usage of Java HashMap put() Method

In the below example, we create an empty 
HashMap
and use the 
put()
method to add two elements. Then, to check that the elements have been added, we use the 
get()
method. 

          
           
import java.util.HashMap;

public class MyClass {
    public static void main(String[] args) {
        HashMap<String, String> map = new HashMap<>();

        // Add elements to the HashMap
        map.put("John", "Smith");
        map.put("Jane", "Doe");
        
        // Print the elements
        System.out.println(map.get("John"));
        System.out.println(map.get("Jane"));
    }
}
The output of the above code will be: 

          
           
Smith
Doe

Example 2: Update Value for an Existing Key

In this example, we create a 
HashMap
, and add three elements. Then, we use the 
put()
method to update the value associated with the existing key 
John
. 

          
           
import java.util.HashMap;

public class MyClass {
    public static void main(String[] args) {
        HashMap<String, String> map = new HashMap<>();

        // Add elements to the HashMap
        map.put("John", "Smith");
        map.put("Jane", "Doe");
        map.put("Jack", "Brown");
        
        // Update value for existing key
        map.put("John", "Watson");
        
        // Print the elements
        System.out.println(map.get("John"));
        System.out.println(map.get("Jane"));
        System.out.println(map.get("Jack"));
    }
}
The output of the above code will be: 

          
           
Watson
Doe
Brown

Example 3: putIfAbsent() Method Vs put() Method

The 
putIfAbsent()
method is similar to the 
put()
method, except that it only adds an element if it does not exist in the Map. In contrast, the 
put()
method adds or updates an element regardless of whether it exists or not. 

          
           
import java.util.HashMap;

public class MyClass {
    public static void main(String[] args) {
        HashMap<String, String> map = new HashMap<>();
        map.put("John", "Smith");

        // Put the new element only if the key does not exist
        map.putIfAbsent("John", "Watson");
        map.putIfAbsent("Jane", "Doe");

        // Print the elements
        System.out.println(map.get("John"));
        System.out.println(map.get("Jane"));
    }
}
The output of the above code will be: 

          
           
Smith
Doe
