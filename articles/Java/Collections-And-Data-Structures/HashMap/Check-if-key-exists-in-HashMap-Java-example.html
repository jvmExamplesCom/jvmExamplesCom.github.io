Check If Key Exists in HashMap Java Example

Java provides many ways to store data in the form of key-value pairs, such as the use of a HashMap. A HashMap is a class that implements the Map interface, and stores key-value pairs. A key is an object that identifies a value, while the value is the actual data stored in the map. In this article we will discuss how to check if a certain key exists in a HashMap using Java examples.

Creating a HashMap

Before we can check if a certain key exists in a HashMap, we must first create the HashMap. We will use the following code to create the HashMap: 

          
           
HashMap<String, Integer> map = new HashMap<>();
map.put("one", 1);
map.put("two", 2);
map.put("three", 3);
This code creates a HashMap with three entries, each with a String key and an Integer value.

Checking If a Key Exists

Now that we have a HashMap created, we can now check if a certain key exists in it. We can use the 

          
           
containsKey()
method to check if a certain key exists in the HashMap. The following code shows an example of how to use this method: 

          
           
boolean checkKeyExists = map.containsKey("two");
The above code checks if the key "two" exists in the HashMap, and will return true if it does. We can also use the 

          
           
containsValue()
method to check if a certain value exists in the HashMap. The following code shows an example of how to use this method: 

          
           
boolean checkValueExists = map.containsValue(3);
The above code checks if the value 3 exists in the HashMap, and will return true if it does.

Conclusion

In this article we discussed how to check if a certain key or value exists in a HashMap using Java examples. We first created a HashMap and then used the 

          
           
containsKey()
and 

          
           
containsValue()
methods to check if a certain key or value exists in the HashMap.
