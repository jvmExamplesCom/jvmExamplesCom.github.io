How to Create a HashMap in Java

The HashMap class is part of the Java Collection Framework and it is an advanced version of the Map interface. It is used to store key-value pairs and is based on the idea of hashing. It offers constant time performance for basic operations like get() and put(). In this tutorial, we will be looking at the various constructors that are available in HashMap. We will be looking at how they work and also examining a few examples of how they can be used.

HashMap Constructors

The HashMap class has four different constructors that we can use. This constructor is used when we want to create a HashMap and add the key-value pairs later. Here's an example of how to use it: 

          
           
//No-args constructor
HashMap<String, Integer> hmap = new HashMap<>;
The second constructor takes a single argument which is a Map object. This constructor creates a new HashMap with the same mappings as the specified map. Here's an example of how to use it: 

          
           
//Constructor with a map
Map<String, Integer> map = new HashMap<String, Integer>();

map.put("one", 1);

HashMap<String, Integer> hmap = new HashMap<String, Integer>(map);

//Constructor with initial capacity and load factor
