Get All Keys of Java HashMap Example

Java HashMap is a powerful data structure for storing mappings of unique keys to values. The HashMap class provides a few useful methods for accessing and manipulating the underlying stored data. One such method is 

          
           
keySet()
, which returns a 

          
           
Set
containing all the keys of the map. In this tutorial, we discuss how to use the 

          
           
keySet()
method to get all the keys of a 

          
           
HashMap
.

Java HashMap keySet() Method Syntax

The full syntax of the 

          
           
keySet()
method is as follows: 

          
           
Set<K> keySet()
It returns a 

          
           
Set
that contains all the keys of the map.

Example of Using keySet()

Let's create a 

          
           
HashMap
with some key-value pairs and see how we can use the 

          
           
keySet()
method to retrieve all the keys. 

          
           
import java.util.HashMap;

public class Main {
public static void main(String[] args) {
    // Create a HashMap with strings as keys and integers as values
    HashMap<String, Integer> map = new HashMap<>();

    // Put some key-value pairs in the map
    map.put("one", 1);
    map.put("two", 2);
    map.put("three", 3);
    map.put("four", 4);
    map.put("five", 5);

    // Use the keySet() method to get all the keys
    Set<String> keys = map.keySet();

    // Print out the keys
    for (String key : keys) {
        System.out.println(key);
    }
}
}
The output of this code is: 

          
           
one
two
three
four
five
As you can see, the 

          
           
keySet()
method returns a 

          
           
Set
containing all the keys in the map.
