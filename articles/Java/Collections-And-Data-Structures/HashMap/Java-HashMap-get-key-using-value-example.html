Java HashMap Get Key Using Value Example

In this tutorial, we will learn how to get a key from Java HashMap using the provided value. We will learn how to use the getKey() method of the Map interface with various examples.

Example 1: Get Key From Value In Java HashMap Using KeySet()

We can use the 
keySet()
method of the HashMap to get the Set of the keys and then we can iterate through the set and check if the given value is equal to the value in the HashMap using the 
get()
method. The following example demonstrates the same. 

          
           
Map<String, String> myMap = new HashMap<>();
myMap.put("A", "Apple");
myMap.put("B", "Banana");
myMap.put("C", "Cat");

String value = "Cat";
String key = null;

Set<String> keys = myMap.keySet();
for(String k : keys) {
    if(value.equals(myMap.get(k))) {
        key = k;
        break;
    }
}

System.out.println("Key for the value '" + value + "' is " + key);
//Output: Key for the value 'Cat' is C

Example 2: Get key from value in Java HashMap using entrySet()

We can also use the 
entrySet()
method of the HashMap to get the Set of the entries and then we can iterate through the set and check if the given value is equal to the value in the HashMap using the 
getValue()
method. The following example demonstrates the same. 

          
           
Map<String, String> myMap = new HashMap<>();
myMap.put("A", "Apple");
myMap.put("B", "Banana");
myMap.put("C", "Cat");

String value = "Apple";
String key = null;

Set<Entry<String, String>> entries = myMap.entrySet();
for(Entry<String, String> entry : entries) {
    if(value.equals(entry.getValue())) {
        key = entry.getKey();
        break;
    }
}

System.out.println("Key for the value '" + value + "' is " + key);
//Output: Key for the value 'Apple' is A

Example 3: Get Key From Value In Java HashMap Using Stream API

We can also use the Stream API of Java 8 to get the key from the value in the HashMap. We can first obtain the 

Stream
from the 
entrySet()
method and use the 
filter()
and 
findFirst()
methods to find the key from the value. 

          
           
Map<String, String> myMap = new HashMap<>();
myMap.put("A", "Apple");
myMap.put("B", "Banana");
myMap.put("C", "Cat");

String value = "Banana";
String key = myMap.entrySet().stream()
                .filter(entry -> value.equals(entry.getValue()))
                .map(Map.Entry::getKey)
                .findFirst()
                .orElse(null);

System.out.println("Key for the value '" + value + "' is " + key);
//Output: Key for the value 'Banana' is B

Example 4: Get Key From value In Java HashMap Using forEach()

We can also use the 
forEach()
method of the HashMap to iterate through each entry and check for the value to find the corresponding key. The following example demonstrates the same. 

          
           
Map<String, String> myMap = new HashMap<>();
myMap.put("A", "Apple");
myMap.put("B", "Banana");
myMap.put("C", "Cat");

String value = "Cat";
String key = null;

myMap.forEach((k, v) -> {
    if(value.equals(v)) {
        key = k;
    }
});

System.out.println("Key for the value '" + value + "' is " + key);
//Output: Key for the value 'Cat' is C
