Simple Java HashMap Example

HashMaps are a great way of storing data in Java. They provide an efficient and fast way of accessing information. In this article, we will look at a simple example of how to use a HashMap in Java.

Creating the HashMap

The first step is to create the HashMap. This is done using the following code:


          
           
Map<String, Integer> myMap = new HashMap<>();

This code creates an empty HashMap with keys that are of type String and values that are of type Integer. We will use this HashMap throughout this article.

Adding Elements to the HashMap

Elements can be added to the HashMap using the 
put()
method. The syntax of this method is as follows: 

          
           
myMap.put(key, value);

This code adds the specified key-value pair to the HashMap. For example, to add the key-value pair 

("John", 10)
, you would use the following code: 

          
           
myMap.put("John", 10);

Retrieving Elements from the HashMap

Elements can be retrieved from the HashMap using the 

get()
method. The syntax of this method is as follows: 

          
           
myMap.get(key);

This code returns the value associated with the specified key. For example, to retrieve the value associated with the key "John", you would use the following code:


          
           
myMap.get("John");

Updating Elements in the HashMap

Elements can be updated in the HashMap using the 
put()
method. The syntax of this method is as follows: 

          
           
myMap.put(key, newValue);

This code updates the value associated with the specified key. For example, to update the value associated with the key "John" to 20, you would use the following code:


          
           
myMap.put("John", 20);

Removing Elements from the HashMap

Elements can be removed from the HashMap using the 

remove()
method. The syntax of this method is as follows: 

          
           
myMap.remove(key);

This code removes the specified key-value pair from the HashMap. For example, to remove the key-value pair 

("John", 10)
, you would use the following code: 

          
           
myMap.remove("John");

Iterating Through the HashMap

The entries in the HashMap can be iterated over using the 

forEach()
method. The syntax of this method is as follows: 

          
           
myMap.forEach((key, value) -> {
   //do something with the key and value
});

This code iterates over each entry in the HashMap, giving you access to the key and value of each entry. For example, to print out each key and value pair in the HashMap you would use the following code:


          
           
myMap.forEach((key, value) -> {
   System.out.println("key: " + key + ", value: " + value);
});

Conclusion

In this article, we looked at a simple example of how to use a HashMap in Java. We saw how to create a HashMap, add elements to it, retrieve elements from it, update elements in it, remove elements from it, and iterate through it. We also saw how to use the various methods associated with a HashMap.
