Convert String or String Array to HashMap

Java provides an easy way to convert String or String array values into a HashMap. This implementation simplifies the codebase, resulting in cleaner and more optimized solutions. In this guide, we'll discuss how to convert String or String array values into a HashMap.

Creating HashMap from String

The simplest implementation of converting a String to a HashMap is using the 

          
           
put()
method. This implementation allows us to freely map our desired key-value pair without any complex string parsing. 

          
           
HashMap<String, String> map = new HashMap<String, String>();
String keyValuePair = "key=value";
String[] splitKeyValuePair = keyValuePair.split("=");
map.put(splitKeyValuePair[0], splitKeyValuePair[1]);
The above code takes in a 

          
           
String
value and parses it into two separate parts: the 

          
           
key
and 

          
           
value
. It then uses the 

          
           
put()
method to add these two values into the HashMap.

Creating HashMap from String Array

Creating a HashMap from a String array is slightly more complicated as compared to creating one using a single String. This is due to the need to iterate over the array and store the values into the HashMap. 

          
           
HashMap<String, String> map = new HashMap<String, String>();
String[] keyValuePairs = {“key1=value1”, “key2=value2”};
for (String keyValuePair : keyValuePairs) { 
  String[] splitKeyValuePair = keyValuePair.split("="); 
  map.put(splitKeyValuePair[0], splitKeyValuePair[1]);
}
In the above code snippet, we first create an array variable containing two 

          
           
key=value
values. We then proceed to loop through it using a 

          
           
for
loop. Inside our loop, we split the 

          
           
key=value
into two parts. Finally, we use the 

          
           
put()
method to add the resulting 

          
           
key
and 

          
           
value
variables into the HashMap.

Conclusion

We have learned how to convert a single String or an array of Strings into a HashMap in Java. This can be accomplished by using the 

          
           
put()
method. We can also iterate over the array and map each individual 

          
           
key=value
pair into the HashMap.
