Writing HashMap to a Text File in Java

Writing a HashMap to a text file requires a few steps. First, the text file needs to be created or opened for writing. Then, the HashMap can be iterated over to write each of its values to the text file. The following code example shows how to do this in Java. 

          
           
// Create or open the text file
FileOutputStream fout = new FileOutputStream("file.txt"); 

// Get a set of the entries
Set set = hashmap.entrySet();

// Get an iterator
Iterator i = set.iterator();

// Loop through the entries
while(i.hasNext()) {
   Map.Entry me = (Map.Entry)i.next();

   // Write the key to the text file
   String key = (String)me.getKey();
   fout.write(key.getBytes());
   fout.write("\n".getBytes());

   // Write the value to the text file
   String value = (String)me.getValue();
   fout.write(value.getBytes());
   fout.write("\n".getBytes());
}

// Close the text file
fout.close();
In the above code example, we first create or open the text file to write to it. We then use the HashMap's entrySet() method to get a set of the entries. An iterator is obtained from this set of entries with the iterator() method. This iterator is then used to loop through all the entries and write their keys and values to the text file. Finally, the text file is closed.

FileOutputStream fout = new FileOutputStream("file.html");

// Get an iterator
Iterator i = set.iterator();

// Loop through the entries
while(i.hasNext()) {
    Map.Entry me = (Map.Entry)i.next();

    // Write the key to the HTML file
    String key = (String)me.getKey();
    fout.write("<pre>".getBytes());

    if(value.length() > 30){
        fout.write("<pre><code>".getBytes());

        fout.write("</code></pre>".getBytes());
    }
    else{
        fout.write("<pre>".getBytes());
        fout.write(value.getBytes());
        fout.write("</pre>".getBytes());
    }
    
}

// Close the HTML file
fout.close();

We then use the HashMap's entrySet() method to get a set of the entries and obtain an iterator for that set.
