Replacing the Value for a Key in Java HashMap

Introduction

HashMaps are an essential feature of Java programming. They are important data structures that are used to store key/value pairs. The keys are used to retrieve associated values, which makes them ideal for storing data efficiently. In this article, we'll look at how to replace the value for a key in Java HashMap.

Example

To demonstrate this, let's take a look at a simple example. Let's say we have the following HashMap: 

          
           
HashMap<String, Integer> numbersMap = new HashMap<>();

numbersMap.put("One", 1);
numbersMap.put("Two", 2);
numbersMap.put("Three", 3);

System.out.println(numbersMap); // prints {One=1, Two=2, Three=3}
Now, if we want to change the value associated with one of these keys, all we need to do is use the 

          
           
.replace()
method. For example, let's say we want to replace the value associated with the 

          
           
"Two"
key. We can do that as follows: 

          
           
numbersMap.replace("Two", 4);

System.out.println(numbersMap); // prints {One=1, Two=4, Three=3}
As you can see, the 

          
           
.replace()
method takes two arguments: the key whose value we want to replace, and the new value that we want to associate with it.

Using the replaceAll() Method

In addition to the 

          
           
.replace()
method, there is also the 

          
           
.replaceAll()
method that can be used to update multiple values at once. This method takes a 

          
           
BiFunction<K,V,V>
object as an argument. This object defines a function that takes the key and value and returns the updated value. Let's take a look at an example. Here, we will use a 

          
           
BiFunction<K,V,V>
to add 1 to each value: 

          
           
numbersMap.replaceAll((key, value) -> value + 1);

System.out.println(numbersMap); // prints {One=2, Two=5, Three=4}
As you can see, the 

          
           
.replaceAll()
method allows us to quickly and easily update multiple values with a single call.

Conclusion

In this article, we looked at how to replace the value for a key in Java HashMap. We saw how to use the 

          
           
.replace()
method to update a single value, and how to use the 

          
           
.replaceAll()
method to update multiple values at once. With these methods, you should have no trouble manipulating the values in your HashMaps.
