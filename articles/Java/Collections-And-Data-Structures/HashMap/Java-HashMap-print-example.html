Java HashMap print example

The following code examples demonstrate how to use the 

          
           
HashMap
class in Java. The code samples print out the contents of a 

          
           
HashMap
, showing what the keys and values are.

Example 1

This example uses the 

          
           
for
loop to iterate over the 

          
           
HashMap
and print out each key and value pair. 

          
           
HashMap<String, String> myMap = new HashMap<String, String>();
myMap.put("key1", "value1");
myMap.put("key2", "value2");

//Print out the contents of the HashMap
for (String key : myMap.keySet()) {
    System.out.println("Key: " + key + ", Value: " + myMap.get(key));
}
