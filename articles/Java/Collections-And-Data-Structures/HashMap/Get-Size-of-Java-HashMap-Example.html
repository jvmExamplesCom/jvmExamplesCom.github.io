Get Size of Java HashMap Example

In this article, we will learn how to get the size of a Java HashMap. We will also provide example code snippets to demonstrate how the size of a HashMap can be determined.

The Size of a Java HashMap

A Java HashMap is an implementation of the Map interface. A HashMap stores data in key-value pairs and has a maximum capacity that must be specified when it is initialized. The size of a HashMap is the number of key-value pairs that are stored in it; this is also known as its length.

Example Code

Below is an example of how to get the size of a HashMap in Java:


          
           
HashMap<String, Integer> map = new HashMap<>();
map.put("one", 1);
map.put("two", 2);

int size = map.size();
System.out.println("Size is " + size);
The output of the above example is: 

          
           
Size is 2

To get the size of the HashMap at any time, you can use the .size() method, which returns an int value representing the size of the HashMap. This method does not take any parameters and does not modify the contents of the HashMap.
