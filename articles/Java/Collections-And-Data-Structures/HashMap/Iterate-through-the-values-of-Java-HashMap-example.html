Iterating Through Values of Java HashMap Example

Java HashMaps are a powerful tool when it comes to data storage and manipulation. Understanding how to iterate through the values stored in a Java HashMap provides great utility. In this article, we will discuss the various methods of iterating through Java HashMap values, demonstrating their use with code examples.

Using for-each Loop

One of the most straightforward ways to iterate through the values of a Java HashMap is using a for-each loop. This loop iterates through a Map's entrySet(), which contains all the key and value pairs stored in the HashMap. This code example demonstrates the use of the for-each loop for iterating through HashMap values.


          
           
HashMap<String, String> hashMap = new HashMap<>();
hashMap.put("Key1", "Value1");
hashMap.put("Key2", "Value2");

for (Map.Entry<String, String> entry : hashMap.entrySet()) {
    System.out.println(entry.getValue());
}

// Output:
// Value1
// Value2

In this example, two values ("Value1" and "Value2") were stored in a HashMap under two different keys ("Key1" and "Key2"). By iterating through the entrySet() with a for-each loop, we can easily access and print out the values stored in the HashMap.

Iterator Class

Another way to iterate through the values of a Java HashMap is by using the Iterator Class. This class provides access to iterator objects which can be used to loop through any collection of objects. This code example demonstrates the use of the Iterator Class for iterating through HashMap values.


          
           
HashMap<String, String> hashMap = new HashMap<>();
hashMap.put("Key1", "Value1");
hashMap.put("Key2", "Value2");

Iterator<String> iterator = hashMap.values().iterator();
while (iterator.hasNext()) {
    System.out.println(iterator.next());
}

// Output:
// Value1
// Value2

This example uses an iterator object to loop through the values stored in the HashMap. An while loop is used to call the hasNext() method on the iterator object. The hasNext() method returns true if the iterator object still has elements to loop through. For each iteration of the loop, the next() method is called on the iterator object to get the value of the current element and then it is printed out.

Java 8 Streams

Java 8 introduced the concept of streams, which provide a new way of iterating through collections of objects. Streams can be used to filter elements and perform various other operations on collections. This code example demonstrates the use of streams to iterate through the values stored in a Java HashMap.


          
           
HashMap<String, String> hashMap = new HashMap<>();
hashMap.put("Key1", "Value1");
hashMap.put("Key2", "Value2");

hashMap.values().stream()
        .forEach(value -> System.out.println(value));

// Output:
// Value1
// Value2

The stream() method is used to acquire a stream from the values stored in the HashMap. The stream is then filtered using the forEach() method. Within the forEach() method, the value of each element is printed out with the println() method.
