Check if value exists in HashMap Java example

Java is a powerful programming language that provides many features to make coding easier. One of these features is the ability to create and use HashMaps. A HashMap is a type of collection that stores key-value pairs, allowing for efficient retrieval of data based on certain keys. This tutorial will discuss how to check if a specific value exists in a HashMap in Java.

Creating a HashMap

Before we can begin checking if a value is present in our HashMap, we need to first create a HashMap. To do this, we will use the following code: 

          
           
Map<String, Integer> map = new HashMap<>();
This code creates a Map object with a String as the key and an Integer as the value. If we want to add entries to this HashMap, we can use the put() method, which takes two parameters—the key and the value—and adds them to the HashMap. For example: 

          
           
map.put("one", 1);
map.put("two", 2);
map.put("three", 3);
This code adds three entries to our HashMap with the keys “one”, “two”, and “three” and the values 1, 2, and 3 respectively.

Checking if value exists in HashMap

Now that we have created our HashMap, we can begin to check if certain values exist within it. To do this, we will use the containsValue() method, which takes one parameter—the value we are looking for—and returns a boolean indicating whether or not the value is present. For example, if we wanted to check if the number 3 is present in our HashMap, we would use the following code: 

          
           
boolean doesContain3 = map.containsValue(3);
This code assigns the result of the containsValue() method (true or false) to the variable doesContain3. If 3 is present in our HashMap, the variable will be assigned the value true. We can then use this variable in an if statement to see if the number 3 is present in the HashMap. 

          
           
if (doesContain3) {
    System.out.println("Number 3 is present!");
} else {
    System.out.println("Number 3 is not present!");
}
This code checks the value of the doesContain3 variable and prints out the appropriate message.

Conclusion

In this tutorial, we discussed how to use the containsValue() method in Java to check if a specific value is present in a HashMap. We also looked at how to create a HashMap and add entries to it. Hopefully, this tutorial has been useful and you now have a better understanding of how to work with HashMaps in Java.
