Java TreeMap forEach example

TreeMap is a class that implements the Map interface, which is used to store elements in key-value pairs, similar to what a HashMap does. The main difference between TreeMap and HashMap is that TreeMap sorts its keys in the order of their natural ordering or based on a Comparator that you can provide to it. This tutorial will show an example of how to use the forEach() method with a TreeMap.

Creating a TreeMap

Before we look at an example of the forEach() method, let's first create a simple TreeMap: 

          
           
// Create TreeMap
TreeMap<String, String> myTreeMap = new TreeMap<>();
 
// Add some items to TreeMap
myTreeMap.put("Key1", "Value1");
myTreeMap.put("Key2", "Value2");
myTreeMap.put("Key3", "Value3");
This code creates a TreeMap called myTreeMap and adds three items to it with the put() method, which takes a key and a value as parameters.

forEach() Method Example

The forEach() method takes a lambda expression as a parameter and executes it for each item in the TreeMap. Here is an example of how you can use it to iterate over all of the items in the TreeMap we just created: 

          
           
// Iterate over all items in TreeMap
myTreeMap.forEach((k,v) -> System.out.println("Key: " + k + " Value: " + v));
This code will iterate over all of the items in the TreeMap and print out their keys and values. The output of this code would be: 

          
           
Key: Key1 Value: Value1
Key: Key2 Value: Value2
Key: Key3 Value: Value3
It is important to note that the forEach() method does not return anything, it simply executes the code inside the lambda expression for each item in the TreeMap.

Conclusion

In this tutorial, we looked at an example of how to use the forEach() method with a TreeMap in Java. We first created a TreeMap and added some items to it, then used the forEach() method to iterate over each item in the TreeMap.
