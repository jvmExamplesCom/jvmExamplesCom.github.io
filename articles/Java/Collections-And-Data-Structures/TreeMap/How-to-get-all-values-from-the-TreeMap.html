How to Get All Values from the TreeMap in Java

The TreeMap is an implementation of the Map interface in Java which stores key-value pairs in a sorted order. This feature makes it very useful for retrieving values based on a specific order. In this article, we will look at how to get all values from a TreeMap in Java and how to use those values to perform operations further down the line.

The Basics of Retrieving Values from the TreeMap

Retrieving values from a TreeMap can be done using the get() method. This method takes the key of the desired value as its parameter and then returns the associated value. Here is an example of this method in action: 

          
           
TreeMap<String, Integer> map = new TreeMap<>();
map.put("a", 1);
map.put("b", 2);

int value = map.get("a"); // value is 1
The above example demonstrates how to use the get() method to retrieve a single value from a TreeMap. However, what if you want to retrieve all of the values from the TreeMap, rather than just one?

Using the Values() Method to Retrieve All Values from the TreeMap

The Values() method of the TreeMap can be used to retrieve all of the values stored in the TreeMap. This method returns a Collection of the values stored in the TreeMap. Here is an example of how this method can be used: 

          
           
TreeMap<String, Integer> map = new TreeMap<>();
map.put("a", 1);
map.put("b", 2);
map.put("c", 3);

Collection<Integer> values = map.values(); // values contains [1, 2, 3]
The above example demonstrates how to use the values() method to retrieve all of the values stored in the TreeMap. However, these values are stored within a Collection, which may not be ideal for some applications.

Using the EntrySet() Method to Retrieve All Values from the TreeMap

An alternative way to retrieve the values stored in a TreeMap is to use the entrySet() method. This method returns a Set of Map.Entry objects, each of which holds a key and its associated value. Here is an example of how this method can be used: 

          
           
TreeMap<String, Integer> map = new TreeMap<>();
map.put("a", 1);
map.put("b", 2);
map.put("c", 3);

Set<Map.Entry<String, Integer>> entries = map.entrySet();

for(Map.Entry<String, Integer> entry : entries) {
    System.out.println("Key: " + entry.getKey() + ", Value: " + entry.getValue());
}
In the above example, we use the entrySet() method to retrieve the set of key-value pairs stored in the TreeMap, and then we loop through each of the entries to print out the key and value associated with it.

Conclusion

In this article, we looked at how to get all values from the TreeMap in Java. We looked at the different methods that can be used to retrieve the values stored in the TreeMap, and how to use those values further down the line.
