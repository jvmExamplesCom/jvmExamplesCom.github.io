How to Convert HashMap to TreeMap in Java

Java offers many options when it comes to converting a HashMap to a TreeMap. This tutorial will provide detailed instructions on how to convert HashMaps to TreeMaps, as well as sample programs illustrating the conversion.

Why Use HashMaps and TreeMaps?

The HashMap and TreeMap classes are two of the Collection classes available in the Java programming language. The primary difference between the two is that TreeMaps use sorted key-value pairs, and HashMaps do not. HashMaps offer better performance when it comes to inserting, updating or deleting elements, while TreeMaps offer better performance when it comes to retrieving elements. In addition, TreeMaps also offer ordering guarantees (i.e. they will always return the same values in the same order).

Converting HashMaps to TreeMaps

In order to convert a HashMap to a TreeMap, we first need to create a new TreeMap and populate it with the elements from the existing HashMap. We can do this using the putAll() method. For example, if we have a HashMap with String keys and Integer values: 

          
           
HashMap<String, Integer> map = new HashMap<>();
map.put("one", 1);
map.put("two", 2);
map.put("three", 3);
We can create a new TreeMap and populate it with the elements from the HashMap like so: 

          
           
TreeMap<String, Integer> sortedMap = new TreeMap<>();
sortedMap.putAll(map);
The sortedMap TreeMap will contain the values from the map HashMap in ascending order. Alternatively, we can also create a TreeMap directly from the HashMap using the constructor: 

          
           
TreeMap<String, Integer> sortedMap = new TreeMap<>(map);
This will also create a TreeMap populated with the values from the map HashMap, in ascending order.

Printing the Keys and Values

Once we have created a TreeMap from the HashMap, we can iterate over the entries and print out the keys and values: 

          
           
for (Map.Entry<String, Integer> entry : sortedMap.entrySet()) {
	System.out.println(entry.getKey() + " : " + entry.getValue());
}
The output of the above code will be: 

          
           
one : 1
two : 2
three : 3
