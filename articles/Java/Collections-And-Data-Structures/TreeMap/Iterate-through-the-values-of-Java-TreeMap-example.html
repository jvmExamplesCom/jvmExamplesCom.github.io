Iterate through the values of Java TreeMap example

The TreeMap in Java is a sorted, keys-based collection that allows for fast retrieval. It implements the Map interface and extends the AbstractMap class. This tutorial will discuss how to iterate through the values of a TreeMap in Java. We will also look at some code examples to illustrate the concepts further.

TreeMap Class

Before we discuss how to iterate through values of a TreeMap, let us first take a look at its class declaration. Here is the complete code for the TreeMap class: 

          
           
public class TreeMap<K,V> extends AbstractMap<K,V> implements NavigableMap<K,V>, Cloneable, Serializable
As you can see, the TreeMap class implements the Map interface and extends the AbstractMap class. This means you can use all the methods of the Map interface in addition to the methods provided by the TreeMap class.

Iterator Through Values in Java TreeMap

To iterate through the values of a TreeMap in Java, we need to first get the keyset of the map using the keySet() method. After that, we can iterate through the values using the get(Object key) method. Here is an example: 

          
           
Map<String, Integer> treeMap = new TreeMap<>();
treeMap.put(“A”, 1);
treeMap.put(“B”, 2);
treeMap.put(“C”, 3);

Set<String> keySet = treeMap.keySet();
for (String key : keySet) {
    int value = treeMap.get(key);
    System.out.println(key + “ - “ + value);
}
In this example, we are creating a TreeMap with String keys and Integer values. Then we get the keyset of the map and iterate through it. Finally, we print out the key and corresponding value. Here is the output of the above example: 

          
           
A - 1
B - 2
C - 3
As you can see, we have successfully iterated through the values of a TreeMap in Java.
