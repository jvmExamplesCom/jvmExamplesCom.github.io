Get Size of Java TreeMap Example

The TreeMap class in Java is a data structure that allows user code to store and access data in the form of key-value pairs. It is widely used in applications when data needs to be sorted and accessed quickly. This article will provide an example of how to get the size of a TreeMap instance in Java.

Code Example


          
           
TreeMap<String, String> treeMap = new TreeMap<>();
treeMap.put("Key1", "Value1");
treeMap.put("Key2", "Value2");
int size = treeMap.size();
System.out.println("TreeMap size is: " + size);
In the above code example, we create a TreeMap instance with two key-value pairs and then use the size() method to get its size. The output of this code example is: 

          
           
TreeMap size is: 2
It is important to note that the size() method will always return an integer that represents the number of elements stored in the TreeMap.
