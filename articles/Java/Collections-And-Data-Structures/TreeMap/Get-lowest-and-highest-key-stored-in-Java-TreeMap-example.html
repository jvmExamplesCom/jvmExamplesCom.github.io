Get lowest and highest key stored in Java TreeMap example

Java TreeMap is a class which implements the Map interface. This class allows storing of key-value pairs and provides various methods to traverse and access the stored data.

The TreeMap class uses a red-black tree as its underlying structure and provides efficient access and retrieval of stored keys and values. It maintains the keys in ascending order, so the lowest and highest key stored in a TreeMap is easy to access.

In this article, we will discuss how to get the lowest and highest key stored in a TreeMap in Java. We will provide an example program to demonstrate this with code snippets.

Example Program

Let’s first create a TreeMap and add some entries to it.


          
           
TreeMap<String, Integer> map = new TreeMap<>();
map.put("Key1", 1);
map.put("Key2", 2);
map.put("Key3", 3);
map.put("Key4", 4);

Now, let’s see how to get the lowest and highest key stored in the TreeMap.

Getting The Lowest Key

The TreeMap class provides a method called firstKey() which returns the lowest key stored in the TreeMap. This method returns the first key stored in the TreeMap, according to its natural ordering.


          
           
String lowestKey = map.firstKey();
System.out.println("Lowest Key: " + lowestKey); // Output: Lowest Key: Key1

Getting The Highest Key

The TreeMap class provides a method called lastKey() which returns the highest key stored in the TreeMap. This method returns the last key stored in the TreeMap, according to its natural ordering.


          
           
String highestKey = map.lastKey();
System.out.println("Highest Key: " + highestKey); // Output: Highest Key: Key4

We can also use the firstEntry() and lastEntry() methods to get the lowest and highest entry stored in the TreeMap. These methods return a Map.Entry object containing the key and value of the corresponding entry.


          
           
Map.Entry<String, Integer> lowestEntry = map.firstEntry();
System.out.println("Lowest Entry: " + lowestEntry); // Output: Lowest Entry: Key1=1
 
Map.Entry<String, Integer> highestEntry = map.lastEntry();
System.out.println("Highest Entry: " + highestEntry); // Output: Highest Entry: Key4=4

That's it! In this article, we discussed how to get the lowest and highest key stored in a TreeMap in Java. We also provided an example program along with code snippets.
