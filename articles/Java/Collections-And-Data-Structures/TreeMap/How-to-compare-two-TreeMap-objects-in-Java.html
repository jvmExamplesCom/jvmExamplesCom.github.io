Comparing Two TreeMap Objects in Java

The TreeMap object is a powerful tool to store and access data. Its ability to automatically sort its keys in ascending order makes it a great alternative to an unordered list. In this article, we'll learn how to compare two TreeMap objects in Java.

Comparing Two TreeMaps Using equals() Method

We can use the 

          
           
equals()
method to compare two TreeMap objects in Java. The method takes a single argument, which is the object we want to compare to. If the argument is not a TreeMap, then 

          
           
false
will be returned. Otherwise, if both TreeMaps have the same size and all key-value pairs are equal, then the result will be 

          
           
true
. The following code example illustrates the usage of 

          
           
equals()
method: 

          
           
TreeMap<Integer,String> map1 = new TreeMap<Integer,String>();
map1.put(1,"One");
map1.put(2,"Two");

TreeMap<Integer,String> map2 = new TreeMap<Integer,String>();
map2.put(1,"One");
map2.put(2,"Two");

System.out.println(map1.equals(map2)); //prints true

Another method to compare two TreeMap objects is by using 

          
           
compareTo()
method. This method takes an argument of type TreeMap and returns an integer which indicates the difference between the two TreeMaps. If the returned value is 0, then both TreeMaps are equal. If the value is positive, then the first TreeMap is greater than the second one. Conversely, if the returned value is negative, then the second TreeMap is greater than the first one. TreeMap<Integer,String> map1 = new TreeMap<Integer,String>();
map1.put(1,"One");
map1.put(2,"Two");

TreeMap<Integer,String> map2 = new TreeMap<Integer,String>();
map2.put(1,"One");
map2.put(3,"Three");
