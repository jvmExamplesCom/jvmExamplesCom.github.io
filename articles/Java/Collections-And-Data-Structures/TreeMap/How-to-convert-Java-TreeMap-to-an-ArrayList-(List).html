How to convert a Java TreeMap to an ArrayList (List)

Java provides several ways to convert a TreeMap to an ArrayList. Here we will discuss each of them in detail and provide examples of how to use them.

Using the Collection Interface

The Collection interface provides a method called toArray() which allows us to convert a TreeMap to an ArrayList by providing an array of objects. The syntax of the method is as follows: 

          
           
public Object[] toArray()
The following code example shows how to use this method to convert a TreeMap to an ArrayList: 

          
           
TreeMap<String, Double> map = new TreeMap<>(); 
map.put("one", 1.0); 
map.put("two", 2.0); 
map.put("three", 3.0); 

Object[] arr = map.values().toArray(); 

ArrayList<Object> list = new ArrayList<>(Arrays.asList(arr)); 

System.out.println(list); // prints [1.0, 2.0, 3.0]
In the above example, we first create a TreeMap with some key-value pairs. We then use the toArray() method to convert the TreeMap's values to an array. This array is then used to create a new ArrayList.

Using Java 8 Streams

Java 8 Streams provide another way to convert a TreeMap to an ArrayList. The syntax of the Stream API's collect() method is as follows: 

          
           
public static <T> Collector<T, ?, List<T>> toList()
The following code example shows how to use this method to convert a TreeMap to an ArrayList: 

          
           
TreeMap<String, Double> map = new TreeMap<>(); 
map.put("one", 1.0); 
map.put("two", 2.0); 
map.put("three", 3.0); 

List<Double> list = map.values().stream() 
              .collect(Collectors.toList()); 

System.out.println(list); // prints [1.0, 2.0, 3.0]
In the above example, we first create a TreeMap with some key-value pairs. We then use the Stream API's collect() method to collect the TreeMap's values into a list. This list is then converted to an ArrayList.

Using An Iterator

An Iterator can also be used to convert a TreeMap to an ArrayList. The syntax of the Iterator's hasNext() method is as follows: 

          
           
public boolean hasNext()
The following code example shows how to use this method to convert a TreeMap to an ArrayList: 

          
           
TreeMap<String, Double> map = new TreeMap<>(); 
map.put("one", 1.0); 
map.put("two", 2.0); 
map.put("three", 3.0); 

Iterator<Double> itr = map.values().iterator(); 

ArrayList<Double> list = new ArrayList<>(); 
while (itr.hasNext()) { 
    list.add(itr.next()); 
} 

System.out.println(list); // prints [1.0, 2.0, 3.0]
In the above example, we first create a TreeMap with some key-value pairs. We then use an Iterator to iterate over the TreeMap's values and add them to a list. This list is then converted to an ArrayList.

Conclusion

In this article, we discussed how to convert a TreeMap to an ArrayList in Java. We saw three different ways to do this: using the Collection interface, using Java 8 Streams, and using an Iterator. Each of these methods has its own advantages and disadvantages, so it's important to consider which one is best for your needs before implementing it.
