How to Get the Size of a TreeMap in Java

TreeMaps are one of the most powerful and useful data structures in Java. They offer efficient key-value storage, making them great for tasks such as storing mappings of user IDs to usernames, or mapping product IDs to objects representing the product. TreeMaps also have some useful methods for finding information about the structure and contents of the map. One such method is size(), which returns the number of key-value pairs stored in the map. In this article, we will explore how to use the size() method in TreeMap, as well as look at other useful options for getting the size of a TreeMap in Java.

Using size()

The easiest way to get the size of a TreeMap is to use its size() method. This method returns an integer representing the number of key-value pairs stored in the TreeMap. Here is an example of how to use this method: 

          
           
TreeMap<String, Integer> map = new TreeMap<String, Integer>();
map.put("One", 1);
map.put("Two", 2);
int size = map.size();  // size is now 2

Using entrySet() and size()

If you don't want to use the size() method directly on your TreeMap, you can also use the entrySet() method to get the same information. The entrySet() method returns a Set of Map.Entry objects, each representing a single key-value pair in the TreeMap. You can then call the size() method on the set: 

          
           
TreeMap<String, Integer> map = new TreeMap<String, Integer>();
map.put("One", 1);
map.put("Two", 2);
Set<Map.Entry<String, Integer>> entries = map.entrySet();
int size = entries.size();  // size is still 2
Using this method is slightly more verbose than using the size() method directly, but it can be useful if you also need to do something with the key-value pairs in the TreeMap.

Using forEach()

You can also use the forEach() method to get the size of a TreeMap. This method takes a BiConsumer as a parameter, and it calls the method for each entry in the TreeMap. Using this method, you can keep a counter variable and increment it for each key-value pair that the forEach() method finds: 

          
           
TreeMap<String, Integer> map = new TreeMap<String, Integer>();
map.put("One", 1);
map.put("Two", 2);
int counter = 0;
map.forEach((key, value) -> {
    counter++;
});
int size = counter;  // size is still 2

Conclusion

In this article, we looked at how to get the size of a TreeMap in Java. We saw that the easiest way to do this is to use the size() method directly on the TreeMap. However, if you need to do more with the key-value pairs in the TreeMap, you can use the entrySet() method, or the forEach() method.
