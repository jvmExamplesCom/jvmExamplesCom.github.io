How to Add Key-Value Mapping to the TreeMap (TreeMap put Method)

The TreeMap put method is used to add key-value mapping to a TreeMap collection. This can be done by passing in a key and a corresponding value, which will then be inserted into the TreeMap.

The following code examples show different ways to use the TreeMap put method for adding key-value mapping to a TreeMap:

Adding a Single Key-Value Pair


          
           
TreeMap<String, Integer> treeMap = new TreeMap<>();

treeMap.put("Key", 1);

Adding Multiple Key-Value Pairs


          
           
TreeMap<String, Integer> treeMap = new TreeMap<>();

treeMap.put("Key1", 1);
treeMap.put("Key2", 2);
treeMap.put("Key3", 3);

Adding a Key-Value Pair with a Replacement Value


          
           
TreeMap<String, Integer> treeMap = new TreeMap<>();

treeMap.put("Key1", 1);
treeMap.put("Key1", 4); // Replaces the existing value of 1 with 4

Adding a Key-Value Pair Using a Comparator


          
           
// Create a custom comparator
Comparator<String> keyComparator = new Comparator<String>() { 
    public int compare(String str1, String str2) { 
        return str1.compareTo(str2); 
    } 
}; 
  
// Create the TreeMap using the comparator
TreeMap<String, Integer> treeMap = new TreeMap<>(keyComparator);

treeMap.put("Key1", 1);

Adding a Key-Value Pair with Return Value


          
           
TreeMap<String, Integer> treeMap = new TreeMap<>();

Integer oldValue = treeMap.put("Key1", 1); // oldValue will now be null

Adding a Key-Value Pair with Optional Return Value


          
           
TreeMap<String, Integer> treeMap = new TreeMap<>();

Integer oldValue = treeMap.putIfAbsent("Key1", 1);

The TreeMap put method can be used to add key-value mapping to a TreeMap collection. In addition to adding a single key-value pair, it is also possible to add multiple key-value pairs, replace an existing value, specify a custom comparator, and get the previous value that was associated with the key.
