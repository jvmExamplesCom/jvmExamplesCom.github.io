Creating a New TreeMap Object in Java

Java offers powerful data structures, such as the 

          
           
TreeMap
, to increase productivity and efficiency. A TreeMap stores key-value pairs and provides an efficient way to look up values based on their keys. In this guide, we will discuss how to create a TreeMap object in Java utilizing its various constructors.

Using the Default Constructor

The most straightforward way of creating a TreeMap object is to use the default constructor. This constructor simply creates an empty TreeMap object that can be populated with key/value pairs later: 

          
           
TreeMap<Integer, String> map = new TreeMap<>();
In this example, we have created an empty TreeMap object that is capable of storing integer keys and string values.

Using the TreeMap Constructor with Custom Comparator

A TreeMap constructor also accepts an instance of the 

          
           
Comparator
interface, allowing you to customize the ordering of the TreeMap. For example, if you wanted to sort the TreeMap entries in descending order, you could pass an anonymous Comparator to the TreeMap constructor: 

          
           
Comparator<Integer> myComp = new Comparator<Integer>() {
   public int compare(Integer a, Integer b) {
     return b - a;
   }
};

TreeMap<Integer, String> map = new TreeMap<>(myComp);

Using the TreeMap Constructor With Existing Map

The TreeMap constructor also takes in a 

          
           
Map
object and builds a TreeMap based on the contents of that map. For example, if you had an existing 

          
           
HashMap
that you wanted to convert to a TreeMap, you could use the following code: 

          
           
HashMap<Integer, String> hashMap = new HashMap<Integer, String>();
// Populate the hashMap with values 

TreeMap<Integer, String> map = new TreeMap<>(hashMap);

Using the TreeMap Constructor With Existing SortedMap

Finally, the TreeMap constructor can also accept an instance of a 

          
           
SortedMap
and build a TreeMap based on its contents. For example, if you had an existing 

          
           
TreeMap
that you wanted to convert to another TreeMap, you could use the following code: 

          
           
TreeMap<Integer, String> treeMap = new TreeMap<>();
// Populate the treeMap with values

TreeMap<Integer, String> map = new TreeMap<>(treeMap);
That's all there is to creating TreeMap objects in Java! With these constructors, you can quickly and easily create and manipulate TreeMaps of any size.
