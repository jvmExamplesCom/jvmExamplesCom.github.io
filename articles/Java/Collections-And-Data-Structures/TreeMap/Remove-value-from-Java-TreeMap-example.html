Remove Value from Java TreeMap Example

Java TreeMap is an implementation of the Map interface that allows for storing data key-value pairs and retrieving the stored values based on the provided keys. The Java TreeMap class implements the Map interface and can store entries in a natural order or a custom order. This article will explain how to remove a value from the TreeMap using the remove() method.

Remove Method Syntax

The syntax for the remove() method is: 

          
           
Object remove(Object key)

Example Usage

In this example, we will create an instance of the TreeMap class, add some key-value pairs, and then remove a value using the remove() method. Firstly, we will declare the TreeMap and add some key-value pairs to it: 

          
           
TreeMap<String, String> treeMap = new TreeMap<>();
treeMap.put("key1", "value1");
treeMap.put("key2", "value2");
treeMap.put("key3", "value3");
Now, we can call the remove() method to remove a value from the TreeMap: 

          
           
treeMap.remove("key2");

Output

When run, the above code will output the following: 

          
           
{key1=value1, key3=value3}
