Get Sub Map from Java TreeMap Example

The TreeMap class in Java is used to store elements in a tree-like structure. The TreeMap class provides various methods to access and traverse a TreeMap, one of these methods is the subMap() method which allows you to get a submap of the TreeMap. In this article, we will discuss how to use the subMap() method to get a subset of a TreeMap.

Java TreeMap subMap() Syntax

The syntax for the subMap() method is as follows: 

          
           
public SortedMap subMap(K startKey, K endKey)
This method takes two arguments, a start key of type K and an end key of type K. It returns a SortedMap which contains key-value pairs in the specified range defined by the startKey and endKey.

Java TreeMap subMap() Example

Let us look at an example to understand how the subMap() method works. 

          
           
import java.util.*;

public class Main {
    public static void main(String[] args) {
        // Create a TreeMap
        TreeMap<Integer, String> map = new TreeMap<Integer, String>();
        
        // Add elements to the map
        map.put(1, "One");
        map.put(2, "Two");
        map.put(3, "Three");
        map.put(4, "Four");
        map.put(5, "Five");
        
        // Get a submap between 2 and 4
        SortedMap<Integer, String> submap = map.subMap(2, 4);
        
        // Print the submap
        System.out.println(submap);
    }
}
In the above example, we created a TreeMap and added some elements to it. Then, we used the subMap() method to get a submap between the keys 2 and 4. Lastly, we printed the resulting submap. The output of the above program will be: 

          
           
{2=Two, 3=Three}
