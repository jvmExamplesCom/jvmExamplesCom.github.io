Check if a particular value exists in Java TreeMap example

Using the TreeMap class of the Java Collections Framework, one can easily check whether a particular value exists in the map or not. In this article, we will discuss how to do that with a code example.

Creating the TreeMap

Before we can look for values in the TreeMap, we must first create the TreeMap object. To do this, we'll use the constructor shown below. This constructor takes in two parameters: a Comparator object, and a boolean value which will determine whether the map should be sorted in ascending or descending order. 

          
           
TreeMap<String, String> treeMap = new TreeMap<>(Comparator.naturalOrder(), false);
In the code above, we've created a TreeMap that is sorted in descending order. For more information on the different sorts of TreeMap constructors, please consult the documentation.

Adding Values to the TreeMap

Once we have our TreeMap object, we can now add values to it. To do this, we'll use the put() method as shown below. 

          
           
treeMap.put("key1", "value1");
treeMap.put("key2", "value2");
treeMap.put("key3", "value3");
The code above will add three entries to our TreeMap, with the keys being "key1", "key2", and "key3", and their corresponding values being "value1", "value2", and "value3".

Checking if a Value Exists in the TreeMap

Now that we have our TreeMap up and running, we're ready to answer the question: How can we check if a particular value exists in the TreeMap? Fortunately, this is quite easy to do. We use the containsValue() method as shown below. 

          
           
boolean containsValue = treeMap.containsValue("value2");
The code above will check if the TreeMap contains the value "value2", and store the result in the boolean variable "containsValue". If the TreeMap does contain the value, then the variable will be set to true, otherwise it will be set to false.

Conclusion

In this article, we discussed how to use the TreeMap class to check if a particular value exists in the map or not. To do this, we first created the TreeMap object using the appropriate constructor, then added values to it using the put() method, and finally used the containsValue() method to check if a particular value existed in the map or not.
