How to Get TreeMap Key, Value or Entry Greater than or Less than the Specified Key

TreeMap is a popular implementation of the Map interface in Java. It stores its elements according to a specific ordering, usually sorted by keys. This makes it easier to locate items with a specific key in the TreeMap. In addition, TreeMap provides you with methods to access the elements that come after or before the specified key. In this article, we will discuss how to use various methods to get TreeMap key, value or entry greater than or less than the specified key.

Getting TreeMap Entry Greater than a Specified Key

You can get the TreeMap entry greater than a specified key using the higherEntry method of the TreeMap class. This method returns a Map.Entry object for the entry whose key is greater than the specified key. The syntax of this method is as follows: 

          
           
Map.Entry<K,V> higherEntry(K key)
It takes the specified key as its parameter and returns the next term in the ascending order. If there is no such entry, then it returns null. For example, consider the following TreeMap: 

          
           
TreeMap<Integer, String> map = new TreeMap<>();
map.put(3, "Three");
map.put(1, "One");
map.put(5, "Five");
map.put(2, "Two");
map.put(4, "Four");
We can get the TreeMap entry greater than the specified key using the following code: 

          
           
Map.Entry<Integer, String> entry = map.higherEntry(3);
System.out.println(entry.getKey() + " - " + entry.getValue());
The output of this program will be: 

          
           
4 - Four
