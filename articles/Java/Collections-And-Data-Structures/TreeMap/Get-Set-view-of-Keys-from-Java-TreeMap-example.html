Java TreeMap Example: Getting Set of Keys

In Java, a TreeMap is an implementation of the Map interface which stores its elements in a tree. It allows you to store and retrieve key-value pairs in a sorted order. One way to access the keys stored in a TreeMap is through the keySet() method. This article walks through an example to demonstrate how to get a set of keys from a TreeMap.

Java TreeMap KeySet() Example

The following example creates a TreeMap which stores a mapping of names and ages. It then uses the keySet() method to get a Set of the keys. 


          
           
import java.util.TreeMap;
import java.util.Set;
 
public class MyTreeMap {
 
    public static void main(String[] args) {
        // create a TreeMap
        TreeMap<String, Integer> map = new TreeMap<>();
 
        // add elements to the map
        map.put("John", 25);
        map.put("Paul", 28);
        map.put("George", 27);
        map.put("Ringo", 32);
 
        // get the keys
        Set<String> keys = map.keySet();
 
        System.out.println(keys);
    }
}

When this code is executed, it prints out the following output:


          
           
[George, John, Paul, Ringo]

As you can see, the keys in the TreeMap are printed out in sorted order. This is because the TreeMap is sorted according to the keys.

The keySet() method returns a Set of the keys stored in the TreeMap. This makes it easy to iterate over all the keys or check for the existence of a specific key. It is also useful for extracting subsets of keys from the map.

Conclusion

In this article, we walked through an example of how to get a set of keys from a Java TreeMap. We saw that the keySet() method returns a set of the keys in the TreeMap, and that they are returned in sorted order. This makes it easy to access the keys stored in a TreeMap.
