Simple Java TreeMap Example

The TreeMap class is an implementation of a map interface and allows the storage of key-value pairs. The TreeMap is sorted according to the natural ordering of its keys or you can use a Comparator if you want your own ordering. It is a red-black tree based NavigableMap implementation. In this tutorial, we will focus on the TreeMap example programs in Java.

TreeMap Constructors

The TreeMap class provides two constructors to create a TreeMap. 

          
           
TreeMap tm = new TreeMap();

Comparator c = new MyComparator();
TreeMap tm = new TreeMap(c);
The first constructor creates a TreeMap that is sorted according to the natural ordering of its keys. The second constructor creates a TreeMap that is sorted according to the specified Comparator.

TreeMap Methods

The TreeMap class provides several methods to add, remove, and query the elements. 

          
           
tm.put(key, value); // adds the given key-value pair
tm.get(key); // returns the value stored for the given key
tm.remove(key); // removes the given key and its associated value
tm.containsKey(key); // returns true if the given key is present in the map
tm.containsValue(value); // returns true if the given value is present in the map
tm.size(); // returns the number of elements in the map
tm.isEmpty(); // returns true if the map is empty
tm.keySet(); // returns a Set view of all the keys in the map
tm.values(); // returns a Collection view of all the values in the map

TreeMap Example

Now letâ€™s look at a simple example to demonstrate the usage of TreeMap in Java. 

          
           
import java.util.TreeMap;

public class TreeMapExample {
    public static void main(String[] args) {
        // Create a TreeMap
        TreeMap<Integer, String> tm = new TreeMap<Integer, String>();
        
        // Add key-value pairs to the TreeMap
        tm.put(1, "One");
        tm.put(2, "Two");
        tm.put(3, "Three");
        tm.put(4, "Four");
        tm.put(5, "Five");
        
        // Print out the TreeMap
        System.out.println(tm); // {1=One, 2=Two, 3=Three, 4=Four, 5=Five}
        
        // Print out the size
        System.out.println("Size: " + tm.size()); // Size: 5
        
        // Get the value for key 3
        System.out.println("Value for key 3: " + tm.get(3)); // Value for key 3: Three
        
        // Remove the key-value pair for key 4
        tm.remove(4);
        
        // Print out the updated TreeMap
        System.out.println(tm); // {1=One, 2=Two, 3=Three, 5=Five}
    }
}
In this example, we created a TreeMap and added some key-value pairs to it. Then we printed out the TreeMap, its size, and the value associated with a given key. Finally, we removed one of the key-value pairs and printed out the resulting TreeMap.
