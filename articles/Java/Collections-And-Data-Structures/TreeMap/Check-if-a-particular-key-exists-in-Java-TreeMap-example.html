Checking if a Particular Key Exists in Java TreeMap Example

A TreeMap is a map implementation that keeps its entries sorted according to the natural ordering of its keys. It is an implementation of a Red-Black tree, and it provides guaranteed log(n) time cost for the containsKey, get, put and remove operations. The structure of the TreeMap is essentially a binary search tree with additional state associated with each node. In this article, we'll look at how to check if a particular key exists in a Java TreeMap example. We'll go over the methods available to do this, as well as walk through some code examples along the way.

containsKey() Method

The most straightforward way to check if a particular key exists in a TreeMap is to use the containsKey() method. This method takes a key as a parameter and returns true if the key is found in the TreeMap. Here's an example of the containsKey() method in action: 

          
           
TreeMap<String, String> map = new TreeMap<>(); 
map.put("key1", "value1"); 
map.put("key2", "value2"); 

if (map.containsKey("key1")) { 
  // do something 
}
In this example, we create a TreeMap and add two items to it. Then we use the containsKey() method to check if the key "key1" is present. If it is, then we can do something.

get() Method

Another way to check if a particular key exists in a TreeMap is to use the get() method. This method takes a key as a parameter and returns the corresponding value if the key is found. Here's an example: 

          
           
map.get("key1");
If the key is found, then this method will return the corresponding value. Otherwise, it will return null.

isEmpty() Method

We can also use the isEmpty() method to check if a TreeMap is empty or not. This method returns true if the TreeMap is empty and false otherwise. Here's an example: 

          
           
if (map.isEmpty()) { 
  // do something 
}
If the TreeMap is empty, then we can do something. Otherwise, we can assume that there are some items in the TreeMap.

size() Method

Finally, we can use the size() method to check how many items are in a TreeMap. This method returns the number of key-value mappings in the TreeMap. Here's an example: 

          
           
int size = map.size(); 
System.out.println("Size: " + size);
This example prints out the size of the TreeMap.
