How to Iterate TreeMap in Java

The TreeMap class is a part of the Java Collections Framework. It is an implementation of a map and uses a red-black tree to store the elements. This allows for fast insertion, deletion and retrieval of elements.

The TreeMap provides an easy way to iterate over its key-value pairs. There are three ways to iterate over a TreeMap: with a for-each loop, using the entrySet() method, and using the keySet() method.

Using a For-Each Loop

The simplest approach to iterating over a TreeMap is using a for-each loop. With this approach, you can get both the keys and the values without having to use any additional methods. The syntax is easy to remember and it is the fastest way to iterate.


          
           
TreeMap<String,Integer> map = new TreeMap<>();

map.put("foo", 1);
map.put("bar", 2);
map.put("baz", 3);

for (Map.Entry<String,Integer> entry : map.entrySet()) {
  System.out.println(entry.getKey() + " : " + entry.getValue());
}
This code will produce the following output: 

          
           
bar : 2
baz : 3
foo : 1

Using the EntrySet() Method

The entrySet() method returns a Set of Map.Entry objects. Each Map.Entry object contains the key and the value for the corresponding entry in the TreeMap. This approach is useful if you need to perform additional processing on each key-value pair.


          
           
TreeMap<String,Integer> map = new TreeMap<>();

map.put("foo", 1);
map.put("bar", 2);
map.put("baz", 3);

for (Map.Entry<String,Integer> entry : map.entrySet()) {
  String key = entry.getKey();
  Integer value = entry.getValue();
  System.out.println(key + " : " + value);
}

This code will produce the same output as the previous example.

Using the KeySet() Method

The keySet() method returns a Set of the keys in the TreeMap. This approach is useful if you only need to access the keys without the associated values.


          
           
TreeMap<String,Integer> map = new TreeMap<>();

map.put("foo", 1);
map.put("bar", 2);
map.put("baz", 3);

for (String key : map.keySet()) {
  System.out.println(key);
}
This code will produce the following output: 

          
           
bar
baz
foo

Iterating over a TreeMap is an easy and efficient task. You can use a for-each loop, the entrySet() method, or the keySet() method to accomplish the task. All three approaches are easy to remember and provide the same result.
