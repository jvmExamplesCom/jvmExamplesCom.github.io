How to check if the key exists in the TreeMap

The TreeMap is a class in the Java library that contains a key and its associated value. The TreeMap class is used to store key-value pairs so it can quickly access stored data by searching for the specific key associated with the value. This makes it an ideal choice for tasks such as finding the value of a given key or checking whether a key exists in the TreeMap. In this article, we will discuss how to check if a key exists in the TreeMap.

Using the containsKey() Method

One way to check if a key exists in the TreeMap is to use the 

containsKey()
method. This method takes a single parameter, which is the key you want to check. If the key exists in the TreeMap, the 
containsKey()
method will return 
true
, otherwise it will return 
false
. A code example of using 
containsKey()
is shown below. 

          
           
TreeMap<Integer, String> map = new TreeMap<>();
map.put(1, "Apple");
map.put(2, "Banana");

if (map.containsKey(1)) {
    System.out.println("Key exists!");
} else {
    System.out.println("Key does not exist!");
}

In the above example, we are creating a TreeMap called 

map
and adding two key-value pairs to it. We then use the 
containsKey()
method to check if the key 
1
exists in the TreeMap. If it does, we print out the message “Key exists!”, otherwise we print out the message “Key does not exist!”.

Using the get() Method

Another way to check if a key exists in the TreeMap is to use the 

get()
method. This method takes a single parameter, which is the key you want to check for. If the key exists in the TreeMap, the 
get()
method will return the value associated with the key. If the key does not exist, the 
get()
method will return 
null
. A code example of using the 
get()
method is shown below. 

          
           
TreeMap<Integer, String> map = new TreeMap<>();
map.put(1, "Apple");
map.put(2, "Banana");

if (map.get(1) != null) {
    System.out.println("Key exists!");
} else {
    System.out.println("Key does not exist!");
}

In the above example, we are creating a TreeMap called 

map
and adding two key-value pairs to it. We then use the 
get()
method to check if the key 
1
exists in the TreeMap. If it does, we print out the message “Key exists!”, otherwise we print out the message “Key does not exist!”.

Using the for-each Loop

Finally, another way to check if a key exists in the TreeMap is to use a for-each loop. A code example of using a for-each loop is shown below.


          
           
TreeMap<Integer, String> map = new TreeMap<>();
map.put(1, "Apple");
map.put(2, "Banana");

boolean found = false;
for (Integer key : map.keySet()) {
    if (key == 1) {
        found = true;
        break;
    }
}

if (found == true) {
    System.out.println("Key exists!");
} else {
    System.out.println("Key does not exist!");
}

In the above example, we are creating a TreeMap called 

map
and adding two key-value pairs to it. We then use a for-each loop to loop through each key in the TreeMap and check if it is equal to the key we are looking for ( 
1
). If it is, we set the 
found
variable to 
true
. After the loop finishes, we can check the value of the 
found
variable to see if the key exists. If it does, we print out the message “Key exists!”, otherwise we print out the message “Key does not exist!”.

In this article, we discussed how to check if a key exists in the TreeMap. We saw three different ways of doing this, using the 

containsKey()
, 
get()
and for-each loop methods.
