Java LinkedList remove the last element

The Java LinkedList class provides a variety of methods for removing elements from the list. One of the most common operations is to delete the last element from the list. This particular operation can be performed in a number of different ways. In this article, we'll explore the various methods that can be used to delete the last element of a Java LinkedList.

Using ListIterator.remove()

The LinkedList class provides a method called ListIterator.remove() which can be used to delete the last element of a list. The following code example shows how the remove() method can be used to delete the last element of a LinkedList: 

          
           
LinkedList<String> list = new LinkedList<>(); 
list.add("Apple"); 
list.add("Banana"); 
list.add("Orange"); 

ListIterator<String> iterator = list.listIterator(list.size());
iterator.remove();

System.out.println(list); // Outputs [Apple, Banana]
In the above example, we create a LinkedList of type String and add three elements to it. We then create a ListIterator to traverse the list in reverse (from the end of the list to the beginning). We then call the ListIterator.remove() method which will delete the element at the current position (which happens to be the last element in the list). Finally, we print the list which now contains only two elements - Apple and Banana.
