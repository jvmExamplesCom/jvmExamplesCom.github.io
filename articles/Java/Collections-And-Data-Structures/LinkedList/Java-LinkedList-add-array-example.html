Java LinkedList add Array Example

Java's LinkedList class provides an easy way to store and manipulate lists of data. The add() method can be used to add items to the list, but it can also be used to add arrays of data to the LinkedList. In this article, we'll look at an example of how to add an array to a LinkedList in Java.

Creating the Array and LinkedList

To begin, let's create a simple array of integers and a LinkedList object to hold the elements of the array:


          
           
int[] array = {1, 2, 3, 4, 5};
LinkedList<Integer> list = new LinkedList<>();

Adding the Array to the LinkedList

Now that we have our array and LinkedList, we can use the add() method to add the contents of the array to the LinkedList. This can be done in one line of code:


          
           
list.add(array);

This will add the elements of the array to the LinkedList in the order they appear in the array. This means that the first element of the array (1) will be the first element of the LinkedList, the second element of the array (2) will be the second element of the LinkedList, and so on.

Verifying the Results

Let's verify that the elements of the array were added correctly by printing out the contents of the LinkedList:


          
           
for (Integer i : list) {
    System.out.println(i);
}
This will produce the following output: 

          
           
1
2
3
4
5

As you can see, the elements of the array were successfully added to the LinkedList.

Conclusion

In this article, we looked at an example of how to add an array to a LinkedList in Java. We created an array and a LinkedList, then used the add() method to add the elements of the array to the list. Finally, we verified that the elements had been added correctly by printing out the contents of the LinkedList.
