Adding Elements to a Java LinkedList Example

The Java LinkedList is an excellent data structure for storing and manipulating linear collections of data. It provides direct access to elements in constant time, as well as several other useful methods for traversing and manipulating the list. This article will walk through adding elements to a LinkedList using the add method.

Using the add Method

Adding elements to a LinkedList is done using the 
add()
method. The add method takes one argument that specifies which element should be added at the end of the list. For example, to add the element 4 to the list, the following code can be used: 

          
           
LinkedList<Integer> list = new LinkedList<>();
list.add(4);

This will add the element 4 to the end of the list, regardless of whether the list is empty or has existing elements. To add an element to the beginning of the list, the 

addFirst()
method can be used: 

          
           
list.addFirst(3);

The addFirst method adds the specified element to the beginning of the list. Similarly, the 

addLast()
method adds the specified element to the end of the list: 

          
           
list.addLast(5);

It is important to note that the add method takes an arbitrary number of arguments, so it is possible to add multiple elements to the list in one line of code. For example, the following code will add the elements 3, 4, and 5 to the end of the list in that order:


          
           
list.add(3, 4, 5);

The add method also has an optional parameter that allows the element to be added at an index other than the end of the list. This is done by passing the index as the second parameter. For example, to add the element 2 at the third index in the list, the following code can be used:


          
           
list.add(2, 2);

Finally, the add method can be used to add an existing list to the current list. For example, if there is a second list containing the elements 6, 7, and 8, these can be added to the existing list using the following code:


          
           
list.addAll(secondList);

Example

Here is an example of adding elements to a LinkedList using the add method:


          
           
LinkedList<Integer> list = new LinkedList<>();
list.add(1);
list.addFirst(0);
list.addLast(2);
list.add(3, 3);
list.addAll(secondList);

//Output:
[ 0, 1, 3, 2, 6, 7, 8 ]

In this example, the list starts out empty with just the element 1. Then the element 0 is added to the beginning of the list using the addFirst method. Next, the element 2 is added to the end of the list using the addLast method. The element 3 is then added at the third index using the add method, and the second list is added to the existing list using the addAll method.

Conclusion

In this article, we looked at how to use the add method to add elements to a Java LinkedList. We saw how to use the add, addFirst, addLast, and addAll methods to add elements to the list. We also saw an example of how to combine these methods to add multiple elements to the list. By using the add method, we can easily add new elements to a LinkedList.
