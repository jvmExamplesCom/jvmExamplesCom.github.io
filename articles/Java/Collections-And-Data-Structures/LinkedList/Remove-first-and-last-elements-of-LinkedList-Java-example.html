Remove first and last elements of LinkedList Java example

Linked Lists are a type of data structure that stores elements in a linear fashion. They are commonly used to store data that needs to be accessed and modified frequently. In this article, we will discuss the process of removing the first and the last elements from a Linked List in Java.

Creating the Linked List

Before we can start removing elements from a Linked List, we need to create one. We can use the Java built-in LinkedList class to do this. Here is an example of how a simple Linked List with 5 elements can be created: 

          
           
LinkedList<integer> list = new LinkedList<>(); 
list.add(11); 
list.add(16); 
list.add(19); 
list.add(24); 
list.add(28);
</integer>

Removing the First and Last Elements

Now that we have our Linked List created, we can begin removing the first and last elements. This can be done by invoking the removeFirst() and removeLast() methods on the LinkedList object. These methods will remove the respective elements from the beginning and the end of the Linked List. Here is an example of how we can use these methods: 

          
           
list.removeFirst(); // remove the first element 
list.removeLast(); // remove the last element
The removeFirst() and removeLast() methods return the element that was removed from the Linked List. If we need this value for further processing, we can save it in a variable. Here is an example of how we can achieve this: 

          
           
// Remove the first element
int removedElement = list.removeFirst(); 

// Remove the last element
removedElement = list.removeLast();

Resulting Linked List

At this point, the resulting Linked List should look like this: 

          
           
[16, 19, 24]
As you can see, the first and last elements have been removed from the original Linked List.

Conclusion

In this article, we have discussed the process of removing the first and last elements from a Linked List in Java. We have also seen how to save the Removed element for further processing, if needed.
