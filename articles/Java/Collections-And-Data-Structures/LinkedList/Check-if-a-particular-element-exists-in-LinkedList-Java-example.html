Check if a Particular Element Exists in LinkedList Java Example

Linked lists are a type of data structure in which each element is linked to the next element in the list. In Java, the LinkedList class is used to create and manipulate linked lists. It is important to note that a LinkedList is different from an array, as it does not have indexes, instead elements are linked together. A LinkedList can contain duplicate elements and allows for null elements.

This tutorial will discuss how to determine if a particular element exists in a Java LinkedList. We will cover how to check if the element exists using both an enhanced loop and a while loop.

Enhanced Loop

The enhanced loop, also referred to as the for-each loop, is a more efficient way of iterating through a sequence like an array or LinkedList. The syntax of the enhanced loop is as follows:


          
           
for (Type variable : sequence) {
    // code block to be executed
}

In this case, the sequence we are looping through is the LinkedList and the type is whatever type of elements are stored in the LinkedList. So, for example, if our LinkedList contains integers, the type would be 

Integer
.

To determine if a particular element exists in the LinkedList, we can use an if statement within the enhanced loop to check if the current element is equal to the element we are looking for. If so, we can set a flag to true, indicating that the element exists in the list. The code below demonstrates how this might be done: 


          
           
boolean exists = false;
// loop through the LinkedList
for (Object obj : list) {
    // check if the current element equals the element we are looking for
    if (obj.equals(element)) {
        exists = true;
        break;  // element was found, so no need to continue looping
    }
}
// check the flag
if (exists) {
    System.out.println("Element exists in list");
} else {
    System.out.println("Element does not exist in list");
}

In the code above, we first declare a boolean flag and set it to false. We then loop through the LinkedList, comparing each element to the element we are looking for. If we find a match, we set the flag to true and break out of the loop. Lastly, we check the flag to determine if the element exists in the list.

While Loop

We can also use a while loop to determine if an element exists in a LinkedList. To do this, we need to use the 

listIterator()
method of the LinkedList class to create an instance of the ListIterator. This iterator allows us to traverse the list in either direction and access the element at the current position. We can use this iterator to check if the element we are looking for is in the list. The syntax for the while loop is as follows: 

          
           
while (condition) {
    // code block to be executed
}

In this case, the condition will be whether or not the iterator has reached the end of the list. To begin, we need to set the iterator to the beginning of the list using the 

listIterator(0)
method. Then, we can use a while loop to check if the element at the current position is equal to the element we are looking for. If so, we set a flag to true and break out of the loop. Otherwise, we move to the next element in the list using the 
next()
method. The code below demonstrates how this might be done: 

          
           
boolean exists = false;
// create an iterator pointing to the beginning of the list
ListIterator<Object> iter = list.listIterator(0);
while (iter.hasNext()) {
    // check if the current element equals the element we are looking for
    if (iter.next().equals(element)) {
        exists = true;
        break;  // element was found, so no need to continue looping
    }
}
// check the flag
if (exists) {
    System.out.println("Element exists in list");
} else {
    System.out.println("Element does not exist in list");
}

In the code above, we first create an iterator and set it to the beginning of the list. We then use a while loop to check if the element at the current position is equal to the element we are looking for. If so, we set the flag to true and break out of the loop. Lastly, we check the flag to determine if the element exists in the list.
