Java LinkedList Print Elements Example

LinkedList is a part of the Java Collections Framework and is a powerful data structure capable of storing objects in memory. LinkedList allows for efficient insertions, removals, and accesses of its members or elements. In this article, we will look at an example of how to print the elements of a LinkedList in Java.

Example 1 - Using Iterator

The most common way to print the elements of a LinkedList is to use a loop with an Iterator. An Iterator is an interface that allows one to traverse a collection or data structure in a certain order. 

          
           
import java.util.*;

public class MyClass {
    public static void main(String[] args) {
        LinkedList<string> myList = new LinkedList<>();
        myList.add("Apple");
        myList.add("Banana");
        myList.add("Cherry");
        myList.add("Durian");
        
        Iterator itr = myList.iterator();
        while (itr.hasNext()) {
            System.out.println(itr.next());
        }
    }
}
</string>
In the above example, we have created a LinkedList of Strings named myList and populated it with four elements. We then used an Iterator to traverse the elements of the LinkedList and printed them to the console. The output of the above code is as follows: 

          
           
Apple
Banana
Cherry
Durian

Example 2 - Using For Each Loop

Another way to iterate through the elements of a LinkedList is to use a for each loop. This method is simpler than using an Iterator and does not require any extra declarations. 

          
           
import java.util.*;

public class MyClass {
    public static void main(String[] args) {
        LinkedList<string> myList = new LinkedList<>();
        myList.add("Apple");
        myList.add("Banana");
        myList.add("Cherry");
        myList.add("Durian");
        
        for (String element : myList) {
            System.out.println(element);
        }
    }
}
</string>
The output of the above code is as follows: 

          
           
Apple
Banana
Cherry
Durian

Example 3 - Using ListIterator

ListIterator is an interface that allows one to traverse the list in both directions. It can also be used to set and get values from the list. 

          
           
import java.util.*;

public class MyClass {
    public static void main(String[] args) {
        LinkedList<string> myList = new LinkedList<>();
        myList.add("Apple");
        myList.add("Banana");
        myList.add("Cherry");
        myList.add("Durian");
        
        ListIterator itr = myList.listIterator();
        while (itr.hasNext()) {
            System.out.println(itr.next());
        }
    }
}
</string>
The output of the above code is as follows: 

          
           
Apple
Banana
Cherry
Durian
In this article, we looked at three different examples of how to print the elements of a LinkedList in Java. We began by using an Iterator to traverse the elements of the LinkedList and print them to the console. We then used a for each loop which does not require any extra declarations. Finally, we looked at how to use ListIterator which allows us to traverse the list in both directions.
