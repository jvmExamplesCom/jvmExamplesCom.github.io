Get First and Last Elements from LinkedList Java Example

LinkedList is a Java-based data structure used to build a list of elements, each element pointing to the next. LinkedLists allow us to access elements in constant time because of the way they are structured. However, when we are working with LinkedLists, sometimes we need to access the first or last elements in the list. In this article, we'll explore how to do this in Java.

Getting the First Element

To get the first element in a LinkedList, we can use the method 

          
           
getFirst()
. This method returns the element at the beginning of the list, or 

          
           
null
if the list is empty. Here is an example: 

          
           
LinkedList<String> list = new LinkedList<>();
list.add("apple");
list.add("banana");
list.add("cherry");

String firstElement = list.getFirst();
System.out.println(firstElement); //prints "apple"

Getting the Last Element

To get the last element in a LinkedList, we can use the method 

          
           
getLast()
. This method returns the element at the end of the list, or 

          
           
null
if the list is empty. Here is an example: 

          
           
LinkedList<String> list = new LinkedList<>();
list.add("apple");
list.add("banana");
list.add("cherry");

String lastElement = list.getLast();
System.out.println(lastElement); //prints "cherry"

Removing the First Element

If we want to remove the first element from a LinkedList, we can use the method 

          
           
removeFirst()
. This method removes the first element from the list and returns it, or 

          
           
null
if the list is empty. Here is an example: 

          
           
LinkedList<String> list = new LinkedList<>();
list.add("apple");
list.add("banana");
list.add("cherry");

String removedElement = list.removeFirst();
System.out.println(removedElement); //prints "apple"
System.out.println(list); //prints ["banana", "cherry"]

Removing the Last Element

To remove the last element from a LinkedList, we can use the method 

          
           
removeLast()
. This method removes the last element from the list and returns it, or 

          
           
null
if the list is empty. Here is an example: 

          
           
LinkedList<String> list = new LinkedList<>();
list.add("apple");
list.add("banana");
list.add("cherry");

String removedElement = list.removeLast();
System.out.println(removedElement); //prints "cherry"
System.out.println(list); //prints ["apple", "banana"]

Conclusion

In this article, we explored how to get and remove the first and last elements from a LinkedList in Java. We saw how to use the methods 

          
           
getFirst()
, 

          
           
getLast()
, 

          
           
removeFirst()
, and 

          
           
removeLast()
to accomplish our task. With these methods, we can access and modify a LinkedList with ease.
