How to sort LinkedList in Java

Sorting a LinkedList of objects in Java is an important concept, and one that can be tricky for novice developers. The most efficient way of accomplishing this task is by using the Collections.sort() method. In this article, we will go over how to use this method to sort a LinkedList of objects and provide several examples of its implementation.

Overview of Collections.sort()

Collections.sort() is a static method used to sort a list of objects. It takes a List as a parameter and uses a Comparator object to determine the sorting order. This method can be used to sort ArrayList, Vector, LinkedList and other List types. In order to use this method, it is necessary to define a Comparator object which will be used to determine the sorting order.

Defining Comparator for Sorting LinkedList of Objects

In order to sort a LinkedList of objects, we must first define a Comparator object. This is an interface which is used to compare two objects and determine which one is greater or lesser. To define a Comparator object, we must override the compare() method and implement our own logic. For example, if we want to sort a LinkedList of integers in ascending order, we can define a Comparator like this: 

          
           
Comparator<Integer> ascOrder = (x, y) -> x - y;
We can also use the Comparator.comparingInt() method to define the Comparator like this: 

          
           
Comparator<Integer> ascOrder = Comparator.comparingInt(x -> x);

Using Collections.sort() to Sort LinkedList

Once the Comparator has been defined, we can use the Collections.sort() method to sort the LinkedList. This method takes a List and a Comparator object as parameters and returns a sorted List. Here is an example of using the Collections.sort() method to sort a LinkedList of integers in ascending order: 

          
           
public List<Integer> sortLinkedListAscending(LinkedList<Integer> list) {
  Comparator<Integer> ascOrder = (x, y) -> x - y;
  Collections.sort(list, ascOrder);
  return list;
}
And here is an example of using the Collections.sort() method to sort a LinkedList of strings in descending order: 

          
           
public List<String> sortLinkedListDescending(LinkedList<String> list) {
  Comparator<String> descOrder = (x, y) -> y.compareTo(x);
  Collections.sort(list, descOrder);
  return list;
}
