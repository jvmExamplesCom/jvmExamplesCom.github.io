Check if Element Exists in Java LinkedList

LinkedList is one of the most popular data structures used in Java. It consists of a linked list of elements, which are easily traversable and can store data in their nodes. This structure allows for efficient memory management and quick access to elements. In this article, we will look at how to check if an element exists in a Java LinkedList.

Looping Through the List

The most common method to check if an element exists in a LinkedList is to loop through the list and compare each node's data with the desired element. We can do this using the following steps: 

          
           
// Create a LinkedList of integers 
LinkedList<integer> list = new LinkedList<>();

// Add some elements to the list 
list.add(1);
list.add(2);
list.add(3);

// Desired element 
int element = 3;

// Loop through the list 
for (int i = 0; i < list.size(); i++) {
    // Check if element matches the current node 
    if (list.get(i) == element) {
        // Element found 
        System.out.println("Element " + element + " exists in the list.");
        break;
    }
}
</integer>
In the above example, we create a LinkedList of integers and add several elements to it. Then, we define an element whose existence we want to check. Finally, we loop through the list, comparing each node’s data with the desired element. If a match is found, we print a message indicating that the element exists in the list.

Using Streams

Another way to check if an element exists in a LinkedList is to use the 

          
           
anyMatch()
method of the 

          
           
Stream
class. This method checks if any elements of the stream match the given predicate. We can use this method as follows: 

          
           
// Create a LinkedList of integers 
LinkedList<integer> list = new LinkedList<>();

// Add some elements to the list 
list.add(1);
list.add(2);
list.add(3);

// Desired element 
int element = 3;

// Check if element exists in list 
boolean exists = list.stream().anyMatch(e -> e == element);

if (exists) {
    // Element found 
    System.out.println("Element " + element + " exists in the list.");
}
</integer>
In this example, we use the same LinkedList as in the previous example. Then, we define the desired element and use the 

          
           
anyMatch()
method to check if any elements in the list match the given predicate. If a match is found, we print a message indicating that the element exists in the list.

Conclusion

In this article, we looked at two different ways to check if an element exists in a Java LinkedList. The first approach was to loop through the list and compare each node’s data with the desired element. The second approach was to use the 

          
           
anyMatch()
method of the 

          
           
Stream
class. Both approaches can be used to efficiently check if an element exists in a LinkedList.
