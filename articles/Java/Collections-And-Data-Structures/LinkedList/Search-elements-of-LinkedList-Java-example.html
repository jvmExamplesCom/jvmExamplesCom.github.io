Search Elements of LinkedList Java Example

A LinkedList is a data structure that stores elements in a linear order. It is composed of nodes, where each node contains a data element, and a pointer to the next node in the list. The LinkedList can be useful for many applications, such as a stack or a queue.

In this example, we will explore how to search for an element in a LinkedList. We will be writing code in Java and displaying the output in HTML.

Example Code

The following code demonstrates a basic search function. It takes a data element as input, and searches through the list of nodes until it finds one with a matching value. If a match is found, it returns the node; otherwise it returns null.


          
           
public static Node search(Object data) {
    Node current = head;
    while (current != null) {
        if (current.data.equals(data)) {
            return current;
        }
        current = current.next;
    }
    return null;
}

This code can be used to search for the node which contains a specific value. For example, if we wanted to search for the node which contains the value "apples", we could call the search function like so:


          
           
Node node = search("apples");

If a node with the value "apples" exists in the list, it will be returned; otherwise it will return null.

Example Output

The output of searching for a node with a specific value would be either the node which contained that value, or null if no such node exists. For example, let's say our list has three nodes, with values "apples", "oranges", and "bananas". If we search for the value "oranges", the result would be the node which contained that value:

System.out.println(node); // prints null
