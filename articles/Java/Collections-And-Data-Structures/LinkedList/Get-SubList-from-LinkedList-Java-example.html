Retrieving a SubList from a LinkedList in Java

LinkedList in Java provides a powerful data structure to store objects and search for items quickly. Since LinkedList is an implementation of the List interface, it provides several methods to retrieve objects from the list. One such useful approach is to obtain a sublist from an existing LinkedList. This article aims to provide a comprehensive guide on how to retrieve a sublist from a LinkedList in Java.

Understanding the LinkedList Class

LinkedList is a linear data structure that stores elements in nodes linked together. Elements can be added or removed quickly using a few methods provided by the class. The LinkedList class implements the List and Deque interfaces, which provides several methods to add, retrieve and remove elements from the linked list.

Retrieving a SubList from a LinkedList in Java

In Java, obtaining a sublist from a LinkedList is done using the 


          
           
subList()
method. This method returns a view of the portion of this list between the specified fromIndex, inclusive, and toIndex, exclusive. In other words, it creates a sublist from an existing list given the start index and end index of the original list. It should be noted that the subList returned by this method is backed by the original list. Therefore, any modifications made to the subList will also be reflected in the original list.

Example

Let us consider a simple example to understand how to use the 


          
           
subList()
method to obtain a sublist from a LinkedList. We will create a LinkedList and fill it with some sample elements. Then, we will use the 

          
           
subList()
method to obtain the sublist of the original LinkedList. 

          
           
// Create a LinkedList and fill it with some sample elements 
LinkedList<String> list = new LinkedList<String>(); 
list.add("apple"); 
list.add("orange"); 
list.add("banana"); 
list.add("strawberry"); 
list.add("pineapple");

// Retrieve the sublist from the original list 
// This will return a list containing elements from index 1 to 3 
List<String> subList = list.subList(1, 4); 

// Print the sublist 
System.out.println(subList);
The output of the above program is: 

          
           
[orange, banana, strawberry]
As seen from the above example, the 

          
           
subList()
method takes two parameters â€” the first parameter is the starting index (inclusive) of the original list, and the second parameter is the ending index (exclusive) of the original list.

Conclusion

In this article, we looked at how to retrieve a sublist from a LinkedList in Java. The 


          
           
subList()
method provided by the LinkedList class helps to easily and quickly retrieve a sublist from an existing list. It should be noted that the sublist returned by this method is backed by the original list, so any changes made to the sublist will be reflected in the original list as well.
