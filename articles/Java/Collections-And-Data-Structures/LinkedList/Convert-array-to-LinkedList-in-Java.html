Convert Array to LinkedList in Java

Java provides a number of different ways to convert an array to a LinkedList. The most common way is using the addAll() method of the LinkedList class. This method takes an array as an argument, and then adds each element of the array to the LinkedList.

Using addAll() for Array to LinkedList Conversion

To use the addAll() method, our first step is to create an array: 

          
           
int[] arr = {1, 2, 3, 4, 5};
Next, we will create an empty LinkedList: 

          
           
LinkedList<Integer> ll = new LinkedList<>();
Finally, we can use the addAll() method to convert our array to a LinkedList: 

          
           
ll.addAll(Arrays.asList(arr));
Our LinkedList now contains the elements of our array:

5
This is the simplest and most straightforward way to convert an array to a LinkedList in Java.

Using Iteration for Array to LinkedList Conversion

We can also use iteration to convert an array to a LinkedList. This approach requires us to loop through each element of the array, and add it to the LinkedList. Let's start by creating an array: 

          
           
int[] arr = {1, 2, 3, 4, 5};
Next, we need to create an empty LinkedList: 

          
           
LinkedList<Integer> ll = new LinkedList<>();
Now, we can iterate over the array elements, and add them to the LinkedList: 

          
           
for (int i : arr) {
    ll.add(i);
 }

Using iteration is a bit more involved than the addAll() approach, but it may be necessary if you need to add additional logic when converting your array to a LinkedList.

Using Streams for Array to LinkedList Conversion

Java 8 introduces the concept of streams, which can be used to simplify the conversion of an array to a LinkedList. Let's begin by creating an array: 

          
           
int[] arr = {1, 2, 3, 4, 5};
Next, we will create an empty LinkedList: 

          
           
LinkedList<Integer> ll = new LinkedList<>();
Now, we can use the stream() method to convert the array to a LinkedList: 

          
           
ll = Arrays.stream(arr).collect(Collectors.toCollection(LinkedList::new));

Conclusion
