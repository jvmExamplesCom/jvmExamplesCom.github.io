Creating an Object Array from Elements of a LinkedList in Java

Java allows users to create a variety of objects, including arrays and linked lists. In this article, we'll discuss how to create an array object that contains the contents of a linked list. We'll be discussing two types of data structures: linked lists and arrays. Linked lists are made up of nodes that hold a piece of data and a pointer to the next node in the list. Arrays are collections of elements that occupy a set amount of memory. Let's start by creating a linked list in Java. We'll use the following Node class, which holds a piece of data and a reference to the next node in the list. 

          
           
class Node {
  int data;
  Node next;

  public Node(int d) {
    data = d;
  }
}
Now we can create a linked list. The following code creates a linked list with three elements: 

          
           
Node head = new Node(1);
Node second = new Node(2);
Node third = new Node(3);

head.next = second;
second.next = third;
Now that we have a linked list, let's see how to create an object array from its elements. We'll use the 

          
           
toArray()
method of the LinkedList class provided by Java. This method takes an array of type Object and returns an array containing the elements of the linked list. 

          
           
Object[] array = list.toArray();
This code creates an array of type Object that contains the elements of the linked list. The array will be the same size as the linked list, and the elements will be in the same order. For example, if the linked list contains three elements, the array will contain three elements. Let's try it out with our example linked list. The following code creates a linked list, then uses the 

          
           
toArray()
method to create an array of type Object that contains the list's elements. 

          
           
Node head = new Node(1);
Node second = new Node(2);
Node third = new Node(3);

head.next = second;
second.next = third;

LinkedList list = new LinkedList(head);
Object[] array = list.toArray();
The array 

          
           
array

for (Object element : array)

toArray()

Integer[] integerArray = (Integer[]) list.toArray(new Integer[list.size()]);
