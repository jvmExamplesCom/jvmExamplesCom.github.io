Java LinkedList replace Elements

LinkedLists are an important data structure in Java. They are used to store and manipulate data in a linear fashion. In this tutorial, we will go over how to replace elements within a Java LinkedList.

Replacing Elements in a LinkedList

In order to replace elements in a LinkedList, you must first know the index of the element that you want to replace. To get the index of an element, use the indexOf() method. Once you know the index of the element you want to replace, use the set() method to replace it. The set() method takes two parameters: the index of the element you want to replace and the replacement value. 

          
           
int index = linkedList.indexOf(elementToReplace);
linkedList.set(index, replacementElement);
For example, if we have a LinkedList of Strings and we want to replace the element “Two” with “Three”, our code would look like this: 

          
           
import java.util.LinkedList;

public class Test {
    public static void main(String[] args) {
        LinkedList<String> linkedList = new LinkedList<>();
        linkedList.add("One");
        linkedList.add("Two");
        linkedList.add("Four");

        int index = linkedList.indexOf("Two");
        linkedList.set(index, "Three");
        System.out.println(linkedList);
    }
}
This code will print out: [One, Three, Four].

Replacing Multiple Elements in a LinkedList

If you want to replace multiple elements in a LinkedList, you can use the replaceAll() method. The replaceAll() method takes two parameters: a filter for identifying elements to replace, and the replacement element. 

          
           
linkedList.replaceAll(elementToReplace, replacementElement);
For example, if we have a LinkedList of Strings and we want to replace all instances of “Two” with “Three”, our code would look like this: 

          
           
import java.util.LinkedList;

public class Test {
    public static void main(String[] args) {
        LinkedList<String> linkedList = new LinkedList<>();
        linkedList.add("One");
        linkedList.add("Two");
        linkedList.add("Two");
        linkedList.add("Four");

        linkedList.replaceAll("Two", "Three");
        System.out.println(linkedList);
    }
}
This code will print out: [One, Three, Three, Four].

Replacing a Range of Elements in a LinkedList

If you want to replace a range of elements in a LinkedList, you can use the replaceRange() method. The replaceRange() method takes three parameters: the start index of the range (inclusive), the end index of the range (exclusive), and the replacement element. 

          
           
linkedList.replaceRange(startIndex, endIndex, replacementElement);
For example, if we have a LinkedList of Strings and we want to replace the elements at positions 1 and 2 with “Three”, our code would look like this: 

          
           
import java.util.LinkedList;

public class Test {
    public static void main(String[] args) {
        LinkedList<String> linkedList = new LinkedList<>();
        linkedList.add("One");
        linkedList.add("Two");
        linkedList.add("Two");
        linkedList.add("Four");

        linkedList.replaceRange(1, 3, "Three");
        System.out.println(linkedList);
    }
}
This code will print out: [One, Three, Four].

Conclusion

In this tutorial, we went over how to replace elements in a Java LinkedList. We saw how to use the indexOf() method, the set() method, the replaceAll() method, and the replaceRange() method to replace elements in a LinkedList.
