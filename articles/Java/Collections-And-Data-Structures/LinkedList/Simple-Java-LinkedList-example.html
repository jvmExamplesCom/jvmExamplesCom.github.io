Simple Java LinkedList example

This article will provide a simple example of implementing a LinkedList in a Java program. A LinkedList is a particular data structure which stores elements in a list, such as an array but with additional functionality. This example will explain how to add elements to a LinkedList, remove them, and iterate over all elements in the LinkedList.

In order to create a LinkedList, the LinkedList constructor needs to be called. The constructor will take the generic type of the elements that the LinkedList will contain. This example will contain integers so the type should be Integer. 

          
           
LinkedList<integer> list = new LinkedList<>();

Adding elements to a LinkedList

To add elements to a LinkedList, the add method can be called. The add method takes the element to be added as an argument. This example will add the number 5 and 6 to the LinkedList. 

          
           
list.add(5);

Removing elements from a LinkedList

list.remove(list.size() - 1);
