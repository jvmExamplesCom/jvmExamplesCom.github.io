Java LinkedList Get Size Example (LinkedList Length)

LinkedList is an interface of the Java Collection Framework, which has methods to find its size. The getSize() method returns the number of the elements present in the list. In this article, we will discuss how to obtain the size of a LinkedList using the getSize() method and other means.

Using Java getSize() Method

The LinkedList class provides the getSize() method, which is used to find the length of a LinkedList, i.e., the number of elements it contains. This method returns an integer value representing the size of the LinkedList. Here is an example demonstrating the usage of the getSize() method: 

          
           
LinkedList<String> list = new LinkedList<>();
list.add("hello"); 
list.add("world");
int size = list.getSize();
System.out.println("Size of LinkedList : " + size); 
// Output: Size of LinkedList : 2

Using Java size() Method

The LinkedList class also provides the size()method, which can be used to find its size. Similarly, this method also returns an integer value representing the size of the list. Here is an example: 

          
           
LinkedList<String> list = new LinkedList<>();
list.add("hello"); 
list.add("world");
int size = list.size();
System.out.println("Size of LinkedList : " + size); 
// Output: Size of LinkedList : 2

Using Iterator Interface

The Linked List also provides the Iterator interface, which can be used to iterate over its elements one at a time. We can use a while loop with the iterator() method of the LinkedList to find its size. Here is an example: 

          
           
LinkedList<String> list = new LinkedList<>();
list.add("hello"); 
list.add("world");
Iterator<String> litr = list.iterator(); 
int count = 0; 
while (litr.hasNext()) {
    litr.next(); 
    count++; 
}
System.out.println("Size of LinkedList : " + count);

The LinkedList class also provides a toArray() method, which is used to return an array containing all of its elements. We can use this array and its length property to find the size of the LinkedList. Here is an example demonstrating the same:
