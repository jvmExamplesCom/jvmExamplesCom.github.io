Getting Elements from a Java LinkedList

LinkedLists are one of the most frequently used implementations of the List interface in Java. Knowing how to navigate and access elements from a LinkedList is essential for any code utilizing them. In this article, we'll go through some examples of how to get elements from a LinkedList in Java.

Using the ‘get’ Method

The simplest way to get an element from a LinkedList is by using the 

          
           
get
method: 

          
           
LinkedList<Integer> linkedList = new LinkedList<Integer>();
linkedList.add(1); // add the value 1 to the list
int firstElement = linkedList.get(0); // Save first element to the variable
This method takes the given index and returns the element at that position in the list.

Using an Iterator

If you don't know the exact position of the elements in the list, you can use an Iterator to access them: 

          
           
LinkedList<String> linkedList = new LinkedList<String>();
linkedList.add("Java");
linkedList.add("Kotlin");
linkedList.add("Python");

Iterator<String> iterator = linkedList.iterator();
while (iterator.hasNext()) {
    String language = iterator.next(); // Get value of current element
    System.out.println(language);
}
This approach allows you to loop through the elements of the list and access each of them.

Using the ‘forEach’ Method

If you want to loop through the elements of the list, but you don't need the iterator object itself, you can use the 

          
           
forEach
method: 

          
           
LinkedList<Float> linkedList = new LinkedList<Float>();
linkedList.add(1.5f);
linkedList.add(2.3f);
linkedList.add(3.7f);

linkedList.forEach(number -> System.out.println(number));
This way, the code becomes shorter and more readable.

Using the ‘stream’ Method

The last way of accessing elements from a LinkedList is by using the 

          
           
stream()
method: 

          
           
LinkedList<Character> linkedList = new LinkedList<Character>();
linkedList.add('A');
linkedList.add('B');
linkedList.add('C');

linkedList.stream()
    .forEach(letter -> System.out.println(letter));
This approach is similar to the 

          
           
forEach
method, but it's more efficient, since it uses streams.

Conclusion

In this article, we've gone through some examples of how to get elements from a LinkedList in Java. We've seen three different ways to do it: using the 

          
           
get
method, using an Iterator or using the 

          
           
forEach
or 

          
           
stream
method.
