Iterating a LinkedList in Reverse Direction (Backward) in Java

Java LinkedList is an implementation of the doubly linked list data structure. Iteration over a LinkedList can be done in both directions, from left to right and from right to left. This article will focus on how to iterate a LinkedList in reverse direction (backward) using Java.

Using for loop to iterate LinkedList Backward

The simplest way to iterate a LinkedList in the backward direction is to use a for loop. To do so, we need to get the size of the LinkedList and then loop until the index of the element reaches 0. In each iteration, we will access the element at the current index and perform our operations. 

          
           
//Create a LinkedList
LinkedList<String> linkedList = new LinkedList<>();

//Add elements to the LinkedList
linkedList.add("Element 1");
linkedList.add("Element 2");
linkedList.add("Element 3");

//Get the size of the LinkedList
int length = linkedList.size();

//Iterate over the LinkedList in reverse direction
for(int i = length-1; i >= 0; i--) {
    String element = linkedList.get(i);
    System.out.println(element);
}
The output of this code will be: 

          
           
Element 3
Element 2
Element 1

Using ListIterator to iterate LinkedList Backward

The ListIterator interface provides methods to iterate the elements of a LinkedList in both the forward and the backward direction. To iterate the LinkedList backward, we need to first get the ListIterator object and then call the previous() method until the iterator has reached the beginning of the LinkedList. 

          
           
//Create a LinkedList
LinkedList<String> linkedList = new LinkedList<>();

//Add elements to the LinkedList
linkedList.add("Element 1");
linkedList.add("Element 2");
linkedList.add("Element 3");

//Get the ListIterator
ListIterator<String> iterator = linkedList.listIterator();

//Iterate over the LinkedList in reverse direction
while(iterator.hasPrevious()) {
    String element = iterator.previous();
    System.out.println(element);
}
The output of this code will be: 

          
           
Element 3
Element 2
Element 1

Using Stream API to iterate LinkedList Backward

Java 8 introduced the Stream API which allows us to process data in a declarative manner. We can also use the Stream API for iterating over a LinkedList in the backward direction. To do so, we need to use the Stream API to create a stream from the LinkedList and then use the skip() and limit() methods to iterate from the last element of the LinkedList to the first one. 

          
           
//Create a LinkedList
LinkedList<String> linkedList = new LinkedList<>();

//Add elements to the LinkedList
linkedList.add("Element 1");
linkedList.add("Element 2");
linkedList.add("Element 3");

//Get the stream from the LinkedList
Stream<String> stream = linkedList.stream();

//Iterate over the LinkedList in reverse direction
stream.skip(linkedList.size()-1).limit(linkedList.size())
        .forEach(e -> System.out.println(e));
The output of this code will be: 

          
           
Element 3
Element 2
Element 1
