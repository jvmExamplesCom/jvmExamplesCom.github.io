Adding an element to the beginning of a Java LinkedList

LinkedLists are a type of data structure that is widely used in computer programming. They are incredibly useful due to their ability to store and manipulate data in an efficient manner. LinkedList<String> myList = new LinkedList<>();
myList.addFirst("Hello");
