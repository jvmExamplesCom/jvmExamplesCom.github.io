How to clear Java LinkedList

LinkedList is one of the most common data structures used in Java, and it can be used to store data in a linear fashion. Clearing a LinkedList is an important operation, and is useful when you want to empty the list before adding new elements. In this article, we will explore the different ways to clear a LinkedList in Java.

Using clear()

The easiest and simplest way to clear a LinkedList is to use the clear() method provided by the List interface. This method removes all elements from the list, leaving it empty. The syntax for using this method is as follows: 

          
           
list.clear();
For example, 

          
           
LinkedList<Integer> numbers = new LinkedList<>();
numbers.add(1);
numbers.add(2);
numbers.add(3);

// Clear the list
numbers.clear();

System.out.println(numbers);   // []
As you can see, the list is now empty.

Using removeAll()

Another way to clear a LinkedList is by using the removeAll() method provided by the List interface. This method takes another list as an argument and removes all elements from the list that are present in the passed list. The syntax for using this method is as follows: 

          
           
list.removeAll(otherList);
For example, 

          
           
LinkedList<Integer> numbers = new LinkedList<>();
numbers.add(1);
numbers.add(2);
numbers.add(3);

// Create a list of elements to be removed
LinkedList<Integer> removeElements = new LinkedList<>();
removeElements.add(1);
removeElements.add(3);

// Remove elements
numbers.removeAll(removeElements);

System.out.println(numbers);   // [2]
As you can see, the list now contains only the element 2.

Using loop

Finally, you can also clear a LinkedList using a loop. You can use a while or for loop to loop over the elements in the list, and use the remove() method to remove them one by one. The syntax for using this method is as follows: 

          
           
while (list.size() > 0) {
    list.remove();
}
For example, 

          
           
LinkedList<Integer> numbers = new LinkedList<>();
numbers.add(1);
numbers.add(2);
numbers.add(3);

// Use while loop to loop over elements
while (numbers.size() > 0) {
    numbers.remove();
}

System.out.println(numbers);   // []
As you can see, the list is now empty.

Conclusion

In this article, we discussed the different ways to clear a LinkedList in Java. We explored the clear(), removeAll(), and looping approaches. Each approach has its own strengths and weaknesses, so it is important to understand the different approaches before choosing the best one for your project.
