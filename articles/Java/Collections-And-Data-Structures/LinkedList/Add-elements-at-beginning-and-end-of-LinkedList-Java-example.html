Add Elements at Beginning and End of LinkedList Java Example

LinkedLists are an essential part of the Java programming language. The ability to add and remove elements from the beginning and end of a LinkedList provides developers with considerable flexibility when managing collections of data. This article will provide a comprehensive guide to adding elements at the beginning and end of LinkedLists in the Java programming language.

Adding Elements at the Beginning

The API of the java.util.LinkedList class provides an addFirst() method that can be used to add an element at the beginning of a LinkedList. It takes one parameter, which is the element to add. Below is an example using the addFirst() method: 

          
           
LinkedList<String> list = new LinkedList<>();
list.addFirst("Foo");
list.addFirst("Bar");
System.out.println(list); // Prints "[Bar, Foo]"
In this example, two strings have been added to the beginning of a LinkedList using the addFirst() method. The resulting LinkedList contains the elements in the same order they were added. Alternatively, the LinkedList class provides an add() method that can be used to add an element at the beginning of a LinkedList. It takes two parameters, the first is the index at which the element should be added, and the second is the element to add. Below is an example using the add() method: 

          
           
LinkedList<String> list = new LinkedList<>();
list.add(0, "Foo");
list.add(0, "Bar");
System.out.println(list); // Prints "[Bar, Foo]"
In this example, two strings have been added to the beginning of a LinkedList using the add() method. The resulting LinkedList contains the elements in the same order they were added.

Adding Elements at the End

The API of the java.util.LinkedList class also provides an addLast() method that can be used to add an element at the end of a LinkedList. It takes one parameter, which is the element to add. Below is an example using the addLast() method: 

          
           
LinkedList<String> list = new LinkedList<>();
list.addLast("Foo");
list.addLast("Bar");
System.out.println(list); // Prints "[Foo, Bar]"
In this example, two strings have been added to the end of a LinkedList using the addLast() method. The resulting LinkedList contains the elements in the same order they were added. Alternatively, the LinkedList class provides an add() method that can be used to add an element at the end of a LinkedList. It takes two parameters, the first is the index at which the element should be added, and the second is the element to add. Below is an example using the add() method: 

          
           
LinkedList<String> list = new LinkedList<>();
list.add(list.size(), "Foo");
list.add(list.size(), "Bar");
System.out.println(list); // Prints "[Foo, Bar]"
In this example, two strings have been added to the end of a LinkedList using the add() method. The resulting LinkedList contains the elements in the same order they were added.

Conclusion

Adding elements to the beginning and end of a LinkedList in the Java programming language is a straightforward task that can be accomplished using either the addFirst(), addLast(), or add() methods provided by the java.util.LinkedList API. This article has provided several examples demonstrating these three methods and showed that the resulting LinkedLists contain the elements in the same order they were added.
