Removing all Elements or Clearing a LinkedList Java Example

LinkedList is a collection that is used to store elements in a linked list structure. It is one of the most common data structures used in Java, and it can be used to store any type of object.

In this tutorial, we will learn how to remove all elements from a LinkedList or 'clear' it using different methods.

Using clear() Method

The clear() method removes all the elements from the list. It does not return any value.


          
           
LinkedList<String> list = new LinkedList<String>();
   list.add("Java");
   list.add("Python");
   list.add("C++");

   // clears the list
   list.clear();

Using removeAll() Method

The removeAll() method removes all of the elements from this list that are also contained in the specified collection, as determined by the equals() method.


          
           
LinkedList<String> list = new LinkedList<String>();
   list.add("Java");
   list.add("Python");
   list.add("C++");
   ArrayList<String> alist = new ArrayList<String>();
   alist.add("Java");

   // clears the list
   list.removeAll(alist);

Using removeIf() Method

The removeIf() method removes all of the elements of this list that satisfy the given predicate.


          
           
LinkedList<String> list = new LinkedList<String>();
   list.add("Java");
   list.add("Python");
   list.add("C++");

   // clears the list
   list.removeIf(str->str.length() <= 3);

Using Iterator

We can iterate over the elements of the LinkedList and remove them one by one using remove() method of the Iterator.


          
           
Iterator itr = list.iterator();
 while (itr.hasNext()) {
      itr.next();
      itr.remove();
 }

Using RetainAll() Method

The retainAll() method retains only those elements of the list which appear in the specified collection, as determined by the equals() method.


          
           
LinkedList<String> list = new LinkedList<String>();
   list.add("Java");
   list.add("Python");
   list.add("C++");
   ArrayList<String> alist = new ArrayList<String>();

   // clears the list
   list.retainAll(alist);
