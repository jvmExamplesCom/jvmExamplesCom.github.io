Insert an Element into LinkedList Java Example

In this article, we will be discussing the topic of inserting new elements into a LinkedList in Java. A LinkedList is one of the most commonly used data structures in programming because of its flexibility and efficiency. In this article, we'll explore how to insert new elements into a LinkedList and provide some code examples to demonstrate it. So let's get started!

LinkedList and Inserting Elements

A LinkedList is a linear data structure that stores its elements in a node-based structure rather than an array-style structure. Each element in the LinkedList is stored in a node, which contains data and a reference to the next node in the list. Inserting new elements into a LinkedList is relatively straightforward. We must first create a node that contains the data we want to insert and then link it to the appropriate node in the LinkedList.

Inserting a New Element at the Beginning of a LinkedList

To insert a new element at the beginning of a LinkedList, we must first create a node that contains the data we want to insert and assign it to a variable. We can then set the node's 'next' reference to the head of the LinkedList, as shown below:


          
           
Node node = new Node(data);
node.next = head;
head = node;

We then assign the new node to the 'head' of the LinkedList and the new element is inserted at the beginning of the list.

Inserting a New Element at the End of a LinkedList

To insert a new element at the end of a LinkedList, we must first create a node that contains the data we want to insert and assign it to a variable. We then iterate through the list to find the last node in the LinkedList and set the node's 'next' reference to the new node, as shown below:


          
           
Node node = new Node(data);
Node temp = head;
while (temp.next != null) {
    temp = temp.next;
}
temp.next = node;

We then set the last node's 'next' reference to the new node and the new element is inserted at the end of the list.

Inserting a New Element at a Specific Position

To insert a new element at a specific position in a LinkedList, we must first create a node that contains the data we want to insert and assign it to a variable. We then iterate through the list until we find the node with the desired position and set the nodes 'next' reference to the new node, as shown below:


          
           
Node node = new Node(data);
Node temp = head;
int position = 3;
while (position > 0) {
    temp = temp.next;
    position--;
}
node.next = temp.next;
temp.next = node;

We then set the node's 'next' reference to the new node and the new element is inserted at the desired position.
