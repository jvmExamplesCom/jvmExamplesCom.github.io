Get First & Last Elements of LinkedList in Java

LinkedList is a commonly used data structure in Java. It is a doubly linked list implementation of the List and Deque interfaces. It has many useful methods, including those related to accessing the first and last elements. In this article, we'll take a look at these methods and how they can be used.

getFirst() & getLast() Methods

The getFirst() and getLast() methods are used to retrieve the first and last elements of a LinkedList respectively. These methods do not remove the elements from the list, they simply return references to the elements.

Example Code

Here's an example of how to use the getFirst() and getLast() methods: 

          
           
LinkedList<String> list = new LinkedList<>();
list.add("Element 1");
list.add("Element 2");
list.add("Element 3");

String firstElement = list.getFirst(); // returns "Element 1"
String lastElement = list.getLast(); // returns "Element 3"
The code above creates a String LinkedList with three elements and stores references to the first and last elements in variables.

Using peekFirst() & peekLast()

The getFirst() and getLast() methods both have equivalent methods that do not throw exceptions if the list is empty: peekFirst() and peekLast(). These methods will return null if the list is empty, otherwise they behave the same as their counterparts. 

          
           
LinkedList<String> list = new LinkedList<>();
String firstElement = list.peekFirst(); // returns null
