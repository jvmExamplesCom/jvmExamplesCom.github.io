Convert List to Set in Java

List
to 
Set

Using Collection.toSet() method

The 

          
           
java.util.Collection
interface provides an easy way to convert a 
List
to a 
Set
. The 
toSet()
method of the 
Collection
interface returns a 
Set
containing the elements of the specified 
List
. 

          
           
List<String> list = new ArrayList<String>(); 
list.add("A"); 
list.add("B"); 
list.add("C"); 

Set<String> set = list.stream().collect(Collectors.toSet());

Using HashSet and for-each loop

We can also convert a

Set
