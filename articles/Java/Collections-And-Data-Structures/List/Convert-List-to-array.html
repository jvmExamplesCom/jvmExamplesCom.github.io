Convert List to Array in Java

Java provides numerous ways of converting a List to an Array. The most straightforward way to do this is by using the 

          
           
toArray()
method provided by the 

          
           
java.util.List
interface. In this article, we will discuss some of the different ways to convert List to Array in Java, with examples and code snippets.

Using the toArray() Method

This is a simple yet effective approach to converting a List to an array. The 

          
           
toArray()
method takes an argument which is used to specify the type of array that should be returned. The types can be primitive data types or objects. Here is an example: 

          
           
List<String> list = new ArrayList<>();
list.add("A");
list.add("B");
list.add("C");

String[] array = list.toArray(new String[list.size()]);
In the above example, we create a 

          
           
List
of strings, add some elements to it, and then convert it to an 

          
           
Array
. The 

          
           
toArray()
method takes a 

          
           
String
array as an argument which is used to specify the type of array to be returned.

Using the Stream API

The Java 8 Stream API is another popular way to convert a 

          
           
List
to an array. This approach allows us to convert a 

          
           
List
containing objects of a specified type to an array of the same type. Here is an example: 

          
           
List<String> list = new ArrayList<>();
list.add("A");
list.add("B");
list.add("C");

String[] array = list.stream().toArray(String[]::new);
In the above example, we are using the 

          
           
stream()
method to convert the 

          
           
List
to a 

          
           
Stream
, and then using the 

          
           
toArray()
method to convert the 

          
           
Stream
to an array. The 

          
           
toArray()
method takes a 

          
           
java.util.function.IntFunction
as an argument which is used to specify the type of array to be returned.

Using the forEach() Method

The 

          
           
forEach()
method of the 

          
           
java.util.List
interface is another useful way to convert a 

          
           
List
to an array. This approach is similar to the 

          
           
toArray()
method, but it does not require us to specify the type of array to be returned. Here is an example: 

          
           
List<String> list = new ArrayList<>();
list.add("A");
list.add("B");
list.add("C");

String[] array = new String[list.size()];
list.forEach(i -> array[i] = list.get(i));
In the above example, we are creating an empty array of strings, and then using the 

          
           
forEach()
method to iterate through the 

          
           
List
and add each element to the array.

Using the Arrays Class

The 

          
           
java.util.Arrays
class provides a static 

          
           
asList()
method which allows us to convert an array to a 

          
           
List
. We can use this method to convert a 

          
           
List
to an array as well. Here is an example: 

          
           
List<String> list = new ArrayList<>();
list.add("A");
list.add("B");
list.add("C");

String[] array = Arrays.asList(list.toArray()).toArray(new String[0]);
In the above example, we are first converting the 

          
           
List
to an array using the 

          
           
toArray()
method, and then using the 

          
           
Arrays.asList()
method to convert the array to a 

          
           
List
, and finally using the 

          
           
toArray()
method to convert the 

          
           
List
to an array.

Using the Collections Class

The 

          
           
java.util.Collections
class provides several static methods which allow us to convert a 

          
           
List
to an array. These methods allow us to specify the type of array to be returned. Here are a few examples: 

          
           
List<String> list = new ArrayList<>();
list.add("A");
list.add("B");

String[] array1 = (String[]) list.toArray();

java.util.Collections

List

List

List
