Convert String to Character Array and Character Array to String in Java

String is an array of characters in Java, it can be converted to Character array by using the 

          
           
toCharArray()
method. The reverse process of converting a Character array to String can be done by using the 

          
           
String(char[] array)
constructor of the String class. In this article, we will discuss both the methods for conversion of String to Character array, and character array to String in Java.

Converting String to Character Array in Java

We can use the 

          
           
toCharArray()
method of the String class to convert a String object to a Character array in Java. The method has no arguments to pass and returns an array of Character. The general syntax of the 

          
           
toCharArray()
method looks like this: 

          
           
public char[] toCharArray()
Let's see its example. Consider the following statement: 

          
           
String str = "Hello World";
The following statement will create a Character array that contains the characters of the String object: 

          
           
char[] charArray = str.toCharArray();
The output of the above statement will be: 

          
           
['H', 'e', 'l', 'l', 'o', ' ', 'W', 'o', 'r', 'l', 'd']

Converting Character Array to String in Java

We can use the 

          
           
String(char[] array)
constructor of the String class to convert a Character array to String. The constructor takes an array of Character as argument and returns a String representation of the character array. The general syntax of the constructor looks like this: 

          
           
public String (char[] array)
Let's take the same example that we used before. The following statement will create a String object from the Character array: 

          
           
String str = new String(charArray);
The output of the above statement will be: 

          
           
Hello World

toCharArray()
method of the String class to convert a String to a Character array, and the second method uses the 

          
           
String(char[] array)
