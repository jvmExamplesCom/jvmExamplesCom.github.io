Create List from Java Object Array Example

Java provides us a great way to create lists from arrays of objects, with just one single line of code. This can be especially helpful for developers who are just starting out, as it is easy to understand and follow. In this article, we will go over how to create a List from an array of objects in Java.

Creating a List from an Array of Objects in Java

To create a List from an array of objects in Java, you simply need to call the `Arrays.asList` method, passing in the array of objects as an argument. This method will return a list of the objects in the array, which can then be manipulated as needed. Here is a simple example to demonstrate how this works: 

          
           
String[] stringArray = new String[]{"one", "two", "three"};
List<String> strList = Arrays.asList(stringArray);
In this example, we create an array of strings called `stringArray`, and then pass this array into the `Arrays.asList` method to create a list of strings called `strList`. We can then iterate over this list or manipulate it as needed.

Using the List Created from an Array of Objects in Java

Once we have created a List from an array of objects in Java, we can then use this list in the same way that we would use any other list. For example, we can iterate over the list using a traditional `for` loop, like so: 

          
           
for (String str : strList) {
    System.out.println(str);
}
In this example, we are looping through the list of strings we created from the array of strings, and printing out each string to the console. We can also add or remove items from the list, as well as access specific items in the list directly. Here are a few examples of how this can be done: 

          
           
// Add an item to the list
strList.add("four");

// Remove first item from the list
strList.remove(0);

// Get the third item from the list
String third = strList.get(2);
Here, we are adding an item to the list, removing the first item from the list, and then getting the third item from the list and storing it in a string.

Conclusion

In this article, we went over how to create a List from an array of objects in Java. We saw how to use the `Arrays.asList` method to easily create a list from an array of objects, and then how to manipulate this list using standard Java techniques. With this knowledge, you will now be able to easily create and manipulate lists of objects in Java.
