Convert array to LinkedHashSet

A LinkedHashSet is an ordered version of a HashSet. It preserves the insertion order of elements and offers fast retrieval of elements. It can be used to create a set from an array of objects. In this article, we will discuss how to convert an array to a LinkedHashSet in Java.

Example 1: Basic Conversion

In this example, we will convert an array of Strings to a LinkedHashSet. We will use the addAll() method of the LinkedHashSet class to achieve this.


          
           
String[] arr = {"one", "two", "three"}; 
  
//Creating a LinkedHashSet 
LinkedHashSet<String> lh_set = new LinkedHashSet<String>(); 
  
//Adding elements to the LinkedHashSet 
lh_set.addAll(Arrays.asList(arr)); 
  
System.out.println("LinkedHashSet: "+lh_set);
Output: 

          
           
LinkedHashSet: [one, two, three]

Example 2: Using Stream API To Convert Array to LinkedHashSet

In this example, we will use the streams API to convert an array to a LinkedHashSet. We will use the collect() method of the Stream class to achieve this.


          
           
String[] arr = {"one", "two", "three"}; 
  
//Creating a LinkedHashSet using streams API 
LinkedHashSet<String> lh_set = Arrays.stream(arr).collect(Collectors.toCollection(LinkedHashSet::new)); 
  
System.out.println("LinkedHashSet: "+lh_set);
Output: 

          
           
LinkedHashSet: [one, two, three]

Example 3: Creating Custom Objects

In this example, we will convert an array of custom objects to a LinkedHashSet. We will define our own class and override equals() and hashCode() methods.


          
           
//Defining a custom class 
class Student{ 
    int id; 
    String name; 
    // Parameterized constructor 
    Student(int id, String name){ 
        this.id = id; 
        this.name = name; 
    } 
    // Overriding equals() and hashcode() 
    @Override
    public boolean equals(Object object){ 
        if (object instanceof Student) { 
            Student student = (Student) object; 
            if (this.id==student.id &amp;&amp; this.name==student.name)  
                return true; 
        } 
        return false; 
    } 

    @Override
    public int hashCode(){ 
        return id*name.length(); 
    } 
} 

//Creating an array 
Student[] arr = {new Student(1, "Ann"), new Student(2, "John")}; 
  
//Creating a LinkedHashSet 
LinkedHashSet<Student> lh_set = new LinkedHashSet<Student>(); 
  
//Adding elements to the LinkedHashSet 
lh_set.addAll(Arrays.asList(arr)); 
  
System.out.println("LinkedHashSet: "+lh_set);
Output: 

          
           
LinkedHashSet: [Student{id=1, name='Ann'}, Student{id=2, name='John'}]
