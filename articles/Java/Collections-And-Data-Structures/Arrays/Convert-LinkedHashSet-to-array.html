Converting LinkedHashSet to Array

Java provides a rich set of Collections classes and APIs which allow developers to quickly and easily store, manage and retrieve data. The LinkedHashSet class is one of the most commonly used collections classes and is designed to be used to store data in a hash table. LinkedHashSet stores elements in insertion order and offers fast access to elements by their index. One of the useful features of LinkedHashSet is that it can be converted to an array. This can be done using the toArray() method. This method takes a single parameter, which is the type of array to be created, and returns an array of the specified type.

Example 1 – Converting LinkedHashSet to String Array

In this example, we will convert a LinkedHashSet containing strings to an array of strings: 

          
           
LinkedHashSet<String> names = new LinkedHashSet<>();
names.add("John");
names.add("David");

String[] namesArray = names.toArray(new String[names.size()]);

// Prints [John, David]
System.out.println(Arrays.toString(namesArray));
In the above example, we first create a LinkedHashSet containing two strings. We then call the toArray() method, passing in a new String array of the same size as the LinkedHashSet. This will convert the LinkedHashSet to an array of strings. We then use Arrays.toString() to print out the contents of the array.

Example 2 – Converting LinkedHashSet to Integer Array

In this example, we will convert a LinkedHashSet containing integers to an array of integers: 

          
           
LinkedHashSet<Integer> numbers = new LinkedHashSet<>();
numbers.add(1);
numbers.add(2);

Integer[] numbersArray = numbers.toArray(new Integer[numbers.size()]);

// Prints [1, 2]
System.out.println(Arrays.toString(numbersArray));
In the above example, we first create a LinkedHashSet containing two integers. We then call the toArray() method, passing in a new Integer array of the same size as the LinkedHashSet. This will convert the LinkedHashSet to an array of integers. We then use Arrays.toString() to print out the contents of the array.

Example 3 – Converting LinkedHashSet to Object Array

In this example, we will convert a LinkedHashSet containing objects to an array of objects: 

          
           
LinkedHashSet<Object> objects = new LinkedHashSet<>();
objects.add("Hello");
objects.add(42);

Object[] objectsArray = objects.toArray(new Object[objects.size()]);

// Prints [Hello, 42]
System.out.println(Arrays.toString(objectsArray));
In the above example, we first create a LinkedHashSet containing two objects — a string and an integer. We then call the toArray() method, passing in a new Object array of the same size as the LinkedHashSet. This will convert the LinkedHashSet to an array of objects. We then use Arrays.toString() to print out the contents of the array.
