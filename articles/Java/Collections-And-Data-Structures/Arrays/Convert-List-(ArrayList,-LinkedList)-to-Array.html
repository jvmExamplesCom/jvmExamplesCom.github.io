Converting a List to an Array in Java

Java provides the developers with a powerful and versatile platform to write efficient code. The ability to quickly and efficiently convert different types of data structures can enable developers to code faster and more efficiently. In this article, we will look at how to convert a List (ArrayList or LinkedList) to an Array in Java.

Using Stream API to Convert List to Array

One of the most effective ways to convert a List to an array is to use the Stream API in Java. This is done by using the toArray() method associated with the Stream API. The following code example demonstrates how to use the toArray() method to convert a List to an array in Java: 

          
           
// Create list
List<String> list = new ArrayList<>();
list.add("Java");
list.add("Programming");
list.add("Language");

// Convert list to array
String[] array = list.stream().toArray(String[]::new);

// Print array
System.out.println(Arrays.toString(array));
The output of the above code is as follows:

Using Arrays Class to Convert List to Array

Another method to convert a List to an array is to use the Arrays class in Java. This is done by using the toArray() method provided by the Arrays class. The following code example demonstrates how to use the toArray() method to convert a List to an array in Java: 

          
           
// Create list
List<String> list = new ArrayList<>();
list.add("Java");
list.add("Programming");
list.add("Language");

// Convert list to array
String[] array = list.toArray(new String[list.size()]);

// Print array
System.out.println(Arrays.toString(array));
The output of the above code is as follows: 

          
           
[Java, Programming, Language]

Using for-each Loop to Convert List to Array

The last method to convert a List to an array is to use the for-each loop in Java. This is done by creating a new array of the same size as the List, and then copying the elements of the List into the array. The following code example demonstrates how to use the for-each loop to convert a List to an array in Java: 

          
           
// Create list
List<String> list = new ArrayList<>();
list.add("Java");
list.add("Programming");
list.add("Language");

// Convert list to array
String[] array = new String[list.size()];
int index = 0;
for(String str: list) {
    array[index++] = str;
}

// Print array
System.out.println(Arrays.toString(array));
The output of the above code is as follows:
