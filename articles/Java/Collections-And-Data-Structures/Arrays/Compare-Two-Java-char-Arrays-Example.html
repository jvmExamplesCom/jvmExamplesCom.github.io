Compare Two Java char Arrays Example

Java char arrays are an important data type that can be used to store characters. In this example, we will cover how to compare two char arrays. To do this, we will create two char arrays with the same size and content and then use the Java Arrays class to see if they are equal. First, let's create two char arrays with the same size and content: 

          
           
char[] array1 = {'a', 'b', 'c'};
char[] array2 = {'a', 'b', 'c'};
Now, let's use the Arrays class to check if they are equal: 

          
           
boolean isEqual = Arrays.equals(array1, array2);
System.out.println("The arrays are equal = " + isEqual);
This should print out "The arrays are equal = true". Now let's create two char arrays with different sizes and content: 

          
           
char[] array3 = {'a', 'b'};
char[] array4 = {'a', 'b', 'c'};
Now, let's use the Arrays class to check if they are equal: 

          
           
boolean isEqual2 = Arrays.equals(array3, array4);
System.out.println("The arrays are equal = " + isEqual2);
This should print out "The arrays are equal = false". We can also use the Arrays class to compare two char arrays of the same size but with different contents: 

          
           
char[] array5 = {'a', 'b', 'c'};
char[] array6 = {'d', 'e', 'f'};

boolean isEqual3 = Arrays.equals(array5, array6);
System.out.println("The arrays are equal = " + isEqual3);
This should print out "The arrays are equal = false". We can also compare two char arrays of different sizes but with the same elements: 

          
           
char[] array7 = {'a', 'b'};
char[] array8 = {'a', 'b', 'c', 'd'};

boolean isEqual4 = Arrays.equals(array7, array8);
System.out.println("The arrays are equal = " + isEqual4);
This should print out "The arrays are equal = false". In summary, we can use the Arrays class to compare two char arrays in Java. We can compare two char arrays of the same size with or without the same elements, or we can compare two char arrays of different sizes with the same elements.
