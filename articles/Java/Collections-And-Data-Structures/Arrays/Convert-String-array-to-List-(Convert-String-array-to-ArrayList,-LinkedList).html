Convert String Array to List

One of the most versatile and useful data structures used in programming is the list. The list is an ordered, indexable collection of elements that can store more than one item at a time. In Java, a list can be created from a String array. The process for doing this is relatively simple and straight-forward.

Convert String Array to ArrayList

The easiest way to convert a String array to a List is to use the 


          
           
Arrays.asList(String[])
method. This method takes an array and returns a fixed-size List of the array elements. The following code example shows how to use this method: 

          
           
String[] strArray = {"a", "b", "c"};
List<String> list = Arrays.asList(strArray);
System.out.println(list);

When the above code is compiled and run, it will produce the following output:


          
           
[a, b, c]

Note that this method does not create a new List object. It simply returns a fixed-size List view of the specified array. Any changes made to the List will be reflected in the original array, and vice versa.

Convert String Array to LinkedList

Another way to convert a String array to a List is to use the 

LinkedList
class. This class provides a constructor that takes an array of objects, which can then be used to initialize the List. The following code example shows how to use this constructor: 

          
           
String[] strArray = {"a", "b", "c"};
List<String> list = new LinkedList<>(Arrays.asList(strArray));
System.out.println(list);

When the above code is compiled and run, it will produce the following output:


          
           
[a, b, c]

In this case, a new List object is created and the elements of the array are added to the List. Note that any changes made to the List will not be reflected in the original array.

In summary, there are two ways to convert a String array to a List in Java. The 


          
           
Arrays.asList(String[])
method can be used to return a fixed-size List view of the array, while the 
LinkedList
constructor can be used to create a new List object containing the elements of the array. Both methods have their own advantages and disadvantages, and should be used according to the particular requirements of the task at hand.
