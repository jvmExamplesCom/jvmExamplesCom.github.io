Comparing Two Java long Arrays Example

Java allows you to compare two long arrays that are of the same size. This can be done using the 

          
           
Arrays.equals(long[],long[])
method. This task is particularly useful for checking if two separate long arrays contain the same values.

Example Code

The following code snippet demonstrates how to use the 

          
           
Arrays.equals(long[],long[])
method to compare two long arrays: 

          
           
long[] array1 = { 1, 2, 3 };
long[] array2 = { 1, 2, 3 };

boolean result = Arrays.equals(array1, array2);
In this example, we first define two long arrays with the same values. We then call the 

          
           
Arrays.equals(long[],long[])
method and assign the result to the 

          
           
result
variable.

Output

In this example, the 

          
           
result
variable will be set to 

          
           
true
if the two arrays contain the same values. Otherwise, it will be set to 

          
           
false
.
