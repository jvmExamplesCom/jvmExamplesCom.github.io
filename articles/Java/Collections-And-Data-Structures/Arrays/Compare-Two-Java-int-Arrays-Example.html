Comparing Two Java int Arrays Example

In this example, we will compare two Java int arrays and will output the result in the console.

Java Program to Compare Two int Arrays


          
           
public class CompareTwoIntArraysExample {
    public static void main(String[] args) {
        int[] arrayOne = {2, 5, 1, 7, 4};
        int[] arrayTwo = {2, 5, 1, 7, 4};

        if (Arrays.equals(arrayOne, arrayTwo)) {
            System.out.println("Two Java int arrays are equal");
        } else {
            System.out.println("Two Java int arrays are not equal");
        }
    }
}
Output: Two Java int arrays are equal

Using another approach to compare two int arrays

We can use another approach to compare two int arrays i.e. looping through each elements of the two arrays and compare them. We will return true only if all the elements of both the arrays are same.


          
           
public class CompareIntArrays {
    public static boolean compareTwoIntArrays(int[] arr1, int[] arr2) {
        if (arr1.length != arr2.length) {
            return false;
        } else {
            for (int i = 0; i < arr1.length; i++) {
                if (arr1[i] != arr2[i]) {
                    return false;
                }
            }
            return true;
        }
    }
}
