Convert String to ArrayList in Java

Java provides a number of ways to convert a String to an ArrayList. This can be useful for manipulating strings or data structures in programs, as ArrayLists are more versatile than arrays and strings. In this article, we will discuss the various ways to convert a String to an ArrayList in Java.

1. Using the split() Method

The simplest way to convert a String to an ArrayList is by using the split() function. The split() function takes the given string and splits it into an array of words, based on a separator. The separator can be specified as a single character, a string or a regular expression. The following example converts a String to an ArrayList using the split() method: 

          
           
String str = "java,language,programming";
String[] words = str.split(","); 
List<String> list = Arrays.asList(words);
ArrayList<String> arrayList = new ArrayList<String>(list);
In the above example, the string is split on the comma character and the resulting words are converted to an ArrayList using the Arrays.asList() function.

String str = "java,language,programming";
