Finding Array Element Index using Arrays Class

Java provides multiple methods to search through an array and find the desired element. In this tutorial, we will discuss how to use the Arrays class to find the index of a particular element in an array. The Arrays class contains a static method called 
indexOf()
 which takes two parameters - the array and the element whose index needs to be found. It returns the first index of this element within the array or -1 if not found. Here is an example of finding the first index element of the value 10 in an integer array: 

          
           
int[] my_array = {1, 5, 10, 15, 20};
int index = Arrays.indexOf(my_array, 10);
System.out.println("Index position of 10 is: " + index);
// Outputs: Index position of 10 is: 2
The 
lastIndexOf()
 method can also be used to find the index of an element starting from the end of the array. 

          
           
int lastIndex = Arrays.lastIndexOf(my_array, 10);
System.out.println("Last index position of 10 is: " + lastIndex);
// Outputs: Last index position of 10 is: 2

Find Element in Multidimensional Array

If the array contains multiple dimensions, the 
deepIndexOf()
 method can be used to find the index of the element. 

          
           
int[][] array2D = {{1, 2, 10}, {4, 5, 6}, {7, 8, 9}};
int deepIndex = Arrays.deepIndexOf(array2D, 10);
System.out.println("Index of 10 is: " + deepIndex);
// Outputs: Index of 10 is: [0, 2]
The 
deepIndexOf()
 method returns an array which contains the indexes of the element in the multidimensional array. In the example above, the index is returned as 

          
           
[0,2]
, which means that the element is at row 0, column 2.

Searching for Multiple Elements

The 
indexOf()
 and 
lastIndexOf()
 methods can be used to find the index of multiple elements in an array. 

          
           
int[] array = {1, 5, 10, 10, 20};
 int first10Index = Arrays.indexOf(array, 10); 
 int last10Index = Arrays.lastIndexOf(array, 10); 
 System.out.println("First index of 10 is: " + first10Index);
 System.out.println("Last index of 10 is: " + last10Index);
 // Outputs: First index of 10 is: 2
 //          Last index of 10 is: 3
In the above example, the 
indexOf()
 method returns the first index of 10 in the array i.e 2 and 
lastIndexOf()
 returns the last index of 10 i.e 3. The 
deepIndexOf()
 method can also be used to search for multiple elements in the multidimensional array. 

          
           
int[][] array2D = {{1, 2, 10}, {4, 5, 10}, {7, 8, 9}};
int[][] deepIndices = Arrays.deepIndexOf(array2D, 10);

System.out.println("Indices of 10 are:");
for (int[] indices : deepIndices) {
 System.out.println(Arrays.toString(indices));
}
// Outputs: Indices of 10 are: 
//          [0, 2]
//          [1, 2]
In the above example, the 
deepIndexOf()
 method returns an array of indices where the element can be found inside the 2D array. Here it returns two indices 

          
           
[0, 2]
and 

          
           
[1,2]
. This tutorial discussed how to use the Arrays class to find the index of an element in an array using multiple methods such as 
indexOf()
, 
lastIndexOf()
 and 
deepIndexOf()
.
