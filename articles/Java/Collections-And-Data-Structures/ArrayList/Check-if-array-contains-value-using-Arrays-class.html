Check if array contains value using Arrays Class

Java's Arrays class contains a variety of methods to help work with arrays. One of these methods, 

contains()
, allows you to determine if an array contains a certain value. In this article, we will take a look at how to use the 
contains()
method in Java.

How to Check if an Array Contains Value

Using the 
contains()
method of the Arrays class, you can quickly check if an array contains a certain value. This method takes two parameters: an array and a value that you want to search for. Note that this type of search will only work with primitive types and objects. 

          
           
boolean containsValue = Arrays.contains(arrayName, value);
In the example above, the 
arrayName
parameter is the array name (the array you are searching through), and the 
value
parameter is the value that you are searching for.

The result of this search is a boolean. If the specified array contains the specified value, the result will be 

true
. If it does not contain the value, the result will be 
false
.

Example Usage of Arrays.contains()

Let's take a look at an example usage of the 
Arrays.contains()
method. Let's say we have an array of integers: 

          
           
int[] array = {1, 2, 3, 4, 5};

Now let's say we want to check if the array contains the number 2. We can use the 

Arrays.contains()
method to do this: 

          
           
boolean containsValue = Arrays.contains(array, 2);
System.out.println("Array contains 2? " + containsValue);
This code will print out the following: 

          
           
Array contains 2? true
As you can see, the 
Arrays.contains()
method correctly found that the array contains the value 2.

Conclusion

In this article, we took a look at how to use the 
Arrays.contains()
method in Java. This method allows you to quickly check if an array contains a certain value. We saw an example usage of this method, and saw that it correctly determined that the array did indeed contain the value 2.
