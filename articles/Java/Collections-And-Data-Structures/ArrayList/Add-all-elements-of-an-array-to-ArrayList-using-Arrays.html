Adding All Elements of an Array to an ArrayList using Arrays

The Arrays class provides a static method 

          
           
asList()
which can be used to convert a given array into a list. This method is available in the 

          
           
java.util
package and it was added to Java 1.4. It can be used for adding all elements of an array to an ArrayList. The following example shows how to add all elements of an array, passed as argument to the List constructor, to an ArrayList. 

          
           
String[] myArray = new String[]{"Apple", "Banana", "Grape"};

List<String> myList = new ArrayList<>(Arrays.asList(myArray));
The above code will create an 

          
           
ArrayList
object with all the elements of 

          
           
myArray
.
