Copying ArrayList Elements to Other ArrayList in Java

ArrayLists are an essential part of programming in Java. They provide an easy way to store, modify, and access data stored in lists. In this article, we will discuss copying elements from one ArrayList to another.

The most straightforward technique for copying the elements of one ArrayList to another is the 

addAll()
method. This method is available with the List interface, so you can use it with any type of List. It takes as a parameter another List, which contains the elements that you want to copy. Here is an example of using the 
addAll()
method to copy the elements from one ArrayList to another: 

          
           
// Create two ArrayLists
ArrayList<String> list1 = new ArrayList<>();
ArrayList<String> list2 = new ArrayList<>();
 
// Add some elements to list1
list1.add("Apple");
list1.add("Banana");
list1.add("Mango");
 
// Copy all elements from list1 to list2
list2.addAll(list1);
 
// Print list2
System.out.println(list2);
This code prints out: 

          
           
[Apple, Banana, Mango]
The 
addAll()
method also has an overloaded version, which accepts a parameter of type Collection. This means that you can use the method to copy elements from any type of Collection, not just List. Here is an example of using the overloaded version of 
addAll()
: 

          
           
// Create a LinkedHashSet
LinkedHashSet<String> set = new LinkedHashSet<>();
 
// Add some elements to set
set.add("Cat");
set.add("Dog");
set.add("Horse");
 
// Create an ArrayList
ArrayList<String> list = new ArrayList<>();
 
// Copy all elements from set to list
list.addAll(set);
 
// Print list
System.out.println(list);
This code prints out: 

          
           
[Cat, Dog, Horse]

Using The Java Stream API

The Java Stream API can also be used to copy the elements of one ArrayList to another. The 

collect()
method can be used to create a new list. Here is an example of using the Stream API to copy the elements of one ArrayList to another: 

          
           
// Create two ArrayLists
ArrayList<String> list1 = new ArrayList<>();
ArrayList<String> list2 = new ArrayList<>();
 
// Add some elements to list1
list1.add("Apple");
list1.add("Banana");
list1.add("Mango");
 
// Copy all elements from list1 to list2
list2 = list1.stream().collect(Collectors.toCollection(ArrayList::new));
 
// Print list2
System.out.println(list2);
This code prints out: 

          
           
[Apple, Banana, Mango]

Manually Copying Elements

It is also possible to manually copy elements from one ArrayList to another. This can be done by iterating through the source list and adding the elements to the target list. Here is an example of manually copying elements from one ArrayList to another:


          
           
// Create two ArrayLists
ArrayList<String> list1 = new ArrayList<>();
ArrayList<String> list2 = new ArrayList<>();
 
// Add some elements to list1
list1.add("Apple");
list1.add("Banana");
list1.add("Mango");
 
// Iterate through list1 and add elements to list2
for (String element : list1) {
    list2.add(element);
}
 
// Print list2
System.out.println(list2);
This code prints out: 

          
           
[Apple, Banana, Mango]

Conclusion

In this article, we discussed how to copy the elements from one ArrayList to another in Java. We looked at three different techniques for copying elements: the 

addAll()
method, the Stream API, and manual iteration. Each technique has its own advantages and disadvantages, so you should choose the one that is the most suitable for your application.
