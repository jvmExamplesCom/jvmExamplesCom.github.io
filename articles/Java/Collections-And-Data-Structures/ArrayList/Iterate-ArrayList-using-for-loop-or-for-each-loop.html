Iterating ArrayList Using for Loop and for each Loop

Overview of for Loop and for each Loop

For loop and for each loop are control structures used to iterate over elements in a collection. For loop was the traditional way to loop through an ArrayList prior to Java 5. It is more flexible than for each loop because it supports additional features like nested looping and using the index of the element being processed. For each loop, introduced in Java 5, simplifies the syntax of looping and is designed specifically to process collections.

for Loop Syntax

The syntax for the for loop is: 

          
           
for(initialization; condition; increment/decrement) {
   statement;
}
The following code example iterates over an ArrayList using a for loop and prints each element in the list: 

          
           
ArrayList<String> list = new ArrayList<String>();
list.add("Hello");
list.add("World");

for (int i = 0; i < list.size(); i++) {
   System.out.println(list.get(i));
}
The output of this code is: 

          
           
Hello
World

for-each Loop Syntax

The syntax for the for-each loop is: 

          
           
for (declaration : expression) {
   statement;
}
The following code example uses a for-each loop to iterate over an ArrayList and print each element in the list: 

          
           
ArrayList<String> list = new ArrayList<String>();
list.add("Hello");
list.add("World");

for (String s : list) {
   System.out.println(s);
}
The output of this code is: 

          
           
Hello
World

Conclusion

For loop and for each loop are essential parts of any programming language. They allow you to iterate over a collection and perform operations on the elements within it. The for loop is more flexible than the for each loop, but the for each loop provides a simpler syntax for looping over collections.
