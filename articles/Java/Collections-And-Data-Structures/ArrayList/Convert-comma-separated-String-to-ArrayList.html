Convert comma separated String to ArrayList in Java

Converting a comma separated String to an ArrayList involves using the split method of the String class on the input String to create an array of String objects, then copying each element of the array into an ArrayList. This can be done as follows: 

          
           
String str = "1,2,3,4,5"; 
String[] items = str.split(","); 
ArrayList<String> list = new ArrayList<>(); 
for (String item : items) { 
    list.add(item); 
}
The first line of this code creates a String object named "str" and assigns it a comma separated values string. The second line calls the split() method on "str", passing in a comma as the separator. The split() method returns an array of String objects that are the elements of the comma separated string. The third line creates a new instance of the ArrayList class, of type String. The fourth line iterates through the array of strings created when we called split(), and adds each element to the ArrayList. Once this code has been executed, the ArrayList "list" contains five elements, one for each element of the comma separated string we started out with. We can also use the Arrays.asList() method to achieve the same result. This method takes an array and returns a List object, which is a type of Collection. Here is an example: 

          
           
String str = "1,2,3,4,5"; 
String[] items = str.split(","); 
List<String> list = Arrays.asList(items);
This code produces the same result as the previous example - the "list" variable now contains a List of five elements, one for each element of the comma separated string. However, there are some important differences between the two approaches. One difference is that if you try to modify the list returned by Arrays.asList() - for example, by adding or removing an element - then you will get an UnsupportedOperationException. This is because the list returned by Arrays.asList() is not a true ArrayList, but rather a fixed-length List, and therefore it cannot be modified. If you need to add or remove elements from the List, then you must use the first approach and construct a true ArrayList.

Conclusion

In this article we discussed two different ways to convert a comma-separated String to an ArrayList in Java. The first approach involves creating an array of String objects, then copying each element of the array into an ArrayList. The second approach involves using the Arrays.asList() method, which returns a List object. Each approach has its advantages and disadvantages - the first approach allows additions and removal of elements from the ArrayList, while the second approach returns a fixed-length List that cannot be modified.
