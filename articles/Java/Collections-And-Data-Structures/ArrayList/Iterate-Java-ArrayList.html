Iterate Java ArrayList

ArrayList is one of the most popular data structures used in Java. It is a dynamic array that can be resized when needed and it allows for easy insertion and removal of elements. It also supports random access, which means elements can be accessed directly by their index. Iterating through an ArrayList is an essential task for any Java developer. In this article, we will explore different ways to iterate an ArrayList in Java.

Using forEach() Method

Java 8 introduced the forEach() method which provides a convenient way to iterate over elements in an ArrayList. Here is an example of using the forEach() method: 

          
           
List<String> list = new ArrayList<>();
list.add("Hello");
list.add("World");
list.forEach(str -> System.out.println(str));
The output of the above example will be: 

          
           
Hello
World

Using Iterator

Iterator is a universal cursor for traversing data structures in Java. It provides an interface for iterating over the elements of an ArrayList. Here is an example of using the Iterator interface to iterate an ArrayList: 

          
           
List<String> list = new ArrayList<>();
list.add("Hello");
list.add("World");
Iterator it = list.iterator();
while (it.hasNext()) {
	String str = (String)it.next();
	System.out.println(str);
}
The output of the above example will be: 

          
           
Hello
World

Using for Loop

A for loop can also be used to iterate over an ArrayList. Here is an example of using a for loop to traverse an ArrayList: 

          
           
List<String> list = new ArrayList<>();
list.add("Hello");
list.add("World");
for (int i = 0; i < list.size(); i++) {
	String str = list.get(i);
	System.out.println(str);
}
The output of the above example will be: 

          
           
Hello
World

Using for-each Loop

Java 5 introduced the for-each loop which provides a simpler way to traverse an ArrayList. Here is an example of using the for-each loop to iterate an ArrayList: 

          
           
List<String> list = new ArrayList<>();
list.add("Hello");
list.add("World");
for (String str : list) {
	System.out.println(str);
}
The output of the above example will be: 

          
           
Hello
World
As you can see, there are several different ways to iterate an ArrayList in Java. Each method has its own pros and cons so you should pick the one that best suits your needs.
