Remove All Elements from Java ArrayList Example

In this article, we will discuss how to remove all elements from Java ArrayList. We will also provide code examples that demonstrate how to remove all elements from Java ArrayList.

Before You Begin

Before you begin, it is important to understand the ArrayList class in Java. ArrayList is a resizable array implementation of the List interface. It provides methods to manipulate both elements in the list as well as the size of the list.

How to Remove All Elements from Java ArrayList

The easiest way to remove all elements from Java ArrayList is by using its clear() method. This method removes all of the elements from the list, setting its size to zero. The syntax for using this method is as follows:


          
           
list.clear();

For example, if we have an ArrayList called "list" that contains the following elements:


          
           
1, 2, 3, 4, 5

We can use the clear() method to remove all elements from the list:


          
           
list.clear();

After calling the clear() method, our list will be empty. It will contain no elements and its size will be set to zero: 


          
           
[]

Remove All Elements from Java ArrayList Example

Let's look at an example that demonstrates how to remove all elements from a Java ArrayList. We will create an ArrayList of integers and add some elements to it. Then, we will use the clear() method to remove all elements from the list.


          
           
import java.util.ArrayList;

public class RemoveAllElementsExample {

    public static void main(String[] args) {

        // Create an ArrayList of integers
        ArrayList<Integer> list = new ArrayList<>();

        // Add some elements to the list
        list.add(1);
        list.add(2);
        list.add(3);
        list.add(4);
        list.add(5);

        // Print the list
        System.out.println("List before removing all elements: " + list);

        // Remove all elements from the list
        list.clear();

        // Print the list
        System.out.println("List after removing all elements: " + list);
    }
}

When we run the above program, we get the following output:


          
           
List before removing all elements: [1, 2, 3, 4, 5]
List after removing all elements: []

As we can see, we were able to successfully remove all elements from the Java ArrayList.
