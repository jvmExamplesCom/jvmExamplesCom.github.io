Removing Duplicate Elements from ArrayList

ArrayList is a very popular data structure in Java for managing collections of objects. It allows us to add, remove, and access elements in an easy and efficient way. However, it is possible that the same element or object may be repeated multiple times within the list. This article explains how to remove duplicate elements from an ArrayList in Java.

The Code to Remove Duplicates from an ArrayList

The following code example shows a method which removes duplicate elements from an ArrayList:


          
           
public static void removeDuplicates(ArrayList<Integer> list) {
    Set<Integer> set = new HashSet<>();
    set.addAll(list);
    list.clear();
    list.addAll(set);
}
The code iterates over the given 
list
and uses a 
HashSet
to store unique elements. The elements from the 
list
are then added to the 
set
. Finally, the contents of the 
list
are cleared, and the elements from the 
set
are added back to the 
list
. This ensures that the 
list
only contains unique elements, effectively removing all duplicates.

Example Usage

Let's see the above code in action by creating an 
ArrayList
with duplicate elements and calling the 
removeDuplicates
method on it: 

          
           
ArrayList<Integer> list = new ArrayList<>();
list.add(1);
list.add(2);
list.add(3);
list.add(1);
list.add(2);

removeDuplicates(list);

for (Integer e : list) {
    System.out.println(e);
}

The above code prints the following output to the console:


          
           
1
2
3
As we can see, the duplicates from the 
ArrayList
have been successfully removed.

Conclusion

This article explained how to remove duplicate elements from an 

ArrayList
in Java. The main idea was to use a 
Set
to store unique elements, and then add these elements back to the 
ArrayList
, effectively removing the duplicates. We also saw an example of the code in action, demonstrating that it works as expected.
