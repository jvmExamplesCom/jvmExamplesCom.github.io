Find the Minimum or Maximum Value in ArrayList

Java makes searching through an array list for the minimum or maximum value easy. In this article, we'll discuss several ways to find the smallest or largest value in an ArrayList.

Using Java Stream API

The Stream API makes finding the minimum or maximum value of an ArrayList easy and straightforward. The following code shows how to find the minimum number in an ArrayList using Stream API: 

          
           
List<Integer> numbers = Arrays.asList(4, 5, 1, 8, 7, 9);
int minNumber = numbers.stream().min(Integer::compare).get();
System.out.println(minNumber); // Output: 1
Similarly, the following code shows how to find the maximum number in an ArrayList using Stream API: 

          
           
List<Integer> numbers = Arrays.asList(4, 5, 1, 8, 7, 9);
int maxNumber = numbers.stream().max(Integer::compare).get();
System.out.println(maxNumber); // Output: 9

Using forEach() and Lambda Expression

We can also use the forEach() method and a lambda expression to find the minimum or maximum value in an ArrayList. The following code shows how to find the minimum number in an ArrayList using forEach() and a lambda expression: 

          
           
List<Integer> numbers = Arrays.asList(4, 5, 1, 8, 7, 9);
int minNumber = Integer.MAX_VALUE;
numbers.forEach(n -> {
   if (n < minNumber) {
       minNumber = n;
   }
});
System.out.println(minNumber); // Output: 1
Similarly, the following code shows how to find the maximum number in an ArrayList using forEach() and a lambda expression: 

          
           
List<Integer> numbers = Arrays.asList(4, 5, 1, 8, 7, 9);
int maxNumber = Integer.MIN_VALUE;
numbers.forEach(n -> {
   if (n > maxNumber) {
       maxNumber = n;
   }
});
System.out.println(maxNumber); // Output: 9

Using for-loop

Finally, we can use a simple for-loop to find the minimum or maximum value in an ArrayList. The following code shows how to find the minimum number in an ArrayList using a for-loop: 

          
           
List<Integer> numbers = Arrays.asList(4, 5, 1, 8, 7, 9);
int minNumber = Integer.MAX_VALUE;
for (int n : numbers) {
   if (n < minNumber) {
       minNumber = n;
   }
}
System.out.println(minNumber); // Output: 1
Similarly, the following code shows how to find the maximum number in an ArrayList using a for-loop: 

          
           
List<Integer> numbers = Arrays.asList(4, 5, 1, 8, 7, 9);
int maxNumber = Integer.MIN_VALUE;
for (int n : numbers) {
   if (n > maxNumber) {
       maxNumber = n;
   }
}
System.out.println(maxNumber); // Output: 9
In summary, there are several ways to find the minimum or maximum value of an ArrayList in Java. We can use the Stream API, the forEach() method and a lambda expression, or a simple for-loop.
