Java ArrayList Reverse Example

Getting Started

An ArrayList in Java is a type of collection that stores data as objects in an array. It is a resizable array, and an ordered list. The ArrayList class provides methods to manipulate the size of the array, to access the individual objects, add and remove objects, search for items within the arraylist, etc. In this tutorial, we will look at how to reverse an ArrayList in Java.

Reverse an ArrayList in Java

To reverse an ArrayList in Java, we can first use the 

ArrayList
class method 
.reverse()
, which reverses the order of the elements in the ArrayList. We can also use a for loop to manually iterate through the ArrayList and add each element to the beginning of the new ArrayList. The following example demonstrates these two approaches: 

          
           
import java.util.ArrayList;
import java.util.Collections;

public class ReverseArrayList {

    public static void main(String[] args) {
        // Create an ArrayList of Strings
        ArrayList<String> fruits = new ArrayList<>();
        fruits.add("apple");
        fruits.add("banana");
        fruits.add("orange");

        // Print the original ArrayList
        System.out.println("Original ArrayList:");
        for (String fruit : fruits) {
            System.out.println(fruit);
        }

        // Reverse the ArrayList using Collections
        Collections.reverse(fruits);
        System.out.println("\nReversed ArrayList using Collections.reverse():");
        for (String fruit : fruits) {
            System.out.println(fruit);
        }

        // Reverse the ArrayList using a for loop
        ArrayList<String> revFruits = new ArrayList<>();
        for (int i = fruits.size() - 1; i >= 0; i--) {
            revFruits.add(fruits.get(i));
        }
        System.out.println("\nReversed ArrayList using for loop:");
        for (String fruit : revFruits) {
            System.out.println(fruit);
        }

    }

}

Reversed ArrayList using Collections.reverse():

apple

Reversed ArrayList using for loop:
orange
