Java ArrayList Example: Adding an Element to a Specified Index

The Java 
ArrayList
class provides the functionality of resizing arrays in the Java programming language. It allows the programmer to add and remove elements from the list with ease.

In this tutorial, we will discuss how to add an element to a specified index in the Java 

ArrayList
. We will also provide examples of code to better explain the concept.

Creating a Java 
ArrayList

Before adding an element to a specified index in an 

ArrayList
, we must first create an 
ArrayList
. This can be done using the 
ArrayList
constructor, like so: 

          
           
ArrayList<String> list = new ArrayList<String>();
This line of code creates a new 
ArrayList
object named 
list
that can hold 
String
objects.

Adding an Element to a Specified Index

We can add an element to a specified index in the list using the 


          
           
add(int index, E element)
method, which adds the element at the specified index. The following example adds the String 
"Test"
at the 0th index in the list: 

          
           
list.add(0, "Test");

We can also add multiple elements to the list in a single statement. The following example adds the Strings 

"Test1"
and 
"Test2"
to indices 0 and 1 in the list, respectively: 

          
           
list.add(0, "Test1"); 
list.add(1, "Test2");

Printing the Contents of the List

We can print the contents of the list using the 
toString()
method. The following code prints the contents of the list we just created: 

          
           
System.out.println(list);
This code will produce the following output: 

          
           
[Test1, Test2]

ArrayList

ArrayList
using the
