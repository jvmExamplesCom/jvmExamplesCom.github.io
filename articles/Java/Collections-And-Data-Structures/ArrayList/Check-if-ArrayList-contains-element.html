Checking if an ArrayList Contains an Element

Java provides a class called ArrayList that allows the user to create dynamic arrays. A dynamic array is an array that resizes itself as elements are added to it or removed from it. ArrayLists are used to store collections of objects, each object representing a single element. It is often necessary to check if an ArrayList contains a certain element or not. This article will discuss how to check if an ArrayList contains an element using Java programing.

Using the `contains()` Method

The most efficient way to check if an ArrayList contains a certain element is to use the `contains()` method. This method returns a boolean value which indicates if a certain element is present in the ArrayList or not. Syntax for the `contains()` method is as follows: 

          
           
boolean contains(Object element)
This method takes a single argument, which is the element whose presence is to be checked in the list. It then returns a boolean value - `true` if the list contains the element, and `false` otherwise. Example code illustrating the use of `contains()` method for checking if an ArrayList contains a certain element is as follows: 

          
           
ArrayList<String> list = new ArrayList<String>();
list.add("Apple");
list.add("Orange");
list.add("Mango");
if (list.contains("Apple")) {
	System.out.println("Element found!");
} else {
	System.out.println("Element not found!");
}
In this example, we have created an ArrayList named `list` that contains three elements - `Apple`, `Orange`, and `Mango`. The `contains()` method is then used to check if the list contains the `Apple` element, and the result is printed on the screen.

Using the `indexOf()` Method

Another way to check if an ArrayList contains a certain element is to use the `indexOf()` method. This method can be used to get the index of a certain element in the list, and then check if the result is greater than or equal to `0` or not. Syntax for the `indexOf()` method is as follows: 

          
           
int indexOf(Object element)
This method takes a single argument, which is the element whose index is to be retrieved. It then returns the index of the element, or `-1` if it is not present in the list. Example code illustrating the use of `indexOf()` method for checking if an ArrayList contains a certain element is as follows: 

          
           
ArrayList<String> list = new ArrayList<String>();
list.add("Apple");
list.add("Orange");
list.add("Mango");
if (list.indexOf("Apple") >= 0) {
	System.out.println("Element found!");
} else {
	System.out.println("Element not found!");
}
In this example, we have created an ArrayList named `list` with three elements - `Apple`, `Orange`, and `Mango`. The `indexOf()` method is then used to check if the list contains the `Apple` element, and the result is printed on the screen.

Using the `stream()` Method

It is also possible to check if an ArrayList contains a certain element using the `stream()` method. This method allows us to filter the list and check if any element matches a certain condition or not. Syntax for the `stream()` method is as follows: 

          
           
Stream<T> stream()
This method does not take any argument and returns a sequential `Stream` of all the elements in the list. Example code illustrating the use of `stream()` method for checking if an ArrayList contains a certain element is as follows: 

          
           
ArrayList<String> list = new ArrayList<String>();
list.add("Apple");
list.add("Orange");
list.add("Mango");
if (list.stream().anyMatch("Apple"::equals)) {
	System.out.println("Element found!");
} else {
	System.out.println("Element not found!");
}
In this example, we have created an ArrayList named `list` that contains three elements - `Apple`, `Orange`, and `Mango`. The `stream()` method is then used to check if the list contains the `Apple` element, and the result is printed on the screen.

Conclusion

In this article, we discussed various ways of checking if an ArrayList contains a certain element using Java programming. We saw how the `contains()`, `indexOf()`, and `stream()` methods can be used for this purpose. With these methods, it is easy to quickly check if an ArrayList contains a certain element or not.
