Insert all elements of other Collection to Specified Index of Java ArrayList Example

In this tutorial, we will explore how to insert all elements of other Collection to a specified index of Java ArrayList with example. Firstly, let's look at the syntax of the method used to insert all elements of other Collection to a specified index of Java ArrayList. 

          
           
public boolean addAll(int index, Collection c)
This method takes two arguments- an index and a Collection. This method returns a boolean value. Let's look at an example that uses this method. 

          
           
import java.util.ArrayList;
import java.util.Collection;
 
public class InsertElementsExample {
   public static void main(String[] args) {
      // create an empty array list with an initial capacity
      ArrayList<String> arrlist = new ArrayList<String>(5);
               
      // use add() method to add elements in the list
      arrlist.add("A");
      arrlist.add("B");
      arrlist.add("C");
      arrlist.add("D");
       
      System.out.println("Size of arraylist before addAll:"+arrlist.size());
       
      Collection<String>collection=new ArrayList<String>();
      collection.add("X");
      collection.add("Y");
      collection.add("Z");
       
      // insert the specified collection elements at the specified position in this list
      arrlist.addAll(2,collection);
       
      System.out.println("Size of arraylist after addAll:"+arrlist.size());
       
      // print the arraylist
      System.out.println(arrlist);
   }    
}
In the above example, we have first created an empty arraylist called arrlist with an initial capacity of 5. We have then used the add() method to add 4 elements, A, B, C and D. We have then created a new collection containing 3 elements X, Y and Z. The addAll() method is used to insert all elements from the specified collection into arrlist at the specified position 2. The size of the arraylist is printed before and after the addition of the new collection's elements. Finally, the arraylist is printed. When we run the above code, we get the following output: 

          
           
Size of arraylist before addAll:4
Size of arraylist after addAll:7
[A, B, X, Y, Z, C, D]
Thus, our code example has effectively demonstrated how to insert all elements of other Collection to a specified index of Java ArrayList.
