Print ArrayList Elements using Arrays

ArrayList is an important part of Java, since it allows you to easily store and work with data. This tutorial will show you how to use Arrays to print the elements of an ArrayList.

Basic ArrayList Syntax

First, let's review some basic ArrayList syntax. Here is an example of an ArrayList declared using generics:


          
           
ArrayList<String> list = new ArrayList<>();
You can add elements to the list using the 
add()
method: 

          
           
list.add("apple");
list.add("banana");
list.add("cherry");

Convert ArrayList to Array

In order to print the elements of an ArrayList using Arrays, we need to first convert it to an array. This can be done using the 

toArray()
method. This method takes one parameter, which is the type of array to be created. 

          
           
String[] array = list.toArray(new String[list.size()]);
This code creates an array of type 
String[]
, with the same size as the ArrayList. It then copies all of the elements from the ArrayList into the newly created array.

Print ArrayList Elements

Once the ArrayList has been converted to an array, we can use a 

for
loop to iterate over the elements and print them out. Here is an example: 

          
           
for (String s : array) {
    System.out.println(s);
}
This code will print out the following: 

          
           
apple
banana
cherry

And that's all there is to printing the elements of an ArrayList using Arrays!
