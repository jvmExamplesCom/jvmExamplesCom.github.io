Convert Java Array to Vector Using Arrays Class

Java’s Arrays class provides the method 

          
           
toVector()
which allows for conversion of an array into a Vector object. Vectors are a specialized type of data structure that fit sequence into memory, allowing for more efficient data access than with a standard array. This conversion can be applied to any type of array and supports all of the same types and sizes as the original array.

Using toVector() for Simple Arrays

The following example demonstrates how to use toVector() to convert a simple array of Integer values: 

          
           
Integer[] nums = {1, 2, 3, 4}; 
Vector<Integer> numsVec = Arrays.toVector(nums);

Integer

nums

double[][] arr = {{1.2, 3.4, 5.6}, {7.8, 9.0, 11.12}}; 
Vector<double[]> arrVec = Arrays.toVector(arr);
