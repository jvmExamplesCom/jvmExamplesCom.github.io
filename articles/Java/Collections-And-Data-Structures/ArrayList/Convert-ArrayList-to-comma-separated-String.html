Convert ArrayList to Comma Separated String in Java

In this article, we'll discuss how to convert a Java ArrayList to a comma-separated string. We will look at several examples of code and explore their output.

Using Iterator

The following example uses an Iterator to traverse the ArrayList and concatenate each item with a comma separator and store it in a StringBuilder object: 

          
           
ArrayList<String> list = new ArrayList<>();
list.add("apple");
list.add("banana");
list.add("orange");

StringBuilder sb = new StringBuilder();
Iterator<String> iter = list.iterator();
while (iter.hasNext()) {
    sb.append(iter.next());
    if (iter.hasNext()) {
    sb.append(",");
    }
}
String str = sb.toString();
System.out.println(str);
This code will output the following comma-separated string: 

          
           
apple,banana,orange

Using Stream API

The following example uses the Stream API and the map() method to create a stream of strings from the ArrayList, and then uses the collect() method to join the strings with a comma separator: 

          
           
ArrayList<String> list = new ArrayList<>();
list.add("apple");
list.add("banana");
list.add("orange");

String str = list.stream()
                .map(e -> e.toString())
                .collect(Collectors.joining(","));
System.out.println(str);
This will output the same string as the previous example: 

          
           
apple,banana,orange

Using For-Each Loop

The following example uses a for-each loop to traverse the ArrayList and append the elements to a StringBuilder object with a comma separator: 

          
           
ArrayList<String> list = new ArrayList<>();
list.add("apple");
list.add("banana");
list.add("orange");

StringBuilder sb = new StringBuilder();
for (String element : list) {
    sb.append(element);
    sb.append(",");
}
String str = sb.deleteCharAt(sb.length()-1).toString();
System.out.println(str);
This code will also output the same string: 

          
           
apple,banana,orange

Using StringJoiner

The following example uses the StringJoiner class and the add() method to add the strings from the ArrayList and creates a comma-separated string: 

          
           
ArrayList<String> list = new ArrayList<>();
list.add("apple");
list.add("banana");
list.add("orange");

StringJoiner sj = new StringJoiner(",");
for (String element : list) {
    sj.add(element);
}
String str = sj.toString();
System.out.println(str);
This code will also output the same string: 

          
           
apple,banana,orange
