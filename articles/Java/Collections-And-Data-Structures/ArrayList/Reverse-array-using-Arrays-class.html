Reverse Array Using Arrays Class

The 
java.util.Arrays
class provides various methods for manipulating arrays. One of these methods is the 
reverse()
method, which reverses the order of the elements within an array. This can be useful in a variety of scenarios, such as sorting algorithms or when you need to manipulate the elements in an array.

Example: Reversing an Array

Let's look at a simple example of how to reverse an array using the 

reverse()
method. We'll create an array with three elements and then reverse the order of those elements. 

          
           
int[] numbers = {1, 2, 3};

Arrays.reverse(numbers);

for(int i : numbers){
    System.out.println(i);
}
In this code, we create an array 
numbers
with three elements (1, 2, and 3). We then call 
Arrays.reverse()
on the array, which reverses the order of the elements. Finally, we use a 
for
loop to print out the elements of the reversed array. The output of this code will be: 

          
           
3
2
1

As you can see, the elements of the array have now been reversed.

Example: Reversing a Sorted Array

reverse()

int[] numbers = {1, 2, 3, 4, 5, 6};

Arrays.reverse(numbers);

for(int i : numbers){
    System.out.println(i);
}
In this code, we create a sorted array 
numbers
with six elements (1, 2, 3, 4, 5, and 6). We then call 
Arrays.reverse()
on the array, which reverses the order of the elements. Finally, we use a 
for
loop to print out the elements of the reversed array. The output of this code will be: 

          
           
6
5
4
3
2
1

As you can see, the elements of the array have now been reversed.
