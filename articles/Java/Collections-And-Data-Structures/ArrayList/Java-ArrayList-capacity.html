Java ArrayList Capacity

ArrayList is one of the most commonly used collection classes in Java. It is a part of the Java Collection Framework, and it offers dynamic arrays that can grow or shrink as needed. The capacity of an ArrayList object is the number of elements that the list can hold. To understand the concept better, let us look at some examples.

Example 1: Creating an ArrayList with Default Capacity

The following code shows how to create an ArrayList object using its default capacity. 

          
           
ArrayList<Integer> numbers = new ArrayList<>();
In this example, the default capacity of the list is 10. If the list has fewer than 10 elements, the extra space will remain unused. However, if the list contains more than 10 elements, the capacity will automatically be increased by creating a new array of double the original size and copying all the elements from the old array to the new one.

Example 2: Creating an ArrayList with a Specific Capacity

It is also possible to specify the initial capacity of the list when creating it. The following code shows how to do this. 

          
           
ArrayList<Integer> numbers = new ArrayList<>(20);
In this example, the initial capacity of the list is 20. If the list contains more than 20 elements, the capacity will be increased in the same way as described in the previous example.

Example 3: Adding Elements to an ArrayList

When adding elements to an ArrayList, the capacity of the list will be increased if necessary. The following code shows an example of adding elements to a list. 

          
           
ArrayList<Integer> numbers = new ArrayList<>(); 
numbers.add(1); 
numbers.add(2); 
numbers.add(3);
In this example, the list starts with a default capacity of 10. After adding three elements, the capacity of the list will be increased to 20.

Example 4: Removing Elements from an ArrayList

Removing elements from an ArrayList does not affect the capacity of the list. The following code shows an example of removing elements from a list. 

          
           
ArrayList<Integer> numbers = new ArrayList<>(); 
numbers.add(1); 
numbers.add(2); 
numbers.add(3); 
numbers.remove(2);
In this example, the list starts with a default capacity of 10. After removing one element, the capacity remains unchanged and is still 10.

Conclusion

In this article, we discussed the concept of capacity in an ArrayList object in Java. We looked at examples of how to create a list with a specific capacity, how to add and remove elements from a list, and how the capacity of a list is affected.
