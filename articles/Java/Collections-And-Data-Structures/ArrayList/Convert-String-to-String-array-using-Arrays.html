Converting String to String Array Using Arrays

Java's Arrays class provides various methods for working with arrays, such as sorting, searching, and converting from one data type to another. In this tutorial, we'll look at how to convert a string to an array of strings using the Arrays class.

Example 1: Split Method

The easiest way to convert a string to an array is by using the split() method of the String class. This method accepts a string as an argument and splits it into an array of strings based on the specified delimiter. The following example shows how to split a string into an array of words: 

          
           
String str = "Hello World";
String[] words = str.split("\\s");
In this example, the delimiter is a whitespace character ("\s"), which will cause the string to be split at each word. The resulting array will contain two elements: "Hello" and "World".

Example 2: Stream API

Another approach is to use the Stream API of the Arrays class. This method takes a String and returns a stream of String objects. The following example demonstrates how to convert a String to an array of Strings using the Stream API: 

          
           
String str = "Hello World";
String[] words = Arrays.stream(str.split("\\s"))
                        .map(w -> w.toUpperCase())
                        .toArray(String[]::new);
This example uses the split() method to split the string into an array of words, then uses the Stream API to map each word to uppercase and creates a new array of Strings. The resulting array will contain two elements: "HELLO" and "WORLD".

Example 3: Copy Of Method

The Arrays class also provides a copyOf() method that can be used to create a copy of an array with a specified length. The following example shows how to use this method to convert a String to an array of Strings: 

          
           
String str = "Hello World";
String[] words = Arrays.copyOf(str.split("\\s"), 2);
In this example, the split() method is used to split the string into an array of words, and then the copyOf() method is used to create a new array with a length of two. The resulting array will contain two elements: "Hello" and "World".

Conclusion

In this tutorial, we looked at three different ways to convert a String to an array of Strings in Java. All three approaches use the Arrays class in combination with other methods, such as the split() method or Stream API.
