Get sublist from ArrayList in Java

Java has a powerful library for working with collections. One of the most useful features of the java.util.ArrayList class is its capability to create a sublist. A sublist is a smaller list of elements contained within the original list. In this article, we'll look at how to create a sublist from an existing ArrayList. First, we'll begin by talking about the syntax of the sublist operation and then we'll learn how to use it to create a sublist from an existing list.

Syntax of Sublist Operations

The general syntax for creating a sublist from an existing ArrayList is as follows: 

          
           
ArrayList<e> subList (int startIndex, int endIndex);</e>
The startIndex specifies the index of the first element that will be included in the sublist while the endIndex defines the index of the last element that will be included in the sublist. Both indices are inclusive, meaning that they will both be included in the result. For example, if you had a list called nums with 10 elements and you wanted to create a sublist of the first 5 elements of the list, you could use the following command: 

          
           
List<integer> firstFive = nums.subList(0,4);</integer>
This would create a list called firstFive that contains the first 5 elements of the nums list.

Examples of Sublist Operations

Now that we've seen the syntax of the sublist method, let's look at some examples of how to use it in real-world scenarios. First, let's suppose we have a list of integers called nums and we want to create a sublist of the elements from index 2 to index 5. We could do this with the following command: 

          
           
List<integer> subNums = nums.subList(2,5);</integer>
This would create a list called subNums that contains the elements from index 2 to index 5 of the nums list. Another example would be if we had a list of Strings called names and we wanted to create a sublist of the first 3 elements of the list. We could do this with the following command: 

          
           
List<string> firstThree = names.subList(0,2);</string>
This would create a list called firstThree that contains the first 3 elements of the names list. Lastly, let's suppose we had a list of Person objects called people and we wanted to create a sublist of the elements from index 4 to index 7. We could do this with the following command: 

          
           
List<person> subPeople = people.subList(4,7);</person>
This would create a list called subPeople that contains the elements from index 4 to index 7 of the people list.
