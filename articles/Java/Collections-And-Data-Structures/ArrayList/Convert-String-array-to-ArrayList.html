How to convert String array to ArrayList in Java?

Converting a String array to an ArrayList can be useful when programming in Java. By converting a String array to an ArrayList, you can access and modify the elements more easily. In this article, we'll take a look at how to convert a String array to an ArrayList in Java.

Method 1: Using for loop

Using a for loop is the most straightforward way to add elements from a String array to an ArrayList. All you need to do is loop over the elements of the String array and add them to the ArrayList one by one. Here's an example: 

          
           
// Declare String array
String[] strArray = {"Hello", "World", "!"}; 

// Create ArrayList
ArrayList<string> arrList = new ArrayList<string>();

// Loop over String array
for(String str : strArray) {
   // Add each element to the ArrayList
   arrList.add(str);
}
</string></string>
In the above code, we've declared a String array and created an empty ArrayList. We've then used a for loop to loop over the elements of the String array and add them to the ArrayList. After running this code, our ArrayList will contain the elements of the String array.

Method 2: Using Arrays.asList() method

Another way to convert a String array to an ArrayList is by using the Arrays.asList() method. This method takes an array as an argument and returns a fixed-size list backed by the specified array. Here's an example of how to use it: 

          
           
// Declare String array
String[] strArray = {"Hello", "World", "!"};

// Convert String array to ArrayList
ArrayList<string> arrList = new ArrayList<string>(Arrays.asList(strArray));
</string></string>
In the above code, we've declared a String array and used the Arrays.asList() method to convert it to an ArrayList. After running this code, our ArrayList will contain the same elements as the String array.

Conclusion

In this article, we've looked at how to convert a String array to an ArrayList in Java. We've seen two different methods for doing this, both of which are straightforward and easy to use. Try them out for yourself and see which one works best for your particular situation.
