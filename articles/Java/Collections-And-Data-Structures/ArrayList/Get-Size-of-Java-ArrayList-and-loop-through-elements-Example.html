Get Size of Java ArrayList and Loop Through Elements Example

The Java ArrayList class is a powerful data structure that allows developers to store, manipulate, and access data. As with any collection class, it is useful to be able to determine the size of the ArrayList, as well as iterate through the contents of the list. This article provides examples of how to get the size of an ArrayList and loop through its elements.

Getting the Size of an ArrayList in Java

The 
size()
method of the 
ArrayList
class returns the number of elements contained in the list. Here is an example of how to use the 
size()
method: 

          
           
// Create an ArrayList of Strings 
ArrayList<String> myList = new ArrayList<String>();

// Add some elements 
myList.add("Hello"); 
myList.add("World");

// Get the size of the list 
int size = myList.size(); 
System.out.println("Size: " + size); // outputs "Size: 2"
This code creates an 
ArrayList
and adds two elements. The 
size()
method is then used to get the size of the list, which is outputted to the console.

Looping Through Elements in an ArrayList

The most common way to loop through elements in an ArrayList is by using a 

for
loop. Here is an example of how to use a 
for
loop to iterate over the elements in an 
ArrayList
of Strings: 

          
           
// Create an ArrayList of Strings 
ArrayList<String> myList = new ArrayList<String>();

// Add some elements 
myList.add("Hello"); 
myList.add("World");

// Iterate through the list 
for (int i = 0; i < myList.size(); i++) { 
    String element = myList.get(i); 
    System.out.println(element); 
} 

// Outputs "Hello", followed by "World"
The 
for
loop in this example iterates from 
0
to 
myList.size()
(which is 
2
in this case). On each iteration, the 
get()
method is used to get the element at the current index, and then this element is outputted to the console.

Conclusion

In this article, we have seen how to use the 
size()
method to determine the number of elements in an 
ArrayList
, as well as how to loop through the elements in the list. Both of these operations are essential for working with 
ArrayList
s in Java.
