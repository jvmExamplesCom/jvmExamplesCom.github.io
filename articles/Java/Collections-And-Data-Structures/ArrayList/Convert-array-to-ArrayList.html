Converting an Array to an ArrayList

Java provides numerous utilities for working with arrays and collections. Often, you may find yourself needing to convert an array into an ArrayList. This article will demonstrate how to do so in an efficient manner.

Using the ArrayList Constructor

The most straightforward way to convert an array to an ArrayList is through the use of the ArrayList constructor. This method requires the passing of a valid array object as an argument. The following example illustrates this technique: 

          
           
String[] animals = { "dog", "cat", "horse" };
ArrayList<String> animalList = new ArrayList<>(Arrays.asList(animals));
In this code sample, we use the Arrays class in order to convert the 

          
           
animals
array into an 

          
           
ArrayList
. The resulting list 

          
           
animalList
contains the same elements as the original array.

Using the Collections Class

Another option for converting an array to an ArrayList is to use the Collections class. This class contains a number of useful methods for manipulating collections. The 

          
           
addAll
method can be used to append the elements of an array to an existing list. The following example illustrates this technique: 

          
           
String[] animals = { "dog", "cat", "horse" };
ArrayList<String> animalList = new ArrayList<>();
Collections.addAll(animalList, animals);
This code does the same as the previous example, but the resulting list 

          
           
animalList
is created before any elements are added.

Conclusion

In this article, we have demonstrated two techniques for converting an array to an ArrayList in Java. By using either the ArrayList constructor or the Collections class, it is easy to create a list from an array in an efficient manner.
