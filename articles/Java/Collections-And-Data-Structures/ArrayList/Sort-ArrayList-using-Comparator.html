Sort ArrayList Using Comparator in Java

Java provides several ways to sort ArrayList. One of the simplest way to sort ArrayList is using the Collections.sort() method. This method takes Comparator as an argument which can be used to sort the list elements. In this article, we'll explore some of the ways to sort an ArrayList using a Comparator.

Using Comparator to Sort List Elements

To sort an ArrayList in ascending order, you can use the Collections.sort() method. This method takes a Comparator as an argument and uses it to sort the list elements. The following code snippet shows how to use this approach to sort a list of strings in ascending order:


          
           
ArrayList<String> list = new ArrayList<>(); 
list.add("Foo"); 
list.add("Bar"); 
list.add("Moo"); 
Collections.sort(list, Comparator.naturalOrder());
// Output: [Bar, Foo, Moo]

You can also use a custom Comparator to sort elements in a different order. For example, the following code snippet shows how to sort a list of integers in descending order:


          
           
ArrayList<Integer> list = new ArrayList<>(); 
list.add(5); 
list.add(4); 
list.add(3);
Collections.sort(list, Comparator.reverseOrder());
// Output: [5, 4, 3]

By default, the Comparator will sort the elements in ascending order. However, you can also use the compare() method to specify a custom sorting order. For example, the following code snippet shows how to sort a list of strings in reverse alphabetical order:


          
           
ArrayList<String> list = new ArrayList<>(); 
list.add("Foo"); 
list.add("Bar"); 
list.add("Moo"); 
Collections.sort(list, new Comparator<String>() {
    public int compare(String s1, String s2) {
        return s2.compareTo(s1); 
    }
});
// Output: [Moo, Foo, Bar]

Using Lambda Expression to Sort List Elements

Java 8 introduces a new feature called lambda expressions which allows us to create anonymous functions. We can use these lambda expressions to simplify our code and make it more readable. The following code snippet shows how to use a lambda expression to sort a list of integers in ascending order:


          
           
ArrayList<Integer> list = new ArrayList<>(); 
list.add(5); 
list.add(4); 
list.add(3);
list.sort((a, b) -> a.compareTo(b));
// Output: [3, 4, 5]

The lambda expression provided to the sort() method compares two integers and returns a negative value if the first argument is smaller than the second one, a positive value if the first argument is larger than the second one or zero if they are equal. You can also use a lambda expression to sort a list of strings in reverse alphabetical order:


          
           
ArrayList<String> list = new ArrayList<>(); 
list.add("Foo"); 
list.add("Bar"); 
list.add("Moo"); 
list.sort(Comparator.reverseOrder());
// Output: [Moo, Foo, Bar]

Conclusion

In this article, we explored some of the ways to sort an ArrayList using a Comparator. We saw how to use the Collections.sort() method and how to use lambda expressions to simplify our code. Sorting an ArrayList is a common task and it's important to know the different approaches to achieve this.
