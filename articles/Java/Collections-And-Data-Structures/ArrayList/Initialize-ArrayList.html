Initializing an ArrayList in Java

An 
ArrayList
is a type of collection used to store types of objects. It is part of the Java Collections Framework, and it provides various methods to manage the items that are stored within it. Initializing an 
ArrayList
instance allows you to create a new collection that can be used for your program.

Syntax for Creating an ArrayList

The syntax for creating an 
ArrayList
is as follows: 

          
           
ArrayList<Type> list = new ArrayList<>();
Where 
Type
is the type of objects you want to store within the list. You can also use generic wildcards in place of a specific type, such as the following: 

          
           
ArrayList<?> list = new ArrayList<>();

Examples of Initializing an ArrayList

Here are some examples of initializing an 
ArrayList
with different types of objects: 

          
           
// String ArrayList
ArrayList<String> names = new ArrayList<>();

// Integer ArrayList
ArrayList<Integer> numbers = new ArrayList<>();

// Wildcard ArrayList
ArrayList<?> list = new ArrayList<>();

Initializing an ArrayList from an Existing Collection

It’s also possible to initialize an 
ArrayList
from an existing collection. The syntax for this is as follows: 

          
           
ArrayList<Type> list = new ArrayList<>(otherCollection);
Where 
otherCollection
is the existing collection you want to create the 
ArrayList
from. Here’s an example: 

          
           
List<String> names = Arrays.asList("John", "Paul", "George", "Ringo");
ArrayList<String> namesList = new ArrayList<>(names);

Conclusion

In this article, we discussed how to initialize an 

ArrayList
instance in Java. We looked at the syntax for creating a new 
ArrayList
and for initializing one from an existing collection. We also looked at some examples of each.
