Convert array to Set using Arrays

Java provides the Arrays class to help with conversion of an array to a Set. The following code snippet shows how to convert an array to a Set using Arrays: 

          
           
Set<Integer> mySet = Arrays.stream(myArray).collect(Collectors.toSet());
The code example above will convert an array of type integer to a Set. The same concept can be applied to other types of arrays as well.

int[] myArray = {1, 2, 3, 4, 5};

Set<Integer> mySet = Arrays.stream(myArray).collect(Collectors.toSet());
