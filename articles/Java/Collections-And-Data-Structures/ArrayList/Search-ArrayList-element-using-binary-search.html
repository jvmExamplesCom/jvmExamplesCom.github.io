Searching ArrayList Element Using Binary Search

Java's 

          
           
ArrayList
includes a powerful utility method for searching for elements by using a binary search algorithm. Binary search performs a comparison between each element in the list and the sought-after element and checks for a logical match in order to determine its search result. This tutorial will discuss how to use binary search for searching for an element in Java's 

          
           
ArrayList
.

Create a Comparator

Before you can use the binary search utility in ArrayList, you'll need to create a Comparator object. This object is used to compare two elements in the list. The following code example shows how to do this: 

          
           
Comparator<Integer> cmp = new Comparator<Integer>() {
            @Override
            public int compare(Integer a, Integer b) {
                return a - b;
            }
        };
The Comparator object defined above compares two 

          
           
Integer
objects. If a is greater than b, it returns a positive value. If a is less than b, it returns a negative value. If a is equal to b, it returns 0.

Search for Element in an ArrayList

Once you have created the Comparator object, you can use the binary search utility to search for a specific element in an 

          
           
ArrayList
. The following code example illustrates this: 

          
           
int index = Collections.binarySearch(list, 5, cmp);
In this code example, 

          
           
list
is an 

          
           
ArrayList
of integers, and 

          
           
5
is the element that you are searching for. The 

          
           
cmp
variable is the Comparator object that you created earlier. This method returns the index of the element if it is found, or a negative value if it is not found.

Example Usage

Now letâ€™s look at an example of binary search being used to find the index of an element in an 

          
           
ArrayList
. The following code example creates an 

          
           
ArrayList
of integers and uses binary search to find the index of a specific element: 

          
           
import java.util.ArrayList;
import java.util.Collections;
import java.util.Comparator;

public class BinarySearchExample {

    public static void main(String[] args) {
        ArrayList<Integer> list = new ArrayList<>();
        list.add(1);
        list.add(2);
        list.add(3);
        list.add(4);
        list.add(5);

        Comparator<Integer> cmp = new Comparator<Integer>() {
            @Override
            public int compare(Integer a, Integer b) {
                return a - b;
            }
        };

        int index = Collections.binarySearch(list, 4, cmp);
        System.out.println("Index of 4: " + index);
    }
}
The output of this code is: 

          
           
Index of 4: 3
As you can see, the binary search utility was able to find the index of the element 

          
           
4
in the 

          
           
ArrayList
.
