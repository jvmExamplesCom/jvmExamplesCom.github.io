Convert String to ArrayList using Arrays

In Java, there are many methods for converting a String to an ArrayList. One of the most commonly used methods is to use the Arrays class.

Examples

Here are some examples of how to convert a String to an ArrayList using Arrays:


          
           
String[] words = str.split("\\s+");
ArrayList<String> list = new ArrayList<String>(Arrays.asList(words));

          

          
           
list.add(str);

ArrayList<String> list = new ArrayList<>();
