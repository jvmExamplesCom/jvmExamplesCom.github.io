Converting Java TreeSet to Array

Java TreeSet is an implementation of a set data structure. It stores elements in a sorted order, and allows for efficient retrieval of values. It is also possible to convert a Java TreeSet into an array, which can be useful for certain operations. This article will explain how to do this, with code examples and outputs.

Example 1: Simple TreeSet to Array Conversion

In this example, we will create a simple TreeSet and convert it to an array. We start by creating the TreeSet and adding some elements to it:


          
           
TreeSet<String> treeSet = new TreeSet<String>();
treeSet.add("Apple");
treeSet.add("Banana");
treeSet.add("Carrot");
treeSet.add("Date");
Next, we can use the 
toArray()
method to convert the TreeSet to an array: 

          
           
String[] array = treeSet.toArray(new String[treeSet.size()]);

With the above code, we have successfully converted the TreeSet to an array. The resulting array will look like this:


          
           
["Apple", "Banana", "Carrot", "Date"]

public class Fruit {
  private String name;
  private Double price;

  // constructor and getters omitted for brevity

treeSet.add(new Fruit("Carrot", 0.89));
