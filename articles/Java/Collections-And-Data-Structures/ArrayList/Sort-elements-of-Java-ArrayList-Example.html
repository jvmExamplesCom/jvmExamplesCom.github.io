Sorting Elements of Java ArrayList Example

In Java, there is a common requirement to sort elements of an ArrayList according to certain criteria. In this article we will learn how to accomplish this in a few simple steps.

Creating the ArrayList

We can create an ArrayList of integers that we would like to sort using the following code:


          
           
ArrayList<Integer> numbers = new ArrayList<>();
numbers.add(10);
numbers.add(3);
numbers.add(7);
numbers.add(34);
numbers.add(2);

Implementing a Comparator

Now we need to create an implementation of the Comparator interface that will define the sorting criteria. This is done by overriding the compare method of the Comparator interface to specify how two objects should be compared. In this example, we will sort the ArrayList in ascending order. The code for this is given below:


          
           
Comparator<Integer> comparator = new Comparator<Integer>() {
    @Override
    public int compare(Integer o1, Integer o2) {
        return o1.compareTo(o2);
    }
};

Sorting the ArrayList

Once we have implemented our Comparator, we can now use it to sort the ArrayList using the Collections.sort method. This takes the ArrayList and the Comparator as parameters. The code for this is given below:


          
           
Collections.sort(numbers, comparator);

Printing the Sorted ArrayList

We can now print the sorted ArrayList to check that the elements have been ordered correctly using the following code:


          
           
for(Integer number : numbers) {
    System.out.println(number);
}
This will output the following result: 

          
           
2
3
7
10
34

Conclusion

In this article we have seen how to sort an ArrayList of integers using the Collections.sort method and an implementation of the Comparator interface. We have also seen how to print the contents of the sorted ArrayList to verify that it has been sorted correctly.
