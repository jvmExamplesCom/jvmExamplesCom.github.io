Java ArrayList replace Element

The Java ArrayList class is one of the most popular implementations of the List interface. It allows for the dynamic storage of elements and provides utility methods for manipulating its content. This article will demonstrate how to replace an existing element within an ArrayList instance.

Using ArrayList.set(int index, E element)

The easiest way of replacing an existing element within an ArrayList is via the 

          
           
set
method. This method takes two parameters, an index and the object to add to the list. The existing element at the specified index will be replaced with the new object. Here is an example of this method in action: 

          
           
List<String> list = new ArrayList<>();
list.add("foo");
list.add("bar");

list.set(0, "baz");
System.out.println(list); // [baz, bar]

Using ArrayList.remove(int index) + ArrayList.add(int index, E element)

Another approach to replacing an existing element involves using the 

          
           
remove
/ 

          
           
add
combo. These two methods can be used in succession to remove the existing element at the specified index and insert a new object. Here is an example: 

          
           
List<String> list = new ArrayList<>();
list.add("foo");
list.add("bar");

list.remove(0);
list.add(0, "baz");
System.out.println(list); // [baz, bar]
