Sort String Array Using Arrays Class

The 

          
           
java.util.Arrays
class is used to sort the elements of an array. It consists of various static methods which can be used for sorting arrays of different types. This article looks at how to sort a String array using one of these methods.

Sorting a String Array

The 

          
           
sort()
method of the 

          
           
Arrays
class is used to sort an array. It has two overloaded versions, one which sorts an array of objects and another which sorts an array of primitive data types. The version which sorts an array of objects takes an optional parameter which is an object of 

          
           
Comparator
interface. It is used to compare the objects of the array and define their order of arrangement. The version which sorts primitive data types takes an array and its length as parameters. Since a String array is an array of objects, we’ll be using the version which takes an object of 

          
           
Comparator
interface as an argument. Let’s look at an example of how to use it: 

          
           
public static void main(String[] args) {
    String[] arr = { "Java", "Sorting", "Examples" };
    
    Comparator<String> com = new Comparator<String>() {
        public int compare(String s1, String s2) {
            return s1.compareTo(s2);
        }
    };
    
    Arrays.sort(arr, com);
    
    for (String str : arr) {
        System.out.println(str);
    }
}
In the above code, we have created an array of String objects. We have then created an object of the 

          
           
Comparator
interface and implemented the 

          
           
compare()
method. The 

          
           
compare()
method is used to compare the Strings and define their relative order. Finally, we have used the 

          
           
sort()
method with the String array and the comparator object as arguments. The output of the above code is:

Sorting
