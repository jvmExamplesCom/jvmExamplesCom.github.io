Iterate through Elements in Java ArrayList using Iterator Example

In this example, we will discuss how to iterate over elements Java ArrayList using Iterator.

Introduction to ArrayList and Iterator:

ArrayList is a collection class that implements the List interface. It allows us to store elements of different data types in an array-like structure. The Iterator interface is used to traverse through the elements of a collection like the List interface.

Java Program to Iterate over Elements of ArrayList:

Here is a complete Java program to illustrate the iteration over elements of the ArrayList. 

          
           
import java.util.ArrayList;
import java.util.Iterator;

public class IteratorDemo {

   public static void main(String[] args) {

           //Creating an ArrayList of Integer
           ArrayList<Integer> numbers = new ArrayList<Integer>();

           //Adding elements to ArrayList
           numbers.add(10);
           numbers.add(20);
           numbers.add(30);
           numbers.add(40);
           numbers.add(50);

           /*
            * Get iterator instance
            * hasNext() method return true if the list
            * has more elements
            */
           Iterator<Integer> itr = numbers.iterator();
           while(itr.hasNext())
           {
               //Moves the cursor/iterator to next element
               int num = itr.next();
               System.out.println(num);
           }
       }
}

Output:


          
           
10 
20
30
40
50

Explanation:

The above program creates an ArrayList of type Integer and adds five elements to it. It then obtains the iterator object for the list using the iterator() method of the ArrayList class. The while loop is used to iterate over the elements of the list. The hasNext() method of the Iterator class checks if there are more elements in the list. The next() method moves the cursor to the next element in the list and returns the element. Finally, the element is printed to the standard output.
