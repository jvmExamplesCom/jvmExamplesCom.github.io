Convert Comma Separated String to ArrayList Using Arrays in Java

In this article we'll take a look at how to convert a comma separated String to an ArrayList using the Arrays class in Java. We'll also provide some code examples to help you understand how to use this powerful tool.

Creating a Comma Separated String

Before we can use the Arrays class to convert a comma separated String to an ArrayList, we must first create the String itself. This can be done by simply declaring and initializing a String with the values that you wish to separate. Let's take a look at an example: 

          
           
String str = "one,two,three,four,five";
The above code creates a String object called str that contains the values "one", "two", "three", "four" and "five" separated by commas.

Using the Arrays Class

Now that we have the comma separated String, we can use the Arrays class to create the ArrayList. The Arrays class provides the 

          
           
asList()
method to convert an array of objects into a List. We can use this method to convert our comma separated string into an ArrayList like so: 

          
           
String[] items = str.split(",");
ArrayList<String> list = new ArrayList<String>(Arrays.asList(items));
In the above code, we first use the 

          
           
split()
method to split the comma separated string into an array of strings. We then use the 

          
           
asList()
method to convert the array of strings into an ArrayList. The 

          
           
asList()
method takes an array of objects, so we must pass in the array of strings that was created by the 

          
           
split()
method.

Printing Out the ArrayList

Now that we have our ArrayList, we can print out the contents to verify that it was created correctly. To do this, we can use a 

          
           
for
loop to iterate over the ArrayList and print out each item: 

          
           
for (String item : list) {
    System.out.println(item);
}
The output of the above code should be as follows: 

          
           
one
two
three
four
five
As you can see, the items are printed out exactly as they were stored in the comma separated String.

Conclusion

In this article, we discussed how to convert a comma separated String to an ArrayList using the Arrays class in Java. We first created a comma separated String, then used the 

          
           
asList()
method of the Arrays class to create the ArrayList. Finally, we printed out the ArrayList to verify that it was created correctly.
