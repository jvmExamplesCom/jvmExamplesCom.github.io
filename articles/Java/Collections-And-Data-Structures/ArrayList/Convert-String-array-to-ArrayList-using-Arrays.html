Convert String Array to ArrayList Using Arrays

Java provides the Arrays class which has several useful static methods for manipulating arrays. One of the most common uses of these methods is to convert a Java String array to an ArrayList. This tutorial will demonstrate how to convert a string array to an ArrayList using the Arrays class.

Example 1 – Convert String Array to ArrayList using Arrays.asList()

The easiest way to convert a string array to an ArrayList is to use the Arrays.asList() static method. This method takes an array of objects and returns a fixed-size list wrapper over the provided array. 

          
           
String[] colors = {"red", "green", "blue"};
List<String> colorsList = Arrays.asList(colors);
System.out.println(colorsList);
The code above will produce the following output: 

          
           
[red, green, blue]

Example 2 – Convert String Array to ArrayList using for loop

We can also convert a string array to an ArrayList by iterating through each element of the string array and adding it to the ArrayList. 

          
           
String[] numbers = {"one", "two", "three"};
List<String> numbersList = new ArrayList<>();
for (String number : numbers) {
    numbersList.add(number);
}
System.out.println(numbersList);
The code above will produce the following output: 

          
           
[one, two, three]

Example 3 – Convert String Array to ArrayList using Collection.addAll()

The Collection.addAll() method allows us to add all elements in a string array to a list. 

          
           
String[] fruits = {"apple", "banana", "cherry"};
List<String> fruitsList = new ArrayList<>();
Collections.addAll(fruitsList, fruits);
System.out.println(fruitsList);
The code above will produce the following output: 

          
           
[apple, banana, cherry]
