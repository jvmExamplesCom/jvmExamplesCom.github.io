Get ArrayList Length in Java

Arrays are a fundamental part of Java, and ArrayList is one of the most popular collections. Knowing how to get the length of an ArrayList is an important skill for any Java programmer. In this article, we will go over the different ways to get the length of an ArrayList in Java and discuss the pros and cons of each approach.

Using size() Method

The easiest way to get the length of an ArrayList is to use the size() method. The size() method returns the number of elements in the ArrayList. Here is an example of how to use the size() method: 

          
           
ArrayList<String> list = new ArrayList<>();
list.add("Hello");
list.add("World");
int size = list.size();
System.out.println("Size of list: " + size);
// Output: Size of list: 2
Using the size() method is easy and straightforward, but it has some drawbacks. The size() method has to iterate over each element in the list in order to count the elements. This can be a problem for large lists as it may take a long time to count the elements.

Using .length Property

Another way to get the length of an ArrayList is to use the .length property. This is not an actual method on the ArrayList, but is a property of the underlying array that stores the ArrayList data. Here is an example that shows how to use the .length property to get the size of an ArrayList: 

          
           
String[] array = {"Hello", "World"};
ArrayList<String> list = new ArrayList<>(Arrays.asList(array));
int size = list.toArray().length;
System.out.println("Size of list: " + size);
// Output: Size of list: 2
Using the .length property is faster than using the size() method, but it has a couple of drawbacks. First, since the .length property relies on an internal array, it is not a reliable way to get the length of an ArrayList if the list is modified after the array was created. Second, it requires the list to be converted to an array first, which can be inefficient.

Using Iterator

The Iterator interface is another way to get the length of an ArrayList. The Iterator interface allows you to loop through the elements of an ArrayList and count the elements. Here is an example of how to use the Iterator interface to get the length of an ArrayList: 

          
           
ArrayList<String> list = new ArrayList<>();
list.add("Hello");
list.add("World");

int size = 0;
Iterator<String> it = list.iterator();
while (it.hasNext()) {
    it.next();
    size++;
}
System.out.println("Size of list: " + size);
// Output: Size of list: 2
Using the Iterator interface is more efficient than using the size() method, but it requires more code and can be a bit more difficult to understand.

Conclusion

In this article, we have gone over the different ways to get the length of an ArrayList in Java. We discussed the advantages and disadvantages of each approach and how to use them in practice. In most cases, the size() method is the simplest and most efficient way to get the length of an ArrayList, but if you are dealing with large lists or need to get the length quickly, then the .length property or the Iterator interface may be the better options.
