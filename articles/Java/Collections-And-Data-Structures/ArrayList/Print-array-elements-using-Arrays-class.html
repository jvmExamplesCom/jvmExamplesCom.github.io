Printing Array Elements Using Arrays Class

The Arrays class provides various methods for manipulating and printing array elements. This article will explain how to use the Arrays class to print array elements in Java.

Example 1: Printing an Integer Array

Letâ€™s start with a simple example. We have an integer array declared as following:

We can print its elements using the Arrays class like this:


          
           
System.out.println(Arrays.toString(arr));
This code will produce the following output: 

          
           
[1, 2, 3, 4, 5]
