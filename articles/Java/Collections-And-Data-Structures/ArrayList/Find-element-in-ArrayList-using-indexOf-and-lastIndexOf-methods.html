Find Element in ArrayList using indexOf and lastIndexOf methods

Introduction

Java provides several useful ways to find an element inside the ArrayList. The two most commonly used methods are indexOf() and lastIndexOf(). Both these methods return the index of the given element in the ArrayList.

indexOf Method

The indexOf() method is used to finds the first occurrence of the specified element in this list. It searches the list from index position 0 till the end and returns the first matching element. Syntax: 

          
           
int indexOf(Object o)
Example: 

          
           
import java.util.ArrayList;

public class Main {
   public static void main(String args[]) {
      // Create an array list
      ArrayList<String> al = new ArrayList<String>();

      // Add elements to the array list
      al.add("John");
      al.add("Jack");
      al.add("Sam");

      // Print the array list
      System.out.println("ArrayList: " + al);

      // Get the index of "Jack"
      int pos = al.indexOf("Jack");

      // Print the index
      System.out.println("Index of 'Jack': " + pos);
   }
}
Output: 

          
           
ArrayList: [John, Jack, Sam]
Index of 'Jack': 1

lastIndexOf Method

The lastIndexOf() method is used to finds the last occurrence of the specified element in this list. It searches the list from last index position and returns the last matching element. Syntax: 

          
           
int lastIndexOf(Object o)
Example: 

          
           
import java.util.ArrayList;

public class Main {
   public static void main(String args[]) {
      // Create an array list
      ArrayList<String> al = new ArrayList<String>();

      // Add elements to the array list
      al.add("John");
      al.add("Jack");
      al.add("Sam");
      al.add("Jack");

      // Print the array list
      System.out.println("ArrayList: " + al);

      // Get the last index of "Jack"
      int pos = al.lastIndexOf("Jack");

      // Print the index
      System.out.println("Last Index of 'Jack': " + pos);
   }
}
Output: 

          
           
ArrayList: [John, Jack, Sam, Jack]
Last Index of 'Jack': 3
