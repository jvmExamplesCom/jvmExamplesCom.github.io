Java ArrayList Example

The Java ArrayList is a powerful tool for storing data in an organized and efficient fashion. It allows you to store objects of different types in the same collection, which can be accessed quickly and easily. In this article, we will look at an example of how to use an ArrayList in Java.

Creating an ArrayList

The first step in using an ArrayList is creating one. To do this, you use the 'new' keyword, followed by the type of list you want to create. For example, to create an ArrayList of integers, you would write: 

          
           
ArrayList<Integer> myList = new ArrayList<>();
This creates an empty ArrayList that can hold Integer objects.

Adding Elements

Once you have created your ArrayList, you can start adding elements to it. There are several ways to do this. The simplest is to use the add() method. This takes an object as an argument and adds it to the end of the list. For example: 

          
           
myList.add(10); //Add the number 10 to the list
myList.add("Hello"); //Add the string "Hello" to the list
You can also add multiple elements at once, using the addAll() method. This takes an Iterable such as a Collection, List, or Set as an argument and adds all of its elements to the ArrayList. For example: 

          
           
Set<String> mySet = new HashSet<>();
mySet.add("Foo");
mySet.add("Bar");

myList.addAll(mySet); //Adds both "Foo" and "Bar" to the list

Accessing Elements

Once you have added elements to your ArrayList, you can access them using the get() method. This takes an index as an argument and returns the element at that index. For example: 

          
           
int firstElement = myList.get(0); //Returns the first element in the list
You can also use the indexOf() method to find the index of a given element. This takes an element as an argument and returns its index in the list. For example: 

          
           
int index = myList.indexOf("Hello"); //Returns the index of the element "Hello"

Removing Elements

To remove an element from an ArrayList, you can use the remove() method. This takes an index or an element as an argument, and removes the corresponding element from the list. For example: 

          
           
myList.remove(10); //Removes the element with the value 10
myList.remove("Hello"); //Removes the element with the value "Hello"
You can also remove multiple elements at once, using the removeAll() method. This takes an Iterable such as a Collection, List, or Set as an argument and removes all of its elements from the ArrayList. For example: 

          
           
Set<String> mySet = new HashSet<>();
mySet.add("Foo");
mySet.add("Bar");

myList.removeAll(mySet); //Removes both "Foo" and "Bar" from the list

Iterating Over Elements

To iterate over all the elements in an ArrayList, you can use an enhanced for loop. This takes the form of: 

          
           
for (Type element : list){
  //Do something with element
}
For example, to print all the elements in an ArrayList, you could use: 

          
           
for (Object element : myList){
  System.out.println(element);
}
Alternatively, you can use the forEach() method to iterate over the elements of the list. This takes a Consumer as an argument and runs it on each element in the list. For example: 

          
           
myList.forEach(e -> System.out.println(e)); //Prints each element of the list

Conclusion

In this article, we looked at an example of how to use an ArrayList in Java. We created an ArrayList, added elements to it, accessed elements from it, removed elements from it, and iterated over it. Hopefully, this has given you a better understanding of how to use this powerful data structure.
