Print ArrayList Example in Java

Java programming language provides a data structure, the array list, which stores a fixed-size sequential collection of elements of the same type. An array list is similar to an array, but there are no size restrictions. Elements can be added or removed from an array list. In this article we will discuss how to print an array list.

Print ArrayList using for loop

The most common way to print an ArrayList is to use a for loop. For example: 

          
           
ArrayList<String> fruits = new ArrayList<String>();
fruits.add("Apple"); 
fruits.add("Banana");
fruits.add("Orange"); 

for (String fruit : fruits) {
   System.out.println(fruit);
}
Output: Apple Banana Orange

Print ArrayList using forEach() method

The forEach() method is another way to traverse the ArrayList. The forEach() method takes a single parameter which is a functional interface. This interface has a single method named accept() that takes a single parameter and returns nothing. The code snippet below shows how to use the forEach() method to print an ArrayList. 

          
           
ArrayList<String> fruits = new ArrayList<String>();
fruits.add("Apple"); 
fruits.add("Banana");
fruits.add("Orange"); 

fruits.forEach(fruit -> System.out.println(fruit));
Output: Apple Banana Orange

Print ArrayList using iterator() method

The iterator() method returns an iterator object which can be used to traverse the ArrayList. The iterator has methods hasNext() and next() which can be used to iterate through the ArrayList. The code snippet below shows how to use the iterator() method to print an ArrayList. 

          
           
ArrayList<String> fruits = new ArrayList<String>();
fruits.add("Apple"); 
fruits.add("Banana");
fruits.add("Orange"); 

Iterator<String> itr = fruits.iterator();
while (itr.hasNext()) {
   System.out.println(itr.next());
}
Output: Apple Banana Orange

Print ArrayList using toString() method

The toString() method is defined in the java.util.AbstractCollection class which is the parent class of ArrayList. The toString() method returns a string representation of the object. The code snippet below shows how to use the toString() method to print an ArrayList. 

          
           
ArrayList<String> fruits = new ArrayList<String>();
fruits.add("Apple"); 
fruits.add("Banana");
fruits.add("Orange"); 

System.out.println(fruits.toString());
Output: [Apple, Banana, Orange]
