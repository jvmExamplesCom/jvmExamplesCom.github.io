Java ArrayList of Arrays

ArrayLists are a generic form of a List interface in Java. They store elements of various types and have the ability to resize themselves dynamically. An ArrayList can also contain another ArrayList, allowing us to create an array of arrays within a single ArrayList. Generally, the purpose of using an array of arrays is to improve space utilization by arranging data in a matrix-like format.

Creating an empty ArrayList of Arrays

You can create an empty ArrayList of arrays with the following code: 

          
           
ArrayList<Object[]> list = new ArrayList<Object[]>();
This ArrayList will now be able to hold any type of array.

Adding Arrays to an ArrayList of Arrays

We can add any array to our list with the following code: 

          
           
int[] array1 = {1, 2, 3};
list.add(array1);
You can add multiple arrays in the same manner.

Accessing Elements from an ArrayList of Arrays

You can access elements from an ArrayList of Arrays with the following code: 

          
           
Object[] array2 = list.get(0);
int element = array2[1]; // element is 2
Here, we are accessing the element at position 1 in the first array in the list.

Removing Elements from an ArrayList of Arrays

You can remove elements from an ArrayList of Arrays with the following code: 

          
           
list.remove(0); // Removes the first array in the list

Iterating Over an ArrayList of Arrays

You can iterate over an ArrayList of Arrays with the following code: 

          
           
for (Object[] arr : list) {
   // arr is a reference to the current array
   for (Object el : arr) {
      // el is a reference to the current element
   }
}
In this example, we are iterating over each array in the list and then each element in each array.

Sorting an ArrayList of Arrays

You can sort an ArrayList of Arrays with the following code: 

          
           
Collections.sort(list, new Comparator<Object[]>() {
   @Override
   public int compare(Object[] a, Object[] b) {
      // Compare two arrays here
   }
});
In this example, we are using a custom Comparator to sort the list according to the logic we define in the compare method.
