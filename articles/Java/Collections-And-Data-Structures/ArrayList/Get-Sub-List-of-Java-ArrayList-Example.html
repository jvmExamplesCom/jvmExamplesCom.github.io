Get Sub List of Java ArrayList Example

In this article we'll provide an example of how to get a sub list from an 

          
           
ArrayList
in Java. We'll go over the process step-by-step, including example code and output.

Create the Main ArrayList

Let's look at the following code example: 

          
           
ArrayList<String> names = new ArrayList<String>();

names.add("Paul"); 
names.add("Ben");
names.add("John");
names.add("James");
names.add("David");
names.add("Sam");
names.add("Roger");
names.add("Joe");
Here, we create the main 

          
           
ArrayList
called 

          
           
names
. This list contains eight different strings.

Sublist Method

To get a sub list from this 

          
           
ArrayList
, we use the 

          
           
subList()
method. This method takes two parameters: the starting index (inclusive) and the ending index (exclusive). Let's look at an example of using the 

          
           
subList()
method. 

          
           
ArrayList<String> sublist = new ArrayList<String>(names.subList(2, 6));
The above code will create a new 

          
           
ArrayList
called 

          
           
sublist
. This list will contain elements from the original 

          
           
names
list, starting at index 2 and ending before index 6. The resulting list contains the following elements:
