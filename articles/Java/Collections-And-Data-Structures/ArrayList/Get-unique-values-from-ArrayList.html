Get Unique Values from ArrayList in Java

Using Java to get the unique values from an ArrayList can be done in multiple ways. In this article, we will go through several examples to showcase different approaches that can be used to achieve the desired result.

Using a Set

An easy way to get the unique values from an ArrayList is to first convert it into a set. Sets are a collection of elements that only contain unique values. So, by converting the ArrayList elements into a set, any duplicate values present in the list will be automatically removed.


          
           
List<Integer> numbers = new ArrayList<>();
numbers.add(1);
numbers.add(2);
numbers.add(3);
numbers.add(4);
numbers.add(4);

Set<Integer> uniqueNums = new HashSet<>(numbers);
System.out.println(uniqueNums); // Outputs [1, 2, 3, 4]

Using a Loop

Another way to get the unique values from an ArrayList is to use a loop. In this approach, a loop is used to iterate over each element in the list and a set is used to check if the element already exists in the list. If not, then the element is added to the set and the set is returned.


          
           
List<Integer> numbers = new ArrayList<>();
numbers.add(1);
numbers.add(2);
numbers.add(3);
numbers.add(4);
numbers.add(4);

Set<Integer> uniqueNums = new HashSet<>();
for (int num : numbers) {
    if (!uniqueNums.contains(num)) {
        uniqueNums.add(num);
    }
}
System.out.println(uniqueNums); // Outputs [1, 2, 3, 4]

Using Java 8 Streams

Java 8 Streams can also be used to get the unique values from an ArrayList. In this approach, the stream() method is used to create a Stream of elements from the list and then the distinct() method is used to filter out any duplicate elements.


          
           
List<Integer> numbers = new ArrayList<>();
numbers.add(1);
numbers.add(2);
numbers.add(3);
numbers.add(4);
numbers.add(4);

Set<Integer> uniqueNums = numbers.stream().distinct().collect(Collectors.toSet());
System.out.println(uniqueNums); // Outputs [1, 2, 3, 4]

In conclusion, there are multiple ways to get the unique values from an ArrayList in Java. Each approach has its own advantages and disadvantages, so it is best to choose the one that suits your needs the most.
