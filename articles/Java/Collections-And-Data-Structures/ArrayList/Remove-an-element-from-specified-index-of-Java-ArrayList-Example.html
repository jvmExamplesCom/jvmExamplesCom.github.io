Remove an Element From Specified Index of Java ArrayList Example

Java provides many different tools for manipulating elements of an 

          
           
ArrayList
. In this article, we will look at how to remove an element from a specified index in a Java ArrayList using the 

          
           
remove()
method.

ArrayList Remove() Method

The 

          
           
remove()
method is used to remove the element at a specified index of an 

          
           
ArrayList
. This method takes one parameter, which is the index of the element to be removed. The syntax of the remove() method is as follows: 

          
           
ArrayList.remove(int index)
The following is an example of using the remove() method to delete an element from a specific index from an 

          
           
ArrayList
: 

          
           
ArrayList<String> colors = new ArrayList<>(); 
colors.add("Red"); 
colors.add("Blue"); 
colors.add("Green"); 

// Remove the element at index 1 
colors.remove(1); 

// Prints [Red, Green] 
System.out.println(colors);
In this example, we have created an 

          
           
ArrayList
named 

          
           
colors
and added three different colored strings to it. We then used the 

          
           
remove()
method to remove the element at index 1, which is the string 

          
           
Blue
. The final 

          
           
ArrayList
contains only two elements: 

          
           
Red
and 

          
           
Green
.

Removing Multiple Elements From an ArrayList

The 

          
           
remove()
method can also be used to remove multiple elements from an 

          
           
ArrayList
. To do this, you will need to pass in a 

          
           
List
of indexes that represent the elements to be removed from the 

          
           
ArrayList
. The syntax for the remove() method to remove multiple elements from an 

          
           
ArrayList
is as follows: 

          
           
ArrayList.removeAll(Collection c)
Where 

          
           
c
is a 

          
           
Collection
(e.g. a 

          
           
List
) of indexes representing the elements to be removed. The following is an example of using the remove() method to delete multiple elements from an 

          
           
ArrayList
: 

          
           
ArrayList<String> colors = new ArrayList<>(); 
colors.add("Red"); 
colors.add("Blue"); 
colors.add("Green"); 
colors.add("Yellow"); 

// Create a List containing the indexes of the elements to be removed 
List<Integer> indexesToRemove = Arrays.asList(1, 3); 

// Remove the elements at the specified indexes 
colors.removeAll(indexesToRemove); 

// Prints [Red, Green] 
System.out.println(colors);
In this example, we have created an 

          
           
ArrayList
containing four colored strings and then used the 

          
           
removeAll()
method to remove the elements at the specified indexes, which are represented by the 

          
           
List
named 

          
           
indexesToRemove
. The final 

          
           
ArrayList
contains only two elements: 

          
           
Red
and 

          
           
Green
.

remove()

ArrayList
. We have also seen how to use the 

          
           
removeAll()
