Convert Array to LinkedHashSet using Arrays Class

The Arrays class of the Java collections framework provides the asList() method which makes it very easy to convert an array into a LinkedHashSet.

Example 1: Convert Integer Array to LinkedHashSet

In the following example we will convert an integer array to LinkedHashSet.


          
           
import java.util.*;

public class Main {

    public static void main(String[] args) {

        int[] arr = {1,2,3,4,5};

        // Converting array to LinkedHashSet
        LinkedHashSet<Integer> lhSet = new LinkedHashSet<>(Arrays.asList(arr));

        System.out.println("LinkedHash Set : " + lhSet);
    }
}
Output: 

          
           
LinkedHash Set : [1, 2, 3, 4, 5]

Example 2: Convert String Array to LinkedHashSet

In the following example we will convert an string array to LinkedHashSet.


          
           
import java.util.*;

public class Main {

    public static void main(String[] args) {

        String[] arr = {"one", "two", "three", "four", "five"};

        //Converting array to LinkedHashSet
        LinkedHashSet<String> lhSet = new LinkedHashSet<>(Arrays.asList(arr));

        System.out.println("LinkedHash Set : " + lhSet);
    }
}
Output: 

          
           
LinkedHash Set : [one, two, three, four, five]
