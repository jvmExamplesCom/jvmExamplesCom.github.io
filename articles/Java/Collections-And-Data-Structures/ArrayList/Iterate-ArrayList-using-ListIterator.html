Iterating ArrayList Using ListIterator

The 

          
           
ListIterator
interface is a powerful tool for iterating an 

          
           
ArrayList
. It provides a cursor-based mechanism for accessing elements of the list, allowing both forward and backward movement through the list. This article will explain how to use 

          
           
ListIterator
to iterate over an 

          
           
ArrayList
, with code examples.

Instantiating the ListIterator Object

To use a ListIterator, you must first instantiate it. To do this, you must call the 

          
           
listIterator()
method on the 

          
           
ArrayList
object. The result of this method call is the 

          
           
ListIterator
object, which can then be used to traverse the list. 

          
           
ArrayList<String> myList = new ArrayList<>();
ListIterator<String> listIterator = myList.listIterator();

Iterating Through the List

Once the 

          
           
ListIterator
object has been instantiated, it can be used to iterate over the 

          
           
ArrayList
. There are two methods that can be used to move through the list: the 

          
           
next()
method and the 

          
           
previous()
method. The 

          
           
next()
method moves the iterator forward one position in the list, while the 

          
           
previous()
method moves it backward. The 

          
           
next()
and 

          
           
previous()
methods both return the element at the current position of the iterator. The element is returned as an 

          
           
Object
, so it must be cast to its correct type before it can be used. 

          
           
while (listIterator.hasNext()) {        
    Object element = listIterator.next();
    String strElement = (String) element;
    System.out.println(strElement);    
}
The 

          
           
hasNext()
method returns 

          
           
true
if the iterator is at a valid element, and 

          
           
false
if it has reached the end of the list. For this reason, it is often used in conjunction with the 

          
           
next()
method to loop over all the elements in a list. 

          
           
while (listIterator.hasPrevious()) {       
    Object element = listIterator.previous();  
    String strElement = (String) element;
    System.out.println(strElement);    
}
The 

          
           
hasPrevious()
method works just like the 

          
           
hasNext()
method, but it returns true if the iterator is at a valid element, and false if it has reached the beginning of the list. The ListIterator can also be used to modify the list. The 

          
           
add()
method inserts an element at the current position of the iterator, and the 

          
           
remove()
method deletes the element at the current position. 

          
           
//Insert element at the current position
listIterator.add("john");

//Delete element at the current position
listIterator.remove();

Conclusion

The ListIterator provides a powerful mechanism for iterating an ArrayList. With the ListIterator, it is easy to traverse the list in both directions, as well as modify the list.
