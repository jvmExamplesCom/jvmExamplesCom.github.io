Shuffling Elements of ArrayList in Java

Shuffling the order of elements within an ArrayList can be a useful technique in many programming tasks. Fortunately, there is an easy method built into the Java language called Collections.shuffle() that can quickly and easily randomize the elements of an ArrayList. In this article, we will go over how to use the Collections.shuffle() method and take a look at some code examples demonstrating its application.

Using the Collections.shuffle() Method

The Collections.shuffle() method is part of the java.util package, so the first line of any code example should be the following to import the necessary classes: 

          
           
import java.util.ArrayList;
import java.util.Collections;
Once these are imported, the ArrayList can be declared and populated with elements in any way desired. For example: 

          
           
ArrayList<String> animals = new ArrayList<String>();
animals.add("cat");
animals.add("dog");
animals.add("mouse");
animals.add("rabbit");
animals.add("turtle");
To shuffle the elements of this ArrayList, the Collections.shuffle() method needs to be passed the variable containing the list as an argument. For example: 

          
           
Collections.shuffle(animals);
After executing this code, the order of the elements in the ArrayList will be rearranged. To demonstrate this more clearly, the following example shows a before and after output of the ArrayList’s contents: 

          
           
System.out.println("Before Shuffling: " + animals);
Collections.shuffle(animals);
System.out.println("After Shuffling: " + animals);
// Output 
// Before Shuffling: [cat, dog, mouse, rabbit, turtle]
// After Shuffling: [rabbit, mouse, cat, turtle, dog]

Randomizing Elements with a Random Object

It is also possible to randomize the elements of the ArrayList using a Random object. This method of shuffling the elements can be used when the desired outcome is a more random distribution than what the Collections.shuffle() method can provide. The Random object needs to be instantiated with the new keyword before it can be used. For example: 

          
           
Random rand = new Random();
Then, a for loop can be established that will iterate through each element of the ArrayList and reassign its index value: 

          
           
for(int i = 0; i < animals.size(); i++) {
    int randomIndex = rand.nextInt(animals.size());
    String temp = animals.get(randomIndex);
    animals.set(randomIndex, animals.get(i));
    animals.set(i, temp);
}
Here, the rand.nextInt() method is used to generate a random integer between 0 and the size of the ArrayList. This number is assigned to an integer named randomIndex. Then the value of the element at the current index is saved to a temporary variable named temp. The value at randomIndex is then set to the element at the current index, and the value stored in temp is set to the element at randomIndex. For example, if the first iteration of the loop occurred with i = 0, randomIndex was assigned an integer of 3, and the ArrayList looked like this: 

          
           
[cat, dog, mouse, rabbit, turtle]
Then the value of “cat” would be saved to a variable named temp, the value of “rabbit” would be set to the position at index 0, and the value of “cat” stored in temp would be set at index 3. Once the loop has completed its iterations, the elements of the ArrayList will be shuffled. To demonstrate, the following example shows a before and after output of the ArrayList’s contents: 

          
           
System.out.println("Before Shuffling: " + animals);
for(int i = 0; i < animals.size(); i++) {
    int randomIndex = rand.nextInt(animals.size());
    String temp = animals.get(randomIndex);
    animals.set(randomIndex, animals.get(i));
    animals.set(i, temp);
}
System.out.println("After Shuffling: " + animals);
// Output 
// Before Shuffling: [cat, dog, mouse, rabbit, turtle]
// After Shuffling: [turtle, mouse, dog, rabbit, cat]

Conclusion

In this article, we have gone over how to use the Collections.shuffle() method to shuffle an ArrayList in Java. We have also shown how to use a Random object to achieve a similar result. After reading this article, you should have a better understanding of how to randomize the elements within an ArrayList in Java.
