Search an Element of Java ArrayList Example

Java collections such as ArrayList are immensely helpful when dealing with large amounts of data. It helps us to store, access, and manipulate our data with ease. One of the many operations we can perform on the collections is the search operation. In this article, we will be looking at an example of searching an element of a Java ArrayList.

Creating an ArrayList

In order to search an element of a Java ArrayList, we first need to create an ArrayList. This can be done in the following way. 

          
           
ArrayList<Integer> myList = new ArrayList<Integer>();
The above code creates an ArrayList called myList that can store Integer objects.

Adding Elements to the ArrayList

Once the ArrayList has been created, we can then add elements to it. Elements can be added using the add() method. 

          
           
myList.add(12);
myList.add(43);
myList.add(56);
myList.add(19);
myList.add(67);
The above code adds five Integer objects to the myList ArrayList.

Searching an Element of an ArrayList

Now that we have an ArrayList filled with elements, we can search for an element. The most common way to search an element of an ArrayList is to use the contains() method. The contains() method takes in one argument, which is the element to search for. 

          
           
boolean result = myList.contains(56);
The above code searches for the element 56 in the myList ArrayList and stores the result in the boolean variable result. If the element is found in the ArrayList, the result will be true and if the element is not found in the ArrayList, the result will be false.

Output

Now that we have searched for an element, we can output the result. We can use the System.out.println() method to output the result. 

          
           
System.out.println(result);
The above code outputs the boolean result variable, which will be either true or false depending on whether the element was found in the ArrayList or not.

Conclusion

In this article, we looked at an example of searching an element of a Java ArrayList. We first created an ArrayList and added elements to it. Then we used the contains() method to search for an element in the ArrayList. Finally, we output the result of the search.
