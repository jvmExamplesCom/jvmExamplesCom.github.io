Get first and last elements from ArrayList

ArrayList is a great data structure for storing collections of objects. It is powerful, easy to use and provides efficient access to array elements, including the first and last element. In this article, we will look at how to get the first and last element from an ArrayList in Java.

How to Get First Element from ArrayList

To get the first element of an ArrayList in Java, we will use the get() method of the ArrayList class. This method requires two parameters. The first parameter is the index of the element that we want to retrieve, and the second parameter is an optional default value, which will be returned if there are no elements at the given index.

For example, if we want to get the first element from an ArrayList named “list”, we can do the following:


          
           
Object first = list.get(0);

This code will retrieve the first element from the list, and store it in a variable named “first”. If there are no elements at index 0, then the default value (which is null) will be returned.

How to Get Last Element from ArrayList

To get the last element from an ArrayList, we will use the size() method of the ArrayList class. This method returns the number of elements stored in the list, which can then be used to calculate the index of the last element.

For example, if we have a list named “list” with 10 elements, we can use the following code to get the last element:


          
           
int lastIndex = list.size() - 1;
Object last = list.get(lastIndex);

This code will get the size of the list (which is 10 in this case), subtract 1 from it (since arrays are zero-indexed) and then use this index to get the last element in the list. The element will be stored in a variable called “last”.

Conclusion

In this article, we looked at how to get the first and last element from an ArrayList in Java. We saw that we can use the get() method to get the first element, and we can use the size() method to get the last element. Using these methods, we can efficiently access array elements at any index.
