Clearing and Empty an ArrayList in Java

ArrayList is a popular choice of collection class in Java to store dynamic arrays of objects. It is part of the Java Collection Framework and implements the List interface. While we normally use ArrayList to store elements, it is also possible to clear or empty an ArrayList using various methods.

Example 1: Using the clear() Method

The easiest way to empty an arraylist is to use the clear() method which removes all elements from the list and sets its size to 0. This method does not accept any parameters and returns nothing.


          
           
import java.util.ArrayList; 
public class Main { 
   public static void main(String[] args){ 
      ArrayList<String> list = new ArrayList<String>(); 
      list.add("Hello"); 
      list.add("World"); 
      list.add("!"); 

      // Print the list 
      System.out.println("Original list: " + list); 

      // Clear the list 
      list.clear(); 

      // Print the list 
      System.out.println("List after clearing: " + list); 
   } 
}

Output:
Original list: [Hello, World, !]
List after clearing: []

Example 2: Using the removeAll() Method

Another way to empty an arraylist is to use the removeAll() method which removes all elements from the list and returns true if the list was modified. This method accepts Collection as an argument, so you can pass an empty collection to this method to remove all elements.


          
           
import java.util.ArrayList; 
import java.util.Collections; 
public class Main { 
   public static void main(String[] args){ 
      ArrayList<String> list = new ArrayList<String>(); 
      list.add("Hello"); 
      list.add("World"); 
      list.add("!"); 

      // Print the list 
      System.out.println("Original list: " + list); 

      // Remove all elements from the list 
      list.removeAll(Collections.emptyList()); 

      // Print the list 
      System.out.println("List after clearing: " + list); 
   } 
}

Output:
Original list: [Hello, World, !]
List after clearing: []

Example 3: Using the retainAll() Method

The retainAll() method can also be used to empty an arraylist. This method removes all elements from the list that are not present in the specified collection and returns true if the list was modified.


          
           
import java.util.ArrayList; 
import java.util.Collections; 
public class Main { 
   public static void main(String[] args){ 
      ArrayList<String> list = new ArrayList<String>(); 
      list.add("Hello"); 
      list.add("World"); 
      list.add("!"); 

      // Print the list 
      System.out.println("Original list: " + list); 

      // Remove all elements from the list 
      list.retainAll(Collections.emptyList()); 

      // Print the list 
      System.out.println("List after clearing: " + list); 
   } 
}

Output:
Original list: [Hello, World, !]
List after clearing: []

Example 4: Using for-each Loop

The for-each loop can also be used to empty an arraylist. This loop iterates through all elements in the list and removes each element one by one. In order to do this, you need to use the remove() method which removes the specified element from the list and returns true if the element was removed successfully.


          
           
import java.util.ArrayList; 
public class Main { 
   public static void main(String[] args){ 
      ArrayList<String> list = new ArrayList<String>(); 
      list.add("Hello"); 
      list.add("World"); 
      list.add("!"); 

      // Print the list 
      System.out.println("Original list: " + list); 

      // Iterate through the list and remove each element 
      for(String str: list) { 
         list.remove(str); 
      } 

      // Print the list 
      System.out.println("List after clearing: " + list); 
   } 
}

Output:
Original list: [Hello, World, !]
List after clearing: []

Conclusion

In this tutorial, we looked at different ways of clearing or emptying an ArrayList in Java, including the clear() method, removeAll() method, retainAll() method and for-each loop. Each of these methods has its advantages and disadvantages, so you should choose the most suitable one based on your requirements.
