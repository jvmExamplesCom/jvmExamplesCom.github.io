Remove the Last Element from ArrayList

In Java, it is sometimes necessary to remove elements from an array list. Removing the last element from an array list is one such example. This article explains how you can remove the last element from an ArrayList in Java.

Example 1: Using the ArrayList.remove() Method

The easiest and most straightforward way of removing the last element from an ArrayList is to use the 

ArrayList.remove()
method. This method takes an int parameter, which is the index of the element to be removed. Since the last element of the list has the highest index, we can simply use the size of the list as the parameter in the remove method: 

          
           
ArrayList<Integer> list = new ArrayList<>();
list.add(1);
list.add(2);
list.add(3);

// Remove the last element (with index = 2)
list.remove(list.size() - 1);

// Print the list
System.out.println(list);  // [1, 2]

Another way of removing the last element from an ArrayList is to use the 

ArrayList.subList()
method. This method creates a view of the original list, starting at the specified index and ending at the index before the specified index. We can use this method to create a view of the original list without its last element, and then assign it to the original list: 

          
           
ArrayList<Integer> list = new ArrayList<>();
list.add(1);
list.add(2);
list.add(3);

// Create a sublist without the last element (index 2)
ArrayList<Integer> sublist = new ArrayList<>(
    list.subList(0, list.size() - 1));

// Assign the sublist to the original list
list = sublist;

// Print the list
System.out.println(list);  // [1, 2]

Iterator.remove()
