How to Iterate through List using ListIterator in Java

ListIterator is a built-in interface of Java that provides the iterator and listIterator methods for traversing the list element. It is used to traverse the element in both the forward and backward direction. This interface provides a set of methods to access the elements one by one and allows to modify the list during the Iteration.

Creating ListIterator Instance

To create a ListIterator instance, we need to use its listIterator() method that returns the instance of ListIterator. This method signature is as follows: 

          
           
public ListIterator<E> listIterator(int index)
where index is the starting position of the element. Now let us look at an example of creating ListIterator: 

          
           
List<String> list = new ArrayList<>();
list.add("Geeks");
list.add("for");
list.add("Geeks");
 
// Creating a listIterator
ListIterator<String> i = list.listIterator(0);
The code snippet above will create a listIterator object i that will start iterating the list from the 0th index.

Methods of ListIterator Interface

The various methods provided by the ListIterator interface are as follows:
  • boolean hasNext() – returns true if there is an element next to the current element.
  • boolean hasPrevious() – returns true if there is an element before the current element.
  • E next() – returns the value of the next element.
  • E previous() – returns the value of the previous element.
  • void add(E e) – adds the specified element to the list.
  • void remove() – removes the last element returned by next or previous.
  • void set(E e) – replaces the last element returned by next or previous.

Example of Iterating a List using ListIterator

Let us look at an example for Iterating a list using the ListIterator. 

          
           
List<String> list = new ArrayList<>();
list.add("Geeks");
list.add("for");
list.add("Geeks");
 
// Creating a listIterator
ListIterator<String> i = list.listIterator(0);
 
// Iterating the list in forward direction
System.out.println("Iterating list in forward direction");
while (i.hasNext())
    System.out.println(i.next());
 
// Iterating the list in backward direction
System.out.println("Iterating list in backward direction");
while (i.hasPrevious())
    System.out.println(i.previous());
When we run the code snippet above, it produces the following output: 

          
           
Iterating list in forward direction
Geeks
for
Geeks
Iterating list in backward direction
Geeks
for
Geeks
