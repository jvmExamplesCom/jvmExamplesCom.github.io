How to Iterate LinkedHashMap using an Iterator

Java provides various ways to iterate over a LinkedHashMap. Iterating a LinkedHashMap using the Java Iterator interface is one of the most efficient ways. By using an Iterator, we can loop through each entry in the LinkedHashMap and perform any required operations on the entries. In this article, we will discuss how to iterate a LinkedHashMap using an Iterator and also provide some code examples to demonstrate the usage.

Creating a LinkedHashMap Collection

Before we start iterating a LinkedHashMap, we need to create it. We can use the LinkedHashMap class constructor to create a LinkedHashMap. The following code snippet creates a LinkedHashMap with keys and values of type String. 

          
           
LinkedHashMap<String, String> linkedHashMap = new LinkedHashMap<>();
We can also use the put() method from the Map interface to add elements to the LinkedHashMap. The following code snippet adds elements to the LinkedHashMap that we created above. 

          
           
linkedHashMap.put("key1", "value1");
linkedHashMap.put("key2", "value2");
linkedHashMap.put("key3", "value3");

Iterating a LinkedHashMap using an Iterator

Now that we have created our LinkedHashMap, we can start iterating it. We can use the iterator() method from the LinkedHashMap class to get an Iterator to the elements in the LinkedHashMap. The following code snippet shows how to do this. 

          
           
Iterator<Map.Entry<String, String>> iterator = linkedHashMap.entrySet().iterator();
Next, we can use the hasNext() and next() methods from the Iterator interface to iterate through the elements in the LinkedHashMap. The hasNext() method returns true if there are more elements in the LinkedHashMap to be iterated. The next() method returns the next element in the LinkedHashMap. The following code snippet illustrates the use of these two methods. 

          
           
while (iterator.hasNext()) {
    Map.Entry<String, String> entry = iterator.next();
    System.out.println("Key: " + entry.getKey() + ", Value: " + entry.getValue());
}
The output of the above code snippet is as follows: 

          
           
Key: key1, Value: value1
Key: key2, Value: value2
Key: key3, Value: value3
As shown in the output, the elements in the LinkedHashMap are iterated in the same order they were added. This is because LinkedHashMap maintains the insertion order of the elements.

Conclusion

In this article, we discussed how to iterate LinkedHashMap using an Iterator. We also provided some code examples to demonstrate the usage. Iterating a LinkedHashMap using an Iterator is an efficient way to loop through the elements in the collection.
