The Java ListIterator is an interface that allows us to traverse the elements of a list in either direction. ListIterator Class

The java.util.ListIterator class is an implementation of Iterator which lets us traverse the elements of a list in either direction. Similarly, we can use the hasNext() and nextIndex() methods to get the next element's index in the list.
