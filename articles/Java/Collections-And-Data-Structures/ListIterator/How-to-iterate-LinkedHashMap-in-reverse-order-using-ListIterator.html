How to Iterate LinkedHashMap in Reverse Order Using ListIterator

Iterating a LinkedHashMap in reverse order can be done using the ListIterator implemented in Java's Collection framework. This article will discuss the basics of how to use this iterator to iterate a LinkedHashMap in reverse order, and it will also include some code examples to illustrate the process.

What is ListIterator?

ListIterator is an interface in the Java Collection framework. It allows you to iterate a collection in both directions (forward and backward). This interface provides methods to check the position of the iterator, to fetch the next and previous elements, and to set the value of the current element.

How to use ListIterator with LinkedHashMap

Using the ListIterator with LinkedHashMap enables you to iterate the LinkedHashMap elements in reverse order, starting from the last element. To do this, you need to create an instance of the ListIterator using the LinkedHashMap's listIterator() method. This method takes two parameters: the start index and the end index.

Once you have created an instance of the ListIterator, you can use the previous() method to get the previous element in the LinkedHashMap. The following code example demonstrates how to use the ListIterator to iterate a LinkedHashMap in reverse order:


          
           
LinkedHashMap<String, String> linkedHashMap = new LinkedHashMap<>();
linkedHashMap.put("key1", "value1");
linkedHashMap.put("key2", "value2");
linkedHashMap.put("key3", "value3");

// Get the list iterator
ListIterator<String> listIterator = linkedHashMap.listIterator(linkedHashMap.size(), 0);

// Iterate the LinkedHashMap in reverse order
while (listIterator.hasPrevious()) {
    String key = listIterator.previous();
    String value = linkedHashMap.get(key);

    System.out.println("key: " + key + ", value: " + value);
}

In the above code example, we first created an instance of the ListIterator using the LinkedHashMap's listIterator() method. We then used the ListIterator's hasPrevious() and previous() methods to iterate the LinkedHashMap in reverse order. Finally, we printed out the key and value of each element in the LinkedHashMap.
