Iterate LinkedList in Reverse Direction Using the ListIterator

A Java LinkedList object is a list of data elements stored in memory. It is a type of data structure which stores items in a linear fashion, allowing fast access to elements. In order to iterate the values stored in a LinkedList, the ListIterator can be used. One of the features of the ListIterator is that it allows iteration in a reverse direction. This article will discuss how to use the ListIterator to iterate a Java LinkedList in reverse.

Creating a Linked List

Before we can iterate a Java LinkedList in reverse, we must first create a LinkedList object. This can be done using the following code:


          
           
import java.util.LinkedList;

public class Main
{
    public static void main(String[] args)
    {
        // create an empty linked list
        LinkedList<String> list = new LinkedList<String>();
    
        // populate with some values
        list.add("Hello");
        list.add("World");
        list.add("!");

    }
}

In this example, we have created a new LinkedList object and populated it with the values “Hello”, “World” and “!”. We can now proceed to iterating the elements in this list in reverse order.

Iterating a LinkedList in Reverse With ListIterator

To iterate a Java LinkedList in reverse, we can use the ListIterator’s 

hasPrevious()
and 
previous()
methods. The 
hasPrevious()
method returns true if there are more elements in the list and false if there are none left. The 
previous()
method returns the previous element in the list. The following code snippet shows how this can be used to iterate a LinkedList in reverse: 

          
           
import java.util.LinkedList;
import java.util.ListIterator;

public class Main
{
    public static void main(String[] args)
    {
        // create an empty linked list
        LinkedList<String> list = new LinkedList<String>();
    
        // populate with some values
        list.add("Hello");
        list.add("World");
        list.add("!");

        // create a list iterator
        ListIterator<String> listIterator = list.listIterator();
        
        // move the iterator to the end of the list
        while (listIterator.hasNext())
        {
            listIterator.next();
        }

        // iterate in reverse
        while (listIterator.hasPrevious())
        {
            String element = listIterator.previous();
            System.out.println(element);
        }
    }
}

The output of this code will be the values “!”, “World”, and “Hello” in reverse order. The ListIterator’s 

hasPrevious()
and 
previous()
methods are the key to iterating a Java LinkedList in reverse order.

Conclusion

In this article, we discussed how to iterate a Java LinkedList in reverse using the ListIterator’s 

hasPrevious()
and 
previous()
methods. This is an effective way to traverse the list in reverse order, allowing access to all of the elements in the list.
