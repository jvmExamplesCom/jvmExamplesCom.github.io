Traverse through ArrayList in Reverse Direction Using Java ListIterator Example

Java ListIterator provides easy way to traverse a list both in forward and reverse direction. We can obtain ListIterator from the list by using listIterator() method. In this tutorial, we will learn how to traverse a list in reverse direction using Java ListIterator with code example.

Points to Remember while Traversing List in Reverse Direction

Following are the few points to remember while traversing a list in reverse direction using ListIterator:
  • ListIterator provide hasPrevious() method to check if there is an element before the current element.
  • It also provides previous() method to get the previous element.
  • It starts from the last index or last element for reverse traversal.
  • We need to call the previous() method after the hasPrevious() method to obtain previous element.

Traverse through ArrayList in Reverse Direction using Java ListIterator

Let's create an ArrayList to illustrate how to traverse a list in reverse direction. 

          
           
ArrayList<String> colors = new ArrayList<>();
colors.add("Blue");
colors.add("Green");
colors.add("Red");
Now, let's see how to traverse this ArrayList in reverse direction using ListIterator. 

          
           
ListIterator iterator = colors.listIterator(colors.size());
while(iterator.hasPrevious()) {
    System.out.println(iterator.previous());
}
Output of the above program is: 

          
           
Red
Green
Blue
