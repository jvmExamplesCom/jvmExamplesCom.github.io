Traverse through ArrayList in forward direction using Java ListIterator Example

Java ListIterator provides the facility of iterating the list in the forward direction. It is an interface and extends Iterator interface. It provides two methods for forward traversal, which are hasNext() and next().

Working of hasNext()

The hasNext() method returns true if the iteration has more element otherwise it returns false. For example: 

          
           
ListIterator iterator = list.listIterator();
while (iterator.hasNext()) {
    System.out.println(iterator.next());
}
In the above code snippet, we are checking if the iterator has more elements or not using hasNext() method.

Working of next()

The next() method returns the next element in the list and advances the cursor position. For example: 

          
           
ListIterator iterator = list.listIterator();
while (iterator.hasNext()) {
    String element = iterator.next();
    System.out.println(element);
}
In the above code snippet, we are getting the next element from the list and printing it.

ListIterator iterator = list.listIterator();

In the above code snippet, we are getting the iterator of the list using listIterator() method. Then using a while loop, we are checking if there is any next element in the list using hasNext() method.
