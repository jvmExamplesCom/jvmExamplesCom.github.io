Replace an Element from ArrayList Using Java ListIterator Example

Java ListIterator provides a way to traverse the list bidirectional. It's also used to replace an element in the list. This tutorial will show you an example of how to do this using the ListIterator. Before we get into the example, let's take a look at the syntax for replacing an element. The method takes two parameters: an Object oldValue and an Object newValue. The method will replace the specified oldValue with the newValue. 

          
           
listIterator.set(oldValue, newValue);
Now let's create an example program to demonstrate this. In this example, we will use a simple ArrayList with Integer values. We will use the ListIterator to remove an element from the list and replace it with a new value. Here is the code: 

          
           
import java.util.ArrayList;
import java.util.ListIterator;
public class ListIteratorExample {
public static void main(String[] args) {
    //Create an ArrayList
    ArrayList<Integer> myList = new ArrayList<>();
    //Add some elements to it
    myList.add(10);
    myList.add(20);
    myList.add(30);
    myList.add(40);
    myList.add(50);
    System.out.println("Original list: "+myList);

    //Create a ListIterator
    ListIterator<Integer> listIterator = myList.listIterator();
    listIterator.next();
    listIterator.next();
    listIterator.next();
    //Replace the element
    listIterator.set(35);
    System.out.println("After replacement: "+myList);
    }
}
In the code above, we created an ArrayList, added some elements and printed out the original list. Then, we created a ListIterator and used the next() and set() methods to replace the third element with a new value. Finally, we printed out the list after the element is replaced. The output of the program is as follows: 

          
           
Original list: [10, 20, 30, 40, 50]
After replacement: [10, 20, 35, 40, 50]
We can see that the third element in the list was successfully replaced with the new value. In this tutorial, we looked at how to replace an element from ArrayList using ListIterator. We saw how to use the set() method to replace the oldValue with the new value. We also created a simple example program to demonstrate this.
