Remove All Elements from a Java LinkedHashSet

A LinkedHashSet is a type of hash set that contains distinct elements in the form of a doubly-linked list. This class is useful for maintaining insertion order between elements and for performing fast lookups. In this article, we will explore how to remove all elements from a LinkedHashSet in Java.

Java Code Examples

The code examples below provide two ways to remove all elements from a LinkedHashSet. The first example uses the 

clear()
method and the second example uses the Iteratorâ€™s 
remove()
method.

Example 1: Using clear() Method


          
           
LinkedHashSet<String> set = new LinkedHashSet<>();
set.add("Apple");
set.add("Orange");
set.add("Banana");

System.out.println("Before clear(): " + set);

// Clear all elements from set
set.clear();

System.out.println("After clear(): " + set);

Example 2: Using Iterator's remove() Method


          
           
LinkedHashSet<String> set = new LinkedHashSet<>();
set.add("Apple");
set.add("Orange");
set.add("Banana");

System.out.println("Before remove(): " + set);

Iterator<String> itr = set.iterator();
while(itr.hasNext()) {
    itr.next();
    itr.remove();
}

System.out.println("After remove(): " + set);

Both of these examples produce the same outcome - all elements are removed from the LinkedHashSet. You can modify any of the examples to suit your specific needs.
