How To Check If Element Exists In LinkedHashSet

LinkedHashSet is a data structure which stores elements in a certain order. It's typically used when you need to maintain an order of elements while iterating over them. You can check if an element exists in a LinkedHashSet by using the contains() method. If the element is present, this method will return true; otherwise it will return false.

Example 1: Check if an Element Exists in a LinkedHashSet


          
           
LinkedHashSet<String> lhs = new LinkedHashSet<>();
lhs.add("Hello");
lhs.add("World");

boolean result = lhs.contains("Hello");
System.out.println(result); // Prints true
In the above code example, we create a LinkedHashSet object and add two elements: “Hello” and “World”. We then use the contains() method to check if the element “Hello” exists in the set. This method returns true if the element is present, and false if it isn’t.

Example 2: Check If an Element Is in a LinkedHashSet Using a Lambda Expression


          
           
LinkedHashSet<Integer> lhs = new LinkedHashSet<>();
lhs.add(1);
lhs.add(2);
lhs.add(3);

boolean result = lhs.stream()
    .anyMatch(x -> x == 2);
System.out.println(result); // Prints true
In this code example, we create a LinkedHashSet object and add three integer elements to it: 1, 2, and 3. Then, we use a lambda expression with the anyMatch() method to check if an element is present in the set. This method returns true if an element is present; otherwise it returns false.

Example 3: Check If an Element Is in a LinkedHashSet Using the forEach() Method


          
           
LinkedHashSet<Integer> lhs = new LinkedHashSet<>();
lhs.add(1);
lhs.add(2);
lhs.add(3);

boolean result = false;
lhs.forEach(x -> {
    if (x == 2) {
        result = true;
    }
});
System.out.println(result); // Prints true
In this code example, we create a LinkedHashSet object and add three integer elements to it: 1, 2, and 3. Then, we use the forEach() method to loop through each element in the set and check if the element is equal to 2. If it is, we set the result variable to true. This variable will then be printed out, which will print true if the element is present, and false if it isn’t.

Conclusion

In this tutorial, we discussed how to check if an element exists in a LinkedHashSet. We went over three different examples to demonstrate how to do this using the contains() method, the anyMatch() method, and the forEach() method. By using these methods, we can easily check if an element is present in the set or not.
