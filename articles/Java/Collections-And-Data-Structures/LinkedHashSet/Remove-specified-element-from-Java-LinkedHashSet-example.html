Remove Specified Element from Java LinkedHashSet Example

LinkedHashSet is a data structure included in the Java Standard Library that combines the regular HashSet which provides fast lookup, and LinkedList, which allows insertion and removal at the beginning or end of the list in constant time. It is an ordered version of HashSet that maintains insertion order. This tutorial will show you how to remove a specified element from a LinkedHashSet.

Using Iterator

The easy way to remove a specified element from a LinkedHashSet is by using the iterator and then calling the remove() method. 

          
           
// Create a LinkedHashSet 
LinkedHashSet<String> lhset = new LinkedHashSet<String>();

// Adding element to LinkedHashSet
lhset.add("Element 1");
lhset.add("Element 2");
lhset.add("Element 3");

// Print the LinkedHashSet 
System.out.println("LinkedHashSet Elements: "+lhset);

// Get the iterator
Iterator<String> itr=lhset.iterator();

// Remove the specified element 
while(itr.hasNext()){
  if(itr.next().equals("Element 1")){
    itr.remove();
  }
}

// Print the LinkedHashSet 
System.out.println("After Removal: "+lhset);
The output of this code will be: 

          
           
LinkedHashSet Elements: [Element 1, Element 2, Element 3]
After Removal: [Element 2, Element 3]

Using remove() Method

If you know the exact element you want to remove from the LinkedHashSet, you can call the 

          
           
remove()
method on the set itself. For example: 

          
           
lhset.remove("Element 2");
This will remove the "Element 2" from the LinkedHashSet.

Using retainAll() Method

If you want to remove all the elements from the LinkedHashSet except for some specified elements, you can use the 

          
           
retainAll()
method. 

          
           
// Create a LinkedHashSet 
LinkedHashSet<String> lhset = new LinkedHashSet<String>();

// Adding element to LinkedHashSet
lhset.add("Element 1");
lhset.add("Element 2");
lhset.add("Element 3");
lhset.add("Element 4");

// Print the LinkedHashSet 
System.out.println("LinkedHashSet Elements: "+lhset);

// Retain only specific elements
lhset.retainAll(Collections.singleton("Element 2"));

// Print the LinkedHashSet 
System.out.println("After Retaining: "+lhset);
The output of this code will be: 

          
           
LinkedHashSet Elements: [Element 1, Element 2, Element 3, Element 4]
After Retaining: [Element 2]
As you can see, this code will retained only the specified element and removed all other elements from the LinkedHashSet.

Conclusion

In this tutorial, we saw how to remove a specified element from a LinkedHashSet in Java. There are multiple ways to do this including using the iterator, remove() method, and the retainAll() method. All of these methods allow you to remove the desired elements from the LinkedHashSet and give you the flexibility to choose which one to use based on your specific use case.
