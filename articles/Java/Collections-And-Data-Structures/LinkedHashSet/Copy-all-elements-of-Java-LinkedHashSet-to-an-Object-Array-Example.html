Copy All Elements of Java LinkedHashSet to an Object Array Example

In this article, we will look at how to copy all the elements of a Java LinkedHashSet to an object array example. We will look at the various methods available for doing this and some examples of each.

Using toArray() Method

The simplest method for copying all the elements of a LinkedHashSet to an object array is by using the 

toArray()
method. This method takes in no parameters and returns an object array containing all the elements of the LinkedHashSet. 

          
           
LinkedHashSet<Integer> linkedHashSet = new LinkedHashSet<>();
linkedHashSet.add(1);
linkedHashSet.add(2);
linkedHashSet.add(3);
Object[] array = linkedHashSet.toArray();
System.out.println(Arrays.toString(array));

The above code snippet will print out the following:


          
           
[1, 2, 3]
As we can see, the 
toArray()
method copies all the elements of the LinkedHashSet into the array.

Using addAll() Method

Another way to copy all the elements of a LinkedHashSet to an object array is by using the 

addAll()
method. This method takes in a collection and adds all the elements of that collection to the calling instance of LinkedHashSet. 

          
           
LinkedHashSet<Integer>linkedHashSet = new LinkedHashSet<>();
linkedHashSet.add(1);
linkedHashSet.add(2);
linkedHashSet.add(3);
Integer[] array = new Integer[linkedHashSet.size()];
linkedHashSet.addAll(Arrays.asList(array));
System.out.println(Arrays.toString(array));

The above code snippet will print out the following:


          
           
[1, 2, 3]
As we can see, the 
addAll()
method adds all the elements of the LinkedHashSet to the given array.

Using stream() Method

One more way to copy all the elements of a LinkedHashSet to an object array is by using the 

stream()
method. This method takes in no parameters and returns a 
Stream
instance containing all the elements of the LinkedHashSet. 

          
           
LinkedHashSet<Integer> linkedHashSet = new LinkedHashSet<>();
linkedHashSet.add(1);
linkedHashSet.add(2);
linkedHashSet.add(3);
Object[] array = linkedHashSet.stream().toArray();
System.out.println(Arrays.toString(array));

The above code snippet will print out the following:


          
           
[1, 2, 3]
As we can see, the 
stream()
method copies all the elements of the LinkedHashSet into the given array.
