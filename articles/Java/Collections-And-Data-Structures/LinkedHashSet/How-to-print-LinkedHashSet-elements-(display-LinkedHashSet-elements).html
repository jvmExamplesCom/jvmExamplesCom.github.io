How to Print LinkedHashSet Elements (Display LinkedHashSet Elements)

Printing the elements of a LinkedHashSet can be done in several ways. In this article we will discuss the different methods of displaying LinkedHashSet elements.

Using for-each Loop

One way to print the elements of a LinkedHashSet is to use a for-each loop to iterate over the set and print each element. 

          
           
LinkedHashSet<String> linkedHashSet = new LinkedHashSet<String>();

linkedHashSet.add("Element 1");
linkedHashSet.add("Element 2");
linkedHashSet.add("Element 3");

for (String element : linkedHashSet) {
    System.out.println(element);
}
When executed, this code will print the following output: 

          
           
Element 1
Element 2
Element 3

Using Iterator

Another way to print the elements of a LinkedHashSet is to use an iterator to iterate over the set and print each element. 

          
           
LinkedHashSet<String> linkedHashSet = new LinkedHashSet<String>();

linkedHashSet.add("Element 1");
linkedHashSet.add("Element 2");
linkedHashSet.add("Element 3");

Iterator<String> iterator = linkedHashSet.iterator();
while (iterator.hasNext()) {
    System.out.println(iterator.next());
}
When executed, this code will print the following output: 

          
           
Element 1
Element 2
Element 3

Using forEach Method

Another way to print the elements of a LinkedHashSet is to use the forEach method to iterate over the set and print each element. 

          
           
LinkedHashSet<String> linkedHashSet = new LinkedHashSet<String>();

linkedHashSet.add("Element 1");
linkedHashSet.add("Element 2");
linkedHashSet.add("Element 3");

linkedHashSet.forEach(element -> System.out.println(element));
When executed, this code will print the following output: 

          
           
Element 1
Element 2
Element 3

Using Stream API

The Java 8 Stream API provides a powerful and convenient way to process collections. We can use the Stream API to print the elements of a LinkedHashSet. 

          
           
LinkedHashSet<String> linkedHashSet = new LinkedHashSet<String>();

linkedHashSet.add("Element 1");
linkedHashSet.add("Element 2");
linkedHashSet.add("Element 3");

linkedHashSet.stream().forEach(System.out::println);
When executed, this code will print the following output: 

          
           
Element 1
Element 2
Element 3

Conclusion

In this article, we discussed the various ways to print the elements of a LinkedHashSet. We explored four different methods: using a for-each loop, using an iterator, using the forEach method, and using the Stream API.
