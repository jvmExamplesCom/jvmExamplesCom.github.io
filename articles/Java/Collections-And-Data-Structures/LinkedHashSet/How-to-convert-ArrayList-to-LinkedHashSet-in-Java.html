How to Convert Arraylist to LinkedHashSet in Java

Java provides a number of ways for converting an ArrayList to LinkedHashSet. This article discusses the various techniques available to do this conversion.

Using Constructor and addAll() Method

One of the most common and straightforward approaches to convert ArrayList to LinkedHashSet is to use the constructor of LinkedHashSet and addAll() method of ArrayList. Here is a sample code demonstrating this approach: 

          
           
ArrayList<String> arrayList = new ArrayList<>();
arrayList.add("a");
arrayList.add("b");
arrayList.add("c");

LinkedHashSet<String> linkedHashSet = new LinkedHashSet<>(arrayList);
linkedHashSet.addAll(arrayList);

System.out.println(linkedHashSet);
The output of this code snippet would be as follows : 

          
           
[a, b, c]

Using Stream API and collect() Method

Another approach to convert ArrayList to LinkedHashSet is to use Stream API and collect() method of Stream. Here is a sample code demonstrating this approach: 

          
           
ArrayList<String> arrayList = new ArrayList<>();
arrayList.add("d");
arrayList.add("e");
arrayList.add("f");

LinkedHashSet<String> linkedHashSet = arrayList.stream()
    .collect(Collectors.toCollection(LinkedHashSet::new));

System.out.println(linkedHashSet);
The output of this code snippet would be as follows : 

          
           
[d, e, f]

Using forEach() Method and add() Method

Another approach to convert ArrayList to LinkedHashSet is to use forEach() method and add() method of LinkedHashSet. Here is a sample code demonstrating this approach: 

          
           
ArrayList<String> arrayList = new ArrayList<>();
arrayList.add("g");
arrayList.add("h");
arrayList.add("i");

LinkedHashSet<String> linkedHashSet = new LinkedHashSet<>();
arrayList.forEach(linkedHashSet::add);

System.out.println(linkedHashSet);
The output of this code snippet would be as follows : 

          
           
[g, h, i]

Using iterator() Method and add() Method

Yet another approach to convert ArrayList to LinkedHashSet is to use iterator() method and add() method of LinkedHashSet. Here is a sample code demonstrating this approach: 

          
           
ArrayList<String> arrayList = new ArrayList<>();
arrayList.add("j");
arrayList.add("k");
arrayList.add("l");

LinkedHashSet<String> linkedHashSet = new LinkedHashSet<>();
Iterator<String> iterator = arrayList.iterator();
while (iterator.hasNext()) {
    linkedHashSet.add(iterator.next());
}

System.out.println(linkedHashSet);
The output of this code snippet would be as follows : 

          
           
[j, k, l]
In conclusion, there are several approaches available to convert an ArrayList to LinkedHashSet in Java. Each of these approaches has its own advantages and disadvantages and should be used according to the requirement.
