How to Clear or Remove all Elements from LinkedHashSet in Java

LinkedHashSet is a collection of elements (objects) linked together in a sequence. It has unique elements and preserves the order of insertion, making it easier for the user to navigate through the list. The LinkedHashSet class is part of the Java Collection Framework, which provides a set of interfaces and classes to store and manipulate collections of objects.

The LinkedHashSet class allows us to clear or remove all elements from a set by calling the clear() method. This method removes all of the elements from the set and clears the underlying storage space. It is important to note that this method has no effect on the backing LinkedHashMap object.

Example 1: Clearing a LinkedHashSet containing String elements

In this example, we will create a LinkedHashSet of type String using the following code:


          
           
LinkedHashSet<String> set = new LinkedHashSet<>();
 set.add("Apple");
 set.add("Banana");
 set.add("Orange");
 set.add("Mango");

Now, to clear the set, we will call the clear() method as shown below:


          
           
set.clear();

After the above line is executed, the set will be cleared and will not contain any elements as shown below:
