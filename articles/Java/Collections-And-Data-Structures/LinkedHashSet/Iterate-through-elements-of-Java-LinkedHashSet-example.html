Iterate through Elements of Java LinkedHashSet Example

LinkedHashSet is a Collection class that stores elements in a linked list and provides fast iteration. It extends the HashSet class and maintains the insertion order of elements. The following example shows how to iterate through the elements of a LinkedHashSet.

Example 1: Iterating through LinkedHashSet


          
           
// Create a LinkedHashSet
LinkedHashSet linkSet = new LinkedHashSet();

// Add elements to LinkedHashSet
linkSet.add("element 1");
linkSet.add("element 2");
linkSet.add("element 3");
linkSet.add("element 4");

// Iterate through the LinkedHashSet
Iterator itr = linkSet.iterator();
while (itr.hasNext())
    System.out.println(itr.next());
The output of the above code will be: 

          
           
element 1
element 2
element 3
element 4

Example 2: Iterating over LinkedHashSet Using for-each loop

We can also use the for-each loop to iterate over the elements of LinkedHashSet. The following example shows how to use the for-each loop to iterate over the elements of a LinkedHashSet:

// Add elements to LinkedHashSet
linkSet.add("element 1");
linkSet.add("element 2");

// Iterate through the LinkedHashSet

Example 3: Iterating through LinkedHashSet using Iterator with while loop

We can also use an Iterator with a while loop to iterate over the elements of LinkedHashSet. The following example shows how to use an Iterator with a while loop to iterate over the elements of a LinkedHashSet: 

          
           
// Create a LinkedHashSet
LinkedHashSet linkSet = new LinkedHashSet();

// Add elements to LinkedHashSet
linkSet.add("element 1");
linkSet.add("element 2");
linkSet.add("element 3");
linkSet.add("element 4");

// Iterate through the LinkedHashSet
Iterator itr = linkSet.iterator();
while (itr.hasNext()) {
    String element = (String) itr.next();

}
