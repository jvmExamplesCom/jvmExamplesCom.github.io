How to sort LinkedHashSet in Java

Java provides a data structure called LinkedHashSet, which is basically a combination of a HashSet and a LinkedList. This allows the items in the set to be both unique and ordered. In this article, we will discuss how to sort LinkedHashSet in Java. We'll look at the various sorting algorithms, the differences between them, and a few code examples.

Types of Sorting Algorithms

There are a variety of sorting algorithms available for use in Java. The most popular ones are insertion sort, selection sort, merge sort, and quick sort. Each of these algorithms has its own advantages and disadvantages.
  • Insertion Sort: This is an efficient algorithm which sorts elements by inserting them into their correct place within the array. It is generally considered the best algorithm for small datasets.
  • Selection Sort: This algorithm sorts elements by selecting the appropriate element from the array and inserting it into its correct position. It is generally faster for larger datasets.
  • Merge Sort: This algorithm divides the list into two halves, sorts each half separately, and then merges them back together. It is generally considered the fastest algorithm when sorting large datasets.
  • Quick Sort: This algorithm sorts elements by picking a pivot element and partitioning the list around the pivot. It is generally considered the fastest algorithm for large datasets.

Sorting a LinkedHashSet in Java

To sort a LinkedHashSet in Java, we can use any of the above sorting algorithms. Let's look at an example of using the merge sort algorithm to sort a LinkedHashSet. 

          
           
public static void mergeSort(LinkedHashSet<String> list) {
   // Base case. List is empty or size 1
   if (list == null || list.size() <= 1) return;
 
   // Split list in half
   LinkedHashSet<String> left = new LinkedHashSet<>();
   LinkedHashSet<String> right = new LinkedHashSet<>();
   for (String s : list) {
     if (list.indexOf(s) % 2 == 0)
        left.add(s);
     else
        right.add(s);
   }
 
   // Recursively sort left and right lists
   mergeSort(left);
   mergeSort(right);
 
   // Merge sorted left and right lists
   list.clear();
   list.addAll(merge(left, right));
}
The merge() method is a helper function that takes two sorted LinkedHashSets and combines them into a single sorted LinkedHashSet. 

          
           
private static LinkedHashSet<String> merge(LinkedHashSet<String> left, LinkedHashSet<String> right) {
   LinkedHashSet<String> result = new LinkedHashSet<>();
 
   while (!left.isEmpty() &amp;&amp; !right.isEmpty()) {
      if (left.iterator().next().compareTo(right.iterator().next()) <= 0) {
         result.add(left.iterator().next());
         left.remove(left.iterator().next());
      } else {
         result.add(right.iterator().next());
         right.remove(right.iterator().next());
      }
   }
 
   if (!left.isEmpty()) {
      result.addAll(left);
   }
 
   if (!right.isEmpty()) {
      result.addAll(right);
   }
 
   return result;
}
