Check if a particular element exists in Java LinkedHashSet Example

LinkedHashSet is an implementation of the Set interface that uses a hash table to store the objects. It has two main features which distinguish it from other implementations of Set â€“ it maintains an insertion order, and it allows null values. In this example, we will see how to check if a particular element exists in the LinkedHashSet.

Example:


          
           
import java.util.LinkedHashSet;

public class LinkedHashSetExample {
   public static void main(String args[]) {
      
      //create LinkedHashSet object
      LinkedHashSet<String> lhs = new LinkedHashSet<String>();
      
      //add elements to LinkedHashSet
      lhs.add("Element1");
      lhs.add("Element2");
      lhs.add("Element3");
      lhs.add("Element4");
      
      //check if "Element2" exists
      boolean flag = lhs.contains("Element2");
      
      //displaying the result of contains method
      if (flag) {
         System.out.println("Element2 exists in LinkedHashSet.");
      } else {
         System.out.println("Element2 does not exist in LinkedHashSet.");
      }
   }
}

Element2 exists in LinkedHashSet.
