How to Convert an Array to LinkedHashSet in Java

Java provides a variety of ways to convert array to other data structures. One of the most common is to convert an array to a LinkedHashSet. LinkedHashSet preserves the order of insertion in which elements were added to the set. It is an implementation of the Set interface that guarantees that the order will remain constant even over time. This makes it an ideal choice for the conversion of an array to a LinkedHashSet. In this article, we will take a look at how to convert an array to a LinkedHashSet in Java and provide code examples of each approach.

Using Collection.addAll() method

The 

          
           
Collection.addAll()
method can be used to add all the elements of an array to a LinkedHashSet. The syntax of the 

          
           
addAll()
method is as follows: 

          
           
LinkedHashSet.addAll(Collection<!--? extends E--> c);
where 

          
           
Collection
is the collection of type LinkedHashSet and 

          
           
E
is the type of elements present in the array. Here is an example of how to use this method to convert an array of integers to a LinkedHashSet: 

          
           
//exampleArray
int[] exampleArray = {1, 2, 3, 4, 5};

// LinkedHashSet
LinkedHashSet<Integer> linkedHashSet = new LinkedHashSet<>();

// add all elements of exampleArray to linkedHashSet
linkedHashSet.addAll(Arrays.asList(exampleArray));

System.out.println("LinkedHashSet: " + linkedHashSet);
The output of the above code will be: 

          
           
LinkedHashSet: [1, 2, 3, 4, 5]

Using a Stream API

The Stream API provides an easy way to convert an array to a LinkedHashSet in Java. The following code snippet shows how to use the Stream API to convert an array of strings to a LinkedHashSet: 

          
           
// exampleStringArray
String[] exampleStringArray = {"Hello", "World", "!"};

// create a LinkedHashSet from the exampleStringArray
LinkedHashSet<String> linkedHashSet = 
    Arrays.stream(exampleStringArray)
          .collect(Collectors.toCollection(LinkedHashSet::new));

System.out.println("LinkedHashSet: " + linkedHashSet);
The output of the above code will be: 

          
           
LinkedHashSet: [Hello, World, !]

Using a For Loop

Using a for loop is also a straightforward way to convert an array to a LinkedHashSet in Java. The following code snippet shows how to use a for loop to convert an array of doubles to a LinkedHashSet: 

          
           
// exampleDoubleArray
double[] exampleDoubleArray = {1.2, 3.4, 5.6};

// create a LinkedHashSet to store the elements
LinkedHashSet<Double> linkedHashSet = new LinkedHashSet<>();

// add each element of the exampleDoubleArray
for (double d : exampleDoubleArray) {
    linkedHashSet.add(d);
}

System.out.println("LinkedHashSet: " + linkedHashSet);
The output of the above code will be: 

          
           
LinkedHashSet: [1.2, 3.4, 5.6]

Conclusion

In this article, we discussed how to convert an array to a LinkedHashSet in Java. We looked at three different approaches, each with code examples. We saw that the Collection.addAll() method, Stream API, and a for loop are all suitable options for the conversion process.
