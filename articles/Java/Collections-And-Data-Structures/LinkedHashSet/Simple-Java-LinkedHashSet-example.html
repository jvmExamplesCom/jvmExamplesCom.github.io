Simple Java LinkedHashSet example

LinkedHashSet is a class in the Java collections framework. It is an ordered collection of objects which uses a doubly linked list to store elements that are accessed in a particular order. This class is useful when you want to maintain a specific sequence of elements, such as insertion order or access order. This tutorial explains how to use LinkedHashSet in Java with code examples. We also discuss some common operations on LinkedHashSet, such as adding elements, removing elements, checking if the LinkedHashSet contains an element, and iterating over a LinkedHashSet.

Creating an empty LinkedHashSet

You can create an empty LinkedHashSet by calling its default constructor. For example: 

          
           
LinkedHashSet<String> set = new LinkedHashSet<>();

Adding elements to a LinkedHashSet

You can add elements to a LinkedHashSet using the add() method. This method returns true if the element is added successfully, and false if the element already exists in the set. For example: 

          
           
set.add("John");
set.add("Alice");
set.add("Bob");
// returns true

set.add("Bob");
// returns false

Removing elements from a LinkedHashSet

You can remove elements from the LinkedHashSet using the remove() method. This method returns true if the element is removed successfully, and false if the element does not exist in the set. For example: 

          
           
set.remove("John");
// returns true

set.remove("John");
// returns false

Checking if a LinkedHashSet contains an element

You can check if a LinkedHashSet contains an element by using the contains() method. This method returns true if the element is present, and false if the element is not present in the set. For example: 

          
           
set.contains("Alice");
// returns true

set.contains("John");
//returns false

Iterating over a LinkedHashSet

You can iterate over a LinkedHashSet using the for-each loop or the Iterator interface. Using 

          
           
for-each loop
: 

          
           
for (String element : set) {
    System.out.println(element);
}
Using 

          
           
Iterator
: 

          
           
Iterator<String> iterator = set.iterator();
while (iterator.hasNext()) {
    String element = iterator.next();
    System.out.println(element);
}

Conclusion

In this tutorial, we discussed how to use LinkedHashSet in Java. We discussed how to create an empty LinkedHashSet, add elements to it, remove elements from it, check if it contains an element, and iterate over it.
