Getting the First or Last Element From LinkedHashSet in Java

LinkedHashSet is a popular data structure in Java that uses a doubly-linked list to store its elements. It provides insertion order iterators that can be used to access the elements in the order they were added. This article will discuss how to get the first or last element from a LinkedHashSet using these iterators.

Using Iterators To Get The First Element

The first step is to create an instance of the LinkedHashSet class and add our desired elements to it. Once the set is populated, we can use the iterator that it provides to access the elements in their insertion order. 

          
           
LinkedHashSet<Integer> set = new LinkedHashSet<>();
set.add(1);
set.add(2);
set.add(3);

Iterator<Integer> iterator = set.iterator();
We can then use the iterator's 

          
           
hasNext()
and 

          
           
next()
methods to iterate through the elements of the set. In the case of getting the first element, we can use the following code: 

          
           
if (iterator.hasNext()) {
    Integer firstElement = iterator.next();
    System.out.println("First element is: " + firstElement);
}
This code will first check if there is a next element in the iteration and if so, it will assign it to a variable and print it out.

Using Iterators To Get The Last Element

To get the last element of the LinkedHashSet, we can use similar code to above. This time, however, instead of just calling the 

          
           
next()
method once, we will need to loop through all the elements using a while loop until we reach the last one. 

          
           
while (iterator.hasNext()) {
    Integer lastElement = iterator.next();
}

System.out.println("Last element is: " + lastElement);
This code will keep looping through the elements and each time assign the current element to the 

          
           
lastElement
variable. Once no more elements can be found, the loop will stop and the code will print out the last element.

Conclusion

In this article, we discussed how to get the first or last element from a LinkedHashSet in Java. We showed how to use the iterator provided by the set to iterate through all its elements and access either the first or last one. With this information, you should now be able to easily get the desired elements from any LinkedHashSet.
