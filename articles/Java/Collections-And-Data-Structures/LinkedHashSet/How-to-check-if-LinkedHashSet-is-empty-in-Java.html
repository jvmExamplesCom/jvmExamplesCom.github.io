Checking if LinkedHashSet is Empty in Java

LinkedHashSet is a type of data structure that is used to store elements in a collections framework. It's similar to a standard HashSet, but with the addition of a linked list added to it that ensures that the order of insertion is preserved in the set. As a result, when you want to check if the LinkedHashSet is empty or not, you will need to take into account this additional aspect. In this article, we'll be exploring how to check if a LinkedHashSet is empty in Java using a few different methods. Let's get started!

Method 1: Using the size() Method

The most straightforward way to check if a LinkedHashSet is empty is by using the size() method. This method returns an int value specifying the number of elements currently stored in the LinkedHashSet. If the size of the LinkedHashSet is 0, this means it is empty. Here's an example of how you can use the size() method to check if a LinkedHashSet is empty: 

          
           
import java.util.LinkedHashSet;

public class CheckLinkedHashSet {
   public static void main(String args[]) {
      // Create an empty LinkedHashSet 
      LinkedHashSet<String> lhs = new LinkedHashSet<String>();
      
      // Check if the LinkedHashSet is empty
      if (lhs.size() == 0) {
         System.out.println("LinkedHashSet is empty!");
      }
   }
}
If you run the above program, you will see the following output: 

          
           
LinkedHashSet is empty!

Method 2: Using the isEmpty() Method

Another way to check if a LinkedHashSet is empty is by using the isEmpty() method. This method returns a boolean value, which will be true if the LinkedHashSet is empty and false if it isn't. Here's an example of how you can use the isEmpty() method to check if a LinkedHashSet is empty: 

          
           
import java.util.LinkedHashSet;

public class CheckLinkedHashSet {
   public static void main(String args[]) {

      // Create an empty LinkedHashSet 
      LinkedHashSet<String> lhs = new LinkedHashSet<String>();
      
      // Check if the LinkedHashSet is empty
      if (lhs.isEmpty()) {
         System.out.println("LinkedHashSet is empty!");
      }
   }
}
If you run the above program, you will see the following output: 

          
           
LinkedHashSet is empty!

Method 3: Using the contains() Method

Finally, you can also check if a LinkedHashSet is empty by using the contains() method. This method returns a boolean value, which will be true if the LinkedHashSet contains the specified element, and false if it doesn't. Here's an example of how you can use the contains() method to check if a LinkedHashSet is empty: 

          
           
import java.util.LinkedHashSet;

public class CheckLinkedHashSet {
   public static void main(String args[]) {

      // Create an empty LinkedHashSet 
      LinkedHashSet<String> lhs = new LinkedHashSet<String>();
      
      // Check if the LinkedHashSet is empty
      if (!lhs.contains("a")) {
         System.out.println("LinkedHashSet is empty!");
      }
   }
}
If you run the above program, you will see the following output: 

          
           
LinkedHashSet is empty!
As you can see, there are several ways to check if a LinkedHashSet is empty in Java. Each of these methods have their own advantages and disadvantages, so it's important to choose the one that best suits your particular needs.
