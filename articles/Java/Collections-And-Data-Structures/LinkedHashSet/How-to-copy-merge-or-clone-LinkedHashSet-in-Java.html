How to Copy Merge or Clone LinkedHashSet in Java

In this article, we'll be exploring the various ways to copy, merge, or clone a LinkedHashSet in Java. LinkedHashSet is a data structure that organizes elements in an ordered manner while still allowing fast access and modifications. This article will walk through the different ways of copying, merging, or cloning LinkedHashSet, complete with code examples and their output.

Using for Loop for Copying Elements

One way to copy elements from one LinkedHashSet to another is to use a for loop. This approach can be used to copy all the elements from one set to another, as seen in the example below. 

          
           
LinkedHashSet<String> source = new LinkedHashSet<>();
source.add("A");
source.add("B");
source.add("C");

LinkedHashSet<String> target = new LinkedHashSet<>();
for (String s : source) {
    target.add(s);
}

System.out.println(target);
Output: 

          
           
[A, B, C]

Using Iterator

We can also use an iterator to copy the elements from one LinkedHashSet to another. This approach requires us to create an iterator for the source LinkedHashSet and then use it to iterate through the elements one by one, copying them into the target LinkedHashSet. This approach is shown in the example below. 

          
           
LinkedHashSet<String> source = new LinkedHashSet<>();
source.add("A");
source.add("B");
source.add("C");

LinkedHashSet<String> target = new LinkedHashSet<>();
Iterator<String> itr = source.iterator();
while (itr.hasNext()) {
    target.add(itr.next());
}

System.out.println(target);
Output: 

          
           
[A, B, C]

Using Stream API

The Stream API in Java 8 enables us to easily copy elements from one LinkedHashSet to another via the forEach() method. It also allows us to use lambda functions as well, as shown in the example below. 

          
           
LinkedHashSet<String> source = new LinkedHashSet<>();
source.add("A");
source.add("B");
source.add("C");

LinkedHashSet<String> target = new LinkedHashSet<>();
source.stream().forEach(e -> target.add(e));

System.out.println(target);
Output: 

          
           
[A, B, C]

Using addAll() Method

The addAll() method is the most straightforward way of copying elements from one LinkedHashSet to another. It takes the source LinkedHashSet as an argument, and simply adds all its elements to the target LinkedHashSet. This is demonstrated in the example below. 

          
           
LinkedHashSet<String> source = new LinkedHashSet<>();
source.add("A");
source.add("B");
source.add("C");

LinkedHashSet<String> target = new LinkedHashSet<>();
target.addAll(source);

System.out.println(target);
Output: 

          
           
[A, B, C]

Using clone() Method

The clone() method is a built-in method that can be used to copy a LinkedHashSet. It creates a shallow copy, meaning that any changes made to the source LinkedHashSet will also be reflected in the cloned LinkedHashSet. This approach is seen in the example below. 

          
           
LinkedHashSet<String> source = new LinkedHashSet<>();
source.add("A");
source.add("B");
source.add("C");

LinkedHashSet<String> target = (LinkedHashSet<String>) source.clone();

System.out.println(target);
Output: 

          
           
[A, B, C]

Merging LinkedHashSets

LinkedHashSets can also be merged. This can be done with the addAll() method, as seen in the example below. This method adds all the elements of one LinkedHashSet to the other, resulting in a merged LinkedHashSet with all the elements of both sets. 

          
           
LinkedHashSet<String> set1 = new LinkedHashSet<>();
set1.add("A");
set1.add("B");

LinkedHashSet<String> set2 = new LinkedHashSet<>();
set2.add("C");
set2.add("D");

set1.addAll(set2);

System.out.println(set1);
Output: 

          
           
[A, B, C, D]
