How to convert LinkedHashSet to ArrayList in Java

The LinkedHashSet is a collection class in the Java programming language that extends HashSet, which provides implementation for the Set interface. It also implements the List interface and thus contains elements in a specific order. It allows for faster iteration over its elements. On the other hand, an ArrayList is an ordered list commonly used to store collections of objects.

Using the addAll() Method

One way of converting a LinkedHashSet to an ArrayList is by using the addAll() method. The addAll() method is a Collection class method that adds all of the elements from one collection to another.


          
           
LinkedHashSet<Integer> set = new LinkedHashSet<Integer>();
set.add(1);
set.add(2);
set.add(3);

ArrayList<Integer> list = new ArrayList<Integer>();
list.addAll(set);

System.out.println(list); // Output: [1, 2, 3]

Using the Stream API

Another way of converting the LinkedHashSet to an ArrayList is by using the Stream API. This API was introduced in Java 8 and allows developers to work with sequences of data using functional-style operations.


          
           
LinkedHashSet<Integer> set = new LinkedHashSet<Integer>();
set.add(1);
set.add(2);
set.add(3);

List<Integer> list = set.stream().collect(Collectors.toList());

System.out.println(list); // Output: [1, 2, 3]

Using the for-each Loop

It is also possible to convert a LinkedHashSet to an arraylist by using a for-each loop. This loop allows us to iterate over each element in the LinkedHashSet and add it to the ArrayList.


          
           
LinkedHashSet<Integer> set = new LinkedHashSet<Integer>();
set.add(1);
set.add(2);
set.add(3);

ArrayList<Integer> list = new ArrayList<Integer>();
for(Integer i: set) {
  list.add(i);
}

System.out.println(list); // Output: [1, 2, 3]
