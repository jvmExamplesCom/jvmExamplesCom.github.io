Creating New Objects of LinkedHashSet Class Using Constructors

LinkedHashSet is a Java class that can store unique elements in an ordered manner. It is a part of the Java Collections Framework which is responsible for managing collections of objects. This article will discuss how to create LinkedHashSet objects using various constructors.

Constructors

LinkedHashSet class has several constructors that allow its users to create LinkedHashSet objects in different ways. The constructors are listed below: 

          
           
LinkedHashSet()
LinkedHashSet(Collection c)
LinkedHashSet(int initialCapacity)
LinkedHashSet(int initialCapacity, float loadFactor)
LinkedHashSet(int initialCapacity, float loadFactor, boolean accessOrder)

Example 1: Creating an Empty LinkedHashSet Object

The following example demonstrates how an empty LinkedHashSet object can be created by calling the no-args constructor. 

          
           
LinkedHashSet list = new LinkedHashSet();

Example 2: Creating a LinkedHashSet Object With Elements Copied From Another Collection (Collection c Constructor)

The following example demonstrates how to create a LinkedHashSet object by ‘copying’ elements from another collection. In this case, a List object is used as the source for the elements. 

          
           
List<String> source = Arrays.asList("elem1", "elem2", "elem3");
LinkedHashSet list = new LinkedHashSet(source);

Example 3: Creating a LinkedHashSet Object With Specific Initial Capacity (int initialCapacity Constructor)

The following example demonstrates how to create a LinkedHashSet object with a specific initial capacity. 

          
           
LinkedHashSet list = new LinkedHashSet(10);

Example 4: Creating a LinkedHashSet Object With Specific Initial Capacity & Load Factor (int initialCapacity, float loadFactor Constructor)

The following example demonstrates how to create a LinkedHashSet object with a specific initial capacity and load factor. 

          
           
LinkedHashSet list = new LinkedHashSet(10, 0.75f);

Example 5: Creating a LinkedHashSet Object With Specific Initial Capacity, Load Factor & Access Order (int initialCapacity, float loadFactor, boolean accessOrder Constructor)

The following example demonstrates how to create a LinkedHashSet object with a specific initial capacity, load factor, and access order. 

          
           
LinkedHashSet list = new LinkedHashSet(10, 0.75f, true);
