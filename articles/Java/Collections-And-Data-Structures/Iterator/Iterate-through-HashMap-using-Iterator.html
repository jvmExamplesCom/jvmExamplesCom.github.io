Iterate Through HashMap Using Iterator

Java provides different ways of iterating through a HashMap. One way is to use the Iterator type. Using Iterator, you can traverse elements of the map in any order and retrieve both keys and values. The following code example shows how to do this.

Create and Fill a HashMap

First, we will create a HashMap and fill it with some test data: 

          
           
HashMap<String, Integer> myHashMap = new HashMap<>();
myHashMap.put("Apple", 1);
myHashMap.put("Banana", 2);
myHashMap.put("Cherry", 3);

Get the Iterator

Next, we will get an Iterator from the HashMap: 

          
           
Iterator<Map.Entry<String,Integer>> iterator = myHashMap.entrySet().iterator();

Iterate Through the HashMap

Finally, we will loop through the HashMap using the Iterator: 

          
           
while (iterator.hasNext()) {
    Map.Entry<String,Integer> entry = iterator.next();
    String key = entry.getKey();
    Integer value = entry.getValue();
    System.out.println("Key: " + key + ", Value: " + value);
}

Output

The above code example should output the following: 

          
           
Key: Apple, Value: 1
Key: Banana, Value: 2
Key: Cherry, Value: 3
