Iterate through a Collection using Java Iterator

Java Iterator is an interface that provides methods to iterate over any Collection. To use an Iterator, you must import the 

          
           
java.util.Iterator
package.

Creating an Iterator

To create an Iterator for a Collection, use the iterator() method of the Collection interface. For example: 

          
           
List<String> list = new ArrayList<String>();
list.add("A");
list.add("B");
list.add("C");

Iterator<String> itr = list.iterator();
The output of this code will be an Iterator pointing to the first element of the list.

Iterating over a Collection

We can use the Iterator to access and traverse through the elements of a Collection. The next() method of the Iterator returns the next element in the Collection and moves the cursor ahead by one position. 

          
           
while(itr.hasNext()) {
   String data = itr.next();
   System.out.println(data);
}
The output of this code will be the following: 

          
           
A
B
C

Removing Elements from Collection

The remove() method of the Iterator interface can be used to remove elements from a Collection. This method removes the current element from the Collection. For example: 

          
           
while(itr.hasNext()) {
   String data = itr.next();
   if(data.equals("B")) {
      itr.remove();
   }
}
This code will remove the element “B” from the Collection.

Using an Iterator with a for each loop

We can also use the Iterator with a for each loop to iterate over a Collection. For example: 

          
           
for(Iterator<String> itr = list.iterator(); itr.hasNext(); ) {
   String data = itr.next();
   System.out.println(data);
}
The output of this code will be the following: 

          
           
A
B
C

Accessing Multiple Iterators on a Collection

We can create multiple Iterators on a single Collection. For example: 

          
           
Iterator<String> itr1 = list.iterator();
Iterator<String> itr2 = list.iterator();
The output of this code will be two Iterators pointing to the same Collection. We can use these Iterators to iterate over the same Collection but at different times or in different ways.

Conclusion
