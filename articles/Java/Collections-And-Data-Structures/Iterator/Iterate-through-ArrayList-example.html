Iterate Through ArrayList Example

Iterating through an ArrayList is an important task in Java, as it allows you to access all of the items in the list and perform operations on them. In this article, we will look at an example of how to iterate through an ArrayList using various methods.

Using for Loop

Using a for loop is the most common way to iterate through an ArrayList. The following code example shows how to use a for loop to iterate through the ArrayList: 

          
           
ArrayList<Integer> myList = new ArrayList<>();
for (int i=0; i<myList.size(); i++) {
    System.out.println(myList.get(i));
}
In the above code example, we create an ArrayList of Integers, and then use a for loop to iterate through it and print out each element.

Using forEach() Method

Another way to iterate through an ArrayList is to use the forEach() method. The following code example shows how to use the forEach() method to iterate through the ArrayList: 

          
           
ArrayList<Integer> myList = new ArrayList<>();
myList.forEach(i -> System.out.println(i));
In the above code example, we create an ArrayList of Integers, and then use the forEach() method to iterate through it and print out each element.

Using Iterator

Another way to iterate through an ArrayList is to use an Iterator. The following code example shows how to use an Iterator to iterate through the ArrayList: 

          
           
ArrayList<Integer> myList = new ArrayList<>();
Iterator<Integer> myIterator = myList.iterator();
while (myIterator.hasNext()) {
    System.out.println(myIterator.next());
}
In the above code example, we create an ArrayList of Integers, and then use an Iterator to iterate through it and print out each element.

Using forEachRemaining() Method

Another way to iterate through an ArrayList is to use the forEachRemaining() method. The following code example shows how to use the forEachRemaining() method to iterate through the ArrayList: 

          
           
ArrayList<Integer> myList = new ArrayList<>();
Iterator<Integer> myIterator = myList.iterator();
myIterator.forEachRemaining(i -> System.out.println(i));
In the above code example, we create an ArrayList of Integers, and then use the forEachRemaining() method to iterate through it and print out each element.

Conclusion

In this article, we looked at an example of how to iterate through an ArrayList using various methods. We saw how to use a for loop, forEach() method, Iterator, and forEachRemaining() method in order to iterate through the ArrayList and access its elements.
