Iterate over Vector elements in Reverse or Backward Direction

Java provides a number of classes that are useful for quickly iterating over a collection of items, such as 

Vector
. One of the lesser-known features of Java is the ability to iterate over a Vector's elements in reverse or backward direction. This tutorial will explain how to do this.

Understanding Iteration in Java

In Java, the most common method for iterating over a collection of items is the for-each loop, also known as the enhanced for loop. This is a shorthand for iterating over a collection, typically used when you don't need to access the index of each element in the collection. Here is an example of a for-each loop with a Vector of Integers:


          
           
Vector<Integer> numbers = new Vector<>();
numbers.add(1);
numbers.add(2);
numbers.add(3);

for (Integer num : numbers) {
    System.out.println(num);
}

This prints out the numbers 1, 2 and 3, one per line.

The other option for iteration is the traditional for loop, which allows us to access the index of each element and use it for something. Here is an example of a traditional for loop with a Vector of String objects:


          
           
Vector<String> strings = new Vector<>();
strings.add("foo");
strings.add("bar");
strings.add("baz");

for (int i = 0; i < strings.size(); i++) {
    System.out.println(strings.get(i));
}

This prints out the three strings “foo”, “bar”, and “baz”, one per line.

Iterating Over Vector Elements in Reverse Direction

Now that you have some understanding of iteration in Java, let's look at how to iterate over a Vector's elements in reverse direction. To do this, we will use the traditional for loop, but with a slightly modified condition:


          
           
Vector<Integer> numbers = new Vector<>();
numbers.add(1);
numbers.add(2);
numbers.add(3);

for (int i = numbers.size() - 1; i >= 0; i--) {
    System.out.println(numbers.get(i));
}

This will print out the numbers 3, 2 and 1, one per line, meaning that we have effectively iterated over the Vector in reverse direction.

Conclusion

As you can see, iterating over a Vector's elements in reverse direction is quite simple. By simply changing the condition of the for loop, we can easily iterate over a Vector's elements in the order that we want.
