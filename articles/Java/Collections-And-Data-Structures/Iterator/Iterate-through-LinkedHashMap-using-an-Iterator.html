Creating a LinkedHashMap

First, let's create a LinkedHashMap in Java. To do this, we can use the following code: 

          
           
LinkedHashMap<Integer, String> map = new LinkedHashMap<>();
map.put(1, "one");
map.put(2, "two");
map.put(3, "three");
map.put(4, "four");
map.put(5, "five");
