Iterating Through a List Using an Iterator in Java

Iterators are an essential tool for working with collections of objects in Java. They provide a simple and effective way to traverse any collection, whether it is a list or a map. In this article, we will discuss how to iterate through a List using an Iterator in the Java programming language.

Creating a List of Objects

Before we can learn how to iterate through a List using an Iterator, let's create a List of objects. To do so, we will use the 

          
           
ArrayList
class: 

          
           
List<String> list = new ArrayList<>();
list.add("Apple");
list.add("Banana");
list.add("Cherry");

Creating an Iterator for a List

Now that we have created a List of objects, let's create an Iterator for it. To do so, we will use the 

          
           
iterator()
method from the 

          
           
List
interface: 

          
           
Iterator<String> iterator = list.iterator();

Iterating Through a List Using an Iterator

We can now use the Iterator to iterate through the List. The 

          
           
hasNext()
method returns 

          
           
true
if the Iterator has more elements in the List. We can use this method to determine when to stop iterating: 

          
           
while (iterator.hasNext()) {
    String element = iterator.next();
    // Do something with 'element'
}
The above code will iterate through the List and store each element in the 

          
           
element
variable.

Example Program

Let's put what we've learned together in an example program that will print out each element in the List: 

          
           
public class IteratorExample {
    public static void main(String[] args) {
        List<String> list = new ArrayList<>();
        list.add("Apple");
        list.add("Banana");
        list.add("Cherry");
    
        Iterator<String> iterator = list.iterator();
    
        while (iterator.hasNext()) {
            String element = iterator.next();
            System.out.println(element);
        }
    }
}
Running the program will produce the following output: 

          
           
Apple
Banana
Cherry

Conclusion

In this article, we discussed how to iterate through a List using an Iterator in Java. We explored how to create a List of objects and an Iterator for the List. We also looked at an example program that uses an Iterator to traverse the List and print out each element.
