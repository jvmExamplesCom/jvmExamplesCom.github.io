Iterating Through Elements of HashSet in Java using an Iterator

Java provides a set of collection classes which implement a group of objects as a single unit. The objects stored in a collection are known as its elements. One of the most widely used collections is the 

          
           
HashSet
. A 

          
           
HashSet
stores a group of unique elements, in no particular order. It does not permit duplicate elements. In this article, we will take a look at how to iterate through the elements of a 

          
           
HashSet
in Java using an iterator. First, let's review what an iterator is.

What is an Iterator?

An iterator is an object that allows you to traverse the elements in a collection. It provides methods like 

          
           
hasNext()
and 

          
           
next()
to step through each element in the collection. It also provides other methods that can be used to manipulate the collection, such as adding or removing elements.

Iterating Through a HashSet in Java

We can use an iterator to iterate through the elements in a 

          
           
HashSet
. To do this, we need to first get an instance of an iterator. We can do this by calling the 

          
           
iterator()
method on the 

          
           
HashSet
object. 

          
           
HashSet<String> hashSet = new HashSet<>();
hashSet.add("A");
hashSet.add("B");
hashSet.add("C");

Iterator<String> iterator = hashSet.iterator();
Once we have an instance of an iterator, we can use it to step through each element in the 

          
           
HashSet
. We will use a 

          
           
while
loop to do this, and check if there is another element using the 

          
           
hasNext()
method of the iterator. When the 

          
           
hasNext()
method returns 

          
           
true
, we call the 

          
           
next()
method to get the next element in the collection. 

          
           
while (iterator.hasNext()) {
    String element = iterator.next();
    System.out.println(element);
}
The above code will print out all the elements in the 

          
           
HashSet
.

Example Program

Let's look at a complete example program that demonstrates how to iterate through a 

          
           
HashSet
in Java using an iterator. This program creates a 

          
           
HashSet
and adds three elements to it. It then creates an iterator and uses it to iterate through the 

          
           
HashSet
and print out each element. 

          
           
import java.util.HashSet;
import java.util.Iterator;

public class IterateHashSetExample {

    public static void main(String[] args) {

        // create a HashSet
        HashSet<String> hashSet = new HashSet<>();

        // add elements to the HashSet
        hashSet.add("A");
        hashSet.add("B");
        hashSet.add("C");

        // get an iterator
        Iterator<String> iterator = hashSet.iterator();

        // iterate through the HashSet
        while (iterator.hasNext()) {
            String element = iterator.next();
            System.out.println(element);
        }
    }
}
When we compile and run the above program, it will print out the following output: 

          
           
A
B
C

iterator()

while
loop with the 

          
           
hasNext()
and 

          
           
next()
methods to step through each element in the 

          
           
HashSet
. Finally, we saw a complete example program that demonstrated how to iterate through a 

          
           
HashSet
in Java.
