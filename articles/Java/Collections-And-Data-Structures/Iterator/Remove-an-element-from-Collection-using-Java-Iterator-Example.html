Remove an Element from Collection Using Java Iterator Example

Java provides various ways of iterating over the elements of a collection, such as the for-each loop, the for-in loop, and the iterator. One of these techniques is removing an element from a collection using an iterator. This tutorial will show you how to use the Iterator class to remove an element from a collection in Java.

Remove an Element from a Collection using Iterator Example

Let's consider a simple example of removing an element from a collection using an Iterator. In this example, we will create an array list and add some elements to it. Then we will create an Iterator object and use it to remove an element from the array list. Here is the code example: 

          
           
import java.util.ArrayList;
import java.util.Iterator;

public class Main {
    public static void main(String[] args) {
        // Create an ArrayList
        ArrayList<String> arrList = new ArrayList<>();
        // Add elements to the ArrayList
        arrList.add("A");
        arrList.add("B");
        arrList.add("C");
 
        // Create an Iterator
        Iterator<String> iterator = arrList.iterator();
 
        // Remove an element from the ArrayList
        while (iterator.hasNext()) {
            if (iterator.next().equals("B")) {
                iterator.remove();
            }
        }
    }
}
In the above example, we have created an ArrayList with three elements "A", "B", and "C". Then we have created an Iterator object from the ArrayList and used it to remove the element "B" from the ArrayList.
