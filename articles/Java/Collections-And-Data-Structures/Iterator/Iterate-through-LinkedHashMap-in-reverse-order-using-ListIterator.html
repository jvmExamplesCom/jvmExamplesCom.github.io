Iterating Through LinkedHashMap in Reverse Order using ListIterator in Java

LinkedHashMap is a popular class from the Java collections framework that makes it easy to create and access a map of key-value pairs. LinkedHashMap is an implementation of the Map interface and extends HashMap, meaning we get all of the benefits of the HashMap implementation in addition to more advanced features like iteration order. In this article, we'll discuss how to iterate through a LinkedHashMap in reverse order using ListIterator. We'll also look at some example code to illustrate the concept.

What is ListIterator?

ListIterator is an interface from the Java collections framework which provides bidirectional access to a list of elements. It has methods for adding, removing, and replacing elements in a list, as well as methods for moving the cursor forward and backward within the list. It also provides methods for getting the previous and next element in the list, which makes it useful for looping over lists in reverse order.

Iterating Through LinkedHashMap Using ListIterator

To iterate through a LinkedHashMap in reverse order using ListIterator, we first need to create a ListIterator object by calling the iterator() method on the LinkedHashMap instance. This will return an Iterator object, which we can then cast to a ListIterator. We can use the ListIterator's hasPrevious() and previous() methods to loop over the LinkedHashMap elements in reverse order. 

          
           
// Create a LinkedHashMap 
LinkedHashMap<String, String> map = new LinkedHashMap<>();

// Add some elements to the map 
map.put("A", "Apple");
map.put("B", "Banana");
map.put("C", "Cherry");

// Get an Iterator object 
Iterator<String> iterator = map.keySet().iterator();

// Cast iterator to ListIterator 
ListIterator<String> listIterator = (ListIterator<String>) iterator;

// Loop over the list in reverse order 
while(listIterator.hasPrevious()) {
    String key = listIterator.previous();
    String value = map.get(key);
    System.out.println(key + " : " + value);
}
The code above will print the following output: 

          
           
C : Cherry 
B : Banana 
A : Apple
As we can see, the ListIterator was able to iterate over the LinkedHashMap elements in reverse order.

Conclusion

In this article, we looked at how to iterate through a LinkedHashMap in reverse order using ListIterator in Java. ListIterator provides several useful methods for looping over elements in a list and makes it easy to iterate over a LinkedHashMap in reverse order.
