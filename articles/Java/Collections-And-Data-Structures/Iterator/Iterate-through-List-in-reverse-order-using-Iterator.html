Iterate Through List in Reverse Order Using Iterator

If you’re looking for a way to iterate through a list in reverse order, using an iterator is the best way to go. With an iterator, you can traverse the list one element at a time, starting from the last element and ending at the first element. In this article, we’ll discuss how to use an iterator to traverse a list in reverse order.

Using an Iterator

To use an iterator to traverse the list in reverse order, first, we need to get the iterator from the list. This is done using the list’s iterator() method:


          
           
List<String> list = new ArrayList<>();
list.add("foo");
list.add("bar");
list.add("baz");

Iterator<String> iter = list.iterator();

The next step is to move the iterator to the end of the list. This is done using the iterator's 

hasNext()
and 
next()
methods: 

          
           
while (iter.hasNext()) {
    iter.next();
}

Now that the iterator has reached the end of the list, we can begin traversing it in reverse order using the iterator's 

hasPrevious()
and 
previous()
methods: 

          
           
while (iter.hasPrevious()) {
    String element = iter.previous();
    System.out.println(element);
}
The output of the above code is: 

          
           
baz
bar
foo

This demonstrates how to use an iterator to traverse a list in reverse order.

Alternatives

If you don't want to use an iterator to traverse the list in reverse order, there are other options available. For example, you can use the 

forEachRemaining()
method to iterate over the elements in the list in reverse order: 

          
           
List<String> list = new ArrayList<>();
list.add("foo");
list.add("bar");
list.add("baz");

Iterator<String> iter = list.iterator();

iter.forEachRemaining(e -> System.out.println(e));
The output of the above code is:

foo

sort()
method with a custom Comparator to sort the list in reverse order: 

          
           
List<String> list = new ArrayList<>();
list.add("foo");
list.add("bar");
list.add("baz");

list.sort((a, b) -> -a.compareTo(b));

list.forEach(e -> System.out.println(e));
The output of the above code is:

These are just two alternatives to using an iterator to traverse the list in reverse order. There are many other options available.

Conclusion

In this article, we discussed how to iterate through a list in reverse order using an iterator. We also looked at two alternative methods of achieving the same result. An iterator is the most efficient way of traversing a list in reverse order, but other methods are available if you don’t want to use an iterator.
