Iterating Through Elements of a LinkedHashSet in Java

LinkedHashSet is an implementation of Set interface which provides insertion order iteration among all the elements of the set. It is an ordered version of HashSet that maintains a doubly-linked List across all elements. In this article, we'll take a look at how to iterate through the elements of a LinkedHashSet in Java.

Using For Loop

One of the most straightforward ways to iterate through the elements of a LinkedHashSet in Java is using a for loop: 

          
           
LinkedHashSet<String> linkedHashSet = new LinkedHashSet<>();
linkedHashSet.add("Element 1");
linkedHashSet.add("Element 2");
linkedHashSet.add("Element 3");

for (String s : linkedHashSet) {
    System.out.println(s);
}
The above code will output the following: 

          
           
Element 1
Element 2
Element 3

Using Iterator

Another way to iterate through the elements of a LinkedHashSet in Java is by using an Iterator: 

          
           
LinkedHashSet<String> linkedHashSet = new LinkedHashSet<>();
linkedHashSet.add("Element 1");
linkedHashSet.add("Element 2");
linkedHashSet.add("Element 3");

Iterator<String> iterator = linkedHashSet.iterator();

while (iterator.hasNext()) {
    System.out.println(iterator.next());
}
The above code will output the following: 

          
           
Element 1
Element 2
Element 3

Using For Each Lambda Expression

Another convenient way to iterate through the elements of a LinkedHashSet in Java is by using a for each lambda expression: 

          
           
LinkedHashSet<String> linkedHashSet = new LinkedHashSet<>();
linkedHashSet.add("Element 1");
linkedHashSet.add("Element 2");
linkedHashSet.add("Element 3");

linkedHashSet.forEach(s -> {
    System.out.println(s);
});
The above code will output the following: 

          
           
Element 1
Element 2
Element 3

Using Stream API

LinkedHashSet also supports the Java Stream API. This makes it easy to process the elements of a LinkedHashSet in a functional manner. The following example uses the Stream API to print the elements of a LinkedHashSet: 

          
           
LinkedHashSet<String> linkedHashSet = new LinkedHashSet<>();
linkedHashSet.add("Element 1");
linkedHashSet.add("Element 2");
linkedHashSet.add("Element 3");

linkedHashSet.stream().forEach(System.out::println);
The above code will output the following: 

          
           
Element 1
Element 2
Element 3
In conclusion, there are several ways to iterate through the elements of a LinkedHashSet in Java. The choice of which method to use depends on the specific needs of the application and the context in which the iteration is taking place.
