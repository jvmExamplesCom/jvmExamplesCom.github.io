Java ArrayList Iterator Example

ArrayList Iterator is one of the most useful classes in Java that helps to iterate over a list. The Iterator interface provides methods hasNext() and next() to go through its elements in the order they were inserted. This article provides an example code of the ArrayList Iterator in Java with the following sections:

  • Declaring and Instantiating an Iterator Object
  • Iterating Through an ArrayList Using Iterator
  • Using Iterator with For-Each Loop

Declaring and Instantiating an Iterator Object

We create an iterator object by calling its static factory method iterator(), which is present in the java.util.Collection interface. An instance of the Iterator class can be declared and instantiated in the following way:


          
           
Iterator<Integer> itr = arraylist.iterator();       // Declares an Iterator<Integer> object

while (itr.hasNext()) {                                  // Iterates over the elements in the list
    System.out.print(itr.next() + " ");                  // Prints the current element
}

Iterating Through an ArrayList Using Iterator

The iterator object can be used to traverse the elements of the list. The hasNext() method returns true if the iterator has more elements. The next() method returns the current element and advances the iterator.


          
           
ArrayList<Integer> arraylist = new ArrayList<Integer>();  // Declares an ArrayList of Integers

arraylist.add(1);                                         // Adds 1 to the list
arraylist.add(2);                                         // Adds 2 to the list

Iterator<Integer> itr = arraylist.iterator();             // Declares an Iterator<Integer> object

while (itr.hasNext()) {                                   // Iterates over the elements in the list
    System.out.print(itr.next() + " ");                   // Prints the current element
}

Using Iterator with For-Each Loop

An iterator object can also be used in the for-each loop to traverse the elements of the list.


          
           
ArrayList<Integer> arraylist = new ArrayList<Integer>();  // Declares an ArrayList of Integers

arraylist.add(1);                                         // Adds 1 to the list
arraylist.add(2);                                         // Adds 2 to the list

Iterator<Integer> itr = arraylist.iterator();             // Declares an Iterator<Integer> object

for (Integer integer : arraylist) {                       // Iterates over the elements in the list
    System.out.print(integer + " ");                      // Prints the current element
}
