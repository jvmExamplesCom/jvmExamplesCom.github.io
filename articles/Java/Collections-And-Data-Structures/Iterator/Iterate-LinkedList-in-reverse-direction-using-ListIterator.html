Iterate LinkedList in Reverse Direction Using ListIterator

LinkedList is an implementation of the java.util.List interface and provides a doubly-linked list structure for storing elements. It has many advantages over ArrayList such as providing efficient insertion and deletion operations, as well as iteration over the elements in both directions. To achieve this, LinkedList utilizes the ListIterator interface to allow traversal of the list in either direction. In this tutorial, we will learn how to iterate LinkedList in reverse order using ListIterator.

Using ListIterator to Iterate Over LinkedList in Reverse Direction

The LinkedList class in Java provides an implementation of the List interface, which provides methods to traverse the list in both directions using the ListIterator interface. To iterate over the LinkedList in reverse order, we will use the ListIterator's previous() method. The previous() method returns the previous element in the list and moves the cursor position backwards. We will use it in a while loop, which will terminate when there are no more elements left in the list. Let us see the following example to understand how to iterate over a LinkedList in reverse order: 

          
           
LinkedList<String> list = new LinkedList<String>(); 
list.add("A"); 
list.add("B"); 
list.add("C"); 

ListIterator<String> iterator = list.listIterator(list.size()); 
while (iterator.hasPrevious()) { 
    String element = iterator.previous(); 
    System.out.println(element); 
}
In the above example, we have created a new LinkedList and added some elements to it. Then, we have created a ListIterator using the listIterator() method and passed the size of the list as its argument. This tells the ListIterator to begin at the end of the list, so we can traverse it in reverse direction. We have used the while loop to iterate over the list in reverse order. Inside the while loop, we have called the previous() method, which returns the previous element in the list and moves the cursor position backwards. Finally, we have printed the element. When we run the above program, we will get the following output: 

          
           
C
B
A
As you can see, the output contains the elements in reverse order, from last element of the list to the first element.
