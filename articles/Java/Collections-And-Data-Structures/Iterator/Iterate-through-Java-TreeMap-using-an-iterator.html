Iterating Through a Java TreeMap with an Iterator

Java TreeMap is a collection class that uses a tree structure to store key-value pairs. It provides an easy way to traverse the elements in order based on the key. An iterator can be used to loop through the TreeMap and access its values. In this article, we'll look at how to iterate through a TreeMap using an iterator in Java.

Creating a Java TreeMap

Before we can use an iterator to traverse through a TreeMap, we need to create one first. A TreeMap can be created by calling the constructor of the TreeMap class. We can also specify a Comparator to be used for ordering the elements in the TreeMap. 

          
           
TreeMap<String, Integer> numbers = new TreeMap<>(Comparator.naturalOrder());
In this example, we've created a TreeMap which will order the elements in natural order.

Adding Elements to a TreeMap

Once we have a TreeMap, we can add elements to it by calling the 

          
           
put()
method. 

          
           
numbers.put("one", 1);
numbers.put("two", 2);
numbers.put("three", 3);
numbers.put("four", 4);
numbers.put("five", 5);
We can add any number of elements to the TreeMap in this way.

Iterating Through a TreeMap

Once we have added all the elements we want to the TreeMap, we can use an iterator to traverse through it and access its values. To get an iterator for the TreeMap, we call the 

          
           
iterator()
method. 

          
           
Iterator<Map.Entry<String, Integer>> iterator = numbers.entrySet().iterator();
The iterator will return a Map.Entry for each element in the TreeMap. We can then use the 

          
           
next()
method of the iterator to access the value of each element. 

          
           
while(iterator.hasNext()) {
    Map.Entry<String, Integer> entry = iterator.next();
    String key = entry.getKey(); // get the key
    Integer value = entry.getValue(); // get the value
    System.out.println("Key: " + key + " Value: " + value);
}
In the example above, we loop over the elements in the TreeMap using the iterator and print out the key and value for each element.

Conclusion

In this article, we've looked at how to iterate through a Java TreeMap using an iterator. We saw how to create a TreeMap and add elements to it, as well as how to loop over the elements using an iterator and access its values.
