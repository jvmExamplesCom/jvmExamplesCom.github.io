Iterate Vector in Reverse Direction with ListIterator

In this article, we will be discussing how to iterate Vector in reverse direction using ListIterator. We will be looking at a few code examples to understand more about ListIterator.

What is ListIterator?

ListIterator is an interface which is part of Java's collection framework and it provides bidirectional cursor for the list. It is an iterator which can move both forward and backward.

ListIterator Methods

The following are the methods provided by ListIterator:

  • 
            
             
boolean hasPrevious()
    : This method returns true if the list has more elements when iterating in the reverse direction.
  • int nextIndex()
    : This method returns the index of the element that would be returned by a call to the next() method.
  • E previous()
    : This method returns the previous element in the list.
  • int previousIndex()
    : This method returns the index of the element that would be returned by a call to the previous() method.

Code Examples

Let us take a look at a few code examples to understand how to use ListIterator to iterate the vector in reverse direction.


          
           
import java.util.*;

public class IterateVector {
    public static void main(String[] args) {
        // create a Vector
        Vector<String> vector = new Vector<>();

        // add elements to the Vector
        vector.add("Apple");
        vector.add("Mango");
        vector.add("Orange");

        // get the list iterator
        ListIterator<String> listIterator = vector.listIterator(vector.size());

        System.out.println("Iterating Vector in Reverse Direction");

        while (listIterator.hasPrevious()) {
            System.out.println(listIterator.previous());
        }
    }
}
