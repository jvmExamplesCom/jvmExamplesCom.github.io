Iterate Through Elements of TreeSet in Java Using an Iterator

TreeSet is a collection class in Java which extends AbstractSet and implements NavigableSet interface. It uses Red-Black tree to maintain elements order maintaining natural ordering of elements or using Comparator that are provided at creation time. A TreeSet stores unique elements in sorted order, so it does not allow duplicate elements.

Iterator in TreeSet

Java provides Iterator for TreeSet. An Iterator is an interface which helps iterate through the elements of a collection. It is an object which has the capability to traverse through all the elements of a collection regardless of its implementation.

Iterator interface provides three methods: 
  • boolean hasNext()
    – returns true if the iteration has more elements.
  • Object next()
    – returns the next element in the iteration.
  • void remove()
    – removes from the underlying collection the last element returned by this iterator.

Example: How to Iterate Through Elements of TreeSet using Iterator?

Let’s see how to iterate through the elements of TreeSet using Iterator.


          
           
import java.util.Iterator;
import java.util.TreeSet;

public class IteratorExample { 
   public static void main(String[] args) 
   { 
      // create Treeset object 
      TreeSet<String> tree_set = new TreeSet<String>(); 

      // populate TreeSet 
      tree_set.add("Red"); 
      tree_set.add("Green"); 
      tree_set.add("Black"); 
      tree_set.add("White"); 

      // create instance of Iterator 
      Iterator<String> itr = tree_set.iterator(); 

      while (itr.hasNext()) { 
         System.out.println(itr.next()); 
      } 
   } 
}
