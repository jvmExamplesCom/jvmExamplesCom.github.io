How to Print Java HashSet Elements

The 
HashSet
in Java is a powerful data structure useful for managing large amounts of data. It also allows you to quickly access elements with complexity time O(1). In this article, we will learn how to print the elements of an existing 
HashSet
.

Using the 
for-each
loop

The most straightforward way to print out the elements of a 

HashSet
is with a 
for-each
loop. Here is the example code: 

          
           
HashSet<Integer> myHashSet = new HashSet<>();
myHashSet.add(1);
myHashSet.add(2);
myHashSet.add(3);

for (Integer num : myHashSet) {
 System.out.println(num); 
}
This code will output the following: 

          
           
1
2
3

Using the 
iterator()
Method

Another way to print out the elements of a 
HashSet
is to use the 
iterator()
method. This method returns an 
Iterator
object, which can be used to loop through the set and print out each element. Here is the example code: 

          
           
HashSet<Integer> myHashSet = new HashSet<>();
myHashSet.add(1);
myHashSet.add(2);
myHashSet.add(3);

Iterator<Integer> it = myHashSet.iterator();

while (it.hasNext()) {
    Integer num = it.next();
    System.out.println(num); 
}
This code will output the same as the 
for-each
loop example above: 

          
           
1
2
3

Using the 
stream()
Method

The 
stream()
method is available in Java 8 and allows us to create and process streams of objects. We can use it to print out the elements of our 
HashSet
. Here is the example code: 

          
           
HashSet<Integer> myHashSet = new HashSet<>();
myHashSet.add(1);
myHashSet.add(2);
myHashSet.add(3);

myHashSet.stream().forEach(System.out::println);

This code will also output the same as the other two examples:


          
           
1
2
3

Conclusion

In this article, we learned how to print out the elements of an existing 

HashSet
in Java. We looked at three different approaches - using the 
for-each
loop, the 
iterator()
method, and the 
stream()
method. All three approaches yielded the same result.
