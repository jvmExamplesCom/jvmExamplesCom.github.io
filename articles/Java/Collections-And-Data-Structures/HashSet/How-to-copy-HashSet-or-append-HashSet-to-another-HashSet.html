Creating a New HashSet

HashSet<String> cloneSet = (HashSet<String>) originalSet.clone();
This will create a shallow copy of the original HashSet and all the elements in the original set will be present in the cloned set.

Variable Assignment

You can also copy a HashSet to another by using the assignment operator. This will create a shallow copy of the HashSet as shown below: 

          
           
HashSet<String> cloneSet = originalSet;
Note that if you modify the original Set, the changes will be reflected in the cloned set as well.

Adding Elements from One Set to Another

If you want to add all the elements from one HashSet to another, you can use the addAll() method as shown below: 

          
           
cloneSet.addAll(originalSet);
This will add all the elements from the original Set to the cloned Set.
