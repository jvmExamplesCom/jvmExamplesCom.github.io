How to Create New HashSet Objects Using the HashSet Constructor

The Java Collection Framework provides an implementation of the Set interface called HashSet. The HashSet class enables you to store a set of unique elements in an unordered collection. This article will explain how to create new HashSet objects using the HashSet constructor.

HashSet Constructor with No Argument

The HashSet constructor with no argument creates a HashSet object with a default initial capacity of 16 and a load factor of 0.75. The following example creates a HashSet object without any argument.


          
           
HashSet hs = new HashSet();

HashSet Constructor with Capacity

The HashSet constructor that takes an integer value as an argument creates a HashSet object with the specified capacity and a load factor of 0.75. The following example creates a HashSet object with an initial capacity of 10.


          
           
HashSet hs = new HashSet(10);

HashSet Constructor with Capacity and Load Factor

The HashSet constructor taking two arguments, an integer value and a floating-point value, creates a HashSet object with the specified capacity and load factor. The load factor is a measure of how full the HashSet can get before it is resized. The following example creates a HashSet object with an initial capacity of 10 and a load factor of 0.5.


          
           
HashSet hs = new HashSet(10, 0.5);

HashSet Constructor with Collection Argument

The HashSet constructor taking a Collection argument creates a HashSet object that contains all of the elements in the specified collection. The following example creates a HashSet object containing elements from an ArrayList.


          
           
ArrayList<Integer> list = new ArrayList<Integer>();
list.add(1);
list.add(2);
list.add(3);

HashSet hs = new HashSet(list);
