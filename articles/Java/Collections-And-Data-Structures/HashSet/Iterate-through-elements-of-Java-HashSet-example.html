Iterating Through Elements of a Java HashSet

The Java HashSet is a class that implements the Set interface. It provides a powerful and efficient way to store and manipulate data in a collection.

In this article, we'll look at how to iterate through the elements of a Java HashSet using the for-each loop and the Iterator class.

Using the For-Each Loop

The simplest way to iterate through the elements of a Java HashSet is to use the for-each loop. The following code shows how to do this:


          
           
HashSet<String> fruits = new HashSet<>();
fruits.add("Apple");
fruits.add("Banana");
fruits.add("Orange");

for (String fruit : fruits) {
    System.out.println(fruit);
}
This code will output the following: 

          
           
Apple
Banana
Orange

As you can see, the for-each loop makes it easy to iterate through the elements of a Java HashSet.

Using the Iterator Class

Another way to iterate through the elements of a Java HashSet is by using the Iterator class. The following code shows how to do this:


          
           
HashSet<String> fruits = new HashSet<>();
fruits.add("Apple");
fruits.add("Banana");
fruits.add("Orange");

Iterator<String> iterator = fruits.iterator();
while (iterator.hasNext()) {
    String fruit = iterator.next();
    System.out.println(fruit);
}
This code will also output the following: 

          
           
Apple
Banana
Orange

As you can see, the Iterator class makes it easy to iterate through the elements of a Java HashSet.

Conclusion

In this article, we looked at two ways to iterate through the elements of a Java HashSet: using the for-each loop and using the Iterator class. We hope you found this article helpful.
