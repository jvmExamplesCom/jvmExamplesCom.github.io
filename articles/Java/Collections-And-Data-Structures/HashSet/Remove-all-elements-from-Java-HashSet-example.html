Remove All Elements from Java HashSet Example

Java HashSet is a Collection implementation backed by a hashing mechanism, which helps it store and retrieve elements faster compared to other Collection implementations such as ArrayList and LinkedList. The core properties of HashSet are that it doesn't maintain insertion order of the elements, allows null values and contains only unique elements. In this article, we'll look at some code examples that explain how to remove all elements from a Java HashSet.

Method 1: Using clear() Method

The most intuitive way to remove all elements from a Java HashSet is using its in-built 

          
           
clear()
method. This method clears the set so that it contains no elements. Let's take a look at an example. 

          
           
HashSet<String> set = new HashSet<>();
  
set.add("One");
set.add("Two");
set.add("Three");
  
System.out.println("HashSet before removal: "+set);
  
set.clear();
  
System.out.println("HashSet after removal: "+set);
The above code will produce the following output: 

          
           
HashSet before removal: [One, Two, Three]
HashSet after removal: []

Method 2: Using removeAll() Method

Another way to remove all elements from a Java HashSet is by using the in-built 

          
           
removeAll()
method. This method takes another Collection instance as argument and removes all the elements present in it from the invoking Set. Let's take a look at an example that explains how 

          
           
removeAll()
method can be used to remove all elements from a HashSet. 

          
           
HashSet<String> set1 = new HashSet<>();
  
set1.add("One");
set1.add("Two");
set1.add("Three");

HashSet<String> set2 = new HashSet<>();
  
set2.add("Four");
set2.add("Five");
set2.add("Six");
  
System.out.println("HashSet1 before removal: "+set1);
  
set1.removeAll(set2);
  
System.out.println("HashSet1 after removal: "+set1);
The above code will produce the following output: 

          
           
HashSet1 before removal: [One, Two, Three]
HashSet1 after removal: [One, Two, Three]

Conclusion

In this article, we looked at two ways to remove all elements from a Java HashSet. First, we saw how to use the in-built 

          
           
clear()
method to remove all elements from a HashSet. We then saw how to use the in-built 

          
           
removeAll()
method to remove all elements from a HashSet. I hope this article was helpful to you.
