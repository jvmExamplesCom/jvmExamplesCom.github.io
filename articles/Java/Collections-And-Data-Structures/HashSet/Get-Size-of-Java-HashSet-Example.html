Get Size of Java HashSet Example

Java HashSet is an implementation of the Set interface, which stores no duplicate values in its collection. This tutorial will provide an example of how to get the size of a Java HashSet.

Example of Getting Size of Java HashSet

The following Java example shows how to get the size of a HashSet: 

          
           
HashSet<String> set = new HashSet<String>();
 
set.add("Apple");
set.add("Banana");
set.add("Orange");
 
int size = set.size();
System.out.println(size); // Outputs 3
To get the size of the HashSet, we first need to declare a HashSet object and add elements to it. In this example, we declared a HashSet of type String, which means that it can store all types of strings within it. We then added three strings (Apple, Banana, and Orange) inside of the set, using the add() method. After the set has been initialized and populated with elements, we can retrieve the size of the set with the size() method, which returns an integer representing the size of the set – in this case, three. Finally, we print out the size of the set in the console. This example will then output “3”, as there are three elements stored in the set.

Retrieving Size of Empty HashSet

The following Java example shows what happens if you try to get the size of an empty HashSet: 

          
           
HashSet<String> set = new HashSet<String>(); 
int size = set.size();
System.out.println(size); // Outputs 0
In this example, we declare the same HashSet of type String, but without initializing it with any elements. We then use the size() method to retrieve the size of the HashSet, which should be 0, as the set is currently empty. This example will also print out “0” in the console as the size of the HashSet.

Conclusion

In this tutorial, we provided an example of how to get the size of a Java HashSet. We discussed both the case where the set is populated with elements, as well as the case where the set is empty.
