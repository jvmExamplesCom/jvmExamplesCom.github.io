Finding the Minimum or Maximum Value from Java HashSet

Java HashSet is an important part of the Java Collection Framework. It is an unordered collection of objects and allows for a unique instance of an object to exist in the set. It does not allow duplicates and is much faster than searching through an ArrayList. In this article, we will discuss how to find the minimum or maximum value from Java HashSet.

Finding the Minimum Value from a Java HashSet

The best method for finding the minimum value from a Java HashSet is by using the Collections.min() method. This method takes a Collection as an argument, and returns the element with the lowest value in that collection. The syntax for this method is:


          
           
Collections.min(Collection c);
Let's look at an example of its use: 

          
           
Set set = new HashSet();
set.add(2);
set.add(5);
set.add(1);
int min = (int) Collections.min(set);
System.out.println("Minimum Value: " + min);

This code snippet will print out the following output:


          
           
Minimum Value: 1

Finding the Maximum Value from a Java HashSet

The same way that we can find the minimum value from a Java HashSet, we can also find the maximum value from it. The method used for this is the Collections.max() method. This method takes a Collection as an argument, and returns the element with the highest value in that collection. The syntax for this method is:


          
           
Collections.max(Collection c);
Let's look at an example of its use: 

          
           
Set set = new HashSet();
set.add(2);
set.add(5);
set.add(1);
int max = (int) Collections.max(set);
System.out.println("Maximum Value: " + max);

This code snippet will print out the following output:


          
           
Maximum Value: 5

Conclusion

In this article, we discussed how to find the minimum and maximum values from a Java HashSet. We saw how to use the Collections.min() and Collections.max() methods for this purpose.
