Check if a Particular Element Exists in a Java HashSet Example

Java HashSet is a collection that stores its elements using a hash code. HashSet provides constant time performance for basic operations like add, remove, contains and size. It allows the storage of a single null element.

Example: Check if an Element Exists in a Java HashSet

The following example demonstrates how to check if a particular element exists in a Java HashSet.


          
           
import java.util.HashSet;

public class HashSetDemo {
 
   public static void main(String[] args) {
      // create a new HashSet
      HashSet<String> hset = new HashSet<String>(); 
      
      // add elements to HashSet
      hset.add("Apple"); 
      hset.add("Mango"); 
      hset.add("Guava");  
      
      // check if an element exists in HashSet
      boolean elementExists = hset.contains("Orange"); 
      
      System.out.println("Does HashSet contains Orange? : " + elementExists);         
   }
}
Output: 

          
           
Does HashSet contains Orange? : false
