How to convert ArrayList to HashSet in Java

Converting an ArrayList to a HashSet is a common operation, and is often used for tasks such as sorting and de-duplicating a list of items. There are various ways to do this in Java, and this article will go through each method and demonstrate how to use it.

Using the .retainAll() Method

The first option is to convert an ArrayList to a HashSet is the 

          
           
retainAll()
method. This method takes an existing HashSet and only keeps elements that are present in the input collection. The example below shows how this method can be used to convert an ArrayList to a HashSet: 

          
           
ArrayList<String> names = new ArrayList<>();
names.add("Alice");
names.add("Bob");
names.add("Charlie");

HashSet<String> namesSet = new HashSet<>();

// convert the arraylist to a hashset
namesSet.retainAll(names);

// print out the contents of the hashset
System.out.println(namesSet);
This will produce the following output: 

          
           
[Alice, Bob, Charlie]

Using the .addAll() Method

The second option is to use the 

          
           
addAll()
method, which takes an existing collection and adds all elements from the input collection to it. This method can also be used to convert an ArrayList to a HashSet. The example below shows how this method can be used: 

          
           
ArrayList<String> names = new ArrayList<>();
names.add("Alice");
names.add("Bob");
names.add("Charlie");

HashSet<String> namesSet = new HashSet<>();

// convert the arraylist to a hashset
namesSet.addAll(names);

// print out the contents of the hashset
System.out.println(namesSet);
This will produce the same output as before: 

          
           
[Alice, Bob, Charlie]

Using the Stream API

The Java 8 Stream API includes many powerful features for manipulating collections, and can be used to convert an ArrayList to a HashSet. The example below demonstrates one way of doing this conversion: 

          
           
ArrayList<String> names = new ArrayList<>();
names.add("Alice");
names.add("Bob");
names.add("Charlie");

HashSet<String> namesSet = names.stream()
    .collect(Collectors.toCollection(HashSet::new));

// print out the contents of the hashset
System.out.println(namesSet);
This will again produce the same output: 

          
           
[Alice, Bob, Charlie]

Conclusion

The examples above show three different ways to convert an ArrayList to a HashSet in Java. Depending on the requirements of the task, any of these methods may be used to quickly and easily convert a list of items into a Set.
