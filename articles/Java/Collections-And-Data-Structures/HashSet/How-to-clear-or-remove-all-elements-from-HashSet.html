How to Clear or Remove all Elements from HashSet in Java

Java provides a number of classes and methods for manipulating collections. One of the most useful is the HashSet class, which is a type of Set that allows objects to be stored in a unordered collection with no duplicates. In this tutorial, we'll discuss how to clear or remove all elements from a Java HashSet.

Method 1: Using the Clear Method

One of the simplest ways to clear or remove all elements from a Java HashSet is by using the 

          
           
clear()
method. This method will empty the HashSet of all its elements and is done in-place, meaning it happens on the same object without creating a new instance. Here is an example of how to use the 

          
           
clear()
method to remove all elements from a Java HashSet: 

          
           
HashSet<String> names = new HashSet<>();
names.add("John");
names.add("Jane");
names.add("Bob");

// Clear the HashSet
names.clear();

// Check if the HashSet is now empty
if (names.isEmpty()) {
    System.out.println("The HashSet is now empty");
}
In this example, we create a HashSet of Strings called 

          
           
names
and add three elements to it. We then call the 

          
           
clear()
method to remove all elements from the HashSet, and then check if it is indeed empty.

Method 2: Using the RemoveAll Method

Another way of removing all elements from a Java HashSet is by using the 

          
           
removeAll()
method. This method takes another collection as a parameter, and then removes all elements in that collection from the HashSet. Here is an example of how to use the 

          
           
removeAll()
method to remove all elements from a Java HashSet: 

          
           
HashSet<String> names = new HashSet<>();
names.add("John");
names.add("Jane");
names.add("Bob");

// Create a new List 
List<String> toRemove = new ArrayList<>();
toRemove.add("John");
toRemove.add("Jane");

// Remove all elements in the List from the HashSet
names.removeAll(toRemove);

// Check if the HashSet is now empty
if (names.isEmpty()) {
    System.out.println("The HashSet is now empty");
}
In this example, we create a HashSet of Strings called 

          
           
names
and add three elements to it. We then create a List with two of the elements from the HashSet, and use the 

          
           
removeAll()
method to remove these elements from the HashSet. We then check if the HashSet is indeed empty.

Method 3: Looping Over the HashSet

It is also possible to loop over a Java HashSet and remove each element one at a time. This can be done with a simple for loop or for-each loop. Here is an example of how to use a for-each loop to remove all elements from a Java HashSet: 

          
           
HashSet<String> names = new HashSet<>();
names.add("John");
names.add("Jane");
names.add("Bob");

// Loop over the HashSet and remove each element
for (String name : names) {
    names.remove(name);
}

// Check if the HashSet is now empty
if (names.isEmpty()) {
    System.out.println("The HashSet is now empty");
}
In this example, we create a HashSet of Strings called 

          
           
names
and add three elements to it. We then loop over the HashSet and remove each element from it. Finally, we check if the HashSet is indeed empty.

Conclusion

In this tutorial, we discussed three methods for clearing or removing all elements from a Java HashSet. We first looked at the 

          
           
clear()
method, which will empty the HashSet of all its elements. Then, we looked at the 

          
           
removeAll()
method, which takes another collection as a parameter and removes all elements in that collection from the HashSet. Finally, we looked at a more manual approach of looping over the HashSet and removing each element one at a time.
