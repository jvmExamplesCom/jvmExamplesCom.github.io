How to Check HashSet Size (Size Method)

When working with Java and its collections, you may need to check how many items are stored in a particular set. This can be accomplished by using the "size" method available on the HashSet class. In this article, we'll demonstrate how to use this method to calculate the size of a HashSet.

The Size Method

The size method is a built-in function that will return the number of elements stored in a particular collection. It can be used on the HashSet class, as well as other collection classes, like ArrayList or LinkedList.

Here is the syntax for the size method: 

          
           
int size = myHashSet.size();

Example Usage of the Size Method

Let's look at an example of how to use the size method. We'll create a simple HashSet, add some values to it, and then use size to calculate how many items are stored in it.


          
           
HashSet<String> myHashSet = new HashSet<>();
myHashSet.add("cat");
myHashSet.add("dog");
myHashSet.add("bird");

int size = myHashSet.size(); // size is 3

In the above example, we create a HashSet and add three elements to it: "cat", "dog", and "bird". We then use the size method to calculate the number of elements stored in the HashSet. Since we added three elements, the size is 3.
