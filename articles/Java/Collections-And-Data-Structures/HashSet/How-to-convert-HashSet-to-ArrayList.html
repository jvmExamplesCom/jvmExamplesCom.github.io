How to Convert HashSet to ArrayList in Java

The HashSet class provides many useful methods for manipulating java collections. One of the most common operations when using a HashSet is converting it to an ArrayList. This is relatively straightforward but there are several different ways to accomplish it. In this article we will discuss the different approaches and compare their performance.

Using Iterator

The first approach is to use an Iterator object to traverse the HashSet and add each element to an ArrayList: 

          
           
Iterator iterator = hashSet.iterator();
ArrayList<String> arrayList = new ArrayList<String>();
while (iterator.hasNext()) {
    arrayList.add(iterator.next());
}
This approach is simple and straightforward, but itâ€™s also relatively slow as it has to iterate over the entire HashSet.

Using For-Each Loop

A more efficient approach is to use a for-each loop to iterate over the HashSet and add each element to the ArrayList: 

          
           
ArrayList<String> arrayList = new ArrayList<String>();
for (String item : hashSet) {
    arrayList.add(item);
}
This approach is much faster as it eliminates the need to create an Iterator object.

Using Streams

Another approach is to use Java 8 Streams to convert the HashSet to an ArrayList: 

          
           
ArrayList<String> arrayList = hashSet.stream()
                                        .collect(Collectors.toCollection(ArrayList::new));
This approach is the fastest of the three as it leverages Java 8 Streams and the Collector facility.

Conclusion

In this article we discussed the different approaches for converting a HashSet to an ArrayList in Java. We compared each approach and concluded that using Java 8 Streams is the fastest and most efficient way to do this operation.
