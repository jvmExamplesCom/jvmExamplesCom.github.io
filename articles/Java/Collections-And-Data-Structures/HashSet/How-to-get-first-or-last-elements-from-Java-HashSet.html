How to Get First or Last Elements from Java HashSet

Java provides various ways to access the first or last element of a HashSet. This article takes a look at some of those methods and explains how they can be used in practical programming scenarios.

Using Iterator

The most common way to access the first or last element in a HashSet is to use an Iterator. An Iterator is an object that allows for the traversal of a collection such as a HashSet - it provides a way to iterate over the elements in a collection. We can use the Iterator's 

          
           
next()
method to access the first element in a HashSet, and the 

          
           
previous()
method to access the last element. 

          
           
HashSet<String> mySet = new HashSet<String>(); 
mySet.add("A"); 
mySet.add("B"); 
mySet.add("C"); 

Iterator<String> iterator = mySet.iterator(); 
String firstElement = iterator.next(); // "A" 
String lastElement = iterator.previous(); // "C"

Using the First() and Last() Methods

The Java Collections framework also provides two methods for accessing the first and last elements of a HashSet. The 

          
           
first()
method returns the first element of the HashSet, and the 

          
           
last()
method returns the last element. 

          
           
HashSet<String> mySet = new HashSet<String>(); 
mySet.add("A"); 
mySet.add("B"); 
mySet.add("C"); 

String firstElement = mySet.first(); // "A" 
String lastElement = mySet.last(); // "C"

Using the toArray() Method

Another way to access the first or last element of a HashSet is to use the 

          
           
toArray()
method. This method creates an array containing all the elements of the HashSet and thus allows us to directly access the first and last element. 

          
           
HashSet<String> mySet = new HashSet<String>(); 
mySet.add("A"); 
mySet.add("B"); 
mySet.add("C"); 

Object[] myArray = mySet.toArray(); 
String firstElement = (String) myArray[0]; // "A" 
String lastElement = (String) myArray[myArray.length - 1]; // "C"

Conclusion

In this article, we looked at how to access the first or last element of a Java HashSet using several different methods. We saw how to use an Iterator, the first() and last() methods, and the toArray() method. These techniques can be used in any programming scenario where it is necessary to access the first or last element of a HashSet.
