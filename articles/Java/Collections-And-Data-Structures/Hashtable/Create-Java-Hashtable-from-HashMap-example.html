Create Java Hashtable from HashMap Example

A Hashtable is a Java collection class that maps keys to values. It works much like a HashMap, but with certain differences. In this article, we'll show how to create a Hashtable from a HashMap.

Example 1: Basic Usage of Hashtable

Let's start with a basic example. We'll use the put() and get() methods to add and retrieve elements in the Hashtable. 

          
           
Hashtable<String, String> fruits = new Hashtable<String, String>();
fruits.put("Apple", "Fruit");
fruits.put("Banana", "Fruit");

String appleType = fruits.get("Apple"); // Outputs: "Fruit"
As you can see, we created a Hashtable object and added two elements to it. Then, we retrieved the value for the key "Apple".

Example 2: Create a Hashtable from a HashMap

We can also create a Hashtable from a HashMap. This is useful when we want to convert existing HashMap objects into Hashtables. To do so, we'll use the constructor Hashtable(Map map): 

          
           
HashMap<String, String> fruitMap = new HashMap<String, String>();
fruitMap.put("Apple", "Fruit");
fruitMap.put("Banana", "Fruit");

Hashtable<String, String> fruits = new Hashtable<String, String>(fruitMap);
This code creates a new Hashtable and adds all the elements of the HashMap fruitMap to it.

Example 3: Use a Custom Comparator

We can also use a custom Comparator when creating a Hashtable from a HashMap. For example, let's say we want to sort the elements in our Hashtable by their values: 

          
           
Comparator<Object> valueComparator = new Comparator<Object>() {
  @Override public int compare(Object o1, Object o2) {
    return ((String) o1).compareTo((String) o2);
  }
};

HashMap<String, String> fruitMap = new HashMap<String, String>();
fruitMap.put("Apple", "Fruit");
fruitMap.put("Banana", "Fruit");

Hashtable<String, String> fruits = new Hashtable<String, String>(fruitMap, valueComparator);
This code creates a new Hashtable with elements sorted by their values using the custom Comparator.

Conclusion

In this article, we showed how to create a Hashtable from a HashMap in Java. We used the constructor Hashtable(Map map) and demonstrated how to use a custom Comparator.
