Check if a Particular Key Exists in Java Hashtable Example

Java provides a class called 

          
           
Hashtable
which is used to implement a hash table. It stores key/value pairs of objects and allows access to them using the corresponding key. A key may exist multiple times in the table, but it can only have one value associated with it.

Checking if a Key Exists in a Hashtable

To check if a particular key exists in a 

          
           
Hashtable
, one can use the 

          
           
containsKey()
method of the 

          
           
Hashtable
class. This method takes a single argument which is the key to check for and returns a boolean - true if the key exists, false otherwise. Here's an example: 

          
           
import java.util.Hashtable;

public class HashTableExample{ 
   public static void main(String[] args){ 
      // Create a Hashtable object 
      Hashtable<String, String> h = new Hashtable<String, String>(); 
       
      // Add some elements to the Hashtable object 
      h.put("key_1", "Value_1"); 
      h.put("key_2", "Value_2"); 
       
      // Check if the key exists 
      boolean keyExists = h.containsKey("key_1"); 
       
      // Print the result 
      System.out.println("Key exists: " + keyExists); 
   } 
}
In this example, we create a 

          
           
Hashtable
object, add two elements to it, and then use the 

          
           
containsKey()
method to check whether the key 

          
           
key_1
exists or not. The output of this program will be: 

          
           
Key exists: true
