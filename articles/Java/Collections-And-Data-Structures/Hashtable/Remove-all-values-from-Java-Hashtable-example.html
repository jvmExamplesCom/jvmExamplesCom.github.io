Removing All Values From Java Hashtable

Java Hashtable is a data structure used to store values using a key-value pair. It is an implementation of the Map interface. It offers many advantages such as fast lookup and retrieval of items, thread safety, and ability to store null values. In this article, we will learn how to remove all values from Java Hashtable in different ways. We will review the relevant methods of the Java library and offer code examples to showcase their usage.

Using `clear()` Method

The `clear()` method of Hashtable can be used to remove all the key-value pairs from the table. This method does not take any argument and returns void. Here is an example: 

          
           
// create a Hashtable 
Hashtable<Integer, Integer> ht = new Hashtable<Integer, Integer>(); 

// populate the Hashtable with values 
ht.put(1, 10); 
ht.put(2, 20); 
ht.put(3, 30); 

// print the Hashtable 
System.out.println("Hashtable before clear(): " + ht); 

// remove all the values 
ht.clear(); 

// print the Hashtable 
System.out.println("Hashtable after clear(): " + ht);
Output: 

          
           
Hashtable before clear(): {1=10, 2=20, 3=30}
Hashtable after clear(): {}

Using `remove()` Method

The `remove()`method of Hashtable can be used to remove a specific key-value pair from the table. This method takes the key of the pair as an argument and returns its value or `null` if no such key exists in the table. Here is an example: 

          
           
// create a Hashtable 
Hashtable<Integer, Integer> ht = new Hashtable<Integer, Integer>(); 

// populate the Hashtable with values 
ht.put(1, 10); 
ht.put(2, 20); 
ht.put(3, 30); 

// print the Hashtable 
System.out.println("Hashtable before remove(): " + ht); 

// remove a value 
int removedValue = ht.remove(2); 

// print the Hashtable 
System.out.println("Hashtable after remove(): " + ht);
Output: 

          
           
Hashtable before remove(): {1=10, 2=20, 3=30}
Hashtable after remove(): {1=10, 3=30}

Using `entrySet()` Method

The `entrySet()` method can be used to return a set of Map.Entry objects iterated in the order of their keys. We can loop over this set and call `remove()` method on each item to remove it from the table. Here is an example: 

          
           
// create a Hashtable 
Hashtable<Integer, Integer> ht = new Hashtable<Integer, Integer>(); 

// populate the Hashtable with values 
ht.put(1, 10); 
ht.put(2, 20); 
ht.put(3, 30); 

// print the Hashtable 
System.out.println("Hashtable before removal: " + ht); 

// get the entry set 
Set<Map.Entry<Integer, Integer>> entries = ht.entrySet(); 

// loop over the set 
for (Map.Entry<Integer, Integer> entry : entries) 
{ 
	// remove each item 
	ht.remove(entry.getKey()); 
} 

// print the Hashtable 
System.out.println("Hashtable after removal: " + ht);
Output: 

          
           
Hashtable before removal: {1=10, 2=20, 3=30}
Hashtable after removal: {}

Conclusion

In this article, we learned different ways to remove all values from Java Hashtable. We saw how to use `clear()`, `remove()` and `entrySet()` methods to achieve our goal.
