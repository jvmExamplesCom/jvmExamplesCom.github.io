Iterate through Values of Java Hashtable Example

Hashtables in Java are used to store key/value pairs and can be useful for various programming operations. Iterating through a Hashtable is an important skill to have when accessing its values. In this article, we will explore how to iterate through the values of a Hashtable example in Java.

Using the Enumeration Interface

The first method of iterating through the values of a Hashtable is to use the 

Enumeration
interface. This is an iterator-like interface that provides access to the elements of the table. To use it, we must first obtain the enumeration using the 
elements()
method. 

          
           
Hashtable<String, Integer> ht = new Hashtable<String, Integer>();

ht.put("One", 1);
ht.put("Two", 2);
ht.put("Three", 3);

Enumeration<Integer> e = ht.elements();
Once we have the enumeration, we can use the 
hasMoreElements()
method to check if there are any elements left in the enumeration and the 
nextElement()
method to get the next element. Here is an example of a loop to iterate through the values of the Hashtable: 

          
           
while (e.hasMoreElements()) {
  Integer value = e.nextElement();
  System.out.println(value);
}
The output of the above code is: 

          
           
1
2
3

Using the For-Each Loop

The second method of iterating through the values of a Hashtable is to use the Java 5 

for-each
loop. This loop allows us to iterate through each element of an iterable structure, such as a Hashtable. Here is an example of a 
for-each
loop to iterate through the values of the Hashtable: 

          
           
for (Integer value : ht.values()) {
  System.out.println(value);
}

The output of the above code is the same as the previous example:


          
           
1
2
3

Conclusion

In this article, we explored two ways to iterate through the values of a Java Hashtable example. We discussed how to use the 

Enumeration
interface and the Java 5 
for-each
loop to do this. With these two approaches, you should now be able to easily iterate through the values in a Hashtable.
