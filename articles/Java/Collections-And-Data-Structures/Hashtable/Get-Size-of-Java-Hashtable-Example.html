Get Size of Java Hashtable Example

Java Hashtable is a class which allows us to store and access key-value pairs. It provides us various methods like 

          
           
size()
, 

          
           
put()
, 

          
           
get()
, 

          
           
remove()
etc. In this article, we will discuss how to get the size of a Map or Hashtable in Java.

Example: Get Size of Java Hashtable

In the following example, we will create a Hashtable object and add some key-value pairs to it and then we will use the 

          
           
size()
method to get the size of the Hashtable. 

          
           
import java.util.Hashtable;

public class GetSizeJavaHashtableExample {

   public static void main(String[] args) {

      // Create a Hashtable object
      Hashtable<String, Integer> hashtable = new Hashtable<String, Integer>();

      // Add key value pairs to the Hashtable
      hashtable.put("Key1", 10);
      hashtable.put("Key2", 20);
      hashtable.put("Key3", 30);

      // Get the size of the Hashtable
      int size = hashtable.size();
      System.out.println("Size of the Hashtable: " + size);
   }
}
Output: 

          
           
Size of the Hashtable: 3
