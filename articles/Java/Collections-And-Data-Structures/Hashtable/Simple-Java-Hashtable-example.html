Java Hashtable Example

A Java Hashtable is a data structure consisting of a collection of values associated with keys. These values can be retrieved through their related key. A hashtable is an example of a mapping of keys to objects, or in other words - an associative array. The basic operations of a hashtable are get and put. In this tutorial, the core concepts of Java Hashtables will be introduced along with a working example.

Creating a Hashtable

To create a hashtable, you must first create an instance of Hashtable with the desired data type for its keys and values. This could look something like the following:


          
           
Hashtable<String, Integer> myTable = new Hashtable<>();

In this example, both the keys and the values will be of type String and Integer respectively.

Adding Values to the Hashtable

Adding elements to a hashtable is done using the put() method. This method takes two parameters - the key and the value. This could look something like the following:


          
           
myTable.put("Key1", 10);
myTable.put("Key2", 20);
myTable.put("Key3", 30);

In this example, three elements have been added to the hashtable with keys "Key1", "Key2", and "Key3" and corresponding values 10, 20, and 30 respectively.

Retrieving Values From a Hashtable

To retrieve a value from a hashtable, you must use the get() method. This method takes the key as its parameter and returns the corresponding value. This could look something like the following:


          
           
int value1 = myTable.get("Key1"); // Returns 10
int value2 = myTable.get("Key2"); // Returns 20
int value3 = myTable.get("Key3"); // Returns 30

Removing Values From a Hashtable

To remove a value from a hashtable, you must use the remove() method. This method takes the key as its parameter and removes the element associated with that key. This could look something like the following:


          
           
myTable.remove("Key2"); // Key2 is removed and its associated value (20) is no longer in the hashtable

Iterating Through a Hashtable

Iterating through a hashtable is done using the iterator() method. This method returns a set of elements contained in the hashtable. This set of elements can then be iterated over using a looping construct such as a for loop. This could look something like the following:


          
           
Set<String> keys = myTable.keySet();
for (String key : keys) {
    System.out.println(key + " " + myTable.get(key));
}

A Working Example

To tie together all of the concepts discussed in this tutorial, here is a working example of a Hashtable:


          
           
import java.util.Hashtable; 

public class MyClass { 
  public static void main(String[] args) { 
    // Create a new Hashtable 
    Hashtable<String, Integer> myTable = new Hashtable<>(); 
  
    // Add some elements 
    myTable.put("Key1", 10); 
    myTable.put("Key2", 20); 
    myTable.put("Key3", 30); 
  
    // Get an element 
    int value1 = myTable.get("Key1"); 
    System.out.println(value1); // Prints 10 
  
    // Remove an element 
    myTable.remove("Key2"); 
  
    // Iterate through the Hashtable 
    Set<String> keys = myTable.keySet(); 
    for (String key : keys) { 
      System.out.println(key + " " + myTable.get(key)); 
    } 
  } 
}
When run, this code will output the following: 

          
           
Key1 10
Key3 30
