Java Hashtable Example - Get Collection of Values

The Java Hashtable class provides a method to get all the values contained in the hashtable object. The following example demonstrates how to collect all the values from a hashtable object using the 

          
           
elements()
method.

Example

Let's look at an example to demonstrate how to retrieve all the values from a hashtable object: 

          
           
// Create a HashTable 
Hashtable<Integer, String> hashtable = new Hashtable<Integer, String>(); 
  
// Add key value pairs to HashTable 
hashtable.put(1, "Apple"); 
hashtable.put(2, "Pear"); 
hashtable.put(3, "Banana"); 
  
// Get all the values 
Collection<String> valueCollection = hashtable.values(); 
  
// Print all the values 
for (String value : valueCollection) { 
    System.out.println(value); 
}
Output: 

          
           
Apple
Pear
Banana
In the above example, we first create a hashtable object that contains 3 key-value pairs. Next, we called the 

          
           
values()
method on the hashtable object and assigned the returned value to a collection object. Finally, we used an enhanced for loop to iterate over the collection object and print out the values contained in the hashtable object.
