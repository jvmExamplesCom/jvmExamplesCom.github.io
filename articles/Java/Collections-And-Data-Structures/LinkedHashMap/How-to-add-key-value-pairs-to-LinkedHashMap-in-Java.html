Adding Key-Value Pairs to LinkedHashMap in Java

LinkedHashMap is a type of Collection in Java that allows storing of key-value pairs. It is useful for Creating, Retrieving, and assigning specific values to keys in a map. To add key-value pairs to this collection, we use the 

          
           
put(K, V)
method. This method accepts two arguments – a key and a value. The most basic syntax for using this method is as follows: 

          
           
LinkedHashMap<K,V> linkedHashMap = new LinkedHashMap<>();
linkedHashMap.put(K key, V value);
In the above example, we first create a LinkedHashMap object with the generic type of K and V. Then we call the 

          
           
put
method to add the specified key-value pair to the object. Let's see an example of adding a simple String key and Integer value to the LinkedHashMap. 

          
           
LinkedHashMap<String, Integer> phoneBook = new LinkedHashMap<>(); 
phoneBook.put("John", 123456); 
phoneBook.put("Jane", 789012);
In the above example, We created a LinkedHashMap called 

          
           
phoneBook
with the generic type of 

          
           
String
and 

          
           
Integer
. Then we added two key-value pairs to the Map – 

          
           
("John", 123456)
and 

          
           
("Jane", 789012)
. It's important to note that duplicate keys are not allowed in a LinkedHashMap. So if we try to insert the same key twice, it will replace the previous key-value pair with the new one. For example: 

          
           
phoneBook.put("John", 654321);
In the above example, we are trying to add a new key-value pair 

          
           
("John", 654321)
, which will replace the existing one 

          
           
("John", 123456)
. Apart from the basic syntax, we can also use the putIfAbsent() method to add key-value pairs. This method allows us to insert the key-value pair only if the specified key is not present in the Map. Let's take the same example of adding the name-phone number pair to the phoneBook LinkedHashMap. 

          
           
phoneBook.putIfAbsent("John", 654321);
Here, the 

          
           
putIfAbsent
method will check if the key 

          
           
"John"
is absent in the Map or not. If it is absent, then it will add the key-value pair 

          
           
("John", 654321)
to the Map. But if it is already present, then nothing will happen. It is also possible to add multiple key-value pairs to the Map using the 

          
           
putAll(Map m)
method. This method accepts a Map object as a parameter and adds all its key-value pairs to the current LinkedHashMap. Let's take an example of adding two Maps to a single LinkedHashMap. 

          
           
LinkedHashMap<String, Integer> phoneBook1 = new LinkedHashMap<>();
phoneBook1.put("John", 123456);
phoneBook1.put("Jane", 789012);

LinkedHashMap<String, Integer> phoneBook2 = new LinkedHashMap<>();
phoneBook2.put("Emily", 345678);
phoneBook2.put("Tom", 901234);

LinkedHashMap<String, Integer> phoneBook3 = new LinkedHashMap<>(); 
phoneBook3.putAll(phoneBook1); 
phoneBook3.putAll(phoneBook2);
In the above example, we first created two Maps - 

          
           
phoneBook1
and 

          
           
phoneBook2
, and added key-value pairs to them. Then we created a third Map 

          
           
phoneBook3
and used the 

          
           
putAll()
method to add all the entries from 

          
           
phoneBook1
and 

          
           
phoneBook2
into it. We can also use the 

          
           
replace(K key, V value)
method to update the existing key-value pair in the Map. This method accepts two parameters – the key and the new value to be assigned to it. For example: 

          
           
phoneBook.replace("John", 654321);
In the above example, we are replacing the existing value for the key 

          
           
"John"
with the new value 

          
           
654321
. To remove key-value pairs from the LinkedHashMap, we use the 

          
           
remove(K key)
method. This method accepts the key of the entry to be removed and returns the associated value. Let's take an example of removing the entry for John from the phoneBook. 

          
           
phoneBook.remove("John");
In the above example, We are using the 

          
           
remove()
method to delete the entry associated with the key 

          
           
"John"
. We can also use the 

          
           
clear()
method to remove all the entries from the Map. This method deletes all the key-value pairs and initializes the Map to the empty state. 

          
           
phoneBook.clear();
That's it! In this article, we discussed how to add key-value pairs to LinkedHashMap in Java. We looked at different methods such as 

          
           
put()
, 

          
           
putIfAbsent()
, 

          
           
putAll()
and 

          
           
replace()
to add entries to the Map. We also learned how to remove entries by using 

          
           
remove()
and 

          
           
clear()
methods.
