How to Check if LinkedHashMap Contains a Key

Java's 

          
           
LinkedHashMap
class is an implementation of the popular 

          
           
Map
interface. It stores objects in key-value pairs and maintains an ordering between elements that is determined by when they are inserted. A 

          
           
LinkedHashMap
instance may need to be tested to see if it contains a given key. This can be done in several ways.

Using the containsKey() Method

The most direct way to determine if a 

          
           
LinkedHashMap
has a specific key is to use the 

          
           
containsKey()
method. It takes a single argument of type 

          
           
Object
, which should be the key for which to search. 

          
           
LinkedHashMap<String, String> map = new LinkedHashMap<>();
map.put("key1", "value1");
map.put("key2", "value2");
boolean containsKey1 = map.containsKey("key1"); // returns true
boolean containsKey2 = map.containsKey("key2"); // returns true
boolean containsKey3 = map.containsKey("key3"); // returns false

Using the get() Method

The 

          
           
get()
method can also be used to search for a key in a 

          
           
LinkedHashMap
. The method takes a single argument of type 

          
           
Object
, which should be the key for which to search. If the key is found, the corresponding value will be returned; if not, 

          
           
null
will be returned. 

          
           
LinkedHashMap<String, String> map = new LinkedHashMap<>();
map.put("key1", "value1");
map.put("key2", "value2");
String value1 = map.get("key1"); // returns "value1"
String value2 = map.get("key2"); // returns "value2"
String value3 = map.get("key3"); // returns null

Using the keySet() Method

The 

          
           
keySet()
method returns a 

          
           
Set
containing all of the keys in the 

          
           
LinkedHashMap
. This set can then be tested for the presence of a given key using the 

          
           
contains()
method. 

          
           
LinkedHashMap<String, String> map = new LinkedHashMap<>();
map.put("key1", "value1");
map.put("key2", "value2");
Set<String> keys = map.keySet();
boolean containsKey1 = keys.contains("key1"); // returns true
boolean containsKey2 = keys.contains("key2"); // returns true
boolean containsKey3 = keys.contains("key3"); // returns false

Using the entrySet() Method

The 

          
           
entrySet()
method returns a 

          
           
Set
containing all of the entries in the 

          
           
LinkedHashMap
. Each entry is represented by a 

          
           
Map.Entry
object that contains both the key and the corresponding value. This set can then be searched for a given key using the 

          
           
contains()
method. 

          
           
LinkedHashMap<String, String> map = new LinkedHashMap<>();
map.put("key1", "value1");
map.put("key2", "value2");
Set<Map.Entry<String, String>> entries = map.entrySet();
boolean containsKey1 = false;
for (Map.Entry<String, String> entry : entries) {
    if (entry.getKey().equals("key1")) {
        containsKey1 = true;
    }
} // returns true
boolean containsKey2 = false;
for (Map.Entry<String, String> entry : entries) {
    if (entry.getKey().equals("key2")) {
        containsKey2 = true;
    }
} // returns true
boolean containsKey3 = false;
for (Map.Entry<String, String> entry : entries) {
    if (entry.getKey().equals("key3")) {
        containsKey3 = true;
    }
} // returns false
By using one of the methods described above, it is possible to check if a 

          
           
LinkedHashMap
contains a specific key.
