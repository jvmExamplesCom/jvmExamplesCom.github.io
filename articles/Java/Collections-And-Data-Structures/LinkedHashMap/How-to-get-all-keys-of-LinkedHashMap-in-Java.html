How to Get All Keys of LinkedHashMap in Java

A 
LinkedHashMap
is an implementation of the 
Map
interface with predictable iteration order. It maintains a doubly-linked list running through all of its entries, and allows insertion-order iteration over the whole map. This makes it useful for caching applications and other cases where you want to iterate over the entries in an ordered fashion.

In this tutorial we’ll demonstrate how to get all keys of the 

LinkedHashMap
using the 
keySet()
and 
entrySet()
methods. We’ll also show how to use a 
forEach()
loop to iterate over the keys of the 
LinkedHashMap.

Prerequisites

To follow along with this tutorial you need to have a basic understanding of Java 8.

Getting Started with LinkedHashMap

In order to get started working with 
LinkedHashMap
, we’ll first need to create an instance of the object. To do this we can use the following code: 

          
           
LinkedHashMap<String, Integer> linkedHashMap = new LinkedHashMap<>();
This creates an instance of 
LinkedHashMap
with a generic type of 
String
and 
Integer
. Now we can start adding items to the map. For example, we can add some key-value pairs like this: 

          
           
linkedHashMap.put("Hello", 1);
linkedHashMap.put("World", 2);

Getting All Keys of a LinkedHashMap with the keySet() Method

The 
keySet()
method is used to return a 
Set
of all the keys of a 
LinkedHashMap.
This is useful if you only need the keys of the map and don’t need the corresponding values. To get the keys of the 
LinkedHashMap
we created earlier, we can use the following code: 

          
           
Set<String> keys = linkedHashMap.keySet();

// print all the keys
keys.forEach(System.out::println);

This prints out the two keys that we added earlier:


          
           
Hello
World

Getting All Keys of a LinkedHashMap with the entrySet() Method

The 
entrySet()
method is used to return a 
Set
of all the entries of a 
LinkedHashMap.
This is useful if you need both the keys and the associated values. To get the entries of the 
LinkedHashMap
we created earlier, we can use the following code: 

          
           
Set<Map.Entry<String, Integer>> entries = linkedHashMap.entrySet();

// print all the keys
entries.forEach(entry -> System.out.println(entry.getKey()));

This prints out the two keys that we added earlier:


          
           
Hello
World

Iterating Over All Keys of a LinkedHashMap with a forEach() Loop

We can also use a 
forEach()
loop to iterate over the keys of a 
LinkedHashMap.
This is useful if you need to do something with each key in the map, such as printing it out or performing some operation on it. To do this with the 
LinkedHashMap
we created earlier, we can use the following code: 

          
           
linkedHashMap.forEach((key, value) -> {
    System.out.println(key);
});

This prints out the two keys that we added earlier:


          
           
Hello
World

And there you have it, three ways to get all of the keys of a 

LinkedHashMap
in Java.
