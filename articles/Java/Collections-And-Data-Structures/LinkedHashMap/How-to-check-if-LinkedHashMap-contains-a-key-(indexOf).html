How to Check if LinkedHashMap Contains a Key (indexOf)

LinkedHashMap is a Java implementation of the Map interface, which is a basic feature of the Java Collections Framework. This class stores key-value pairs in an ordered manner, allowing for efficient retrieval based on the key. As such, it is often used when there is a need to track the order in which elements were added. In this tutorial, we will discuss how to check if LinkedHashMap contains a specific key.

Using the indexOf() Method

The easiest way to check if LinkedHashMap contains a certain key is to use the indexOf() method, which returns the index of the specified key in the map. The syntax for this method is as follows: 

          
           
int indexOf(Object key);
This method accepts a single argument, namely the key whose index is to be determined. If the specified key is present in the LinkedHashMap, the index of that key is returned. Otherwise, -1 is returned to indicate that the key does not exist in the map.

An Example Program

Let's take a look at a simple program that illustrates how to use the indexOf() method to check if LinkedHashMap contains a certain key: 

          
           
import java.util.LinkedHashMap;

public class LinkedHashMapExample {
    public static void main(String[] args) {
        LinkedHashMap<String, Object> map = new LinkedHashMap<>();
        map.put("A", 1);
        map.put("B", 2);
        map.put("C", 3);

        String key = "A";
        int index = map.indexOf(key);

        System.out.println("Index of " + key + " is " + index);
    }
}
In this example, we first create a LinkedHashMap with three key-value pairs. We then define a String variable called key and assign it the value "A"; this is the key whose index we want to determine. Next, we call the indexOf() method, passing in the key as an argument. Finally, we print out the index of the key to the console. If we run this program, its output should look something like this: 

          
           
Index of A is 0
This indicates that the key "A" is present in the LinkedHashMap, and its index is 0.
