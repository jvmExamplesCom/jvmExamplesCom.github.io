How to get first or last entry from Java LinkedHashMap?

Java LinkedHashMap is a type of Map that preserves the order of insertion. This makes it an excellent option for storing data if you need to retain the order of elements.

The following code snippet illustrates how to get the first and last entries from a LinkedHashMap:


          
           
LinkedHashMap<String, String> map = new LinkedHashMap<>();
map.put("A", "Apple");
map.put("B", "Banana");
map.put("C", "Cherry");

String firstEntry = map.entrySet().iterator().next().getValue();
String lastEntry = map.entrySet().toArray()[map.size()-1].getValue();

System.out.println("First Entry: " + firstEntry);
System.out.println("Last Entry: " + lastEntry);
This code produces the following output: 

          
           
First Entry: Apple
Last Entry: Cherry

In this code, we first create a LinkedHashMap with three key/value pairs. We then use the entrySet() method to obtain the Set of Map.Entry objects in the LinkedHashMap. We can then use the iterator() method to obtain an Iterator object, which we can use to access the first element in the LinkedHashMap. Finally, we use the toArray() method to convert the Set of Map.Entry objects into an array and then use the size() method to get the last element.

We can also use the keySet() method to obtain a Set of keys in the LinkedHashMap, and then use the get() method to get the values. The following code shows how to do this:


          
           
LinkedHashMap<String, String> map = new LinkedHashMap<>();
map.put("A", "Apple");
map.put("B", "Banana");
map.put("C", "Cherry");

String firstEntry = map.get(map.keySet().iterator().next());
String lastEntry = map.get(map.keySet().toArray()[map.size()-1]);

System.out.println("First Entry: " + firstEntry);
System.out.println("Last Entry: " + lastEntry);

This code produces the same output as the previous code:
