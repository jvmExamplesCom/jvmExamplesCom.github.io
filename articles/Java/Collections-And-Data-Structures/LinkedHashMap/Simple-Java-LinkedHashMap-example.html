Simple Java LinkedHashMap example

A LinkedHashMap uses a hash table and linked list to store entries, ensuring the order of insertion remains intact. LinkedHashMaps also allow null values with a unique NULL key. This article will provide detailed examples of how to implement and use LinkedHashMaps in Java.

Getting started

To create and use a LinkedHashMap in Java, you must first create the map. The map is created by initializing the LinkedHashMap using its constructor. The constructor takes two parameters: 1. Initial capacity of the map 2. Load factor for the map The initial capacity is the size of the map, or the number of available key-value pairs it can hold, before it starts growing. The load factor defines how full the LinkedHashMap should be before it grows in size. Below is an example of creating a LinkedHashMap with initial capacity of 5 and a load factor of 0.75. 

          
           
LinkedHashMap<String, Integer> map = new LinkedHashMap<>(5, 0.75);

Adding Elements

Now that the map is initialized, you can begin adding elements. LinkedHashMaps support all types of key and value combinations such as primitives, objects, and strings. 

          
           
map.put("one", 1);
In this example, we’re adding a String key “one” and an Integer value 1. You can also add objects and primitives to the map. 

          
           
map.put(2, "two");
map.put(true, 3);
In this example, we’ve added an Integer key 2 and a String value “two” as well as a Boolean key true and an Integer value 3. You can also add elements to the LinkedHashMap using the putIfAbsent() method which only adds the specified key-value pair if the key does not already exist in the map. 

          
           
map.putIfAbsent(4, "four");

Iterating Over Elements

You can iterate over the elements in a LinkedHashMap using the entrySet() method which returns a Set of Map.Entry objects containing the key-value pairs contained in the map. 

          
           
for (Map.Entry<String, Integer> entry : map.entrySet()) {
    System.out.println("key: " + entry.getKey() + ", value: " + entry.getValue());
}
This code will iterate over each key-value pair in the map and print the key and value to the console.

Removing Elements

You can use the remove() method to remove a specific key from the LinkedHashMap. 

          
           
map.remove("one");
This code will remove the element with the key “one” from the map. You can also use the remove() method to remove a key-value pair from the map if the value matches the specified value. 

          
           
map.remove("two", 2);
This code will remove the element with the key “two” and the value 2 if it exists in the map. You can also use the clear() method to remove all elements from the LinkedHashMap. 

          
           
map.clear();

Conclusion

LinkedHashMaps are useful when you need to maintain the order of insertion. They are also useful when you need to add or remove elements based on keys or values. In this article, we covered how to initialize, add elements, iterate over elements, and remove elements from a LinkedHashMap.
