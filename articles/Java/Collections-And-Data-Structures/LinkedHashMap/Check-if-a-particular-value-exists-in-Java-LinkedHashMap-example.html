Check if a particular value exists in Java LinkedHashMap example

Using the LinkedHashMap Class, you can check to see if a particular value exists in the map. This can be done by calling the containsValue() method, which takes an Object as an argument. The method will return true if a particular value is present in the map and false if it is not.

Java Example

Below is a simple example that demonstrates how to check if a particular value exists in a LinkedHashMap. 

          
           
// Create a LinkedHashMap  
LinkedHashMap<Integer, String> names = new LinkedHashMap<>(); 
        
// Add values to the LinkedHashMap  
names.put(1, "John"); 
names.put(2, "Mike"); 
names.put(3, "Tom"); 
        
// Check if "Tom" exists in the LinkedHashMap  
boolean exists = names.containsValue("Tom"); 
System.out.println("Does 'Tom' exist? " + exists);
The output of the code above will be: 

          
           
Does 'Tom' exist? true
As can be seen from the example, the containsValue() method will return true if the value is present in the map and false if it is not. This can be used to easily check if a particular value exists in the LinkedHashMap.
