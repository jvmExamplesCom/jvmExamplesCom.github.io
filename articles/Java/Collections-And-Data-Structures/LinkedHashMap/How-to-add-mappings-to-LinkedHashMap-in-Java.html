How to Add Mappings to LinkedHashMap in Java

LinkedHashMap is one of the basic data structures in Java. It offers a combination of fast lookup times and sorting performance, enabling efficient use of various operations like accessing and updating elements. This tutorial will show you how to add mappings to LinkedHashMap in Java.

Creating a LinkedHashMap Instance

First of all, let’s create a LinkedHashMap instance: 

          
           
LinkedHashMap<String, Integer> linkedHashMap = new LinkedHashMap<>();
We just created an empty LinkedHashMap object that can store String keys and Integer values.

Adding Elements to LinkedHashMap Instance

Once we have a LinkedHashMap instance, we can start adding elements to it using the 

          
           
.put()
method. We need to pass the key of the element and its value to the 

          
           
.put()
method. For example, to add a mapping for the key "apple" with value 10, we can do this: 

          
           
linkedHashMap.put("apple", 10);
We can also use the 

          
           
.putIfAbsent()
method if we only want to add the mapping if the key does not already exist. For example: 

          
           
linkedHashMap.putIfAbsent("apple", 20);
This adds the mapping if there isn't an existing mapping for the key "apple". Otherwise, it does nothing.

Updating Elements in a LinkedHashMap Instance

The 

          
           
.put()
and 

          
           
.putIfAbsent()
methods can also be used for updating existing mappings. All we need to do is pass the same key and a different value from the one currently stored in the LinkedHashMap instance. For example, to change the value of the key "apple" from 10 to 25: 

          
           
linkedHashMap.put("apple", 25);

Retrieving Elements from a LinkedHashMap Instance

We can use the 

          
           
.get()
method to retrieve the value associated with a particular key. Just pass the key as an argument to the 

          
           
.get()
method and it will return the corresponding value. For example, to get the value of the key "apple": 

          
           
Integer appleValue = linkedHashMap.get("apple");
At this point, 

          
           
appleValue
will be equal to 25 as we added the mapping for "apple" with value 25 in the previous step.

Removing Elements from a LinkedHashMap Instance

To remove a mapping from the LinkedHashMap instance, we can use the 

          
           
.remove()
method. Just pass the key as an argument to the 

          
           
.remove()
method and the corresponding mapping will be removed from the LinkedHashMap instance. For example, to remove the mapping for the key "apple": 

          
           
linkedHashMap.remove("apple");
At this point, the mapping for the key "apple" will no longer exist in the LinkedHashMap instance.

Conclusion

In this tutorial, we saw how to add mappings to LinkedHashMap in Java. We started by creating a LinkedHashMap instance. Then, we learned how to add elements using the 

          
           
.put()
and 

          
           
.putIfAbsent()
methods. We also learned how to update existing mappings, retrieve elements and remove elements from the LinkedHashMap instance.
