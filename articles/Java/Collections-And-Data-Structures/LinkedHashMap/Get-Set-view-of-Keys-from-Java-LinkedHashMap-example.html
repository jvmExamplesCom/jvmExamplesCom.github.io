Keys from Java LinkedHashMap Example

Java LinkedHashMap is a powerful data structure that can be used to store data in key-value pairs. Keys are often used for searching, sorting and retrieving values stored in a LinkedHashMap. In this article, we will look at how to get the keys from a LinkedHashMap using the Java API.

Using the KeySet() Method

The LinkedHashMap class provides a number of methods to access the keys stored in the map. The simplest of these methods is the 

          
           
keySet()
method which returns a Set of the keys stored in the map. This can then be iterated over using a for-each loop, as shown below: 

          
           
LinkedHashMap<String, Integer> map = new LinkedHashMap<>();
map.put("A", 1);
map.put("B", 2);
map.put("C", 3);

Set<String> keys = map.keySet();
for (String key : keys) {
	System.out.println(key);
}
// Outputs:
// A
// B
// C

Using the EntrySet() Method

Another way to get the keys from a LinkedHashMap is to use the 

          
           
entrySet()
method. This returns a set of Map.Entry objects which can then be used to get the keys stored in the map. 

          
           
LinkedHashMap<String, Integer> map = new LinkedHashMap<>();
map.put("A", 1);
map.put("B", 2);
map.put("C", 3);

Set<Map.Entry<String, Integer>> entries = map.entrySet();
for (Map.Entry<String, Integer> entry : entries) {
	System.out.println(entry.getKey());
}
// Outputs:
// A
// B
// C

Using the Values() Method

Finally, another way to get the keys from a LinkedHashMap is to use the 

          
           
values()
method. This returns a Collection of the values stored in the map, which can then be used to get the keys. 

          
           
LinkedHashMap<String, Integer> map = new LinkedHashMap<>();
map.put("A", 1);
map.put("B", 2);
map.put("C", 3);

Collection<Integer> values = map.values();
for (Integer value : values) {
	System.out.println(map.getKey(value));
}
// Outputs:
// A
// B
// C
As you can see, there are a number of ways to get the keys from a LinkedHashMap using the Java API. Each of these methods has its own advantages and disadvantages, and it is important to choose the one that best suits your needs.
