Get Size of Java LinkedHashMap Example

This article provides an example of how to get the size of a Java LinkedHashMap. A LinkedHashMap is a hash table and linked list implementation of the Map interface, with predictable iteration order. Before getting started, let’s first import the necessary packages: 

          
           
import java.util.LinkedHashMap;
import java.util.Map;
Next, we’ll create a new LinkedHashMap and add some key-value pairs to it: 

          
           
Map<String, Integer> map = new LinkedHashMap<>();

map.put("John", 25);
map.put("Mark", 32);
map.put("Ann", 19);
map.put("Tom", 28);
Now that our map is ready, let’s take a look at how to get the size of it. We can use the size() method of the LinkedHashMap class to get the size of the map. The size() method returns an int value that represents the number of key-value pairs in the map. 

          
           
int size = map.size();
Let’s print out the size of our map to the console. 

          
           
System.out.println("The size of the map is: " + size);
If we run the program, we should get the following output: 

          
           
The size of the map is: 4
As you can see, the size of our map is 4, as we have added 4 key-value pairs to it. We can also use the isEmpty() method to determine if our map is empty or not. This method returns a boolean value - true if the map is empty and false if it contains elements. 

          
           
boolean isEmpty = map.isEmpty();
And that’s it! In this article, we learned how to get the size of a Java LinkedHashMap using the size() method. We also saw how to use the isEmpty() method to check if the map is empty or not.
