Iterating a LinkedHashMap in Reverse Order (Backward Direction)

The Java LinkedHashMap is an implementation of the Map interface that maintains the order of entries inserted into the map. This is done by keeping the entries in doubly-linked list. The advantage of using a LinkedHashMap is that you can traverse the keys in the same order as they were entered, as well as in reverse order. In this article, we will discuss how to use the LinkedHashMap's descendingKeySet method to iterate the entries in reverse order.

Using Iterator

The first method of iterating a LinkedHashMap in reverse order is to use the iterator and traverse the LinkedHashMap in reverse. The following snippet shows how this can be done. 


          
           
LinkedHashMap<String, Integer> lhm = new LinkedHashMap<>();
lhm.put("one", 1);
lhm.put("two", 2);
lhm.put("three", 3);

// Create a reverse view of the LinkedHashMap
Map<String, Integer> reverseView = 
    Collections.unmodifiableMap(
        new LinkedHashMap<>(lhm).descendingMap());

// Get an iterator on the reverse view and traverse it
Iterator<String> iterator = reverseView.keySet().iterator();
while (iterator.hasNext()) {
    String key = iterator.next();
    Integer value = reverseView.get(key);
    System.out.println(key + ": " + value);
}

When you run this code, you should see the following output.


          
           
three: 3
two: 2
one: 1

Using for-each Loop

Another way to iterate a LinkedHashMap in reverse order is to use the for-each loop and traverse the LinkedHashMap in reverse. The following snippet shows how this can be done. 


          
           
LinkedHashMap<String, Integer> lhm = new LinkedHashMap<>();
lhm.put("one", 1);
lhm.put("two", 2);
lhm.put("three", 3);

// Create a reverse view of the LinkedHashMap
Map<String, Integer> reverseView = 
    Collections.unmodifiableMap(
        new LinkedHashMap<>(lhm).descendingMap());

for(String key : reverseView.keySet()) {
    Integer value = reverseView.get(key);
    System.out.println(key + ": " + value);
}

When you run this code, you should see the following output.


          
           
three: 3
two: 2
one: 1

Conclusion

In this article, we have discussed two ways to iterate a LinkedHashMap in reverse order. Both of these methods use the LinkedHashMap's descendingKeySet method to create a reverse view of the LinkedHashMap. The first method uses an iterator while the second method uses a for-each loop.
