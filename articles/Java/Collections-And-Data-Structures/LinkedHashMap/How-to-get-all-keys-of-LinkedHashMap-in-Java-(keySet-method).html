How to Get All Keys of LinkedHashMap in Java (keySet Method)

LinkedHashMap is a useful class for storing key-value mappings of any type. It stores the values in insertion order and can be used in situations where order matters. Moreover, it provides a number of methods for accessing the data it stores. One of these methods is the 

keySet
method, which is used to return a set of all keys stored in the LinkedHashMap.

Using keySet Method

The keySet method is an inbuilt method in Java that belongs to the 

LinkedHashMap
class. This method returns a set view of the keys stored in the LinkedHashMap. 

          
           
public Set<K> keySet()

This method returns a set view of the keys in this LinkedHashMap. The set view is backed by the LinkedHashMap, so changes to one are reflected in the other.

Syntax


          
           
LinkedHashMap<key, value> object = new LinkedHashMap<key, value>; 

Set<key> = object.keySet();

Example 1: Getting all keys from LinkedHashMap

In this example, we will use the LinkedHashMap's keySet method to get all the keys in the map. 


          
           
import java.util.*; 

public class GettingKeys { 
	public static void main(String[] args) 
	{ 
		// Creating LinkedHashMap 
		LinkedHashMap<String, Integer> lhm 
			= new LinkedHashMap<String, Integer>(); 
		
		// Adding elements to LinkedHashMap 
		lhm.put("Summer", 1); 
		lhm.put("Winter", 2); 
		lhm.put("Spring", 3); 
		lhm.put("Autumn", 4); 

		// Fetching all keys from LinkedHashMap 
		Set<String> keys = lhm.keySet(); 
		System.out.println("Keys of LinkedHashMap: "
						+ keys); 
	} 
}
Output: 

          
           
Keys of LinkedHashMap: [Summer, Winter, Spring, Autumn]

Example 2: Getting an array of keys from a LinkedHashMap

We can also use the LinkedHashMap's keySet method to get an array of all the keys in the map. 


          
           
import java.util.LinkedHashMap; 
import java.util.Set; 

public class GettingArrayKeys { 
	public static void main(String[] args) 
	{ 
		// Creating LinkedHashMap 
		LinkedHashMap<String, Integer> lhm 
			= new LinkedHashMap<String, Integer>(); 
		
		// Adding elements to LinkedHashMap 
		lhm.put("Summer", 1); 
		lhm.put("Winter", 2); 
		lhm.put("Spring", 3); 
		lhm.put("Autumn", 4); 

		// Fetching all keys from LinkedHashMap 
		Set<String> keys = lhm.keySet(); 

		// Converting the set of keys to an array 
		String[] array = keys.toArray(new String[keys.size()]); 

		for (String key : array) 
			System.out.print(key + " "); 
	} 
}
Output: 

          
           
Summer Winter Spring Autumn
Thus, the LinkedHashMap's 
keySet
method is a useful and efficient way to get all of the keys stored in the map.
