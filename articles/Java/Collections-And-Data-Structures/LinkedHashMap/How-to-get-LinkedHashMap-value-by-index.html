How to get LinkedHashMap Value by Index in Java

LinkedHashMaps, like other Maps, offer a way of accessing an element from the Map based on its index. It is important to note that, unlike in an array, LinkedHashMaps do not have an intrinsic order and therefore don’t necessarily return items in the same order you put them in. Therefore, accessing an item based on its index does not guarantee any particular result.

Creating a LinkedHashMap

Before we look at how to get LinkedHashMap values based on their index, let’s first look at how to create a LinkedHashMap. To illustrate this, we will create a simple LinkedHashMap with five String elements: 

          
           
Map<String, String> myMap = new LinkedHashMap<>();

myMap.put("key1", "value1");
myMap.put("key2", "value2");
myMap.put("key3", "value3");
myMap.put("key4", "value4");
myMap.put("key5", "value5");
This will create a LinkedHashMap containing five key-value pairs.

Accessing LinkedHashMap values by Index

Now that we have created our LinkedHashMap, we can look at how to access its values by index. The LinkedHashMap class provides two methods for returning elements from the map by index—entrySet() and keySet(). These two methods return Set views of the Map, with the entrySet() method returning a Set view of the Map’s entries, and the keySet() method returning a Set view of the Map’s keys. 

          
           
Set<Map.Entry<String, String>> entries = myMap.entrySet();
Set<String> keys = myMap.keySet();
Using these Set views, we can use the Set.toArray() method to create an array of the entries/keys in the Map, and access the elements by index. For example, if we want to access the value for “key3” from our Map, we can use the following code: 

          
           
String[] keysArray = keys.toArray(new String[keys.size()]);
String value3 = myMap.get(keysArray[2]);
Here, we are using the toArray() method to create an array of String objects containing all of the Map’s keys. We then access the third element in the array—the key for “key3”—and use it to return the value associated with it using the Map.get() method. In this case, the value returned would be “value3”. We could use the same approach to access the third element in the Map’s entrySet(). To do this, we would use the following code: 

          
           
Map.Entry<String, String>[] entriesArray = entries.toArray(new Map.Entry[entries.size()]);
String value3 = entriesArray[2].getValue();
Here, we create an array of Map.Entry objects representing the entries in the Map, and then use the array’s third element—in this case, the entry for “key3”—to return the associated value using the Map.Entry.getValue() method. Again, the value returned would be “value3”.

Conclusion

In this tutorial, we looked at how to access LinkedHashMap values by index. We saw how to create a LinkedHashMap and how to use the LinkedHashMap.entrySet() and LinkedHashMap.keySet() methods to access values from the Map. We then looked at examples of how to use the toArray() method to create an array of Map entries or keys, and then use the resulting array to access the value associated with a particular index.
