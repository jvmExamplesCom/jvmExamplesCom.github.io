Java LinkedHashMap forEach Example

In this article, we will be discussing the Java LinkedHashMap forEach example. Before we start to discuss about this example, let us first understand what LinkedHashMap is and how it is different from other Map implementations like HashMap. A LinkedHashMap is an implementation of the Map interface which stores its entries in a doubly-linked list. It provides predictable iteration order for its elements which makes it an ideal choice when order matters. Now, let us see how to use the forEach method with LinkedHashMap. The forEach() method is a recently added method in Java which is defined in the Iterable and Stream Interface. This method takes a single argument which is an object of type Consumer and it performs the given action on each element of the LinkedHashMap. Here is an example given below which illustrates the use of forEach() method with LinkedHashMap: 

          
           
LinkedHashMap<String,Integer> map = new LinkedHashMap<>();
map.put("A", 1);
map.put("B", 2);
map.put("C", 3);

map.forEach((key,value) -> System.out.println(key + " : " + value));
In the above code, we have created a LinkedHashMap with three key-value pairs and then we are using forEach() method to iterate through each pair and print them on the console. The output of the above program is as follows: 

          
           
A : 1
B : 2
C : 3
The forEach() method also supports parallel operations by using the Java 8 feature of parallel streams. Here is an example of how to use the parallel feature of the forEach() method: 

          
           
LinkedHashMap<String,Integer> map = new LinkedHashMap<>();
map.put("A", 1);
map.put("B", 2);
map.put("C", 3);

map.entrySet().parallelStream().forEach(entry ->
  System.out.println(entry.getKey() + " : " + entry.getValue()));
In the above code, we are using the entrySet() of the LinkedHashMap and then using a parallel stream API to perform a parallel operation on all the elements of the map. The output of the above program is as follows: 

          
           
C : 3
A : 1
B : 2
The forEach() operation can be used to perform various tasks such as printing, sorting, filtering etc. It is a very convenient way to iterate through the elements of the LinkedHashMap and perform some operation on them without having to write the Iterator loop.

Conclusion

In this article, we discussed the Java LinkedHashMap forEach example. We saw how to use forEach() with LinkedHashMap and also saw how to use the parallel feature of forEach() for parallel operations.
