Removing Values From Java LinkedHashMap Example

LinkedHashMap is an implementation of a Map interface that implements the Map interface and uses the doubly linked list to store the entries in the order of insertion. It stores the Keys and the corresponding value in the form of a key-value mapping. LinkedHashMap allows the insertion, removal and search time complexity of O(1). The LinkedHashMap can also be iterated in the same order as the insertions.

Example

Let's look at the example below to understand how to remove elements from a LinkedHashMap.


          
           
import java.util.LinkedHashMap;

public class LinkedHashMapExample {

  public static void main(String[] args) {
    // Create a LinkedHashMap
    LinkedHashMap<String, Integer> linkedHashMap = new LinkedHashMap<>();

    // Add keys and values (Key, Value)
    linkedHashMap.put("One", 1);
    linkedHashMap.put("Two", 2);
    linkedHashMap.put("Three", 3);
    linkedHashMap.put("Four", 4);           

    System.out.println("LinkedHashMap: " + linkedHashMap);

    // Remove value for key "Three"
    linkedHashMap.remove("Three");  

    System.out.println("LinkedHashMap after removal: " + linkedHashMap);    
  }
}

When we run the above program, the output will be:


          
           
LinkedHashMap: {One=1, Two=2, Three=3, Four=4}
LinkedHashMap after removal: {One=1, Two=2, Four=4}

In this example, we have created a LinkedHashMap with the keys and values. We have then removed the value for key “Three” using the remove() method.

Removing All Values

To remove all the values from a LinkedHashMap, you can use the clear() method of the LinkedHashMap class.


          
           
import java.util.LinkedHashMap;

public class LinkedHashMapExample {

  public static void main(String[] args) {
    // Create a LinkedHashMap
    LinkedHashMap<String, Integer> linkedHashMap = new LinkedHashMap<>();

    // Add keys and values (Key, Value)
    linkedHashMap.put("One", 1);
    linkedHashMap.put("Two", 2);
    linkedHashMap.put("Three", 3);
    linkedHashMap.put("Four", 4);           

    System.out.println("LinkedHashMap before removal: " + linkedHashMap);

    // Remove all values
    linkedHashMap.clear();  

    System.out.println("LinkedHashMap after removal: " + linkedHashMap);    
  }
}

When we run the above program, the output will be:


          
           
LinkedHashMap before removal: {One=1, Two=2, Three=3, Four=4}
LinkedHashMap after removal: {}

In this example, we have created a LinkedHashMap with the keys and values. We have then removed all the values in the LinkedHashMap using the clear() method.
