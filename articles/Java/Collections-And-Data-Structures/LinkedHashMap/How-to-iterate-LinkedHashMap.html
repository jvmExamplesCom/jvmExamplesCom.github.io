How to Iterate LinkedHashMap in Java

LinkedHashMap is a hashtable and linked list implementation of the Map interface, with predictable iteration order. It provides several ways to iterate an instance of LinkedHashMap. The easiest way to iterate a LinkedHashMap is through its entrySet, which can be accessed using the entrySet() method. This returns a Set view of the mappings contained in the map.

Using for-each loop to Iterate a LinkedHashMap

For iteration over the entire LinkedHashMap, you can use a foreach loop. This example shows how to use a forEach loop to iterate a LinkedHashMap instance: 

          
           
LinkedHashMap<Integer, String> hm = new LinkedHashMap<Integer, String>(); 

hm.put(1, "Apple"); 
hm.put(2, "Mango"); 

for (Map.Entry<Integer, String> entry : hm.entrySet()) { 
	System.out.println("Key = " + entry.getKey() + 
				", Value = " + entry.getValue()); 
}
This code will output the following result: 

          
           
Key = 1, Value = Apple
Key = 2, Value = Mango

Using Iterator to Iterate a LinkedHashMap

You can also use the iterator to iterate a LinkedHashMap instance. To do this, you can use the iterator() method of the LinkedHashMap class. The following example shows how to use the iterator to iterate over the entire LinkedHashMap: 

          
           
LinkedHashMap<Integer, String> hm = new LinkedHashMap<Integer, String>(); 

hm.put(1, "Apple"); 
hm.put(2, "Mango"); 

Iterator<Map.Entry<Integer, String>> it = hm.entrySet().iterator();

while (it.hasNext()) {
	Map.Entry<Integer, String> entry = it.next();
	System.out.println("Key = " + entry.getKey() + 
				", Value = " + entry.getValue()); 
}
This code will output the following result: 

          
           
Key = 1, Value = Apple
Key = 2, Value = Mango

Using forEach and Lambda Expression to Iterate a LinkedHashMap

You can also use the forEach() method of the LinkedHashMap class along with the lambda to iterate over the LinkedHashMap. The following example shows how to use forEach() and lambda expression to iterate over the entire LinkedHashMap instance: 

          
           
LinkedHashMap<Integer, String> hm = new LinkedHashMap<Integer, String>(); 

hm.put(1, "Apple"); 
hm.put(2, "Mango"); 

hm.forEach((k,v)->System.out.println("Key = " + k + 
				", Value = " + v));

Using Stream API to Iterate a LinkedHashMap

Java 8 added a Stream API that gives you the ability to process data declaratively and leverage multicore architectures without any specific knowledge. You can use the stream API to iterate a LinkedHashMap instance as shown in the following example: 

          
           
LinkedHashMap<Integer, String> hm = new LinkedHashMap<Integer, String>(); 

hm.put(1, "Apple"); 
hm.put(2, "Mango"); 

hm.entrySet().stream().forEach(e -> System.out.println("Key = " + e.getKey() +
