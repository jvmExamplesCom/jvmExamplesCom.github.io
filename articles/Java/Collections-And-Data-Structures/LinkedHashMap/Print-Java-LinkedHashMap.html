Java LinkedHashMap

LinkedHashMap is a powerful class that is part of the Java Collections Framework. It is an implementation of the Map interface and provides key/value mappings. It is also similar to HashMap, but it maintains insertion order of elementsâ€”allowing you to iterate over them in the same order they were added. In this article, we will look at the syntax and usage of LinkedHashMap and explore some examples to help understand how it works.

Syntax of a LinkedHashMap

The syntax of LinkedHashMap is as follows: 

          
           
LinkedHashMap<k, v> map = new LinkedHashMap<k, v>();</k,></k,>
Here, K and V refer to the type of the key and value stored in the LinkedHashMap respectively.

Methods in LinkedHashMap

LinkedHashMap has several important methods for adding elements, removing elements and updating values associated with keys.

put(K key, V value)

This method inserts a mapping of key and value into the LinkedHashMap. The key must be unique.

get(Object key)

This method returns the element associated with the given key.

remove(Object key)

This method removes the element associated with the given key.

clear()

This method removes all elements from the LinkedHashMap.

containsKey(Object key)

This method returns true if the given key is present in the LinkedHashMap.

containsValue(Object value)

This method returns true if the given value is present in the LinkedHashMap.

Examples of LinkedHashMap

Let's look at a few examples to understand how LinkedHashMap works.

Example 1: Adding elements to a LinkedHashMap

In this example, we will create a LinkedHashMap and add elements to it. 

          
           
public static void main(String[] args) { 
  LinkedHashMap<Integer, String> map = new LinkedHashMap<Integer, String>(); 
 
  map.put(1, "Apple"); 
  map.put(2, "Banana"); 
  map.put(3, "Cherry"); 
  
  System.out.println(map); 
}
Output 

          
           
{1=Apple, 2=Banana, 3=Cherry}

Example 2: Accessing elements of a LinkedHashMap

In this example, we will access the elements of a LinkedHashMap using the get() method. 

          
           
public static void main(String[] args) { 
  LinkedHashMap<Integer, String> map = new LinkedHashMap<Integer, String>(); 
 
  map.put(1, "Apple"); 
  map.put(2, "Banana"); 
  map.put(3, "Cherry"); 
  
  System.out.println(map.get(2));

Example 3: Updating elements of a LinkedHashMap

In this example, we will update the elements of a LinkedHashMap using the put() method. 

          
           
public static void main(String[] args) { 
  LinkedHashMap<Integer, String> map = new LinkedHashMap<Integer, String>(); 
 
  map.put(1, "Apple"); 
  map.put(2, "Banana"); 
  map.put(3, "Cherry"); 
  
  map.put(2, "Strawberry"); 
  
  System.out.println(map);
