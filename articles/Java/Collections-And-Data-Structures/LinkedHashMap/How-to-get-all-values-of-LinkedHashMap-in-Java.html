How to Get All Values of LinkedHashMap in Java

The 
LinkedHashMap
class is a subclass of the 
HashMap
class. It's a combination of a 
HashMap
and a 
DoublyLinkedList
. It basically provides the functionality of both classes. It is used when you want to keep the order of entries in a 
Map
. The 
LinkedHashMap
class has several useful methods which will help you get all the values in an instance of a 
LinkedHashMap
. In this article, we'll look at methods for getting all the values from a 
LinkedHashMap
in Java.

Method 1: Using the 
values()
Method

The 
LinkedHashMap
class provides the 
values()
method which returns a 
Collection
view of all the values in the 
LinkedHashMap
. It can be used as follows: 

          
           
LinkedHashMap<Integer, String> map = new LinkedHashMap<Integer, String>();
map.put(1, "One");
map.put(2, "Two");
map.put(3, "Three");

Collection<String> values = map.values();

for (String value : values) {
    System.out.println(value);
}
The output of the above code is: 

          
           
One
Two
Three

Method 2: Using the 
forEach()
Method

The 
LinkedHashMap
class also provides the 
forEach()
method which can be used to iterate over the values in the 
LinkedHashMap
. This method takes a 
BiConsumer
object as an argument. The 
BiConsumer
object is invoked for each entry in the 
LinkedHashMap
. We can use this method to get all the values in the 
LinkedHashMap
as follows: 

          
           
LinkedHashMap<Integer, String> map = new LinkedHashMap<Integer, String>();
map.put(1, "One");
map.put(2, "Two");
map.put(3, "Three");

map.forEach((key, value) -> System.out.println(value));
The output of the above code is: 

          
           
One
Two
Three

Method 3: Using the 
entrySet()
Method

The 
LinkedHashMap
class also provides the 
entrySet()
method which returns a 
Set
view of all the entries in the 
LinkedHashMap
. We can use this method to get all the values in the 
LinkedHashMap
as follows: 

          
           
LinkedHashMap<Integer, String> map = new LinkedHashMap<Integer, String>();
map.put(1, "One");
map.put(2, "Two");
map.put(3, "Three");

Set<Map.Entry<Integer, String>> entries = map.entrySet();

for (Map.Entry<Integer, String> entry : entries) {
    System.out.println(entry.getValue());
}
The output of the above code is: 

          
           
One
Two
Three

In this article, we looked at three different methods of getting all the values from a 

LinkedHashMap
in Java. We looked at using the 
values()
, 
forEach()
and 
entrySet()
methods. Each of these methods can be used to get all the values in a 
LinkedHashMap
.
