Clearing or Removing All Mappings from a LinkedHashMap

LinkedHashMap is an implementation of the Map interface that preserves the insertion order of elements. It maintains a linked list of all its entries. In this article, we will learn about how to clear or remove all mappings from a LinkedHashMap in Java.

Using the clear() Method:

The easiest and most efficient way to remove all the elements from a LinkedHashMap is using the 

          
           
clear()
method. It removes all of the elements from the map. The syntax of this method is as follows: 

          
           
public void clear()
After calling this method on a LinkedHashMap, the size of the map will be zero. The following code snippet demonstrates the use of the 

          
           
clear()
method to remove all the elements from a LinkedHashMap: 

          
           
LinkedHashMap<String, String> map = new LinkedHashMap<>();
map.put("key1", "value1");
map.put("key2", "value2");
map.put("key3", "value3");

System.out.println("Map before clearing: " + map);
map.clear();
System.out.println("Map after clearing: " + map);
The output of this code snippet will be: 

          
           
Map before clearing: {key1=value1, key2=value2, key3=value3}
Map after clearing: {}

Using the removeAll() Method:

Another way to clear all the elements from a LinkedHashMap is using the 

          
           
removeAll(Collection c)
method. This method takes a collection of keys as an argument and removes all the elements with those keys from a LinkedHashMap. The syntax of this method is as follows: 

          
           
public boolean removeAll(Collection c)
The following code snippet demonstrates the use of the 

          
           
removeAll()
method to remove all the elements from a LinkedHashMap: 

          
           
LinkedHashMap<String, String> map = new LinkedHashMap<>();
map.put("key1", "value1");
map.put("key2", "value2");
map.put("key3", "value3");

System.out.println("Map before removing all: " + map);
map.removeAll(map.keySet());
System.out.println("Map after removing all: " + map);
The output of this code snippet will be: 

          
           
Map before removing all: {key1=value1, key2=value2, key3=value3}
Map after removing all: {}

Conclusion

In this article, we have learned about how to clear or remove all the elements from a LinkedHashMap in Java. We have seen the two different ways of doing this, which are using the 

          
           
clear()
method and the 

          
           
removeAll(Collection c)
method.
