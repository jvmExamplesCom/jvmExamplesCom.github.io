Creating LinkedHashMap in Java (LinkedHashMap Constructors)

Java has a class called 

          
           
LinkedHashMap
which is a Hash table and linked list implementation of the 

          
           
Map
interface. It preserves the insertion order of the elements, and it is similar to 

          
           
HashMap
with the only difference being that it maintains the order in which the entries were added to the Map. In addition, it can retrieve elements in the same order they were added. LinkedHashMap provides the following two constructors to create an empty LinkedHashMap: 

          
           
LinkedHashMap()
LinkedHashMap(int initialCapacity)
The first constructor creates an empty LinkedHashMap with the default initial capacity (16) and load factor (0.75). The second constructor creates an empty LinkedHashMap with the specified initial capacity and the default load factor (0.75). To add elements to the LinkedHashMap, use the 

          
           
put
method. This method adds the mapping for the key and the value: 

          
           
public V put(K key, V value)
The following example demonstrates how to create a 

          
           
LinkedHashMap
and add elements to it: 

          
           
// Creating a LinkedHashMap 
LinkedHashMap<String, Integer> map = new LinkedHashMap<String, Integer>(); 

// Adding elements to the LinkedHashMap 
map.put("apple", 10); 
map.put("orange", 20); 
map.put("mango", 30);
You can also use the 

          
           
putAll()
method to add the contents of another 

          
           
Map
to the LinkedHashMap: 

          
           
public void putAll(Map<? extends K, ? extends V> m)

putAll()

// Creating another Map

Map

LinkedHashMap(Map) constructor
: 

          
           
public LinkedHashMap(Map<? extends K, ? extends V> m)

LinkedHashMap(Map)

// Creating a LinkedHashMap from a Map

You can also specify the load factor and the ordering mode while creating a LinkedHashMap using the following constructor: 

          
           
public LinkedHashMap(int initialCapacity, float loadFactor, boolean accessOrder)
The parameter 

          
           
accessOrder

true
