Convert ArrayList to LinkedHashMap in Java

Java provides several ways to convert an ArrayList to a LinkedHashMap. The following code example demonstrates how to do this using the collect and toMap methods of the Stream API.

Using Stream API and Collectors.toMap

The Stream API provides an efficient way to convert an ArrayList to a LinkedHashMap. In the following example, an ArrayList of Person objects is created, and then converted to a LinkedHashMap. 

          
           
ArrayList<Person> people = new ArrayList<>();
people.add(new Person("John", 25));
people.add(new Person("Mary", 32));
people.add(new Person("David", 28));

Map<String, Integer> peopleMap =
  people.stream()
    .collect(Collectors.toMap(
      Person::getName, 
      Person::getAge, 
      (existingVal, newVal) -> existingVal, 
      LinkedHashMap::new));
The collect() method takes a Collector object as parameter and calls the toMap() method of the Collectors class with four arguments. The first argument is a Function which returns the key for the Map. The second is a Function which returns the value for the Map. The third argument (optional) is a BinaryOperator which merges two values if there is a collision. The fourth argument is a Supplier which is used to create the Map instance. In this example, the LinkedHashMap constructor is used as a Supplier to create a new LinkedHashMap instance. The resulting LinkedHashMap will have the same order as the original ArrayList.

Using Stream API and Collectors.toMap with MergeFunction

If we need to do something more advanced, such as merging two values if there is a collision, we can use the third argument of the toMap() method, which is a BinaryOperator. In the following example, a map of names and ages is created, and then merged with the result of the stream() operation. If there is a collision, the name and age from the map will be used instead of the one from the stream. 

          
           
// existing map containing names and ages

  people.stream()

In this example, the third argument of the toMap() method is a BinaryOperator which takes two values and returns one. If there is a collision between the existing map and the stream, the existing value (from the map) will be used.

In this example, the forEach method is called on the ArrayList and the put method of the LinkedHashMap is called for each item in the list.
