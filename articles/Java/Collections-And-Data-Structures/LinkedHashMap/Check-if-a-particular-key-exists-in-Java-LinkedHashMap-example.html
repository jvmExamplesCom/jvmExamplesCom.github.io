Checking if a Particular Key Exists in Java LinkedHashMap Example

A LinkedHashMap is an implementation of the interface Map, which is part of the Java Collections Framework. It is a hash table and linked list combination, allowing for fast access to individual elements of the structure. LinkedHashMaps are particularly useful when you need to maintain the order in which your keys and values were placed, or when you need to access elements quickly.

In this article we will discuss how to check if a particular key exists in a Java LinkedHashMap. We will be using the containsKey() method. This method returns a boolean value, true if the specified key is found in the LinkedHashMap, and false if it is not.

Example 1:

For our first example, we will create a LinkedHashMap with two elements, and then check to see if a particular key exists. The code below demonstrates this process.


          
           
LinkedHashMap<String, Integer> map = new LinkedHashMap<>();
map.put("One", 1);
map.put("Two", 2);

boolean keyExists = map.containsKey("One");
System.out.println(keyExists); //prints true

In this example, we have created a LinkedHashMap named 

map
, and added two elements to it. We then use the 
containsKey()
method to check if the key 
"One"
is present in the LinkedHashMap. The method returns a boolean value, which is printed to the console. As expected, we get 
true
as the result.

Example 2:

In this example, we will check to see if a key that does not exist in the LinkedHashMap returns false when we search for it.


          
           
LinkedHashMap<String, Integer> map = new LinkedHashMap<>();
map.put("One", 1);
map.put("Two", 2);

boolean keyExists = map.containsKey("Three");
System.out.println(keyExists); //prints false

In this example, we have the same LinkedHashMap structure as before, but this time we use the 

containsKey()
method to check if the key 
"Three"
is present. We get 
false
as a result, since this key does not exist in the LinkedHashMap.

Conclusion

In this article, we discussed how to check if a particular key exists in a Java LinkedHashMap. We used the 

containsKey()
