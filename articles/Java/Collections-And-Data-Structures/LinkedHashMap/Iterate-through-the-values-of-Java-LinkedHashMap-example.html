Iterate Through the Values of Java LinkedHashMap Example

Java LinkedHashMap is an ordered version of HashMap, which means that it maintains the order in which entries are added. It stores key-value pairs and provides constant time performance for the basic operations such as get() and put(). LinkedHashMap also provide methods to iterate through the values of the map. In this article, we'll look at how to iterate through the values of a Java LinkedHashMap example.

Using For Each Loop

The most straightforward way to iterate through the values of a LinkedHashMap is to use a for each loop: 

          
           
LinkedHashMap<String, Integer> map = new LinkedHashMap<>();

map.put("one", 1);
map.put("two", 2);
map.put("three", 3);

for (Integer value : map.values()) {
    System.out.println(value);
}
This code snippet creates a LinkedHashMap and adds three key-value pairs. We then use a for each loop to iterate through the values of the map. The output of this code will be: 

          
           
1
2
3

Using Iterator

Another way to iterate through the values of a LinkedHashMap is to use an iterator. An iterator is an object that allows us to traverse through a collection such as a LinkedHashMap and access its elements one by one. To iterate through the values of a LinkedHashMap, we can use the values() method, which returns a collection-view of all the values in the map. We can then use the Iterator interface to traverse through the collection. Here's an example: 

          
           
LinkedHashMap<String, Integer> map = new LinkedHashMap<>();

map.put("one", 1);
map.put("two", 2);
map.put("three", 3);

Iterator<Integer> it = map.values().iterator();
while (it.hasNext()) {
    System.out.println(it.next());
}
This code snippet creates a LinkedHashMap and adds three key-value pairs. We then use the values() method to get a collection-view of the values. We then use the Iterator interface to traverse through the collection. The output of this code will be: 

          
           
1
2
3
