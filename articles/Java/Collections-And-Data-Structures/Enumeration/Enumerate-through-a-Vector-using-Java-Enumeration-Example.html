Enumerate Through a Vector Using Java Enumeration Example

Java provides a special class called 'Enumeration' that allows us to iterate over the objects of a collection, such as a Vector. This tutorial will demonstrate how to enumerate through a Vector using the Java Enumeration class.

Creating a Vector Object

First, we will create a Vector object that contains some strings. 

          
           
Vector<String> words = new Vector<>();
words.add("Apple");
words.add("Orange");
words.add("Banana");
words.add("Grape");

Creating an Enumeration Object

Next, we will create an Enumeration object from our Vector object. We will use the elements() method from the Vector class to create the Enumeration object. 

          
           
Enumeration<String> enumeration = words.elements();

Iterating Through the Enumeration Object

Finally, we will use a while loop to iterate through the Enumeration object and print out each element. We can use the hasMoreElements() method from the Enumeration class to determine if there are any more elements in the enumeration. 

          
           
while(enumeration.hasMoreElements()) {
    String word = enumeration.nextElement();
    System.out.println(word);
}
