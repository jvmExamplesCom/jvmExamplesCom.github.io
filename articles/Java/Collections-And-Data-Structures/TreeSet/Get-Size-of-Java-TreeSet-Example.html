Get Size of Java TreeSet Example

TreeSet is a collection in Java that stores its elements in a sorted and ascending order. The size of a TreeSet is the number of elements it contains. In this article, we'll look at an example of how to calculate the size of a TreeSet using the size() method.

Example

Let's look at a quick example of how to get the size of a TreeSet. In the following code, we'll create a TreeSet and add some elements to it. We'll then use the size() method to calculate the number of elements in the TreeSet. 

          
           
TreeSet<Integer> numbers = new TreeSet<>();
numbers.add(1);
numbers.add(2);
numbers.add(3);

int size = numbers.size();
System.out.println("Size of TreeSet: " + size);
In the above code, we create a TreeSet called numbers and add three Integer elements to it. We then calculate the size of the TreeSet by calling the size() method on it. This will return an int value which is the number of elements in the TreeSet.

Output

If we run the above code, we'll get the following output: 

          
           
Size of TreeSet: 3
