Get Sub Set from Java TreeSet example

Java TreeSet is a powerful and versatile data structure that allows users to store data in a manner that's both efficient and easy to use. One of the advantages of using TreeSet is that it provides ways to access a subset of the stored elements quickly and easily. In this article, we'll explore how to get a subSet from a TreeSet.

Creating a TreeSet Instance

In order to get a subSet from a TreeSet, we need to first create an instance of a TreeSet to work with. We can do this by creating a new TreeSet object, passing in the appropriate type argument: 

          
           
TreeSet<Integer> tset = new TreeSet<>();
In this example, we've created a TreeSet object called "tset" which will store Integer values.

Populating the TreeSet

Once we have our TreeSet instance, we can start adding elements to it. We can add elements one at a time using the add() method, or we can use the addAll() method to add multiple elements at once: 

          
           
// Add single element
tset.add(3);

// Add a range of elements
List<Integer> list = Arrays.asList(1, 2, 3, 4, 5, 6, 7, 8, 9);
tset.addAll(list);
In this example, we've added a single element (an Integer with the value of 3) to our TreeSet, as well as a range of elements from 1 to 9.

Get a SubSet from the TreeSet

Now that we've populated our TreeSet, we can retrieve a subSet of elements from it. This is done using the subSet() method, which takes two arguments: the lower and upper bounds of the subSet. Both of these arguments must be of the same type as the type argument passed in when creating our TreeSet instance. For example, if we wanted to get a subSet of elements from 3 to 7, we could call the subSet() method like this: 

          
           
SortedSet<Integer> subSet = tset.subSet(3, 7);
The subSet() method returns a SortedSet, which is a sub-interface of the Set interface. We can then iterate over this SortedSet to view the elements within it: 

          
           
for (Integer i : subSet) {
    System.out.println(i);
}

Conclusion

In this article, we looked at how to get a subSet from a TreeSet in Java. We started by creating a TreeSet instance and then added some elements to it. We then used the subSet() method to retrieve a subSet of elements from the TreeSet, and finally, we iterated over the subSet to view its contents.
