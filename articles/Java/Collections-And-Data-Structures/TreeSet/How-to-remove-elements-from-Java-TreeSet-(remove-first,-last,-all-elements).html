How to Remove Elements from Java TreeSet

A Java 
TreeSet
is a collection that stores its elements in a tree structure. It allows you to store data in a sorted order, making it easier to find an element quickly. It also has methods to remove elements from the set.

Removing the First Element

To remove the first element in the tree set, you can use the 

pollFirst()
method. It will remove and return the first element in the tree set. The example below shows how to use the 
pollFirst()
method: 

          
           
TreeSet<Integer> mySet = new TreeSet<Integer>();
mySet.add(1);
mySet.add(2);
mySet.add(3);

Integer removedElement = mySet.pollFirst();
System.out.println("Removed element: "+removedElement);
// Outputs: Removed element: 1
The code above creates a 
TreeSet
and adds three elements to it (1, 2 and 3). Then, it uses the 
pollFirst()
method to remove and return the first element (1) from the set. The output of this code is "Removed element: 1".

Removing the Last Element

Similarly, to remove the last element in the tree set, you can use the 

pollLast()
method. It will remove and return the last element in the tree set. The example below shows how to use the 
pollLast()
method: 

          
           
TreeSet<Integer> mySet = new TreeSet<Integer>();
mySet.add(1);
mySet.add(2);
mySet.add(3);

Integer removedElement = mySet.pollLast();
System.out.println("Removed element: "+removedElement);
// Outputs: Removed element: 3
The code above creates a 
TreeSet
and adds three elements to it (1, 2 and 3). Then, it uses the 
pollLast()
method to remove and return the last element (3) from the set. The output of this code is "Removed element: 3".

Removing All Elements

To remove all elements from the tree set, you can use the 

clear()
method. It will remove all elements from the tree set. This can be useful if you want to start with a blank set. The example below shows how to use the 
clear()
method: 

          
           
TreeSet<Integer> mySet = new TreeSet<Integer>();
mySet.add(1);
mySet.add(2);
mySet.add(3);

mySet.clear();
System.out.println("Size of set: "+mySet.size());
// Outputs: Size of set: 0
The code above creates a 
TreeSet
and adds three elements to it (1, 2 and 3). Then it uses the 
clear()
method to remove all elements from the set. Finally, it prints out the size of the set (which should be 0). The output of this code is "Size of set: 0".

Conclusion

In this tutorial, you have learned how to remove elements from a Java 

TreeSet
. You have seen how to use the 
pollFirst()
, 
pollLast()
and 
clear()
methods to remove one, the last and all elements from the set respectively.
