How to Clear TreeSet (Remove All Elements)

Clearing a TreeSet is an incredibly useful tool for any Java programmer, as it allows you to quickly and easily remove all of the elements from a TreeSet. Examples of Clearing TreeSet

clear()
method on the TreeSet instance. This will remove all elements from the TreeSet. The following code example demonstrates how to do this: 

          
           
TreeSet<String> mySet = new TreeSet<String>();
mySet.add("foo");
mySet.add("bar");
mySet.add("baz");

mySet.clear();

mySet
