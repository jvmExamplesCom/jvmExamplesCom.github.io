Adding Elements to TreeSet in Java

Java’s TreeSet is a specialized version of the common Set interface, which provides several useful benefits for managing large collections of data elements. It supports operations that allow for fast search, insertion, and removal of element values, as well as maintaining order among elements despite frequent additions and deletions. In this article, we will explore how to add elements to a TreeSet collection in Java.

Creating a TreeSet Collection

To use TreeSet in a Java program, you must first initialize a new instance of the collection with the new keyword. You may also need to specify the type of elements the collection will contain, as in the following example: 

          
           
TreeSet<Integer> intSet = new TreeSet<>();
This code creates a new TreeSet of Integer type called intSet. You can also use this general template to create other types of TreeSets, such as String, Float, or Double.

Adding Elements to a TreeSet

The most common way to add elements to a TreeSet is using the 

          
           
add()
method, which adds a single element to the collection. For example, to add 10 to the above intSet, use the following code: 

          
           
intSet.add(10)
You can also use the 

          
           
addAll()
method to add all elements from another collection, such as an ArrayList or TreeMap, to a TreeSet, as in the following example: 

          
           
List<Integer> list = new ArrayList<>();
list.add(11);
list.add(12);
intSet.addAll(list);
This code adds two elements (11 and 12) to intSet by adding them from the list collection.

Remove Elements from a TreeSet

In addition to adding elements to a TreeSet, you can also remove elements using the 

          
           
remove()
method, which removes a single element from the collection. For example, to remove 10 from the above intSet, use the following code: 

          
           
intSet.remove(10)
You can also use the 

          
           
removeAll()
method to remove multiple elements from the TreeSet at once, as in the following example: 

          
           
List<Integer> list2 = new ArrayList<>();
list2.add(13);
list2.add(14);
intSet.removeAll(list2);
This code removes two elements (13 and 14) from intSet by removing them from the list2 collection.

Conclusion

In this article, we explored how to add and remove elements from a TreeSet collection in Java. We saw how to use the 

          
           
add()
, 

          
           
addAll()
, 

          
           
remove()
and 

          
           
removeAll()
methods to add and remove elements from a TreeSet collection. With these methods in hand, you should have no trouble adding and removing elements from TreeSets in your Java programs.
