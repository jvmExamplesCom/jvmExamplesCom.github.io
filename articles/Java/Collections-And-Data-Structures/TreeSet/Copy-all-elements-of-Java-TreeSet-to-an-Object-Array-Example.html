Copy All Elements of Java TreeSet to an Object Array Example

In this article, we will be looking at how to copy the elements of a Java TreeSet into an object array.

Example Code

Let's look at a simple example of how to do this: 

          
           
TreeSet<String> strings = new TreeSet<>();
strings.add("Foo");
strings.add("Bar");

// Copy the elements of the TreeSet into an Object array

Object[] objArray = strings.toArray();

System.out.println(Arrays.toString(objArray);

In this example, we create a TreeSet called "strings" and add two elements to it ("Foo" and "Bar"). We then use the toArray() method from the TreeSet class to copy its elements into an Object array, and use the static toString() method from the Arrays class to print out the contents of the array.

The output of this code would be: 

          
           
[Foo, Bar]
