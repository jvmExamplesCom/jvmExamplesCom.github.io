How to convert HashSet to TreeSet in Java

Java provides a vast array of data structures to store collections of data. Two of the most common data structures are HashSet and TreeSet. HashSet is an unordered collection of elements, while TreeSet is an ordered collection. Sometimes, it may be useful to convert a HashSet to a TreeSet, so that its contents can be displayed in an ordered fashion. In this article, we will discuss how to convert HashSet to TreeSet in Java.

Converting a HashSet to a TreeSet

Let's start by looking at a simple example of how to convert a HashSet to a TreeSet in Java. First, we need to create a new HashSet: 

          
           
HashSet<String> hs = new HashSet<String>();
hs.add("Apple");
hs.add("Banana");
hs.add("Orange");
Now, we can convert this HashSet to a TreeSet using the following code: 

          
           
TreeSet<String> ts = new TreeSet<String>(hs);
This will create a TreeSet containing the same elements as the HashSet.

Converting a HashSet to a TreeSet with a Comparator

Sometimes, it can be useful to convert a HashSet to a TreeSet using a custom Comparator. This allows us to define a custom ordering for the TreeSet using the Comparator’s compare() method. For example, let’s say we have a custom class called Person that contains two fields: name and age. We can create a Comparator to sort the elements of the TreeSet by age: 

          
           
Comparator<Person> comparator = new Comparator<Person>(){
    public int compare(Person p1, Person p2){
        if(p1.getAge() < p2.getAge())
            return -1;
        else if(p1.getAge() > p2.getAge())
            return 1;
        else
            return 0;
    }
};
Now, we can use this Comparator to convert our HashSet to a TreeSet: 

          
           
TreeSet<Person> ts = new TreeSet<Person>(comparator);
ts.addAll(hs);
This will create a TreeSet containing the same elements as the HashSet, but they will be sorted by age.

Conclusion

In this article, we discussed how to convert a HashSet to a TreeSet in Java. We looked at how to do this without a Comparator, as well as how to do it with a Comparator to define a custom ordering for the TreeSet.
