Java TreeSet Example

A TreeSet is a collection that stores its elements in an ordered tree and allows you to access elements quickly. It is part of the Java Collections Framework and is similar to a tree map. This example shows how to create a TreeSet and how to add, remove, and search elements in it.

Creating a TreeSet

To create a TreeSet, you need to specify the type of the elements that the TreeSet will store. You can then create an instance of the TreeSet with the following code:


          
           
TreeSet<String> words = new TreeSet<>();

In this example, we are creating a TreeSet of strings. The angle brackets (<>) indicate that the type of elements is String.

Adding Elements to a TreeSet

You can add elements to a TreeSet using the add() method. For example, to add the strings "apple", "banana", and "cherry" to the TreeSet, you would use the following code:


          
           
words.add("apple");
words.add("banana");
words.add("cherry");

If the element already exists in the TreeSet, it will not be added again.

Removing Elements from a TreeSet

You can remove elements from a TreeSet using the remove() method. For example, to remove the string "cherry" from the TreeSet, you would use the following code:


          
           
words.remove("cherry");

If the element does not exist in the TreeSet, the remove() method will do nothing.

Searching a TreeSet

You can search a TreeSet using the contains() method. For example, to check if the TreeSet contains the string "apple", you would use the following code:


          
           
boolean hasApple = words.contains("apple");

The contains() method returns true if the element is found, or false if it is not found.

Iterating Through a TreeSet

You can loop through all the elements in a TreeSet using the for-each loop. For example, to loop through all the elements in the TreeSet and print them out, you would use the following code:


          
           
for (String word : words) {
    System.out.println(word);
}

This code will print out each element in the TreeSet, one per line.
