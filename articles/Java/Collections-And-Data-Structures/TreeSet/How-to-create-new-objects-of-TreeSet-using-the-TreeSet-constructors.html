How to Create New Objects of TreeSet Using the TreeSet Constructors

Constructor 1: TreeSet()

Constructs a new, empty tree set, sorted according to the natural ordering of its elements.


          
           
TreeSet<Integer> ts1 = new TreeSet<Integer>();

Constructor 2: TreeSet(Collection c)

Builds a new tree set containing the elements in the specified collection c, sorted according to the natural ordering of its elements.


          
           
ArrayList<Integer> al = new ArrayList<Integer>();
al.add(1);
al.add(2);
al.add(3);

TreeSet<Integer> ts2 = new TreeSet<Integer>(al);

Constructor 3: TreeSet(Comparator comparator)

Constructs a new, empty tree set, sorted according to the specified comparator.


          
           
Comparator<Integer> comp = new Comparator<Integer>(){
    public int compare(Integer i1, Integer i2){
        if(i1 < i2)
            return -1;
        else if(i1 == i2)
            return 0;
        else 
            return 1;
    }
};

TreeSet<Integer> ts3 = new TreeSet<Integer>(comp);

Constructor 4: TreeSet(SortedSet s)

Builds a new tree set containing the same elements and using the same ordering as the specified sorted set s.


          
           
TreeSet<Integer> ts4 = new TreeSet<Integer>(ts3);
