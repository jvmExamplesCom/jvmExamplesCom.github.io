Check if a Particular Value Exists in Java TreeSet Example

Java's TreeSet class is an implementation of the Set interface that stores its elements in a tree structure. A TreeSet maintains its elements in order and provides quick and efficient methods for checking whether a particular element exists in the set, as well as for adding or removing elements from the set. In this tutorial, we will take a look at how to check if a particular value exists in a Java TreeSet.

Creating a TreeSet

To create a TreeSet, we simply need to call the TreeSet constructor and pass it our initial values. For example: 

          
           
TreeSet<Integer> treeSet = new TreeSet<>(Arrays.asList(1, 2, 3));
The code above creates a TreeSet containing the values 1, 2, and 3.

Checking for the Presence of a Value

Once we have created our TreeSet, we can check for the presence of a particular value using the 

          
           
contains
method. For example: 

          
           
boolean containsValue = treeSet.contains(2); // true
The code above checks whether the value 2 is present in our TreeSet and assigns the result to the variable 

          
           
containsValue
. In this case, the result is 

          
           
true
, as the TreeSet contains the value 2.

Adding a Value to the TreeSet

If we want to add a new value to the TreeSet, we can use the 

          
           
add
method. For example: 

          
           
treeSet.add(4);
The code above adds the value 4 to our TreeSet.

Removing a Value from the TreeSet

If we want to remove a value from the TreeSet, we can use the 

          
           
remove
method. For example: 

          
           
treeSet.remove(2);
The code above removes the value 2 from our TreeSet.

Conclusion

In this tutorial, we looked at how to check if a particular value exists in a Java TreeSet. We also saw how to add and remove values from the TreeSet.
