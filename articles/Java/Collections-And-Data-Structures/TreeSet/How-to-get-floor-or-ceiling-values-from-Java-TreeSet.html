How to get Floor or Ceiling Values from Java TreeSet

The Java TreeSet is a powerful class that implements the Set interface. The TreeSet stores its elements in a tree-like data structure, which allows it to provide high performance access time. It also provides several useful methods for efficiently finding elements in the set. One such method is the floor() and ceiling() methods, which can be used to find the floor or ceiling element in the set.

What are Floor and Ceiling Elements?

Floor and ceiling elements are elements that are either lower or higher than a given element in a set. For example, if you have a set containing the numbers 1, 2, 4, 6, 8, the floor of 4 is 2, and the ceiling is 6. Floor and ceiling elements are useful when you need to find a range of elements in a set, or perform certain calculations based on the values of the elements.

Using the floor() and ceiling() Methods

The floor() and ceiling() methods are both used to find the floor and ceiling elements in a set. They both take a single parameter, which is the element for which you want to find the floor or ceiling. Both methods return the element that is either lower or higher than the given element.

Here's an example of how to use the floor() and ceiling() methods to find the floor and ceiling elements of the set 3, 5, 7, 9:


          
           
TreeSet<Integer> set = new TreeSet<>();
set.add(3);
set.add(5);
set.add(7);
set.add(9);

int floor = set.floor(6); // floor is 5
int ceiling = set.ceiling(6); // ceiling is 7

In this example, we created a TreeSet containing the elements 3, 5, 7, and 9. Then, we used the floor() and ceiling() methods to find the floor and ceiling elements of 6. The floor is 5, and the ceiling is 7.

Example Using a Comparator

The floor() and ceiling() methods can also take a second parameter, which is a Comparator object that is used to compare the elements. This can be useful if you want to define a custom comparison that does not use the element's natural ordering.

Here's an example using a Comparator to find the floor and ceiling of the set 4, 6, 8, 10:


          
           
Comparator<Integer> comparator = new Comparator<Integer>() {
    @Override
    public int compare(Integer o1, Integer o2) {
        if (o1 == o2) {
            return 0;
        }
        return o1 < o2 ? -1 : 1;
    }
};

TreeSet<Integer> set = new TreeSet<>(comparator);
set.add(4);
set.add(6);
set.add(8);
set.add(10);

int floor = set.floor(7); // floor is 6
int ceiling = set.ceiling(7); // ceiling is 8

Conclusion

In this article, we looked at how to use the floor() and ceiling() methods in the Java TreeSet class to find the floor or ceiling elements of a given element. We saw how to use both methods with or without a Comparator, and how to use them to find the range of elements in a set.
