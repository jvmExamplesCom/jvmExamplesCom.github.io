Find element index in Java TreeSet

Java TreeSet is a powerful data structure that can be used to store and retrieve data in an efficient manner. It provides many useful methods to access, modify, and manipulate the data stored in a TreeSet. One of the commonly used operations is finding the index of an element in a TreeSet. In this article, we will discuss how to find the index of an element in a Java TreeSet using various methods like 'subSet', ‘headSet’, ‘tailSet’ and ‘first’ and ‘last’.

Using subSet() to find Element Index

The ‘subSet’() method in Java TreeSet allows us to get the elements between two specified values. We can traverse the returned element set and compute the index of the desired element. 

          
           
TreeSet<Integer> treeSet = new TreeSet<>();
treeSet.add(10);
treeSet.add(20);
treeSet.add(30);
treeSet.add(40);
treeSet.add(50);

int index = 0;
int element = 40;

SortedSet<Integer> subSet = treeSet.subSet(treeSet.first(), element);

for(Integer i : subSet) {
    if(i == element) {
        break;
    }
    index++;
}

System.out.println("Index of element " + element + " is " + index);
// Output: Index of element 40 is 3
In the above example, we have used the subSet() method to get all the elements between the first element and the given element. Then we have used the ‘for-each’ loop to traverse the returned element set and find the index of the given element.
