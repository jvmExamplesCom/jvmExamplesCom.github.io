How to Check if TreeSet is Empty (isEmpty Method)

Java's TreeSet class has a useful method to determin eif a set is empty. But first, let's review the basics of TreeSet. TreeSet is a collection that stores elements in sorted order. In order to use TreeSet, you must specify the type of elements being stored. For example: 

          
           
TreeSet<String> myTreeSet = new TreeSet<String>();
The syntax above creates a TreeSet of type String. The elements stored in the set will be of type String and sorted in lexicographic order. Now, let's consider the isEmpty() method. The isEmpty() method returns true if the TreeSet is empty and false if it contains elements. The syntax looks like this: 

          
           
boolean result = myTreeSet.isEmpty();
The value stored in result will either be true or false depending on whether the TreeSet is empty. Now, let's look at a few code examples.

Example 1: An Empty TreeSet


          
           
TreeSet<String> myTreeSet = new TreeSet<String>();

boolean result = myTreeSet.isEmpty();

System.out.println(result); // prints true
In this example, we create an empty TreeSet. Then, we use the isEmpty() methods to check if the TreeSet is empty. As expected, the method returns true.

Example 2: A NonEmpty TreeSet


          
           
TreeSet<String> myTreeSet = new TreeSet<String>();

myTreeSet.add("Hello");

boolean result = myTreeSet.isEmpty();

System.out.println(result); // prints false
In this example, we create a TreeSet and add an element to it. Then, we use the isEmpty() method to check if the TreeSet is empty. As expected, the method returns false.

Example 3: A NonEmpty TreeSet After Removing Elements


          
           
TreeSet<String> myTreeSet = new TreeSet<String>();

myTreeSet.add("Hello");
myTreeSet.remove("Hello");

boolean result = myTreeSet.isEmpty();

System.out.println(result); // prints true
In this example, we create a TreeSet and add an element to it. Then, we remove the element from the TreeSet. Then, we use the isEmpty() method to check if the TreeSet is empty. As expected, the method returns true. To summarize, the isEmpty() method is a useful way to check if a TreeSet is empty. It returns true if the set is empty and false if it contains elements.
