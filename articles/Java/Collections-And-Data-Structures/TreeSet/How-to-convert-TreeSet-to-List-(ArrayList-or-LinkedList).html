Convert TreeSet to List (ArrayList or LinkedList)

Java provides a powerful data structure which is known as TreeSet. It can store and manage large collections of data in a hierarchical manner. In this article, we will discuss how to convert a TreeSet to either an ArrayList or LinkedList.

Using Iterator

We can iterate over the elements of the TreeSet and add each element to either an ArrayList or LinkedList. Here is the code snippet for this approach: 

          
           
TreeSet<String> treeSet = new TreeSet<>();
treeSet.add("apple");
treeSet.add("banana");

// Create a new List
List<String> list = new ArrayList<>();
 
// Iterate over treeSet
Iterator<String> itr = treeSet.iterator();
while(itr.hasNext()) {
    // Add the value from iterator to list
    list.add(itr.next());
}

Using Streams

Java 8 added the Streams API which allows us to use lambda expressions to process collections of data. We can use the Streams API to convert a TreeSet to either an ArrayList or LinkedList. Here is the code snippet for this approach: 

          
           
TreeSet<String> treeSet = new TreeSet<>();
treeSet.add("apple");
treeSet.add("banana");

// Convert the TreeSet to a Stream
Stream<String> stream = treeSet.stream();

// Create a new List
List<String> list = new ArrayList<>();

// Use the stream to populate the list
stream.forEach(list::add);

Using toArray() Method

The toArray() method can be used to retrieve all elements in a TreeSet as an array. We can then convert the array to either an ArrayList or LinkedList. Here is the code snippet for this approach: 

          
           
TreeSet<String> treeSet = new TreeSet<>();
treeSet.add("apple");
treeSet.add("banana");

// Convert the TreeSet to an array
String[] array = treeSet.toArray(new String[treeSet.size()]);

// Create a new List
List<String> list = new ArrayList<>();

// Use the array to populate the list
for(String str : array) {
    list.add(str);
}

Conclusion

In this article, we have discussed three approaches to convert a TreeSet to either an ArrayList or LinkedList. Each approach has its own advantages and tradeoffs so choose whichever one best suits your needs.
