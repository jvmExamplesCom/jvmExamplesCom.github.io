How to Reverse Order of TreeSet Elements (Iterate TreeSet in Reverse Order)

Java provides the powerful TreeSet data structure which is an implementation of the SortedSet interface that keeps its elements ordered using their natural ordering, or by an explicitly provided Comparator. However, many developers are faced with the difficulty of iterating through a TreeSet in reverse order. In this tutorial, we will discuss various approaches for reversing the order of TreeSet elements.

TreeSet Reverse Iteration using DescendingSet() Method

The simplest way to iterate a TreeSet in reverse order is to use the TreeSet’s descendingSet() method. This method returns a reverse order view of the given TreeSet. We can then retrieve the elements from the reverse view using the iterator() method. Consider this example: 

          
           
// Create a TreeSet of Integer
TreeSet<Integer> set = new TreeSet<>();

// Add items to TreeSet
set.add(1);
set.add(2);
set.add(3);
set.add(4);
set.add(5);

// Retrieve in reverse order
for (Integer i : set.descendingSet()) {
    System.out.print(i + " ");
}
The output of this code will be: 

          
           
5 4 3 2 1

TreeSet Reverse Iteration Using Comparator

We can also reverse the order of TreeSet elements using a Comparator. The Comparator class defines the compare(Object o1, Object o2) method which compares two objects and returns an int value. We can define a custom Comparator that takes the natural ordering of Comparable objects and reverses it. 

          
           
// Create a TreeSet of Integer
TreeSet<Integer> set = new TreeSet<>();

// Add items to TreeSet
set.add(1);
set.add(2);
set.add(3);
set.add(4);
set.add(5);

// Define a custom Comparator that reverses the order of Comparable objects
Comparator<Integer> comparator = new Comparator<Integer>() {
    @Override
    public int compare(Integer o1, Integer o2) {
        return o2.compareTo(o1);
    }
};

// Retrieve in reverse order
TreeSet<Integer> reverseSet = new TreeSet<>(comparator);
reverseSet.addAll(set);
for (Integer i : reverseSet) {
    System.out.print(i + " ");
}
The output of this code will be: 

          
           
5 4 3 2 1

TreeSet Reverse Iteration Using Comparator and Iterator

It is also possible to iterate a TreeSet object in reverse order using a Comparator and an Iterator. We can create a Comparator that reverses the order of Comparable objects and pass it to the TreeSet constructor. Then we can retrieve an iterator pointing towards the last element in the set, and traverse it in reverse order. 

          
           
// Create a TreeSet of Integer
TreeSet<Integer> set = new TreeSet<>();

// Add items to TreeSet
set.add(1);
set.add(2);
set.add(3);
set.add(4);
set.add(5);

// Create a Comparator that reverses the order of Comparable objects
Comparator<Integer> comparator = new Comparator<Integer>() {
    @Override
    public int compare(Integer o1, Integer o2) {
        return o2.compareTo(o1);
    }
};

// Retrieve an iterator pointed to the last element
Iterator<Integer> itr = set.iterator(set.last());

// Traverse the TreeSet in reverse order
while (itr.hasPrevious()) {
    System.out.print(itr.previous() + " ");
}
The output of this code will be: 

          
           
5 4 3 2 1

Conclusion

In this tutorial, we discussed various approaches for iterating a TreeSet in reverse order. We discussed using the descendingSet() method, creating a custom Comparator, and using a Comparator and the Iterator method. All these approaches allow us to traverse the TreeSet in reverse order.
