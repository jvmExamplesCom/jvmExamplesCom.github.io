Java TreeSet Example - Get Lowest and Highest Value

In this article, we will explore what a Java TreeSet is, how it works, and how to use its methods to get the lowest and highest value stored in it.

What is a Java TreeSet?

A Java TreeSet is a type of collection that uses a tree structure to store elements. It is an ordered collection and provides fast lookups, adding, and removing of elements. It also allows for efficient sorting of elements. Unlike other collections, a Java TreeSet does not allow for duplicate elements. If a duplicate element is attempted to be added, the set will ignore it and retain the original element.

Example: Adding Elements to a TreeSet

The following code example shows the syntax for adding elements to a TreeSet. 

          
           
TreeSet myTreeSet = new TreeSet();
myTreeSet.add("Element 1");
myTreeSet.add("Element 2");
myTreeSet.add("Element 3");
This adds the three elements to the TreeSet, as long as they are not duplicates.

Example: Getting the Lowest and Highest Value

To get the lowest and highest value stored in a Java TreeSet, you can use the first() and last() methods. The following code example shows how to use the first() and last() methods to get the lowest and highest values stored in a TreeSet: 

          
           
String lowestValue = myTreeSet.first();
String highestValue = myTreeSet.last();
This will set the lowestValue and highestValue strings to the respective lowest and highest values that are stored in the TreeSet.

Conclusion
