How to Iterate over Elements of TreeSet in Java

Java's TreeSet class is an implementation of the Set interface, which is a collection that cannot contain duplicates. TreeSet elements are ordered using their natural ordering, or by a Comparator provided at set creation time. It is possible to iterate over the elements of a TreeSet in sorted order.

Using Iterators

The most basic way to iterate over the elements of a TreeSet is to use three Iterator classes associated with TreeSet. The iterator() method returns an Iterator object that can traverse the elements of the TreeSet in proper sequence. 

          
           
TreeSet<String> mySet = new TreeSet<String>();

mySet.add("Hello");
mySet.add("World");
mySet.add("!");

Iterator<String> iter = mySet.iterator();

while (iter.hasNext()) {
  System.out.println(iter.next());
}
Output: 

          
           
Hello
World
!
The descendingIterator() method returns an Iterator object that traverses the elements of the TreeSet in reverse order. 

          
           
TreeSet<String> mySet = new TreeSet<String>();

mySet.add("Hello");
mySet.add("World");
mySet.add("!");

Iterator<String> iter = mySet.descendingIterator();

while (iter.hasNext()) {
  System.out.println(iter.next());
}
Output: 

          
           
!
World
Hello
The spliterator() method returns a Spliterator object, which is an advanced Iterator that can be used to partition off portions of the TreeSet, allowing for parallel operations on multiple threads. 

          
           
TreeSet<String> mySet = new TreeSet<String>();

mySet.add("Hello");
mySet.add("World");
mySet.add("!");

Spliterator<String> split = mySet.spliterator();

split.forEachRemaining(System.out::println);
Output: 

          
           
Hello
World
!

Using For-each Loop

It is also possible to iterate over the elements of a TreeSet using the for-each loop. This is the most concise way of iterating over the elements of the TreeSet. 

          
           
TreeSet<String> mySet = new TreeSet<String>();

mySet.add("Hello");
mySet.add("World");
mySet.add("!");

for (String s : mySet)
  System.out.println(s);
Output: 

          
           
Hello
World
!

Looping Over Subsets

It is also possible to iterate over a subset of the elements of a TreeSet. This can be done using the subSet(), headSet(), and tailSet() methods to create subsets containing specific elements. 

          
           
TreeSet<Integer> mySet = new TreeSet<Integer>();

mySet.add(10);
mySet.add(20);
mySet.add(30);
mySet.add(40);

TreeSet<Integer> subSet = (TreeSet<Integer>)mySet.subSet(20, 40);

for (Integer i : subSet)
  System.out.println(i);
Output: 

          
           
20
30
In conclusion, it is possible to iterate over the elements of a TreeSet in various ways, depending on the desired output and complexity.
