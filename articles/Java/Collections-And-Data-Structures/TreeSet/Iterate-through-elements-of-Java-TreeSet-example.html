Iterating Through Elements of a Java TreeSet

A TreeSet is an implementation of the Set interface that uses a tree for storage. It stores its elements in an ordered fashion and provides useful methods for getting, adding, and removing elements. This article will demonstrate how to iterate through the elements of a TreeSet using Java.

Creating a TreeSet

The first step is to create a TreeSet. The following code creates a TreeSet with elements of type String.


          
           
TreeSet<String> strings = new TreeSet<>();
strings.add("Hello");
strings.add("World!");
strings.add("Java");

This TreeSet now contains the three Strings "Hello", "World!", and "Java".

Iterating Through a TreeSet

Once we have created a TreeSet, we can use it to iterate through its elements. There are several approaches to doing this.

Using "for each" Loop

A simple approach for iterating through a TreeSet is to use a "for each" loop. The following code uses a "for each" loop to print out each element in the TreeSet:


          
           
for (String s : strings) {
 System.out.println(s);
}
This code prints out the following: 

          
           
Hello
World!
Java

Using Iterator

We can also use an Iterator to iterate through the TreeSet. An Iterator allows us to access the elements one by one. The following code uses an Iterator to print out each element in the TreeSet:


          
           
Iterator<String> iterator = strings.iterator();
while (iterator.hasNext()) {
 System.out.println(iterator.next());
}

Like the previous example, this code will print out the following:


          
           
Hello
World!
Java

Using forLoop()

The forLoop() method of TreeSet is another way to iterate through its elements. This method takes a lambda expression as an argument, which allows us to perform an action on each element. The following code uses forLoop() to print out each element in the TreeSet:


          
           
strings.forLoop(s -> System.out.println(s));
Again, this code prints out the following: 

          
           
Hello
World!
Java

Each of these approaches can be used to iterate through the elements of a TreeSet.

Conclusion

In this article, we demonstrated how to iterate through the elements of a TreeSet using Java. We looked at three different approaches for doing so: using a "for each" loop, using an Iterator, and using the forLoop() method. Using any of these approaches, we can easily access the elements of a TreeSet.
