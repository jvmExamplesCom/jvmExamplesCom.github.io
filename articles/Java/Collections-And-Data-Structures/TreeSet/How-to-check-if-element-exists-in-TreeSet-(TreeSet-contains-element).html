Checking if Element Exists in TreeSet

A TreeSet is a collection that stores elements in a sorted set. In this article, we will be discussing how to check if an element exists within the TreeSet.

Using The Contains Method

The simplest and most straightforward method of determining if an element exists within a TreeSet is by using the contains method. This method takes a single parameter and returns true or false depending on whether the element is found within the TreeSet. Here is an example of this method in action:


          
           
TreeSet<String> ts = new TreeSet<>();
ts.add("Apple");
ts.add("Banana");
ts.add("Cherry");

boolean exists = ts.contains("Banana");
// returns true

In this example, we first create a TreeSet and add three strings to it. We then call the contains method with a parameter of "Banana" to check if that element exists in the TreeSet. The method returns a boolean value of true, indicating that the element was found.

Iterating Over The Set

Another way of checking if an element exists within a TreeSet is by iterating over it. This can be done using a for-each loop. Here is an example of this approach:


          
           
TreeSet<String> ts = new TreeSet<>();
ts.add("Apple");
ts.add("Banana");
ts.add("Cherry");

boolean exists = false;
for (String element : ts) {
    if (element.equals("Banana")) {
        exists = true;
        break;
    }
}
//returns true

In this example, we first create a TreeSet and add three strings to it. We then iterate over the TreeSet using a for-each loop, comparing each element with the string "Banana". If a match is found, the boolean variable is set to true, and the loop is broken out of. This approach can be more efficient than the contains method when dealing with large sets.

Using An Iterator

Finally, we can also use an Iterator to check if an element exists within a TreeSet. Here is an example of this approach:


          
           
TreeSet<String> ts = new TreeSet<>();
ts.add("Apple");
ts.add("Banana");
ts.add("Cherry");

boolean exists = false;
Iterator<String> iter = ts.iterator();
while (iter.hasNext()) {
    String element = iter.next();
    if (element.equals("Banana")) {
        exists = true;
        break;
    }
}
// returns true

In this example, we first create a TreeSet and add three strings to it. We then create an Iterator and use it to iterate over the TreeSet. If a match for the string "Banana" is found, the boolean variable is set to true and the iterator is broken out of. This approach can also be more efficient than the contains method as it only needs to traverse the TreeSet once.

In conclusion, there are three different ways of checking if an element exists within a TreeSet. The simplest and most straightforward approach is the contains method. Alternatively, we can iterate over the TreeSet using either a for-each loop or an Iterator.
