How to Get the First and Last Elements of the TreeSet in Java

To get the first and last elements of the TreeSet in Java, we can use the following methods: 
  • first()

  • last()
    – This method returns the last element in the TreeSet.
For example, suppose we have a TreeSet of integers: 

          
           
TreeSet<Integer> set = new TreeSet<>();

int firstElement = set.first(); // 1
int lastElement = set.last(); // 3
