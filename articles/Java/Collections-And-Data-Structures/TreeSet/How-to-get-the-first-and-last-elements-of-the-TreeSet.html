TreeSet and Java

Java is a powerful and versatile programming language used to create anything from web applications to desktop software. How to Get the First and Last Elements of the TreeSet in Java

To get the first and last elements of the TreeSet in Java, we can use the following methods: 
  • first()
    – This method returns the first element in the TreeSet.
  • last()
    – This method returns the last element in the TreeSet.
For example, suppose we have a TreeSet of integers: 

          
           
TreeSet<Integer> set = new TreeSet<>();
set.add(1);
set.add(2);
set.add(3);
We can access the first and last elements of the TreeSet as follows: 

          
           
int firstElement = set.first(); // 1
int lastElement = set.last(); // 3
We can also retrieve the first and last elements using a 

          
           
for
loop: 

          
           
for(Integer element : set){
    if(element == set.first()) {
        int firstElement = element; // 1
    } else if (element == set.last()) {
        int lastElement = element; // 3
    }
}
Using the methods and techniques outlined above, we can easily retrieve the first and last elements of the TreeSet in Java.
