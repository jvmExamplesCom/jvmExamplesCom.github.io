Java TreeSet Example

The TreeSet class in Java is an implementation of a Set interface that stores elements in a tree. It provides an efficient way to store and retrieve elements. It also keeps the elements sorted according to their natural ordering, or according to a given Comparator. To illustrate how this works, we'll walk through a simple example that prints out a set of integers stored in a TreeSet.

Example Code

In this example, we'll create a TreeSet and add several integer values to it. Then, we'll print out the values in the TreeSet:


          
           
import java.util.TreeSet;

public class TreeSetExample {
  public static void main(String[] args) {
    TreeSet<Integer> ts = new TreeSet<>();
    ts.add(3);
    ts.add(2);
    ts.add(1);
    ts.add(4);

    System.out.println(ts);
  }
}

When we run this code, we get the following output:


          
           
[1, 2, 3, 4]

As you can see, the values were added to the TreeSet in ascending order. This is because the TreeSet uses a binary search tree to store its elements, and the elements are sorted according to their natural ordering. We can also use a Comparator to sort the elements in a different way.

Using a Comparator

If we want to sort the elements in a different way, we can use a Comparator. For example, if we want to sort the elements in descending order, we can do the following:


          
           
import java.util.Comparator;
import java.util.TreeSet;

public class TreeSetExample {
  public static void main(String[] args) {
    TreeSet<Integer> ts = new TreeSet<>(Comparator.reverseOrder());
    ts.add(3);
    ts.add(2);
    ts.add(1);
    ts.add(4);

    System.out.println(ts);
  }
}

Now, when we run this code, we get the following output:


          
           
[4, 3, 2, 1]

As you can see, the elements were printed in descending order. This is because we used the reverseOrder() comparator to sort the elements in the TreeSet.
