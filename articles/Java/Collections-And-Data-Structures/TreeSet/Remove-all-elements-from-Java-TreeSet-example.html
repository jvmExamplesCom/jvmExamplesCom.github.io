Removing All Elements From a Java TreeSet

Java's TreeSet is an implementation of Set which is a collection that cannot contain duplicate elements and is backed by a red-black tree. The TreeSet class is part of the Java Collections Framework and provides various methods for adding, removing and querying elements.

Using the clear() Method

The easiest way to remove all elements from a TreeSet is using the 

clear()
method. This method removes all of the elements from the set, leaving it empty. 

          
           
TreeSet<String> ts = new TreeSet<String>();
ts.add("A");
ts.add("B");
ts.add("C");

ts.clear();

System.out.println(ts.size());
// Outputs 0

Using the removeAll() Method

The 
removeAll()
method can also be used to remove all elements from a TreeSet. This method takes a Collection as a parameter and removes all elements that are contained in the Collection. This can be useful when removing a subset of elements from a TreeSet. 

          
           
TreeSet<String> ts = new TreeSet<String>();
ts.add("A");
ts.add("B");
ts.add("C");

Collection<String> c = new ArrayList<String>();
c.add("A");
c.add("B");

ts.removeAll(c);

System.out.println(ts.size());
// Outputs 1

Using the Iterator.remove() Method

The 
Iterator.remove()
method can also be used to remove all elements from a TreeSet. This method takes an Iterator as a parameter and removes the element that the Iterator is pointing to. Since an Iterator iterates over all elements in a set, this method can be used to loop over all elements and remove them one by one. 

          
           
TreeSet<String> ts = new TreeSet<String>();
ts.add("A");
ts.add("B");
ts.add("C");

Iterator<String> it = ts.iterator();
while (it.hasNext()) {
    it.next();
    it.remove();
}

System.out.println(ts.size());
// Outputs 0
