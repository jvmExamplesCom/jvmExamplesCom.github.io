How to Print TreeSet Elements in Java

TreeSet is a collection class from the Java Collection Framework. It is an implementation of the Set interface which stores elements in a tree structure. This allows for fast and efficient retrieval of elements. In this article, we'll explain how to print TreeSet elements in Java.

Creating a TreeSet Object

To use a TreeSet in Java, we need to create a TreeSet object. This object is used to store and retrieve objects which extend the Comparable interface or implement the Comparator interface. We can create a TreeSet by passing a Comparator object as a parameter in the constructor, or by just calling the constructor with no parameters. In the following example, we are creating a TreeSet object which stores strings. 

          
           
TreeSet<String> treeSet = new TreeSet<>();

Adding Elements to a TreeSet

Now that we have created our TreeSet, we can add elements to it using the add() method. Here, we will add some string elements to our TreeSet. 

          
           
treeSet.add("Java");
treeSet.add("Python");
treeSet.add("C++");
treeSet.add("C#");
treeSet.add("Ruby");

Printing TreeSet Elements

Now that we have added elements to our TreeSet, we can print them out. There are two main ways to do this: using the for-each loop or the iterator.

Using For-Each Loop

Using the for-each loop is the simpler way of printing TreeSet elements. In this approach, we first use the for-each loop to iterate through each element in the TreeSet and print it out. 

          
           
for(String element : treeSet){
    System.out.println(element);
}

Using Iterator

The other way to print TreeSet elements is by using an iterator. The iterator can be obtained by calling the iterator() method on the TreeSet object. We then use the next() method to iterate through the elements in the TreeSet and print them out. 

          
           
Iterator iter = treeSet.iterator();
while(iter.hasNext()){
    String element = (String)iter.next();
    System.out.println(element);
}
