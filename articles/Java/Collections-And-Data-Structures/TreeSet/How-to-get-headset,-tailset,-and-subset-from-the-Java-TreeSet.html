Getting Subset, Headset and Tailset from Java TreeSet

A TreeSet is an ordered collection of elements in which duplicate values cannot be stored. It provides an efficient means to store and retrieve data. The TreeSet also provides various methods to determine the subset, headsets and tailsets of the set. In this article, we will take a closer look at these methods and demonstrate their uses with code examples.

Subset

The 
subSet()
method is used to retrieve the subSet of the given set. This method returns a view of the portion of the set which lies between two given element in the sorted order. The first argument of the 
subSet()
method should be the beginning element and the other argument should be the ending element of the subset range. 

          
           
TreeSet<String> set = new TreeSet<>();

set.add("Java");
set.add("C");
set.add("C++");
set.add("Python");
 
SortedSet<String> subset = set.subSet("C", "Python");

System.out.println(subset);

// Output: [C, C++]

In the example above, we have created a TreeSet of String elements and then retrived the subset of the set which lies between the elements 'C' and 'Python'. As can be seen, the output contains all the elements between 'C' and 'Python' including both of them.

HeadSet

The 
headSet()
method is used to retrieve the headSet of the given set. This method returns a view of the portion of the set which contains all the elements which are less than the specified element in the sorted order. The argument of the 
headSet()
method should be the element till which the headSet has to be generated. 

          
           
TreeSet<String> set = new TreeSet<>();

set.add("Java");
set.add("C");
set.add("C++");
set.add("Python");
 
SortedSet<String> headSet = set.headSet("C++");

System.out.println(headSet);

// Output: [C]

In the example above, we have created a TreeSet of String elements and then retrived the headSet of the set which contains all the elements which are lesser than 'C++'. As can be seen, the output contains only one element which is 'C' as it is the only element which is lesser than 'C++'.

TailSet

The 
tailSet()
method is used to retrieve the tailSet of the given set. This method returns a view of the portion of the set which contains all the elements which are greater than the specified element in the sorted order. The argument of the 
tailSet()
method should be the element from which the tailSet has to be generated. 

          
           
TreeSet<String> set = new TreeSet<>();

set.add("Java");
set.add("C");
set.add("C++");
set.add("Python");
 
SortedSet<String> tailSet = set.tailSet("C++");

System.out.println(tailSet);

// Output: [C++, Java, Python]

In the example above, we have created a TreeSet of String elements and then retrived the tailSet of the set which contains all the elements which are greater than 'C++'. As can be seen, the output contains all the elements which are greater than 'C++' including itself.
