Create List containing n Copies of Specified Object Example

Overview

Writing a program in Java to create a list containing a certain number of copies of a specified object example requires familiarity with the basic facilities of the language. This guide will demonstrate an example of how to implement the program and show the output of the various stages.

Implementation

Firstly, we will need to import some packages: 

          
           
import java.util.*;
import java.util.Collections;
import java.util.List;
We then create our class and a "List" to store our data: 

          
           
public class CreateListCopies {
	public static void main(String args[]) {

		//Create a List
		List list = new ArrayList();
Next, we define a method that creates a list containing a certain number of copies of the specified object: 

          
           
//Create a method to create a list containing n copies of specified object
		public static List createListCopies(Object obj, int n) {
			
			//Create a list to store the elements
			List copyList = new ArrayList();
			
			//Loop through and add the objects
            for (int i = 0; i < n; i++) {
               copyList.add(obj);
            }
            
            //return list
            return copyList;
        }
Now we can call the method: 

          
           
//Call the method to create the list
        List copyList = createListCopies("Hello World!", 5);
Finally, we print the results of the list: 

          
           
//Print contents of the list
        System.out.println("Contents of the List : " + copyList);
    }
    
}
