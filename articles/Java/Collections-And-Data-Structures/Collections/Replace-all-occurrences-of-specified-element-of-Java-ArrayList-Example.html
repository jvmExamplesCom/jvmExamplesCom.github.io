Replace All Occurrences of Specified Element of Java ArrayList Example

Java ArrayList is an ordered list of objects in which duplicate values can be stored. It provides access and manipulation of a list of objects using various methods.

Example

In the following example, we will replace all occurrences of the element 'apple' with 'orange' in the given ArrayList. 

          
           
import java.util.ArrayList;

public class ReplaceElements {
	public static void main(String[] args) {
		//Initialize an ArrayList
		ArrayList<string> fruits = new ArrayList<string>();

		//Add elements to ArrayList
		fruits.add("apple");
		fruits.add("banana");
		fruits.add("guava");
		fruits.add("apple");
		System.out.println("Original ArrayList: "+fruits);

		//Replace 'apple' with 'orange'
		for(int i=0; i < fruits.size(); i++) {
            if(fruits.get(i).equals("apple")) {
                fruits.set(i, "orange");
            }
        }

		System.out.println("After Replacement: "+fruits);
	}
}
</string></string>
Output: 

          
           
Original ArrayList: [apple, banana, guava, apple]
After Replacement: [orange, banana, guava, orange]
In the above example, we first initialized an instance of 

          
           
ArrayList
, then added four fruit names to it. We used 

          
           
for
loop to iterate through the list and used 

          
           
equals()
method to check if the element is 'apple'. If the element is 'apple', 

          
           
set()
method is used to replace it with 'orange'. Lastly, we print the original list and after replacement list on console.
