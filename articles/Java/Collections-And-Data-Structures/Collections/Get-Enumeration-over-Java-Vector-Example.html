Enumeration over Java Vector Example

Java provides the 

          
           
Vector
class which is one of the ways to store a collection of data. The 

          
           
Vector
class provides an enumeration interface that allows you to iterate a collection of data. In this tutorial, we will be looking at how to use the 

          
           
Enumeration
interface to access and iterate through a 

          
           
Vector
.

Creating a Vector

The first step is to create a 

          
           
Vector
with some data. To create a vector, you can use either the default constructor, 

          
           
Vector()
, or the overloaded constructor, 

          
           
Vector(Collection c)
. To demonstrate, let's create a 

          
           
Vector
with some strings: 

          
           
Vector<String> myVector = new Vector<String>();
myVector.add("Apple");
myVector.add("Banana");
myVector.add("Orange");
This 

          
           
Vector
now contains three strings: "Apple", "Banana", and "Orange".

Accessing Elements in the Vector using Enumeration

Once you have created a 

          
           
Vector
, you can access its elements by using the methods provided by the 

          
           
Enumeration
interface. The 

          
           
Enumeration
interface provides methods that allow you to iterate through the elements in a 

          
           
Vector
. To create an 

          
           
Enumeration
, you can use the 

          
           
elements()
method provided by the 

          
           
Vector
class: 

          
           
Enumeration<String> e = myVector.elements();
This will create an 

          
           
Enumeration
object with all of the elements in the 

          
           
Vector
. The 

          
           
Enumeration
interface provides two methods for accessing the elements of a 

          
           
Vector
: 

          
           
hasMoreElements()
and 

          
           
nextElement()
. The 

          
           
hasMoreElements()
method returns 

          
           
true
if there are more elements in the 

          
           
Vector
, and 

          
           
false
if there are no more elements. The 

          
           
nextElement()
method returns the next element in the 

          
           
Vector
. To demonstrate, let's iterate through our 

          
           
Vector
and print out each element: 

          
           
while(e.hasMoreElements()) {
    String element = e.nextElement();
    System.out.println(element);
}
This will print out each element in the 

          
           
Vector
: 

          
           
Apple
Banana
Orange
