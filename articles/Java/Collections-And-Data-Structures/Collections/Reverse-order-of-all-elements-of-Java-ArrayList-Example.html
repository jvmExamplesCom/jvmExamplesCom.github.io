Reverse Order of All Elements of Java ArrayList Example

In this tutorial, we will learn how to reverse the order of all elements of Java ArrayList with a few simple examples.

Example 1: Using Collection.reverse() Method

The Collection.reverse() method can be used to reverse the order of all elements in a given Java ArrayList.


          
           
import java.util.ArrayList;
import java.util.Collections;

public class ReverseArrayList {

   public static void main(String[] args) {

      //Create an ArrayList
      ArrayList<Integer> myList = new ArrayList<>();

      //Add elements to ArrayList
      myList.add(1);
      myList.add(3);
      myList.add(4);
      myList.add(2);
      myList.add(5);

      //Print original list
      System.out.println("Original ArrayList: "+myList);

      //Reverse list using Collections.reverse()
      Collections.reverse(myList);

      //Print reversed list
      System.out.println("Reversed ArrayList: "+myList);

   }
}
When you run the program, the output will be: 

          
           
Original ArrayList: [1, 3, 4, 2, 5]
Reversed ArrayList: [5, 2, 4, 3, 1]

Example 2: Using ListIterator.previous() Method

The ListIterator.previous() method can also be used to reverse the order of all elements in a given Java ArrayList.


          
           
import java.util.ArrayList;
import java.util.ListIterator;

public class ReverseArrayList {

   public static void main(String[] args) {

      //Create an ArrayList
      ArrayList<Integer> myList = new ArrayList<>();

      //Add elements to ArrayList
      myList.add(1);
      myList.add(3);
      myList.add(4);
      myList.add(2);
      myList.add(5);

      //Print original list
      System.out.println("Original ArrayList: "+myList);

      //Create a ListIterator
      ListIterator<Integer> iterator = myList.listIterator(myList.size());

      //Iterate over list in reverse order
      while (iterator.hasPrevious()) {
          Integer number = iterator.previous();
          System.out.print(number + " ");
      }

   }
}
When you run the program, the output will be: 

          
           
Original ArrayList: [1, 3, 4, 2, 5]
5 2 4 3 1

Example 3: Using For Loop

We can also use a simple for loop to reverse the order of all elements in a Java ArrayList.


          
           
import java.util.ArrayList;

public class ReverseArrayList {

   public static void main(String[] args) {

      //Create an ArrayList
      ArrayList<Integer> myList = new ArrayList<>();

      //Add elements to ArrayList
      myList.add(1);
      myList.add(3);
      myList.add(4);
      myList.add(2);
      myList.add(5);

      //Print original list
      System.out.println("Original ArrayList: "+myList);

      //Iterate through list in reverse order
      for (int i = myList.size()-1; i >= 0; i--) {
          System.out.print(myList.get(i) + " ");
      }

   }
}
When you run the program, the output will be: 

          
           
Original ArrayList: [1, 3, 4, 2, 5]
5 2 4 3 1
