Shuffling Elements of Java Vector Example

The class in Java provides the ability to access, modify and shuffle elements. In this example, we'll look at how to shuffle elements of a Vector using the built-in Java API.

Example Code


          
           
Vector<String> vector = new Vector<>();
vector.add("one");
vector.add("two");
vector.add("three");

Collections.shuffle(vector);

System.out.println(vector);
The output of the above code is: 

          
           
[two, one, three]

In this example, we created a Vector of Strings and added three elements ("one", "two" and "three"). Then we used the Collections.shuffle() method to randomly shuffle the elements in the vector. Lastly, we printed out the resulting vector.

Using Vector.setElementAt() to Shuffle Elements

We can also use the Vector.setElementAt() method to manually shuffle the elements of a Vector. Let's take a look at an example:


          
           
Vector<String> vector = new Vector<>();
vector.add("one");
vector.add("two");
vector.add("three");

Random random = new Random();
int index1 = random.nextInt(vector.size());
int index2 = random.nextInt(vector.size());

String temp = vector.elementAt(index1);
vector.setElementAt(vector.elementAt(index2),index1);
vector.setElementAt(temp,index2);

System.out.println(vector);
The output of the above code is: 

          
           
[one, two, three]

In this example, we first create a Vector and add three elements to it. Then we generate two random numbers between 0 and the size of the vector (in this case 3). We use these numbers as indices to swap the elements at those indices. Finally, we print out the resulting vector.

Conclusion

In this tutorial, we looked at two ways to shuffle elements of a Vector in Java. The first way was by using the Collections.shuffle() method and the second was by using the Vector.setElementAt() method. Both methods are fairly straightforward and can be used to randomly shuffle elements in a Vector.
