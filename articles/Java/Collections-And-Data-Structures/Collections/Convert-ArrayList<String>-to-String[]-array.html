Converting ArrayList to String[] Array

Java provides a variety of ways to convert an ArrayList of type String to a String array. There are multiple methods available for converting an ArrayList of Strings to a String array, and the following sections describe each of those methods.

Using the T0Array() Method

The 
toArray()
method can be used to convert an ArrayList of Strings to a String array. This method takes a generic array as an argument, and then creates a new array of the same type as the argument and copies all the elements from the ArrayList into the new array. 

          
           
// Create an ArrayList of Strings
ArrayList<String> list = new ArrayList<>(); 
list.add("One"); 
list.add("Two"); 
list.add("Three"); 

// Convert the ArrayList to a String[] array 
String[] array = list.toArray(new String[0]); 

// Print the array 
for (String s : array) { 
    System.out.println(s); 
}
The output of the above code is: 

          
           
One 
Two 
Three

Using Enhanced For Loop

The enhanced for loop can also be used to convert an ArrayList of Strings to a String array. In this method, a new String array is created with the size equal to the size of the ArrayList, and then each element of the ArrayList is copied into the new array using the enhanced for loop.


          
           
// Create an ArrayList of Strings
ArrayList<String> list = new ArrayList<>(); 
list.add("One"); 
list.add("Two"); 
list.add("Three"); 

// Create a new String[] array with the size of the ArrayList 
String[] array = new String[list.size()]; 

// Copy each element of the ArrayList to the String[] array 
int i = 0; 
for (String s : list) { 
    array[i] = s; 
    i++;
