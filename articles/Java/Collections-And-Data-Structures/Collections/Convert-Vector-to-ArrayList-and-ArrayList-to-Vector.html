Convert Vector to ArrayList

Converting a Vector to an ArrayList is not difficult and provides a great way to manipulate elements in a Vector. By converting a Vector to an ArrayList, you can sort, remove, or add elements to a Vector. This article will detail the process of how to convert a Vector to an ArrayList in Java.

Creating an ArrayList

In order to convert a Vector to an ArrayList, an ArrayList must be created. The ArrayList class can be imported from the Java Collections framework: 

          
           
import java.util.ArrayList;
Once it has been imported, the ArrayList can be created using the following code: 

          
           
ArrayList<String> list = new ArrayList<>();

Iterating Through a Vector

To add elements from a Vector to the newly created ArrayList, we need to loop through the Vector and add each element to the ArrayList. To do this, we'll use the enhanced for loop: 

          
           
for (String s : vector) {
    list.add(s);
}

Converting a Vector to an ArrayList

Now that the elements have been added, the Vector and the ArrayList contain the same elements. In order to convert the Vector to an ArrayList, the Vector must be converted to an array using the toArray() method: 

          
           
String[] array = vector.toArray(new String[vector.size()]);
Once the Vector has been converted to an array, it can now be used to create an ArrayList. To do this, we can use the ArrayList constructor that takes an array as a parameter: 

          
           
ArrayList<String> list = new ArrayList<>(Arrays.asList(array));
