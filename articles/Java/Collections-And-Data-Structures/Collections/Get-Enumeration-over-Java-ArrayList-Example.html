Enumeration over Java ArrayList

Enumeration is a tool in Java that allows the user to traverse through a Collection, such as an ArrayList. This tutorial will provide a brief overview of enumeration, along with examples of how to use it to traverse an ArrayList.

Overview of Enumeration in Java

Enumeration is an interface in the Java Collection Framework. It provides methods for iterating through the elements of a collection, one at a time. It defines two methods: 

hasMoreElements()
and 
nextElement()
. The former method returns a Boolean value indicating whether there are more elements to be processed in the collection. The latter returns an Object from the collection. Here is a basic example of an enumeration in action: 

          
           
Enumeration enumeration = collection.elements();
while (enumeration.hasMoreElements()) {
    Object element = enumeration.nextElement();
    //process element
}

Let's look at an example of how to use enumeration to iterate over an ArrayList.

Enumeration over an ArrayList

To illustrate how to use Enumeration to iterate over an ArrayList, let's examine the following example:


          
           
ArrayList<string> list = new ArrayList<string>();
list.add("Java");
list.add("is");
list.add("cool");

Enumeration enumeration = list.elements();
while (enumeration.hasMoreElements()) {
    String element = (String) enumeration.nextElement();
    System.out.println(element);
}

elements()

nextElement()
