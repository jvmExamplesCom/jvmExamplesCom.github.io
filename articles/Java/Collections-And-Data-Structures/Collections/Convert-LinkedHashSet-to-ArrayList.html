Convert LinkedHashSet to ArrayList in Java

In this article, we'll learn about the process of converting a LinkedHashSet to an ArrayList in Java. This is a useful technique for many applications, such as when you need an easy way to iterate through a set of elements without worrying about order.

Example 1: Using LinkedHashSet and ArrayList

The following example shows how to create a LinkedHashSet and then convert it to an ArrayList using the static method "toArrayList()". 

           
            
// Create LinkedHashSet
LinkedHashSet<String> linkedHashSet = new LinkedHashSet<String>();
linkedHashSet.add("one");
linkedHashSet.add("two");
linkedHashSet.add("three");

// Convert to ArrayList
ArrayList<String> arrayList = LinkedHashSet.toArrayList(linkedHashSet);
The array list contains the elements in the same order as what was added to the linked hash set.

Example 2: Using Iterator

We can also use an iterator to iterate through the elements of a LinkedHashSet and add them to an ArrayList. The following example shows how to do this. 

           
            
// Create LinkedHashSet
LinkedHashSet<String> linkedHashSet = new LinkedHashSet<String>();
linkedHashSet.add("one");
linkedHashSet.add("two");
linkedHashSet.add("three");

// Iterator
Iterator<String> iterator = linkedHashSet.iterator();

// Create ArrayList
ArrayList<String> arrayList = new ArrayList<String>();

// Add elements to ArrayList 
while (iterator.hasNext()) { 
    arrayList.add(iterator.next()); 
}
This approach ensures that the elements in the ArrayList are added in the same order as they were added to the LinkedHashSet.

Example 3: Using Streams

We can also use streams to convert a LinkedHashSet to an ArrayList. The following example shows how to do this. 

           
            
// Create LinkedHashSet
LinkedHashSet<String> linkedHashSet = new LinkedHashSet<String>();
linkedHashSet.add("one");
linkedHashSet.add("two");
linkedHashSet.add("three");

// Stream
Stream<String> stream = linkedHashSet.stream();

// Create ArrayList
ArrayList<String> arrayList = new ArrayList<String>();

// Add elements to ArrayList 
stream.forEach(arrayList::add);
This approach ensures that the elements in the ArrayList are added in the same order as they were added to the LinkedHashSet.

Example 4: Using Collections.list()

The java.util.Collections class provides a static method called “list” which can be used to convert a LinkedHashSet to an ArrayList. The following example shows how to do this. 

           
            
// Create LinkedHashSet
LinkedHashSet<String> linkedHashSet = new LinkedHashSet<String>();

linkedHashSet.add("three");

// Create ArrayList
