Replacing elements With Java Vector Example

In this article, we will look into replacing all occurrences of a specified element in a Java Vector with a new element. We will discuss a simple example program and learn how to write the code for it.

What is a Java Vector?

A Java Vector is an implementation of the List interface. It provides dynamic arrays that can grow or shrink in size when new items are added to it or old items are removed from it. It provides many useful methods for manipulating its elements.

Example Program

We will now look at a code example which demonstrates how to replace all occurrences of an element in a Java Vector with a new element. In this example, we will create a Vector of String objects and then use the 

replaceAll()
method to replace the String "old" with "new". 

          
           
import java.util.Vector;

public class VectorExample {
    public static void main(String[] args) {

        // Create a Vector of Strings
        Vector<String> v = new Vector<String>();
        v.add("old");
        v.add("old");
        v.add("new");
        v.add("old");

        // Replace all occurrences of "old" with "new"
        v.replaceAll("old", "new");

        // Print out the contents of the Vector
        System.out.println(v);
    }
}

In the code example above, we first created a Vector of String objects and added four elements to it. Then, we used the 

replaceAll()
method to replace all occurrences of "old" with "new". Finally, we printed out the contents of the Vector using the 
println()
method.

Output

When we compile and execute the code example above, we get the following output:


          
           
[new, new, new, new]
