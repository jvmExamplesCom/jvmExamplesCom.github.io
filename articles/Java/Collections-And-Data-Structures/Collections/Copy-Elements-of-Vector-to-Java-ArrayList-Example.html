Copy Elements of Vector to Java ArrayList Example

Java provides several classes like Vector and ArrayList which provide collections of objects. Sometimes we need to copy elements from one class to another. In this example, we will learn how to copy elements of java.util.Vector to another java.util.ArrayList.

Example: Copying Vector to ArrayList

To copy elements of Vector to ArrayList, first create an ArrayList and vector using the given code: 

          
           
ArrayList<String> arraylist = new ArrayList<String>();
Vector<String> vector = new Vector<String>();
Then add some elements in Vector and ArrayList using the add() method: 

          
           
arraylist.add("A");
arraylist.add("B");
arraylist.add("C");

vector.add("D");
vector.add("E");
vector.add("F");
Now, use the addAll() method to copy all elements of vector to arraylist as given below: 

          
           
arraylist.addAll(vector);
Finally, print both the collection using the forEach loop: 

          
           
arraylist.forEach(e -> System.out.println(e));
vector.forEach(e -> System.out.println(e));

Output

The output of the above code is: 

          
           
A
B
C
D
E
F
