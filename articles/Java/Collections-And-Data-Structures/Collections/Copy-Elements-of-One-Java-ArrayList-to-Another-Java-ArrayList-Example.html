Copy Elements of One Java ArrayList to Another Java ArrayList Example

The purpose of this article is to demonstrate how to copy elements of one Java ArrayList to another. The techniques used incorporate the ArrayList.addAll() and Collections.copy() methods, as well as a for loop which manually copies the elements from one list to another.

Using ArrayList.addAll()

The ArrayList.addAll() method allows one to add all the elements of one list to another, as demonstrated in the following example: 

          
           
import java.util.ArrayList; 

public class Main { 
   public static void main(String args[]) { 
      // Initializing both the lists 
      ArrayList<string> list1 = new ArrayList<string>(); 
      list1.add("A"); 
      list1.add("B"); 
      ArrayList<string> list2 = new ArrayList<string>(); 
      list2.add("C"); 
      list2.add("D");
  
      // Using addAll() to append all elements 
      list2.addAll(list1); 
  
      // Checking the contents of list2 
      System.out.println("Contents of list2:"); 
      for (String s : list2) 
         System.out.println(s); 
   } 
}

Using Collections.copy()

Another way to copy the elements of one list to another is using the Collections.copy() method, as shown below: 

          
           
import java.util.ArrayList; 
import java.util.Collections; 

public class Main { 
   public static void main(String args[]) { 
      // Initializing both the lists 
      ArrayList<string> list1 = new ArrayList<string>(); 
      list1.add("A"); 
      list1.add("B"); 
      ArrayList<string> list2 = new ArrayList<string>(); 
      list2.add("C"); 
      list2.add("D"); 
  
      // using copy() to copy elements from one list to another 
      Collections.copy(list2, list1); 
  
      // Checking the contents of list2 
      System.out.println("Contents of list2:"); 
      for (String s : list2) 
         System.out.println(s); 
   } 
}

Using For Loop

Lastly, one can also create a loop which manually copies elements from one list to another, as seen here: 

          
           
import java.util.ArrayList; 

public class Main { 
   public static void main(String args[]) { 
      // Initializing both the lists 
      ArrayList<string> list1 = new ArrayList<string>(); 
      list1.add("A"); 
      list1.add("B"); 
      ArrayList<string> list2 = new ArrayList<string>(); 
      list2.add("C"); 
      list2.add("D"); 
  
      // Appending all the elements of list1 to list2 
      for (String s : list1) 
         list2.add(s); 
  
      // Checking the contents of list2 
      System.out.println("Contents of list2:"); 
      for (String s : list2) 
         System.out.println(s); 
   } 
}
