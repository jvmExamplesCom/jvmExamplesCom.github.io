Introduction to ArrayList and HashSet in Java

Java provides two powerful types of collections: ArrayList and HashSet. An ArrayList is a dynamic array that allows user to store elements of any type. It can also store duplicate elements. On the other hand, HashSet is a set which does not allow duplicates and provides fast lookup for existing elements. In this article we will explore the differences between these two collections and learn how to convert from an ArrayList to a HashSet.

ArrayList and HashSet

An ArrayList is a dynamically resizable array-based data structure. It implements the List interface and allows users to store null values and duplicate elements. 

          
           
ArrayList<Integer> numbers = new ArrayList<>();

// inserting values
numbers.add(2);
numbers.add(3);
numbers.add(3); // duplication allowed
numbers.add(null); // null value allowed
The elements in an ArrayList are not stored in any particular order. So if we want to search the list for a particular element, we have to iterate through the entire list. A HashSet on the other hand, is a set which does not allow duplicates and provides faster lookups than an ArrayList. 

          
           
HashSet<Integer> numbers = new HashSet<>();

// inserting values
numbers.add(2);
numbers.add(3);
numbers.add(3); // duplication not allowed
numbers.add(null); // null value not allowed
Since a HashSet does not allow duplicates, it does not maintain the insertion order.

Converting ArrayList to HashSet

We can convert an ArrayList to a HashSet by using the addAll() method of the HashSet class. This method takes a Collection as an argument, so we can pass in our ArrayList and the elements will be added to the HashSet. 

          
           
ArrayList<Integer> numbers = new ArrayList<>();

// adding values to array list
numbers.add(2);
numbers.add(3);
numbers.add(3);

// converting ArrayList to HashSet
HashSet<Integer> set = new HashSet<>();
set.addAll(numbers);

// printing the HashSet
System.out.println(set);
Output: 

          
           
{2, 3}
As you can see, the duplicate element (3) has been eliminated from the HashSet.

Sorting the HashSet

If we want to sort the elements in a HashSet, we can first convert it to a TreeSet. A TreeSet is a sorted version of a HashSet which implements the SortedSet interface. We can use the constructor of the TreeSet class to initialize it with the HashSet, which will copy the elements of the HashSet into the TreeSet. 

          
           
HashSet<Integer> set = new HashSet<>();

// adding values to hash set
set.add(3);
set.add(2);
set.add(1);

// converting to tree set
TreeSet<Integer> sortedSet = new TreeSet<>(set);

// printing the TreeSet
System.out.println(sortedSet);
Output: 

          
           
{1, 2, 3}
As you can see, the elements have been sorted in ascending order.

Conclusion

In this article we have learnt about the differences between ArrayList and HashSet and how to convert from one to another. We have also seen how to sort a HashSet by converting it to a TreeSet. With this knowledge, we can now use these collections more effectively in our code.
