Find Minimum Element of Java Vector Example

Java has a powerful and versatile collection framework, consisting of several different types of collections. One such collection is the Vector, which is an ordered sequence of objects. In this article we will look at how to find the minimum element in a Java Vector using various examples.

Example 1: Using Streams and Lambda Expressions

The following example uses Java 8 Streams and lambda expressions to find the minimum element in a given Vector. 

          
           
Vector<Integer> vec = new Vector<>(Arrays.asList(11, 4, 7, 25, 5)); 
Integer min= vec.stream().min(Comparator.naturalOrder()).orElse(null);
System.out.println(min); //4
In this example, we first create a Vector with some sample integer values. Then, we use the Streams API to get a Stream from the Vector, and apply the min() method with a natural order Comparator. Finally, we print out the result, which is 4, the minimum element in the Vector.

Example 2: Using Enhanced for Loop

The following example uses an enhanced for loop to find the minimum element in a given Vector. 

          
           
Vector<Integer> vec = new Vector<>(Arrays.asList(11, 4, 7, 25, 5));
Integer min = Integer.MAX_VALUE; 
for (Integer i : vec) { 
    if (i < min) 
        min = i; 
} 
System.out.println(min); //4
In this example, we first create a Vector with some sample integer values. Then, we initialize a variable to the maximum possible integer value. Next, we iterate over the Vector and update the variable with the minimum value. Finally, we print out the result, which is 4, the minimum element in the Vector.

Example 3: Using Collections.min()

The following example uses the Collections.min() method to find the minimum element in a given Vector. 

          
           
Vector<Integer> vec = new Vector<>(Arrays.asList(11, 4, 7, 25, 5)); 
Integer min = Collections.min(vec); 
System.out.println(min); //4
In this example, we first create a Vector with some sample integer values. Then, we use the Collections.min() method to find the minimum element in the Vector. Finally, we print out the result, which is 4, the minimum element in the Vector.

Conclusion

In this article we looked at how to find the minimum element in a Java Vector using various examples. We saw how to use Streams and lambda expressions, an enhanced for loop, and the Collections.min() method to achieve the same result.
