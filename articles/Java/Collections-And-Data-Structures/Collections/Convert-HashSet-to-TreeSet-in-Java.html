Convert HashSet to TreeSet in Java

In Java, a HashSet is an unordered collection of elements that are stored in a hash table. This allows for fast retrieval and updates. On the other hand, a TreeSet is a collection of elements that are sorted according to their natural ordering. Converting a HashSet to a TreeSet is a useful task when you need to sort elements that are represented as a HashSet.

Example 1 - Basic Conversion

The following example shows how to convert a HashSet to a TreeSet. It first creates a HashSet of strings and then converts it to a TreeSet. The output of the code is shown below.


          
           
HashSet<String> hashSet = new HashSet<>();
hashSet.add("Apple");
hashSet.add("Banana");
hashSet.add("Orange");

TreeSet<String> treeSet = new TreeSet<>(hashSet);
System.out.println(treeSet);

          

          
           
[Apple, Banana, Orange]

Example 2 - Sorting with a Comparator

The previous example showed how to convert a HashSet to a TreeSet using the default comparison behavior. However, you can also define your own comparison logic using a Comparator object. The following example shows how to sort the elements in a HashSet based on their length.


          
           
HashSet<String> hashSet = new HashSet<>();
hashSet.add("Apple");
hashSet.add("Banana");
hashSet.add("Orange");

Comparator<String> comparator = Comparator.comparingInt(String::length);
TreeSet<String> treeSet = new TreeSet<>(comparator);
treeSet.addAll(hashSet);
System.out.println(treeSet);

          

          
           
[Apple, Orange, Banana]

Example 3 - Convert and Sort in One Step

The previous examples showed how to convert a HashSet to a TreeSet in two steps. However, you can also convert and sort the elements in a single step using the constructor that accepts a Collection and a Comparator. The following example shows how to accomplish this.


          
           
HashSet<String> hashSet = new HashSet<>();
hashSet.add("Apple");
hashSet.add("Banana");
hashSet.add("Orange");

Comparator<String> comparator = Comparator.comparingInt(String::length);
TreeSet<String> treeSet = new TreeSet<>(hashSet, comparator);
System.out.println(treeSet);

          

          
           
[Apple, Orange, Banana]
