Convert Comma Separated String to HashSet in Java

The HashSet class is a widely-used implementation of the Set interface in Java. It also allows you to store objects of any type. This makes it ideal for storing comma-separated strings, as they can be easily converted into a Set. In this article, we'll go through the process of converting a comma-separated string into a HashSet in Java.

Using String.split()

The easiest way to convert a comma-separated string into a HashSet is by using the 

String.split()
method. This method takes a string and splits it into an array of strings, separating each element at the comma (or whatever string you provide). Here's an example: 

          
           
String input = "apple, orange, banana";
String[] elements = input.split(", ");

Now that we have an array of strings, we can use them to create a HashSet. We simply need to loop through the array and add each element to the set:


          
           
HashSet<String> set = new HashSet<>();
for (String element : elements) {
    set.add(element);
}

Using Streams

Java 8 introduced a new way of working with collections, called Streams. They allow us to perform operations on collections more easily and efficiently. We can use streams to convert a comma-separated string into a HashSet, as shown below:


          
           
String input = "apple, orange, banana";
HashSet<String> set = Arrays.stream(input.split(", "))
        .collect(Collectors.toCollection(HashSet::new));

This approach is much simpler than the first one, as it only requires us to write 1 line of code. However, it has the same effect, as it converts the comma-separated string into a HashSet.

Conclusion

In this article, we discussed two ways of converting a comma-separated string into a HashSet in Java. The first method is to use the 

String.split()
method, followed by a loop to add each element to the set. The second method is to use streams, which is a more concise approach. Whichever method you choose, you can now easily convert a comma-separated string into a HashSet in Java.
