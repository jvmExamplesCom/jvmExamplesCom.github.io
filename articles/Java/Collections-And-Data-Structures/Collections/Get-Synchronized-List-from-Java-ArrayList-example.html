Synchronizing Java ArrayLists

Java ArrayLists offer a great way to store data in a flexible manner, but there are times when you might also want to make sure that other programs don't interfere with your data while it is being accessed. This is where synchronization comes in: if you need to make sure that only one thread can access an array list at the same time, you'll need to get a synchronized version of the list.

Getting a Synchronized List from an ArrayList

The Java API provides two methods for getting a synchronized version of a java.util.ArrayList. The first option is to wrap the list using the static synchronizedList() method: 

          
           
List<String> synchronizedList = 
    Collections.synchronizedList(new ArrayList<String>());
The second option is to use the synchronized wrapper constructor: 

          
           
List<String> synchronizedList = new 
    Collections.synchronizedList(new ArrayList<String>());

Using a Synchronized List

Once you have a synchronized list, you will need to use the synchronized keyword when accessing it: 

          
           
List<String> synchronizedList = 
    Collections.synchronizedList(new ArrayList<String>());

synchronized (synchronizedList) {
    for (String s : synchronizedList) {
        // do something 
        System.out.println(s);
    }
}
In this example, the synchronized keyword ensures that only one thread can access the list at any given time.

Example

Let's take a look at a complete example that shows how to create and use a synchronized list: 

          
           
import java.util.ArrayList;
import java.util.Collections;
import java.util.List;

public class SynchronizedListExample {

    public static void main(String[] args) {
        List<String> synchronizedList = 
            Collections.synchronizedList(new ArrayList<String>());

        synchronized (synchronizedList) {
            synchronizedList.add("one");
            synchronizedList.add("two");
            synchronizedList.add("three");
            synchronizedList.add("four");

            for (String s : synchronizedList) {
                System.out.println(s);
            }
        }
    }

}

one
two
three
