Shuffling Elements of Java ArrayList Example

Introduction

Java ArrayList is one of the most commonly used data structures for storing collections of objects. It has many advantages over other data structures, such as being dynamic and resizable, making it a great choice for use in many applications. In this tutorial, we will look at how to shuffle elements of an ArrayList in Java.

Shuffling Elements of ArrayList in Java

Shuffling the elements of an ArrayList in Java can be done in several different ways. In this section, we will look at two of the most popular methods.

Method 1: Using the Collections.shuffle() Method

The simplest way to shuffle elements of an ArrayList in Java is by using the built-in Collections.shuffle() method. This method takes an array or a list as an argument and returns a randomly shuffled version of the array or list. Let's take a look at an example. 

          
           
import java.util.ArrayList;
import java.util.Collections;

public class ShuffleExample {

    public static void main(String[] args) {
        // Create an ArrayList of Strings
        ArrayList<String> list = new ArrayList<>();
        list.add("apple");
        list.add("banana");
        list.add("orange");
        list.add("grapes");
        list.add("kiwi");
        list.add("pineapple");
    
        // Shuffle the elements in the list
        Collections.shuffle(list);
    
        // Print out the new list
        System.out.println(list);
    }
}
Output: 

          
           
[orange, kiwi, pineapple, apple, grapes, banana]
As you can see, the elements of our ArrayList have been randomly shuffled.

Method 2: Using the Random Class

Another way to shuffle the elements of an ArrayList in Java is to make use of the Random class. The Random class is used to generate pseudo-random numbers in Java, and can be used to randomly shuffle the elements of an ArrayList. Here is an example of how to do this: 

          
           
import java.util.ArrayList;
import java.util.Random;

public class ShuffleExample2 {

    public static void main(String[] args) {
        // Create an ArrayList of Strings
        ArrayList<String> list = new ArrayList<>();
        list.add("apple");
        list.add("banana");
        list.add("orange");
        list.add("grapes");
        list.add("kiwi");
        list.add("pineapple");
    
        // Create a Random object
        Random rand = new Random();
    
        // Shuffle the elements in the list
        for (int i = 0; i < list.size(); i++) {
            int randomIndex = rand.nextInt(list.size());
            String temp = list.get(i);
            list.set(i, list.get(randomIndex));
            list.set(randomIndex, temp);
        }
        
        // Print out the new list
        System.out.println(list);
    }
}
Output: 

          
           
[grapes, pineapple, banana, orange, apple, kiwi]

Conclusion

In this tutorial, we looked at two different methods for shuffling the elements of an ArrayList in Java. The first method made use of the built-in Collections.shuffle() method, while the second method made use of the Random class. Both methods can be used to randomly shuffle the elements of an ArrayList in Java.
