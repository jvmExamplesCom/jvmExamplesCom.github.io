Find Minimum element of Java HashSet Example

Java HashSet is a collection that uses a hash table for storage and is used to store objects. This example demonstrates how to find the minimum element from a given HashSet in Java. To find the minimum element from the given HashSet, we can use the methods available in the HashSet class such as iterator(), peek() or forEach().

Example: Find Minimum Element using Iterator()

The following example demonstrates how to find the minimum element from the given HashSet by using the iterator() method. 

          
           
import java.util.HashSet;  
import java.util.Iterator;  
public class MinimumElementExample {  
    public static void main(String[] args) {  
        // create a HashSet  
        HashSet<Integer> hashSet = new HashSet<>(); 

        // add elements to the HashSet  
        hashSet.add(2);  
        hashSet.add(3);  
        hashSet.add(4);  
        hashSet.add(5);  

        // get an iterator  
        Iterator<Integer> iterator = hashSet.iterator();  

        // declare min variable  
        int min = Integer.MAX_VALUE; 

        // iterate through the elements  
        while (iterator.hasNext())  
        {  
            // get the element  
            int element = iterator.next();  

            // compare with current minimum  
            if(element < min)  
            {  
                // update the current minimum  
                min = element;  
            }  
        }  

        System.out.println("Minimum Element in Set : " + min);  
    }  
}
When we run the above program, we get the following output: 

          
           
Minimum Element in Set : 2

Example: Find Minimum Element using peek()

The following example demonstrates how to find the minimum element from the given HashSet by using the peek() method. 

          
           
import java.util.HashSet;  
public class MinimumElementExample2 {  
    public static void main(String[] args) {  
        // create a HashSet  
        HashSet<Integer> hashSet = new HashSet<>();  

        // add elements to the HashSet  
        hashSet.add(2);  
        hashSet.add(3);  
        hashSet.add(4);  
        hashSet.add(5);  

        // get the minimum element  
        int min = hashSet.stream().min(Integer::compare).get();  

        System.out.println("Minimum Element in Set : " + min);  
    }  
}
When we run the above program, we get the following output: 

          
           
Minimum Element in Set : 2

Example: Find Minimum Element using forEach()

The following example demonstrates how to find the minimum element from the given HashSet by using the forEach() method. 

          
           
import java.util.HashSet;  
public class MinimumElementExample3 {  
    public static void main(String[] args) {  
        // create a HashSet  
        HashSet<Integer> hashSet = new HashSet<>();  

        // add elements to the HashSet  
        hashSet.add(2);  
        hashSet.add(3);  
        hashSet.add(4);  
        hashSet.add(5);  

        // declare min variable  
        int min = Integer.MAX_VALUE; 

        // iterate through the HashSet  
        hashSet.forEach((element) -> {  
            // compare with current minimum  
            if(element < min)  
            {  
               // update the current minimum  
               min = element;  
            }  
        });  

        System.out.println("Minimum Element in Set : " + min);  
    }  
}
When we run the above program, we get the following output: 

          
           
Minimum Element in Set : 2
So, in this tutorial, we learned three different methods to find the minimum element in a given Java HashSet. We used the iterator(), peek() and forEach() methods which are available in the HashSet class.
