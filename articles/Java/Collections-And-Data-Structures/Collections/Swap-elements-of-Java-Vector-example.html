Swap Elements of Java Vector Example

Java provides a special kind of 

          
           
Vector
, which is used to store multiple values of the same type. This collection can be used to store a collection of data, such as strings or objects. The 

          
           
Vector
class provides methods for manipulating the elements in the collection. One of these methods is the 

          
           
swap()
method, which allows two elements to be swapped in the collection.

Using the Vector.swap() Method

The 

          
           
swap()
method takes two parameters, which are the indices of the elements that should be swapped. The method then swaps their positions in the collection. The syntax of the method is as follows: 

          
           
vector.swap(int index1, int index2);
The indices must be valid for the collection, or an 

          
           
ArrayIndexOutOfBoundsException
will be thrown.

Example: Swapping Vector Elements

To demonstrate how to use the 

          
           
swap()
method, we can create a 

          
           
Vector
of strings and swap the first and second elements. First, we need to create the 

          
           
Vector
collection, which we can do with the following code: 

          
           
Vector<String> values = new Vector<>();
We can then add some strings to the 

          
           
Vector
: 

          
           
values.add("foo");
values.add("bar");
values.add("baz");
Finally, we can call the 

          
           
swap()
method to swap the first and second elements: 

          
           
values.swap(0, 1);
After this method is called, the 

          
           
Vector
will contain the elements 

          
           
bar
, 

          
           
foo
, and 

          
           
baz
.

Conclusion

The 

          
           
Vector.swap()
method allows two elements in a 

          
           
Vector
collection to be swapped. This is a very useful way of reordering the elements of the collection.
