Converting a String or String array to HashMap (String to Map)

In some cases, it's necessary to convert a String or String array to a HashMap. This is possible with Java and can be done with various approaches. The following sections explain all the different ways in which this can be accomplished.

Method 1: Array Constructor

The simplest way to convert a String or String array to HashMap is through the use of an array constructor. This converts each string in the input String or String array into a key-value pair, with the first element of the string being the key and the second element being the value.

For example, let's say we have an input String as follows:


          
           
String data = "key1:value1,key2:value2,key3:value3";

We can use the following code to parse the input String into a HashMap:


          
           
String[] pairs = data.split(",");
HashMap<String, String> map = new HashMap<>();
for (String pair : pairs) {
   String[] parts = pair.split(":");
   map.put(parts[0], parts[1]);
}

map
has the key-value pairs that were specified in the input String: 

          
           
{key1=value1, key2=value2, key3=value3}
