Reverse Order of All Elements of Java Vector Example

The Java Vector class provides a collection that is very similar to an array. It allows us to store and retrieve objects in an ordered fashion. This example shows how to reverse the order of elements inside a Vector object.

Java Reverse Vector Example

We can use the 

          
           
Collections.reverse()
method to reverse the order of elements inside a Vector object. The below example will create a Vector, add some elements to it, and then reverse the order of the elements. 

          
           
import java.util.Vector; 
import java.util.Collections; 

public class ReverseVectorExample { 
   public static void main(String[] args) 
   { 
      // Create a Vector object 
      Vector<String> vect = new Vector<String>(); 

      // Add elements to the Vector 
      vect.add("Geeks"); 
      vect.add("for"); 
      vect.add("Geeks"); 

      // Print original Vector 
      System.out.println("Original Vector: " + vect); 

      // Reverse the Vector 
      Collections.reverse(vect); 

      // Print the reversed Vector 
      System.out.println("Reversed Vector: " + vect); 
   } 
}
The output of the above code is: 

          
           
Original Vector: [Geeks, for, Geeks]
Reversed Vector: [Geeks, for, Geeks]
