Convert List to Set (Convert ArrayList to HashSet, Convert ArrayList to TreeSet)

Introduction

Java provides the most basic utility when it comes to converting java.util.List objects into java.util.Set objects. In this article, we will discuss how to convert a List into Set (HashSet and TreeSet) using two simple methods. When you are using the power of collections in Java, you will find that there are several different types of collections and each of them can be used in different scenarios. In this article, we will be focusing on lists and sets. A list is an ordered collection of elements, while a set is an unordered collection of elements. So if we want to convert a List object into a Set object, we need to use some specific methods.

Convert ArrayList to HashSet

The following code example shows how to convert an ArrayList object into a HashSet object: 

          
           
ArrayList<String> al = new ArrayList<>();
al.add("One");
al.add("Two");
al.add("Three"); 

HashSet<String> hs = new HashSet<>();
hs.addAll(al);
In the above code, we have created an ArrayList with three strings and then created a HashSet object. We then used the addAll() method to add all the elements of the ArrayList to the HashSet.

Convert ArrayList to TreeSet

The following code example shows how to convert an ArrayList object into a TreeSet object: 

          
           
ArrayList<String> al = new ArrayList<>();
al.add("One");
al.add("Two");
al.add("Three"); 

TreeSet<String> ts = new TreeSet<>();
ts.addAll(al);
In the above code, we have created an ArrayList with three strings and then created a TreeSet object. We then used the addAll() method to add all the elements of the ArrayList to the TreeSet.

Conclusion

In this article, we discussed two methods to convert a List object into a Set object- one to convert an ArrayList to a HashSet and one to convert an ArrayList to a TreeSet. Both of these methods are very easy to use and can be used in any application that makes use of List and Set objects.
