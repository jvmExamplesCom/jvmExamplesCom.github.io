Create Java ArrayList from Enumeration Example

The following code snippet shows how to create a 

          
           
java.util.ArrayList
from an 

          
           
Enumeration
, and how to loop over it. This can be used if you want to work with a standard 

          
           
List
but get the data from an 

          
           
Enumeration
. 

          
           
// Create the Enumeration
Enumeration enum = ...;

// Loop while there are more elements left
List list = new ArrayList();
while (enum.hasMoreElements()) {
    // Get the element and add it to the List
    Object o = enum.nextElement();
    list.add(o);
}

// Do something with the list
for (Object o : list) {
    // Work with the object
}
The snippet above creates a new 

          
           
ArrayList
, and then iterates over the 

          
           
Enumeration
, adding each element to the list. It then loops over the 

          
           
List
, allowing you to perform operations on each element. You can also create an 

          
           
ArrayList
directly from the 

          
           
Enumeration
, using the static method 

          
           
Collections.list()
: 

          
           
List list = Collections.list(enum);
This is much simpler, and has the same effect as the previous snippet.

Example

Here is a complete example that demonstrates both approaches: 

          
           
// Create the Enumeration
ArrayList numbers = new ArrayList();
numbers.add(1);
numbers.add(2);
numbers.add(3);
Enumeration enum = Collections.enumeration(numbers);

// Create the List from the Enumeration
List list1 = new ArrayList();
while (enum.hasMoreElements()) {
    Object o = enum.nextElement();
    list1.add(o);
}

// Create the List directly from the Enumeration
List list2 = Collections.list(enum);

// Print the Lists
System.out.println("list1: " + list1);
System.out.println("list2: " + list2);
The output of the above program is: 

          
           
list1: [1, 2, 3]
list2: [1, 2, 3]
As you can see, both lists have the same content.

Conclusion

In this article, we've seen how to create an 

          
           
ArrayList
from an 

          
           
Enumeration
, both using a loop, and using the static 

          
           
Collections.list()
method.
