Converting Lists to Maps

The 

          
           
java.util.List
and 

          
           
java.util.Map

toMap()

Collectors

Relevant Classes & Methods

Collectors

java.util.stream

Collectors

An Example List-to-Map Conversion

toMap()

We used the third parameter to handle collisions (i.e. if two elements in the list have the same id, the resulting map only contains the entry with the higher index in the list).
