Convert HashSet to ArrayList in Java

Java provides a number of utility classes for working with collections of data. A HashSet is a collection that does not allow duplicate elements, and is unordered. This can be very useful when working with sets of data, but there are times when converting a HashSet to an ArrayList is desirable. In this article, we will look at how to convert a HashSet to an ArrayList in Java.

Example 1: Basic Conversion From HashSet to ArrayList

The simplest way to convert a HashSet to an ArrayList is to use the ArrayList constructor. In this example, we create a HashSet of Strings and then convert it to an ArrayList.


          
           
HashSet<String> hs = new HashSet<>();
hs.add("Apple");
hs.add("Banana");
hs.add("Orange");

ArrayList<String> al = new ArrayList<>(hs);
System.out.println(al);
The output of this code is: 

          
           
[Apple, Banana, Orange]

Example 2: Using the Collection.toArray() Method

The Collection.toArray() method can also be used to convert a HashSet to an ArrayList. This method takes an array as a parameter and then populates it with the elements of the collection. In this example, we convert a HashSet of Strings to an ArrayList.


          
           
HashSet<String> hs = new HashSet<>();
hs.add("Apple");
hs.add("Banana");
hs.add("Orange");

String[] arr = new String[hs.size()];
ArrayList<String> al = new ArrayList<>(Arrays.asList(hs.toArray(arr)));
System.out.println(al);
The output of this code is: 

          
           
[Apple, Banana, Orange]

Example 3: Using the Stream API

The java.util.stream API provides a way to convert a HashSet to an ArrayList. In this example, we use the Stream API to convert a HashSet of Strings to an ArrayList.


          
           
HashSet<String> hs = new HashSet<>();
hs.add("Apple");
hs.add("Banana");
hs.add("Orange");

ArrayList<String> al = hs.stream().collect(Collectors.toCollection(ArrayList::new));
System.out.println(al);
The output of this code is: 

          
           
[Apple, Banana, Orange]

Conclusion

In this article, we looked at how to convert a HashSet to an ArrayList in Java. We looked at three different methods: using the ArrayList constructor, using the Collection.toArray() method, and using the Stream API. With these methods, it is possible to easily convert a HashSet to an ArrayList.
