Convert List to String in Java

List is a collection of objects, it can hold any types of objects such as Integer, String, Boolean, etc. In Java, there are two implementations of the List interface which are ArrayList and LinkedList. Both these classes implement List interface so they inherit common methods from List interface including add(), remove(), contains(), size(), etc. In this tutorial, we will learn how to convert List (ArrayList and LinkedList) to String in Java. We will look into different code examples that illustrate the ways to convert List to String in Java.

1. Using StringJoiner

StringJoiner is a class introduced in Java 8 that allows us to construct a sequence of characters separated by a delimiter. We can use this class to convert a List to String. The following example shows how to convert an ArrayList of Strings to String using StringJoiner. 

          
           
List<String> list = Arrays.asList("Hello", "World");

// convert List to String
String str = String.join(", ", list);

// print result
System.out.println(str);
The output of the above program will be: 

          
           
Hello, World
Similarly, we can use StringJoiner to convert a LinkedList of Strings to String in Java. For example: 

          
           
LinkedList<String> list = new LinkedList<>(Arrays.asList("Java", "is", "fun"));

// convert list to string
String str = String.join(" ", list);

// print result
System.out.println(str);
The output of the above program will be: 

          
           
Java is fun

2. Using String.Join() Method

The String.join() method is a utility method used to join elements of a given Iterable (such as list, set, etc) separated by a specified delimiter. It takes two arguments: the delimiter and the Iterable element. We can use this method to convert List to String. For example, the following example shows how to convert an ArrayList of Strings to String using String.join() method in Java. 

          
           
List<String> list = Arrays.asList("Python", "is", "powerful");

// convert List to String
String str = String.join(" ", list);

// print result
System.out.println(str);
The output of the above program will be: 

          
           
Python is powerful
Similarly, we can use String.join() method to convert a LinkedList of Strings to String in Java, as shown below: 

          
           
LinkedList<String> list = new LinkedList<>(Arrays.asList("Java", "is", "cool"));

// convert list to string
String str = String.join(".", list);

// print result
System.out.println(str);
The output of the above program will be: 

          
           
Java.is.cool

3. Using Stream API

Another way to convert List to String in Java is to use Java Stream API. Java Stream API provides a very powerful tool for processing collections of objects. We can chain multiple methods together to process objects stored in a List. The following example shows how to convert an ArrayList of Strings to String using Stream API in Java. 

          
           
List<String> list = Arrays.asList("Golang", "is", "fast");

// convert list to string
String str = list.stream().collect(Collectors.joining("-"));

// print result
System.out.println(str);
The output of the above program will be: 

          
           
Golang-is-fast
Similarly, we can use Stream API to convert a LinkedList of Strings to String in Java, as shown below: 

          
           
LinkedList<String> list = new LinkedList<>(Arrays.asList("Kotlin", "is", "concise"));

// convert list to string
String str = list.stream().collect(Collectors.joining("_"));

// print result
System.out.println(str);
The output of the above program will be: 

          
           
Kotlin_is_concise

Conclusion

In this tutorial, we have learned three different ways to convert List (ArrayList and LinkedList) to String in Java. We have used StringJoiner, String.join() method and Stream API to convert List to String.
