How to Convert Hashset to Comma Separated String in Java

Converting a Hashset to a comma separated string is a common task required by Java developers. There are various approaches that can be used to achieve this, as outlined below.

Using Iterator and StringBuilder

The first approach is to use an Iterator and StringBuilder for constructing the comma separated string. 

          
           
Set<String> myHashset = new HashSet<>();
myHashset.add("item1");
myHashset.add("item2");
myHashset.add("item3");

Iterator<String> itr = myHashset.iterator();

StringBuilder sb = new StringBuilder();

// Check if iterator has elements
if (itr.hasNext()) {

    //Append the first element
    sb.append(itr.next());

    // Iterate through remaining elements and append them to the StringBuilder 
    while(itr.hasNext()) {
        sb.append(",");
        sb.append(itr.next());
    }
}

// Convert StringBuilder to string
String output = sb.toString();
System.out.println(output);
// Outputs: item1,item2,item3

Using the Stream API

The second approach is to use the Stream API to construct the comma separated string. 

          
           
Set<String> myHashset = new HashSet<>();
myHashset.add("item1");
myHashset.add("item2");
myHashset.add("item3");

String output = myHashset.stream()
                       .map(Object::toString)
                       .collect(Collectors.joining(","));

System.out.println(output);
// Outputs: item1,item2,item3

Using Apache Commons Lang

The third approach is to use the Apache Commons Lang Library for constructing the comma separated string. 

          
           
import org.apache.commons.lang3.StringUtils;

Set<String> myHashset = new HashSet<>();
myHashset.add("item1");
myHashset.add("item2");
myHashset.add("item3");

String output = StringUtils.join(myHashset, ",");
System.out.println(output);
// Outputs: item1,item2,item3

Conclusion

In this article, we discussed how to convert a Hashset to a comma separated string in Java. Three approaches were outlined: using Iterator and StringBuilder, the Stream API, and the Apache Commons Lang Library.
