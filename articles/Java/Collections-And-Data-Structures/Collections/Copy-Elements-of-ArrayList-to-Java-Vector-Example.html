Copy Elements of ArrayList to Java Vector Example

Java's Vector class contains numerous methods for manipulating its contents. One such method, called addAll, allows elements from a collection, such as an ArrayList, to be added to the vector. Here is an example of using this method to copy the contents of an array list to a vector. 

          
           
import java.util.ArrayList;
import java.util.Vector;

public class CopyExample {
    public static void main(String[] args) {
        ArrayList<Integer> al = new ArrayList<Integer>();
        al.add(3);
        al.add(2);
        al.add(1);

        Vector<Integer> v = new Vector<Integer>();
        v.add(4);
        v.add(5);
        v.add(6);

        // copy elements of ArrayList to Vector
        v.addAll(al);

        System.out.println("Vector Elements: " + v);
    }
}
The output of this code is as follows: 

          
           
Vector Elements: [4, 5, 6, 3, 2, 1]
As can be seen, the elements of the array list have been successfully copied to the vector. It is important to note that the order in which the elements are copied determines the order in which the elements are present in the vector. In the above example, the elements were added in the order they were present in the array list, so the vector contains them in the same order. However, if the order of the elements in the array list is changed, then the output would be different. For example, if the elements in the array list are reversed, then the output would be as follows: 

          
           
Vector Elements: [4, 5, 6, 1, 2, 3]
In this example, the last element of the array list (which is now the first element) was also the first element added to the vector. Thus, it is the first element in the vector. The order of elements in the vector is determined by the order in which they were added. It is also possible to add multiple collections to a vector using the addAll method. For example, if there is another array list with elements 7, 8 and 9, then the following code can be used: 

          
           
ArrayList<Integer> al2 = new ArrayList<Integer>();
al2.add(7);
al2.add(8);
al2.add(9);

v.addAll(al2);

System.out.println("Vector Elements: " + v);
The output of this code is as follows: 

          
           
Vector Elements: [4, 5, 6, 3, 2, 1, 7, 8, 9]
As can be seen, the elements of both the array lists have been successfully copied to the vector. Finally, it is important to note that the addAll method only adds the elements of the collection to the vector. It does not delete existing elements from the vector. Thus, if the vector already contains elements, then these elements will remain and the elements from the collection will be added after them. In summary, the Java Vector class provides the addAll method to copy the elements of a collection, such as an ArrayList, to the vector. The order in which the elements are added to the vector determines the order in which they appear. It is also possible to add multiple collections to the vector using this method.
