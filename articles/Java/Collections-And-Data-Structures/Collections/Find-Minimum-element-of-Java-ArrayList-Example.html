Find Minimum Element of Java ArrayList Example

This example shows how to find minimum element of Java ArrayList. The class 

Collections.min()
is used to find the minimum element of the given collection, according to natural ordering.

Java Program Example

Create a Java program containing an 
ArrayList
of Integer and use the 
Collections.min()
method to get the minimum element. 

          
           
import java.util.ArrayList;
import java.util.Collections;

public class FindMinimumElementExample {

   public static void main(String[] args) {
      //create an ArrayList object
      ArrayList arrayList = new ArrayList();

      //Add elements to ArrayList object
      arrayList.add(new Integer("10"));
      arrayList.add(new Integer("25"));
      arrayList.add(new Integer("8"));
      arrayList.add(new Integer("19"));
      arrayList.add(new Integer("9"));

      //Get the minimum element of ArrayList
      Object min = Collections.min(arrayList);

      //Print the minimum element 
      System.out.println("Minimum Element of ArrayList: " + min);
   }
}

When you compile and run the above Java program, it will print the following output:


          
           
Minimum Element of ArrayList: 8
