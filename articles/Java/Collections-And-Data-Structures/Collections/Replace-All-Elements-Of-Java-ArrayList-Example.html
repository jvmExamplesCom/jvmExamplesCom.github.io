Replace All Elements Of Java ArrayList Example

In this tutorial, we are going to learn how to replace all elements of a Java ArrayList with a specified value. Java ArrayList is one of the most useful classes that Java offers. In the following example, we create an ArrayList called “list” and populate it with some elements. After that we use the replaceAll() method to replace all the elements of the list with the value “A”.

list.add("D");
