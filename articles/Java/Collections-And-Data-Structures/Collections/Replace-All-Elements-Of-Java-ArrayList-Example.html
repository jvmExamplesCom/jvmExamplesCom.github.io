Replace All Elements Of Java ArrayList Example

In this tutorial, we are going to learn how to replace all elements of a Java ArrayList with a specified value. Java ArrayList is one of the most useful classes that Java offers. In this example, we are going to learn how to replace all elements of a Java ArrayList with a specified value.

Example 1: Replace All Elements of ArrayList

In the following example, we create an ArrayList called “list” and populate it with some elements. After that we use the replaceAll() method to replace all the elements of the list with the value “A”. 

          
           
ArrayList<string> list = new ArrayList<string>();
list.add("B");
list.add("C");
list.add("D");

list.replaceAll(s -> "A");
