Converting Map to List (Convert HashMap to ArrayList)

Java is a powerful language which provides many different kinds of data structures. One of these data structures is the 

          
           
Map
, which is used to store key-value pairs. A common implementation of the 

          
           
Map
interface is the 

          
           
HashMap
. In this article we will look at how to convert a 

          
           
Map
to a 

          
           
List
using two methods: using the 

          
           
ArrayList
constructor and using Java's 

          
           
Stream
API.

Using the ArrayList Constructor

The first method we will look at involves using the 

          
           
ArrayList
constructor: 

          
           
Map<String, String> map = new HashMap<>();
// populate map

// create a list from entries in map
List<Map.Entry<String, String>> list = 
  new ArrayList<>(map.entrySet());
In this example, we have created a 

          
           
HashMap
to store some key-value pairs. We then create a list of 

          
           
Map.Entry
objects by passing the 

          
           
entrySet()
method of the 

          
           
Map
to the 

          
           
ArrayList
constructor.

Using Java Streams

Another way to achieve the same result is to use the 

          
           
Stream
API: 

          
           
Map<String, String> map = new HashMap<>();
// populate map

// create a list from entries in map
List<Map.Entry<String, String>> list = 
  map.entrySet().stream().collect(Collectors.toList());
This example is almost identical to the previous one, except that we are using the 

          
           
Stream
API to perform the conversion. The 

          
           
stream()
method returns a 

          
           
Stream
of 

          
           
Map.Entry
objects, which is then collected into a 

          
           
List
using the 

          
           
Collectors
class.

Conclusion

In this article, we looked at two methods for converting a 

          
           
Map
to a 

          
           
List
: using the 

          
           
ArrayList
constructor and using Java's 

          
           
Stream
API. Both of these approaches are simple and straightforward, yet powerful enough to perform the task at hand.
