Converting an Array to Vector in Java

Java provides the ability to store collections of objects. Arrays and Vector classes are used for this purpose. An array is a way to store multiple values in a single variable. However, in some cases, it is useful to convert an array to a Vector object so we can make use of the methods specific to that object. In this article, we will look at how to convert an array to Vector in Java.

Converting Array to Vector

To convert an array to Vector, we must first create an empty Vector object. We can then use the addElement() method of the Vector class to add each element of the array to the Vector. This can be done with a simple for loop: 

          
           
// Create an empty Vector object
Vector<Integer> vec = new Vector<>();

// Get an array of integers
int[] array = {1, 2, 3, 4, 5};

// Loop through the array and add each element to the Vector 
for(int element : array) {
    vec.addElement(element);
}
Once the loop is complete, we have a Vector object which contains all of the elements of the array.

Converting Vector to Array

We can also go the other way and convert a Vector to an array. To do this, we use the toArray() method of the Vector class. This method takes an array as an argument and returns a new array containing the elements of the Vector. 

          
           
// Create a Vector object
Vector<Integer> vec = new Vector<>(Arrays.asList(1, 2, 3, 4, 5));

// Create an empty array
int[] array = new int[vec.size()];

// Convert the Vector to an array
array = vec.toArray(array);
Once the Vector has been converted to an array, we can make use of the array as we would any other array.
