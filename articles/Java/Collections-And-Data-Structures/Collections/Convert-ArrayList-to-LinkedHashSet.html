Converting ArrayList to LinkedHashSet

Converting a 
ArrayList
to a 
LinkedHashSet
is a common technique when needing to store a collection of data in a specific order. This process can be done using several different approaches, including using the constructor of the 
LinkedHashSet
class or using the addAll() method of the 
LinkedHashSet
class.

Using Constructor

A quick way to convert an 
ArrayList
to a 
LinkedHashSet
is to use the constructor of the 
LinkedHashSet
class. The constructor contains a single parameter which takes an existing 
Collection
type, such as an 
ArrayList
. The constructor will automatically add the elements from the provided Collection type into the 
LinkedHashSet
. 

          
           
ArrayList<String> list = new ArrayList<String>();
list.add("a");
list.add("b");
list.add("c");

LinkedHashSet<String> set = new LinkedHashSet<String>(list);
In the example above, the elements in the 
list
will be added to the 
set
in the same order they are found in the 
list
. This is due to the 
LinkedHashSet
preserving the order they are added to.

Using addAll()

Alternatively, the 
LinkedHashSet
class contains an 
addAll()
method which also takes a 
Collection
type as an argument and adds all of the elements from the Collection type to the 
LinkedHashSet
. 

          
           
ArrayList<String> list = new ArrayList<String>();
list.add("a");
list.add("b");
list.add("c");

LinkedHashSet<String> set = new LinkedHashSet<String>();
set.addAll(list);
The elements within the 
list
will be added to the 
set
in the same order they are found in the 
list
. As with the constructor, this is due to the 
LinkedHashSet
preserving the order they are added to.

Conclusion

In summary, there are two ways of converting an 
ArrayList
to a 
LinkedHashSet
, either by using the constructor or by using the addAll() method. Both techniques will result in the elements from the 
ArrayList
being added to the

LinkedHashSet
preserving the order they are added in.
