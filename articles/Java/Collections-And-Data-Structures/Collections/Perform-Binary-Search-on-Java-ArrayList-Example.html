Perform Binary Search on Java ArrayList Example

Java provides a powerful way of searching and sorting data through the use of its ArrayList class. In this article, we'll look at how to perform a binary search on an ArrayList and provide a code example for implementation.

What is a Binary Search?

A binary search is an algorithm for finding a particular element in a sorted list. It works by comparing the element in the middle of the list with the target element. If the middle element is smaller than the target, then the algorithm searches the second part of the list. If the middle element is larger than the target, then the search occurs in the first part of the list. This process continues until either the target element is found or the list is exhausted.

How to Perform a Binary Search on an ArrayList

To perform a binary search on an ArrayList, the following steps need to be followed:
  1. Sort the ArrayList using Collections.sort() method.
  2. Find the index of the middle element using ArrayList.size()/2.
  3. Compare the middle element with the target element. If it is equal, then the target element has been found and the search can stop. If it is not equal, proceed to step 4.
  4. If the middle element is smaller than the target element, search the second half of the list. If it is larger than the target element, search the first half of the list.
  5. Repeat steps 2 - 4 until the target element is found or the list is exhausted.

Java Code Example for Binary Search on an ArrayList

Below is an example of how to perform a binary search on an ArrayList using Java: 

          
           
import java.util.*;

public class Main {
    public static void main(String[] args) {
        // Create an ArrayList
        ArrayList<Integer> list = new ArrayList<>();
        list.add(5);
        list.add(10);
        list.add(15);
        list.add(20);
        list.add(25);
        list.add(30);
        
        // Sort the ArrayList
        Collections.sort(list);

        // Target element
        int target = 20;

        // Perform binary search
        int low = 0;
        int high = list.size() - 1;
        while (low <= high) {
            int mid = (low + high) / 2;
            if (list.get(mid) == target) {
                System.out.println("Element found at index " + mid);
                break;
            } else if (list.get(mid) < target) {
                low = mid + 1;
            } else {
                high = mid - 1;
            }
        }
    }
}
