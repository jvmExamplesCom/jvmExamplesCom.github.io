Converting an ArrayList to a LinkedHashMap

Java provides a powerful set of classes for data storage and manipulation through the use of collections. Two of the most popular collection classes are ArrayList and LinkedHashMap. An ArrayList stores elements in order, while LinkedHashMap stores and retrieves them in a specific order. It is often necessary to convert between these two collection types. In this article, we will explore how to convert a Java ArrayList to a LinkedHashMap.

Using an Iterator to Traverse the ArrayList

The first step in converting from ArrayList to LinkedHashMap is to traverse the ArrayList using an Iterator. An Iterator allows us to loop through the list and access each element one at a time. Iterators can be created with the following code: 

          
           
ArrayList<String> arrList = new ArrayList<String>(); 
Iterator<String> itr = arrList.iterator(); 
while(itr.hasNext()) { 
   String str = itr.next(); 
   //do something with the string str
}
This code creates an Iterator over the ArrayList and then iterates through each element until the list is completely traversed. During each iteration, the element is stored in the String variable ‘str’.

Using a for-each Loop to Traverse the ArrayList

Java 8 adds a feature called enhanced for-each loops, which use the ‘for-each’ keyword to iterate through the elements of the list. This is more concise than using an Iterator and makes code more readable. For example, the following code will loop through the elements of an ArrayList: 

          
           
ArrayList<String> arrList = new ArrayList<String>(); 
for(String str : arrList) { 
   //do something with the string str 
}
As before, each element is stored in the String variable ‘str’ during each iteration.

Creating the LinkedHashMap

Once the ArrayList has been traversed, it is time to create the LinkedHashMap. A LinkedHashMap is a key-value mapping, where each element of the list is stored with a unique key. The LinkedHashMap is initialized with the following code: 

          
           
LinkedHashMap<String,Object> linkedHashMap = new LinkedHashMap<String,Object>();
The generic type parameters specify the type of key and value to be stored in the mapping. For example, if the element of the list is a String, then the key must also be a String.

Adding Elements from the ArrayList to the LinkedHashMap

Now that we have our Iterator or enhanced for-each loop set up to traverse the ArrayList, and our LinkedHashMap initialized, it is time to add the elements to the LinkedHashMap. This is accomplished with the following code: 

          
           
int i = 0;
while(itr.hasNext()) { 
   String str = itr.next(); 
   linkedHashMap.put("key" + i, str); 
   i++;
}}
This code uses the Iterator to loop through the ArrayList. For each element, a unique key is created, using an integer ‘i’, and the element is stored in the corresponding value position of the LinkedHashMap.

Conclusion

In this article, we have explored how to convert a Java ArrayList to a LinkedHashMap. We have seen how to use an Iterator or enhanced for-each loop to traverse the ArrayList, how to create a LinkedHashMap, and how to add elements from the ArrayList to the LinkedHashMap. By following these steps, you should now be able to convert between these two important collection types.
