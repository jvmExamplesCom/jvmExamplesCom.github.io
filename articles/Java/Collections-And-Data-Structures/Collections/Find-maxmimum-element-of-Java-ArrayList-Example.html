Find Maximum Element of Java ArrayList Example

In this tutorial, we will learn how to find maximum element of an ArrayList in Java. We will be using the default max() method of the Collections class, which finds the maximum element of all elements stored in an ArrayList.

Java Program to find the Maximum Element of an ArrayList

Below is a simple Java program to find the maximum element of an ArrayList:


          
           
import java.util.ArrayList; import java.util.Collections; public class Main { public static void main(String[] args) { ArrayList<Integer> list = new ArrayList<>(); 
list.add(5); list.add(10); list.add(20); int max = Collections.max(list); System.out.println("Maximum element of the ArrayList is : " + max); }
Output: 

          
           
Maximum element of the ArrayList is : 20

Example Explained

In the above Java Program, we are trying to find the maximum element of the list ArrayList. We first create an ArrayList and add 3 elements to it. Then, we use the Collections.max() method to find out the maximum element of the list. Finally, we print out the result.
