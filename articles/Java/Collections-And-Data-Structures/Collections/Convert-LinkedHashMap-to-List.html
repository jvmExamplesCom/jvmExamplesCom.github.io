Converting a LinkedHashMap to a List in Java

With the help of Java, it's easy to convert a LinkedHashMap to a List. A LinkedHashMap is a type of Map that maintains a linked list of the entries within it, so that the entries get traversed in the order they were inserted into the Map. The process of converting a LinkedHashMap to a List requires the use of some classes and methods defined in the Java collections framework. Some of the important classes which form a part of the collection framework include Maps, Lists, LinkedLists, Set, and Iterators.

Steps to Convert a Linked HashMap to a List

The following steps can be followed to convert a LinkedHashMap to a List:
  • Create an empty List.
  • Create an iterator to traverse the entries in the LinkedHashMap.
  • Traverse the entries in the LinkedHashMap and add each entry to the List.

Example

The following example demonstrates how to convert a LinkedHashMap to a List. In this example, a LinkedHashMap is created and populated with some key-value pairs, and then it is converted to a List using the steps outlined above. 

          
           
import java.util.LinkedHashMap;
import java.util.List;
import java.util.ArrayList;
import java.util.Iterator;
 
public class Main {
 
    public static void main(String[] args) {
 
        //Create a LinkedHashMap
        LinkedHashMap<String, String> map = new LinkedHashMap<String, String>();
        //Populate the map
        map.put("Key 1", "Value 1");
        map.put("Key 2", "Value 2");
        map.put("Key 3", "Value 3");
 
        //Create a list
        List<String> list = new ArrayList<String>();
 
        //Create an iterator
        Iterator<String> it = map.keySet().iterator();
 
        //Traverse the map and add the entries to the list
        while(it.hasNext()){
            String key = it.next();
            list.add(map.get(key));
        }
 
        //Print out the List
        System.out.println(list);
    }
}
When this code is compiled and executed, the output will be a List of all the values in the LinkedHashMap.

Conclusion

In this tutorial, we learned how to convert a LinkedHashMap to a List in Java. This can be done by creating an empty List, creating an Iterator to traverse the entries in the LinkedHashMap, and then traversing the entries and adding each one to the List.
