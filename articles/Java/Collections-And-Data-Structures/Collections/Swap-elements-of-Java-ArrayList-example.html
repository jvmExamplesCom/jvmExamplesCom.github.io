Swapping Elements of Java ArrayList Example

Java provides a variety of classes to work with collections of objects. One of the most popular and versatile of these is the ArrayList. An ArrayList is a type of List, which stores elements in a dynamically-sized array. This allows for fast access to elements in the list, but can present challenges when it comes to swapping elements. In this article, we’ll explore how to swap elements within an ArrayList in Java. We’ll look at some code examples that demonstrate different ways to achieve this. Let’s get started!

Creating an ArrayList

Before we can start swapping elements in an ArrayList, we need to create one. To do this, we can declare an ArrayList variable like so: 

          
           
List<Integer> myList = new ArrayList<Integer>();
This creates an empty ArrayList that holds Integer values. The type of object stored in the ArrayList must be specified within the angle brackets (<>). We can then add elements to the list using the add() method: 

          
           
myList.add(1);
myList.add(2);
myList.add(3);
This adds the values 1, 2, and 3 to the list.

Swapping Elements in an ArrayList

Once we have an ArrayList, swapping elements is relatively simple. We can use the set() method to replace an element at a given index. For example, let’s say we want to swap the first and second elements of our list. We could do this as follows: 

          
           
int temp = myList.get(0);
myList.set(0, myList.get(1));
myList.set(1, temp);
Here, we’ve used a temporary variable to store the value of the first element. We then use the set() method to set the first element to the value of the second element, and the second element to the value of the temporary variable. We can also swap elements using the Collections.swap() method. This method takes two indices as parameters and swaps the elements at those indices: 

          
           
Collections.swap(myList, 0, 1);
The result of either of these methods will be the same. The list will now contain the elements in reverse order: 

          
           
[2, 1, 3]

Conclusion

In this article, we looked at how to swap elements within an ArrayList in Java. We explored the set() and Collections.swap() methods, and saw examples of how to use them to change the order of elements in a list. With these techniques, you’ll be able to perform all sorts of manipulations on your ArrayLists!
