Java Convert TreeSet to array

Converting a TreeSet to an array in Java is relatively straightforward and can be accomplished using the 

          
           
toArray()
method of the 

          
           
TreeSet
class. Converting a TreeSet to an array is useful when you want to access all of the elements in the set quickly, such as in a loop or in some other type of processing.

Creating a TreeSet

Before we can convert a TreeSet to an array, we must first create a TreeSet. A TreeSet is a specialization of the AbstractSet class, which implements the Set interface. A Set is a collection that stores unique elements and does not allow duplicates. The following code creates a TreeSet of type String with five elements: 

          
           
TreeSet<String> myTreeSet = new TreeSet<>();
myTreeSet.add("Apple");
myTreeSet.add("Banana");
myTreeSet.add("Orange");
myTreeSet.add("Kiwi");
myTreeSet.add("Mango");

Converting a TreeSet to an Array

Once we have created a TreeSet, we can convert it to an array using the 

          
           
toArray()
method. This method takes one parameter, a type array, which is used to store the elements of the TreeSet. The following code demonstrates how to use the 

          
           
toArray()
method: 

          
           
String[] myArray = myTreeSet.toArray(new String[myTreeSet.size()]);
The 

          
           
toArray()
method will return an array containing the elements of the TreeSet in the same order that they were added. In this example, the array 

          
           
myArray
will contain the following elements: 

          
           
Apple, Banana, Orange, Kiwi, Mango

Accessing the Elements of the Array

Once the TreeSet has been converted to an array, you can access the elements of the array using a 

          
           
for
loop. The following code demonstrates how to loop through the elements of the array: 

          
           
for (String element : myArray) {
   System.out.println(element);
}
This code will print out each element of the array, one per line: 

          
           
Apple
Banana
Orange
Kiwi
Mango
