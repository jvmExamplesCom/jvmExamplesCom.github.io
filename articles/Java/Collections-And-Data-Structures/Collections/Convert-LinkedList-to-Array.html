Converting a LinkedList to an Array in Java

Java's LinkedList class is one of the most versatile data structures out there. It allows for easy insertion, deletion and manipulation of elements as well as constant-time access to its elements. However, there are certain scenarios where an array may be more appropriate, such as where an array of objects needs to be generated from another list or where an array of primitives is required but a LinkedList is given. In this article, we will explore how to convert a LinkedList to an array in Java.

Using the LinkedList.toArray() Method

The easiest way to convert a LinkedList to an array is to use the toArray() method of the LinkedList class. This method accepts an array of Objects as a parameter and returns an array containing all of the elements of the LinkedList in the same order as they appear in the list. Consider the following example: 

          
           
LinkedList<String> list = new LinkedList<String>();
list.add("foo");
list.add("bar");
list.add("baz");

String[] array = list.toArray(new String[list.size()]);
