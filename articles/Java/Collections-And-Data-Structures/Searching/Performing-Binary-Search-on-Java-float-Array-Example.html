Performing Binary Search on Java Float Array Example

Binary search is an efficient algorithm for searching through a sorted array. It works by comparing the value of the element you are looking for with the middle element in the array, and then discarding half of the array depending on whether the desired element is higher or lower than the middle element. The process is repeated until the desired element is found.

In this example, we are going to go through a step-by-step guide on how to perform a binary search on a Java float array. Let's get started!

Step 1: Create a Sorted Java Float Array

The first step is to create a sorted Java float array that contains the values we want to search through. For this example, we will create an array containing the following elements in ascending order: 1.5, 2.5, 3.5, 4.5, 5.5.


          
           
float[] array = {1.5f, 2.5f, 3.5f, 4.5f, 5.5f};

Step 2: Define Function for Binary Search

The next step is to define a function for performing a binary search on the array. The function should accept the array as an argument, along with the number we want to search for. The function should return the index of the element if it is found, or -1 if it is not found.


          
           
public static int binarySearch(float[] array, float value) {
    int start = 0;
    int end = array.length - 1;
 
    while (start <= end) {
        int mid = (start + end) / 2;
        if (array[mid] == value) {
            return mid;
        }
 
        if (array[mid] < value) {
            start = mid + 1;
        } else {
            end = mid - 1;
        }
    }
 
    return -1;
}

Step 3: Call Function to Perform Binary Search

Now that we have the function defined, we can call it with the array and the element we want to search for. For example, to search for the value 4.5, we can call the function like this:


          
           
int index = binarySearch(array, 4.5f);
