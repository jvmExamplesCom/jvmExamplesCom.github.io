Performing Binary Search on Java int Array Example

Binary search is a very popular and useful algorithm for searching a given item from within a sorted array of items. In this example, we will be discussing how to perform binary search on an integer array in Java.

Search Algorithm

The following is the algorithm for performing binary search on a sorted array:
  • Initialize the starting index as 0 and the ending index as n-1, where n is the length of the array.
  • Calculate the index of the middle element by taking the average of the starting and ending indexes.
  • If the value of the middle element matches the search item, return its index.
  • If the value of the middle element is less than the search item, set the starting index to the middle index + 1.
  • If the value of the middle element is greater than the search item, set the ending index to the middle index - 1.

Code Example

The following is a Java code example for performing binary search on an integer array: 

          
           
public static int binarySearch(int[] array, int key) {
    // Initialize the starting index and the ending index
    int start = 0;
    int end = array.length - 1;
    
    // Loop while the starting index is less than or equal to the ending index
    while (start <= end) {
        // Calculate the middle index
        int mid = (start + end) / 2;
        
        // Check if the search key is equal to the middle element
        if (array[mid] == key) {
            return mid;
		}
        // If the key is less than the middle element, set the ending index to the middle index - 1
        else if (key < array[mid]) {
            end = mid - 1;
		}
        // If the key is greater than the middle element, set the starting index to the middle index + 1
        else {
            start = mid + 1;
		}
    }
    // Return -1 if the item is not found in the array
    return -1;
}

Example Usage

The following is an example of how to use the binary search algorithm to search for an item in an integer array: 

          
           
// Create an int array
int[] array = {1, 2, 3, 4, 5};

// Search for the item with key 4
int key = 4;
int index = binarySearch(array, key);

// Print the index of the item (3)
System.out.println("Index of the item: " + index);
