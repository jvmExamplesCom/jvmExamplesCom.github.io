Performing Binary Search on Java double Array Example

Binary Search is a popular algorithm that can be used to search for an element in a sorted array. It works by repeatedly dividing the search interval in half and selecting the midpoint as the comparison value. The array must be sorted prior to the search being performed. Java provides an efficient implementation of this algorithm in the java.util.Arrays class. In this article, we will discuss the algorithm using a simple example of performing a binary search on a double array. We will also provide code examples which cover the basic concepts behind the algorithm and demonstrate its performance.

Example Double Array

For our example, we will use a simple double array containing the following values: 

          
           
double[] arr = {-4.3, 0.7, 9.1, 15.2, 33.4};
The elements are already sorted in ascending order, so no additional sorting is required.

Search Target Value

Next, we need to define a target value. For this example, let's assume the target value is 9.1.

Binary Search Algorithm Overview

The binary search algorithm works by comparing the target value to the midpoint of the search interval. If the target value is less than the midpoint, the search interval is reduced to the lower half. If the target value is greater than the midpoint, the search interval is reduced to the upper half. This process is repeated until the target value is found or the search interval is empty.

Java Code Example

Now that we understand the basics of the binary search algorithm, let's take a look at the actual Java code that implements it. First, we will define a helper method which searches for our target value in the given array. 

          
           
public static int binarySearch( double[] arr, double value ) 
{
    int low = 0;
    int high = arr.length - 1;
 
    while (low <= high) 
    {
        int mid = (low + high) / 2;
        if (arr[mid] == value) 
            return mid;
        else if (arr[mid] > value) 
            high = mid - 1;
        else 
            low = mid + 1;
    }
 
    return -1;
}
The binarySearch() method takes two parameters: the array and the target value. It then creates an initial search interval consisting of the entire array by setting the low and high variables to 0 and arr.length - 1 respectively. The while loop is used to iterate over the search interval until the target value is found or the search interval becomes empty. At each iteration, the mid point of the search interval is calculated and checked against the target value. If they are equal, the algorithm returns the mid point. If the mid point is greater than the target value, the search interval is reduced to the lower half. Otherwise, the search interval is reduced to the upper half.

Running the Example

Let's now try running our code example with the example array and target value set up earlier. The following code shows how to do this. 

          
           
int index = binarySearch( arr, 9.1 );
System.out.println( "Index of 9.1 is: " + index );
When we run the above code, it prints out the following result: 

          
           
Index of 9.1 is: 2
As expected, the target value is found in the index 2 of the array.
