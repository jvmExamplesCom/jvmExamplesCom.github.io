Performing Binary Search on Java long Array Example

Binary search is an efficient algorithm for finding an element in a sorted array. It works by repeatedly dividing the search interval in half until the desired element is found. This tutorial will show you how to implement a binary search in Java on a Long array.

Java Long Binary Search Algorithm

The following algorithm shows how to perform a binary search in Java on a Long array:


          
           
// start with the whole array
int left = 0;
int right = array.length - 1;
 
// keep splitting the array until the desired index is found
while (left <= right) {
    // determine the middle index
    int middle = (left + right) / 2;
    
    if (array[middle] == value) {
        return middle; 
    } else if (array[middle] < value) {
        // the value is greater than the middle index, so search the right side
        left = middle + 1;
    } else {
        // the value is less than the middle index, so search the left side
        right = middle - 1;
     }
}
 
// not found
return -1;

In this algorithm, we start off by narrowing down the search area by defining the left and right indices. We then enter a loop which will continue to iterate until the desired element is found or until the search area has been exhausted. After calculating the middle index, we compare the value at that index to the desired value. If they are equal, we can return the index. If the value at the middle index is less than the desired value, we need to search the right side of the array, so we move the left index to the middle index plus one. Otherwise, if the value at the middle index is greater than the desired value, we need to search the left side of the array, so we move the right index to one less than the middle index.
