Performing Binary Search on Java short Array Example

Binary search is an essential algorithm that is used in computer science to quickly find a particular value in an array or collections. In this tutorial, we will learn how to perform binary search on a Java Short Array example.

Java Binary Search Algorithm

The basic idea behind binary search is to compare the target value to the middle element of the array. If it matches, then the item has been found. Otherwise, the half in which the target cannot lie is eliminated, and the remaining half is searched again recursively until the target value is found. Here is the Java code for doing binary search: 

          
           
public static int binarySearch(Short[] arr, int l, int r, int x) 
{ 
   if (r >= l) { 
        int mid = l + (r - l) / 2; 
  
        // If the element is present at the 
        // middle itself 
        if (arr[mid] == x) 
            return mid; 
  
        // If element is smaller than mid, then 
        // it can only be present in left subarray 
        if (arr[mid] > x) 
            return binarySearch(arr, l, mid - 1, x); 
  
        // Else the element can only be present 
        // in right subarray 
        return binarySearch(arr, mid + 1, r, x); 
    } 
  
    // We reach here when element is not present 
    // in array 
    return -1; 
}
