Sorting ArrayList elements using Comparator in Java

Java provides several ways to sort the elements of an arraylist. One of the ways is to use a comparator. A comparator is an interface that defines a method called compare which compares two boolean values. By providing an implementation of this compare method, you can sort the elements of an arraylist based on your own logic.

Implementing Comparator interface

The following code example shows how to implement the Comparator interface for sorting ArrayList elements.


          
           
public class ArrayListUtils implements Comparator<Object> {
&nbsp; &nbsp; public int compare(Object obj1, Object obj2) {
&nbsp; &nbsp; &nbsp; &nbsp; //Your custom comparison logic goes here
&nbsp; &nbsp; }
}

Sorting ArrayList elements

Once the Comparator interface has been implemented, it can be passed to the Collections.sort() method. This method sorts the elements of the provided arraylist based on the logic provided by the comparator.


          
           
ArrayList<Object> list = new ArrayList<Object>();

//Add elements to the list

Collections.sort(list, new ArrayListUtils());

Example

Let's take a look at a simple example which sorts an arraylist of strings. In this example, we will sort the elements of the arraylist based on their lengths.


          
           
public class ArrayListExample {    
&nbsp; &nbsp; public static void main(String[] args) {    
&nbsp; &nbsp; &nbsp; &nbsp; ArrayList<String> list = new ArrayList<String>();    
&nbsp; &nbsp; &nbsp; &nbsp; list.add("C");    
&nbsp; &nbsp; &nbsp; &nbsp; list.add("Java");    
&nbsp; &nbsp; &nbsp; &nbsp; list.add("Python");    
&nbsp; &nbsp; &nbsp; &nbsp; list.add("Go");    
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;
&nbsp; &nbsp; //Comparator implementation to sort string based on their length    
&nbsp; &nbsp; &nbsp; &nbsp; Comparator<String> comparator = Comparator.comparingInt(String::length);    
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;
&nbsp; &nbsp; //Sorting the element of the list based on comparator    
&nbsp; &nbsp; &nbsp; &nbsp; Collections.sort(list, comparator);
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;
&nbsp; &nbsp; //Printing the sorted list    
&nbsp; &nbsp; &nbsp; &nbsp; System.out.println("Sorted List: " + list);    
&nbsp; &nbsp; }
}

When the above code is compiled and executed, it produces the following output:


          
           
Sorted List: [C, Go, Java, Python]
