Sort Java ArrayList in Descending Order using Comparator

Sorting a list is an important part of programming. In Java, you can use the Comparator interface to sort an ArrayList in descending order. This tutorial will show you how to sort an ArrayList in descending order using the Comparator interface.

Creating an ArrayList

First, let's create an ArrayList with some elements. We'll add some strings to the list for this example.


          
           
List<String> list = new ArrayList<>();
list.add("Bob");
list.add("Alice");
list.add("Tom");
list.add("John");
list.add("Liz");
list.add("Jill");

This code creates an ArrayList with six elements. The Strings are added in the order they appear in the code. The result is:


          
           
Bob Alice Tom John Liz Jill

Creating a Comparator

Now that we have an ArrayList, let's create a Comparator to sort the list in descending order. To do this, we must implement the Comparator interface and override its compare method. The compare method takes two arguments and returns an integer.


          
           
public class DescendingComparator implements Comparator<String> {
    @Override
    public int compare(String str1, String str2) {
        return str2.compareTo(str1);
    }
}

The compare method in this example compares two strings. It returns a positive number if the first string is "greater" than the second string, a negative number if the first string is "less than" the second string, and 0 if the strings are equal. This is the same behavior as the compareTo method of the String class.

Sorting the ArrayList

Now that we have an ArrayList and a Comparator, we can call the sort method of the Collections class to sort the list. The sort method takes a list and a Comparator as arguments.


          
           
Collections.sort(list, new DescendingComparator());

This code uses our DescendingComparator to sort the list. The result is an ArrayList that is sorted in descending order:


          
           
Jill Liz John Tom Alice Bob

And that's it! We have successfully sorted an ArrayList in descending order using the Comparator interface.
