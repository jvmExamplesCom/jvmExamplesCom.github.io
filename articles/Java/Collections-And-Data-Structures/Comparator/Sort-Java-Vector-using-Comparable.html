Sort Java Vector using Comparable

Java Vector provides a powerful way to store and manipulate a list of objects. By using the Comparable interface, it is possible to sort the contents of the vector according to any criteria. This article will demonstrate how to properly implement the Comparable interface in order to sort a Vector using a specific criteria.

Implementing Comparable

In order to sort Java Vector using Comparable, the elements of the vector must implement the Comparable interface. To do this, the class for the type of object being stored in the vector must implement the Comparable interface. The compareTo() method must be declared as public and override the Object class compareTo() method. The code for a basic implementation of the compareTo() method looks like this: 

          
           
public int compareTo(Object o) {
    //Your code here
}
The code above will allow Vector sorting using Comparator. Now let's look at how to write an actual comparison logic.

Using Comparison Logic

In order to compare two objects stored in a Vector, the compareTo() method needs to define a comparison logic. This logic can be anything you choose, such as comparing object properties, or performing some calculation on them. For example, let's say we have a Vector of Person objects, and we want to sort them by age. We could implement the compareTo() method like this: 

          
           
public int compareTo(Object o) {
    Person p = (Person) o;
    if (this.age == p.age)
        return 0;
    else if (this.age > p.age)
        return 1;
    else 
        return -1;
}
This comparison logic compares the age of the current Person object (this) with the age of the Person object passed as a parameter (p). If the age of both objects is equal, then 0 is returned, indicating that they are equal. Otherwise, if the age of the current Person object is greater than the age of the parameter Person, then 1 is returned, indicating that the current Person should come after the parameter Person in the sorted vector. Otherwise, if the age of the current Person object is less than the age of the parameter Person, then -1 is returned, indicating that the current Person should come before the parameter Person in the sorted vector.

Using Vector.sort()

public static void sort(Vector v)
Where v is the Vector which we want to sort. For example, if we have a Vector of Person objects, we can sort it like this:

The above code will sort the Vector of Person objects according to the comparison logic defined in the compareTo() method.

Conclusion
