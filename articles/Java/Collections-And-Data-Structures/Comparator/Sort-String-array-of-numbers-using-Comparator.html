Sorting a String Array of Numbers Using Comparator in Java

Java provides various ways to order objects of our own type or the provided ones. We can order objects of our own data type by implementing 

Comparable
or 
Comparator
interfaces in our custom classes. In this article, we will discuss how to sort a String array of numbers using 
Comparator
.

The Comparator Interface

The Comparator interface defines two methods: 
compare()
and 
equals()
. The 
compare()
method is used for comparing two objects for sorting whereas the 
equals()
method is used for checking if two objects are equal.

Example Program:


          
           
import java.util.Arrays;
import java.util.Comparator;
 
public class SortStringArray {
   public static void main(String[] args) {
      String[] numbers = {"3", "2", "1", "5", "4"};
      Arrays.sort(numbers, new Comparator<String>() {
         @Override
         public int compare(String s1, String s2) {
            return Integer.compare(Integer.parseInt(s1), Integer.parseInt(s2));
         }
      }); 
      System.out.println(Arrays.toString(numbers));
   }
}

In the above program, we have an unsorted array of strings 

numbers
. In each string, there is a number stored as a string. We use the 
Arrays.sort()
method to sort the array. The 
sort()
method takes two parameters. The first one is the array which needs to be sorted and the second one is a 
Comparator
object. To create a 
Comparator
object, we have created an anonymous class with a 
compare()
method. The 
compare()
method takes two strings as parameters and converts them into integers before comparing them. The output of the above program is: 

          
           
[1, 2, 3, 4, 5]
