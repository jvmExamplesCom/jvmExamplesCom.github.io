Sorting HashMap by Key Using Comparator in Java

HashMap is a generic class that implements Map interface and is used to store key-value pairs. It represents data in a key-value pair format which allows users to store, insert and access data easily. In this article, we will be discussing the process of sorting a HashMap by its keys using Comparator.

Comparator Interface in Java

The Comparator interface in Java is used to impose specific ordering on a collection of objects, such as when sorting them. It provides multiple methods for that purpose, such as compare(), equals(), etc. To sort HashMap by its keys using Comparator, we will first define our own class that implements the Comparator interface and then override the compare () method to compare two keys according to our desired order.

Creating a Comparator Class

We will create a class named KeyComparator that implements the Comparator interface and overrides the compare() method in order to compare keys according to our desired order. This class will take two arguments as input, one representing the first key and the other representing the second key. The compare() method returns an integer value which is negative, zero or positive based on the result of the comparison. 

          
           
public class KeyComparator implements Comparator<Object> {
 
    public int compare(Object k1, Object k2) {
        ...
    } 
}

Sorting the HashMap

Once the comparator class is created, we can now use it to sort the HashMap by its keys. The TreeMap class in Java is used to store key-value pairs in a sorted manner. We can pass an instance of the KeyComparator class to it while creating a TreeMap object. This will ensure that the TreeMap is sorted according to the order defined in the comparator class. We can then iterate over the TreeMap object in order to get the sorted list of keys and values. 

          
           
HashMap<Object, Object> hm = new HashMap<Object, Object>();
// put elements to the HashMap

TreeMap<Object, Object> sortedMap = new TreeMap<Object, Object>(new KeyComparator());
sortedMap.putAll(hm);

// iterate over the TreeMap object in order to get the sorted list
for (Map.Entry<Object, Object> entry : sortedMap.entrySet()) { 
    System.out.println("Key = " + entry.getKey() + 
                       ", Value = " + entry.getValue()); 
}

Example

In this example, we are sorting a HashMap containing Integer keys and String values. We create our Comparator class and implement the compare () method to compare two integers and return a negative, zero or positive integer depending on the result. We then pass this Comparator object to a TreeMap while creating it in order to ensure that the TreeMap is sorted according to the order specified in the Comparator class. 

          
           
// Comparator class to sort the HashMap by keys
public class KeyComparator implements Comparator<Object> {
 
    public int compare(Object k1, Object k2) {
        Integer i1 = (Integer) k1;
        Integer i2 = (Integer) k2;
        return i1.compareTo(i2);
    } 
}

// main method
public static void main(String[] args) {

    HashMap<Integer, String> hm = new HashMap<Integer, String>();

    // put elements to the HashMap
    hm.put(3, "Three");
    hm.put(2, "Two");
    hm.put(1, "One");
    hm.put(4, "Four");

    TreeMap<Integer, String> sortedMap = new TreeMap<Integer, String>(new KeyComparator());
    sortedMap.putAll(hm);

    // iterate over the TreeMap object in order to get the sorted list
    for (Map.Entry<Integer, String> entry : sortedMap.entrySet()) { 
        System.out.println("Key = " + entry.getKey() + 
                           ", Value = " + entry.getValue()); 
    } 
}

Output

When the above code is executed, it produces the following output: 

          
           
Key = 1, Value = One
Key = 2, Value = Two
Key = 3, Value = Three
Key = 4, Value = Four
As we can see, the HashMap is now sorted according to the keys.
