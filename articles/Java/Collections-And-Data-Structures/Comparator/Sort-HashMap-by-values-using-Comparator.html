Sorting HashMap by Values using Comparator in Java

One of the most common tasks a programmer will need to do in Java is sorting a 

          
           
HashMap
by values. This is often required when a data structure needs to be presented in a particular order. In this tutorial, we will explore how to sort a 

          
           
HashMap
by its values using a 

          
           
Comparator
in Java.

Create a Comparator for Sorting HashMap

Before we can sort a 

          
           
HashMap
by its values, we must first create a 

          
           
Comparator
that can compare two different entries from the map and return an indication of which entry should precede the other. The following example shows how to create a 

          
           
Comparator
for a 

          
           
HashMap
of 

          
           
String
keys and Integer values. 

          
           
public class MyComparator implements Comparator<String> {
    HashMap<String, Integer> map = new HashMap<String, Integer>();

    public MyComparator(HashMap<String, Integer> map) {
        this.map.putAll(map);
    }

    @Override
    public int compare(String s1, String s2) {
        if(map.get(s1) <= map.get(s2)){
            return -1;
        }else{
            return 1;
        }
    }
}

Sort HashMap by Values with Comparator

Now that we have a 

          
           
Comparator
for our 

          
           
HashMap
, we can use it to sort the map by its values. The following example shows how to use the 

          
           
MyComparator
class to sort a 

          
           
HashMap
of 

          
           
String
keys and Integer values. 

          
           
HashMap<String, Integer> map = new HashMap<String, Integer>();
map.put("A", 5);
map.put("B", 2);
map.put("C", 10);
map.put("D", 1);

Map<String, Integer> sortedMap = new TreeMap<String, Integer>(new MyComparator(map));
sortedMap.putAll(map);

for (Map.Entry<String, Integer> entry : sortedMap.entrySet()) {
    System.out.println(entry.getKey() + " : " + entry.getValue());
}
The output of the above code would be: 

          
           
D : 1
B : 2
A : 5
C : 10
As you can see, the 

          
           
HashMap
has been sorted by its values in ascending order.
