Sort String Array Using Comparator

Java provides the 

          
           
java.util.Arrays.sort()
method for sorting a string array, which uses either natural ordering or an optional comparator for ordering. In this article, we will look at how to sort a string array using an implementation of the 

          
           
java.util.Comparator
interface.

Creating a Comparator

The first step is creating an implementation of the 

          
           
java.util.Comparator
interface. This interface defines a single method, 

          
           
compare(Object o1, Object o2)
, which takes two objects and returns an int based on their comparison. We will use this to compare two strings in our array sorting example. Here is an example of a simple Comparator implementation that compares strings in natural order: 

          
           
public class StringComparator implements Comparator<String> {
    public int compare(String o1, String o2) {
        return o1.compareTo(o2);
    }
}

Sorting the String Array

Now that we have a Comparator implementation, we can use it to sort our string array. We will use the 

          
           
java.util.Arrays.sort()
method for this task. Here is an example: 

          
           
String[] data = new String[]{"foo", "bar", "baz"};
Arrays.sort(data, new StringComparator());
This will sort the array in-place using the provided Comparator. The output would be the following array: 

          
           
["bar", "baz", "foo"]

Conclusion

In this article, we looked at how to sort a string array using a simple Comparator implementation in Java. This method is easy to use and can be used to sort any array of objects with a custom Comparator.
