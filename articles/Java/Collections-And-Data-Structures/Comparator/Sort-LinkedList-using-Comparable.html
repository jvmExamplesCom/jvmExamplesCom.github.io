Sorting a LinkedList using Comparable in Java

A LinkedList is one of the most commonly used data structures in Java. It allows for efficient insertion and deletion of data elements, and is used for collections that require frequent changes. An important task in working with a LinkedList is sorting the elements it contains. The standard way of doing this is to use the Comparable interface, which provides a compareTo() method that can be used to determine the relative order of two objects. In this article we will look at how to use the Comparable interface to sort a LinkedList in Java.

Creating a LinkedList and Adding Elements

The first step in sorting our LinkedList is to create it, along with some elements to sort. We will do this by creating a class called Person, which will contain a name and age. Our Person objects will be added to the LinkedList, and sorted according to their age. 

          
           
public class Person {
    private String name;
    private int age;
    // constructors, getters, setters
}
Next, we will create the LinkedList and add several Person objects to it: 

          
           
LinkedList<Person> people = new LinkedList<>();
people.add(new Person("John", 25));
people.add(new Person("Jane", 32));
people.add(new Person("Bob", 19));
people.add(new Person("Alice", 27));

Implementing the Comparable Interface

In order to sort the LinkedList, we need to implement the Comparable interface on the Person class. The Comparable interface requires us to provide an implementation of the compareTo() method, which compares two objects and returns an integer indicating their relative order. In our case, we want to compare two Person objects by their age. We can do this by comparing the ages of the two objects, and returning the result as an integer. If the first object's age is greater than the second, the compareTo() method should return a positive number. If the second object's age is greater, the method should return a negative number. If the ages are equal, the return value should be 0. 

          
           
public class Person implements Comparable<Person> {
    private String name;
    private int age;
    
    @Override
    public int compareTo(Person o) {
        return Integer.compare(this.age, o.age);
    }
    // constructors, getters, setters
}

Sorting the LinkedList

Now that our Person objects can be compared and ordered, we can sort the linked list using the Collections.sort() method. All we need to do is pass the LinkedList to the method, and it will automatically sort the elements using our compareTo() method. 

          
           
Collections.sort(people);
After sorting, the LinkedList will contain the Person objects in ascending order of age. We can prove this by printing out the Person objects in the list. 

          
           
for(Person p : people) {
    System.out.println(p.getName() + ", " + p.getAge());
}
This will print out the following: 

          
           
Bob, 19
John, 25
Alice, 27
Jane, 32
As we can see, the LinkedList is now sorted according to the age of the Person objects.

Conclusion

In this article we looked at how to use the Comparable interface to sort a LinkedList in Java. We created a Person class, implemented the compareTo() method, and used the Collections.sort() method to sort the elements. With this approach, we can easily sort a LinkedList with any type of object.
