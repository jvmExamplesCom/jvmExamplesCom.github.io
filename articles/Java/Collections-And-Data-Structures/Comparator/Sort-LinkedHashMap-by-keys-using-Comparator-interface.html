Sorting LinkedHashMap by Keys Using Comparator Interface

The Java language provides a powerful way to sort a collection of objects like LinkedHashMap using the Comparator interface. The Comparator interface allows you to modify the sorting order of objects for a specific type within a class or collection. The Comparator interface can be used to sort LinkedHashMap according to its 

          
           
key
values. In this article, we’ll discuss how to use the Comparator interface to sort LinkedHashMap by keys in Java.

Creating a Comparator

Before we can sort a LinkedHashMap according to its keys, we need to create a Comparator that will provide the logic and rules for the sorting. To do this, we’ll create an anonymous inner class that implements the Comparator interface and override its compare() method. 

          
           
Comparator<String> comparator = new Comparator<String>() {
@Override
public int compare(String key1, String key2) {
    // implementation here…
}
};
In this example, we’re creating a Comparator that will sort 

          
           
String
objects. The compare() method will accept two Strings, compare them, and return an 

          
           
int
value determining if one is smaller, equal to, or greater than the other.

Sorting LinkedHashMap by Keys

Now that we have our Comparator, we’re ready to sort the LinkedHashMap by its keys. To do this, we’ll use the Collections.sort() Method, passing in the LinkedHashMap’s 

          
           
keySet()
method and the Comparator we created earlier. 

          
           
Collections.sort(linkedHashMap.keySet(), comparator);
The sorting of the LinkedHashMap will happen in-place and the Keys will remain in the same order they were before the sorting.

Example

Let’s look at a complete example of how to sort a LinkedHashMap by its keys using the Comparator Interface in Java. In this example, we’re creating a simple LinkedHashMap of 

          
           
String
objects with their corresponding 

          
           
Integer
values. We’re then creating a Comparator that will sort the LinkedHashMap alphabetically according to its keys. Finally, we’re using the Collections.sort() method to sort the LinkedHashMap by its keys. 

          
           
// Create a new LinkedHashMap
LinkedHashMap<String, Integer> linkedHashMap = new LinkedHashMap<>();

// Add some elements to the LinkedHashMap
linkedHashMap.put("John", 21);
linkedHashMap.put("David", 25);
linkedHashMap.put("Alice", 27);

// Create a Comparator to sort the LinkedHashMap by its keys
Comparator<String> comparator = new Comparator<String>() {
@Override
public int compare(String key1, String key2) {
    return key1.compareTo(key2);
}
};

// Sort the LinkedHashMap
Collections.sort(linkedHashMap.keySet(), comparator);

// Output the sorted LinkedHashMap
for (Map.Entry<String, Integer> entry : linkedHashMap.entrySet())
{
    System.out.println(entry.getKey() + ": " + entry.getValue());
}
The output of this code will be: 

          
           
Alice: 27
David: 25
John: 21
As you can see, the LinkedHashMap was successfully sorted by its keys in alphabetical order.
