Sorting LinkedHashMap by Values using Comparable Interface

LinkedHashMaps are a type of Java Collection that store key-value pairs, which can easily be sorted according to some criteria. In this article, we will look at how to sort a LinkedHashMap by values using the Comparable interface in Java.

What is the Comparable Interface?

The Comparable interface is a Java interface that allows us to compare two objects, and determine which one is "greater" than the other. It is used to define the natural ordering of elements in a collection, and it is commonly used with collections like List, Set, and Map. The Comparable interface defines the compareTo() method, which takes an Object as an argument and returns an integer indicating the result of the comparison. If the result is 0, then the two objects are equal; if the result is less than 0, then the first object is smaller than the second object; and if the result is greater than 0, then the first object is larger than the second object.

Using the Comparator Interface to Sort a LinkedHashMap by Values

We can use the Comparator interface in Java to sort a LinkedHashMap by values. The Comparator interface takes two arguments and returns an integer indicating the result of the comparison. We will create a Comparator class which implements the Comparator interface, and override its compare() method. The compare() method takes two objects as arguments, and returns an integer indicating the result of the comparison. We will use this compare() method to compare two values in the LinkedHashMap. If the result is 0, then the two values are equal; if the result is less than 0, then the first value is smaller than the second value; and if the result is greater than 0, then the first value is larger than the second value. Once we have created the Comparator class, we can then use it to sort the LinkedHashMap. We will create a new TreeMap, and pass the Comparator instance to its constructor. We can then loop through the entries in the LinkedHashMap, and add them to the TreeMap. The TreeMap will automatically sort the entries according to the Comparator that we specified.

Example Code

Here is an example of how to use the Comparator interface to sort a LinkedHashMap by values: 

          
           
import java.util.*;

public class MyComparator implements Comparator<Integer, Integer> {

    public int compare(Integer key1, Integer key2) {
        return key1.compareTo(key2);
    }

}

public static void main(String[] args) {

    // Create the LinkedHashMap
    LinkedHashMap<Integer, Integer> map = new LinkedHashMap<>();
    map.put(1, 10);
    map.put(2, 5);
    map.put(3, 15);

    // Create the Comparator instance
    Comparator<Integer, Integer> comp = new MyComparator();

    // Create the TreeMap
    TreeMap<Integer, Integer> sortedMap = new TreeMap<>(comp);

    // Add the entries to the TreeMap
    for (Map.Entry<Integer, Integer> entry : map.entrySet()) {
        sortedMap.put(entry.getKey(), entry.getValue());
    }

    // Print the sorted map
    System.out.println(sortedMap);

}
The output of the above code would be: 

          
           
{2=5, 1=10, 3=15}
As you can see, the LinkedHashMap has been sorted according to the Comparator instance that we created.

Conclusion

In this article, we looked at how to sort a LinkedHashMap by values using the Comparable interface in Java. We created a Comparator class which implements the Comparator interface, and used it to sort the LinkedHashMap. We then added the entries to a TreeMap, which, thanks to our Comparator instance, sorted the entries according to the provided criteria.
