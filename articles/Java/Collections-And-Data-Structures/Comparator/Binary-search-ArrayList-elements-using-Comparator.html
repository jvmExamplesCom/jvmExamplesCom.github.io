Binary Search ArrayList Elements using Comparator

The binary search is an efficient way to search through a sorted array, and is particularly useful when trying to find a specific element within an ArrayList in Java. Binary search is a fast search algorithm that finds a given value in an array or ArrayList by comparing the middle element of the array to the given value. If the middle element is equal to the value, the index is returned. Otherwise, if the value is larger than the middle element then the search will continue in the upper half of the array, and if the value is smaller than the middle element, the lower half of the array will be searched. To efficiently search an ArrayList in Java, we can use the Comparator interface to define a sorting order for the elements in the list.

Example Code


          
           
public static int binarySearch(ArrayList <Integer> list, int key) {

    int low = 0;
    int high = list.size() - 1;

    while (low <= high) {
        int mid = (low + high) / 2;
        int midVal = list.get(mid); 

        if (midVal < key) 
            low = mid + 1;
        else if (midVal > key) 
            high = mid - 1;
        else
            return mid; // key found
    }
    return -1;  // key not found
}

This code example shows how to perform a binary search on an ArrayList in Java. The code creates two variables, low and high, that store the lower and upper boundaries of the range of elements being searched. It then enters a loop and obtains the midpoint value of the list using the mid variable. The midVal variable stores the element at the midpoint of the list, which is compared to the value being searched for (key). If the midVal is less than the key, the low value is increased such that the midpoint is now in the upper half of the list, and the search will be done in the upper half of the list. If the midVal is greater than the key, the search is done in the lower half of the list. Finally, if the midVal is equal to the key, the index of the element is returned. Otherwise, the search returns -1, indicating that the key was not found.
