How to Sort TreeSet Elements using Comparator Interface

Sorting TreeSet elements using the Comparator interface allows you to control the order in which objects are stored and retrieved from a TreeSet. This is done by implementing a custom comparator in place to create a sort ordering that follows specific rules. It also allows you to compare different types of objects, such as String or Integer, etc. In this tutorial, we'll show you how to sort TreeSet elements using the Comparator interface and the TreeSet class.

TreeSet

The TreeSet class in Java is a member of the collection framework. It implements the NavigableSet interface and extends AbstractSet class. Internally, a TreeSet uses a tree structure to store its elements. It allows you to store and retrieve elements in an ascending order. The TreeSet class does not support duplicate elements. If you try to add a duplicate element in the TreeSet, it wonâ€™t be added and the add() method will return false.

Comparator

The Comparator interface is used for sorting the objects of a user-defined class. The Comparator interface contains a single method compare(Object obj1, Object obj2). This method usually returns an integer value indicating the order of two objects. Here, we're going to use this interface to sort TreeSet elements.

Example 1: Sorting TreeSet Elements in Ascending Order

In this example, we're going to sort the TreeSet elements in ascending order. We'll use the TreeSet class and the compare() method of the Comparator interface to achieve this. We'll create a simple class named Student with three attributes and their getter methods. 

          
           
public class Student {  
   int rollno;  
   String name;  
   int age;  
  
   Student(int rollno, String name, int age) {  
   this.rollno = rollno;  
   this.name = name;  
   this.age = age;  
   }  
  
   public int getRollno() {  
   return rollno;  
   }  
  
   public String getName() {  
   return name;  
   }  
  
   public int getAge() {  
   return age;  
   }  
}
Next, we'll create a Comparator that defines an order that we want to use in our sorting. 

          
           
class SortByName implements Comparator<Student> {  
   public int compare(Student a, Student b) {  
   return a.getName().compareTo(b.getName());  
   }  
}
Now, we'll create a TreeSet and add a few Student objects. 

          
           
TreeSet<Student> set=new TreeSet<Student>(new SortByName());  

set.add(new Student(101,"John",20));  
set.add(new Student(102,"Jerry",21));  
set.add(new Student(103,"Devin",20));  
set.add(new Student(104,"James",22));
Finally, we'll print out the sorted set elements. 

          
           
System.out.println("Sorted Set Elements");  
for(Student st:set){  
System.out.println(st.getName()+" "+st.getRollno()+" "+st.getAge());  
}  
  
// Output  
Sorted Set Elements  
Devin 103 20  
James 104 22  
John 101 20  
Jerry 102 21
