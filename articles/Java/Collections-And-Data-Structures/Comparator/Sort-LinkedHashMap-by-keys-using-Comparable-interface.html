Sort LinkedHashMap by Keys Using Comparable Interface

LinkedHashMap is a type of hashmap that maintains insertion order of the elements. It is a common use case to sort the LinkedHashMap keys according to some order. This can be achieved by implementing the Comparable interface. In this tutorial, we will learn how to sort LinkedHashMap by its keys using Comparable interface.

Comparable Interface

The Comparable interface is an in-built interface present in the java.lang package. This interface defines the default single-parameter compareTo() method which compares the current object with another object of the same type. It returns a negative number if the current object is less than the other object, zero if the objects are equal and a positive number if the current object is greater than the other object. The Comparable interface must be implemented by the type whose objects are to be compared. The implementation provides the sorting logic and the objects can then be sorted using the Collections.sort() method.

Example 1

Let us understand this concept with a simple example. The following example sorts a LinkedHashMap of strings according to their length. First, we define a custom Comparator class StringLengthComparator which implements the Comparable interface and provides a comparison logic based on the length of the strings. 

          
           
class StringLengthComparator implements Comparator<String> {
    @Override
    public int compare(String a, String b) {
        return a.length() - b.length();
    }
}
Next, we define a LinkedHashMap containing country and its capital cities. 

          
           
LinkedHashMap<String, String> map = new LinkedHashMap<>();
map.put("India", "Delhi");
map.put("United States", "Washington DC");
map.put("United Kingdom", "London");
map.put("Germany", "Berlin");
Finally, we sort the LinkedHashMap by passing the StringLengthComparator to the sort() method of the Collections class. 

          
           
Collections.sort(map.entrySet(), new StringLengthComparator());
System.out.println(map);
The output will look like this. 

          
           
{Germany=Berlin, India=Delhi, United Kingdom=London, United States=Washington DC}

Example 2

Let us take a more complex example where we have to sort LinkedHashMap of integers according to their absolute values. We define an IntegerAbsoluteValueComparator class which implements the Comparable interface and provides a comparison logic based on the absolute value of the integers. 

          
           
class IntegerAbsoluteValueComparator implements Comparator<Integer> {
    @Override
    public int compare(Integer a, Integer b) {
        return Math.abs(a) - Math.abs(b);
    }
}
Next, we define a LinkedHashMap containing Country Code and Country Name. 

          
           
LinkedHashMap<Integer, String> map = new LinkedHashMap<>();
map.put(12, "France");
map.put(-44, "United Kingdom");
map.put(98, "India");
map.put(-1, "USA");
Finally, we sort the LinkedHashMap by passing the IntegerAbsoluteValueComparator to the sort() method of the Collections class. 

          
           
Collections.sort(map.entrySet(), new IntegerAbsoluteValueComparator());
System.out.println(map);
The output will look like this. 

          
           
{-1=USA, 12=France, -44=United Kingdom, 98=India}
In this tutorial, we learned how to sort LinkedHashMap by its keys using the Comparable interface.
