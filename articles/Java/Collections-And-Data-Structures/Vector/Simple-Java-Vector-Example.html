Simple Java Vector Example

Introduction

Java's Vector class is a particularly useful tool for adding, looking up, and manipulating objects stored in dynamic data structures. In this article, we'll cover the basics of the Vector class, as well as a simple example of how to use it.

What is a Vector?

A Vector is a type of data structure that functions similarly to an array, but is more flexible. Unlike arrays, Vectors can grow and shrink automatically as new elements are added or removed. This makes them a great choice for situations where you don't always know how many elements will be in the data structure beforehand. Vectors are also synchronized, meaning multiple threads can access the same Vector without any issues. As such, they are a great choice for storing data and objects that will be accessed by multiple threads.

Creating a Vector

To create a Vector in Java, you'll need to use the built-in Vector class. This class provides several constructors which allow you to create empty Vectors, preallocate space, or even create Vectors from existing collections. For our example, we'll create a simple empty Vector like this: 

          
           
Vector<String> myVector = new Vector<String>();
Here, we've created a Vector of type String, with no elements in it yet.

Adding Elements to a Vector

To add elements to a Vector, you can use the add() method. This method accepts an object which will be added to the end of the Vector. Let's add some elements to our example Vector: 

          
           
myVector.add("Hello");
myVector.add("World");
Now our Vector contains two String objects – "Hello" and "World".

Accessing Elements in a Vector

As with arrays, the elements in a Vector can be accessed using their index position. The index starts at 0, so the first element in the Vector can be accessed with an index of 0, the second with an index of 1, and so on. To access an element in a Vector, you'll need to use the get() method. For example, if we wanted to access the first element of our Vector: 

          
           
String firstElement = myVector.get(0);
In this case, the value of the firstElement variable would be "Hello".

Iteration Over Elements in a Vector

Iterating over elements in a Vector is similar to iterating over an array. You'll need to use a loop to access each element in turn. The Vector class provides two methods which allow you to quickly iterate over all elements – forEach() and forEachRemaining(). An example of iteration over our Vector: 

          
           
for (String element : myVector) {
    System.out.println(element);
}
This code will output the elements of our Vector one at a time, printing "Hello" followed by "World".

Conclusion

Java's Vector class provides a powerful and flexible way of creating and manipulating data structures which grow and shrink automatically. In this article, we've covered the basics of Vectors and provided a simple example of how to use them.
