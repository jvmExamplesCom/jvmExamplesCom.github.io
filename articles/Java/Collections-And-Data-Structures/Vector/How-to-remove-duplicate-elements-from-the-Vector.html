How to Remove Duplicate Elements from the Vector

In Java, it is possible to remove duplicate elements from a Vector using the removeAll() method. This method takes an argument that specifies which elements will be removed from the Vector. In this article, we will discuss how to remove duplicate elements from a Vector and provide several code examples to illustrate the process.

Example 1 - Using the removeAll() Method

The following code example shows how to use the removeAll() method to remove duplicate elements from a Vector.


          
           
Vector<String> myVector = new Vector<String>();
myVector.add("Hello");
myVector.add("World");
myVector.add("Hello");

//Remove duplicates
myVector.removeAll(Collections.singleton("Hello"));

System.out.println(myVector);
The output of this code is as follows: 

          
           
[World]

Example 2 - Using the HashSet Collection

The following code example shows how to use the HashSet collection to remove duplicate elements from a Vector.


          
           
Vector<String> myVector = new Vector<String>();
myVector.add("Hello");
myVector.add("World");
myVector.add("Hello");

//Create a HashSet to hold unique values
HashSet<String> mySet = new HashSet<String>(myVector);

//Replace Vector with the HashSet
myVector = new Vector<String>(mySet);

System.out.println(myVector);
The output of this code is as follows: 

          
           
[Hello, World]

Example 3 - Using the LinkedHashSet Collection

The following code example shows how to use the LinkedHashSet collection to remove duplicate elements from a Vector.


          
           
Vector<String> myVector = new Vector<String>();
myVector.add("Hello");
myVector.add("World");
myVector.add("Hello");

//Create a LinkedHashSet to maintain order
LinkedHashSet<String> mySet = new LinkedHashSet<String>(myVector);

//Replace Vector with the LinkedHashSet

System.out.println(myVector);

[Hello, World]

Example 4 - Using the ListIterator Interface

The following code example shows how to use the ListIterator interface to remove duplicate elements from a Vector.


          
           
Vector<String> myVector = new Vector<String>();
myVector.add("Hello");
myVector.add("World");
myVector.add("Hello");

//Iterate through the Vector
ListIterator<String> iter = myVector.listIterator();

//Store previously seen elements in a Set
HashSet<String> seenElements = new HashSet<String>();

//Remove any duplicate elements
while (iter.hasNext()) {
    String element = iter.next();
    if (seenElements.contains(element)) {
        iter.remove();
    } else {
        seenElements.add(element);
    }
}

System.out.println(myVector);
The output of this code is as follows: 

          
           
[Hello, World]
