How to Find an Element in the Vector Using indexOf and lastIndexOf Methods

In a Java program, there are certain times when you will want to search through a Vector to find an element that meets certain criteria. You can do this by using the indexOf and lastIndexOf methods. They will both return the index of the element found, or a -1 if it is not found. These methods can also be used to differentiate between two similar elements.

Example 1 – Search for a String

In this example, we will use indexOf to find a String element in a Vector of Strings. We will first create the Vector and fill it with some Strings.


          
           
Vector<String> vector = new Vector<>();
vector.add("Hello");
vector.add("World");
vector.add("!");

Now, we will create a variable to store the index of the element we are looking for.


          
           
int index = 0;
Finally, we will use the 
indexOf
method to search for the element and store the index in the variable. 

          
           
index = vector.indexOf("World");

If the element is found, the index will be stored in the 

index
variable. If the element is not found, the index will be set to -1.

Example 2 – Differentiate Two Similar Elements

In this example, we will use the 
lastIndexOf
method to differentiate between two elements that are identical but appear at different positions in the Vector. We will first create another Vector containing two identical Strings. 

          
           
Vector<String> vector = new Vector<>();
vector.add("Hello");
vector.add("World");
vector.add("!");
vector.add("World");

Now, we will create a variable to store the index of the elements we are looking for.


          
           
int index1 = 0;
int index2 = 0;
Finally, we will use the 
lastIndexOf
method to search for the elements and store the indices in the variables. 

          
           
index1 = vector.indexOf("World");
index2 = vector.lastIndexOf("World");
The 
index1
variable will contain the index of the first occurrence of "World", while the 
index2
variable will contain the index of the second occurrence. In this way, you can differentiate between two identical elements in a Vector.

Conclusion

In this article, we have discussed how to use the indexOf and lastIndexOf methods to find elements in a Vector and differentiate between two similar elements. By using these methods, you can easily locate elements and access their indices in a Vector.
