Iterate Through Elements of Java Vector Using ListIterator Example

It is possible to iterate through the elements of a Java Vector using a ListIterator object. This tutorial provides example code to demonstrate how this can be accomplished. By the end of this tutorial, you will have a working example that you can use and modify in your own applications.

Creating a Sample Vector

The first step is to create a Vector object with some sample elements. The following code creates a Vector containing three Strings:


          
           
Vector<String> vec = new Vector<String>();
vec.add("foo");
vec.add("bar");
vec.add("baz");

Getting a ListIterator Object

Once the Vector has been created, we can get a ListIterator object from it by calling the 

listIterator()
method. The following code shows how this is done: 

          
           
ListIterator<String> iter = vec.listIterator();

Using ListIterator to Iterate Through the Vector

Now that we have a ListIterator object, we can use it to iterate through the Vector. The 

hasNext()
method returns 
true
if there are more elements in the Vector, and the 
next()
method returns the next element in the Vector. The following code shows how this is done: 

          
           
while (iter.hasNext()) {
    String s = iter.next();
    System.out.println(s);
}

This code will print out each of the elements in the Vector, as follows:


          
           
foo
bar
baz

Conclusion

In this tutorial, we have seen how to iterate through the elements of a Java Vector using a ListIterator object.
