Finding Minimum and Maximum Elements from a Vector in Java

In this article, we discuss how to find the minimum and maximum elements from a vector in Java. We'll look at some code examples to help explain this task. A Vector is a data structure that holds an ordered list of elements. It is important to know how to find the minimum and maximum elements from such vectors as this can be useful for sorting or counting certain types of information.

Finding the Minimum and Maximum Element

In order to find the minimum and maximum elements from a Vector, we need to iterate through the Vector and compare each element to the other elements. The one with the smallest value will be the minimum, and the one with the greatest value will be the maximum. An example of this in action is shown below: 

          
           
Vector<Integer> v = new Vector<Integer>();
v.add(3);
v.add(2);
v.add(1);
v.add(4);

int max = Integer.MIN_VALUE;
int min = Integer.MAX_VALUE;

for (int i : v) {
    if (i > max) max = i;
    if (i < min) min = i;
}

System.out.println("Max: " + max + ", Min: " + min);
In this example, we create a vector of integers, then use a for loop to iterate through each integer. After comparing each element to the current maximum and minimum values, the maximum and minimum are updated accordingly. The output for this example is: 

          
           
Max: 4, Min: 1

Using Java 8 Streams

Since Java 8, it is also possible to use streams to find the minimum and maximum elements from a Vector. This is a much simpler approach than the one seen above. An example of using streams to do this is shown below: 

          
           
List<Integer> v = Arrays.asList(3, 2, 1, 4);

int max = v.stream().mapToInt(x -> x).max().getAsInt();
int min = v.stream().mapToInt(x -> x).min().getAsInt();

System.out.println("Max: " + max + ", Min: " + min);
This example produces the same output as before: 

          
           
Max: 4, Min: 1

Conclusion

In this article, we looked at how to find the minimum and maximum elements from a Vector in Java. We saw two different approaches: one using a for loop and one using Java 8 Streams. Both produced the same result, but the Streams approach was much simpler.
