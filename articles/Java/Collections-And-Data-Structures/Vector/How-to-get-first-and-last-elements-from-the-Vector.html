How to Get First and Last Elements from a Vector in Java

Java provides the Vector class which is a type of collection that stores elements in a dynamic array. This class allows you to quickly and easily retrieve the first and last elements from a vector. In this article, we will walk through some examples of how to get the first and last elements from a vector.

Retrieving the First Element from a Vector

The easiest way to retrieve the first element from a vector is by using the get() method. This method takes an index as an argument and will return the element at that index. To get the first element in the vector, pass in an index of 0. For example, consider a vector containing the integers 1, 2, 3 and 4. To retrieve the first element (1) from the vector, use the following code: 

          
           
Vector<Integer> v = new Vector<>();
v.add(1);
v.add(2);
v.add(3);
v.add(4);

// Retrieve the first element
int firstElement = v.get(0);

System.out.println(firstElement);
// Outputs 1

Retrieving the Last Element from a Vector

The easiest way to retrieve the last element from a vector is by using the lastElement() method. This method does not take any arguments and will return the last element from the vector. For example, consider the same vector containing the integers 1, 2, 3 and 4. To retrieve the last element (4) from the vector, use the following code: 

          
           
Vector<Integer> v = new Vector<>();
v.add(1);
v.add(2);
v.add(3);
v.add(4);

// Retrieve the last element
int lastElement = v.lastElement();

System.out.println(lastElement);
// Outputs 4
