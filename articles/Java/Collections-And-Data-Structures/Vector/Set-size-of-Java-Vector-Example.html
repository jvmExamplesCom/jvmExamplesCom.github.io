Set Size of Java Vector Example

In this article, we will learn how to set the size of a Java Vector. We will cover several methods that can be used to set the size of a Vector and provide examples of each approach.

What is a Java Vector?

A Java Vector is a resizable array implementation of the List interface. It is similar to an ArrayList, but it has additional synchronization capabilities. Vectors are useful for when multiple threads need to access the same data structure.

Methods for Setting the Size of a Vector

There are several methods that can be used to set the size of a Vector in Java. We will look at the following methods:
  • setSize()
  • clear()
  • trimToSize()

setSize() Method

The setSize() method is used to set the size of a Vector to a specified size. If the Vector is smaller than the new size, null elements are added to fill the gap. If the Vector is larger than the new size, excess elements will be removed. 

          
           
// Create a Vector with 5 elements 
Vector<String> vector = new Vector<String>(5); 
vector.add("First"); 
vector.add("Second"); 
vector.add("Third"); 
vector.add("Fourth"); 
vector.add("Fifth"); 

// Set the size of the Vector to 10 
vector.setSize(10); 

// Print out the Vector 
System.out.println(vector); 
// Output: [First, Second, Third, Fourth, Fifth, null, null, null, null, null]

clear() Method

The clear() method is used to remove all elements from a Vector. This will set the Vector size to 0. 

          
           
// Create a Vector with 5 elements 
Vector<String> vector = new Vector<String>(5); 
vector.add("First"); 
vector.add("Second"); 
vector.add("Third"); 
vector.add("Fourth"); 
vector.add("Fifth"); 

// Clear the Vector 
vector.clear(); 

// Print out the Vector 
System.out.println(vector); 
// Output: []

trimToSize() Method

The trimToSize() method is used to reduce the size of a Vector to its current length. This method is useful for reducing memory consumption when a given Vector is no longer growing or shrinking. 

          
           
// Create a Vector with 5 elements 
Vector<String> vector = new Vector<String>(5); 
vector.add("First"); 
vector.add("Second"); 
vector.add("Third"); 
vector.add("Fourth"); 
vector.add("Fifth"); 

// Reduce the size of the Vector to its current length
vector.trimToSize(); 

// Print out the Vector 
System.out.println(vector); 
// Output: [First, Second, Third, Fourth, Fifth]

Conclusion

In this article, we looked at several methods for setting the size of a Java Vector. We saw how to use the setSize(), clear(), and trimToSize() methods to set the size of a Vector.
