How to Clear or Remove All Elements from the Vector in Java

Java provides various ways to clear a vector, depending on what needs to be done. In this article, we discuss the various ways in which you can clear a vector and provide example code for each one.

Using the clear() Method

The simplest way to clear a vector is by using the clear() method. This method clears all the elements from a vector in one operation. It also sets the size of the vector to 0. 

          
           
Vector<String> vector = new Vector<>();
            vector.add("Apple");
            vector.add("Banana");
            vector.add("Orange");
            
            vector.clear();
            System.out.println("Vector cleared: " + vector);  
            // Output: Vector cleared: []

Using removeAllElements() Method

Another way to clear a vector is by using the removeAllElements() method. This method also clears all elements from a vector in one operation. 

          
           
Vector<String> vector = new Vector<>();
            vector.add("Apple");
            vector.add("Banana");
            vector.add("Orange");
            
            vector.removeAllElements();
            System.out.println("Vector cleared: " + vector);  
            // Output: Vector cleared: []

Using removeRange() Method

The removeRange() method can be used to remove a range of elements from a vector. This method takes two parameters, the starting index and the ending index. All elements in the vector within the specified range will be removed. 

          
           
Vector<String> vector = new Vector<>();
            vector.add("Apple");
            vector.add("Banana");
            vector.add("Orange");

            vector.removeRange(0, 2);
            System.out.println("Vector cleared: " + vector);  
            // Output: Vector cleared: [Orange]

Using removeElementAt() Method

The removeElementAt() method can be used to remove a single element at a specific position from a vector. This method takes one parameter, the index of the element that needs to be removed. 

          
           
Vector<String> vector = new Vector<>();
            vector.add("Apple");
            vector.add("Banana");
            vector.add("Orange");

            vector.removeElementAt(2);
            System.out.println("Vector cleared: " + vector);  
            // Output: Vector cleared: [Apple, Banana]

Using Iterator

The Iterator class can be used to iterate over all elements in a vector, and remove individual elements one by one. 

          
           
Vector<String> vector = new Vector<>();
            vector.add("Apple");
            vector.add("Banana");
            vector.add("Orange");

            Iterator<String> iterator = vector.iterator();
            while (iterator.hasNext()) {
                iterator.next();
                iterator.remove();
            }
            System.out.println("Vector cleared: " + vector); 
            // Output: Vector cleared: []
