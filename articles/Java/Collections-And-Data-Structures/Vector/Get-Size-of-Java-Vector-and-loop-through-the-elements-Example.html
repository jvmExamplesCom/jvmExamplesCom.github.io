Get Size of Java Vector and Loop Through the Elements Example

To get the size of a Java Vector, use the size() method. This will return the number of elements in the Vector. Here is a code example:


          
           
int size = Vector.size();

To loop through each element in the Vector, you can use a for-each loop. Here is an example:


          
           
for (Object element : Vector) {
  // Process element here
}

If you need to iterate through the Vector in reverse order, you can use a regular for loop:


          
           
for (int i = Vector.size() - 1; i >= 0; i--) {
  Object element = Vector.get(i);
  // Process element here
}

Example Output

Let's take a look at a complete example that makes use of the above methods. This program will print out the size of the Vector and then loop through each element and print it out.


          
           
import java.util.Vector;

public class VectorExample {
  public static void main(String[] args) {
    // Create a Vector
    Vector<string> Vector = new Vector<>();
    Vector.add("Hello");
    Vector.add("World");

    // Get the size
    int size = Vector.size();
    System.out.println("Size of Vector: " + size);

    // Loop through the Vector
    for (String element : Vector) {
      System.out.println(element);
    }
  }
}</string>
