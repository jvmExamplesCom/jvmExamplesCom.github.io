Search an element of Java Vector Example

The following example shows how to search an element in a Java Vector. We will use the 

contains()
method to check if the vector contains a specific element or not.

Code Example


          
           
import java.util.Vector; 

public class VectorSearchExample { 
   public static void main(String[] args) 
    { 
        Vector<String> vec = new Vector<String>(); 
  
        vec.addElement("Apple"); 
        vec.addElement("Orange"); 
        vec.addElement("Mango"); 
        vec.addElement("Fig"); 
  
        System.out.println("Does vector contains Mango? "+vec.contains("Mango")); 
        System.out.println("Does vector contains Jack Fruit? "+vec.contains("Jack Fruit")); 
    } 
}

Output


          
           
Does vector contains Mango? true
Does vector contains Jack Fruit? false
