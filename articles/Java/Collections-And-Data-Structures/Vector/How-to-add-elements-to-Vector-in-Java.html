How to add Elements to Vector in Java

Adding elements to a Vector in Java is a quick and easy way to store data for your application. This tutorial will show you how to add elements to your Vector using the Java Collections Framework.

What is a Vector?

A vector is an ordered collection of objects in Java. It allows for manipulation of the objects it contains, such as inserting, deleting, or replacing them. The elements of a vector can be any type of object, including primitive values or object references. The Vector class implements the List interface in the Java Collections Framework. It provides several methods to add elements to the Vector. These methods are used to add new elements at the end of the Vector or to insert elements at specified positions in the Vector.

Adding Elements to Vector Java

To add elements to a Vector, you can use one of the four methods provided by the Vector class: ‘add’, ‘addElement’, ‘addAll’, and ‘insertElementAt’.

Add Method

One way of adding elements to a Vector is through the 'add' method. The add method takes an element as an argument and inserts it at the end of the Vector. 

          
           
Vector<String> myVector = new Vector<>();
myVector.add("Hello");
myVector.add("World");
//myVector now contains "Hello" and "World"

addElement Method

The addElement method is similar to the add method, but it has been deprecated since Java 2. It has the same functionality as the add method and can be used to add elements to the end of the Vector. 

          
           
Vector<Integer> myVector = new Vector<>();
myVector.addElement(1);
myVector.addElement(2);
//myVector now contains 1 and 2

addAll Method

The addAll method allows you to add all the elements from another collection to the Vector. It takes a collection as an argument and inserts all the elements from that collection into the Vector. The elements are added in the same order as they appear in the collection. 

          
           
Vector<Integer> myVector = new Vector<>();
myVector.addAll(Arrays.asList(1, 2, 3, 4, 5));
//myVector now contains 1, 2, 3, 4, and 5

insertElementAt Method

The insertElementAt method allows you to insert an element at a specific position in the Vector. It takes two arguments: the element to be inserted and the index of the position where the element should be inserted. 

          
           
Vector<String> myVector = new Vector<>();
myVector.add("Hello");
myVector.add("World");
myVector.insertElementAt("Foo", 1);
//myVector now contains "Hello", "Foo", and "World"

Conclusion

In this tutorial, we discussed how to add elements to a Vector in Java. We looked at the four methods provided by the Vector class: add, addElement, addAll, and insertElementAt. We also saw examples of each method in action.
