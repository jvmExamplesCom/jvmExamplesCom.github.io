How to Iterate the Vector Elements in the Reverse Order (Backward Direction)

Java is a widely used programming language that is used to create a wide variety of objects and applications. In Java, the 

Vector
class is one of the most commonly used classes when dealing with collections of items. This class allows users to store items in an ordered fashion and provides various methods for accessing and manipulating the elements within a 
Vector
object. One of the more advanced operations that can be performed on a 
Vector
object is iteration in the reverse order. This means that the elements within the 
Vector
will be accessed in a backward direction.

The following examples show how to iterate the elements of a 

Vector
in the reverse order.

Example 1: Iterating a Vector Using a For-Loop

The first example shows how to iterate the elements of a 

Vector
using a traditional 
for
loop. 

          
           
Vector<Integer> vector = new Vector<>();
vector.add(1);
vector.add(2);
vector.add(3);

for (int i = vector.size() - 1; i >= 0; i--) {
    int element = vector.get(i);
    System.out.println(element);
}

// Output:
// 3
// 2 
// 1
In this example, a 
Vector
object named 
vector
is created and is populated with three 
Integer
elements. Next, a 
for
loop is used to traverse the 
Vector
in the reverse order. The loop begins at the last index and works its way down until it reaches the first index. For each iteration of the loop, the element at the current index is retrieved and printed to the console.

Example 2: Iterating a Vector Using an Iterator

The second example shows how to iterate the elements of a 

Vector
using an 
Iterator

Vector<Integer> vector = new Vector<>();
vector.add(1);
vector.add(2);

Vector

vector

Integer

ListIterator

Vector
. This iterator is then used to traverse the 
Vector
in the reverse order by using the 
hasPrevious()
and 
previous()
methods. For each iteration of the loop, the element at the current index is retrieved and printed to the console.

using a lambda expression.
