Remove an Element from Specified Index of Java Vector Example

In this article, we will learn how to remove an element from a specified index in the Java Vector using Java Collection API. We will look at simple code examples to demonstrate how to use the Vector's remove() method.

How to Remove an Element from Vector by Index?

To remove an element from vector based on the index position, we can use the remove() method of Vector class. The remove() method accepts an int argument which specifies the index of the element to be removed from the Vector. The syntax of remove method is given below.


          
           
public void remove(int index)

It returns the specified element that has been removed from the Vector. Let's take a look at the example code to understand how to use the remove() method.

Java Vector Remove Element by Index Example

In this example, we will create a Vector of String elements, add some elements to it and then delete a particular element from the Vector using its index position. The modified Vector will be then printed.


          
           
import java.util.Vector;

public class VectorExample {

   public static void main(String[] args) {
      // Creating Vector of String Elements
      Vector<String> animals = new Vector<String>();

      // Adding Elements to the Vector
      animals.add("Lion");
      animals.add("Tiger");
      animals.add("Cat");
      animals.add("Dog");
      animals.add("Crocodile");

      System.out.println("Vector before removing an element: " + animals);

      // Removing an element from Vector by index
      String removedEle = animals.remove(2);
 
      System.out.println("Element removed from vector: " + removedEle);
      System.out.println("Vector after removing an element: " + animals);
   }
}

If we run the above code, it gives the following output.


          
           
Vector before removing an element: [Lion, Tiger, Cat, Dog, Crocodile]
Element removed from vector: Cat
Vector after removing an element: [Lion, Tiger, Dog, Crocodile]

We can see that the element at index 2 (Cat) has been successfully removed from the Vector and now our Vector contains four elements.
