Replace an Element at Specified Index of Java Vector Example

In this article we will demonstrate how to replace an element at a specified index in a Java Vector. We will provide code examples so that you can better understand how to accomplish this.

Replacing the Element at a Specified Index

The following code snippet demonstrates how to replace the element at a specified index in a Java Vector:


          
           
Vector <Integer> vector = new Vector<>();
vector.add(0, 10);
vector.add(1, 20);
vector.add(2, 30);

vector.set(1, 40); // Replacing the element at index 1 with 40

In this example we create a Vector containing three elements (10, 20, and 30) and then we replace the element at index 1 with the value 40.

Verifying the Replacement

The following code snippet demonstrates how to verify that the element at index 1 was successfully replaced:


          
           
int elementAtIndex1 = vector.get(1);
System.out.println("The element at index 1 is " + elementAtIndex1);
This code prints out the following result: 

          
           
The element at index 1 is 40
This confirms that the element at index 1 was successfully replaced.
