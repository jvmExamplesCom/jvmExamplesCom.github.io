Append All Elements of Other Collection to Java Vector Example

Java Vector class is a part of the Java Collections Framework. It provides various methods that can be used to manipulate its elements such as add(), remove() and append(). The append() method appends the elements of one collection to another. In this article, we will discuss the append() method in detail and provide some code examples.

Java Vector append() Syntax

The syntax for the append() method is as follows: 

          
           
public void append(Collection c)
It takes a Collection object as parameter and appends all the elements from it to the Vector.

Java Vector append() Example

Let’s look at an example: 

          
           
import java.util.ArrayList;
import java.util.Vector;

public class VectorAppendExample {

    public static void main(String[] args) {
        // Create a Vector instance
        Vector<string> vector = new Vector<>();
        
        // Add some elements
        vector.add("Java");
        vector.add("Python");
        vector.add("C++");
        
        // Display the elements
        System.out.println("Vector Before Appending:");
        System.out.println(vector);

        // Create an ArrayList instance
        ArrayList<string> arrayList = new ArrayList<>();
        
        // Add some elements
        arrayList.add("R");
        arrayList.add("Ruby");
        arrayList.add("Scala");
        
        // Append the elements of arrayList to vector
        vector.append(arrayList);
        
        // Display the elements
        System.out.println("Vector After Appending:");
        System.out.println(vector);
    }
}
</string></string>
When we run the above program, we get the following output: 

          
           
Vector Before Appending:
[Java, Python, C++]
Vector After Appending:
[Java, Python, C++, R, Ruby, Scala]
In the above example, we created two collections to demonstrate the working of the append() method. We first created a Vector and added three strings to it. We then created an ArrayList and added three strings to it. Finally, we used the append() method to append all the elements from the ArrayList to the Vector.
