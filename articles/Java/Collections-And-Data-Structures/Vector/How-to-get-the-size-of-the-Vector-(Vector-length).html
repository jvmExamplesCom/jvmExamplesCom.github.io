Getting the Size of a Vector in Java

The size of a Vector in Java is determined by the number of elements stored inside it. It is possible to get the size of a Vector using its 

          
           
size()
method, as shown in the following example: 

          
           
// Create a new Vector and add some elements
Vector<Integer> vec = new Vector<Integer>();
vec.add(1);
vec.add(2);
vec.add(3);

// Get the size of the Vector
int size = vec.size(); 
System.out.println("Vector size: " + size);
// Outputs: Vector size: 3
As seen in the example, the 

          
           
size()
method returns the number of elements stored in the vector as an 

          
           
int
.

isEmpty()

if (vec.size() == 0) {
 // Vector is empty
}

 // Vector is empty

Iterating through Vector Elements

Iterating through the elements of a Vector can be done with either a for loop or an enhanced for loop, as seen in the following examples: 

          
           
// For loop
for (int i = 0; i < vec.size(); i++) {
 System.out.println(vec.get(i));
}

// Enhanced for loop
for (int element : vec) {
 System.out.println(element);
}
The enhanced for loop is recommended over the regular for loop when iterating through Vector elements, as it is more concise.
