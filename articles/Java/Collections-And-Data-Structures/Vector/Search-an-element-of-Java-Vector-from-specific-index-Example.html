Searching an Element from a Specific Index of Java Vector Example

Java Vector is an important data structure in the Java library, used for storing elements. It works similar to an array, but it has some additional features compared to arrays. In this article, we will discuss how you can search for an element from a specific index in a Java Vector.

Searching an Element from a Specific Index Example

The following code example shows how you can search for an element from a specific index in a Java Vector. 

          
           
Vector v = new Vector();
v.add("A");
v.add("B");
v.add("C");
v.add("D");

String element = (String) v.get(2); //element at index 2 is C
System.out.println("Element: "+element);
The output of the above program will be: 

          
           
Element: C
In the above code example, we first create an instance of the Vector class, and then add some elements using the add() method. We then use the get() method with the index as argument to get the element at that index.

Searching an Element from a Specific Index Using For Each Loop

You can also use a for each loop to search for an element from a specific index in a Java Vector. Here is an example: 

          
           
Vector v = new Vector();
v.add("A");
v.add("B");
v.add("C");
v.add("D");

int index = 2;
for (Object elem : v) {
    if (v.indexOf(elem) == index) {
        System.out.println("Element: "+elem);
    }
}
The output of the above program will be: 

          
           
Element: C
In the above code example, we first create an instance of the Vector class, and then add some elements using the add() method. We then use the for each loop to iterate over all elements of the Vector. Inside the loop, we use the indexOf() method to find the index of the element. If the index matches the index we are searching for, we print out the element.

Conclusion

In this article, we discussed how you can search for an element from a specific index in a Java Vector. We looked at two different approaches using the get() method and the for each loop.
