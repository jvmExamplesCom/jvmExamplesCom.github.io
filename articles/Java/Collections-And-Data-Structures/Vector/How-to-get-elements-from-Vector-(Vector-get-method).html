How to Get Elements from Vector (Vector Get Method)

The Vector class in Java allows us to store and manipulate elements within a collection. This article will explain how to use the Vector get() method to retrieve elements from a Vector.

Syntax

The syntax of the Vector get() method is as follows: 


          
           
public Object get(int index)

The first parameter is the index number of the element in the vector that you wish to retrieve.

Examples

Let’s take a look at a few examples of using the Vector get() method.

Example 1

First, let’s create a Vector and add some elements to it: 


          
           
Vector<String> myVector = new Vector<String>();
myVector.add("Apple");
myVector.add("Orange");
myVector.add("Banana");
myVector.add("Kiwi");

Now, let’s retrieve the element at index 2 using the Vector get() method: 


          
           
Object myElement = myVector.get(2);
System.out.println(myElement);
The output of this code will be: 

          
           
Banana

Example 2

Let’s take a look at another example of using the Vector get() method. In this example, we will create a Vector of Integer objects and retrieve an element from it: 


          
           
Vector<Integer> myVector = new Vector<Integer>();
myVector.add(1);
myVector.add(2);
myVector.add(3);
myVector.add(4);

Now, let’s retrieve the element at index 3 using the Vector get() method: 


          
           
Object myElement = myVector.get(3);
System.out.println(myElement);
The output of this code will be: 

          
           
4
