Get SubList of Java Vector Example

In this article, we will learn how to get the sublist from a Java Vector using the subList() method. We are going to look at an example and discuss it in detail.

Example

In this example, we will get the subList of a Vector. We will first create a Vector, then add some elements to it. Then we will use the subList() method to get the sublist of the Vector.


          
           
Vector<Integer> vector = new Vector<Integer>();
vector.add(1);
vector.add(2);
vector.add(3);
vector.add(4);
vector.add(5);

// Get the subList
List<Integer> subList = vector.subList(1, 4);
System.out.println(subList);
The output of the above code will be: 

          
           
[2, 3, 4]

Explanation

In this example, we first create a 

          
           
Vector<Integer>
and add 5 elements to it. Then we use the 
subList()
method to get the subList from the Vector. The 
subList()
method takes two parameters, the start index and the end index. The start index is inclusive, and the end index is exclusive. So, in our example, 
subList(1, 4)
returns the sublist that starts at index 1 and ends at index 3 (4-1).

The subList returned is a List, so we can iterate over it or perform other operations on it.

Conclusion

subList()
method. We also looked at an example to understand how it works. Understanding the subList() method can help you write more efficient code.
