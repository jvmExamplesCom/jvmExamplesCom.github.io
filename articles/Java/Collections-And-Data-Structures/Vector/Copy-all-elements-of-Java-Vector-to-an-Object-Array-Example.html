Copying Vector Elements to an Object Array Example in Java

Java Vector class is used to create a dynamic array of objects. It is like an ArrayList, but with some additional methods. In this article, we will look at an example of how to copy all elements of a Java Vector to an object array.

Example of Copying All Vector Elements to an Object Array

In this example, we will create a Java Vector and then copy all elements of the vector to an object array.


          
           
// Create a new Vector 
Vector<Object> myVector = new Vector<Object>();

// Add some elements 
myVector.add("Element 1");
myVector.add("Element 2");
myVector.add("Element 3");

// Get the size of the Vector 
int size = myVector.size();

// Create the Object array 
Object[] myArray = new Object[size];

// Copy all elements of the Vector to the array
myVector.copyInto(myArray);

In the above example, we first created a new Vector. Then, we added three elements to it. After that, we got the size of the Vector. We used this size to create an Object array. Finally, we used the copyInto() method to copy all elements of the Vector to the array.

Output

When we run the above code, we get the following output:


          
           
Element 1
Element 2
Element 3
