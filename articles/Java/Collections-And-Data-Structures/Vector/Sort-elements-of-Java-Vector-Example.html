Sorting Elements of a Java Vector Example

Java provides many tools for sorting elements in a Vector. This article explains how to sort Vector elements by using the 

Collections.sort
method.

Creating a Simple Vector

We start by creating a Vector and adding some elements to it: 

          
           
Vector<Integer> numbers = new Vector<>();
numbers.add(3);
numbers.add(11);
numbers.add(2);
numbers.add(7);
numbers.add(8);

Sorting the Vector

Once we have our Vector, we then use the 

          
           
Collections.sort
method to sort the elements: 

          
           
Collections.sort(numbers);

Printing the Sorted Vector

Finally, we can print out the sorted Vector using a 

          
           
for
loop: 

          
           
for (Integer num : numbers) {
    System.out.println(num);
}

Output


          
           
2
3
7
8
11
