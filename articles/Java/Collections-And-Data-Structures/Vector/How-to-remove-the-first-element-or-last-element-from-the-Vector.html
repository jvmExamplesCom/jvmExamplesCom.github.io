Removing the First or Last Element from the Vector in Java

Java provides an easy way to remove the first or last element from a Vector type collection. In this tutorial, we will discuss and demonstrate how to do so.

Using The Vector remove() Method

The simplest way of removing the first or last element from a Vector is by making use of the 

          
           
remove()
method. This method takes a single argument which is the index of the element that the user wishes to remove. Depending on the index specified, it will remove the first or last element of the Vector. The syntax for this method is as follows: 

          
           
public Object remove(int index)
If the index specified is within the valid range of the vector (i.e 0 ≤ index ≤ vector.size()-1), this method will remove the element from the vector, and return the object previously located there. If the index is not within the valid range, then this method will throw an ArrayIndexOutOfBoundsException. Here is an example demonstrating how to use the 

          
           
remove()
method to remove the last element from a Vector. 

          
           
//Create a Vector of type Integer
Vector<Integer> myVector = new Vector<Integer>();
//Add some elements to the Vector
myVector.add(1); 
myVector.add(2); 
myVector.add(3); 

//Remove the last element from the Vector
myVector.remove(myVector.size()-1); //Removes 3
In the above example, we create a Vector of type Integer, and add 3 elements to it (1, 2 and 3). We then use the 

          
           
remove()
method, passing in the size of the vector - 1 as the index. This removes the last element from the Vector.

Using The Vector pop() Method

Another way of removing the last element from a Vector is by making use of the 

          
           
pop()
method. This method is used when the user wishes to remove the last element from the Vector, and also get its value. The syntax for this method is as follows: 

          
           
public Object pop()
This method returns the object that is present at the end of the Vector. At the same time, it also removes the last element from the Vector. If the Vector is empty, then this method returns null. Here is an example demonstrating how to use the 

          
           
pop()
method to remove the last element from a Vector and get its value. 

          
           
//Create a Vector of type Integer
  Vector <Integer> myVector = new Vector <Integer>();
  //Add some elements to the Vector
  myVector.add(1); 
  myVector.add(2); 
  myVector.add(3); 

  //Remove and get the last element from the Vector
  int lastElement = myVector.pop(); //Returns 3
In the above example, we create a Vector of type Integer, and add 3 elements to it (1, 2 and 3). We then use the 

          
           
pop()
method to remove the last element from the Vector and get its value. The last element from the Vector is 3, so this method returns 3.

Conclusion

In this tutorial, we discussed and demonstrated how to remove the first or last element from a Vector in Java. We showed how to do this using the 

          
           
remove()
and 

          
           
pop()
methods. With these methods, removing the first and last elements from a Vector in Java is quite simple.
