Removing All Elements From Java Vector Example

The Java Vector class is an implementation of a dynamic array. It is a resizable collection that implements the List interface. In this article, we'll be discussing how to remove all elements from a Vector using two different approaches.

Using clear() Method

The easiest method to remove all elements from a Vector is by using the clear() method. This method removes all elements from the Vector and sets the size equal to 0. The syntax for the clear() method is as follows:


          
           
vectorName.clear();
Let's see a simple example of using this method. 

          
           
import java.util.Vector;

public class VectorExample {

    public static void main(String[] args) {

        // Create a new Vector of Strings.
        Vector<String> vec = new Vector<String>();

        // Add some elements
        vec.add("Hello");
        vec.add("World");

        System.out.println("Vector before clearing: " + vec);

        // Remove all the elements
        vec.clear();

        System.out.println("Vector after clearing: " + vec);
    }
}

When we run this code, we should see the following output:


          
           
Vector before clearing: [Hello, World]
Vector after clearing: []

As we can see, all elements have been successfully removed from the Vector.

Using removeAllElements() Method

Another approach to remove all elements from a Vector is by using the removeAllElements() method. This method removes all existing elements from the Vector and sets its size to zero. The syntax for the removeAllElements() method is as follows:


          
           
vectorName.removeAllElements();
Let's see a simple example of using this method. 

          
           
import java.util.Vector;

public class VectorExample {

    public static void main(String[] args) {

        // Create a new Vector of Strings.
        Vector<String> vec = new Vector<String>();

        // Add some elements
        vec.add("Hello");
        vec.add("World");

        System.out.println("Vector before removing all elements: " + vec);

        // Remove all the elements
        vec.removeAllElements();

        System.out.println("Vector after removing all elements: " + vec);
    }
}

When we run this code, we should see the following output:


          
           
Vector before removing all elements: [Hello, World]
Vector after removing all elements: []

As we can see, all elements have been successfully removed from the Vector.

In this article, we discussed two different approaches to remove all elements from a Vector in Java. In the first approach, we used the clear() method and in the second approach, we used the removeAllElements() method. Both of these methods are easy to use and achieve the same result.
