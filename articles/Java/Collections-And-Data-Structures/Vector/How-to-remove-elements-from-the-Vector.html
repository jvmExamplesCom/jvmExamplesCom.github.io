Removing Elements from a Vector in Java

A Vector is an ordered collection of objects, and it is available in the java.util package. This tutorial will demonstrate how to remove elements from a Vector in Java.

Using the remove() Method

The 
remove()
method is used to remove an element from a Vector. It accepts an argument, which should be the index of the element that is to be removed, as an integer. The following code example shows the use of the 
remove()
method to remove the element at index 3 from the Vector: 

          
           
Vector v = new Vector();
v.add(1);
v.add(2);
v.add(3);
v.add(4);
v.remove(3);
The output of the above code snippet would be: 

          
           
[1, 2, 3]

Using the removeElement() Method

Another method for removing an element from a Vector is the 

removeElement()
method. This method removes the first occurrence of the specified element from the Vector. It accepts an argument, which should be the element that needs to be removed. The following code example shows the use of the 
removeElement()
method to remove the element with value 2 from the Vector: 

          
           
Vector v = new Vector();
v.add(1);
v.add(2);
v.add(3);
v.add(4);
v.removeElement(2);
The output of the above code snippet would be: 

          
           
[1, 3, 4]

Using the clear() Method

The 
clear()
method can be used to remove all the elements from a Vector. It does not require any arguments and can be used as follows: 

          
           
Vector v = new Vector();
v.add(1);
v.add(2);
v.add(3);
v.add(4);
v.clear();
The output of the above code snippet would be: 

          
           
[]

Conclusion

In this tutorial, we learned how to remove elements from a Vector in Java using the 

remove()
, 
removeElement()
and 
clear()
methods.
