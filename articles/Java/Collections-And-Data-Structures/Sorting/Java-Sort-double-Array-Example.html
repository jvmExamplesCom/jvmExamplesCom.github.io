Java Sort Double Array Example

Here is an example of how to sort a double array in java. This example uses the Arrays class to sort a double array. 

          
           
import java.util.Arrays;

double[] myArray = {2.0, 3.5, 1.2, 4.8, 7.9};
Arrays.sort(myArray);

for (double element : myArray) {
	System.out.println(element);
}
The output of the code would be: 

          
           
1.2
2.0
3.5
4.8
7.9
The example above sorts the elements of an array into ascending order. But what if we wanted to sort them in descending order?

Sorting a Double Array in Descendin Order

To sort a double array using the Arrays class, in descending order, you can use the following code: 

          
           
import java.util.Arrays;

double[] myArray = {2.0, 3.5, 1.2, 4.8, 7.9};
Arrays.sort(myArray, Collections.reverseOrder());

for (double element : myArray) {
	System.out.println(element);
}
This will output the double array in descending order like this: 

          
           
7.9
4.8
3.5
2.0
1.2

Sorting a Part of a Double Array

The Arrays class also provides a way to sort part of a double array. To do this, you can use the method Arrays.sort(double[],int,int), where the first argument is the double array, the second argument is the index of the first element to be sorted, and the third argument is the index of the last element to be sorted. For example, to sort only the third and fourth element of the array, we can use this code: 

          
           
import java.util.Arrays;

double[] myArray = {2.0, 3.5, 1.2, 4.8, 7.9};

This will output the double array like this:

As you can see, only two of the elements have been sorted - the third and fourth elements.

Sorting a Double Array Without Using the Arrays Class

It is possible to sort a double array without using the Arrays class. The easiest way to do this is to use a bubble sort algorithm. A bubble sort algorithm works by comparing each value in the array with the value next to it, swapping them if they are in the wrong order. For example, here is a bubble sort algorithm for a double array: 

          
           
public static void bubbleSortDoubleArray(double[] arr){
    for (int i = 0; i < arr.length - 1; i++)
        for (int j = 0; j < arr.length - i - 1; j++)
            if (arr[j] > arr[j + 1]){
                double temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
            }
}
To use the bubble sort algorithm, you can call the method like this: 

          
           
bubbleSortDoubleArray(myArray);
where myArray is the double array to be sorted. The output of the code would be a double array that is sorted in ascending order.
