Java Sort Short Array Example

In this article we'll go through a simple example on how to sort an array of short integers in Java. We'll examine the code as well as the output created from the code. First, we'll begin by declaring an array of short integers, and assigning values to each of the elements. 

          
           
short[] array = new short[] {2, 3, 1};
Next, we'll use the Arrays.sort() method to sort the array. This method accepts the array as a parameter and returns an sorted array. 

          
           
Arrays.sort(array);
We can now print out the elements of the sorted array to verify that they are indeed sorted. 

          
           
for (short s : array)
    System.out.print(s + " ");
When we run this code, we get the following output: 

          
           
1 2 3
As you can see, the array is now in sorted order. Now let's take a look at how we can sort an array of doubles. We'll use the same Arrays.sort() method, but instead of a short array, we'll pass in a double array as a parameter. 

          
           
double[] doubleArray = new double[] {5.5, 6.5, 4.5};
Arrays.sort(doubleArray);
We can then print the elements of the sorted array to verify that they have been properly sorted. 

          
           
for (double d : doubleArray)
    System.out.print(d + " ");
When we run this code, we get the following output: 

          
           
4.5 5.5 6.5
As you can see, the array is now in sorted order. Finally, let's take a look at how we can sort a String array. We'll again use the Arrays.sort() method, but instead of a short or double array, we'll pass in a String array as a parameter. 

          
           
String[] stringArray = new String[] {"apple", "banana", "pear"};
Arrays.sort(stringArray);
We can then print the elements of the sorted array to verify that they have been properly sorted. 

          
           
for (String s : stringArray)
    System.out.print(s + " ");
When we run this code, we get the following output: 

          
           
apple banana pear
As you can see, the array is now in sorted order. In this article, we went through an example of how to sort an array of short integers, doubles, and strings in Java. We used the Arrays.sort() method, which accepts an array as a parameter and returns an array sorted in ascending order. We also verified the sorting by printing out the elements of the sorted array.
