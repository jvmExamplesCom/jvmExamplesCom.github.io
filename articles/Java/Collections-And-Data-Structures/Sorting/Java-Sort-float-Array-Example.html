Java Sort float Array Example

The following example shows how to sort float array items using the sort() method of the Arrays class. 

          
           
// Create an float array
float[] numbers = {2.3f, 4.5f, 3.2f, 1.7f};

// Sort the array
Arrays.sort(numbers);
Once the sort() method is called on the numbers array, it will be sorted in ascending order, i.e. the smallest value will appear first and the largest value will appear last. 

          
           
// Output the array
for (float number : numbers) {
  System.out.println(number);
}
The above code will output the following: 

          
           
1.7f
2.3f
3.2f
4.5f
If we want to sort the array in descending order, we can use the sort() method of the Collections class. We can use the reverseOrder() comparator to get the desired result. 

          
           
// Create a Comparator
Comparator<Float> comparator = Collections.reverseOrder();

// Sort the array
Arrays.sort(numbers, comparator);
After calling the above code, the array will now be sorted in descending order. The output of the following code should be: 

          
           
4.5f
3.2f
2.3f
1.7f
We can also pass a custom comparator to the sort() method. For example, if we want to sort the array with the even numbers at the beginning and the odd numbers at the end, we can define a custom comparator like this: 

          
           
// Create a Comparator
Comparator<Float> comparator = new Comparator<Float>(){
    @Override
    public int compare(Float f1, Float f2){
        if (f1 % 2 == 0 &amp;&amp; f2 % 2 != 0) 
            return -1;
        else if (f1 % 2 != 0 &amp;&amp; f2 % 2 == 0) 
            return 1;
        else 
            return f1.compareTo(f2);
    }
};

// Sort the array
Arrays.sort(numbers, comparator);
In this case, the above code should output: 

          
           
2.3f
4.5f
1.7f
3.2f
So, as we have seen, the sort() method of the Arrays class can be used to sort float array items in Java. We can also sort the array in reverse order or according to a custom comparator.
