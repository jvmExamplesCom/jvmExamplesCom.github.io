Java - Sorting a Char Array Example

In this tutorial, we are going to explore how to sort a char array in Java. We will look at several examples with different approaches and show the output of each program. Let's start by taking a look at a simple example. In this example, we will sort the characters in an array alphabetically. To do this, we will use the 

          
           
Arrays.sort()
method from the Java util package: 

          
           
//Example 1 
char[] letters = {'a', 'z', 'e', 'd'};
Arrays.sort(letters); 

for (char c : letters) {
    System.out.println(c); 
}
In the code above, we first created an array of characters called 

          
           
letters
and populated it with four letters. Then, we used the 

          
           
Arrays.sort()
method to sort the array. Finally, we printed out the sorted array using a for-each loop. When we run this program, we get the following output: 

          
           
a
d
e
z
As you can see, the characters have been sorted alphabetically. Now let's take a look at another example. In this example, we will sort the characters in descending order. To do this, we need to provide a custom Comparator object that can compare two characters and determine which one should come first: 

          
           
//Example 2
char[] letters = {'a', 'z', 'e', 'd'};
Arrays.sort(letters, (o1, o2) -> Character.compare(o2, o1));

for (char c : letters) {
   System.out.println(c); 
}
In the code above, we used a lambda expression to create a Comparator object which sorted the characters in descending order. This Comparator reversed the result of the traditional 

          
           
Character.compare()
method which returned the smaller character first. When we run this program, we get the following output: 

          
           
z
e
d
a
As you can see, the characters have been sorted in descending order. Finally, let's take a look at one more example. In this example, we will sort the characters in ascending order but ignore the case of the characters. To do this, we will use the 

          
           
String.CASE_INSENSITIVE_ORDER
Comparator object: 

          
           
//Example 3 
char[] letters = {'A', 'Z', 'e', 'D'};
Arrays.sort(letters, String.CASE_INSENSITIVE_ORDER);

for (char c : letters) {
    System.out.println(c); 
}
In the code above, we used the 

          
           
String.CASE_INSENSITIVE_ORDER
Comparator object to sort the characters in ascending order while ignoring their cases. When we run this program, we get the following output: 

          
           
A
D
e
Z
As you can see, the characters have been sorted in ascending order while ignoring their case. In this tutorial, we have explored how to sort a char array in Java. We looked at several examples that used different approaches, including using the 

          
           
Arrays.sort()
method and custom Comparator objects.
