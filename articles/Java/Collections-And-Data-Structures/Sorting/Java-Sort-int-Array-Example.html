Java Sort Int Array Example

Sorting an array of integers can be done quickly and efficiently in Java. In this article we will look at a number of different methods for sorting int arrays, including the bubble sort, insertion sort, selection sort, merge sort, and quick sort algorithms. Bubble Sort

The bubble sort is one of the most classic sorting algorithms. It works by comparing adjacent elements of an array and swapping them if they are out of order. The following example shows the basic structure of the bubble sort algorithm:


          
           
for (int i = 0; i < array.length; i++) {
    for (int j = 0; j < array.length - 1; j++) {
        if (array[j] > array[j + 1]) {
            // swap array[j] and array[j+1]
        }
    }
}

The algorithm can be improved by keeping track of whether or not any swaps were made on each pass. If no swaps were made on a pass then the array must already be sorted and the algorithm can be terminated early. This modified version of the algorithm is known as the optimized bubble sort:

do {
    swapped = false;
    for (int i = 0; i < array.length; i++) {
        if (array[i] > array[i + 1]) {
            // swap array[i] and array[i+1]
            swapped = true;
        }
    }
} while (swapped);

Insertion Sort

The insertion sort is another classic sorting algorithm. It works by starting from the leftmost element of the array and "inserting" it into its correct place within the array. The following example shows the basic structure of the insertion sort algorithm:


          
           
for (int i = 0; i < array.length; i++) {
    int currentValue = array[i];
    int j = i - 1;
    while (j >= 0 &amp;&amp; array[j] > currentValue) {
        array[j + 1] = array[j];
        j--;
    }
    array[j + 1] = currentValue;
}

Selection Sort

The selection sort is similar to the insertion sort, but instead of inserting each element into its correct place, it finds the smallest element in the array and swaps it with the first element. The following example shows the basic structure of the selection sort algorithm:


          
           
for (int i = 0; i < array.length; i++) {
    int minIndex = i;
    for (int j = i + 1; j < array.length; j++) {
        if (array[j] < array[minIndex]) {
            minIndex = j;
        }
    }
    // swap array[i] and array[minIndex]
}

Merge Sort

The merge sort is a recursive algorithm that works by dividing the array into two halves, sorting each half separately, and then merging the two halves back together in sorted order. The following example shows the basic structure of the merge sort algorithm:


          
           
public void mergeSort(int[] array) {
    if (array.length > 1) {
        int[] left  = Arrays.copyOfRange(array, 0, array.length/2);

        mergeSort(left);
        mergeSort(right);
        merge(array, left, right);
    }
