Java Sort Long Array Example

In this example we'll use the Java programming language to illustrate how to sort an array of numbers which is longer than 30 characters. We'll first begin with a basic example, and then move on to more complex code examples.

Basic Example

Let's start by considering an array of integers that contains 5 elements: 

          
           
int[] myArray = {3, 5, 7, 1, 4};
We can sort the array using the inbuilt sort function available in Java: 

          
           
Arrays.sort(myArray);
This will result in the array being sorted in ascending order as follows: 

          
           
{1, 3, 4, 5, 7}

More Complex Code Examples

int[] myArray = {44, 77, 11, 23, 63, 48, 17, 22, 9, 39, 40, 61, 92, 65, 79, 5, 50, 81, 93, 27};
We can use the same sort function as before to sort the array in ascending order: 

          
           
Arrays.sort(myArray);

{5, 9, 11, 17, 22, 23, 27, 39, 40, 44, 48, 50, 61, 63, 65, 77, 79, 81, 92, 93}
